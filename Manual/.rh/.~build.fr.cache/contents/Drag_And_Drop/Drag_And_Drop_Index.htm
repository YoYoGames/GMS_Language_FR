Index Drag And Drop™ Drag and Drop™ (également appelé DnD™), est un langage de script visuel basé sur le GameMaker Language ( GML ). Il consiste à utiliser des blocs d'action pour construire la logique du jeu de manière visuelle et intuitive, et est un outil idéal pour les personnes qui apprennent la programmation, ou pour les personnes qui ont une inclination plus visuelle.
