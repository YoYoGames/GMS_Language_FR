Nouveau Cette action n'est utilisée qu'à l'adresse lors de la création de structures à l'aide d'une fonction qui a été marquée comme une fonction constructrice (voir Déclarer une nouvelle fonction pour plus d'informations). Cette action prend la méthode de fonction pour une fonction précédemment définie, ainsi qu'un certain nombre d'arguments, que vous pouvez développer pour en ajouter d'autres si nécessaire en utilisant l'icône sur la gauche. Ces arguments doivent correspondre aux entrées requises par la fonction, et seront utilisés pour remplir la structure qui est créée. La structure sera retournée à la variable cible, qui peut être marquée comme une variable locale temporaire ou non. temporaire ou non. Syntaxe de l'action : Arguments : Arguments Description Script Le nom du script ou de la fonction définie par l'utilisateur à appeler. Argument0 ... Argument3 Les différents arguments (valeurs) qui doivent être transmis au script ou à la fonction (les arguments non utilisés peuvent être laissés vides). Cible Le nom de la variable qui doit être ciblée pour toutes les valeurs retournées (peut être laissé vide). Exemple étendu : Le code du bloc d'action ci-dessus serait probablement placé dans l'événement Create d'une instance, et il déclare une nouvelle fonction constructrice init_char_struct avec quatre arguments : _str1, _str2, _val1 et _val2. Ces arguments sont ensuite utilisés pour remplir les variables données dans la structure que la fonction crée. Vous appellerez ensuite la fonction à l'aide de l'action New dans tout événement ultérieur, comme ceci : La fonction est appelée et elle renverra une nouvelle struct au tableau "char", et la struct sera remplie avec les variables "name", "location", "hp" et "mana", définies avec les valeurs utilisées pour les arguments correspondants dans la fonction.
