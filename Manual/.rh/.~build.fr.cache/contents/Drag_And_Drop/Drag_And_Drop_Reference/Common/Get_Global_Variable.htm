La variable à laquelle vous assignez la valeur globale peut être une variable précédemment créée, ou vous pouvez cocher la case "Temp" et donner un nouveau nom de variable. et donner un nouveau nom de variable pour que l'action crée une nouvelle variable temporaire (locale) spécifiquement pour contenir la valeur retournée.
