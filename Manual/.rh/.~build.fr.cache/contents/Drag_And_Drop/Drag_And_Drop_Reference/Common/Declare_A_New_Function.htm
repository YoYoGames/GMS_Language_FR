Déclarer une nouvelle fonction Cette action peut être utilisée pour créer une nouvelle fonction définie par l'utilisateur, qui peut ensuite être appelée à tout moment ultérieurement dans votre code. Une fonction définie par l'utilisateur est celle que vous définissez pour effectuer une tâche qui n'est pas couverte par les DnD™ ou les GML, et qui peut avoir une portée globale, d'instance ou locale (voir ici pour plus d'informations sur la portée) et ils peuvent également être utilisés comme constructeurs pour créer des structures. Pour en savoir plus sur l'utilisation de cette action pour créer des fonctions globales, veuillez consulter la section sur la création de fonctions d'action. Vous pouvez également marquer la fonction comme étant statique, ce qui signifie qu'elle ne sera définie qu'une seule fois lors de sa première utilisation. Pour utiliser cette action, vous pouvez l'ajouter à un événement d'instance, puis donner un nom à la fonction que vous souhaitez déclarer. Vous fournissez ensuite les noms d'argument requis pour chacun des paramètres que la fonction prendra. Si la fonction doit être utilisée en tant que constructeur (c'est-à-dire qu'elle va créer une structure), vous devez vérifier le drapeau du constructeur, puis appeler la fonction dont vous avez besoin pour utiliser l'action Nouveau. Si la fonction ne doit être utilisée que localement dans le même événement et nulle part ailleurs, vous pouvez également sélectionner l'indicateur Temp.
