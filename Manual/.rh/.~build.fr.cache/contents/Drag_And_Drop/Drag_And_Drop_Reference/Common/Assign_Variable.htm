Attribuer une variable Cette action permet soit d'affecter une valeur à une variable (en la créant dans le processus), soit de modifier la valeur d'une variable existante. Vous donnez le nom de la variable puis la valeur, qui peut être ajoutée par rapport à la valeur d'origine si nécessaire. Si vous donnez un nouveau nom de variable (c'est-à-dire: une variable qui n'a jamais été assignée auparavant dans l'instance) alors cela créera la nouvelle variable et lui assignera la valeur que vous donnez, tandis que si vous donnez le nom d'une variable existante, elle sera remplacez simplement la valeur par la nouvelle. Notez que le fait de cocher Est relatif ajoutera la valeur à la valeur de la variable existante (utilisez un nombre négatif pour soustraire) et vous pouvez également utiliser l' expression s et les variables créées précédemment comme valeur. Il est à noter que lorsque nous parlons de «valeurs», nous ne parlons pas uniquement de valeurs numériques, car une variable peut être une chaîne, un pointeur, un ID de ressource ou tout autre élément qu'une fonction peut renvoyer ou utiliser. Dans le cas d'une chaîne, la vérification de Relative concaténera la valeur avec la variable précédemment contenue, donc si votre variable était "Hello" et que vous lui attribuez "World" en utilisant le contrôle relatif, la valeur finale de la variable sera "Hello World ". Il faut cependant veiller à ce que les valeurs des variables soient toutes les deux du même type de données, car essayer d'ajouter, par exemple, un entier et une chaîne, entraînera une erreur du compilateur. Notez que vous pouvez ajouter des variables supplémentaires dans la même action en cliquant sur l'icône plus à côté de l'action, et en donnant un autre nom et une autre valeur de variable. Pour des informations plus avancées sur les variables, veuillez consulter la section Variables et portée des variables. Syntaxe de l'action: Arguments: Argument Description Name Le nom de la nouvelle variable à affecter ou le nom d'une variable existante à modifier. Value La valeur que la variable doit contenir. Relative Cochez pour définir la valeur par rapport à la précédente.
