Appel de fonction Cette action peut être utilisée pour appeler toute fonctionGML ou toute fonction définie par l'utilisateur. Vous indiquez la fonction à appeler, puis vous fournissez tous les supplémentaires dont elle a besoin, c'est-à-dire toutes les valeurs (chaînes de caractères, entiers, indices de ressources, etc...) que vous souhaitez entrer dans la fonction pour qu'elle puisse les utiliser. Vous pouvez ajouter des entrées d'arguments supplémentaires en cliquant sur l'icône plus à côté de Vous pouvez ajouter des arguments supplémentaires en cliquant sur l'icône plus à côté de l'action, et vous devez vous assurer que votre nombre d'arguments correspond à celui de la fonction utilisée (c'est-à-dire que si la fonction requiert 3 arguments, vous devez fournir 3 arguments). Certaines fonctions peuvent également renvoyer une valeur à utiliser ultérieurement dans la liste des blocs d'action. et vous pouvez donc également spécifier la variable cible qui doit stocker cette valeur de retour, qui peut avoir été créée précédemment ou être une nouvelle variable locale temporaire (si vous cochez la case " Temp "). Les champs cible et l'argument peuvent être laissés vides s'ils ne sont pas nécessaires. Syntaxe de l'action : Arguments : Arguments Description Fonction La fonction GML à appeler. Argument0 ... ArgumentX Les différents arguments (valeurs) qui doivent être transmis à la fonction. Cible Le nom de la variable qui doit être ciblée pour toutes les valeurs retournées (peut être laissé vide).
