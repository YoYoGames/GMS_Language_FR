Macro Cette action vous permet de définir une ou plusieurs macros. Les macros sont simplement des valeurs constantes uniques - ou des expressions qui s'évaluent à une valeur constante unique - qui peuvent ensuite être utilisées de la même manière qu'une constante régulière. Vous pouvez également utiliser un appel de fonction dans cette action pour générer une valeur de retour unique, auquel cas la macro évaluera la fonction à chaque fois qu'elle est appelée. Notez que les macros sont global dans la portée, et une fois déclarés, ils peuvent être utilisés par n'importe quelle instance, script ou bloc d'action de votre jeu, mais ils ne peuvent pas être modifiés après avoir été définis. Notez que vous pouvez définir des macros supplémentaires dans la même action en cliquant sur l'icône plus à côté de l'action. Pour plus d'informations sur les macros, cliquez ici. Syntaxe de l'action: Arguments: Argument Description Return La valeur de retour. Exemple: L'action ci-dessus créera trois nouvelles macros appelées pour générer de nouvelles constantes de couleur qui peuvent ensuite être utilisées dans toute autre action de votre jeu.
