Définir une variable globale Cette action est utilisée soit pour attribuer une valeur à une variable globale (en la créant au passage), soit pour modifier la valeur d'une variable globale existante. Les variables globales sont des variables qui "n'appartiennent" à aucune instance spécifique, mais plutôt à l'ensemble du projet. projet, et en tant que telles peuvent être accédées et modifiées par n'importe quelle instance, script ou code. Pour des informations plus avancées sur les variables globales, veuillez consulter ici. Vous donnez le nom de la variable globale, puis la valeur, qui peut être ajoutée par rapport à la valeur d'origine si nécessaire. Si vous donnez le nom d'une nouvelle variable (c'est-à-dire une variable qui n'a jamais été assignée auparavant dans tout le jeu), cette fonction créera la nouvelle variable globale et lui attribuera la valeur que vous avez donnée. créera la nouvelle variable globale et lui attribuera la valeur que vous donnez, tandis que si vous donnez le nom d'une variable globale existante, il changera simplement la valeur pour la nouvelle. Notez que si vous cochez la case Relative, la valeur sera ajoutée à la valeur de la variable globale existante (utilisez un nombre négatif). existante (utilisez un nombre négatif pour la soustraire) et vous pouvez également utiliser desexpressions et des variables créées précédemment comme valeur. Il convient de noter que lorsque nous parlons de "valeurs", nous ne parlons pas uniquement de valeurs numériques, car une variable peut être une chaîne de caractères, un pointeur, un ID de ressource ou tout autre élément qu'une fonction peut retourner ou utiliser. Dans le cas d'une chaîne de caractères, la vérification relative concaténera la valeur avec ce que la variable contenait précédemment, donc si votre variable était "Hello" et que vous lui attribuez "World" en utilisant la vérification relative, la valeur finale de la variable sera "Hello World". Il faut cependant veiller Il faut cependant veiller à ce que les valeurs de la variable soient toutes deux du même type de données, car si vous essayez d'ajouter, par exemple, un entier et une chaîne de caractères, vous obtiendrez une erreur de compilation. Vous pouvez également ajouter des variables supplémentaires dans la même action en cliquant sur l'icône plus à côté de l'action, et en donnant une autre variable globale nom et la valeur. Pour des informations plus avancées sur les variables, veuillez consulter la section sur les variables et leur portée.
