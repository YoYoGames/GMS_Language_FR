Retour Cette action est à utiliser lors de la création de fonctions de bloc d'action et est utilisée pour renvoyer une valeur à l'instance qui appelle le script. La valeur de retour peut être n'importe quoi qu'une variable peut variable - comme une chaîne de caractères, une valeur numérique, un pointeur ou un ID de ressource - ou peut être une variable elle-même. Notez que lorsque nous appelons l'action de retour dans une fonction, la fonction sera quittée - en retournant la valeur donnée - et aucune autre action après le retour ne sera appelée, bien que si vous avez des actions dans l'événement après avoir appelé la fonction, elles seront toujours exécutées. Syntaxe de l'action : Arguments : Arguments Description Retourner à La valeur de retour. Exemple : La fonction du bloc d'action ci-dessus définit une variable temporaire avec le mot-clé personne, puis vérifie une variable d'entrée pour voir quelle est sa valeur actuelle. Si la variable est supérieure à 100, une instance est créée et son ID d'instance unique est stocké dans la variable temporaire, qui est ensuite retournée par la fonction.
