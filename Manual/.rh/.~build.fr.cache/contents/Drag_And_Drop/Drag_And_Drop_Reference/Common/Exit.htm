Sortie Cette action fera l'une des deux choses suivantes, selon le contexte dans lequel elle est appelée. S'il est appelé à l'intérieur d'un événement, il quittera l'événement et passera au suivant, en ignorant tout code de bloc d'action qui pourrait être placé après lui. Toutefois, si elle est utilisée à l'intérieur d'une fonction de bloc d'action, elle quittera la fonction uniquement et reviendra à l'événement qui a appelé la fonction, en exécutant toute action ultérieure. Notez que le fait d'appeler au milieu d'une fonction ne retournera aucune valeur (pour cela, utilisez l'action de retour ). Syntaxe de l'action : Exemple : Le code du bloc d'action ci-dessus vérifie une variable et si elle est supérieure ou égale à 100, l'événement est quitté, sinon la variable est incrémentée de un et le sprite de l'instance reçoit une échelle aléatoire.
