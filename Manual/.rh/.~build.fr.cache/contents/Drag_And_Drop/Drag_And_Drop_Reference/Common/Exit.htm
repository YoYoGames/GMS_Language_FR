Sortir Cette action fera l'une des deux choses selon le contexte dans lequel elle est appelée. S'il est appelé depuis un événement, il quittera l'événement et passera au suivant, en ignorant tout code de bloc d'action qui pourrait être placé après lui. Cependant, s'il est utilisé dans une fonction de bloc d'action, il quittera la fonction uniquement et retournera à l'événement qui a appelé la fonction, en exécutant toutes les actions suivantes. Notez que l'appeler au milieu d'une fonction ne retournera aucune valeur (pour cela, utilisez lereturn action). Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus vérifie une variable et si elle est supérieure ou égale à 100, l'événement est quitté, sinon la variable est incrémentée de un et le sprite d'instance reçoit une échelle aléatoire.
