Set Particle Colour Cette action définira la ou les couleurs à utiliser lors de la création de votre sprite et pendant sa durée de vie. Vous devez d'abord spécifier la valeur de l'ID du type de particule (qui doit être la même que la valeur renvoyée lorsque vous avez créé la particule à l'aide de l'action Créer un type de particule ), puis vous pouvez définir les couleurs à utiliser. Vous pouvez choisir trois couleurs à utiliser, et le sprite se fondra entre elles pendant la durée de vie de la particule. Par exemple, si une particule a des couleurs rouge, verte et jaune, et une durée de vie de 60 images de jeu, alors lorsqu'elle est créée (image 0) elle sera rouge, puis elle s'estompera vers le vert à la moitié de sa durée de vie (image 30), avant de s'estomper vers le jaune à la fin de sa vie (image 60). Si vous utilisez un sprite personnalisé au lieu d'une forme de particule, toute couleur autre que le blanc se confondra avec le sprite utilisé. avec le sprite utilisé. REMARQUE: la composante alpha de la couleur n' est pas utilisée, seules les composantes RVB du sélecteur de couleur le sont. Pour définir l'alpha, utilisez l'action Set Particle Alpha.
