Détruire le type de particule Cette action "détruira" le type de particule donné créé par l'action Créer un type de particule (c'est-à-dire: libérer la mémoire utilisée par le type de particule). Cette action doit être appelée chaque fois que vous n'avez plus besoin d'un type de particule dans votre jeu, ou chaque fois que vous souhaitez recréer le type de particule (par exemple, juste avant d'appeler un redémarrage du jeu ). Syntaxe de l'action: Arguments: Argument Description Type ID La valeur d'ID du type à détruire Exemple: Le code de bloc d'action ci-dessus vérifiera si la touche "Espace" est enfoncée et si elle est détectée, il supprime le système de particules et les différents types de particules définis dans les variables globales données, puis redémarre le jeu.
