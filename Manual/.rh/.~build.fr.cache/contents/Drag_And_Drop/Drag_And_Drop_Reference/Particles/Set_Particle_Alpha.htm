Définir l'alpha des particules Cette action définira l'alpha lorsque vous créez la particule ainsi que l'alpha pendant sa durée de vie. Vous spécifiez d'abord la valeur de l'ID du type de particule (qui doit être identique à la valeur renvoyée lorsque vous avez créé la particule à l'aide de l' action Créer un type de particule ), puis vous pouvez définir les valeurs alpha à utiliser. Vous pouvez choisir trois alphas à utiliser, et la particule se fondra entre eux pendant toute la durée de vie de la particule. Par exemple, si une particule a 0,5, 1 et 0 comme valeurs alpha, et une durée de vie de 60 images de jeu, alors quand elle est créée (image 0) elle sera à 0,5 alpha, alors elle disparaîtra vers un alpha complet de 1 (image 30), avant de passer pour être invisible avec un alpha de 0 (image 60). Syntaxe de l'action: Arguments: Argument Description Type ID La valeur d'ID unique du type de particule pour définir la couleur de Start L'alpha initial à utiliser Middle L'alpha intermédiaire à utiliser End L'alpha final à utiliser Exemple: Le code de bloc d'action ci-dessus créera un nouveau type de particule et attribuera sa valeur d'identifiant unique à une variable globale. Il procède ensuite à la définition de toutes les propriétés du type de particule.
