Set Particle Sprite Cette action peut être utilisée pour définir le sprite qu'une particule peut utiliser lors du dessin. Vous fournissez la valeur d'identification unique du type dont vous voulez définir le sprite (telle que renvoyée lorsque vous avez créé le type avec l'action Créer un type de particule) et la ressource de sprite à utiliser. Si le sprite comporte des sous-images, la particule s'animera dans les images à la vitesse définie dans l'éditeur d'images pour le sprite. Notez également que si vous définissez une valeur de couleur pour la particule (à l'aide de l'action Définir la couleur de la particule ), cette couleur sera mélangée avec le sprite utilisé, de la même manière que la définition du mélange d'images sur une instance mélangera une couleur avec le sprite de l'instance (utilisez le blanc pour ne pas mélanger). NOTE: Si vous utilisez cette action, vous ne devez pas utiliser l'action Set Particle Shape car vous ne pouvez avoir qu'un sprite ou une forme, mais pas les deux.
