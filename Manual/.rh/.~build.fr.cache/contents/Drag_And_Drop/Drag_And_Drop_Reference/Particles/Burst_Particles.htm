Particules éclatées Cette action vous permet d'éclater des particules à tout moment sans compter sur un émetteur. Vous fournissez la valeur d'ID unique pour le système pour créer les particules (comme renvoyé lorsque vous avez créé le système avec l'action Créer un système de particules ) et l'ID unique du type de particule à créer (tel que renvoyé lorsque vous avez créé le type de particule avec le action Créer un type de particule ), ainsi que la position x / y pour les créer et le numéro à créer. La position peut être une position absolue dans la pièce ou vous pouvez la marquer comme étant une position relative par rapport à l'instance qui appelle l'action. Vous pouvez appeler cette action à chaque image du jeu pour reproduire l'effet de flux d'un émetteur, ou vous pouvez l'appeler une fois dans un événement (comme l'événement Destroy) pour simplement éclater les particules. Syntaxe de l'action: Arguments: Argument Description System La valeur ID du système de particules à utiliser Type La valeur ID du type de particule à éclater X La position x pour éclater les particules à Y Ils se positionnent pour éclater les particules à Count Le nombre de particules à éclater Exemple: Le code de bloc d'action ci-dessus vérifiera si l'instance a changé de position depuis la dernière mise à jour de la dernière image de jeu et si elle l'a fait, certaines particules éclateront à la nouvelle position.
