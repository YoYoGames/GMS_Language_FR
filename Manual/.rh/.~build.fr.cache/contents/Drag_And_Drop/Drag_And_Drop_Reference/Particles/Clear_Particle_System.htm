Système de particules claires Avec cette action, vous pouvez réinitialiser le système de particules donné à son état par défaut, en supprimant tous les émetteurs (il n'est pas non plus nécessaire d'appeler l' action Détruire l'émetteur de particules car cela est pris en charge automatiquement) et en effaçant toutes les particules de l'écran. Vous fournissez la valeur d'ID unique que le système doit effacer, telle qu'elle est renvoyée lorsque vous avez créé le système avec l'action Créer un système de particules. Des précautions doivent être prises lors de l'utilisation de cette action car elle détruit tous les émetteurs associés au système de particules - cela signifie que toute instance dans la pièce qui utilise un émetteur peut donner une erreur car cet émetteur n'existe plus, alors assurez-vous que l'action est uniquement appelé lorsque vous n'allez pas créer de nouvelles particules à partir de cet émetteur (pour plus d'informations sur les émetteurs de particules, voir Créer un émetteur de particules ). NOTE: Cette action n'effacer les particules visibles dans la salle, mais il ne supprime pas les propriétés de particules ni les supprimer de la mémoire. Pour cela, vous devez utiliser l'action Détruire le type de particule. Syntaxe de l'action: Arguments: Argument Description System La valeur d'identifiant unique du système à détruire Exemple: Le code de bloc d'action ci-dessus effacera toutes les particules de l'écran si le joueur a 0 vies ou moins, puis redémarrera la pièce.
