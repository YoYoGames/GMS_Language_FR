Système de particules claires Avec cette action, vous pouvez remettre le système de particules à son état par défaut, en supprimant tous les émetteurs (il n'est pas nécessaire d'appeler l'action Destroy Particle Emitter car cela est pris en charge automatiquement) . et effacer toutes les particules de l'écran. Vous fournissez la valeur d'identification unique du système à effacer, telle qu'elle a été retournée lorsque vous avez créé le système avec l'action Create Particle System. Il faut faire attention en utilisant cette action car elle détruit tous les emitters associés au système de particules - cela signifie que toute instance dans la pièce qui utilise un emitter peut donner une erreur car cet emitter n'existe plus, donc assurez-vous que l'action est seulement appelée quand vous n'avez pas l'intention de créer de nouvelles particules à partir de cet émetteur (pour plus d'informations sur les émetteurs de particules, voir Create Particle Emitter). NOTE: Cette action n'efface que les particules visibles dans la pièce, mais elle n'efface pas les propriétés des particules et ne les supprime pas de la mémoire. Pour cela, vous devez utiliser l'action Détruire le type de particule. Syntaxe de l'action : Arguments : Arguments Description Système La valeur d'identification unique du système à détruire. Exemple : Le code du bloc d'action ci-dessus effacera toutes les particules de l'écran si le joueur a 0 vie ou moins, puis redémarre la pièce.
