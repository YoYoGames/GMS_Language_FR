Ensemble Forme des particules Cette action peut être utilisée pour définir la forme du sprite intégré qu'une particule doit utiliser lors du dessin. Vous fournissez la valeur d'identification unique du type dont vous voulez définir la forme (telle qu'elle est retournée lorsque vous avez créé le type avec l'action Créer un type de particule) et la forme à utiliser. Les formes de particules sont simplement des sprites blancs qui ont été créés pour GameMaker Studio 2 afin d'être utilisés spécifiquement en tant que particules et se composent des formes suivantes : NOTE: Si vous utilisez cette action, vous ne devez pas utiliser l'action Set Particle Sprite car vous ne pouvez avoir qu'un sprite ou une forme, mais pas les deux. Syntaxe de l'action : Arguments : Arguments Description Type La valeur d'identification du type dont il faut définir la forme. Forme La forme à utiliser pour le type de particule Exemple : Le code du bloc d'action ci-dessus créera un nouveau type de particule et affectera sa valeur d'identification unique à une variable globale. unique à une variable globale.
