Définir la forme des particules Cette action peut être utilisée pour définir la forme de sprite intégrée qu'une particule doit utiliser lors du dessin. Vous fournissez la valeur d'ID unique du type pour définir la forme (telle que renvoyée lorsque vous avez créé le type avec l'action Créer un type de particule ) et la forme à utiliser. Les formes de particules sont simplement des sprites blancs qui ont été créés pour GameMaker Studio 2 pour être utilisés spécifiquement en tant que particules et se composent des formes suivantes: REMARQUE: si vous utilisez cette action, vous ne devez pas utiliser l'action Définir le sprite de particules car vous ne pouvez avoir qu'un sprite ou une forme, mais pas les deux. Il procède ensuite à la définition de toutes les propriétés du type de particule.
