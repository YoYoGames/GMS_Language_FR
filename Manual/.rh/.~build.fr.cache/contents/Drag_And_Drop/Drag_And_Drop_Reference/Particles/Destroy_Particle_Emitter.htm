Détruire l'émetteur de particules Cette action va "détruire" l'émetteur de particules donné, c'est-à-dire: libérer la mémoire utilisée par l'émetteur de particules. Vous fournissez l'ID du système de particules auquel l'émetteur appartient (tel que renvoyé par l'action Créer un système de particules ), puis l'ID réel de l'émetteur (tel que renvoyé par l'action Créer un émetteur de particules ). Cette action doit être appelée chaque fois que vous n'avez plus besoin d'un émetteur de particules dans votre jeu, ou chaque fois que vous souhaitez recréer l'émetteur de particules (par exemple, juste avant d'appeler un redémarrage du jeu ). Notez que si le système de particules auquel l'émetteur a été affecté n'a pas été marqué comme Persistant, vous n'avez pas besoin d'appeler cette action, car l'émetteur sera automatiquement nettoyé du côté de la pièce avec le système de particules. Syntaxe de l'action: Arguments: Argument Description System La valeur ID du système qui contient l'émetteur Emitter La valeur ID de l'émetteur à détruire Exemple: Le code de bloc d'action ci-dessus vérifiera une variable d'instance et si elle est supérieure à 100, il détruira l'émetteur affecté à l'instance, puis passera à la pièce suivante.
