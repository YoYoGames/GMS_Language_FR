Mise à jour du système de particules Cette action va forcer le système de particules donné à mettre à jour la position, la couleur, l'alpha, etc... Vous fournissez la valeur d'identification unique du système à mettre à jour (telle que retournée lorsque vous avez créé le système avec l'action Create Particle System), et chaque fois que vous appelez l'action, le système donné mettra à jour une image de jeu, ce qui signifie que vous pouvez, par exemple, appeler cette action dans une boucle Repeat et faire en sorte que le système de particules soit mis à jour plusieurs fois dans une seule image de jeu. une seule image de jeu (utile lorsque vous voulez qu'un niveau ait un effet qui nécessite la présence de particules au tout début, comme la neige ou la pluie). Vous pouvez également utiliser cette action lorsque le système de particules a été mis en pause à l'aide de la fonction Pause Particle System pour mettre à jour manuellement les particules du système.
