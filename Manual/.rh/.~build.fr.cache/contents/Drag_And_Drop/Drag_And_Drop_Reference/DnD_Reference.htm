Référence Drag And Drop™ Cette section du manuel répertorie toutes les différentes actions disponibles lorsque vous utilisez Drag and Drop™ pour programmer votre jeu. Vous trouverez ci-dessous chacune des différentes bibliothèques d'actions disponibles à partir de l'objet DnD™ Toolbox, et dans chaque section de bibliothèque, les actions individuelles sont expliquées, ainsi que des exemples d'utilisation.
