Bibliothèque d'action des tampons Une mémoire tampon est une région de la mémoire physique utilisée pour contenir temporairement des données pendant qu'elles sont déplacées d'un endroit à un autre ou manipulées d'une manière ou d'une autre. Par exemple, vous pouvez utiliser une mémoire tampon pour contenir des données sur l'état du jeu. GameMaker Studio 2 comporte un certain nombre d'actions spécifiques liées à la création, à la modification et au chargement/à la sauvegarde des tampons, mais avant de les utiliser, veuillez noter les points suivants : Lorsqu'un tampon est créé, il est automatiquement effacé et rempli à 0. L'argument "size" des fonctions de tampon fait toujours référence à la taille en octets du tampon. Les tamponsDnD™sont toujours alignés sur 1 octet. L'"alignement" fait référence à la façon dont les données sont stockées dans un tampon et est utilisé lors de l'utilisation des fonctions de tampon GML , mais pour le glisser-déposer, il n'est pas pertinent. Les données dans les tampons sont lues et écrites de manière séquentielle (c'est-à-dire que si vous appelez l'action Write Buffer trois fois, elle écrira les premières données, avancera, écrira les secondes données, avancera et écrira les dernières données). la dernière donnée), avec une valeur appelée "seek" pour vous indiquer où dans le tampon la prochaine lecture/écriture aura lieu. Toutes les données sont écrites dans le tampon et retournées du tampon sous forme de chaîne de caractères. Si vous avez stocké des nombres réels, vous devrez utiliser l'une des actions de type de données pour les convertir de la chaîne de caractères en valeur. en valeur. La mémoire utilisée pour la création des tampons est la mémoire système. Ainsi, même lorsque le jeu n'est pas au centre de l'attention, les données stockées dans un tampon devraient être protégées, mais si l'application est fermée ou redémarrée, elles seront perdues. Il est également intéressant de noter que les tampons peuvent également être sauvegardés sous forme de fichiers puis rechargés dans le jeu à l'aide des actions de fichier.
