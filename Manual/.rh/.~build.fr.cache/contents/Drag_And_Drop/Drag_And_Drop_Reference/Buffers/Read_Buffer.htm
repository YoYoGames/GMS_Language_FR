Tampon de lecture Cette action peut être utilisée pour lire des données à partir d'un tampon créé précédemment. Vous fournissez la valeur d'ID de tampon unique (telle que renvoyée par l'action Créer un tampon ) et indiquez une variable pour stocker la valeur renvoyée. Si vous marquez la variable comme une variable locale temporaire, elle sera créée pour vous et supprimée à nouveau à la fin de l'événement ou du script d'action. Toutes les données sont renvoyées sous forme de chaîne, donc si vous avez stocké des nombres réels, vous devrez utiliser l'une des actions de type de données pour les convertir de la chaîne en valeur. Gardez à l'esprit également que les tampons sont lus et écrits séquentiellement et que la lecture à partir d'un tampon déplacera la position «recherche» vers l'entrée suivante, ce qui signifie que pour lire divers éléments de données, vous appelez simplement cette action le nombre de fois requis et stockez chaque élément de données renvoyé à une variable. Syntaxe de l'action: Arguments: Argument Description Buffer L'index du tampon (stocké dans une variable) Target La variable cible contenant les données renvoyées Exemple: Le code de bloc d'action ci-dessus définit la position de lecture / écriture du tampon au début du tampon, puis lit deux éléments de données, en les stockant (sous forme de chaînes) dans des variables locales temporaires.
