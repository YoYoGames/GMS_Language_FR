Rechercher le tampon Cette action peut être utilisée pour se déplacer (rechercher) vers une position dans un tampon. Vous fournissez la valeur d'ID de tampon unique renvoyée par l'action Créer un tampon, puis la valeur de décalage en octets vers laquelle se déplacer. La valeur de décalage sera appliquée à partir du début du tampon, donc l'appel de cette fonction déplacera la position de lecture / écriture au début du tampon, puis déplacera le nombre d'octets donné et y positionnera la nouvelle position de lecture / écriture. Cependant, si vous marquez le décalage comme relatif, il déplacera la position de lecture / écriture vers l'avant (ou vers l'arrière si le décalage est une valeur négative) de la position actuelle. Syntaxe de l'action: Arguments: Argument Description Buffer L'index du tampon (stocké dans une variable) offset Le décalage (en octets) par rapport à la position de recherche actuelle Exemple: Le code de bloc d'action ci-dessus obtient la taille de la mémoire tampon, puis utilise cette valeur pour définir la position de lecture / écriture à la fin des données actuellement stockées.
