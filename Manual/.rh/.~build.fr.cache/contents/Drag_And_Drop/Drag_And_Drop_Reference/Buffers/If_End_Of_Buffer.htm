Si fin de tampon Cette action peut être utilisée pour vérifier si la position de lecture / écriture a atteint la fin du tampon. Vous fournissez la valeur d'ID de tampon unique renvoyée par l'action Créer un tampon, puis l'action retournera true si la position de lecture / écriture a atteint la fin de toutes les données disponibles dans le tampon, ou false si ce n'est pas le cas. Vous pouvez marquer "non" pour que l'action vérifie si la position de lecture / écriture n'est pas à la fin du tampon. Cela continue jusqu'à ce que la fin du tampon soit atteinte, moment auquel la boucle est interrompue.
