Tampon d'écriture Cette action peut être utilisée pour écrire des données dans un tampon. Vous fournissez l'ID unique du tampon dans lequel vous souhaitez écrire (tel que retourné par l'action Create Buffer) et ensuite la valeur à écrire, qui peut être un réel ou une chaîne, mais notez que toutes les valeurs sont stockées en tant que chaînes de caractères, donc lorsque vous lisez à partir du tampon (en utilisant Read Buffer), vous pouvez avoir besoin d'utiliser l'une des actions de type de données pour le convertir d'une chaîne de caractères à la valeur originale. en valeur d'origine. Gardez également à l'esprit que les tampons sont lus et écrits séquentiellement et que l'écriture dans un tampon déplace la position de recherche d'une unité, ce qui signifie que pour écrire divers éléments de données, il suffit d'appeler cette action le nombre de fois nécessaire.
