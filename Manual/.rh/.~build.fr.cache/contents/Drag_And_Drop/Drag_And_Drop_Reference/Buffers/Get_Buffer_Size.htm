Obtenir la taille du tampon Cette action peut être utilisée pour récupérer la taille d'un tampon, en octets. Vous fournissez la valeur d'ID de tampon unique renvoyée par l'action Créer un tampon, puis donnez une variable cible à laquelle renvoyer la valeur de taille. La variable cible peut être marquée comme étant une variable locale temporaire, auquel cas elle sera créée pour vous et supprimée à nouveau de la mémoire à la fin du script d'action ou de l'événement en cours. Syntaxe de l'action: Arguments: Argument Description Buffer L'index du tampon (stocké dans une variable) Target La variable cible contenant la valeur renvoyée Exemple: Le code de bloc d'action ci-dessus obtient la taille de la mémoire tampon, puis utilise cette valeur pour définir la position de lecture / écriture à la fin des données actuellement stockées.
