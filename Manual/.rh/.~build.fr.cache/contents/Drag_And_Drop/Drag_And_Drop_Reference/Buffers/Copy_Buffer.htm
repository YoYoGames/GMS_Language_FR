Tampon de copie Cette action peut être utilisée pour copier toutes les données stockées dans un tampon vers un autre. Lorsque vous utilisez deux tampons et copiez de l'un à l'autre, les deux tampons doivent avoir été préalablement créés à l'aide de l'action Créer un tampon, et vous devez fournir l'ID unique du tampon retourné comme arguments de/à. REMARQUE: Vous ne pouvez pas copier dans le même tampon. Syntaxe de l'action : Arguments : Arguments Description Tampon L'index du tampon (stocké dans une variable) à copier à partir de Copie à L'index de la mémoire tampon (stocké dans une variable) à copier . Exemple : Le code du bloc d'action ci-dessus va créer un nouveau tampon et stocker son ID dans une variable temporaire. Il vérifie ensuite si le tampon indexé dans la variable globale donnée existe, et si c'est le cas, il est copié dans le tampon temporaire et la position de lecture/écriture est réinitialisée au début, et s'il n'existe pas, il est créé.
