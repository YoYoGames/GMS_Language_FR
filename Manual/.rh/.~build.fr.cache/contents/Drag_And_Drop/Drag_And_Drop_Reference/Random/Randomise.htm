Randomiser Cette action définit la "graine" du générateur de nombres aléatoires sur une valeur aléatoire. Lorsque vous utilisez les fonctions de nombres aléatoires dans GameMaker Studio 2 la valeur initiale est toujours la même, car cela facilite grandement le traçage des erreurs et le débogage (les jeux compilés n'ont pas ce comportement). Si vous souhaitez tester avec un vrai aléatoire, vous devez appeler cette action une fois au début du jeu. Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus définit la valeur de départ aléatoire sur une nouvelle valeur, puis génère un nombre compris entre 0 et 9 et l'assigne à une variable de portée globale.
