Bibliothèque d'actions aléatoires Le site Random comporte les actions suivantes pour traiter la génération de nombres aléatoires et les opérations de hasard : Obtenir un numéro aléatoire Randomisez Choisissez Il convient de noter que GameMaker Studio 2 utilisera la même graine pour générer des nombres aléatoires chaque fois que vous exécuterez le jeu à partir de l'IDE. Cela permet de faciliter le débogage et signifie que vous obtiendrez les mêmes résultats initiaux lors de la génération de valeurs aléatoires, mais cela ne se produira pas lorsque vous aurez compilé le jeu dans un exécutable (une nouvelle graine sera générée à chaque fois que le jeu sera exécuté). Si vous ne souhaitez pas obtenir ce comportement lors des tests, vous devez appeler l'action Randomise une fois au début du jeu, avant de générer des valeurs aléatoires.
