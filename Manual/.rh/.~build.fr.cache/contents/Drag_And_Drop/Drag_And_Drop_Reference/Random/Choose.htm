Choisir Avec cette action, vous pouvez donner un certain nombre d'options différentes (il peut s'agir d'un nombre réel, d'une chaîne de caractères, d'un ID de ressource ou d'une constante) parmi lesquelles l'action en choisira une à renvoyer. Vous pouvez ajouter des options supplémentaires en cliquant sur l'icône plus à côté de l'action, et en donnant une autre valeur qui pourrait être retournée. La valeur retournée sera alors stockée dans la variable cible qui peut avoir été créée précédemment ou être une nouvelle variable locale temporaire (si vous cochez la case "Temp").
