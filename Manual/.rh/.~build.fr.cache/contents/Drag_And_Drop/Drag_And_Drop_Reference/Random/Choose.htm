Choisir Avec cette action, vous pouvez donner un certain nombre d'options différentes (cela peut être un nombre réel, une chaîne, un ID de ressource ou une constante) à partir desquelles l'action en choisira une à renvoyer. Vous pouvez ajouter des options supplémentaires en cliquant sur l'icône plus à côté de l'action, et en donnant une autre valeur qui pourrait être retournée. La valeur retournée sera alors stockée dans la variable cible qui peut avoir été créée précédemment ou peut être une nouvelle variable locale temporaire (si vous cochez la case "Temp"). Syntaxe de l'action: Arguments: Argument Description Option L'option qui peut être renvoyée par l'action de sélection. Exemple: Le code de bloc d'action ci-dessus choisit l'une des quatre ressources de salle et stocke l'ID renvoyé dans une variable locale temporaire. Ceci est ensuite utilisé pour changer de pièce.
