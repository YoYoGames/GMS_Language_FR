Obtenir un numéro aléatoire Cette action va générer un nombre aléatoire dans les limites d'un ensemble de valeurs minimales et maximales. Vous pouvez choisir le type de nombre à générer (décimal ou entier), puis définir la plage dans laquelle le nombre doit être généré en fournissant une valeur minimale et maximale. La valeur aléatoire générée sera renvoyée à la variable cible (qui peut être marquée comme une variable locale temporaire). REMARQUE: Lors de la génération d'un nombre entier aléatoire, l'action renverra une valeur entière comprise entre la valeur minimale et la valeur maximale incluses, donc une plage de 5 à 10 pourrait renvoyer 5 ou 10 ainsi que tout nombre entier intermédiaire. Cependant, la génération d'un nombre décimal aléatoire ne retournera jamais la valeur maximale, donc la plage de 5 à 10 peut retourner 5 mais ne retournera pas 10 (bien qu'elle puisse retourner 9.999999999999). Syntaxe de l'action : Arguments : Arguments Description Type Le type de chiffres à utiliser (entiers ou décimaux) Minimum Le nombre minimum pour la gamme aléatoire Maximum Le nombre maximum pour la gamme aléatoire Cible La variable cible pour stocker la valeur retournée Exemple : Le code du bloc d'action ci-dessus génère une valeur aléatoire et l'attribue à une locale temporaire. Cette valeur est ensuite utilisée dans une boucle répétitive où d'autres appels aléatoires sont effectués pour générer des positions dans la pièce.
