Renommer un fichier Cette action vous permet de renommer n'importe quel fichier qui a été sauvegardé par votre jeu. Vous donnez le nom du fichier à modifier (sous forme de chaîne), puis le nouveau nom (également sous forme de chaîne et incluant l'extension du fichier), et le fichier sera renommé. Notez que vous ne pouvez pas renommer un fichier qui a été inclus dans votre jeu (voir la section sur le système de fichiers pour plus d'informations). Syntaxe de l'action : Arguments : Arguments Description Nom de fichier Le nom (sous forme de chaîne) du fichier à renommer Nouveau nom Le nouveau nom (sous forme de chaîne) du fichier Exemple : Le code du bloc d'action ci-dessus vérifie si le fichier "checkpoint.sav" existe, et si c'est le cas, il vérifie ensuite si le fichier "checkpoint_OLD.sav" existe. il existe, il vérifie ensuite si le fichier "checkpoint_OLD.sav" existe. Si ce fichier existe également, il est supprimé, et le fichier original "checkpoint.sav" est renommé en "checkpoint_OLD.sav". Enfin, un tampon est sauvegardé en tant que "checkpoint.sav" (essentiellement, nous sauvegardons un fichier tampon à chaque fois que nous le sauvegardons, de sorte qu'il y a toujours un fichier de sauvegarde "actuel" et un fichier de sauvegarde "ancien").
