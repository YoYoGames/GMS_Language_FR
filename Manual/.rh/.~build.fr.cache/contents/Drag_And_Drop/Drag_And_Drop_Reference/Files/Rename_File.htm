Renommer le fichier Avec cette action, vous pouvez renommer n'importe quel fichier qui a été enregistré par votre jeu. Vous donnez le nom du fichier à modifier (sous forme de chaîne), puis le nouveau nom (également sous forme de chaîne et avec l'extension de fichier), et le fichier sera renommé. Notez que vous ne pouvez pas renommer un fichier qui a été fourni en tant que fichier inclus avec votre jeu (voir la section sur le système de fichiers pour plus d'informations). Syntaxe de l'action: Arguments: Argument Description Filename Le nom (sous forme de chaîne) du fichier à renommer New Name Le nouveau nom (sous forme de chaîne) du fichier Exemple: Le code de bloc d'action ci-dessus vérifiera si le fichier "checkpoint.sav" existe, et s'il le fait alors vérifie si le fichier "checkpoint_OLD.sav" existe. Si ce fichier existe également, il est supprimé, puis le fichier d'origine "checkpoint.sav" est renommé "checkpoint_OLD.sav". Enfin, un tampon est enregistré sous "checkpoint.sav" (essentiellement, nous sauvegardons un fichier tampon enregistré à chaque fois que nous le sauvegardons, de sorte qu'il y ait toujours une sauvegarde "actuelle" et un "ancien" fichier de sauvegarde).
