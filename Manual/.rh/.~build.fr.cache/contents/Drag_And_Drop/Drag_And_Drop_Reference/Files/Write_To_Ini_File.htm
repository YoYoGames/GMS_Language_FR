Ecrire dans un fichier Ini Avec cette action, vous pouvez écrire une valeur numérique ou une chaîne de caractères dans un fichier Ini. Vous devez au préalable avoir ouvert le fichier Ini en lecture/écriture à l'aide de l'action Ouvrir un fichier Ini, puis avec cette action vous donnez le type de données à écrire (soit un nombre réel, soit une chaîne de caractères entourée de ""), ainsi que l'en-tête de section, la clé dans cette section et la valeur de la clé. Pour aider à visualiser la structure d'un fichier Ini, voir cette image : Comme vous pouvez le constater, les fichiers Ini sont divisés en sections - où chaque section a son propre nom d'en-tête - et puis chaque section est ensuite divisée en diverses paires clé/valeur (un peu comme une structure de données Map). Notez que vous pouvez créer vous-même des fichiers Ini à l'aide de n'importe quel logiciel de traitement de texte (comme Notepad sous Windows), puis les inclure en tant que fichier inclus à lire et à écrire dans votre jeu. IMPORTANT ! Essayer de lire les données d'un fichier Ini qui n'a pas été ouvert précédemment à l'aide de l'action Ouvrir un fichier Ini provoquera une erreur grave et fera planter votre jeu. IMPORTANT ! Cette opération ne fait qu'écrire les données en mémoire. Pour écrire définitivement les données dans le fichier sur le disque, vous devez appeler Close Ini File, sinon les données ne seront pas enregistrées. Syntaxe de l'action : Arguments : Arguments Description Type Le type de données à écrire, soit une chaîne de caractères (entre guillemets ""), soit un nombreréel. Section La section du fichier Ini dans laquelle il faut écrire (sous forme de chaîne). Clé La clé de la section sur laquelle écrire (sous forme de chaîne). Valeur La valeur à écrire, soit une chaîne de caractères, soit un nombre réel basé sur l'argument "type" (ci-dessus). Exemple : Le code du bloc d'action ci-dessus ouvrira un fichier ini, puis écrira des données dans trois le même en-tête de section (une chaîne de caractères et deux nombres réels) avant de refermer le fichier ini (et d'écrire les données sur le disque).
