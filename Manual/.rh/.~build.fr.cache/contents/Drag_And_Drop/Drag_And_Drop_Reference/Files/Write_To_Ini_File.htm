Écrire dans un fichier Ini Avec cette action, vous pouvez écrire une valeur numérique ou une chaîne dans un fichier Ini. Vous devez au préalable avoir ouvert le fichier Ini en lecture / écriture à l'aide de l'action Ouvrir un fichier Ini, puis avec cette action, vous indiquez le type de données à écrire (soit un nombre réel soit une chaîne entre ""), ainsi que le en-tête de section, la clé dans cette section et la valeur de la clé. Pour aider à visualiser la structure d'un fichier Ini, voir cette image: Comme vous pouvez le voir, les fichiers Ini sont divisés en sections - où chaque section a son propre nom d'en-tête - puis chaque section est ensuite divisée en différentes paires clé / valeur (un peu comme une structure de données cartographiques ). Notez que vous pouvez créer vous-même des fichiers Ini à l'aide de n'importe quel logiciel de traitement de texte (comme le Bloc-notes sous Windows ), puis les inclure en tant que fichier inclus à lire et à écrire dans votre jeu. IMPORTANT! Essayer de lire des données à partir d'un fichier Ini qui n'a pas été ouvert auparavant à l'aide de l' action Ouvrir un fichier Ini provoquera une erreur grave et plantera votre jeu. IMPORTANT! Cela n'écrit que les données dans la mémoire et pour finalement écrire les données dans le fichier sur le disque, vous devez appeler Close Ini File, sinon les données ne seront pas enregistrées. Syntaxe de l'action: Arguments: Argument Description Type Le type de données à écrire, soit un string (entre guillemets ""), ou un real number Section La section du fichier Ini dans laquelle écrire (sous forme de chaîne) Key La clé dans la section dans laquelle écrire (sous forme de chaîne) Value La valeur à écrire, soit une chaîne, soit un nombre réel basé sur l'argument "type" (ci-dessus) Exemple: Le code de bloc d'action ci-dessus ouvrira un ini puis écrira des données sur trois clés différentes sous le même en-tête de section (une chaîne et deux nombres réels) avant de refermer le fichier ini (écrivant ainsi les données sur le disque).
