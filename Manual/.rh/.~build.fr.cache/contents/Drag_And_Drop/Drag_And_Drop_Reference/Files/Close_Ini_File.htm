Fermer le fichier Ini Cette action fermera le fichier Ini actuellement ouvert. Cette action doit être appelée au moment où vous avez fini de lire ou d'écrire dans un fichier Ini ouvert (les fichiers Ini sont ouverts en utilisant l'action Ouvrir un fichier Ini). Si vous n' utilisez pas cette action après avoir utilisé l'action Ecrire dans un fichier Ini, rien ne sera écrit sur le disque car les informations du fichier sont conservées en mémoire jusqu'à ce que cette action soit appelée, moment auquel elles sont extraites de la mémoire et écrites dans le fichier sur le disque. et écrites dans le fichier sur le disque. Notez également que si vous essayez d'ouvrir un fichier Ini sans en avoir préalablement fermé un autre (ou le même), vous obtiendrez une erreur.
