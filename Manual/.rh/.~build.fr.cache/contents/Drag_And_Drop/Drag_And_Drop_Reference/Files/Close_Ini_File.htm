Fermer le fichier Ini Cette action fermera le fichier Ini actuellement ouvert. Cette action doit être appelée au moment où vous avez fini de lire ou d'écrire dans un fichier Ini ouvert (les fichiers Ini sont ouverts à l'aide de l'action Ouvrir un fichier Ini ). Si vous n'utilisez pas cette action après avoir utilisé l'action Écrire dans un fichier Ini, rien ne sera écrit sur le disque car les informations sur le fichier sont conservées en mémoire jusqu'à ce que cette action soit appelée, moment auquel elles sont extraites de la mémoire et écrites dans le fichier sur le disque. Notez également que si vous essayez d'ouvrir un fichier Ini sans en avoir préalablement fermé un autre (ou le même), vous obtiendrez une erreur. Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus ouvrira un fichier ini pour utilisation, puis obtiendra la valeur associée à la clé "nom" sous l'en-tête "player". Si le nom renvoyé correspond à la valeur de nom par défaut (c'est-à-dire: le fichier, la section ou la clé n'existe pas ), alors le fichier a une variable globale écrite avant d'être fermé (et d'écrire les nouvelles données sur le disque).
