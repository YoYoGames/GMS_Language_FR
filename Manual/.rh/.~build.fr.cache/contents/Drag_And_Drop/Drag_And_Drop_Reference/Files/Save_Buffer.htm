Sauvegarder le tampon Cette action vous permet d'enregistrer un fichier contenant les données d'un tampon. Vous fournissez l'ID unique du tampon (tel que retourné lorsque vous avez créé le tampon avec Create Buffer) et donnez un nom au fichier (sous forme de chaîne, avec l'extension de votre choix) et le tampon sera écrit dans ce fichier prêt à être chargé à nouveau en utilisant Load Buffer. Si le fichier existe, il est chargé dans la variable de la mémoire tampon que nous avons créée, mais s'il n'existe pas, il est créé, écrit puis sauvegardé, prêt à être chargé la prochaine fois.
