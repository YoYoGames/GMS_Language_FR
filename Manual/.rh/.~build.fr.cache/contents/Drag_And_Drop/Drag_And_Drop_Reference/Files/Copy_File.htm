Copier un fichier Avec cette action, vous pouvez copier tout fichier qui a été enregistré par votre jeu ou qui a été regroupé dans le cadre des fichiers inclus. Vous indiquez le nom du fichier à copier (sous forme de chaîne), puis le nom du fichier à créer sous forme de copie du fichier d'origine (également sous forme de chaîne et avec l'extension de fichier). Le fichier donné sera dupliqué et recevra le nouveau nom. Syntaxe de l'action: Arguments: Argument Description Filename Le nom (sous forme de chaîne) du fichier à copier Copy To Le nom (sous forme de chaîne) du fichier à créer en tant que copie Exemple: Le code de bloc d'action ci-dessus vérifiera si un fichier ini n'existe pas et si ce n'est pas le cas, il est créé en copiant un autre fichier.
