Ouvrir le fichier Ini Avec cette action, vous pouvez ouvrir un fichier Ini pour la lecture et / ou l'écriture. Si le fichier Ini n'existe pas à l'emplacement que vous vérifiez, GameMaker Studio 2 peut en créer un, mais uniquement si vous y écrivez des données. Si vous n'avez lu que les informations du fichier Ini, les valeurs par défaut de la fonction de lecture seront renvoyées, mais le fichier Ini ne sera pas réellement créé. Veuillez noter que vous ne pouvez ouvrir qu'un seul fichier Ini à la fois et n'oubliez pas d'utiliser Close Ini File une fois que vous avez fini de lire / écrire à partir du fichier Ini car les informations ne sont pas réellement stockées sur le disque jusque-là (elles sont stockées mémoire jusqu'à ce que le fichier soit fermé). Syntaxe de l'action: Arguments: Argument Description Filename Le nom - sous forme de chaîne et avec le .ini extension - du fichier à ouvrir Exemple: Le code de bloc d'action ci-dessus ouvrira un fichier ini pour utilisation, puis obtiendra la valeur associée à la clé "nom" sous l'en-tête "player". Si le nom renvoyé correspond à la valeur de nom par défaut (c'est-à-dire: le fichier, la section ou la clé n'existe pas ), alors le fichier a une variable globale écrite avant d'être fermé (et d'écrire les nouvelles données sur le disque).
