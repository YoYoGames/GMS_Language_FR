Types de données Bibliothèque d'action GameMaker Studio 2 possède un certain nombre de types de données différents, comme les entiers, les chaînes de caractères, les pointeurs, etc... et certains d'entre eux peuvent être convertis en d'autres. Cette section du manuel présente les différentes actions disponibles pour traiter et convertir différents types de données.
