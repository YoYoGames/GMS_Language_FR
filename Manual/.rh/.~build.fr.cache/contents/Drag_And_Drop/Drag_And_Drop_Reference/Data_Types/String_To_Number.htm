Chaîne en nombre Cette action peut convertir une chaîne de caractères en un nombre. Vous fournissez la chaîne à modifier, puis une variable cible à laquelle renvoyer le nombre réel (cette variable peut être marquée comme une variable locale temporaire). Notez que si la chaîne de caractères contient des caractères autres que des chiffres et un point décimal, elle ne sera pas convertie, mais si la chaîne commence par un chiffre, celui-ci sera converti et le reste de la chaîne sera exclu. Il est important de noter que lorsque vous passez des décimales dans cette action la chaîne retournée sera arrondie à deux décimales seulement. Voici un tableau des valeurs d'entrée et de ce que vous pouvez vous attendre à recevoir en retour. Chaîne de caractères Valeur retournée "123" 123 "123 Hello World" 123 "Hello World" 0 (par défaut, si la chaîne ne peut pas être convertie, 0 sera renvoyé) "Hello World 123" 0 "123.45678" 123.46 Syntaxe de l'action : Arguments : Arguments Description Valeur La chaîne de caractères à convertir Cible La variable cible qui contient la valeur retournée Exemple : Le code du bloc d'action ci-dessus initialise une variable globale et charge ensuite un fichier avec un tampon enregistré dans celui-ci. Deux chaînes de caractères sont lues à partir du tampon et stockées dans une variable locale temporaire avant que l'une d'entre elles ne soit convertie en un nombre réel et stockée dans une autre variable locale temporaire. Celles-ci sont ensuite affectées à des variables globales, et le tampon est supprimé.
