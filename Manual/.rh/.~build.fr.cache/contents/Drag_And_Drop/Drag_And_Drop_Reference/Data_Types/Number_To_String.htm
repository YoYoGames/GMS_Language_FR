Nombre en chaîne Cette action convertira un nombre réel en une chaîne. Si le nombre n'est pas une valeur entière et comporte des décimales, celles-ci seront arrondies aux deux décimales les plus proches. Ainsi, par exemple, le nombre 123,45678 renverra la chaîne «123,46». Syntaxe de l'action: Arguments: Argument Description Value Le nombre réel à convertir Target La variable cible contenant la valeur renvoyée Exemple: Le code de bloc d'action ci-dessus prend une valeur décimale et la convertit en entier, en stockant la valeur renvoyée dans une variable locale temporaire. Cette valeur est ensuite transformée en une chaîne et dessinée à l'écran. Retour: Actions de type de données Suivant: Décimal en entier © Copyright YoYo Games Ltd. 2021 Tous droits réservés
