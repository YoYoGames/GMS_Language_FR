Direction du point de consigne Cette action est utilisée pour définir une direction à partir d'un vecteur formé par la position actuelle de l'instance et la position donnée. Vous fournissez les positions x et y sur lesquelles "pointer" - celles-ci peuvent être marquées comme étant relatives à la position de l'instance - et la direction de l'instance vers ce point sera utilisée pour l'instance. de l'instance à ce point sera utilisée pour l'instance. Syntaxe de l'action : Arguments : Arguments Description x La composante x du vecteur directionnel. y La composante y du vecteur directionnel. Exemple : Le code du bloc d'action ci-dessus vérifie si une instance de l'objet "obj_Player" existe dans la pièce et si c'est le cas, il définit la direction de l'instance pour qu'elle pointe vers la position x/y de "obj_Player".
