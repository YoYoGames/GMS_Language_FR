Direction fixe Cette action est utilisée pour définir une direction de mouvement fixe. Les directions disponibles sont indiquées par les différents boutons du bloc d'action, et cliquer sur l'un d'entre eux permet de définir cette direction. Notez que les directions dans GameMaker Studio 2 sont mesurées en degrés, la droite correspondant à 0°, le haut à 90°, la gauche à 180° et le bas à 270°. Syntaxe de l'action : Exemple : Le code du bloc d'action ci-dessus vérifie si une pression sur le bouton gauche de la souris sur l'instance et, s'il en détecte une, il règle la direction sur le bas (270°) et la vitesse sur 5.
