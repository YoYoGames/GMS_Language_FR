Set Direction Random Cette action est utilisée pour régler la direction du mouvement de l'instance sur une ou plusieurs directions aléatoires choisies parmi les 8 points cardinaux. Vous pouvez cliquer sur une ou plusieurs des 8 flèches directionnelles dans l'action et l'action choisira l'une d'entre elles au hasard pour l'utiliser comme direction pour l'instance. à utiliser comme direction pour l'instance. Si vous ne sélectionnez aucune des flèches, l'action n'aura aucun effet sur la direction actuelle de l'instance. Syntaxe de l'action : Exemple : Le code du bloc d'action ci-dessus choisit soit true soit false et stocke le résultat dans une variable (locale) temporaire. Cette variable est ensuite vérifiée et si elle vaut true, un ensemble de directions aléatoires est choisi et si elle vaut false, un autre ensemble est choisi.
