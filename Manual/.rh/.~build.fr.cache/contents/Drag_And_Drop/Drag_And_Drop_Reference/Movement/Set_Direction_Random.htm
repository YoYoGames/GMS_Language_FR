Définir la direction au hasard Cette action permet de définir la direction du mouvement de l'instance sur une ou plusieurs directions aléatoires choisies parmi les 8 points cardinaux. Vous pouvez cliquer sur une ou plusieurs des 8 flèches directionnelles dans l'action et l'action choisira l'une d'entre elles au hasard pour l'utiliser comme direction de l'instance. Si vous ne sélectionnez aucune des flèches, l'action n'aura aucun effet sur la direction actuelle de l'instance. Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus choisit soit true ou false et stocke le résultat dans une variable temporaire (locale). Cette variable est ensuite vérifiée et si elle est true alors un ensemble de directions aléatoires est choisi et s'il false puis un autre ensemble est choisi.
