Définir la variable de direction Cette action est utilisée pour définir une direction pour l'instance qui n' est pas fixée à des incréments de 45°. Vous fournissez la valeur de la direction de 0° à 360° - où la droite est 0°, le haut est 90°, la gauche est 180° et le bas est 270° - et vous pouvez également définir l'indicateur relatif pour ajouter/soustraire la valeur que vous donnez à la direction actuelle. Syntaxe de l'action : Arguments : Arguments Description Direction Direction de 0° à 360° (les valeurs en dehors de la plage seront enveloppées). Exemple : Le code du bloc d'action ci-dessus vérifie si le bouton gauche de la souris est maintenu enfoncé. est maintenu enfoncé et si c'est le cas, il définit la direction sur une valeur aléatoire de 0 à 360 et la vitesse sur 5.
