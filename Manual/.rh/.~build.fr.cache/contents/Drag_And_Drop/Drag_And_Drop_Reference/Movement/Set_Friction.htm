Régler la friction Cette action est utilisée pour définir le "frottement" de l'instance. Le frottement signifie simplement qu'un montant spécifique sera déduit du vecteur de vitesse à chaque image de jeu, où vous spécifiez le montant à déduire. Habituellement, des valeurs plus petites fonctionnent mieux, comme 0,2, mais le montant réel dépendra de la configuration de votre projet. Si vous définissez l'indicateur relatif, la quantité donnée sera ajoutée à la valeur de friction actuelle au lieu de la définir. Syntaxe de l'action: Arguments: Argument Description friction Le frottement à appliquer (valeurs positives uniquement) Exemple: Le code de bloc d'action ci-dessus définit diverses propriétés d'occurrence liées au mouvement: la direction de la gravité est réglée vers le bas (270 °), la force de gravité est définie sur 0,05 et le frottement est défini sur 0,2.
