Bibliothèque d'action de mouvement Dans n'importe quel jeu, le mouvement et la position sont d'une importance primordiale et GameMaker Studio 2 dispose donc d'une sélection complète d'actions pour faire face à chaque situation. Les deux méthodes principales pour déplacer une instance sont soit de définir la position réelle (x / y) dans la pièce, soit de définir un vecteur vitesse / direction. Les actions présentées ici couvrent les deux méthodes et vous pouvez utiliser la variable d'affectation d'action commune pour définir directement vous-même la position x et y.
