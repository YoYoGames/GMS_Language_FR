Obtenir la position le long du chemin Cette action permet d'obtenir la position d'une instance le long d'un chemin. La valeur retournée est normalisée de 0 à 1, donc si vous obtenez un résultat de, par exemple, 0.5, l'instance se trouve exactement au milieu du chemin. Notez que vous devez avoir appelé l'action Start Following Path pour assigner un chemin à l'instance pour que cette action retourne une valeur autre que 0.
