Obtenir la position le long du chemin Cette action obtiendra la position d'une instance le long d'un chemin. La valeur renvoyée est normalisée de 0 à 1, donc si vous obtenez un résultat de, par exemple, 0,5, l'instance exactement au milieu du chemin. Notez que vous devez avoir appelé l' action Démarrer le chemin suivant pour attribuer un chemin à l'instance pour que cette action renvoie une valeur autre que 0. La valeur renvoyée sera stockée dans la variable cible que vous donnez, ou vous pouvez marquer la variable comme un variable locale temporaire pour la créer et attribuer la valeur. Syntaxe de l'action: Arguments: Argument Description Target La variable cible pour contenir la position de chemin renvoyée Exemple: Le code de bloc d'action ci-dessus obtient la position actuelle sur le chemin assigné pour l'instance et si elle est inférieure à 0,5, elle est définie sur 0,5.
