Définir la position le long du chemin Cette action définira la position d'une instance le long d'un chemin. La valeur est normalisée de 0 à 1, donc si vous la définissez sur, par exemple, 0,5, l'instance sera déplacée exactement au milieu du chemin. Notez que vous devez avoir appelé l' action Démarrer le chemin suivant pour attribuer un chemin à l'instance pour que cela ait un effet. Syntaxe de l'action: Arguments: Argument Description Position La position (de 0 à 1) le long du chemin pour placer l'instance Exemple: Le code de bloc d'action ci-dessus démarre l'instance le long d'un chemin, puis génère une valeur décimale aléatoire entre 0 et 1. Cette valeur est stockée dans une variable locale temporaire et utilisée pour définir la position de l'instance le long du chemin.
