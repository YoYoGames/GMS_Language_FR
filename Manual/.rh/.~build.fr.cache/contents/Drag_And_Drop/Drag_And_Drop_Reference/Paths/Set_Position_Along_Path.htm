Définir la position le long de la trajectoire Cette action définira la position d'une instance le long d'un chemin. La valeur est normalisée de 0 à 1, donc si vous la fixez, par exemple, à 0.5, l'instance sera déplacée exactement au milieu du chemin. Notez que vous devez avoir appelé l'action Start Following Path pour assigner un chemin à l'instance pour que cela ait un effet. Syntaxe de l'action : Arguments : Arguments Description Position La position (de 0 à 1) le long du chemin pour placer l'instance. Exemple : Le code du bloc d'action ci-dessus démarre l'instance le long d'un chemin et génère ensuite une valeur décimale aléatoire entre 0 et 1. une valeur décimale aléatoire entre 0 et 1. Cette valeur est stockée dans une variable locale temporaire et utilisée pour définir la position de l'instance le long du chemin.
