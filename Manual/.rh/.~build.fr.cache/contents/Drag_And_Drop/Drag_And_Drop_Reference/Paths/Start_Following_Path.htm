commence à suivre le chemin Cette action démarrera une instance suivant un chemin qui a été créé dans le navigateur d'actifs. Vous utiliserez ensuite cette action pour indiquer à votre instance le chemin à suivre, la vitesse à laquelle elle doit suivre le chemin (mesurée en pixels par image de jeu), comment elle doit se comporter lorsqu'elle atteint la fin du chemin et si elle doit suivre la position absolue ou relative du chemin. comment se comporter lorsqu'elle atteint la fin du chemin, et si elle doit suivre la position absolue ou relative du chemin. Lorsque vous définissez la vitesse, vous pouvez utiliser des valeurs négatives pour signifier que l'instance doit suivre le chemin en sens inverse et il est intéressant de noter que la vitesse que vous définissez ici peut être modifiée par le modificateur de vitesse défini dans l'éditeur de chemin pour chaque point du chemin, ainsi si vous définissez la vitesse du chemin à 2 et que dans le chemin un des points a un modificateur de vitesse de 200%, alors la vitesse réelle à ce point sera de 4 pixels par image de jeu. Une fois que vous avez défini la vitesse initiale du chemin, vous pouvez définir comment l'instance doit se comporter lorsqu'elle atteint la fin du chemin à partir des options suivantes (notez que lorsqu'une instance atteint la fin d'un chemin, cela déclenche également un événement Other - Path Ended) : Stop: Termine le chemin, en arrêtant l'instance sur le dernier point du chemin. Redémarrer: Continuer le chemin depuis le début, en revenant à la position de départ si le chemin n'est pas marqué comme fermé dans l'éditeur de chemin. Boucle: Continuer à partir de la position actuelle pour parcourir à nouveau le chemin. Inverser: Revenir en arrière le long du chemin (en inversant la vitesse de suivi du chemin), et lorsqu'il atteindra le début du chemin, il fera à nouveau marche arrière, etc... Enfin, vous pouvez indiquer que le chemin est absolu ou non. Par défaut, l'instance commencera à suivre le chemin à partir de sa position actuelle (position relative), mais si vous cochez la case Absolute, elle commencera à suivre le chemin exactement comme vous l'avez conçu et placé dans l'éditeur de chemin (position absolue). chemin exactement comme vous l'avez conçu et placé dans l'éditeur de chemin (la position absolue).
