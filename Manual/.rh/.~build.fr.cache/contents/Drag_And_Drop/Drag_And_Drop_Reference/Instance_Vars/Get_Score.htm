Obtenez le score Vous pouvez utiliser cette action pour récupérer la valeur de la variable d'instance créée pour le "score". Vous fournissez la variable cible pour stocker la valeur de «score» renvoyée (qui peut être marquée comme une variable locale temporaire), mais notez que si la variable n'a pas été définie précédemment (à l'aide de Set Score ), la variable sera créée pour vous et l'action renverra 0 (contrairement aux variables d'instance régulières qui donneront une erreur si vous essayez d'y accéder sans les avoir définies au préalable). Syntaxe de l'action: Arguments: Argument Description Target La variable cible pour stocker la valeur de «score» renvoyée (peut être marquée comme une variable locale temporaire) Exemple: Le code de bloc d'action ci-dessus récupérera la valeur de score actuelle et la stockera dans une variable locale temporaire. Cette variable est ensuite convertie en chaîne et stockée dans une autre variable locale temporaire avant d'être dessinée à l'écran.
