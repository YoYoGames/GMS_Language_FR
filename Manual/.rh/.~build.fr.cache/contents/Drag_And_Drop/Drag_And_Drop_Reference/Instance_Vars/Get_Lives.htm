Obtenez des vies Vous pouvez utiliser cette action pour récupérer la valeur de la variable d'instance créée pour les "lives". Vous fournissez la variable cible pour stocker la valeur "lives" retournée (qui peut être marquée comme une variable locale temporaire), mais notez que si la variable n'a pas été définie précédemment (à l'aide de Set Lives ), la variable sera créée pour vous et l'action renverra 0 (contrairement aux variables d'instance régulières qui donneront une erreur si vous essayez d'y accéder sans les avoir définies au préalable). Syntaxe de l'action: Arguments: Argument Description Target La variable cible pour stocker la valeur "lives" renvoyée (peut être marquée comme une variable locale temporaire) Exemple: Le code de bloc d'action ci-dessus récupérera la valeur de "lives" et la stockera dans une variable locale temporaire. Cette variable est ensuite utilisée pour décider du nombre de sprites empilés à dessiner.
