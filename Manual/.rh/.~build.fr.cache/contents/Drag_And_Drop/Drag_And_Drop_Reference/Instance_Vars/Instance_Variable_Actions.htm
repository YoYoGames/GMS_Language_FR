Variable d'instance Bibliothèque d'action Les actions Instance Variables sont conçues pour vous aider à démarrer lorsque vous créez un jeu et sont utilisées pour créer et maintenir des variables d'instance spécifiques pour le score, les vies et la santé (vous pouvez en savoir plus sur la portée des variables ici). ici). Ces actions créeront ces variables pour vous à chaque fois qu'elles seront utilisées et sont plus destinées à la commodité ou aux débutants. Si vous avez besoin d'un contrôle plus complexe des valeurs du jeu, vous devriez créer vos propres variables en utilisant les actions appropriées (voir ici).
