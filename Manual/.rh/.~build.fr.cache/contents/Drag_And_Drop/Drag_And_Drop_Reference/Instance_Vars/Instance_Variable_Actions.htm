Bibliothèque d'actions de variable d'instance Les actions Variables d'instance sont conçues pour vous aider à démarrer lors de la création d'un jeu et sont utilisées pour créer et gérer des variables d'étendue d'instance spécifiques pour score, lives et health (vous pouvez en savoir plus sur la portée des variables ici ). Ces actions créeront ces variables pour vous à tout moment elles sont utilisées et sont destinées davantage à des raisons de commodité ou aux débutants, et si vous avez besoin d'un contrôle plus complexe sur les valeurs du jeu, vous devriez créer vos propres variables en utilisant les actions appropriées (voir ici ) Les actions suivantes existent pour utiliser ces variables d'instance: Définir le scoreObtenez le scoreDéfinir des viesObtenez des viesDéfinir la santéObtenir de la santéSi ScoreSi vitSi la santé Retour: Index Suivant: Actions diverses © Copyright YoYo Games Ltd. 2021 Tous droits réservés
