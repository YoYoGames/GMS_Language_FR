Set Lives Avec cette action, vous pouvez créer ou définir une variable d'instance pour les vies. Vous fournissez la valeur à laquelle définir la variable lives ou à ajouter à la variable lives (si vous cochez la case relative, la valeur sera ajoutée à la valeur actuelle de "lives"). actuelle). Notez que contrairement aux variables d'instance ordinaires, le nom de la variable "lives" est stocké en interne. Pour récupérer la valeur du score, vous devez donc utiliser l'action Get Lives.
