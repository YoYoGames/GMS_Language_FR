Obtenir de la santé Vous pouvez utiliser cette action pour récupérer la valeur de la variable d'instance créée pour la "santé" de l'objet. Vous fournissez la variable cible pour stocker la valeur «santé» retournée (qui peut être marquée comme une variable locale temporaire), mais notez que si la variable n'a pas été définie précédemment (à l'aide de Set Health ), la variable sera créée pour vous et l'action renverra 0 (contrairement aux variables d'instance régulières qui donneront une erreur si vous essayez d'y accéder sans les avoir définies au préalable). Syntaxe de l'action: Arguments: Argument Description Target La variable cible pour stocker la valeur «santé» renvoyée (peut être marquée comme une variable locale temporaire) Exemple: Le code de bloc d'action ci-dessus récupérera la valeur de «santé» et la stockera dans une variable locale temporaire.
