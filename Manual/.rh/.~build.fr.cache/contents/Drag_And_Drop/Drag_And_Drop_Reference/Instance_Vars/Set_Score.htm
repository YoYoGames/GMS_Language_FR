Définir le score Avec cette action, vous pouvez créer ou définir une variable d'instance pour le score. Vous fournissez la valeur pour définir la variable de score ou pour l'ajouter à la variable de score (si vous cochez la case relative, la valeur sera ajoutée à la valeur de «score» actuelle). Notez que contrairement aux variables d'instance régulières, le nom de la variable "score" est stocké en interne. Pour récupérer la valeur du score, vous devez utiliser l'action Obtenir le score. Syntaxe de l'action: Arguments: Argument Description Score Le montant auquel définir le score (ou à ajouter au score si relatif est coché) Exemple: Le code de bloc d'action ci-dessus vérifiera si la salle est la première du jeu et si c'est le cas, il définit les variables «score», «lives» et «health» pour l'instance.
