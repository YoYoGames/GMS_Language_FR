Séquence de lecture Cette action vous permet de lancer la lecture d'une séquence (précédemment mise en pause). Vous fournissez l'ID de l'élément de séquence (tel que renvoyé par l'action Create Sequence), et l'action lancera la lecture. Syntaxe de l'action : Arguments : Arguments Description Variable L'ID de l'élément de séquence à lire. Exemple : Le code du bloc d'action ci-dessus vérifie si une variable globale est vraie. Si c'est le cas, l'élément de séquence référencé dans la variable "my_seq" est mis en pause, sinon il est mis en lecture.
