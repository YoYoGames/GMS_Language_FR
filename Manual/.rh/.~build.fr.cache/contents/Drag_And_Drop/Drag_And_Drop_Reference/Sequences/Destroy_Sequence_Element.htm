Destroy Sequence Element Vous pouvez appeler cette action chaque fois que vous souhaitez "détruire" un élément de séquence précédemment créé. Vous donnez l'ID de l'élément, tel qu'il a été renvoyé par l'action Create Sequence Element (Créer un élément de séquence). le détruira, en le retirant de la pièce. Syntaxe de l'action : Arguments : Arguments Description Variable L'ID de l'élément de séquence à détruire. Exemple : Le code du bloc d'action ci-dessus vérifie si l'élément de séquence stocké dans la variable dans la variable "my_seq" existe et si c'est le cas, il le détruit ensuite.
