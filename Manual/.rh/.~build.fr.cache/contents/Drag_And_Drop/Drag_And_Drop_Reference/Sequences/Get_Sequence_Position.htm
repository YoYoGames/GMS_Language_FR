Obtenir la position de la séquence Cette action vous permet de récupérer la position actuelle de l'élément de séquence donné dans la pièce. Vous fournissez l'ID de l'élément de séquence (tel qu'il est renvoyé par l'action Create Sequence), et cette action renverra sa position X et Y dans des variables cibles. sa position X et Y dans des variables cibles. Notez que les variables cibles peuvent être marquées comme des variables temporaires (locales), qui seront créées pour l'action et pourront être utilisées dans toutes les actions suivantes pour l'événement dans lequel l'action se trouve.
