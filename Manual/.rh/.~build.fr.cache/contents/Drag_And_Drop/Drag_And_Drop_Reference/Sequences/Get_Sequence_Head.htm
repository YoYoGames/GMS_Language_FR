Get Sequence Head Cette action vous permet de récupérer la position actuelle de la tête de lecture de l'élément de séquence donné dans la pièce. Vous fournissez l'ID de l'élément de séquence (tel que renvoyé par l'action Create Sequence), et cette action renvoie l'image actuelle sur laquelle se trouve la tête de lecture de la séquence, dans la variable cible. Pour plus d'informations sur la tête de lecture, reportez-vous à la section Utilisation de la feuille de calcul. Notez que la variable cible peut être marquée comme temporaire (locale), qui sera créée pour l'action et pourra être utilisée dans toutes les actions suivantes pour l'événement dans lequel l'action se trouve. Syntaxe de l'action : Arguments : Arguments Description Variable L'ID de l'élément de la séquence pour obtenir la position de la tête. Cible La variable cible pour maintenir la position de la tête. Exemple : Le bloc d'action ci-dessus récupère la position de tête actuelle de l'ID de l'élément de séquence stocké dans la la variable "seq_id". Cette valeur est ensuite utilisée pour définir la position de tête du même élément de séquence, en l'avançant de 4 images.
