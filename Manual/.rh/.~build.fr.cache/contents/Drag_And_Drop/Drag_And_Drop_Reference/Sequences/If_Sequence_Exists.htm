Si la séquence existe Cette action peut être utilisée pour vérifier si l'élément de séquence donné existe dans la salle de jeu. Vous donnez l'ID de l'élément, tel que retourné par l'action Create Sequence, et si l'élément de séquence existe dans la salle, alors l'action retournera true sinon elle retournera false. Si vous indiquez l'argument " Not ", l'action vérifiera si un tel élément de séquence n'existe pas et si aucun n'est trouvé, elle retournera true. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous : Ces actions seront désormais exécutées si le bloc "if" vaut true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if".
