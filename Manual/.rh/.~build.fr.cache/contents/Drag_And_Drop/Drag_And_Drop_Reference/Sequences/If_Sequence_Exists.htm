Si la séquence existe Cette action peut être utilisée pour vérifier si l'élément de séquence donné existe dans la salle de jeu. Vous donnez l'ID d'élément, tel que renvoyé par l'action Créer une séquence, et si l'élément de séquence existe dans la salle, l'action retournera true sinon il reviendra false. Si vous marquez l' argument "Non ", alors l'action vérifiera si aucun élément de séquence n'existe et si aucun n'est trouvé, il retournera true. Notez que pour ajouter des actions dans le "if ", ils doivent être déposés sur le côté de l'action, comme indiqué dans l'image ci-dessous: Ces actions seront désormais exécutées si le "if "évalue à true, tandis que toutes les actions déposées ailleurs seront exécutées après le "if " bloquer.
