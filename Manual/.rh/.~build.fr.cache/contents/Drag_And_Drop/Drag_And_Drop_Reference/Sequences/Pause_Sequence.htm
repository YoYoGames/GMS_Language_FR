Séquence de lecture Cette action vous permet de mettre en pause une séquence en cours de lecture. Vous fournissez l'ID de l'élément de la séquence (tel que renvoyé par l'action Create Sequence), et l'action interrompra la lecture. Syntaxe de l'action : Arguments : Arguments Description Variable L'ID de l'élément de séquence à mettre en pause. Exemple : Le code du bloc d'action ci-dessus vérifie si une variable globale est vraie. Si c'est le cas, l'élément de séquence référencé dans la variable "my_seq" est mis en pause, sinon il est mis en lecture.
