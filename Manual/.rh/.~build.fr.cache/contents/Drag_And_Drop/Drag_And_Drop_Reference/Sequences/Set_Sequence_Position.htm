Définir la position de la séquence Avec cette action, vous pouvez définir la position de l'élément de séquence donné dans la pièce. Vous fournissez l'ID d'élément de séquence (tel que renvoyé par l'action Créer une séquence ), puis les nouvelles positions X et Y. Si vous cochez la case "relative", alors les valeurs données pour X et Y seront ajoutées à la position actuelle de la séquence, sinon elles seront une position absolue. Syntaxe de l'action: Arguments: Argument Description Variable ID de l'élément de séquence pour lequel définir la position. X La nouvelle position x dans la pièce pour la séquence. Y La nouvelle position y dans la pièce pour la séquence. Exemple: Le code de bloc d'action ci-dessus récupère la position actuelle de l'ID d'élément de séquence stocké dans la variable "my_seq ". Ces valeurs sont ensuite utilisées pour définir la position de la séquence, en la déplaçant de 10 pixels vers la droite.
