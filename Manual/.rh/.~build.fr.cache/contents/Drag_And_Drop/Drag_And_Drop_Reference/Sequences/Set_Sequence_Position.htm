Position de la séquence de réglage Cette action vous permet de définir la position de l'élément de séquence donné dans la pièce. Vous fournissez l'ID de l'élément de séquence (tel que renvoyé par l'action Créer une séquence), puis la nouvelle position X et Y. Si vous cochez la case "relative", les valeurs données pour X et Y seront ajoutées à la position actuelle de la séquence, sinon il s'agira d'une position absolue. Syntaxe de l'action : Arguments : Arguments Description Variable L'ID de l'élément de la séquence dont il faut définir la position. X La nouvelle position x dans la pièce pour la séquence. Y La nouvelle position y dans la pièce pour la séquence. Exemple : Le code du bloc d'action ci-dessus récupère la position actuelle de l'élément de séquence ID stockée dans la variable "my_seq". Ces valeurs sont ensuite utilisées pour définir la position de la séquence, en la déplaçant de 10 pixels vers la droite.
