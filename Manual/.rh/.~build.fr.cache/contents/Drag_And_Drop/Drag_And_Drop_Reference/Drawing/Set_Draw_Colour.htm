Set Draw Colour Avec cette action, vous pouvez définir la couleur à utiliser pour toutes les formes (rectangle/ellipse/ligne) ou pour tout texte qui est dessiné par la suite. Notez que cette action ne doit pas être appelée avant chaque action de dessin, et peut être définie une seule fois au début du jeu. Par exemple, si tout votre texte doit être jaune, vous pouvez appeler cette action dans l'événement de création de la première pièce du jeu, et tout le texte (et les formes) seront alors dessinés en jaune à partir de ce moment-là. Cependant, si vous avez besoin d'utiliser plusieurs couleurs différentes, vous devrez appeler cette action avant chaque texte ou forme qui nécessite une couleur différente. L'action comporte une case à cocher en bas de page pour Use alpha. Cela permet d'utiliser la composante alpha de la couleur pour définir l'alpha du dessin (de la même manière que l'action Set Draw Alpha).
