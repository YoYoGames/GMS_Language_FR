Définir la couleur de dessin Avec cette action, vous pouvez définir la couleur à utiliser pour toutes les formes (rectangle / ellipse / ligne) ou pour tout texte dessiné ultérieurement. Notez que cela ne doit pas être appelé avant chaque action de tirage et ne peut être défini qu'une seule fois au début de la partie. Par exemple, si tout votre texte va être jaune, vous pouvez appeler cette action dans l'événement Créer de la première salle du jeu, et tout le texte (et les formes) seront alors dessinés en jaune à partir de là. Cependant, si vous devez utiliser plusieurs couleurs différentes, vous devrez l'appeler avant chaque texte ou forme nécessitant une couleur différente. L'action a une case à cocher en bas pour Utiliser alpha. Cela permet également d'utiliser le composant alpha de la couleur pour définir l'alpha de dessin (de la même manière que l'option Set Draw Alpha. Si vous ne souhaitez pas inclure le composant alpha et avoir simplement une couleur unie, cela peut être désactivé ( il est activé par défaut).
