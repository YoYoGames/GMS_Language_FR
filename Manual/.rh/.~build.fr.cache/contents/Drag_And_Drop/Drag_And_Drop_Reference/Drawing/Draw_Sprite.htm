Dessiner Sprite Cette action dessinera un sprite quelque part dans la pièce. Vous spécifiez le sprite, la position (soit une position absolue dans la pièce, soit une position relative par rapport à l'occurrence faisant le dessin) et le cadre d'animation du sprite, s'il a plusieurs cadres (notez que les cadres d'image sont numérotés de 0 vers le haut ). Si vous avez un sprite animé et que vous souhaitez qu'il dessine les images comme s'il était affecté à l'objet, utilisez -1 pour l'index d'image ou la variable d'instance intégrée image_index, mais notez que si l'occurrence n'a pas de sprite assigné, elle ne s'animera pas, et si le sprite assigné a un nombre d'images différent de celui en cours de dessin, alors le sprite sera dessiné avec des images supplémentaires ou des images manquées. IMPORTANT: cette action ne dessine pas le sprite transformé, même si vous avez modifié le mélange ou l'échelle de l'image. Pour cela, consultez l'action Dessiner un sprite transformé. REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de tirage au sort et ne tirera rien si elle est utilisée ailleurs.
