Obtenez Draw Alpha Avec cette action, vous pouvez obtenir le dessin général alpha utilisé. L'action renverra l'alpha actuel à la variable cible que vous spécifiez, qui peut avoir été créée précédemment ou peut être une nouvelle temporaire (si vous cochez la case "Temp"). Syntaxe de l'action: Exemple: Le bloc d'action ci-dessus obtient l'alpha de dessin actuel et s'il n'est pas défini sur 1 (complètement opaque), il est défini sur cette valeur.
