Draw Sprite transformé Cette fonction dessinera le sprite donné de façon similaire à l'action Draw Sprite mais avec des options supplémentaires pour changer l'échelle, le mélange, la rotation et le cadre du sprite dessiné. Le changement de ces valeurs ne modifie pas la ressource de quelque manière que ce soit (seulement la manière dont elle est dessinée), et vous pouvez utiliser n'importe quelle variable de propriété de sprite disponible au lieu de valeurs directes pour tous les arguments de la fonction. Cependant, tout mélange de cette manière crée un sprite dupliqué qui est ensuite stocké dans la base de données. sprite qui est ensuite stocké dans le cache et utilisé au besoin. C'est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de votre jeu, à moins que vous n'activiez WebGL. REMARQUE: Cette action est uniquement destinée à être utilisée dans les différents événements de tirage au sort, et ne tirera rien si elle est utilisée ailleurs. Syntaxe de l'action : Arguments : Arguments Description Sprite Le sprite à dessiner Cadre L'image du sprite à dessiner (utilisez la variable intégrée image_index pour l'image actuelle). X La position x à laquelle dessiner dans la pièce Y La position y pour dessiner dans la pièce Échelle X Le facteur d'échelle pour l'axe X (utilisez la variable intégrée image_xscale pour l'échelle X actuelle) Échelle Y Le facteur d'échelle pour l'axe Y (utilisez la variable intégrée image_yscale pour l'échelle Y actuelle) Rotation L'angle de dessin du sprite (de 0 à 360, en tournant dans le sens inverse des aiguilles d'une montre, 0 étant à droite - utilisez la variable image_angle intégrée pour connaître la rotation actuelle). Couleur La couleur à mélanger avec le sprite (la valeur par défaut est le blanc, et vous pouvez utiliser la variable image_blend intégrée pour connaître la couleur actuelle).
