Dessiner une valeur transformée Avec cette action, vous pouvez dessiner une légende de chaîne et / ou une variable ou une valeur à l'écran. L'argument caption est une chaîne que vous souhaitez ajouter avant la valeur, et peut être laissé vide si nécessaire, et la valeur peut être une variable, une valeur fixe ou une autre chaîne - auquel cas elle doit être placée entre guillemets et peut également être laissé vide si nécessaire. Vous indiquez ensuite la position où la chaîne doit être dessinée, soit en tant que position absolue, soit en tant que position par rapport à l'occurrence dessinant le texte. De plus, vous pouvez définir l'échelle du texte, en l'étirant le long de l'axe horizontal ou vertical, ainsi que la rotation (les angles de rotation du texte sont dans le sens inverse des aiguilles d'une montre, donc le régler à 90 ° le fera pivoter vers la gauche et -90 ° à droite). La mise à l'échelle est effectuée par le facteur donné, donc une échelle de 0,5 fera la moitié de la largeur du texte et une échelle de 2 doublera la largeur, 1 étant la valeur par défaut pour aucune mise à l'échelle. Notez que l'apparence du texte une fois dessiné dépendra de la police utilisée, de la couleur utilisée et de l'alignement. Toutes ces choses peuvent être modifiées avant de dessiner, comme le montre l'exemple ci-dessous. REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de tirage au sort et ne tirera rien si elle est utilisée ailleurs. Syntaxe de l'action: Arguments: Argument Description Caption La légende à dessiner (une chaîne), peut être laissée vide Value La valeur à dessiner (réelle, variable ou chaîne) x La position x à laquelle dessiner y La position y à laquelle dessiner xscale Le facteur d'échelle x (horizontal), la valeur par défaut est 1 yscale Le facteur d'échelle y (vertical), la valeur par défaut est 1 Rotation L'angle de rotation pour dessiner le texte Exemple: Le code de bloc d'action ci-dessus définit une couleur, une police, puis aligne le texte à dessiner à partir du centre avant de dessiner la valeur "current_time" avec la légende "TIME:", redimensionnée pour être le double de la taille et "vibrer" légèrement en raison au changement d'angle aléatoire.
