Obtenir la police de dessin Avec cette action, vous pouvez obtenir la valeur d'ID de la police actuellement affectée au dessin. La valeur renvoyée sera soit -1 pour aucune police attribuée, soit une valeur> = 0 qui représente une constante de police du navigateur de ressources. La valeur de retour sera stockée dans la variable cible que vous spécifiez, qui peut avoir été créée précédemment ou peut être une nouvelle variable temporaire (si vous cochez la case "Temp"). Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus vérifie la police actuelle utilisée pour le dessin, et s'il ne s'agit pas de l'actif donné, il est défini sur cet actif.
