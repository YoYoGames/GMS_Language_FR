Définir l'alignement du texte Cette action définira l'alignement de la police pour toutes les actions suivantes de dessin de texte. Vous pouvez définir l'alignement horizontal sur la gauche, le centre ou la droite, et l'alignement vertical sur le haut, le milieu ou le bas et le texte sera aligné par rapport à la position x / y de l'action de dessin, comme indiqué dans les images ci-dessous: Le texte est aligné horizontalement à gauche Le texte est aligné horizontalement au centre Le texte est aligné horizontalement à droite Le texte est aligné verticalement vers le haut Le texte est aligné verticalement au milieu Le texte est aligné verticalement vers le bas Notez que cela définira l'alignement à utiliser pour tous les appels de texte de dessin ultérieurs, et peut donc être appelé, par exemple, une fois au début du jeu - dans tous les cas, pas seulement l'événement de dessin - et toutes les actions de dessin de texte utiliseront l'alignement défini. Cependant, si vous avez besoin de plusieurs actions de dessin de texte pour s'aligner de différentes manières, vous devrez appeler cette action avant chaque action de dessin de texte que vous souhaitez aligner différemment. Syntaxe de l'action: Arguments: Argument Description H.Align La valeur d'alignement horizontal (gauche, centre, droite) V.Align La valeur d'alignement vertical (haut, milieu, bas) Exemple: Le code de bloc d'action ci-dessus définit la police, la couleur de dessin et l'alignement du texte dessiné.
