Tracer une ligne Cette action va dessiner une ligne à une position donnée dans la pièce. Vous donnez la position initiale x/y et ensuite la position finale x/y et la ligne sera dessinée entre les deux en utilisant la couleur de dessin actuelle. Les positions peuvent être soit une position absolue dans la room, ou une position relative à l'instance appelant l'action. REMARQUE: Cette action est uniquement destinée à être utilisée dans les différents événements de tirage au sort, et ne tirera rien si elle est utilisée ailleurs. Syntaxe de l'action : Arguments : Arguments Description x La position x de départ y La position de départ en y x2 La position x d'arrivée y2 La position d'arrivée y Exemple : Le code du bloc d'action ci-dessus dessine une ligne rose en haut de la pièce.
