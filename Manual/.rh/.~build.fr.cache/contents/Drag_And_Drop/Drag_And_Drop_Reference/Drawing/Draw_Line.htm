Dessiner une ligne Cette action dessinera une ligne à une position donnée dans la pièce. Vous donnez la position x / y initiale, puis la position x / y finale et la ligne sera tracée entre eux en utilisant la couleur de dessin actuelle. Les positions peuvent être soit une position absolue dans la pièce, soit une position relative à l'instance appelant l'action. REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de tirage au sort et ne tirera rien si elle est utilisée ailleurs. Syntaxe de l'action: Arguments: Argument Description x La position x de départ y La position y de départ x2 La position x de finition y2 La position y d'arrivée Exemple: Le code du bloc d'action ci-dessus trace une ligne rose en haut de la pièce.
