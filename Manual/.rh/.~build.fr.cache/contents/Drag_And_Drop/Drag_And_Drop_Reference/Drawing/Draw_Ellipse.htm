Dessiner une ellipse Cette action dessinera une ellipse à une position donnée dans la pièce. Vous donnez la position en haut à gauche et en bas à droite de la zone dans laquelle l'ellipse doit "s'insérer", et l'ellipse sera dessinée entre elles. Les positions peuvent être soit une position absolue dans la pièce, soit une position relative à l'instance appelant l'action, et vous pouvez également définir si l'ellipse peut être remplie ou délimitée en cochant la case de remplissage en bas. REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de tirage au sort et ne tirera rien si elle est utilisée ailleurs. Syntaxe de l'action: Arguments: Argument Description Left La position de gauche à partir de laquelle commencer à dessiner Top La position supérieure à partir de laquelle commencer à dessiner Right La bonne position pour terminer le dessin Bottom La position inférieure pour terminer le dessin à partir de Exemple: Le code de bloc d'action ci-dessus dessine une série d'ellipses concentriques soulignées à la position de l'instance.
