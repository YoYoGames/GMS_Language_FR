Dessiner une ellipse de dégradé Cette action dessinera une ellipse à une position donnée dans la pièce, en utilisant un ensemble de couleurs de fusion pour créer un dégradé. Vous donnez la position en haut à gauche et en bas à droite de la zone dans laquelle l'ellipse doit "s'insérer", et l'ellipse sera dessinée entre elles. La position peut être soit une position absolue dans la pièce, soit une position par rapport à l'occurrence appelant l'action et vous pouvez définir les couleurs pour qu'elles se fondent à partir du bord de l'ellipse et de son centre. L'ellipse peut être dessinée remplie ou délimitée en cochant la case de remplissage en bas. REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de tirage au sort et ne tirera rien si elle est utilisée ailleurs. Syntaxe de l'action: Arguments: Argument Description Left La position de gauche à partir de laquelle commencer à dessiner Top La position supérieure à partir de laquelle commencer à dessiner Right La bonne position pour terminer le dessin Bottom La position inférieure pour terminer le dessin à partir de Center La couleur à mélanger à partir du centre Edge La couleur à mélanger à partir du bord Exemple: Le code du bloc d'action ci-dessus dessine une ellipse de gradient à la position de l'instance de "obj_Player".
