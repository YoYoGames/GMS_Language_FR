Dessiner un rectangle Cette action dessinera un rectangle à une position donnée dans la pièce. Vous donnez la position en haut à gauche et en bas à droite et le rectangle sera dessiné entre elles, et la position peut être soit une position absolue dans la pièce, soit une position par rapport à l'instance appelant l'action. Vous pouvez également définir si le rectangle peut être rempli ou délimité en cochant la case de remplissage en bas. REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de tirage au sort et ne tirera rien si elle est utilisée ailleurs. Syntaxe de l'action: Arguments: Argument Description Left La position de gauche à partir de laquelle commencer à dessiner Top La position supérieure à partir de laquelle commencer à dessiner Right La bonne position pour terminer le dessin Bottom La position inférieure pour terminer le dessin à partir de Exemple: Le code de bloc d'action ci-dessus dessine deux rectangles, un vert et rempli et un jaune et un contour uniquement, à une position relative à l'instance appelant les actions.
