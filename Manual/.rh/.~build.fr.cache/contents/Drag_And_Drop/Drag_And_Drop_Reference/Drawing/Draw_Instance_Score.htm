Dessiner le score d'instance Avec cette action, vous pouvez dessiner la valeur de score de l'instance. Lorsque vous utilisez une action de score dans GameMaker Studio 2 l'instance qui l'appelle aura une nouvelle variable d'étendue d'instance appelée score ajoutée. Cette variable peut ensuite être vérifiée et définie comme vous le feriez pour toute autre variable. Dans le cas de cette action, la variable sera dessinée à l'écran à la position donnée, avec tout texte de légende que vous souhaitez fournir. Le texte de la légende sera dessiné avant la valeur du score et s'il s'agit de texte, il doit être entouré de "", et les deux seront formatés en utilisant la police de dessin, la couleur et l'alignement actuels. REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de tirage au sort et ne tirera rien si elle est utilisée ailleurs. Syntaxe de l'action: Arguments: Argument Description Caption La chaîne à dessiner avec la valeur du score (précédente et peut être laissée vide) x La position x à laquelle dessiner le score y La position y à laquelle dessiner le score Exemple: Le code de bloc d'action ci-dessus dessine le score en utilisant le formatage de tirage donné.
