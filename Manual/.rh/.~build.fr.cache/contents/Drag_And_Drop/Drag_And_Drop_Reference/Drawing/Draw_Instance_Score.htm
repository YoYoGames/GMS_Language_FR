Score de l'instance de dessin Cette action vous permet de dessiner la valeur du score de l'instance. Lorsque vous utilisez une action de score dans GameMaker Studio 2, une nouvelle variable de portée d'instance appelée score est ajoutée à l'instance qui l'appelle. Cette variable peut ensuite être vérifiée et définie comme n'importe quelle autre variable. Dans le cas de cette action, la variable sera affichée à l'écran à la position donnée, ainsi que tout texte de légende que vous souhaitez fournir. Le texte de la légende sera dessiné avant la valeur du score et, s'il s'agit d'un texte, il doit être enveloppé dans une enveloppe. Si c'est du texte, il doit être entouré de "", et les deux seront formatés en utilisant la police, la couleur et l'alignement actuels du dessin. REMARQUE: Cette action est uniquement destinée à être utilisée dans les différents événements de tirage au sort, et ne tirera rien si elle est utilisée ailleurs. Syntaxe de l'action : Arguments : Arguments Description Légende La chaîne à dessiner avec la valeur du score (précédent et peut être laissé vide) x La position en x pour dessiner la partition y La position y pour dessiner la partition Exemple : Le code du bloc d'action ci-dessus dessine le score en utilisant le formatage de dessin donné.
