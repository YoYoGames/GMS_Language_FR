Dessiner des Sprites empilés Cette action dessinera un sprite un nombre donné de sprites les uns après les autres à une position donnée dans la salle. Vous donnez le sprite à dessiner et l'ordre de la pile, qui peut être soit une ligne (horizontalement, de gauche à droite), soit une colonne (verticalement, de haut en bas), ainsi que le nombre de sprites à dessiner et la position. La position peut être une position absolue dans la pièce, ou une position relative à la position de l'occurrence faisant le dessin, et l'espacement entre les images sera basé sur la largeur ou la hauteur de l'image-objet. Notez que cela dessine simplement une image statique - l'image unique initiale (image 0) du sprite donné - et toutes les autres images seront ignorées, de même que toutes les transformations qui ont été ajoutées en changeant les variables d'instance (comme image_xscale ou image_blend ). REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de tirage au sort et ne tirera rien si elle est utilisée ailleurs. Syntaxe de l'action: Arguments: Argument Description Sprite Le sprite à dessiner Stack Order L'ordre dans lequel dessiner (ligne ou colonne) Number Le nombre de sprites à dessiner x La position x à laquelle dessiner y La position y à laquelle dessiner Exemple: Le code de bloc d'action ci-dessus obtient le nombre d'instances de l'objet "obj_Player" et l'utilise ensuite pour dessiner un certain nombre de sprites marqueurs à l'écran.
