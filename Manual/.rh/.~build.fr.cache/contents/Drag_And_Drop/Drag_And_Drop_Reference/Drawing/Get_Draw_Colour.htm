Obtenir la couleur de dessin Avec cette action, vous pouvez obtenir la couleur utilisée pour toutes les formes (rectangle / ellipse / ligne) ou pour tout texte dessiné. L'action renverra la couleur à la variable cible que vous spécifiez, qui peut avoir été créée précédemment ou peut être une nouvelle temporaire (si vous cochez la case "Temp"). Syntaxe de l'action: Exemple: Le bloc d'action ci-dessus obtient la couleur de dessin actuelle et s'il n'est pas défini sur $ffffff (blanc) il est réglé sur cette couleur.
