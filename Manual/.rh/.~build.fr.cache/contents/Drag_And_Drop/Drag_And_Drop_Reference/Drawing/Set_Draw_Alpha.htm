Set Draw Alpha Avec cette action, vous pouvez définir l'alpha à utiliser pour toutes les formes (rectangle/ellipse/ligne) ou pour tout texte qui est dessiné par la suite. Notez que cette action ne doit pas être appelée avant chaque action de dessin, et qu'elle peut être définie une seule fois au début du jeu. Alpha est le paramètre qui contrôle le degré de "transparence" des objets dessinés et est basé sur une échelle de 0 à 1, où 0 est totalement transparent et 1 est totalement opaque. Syntaxe de l'action : Arguments : Arguments Description Alpha La valeur alpha à utiliser (de 0 à 1, la valeur par défaut est 1) Exemple : Le code du bloc d'action ci-dessus définit l'alpha du tirage à une valeur aléatoire entre 0 et 1.
