Définir Draw Alpha Avec cette action, vous pouvez définir l'alpha à utiliser pour toutes les formes (rectangle / ellipse / ligne) ou pour tout texte dessiné ultérieurement. Notez que cela ne doit pas être appelé avant chaque action de tirage et ne peut être défini qu'une seule fois au début de la partie. Alpha est le paramètre qui contrôle la façon dont les objets "transparents" seront dessinés et est basé sur une échelle de 0 à 1, où 0 est totalement transparent et 1 est totalement opaque. Syntaxe de l'action: Arguments: Argument Description Alpha La valeur alpha à utiliser (de 0 à 1, la valeur par défaut est 1) Exemple: Le code de bloc d'action ci-dessus définit l'alpha de tirage sur une valeur aléatoire comprise entre 0 et 1.
