Dessiner un rectangle dégradé Cette action va dessiner un rectangle à une position donnée dans la pièce, en utilisant un ensemble de couleurs de mélange pour créer un gradient. Vous donnez la position supérieure gauche et la position inférieure droite et le rectangle sera dessiné entre les deux. La position peut être soit une position absolue dans la pièce, soit une position relative à l'instance qui appelle l'action. Vous définissez ensuite les couleurs à mélanger à partir de chacun des coins et vous pouvez également définir si le rectangle peut être rempli ou délimité en cochant la case en cochant la case de remplissage en bas. REMARQUE: Cette action est uniquement destinée à être utilisée dans les différents événements de tirage au sort, et ne tirera rien si elle est utilisée ailleurs.
