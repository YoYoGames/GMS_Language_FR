Définir l'état de la ligne de temps Cette action peut être utilisée pour démarrer, arrêter, mettre en pause ou reprendre une ligne de temps. La ligne de temps doit d'abord avoir été définie pour l'instance en utilisant l'action Définir la ligne de temps de l'instance, et ensuite vous pouvez utiliser cette action à tout moment pour changer son état de fonctionnement à partir des options suivantes Option Description Début Démarrer la timeline assignée à l'instance (si la timeline a été mise en pause, la position de départ sera de nouveau fixée au moment 0). Curriculum Vitae Reprendre une timeline en pause (si la timeline a été arrêtée et non mise en pause, cela ne fera rien). En pause Met en pause la timeline assignée à l'instance (lorsqu'elle est en pause, les moments de la timeline n'avancent pas avec les images du jeu). Stop Arrête la timeline attribuée à l'instance. Syntaxe : Arguments : Arguments Description État L'état dans lequel il faut placer la chronologie (voir le tableau ci-dessus) Le code du bloc d'action ci-dessus vérifiera la valeur contenue dans une variable globale et si elle est vraie, la timeline est mise en pause, et si elle est fausse, la timeline sera reprise.
