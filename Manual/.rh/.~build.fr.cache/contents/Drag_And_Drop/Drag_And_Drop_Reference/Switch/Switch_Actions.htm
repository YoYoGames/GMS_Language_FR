Switch bibliothèque d'actions La Switch es contient les actions requises pour créer des instructions switch dans le code d'action de votre jeu. Dans un certain nombre de situations, vous souhaitez laisser vos instances effectuer une action en fonction d'une valeur particulière, et bien que vous puissiez le faire en utilisant un certain nombre d'instructions "if" consécutives, cela peut devenir compliqué lorsque les choix possibles dépassent deux ou trois. Dans ces cas, il est préférable d'utiliser l'action "switch". Une action de commutation fonctionne comme suit: Vous fournissez la valeur a (généralement à partir d'un type d'expression ou de variable) La valeur est comparée à la valeur attribuée aux différentes instructions "case" Si la valeur est la même que la valeur du cas, le contenu du cas est exécuté sinon le cas suivant est évalué, etc... Une fois qu'un cas a été exécuté ou que tous les cas ont été évalués et ont échoué, le code continuera à partir de la fin du commutateur Si tous les cas ont échoué et que vous avez un cas par défaut, cela sera effectué avant que le code continue à partir de la fin du commutateur.
