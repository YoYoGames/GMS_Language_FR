Bibliothèque d'actionSwitch La bibliothèque d'actions Switches contient les actions nécessaires pour créer des instructions de commutation dans le code d'action de votre jeu. Dans un certain nombre de situations, vous souhaitez que vos instances effectuent une action en fonction d'une valeur particulière, et bien que vous puissiez le faire en utilisant un certain nombre d'instructions "if" consécutives, cela peut devenir compliqué lorsque le nombre de choix possibles dépasse deux ou trois. mais cela peut devenir compliqué lorsque le nombre de choix possibles dépasse deux ou trois. Dans ce cas, il est préférable d'utiliser l'action "switch". Une action "switch" fonctionne comme suit : Vous fournissez la valeur a (généralement à partir d'une expression ou d'une variable). La valeur est comparée à la valeur attribuée aux différentes déclarations "cas". Si la valeur est la même que celle du cas, le contenu du cas est exécuté, sinon le cas suivant est évalué, etc... Une fois qu'un cas a été effectué ou que tous les cas ont été évalués et ont échoué, le code continuera à partir de la fin du commutateur Si tous les cas ont échoué et que vous avez un cas par défaut, alors celui-ci sera exécuté avant que le code ne continue à partir de la fin du switch.
