Set View Variable Cette action vous permet de définir certaines variables intégrées liées à la façon dont le port de vue affichera les vues de la pièce par la caméra. Vous sélectionnez la variable du port de vue à modifier, puis vous donnez au port de vue une cible (de 0 à 7) et enfin la nouvelle valeur de la variable. pour la variable. Visibility C'est le bouton de visibilité pour le port de vue et attend une valeur de true (vrai) ou false (faux) où la valeur true rendra le port de vue visible (c'est-à-dire qu'il sera dessiné à l'écran), et une valeur false ne dessinera rien à l'écran. rien à l'écran. View Port X Coordinate Il s'agit de la position X du port de vue dans la fenêtre du jeu. View Port Width C'est la largeur (en pixels) du port de vue. La largeur du port d'affichage (ou des ports d'affichage combinés si plusieurs sont actifs) définit la largeur de la fenêtre de jeu ou du canevas d'arrière-plan au début du jeu. jeu n'aura pas d'effet visible sur la taille de la fenêtre de jeu, à moins d'être appelé en même temps que la fonction GML window_set_size(). Notez également que si vous avez une taille de port plus grande ou plus petite que celle assignée à la caméra, la vue de la caméra sera réduite - ou augmentée - pour s'adapter.
