Bibliothèque d'actions du Gamepad GameMaker Studio 2 possède un certain nombre d'actions dédiées qui peuvent être utilisées pour détecter les commandes analogiques et numériques de plusieurs manettes de jeu connectées. à partir de plusieurs manettes de jeu connectées. Ces actions vous demandent de spécifier une valeur d'index de manette de jeu, qui est un nombre (à partir de 0) représentant la manette de jeu sélectionnée. Notez que lorsqu'une manette de jeu est connectée à votre appareil (ou elle est retirée), un événement système asynchrone est déclenché (toutefois, pour gérer cela, vous devrez peut-être utiliser du code). Les emplacements de manettes de jeu sont indexés à partir de 0, les emplacements 0 à 3 inclus étant réservés aux manettes de jeu Xinput , c'est-à-dire les manettes Xbox360 et compatibles. Cependant, vous pouvez également vérifier les emplacements 4 à 11 inclus pour les manettes à entrée directe, ce qui signifie que vous pouvez détecter de nombreux autres modèles de manettes lorsqu'elles sont connectées par ces emplacements.
