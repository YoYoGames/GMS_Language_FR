Set Gamepad Axis Deadzone Cette action peut être utilisée pour définir la "zone morte" de l'axe du stick du gamepad. Vous spécifiez l'index du gamepad à définir, puis vous définissez une valeur entre 0 et 1. Si la quantité d'entrée est inférieure à la valeur donnée, l'axe du stick du gamepad est considéré comme étant à 0. Notez qu'il s'agit d'un paramètre global qui affectera tous les axes du stick connecté. Notez que cela ne modifie pas la sortie de l'action Get Gamepad Axis, qui renverra toujours une valeur comprise entre -1 et 0, et 0 et 1. Par exemple, si vous définissez la zone morte à 0,2, le fait de pousser le stick vers la droite signifiera que ne commencera à renvoyer une valeur de 0 à 1 que lorsque la valeur brute de l'axe sera supérieure à 0,2. Ainsi, lorsque la valeur brute sera de 0,2, la valeur renvoyée sera 0, lorsque la valeur brute sera de 0,5, la valeur renvoyée sera 0,375, ou si la valeur brute est de 0,9, la valeur renvoyée sera 0,875. la valeur de retour sera de 0,875. Syntaxe de l'action : Arguments : Arguments Description Manette de jeu L'index du gamepad. Deadzone La valeur de la zone morte (0 - 1) Exemple : Le code du bloc d'action ci-dessus définit la zone morte pour tous les indices du gamepad à 0,2.
