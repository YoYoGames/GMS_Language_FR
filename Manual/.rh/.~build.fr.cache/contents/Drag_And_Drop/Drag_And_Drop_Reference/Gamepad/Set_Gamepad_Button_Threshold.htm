Seuil des boutons du Gamepad Cette action peut être utilisée pour définir le "seuil" des boutons analogiques du gamepad. Vous spécifiez l'index du gamepad à définir, puis vous définissez une valeur comprise entre 0 et 1. Si le montant de l'entrée du bouton analogique est inférieur à la valeur donnée, le bouton du gamepad est considéré comme étant à 0. Notez qu'il s'agit d'un paramètre global qui affectera tous les boutons analogiques connectés au gamepad. Cette valeur sera utilisée dans toutes les vérifications d'appui, de pression et de relâchement pour le gamepad donné, mais sera ignorée par l'action Get Gamepad Trigger. Syntaxe de l'action : Arguments : Arguments Description Manette de jeu L'index du gamepad. Deadzone La valeur du seuil (0 - 1) Exemple : Le code du bloc d'action ci-dessus définit le seuil du bouton pour tous les indices gamepad à 0,2.
