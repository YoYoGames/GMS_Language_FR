Obtenez Gamepad Axis Vous pouvez utiliser cette action pour obtenir la valeur des différents axes d'une manette de jeu donnée. Vous fournissez l'index de la manette de jeu (c'est le nombre de "slots" qu'une manette de jeu occupe) et sélectionnez l'axe à vérifier à partir du joystick gauche ou droit, horizontal ou vertical. La valeur renvoyée sera comprise entre -1 et 1 pour chacun des axes horizontaux et verticaux disponibles: -1 est à gauche et 1 à droite pour l'axe horizontal, et -1 est vers le haut et 1 est vers le bas pour l'axe vertical. Cette valeur retournée sera stockée dans la variable cible que vous fournissez pour une utilisation future, que vous pouvez marquer comme une variable "Temp", ce qui signifie que l'action créera une nouvelle variable temporaire (locale) pour contenir la valeur de retour pour vous, comme illustré dans l'exemple ci-dessous. Il est à noter que la définition de la zone morte pour l'axe du manche n'affectera pas la valeur renvoyée, qui sera toujours comprise entre -1 et 1. Par exemple, définir la zone morte sur 0,2 signifiera que pousser le manche vers la droite ne commencera qu'à renvoyer une valeur de 0 à 1 lorsque la valeur brute de l' axe est supérieure à 0,2 - donc lorsque la valeur brute est de 0,2, la valeur de retour sera 0, lorsque la valeur brute est à 0,5, la valeur de retour sera de 0,375, ou si la valeur brute est de 0,9, alors la valeur de retour serait 0,875. Syntaxe de l'action: Arguments: Argument Description Gamepad L'index de la manette de jeu. Axis L'axe à vérifier (joystick gauche / joystick droit, horizontal / vertical) Target La variable cible dans laquelle stocker la valeur renvoyée. Exemple: Le code de bloc d'action ci-dessus crée deux variables locales (temporaires), puis stocke les valeurs actuelles pour les axes horizontal et vertical du joystick gauche de la manette de jeu.
