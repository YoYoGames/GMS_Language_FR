Connecter la manette de jeu Vous pouvez utiliser cette action pour vérifier et voir si une "fente" de manette de jeu donnée a une manette de jeu connectée. Vous fournissez l'index de la manette de jeu (c'est le nombre de "slots" qu'une manette de jeu occupe) et la valeur renvoyée sera stockée dans la variable cible que vous fournissez pour une utilisation future (cela peut être marqué comme une variable locale temporaire que vous créez pour cet événement ou script). Syntaxe de l'action: Arguments: Argument Description Gamepad L'index de la manette de jeu à vérifier. Target La variable cible dans laquelle stocker la valeur renvoyée. Exemple: Le code de bloc d'action ci-dessus exécute un for boucle pour vérifier et voir si une manette de jeu est connectée à l'un des 12 "slots" disponibles. Si une manette de jeu est découverte, une variable globale est définie sur le numéro de l'emplacement actuel et la boucle est interrompue.
