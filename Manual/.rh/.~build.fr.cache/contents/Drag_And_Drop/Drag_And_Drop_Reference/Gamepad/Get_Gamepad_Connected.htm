Connectez votre Gamepad Vous pouvez utiliser cette action pour vérifier et voir si un gamepad "slot" donné a un gamepad connecté. Vous fournissez l'index du gamepad (c'est le numéro du "slot" qu'un gamepad occupe) et la valeur renvoyée sera stockée dans la variable cible que vous fournissez pour une utilisation future (elle peut être marquée comme une variable locale temporaire que vous créez pour cet événement ou ce script). Syntaxe de l'action : Arguments : Arguments Description Manette de jeu L'index du gamepad à vérifier. Cible La variable cible dans laquelle stocker la valeur retournée. Exemple : Le code du bloc d'action ci-dessus exécute une boucle for pour vérifier si un gamepad est connectée à l'un des 12 "slots" disponibles. Si un gamepad est découvert, une variable globale est définie sur le numéro de l'emplacement actuel et la boucle est interrompue.
