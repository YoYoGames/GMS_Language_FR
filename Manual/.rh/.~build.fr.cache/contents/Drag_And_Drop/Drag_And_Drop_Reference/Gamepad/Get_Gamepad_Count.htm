Obtenez le nombre de manettes de jeu Vous pouvez utiliser cette action pour obtenir le nombre de manettes de jeu connectées à l'appareil exécutant votre jeu. Le nombre retourné sera compris entre 0 et 12. Syntaxe de l'action: Arguments: Argument Description Target La variable cible dans laquelle stocker la valeur renvoyée. Exemple: Le code de bloc d'action ci-dessus obtient le nombre de manettes de jeu connectées et attribue la valeur à une variable locale temporaire. Ceci est ensuite vérifié pour voir s'il est supérieur à 0 et une variable globale définie en conséquence.
