Si le bouton de la manette de jeu est relâché Cette action vérifie quand un bouton de manette de jeu a été relâché pour l'index de manette de jeu donné. Il reviendra true sur le jeu unique, cochez sur lequel le bouton est relâché et false à tous les autres moments. Si vous activez le not modificateur, cette action sera inversée, et vérifier si le bouton n'a pas été relâché, en retournant false sur le jeu, cochez la date de sa sortie et true à tous les autres moments. Si le bouton à cocher est un bouton analogique, le relâchement ne sera pas détecté tant que la valeur du bouton n'aura pas été inférieure au seuil défini à l'aide de l'action Définir le seuil du bouton de la manette de jeu. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme indiqué dans l'image ci-dessous: Ces actions seront désormais exécutées si le "si" est évalué à true, tandis que toutes les actions déposées ailleurs seront exécutées après le bloc "if". Syntaxe de l'action: Arguments: Argument Description Gamepad L'index de la manette de jeu. Button Le nom du bouton à vérifier. Not Annuler le chèque (vrai devient faux et vice versa) Exemple: Le code de bloc d'action ci-dessus interroge l'état du bouton de la manette de jeu 0 à chaque étape et si un bouton est maintenu enfoncé, il vérifie le bouton initial enfoncé. Si le chèque est true lors de la pression initiale vers le bas, un son est joué, puis, pendant que le bouton est maintenu enfoncé, la vitesse d'instance est réglée sur 10. Si le bouton n'est pas maintenu enfoncé, une vérification est effectuée sur le relâchement du bouton pour réinitialiser la vitesse 0 à nouveau.
