Si le bouton du Gamepad est relâché Cette action vérifie si un bouton de manette de jeu a été relâché pour l'index de manette de jeu donné. Elle retournera vrai sur le seul tick de jeu où le bouton est relâché et faux à tous les autres moments. Si vous activez le modificateur not, cette action sera inversée et vérifiera si le bouton n'a pas été relâché, renvoyant false au tick de jeu où il a été relâché et true à tous les autres moments. Si le bouton vérifié est un bouton analogique, le relâchement ne sera pas détecté tant que la valeur du bouton ne sera pas passée sous le seuil défini à l'aide de l'action Set Gamepad Button Threshold. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous : Ces actions seront maintenant exécutées si le bloc "if" est évalué à true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if". Syntaxe de l'action : Arguments : Arguments Description Manette de jeu L'index du gamepad. Bouton Le nom du bouton à vérifier. Non Négation de la vérification (vrai devient faux et vice versa) Exemple : Le code du bloc d'action ci-dessus interroge l'état du bouton du gamepad 0 à chaque étape et si un bouton est maintenu enfoncé, il vérifie la pression initiale sur le bouton. Si la vérification est vraie lors de l'appui initial sur le bouton, un son est joué, puis, tant que le bouton est maintenu enfoncé, la vitesse de l'instance est fixée à 10. Si le bouton n'est pas le bouton n'est pas maintenu enfoncé, une vérification est effectuée au moment du relâchement du bouton pour remettre la vitesse à 0.
