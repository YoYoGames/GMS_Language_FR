Redémarrer le jeu Cette action vous permet de redémarrer le jeu. Cela revient à lancer le jeu pour la première fois et l'événement Other - Game Start Event sera donc déclenché, ainsi que l'événement Other - Game End Event. Cependant, il faut noter que les variables globales ne seront pas réinitialisées à moins qu'elles ne soient explicitement codées comme telles - ce qui signifie que si vous ne les définissez pas dans la toute première pièce du jeu, elles conserveront leurs valeurs précédentes - et que Les structures de données et les particules ne seront pas non plus libérées, ce qui doit être fait avant l' appel de l'action "Restart Game" ou dans l'événement "Game End". Notez également que le jeu ne redémarrera pas au moment où cette action est appelée, mais plutôt à la fin du script ou de l'événement où l'action est placée, donc si d'autres actions sont placées après celle-ci, elles seront toujours appelées et le jeu redémarrera à la fin du bloc.
