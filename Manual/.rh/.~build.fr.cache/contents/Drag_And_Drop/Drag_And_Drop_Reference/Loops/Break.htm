Casser Cette action ne peut être utilisée que dans une action de boucle (par exemple, Repeat ou While ), et son appel sortira de la boucle à ce moment-là. Normalement, une boucle s'exécutera un nombre fixe de fois ou jusqu'à ce qu'une condition spécifique soit remplie, mais vous pouvez utiliser l'action Break pour terminer prématurément une boucle à tout moment (les actions dans la boucle après le Break ne seront pas exécutées). Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus effectue une boucle de répétition 1000 fois. Dans la boucle, il vérifie si une structure de données de la liste est vide et si elle ne l' est pas supprime un élément de la liste, et si elle est elle brise la boucle.
