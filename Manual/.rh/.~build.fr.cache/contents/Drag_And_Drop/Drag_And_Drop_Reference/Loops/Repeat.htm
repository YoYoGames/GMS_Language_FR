Répétition Cette action répétera toutes les actions qui lui sont ajoutées le nombre de fois spécifié. Vous indiquez le nombre de fois que les actions doivent être répétées, puis vous ajoutez l'action à la boucle de répétition, de la même manière que vous ajouteriez une action à un "if", c'est-à-dire en la plaçant l'action à côté de l'action plutôt que sous celle-ci : Toutes les actions ajoutées sur le côté seront incluses dans la répétition et seront exécutées avant que l'événement ou le script ne continue avec les actions ajoutées en dessous (voir l'exemple ci-dessous). ne continue avec les actions ajoutées en dessous (comme le montre l'exemple ci-dessous). Notez que vous pouvez mettre fin prématurément à toute boucle si vous appelez l'action Break. Syntaxe de l'action : Arguments : Arguments Description Times Le nombre de fois à répéter (une valeur entière) Exemple : Le code du bloc d'action ci-dessus obtient le nombre d'éléments d'une liste et stocke la valeur dans une variable locale temporaire. Cette variable est ensuite utilisée pour définir le nombre de fois qu'une boucle de répétition est exécutée. Dans la boucle de répétition, les entrées de la liste sont lues dans une autre variable temporaire qui est ensuite utilisée pour détruire les instances qu'elle référence.
