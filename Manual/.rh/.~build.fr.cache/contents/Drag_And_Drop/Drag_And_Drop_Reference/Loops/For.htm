Pour L' action For est utilisée pour effectuer des boucles qui ne se terminent que lorsqu'une valeur atteint la condition donnée. Vous donnez une valeur initiale en tant que variable (qui peut être marquée comme une variable locale temporaire - c'est généralement ce que vous voulez), puis vous donnez la condition pour vérifier la variable (par exemple si elle est inférieure à une certaine valeur), puis enfin la quantité d'incrémentation de la variable à chaque itération de la boucle. Ainsi, le flux général est le suivant: Vous initialisez une variable à une valeur Vous définissez la condition pour vérifier cette variable par rapport Vous définissez combien la variable doit être incrémentée à chaque itération de la boucle Vous ajoutez ensuite des actions dans la boucle, et celles-ci seront exécutées pour chaque itération jusqu'à ce que la condition retourne false ou une action Break est rencontrée. Lors de l'ajout d'actions à effectuer dans la boucle, celles-ci sont ajoutées de la même manière que vous ajouteriez une action à un "si", c'est-à-dire en la déposant sur le côté de l'action plutôt qu'en dessous: Toutes les actions ajoutées sur le côté seront incluses dans le For et seront exécutées avant que l'événement ou le script ne se poursuive avec les actions ajoutées sous. Notez que vous pouvez mettre fin prématurément à n'importe quelle boucle si vous appelez l' action Break, et lorsque vous utilisez des boucles For, vous n'avez pas besoin d'incrémenter et vous pouvez la faire décrémenter aussi, tant que la condition que vous utilisez est appropriée. Par exemple, définissez la variable initiale sur 10, définissez la condition pour vérifier la présence de moins de 0, puis définissez la valeur d'incrément sur -1 et la boucle répétera 10 fois, en comptant à rebours de 10 à 0. Si une manette de jeu est découverte, une variable globale est définie sur le numéro de l'emplacement actuel et la boucle est interrompue.
