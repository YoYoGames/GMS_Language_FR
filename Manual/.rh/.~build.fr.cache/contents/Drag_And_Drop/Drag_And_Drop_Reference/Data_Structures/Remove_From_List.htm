Supprimer de la liste Cette action peut être utilisée pour supprimer un élément d'index de la liste donnée. Vous fournissez la variable qui stocke l'index de la liste (tel que retourné par l'action Create List) et la position de l'index dans la liste à supprimer, où la position de l'index est comprise entre 1 et (longueur de la liste -1). position de l'indice est comprise entre 1 et (longueur de la liste -1). Cette action ne renvoie rien, donc si vous avez besoin de la valeur à la position de l'index, vous devez utiliser Get List Item At avant de supprimer l'index. Syntaxe de l'action : Arguments : Arguments Description Liste L'index (stocké dans une variable) de la liste à retirer de la liste Index L'index dans la liste à supprimer Exemple : Le code du bloc d'action ci-dessus vérifie la présence d'une collision à la position de l'instance. Si une collision est trouvée, la valeur d'identification unique de l'instance est stockée dans une variable temporaire, puis vérifiée pour voir si elle existe dans la structure de données de la liste. Si elle existe, l'élément est retiré de la liste et l'instance est détruite.
