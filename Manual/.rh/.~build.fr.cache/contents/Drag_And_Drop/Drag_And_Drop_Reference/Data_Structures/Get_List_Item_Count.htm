Obtenir le nombre d'éléments de liste Vous pouvez utiliser cette action pour savoir combien d'entrées il y a dans une liste. Vous fournissez la variable qui contient l'index de la liste (tel que renvoyé par l'action Créer une liste ) et la variable cible pour stocker le nombre d'éléments retournés dans la liste (qui peut être marquée comme une variable locale temporaire à utiliser jusqu'à la fin de la script ou événement). Syntaxe de l'action: Arguments: Argument Description List L'index (stocké dans une variable) pour obtenir le nombre d'éléments à partir de Target La variable cible pour stocker le nombre d'articles retournés Exemple: Le code de bloc d'action ci-dessus obtient la taille de la liste référencée dans la variable globale, puis utilise une boucle for pour la parcourir en boucle et détruire les instances avec les ID stockés dans la liste.
