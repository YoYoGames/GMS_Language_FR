Définir la valeur de la grille Avec cette action, vous pouvez définir une cellule spécifique dans la structure de données de grille sur une valeur. Vous fournissez la variable qui contient l'index de la grille (tel que renvoyé par l'action Créer une grille ), puis indiquez la position x et y (la colonne et la ligne) de la cellule à définir, puis enfin la valeur de cette cellule. La valeur peut être n'importe quel type de données valide, mais les positions x et y doivent être des entiers entiers dans les limites de la grille. Syntaxe de l'action: Arguments: Argument Description Grid L'index (stocké dans une variable) de la grille à définir x La position de la cellule le long de l'axe x à définir (la colonne) y La position de la cellule le long de l'axe y à définir (la ligne) Value La valeur sur laquelle définir la cellule de la grille Exemple: Le code de bloc d'action ci-dessus crée une nouvelle structure de données de grille et conserve sa valeur d'index dans une variable globale. Il parcourt ensuite toutes les instances de la salle et ajoute leur valeur d'identifiant unique dans la cellule correspondant à leur position.
