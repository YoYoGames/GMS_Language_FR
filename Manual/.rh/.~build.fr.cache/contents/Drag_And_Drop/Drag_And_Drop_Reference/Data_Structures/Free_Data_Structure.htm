Structure de données gratuite Cette action libère la structure de données de la mémoire et doit être appelée chaque fois qu'une structure de données n'est plus nécessaire, par exemple lors de l'événement Destroy even of an instance ou Room End, etc... Libérer une structure de données signifie qu'aucune autre action n'est nécessaire. peut être utilisé pour le référencer car il n'existera plus, bien que vous puissiez également le vérifier à l'aide de l'action Si la structure de données existe. Notez qu'à moins que la structure de données n'ait un index affecté à une variable de portée globale, elle ne sera pas transmise d'une pièce à une autre. Par exemple, si vous créez une instance de lecteur et qu'une structure de données de liste est affectée à une variable dans cette instance, la liste devra être libérée si le lecteur est détruit ou si la salle change avec l'instance de lecteur (sauf si le joueur l'instance est marquée comme persistante), de sorte que l'instance de lecteur aurait deux actions de structure de données gratuite pour ces deux événements. Il est important que toutes les structures de données créées soient détruites de cette manière lorsqu'elles ne sont plus nécessaires pour éviter les fuites de mémoire qui peuvent entraîner des problèmes de performances ou même planter votre jeu. Syntaxe de l'action: Arguments: Argument Description Type Le type de structure de données que vous libérez (Liste, Carte, Grille ou Pile) Variable La variable contenant l'index de la structure de données à libérer Exemple: Le code de bloc d'action ci-dessus obtient la taille de la liste référencée dans la variable globale, puis utilise une boucle for pour la parcourir en boucle et détruire les instances avec les ID stockés dans la liste.
