Obtenir la valeur de la grille Avec cette action, vous pouvez définir une cellule spécifique dans la structure de données de grille sur une valeur. Vous fournissez la variable qui contient l'index de la grille (tel que renvoyé par l'action Créer une grille ), puis indiquez la position x et y (la colonne et la ligne) de la cellule à définir, puis enfin la valeur de cette cellule. Vous devez également définir une variable cible (qui peut être marquée comme variable locale temporaire) pour stocker la valeur de cellule renvoyée, et la valeur peut être n'importe quel type de données valide. Notez que les positions x et y de la cellule doivent être des entiers entiers dans les limites de la grille. Syntaxe de l'action: Arguments: Argument Description Grid L'index (stocké dans une variable) de la grille pour obtenir x La position de la cellule le long de l'axe x à obtenir (la colonne) y La position de la cellule le long de l'axe y pour obtenir (la ligne) Target La variable cible à utiliser pour stocker la valeur renvoyée Exemple: Le code de bloc d'action ci-dessus obtient la valeur de la grille stockée dans une variable globale qui correspond à la position des instances dans la salle. Cette valeur est ensuite vérifiée et s'il s'agit d'une valeur d'ID d'instance, cette instance est détruite et la cellule de la grille définie sur le mot-clé noone.
