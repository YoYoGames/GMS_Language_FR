Insérer dans la liste Avec cette action, vous pouvez récupérer la position d'index d'un élément dans la liste donnée. Vous fournissez la variable qui stocke l'index de la liste (tel que renvoyé par l'action Créer une liste ) et la valeur dont vous souhaitez trouver la position d'index dans la liste, ainsi qu'une variable cible pour stocker la position d'index renvoyée pour l'élément (qui peut être marqué comme une variable locale temporaire à utiliser jusqu'à la fin du script ou de l'événement). Lorsque vous insérez une valeur dans une liste, tous les index au-dessus de l'index donné pour l'insertion seront déplacés vers le haut, donc si vous avez une liste avec 4 éléments (position d'index 0 à 3) et placez un nouvel élément à la position d'index 2, l'élément précédent 2 deviendra l'élément 3 et l'élément précédent 3 deviendra l'élément 4. Notez que si la valeur que vous recherchez n'existe pas dans la liste, il n'y aura pas de position d'index de liste à renvoyer, et donc le retour la valeur sera -1. Syntaxe de l'action: Arguments: Argument Description List L'index (stocké dans une variable) de la liste à insérer dans Index La position d'index dans la liste pour insérer une valeur Value La valeur à stocker à la position Exemple: Le code de bloc d'action ci-dessus utilise une boucle de répétition pour obtenir la taille de la liste, puis générer une position aléatoire dans la liste. Il génère ensuite une valeur aléatoire et insère cette valeur dans la liste à la position aléatoire définie précédemment.
