Effacer la grille Avec cette action, vous pouvez effacer une structure de données de grille à une valeur unique (chaque cellule de la grille sera définie sur la valeur donnée). Vous fournissez la variable qui contient l'index de la grille (tel que renvoyé par l'action Créer une grille ), puis donnez la valeur pour effacer la grille. Syntaxe de l'action: Arguments: Argument Description Grid L'index (stocké dans une variable) de la grille à effacer Value La valeur pour effacer la grille Exemple: Le code de bloc d'action ci-dessus efface la structure de données de grille donnée pour contenir le mot-clé noone dans chaque cellule si une variable globale est inférieure ou égale à 0.
