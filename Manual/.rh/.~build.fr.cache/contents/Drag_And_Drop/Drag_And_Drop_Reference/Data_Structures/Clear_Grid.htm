Grille claire Avec cette action, vous pouvez effacer une structure de données de grille à une valeur unique (chaque cellule de la grille sera définie à la valeur donnée). Vous fournissez la variable qui contient l'index de la grille (tel que retourné par l'action Create Grid) et ensuite vous donnez la valeur à effacer de la grille. Syntaxe de l'action : Arguments : Arguments Description Grille L'index (stocké dans une variable) de la grille à effacer. Valeur La valeur à laquelle la grille doit être effacée Exemple : Le code du bloc d'action ci-dessus efface la structure de données de la grille donnée pour contenir le mot clé noone dans chaque cellule si une variable globale est inférieure ou égale à 0.
