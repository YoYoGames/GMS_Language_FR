Si la structure de données est vide Avec cette action, vous pouvez vérifier si une structure de données donnée est vide ou non. Vous devez d'abord choisir le type de structure de données à vérifier (soit une liste, une carte ou une pile - pas de grilles car elles ne peuvent même pas être vides), puis vous fournissez la variable qui contient l'index de la structure de données (tel que retourné par les actions Create List, Create Map, et Create Stack). L'action retournera vrai si la structure de données est vide, et faux si elle ne l'est pas. Notez que vous pouvez marquer "not" dans l'action pour vérifier si une structure de données contient des données en vérifiant si elle n'est pas vide.
