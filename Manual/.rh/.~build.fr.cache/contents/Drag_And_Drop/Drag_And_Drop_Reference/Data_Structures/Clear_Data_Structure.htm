Effacer la structure des données Cette action effacera toutes les valeurs de la structure de données donnée. Vous devez d'abord sélectionner le type de structure de données à effacer, puis donner l'index de la structure de données (l'index est la valeur qui est retournée et stockée dans une variable lorsque vous avez créé la structure de données à l'origine). Une structure de données effacée est considérée comme "vide" (c'est-à-dire qu'elle n'a pas d'entrées), à l' exception des grilles. Avec les grilles, cette action effacera simplement toutes les cellules de la grille à 0. REMARQUE: cela ne libère pas la structure de données de la mémoire et l'efface uniquement de toute information. Pour libérer la mémoire affectée à la structure de données, vous devez utiliser l'action Libérer la structure de données. Syntaxe de l'action: Arguments: Argument Description Structure Le type de structure de données que vous effacez (Liste, Carte, Grille ou Pile) Variable La variable contenant l'index de la structure de données à effacer Exemple: Le code de bloc d'action ci-dessus vérifie si la structure de données référencée dans la variable globale n'est pas vide, et si tel est le cas, la structure de données est effacée.
