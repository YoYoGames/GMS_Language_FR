Effacer la structure des données Cette action efface toutes les valeurs de la structure de données donnée. Vous devez d'abord sélectionner le type de structure de données à effacer et ensuite donner l'index de la structure de données (l'index est la valeur qui est retournée et stockée dans une variable lorsque vous avez créé la structure de données à l'origine). la structure de données à l'origine). Une structure de données effacée est considérée comme "vide" (c'est-à-dire qu'elle n'a pas d'entrées), à l'exception des grilles. Avec les grilles, cette action effacera simplement toutes les cellules de la grille à 0. REMARQUE: Cette action ne libère pas la structure de données de la mémoire et ne fait que l'effacer de toute information. Pour libérer la mémoire affectée à la structure de données, vous devez utiliser l'action Libérer la structure de données.
