Obtenir l'élément de liste à Avec cette action, vous pouvez récupérer la position d'index d'un élément dans la liste donnée. Vous fournissez la variable qui stocke l'index de la liste (tel que renvoyé par l'action Créer une liste ) et la position d'index dans la liste à partir de laquelle obtenir la valeur de l'élément, ainsi qu'une variable cible pour stocker la valeur de l'élément retourné (qui peut être marquée comme une variable locale temporaire à utiliser jusqu'à la fin du script ou de l'événement). REMARQUE: Si vous donnez une position qui est en dehors de la taille de la liste donnée (c'est-à-dire: la position 11 dans une liste de 10 valeurs), l'action retournera undefined. Vous pouvez vérifier cela à l'aide de l'action Si non défini. Syntaxe de l'action: Arguments: Argument Description List L'index (stocké dans une variable) de la liste à supprimer Index La position d'index dans la liste à partir de laquelle obtenir l'élément Target La variable cible pour stocker la valeur renvoyée Exemple: Le code de bloc d'action ci-dessus obtient la taille de la liste référencée dans la variable globale, puis utilise une boucle for pour la parcourir en boucle et détruire les instances avec les ID stockés dans la liste. Une fois la boucle terminée, la liste est libérée de la mémoire.
