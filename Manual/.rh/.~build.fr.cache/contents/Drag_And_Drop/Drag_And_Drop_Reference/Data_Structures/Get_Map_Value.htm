Obtenir la valeur de la carte Avec cette action, vous pouvez récupérer la valeur associée à une clé donnée dans une structure de données de carte. Vous fournissez la variable qui contient l'index de la carte (tel que renvoyé par l'action Créer une carte ), puis donnez une "clé" (qui est l'identificateur dans la carte pour une valeur), et fournissez également une variable cible pour contenir la valeur retournée ( qui peut être marquée comme une variable locale temporaire). REMARQUE: si vous donnez une clé qui n'existe pas dans la carte, l'action retournera undefined. Vous pouvez vérifier cela à l'aide de l'action Si non défini. Syntaxe de l'action: Arguments: Argument Description Map L'index (stocké dans une variable) de la carte pour obtenir la valeur de Key La clé à obtenir (réelle ou chaîne) Target La variable cible contenant la valeur de la carte renvoyée Exemple: Le code du bloc d'action ci-dessus vérifie la présence d'une collision à la position de l'instance et si une carte est trouvée, une carte est recherchée pour obtenir une valeur de la clé "damage", et cette valeur est ensuite utilisée pour changer la variable d'instance appelante "hp".
