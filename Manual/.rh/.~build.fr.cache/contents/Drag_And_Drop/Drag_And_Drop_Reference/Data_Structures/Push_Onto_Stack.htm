Pousser sur la pile Avec cette action, vous pouvez pousser une valeur sur une structure de données de pile. Vous fournissez la variable qui contient l'index de la pile (tel que renvoyé par l'action Créer une pile ), puis donnez la valeur à pousser vers le haut. Avec les structures de données de pile, les valeurs sont toujours ajoutées en «haut» de la pile, et jamais à la fin ou au milieu (si vous avez besoin de cette fonctionnalité, utilisez une structure de données de liste). Syntaxe de l'action: Arguments: Argument Description Stack L'index (stocké dans une variable) de la pile à utiliser Value La valeur à pousser sur la pile Exemple: Le code de bloc d'action ci-dessus crée une variable d'instance et une nouvelle structure de données de pile. L'index de la pile est stocké dans la nouvelle variable, puis une boucle est effectuée qui crée 10 instances et pousse leurs valeurs d'ID uniques sur la pile.
