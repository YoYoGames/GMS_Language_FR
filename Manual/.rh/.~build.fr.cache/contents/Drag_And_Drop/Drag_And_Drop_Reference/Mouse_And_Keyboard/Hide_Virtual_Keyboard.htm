Cacher le clavier virtuel Cette action peut être utilisée pour masquer le clavier virtuel sur le périphérique exécutant le jeu. L'appel de cette action générera un événement asynchrone du système, dans lequel la carte DS sera remplie des paires clé/valeur suivantes async_load DS map sera remplie avec les paires clé/valeur suivantes : "event_type" - il s'agira de "l'état du clavier virtuel" lorsqu'il sera déclenché par des actions du clavier virtuel. "screen_height " - la hauteur du clavier virtuel (en pixels). Elle sera de 0 si le clavier est invisible. "keyboard_status " - l'état actuel du clavier, renvoyé comme l'une des chaînes suivantes : "hiding" (caché) "caché" "visible" "visible" Syntaxe de l'action : Exemple : Le code du bloc d'action ci-dessus vérifie qu'il n'y a pas d'action de la souris vers le bas. détectée, une vérification est effectuée pour voir si le clavier virtuel du système d'exploitation est affiché.
