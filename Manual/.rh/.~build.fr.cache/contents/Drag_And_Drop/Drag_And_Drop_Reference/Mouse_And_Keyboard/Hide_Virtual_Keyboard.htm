Masquer le clavier virtuel Cette action peut être utilisée pour masquer le clavier virtuel sur l'appareil exécutant le jeu. L'appel de cette action générera un événement asynchrone du système, dans lequel le async_load  La carte DS sera remplie avec les paires clé / valeur suivantes: " event_type " - ce sera "l' état du clavier virtuel " lorsqu'il est déclenché par des actions du clavier virtuel. " screen_height " - la hauteur du clavier virtuel (en pixels). Ce sera 0 si le clavier est invisible. " keyboard_status " - l'état actuel du clavier, renvoyé par l'une des chaînes suivantes: "cache" "caché" "montrant" "visible" Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus vérifie une action de souris enfoncée, et si une action est détectée, une vérification est effectuée pour voir si le clavier virtuel du système d'exploitation est affiché. Si ce n'est pas le cas, il est configuré pour afficher - à l'aide d'un type de pavé numérique - et s'il est déjà affiché, il est masqué.
