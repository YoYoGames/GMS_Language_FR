Bibliothèque d'actions de souris et de clavier La bibliothèque d'actions Souris et clavier est l'endroit où vous pouvez trouver les actions requises pour interroger et traiter les entrées des périphériques clavier ou souris. Les actions disponibles sont toutes répertoriées ci-dessous: Si vous appuyez sur la sourisSi la souris est enfoncéeSi la souris est relâchéeSi la touche est enfoncéeSi touche enfoncéeSi la clé est libérée Les trois autres actions ici se réfèrent au clavier virtuel. Si vous avez besoin d'une saisie de texte ou numérique dans votre jeu sur un appareil à écran tactile, vous pouvez utiliser les actions du clavier virtuel fournies pour utiliser le clavier du système d'exploitation de l'appareil. Ces actions appelleront ou masqueront le clavier du système d'exploitation et accepteront les entrées de celui-ci de la même manière qu'un clavier physique, de sorte que les GameMaker Studio 2 devraient toutes fonctionner avec des claviers virtuels (par exemple, l'action "Si la touche est relâchée"), et toute entrée du clavier virtuel sera stockée dans les différentes variables du clavier, comme keyboard_string ou keyboard_last_char, etc...). L'ouverture ou la fermeture du clavier virtuel générera des événements asynchrones du système, dans lesquels async_load  La carte DS sera remplie avec les paires clé / valeur suivantes: " event_type " - le type d'événement système reçu, qui sera "l' état du clavier virtuel " pour les claviers virtuels. " screen_height " - la hauteur du clavier virtuel (en pixels). Ce sera 0 si le clavier est invisible. " keyboard_status " - l'état actuel du clavier, renvoyé par l'une des chaînes suivantes: "cache" "caché" "montrant" "visible" Les actions du clavier virtuel disponibles sont: Afficher le clavier virtuelMasquer le clavier virtuelObtenir la hauteur du clavier virtuelSi le clavier virtuel s'affiche Retour: Index Suivant: Actions de la manette de jeu © Copyright YoYo Games Ltd. 2021 Tous droits réservés
