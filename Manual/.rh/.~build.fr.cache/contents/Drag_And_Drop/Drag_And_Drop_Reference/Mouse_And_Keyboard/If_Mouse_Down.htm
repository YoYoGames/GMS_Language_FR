Si la souris descend Avec cette action, vous pouvez interroger la souris pour savoir si le bouton sélectionné est actuellement maintenu enfoncé. L'action retournera vrai à chaque étape où le bouton est maintenu enfoncé, ou faux sinon, bien que si vous cliquez sur le modificateur not, vous pouvez alors vérifier si le bouton n'est pas maintenu enfoncé, c'est-à-dire que l'action retournera vrai si aucun bouton n'est enfoncé et faux s'il l'est. Si vous n'avez besoin de vérifier qu'un seul bouton est enfoncé, utilisez l'action Si la souris est pressée. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous : Ces actions seront maintenant exécutées si le "if" vaut true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if".
