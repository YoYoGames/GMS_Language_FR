Si la souris est libérée Cette action vérifie si le bouton de la souris a été relâché. Elle retournera vrai sur le seul pas de jeu (frame) où le bouton de la souris est relâché et faux à tous les autres moments. Si vous activez le modificateur not, cette action sera inversée, et vérifiera si le bouton de la souris n'a pas été relâché, en retournant false sur l'étape de jeu (frame) où il a été relâché et true à tous les autres moments. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous : Ces actions seront maintenant exécutées si le "if" est évalué à true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if". Syntaxe de l'action : Arguments : Arguments Description Bouton de la souris Le nom du bouton de la souris à vérifier (gauche, milieu, droite). Non Négation de la vérification (vrai devient faux et vice versa) Exemple : Le code du bloc d'action ci-dessus interroge l'état du bouton de la souris à chaque étape et, s'il est maintenu enfoncé, il vérifie la pression initiale de la souris. est maintenu enfoncé, il vérifie l'appui initial sur le bouton de la souris. Si la vérification est vraie lors de l'appui initial sur le bouton, la vitesse de l'instance est définie, puis, pendant que le bouton est enfoncé, le mélange du sprite passe au rouge. Si la souris n'est pas maintenue Si la souris n'est pas maintenue enfoncée, une vérification est effectuée lors du relâchement de la souris pour réinitialiser la couleur de mélange en blanc et définir la vitesse à 0.
