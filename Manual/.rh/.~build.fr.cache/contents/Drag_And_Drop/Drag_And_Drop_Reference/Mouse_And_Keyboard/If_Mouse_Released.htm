Si la souris est relâchée Cette action vérifie quand un bouton de la souris a été relâché. Il reviendra true sur la seule étape de jeu (cadre) sur laquelle le bouton de la souris est relâché et false à tous les autres moments. Si vous activez le not modificateur, cette action sera inversée, et vérifier si le bouton de la souris n'a pas été relâché, en retournant false sur l'étape de jeu (cadre) sur laquelle il a été libéré et true à tous les autres moments. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme indiqué dans l'image ci-dessous: Ces actions seront désormais exécutées si le "si" est évalué à true, tandis que toutes les actions déposées ailleurs seront exécutées après le bloc "if". Syntaxe de l'action: Arguments: Argument Description Mouse Button Le nom du bouton de la souris à vérifier (gauche, milieu, droite). Not Annuler le chèque (vrai devient faux et vice versa) Exemple: Le code de bloc d'action ci-dessus interroge l'état du bouton de la souris à chaque étape et s'il est maintenu enfoncé, il vérifie la pression initiale de la souris. Si le chèque est true lors de la pression initiale vers le bas, la vitesse de l'instance est définie, puis, pendant que le bouton est enfoncé, le mélange de sprite passe au rouge. Si la souris n'est pas maintenue enfoncée, une vérification est effectuée au relâchement de la souris pour réinitialiser la couleur de fusion au blanc et régler la vitesse sur 0.
