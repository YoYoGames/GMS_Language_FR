Si la touche est enfoncée Avec cette action, vous pouvez interroger le clavier pour voir si une touche est actuellement maintenue enfoncée. L'action retournera vrai à chaque étape où la touche du clavier est maintenue enfoncée, ou faux sinon, bien que si vous cliquez sur le modificateur not, vous pouvez alors vérifier si la touche n'est pas maintenue enfoncée, c'est-à-dire que l'action retournera vrai si aucune touche n'est enfoncée et faux si elle l'est. Si vous n'avez besoin de vérifier qu'une seule pression sur une touche, utilisez l'action If Key Pressed.
