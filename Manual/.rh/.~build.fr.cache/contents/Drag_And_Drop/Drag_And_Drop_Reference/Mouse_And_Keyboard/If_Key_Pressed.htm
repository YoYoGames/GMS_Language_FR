Si la touche est pressée Cette action vérifie l'appui initial sur une touche du clavier. Elle retournera vrai à l'unique étape du jeu (image) où la touche est initialement enfoncée et faux à tous les autres moments. Si vous activez le modificateur not, cette action sera inversée et vérifiera si la touche n'a pas été enfoncée, et renverra un résultat faux à l'étape du jeu (image) où la touche a été enfoncée et vrai à tous les autres moments. Si vous avez besoin de vérifier si une touche est maintenue enfoncée et non seulement l'événement d'appui initial, utilisez l'action If Key Down. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous : Toute action déposée ailleurs sera exécutée après le bloc "if". Si la vérification est vraie lors de l'appui initial sur la touche, un son est joué, puis, tant que la touche est maintenue enfoncée, le mélange du sprite passe au rouge. Si la touche n'est pas maintenue enfoncée, une vérification est effectuée au relâchement de la touche pour réinitialiser la couleur de mélange en blanc.
