Si vous appuyez sur la souris Cette action vérifie la pression initiale sur un bouton de la souris. Il reviendra true à l'étape de jeu unique (cadre) où le bouton est initialement enfoncé et false à tous les autres moments. Si vous activez le not modificateur, cette action sera inversée, et vérifier si le bouton n'a pas été enfoncé, en retournant false sur le pas de jeu (cadre) sur lequel il a été enfoncé et true à tous les autres moments. Si vous avez besoin de vérifier si un bouton est maintenu enfoncé et pas seulement l'événement de pression initial, utilisez l'action Si la souris est enfoncée. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme indiqué dans l'image ci-dessous: Ces actions seront désormais exécutées si le "si" est évalué à true, tandis que toutes les actions déposées ailleurs seront exécutées après le bloc "if". Syntaxe de l'action: Arguments: Argument Description Mouse Button Le nom du bouton de la souris à vérifier (gauche, milieu, droite). Not Annuler le chèque (vrai devient faux et vice versa) Exemple: Le code de bloc d'action ci-dessus interroge l'état du bouton de la souris à chaque étape et s'il est maintenu enfoncé, il vérifie la pression initiale de la souris. Si le chèque est true lors de la pression initiale vers le bas, la vitesse de l'instance est définie, puis, pendant que le bouton est enfoncé, le mélange de sprite passe au rouge. Si la souris n'est pas maintenue enfoncée, une vérification est effectuée au relâchement de la souris pour réinitialiser la couleur de fusion au blanc et régler la vitesse sur 0.
