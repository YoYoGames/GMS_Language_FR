Si la souris est pressée Cette action vérifie l'appui initial sur un bouton de la souris. Elle renvoie un résultat vrai lors de l'unique étape du jeu (image) où le bouton est initialement enfoncé et un résultat faux à tous les autres moments. Si vous activez le modificateur not, cette action sera inversée et vérifiera si le bouton n'a pas été enfoncé, en renvoyant false à l'étape du jeu (frame) où il a été enfoncé et true à toutes les autres étapes. Si vous avez besoin de vérifier si un bouton a été maintenu enfoncé et et pas seulement l'événement d'appui initial, utilisez l'action If Mouse Down. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous : Ces actions seront maintenant exécutées si le "if" vaut true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if". Syntaxe de l'action : Arguments : Arguments Description Bouton de la souris Le nom du bouton de la souris à vérifier (gauche, milieu, droite). Non Négation de la vérification (vrai devient faux et vice versa) Exemple : Le code du bloc d'action ci-dessus interroge l'état du bouton de la souris à chaque étape et, s'il est maintenu enfoncé, il vérifie la pression initiale de la souris. est maintenu enfoncé, il vérifie l'appui initial sur le bouton de la souris. Si la vérification est vraie lors de l'appui initial sur le bouton, la vitesse de l'instance est définie, puis, pendant que le bouton est enfoncé, le mélange du sprite passe au rouge. Si la souris n'est pas maintenue enfoncée, une vérification est effectuée au moment du relâchement de la souris pour réinitialiser la couleur de mélange en blanc et définir la vitesse à 0.
