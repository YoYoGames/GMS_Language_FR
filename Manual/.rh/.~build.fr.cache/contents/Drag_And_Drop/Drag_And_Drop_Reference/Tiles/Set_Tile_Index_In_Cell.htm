Set Tile Index In Cell Avec cette action, vous pouvez définir une tuile sur une couche donnée à un nouvel indice. La couche est spécifiée à partir du nom de couche donné (une chaîne de caractères, telle qu'elle est utilisée pour nommer la couche dans l'éditeur de pièce), et ensuite vous donnez la ligne et la colonne de la cellule dans la couche pour obtenir la tuile à définir. Enfin, vous fournissez l'index de la tuile à définir (les tuiles sont indexées de gauche à droite, de haut en bas, avec 0 en haut à gauche). Notez que vous pouvez toujours définir l'indice de tuile à 0 pour effacer la tuile.
