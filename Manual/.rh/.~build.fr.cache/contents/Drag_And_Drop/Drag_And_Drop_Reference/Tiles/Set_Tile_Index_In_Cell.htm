Définir l'index des mosaïques dans la cellule Avec cette action, vous pouvez définir une tuile sur une couche donnée sur un nouvel index. Le calque est spécifié à partir du nom du calque donné (une chaîne, comme utilisé pour nommer le calque dans l'éditeur de pièce), puis vous donnez la ligne et la colonne de la cellule dans le calque pour obtenir la vignette à définir. Enfin, vous fournissez l'index de tuile pour définir la tuile (les tuiles sont indexées de gauche à droite de haut en bas, avec 0 en haut à gauche). Notez que vous pouvez toujours définir l'index de la vignette sur 0 pour effacer la vignette. Syntaxe de l'action: Arguments: Argument Description Layer Le calque sur lequel définir la vignette Column La colonne (de gauche à droite) de la cellule Row La ligne (de haut en bas) de la cellule Tile L'index de vignette sur lequel définir la cellule Exemple: Le code de bloc d'action ci-dessus crée une boucle pour parcourir la ligne supérieure de la couche de pièce donnée, et comme il le fait, il vérifie si la tuile indexée dans la cellule de la colonne est définie sur 0. Si ce n'est pas l'index de tuile pour cette cellule est mis à 0 et la variable temporaire (locale) est incrémentée avant que la boucle ne fasse un autre ensemble de vérifications.
