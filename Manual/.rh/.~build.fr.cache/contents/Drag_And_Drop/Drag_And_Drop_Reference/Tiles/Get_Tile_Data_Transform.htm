Obtenir la transformation des données de tuile Avec cette action, vous pouvez savoir si les données de mosaïque d'une mosaïque ont été transformées de l'une des trois manières suivantes: retournées, mises en miroir ou pivotées. Les données de tuile sont simplement une valeur qui reflète l'index de la tuile avec les transformations appliquées, et vous voudrez normalement obtenir les données de tuile d'abord en utilisant soit Obtenir les données de tuile dans la cellule ou Obtenir les données de tuile au pixel avant d'utiliser cette action. L'action reviendra true si la transformation choisie a été appliquée aux données de tuile, ou false sinon, et la valeur retournée sera alors stockée dans la variable cible qui peut avoir été créée précédemment ou peut être une nouvelle variable temporaire (si vous cochez la case "Temp"). Pour plus d'informations sur les données de mosaïque, veuillez consulter la GML sur les fonctions de mappage de mosaïques. Syntaxe de l'action: Arguments: Argument Description Tile Les données de tuile pour la tuile à partir de laquelle obtenir la transformation Transform La transformation à vérifier Exemple: Le code de bloc d'action ci-dessus récupère les données de la vignette à la position de la souris, puis vérifie si la vignette a été soumise à une transformation de rotation. Si tel est le cas, la transformation est réinitialisée et les données de mosaïque réinitialisent la mosaïque à la position.
