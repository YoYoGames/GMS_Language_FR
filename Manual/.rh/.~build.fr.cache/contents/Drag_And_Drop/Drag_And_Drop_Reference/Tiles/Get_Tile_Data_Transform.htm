Transformer les données des tuiles Avec cette action, vous pouvez savoir si les données d'une tuile ont été transformées de l'une des trois façons suivantes : retournement, miroir ou rotation. Les données de tuile sont simplement une valeur qui reflète l'index de la tuile avec les transformations appliquées, et vous voudrez normalement obtenir les données de tuile d'abord en utilisant l'une ou l'autre des méthodes suivantes normalement obtenir les données de la tuile d'abord en utilisant soit Get Tile Data In Cell ou Get Tile Data At Pixel avant d'utiliser cette action. L'action retournera vrai si la transformation choisie a été appliquée aux données de la tuile, ou faux sinon, et la valeur retournée sera alors stockée dans la variable cible qui peut avoir été créée précédemment ou peut être une nouvelle variable temporaire (si vous cochez la case "Temp"). ). Pour plus d'informations sur les données de tuiles, veuillez consulter la section GML sur les fonctions de carte de tuiles.
