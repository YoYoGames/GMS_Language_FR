Définir l'indice de la tuile au pixel Avec cette action, vous pouvez définir une tuile sur une couche donnée à un nouvel indice. La couche est spécifiée à partir du nom de couche donné (une chaîne, telle qu'elle est utilisée pour nommer la couche dans l'éditeur de pièce), puis vous donnez une position x et y pour obtenir la tuile à définir. Les positions x et y peuvent être marquées comme étant relatives à la position de l'instance appelante, et la tuile à définir sera celle dans laquelle se trouve la position donnée dans la room et le jeu de tuiles utilisé sera celui assigné à la couche dans l'éditeur de room éditeur. Enfin, vous fournirez l'index de la tuile à laquelle vous souhaitez affecter la tuile (les tuiles sont indexées de gauche à droite, de haut en bas, avec 0 en haut à gauche). Notez que vous pouvez toujours mettre l'index de la tuile à 0 pour effacer la tuile. Syntaxe de l'action : Arguments : Arguments Description Couche La couche sur laquelle la tuile doit être placée x La position x à vérifier y La position y à vérifier Carreaux Le nouvel index de la tuile à utiliser Exemple : Le code du bloc d'action ci-dessus vérifie si le bouton gauche de la souris a été enfoncé. s'il en détecte un, il obtient l'index de la cellule de la tuile dans laquelle se trouve la position de la souris. Cet indice de tuile est ensuite vérifié et s'il est différent de 0, il est mis à 0.
