Définir l'index de la mosaïque sur le pixel Avec cette action, vous pouvez définir une tuile sur une couche donnée sur un nouvel index. Le calque est spécifié à partir du nom du calque donné (une chaîne, comme utilisé pour nommer le calque dans l'éditeur de pièce), puis vous donnez une position x et y pour obtenir la tuile à définir. Les positions x et y peuvent être marquées comme étant relatives à la position de l'instance appelante, et la tuile en cours de définition sera celle dans laquelle la position donnée se situe dans la pièce et le jeu de tuiles utilisé sera celui assigné au calque dans l'éditeur de salle. Enfin, vous fournissez l'index de tuile pour définir la tuile (les tuiles sont indexées de gauche à droite de haut en bas, avec 0 en haut à gauche). Notez que vous pouvez toujours définir l'index de la vignette sur 0 pour effacer la vignette. Syntaxe de l'action: Arguments: Argument Description Layer Le calque sur lequel définir la vignette x La position x à vérifier y La position y à vérifier Tile Le nouvel index de la vignette à utiliser Exemple: Le code de bloc d'action ci-dessus vérifie si le bouton gauche de la souris est enfoncé et, s'il en est détecté, il obtient l'index de la cellule de tuile dans laquelle se situe la position de la souris.
