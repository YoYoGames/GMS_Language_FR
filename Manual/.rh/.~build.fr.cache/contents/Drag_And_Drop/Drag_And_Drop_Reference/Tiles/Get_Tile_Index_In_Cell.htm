Obtenir l'index des tuiles dans la cellule Avec cette action, vous pouvez obtenir un index de tuiles sur une couche donnée. Le calque est spécifié à partir du nom du calque donné (une chaîne, comme utilisé pour nommer le calque dans l'éditeur de pièce), puis vous donnez la ligne et la colonne de la cellule dans le calque à partir de laquelle obtenir l'index de la tuile. Enfin, vous fournissez la variable cible dans laquelle stocker l'index de tuile retourné, où 0 signifie aucune tuile et un entier positif est l'index de la tuile. Notez que la variable cible peut être marquée comme une variable temporaire (locale) qui peut être utilisée jusqu'à la fin de l'événement. Syntaxe de l'action: Arguments: Argument Description Layer La couche à partir de laquelle obtenir la tuile Column La colonne (de gauche à droite) de la cellule Row La ligne (de haut en bas) de la cellule Target La variable cible pour la valeur de retour Exemple: Le code de bloc d'action ci-dessus crée une boucle pour parcourir la ligne supérieure de la couche de pièce donnée, et comme il le fait, il vérifie si la tuile indexée dans la cellule de la colonne est définie sur 0. Si ce n'est pas l'index de tuile pour cette cellule est mis à 0 et la variable temporaire (locale) est incrémentée avant que la boucle ne fasse un autre ensemble de vérifications.
