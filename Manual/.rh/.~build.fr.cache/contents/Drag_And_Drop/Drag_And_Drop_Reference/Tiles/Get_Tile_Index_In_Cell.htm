Obtenir l'indice de la tuile dans la cellule Avec cette action, vous pouvez obtenir l'index d'une tuile sur une couche donnée. La couche est spécifiée à partir du nom de couche donné (une chaîne de caractères, telle qu'elle est utilisée pour nommer la couche dans l'éditeur de pièce), puis vous donnez la ligne et la colonne de la cellule dans la couche pour obtenir l'index de la tuile de la tuile. Enfin, vous fournissez la variable cible pour stocker l'indice de tuile retourné, où 0 est l'absence de tuile, et un nombre entier positif est l'indice de la tuile. Notez que la variable cible peut être marquée comme une variable temporaire (locale) qui peut être utilisée jusqu'à la fin de l'événement. Si ce n'est pas le cas, l'index de la tuile pour cette cellule est défini sur 0 et la variable temporaire (locale) est incrémentée avant que la boucle n'effectue une autre série de vérifications. effectue une autre série de vérifications.
