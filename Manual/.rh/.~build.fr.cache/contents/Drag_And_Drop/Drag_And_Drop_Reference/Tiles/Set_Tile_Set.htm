Jeu de carreaux Cette action vous permet de configurer un calque pour utiliser des jeux de tuiles différents de ceux créés dans le navigateur d'actifs. Vous donnez le nom de la couche (une chaîne de caractères, telle que définie dans l'éditeur de pièce), et ensuite la ressource de jeu de tuiles à utiliser, et toutes les tuiles de la pièce sur cette couche seront dessinées avec le nouveau jeu de tuiles. Syntaxe de l'action : Arguments : Arguments Description Couche Le calque sur lequel doit être placé le nouveau jeu de tuiles. Jeu de carreaux Le jeu de tuiles à utiliser pour la couche. Exemple : Le code du bloc d'action ci-dessus vérifie si les tuiles du calque "Floor_Tiles" utilisent le jeu de tuiles "tl_PalaceRuins". l'ensemble de tuiles "tl_PalaceRuins", et si ce n'est pas le cas, elles sont configurées pour l'utiliser.
