Ensemble de tuiles Avec cette action, vous pouvez définir un calque pour utiliser des ensembles de tuiles différents de ceux créés dans le navigateur de ressources. Vous donnez le nom du calque (une chaîne, telle que définie dans l'éditeur de pièce), puis la ressource de jeu de tuiles à utiliser, et toutes les tuiles de la pièce sur ce calque seront dessinées avec le nouveau jeu de tuiles. Syntaxe de l'action: Arguments: Argument Description Layer Calque sur lequel définir le nouvel ensemble de tuiles. Tileset Ensemble de tuiles à utiliser pour le calque. Exemple: Le code de bloc d'action ci-dessus vérifie si les tuiles du calque "Floor_Tiles" utilisent le jeu de tuiles "tl_PalaceRuins", et si ce n'est pas le cas, elles sont configurées pour l'utiliser.
