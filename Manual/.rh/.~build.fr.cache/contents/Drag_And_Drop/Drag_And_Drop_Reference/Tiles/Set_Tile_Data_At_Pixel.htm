Définir les données de la tuile au pixel Avec cette action, vous pouvez définir les données de tuile pour une cellule de tuile trouvée à une position spécifique sur une couche de carte de tuile. Les données de la tuile sont simplement une valeur qui reflète l'index de la tuile avec les valeurs d'inversion, de miroir et de rotation ainsi que les masques de tuiles personnalisés que vous avez utilisés. ainsi que les valeurs d'inversion, de miroir et de rotation, ainsi que les masques de tuiles personnalisés que vous avez utilisés. Avec cette action, vous fournissez d'abord le calque à cibler en utilisant le nom du calque (une chaîne) tel que défini dans l'éditeur de pièce, puis les axes x et y dans la pièce pour lesquels vous souhaitez définir les données des tuiles. Notez que les données des tuiles seront définies pour la "cellule" de la carte des tuiles dans laquelle se trouve la position donnée. Par exemple, si les tuiles de votre jeu de tuiles sont de 16x16, alors la couche de la carte des tuiles contiendra des tuiles dans des cellules de 16x16 pixels, donc si vous utilisez cette action pour cibler la position de la pièce (8, 8), vous modifierez la tuile dans la cellule (0, 0) de la carte des tuiles. REMARQUE: La couche sélectionnée doit avoir été définie comme une couche de carte de tuile dans l'éditeur de pièce, sinon vous pouvez obtenir des erreurs. Le dernier argument de l'action est la donnée de la tuile elle-même. Normalement, vous récupérez les données de tuiles pour la cellule de carte de tuiles à l'aide de l'action Obtenir les données de tuiles au pixel, puis vous les manipulez à l'aide de l'action Définir la transformation des données de tuiles avant de les définir à nouveau avec cette action. Pour plus d'informations sur les données de tuiles, veuillez consulter la section GML sur les fonctions de carte de tuiles. Syntaxe de l'action : Arguments : Arguments Description Couche La couche avec la carte des tuiles à cibler X La position le long de l'axe horizontal (X) pour définir les données de la tuile pour Y La position le long de l'axe vertical (Y) pour définir les données de la tuile. Données Les données de la tuile à utiliser pour définir la cellule à la position. Exemple : Le code du bloc d'action ci-dessus teste un clic de souris et, lorsqu'il en reçoit un il récupère les données de la tuile pour la cellule qui coïncide avec la position de la souris. Ces données sont ensuite modifiées et renvoyées à la couche de carte des tuiles, faisant pivoter la tuile dans cette cellule.
