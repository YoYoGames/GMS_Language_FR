Définir la transformation des données de mosaïque Avec cette action, vous pouvez modifier l'une des trois transformations de tuile dans les données de tuile. Vous fournissez la variable qui contient les données de mosaïque pour la mosaïque, puis sélectionnez la transformation à modifier. Cliquer sur la case à cocher "Appliquer" appliquera cette transformation et la décocher la supprimera. Les données de mosaïque sont simplement une valeur qui reflète l'index de la mosaïque ainsi que les transformations appliquées ainsi que tous les masques de mosaïque personnalisés que vous avez utilisés, et cette action vous permet de modifier trois de ces valeurs: Retourner, inverser et faire pivoter. Notez que vous souhaiterez normalement obtenir les données de mosaïque d'abord en utilisant Obtenir les données de mosaïque dans la cellule ou Obtenir les données de mosaïque au pixel, puis appliquez la ou les transformations requises, puis définissez à nouveau la mosaïque dans la carte de mosaïques à l'aide de Définir les données de mosaïque dans la cellule ou Définir les données de mosaïque au pixel, puisque cette action ne définit qu'une valeur mais ne l'applique pas à la mosaïque dont elle a été extraite. Pour plus d'informations sur les données de mosaïque, veuillez consulter la GML sur les fonctions de mappage de mosaïques. Syntaxe de l'action: Arguments: Argument Description Tile Les données de la vignette à modifier Transform La transformation à appliquer / supprimer Exemple: Le code de bloc d'action ci-dessus récupère les données de la vignette à la position de la souris, puis vérifie si la vignette a été soumise à une transformation de rotation. Si tel est le cas, la transformation est réinitialisée et les données de mosaïque réinitialisent la mosaïque à la position.
