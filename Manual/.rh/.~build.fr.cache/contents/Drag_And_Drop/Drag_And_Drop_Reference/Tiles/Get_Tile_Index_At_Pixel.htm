Obtenir l'indice de la tuile au pixel Cette action est utilisée pour obtenir l'index d'une tuile sur une couche donnée. La couche est spécifiée à partir du nom de couche donné (une chaîne de caractères, telle qu'utilisée pour nommer la couche dans l'éditeur de pièce), et ensuite vous donnez une position x et y pour trouver la tuile dont vous voulez obtenir l'index. La tuile sera celle dans laquelle se trouve la position donnée dans la pièce, et l'index sera renvoyé à la variable cible, où 0 est l'absence de tuile, et un entier positif est l'index de la tuile. Les positions x et y peuvent être marquées comme étant relatives à la la position de l'instance appelante, et la variable cible peut être marquée comme étant une variable temporaire (locale) qui peut être utilisée jusqu'à la fin de l'événement. Syntaxe de l'action : Arguments : Arguments Description Couche La couche sur laquelle la tuile doit être placée x La position x à vérifier y La position y à vérifier Cible La variable cible pour stocker l'indice de la tuile Exemple : Le code du bloc d'action ci-dessus vérifie si le bouton gauche de la souris a été enfoncé. et s'il est détecté, il obtient l'index de la cellule de la tuile dans laquelle se trouve la position de la souris. Cet indice de tuile est ensuite vérifié et s'il est différent de 0, il est mis à 0.
