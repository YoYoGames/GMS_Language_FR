Définir les données de la tuile dans la cellule Cette action vous permet de définir les données de tuile pour une cellule de tuile sur une couche de carte de tuile. Les données de la tuile sont simplement une valeur qui reflète l'index de la tuile ainsi que les valeurs d'inversion, de miroir et de rotation, ainsi que tout masque de tuile personnalisé que vous avez utilisé. Avec cette action, vous fournissez d'abord le calque à cibler en utilisant le nom du calque (une chaîne) tel que défini dans l'éditeur de pièce, puis la colonne et la ligne sur le calque de carte de tuiles pour définir les données de tuile pour. La "cellule" est la zone de la grille de la carte des tuiles qui contient la tuile pour laquelle vous voulez définir les données, donc si vos tuiles sont 16x16, par exemple, et que la pièce est 1024x768, la carte des tuiles aura 64 colonnes et 48 lignes. REMARQUE: La couche sélectionnée doit avoir été définie comme une couche de carte de tuile dans l'éditeur de pièce, sinon vous pouvez obtenir des erreurs. Le dernier argument de l'action est la donnée de tuile elle-même. Normalement, vous devriez récupérer les données de tuiles pour la cellule de carte de tuiles en utilisant l'action Obtenir les données de tuiles dans la cellule, puis les manipuler en utilisant l'action Transformer les données de tuiles avant de les définir à nouveau avec cette action. Pour plus d'informations sur les données de tuiles, veuillez consulter la section GML sur les fonctions de carte de tuiles.
