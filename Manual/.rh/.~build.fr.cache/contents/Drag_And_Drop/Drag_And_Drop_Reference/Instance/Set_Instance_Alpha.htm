Set Instance Alpha Ce bloc d'action définit la valeur image_alpha de l'instance. L'alpha de l'image est la valeur de "transparence" pour dessiner tout sprite assigné à l'instance et a une valeur entre 0 et 1, où 0 serait complètement transparent et 1 serait complètement opaque. complètement opaque. Notez que si le sprite possède des pixels transparents ou semi-transparents, leur transparence sera mise à l'échelle par la quantité, donc si vous avez des pixels transparents à 50% dans le sprite et que vous définissez l'alpha de l'instance à 0.5, l'alpha final final des pixels transparents sera de 25 %. Vous pouvez définir cette valeur comme une quantité relative. Veuillez noter que pour que les modifications de cette variable soient visibles, l'instance ne doit pas avoir d'événement de dessin ( GameMaker Studio 2 dessinera donc le sprite par défaut) ou être dessinée à l'aide de l'action Draw Self . Syntaxe de l'action : Arguments : Arguments Description alpha La valeur alpha à définir (de 0 à 1, la valeur par défaut est 1). Exemple : Le code du bloc d'action ci-dessus définit un nouveau sprite ainsi qu'un certain nombre d'autres propriétés pour l'affichage de ce sprite. pour l'affichage de ce sprite, y compris le réglage de l'alpha de l'image à 0,5.
