Définir l'instance alpha Ce bloc d'action définit le image_alpha valeur de l'instance. L'image alpha est la valeur de «transparence» pour dessiner tout sprite affecté à l'instance et a une valeur comprise entre 0 et 1, où 0 serait totalement transparent et 1 serait totalement opaque. Notez que si le sprite a des pixels transparents ou semi-transparents, leur transparence sera mise à l'échelle de la quantité, donc si vous avez environ 50% de pixels transparents dans le sprite et définissez l'alpha de l'instance sur 0,5, l'alpha final pour les pixels transparents serait de 25%. Vous pouvez définir cette valeur comme un montant relatif Veuillez noter que pour que les modifications apportées à cette variable soient visibles, l'instance doit avoir soit aucun événement de dessin (et donc GameMaker Studio 2 dessine par défaut le sprite), soit être dessinée à l'aide de Draw Self. action. Syntaxe de l'action: Arguments: Argument Description alpha La valeur alpha à définir (de 0 à 1, la valeur par défaut est 1). Exemple: Le code de bloc d'action ci-dessus définit un nouveau sprite ainsi qu'un certain nombre d'autres propriétés pour la façon dont ce sprite doit être affiché, y compris la définition de l'alpha de l'image sur 0,5.
