Détruire en position Ce bloc d'action détruira toutes les instances trouvées au point donné, appelant l' événement Destroy, puis l' événement Clean Up pour toutes les instances trouvées. Notez que cette action est basée sur le masque de collision des instances, et si une partie du masque chevauche le point cible, alors la fonction détruira cette instance (si les instances n'ont pas de sprite ou de masque appliqué, elles le seront ne pas être détruit). Syntaxe de l'action: Arguments: Argument Description x La position x dans la pièce à utiliser. y La position y dans la pièce à utiliser. Exemple: Le code de bloc d'action ci-dessus vérifie une pression sur une touche (la touche espace) et si une est détectée, elle détruira toutes les instances trouvées à la position relative des instances.
