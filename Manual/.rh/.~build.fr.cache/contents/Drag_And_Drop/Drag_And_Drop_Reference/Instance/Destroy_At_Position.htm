Destroy At Position Ce bloc d'action détruira toutes les instances trouvées au point donné, en appelant l'événement Destroy et ensuite l'événement Clean Up pour toutes les instances trouvées. Notez que cette action est basée sur le masque de collision des instances, et si une partie du masque chevauche le point cible, la fonction détruira cette instance (si les instances n'ont pas de sprite ou de masque appliqué, alors elles ne seront pas détruites). Syntaxe de l'action : Arguments : Arguments Description x La position x dans la pièce à utiliser. y La position y dans la pièce à utiliser. Exemple : Le code du bloc d'action ci-dessus vérifie la présence d'une pression sur une touche (la touche espace) et, si elle est détectée, il détruit toutes les instances trouvées à la position relative de l'instance. détectée, il détruira toutes les instances trouvées à la position relative de l'instance.
