Définir la vitesse d'animation Ce bloc d'action définit l'image_speed (vitesse d'animation) pour le sprite affecté à l'instance. Vous définissez le nombre d'images par seconde idéal dans l'éditeur de sprites, puis vous pouvez utiliser cette valeur pour le modifier. pour la modifier. La vitesse d'animation par défaut est fixée à 1, ce qui signifie que le jeu essaiera de maintenir le nombre d'images par seconde que vous avez défini pour le sprite, mais vous pouvez utiliser des valeurs supérieures à 1 pour animer plus rapidement (des images peuvent être sautées), ou des valeurs décimales pour une animation plus lente (les images seront affichées en plusieurs étapes) ou même des valeurs négatives pour inverser la vitesse de l'animation.
