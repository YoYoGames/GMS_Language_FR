Définir la vitesse d'animation Ce bloc d'action définit le image_speed (vitesse d'animation) pour le sprite affecté à l'instance. Vous définissez les images idéales par seconde dans l' éditeur de sprites, puis vous pouvez utiliser cette valeur pour la modifier. La vitesse d'animation par défaut est définie sur 1, ce qui signifie que le jeu essaiera de maintenir le nombre d'images par seconde que vous avez défini pour le sprite, mais vous pouvez utiliser des valeurs supérieures à 1 pour animer plus rapidement (les images peuvent être ignorées), ou des valeurs décimales pour animer plus lentement (les images seront affichées sur différentes étapes) ou même des valeurs négatives pour inverser la vitesse de l'animation. Syntaxe de l'action: Arguments: Argument Description Speed La nouvelle valeur de vitesse du modificateur. Exemple: Le code de bloc d'action ci-dessus définit un nouveau sprite ainsi qu'un certain nombre d'autres propriétés pour la façon dont ce sprite doit être affiché, y compris le paramétrage pour qu'il s'anime à la moitié de la vitesse définie.
