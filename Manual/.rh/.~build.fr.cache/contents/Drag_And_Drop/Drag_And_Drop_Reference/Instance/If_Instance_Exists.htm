Si l'instance existe Cette action peut être utilisée pour vérifier si une instance d'un objet spécifique existe dans la salle de jeu, ou si une instance spécifique elle-même existe. Vous donnez l'objet à partir de l'explorateur d'actifs, et si une instance existe dans la salle, l'action retournera true sinon elle retournera false. Notez que vous pouvez utiliser une variable ici avec l'ID unique d'une instance spécifique (par exemple, tel que retourné par l'action Create Object Instance) ou même l'ID d'instance tel que retourné par l'éditeur de la room lorsque vous avez ajouté l'instance. Si vous indiquez l'argument "Not", l'action vérifiera qu'aucune instance n'existe et si aucune n'est trouvée, elle retournera true et si une seule existe, elle retournera false.
