Événement d'appel de l'utilisateur Avec cette action, vous pouvez appeler l'un des 12 autres événements utilisateur disponibles. Un événement utilisateur est un événement auquel vous pouvez ajouter des scripts ou du code d'action, puis appeler à tout moment à l'aide de cette action, ce qui en fait essentiellement des événements d'objet personnalisés. Syntaxe de l'action: Arguments: Argument Description Event Numéro de l'événement utilisateur à appeler (0 - 11). Exemple: Le code de bloc d'action ci-dessus vérifie une variable puis appelle soit l'événement utilisateur [0], soit l'événement utilisateur [1] selon que la vérification est évaluée à true ou false.
