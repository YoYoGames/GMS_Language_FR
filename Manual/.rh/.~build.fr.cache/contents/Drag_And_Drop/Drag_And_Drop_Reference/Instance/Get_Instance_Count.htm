Obtenir le nombre d'instances Cette action vous permet de connaître le nombre d'instances d'un objet donné actuellement actif dans la pièce. Vous pouvez sélectionner n'importe quel objet dans la liste des ressources ou vous pouvez utiliser le mot-clé spécial tous pour obtenir le nombre de toutes les instances actives. Vous devez également fournir une variable cible à laquelle renvoyer la valeur afin de pouvoir l'utiliser dans d'autres actions. Si vous marquez la variable cible comme "Temp", cela créera une nouvelle variable temporaire (locale) pour contenir la valeur retournée. temporaire (locale) qui contiendra la valeur de retour pour vous, comme le montre l'exemple ci-dessous.
