Obtenir le nombre d'instances Avec cette action, vous pouvez trouver le nombre d'instances d'un objet donné actuellement actif dans la salle. Vous pouvez sélectionner n'importe quel objet dans la liste des ressources ou vous pouvez utiliser le mot-clé spécial all pour obtenir le nombre de toutes les instances actives. Vous devez également fournir une variable cible pour renvoyer la valeur afin de pouvoir ensuite l'utiliser dans d'autres actions. Si vous marquez la variable cible comme "Temp", cela créera une nouvelle variable temporaire (locale) pour contenir la valeur de retour pour vous, comme indiqué dans l'exemple ci-dessous. Syntaxe de l'action: Arguments: Argument Description Object Le nom de l'objet à rechercher. Target La variable à cibler pour la valeur de retour. Exemple: Le code de bloc d'action ci-dessus crée une variable locale (temporaire), puis lui attribue le nombre d'instances de l'objet "obj_Squirrel". Cette valeur est ensuite vérifiée et l'instance est détruite si le nombre d'instances est supérieur à 100.
