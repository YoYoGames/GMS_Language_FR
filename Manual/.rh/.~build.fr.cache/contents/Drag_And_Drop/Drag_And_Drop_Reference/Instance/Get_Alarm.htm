Obtenir une alarme Ce bloc d'action récupère la valeur d'un compte à rebours d'alarme et la stocke dans la variable cible donnée. Si vous marquez la variable cible comme "Temp", cela créera une nouvelle variable temporaire (locale) pour contenir la valeur de retour pour vous, comme indiqué dans l'exemple ci-dessous. Syntaxe de l'action: Arguments: Argument Description Alarm L'alarme à obtenir. Target La variable cible à utiliser pour stocker la valeur d'alarme. Exemple: Le code de bloc d'action ci-dessus crée une variable locale (temporaire) et stocke la valeur actuelle de l'alarme [0].
