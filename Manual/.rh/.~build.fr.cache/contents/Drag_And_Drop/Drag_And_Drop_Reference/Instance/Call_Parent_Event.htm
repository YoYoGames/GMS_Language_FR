Événement d'appeler le parent Ce bloc d'action exécutera l'événement de l'objet parent dont l'instance est un enfant avant de continuer à exécuter les actions dans l'instance réelle. Lorsque vous affectez un objet parent à une instance, tous les événements de cet objet sont "hérités" par l'enfant. Cependant, si vous ajoutez ensuite un événement à l'objet enfant, il remplace l'événement parent. À l'aide de ce bloc d'action, vous pouvez forcer un objet enfant à exécuter à la fois l'événement parent et l'événement dans l'instance elle-même. Consultez la section Éditeur d'objets pour plus d'informations sur les parents. Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus appelle l'événement parent de l'objet dont l'instance est un enfant, puis définit une alarme et modifie l'image-objet.
