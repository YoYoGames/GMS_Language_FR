Appel événement pour les parents Ce bloc d'action exécutera l'événement de l'objet parent dont l'instance est un enfant avant de continuer à exécuter les actions dans l'instance actuelle. Lorsque vous assignez un objet parent à une instance, tous les événements de cet objet sont "hérités" par l'enfant. par l'enfant. Cependant, si vous ajoutez ensuite un événement à l'objet enfant, il remplace l'événement parent. En utilisant ce bloc d'action, vous pouvez forcer un objet enfant à exécuter à la fois l'événement parent et l'événement de l'instance elle-même. Pour plus d'informations sur les parents, consultez la section " Object Editor". Syntaxe de l'action : Exemple : Le code du bloc d'action ci-dessus appelle l'événement parent de l'objet dont l'instance est l'enfant. puis déclenche une alarme et modifie le sprite.
