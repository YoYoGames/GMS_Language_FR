Set Sprite Ce bloc d'action définit le sprite_index de l'instance à un autre sprite du navigateur d'actifs. Vous sélectionnez le sprite à modifier, puis vous indiquez l'image (cadre d'animation) à afficher. Notez que si vous ne souhaitez pas que le sprite s'anime après avoir après avoir défini le cadre, vous devrez utiliser l'action Set Animation Speed et la définir à 0, ou si vous souhaitez que le sprite continue à s'animer, définissez la valeur pour utiliser la variable intégrée image_index à la place. La modification du sprite changera également le masque de collision de l'instance, à moins que vous n'ayez également spécifié un mask_index distinct dans l'éditeur d'objets. Veuillez noter que pour que les modifications de cette action soient visibles, l'instance ne doit pas avoir d'événement de dessin (et GameMaker Studio 2 dessinera donc le sprite par défaut) ou doit être dessinée à l'aide de l'action Draw Self. Syntaxe de l'action : Arguments : Arguments Description Sprite Le nouveau sprite à utiliser (-1 si vous voulez supprimer le sprite). Cadre Le cadre d'animation à afficher initialement (si aucun cadre supplémentaire n'est utilisé, utilisez 0). Exemple : Le code du bloc d'action ci-dessus définit un nouveau sprite ainsi qu'un certain nombre d'autres propriétés pour l'affichage de ce sprite.
