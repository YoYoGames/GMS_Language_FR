Destroy Object Instance Vous pouvez appeler cette action chaque fois que vous souhaitez "détruire" une instance, ce qui la supprimera de la room. Vous devez noter que lorsque vous détruisez une instance, son événement de destruction est appelé immédiatement après l'action qui appelle la destruction. Notez également que bien que l'événement Destroy soit exécuté à l'instant où l'action est appelée, l'instance n' est pas immédiatement retirée du jeu, et elle continuera à exécuter le code contenu dans l'action courante. Elle continuera à exécuter le code contenu dans l'événement en cours jusqu'à la fin, et ce n'est que lorsque l'événement en cours sera terminé que l'instance sera retirée du jeu, auquel moment l'événement de nettoyage sera appelé.
