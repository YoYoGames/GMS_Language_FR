Créer une instance d'objet Avec cette action, vous pouvez créer une instance de n'importe quel objet à partir du navigateur d'actifs. Vous donnez le nom de l'objet et une position x/y dans la pièce où il doit être créé, ainsi que le nom de la couche sur laquelle créer l'instance (il doit s'agir d'une couche d'instance). Instance Layer). Vous pouvez cocher les drapeaux "relatifs" pour positionner l'instance par rapport à l'instance qui exécute les blocs de code d'action. Chaque fois que vous créez une instance dans le jeu, une valeur d'identification unique lui est attribuée. Cette action vous permet donc de définir une variable "cible" pour contenir cette valeur d'identification (comme indiqué dans l'exemple ci-dessous). Notez que la cible peut être marquée comme une variable temporaire (locale), qui sera créée pour l'action et qui pourra être modifiée. sera créée pour l'action et pourra être utilisée dans toutes les actions suivantes. Syntaxe de l'action : Arguments : Arguments Description Objet Le nom de l'objet dont on veut créer une instance. x La position x dans la pièce pour créer l'instance (peut être relative à l'instance appelante). y La position y dans la pièce pour créer l'instance (peut être relative à l'instance appelante). Couche Le nom (une chaîne) de la couche d'instance sur laquelle créer l'instance. Cible La variable à cibler pour la valeur de retour. Exemple : Le code du bloc d'action ci-dessus crée une nouvelle instance et stocke sa valeur d'identification unique dans une variable temporaire. variable temporaire. Cette variable est ensuite utilisée dans l'action Applies To... pour définir certaines propriétés de l'instance créée.
