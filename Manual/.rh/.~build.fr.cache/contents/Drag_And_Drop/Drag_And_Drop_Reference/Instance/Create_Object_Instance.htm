Créer une instance d'objet Avec cette action, vous pouvez créer une instance de n'importe quel objet à partir du navigateur d'actifs. Vous donnez le nom de l'objet et une position x / y dans la pièce où il doit être créé ainsi que le nom du calque sur lequel créer l'instance (il doit s'agir d'un calque d'instance ). Vous pouvez cocher les indicateurs «relatifs» pour positionner l'instance par rapport à l'instance exécutant les blocs de code d'action. Chaque fois que vous créez une instance dans le jeu, une valeur d'ID unique lui est attribuée avec laquelle vous pouvez ensuite identifier l'instance dans d'autres actions, par conséquent, cette action vous permet de définir une variable "cible" pour contenir cette valeur d'ID (comme indiqué dans le exemple ci-dessous). Notez que la cible peut être marquée comme une variable temporaire (locale), qui sera créée pour l'action et peut être utilisée dans toutes les actions suivantes. Syntaxe de l'action: Arguments: Argument Description Object Le nom de l'objet pour lequel créer une instance. x La position x dans la salle pour créer l'instance (peut être relative à l'instance appelante). y La position y dans la salle pour créer l'instance (peut être relative à l'instance appelante). Layer Le nom (une chaîne) du calque d'instance sur lequel créer l'instance. Target La variable à cibler pour la valeur de retour. Exemple: Le code de bloc d'action ci-dessus crée une nouvelle instance et stocke sa valeur d'ID unique dans une variable temporaire. Cette variable est ensuite utilisée dans l' action S'applique à... pour définir certaines propriétés de l'instance créée.
