Set Instance Rotation Ce bloc d'action définit l'angle_image (rotation) de l'instance. L'angle est mesuré en degrés, la droite correspondant à 0°, le haut à 90°, la gauche à 180° et le bas à 270°. Définissez cette variable à 0 pour réinitialiser l'instance, ce qui signifie qu'elle dessinera tous les sprites qui lui sont assignés. l'instance, ce qui signifie qu'elle dessinera tout sprite qui lui est assigné tel qu'il a été défini dans l'éditeur de sprites. Veuillez noter que pour que les modifications apportées à cette action soient visibles, l'instance ne doit pas avoir d'événement de dessin (et GameMaker Studio 2 dessinera donc le sprite par défaut) ou doit être dessinée en utilisant l'action Draw Self. Syntaxe de l'action : Arguments : Arguments Description Angle Le nouvel angle à utiliser (de 0 à 360).
