Définir la rotation de l'instance Ce bloc d'action définit le image_angle (rotation) de l'instance. L'angle est mesuré en degrés, la droite étant de 0 °, le haut étant de 90 °, la gauche étant de 180 ° et le bas étant de 270 °. Définissez cette variable sur 0 pour réinitialiser l'instance, ce qui signifie qu'elle dessinera tout sprite affecté tel qu'il a été défini dans l'éditeur de sprite. Veuillez noter que pour que les modifications apportées à cette action soient visibles, l'instance doit avoir soit aucun événement de dessin (et donc GameMaker Studio 2 dessine par défaut le sprite) soit être dessinée à l'aide de l'action Draw Self. Syntaxe de l'action: Arguments: Argument Description Angle Le nouvel angle à utiliser (de 0 à 360). Exemple: Le code de bloc d'action ci-dessus définit un nouveau sprite ainsi qu'un certain nombre d'autres propriétés pour la façon dont ce sprite doit être affiché, y compris la rotation de 180 °.
