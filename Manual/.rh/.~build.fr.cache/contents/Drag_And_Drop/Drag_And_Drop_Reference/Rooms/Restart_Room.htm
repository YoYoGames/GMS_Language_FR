Salle de redémarrage Cette action va redémarrer la pièce actuelle. Notez que la pièce ne redémarrera pas avant la fin de l'événement ou du script où l'action a été appelée, donc tout code après cette action sera toujours exécuté. Syntaxe de l'action : Exemple : Le code du bloc d'action ci-dessus vérifie la présence d'une pression sur une touche et, si elle est détectée, la pièce est redémarrée.
