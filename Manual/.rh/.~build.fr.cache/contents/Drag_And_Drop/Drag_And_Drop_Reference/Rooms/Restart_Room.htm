Redémarrer la salle Cette action redémarrera la salle actuelle. Notez que la salle ne redémarrera pas avant la fin de l'événement ou du script où l'action a été appelée, donc tout code après cette action sera toujours exécuté. Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus vérifie si une touche est enfoncée et si elle est détectée, la salle est redémarrée.
