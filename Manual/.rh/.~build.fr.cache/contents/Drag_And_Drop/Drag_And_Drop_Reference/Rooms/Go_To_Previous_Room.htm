Aller à la pièce précédente Cette action fera passer votre jeu dans la salle précédente, comme indiqué dans le Gestionnaire de salle au moment de la compilation du jeu. Si cette salle n'existe pas, une erreur sera lancée et le jeu sera forcé de se fermer. Notez que la salle ne changera pas avant la fin de l'événement ou du script où l'action a été appelée, donc tout code après cette action sera toujours exécuté. Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus vérifie une collision avec un autre objet et si un objet est détecté, une variable dans l'instance est vérifiée et le jeu ira soit dans la pièce suivante, soit dans la pièce précédente, comme indiqué dans l'ordre du gestionnaire de salle.
