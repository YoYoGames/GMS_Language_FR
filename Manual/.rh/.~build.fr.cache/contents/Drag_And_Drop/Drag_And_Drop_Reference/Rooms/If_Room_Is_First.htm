Si la chambre est la première Cette action vérifiera si la pièce actuelle est la première pièce listée dans le Room Manager. L'action retournera vrai si la pièce actuelle est la première dans le Room Manager et faux si elle ne l'est pas. Vous pouvez cocher la case not pour transformer cette action en "Si la pièce n'est PAS la première".
