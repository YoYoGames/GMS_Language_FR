Obtenir la chambre actuelle Cette action renverra la valeur d'index de pièce unique pour la pièce actuelle. Vous devez fournir une variable cible pour contenir la valeur renvoyée, et cette variable peut être marquée comme une variable locale temporaire. Syntaxe de l'action: Arguments: Argument Description Target La variable cible pour stocker l'index de pièce retourné Exemple: Le code de bloc d'action ci-dessus obtient la valeur d'index de pièce actuelle, puis utilise une Switch pour vérifier la pièce et définir une variable globale en fonction du résultat.
