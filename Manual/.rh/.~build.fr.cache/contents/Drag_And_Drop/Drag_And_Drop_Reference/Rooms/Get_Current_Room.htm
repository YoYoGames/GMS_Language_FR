Obtenir la chambre actuelle Cette action renvoie la valeur unique de l'index de la pièce pour la pièce actuelle. Vous devez fournir une variable cible pour contenir la valeur retournée, et cette variable peut être marquée comme une variable locale temporaire. Syntaxe de l'action : Arguments : Arguments Description Cible La variable cible pour stocker l'index de la pièce retournée Exemple : Le code du bloc d'action ci-dessus récupère la valeur de l'indice de la pièce actuelle, puis utilise une action de type Switch pour vérifier la pièce et définir une variable globale en fonction du résultat.
