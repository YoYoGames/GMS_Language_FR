Si l'audio est mis en pause Cette action peut être utilisée pour vérifier si une instance d'une ressource sonore est actuellement en pause dans votre jeu. Vous donnez la ressource sonore à vérifier à partir de l'explorateur d'actifs et la fonction reviendra true if it is paused and false si ce n'est pas le cas, bien que si vous cochez le drapeau "non", cela est inversé et la fonction retournera true si le son n'est pas interrompu et false autrement. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme indiqué dans l'image ci-dessous: Ces actions seront désormais exécutées si le "si" est évalué à true, tandis que toutes les actions déposées ailleurs seront exécutées après le bloc "if".
