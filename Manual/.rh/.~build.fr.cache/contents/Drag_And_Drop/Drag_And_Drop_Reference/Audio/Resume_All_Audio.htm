Reprendre tout l'audio Cette action peut être utilisée pour reprendre tous les sons précédemment mis en pause qui ont été mis en pause à l'aide de Pause Audio ou Pause All Audio. Cela n'aura aucun effet sur les sons qui n'ont pas été mis en pause auparavant. Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus vérifiera une variable globale et s'il est évalué à true alors tout le son est mis en pause, sinon tout est repris.
