Arrêter l'audio Cette action peut être utilisée pour arrêter la lecture d'un son. Vous sélectionnez le son dans l'explorateur d'actifs et toutes les instances de ce son cesseront de jouer (donc si vous avez défini le même son pour jouer dans trois événements différents, ils s'arrêteront tous). Syntaxe de l'action: Arguments: Argument Description Sound La ressource sonore pour arrêter Exemple: Le code de bloc d'action ci-dessus vérifie si un bouton de la souris est pressé et si un est détecté, il vérifie alors une variable globale pour voir si ce n'est pas vrai. Si ce n'est pas vrai, alors un son est joué (et mis en boucle) et la variable globale définie sur true, sinon le son est arrêté et la variable globale est définie sur false.
