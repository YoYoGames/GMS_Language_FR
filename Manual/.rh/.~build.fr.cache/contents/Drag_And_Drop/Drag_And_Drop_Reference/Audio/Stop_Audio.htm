Arrêter l'audio Cette action peut être utilisée pour arrêter la lecture d'un son. Vous sélectionnez le son dans l'explorateur de ressources et toutes les instances de ce son s'arrêteront de jouer (ainsi, si vous avez défini le même son pour qu'il soit joué dans trois événements différents, tous s'arrêteront). Syntaxe de l'action : Arguments : Arguments Description Son La ressource sonore s'arrête Exemple : Le code du bloc d'action ci-dessus vérifie si le bouton de la souris a été enfoncé et, si c'est le cas, il vérifie une variable globale pour voir si elle n'est pas vraie. pour voir si elle n'est pas vraie. Si ce n'est pas le cas, un son est joué (et mis en boucle) et la variable globale est définie comme vraie, sinon le son est arrêté et la variable globale est définie comme fausse.
