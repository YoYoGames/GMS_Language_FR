Régler le volume principal Avec cette action, vous pouvez régler le volume de l'ensemble du système audio. La valeur de volume que vous donnez peut être comprise entre 0 (silencieux) et 1 (plein volume) et l'échelle est linéaire, de sorte qu'une valeur de 0,5 serait la moitié du volume. Notez que cette action affectera toutes les instances de tous les sons en cours de lecture dans la pièce et affectera toutes les futures instances de tous les sons joués. Cette action est également indépendante du volume réglé pour tout son individuel, donc si vous avez un son réglé pour jouer à 0,5 volume, puis réglez le volume principal sur 0,5, le son sera lu à 0,25 volume. Syntaxe de l'action: Arguments: Argument Description Volume Le nouveau volume principal pour tous les sons (de 0 à 1, par défaut 1) Exemple: Le code de bloc d'action ci-dessus vérifie une variable globale pour voir si elle est true ou fausse. Si c'est true puis le volume principal du système audio est récupéré et stocké dans une variable locale temporaire. La valeur est ensuite vérifiée pour voir si elle n'est pas égale à 0,5, et si ce n'est pas le cas, le volume principal du système audio est réglé sur 0,5. Si la variable globale est évaluée à false, puis le volume principal est réglé sur 1. Retour: Actions audio Suivant: Obtenir le volume principal © Copyright YoYo Games Ltd. 2021 Tous droits réservés
