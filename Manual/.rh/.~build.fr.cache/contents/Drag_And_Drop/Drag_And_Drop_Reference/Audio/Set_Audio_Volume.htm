Régler le volume audio Cette action vous permet de régler le volume d'un son donné. Vous sélectionnez le son dans l'explorateur d'actifs, puis vous définissez la valeur du volume. Le volume peut être compris entre 0 (silencieux) et 1 (plein volume) et l'échelle est linéaire, de sorte qu'une valeur de 0,5 correspond à la moitié du volume. la moitié du volume. Notez que cette action affectera toutes les instances du son qui sont actuellement en cours de lecture dans la pièce, et affectera toutes les instances futures du son joué. Syntaxe de l'action : Arguments : Arguments Description Son La ressource sonore pour régler le volume de Volume Le nouveau volume pour le son (de 0 à 1, par défaut 1) Exemple : Le code du bloc d'action ci-dessus vérifie une variable globale pour voir si elle est vraie ou fausse. Si elle est vraie, le volume du son donné est récupéré et stocké dans une variable locale temporaire. La valeur est ensuite vérifiée pour voir si elle n'est pas égale à 0,5, et si elle ne l'est pas, alors le volume du son est fixé à 0,5. Si la variable globale vaut false, le volume du son est fixé à 1. Retour : Actions audio Suivant : Obtenir le volume audio Copyright YoYo Games Ltd. 2021 Tous droits réservés
