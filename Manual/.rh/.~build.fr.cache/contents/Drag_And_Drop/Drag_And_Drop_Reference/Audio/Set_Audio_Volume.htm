Régler le volume audio Avec cette action, vous pouvez régler le volume d'un son donné. Vous sélectionnez le son dans l'explorateur d'actifs, puis définissez la valeur du volume. Le volume peut être compris entre 0 (silencieux) et 1 (plein volume) et l'échelle est linéaire, de sorte qu'une valeur de 0,5 serait la moitié du volume. Notez que cette action affectera toutes les instances du son en cours de lecture dans la pièce et affectera toutes les instances futures du son joué. Syntaxe de l'action: Arguments: Argument Description Sound La ressource sonore pour régler le volume de Volume Le nouveau volume du son (de 0 à 1, par défaut 1) Exemple: Le code de bloc d'action ci-dessus vérifie une variable globale pour voir si elle est true ou fausse. Si c'est true puis le volume du son donné est récupéré et stocké dans une variable locale temporaire. La valeur est ensuite vérifiée pour voir si elle n'est pas égale à 0,5, et si ce n'est pas le cas, le volume du son est réglé sur 0,5. Si la variable globale est évaluée à false, le son a alors son volume réglé sur 1. Retour: Actions audio Suivant: Obtenir le volume audio © Copyright YoYo Games Ltd. 2021 Tous droits réservés
