Obtenir le volume principal Cette action renvoie le volume principal du système audio. Vous donnez une variable cible à laquelle renvoyer la valeur du volume (vous pouvez marquer cela comme étant une variable locale temporaire). La valeur renvoyée sera comprise entre 0 et 1, où 0 est silencieux et 1 est plein volume. Syntaxe de l'action: Arguments: Argument Description Target La variable cible pour stocker le volume retourné Exemple: Le code de bloc d'action ci-dessus vérifie une variable globale pour voir si elle est true ou fausse. Si c'est true puis le volume principal du système audio est récupéré et stocké dans une variable locale temporaire. La valeur est ensuite vérifiée pour voir si elle n'est pas égale à 0,5, et si ce n'est pas le cas, le volume principal du système audio est réglé sur 0,5. Si la variable globale est évaluée à false, puis le volume principal est réglé sur 1.
