Pause de tout l'audio Cette action peut être utilisée pour mettre en pause tous les sons en cours de lecture, mais cela ne mettra pas en pause les sons joués après que l'action ait été appelée. Vous pouvez les relancer en utilisant l'action Reprendre tous les sons. Syntaxe de l'action : Exemple : Le code du bloc d'action ci-dessus vérifie une variable globale et si elle vaut true, tout le son est mis en pause, sinon il est repris.
