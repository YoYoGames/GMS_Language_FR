Définir la position audio Avec cette action, vous pouvez définir un son à jouer à un moment précis. Vous donnez le son à jouer et l'heure à partir de laquelle le démarrer, mais cette action ne jouera pas le son à votre place, elle définira simplement la position de lecture du son. Pour utiliser cette action, vous devez d' abord définir la position puis jouer le son à l'aide de l'action Lire l'audio, sinon vous ne changerez pas de position jusqu'à la prochaine fois que vous jouerez la même ressource sonore (qui commencera à la nouvelle position). La position est définie en secondes - vous pouvez utiliser des valeurs décimales - et vous pouvez obtenir la longueur totale du son en utilisant l'action Obtenir la longueur audio, bien que si vous fournissez une valeur supérieure à la longueur du son souhaité, elle sera enveloppée si le son dure 10 secondes et vous donnez 12 secondes comme position de départ, le son commencera à 2 secondes. Syntaxe de l'action: Arguments: Argument Description Sound La ressource sonore pour définir la position de lecture de Time Le temps à régler (en secondes) Exemple: Le code de bloc d'action ci-dessus obtient la durée de lecture de la ressource audio donnée et la stocke dans une variable locale temporaire. Ceci est ensuite utilisé pour générer une valeur aléatoire entre 0 et la longueur du son, qui est ensuite stockée dans une variable locale temporaire différente. Cette nouvelle valeur aléatoire est ensuite utilisée pour définir la position de départ du son et le son est alors joué.
