Définir la position audio Avec cette action, vous pouvez définir un son à jouer à un moment spécifique. Vous donnez le son à jouer, et l'heure à laquelle il doit commencer, mais cette action ne jouera pas le son pour vous, elle définira simplement la position de lecture pour le son. Pour utiliser cette action vous devez d'abord définir la position puis jouer le son en utilisant l'action Jouer Audio, sinon vous n'aurez aucun changement de position jusqu'à la prochaine fois que vous jouerez la même ressource sonore (qui commencera à la nouvelle position). nouvelle position). La position est définie en secondes - vous pouvez utiliser des valeurs décimales - et vous pouvez obtenir la longueur totale du son à l'aide de l'action Get Audio Length, bien que si vous fournissez une valeur supérieure à la longueur du son souhaité, celle-ci sera enveloppée. Ainsi, si le son dure 10 secondes et que vous indiquez 12 secondes comme position de départ, le son commencera à 2 secondes. Syntaxe de l'action : Arguments : Arguments Description Son La ressource sonore pour définir la position de lecture de Temps Le temps à régler (en secondes) Exemple : Le code du bloc d'action ci-dessus récupère la durée de lecture de la ressource sonore donnée et la stocke dans une variable locale temporaire. dans une variable locale temporaire. Cette valeur est ensuite utilisée pour générer une valeur aléatoire comprise entre 0 et la longueur du son, qui est ensuite stockée dans une autre variable locale temporaire. Cette nouvelle valeur aléatoire est ensuite utilisée pour définir la position de départ du son. et le son est alors joué.
