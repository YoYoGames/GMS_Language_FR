Si l'audio est en cours de lecture Cette action peut être utilisée pour vérifier si une instance d'une ressource sonore est en cours de lecture dans votre jeu. Vous donnez la ressource sonore à vérifier à partir de l'explorateur d'actifs et la fonction reviendra true if it is playing and false si ce n'est pas le cas, bien que si vous cochez le drapeau "non", cela est inversé et la fonction retournera true si le son ne joue pas et false autrement. Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme indiqué dans l'image ci-dessous: Ces actions seront désormais exécutées si le "si" est évalué à true, tandis que toutes les actions déposées ailleurs seront exécutées après le bloc "if". Syntaxe de l'action: Arguments: Argument Description Sound La ressource sonore à vérifier Exemple: Le code de bloc d'action ci-dessus vérifie si le son donné est en cours de lecture et s'il l'est, il l'arrête.
