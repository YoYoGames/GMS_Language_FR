Obtenir le volume audio Cette action renvoie le volume du son donné. Vous choisissez le son dans l'explorateur d'actifs et donnez une variable cible à laquelle renvoyer la valeur de volume (vous pouvez marquer cela comme étant une variable locale temporaire). La valeur renvoyée sera comprise entre 0 et 1, où 0 est silencieux et 1 est plein volume. Syntaxe de l'action: Arguments: Argument Description Sound La ressource sonore pour obtenir le volume de Target La variable cible pour stocker le volume retourné Exemple: Le code de bloc d'action ci-dessus vérifie une variable globale pour voir si elle est true ou fausse. Si c'est true puis le volume du son donné est récupéré et stocké dans une variable locale temporaire. La valeur est ensuite vérifiée pour voir si elle n'est pas égale à 0,5, et si ce n'est pas le cas, le volume du son est réglé sur 0,5. Si la variable globale est évaluée à false, le son a alors son volume réglé sur 1. Retour: Actions audio Suivant: Régler le volume principal © Copyright YoYo Games Ltd. 2021 Tous droits réservés
