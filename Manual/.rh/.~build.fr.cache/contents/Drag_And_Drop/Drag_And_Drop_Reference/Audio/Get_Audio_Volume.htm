Obtenir le volume audio Cette action retourne le volume du son donné. Vous choisissez le son dans l'explorateur d'actifs et vous donnez une variable cible à laquelle renvoyer la valeur du volume (vous pouvez la marquer comme étant une variable locale temporaire). La valeur renvoyée sera comprise entre 0 et 1, où 0 est silencieux et 1 est le volume maximum. Syntaxe de l'action : Arguments : Arguments Description Son La ressource sonore pour obtenir le volume de Cible La variable cible pour stocker le volume retourné Exemple : Le code du bloc d'action ci-dessus vérifie une variable globale pour voir si elle est vraie ou fausse. Si elle est vraie, le volume du son donné est récupéré et stocké dans une variable locale temporaire. La valeur est ensuite vérifiée pour voir si elle n'est pas égale à 0,5, et si elle ne l'est pas, alors le volume du son est fixé à 0,5. Si la variable globale vaut false, le volume du son est fixé à 1. Retour : Actions audio Suivant : Régler le volume principal Copyright YoYo Games Ltd. 2021 Tous droits réservés
