Lire l'audio Cette action définira la lecture du son donné. Vous sélectionnez le son dans l'explorateur d'actifs, puis définissez si vous souhaitez que le son soit mis en boucle ou non (un son en boucle sera lu encore et encore jusqu'à l'arrêt) en définissant l'indicateur «Boucle». Syntaxe de l'action: Arguments: Argument Description Sound La ressource sonore pour jouer Loop Que ce soit pour boucler le son ou non (désactivé par défaut) Exemple: Le code de bloc d'action ci-dessus vérifie si un bouton de la souris est pressé et si un est détecté, il vérifie alors une variable globale pour voir si ce n'est pas vrai. Si ce n'est pas vrai, alors un son est joué (et mis en boucle) et la variable globale définie sur true, sinon le son est arrêté et la variable globale est définie sur false.
