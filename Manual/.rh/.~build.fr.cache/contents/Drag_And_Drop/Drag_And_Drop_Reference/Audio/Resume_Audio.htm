Reprendre l'audio Cette action peut être utilisée pour reprendre un son précédemment mis en pause. Cela peut être utilisé sur n'importe quel son qui a été mis en pause à l'aide de Pause Audio ou Pause All Audio, et n'aura aucun effet sur les sons qui n'ont pas été mis en pause auparavant. Syntaxe de l'action: Arguments: Argument Description Sound La ressource sonore pour reprendre la lecture Exemple: Le code de bloc d'action ci-dessus vérifiera une variable globale et s'il est évalué à true puis le son donné est mis en pause, sinon il est repris.
