Obtenez la tonalité audio Cette action récupérera la hauteur de la ressource sonore que vous fournissez. La valeur de retour sera un multiplicateur de la hauteur réelle enregistrée de l'échantillon, où une valeur de 1 ne signifie aucun changement. Pour plus d'informations sur la hauteur, consultez l'action Définir la hauteur audio. Syntaxe de l'action: Arguments: Argument Description Sound La ressource sonore pour obtenir la hauteur de Target La variable cible pour stocker le pas retourné Exemple: Le code de bloc d'action ci-dessus obtient d'abord la hauteur du son donné et le stocke dans une variable locale temporaire. Il vérifie ensuite si une touche est enfoncée, et si une est détectée, la valeur de la hauteur est vérifiée pour voir si elle est inférieure à 2. Si c'est le cas, alors le son est arrêté, la variable de hauteur a 0,1 ajouté, et puis cette variable est utilisée pour régler à nouveau la hauteur du son avant qu'il ne soit joué.
