Obtenir la longueur de l'audio Cette action peut être utilisée pour récupérer la longueur d'un son dans votre jeu. Vous fournissez la ressource sonore pour obtenir la longueur et une variable cible pour retourner la valeur (ou vous pouvez marquer la cible comme une variable temporaire pour que l'action crée la variable qui sera utilisée jusqu'à la fin de l'événement ou du script). La durée sera renvoyée à la variable cible (qui peut être temporaire) en secondes et peut être une valeur décimale. Syntaxe de l'action : Arguments : Arguments Description Son La ressource sonore pour obtenir la longueur de Cible La variable cible pour stocker la valeur de retour Exemple : Le code du bloc d'action ci-dessus récupère la durée de lecture de la ressource sonore donnée et la stocke dans une variable locale temporaire. dans une variable locale temporaire. Cette valeur est ensuite utilisée pour générer une valeur aléatoire comprise entre 0 et la longueur du son, qui est ensuite stockée dans une autre variable locale temporaire. Cette nouvelle valeur aléatoire est ensuite utilisée pour définir la position de départ du son. et le son est alors joué.
