Obtenir la longueur audio Cette action peut être utilisée pour récupérer la durée d'un son dans votre jeu. Vous fournissez la ressource audio pour obtenir la longueur et une variable cible pour renvoyer la valeur également (ou vous pouvez marquer la cible comme une variable temporaire pour que l'action crée la variable à utiliser jusqu'à la fin de l'événement ou du script). La longueur sera renvoyée à la variable cible (qui peut être rendue temporaire) en secondes et peut être une valeur décimale. Syntaxe de l'action: Arguments: Argument Description Sound La ressource sonore pour obtenir la longueur de Target La variable cible pour stocker la valeur de retour Exemple: Le code de bloc d'action ci-dessus obtient la durée de lecture de la ressource audio donnée et la stocke dans une variable locale temporaire. Ceci est ensuite utilisé pour générer une valeur aléatoire entre 0 et la longueur du son, qui est ensuite stockée dans une variable locale temporaire différente. Cette nouvelle valeur aléatoire est ensuite utilisée pour définir la position de départ du son et le son est alors joué.
