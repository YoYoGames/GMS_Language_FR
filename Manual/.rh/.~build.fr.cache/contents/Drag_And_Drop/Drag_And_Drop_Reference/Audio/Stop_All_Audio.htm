Arrêter tout l'audio Cette action peut être utilisée pour arrêter tout l' audio en cours de lecture (notez que même si l'audio est marqué comme mis en pause, il sera arrêté avec cette fonction et vous ne pourrez plus le reprendre plus tard). Syntaxe de l'action: Exemple: Le code de bloc d'action ci-dessus vérifie si un bouton de la souris est pressé et si un est détecté, il vérifie alors une variable globale pour voir si ce n'est pas vrai. Si ce n'est pas vrai, alors un son est joué et la variable globale définie sur true, sinon tout le son est arrêté et la variable globale est définie sur false.
