Arrêter tout le son Cette action peut être utilisée pour arrêter tout le son en cours de lecture (notez que même si le son est marqué comme étant en pause, il sera arrêté avec cette fonction et vous ne pourrez pas le reprendre plus tard). Syntaxe de l'action : Exemple : Le code du bloc d'action ci-dessus vérifie si le bouton de la souris a été enfoncé et, si c'est le cas, il vérifie une variable globale pour voir si elle n'est pas vraie. une variable globale pour voir si elle n'est pas vraie. Si ce n'est pas le cas, un son est joué et la variable globale prend la valeur true, sinon tout son est arrêté et la variable globale prend la valeur false.
