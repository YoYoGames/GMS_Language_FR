Afficher le message de débogage Avec cette action, vous pouvez envoyer un message de débogage à la sortie du compilateur. Ce message peut être une chaîne ou un nombre réel, mais si vous souhaitez utiliser les deux, le nombre doit être converti en chaîne (voir l'action Number To String ). Notez que vous pouvez générer plusieurs messages à l'aide de cette action en cliquant sur le bouton icône pour étendre l'action avec d'autres champs de sortie. Syntaxe de l'action: Arguments: Argument Description Message Le message à afficher dans la fenêtre Sortie Exemple: Le code de bloc d'action ci-dessus vérifiera une variable et si elle retourne true puis quelques messages de débogage seront affichés dans la fenêtre Sortie.
