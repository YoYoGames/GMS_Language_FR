Définir l'état de la fenêtre Cette action vous permet de définir l'état de la fenêtre de jeu, soit plein écran, soit fenêtré. Il y a trois options ici : Toggle - Lorsque cette option est sélectionnée, chaque fois que vous appelez l'action, la fenêtre basculera entre les états plein écran et fenêtré. Plein écran - Ceci fera passer la fenêtre en plein écran à partir d'un état fenêtré. Windowed - Ceci permet à la fenêtre de revenir à un état fenêtré si elle est en plein écran. Notez que l'aspect du jeu en mode plein écran dépendra des options de jeu - paramètres graphiques de vos plates-formes cibles. aux différents rapports d'aspect et résolutions, vous pouvez obtenir un étirement (si l'option du jeu est définie sur Pleine échelle), ou vous pouvez obtenir des barres noires ajoutées (si l'option du jeu est définie sur Conserver le rapport d'aspect). En général, si vous n'avez rien mis en place pour pour gérer les différentes résolutions, vous devriez utiliser l'option Keep Aspect Ratio pour les plateformes cibles vers lesquelles vous voulez compiler.
