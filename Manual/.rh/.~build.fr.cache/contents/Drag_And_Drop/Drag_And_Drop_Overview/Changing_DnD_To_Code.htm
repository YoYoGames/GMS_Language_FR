Modification du DnD™ en code Une autre caractéristique de DnD™ est que vous pouvez le convertir en GML ( GameMaker Language ), et ainsi voir ce qui se passe exactement dans les coulisses lorsque vous utilisez des actions. Cependant, avant de modifier réellement les actions en GML, vous pouvez d'abord les prévisualiser en utilisant le bouton droit de la souris option de menu Aperçu en direct: Ce qui ouvrira une nouvelle fenêtre: Au fur et à mesure que vous ajoutez, modifiez ou supprimez des DnD™, l' GML se met à jour pour vous montrer le code réel qui est créé "en coulisses". Le code de l'aperçu en direct ne peut pas être modifié directement, mais vous pouvez sélectionner des sections et les copier pour les coller dans des GML ou des actions de code (par exemple). Si vous décidez que vous souhaitez convertir le DnD™ en code après avoir vu l'aperçu, vous pouvez le faire à nouveau en cliquant sur le bouton droit de la souris dans n'importe quel espace de travail d'événement avec des actions et en sélectionnant Convertir en GML. La première fois que vous faites cela, vous recevrez un message d'avertissement indiquant qu'il s'agit d'une conversion à sens unique, car vous pouvez convertir des actions en code, mais vous ne pouvez pas les reconvertir en actions individuelles plus tard. Cliquez sur "OK" ici pour effectuer la conversion. Le code résultant utilisera {} pour délimiter des actions individuelles, et vous pouvez voir clairement quelles actions se rapportent à quelles fonctions ou déclarations de variables dans le code. Si le DnD™ est plus complexe alors le code le sera aussi, mais les mêmes règles générales s'appliquent et le code sera des structures séquentiellement exactement les mêmes que vous avez écrit le DnD™. Lorsque vous apprenez à programmer en utilisant le Drag and Drop, cela peut être un outil important pour passer à l'utilisation de GML à un stade ultérieur, mais ce n'est en aucun cas obligatoire et vous pouvez toujours créer d'excellents jeux en utilisant DnD™! Il convient également de noter que même si le processus de conversion est à sens unique, après la conversion des actions en code, vous pouvez à nouveau utiliser DnD™ en utilisant le clic droit dans l'éditeur de code et en sélectionnant " Convertir en DnD™ ".
