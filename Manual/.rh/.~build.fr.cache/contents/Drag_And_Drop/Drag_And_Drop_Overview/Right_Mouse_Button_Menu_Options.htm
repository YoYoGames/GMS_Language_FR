Options du menu du bouton droit de la souris Lorsque vous travaillez sur votre code DnD™, vous pouvez cliquer avec le bouton droit de la souris sur l'action, sur l'espace de travail ou sur le texte dans la liste restreinte à gauche pour ouvrir le menu du bouton droit de la souris : Ce menu comporte les options suivantes : Couper / Copier / Coller: Ceci vous permettra de couper, copier ou coller toute action sélectionnée dans l'espace de travail. Notez que vous pouvez utiliser + en dehors des actions et les faire glisser, ou utiliser / + sur des actions individuelles, pour en sélectionner plusieurs et appliquer la commande à chacune d'entre elles. Commentaire / Supprimer le commentaire: Cette option vous permet d'ajouter et de supprimer des commentaires de vos blocs de code DnD™. Si le commentaire n'est plus nécessaire, vous pouvez soit supprimer tout le texte du commentaire, soit utiliser la fonction d'effacement des commentaires. le texte du commentaire, soit utiliser l'option Supprimer le commentaire du menu.
