Extraits de code Un outil très pratique dont vous disposez lorsque vous modifiez votre code est l'utilisation des extraits de code. Lorsque vous utilisez l'éditeur de script, vous pouvez appuyer sur F4 pour ouvrir la fenêtre contextuelle des extraits de code, qui vous permet de sélectionner l'une des méthodes de code les plus utilisées dans une liste : Dans cette fenêtre, vous pouvez sélectionner avec la souris le fragment à utiliser ou appuyer sur la touche de raccourci associée indiquée à droite. Ceci ajoutera l'extrait à votre code pour le modifier. Vous pouvez également définir vos propres extraits de code si vous le souhaitez. Avant de faire cela, vous devez créer un fichier appelé "snippets.txt" dans l'un des répertoires suivants : Windows: %ProgramData%\GameMakerStudio2\User\\ macOS: /Users/Shared/GameMakerStudio2/User/ Ce dossier n'est pas modifié lors d'une mise à jour de GameMaker Studio 2, de sorte que le fichier que vous avez modifié restera intact (mais toute modification apportée au fichier des extraits de base dans le répertoire d'installation sera annulée). Une fois le fichier créé dans le répertoire de l'utilisateur, vous pouvez le modifier avec n'importe quel éditeur de texte en respectant les règles suivantes : Chaque extrait doit être sur une ligne distincte (il ne doit pas y avoir de lignes vides). Chaque snippet doit commencer par la touche de raccourci à utiliser, suivie de "-", puis du nom du snippet (qui est celui qui est affiché dans le menu), suivi de deux points ":", par exemple : I - Instance Create : Après les deux points, vous ajoutez le bout de code suivant I - Création d'instance:instance_create_layer(x, y, |layer|, object) ; Le code que vous ajoutez doit également respecter un format spécifique où : Tout le code est sur une seule ligne Vous utilisez "$" pour une nouvelle ligne. Vous placez le texte qui doit être sélectionné en premier entre les symboles "|" (le curseur sélectionnera ce texte pour l'éditer lorsque le snippet sera ajouté). Consultez les exemples qui se trouvent déjà dans le fichier snippets de base pour voir comment il est mis en place en suivant les règles ci-dessus.
