Tampons Lorsque vous utilisez les fonctions buffer_save_async() ou buffer_load_async() un événement asynchrone événement asynchrone sera déclenché lorsque le transfert de données sera terminé. Cet événement remplira la carte async_load avec les paires clé/valeur suivantes "id" : l'ID de la fonction asynchrone telle que retournée par la fonction utilisée. "status" : retournera vrai si les données ont été sauvegardées/chargées correctement, et faux sinon. Cela vous permet d'interroger la progression de la sauvegarde/du chargement et d'afficher un message ou de changer de salle, etc... lorsque le processus est terminé.
