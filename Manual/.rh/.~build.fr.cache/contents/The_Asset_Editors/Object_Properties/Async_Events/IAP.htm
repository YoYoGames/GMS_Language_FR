Achat dans l'application Cet événement ne peut être déclenché que par les diverses extensions In App Purchase (IAP) pour les différentes plates-formes cibles disponibles sur le site Web de l'entreprise. Place de marché. Cet événement génère toujours une carte DS qui est exclusive à cet événement et stockée dans la variable spéciale async_load. Cette carte DS aura différentes paires clé/valeur en fonction de la fonction d'extension qui l'a déclenchée, mais il y aura toujours une clé "id" qui pourra être utilisée pour identifier le type d'événement In App Purchase dont il s'agit. Pour connaître les détails exacts des valeurs de retour possibles et des fonctions qui génèrent cet événement, veuillez consulter les manuels PDF individuels inclus avec chacune des extensions.
