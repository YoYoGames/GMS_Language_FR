L'éditeur de polices Lorsque vous souhaitez dessiner du texte dans votre jeu, ce texte est dessiné dans une police Arial standard de 12 points, mais pour créer des textes plus intéressants ou uniques, vous voudrez probablement utiliser des polices différentes. Par conséquent, pour utiliser différentes polices que vous avez sur votre ordinateur, vous devez créer une ressource de police dans GameMaker Studio 2. Cela se fait par un clic droit sur le navigateur de ressources et en sélectionnant " Créer une police ". Pour chaque ressource de police que vous créez, vous spécifiez un type particulier de police de votre ordinateur qui peut ensuite être utilisé dans votre jeu en utilisant l'action ou le code pour définir une police pour le dessin. Lorsque vous créez une ressource de police pour la première fois, la fenêtre de l'éditeur de polices s'ouvre avec les options suivantes: Des détails Ici, vous pouvez donner à votre police un nom unique (caractères alphanumériques et le symbole sous-barre "_" uniquement), ainsi que choisir la police à utiliser. La police doit être installée sur votre ordinateur et en raison de problèmes de licence, GameMaker Studio 2 ne stocke pas les fichiers de police réels avec le fichier de projet par défaut, mais plutôt la police est rendue sur une page de texture sous la forme d'une série de glyphes bitmap. La page de texture sera affectée à un groupe de textures, que vous pouvez également choisir dans l'option Groupe. Si vous souhaitez distribuer le fichier de police réel ( *.ttf uniquement) avec le projet lors de la création d'un Marketplace ou pour toute autre raison, vous pouvez cocher l'option Copier dans le projet. Cela ouvrira l'explorateur de fichiers où vous pourrez sélectionner le fichier de police et il sera ensuite ajouté aux fichiers fournis avec le projet. IMPORTANT! Vous ne devez utiliser que des polices que vous êtes légalement autorisé et autorisé à redistribuer ou à utiliser dans votre projet. Préconfigurations Dans cette section, vous pouvez définir la taille (en points - la valeur par défaut est 12pt) et si la police doit être dessinée en gras ou en italique ou avoir un anti-aliasing appliqué. La fenêtre d'aperçu sur la droite vous montrera le texte à peu près tel qu'il apparaîtra dans votre jeu avec la taille et les transformations que vous avez spécifiées. Commandes de gamme Lors de la création d'une ressource de police, il est rare que vous souhaitiez inclure chaque glyphe pour la police. C'est pourquoi GameMaker Studio 2 utilise par défaut uniquement les caractères de 32 à 127 de la police choisie, car cela nous donne les chiffres et les lettres minuscules et majuscules ainsi que certains signes de ponctuation. Vous pouvez cependant modifier la plage de caractères utilisée pour optimiser vos jeux, c'est-à-dire: si vous n'avez besoin que des chiffres d'une police spécifique, sélectionnez uniquement les chiffres. Pour ce faire, vous devez d'abord cliquer sur le bouton Supprimer pour effacer la plage actuelle (vous devrez peut-être sélectionner la plage pour que cela fonctionne en cliquant sur dessus dans la fenêtre d'aperçu) puis cliquez sur le bouton Ajouter pour ajouter une nouvelle gamme. Cela ouvrira la fenêtre chaînée suivante: Cette fenêtre contient quelques boutons pour vous aider à établir une plage standard pour votre police: - La gamme normale de 32 à 127 - La plage ASCII de 0 à 255 - La plage de chiffres qui ne contient que les 10 chiffres - La plage de lettres qui contient toutes les lettres majuscules et minuscules uniquement D'autres plages peuvent être utilisées en tapant le premier et le dernier index de caractère de la plage que vous souhaitez définir dans les zones de saisie de la plage de caractères (si un caractère ne se trouve pas dans la plage, il est remplacé par un espace). En plus de ces boutons et zones de saisie, vous avez également deux autres options qui sont particulièrement utiles: À partir du code: si vous cliquez sur ce bouton puis sur OK, GameMaker Studio 2 analysera automatiquement votre code de jeu à la recherche de chaînes, puis créera différentes plages de caractères pour couvrir tout le texte de votre jeu. Notez qu'il recherche toutes les chaînes (entre "" ou "") et peut donc également inclure des noms de fichiers dans les plages de caractères. À partir du fichier: cela fonctionne de la même manière que l'option «à partir du code» expliquée ci-dessus. Si vous cliquez dessus, il vous sera demandé de fournir un fichier, puis une fois cela fait, vous devrez cliquer sur OK. GameMaker Studio 2 analysera ensuite le fichier et créera des plages de caractères pour le texte qui s'y trouve. Notez que vous pouvez modifier manuellement la plage de texte en ajoutant ou en supprimant des glyphes de texte directement dans la fenêtre d'aperçu. Contrôles de texture Comme mentionné ci-dessus (dans la section sur les détails de la police ) par défaut, tous les glyphes de police seront rendus sur une page de texture dans le cadre d'un groupe de textures pour les convertir en ressources graphiques, et lorsque le jeu est compilé, la police sera ajoutée à un fichier temporaire. texture pour créer une seule image avec tous les glyphes de police requis, puis cela sera ajouté au groupe de texture principal pour l'exportation. Les boutons ici servent à modifier la façon dont la texture de police est générée ou à régénérer une nouvelle texture de police. Le bouton Options ouvrira les options avancées de texture de police: Ces options peuvent, parfois radicalement, modifier la façon dont une police sera affichée dans le jeu, et toutes les polices ne nécessiteront pas les mêmes paramètres. Les options suivantes sont conçues pour vous offrir la plus grande personnalisation possible pour votre police: Indication: l'indication de police (également appelée instruction) consiste à utiliser des instructions mathématiques pour ajuster l'affichage d'une police de contour afin qu'elle s'aligne avec une grille pixellisée. À de faibles résolutions d'écran, les indices sont essentiels pour produire un texte clair et lisible, et ici vous pouvez choisir parmi différents algorithmes d'indices pour obtenir le meilleur résultat possible. La valeur par défaut est "normal". Options des glyphes: ces options peuvent être utilisées pour modifier la façon dont les glyphes sont chargés en activant / désactivant différentes opérations. Ceux-ci sont: Désactiver l'indication: Cela désactivera l'indication native des polices et l'auto-hinter. Désactiver l'auto-hinter: Cela désactivera l'auto-hinter et utilisera à la place l'hinter natif des polices (s'il en a un). Notez que cela remplace l'option "Préférer l'auto-hinter". Préférer l'auto-hinter: Cela indiquera à GameMaker Studio 2 de préférer l'utilisation de l'auto-hinter à toute indication native que la police peut avoir. Notez que ceci est remplacé par les options «Désactiver l'auto-hinter» et «Désactiver l'indication». Warping: l'activation de cette option déformera légèrement les glyphes pour améliorer l'alignement de la grille. Notez que cela nécessite l'auto-hinter et un mode d'indication "Normal". Pas d'échelle: lorsque cette option est cochée, elle conservera les glyphes dans les unités de police et ignorera la taille demandée. Cela implique que l'indication est désactivée. L'état par défaut de toutes les options ci-dessus est désactivé. Interpréteur: ici, vous pouvez sélectionner la version TrueType GameMaker Studio 2 doit utiliser pour le rendu des polices. La valeur par défaut est TrueType v40. Arrondi: Cette option permet de définir la méthode d'arrondi lors de la conversion d'une taille décimale à virgule fixe en un entier. La valeur par défaut est "étage". Le bouton Régénérer peut être utilisé pour régénérer la texture de la police. Cela est nécessaire si vous souhaitez mettre à jour les modifications que vous avez apportées à une police existante ou si vous avez créé une nouvelle police et que vous souhaitez voir à quoi elle ressemble dans la fenêtre d'aperçu. Notez que vous pouvez également régénérer les polices à partir du bouton droit menu pour la police (ou les polices) dans le navigateur de ressources. Il est important de noter que lorsque vous créez une nouvelle police pour la première fois, si vous ne régénérez pas la texture dans l'éditeur, la texture sera créée la première fois que vous exécuterez ou testerez votre jeu, mais c'est la seule fois sera généré automatiquement. Cela signifie qu'une fois qu'une police a une texture générée, toute modification apportée à la ressource de police nécessitera que la police soit régénérée pour que les modifications soient visibles dans le jeu, sinon la "vieille" texture de police sera toujours utilisée. GameMaker Studio 2 rendra la texture de police générée sous la forme d'un PNG avec le reste des fichiers du projet, et si vous chargez le projet et que la police de base n'est pas trouvée, le jeu affichera toujours le texte correctement en utilisant la texture stockée. Une police manquante signifie cependant que vous ne pouvez plus modifier aucune de ses propriétés, à l'exception de son nom, de sa famille de polices et de son groupe de textures (à partir de l' éditeur de groupe de textures ). Mode aperçu Dans cette section, vous pouvez choisir d'afficher dans la fenêtre d'aperçu un exemple de texte de la police sélectionnée ou l'image de texture rendue qui a été générée pour la police. Aperçu La fenêtre d'aperçu est l'endroit où vous pouvez voir un aperçu de la police, soit sous forme d'échantillon de texte, soit sous forme de texture. Lorsque le mode d'aperçu est défini sur Échantillon, vous pouvez taper le texte que vous souhaitez tester directement dans la fenêtre d'aperçu pour voir à quoi il ressemble, tandis que la sélection de l' aperçu du jeu affichera l'image de texture rendue qui a été générée pour la police. Si aucune image n'est présente, vous devez cliquer sur le bouton Régénérer pour générer la texture. Notez que l'image de texture peut être différente de l'échantillon, car la texture ne sera pas régénérée sauf indication explicite. Cela signifie qu'après avoir initialement généré une texture de police, les modifications apportées à la ressource de police ne seront pas reportées dans le jeu à moins que la texture ne soit régénérée. Aperçu de la plage Ici, les différentes plages de caractères que vous avez ajoutées pour la police seront répertoriées. Les valeurs de plage sont affichées à gauche et les glyphes de plage qui correspondent à ces valeurs sont affichés à droite.
