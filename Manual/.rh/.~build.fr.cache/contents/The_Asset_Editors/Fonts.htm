L'éditeur de polices Lorsque vous voulez dessiner du texte dans votre jeu, ce texte est dessiné dans une police standard Arial 12 points, mais pour créer des textes plus intéressants ou uniques, vous voudrez probablement utiliser des polices différentes. Pour utiliser les différentes polices dont vous disposez sur votre ordinateur, vous devez donc créer une ressource de police dans GameMaker Studio 2. Pour ce faire, cliquez avec le bouton droit de la souris sur dans le navigateur de ressources et sélectionnez "Create" -> "Font". Pour chaque ressource de police que vous créez, vous spécifiez un type de police particulier sur votre ordinateur, qui peut ensuite être utilisé dans votre jeu à l'aide de l'outil de création de police. qui peut ensuite être utilisée dans votre jeu à l'aide de l'action ou du code permettant de définir une police pour le dessin. Lorsque vous créez une ressource de police pour la première fois, la fenêtre de l'éditeur de police s'ouvre avec les options suivantes : Détails Vous pouvez y donner un nom unique à votre police (caractères alphanumériques et symbole "_" sous la barre uniquement), et choisir la police à utiliser. La police doit être installée sur votre ordinateur et, pour des raisons de licence, GameMaker Studio 2 ne stocke pas les fichiers de police avec le fichier de projet par défaut. stocke pas les fichiers de police réels avec le fichier de projet par défaut, mais la police est rendue sur une page de texture sous la forme d'une série de glyphes bitmap. de glyphes bitmap. La page de texture sera affectée à un groupe de texture, que vous pouvez également choisir dans l'option Groupe. Si vous souhaitez distribuer le fichier de police réel ( *.ttf ou *.otf uniquement) avec le projet lors de la création d'une ressource Marketplace ou pour toute autre raison, vous pouvez cocher l'option Copy To Project. Cette option ouvrira l'explorateur de fichiers où vous pourrez sélectionner le fichier de police et il sera alors ajouté aux fichiers fournis avec le projet. IMPORTANT ! Vous ne devez utiliser que des polices que vous êtes légalement autorisé à redistribuer ou à utiliser dans votre projet. Préréglages Dans cette section, vous pouvez définir la taille (en points - la valeur par défaut est 12pt) et indiquer si la police doit être dessinée en gras ou en italique ou si un anticrénelage doit être appliqué. La fenêtre de prévisualisation sur la droite vous montrera le texte tel qu'il apparaîtra dans votre jeu avec la taille et les transformations que vous avez spécifiées. Contrôles de la gamme Lorsque vous créez une ressource de police, il est rare que vous souhaitiez inclure tous les glyphes de la police. C'est pourquoi GameMaker Studio 2 n'utilise par défaut que les caractères de 32 à 127 de la police choisie. les lettres minuscules et majuscules, ainsi que certains signes de ponctuation. Vous pouvez toutefois modifier la plage de caractères utilisée pour optimiser vos jeux. Par exemple, si vous n'avez besoin que des chiffres d'une police spécifique, ne sélectionnez que les chiffres. Pour ce faire Pour ce faire, vous devez d'abord cliquer sur le bouton Supprimer pour effacer la plage actuelle (il se peut que vous deviez sélectionner la plage pour que cela fonctionne en cliquant sur dans la fenêtre d'aperçu). fenêtre d'aperçu), puis cliquez sur le bouton Ajouter pour ajouter une nouvelle plage. Cela ouvrira la fenêtre chaînée suivante : Cette fenêtre comporte quelques boutons qui vous aideront à établir une gamme standard pour votre police : - La plage normale de 32 à 127 - La plage ASCII de 0 à 255 - La plage des chiffres qui contient uniquement les 10 chiffres - La plage des lettres qui contient uniquement les lettres majuscules et minuscules D'autres plages peuvent être utilisées en saisissant le premier et le dernier indice de caractère de la plage que vous souhaitez définir dans les zones de saisie de la plage de caractères (si un caractère ne se trouve pas dans la plage, il est remplacé par un espace). Outre ces boutons et boîtes de saisie, vous disposez également de deux autres options particulièrement utiles : À partir du code: si vous cliquez sur ce bouton, puis sur OK, GameMaker Studio 2 analysera automatiquement le code de votre jeu à la recherche de chaînes de caractères, puis créera différentes plages de caractères pour couvrir tout le texte de votre jeu. Notez qu'il recherche toutes les chaînes de caractères (que ce soit dans "" ou '') et peut donc également inclure les noms de fichiers dans les plages de caractères. From File: cette option fonctionne de la même manière que l'option "from code" expliquée ci-dessus. Si vous cliquez dessus, il vous sera demandé de fournir un fichier, et une fois que c'est fait, vous devez cliquer sur OK. GameMaker Studio 2 analysera alors le fichier et créera des plages de caractères pour le texte qu'il contient. Notez que vous pouvez modifier manuellement la plage de texte en ajoutant ou supprimant des glyphes de texte directement dans la fenêtre d'aperçu. Contrôles des textures Comme mentionné ci-dessus (dans la section sur les détails de la police), par défaut, tous les glyphes de la police seront rendus sur une page de texture en tant que partie d'un groupe de texture pour les convertir en ressources graphiques. Lorsque le jeu est compilé, la police est ajoutée à une texture temporaire pour créer une image unique avec tous les glyphes de police requis, puis elle est ajoutée au groupe de textures principal pour l'exportation. Les boutons ici permettent de modifier comment la texture de la police est générée ou pour régénérer une nouvelle texture de police. Le bouton Options permet d'ouvrir les options avancées de texture de la police : Ces options peuvent modifier, parfois radicalement, la façon dont une police sera affichée dans le jeu, et toutes les polices ne nécessitent pas les mêmes paramètres. Les options suivantes sont conçues pour vous offrir la plus grande personnalisation possible de votre police : Indication: L'indication de la police (également appelée instruction) est l'utilisation d'instructions mathématiques pour ajuster l'affichage d'une police de contour afin qu'elle s'aligne sur une grille tramée. Dans le cas de résolutions d'écran faibles, l'indication est essentielle pour produire un texte clair et lisible. Dans ce cas, vous pouvez choisir parmi différents algorithmes d'indication pour obtenir le meilleur résultat possible. La valeur par défaut est "normal". Options des glyphes: Ces options peuvent être utilisées pour modifier la façon dont les glyphes sont chargés en activant/désactivant différentes opérations. Ces options sont les suivantes : Désactiver le hinting: Ceci désactivera le hinting natif des polices et l'auto-hinter. Désactiver l'auto-interface: Ceci désactivera l'auto-interprétation et utilisera à la place l'intercalaire natif de la police (si elle en a un). Notez que cela remplace l'option "Prefer Auto-hinter". Préférer l'auto-interface: Cette option indique à GameMaker Studio 2 qu'il faut préférer l'utilisation de l'auto-interface à toute indication native que la police peut avoir. Notez que cette option est remplacée par les options "Disable Auto-hinter" et "Disable Hinting". sont prioritaires. Déformation: En activant cette option, les glyphes seront légèrement déformés pour améliorer l'alignement de la grille. Notez que cela nécessite l'auto-interprétation et un mode d'affichage "Normal". Pas d'échelle: Lorsque cette option est cochée, les glyphes restent en unités de police et ignorent la taille demandée. Cela implique que le hinting est désactivé. L'état par défaut pour toutes les options ci-dessus est désactivé. Interpreter: vous pouvez sélectionner ici la version TrueType que GameMaker Studio 2 doit utiliser pour le rendu des polices. La valeur par défaut est TrueType v40. Arrondi: Cette option permet de définir la méthode d'arrondi lors de la conversion d'une taille décimale à virgule fixe en un nombre entier. La valeur par défaut est "floor". Le bouton Régénérer permet de régénérer la texture de la police. Cette opération est nécessaire si vous souhaitez mettre à jour les modifications apportées à une police existante ou si vous avez créé une nouvelle police et que vous souhaitez voir à quoi elle ressemble dans la fenêtre d'aperçu. de la fenêtre d'aperçu. Notez que vous pouvez également régénérer les polices à partir du menu du bouton droit pour la (ou les) police(s) dans le navigateur d'actifs. Il est important de noter que lorsque vous créez une nouvelle police pour la première fois, si vous ne régénérez pas la texture dans l'éditeur, la texture sera créée la première fois que vous exécuterez ou testerez votre jeu, mais ce sera la seule fois où elle sera générée automatiquement. Cela signifie qu'une fois qu'une police a une texture générée, toute modification apportée à la ressource de la police nécessitera que la police soit régénérée pour que les changements soient visibles dans le jeu, sinon la "vieille" texture de la police sera toujours utilisée. GameMaker Studio 2 rendra la texture de la police générée sous la forme d'un fichier PNG avec le reste des fichiers du projet, et si vous chargez le projet et que la police de base n'est pas trouvée, le jeu affichera quand même le texte correctement en utilisant la texture stockée. Une police manquante signifie toutefois que vous ne pouvez plus modifier aucune de ses propriétés, à l'exception de son nom, de sa famille de polices et de son groupe de textures (à partir de l'éditeur de groupe de textures).
