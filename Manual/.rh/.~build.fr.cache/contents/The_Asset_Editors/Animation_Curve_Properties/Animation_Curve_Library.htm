La bibliothèque de courbes d'animation La fenêtre Bibliothèque de courbes d'animation vous permet de modifier le type de courbe de la courbe d'animation et d'appliquer un préréglage au canal sélectionné. Il peut être ouvert par clic gauche sur le dans l' éditeur de courbes d'animation ou la Dope Sheet séquence, ou en cliquez avec le bouton droit n'importe où dans l'éditeur de chaînes. Type de courbe Le type de courbe contrôle le flux d'un canal d'un point à un autre, qui peut être défini sur l'une des options suivantes: Linéaire: Cela créera une progression linéaire entre les points sur le canal Lisse: Cela utilisera l' interpolation catmull-rom pour créer une progression fluide entre les points sur le canal Bézier: Cela utilisera les mathématiques de la courbe de Bézier pour créer une progression douce entre les points sur le canal, où vous pouvez contrôler l'apparence de la courbe en manipulant des «poignées» sur les points Notez que la modification du type de courbe appliquera ce type à tous les canaux de la courbe d'animation. Préréglages de courbes Un C Urve Preset change la forme de votre courbe. Un clic gauche sur un préréglage dans le menu Préréglages appliquera ce préréglage à votre canal sélectionné, et un clic sur Réinitialiser ramènera le canal à ce qu'il était avant l'ouverture de la fenêtre Bibliothèque de courbes d'animation. REMARQUE: les préréglages ne peuvent être appliqués que si le type de courbe est défini sur Bézier. Par défaut, un préréglage est appliqué à l'ensemble du canal. Vous pouvez sélectionner une section de votre chaîne en faisant glisser avec le bouton gauche de la souris dans l' éditeur de canaux, puis ouvrez la fenêtre Bibliothèque de courbes d'animation via le bouton ou par un clic droit, qui appliquera les préréglages à la section sélectionnée uniquement. Notez que votre canal de courbe (ou sélection dans ce canal) doit avoir une pente pour qu'un préréglage ait un effet, car toute interpolation appliquée est basée sur la différence verticale entre les points de départ et de fin. Une courbe plate n'a pas une telle différence entre les deux points, vous ne verrez donc aucun résultat après avoir appliqué un préréglage à une telle courbe. Vous pouvez choisir parmi deux modes d'application, qui affectent la manière dont un préréglage est appliqué à votre canal (ou à votre sélection dans ce canal): Écraser: il s'agit de la valeur par défaut lorsque la fenêtre est ouverte via le dans l'éditeur de courbes d'animation ou la Dope Sheet séquence. Il écrase tous les points entre les points de début et de fin de votre chaîne ou de votre sélection. Entre: Il s'agit de la valeur par défaut lorsque la fenêtre est ouverte par cliquez avec le bouton droit n'importe où dans l'éditeur de chaînes. Il applique le préréglage entre chaque paire de points sélectionnés et ajoute tous les nouveaux points nécessaires sans supprimer aucun point existant. Ecraser supprime tous les points entre les points de départ et de fin, puis applique le préréglage. Vous avez également le choix entre trois modes d'accélération, qui contrôlent la façon dont une courbe accélère ou décélère. Le mode d'accélération par défaut est Ease Out. Facilité d'entrée: la courbe commence par une accélération plus rapide, puis décélère Ease Out: la courbe commence par une accélération plus lente, puis s'accélère avec le temps Facilité d'entrée-sortie: la courbe commence par une accélération plus lente, accélère avec le temps, puis décélère Le visuel suivant montre le préréglage de la courbe Quart dans les trois modes d'accélération: Retour: Courbes d'animation Suivant: Extensions © Copyright YoYo Games Ltd. 2021 Tous droits réservés
