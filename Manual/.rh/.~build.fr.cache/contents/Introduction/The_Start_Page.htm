Démarrer un nouveau projet Vous pouvez cliquer sur le bouton Nouveau pour créer un nouveau projet, sur le bouton Ouvrir pour ouvrir un projet existant ou sur le bouton Importer pour ouvrir un fichier de projet GameMaker: Studio 1.4 GMX ou pour ouvrir un fichier de projet GameMaker Studio 2 compressé YYZ. Les boutons Ouvrir et Importer ouvrent l'explorateur de fichiers pour vous permettre de naviguer jusqu'au fichier de projet souhaité. REMARQUE: GameMaker Studio 2 n'est pas totalement rétrocompatible avec GameMaker: Studio 1.4, mais les projets 1.4 importés devraient tout de même fonctionner, car les fonctionnalités obsolètes ont été recréées automatiquement pour vous à l'aide de scripts de compatibilité. Pour obtenir tous les détails sur les problèmes possibles et les modifications apportées à GML, veuillez consulter l'article du Centre d'aide intitulé Portage d'un projet GMS 1.4 vers GameMaker Studio 2, ainsi que la section du manuel consacrée aux fonctions obsolètes.
