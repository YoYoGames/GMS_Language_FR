La page de démarrage Lorsque vous ouvrez GameMaker Studio 2, il vous sera demandé de vous connecter à votre compte YoYo (ou d'en créer un si vous ne l'avez pas déjà fait - voir ici ). Une fois inscrit et connecté, le principal GameMaker Studio 2 IDE ouvrira la page de démarrage: En haut, vous trouverez les menus généraux de l'IDE qui sont expliqués dans les sections suivantes du manuel: Le menu fichier Le menu d'édition Le menu de construction Le menu Windows Le menu Outils Le menu du Marketplace Le menu des mises en page Le menu d'aide Notez que des options de menu spécifiques au contexte apparaîtront également dans la barre de menu supérieure, en fonction de la fenêtre sur laquelle vous vous êtes concentré actuellement. Par exemple, si vous avez la fenêtre de l'éditeur d'images en focus, vous aurez ici des éléments de menu supplémentaires pour " Images ", " Affichage " et " Effets ". Ces éléments de menu supplémentaires sont expliqués dans les sections pertinentes du manuel pour l'espace de travail ou la fenêtre qui les a générés. Le reste de la page d'accueil se présente comme suit: Accueil - Lors de la navigation dans les différentes sections de la page d'accueil, cliquez sur le bouton gauche de la souris sur cela vous ramènera simplement à la page principale. Détails de la version - Cette section fournit des détails sur la version actuelle de l'EDI utilisée, ainsi que sur la version Runtime Vous serez également informé de tout changement disponible pour l'EDI ou le Runtime dans cette section. Projets récents - Sur la gauche de la page de démarrage, vous pouvez voir une liste des projets précédents que vous pouvez ouvrir. Si vous passez la souris sur eux, vous pouvez voir le nom complet du projet et le chemin, et en cliquant sur le bouton gauche de la souris ouvrira le projet. Social - La barre supérieure de la page d'accueil contient un certain nombre de liens pour faciliter la navigation. Ils sont listés ci-dessous: YoYo Games - vous emmène sur le site principal de YoYo Games Icônes de médias sociaux - elles vous amènent aux différents YoYo Games où vous pouvez vous tenir au courant des actualités de GameMaker Studio 2 Options de projet - La section Projets est l'endroit où vous pouvez créer, ouvrir ou importer des projets. Ceci est expliqué plus en détail plus bas sur cette page. Informations - La section d'informations de la page de démarrage vous permet de parcourir le YoYo Games Marketplace ainsi que de parcourir et d'ouvrir des didacticiels et des démos créés par l' YoYo Games pour montrer des aspects spécifiques de GameMaker Studio 2 (voir ci-dessous pour plus d'informations). Démarrer un nouveau projet Vous pouvez cliquer sur le bouton Nouveau pour créer un nouveau projet, sur le bouton Ouvrir pour ouvrir un projet existant ou sur le bouton Importer pour ouvrir un GameMaker: Studio 1.4 GMX fichier projet ou pour ouvrir un fichier compressé YYZ Fichier de projet GameMaker Studio 2 Ouvrir et importer ouvriront tous les deux l'explorateur de fichiers pour vous permettre de parcourir le fichier de projet dont vous avez besoin. REMARQUE: GameMaker Studio 2 n'est pas complètement rétrocompatible avec les GameMaker: Studio 1.4, mais les projets 1.4 importés doivent toujours s'exécuter, car des fonctionnalités obsolètes ont été recréées automatiquement à l'aide de scripts de compatibilité. Pour plus de détails sur les problèmes possibles et les modifications apportées à GML, veuillez consulter l'article du centre d'aide GameMaker Studio 2 projet GMS 1.4 vers GameMaker Studio 2, ainsi que la section du manuel sur les fonctions obsolètes. Pour créer un nouveau projet, cliquez simplement sur le bouton intitulé Nouveau pour accéder à l'écran suivant: GameMaker Studio 2 vous permet de spécifier le type de projet que vous souhaitez créer, et vous pouvez choisir en fonction de vos connaissances ou habitudes en matière de codage. Les options disponibles sont: Nouveau projet Drag and Drop™ - Cela créera un nouveau projet dans lequel l' éditeur d'objets (et tout autre éditeur autorisant le code) sera configuré pour utiliser l' GameMaker Studio 2 Drag and Drop™. Pour plus d'informations sur le Drag and Drop ( DnD™), cliquez ici. Notez qu'il ne s'agit pas d'une DnD™ exclusive et que vous pouvez basculer entre le DnD™ et GML utilisant le bouton droit de la souris. Nouveau GameMaker Language: Cela créera un projet dans lequel la programmation par défaut est effectuée à l'aide du GameMaker Language ( GML ). Vous pouvez trouver plus d'informations sur GML ici. Notez qu'il ne s'agit pas d'une GML exclusive et que vous pouvez basculer entre le code GML DnD™ en utilisant le bouton droit de la souris. Peu importe ce que vous sélectionnez, vous serez alors invité à choisir un emplacement de sauvegarde pour le projet, puis il sera ouvert dans GameMaker Studio 2. Tutoriels et démos Lorsque vous cliquez sur les boutons Tutoriels ou Démos, vous serez redirigé vers un écran qui montre les tutoriels / démos que vous possédez et les tutoriels / démos officiels disponibles au téléchargement: Pour télécharger l'un des didacticiels ou démos officiels, cliquez simplement sur l'icône de celui que vous souhaitez parcourir et il sera ajouté à votre section "Possédé " en haut (notez que vous devrez peut-être vous connecter au Marketplace avant cela peut être fait). Une fois que le didacticiel ou la démo est dans la section "Possédé", vous pouvez passer la souris dessus et l'installer ou afficher les informations qu'il Marketplace partir de la place de marché: Une fois qu'un didacticiel ou une démo a été installé, il suffit de cliquer dessus pour l'ouvrir et vous serez invité à fournir un emplacement pour enregistrer le projet de didacticiel avant que l'EDI ne s'ouvre pour que vous puissiez commencer. Notez que les didacticiels et démos installés auront une icône supplémentaire qui peut être utilisé pour désinstaller le projet de votre ordinateur (cela ne supprime pas le didacticiel ou la démo de votre liste "Possédé"). Les sections didacticiels / démos de la page de démarrage comportent également quelques boutons pour vous aider à mieux utiliser et naviguer dans vos ressources: Cliquez dessus pour actualiser la liste des projets à partir du Marketplace afin que vous puissiez vérifier si quelque chose a été mis à jour et obtenir les dernières versions. Cliquez dessus pour basculer la vue entre les didacticiels ou les démos que vous possédez et ceux que vous avez installés. Il convient également de noter que vous pouvez également trouver des didacticiels et des démos tiers sur la place de Marketplace. Une fois qu'ils ont été achetés et ajoutés à votre compte, ils apparaîtront également dans la section "Possédé" de la page appropriée.
