Espaces de travail Après s'être connecté et avoir démarré un nouveau projet, GameMaker Studio 2 vous amène à l'espace de travail initial avec quelques fenêtres de base ancrées dans l'IDE. En général, l'espace de travail est simplement une zone où vous pouvez organiser les différentes ressources de votre jeu pendant que vous travaillez : Comme vous pouvez le voir, l'espace de travail initial se trouve sur un onglet en haut de l'écran (et vous pouvez le renommer en double-cliquant sur l'onglet), mais vous pouvez créer d'autres espaces de travail pour le projet en cliquant sur le site sur le côté, ce qui vous donne plusieurs espaces de travail possibles pour un même projet. Par exemple, vous travaillez peut-être sur les interactions entre le joueur et plusieurs objets ennemis. et plusieurs objets ennemis, vous pourriez avoir le joueur dans son propre espace de travail et les objets ennemis dans un autre, et peut-être un autre espace de travail uniquement pour montrer les scripts que les deux utilisent. Une autre caractéristique importante des espaces de travail est que vous pouvez cliquer sur sur l'onglet et - tout en maintenant le bouton de la souris enfoncé - le faire glisser hors de la fenêtre principale de l'IDE dans sa propre fenêtre individuelle, ce qui facilite grandement l'organisation des choses si vous utilisez - par exemple - plusieurs écrans. Vous pouvez également fusionner ces fenêtres secondaires Vous pouvez également réintégrer ces fenêtres d'espace de travail secondaires dans la fenêtre principale en faisant glisser l'onglet vers la première fenêtre. Notez que même si vous semblez avoir deux instances de l'IDE en cours d'exécution lorsque vous faites cela, elles sont toutes deux pour le même projet et vous ne pouvez pas avoir un projet dans l'une et un autre dans l'autre, sauf si vous ouvrez spécifiquement deux instances de GameMaker Studio 2.
