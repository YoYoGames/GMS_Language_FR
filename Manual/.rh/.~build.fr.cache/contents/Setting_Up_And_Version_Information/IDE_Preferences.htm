Préférences de l'IDE La fenêtre des préférences est un élément essentiel de GameMaker Studio 2, car c'est là que vous pouvez configurer et personnaliser votre expérience de l'utilisation de l'IDE, ainsi que la configuration de tout kit SDK de plate-forme spécifique, etc. l'utilisation de l'IDE ainsi que la configuration de tout SDK de plateforme spécifique, etc... Dans cette section, nous couvrons les zones de la fenêtre des préférences spécifiques à l'IDE - pour les plateformes cibles individuelles, veuillez consulter la section sur les préférences des plateformes cibles. Notez qu'après avoir modifié les préférences dans cette fenêtre, vous devez cliquer sur le bouton Appliquer pour qu'elles prennent effet. Si, après avoir modifié les options ici pour personnaliser GameMaker Studio 2 à votre goût, vous souhaitez sauvegarder les préférences que vous avez définies, vous devez vous rendre dans le dossier : %AppData%/GameMakerStudio2// sous Windows, ou le dossier : ~/.config/GameMakerStudio2 sur Mac, puis copiez les deux fichiers suivants dans un endroit sûr : local_settings.json cloud_settings.json Toutefois, si vous souhaitez réinitialiser les préférences que vous avez modifiées, vous devez cliquer sur le bouton Restaurer les valeurs par défaut en haut de la fenêtre. Notez que cette opération ne rétablira les valeurs par défaut que pour la section en cours d'édition.
