Rapport d'erreur Bien que le plus grand soin ait été apporté à la création de GameMaker Studio 2 afin d'en faire un outil stable et fiable, il est toujours possible qu'un problème survienne, étant donné la nature du logiciel et des systèmes d'exploitation sur lesquels il fonctionne. Dans la plupart des cas Dans la plupart des cas, toute erreur sera détectée par GameMaker Studio 2 et il vous sera peut-être demandé d'enregistrer vos projets, puis de fermer et de redémarrer le programme. Si cela se produit, nous vous demandons de signaler un bug au service d'assistance de YoYo Games. Vous pouvez le faire en utilisant l'option Report A Bug du menu Help et nous vous demandons de donner le plus d'informations possible et de fournir un lien vers le fichier GameMaker Studio 2 ui.log, qui se trouve aux endroits suivants, selon le système d'exploitation : Sous Windows - C:\ProgramData\GameMakerStudio2 Sur macOS - /Users/Shared/GameMakerStudio2/ (Macintosh HD > Users > Shared > GameMakerStudio2 ) En de très rares occasions, si l'erreur est grave, le produit peut se bloquer sans donner l'avertissement mentionné ci-dessus, et si cela se produit, la fenêtre suivante s'affichera : Cette fenêtre vous montrera certaines des données du rapport d'accident qui ont été compilées par GameMaker Studio 2 et vous offre la possibilité de les envoyer à YoYo Games. les envoyer à YoYo Games afin qu'ils puissent voir ce qui s'est passé et corriger le problème si possible dans les futures mises à jour. Si vous souhaitez savoir exactement quelles données sont envoyées, veuillez consulter cet article sur le Helpdesk de YoYo Games.
