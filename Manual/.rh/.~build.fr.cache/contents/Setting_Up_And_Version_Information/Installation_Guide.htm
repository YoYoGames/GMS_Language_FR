Guide d'installation GameMaker Studio 2 est conçu pour être utilisé sur macOS et Windows avec les spécifications minimales mentionnées ici. Vous pouvez télécharger le GameMaker Studio 2 partir de la YoYo sur le site Web de YoYo Games. Une fois installé, la première fois que vous exécutez GameMaker Studio 2 vous pouvez être invité à effectuer une mise à jour avant de pouvoir utiliser le produit. Une fois les mises à jour installées, vous devrez fournir les détails de la licence de votre YoYo Games (voir ici pour plus d'informations) avant de pouvoir utiliser le produit, ce qui signifie que la première fois que vous utilisez GameMaker Studio 2 vous devez avoir un connexion Internet. Après la connexion initiale, vous devriez être en mesure d'utiliser le produit hors ligne, mais vous devrez périodiquement vous connecter pour vérifier le compte de connexion. Il convient de noter que, selon le type de licence dont vous disposez, vous devrez peut-être installer des composants supplémentaires ou suivre des étapes supplémentaires pour configurer et utiliser GameMaker Studio 2 pour des plates-formes spécifiques. Pour obtenir des informations complètes sur les besoins de chaque plate-forme, veuillez consulter cet article sur le Helpdesk YoYo Games : GMS2 Required SDKs. Versions stables et bêta La version que vous téléchargez à partir de la page Web de YoYo Games doit être la version stable GameMaker Studio 2, mais il peut également y avoir une autre version "Beta " disponible qui est utilisée pour présenter ou tester de nouvelles fonctionnalités avant de passer à une version finale stable. Les détails de la version complète sont les suivants: Stable: Cette version est celle que vous utiliseriez pour le développement général et est recommandée pour une utilisation avec tout produit commercial. Certaines fonctionnalités disponibles dans les versions bêta peuvent manquer, mais c'est parce qu'elles n'ont pas été entièrement testées ou nécessitent un travail supplémentaire. Bêta: La version bêta de GameMaker Studio 2 est destinée à ceux qui souhaitent essayer de nouvelles fonctionnalités avant d'être ajoutées à la version Stable, ou à ceux qui nécessitent un correctif à venir pour un problème. Il n'est pas recommandé d'utiliser la version bêta pour la production finale du jeu à moins qu'elle ne contienne un correctif important pour un bogue qui retarde votre projet. Vous pouvez obtenir la version bêta de GameMaker Studio 2 partir de la YoYo, et elle peut être installée et exécutée côte à côte avec la version stable. Veuillez noter que si vous exécutez GameMaker Studio 2 via Steam vous n'aurez pas accès à la version bêta du produit en utilisant Steam lui-même, bien que vous puissiez toujours télécharger et utiliser la version de bureau.
