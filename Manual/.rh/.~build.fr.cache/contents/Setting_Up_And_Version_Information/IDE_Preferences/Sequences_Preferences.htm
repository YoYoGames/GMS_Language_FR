Préférences des séquences Les préférences de séquences sont utilisées pour définir certaines propriétés du fonctionnement de l' éditeur de séquences et de ses éléments. Les principales options sont: Afficher le cadre du canevas sur les séquences nouvellement créées: lorsqu'il est activé, le guide intelligent du cadre du canevas sera affiché dans le canevas de l'éditeur de séquence pour toutes les nouvelles séquences. Cette option peut être activée / désactivée selon les besoins à partir des propriétés du canevas dans l'éditeur lui-même. La valeur par défaut est activée. Largeur du cadre du canevas sur les séquences nouvellement créées: Ici, vous pouvez définir la largeur du cadre par défaut pour toutes les nouvelles séquences. La valeur par défaut est 1366px. Ca nvas frame height sur les séquences nouvellement créées: Ici, vous pouvez définir la hauteur de cadre par défaut pour toutes les nouvelles séquences. La valeur par défaut est 768px. Pause en cliquant sur le canevas: lorsque cette option est cochée, cliquer n'importe où dans le canevas de l'éditeur de séquence mettra en pause la lecture de la séquence. C'est activé par défault. Délai de sélection de chevauchement en millisecondes: cela contrôle le temps que vous devez attendre avant que la boîte à outils d'élément apparaisse pour les éléments qui se chevauchent sur le canevas de l'éditeur de séquence. La valeur par défaut est de 1000 ms. Marge d'insertion du point de chemin de traduction: dans le canevas de l'éditeur de séquence, lorsque vous avez ajouté un élément et créé une piste de paramètre de position, des lignes pointillées seront dessinées pour illustrer le chemin emprunté par l'élément lors du déplacement. Si vous placez votre souris près de cette ligne, vous pouvez y ajouter un nouveau point de suivi de paramètre, et cette option définit la distance par rapport à la ligne de translation que la souris doit être avant que le marqueur de point ne soit affiché. La valeur par défaut est 6px. Créer automatiquement des pistes de paramètres par défaut: lorsqu'elle est activée, cette option ajoute automatiquement des pistes de paramètres pour la position, l'échelle, la rotation et l'origine à toutes les nouvelles pistes créées dans le panneau des pistes de la séquence. La valeur par défaut est désactivée. Étirer automatiquement les actifs nouvellement créés: cette option étendra toutes les clés d'actif nouvellement créées pour correspondre à la longueur de la séquence. La valeur par défaut est désactivée. Hauteur de la piste de courbe animée dans la feuille d'expo: cette option peut être utilisée pour définir le nombre de pistes de paramètres de lignes de piste qui utilisent des courbes dans la feuille dopw lorsqu'elles sont dans la vue d' keyframe La valeur par défaut est 2, mais vous pouvez également sélectionner 1 ou 3. Confirmer les boîtes de dialogue Les préférences de confirmation des boîtes de dialogue sont utilisées pour modifier les messages affichés dans l'éditeur de séquence lorsque vous effectuez certaines actions. Les messages que vous pouvez modifier sont: Choix automatique d'incorporer la courbe modifiée: Si vous essayez d'incorporer une courbe d'animation que vous avez modifiée, un message s'affiche par défaut pour confirmer l'action. Cependant, la définition de cette option sur "Oui" n'affichera pas le message et ne l'incorporera pas à chaque fois. Le paramétrer sur "Non" empêchera l'incorporation à chaque fois. Choix automatique de convertir la courbe en clés: Si vous essayez de convertir une courbe d'animation en keyframes, un message s'affiche par défaut vous demandant si vous souhaitez continuer. Cependant, le réglage de cette option sur "Ok" n'affichera pas le message et ne vous permettra tout simplement pas de convertir la courbe. Choix automatique d'afficher un avertissement lors de la modification de la courbe externe: Si vous essayez de modifier une courbe appliquée à une piste de paramètres et qu'il s'agit d'un élément de courbe d'animation externe, un message d'avertissement s'affiche par défaut. Cependant, la définition de cette option sur "Ok" n'affichera pas le message et vous permettra de le modifier de toute façon. Guides intelligents Ici, vous pouvez sélectionner le comportement des guides intelligents lorsque vous travaillez dans l'éditeur de séquences. Les options disponibles sont: Activer les guides intelligents: ici, vous pouvez activer ou désactiver les guides intelligents dans le canevas de l'éditeur de séquences. C'est activé par défault. Activer les règles: ici, vous pouvez activer ou désactiver les guides intelligents dans le canevas de l'éditeur de séquences. C'est activé par défault. Aligner les règles sur les bords: cette option permet aux règles de «s'aligner» sur les bords des éléments dans le canevas des séquences (et vice versa). La valeur par défaut est activée. Distance d'accrochage: C'est la distance (en pixels) qu'un élément doit être à une règle (ou une règle à un élément) pour qu'il "s'accroche" à la position. La valeur par défaut est 6px.
