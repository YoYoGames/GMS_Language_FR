Préférences de l'éditeur de jeux de mosaïques Les préférences de l'éditeur de jeux de mosaïques ont l'option suivante pour la mosaïque automatique: Couleur d'arrière-plan de la mosaïque automatique: cette option définit la couleur d'arrière-plan dans l'éditeur de jeu de mosaïques dans la section de création de mosaïques automatiques. La couleur par défaut est (rgba) # 808080FF. Les préférences de l'éditeur de jeux de mosaïques comportent également une sous-section de dialogue de confirmation: L'option disponible ici contrôle le message qui apparaît lorsque vous modifiez la taille de cellule du jeu de tuiles et cela affectera la façon dont les couches de plan de mosaïque de pièce sont rendues (résultant en des couches rognées). Par défaut, vous serez invité à continuer ou à annuler l'opération, et cette option vous permet de supprimer ce message en sélectionnant "OK", ce qui ira de l'avant avec le changement de cellule du jeu de tuiles quel que soit l'impact sur les couches de plan de tuiles de la pièce.
