Préférences de la chronologie Les préférences de montage permettent de définir certaines propriétés lorsque vous travaillez avec l' éditeur de chronologie. Les options disponibles sont: Réponse automatique à la DnD™ en GameMaker Language: lorsque vous utilisez le bouton droit de la souris option de menu sur un moment de la chronologie pour convertir les Drag and Drop™ en script (code), vous obtiendrez une invite vous demandant si vous souhaitez continuer, car il s'agit d'une conversion à sens unique. Changer cette option sur "OK" supprimera le message et poursuivra la conversion comme si vous aviez cliqué sur "OK" dans la boîte de dialogue. Le paramètre par défaut est "Afficher le message". Une réponse automatique à l'avertissement d'action permanente: certaines actions dans une chronologie ne peuvent pas être annulées, et lorsque cela se produit, vous recevrez un message d'avertissement vous demandant si vous souhaitez continuer. Changer cette option sur "OK" supprimera le message et poursuivra l'action comme si vous aviez cliqué sur "OK" dans la boîte de dialogue. Le paramètre par défaut est "Afficher le message". Ouvrir de nouveaux moments à modifier lors de la création: Normalement, lorsque vous ajoutez un nouveau moment à une chronologie, GameMaker Studio 2 ouvrira automatiquement l'éditeur pour que vous puissiez y ajouter votre code ou Drag and Drop. Cependant si vous décochez cette option, le moment sera ajouté sans ouvrir l'éditeur (vous pouvez l'ouvrir en double-cliquant en ce moment). Par défaut, cette option est activée.
