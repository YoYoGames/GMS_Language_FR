Préférences de l'éditeur d'images Les préférences de l'éditeur d'images sont utilisées pour définir l'apparence et l'apparence de la fenêtre de l' éditeur d'images. Voici les options générales: Développer la superposition des contrôles de sprite par défaut: lorsque cette option est cochée, les contrôles de sprite et les informations sur le cadre seront affichés dans l'éditeur d'images chaque fois que vous ouvrez un sprite pour modification (cela peut être refermé à l'aide du bouton Réduire, mais il rouvrira lors de l'édition du sprite à nouveau plus tard). Lorsque cette option n'est pas cochée, les commandes de sprite et les informations sur le cadre seront masquées lors de l'ouverture d'un sprite. La valeur par défaut est de ne pas cocher cette case. Mode de boucle par défaut: le bouton de boucle dans l'éditeur d'images peut être configuré pour fonctionner dans deux modes différents: soit jouer une fois, où l'animation s'arrêtera quand elle atteindra la dernière image, soit en boucle depuis le début, où l'animation reviendra. la première image à la fin, ou ping-pong, où l'animation remontera à travers les images jusqu'à la fin, puis les redescendra au début. La valeur par défaut ici est "boucle". Dessiner un contour de sélection fin: si cette option est cochée, le contour des sélections aura une largeur absolue de 1px, quelle que soit l'échelle de l'image, tandis que le fait de ne pas cocher le contour de la sélection donnera au contour de la sélection une largeur et une échelle relatives de 1px avec le image. Par défaut, cette option est cochée. Couleur primaire du contour de la sélection: le contour de la section est une ligne en pointillés qui alterne entre deux couleurs. Cette option vous permet de définir la première de ces couleurs à utiliser (y compris alpha) et a une valeur par défaut de ARVB # FFFFFD0. Couleur secondaire du contour de la sélection: le contour de la section est une ligne en pointillés qui alterne entre deux couleurs. Cette option vous permet de définir la seconde de ces couleurs à utiliser (y compris l'alpha) et a une valeur par défaut ARVB # 0000D0. Couleur d'arrière-plan de l'éditeur: lors de l'édition d'un cadre de sprite, l'image peut être mise à l'échelle ou ne pas s'adapter à l'ensemble du dessin dans la fenêtre, auquel cas une couleur d'arrière-plan sera affichée que vous pouvez définir ici. La valeur par défaut est ARVB # 282828FF. Les préférences de l'éditeur d'images comportent également les sous-catégories suivantes: Fond Ici, vous pouvez définir les options de l'arrière-plan sur lequel vous modifiez vos images de sprite. Par défaut, il s'agit d'une grille de carrés gris, mais vous pouvez utiliser les paramètres suivants pour la modifier: Style de canevas: Ici, vous définissez le "style" pour l'arrière-plan, qui peut être " Motif de bloc " ou " Couleur unique ". La couleur unique définit simplement l'arrière-plan sur la couleur de votre choix (en utilisant le paramètre Couleur 1, comme expliqué ci-dessous) tandis que le motif de bloc est le paramètre par défaut (qui peut également être modifié à l'aide du reste des options de dialogue ci-dessous). La valeur par défaut est " Modèle de bloc ". Taille du bloc de canevas: lorsque vous avez sélectionné Motif de bloc pour l'arrière-plan, vous pouvez définir ici la taille du bloc individuel. La valeur par défaut est 16px, avec une valeur minimale de 2px et un maximum de 256px autorisé. Couleur principale de la toile: si vous avez choisi une couleur pour l'arrière-plan, la couleur utilisée est celle-ci. Vous pouvez cliquer sur l'échantillon de couleur pour ouvrir l'éditeur de couleurs et le modifier. Lorsque vous utilisez l' option Motif de bloc, cela définit la première couleur des blocs alternatifs. La couleur par défaut est ( ARVB ) # FFC3C3C3. Couleur secondaire du canevas (style de motif de bloc): Si vous avez choisi un motif de bloc pour l'arrière-plan, la couleur utilisée pour alterner avec la couleur 1 (ci-dessus) est celle-ci. Vous pouvez cliquer sur l'échantillon de couleur pour ouvrir l'éditeur de couleurs et le modifier. La couleur par défaut est (ARVB) # FF808080. Confirmer les boîtes de dialogue Ici, vous pouvez sélectionner le comportement des boîtes de dialogue de confirmation lorsque vous effectuez différentes actions sur le sprite actuel. Elles sont: Réponse automatique à la suppression de cadres: lorsque vous supprimez un cadre par défaut, un message s'affiche pour vous demander si vous êtes sûr de vouloir le faire. En définissant cette option sur "OK", vous pouvez supprimer ce message et autoriser la suppression du cadre sans que d'autres messages ne soient affichés. La valeur par défaut est "Afficher le message". Réponse automatique au remplacement des images lors de l'importation de bandes: lorsque vous sélectionnez Importer une bande dans le menu déroulant Image, vous recevrez un message vous avertissant que vous allez écraser toutes les images déjà dans la ressource de sprite, avec l'option de continuer ou non. Vous pouvez définir ici si afficher ce message ou demander à l'EDI de choisir simplement l'option "Oui" ou "Non". La valeur par défaut est "Afficher le message". Réponse automatique à la validation des modifications lors du changement d'outil: lorsque vous modifiez les outils de dessin après avoir modifié un cadre, un message s'affiche par défaut pour vous demander si vous souhaitez valider les modifications apportées. En définissant cette option sur "OK", vous pouvez supprimer ce message et autoriser la validation de l'édition de trame sans que d'autres messages ne soient affichés lorsque vous changez d'outils. Si vous choisissez "Non", l'édition du cadre sera annulée silencieusement lorsque vous changez d'outil. La valeur par défaut est "Afficher le message". Réponse automatique à l'avertissement de couche verrouillée: si vous essayez de modifier une couche verrouillée, vous recevrez un message par défaut vous informant que vous ne pouvez pas faire cela. Si vous définissez cette option sur "OK", aucun message ne sera affiché et l'édition échouera silencieusement. Réponse automatique à l'avertissement de groupe sélectionné: Si vous sélectionnez plusieurs calques dans l'éditeur de calques, puis essayez de dessiner ou de modifier, un message s'affichera par défaut pour vous informer que ce n'est pas possible. Vous pouvez sélectionner "OK" ici pour supprimer ce message et échouer silencieusement à éditer les images ensemble jusqu'à ce qu'un seul calque soit sélectionné. La valeur par défaut est "Afficher le message". Paramètres de la grille Les préférences Paramètres de la grille sont utilisées pour définir la façon dont la grille de l'image sera présentée, avec les options suivantes disponibles: Espacement horizontal de la grille (px): cette valeur définit l'espacement des lignes horizontales de la grille de l'image. La valeur par défaut est 1px. Espacement vertical de la grille (px): cette valeur définit l'espacement des lignes verticales de la grille de l'image. La valeur par défaut est 1px. Afficher la grille: si cette case est cochée, l'éditeur d'images s'ouvrira avec la grille d'image activée. La valeur par défaut est désactivée. Aligner sur la grille: si cette case est cochée, l'éditeur d'image alignera automatiquement certains outils sur la grille d'image. La valeur par défaut est désactivée. Couleur de la ligne de la grille: Ici, vous pouvez définir la couleur (y compris la valeur alpha) de la grille de l'image. La valeur par défaut est ( ARVB ) # FF000000. Outils Ici, vous pouvez définir les différentes options de fonctionnement et d'affichage de certains des outils de dessin. Si vous utilisez l' outil Arc alors ici vous pouvez définir les différentes couleurs utilisées pour dessiner les propriétés du nœud: Couleur du point d'ancrage: définissez la couleur des différents points d'ancrage sur la spline / l'arc. La valeur par défaut est ( ARVB ) # FFFF0000. Couleur du point de contrôle: définissez la couleur de chacune des «poignées» de point de contrôle sur un point d'ancrage. La valeur par défaut est (ARVB) # FFFF0000. Couleur de la ligne tangente: définissez la couleur des lignes reliant les points de contrôle aux ancres. La valeur par défaut est (ARVB) # FF00FFFF. Couleur du point sélectionné: définissez la couleur à utiliser pour tous les points d'ancrage sélectionnés sur la spline / l'arc. La valeur par défaut est (ARVB) # FF00FF00. Taille du point: définissez la taille des différents points d'ancrage et points de contrôle à utiliser. La valeur par défaut est 10 px.
