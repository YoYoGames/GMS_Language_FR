Préférences de chemin Les préférences Chemin contrôlent les différents chemins de fichiers requis par les éléments de l' GameMaker Studio 2 ainsi que pour les éditeurs externes dont vous avez besoin. Les options suivantes existent pour l'EDI: Répertoire temporaire: emplacement de sauvegarde de tous les dossiers temporaires. Par défaut sur Windows c'est\Users\\AppData\Local\Temp\GameMakerStudio2 Et sur macOS c'est:/var/folders//GameMakerStudio2 Répertoire de cache IDE: emplacement de sauvegarde du cache IDE. Par défaut sur Windows c'est\Users\\AppData\Roaming\GameMakerStudio2\Cache Et sur macOS c'est:/Users//.config/GameMakerStudio2/Cache Répertoire du cache d'actifs: emplacement d'enregistrement du cache d'actifs pour chaque projet. Par défaut sur Windows c'est\Users\\AppData\Roaming\GameMakerStudio2\Cache Et sur macOS c'est:/Users//.config/GameMakerStudio2/Cache Emplacement «Mes projets»: l'emplacement où GameMaker Studio 2 créera initialement de nouveaux projets. Par défaut sur Windows c'est\Users\\Documents\GameMakerStudio2 Et sur macOS c'est/Users//GameMakerStudio2 Supprimer automatiquement le répertoire temporaire à la fermeture: GameMaker Studio 2 à supprimer automatiquement le dossier Temp qu'il crée par projet pour la compilation etc... Ce paramètre est activé par défaut, et le décocher le désactivera (ce qui signifie que vous devrez supprimer manuellement tous les fichiers temporaires plus tard). Supprimer automatiquement le cache d'actifs à la fermeture: si cette case est cochée, le dossier de cache du compilateur d'actifs sera supprimé lorsque vous GameMaker Studio 2. Cette option est désactivée par défaut, et l'activer signifie que chaque fois que vous chargez et exécutez un projet, le cache devra être reconstruit (ce qui peut prendre du temps en fonction de la taille du jeu). Supprimer automatiquement le cache IDE à la fermeture: si cette case est cochée, le dossier de cache du compilateur IDE sera supprimé lorsque vous GameMaker Studio 2. Cette option est désactivée par défaut, et l'activer signifie que chaque fois que vous démarrez GameMaker Studio 2, le cache devra être reconstruit. Supprimer le dossier temporaire: cliquez sur ce bouton pour supprimer le dossier temporaire du projet. Supprimer le cache d'actifs: Cliquer sur ce bouton supprimera le cache d'actifs du compilateur pour le projet. Supprimer le cache IDE: Cliquer sur ce bouton supprimera le cache IDE Les options suivantes existent pour définir les chemins vers les éditeurs externes: Chemin d'accès à l'éditeur / visualiseur externe pour les fichiers SWF: Si vous travaillez avec des sprites au format SWF, vous pouvez le définir sur le chemin de votre visualiseur / éditeur préféré et lorsque vous cliquez sur le bouton Modifier l'image dans l'éditeur de sprite, il ouvrira le fichier donné. programme. La valeur par défaut ici est de ne pas avoir de chemin. Chemin d'accès à l'éditeur / visionneuse externe pour les fichiers Spine: Si vous travaillez avec des sprites au format Spine, vous pouvez le définir sur le chemin de votre visionneuse / éditeur préférée et lorsque vous cliquez sur le bouton Modifier l'image dans l'éditeur de sprite, il ouvrira le fichier donné. programme.
