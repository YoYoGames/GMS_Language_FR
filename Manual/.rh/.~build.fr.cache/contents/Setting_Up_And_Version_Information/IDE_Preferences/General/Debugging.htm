Préférences de débogage Les préférences sont ici liées au débogueur deGameMaker Studio 2 et vous disposez des options suivantes : Réponse automatique à la fermeture du débogueur: Cette option vous permet de définir la réponse automatique à la fermeture du débogueur. Si elle est réglée sur "Show message", le dialogue s'affichera lorsque vous voudrez fermer le débogueur. Si elle est réglée sur "Oui" ou "Non", ce dialogue ne s'affichera pas et la tâche sera exécutée comme indiqué. Activer le débogage en temps réel: Si vous cochez cette option, vous pourrez déboguer vos projets de jeu en temps réel lorsque vous les exécuterez à partir de l'IDE. Cette option est cochée par défaut. Port du débogueur: Ce paramètre définit le numéro de port à utiliser par le module de débogage. La valeur par défaut est 6509. Afficher les icônes d'événements dans la vue des ressources du débogueur: Indiquez si les icônes doivent être affichées dans la vue des ressources du débogueur. Cette option est cochée par défaut.
