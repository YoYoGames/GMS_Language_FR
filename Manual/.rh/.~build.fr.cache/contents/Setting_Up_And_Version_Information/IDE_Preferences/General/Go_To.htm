Aller aux préférences Les préférences concernent ici la fenêtre Aller à, que vous pouvez ouvrir à l'aide du raccourci clavier / + T ou en cliquant avec le bouton droit de la souris sur la zone vide d'un espace de travail ( ). Nombre maximal de résultats de recherche à afficher: Cette option contrôle le nombre maximum de résultats renvoyés lorsque vous utilisez la fenêtre de recherche "Aller à". Activer le style des résultats de recherche (à désactiver pour de meilleures performances): Cette option active ou désactive le style du texte pour les actifs trouvés dans la fenêtre Aller à, et est activée par défaut. La désactivation de cette option devrait améliorer les performances si vous trouvez que la fenêtre de recherche Aller à est lente sur votre système. Rechercher dans les actifs: Cette option, activée par défaut, permet à la fenêtre Go To de rechercher dans le navigateur d'actifs tous les actifs qui correspondent au terme recherché. Rechercher dans les préférences: Cette option, activée par défaut, permet à la fenêtre "Aller à" de rechercher dans les préférences toutes les options qui correspondent au terme recherché. Rechercher dans les options du jeu: Cette option permet à la fenêtre "Aller à" de rechercher dans les options du jeu toutes les options qui correspondent au terme recherché. Elle est activée par défaut. Recherche dans les fonctions définies par l'utilisateur: Cette option permet à la fenêtre Aller à de rechercher dans vos scripts et objets toutes les fonctions qui correspondent au terme de recherche, et est activée par défaut. Notez que cette option ne fonctionne qu'avec la syntaxe de fonction de script(function name()) et non avec la syntaxe de variable de méthode(name = function()).
