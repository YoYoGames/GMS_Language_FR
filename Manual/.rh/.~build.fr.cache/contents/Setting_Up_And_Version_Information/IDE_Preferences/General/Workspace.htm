Préférences de l'espace de travail Les préférences définies ici influent sur la manière dont vous vous déplacez dans les différents espaces de travail de GameMaker Studio 2 espaces de travail. Les options sont les suivantes : Portée des degrés de navigation du clavier de l'espace de travail: Dans un espace de travail, vous pouvez avoir plusieurs fenêtres ouvertes pour différents objets et ressources. Pour vous déplacer rapidement entre elles, vous pouvez utiliser le raccourci clavier / + + . Étant donné que les fenêtres d'un espace de travail ne sont pas alignées sur une grille, l'IDE doit choisir la fenêtre que vous souhaitez ouvrir lorsque vous appuyez sur l'une des 8 directions possibles. cette option vous permet de définir l'"angle de vue" utilisé. La valeur par défaut est de 40°. Remplissage des colonnes de la chaîne de l'espace de travail (px): Les éléments de l'espace de travail sont souvent "enchaînés" les uns aux autres (par exemple, l'éditeur de code est enchaîné à l'éditeur d'événements est enchaîné à l'éditeur d'objets). la distance horizontale entre deux fenêtres chaînées. La valeur par défaut est de 30px. Remplissage des rangées de la chaîne de l'espace de travail (px): Comme mentionné ci-dessus, les éléments de l'espace de travail sont souvent "enchaînés" les uns aux autres et cette option vous permet de définir la distance verticale entre deux fenêtres enchaînées. La valeur par défaut est de 20px. Nombre de segments de chaîne de rendu de l'espace de travail: Vous pouvez définir ici le nombre maximum de segments pour les chaînes de rendu. Réduire cette valeur peut améliorer les performances sur les machines les moins performantes. La valeur par défaut est de 20. Couleur des chaînes de l'espace de travail: il s'agit de la couleur qui sera utilisée pour afficher les chaînes entre les éléments de l'espace de travail. La couleur par défaut est ARGB $99DD00. La navigation au clavier dans l'espace de travail réinitialise le zoom: Comme mentionné ci-dessus, vous pouvez passer d'une fenêtre à l'autre dans l'espace de travail à l'aide de raccourcis clavier. Vous pouvez également effectuer un zoom avant et arrière dans l'espace de travail en utilisant les touches / + la molette de la souris . Si vous activez cette option, si vous avez zoomé sur l'espace de travail et que vous utilisez les raccourcis pour passer à une autre fenêtre, l'espace de travail sera ramené à une vue 1:1 et centré sur la fenêtre sélectionnée. Cette option est désactivée par défaut. Les chaînes d'espaces de travail peuvent se chevaucher: Lorsqu'elle est activée, cette option permet aux chaînes d'espace de travail de se croiser et de se chevaucher.
