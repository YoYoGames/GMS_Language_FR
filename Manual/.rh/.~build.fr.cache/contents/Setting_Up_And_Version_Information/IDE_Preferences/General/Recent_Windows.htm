Préférences récentes de Windows Les préférences contiennent des options sur la façon dont la liste des fenêtres récentes affiche les éléments : Déplacer les fenêtres ciblées vers le haut: Lorsque cette option est activée et que vous cliquez sur un élément de la liste des fenêtres récentes, l'élément sélectionné est déplacé vers le haut de la liste. Par défaut, cette option est désactivée. Afficher uniquement les fenêtres actuellement ouvertes: En activant cette option, la liste des fenêtres récentes supprimera les éléments qui ne sont plus ouverts dans un espace de travail (c'est-à-dire que la fermeture d'une fenêtre la supprimera également de la liste). Par défaut, cette option est désactivée. Nombre maximal de fenêtres à afficher: Ceci définit le nombre maximum de fenêtres précédemment ouvertes qui seront affichées dans la liste des fenêtres récentes. La valeur par défaut est 10.
