Préférences d'aide Le fichier d'aide est conçu pour fonctionner comme une page Web et même si les fichiers sont stockés localement, il faut utiliser un navigateur externe pour les visualiser. Les paramètres suivants relatifs au fichier d'aide peuvent être modifiés ici : Langue du manuel: Ceci définit la langue à utiliser par l'IDE. La valeur par défaut est "English". Port manuel d'aide: Ce paramètre définit le numéro de port à utiliser par le serveur web micro manuel. La valeur par défaut est 51291.
