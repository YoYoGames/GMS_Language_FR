Préférences de ressources Les préférences ici contiennent des options sur la façon dont certaines ressources seront dessinées dans vos projets: Couleur de dessin par défaut initiale pour les nouveaux projets: lorsque vous utilisez une police ou dessinez une forme à l'aide des fonctions GameMaker Studio 2 ou des DnD™ sans spécifier de couleur, la couleur définie ici est celle qui sera utilisée. La couleur par défaut est le blanc ( ARVB: $FFFFFFFF ). Couleur de fenêtre par défaut initiale pour les nouveaux projets: cette option est utilisée pour définir la couleur d'arrière-plan de la page HTML pour la cible HTML5, ainsi que pour définir la couleur affichée à l'extérieur des pièces sur toutes les cibles (y compris HTML5).
