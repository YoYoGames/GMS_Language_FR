Préférences deWindows Les préférences de Windows comportent les options suivantes : Emplacement deVisual Studio: Cette option permet de faire pointer GameMaker Studio 2 vers l'installation de Microsoft Visual Studio sur le PC de construction. Type d'emballage par défaut: Lorsque vous créez un paquet final exécutable pour Windows, vous pouvez choisir de créer soit un installateur (en utilisant NSIS), soit un fichier compressé ZIP . qui contiendra le jeu et tous les fichiers supplémentaires. Par défaut, une boîte de dialogue vous permettra de choisir entre ces deux options lors de la compilation, mais vous pouvez définir cette préférence pour toujours utiliser l'une ou l'autre. Notez que lorsque cette option est réglée sur Afficher la boîte de dialogue, dans la fenêtre de dialogue qui s'affiche lors de la compilation, l'activation de l'option "Se souvenir de l'option d'emballage" définira cette préférence en fonction de l'option sélectionnée.
