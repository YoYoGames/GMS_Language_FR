Préférences macOS Les macOS ont les options suivantes: Identifiant d'équipe par défaut: ici, vous pouvez ajouter votre identifiant d'équipe par défaut, tel qu'il vous a été attribué par Apple. Cet identifiant d'équipe sera utilisé lorsque vos fichiers de jeu seront envoyés à Xcode pour créer l'application et permettra à Xcode de générer les certificats de signature requis. Notez que ce paramètre sera appliqué par défaut à tous les jeux conçus pour macOS, mais il peut être remplacé par projet à partir des options de jeu macOS. Choix d'empaquetage automatique: ici, vous pouvez définir une préférence par défaut pour créer un exécutable pour la cible macOS L'option "Afficher la boîte de dialogue" affichera toujours une boîte de dialogue avant de créer un exécutable, vous demandant de choisir entre DMG et Zip, mais vous pouvez définir cette option sur l'une de ces options dans les Préférences pour ignorer cette boîte de dialogue. Notez que l'activation de "Mémoriser l'option de package" dans la boîte de dialogue définira également cette option sur le type de package sélectionné.
