PréférencesWindows UWP Les préférences de Windows UWP comportent les options suivantes : Emplacement deVisual Studio: Cette option permet de faire pointer GameMaker Studio 2 vers l'installation de Microsoft Visual Studio sur le PC de construction. Emplacement des outils UWP: Cette option permet de faire pointer GameMaker Studio 2 vers le chemin d'installation des outils UWP sur le PC de construction. Architecture du paquet par défaut: Par défaut, une fenêtre de dialogue vous permet de choisir entre les architectures x86, x64 et ARM lors de la compilation d'un exécutable UWP. Vous pouvez définir cette option à l'une de ces trois architectures qui sera alors sélectionnée par défaut lorsque vous choisirez de compiler un exécutable. Réponse automatique à la création d'un paquet de magasin: dans la fenêtre de dialogue affichée lors de la compilation, vous pouvez choisir de créer un paquet de magasin. Vous pouvez définir cette option sur Oui ou Non qui sera utilisée par défaut chaque fois que vous choisissez de compiler un exécutable.
