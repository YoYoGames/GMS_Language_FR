Préférences iOS Les iOS ont les options suivantes: Supprimer la génération et l'exécution: par défaut, lorsque vous créez un iOS, puis que vous créez un package exécutable final, GameMaker Studio 2 construira le package, puis tentera de l'exécuter sur tous les appareils connectés. Cocher cette option supprimera la création de l'application et son exécution sur l'appareil et créera simplement un projet XCode sur le Mac hôte. Chemin d'installation macOS: chemin d'installation des fichiers de projet sur le Mac de construction. Identifiant d'équipe par défaut: ici, vous pouvez ajouter votre identifiant d'équipe par défaut, tel qu'il vous a été attribué par Apple. Cet identifiant d'équipe sera utilisé lorsque vos fichiers de jeu seront envoyés à Xcode pour créer l'application et permettra à Xcode de générer les certificats de signature requis. Notez que ce paramètre sera appliqué par défaut à tous les jeux conçus pour iOS, mais il peut être remplacé par projet à partir des options iOS.
