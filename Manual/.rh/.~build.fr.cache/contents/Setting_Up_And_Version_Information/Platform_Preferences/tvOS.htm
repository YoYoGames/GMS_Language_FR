PréférencestvOS Les préférences de tvOS comportent les options suivantes : Réponse automatique à l'avertissement de limitation du stockage persistant de tvOS: les jeux tvOS ont un stockage local limité pour les fichiers, et lorsque vous compilez pour cette cible, un avertissement sera affiché par défaut pour vous rappeler ce fait. Vous pouvez modifier Vous pouvez modifier le paramètre ici pour supprimer cet avertissement en sélectionnant "OK". Le paramètre par défaut est "Show Message". Supprimer la construction et l'exécution: Par défaut, lorsque vous créez un projet tvOS, puis un package exécutable final, GameMaker Studio 2 construit le package, puis tente de l'exécuter sur tous les appareils connectés. En cochant cette option, vous supprimera la construction et l'exécution de l'application sur l'appareil et créera simplement un projet XCode sur le Mac hôte. Chemin d'installation demacOS: Le chemin d'installation des fichiers du projet sur le Mac de construction. Identifiant d'équipe par défaut: Vous pouvez ajouter ici votre identifiant d'équipe par défaut, tel qu'il vous a été attribué par Apple. Cet identifiant d'équipe sera utilisé lorsque vos fichiers de jeu seront envoyés à Xcode pour construire l'application, et permettra à Xcode de générer les certificats de signature requis. Notez que ce paramètre sera appliqué par défaut à tous les jeux construits pour tvOS, mais qu'il peut être modifié pour chaque projet à partir de la section Options générales tvOS tvOS.htm">Options du jeu.
