Préférences HTML5 Les préférences HTML5 ont les options suivantes: Type de package par défaut: lors de la construction d'un package exécutable HTML5 final, vous pouvez choisir entre Package en tant que fichiers en vrac - qui générera simplement tous les fichiers requis dans un dossier - ou Package en Zip - qui créera ZIP fichier qui contiendra le jeu et tous les fichiers supplémentaires qu'il contient. Par défaut, vous aurez le choix parmi ces deux options que vous souhaitez choisir lors de la compilation, mais vous pouvez définir cette préférence pour toujours utiliser l'une ou l'autre. Notez que lorsque cette option est définie sur Afficher le dialogue, dans la fenêtre de dialogue qui apparaît lors de la compilation, choisir "Mémoriser l’option d’emballage" définira cette préférence sur l’option sélectionnée. Port du serveur Web par défaut: cela vous permet de modifier le port par défaut de tous les périphériques créés via le Gestionnaire de périphériques. La valeur par défaut est 51264. Adresse du serveur Web par défaut: ici, vous pouvez modifier l'adresse du serveur Web par défaut pour les périphériques que vous ajoutez au Gestionnaire de périphériques. La valeur par défaut est 127.0.0.1. IP autorisées par le serveur Web par défaut: Ici, vous pouvez ajouter toutes les adresses IP autorisées pour le serveur Web et elles seront définies par défaut pour tous les périphériques ajoutés via Target Manager.
