Support Proxy Si vous avez besoin d'un support Proxy pour votre installation de GameMaker Studio 2, vous pouvez l'ajouter en cliquant simplement sur le Proxy. La fenêtre suivante s'affiche alors : Vous pouvez maintenant ajouter les détails de votre procuration. Certains des champs indiqués ci-dessus sont facultatifs et s'il y en a un dont vous n'avez pas besoin alors laissez simplement le champ vide. Avant de poursuivre, il convient de noter que si vous avez défini un proxy dans les options Internet deWindows et qu'il ne requiert pas de nom d'utilisateur ou de mot de passe, alors il ne sera pas nécessaire d'entrer un mot de passe. qu'il ne nécessite pas de nom d'utilisateur/mot de passe, il doit rester tel quel et vous n'aurez probablement pas à le configurer dans GameMaker Studio 2. Si votre proxy nécessite un nom d'utilisateur et/ou un mot de passe, vous devrez les ajouter dans les champs indiqués dans la fenêtre ci-dessus. Notez que dans ces cas, les détails du nom d'utilisateur/mot de passe/proxy seront toujours requis, et que l'IP du proxy doit également avoir le port ajouté, c'est-à-dire : . le port doit être ajouté à l'adresse IP du proxy, par exemple : xx.xx.xx.xx:port. Une fois que vous avez ajouté les informations pertinentes, cliquez sur OK et connectez-vous à GameMaker Studio 2 comme d'habitude.
