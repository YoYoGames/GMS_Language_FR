Informations sur la licence Une fois que vous avez téléchargé et installé GameMaker Studio 2, la première fois que vous l'utilisez, vous serez invité à saisir les détails de votre licence: Votre licence est liée à un compte YoYo (YYA) et vous devez en avoir créé un pour pouvoir utiliser GameMaker Studio 2. Ce compte contient tous les détails de votre licence pour les YoYo Games que vous possédez, et est également utilisé pour le YoYo Games Marketplace. Vous trouverez de plus amples informations sur le YYA sur la page suivante: Créer un compte YoYo IMPORTANT! Après avoir acheté une nouvelle licence pour GameMaker Studio 2 nous vous recommandons de vous déconnecter de l'EDI, puis de redémarrer GameMaker Studio 2 et de vous reconnecter pour vous assurer que la nouvelle licence est correctement appliquée. Vous pouvez vous déconnecter du menu Fichier. Support proxy Si vous avez besoin du support proxy pour votre installation de GameMaker Studio 2, vous pouvez l'ajouter simplement en cliquant sur le proxy. Cela fera apparaître la fenêtre suivante: Vous pouvez maintenant ajouter les détails de votre proxy. Certains des champs indiqués ci-dessus sont facultatifs et s'il y en a un dont vous n'avez pas besoin, laissez simplement le champ vide. Avant de continuer cependant, il convient de noter que si vous avez défini un proxy dans vos options Internet Windows, et qu'il ne nécessite pas de nom d'utilisateur / mot de passe, il devrait simplement "tel quel" et vous ne serez probablement pas obligé de configurer le proxy dans GameMaker Studio 2. Si votre proxy nécessite un nom d'utilisateur et / ou un mot de passe, vous devrez les ajouter aux champs affichés dans la fenêtre ci-dessus. Notez que dans ces cas, le nom d'utilisateur / mot de passe / les détails du proxy seront toujours requis, et que l'adresse IP du proxy devrait également avoir le port ajouté, c'est-à-dire: xx.xx.xx.xx:port. Une fois que vous avez ajouté les informations pertinentes, cliquez sur OK et connectez-vous à GameMaker Studio 2 comme d'habitude.
