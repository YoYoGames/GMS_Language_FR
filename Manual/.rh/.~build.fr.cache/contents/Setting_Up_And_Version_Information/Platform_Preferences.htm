Préférences de la plate-forme Cette section du manuel couvre les différentes préférences spécifiques à la plate-forme qui doivent être configurées avant de pouvoir compiler vos jeux vers certaines cibles. Celles-ci ne doivent généralement être configurées qu'une seule fois pour commencer (bien que les futures mises à jour des runtimes puissent vous obliger à modifier certaines choses, mais celles-ci seront mentionnées dans les notes de publication ) et sont utilisées pour pointer vers tous les SDK requis et définis de base. compiler options pour la façon dont vous voulez GameMaker Studio 2 pour compiler vos projets de jeux pour toute plate - forme donnée. Notez que les plates-formes disponibles dépendront de la licence dont vous disposez et que toutes ne seront peut-être pas disponibles. Vous trouverez ci-dessous une liste de toutes les plates-formes cibles disponibles pour les licences standard: Android HTML5 iOS tvOS macOS Steam Windows Windows UWP Notez qu'en plus de configurer ces préférences, vous devrez également configurer les périphériques ciblés par certaines de ces plates-formes, sinon vous ne pourrez peut-être pas compiler ni tester vos projets. Voir la section sur la compilation et la section sur le gestionnaire de périphériques pour plus d'informations. Le manuel ne montre pas les différentes préférences de la console (XboxOne, PS4 et Nintendo ) en raison des différentes restrictions légales maintenues par les différentes sociétés. Si vous avez une licence de console, vous pouvez obtenir ces informations YoYo Games Helpdesk de YoYo Games, mais vous devez d'abord contacter YoYo Games avec votre adresse e-mail d'ID de console et demander l'autorisation d'afficher les différentes sections.
