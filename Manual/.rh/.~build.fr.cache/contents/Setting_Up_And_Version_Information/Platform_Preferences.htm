Préférences de la plate-forme Cette section du manuel couvre les différentes préférences spécifiques à la plate-forme qui doivent être configurées avant que vous puissiez compiler vos jeux vers certaines cibles. Ces préférences ne doivent généralement être configurées qu'une fois pour commencer (bien que les futures mises à jour de l'application runtimes peuvent vous obliger à modifier certaines choses, mais celles-ci seront mentionnées dans les notes de mise à jour) et sont utilisées pour indiquer les SDKrequis et définir les options de compilation de base pour la façon dont vous souhaitez que GameMaker Studio 2 compile vos projets de jeu pour une plate-forme donnée. Notez que les plates-formes disponibles dépendent de la licence que vous possédez et qu'elles ne sont pas toutes disponibles. Vous trouverez ci-dessous une liste de toutes les plateformes cibles disponibles pour les licences standard : Android HTML5 iOS tvOS macOS Vapeur Windows Windows UWP Notez qu'en plus de configurer ces préférences, vous devrez également configurer les périphériques ciblés par certaines de ces plates-formes, sinon vous ne pourrez pas compiler ni tester vos projets. Voir la section sur la compilation et la section sur le gestionnaire de périphériques pour plus d'informations. Le manuel n'indique pas les préférences des différentes consoles (Xbox One, PS4 et Nintendo) en raison des différentes restrictions légales maintenues par les différentes sociétés. Si vous possédez une licence Console, vous pouvez obtenir ces informations auprès du Helpdesk YoYo Games. le Helpdesk deYoYo Games, mais vous devez d'abord contacter YoYo Games avec l'email de votre ID de console et demander la permission de voir les différentes sections.
