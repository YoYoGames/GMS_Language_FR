Le menu d'édition Le menu d'édition est l'endroit où vous pouvez utiliser les outils d'édition généraux de l'interface de GameMaker Studio 2. Les options disponibles sont les suivantes : Annuler - Annule une action. Redo - Refaire une action précédente. Rechercher et remplacer - Cela ouvrira une fenêtre de recherche qui vous permettra de rechercher dans l'ensemble de votre code (y compris le code de création de la pièce, les scripts et les événements) pour une entrée donnée - voir ci-dessous pour plus d'informations. Lorsque vous sélectionnez Search & Replace dans le menu Edit, la fenêtre suivante s'ouvre : Vous pouvez l'utiliser pour rechercher tous les actifs scriptés de votre jeu, y compris le code de la pièce, le code de l'événement, les scripts et les shaders. Lors de la recherche, vous pouvez saisir un nouveau terme de recherche dans le champ situé en haut de l'écran, ou cliquez sur l'icône en forme de flèche à droite pour ouvrir une liste de termes précédemment recherchés. Vous avez également la possibilité d'indiquer un terme de remplacement qui sera utilisé pour remplacer le terme de recherche initial. Vous avez également la possibilité d'indiquer un terme de remplacement qui sera utilisé pour remplacer le terme de recherche initial, et cette option permet également d'utiliser une chaîne de remplacement précédemment utilisée à partir du bouton fléché situé à droite. Les recherches peuvent être générales ou vous pouvez les affiner en utilisant les options situées sous les entrées de termes de recherche : Vous pouvez choisir de rendre la recherche sensible à la casse, auquel cas elle n'inclura que les mots qui correspondent exactement à la casse de la chaîne de recherche. Par exemple, si vous recherchez "Mon_Var" sans cette option, vous obtiendrez des résultats avec "ma_var", "mY_VAR", ou "My_Var", mais si cette option est cochée, seul "My_Var" sera inclus dans les résultats de la recherche. Vous pouvez choisir de n'inclure que les résultats de recherche qui incluent le mot entier uniquement, auquel cas tous les résultats qui incluent le terme recherché mais qui ne sont pas une correspondance exacte seront ignorés, c'est-à-dire qu'une recherche pour "game" sans cocher cette option affichera des éléments tels que la fonction "game_end()" ou des variables telles que "game_state", alors que si vous cochez cette option et que vous effectuez la recherche, seuls les résultats contenant le mot "game" lui-même seront affichés. Vous pouvez choisir de faire en sorte que la recherche ignore les commentaires. Lorsque cette option est cochée, tous les commentaires de votre code qui contiennent le terme recherché seront simplement ignorés et n'apparaîtront pas dans les résultats. Vous pouvez appliquer un filtre sur les résultats de la recherche en fonction des différents types de biens dans le navigateur de biens. Par défaut, ce filtre est réglé sur "Tout", mais en cliquant sur ce bouton, le menu suivant s'ouvre et vous pouvez cocher ou décocher les éléments suivants les actifs à inclure dans la recherche : Une fois que vous avez donné un terme de recherche (et éventuellement un terme de remplacement) et que vous avez configuré les options de recherche, vous pouvez utiliser les différents boutons en bas pour choisir comment effectuer la recherche et les actions à entreprendre. Ces boutons sont les suivants : Rechercher tout: Trouver tous les termes recherchés et les afficher dans la fenêtre de sortie. Find Previous: Trouver l'instance précédente du terme recherché. Cela ouvrira le poste contenant la référence dans l'espace de travail (s'il n'est pas déjà ouvert) et effectuera un panoramique vers celui-ci. Trouver le suivant: Trouver la prochaine instance du terme recherché. Cela ouvrira la ressource contenant la référence dans l'espace de travail (si elle n'est pas déjà ouverte) et effectuera un panoramique vers celle-ci. Remplacer tout: Cette option permet de rechercher le terme donné avec les options définies et de le remplacer dans les ressources par le nouveau terme de remplacement que vous avez spécifié. Remplacer: Cela remplacera le terme actuellement sélectionné dans l'éditeur actuel où il est mis en évidence par le terme de remplacement sélectionné. Arrêter la recherche: Arrêter la recherche en cours si une recherche est en cours. Si vous appuyez sur la touche Échap pendant que cette fenêtre est ouverte, vous effacerez le champ sélectionné, mais si ce champ est déjà vide, vous fermerez la fenêtre. Tous les résultats de l'utilisation de la fenêtre Rechercher et Remplacer s'affichent dans la fenêtre de sortie de recherche au bas de l'IDE. Vous pouvez double-cliquer sur chaque résultat à l'adresse afin d'ouvrir le poste contenant le terme dans l'espace de travail actuel et d'effectuer un panoramique vers celui-ci, prêt à être édité.
