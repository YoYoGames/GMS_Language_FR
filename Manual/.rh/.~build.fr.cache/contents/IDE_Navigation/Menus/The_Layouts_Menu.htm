Le menu des mises en page Ce menu est lié à la sauvegarde et au chargement de différents agencements de l'IDE. Vous pouvez configurer l'IDE comme vous le souhaitez, avec certains éléments (comme le navigateur d'actifs, ou les fenêtres de sortie) disposés différemment de la configuration par défaut. Cela vous permet d'adapter GameMaker Studio 2 à votre flux de travail et, grâce aux options de ce menu, vous pouvez enregistrer les modèles du projet en cours dans un fichier *.xml qui peut ensuite être chargé dans n'importe quel projet pour que les modèles soient identiques. Les options de ce menu sont les suivantes : Charger la mise en page - Cela ouvrira un explorateur de fichiers où vous pouvez sélectionner un fichier *.xml de mise en page précédemment enregistré pour le charger. Enregistrer la mise en page - Cela ouvrira un explorateur de fichiers où vous pourrez nommer et enregistrer la mise en page actuelle de l'IDE en tant que fichier *.xml. Réinitialiser la disposition - Ceci réinitialisera la disposition actuelle de l'IDE aux paramètres par défaut.
