Le menu Outils Ce menu vous permet d'ouvrir les différents outils autonomes qui sont à votre disposition dans GameMaker Studio 2, avec les options suivantes : Groupes de textures - Affiche la fenêtre des groupes de textures. Groupes audio - Affiche la fenêtre des groupes audio. Sound Mixer - Affiche la fenêtre Sound Mixer. Generate Project Images - Ouvre la fenêtre Generate Project Images. Créer un package local - Ouvre la fenêtre Asset Packager, afin que vous puissiez créer et enregistrer des packages d'actifs locaux pour une utilisation ultérieure. Importer un package local - Ouvre une fenêtre d'exploration de fichiers dans laquelle vous pouvez sélectionner un package d'actifs locaux et l'importer, soit en l'ajoutant au projet actuel, soit en créant un nouveau projet pour lui. pour celui-ci.
