Le menu du marché Outre la création de vos propres ressources et de votre propre code pour vos projets, vous avez également accès à la place de marché de GameMaker Studio 2, où vous pouvez télécharger des ressources gratuites et payantes à ajouter à vos projets. Vous pouvez en savoir plus sur le Marché en consultant la section sur le GameMaker MarketplaceLes options de ce menu sont indiquées ci-dessous : Aller sur la place de marché - Ouvre la fenêtre de la place de marché où vous pouvez rechercher des actifs. Ma bibliothèque - Affichez votre bibliothèque de contenu Marketplace actuelle. Historique de mes achats - Cette option ouvrira un navigateur et vous amènera à la page du compte de l'utilisateur connecté afin que vous puissiez voir l'historique des achats pour ce compte. Mettre à jour un paquet existant - Cette option ouvrira l'assistant de mise à jour de paquet afin que vous puissiez mettre à jour un paquet Marketplace existant que vous avez précédemment créé. Créer un package - Cette option vous permet d'ouvrir l'assistant de création de package pour créer un nouveau package d'actifs ou un tutoriel pour la place de marché.
