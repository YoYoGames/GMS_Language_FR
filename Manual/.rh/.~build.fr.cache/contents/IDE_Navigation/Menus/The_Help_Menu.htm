Le menu d'aide Ce menu permet d'obtenir de l'aide lors de l'utilisation de GameMaker Studio 2. Il contient les éléments de menu suivants: Page de démarrage - Cela ouvrira un nouvel espace de travail de la page de démarrage. Page d'accueil - Cela ouvrira le diaporama de la page d'accueil initiale dans un nouvel espace de travail. Ouvrir le manuel - Ouvre le document d'aide. Notez que cliquer sur le bouton d'aide dans la barre de menu ouvrira également le document, tout comme en appuyant sur. Base de connaissances - Cela vous mènera au centre d'aide GameMaker Studio 2 où vous pouvez trouver des articles sur la configuration de la plate-forme, les problèmes connus, l'utilisation de GameMaker Studio 2 et un certain nombre d'autres choses. Signaler un bogue - Si vous trouvez des bogues ou rencontrez des problèmes lors de l'utilisation de GameMaker Studio 2 vous pouvez les signaler ici. YoYo Games.com - Ouvre la page YoYo Games. Communauté GameMaker - Cela ouvrira les forums de la communauté GameMaker dans votre navigateur. Notez qu'il s'agit d'un forum géré par la communauté, donc si vous avez un rapport de bogue ou avez besoin d'une assistance technique, vous devez d'abord vous rendre sur le site de support officiel (voir ci-dessus). Notes de mise à jour - Cela ouvrira les notes de mise à GameMaker Studio 2 dans votre navigateur. Si vous avez besoin d'annuler une mise à jour, vous pouvez obtenir les programmes d'installation de la version précédente en cliquant sur le lien du numéro de version sur cette page. Notes de Runtime actuel - Étant donné que le Runtime peut se mettre à jour indépendamment de l'EDI, il dispose de son propre ensemble de Notes Runtime. La sélection de cette option les ouvrira dans votre navigateur, mais notez que le runtime actuel n'est peut-être pas celui que vous utilisez. Pour savoir sur quel runtime d'exécution vous vous trouvez, ou pour savoir comment ajouter, supprimer ou activer des runtimes, veuillez consulter la section Préférences d' Runtime SDK requis - En raison de la nature multiplateforme de GameMaker Studio 2 un certain nombre de SDK tiers sont nécessaires. En cliquant dessus, vous accédez à un article du forum d'assistance qui détaille ce dont vous avez besoin par plate-forme. Ouvrir le projet dans l'explorateur - Cela ouvrira l'emplacement de sauvegarde du projet actuellement chargé dans l'explorateur de fichiers du système d'exploitation. À propos de - Ouvre la fenêtre «À propos de GameMaker Studio 2 » dans laquelle vous pouvez obtenir des informations sur la version du produit (indispensable pour signaler tout bogue ou problème).
