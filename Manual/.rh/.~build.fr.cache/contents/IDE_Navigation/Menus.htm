Les menus Le haut de l' GameMaker Studio est occupé par la barre de menus. Vous trouverez ici un certain nombre de menus déroulants qui peuvent être utilisés pour naviguer dans l'EDI, effectuer des tâches ou ouvrir des fenêtres supplémentaires avec des outils pour vous aider à créer vos jeux. Notez que nous listons ici les menus par défaut qui sont disponibles, mais certains des éditeurs d'actifs ajouteront des menus supplémentaires en fonction du contexte. Ces menus supplémentaires sont expliqués sur les pages relatives à l'éditeur qui les a créés.
