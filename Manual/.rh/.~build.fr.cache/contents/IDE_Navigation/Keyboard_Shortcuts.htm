Raccourcis clavier Il existe un grand nombre de raccourcis clavier disponibles pour GameMaker Studio 2, et bien que la plupart d'entre eux soient indiqués dans l'IDE à côté des éléments de menu auxquels ils se rapportent, tous ne le sont pas ou ne sont pas évidents pour l'utilisateur au départ. Vous trouverez ici une liste complète de tous les raccourcis clavier pour un guide de référence facile. En général, les raccourcis se répartissent dans les trois catégories suivantes : Global - Les raccourcis globaux sont ceux qui fonctionnent quel que soit l'endroit où vous vous trouvez dans l'IDE de GameMaker Studio 2, à moins que votre attention ne se porte sur une fenêtre ou un gadget spécifique qui le remplace. Fenêtre - Les raccourcis de fenêtre sont ceux qui sont spécifiques à une seule fenêtre ou un seul onglet, sauf si un raccourci de gadget le remplace. Gadget - Les raccourcis des gadgets sont très spécifiques et ne fonctionnent que lorsqu'on utilise un seul gadget (le sélecteur de couleur, par exemple).
