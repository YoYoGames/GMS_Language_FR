Windows récentes La Windows récentes peut être ouverte à partir du Windows et affichera une courte liste (par défaut, elle contient un maximum de 10 éléments) des fenêtres que vous avez précédemment ouvertes. Pour commencer, cette liste sera vide, mais lorsque vous ouvrez différentes fenêtres dans l'EDI (comme un sprite ou un script ou un objet, etc.), la liste sera remplie. Si vous avez besoin de revenir à l'une de ces fenêtres précédemment ouvertes, il vous suffit de cliquer sur le bouton gauche de la souris sur l'élément de la liste et l'espace de travail se déplacera vers la fenêtre. Si vous cliquez sur le bouton droit de la souris sur un élément, vous ouvrirez alors le menu suivant: Ici, vous pouvez choisir de supprimer un élément (ou des éléments, comme vous pouvez + ou / + pour sélectionner plusieurs) dans la liste ou vous pouvez les déplacer vers le haut de la liste. Le menu RMB a également une option supplémentaire vous permettant d'effacer complètement la liste. Il convient de noter que la Windows récentes peut être ancrée sur les côtés de l'EDI et que vous pouvez également ajuster certaines préférences pour modifier son comportement.
