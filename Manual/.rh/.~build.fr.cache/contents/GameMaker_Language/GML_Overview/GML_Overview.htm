Aperçu dulangage GameMaker Cette section du manuel contient toutes les informations nécessaires pour comprendre et utiliser le logiciel GameMaker Studio 2 langage de script GML. Les fonctions d'exécution de ce langage peuvent être utilisées pour créer vos jeux et sont ajoutées aux objets à partir de l éditeur d'objetsnormalement lorsque vous avez choisi de créer un nouveau projetGameMaker Language bien qu'il puisse également être utilisé avec le langage DnD™(glisser-déposer). Ci-dessous, vous pouvez voir une image typique d'un objet avec l'éditeur de code ouvert sur un événement : Chaque événement a son propre onglet dans l'éditeur et vous pouvez ajouter, modifier ou supprimer du code à tout moment (pour plus d'informations sur les événements, voir Événements d'objets). Le code lui-même doit avoir une structure de base et peut contenir des indices de ressources, des variables, des fonctions, des expressions, des mots-clés... etc... tous ces éléments sont expliqués dans les sections ci-dessous. Si vous êtes novice en matière de programmation ou si vous faites la transition depuis DnD™, il est recommandé de commencer par la page sur la Structure de code de base puis de lire toutes les autres pages de cette section, en testant le code de chacune d'entre elles au sein de GameMaker Studio 2 lui-même.
