vies Cette variable a une portée globale et est utilisée pour contenir une valeur numérique qui est généralement utilisée pour les vies des joueurs. Cette variable est uniquement conçue pour prendre en charge les projets hérités des versions précédentes de GameMaker et devrait ne pas utiliser dans les nouveaux projets car elle pourrait être obsolète à l'avenir. Syntaxe : vies ; Les retours : Réel (valeur à virgule flottante de simple précision) Exemple if (lives < 5) { lives += 1 ; } Le code ci-dessus vérifie la variable lives et si elle est inférieure à 5, on lui ajoute 1.
