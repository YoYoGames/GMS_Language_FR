nombre d'arguments Cette variable en lecture seule renvoie le nombre d'"arguments" qui sont transmis à une fonction de script ou à une méthode. Normalement, elle est utilisée en conjonction avec un tableau d'arguments (argument [0...15]) pour permettre de varier les arguments d'entrée pour une fonction donnée. NOTE: Ceci n'est plus strictement requis et est fourni pour le support de l'héritage plus qu'autre chose. Toutes les fonctions définies par l'utilisateur accepteront des arguments variables, et tout argument passé dans une fonction qui ne fait pas partie de la fonction nommée sera initialisé à undefined par défaut, ce qui peut être vérifié en utilisant la fonction is_undefined(). Syntaxe : argument_count ; Les retours : Entier Exemple : function my_array() { var i, arg ; for (i = 0 ; i < 5 ; i += 1 ;) { if argument_count > i { arg[i] = argument[i] } else { arg[i] = -1 ; } } } Le code ci-dessus utilise la variable argument_count ainsi que le tableau argument pour initialiser les variables, et constitue un exemple de la façon dont on peut permettre à une fonction définie par l'utilisateur d'accepter un nombre variable d'arguments.
