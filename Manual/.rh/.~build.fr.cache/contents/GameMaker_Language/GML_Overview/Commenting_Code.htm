Code de commentaire Lorsque vous travaillez en collaboration avec d'autres personnes, lorsque vous vous occupez d'un grand projet, ou même simplement pour votre propre référence et le débogage, il est très important de laisser des commentaires dans votre code d'événement et vos scripts. Dans GameMaker Studio 2, vous disposez de plusieurs mécanismes pour vous aider à laisser des notes et des commentaires sur des sections de code, voire des blocs entiers de code, visibles dans la liste d'événements de l'objet que vous êtes en train de modifier. Vous pouvez ainsi laisser des notes à vous-même et à vos collègues, ou expliquer un morceau de code particulièrement délicat. Pour laisser un simple commentaire d'une ligne, la première chose à faire est d'utiliser // avant le texte. Par exemple : //initialiser les variables sr = 10 ; hp = 100 ; Vous pouvez également laisser des commentaires de plusieurs lignes, pour donner du crédit, pour omettre une section complète du code pour le débogage, ou même pour laisser des notes sur la façon dont vous voulez modifier le code à l'avenir.
