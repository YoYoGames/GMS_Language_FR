Variables interdites dans la structure Lors de la création d'une structure, il est possible d'utiliser certaines variables intégrées comme noms de variables membres, par exemple : mystruct = { object_index : obj_Player, speed : 3, image_blend : c_red } Toutefois, seules les variables intégrées locales et d'instance peuvent être utilisées de cette manière et se voir attribuer des valeurs comme s'il s'agissait de variables membres de structures ordinaires. L'utilisation de variables intégrées de portée globale est interdite et entraînera des problèmes dans votre jeu. Vous trouverez ci-dessous une liste complète de ces variables afin que vous sachiez lesquelles éviter. Les variables globales intégrées suivantes sont toutes disponibles pour être utilisées dans vos projets mais ne peuvent pas être utilisées comme variables membres de la structure : argument [n]
