début / fin Le langage GameMaker vous permet d'utiliser les mots-clés begin et end au lieu des accolades {} plus habituelles lors de la création de blocs de code. L'exemple de code ci-dessous montre comment cela fonctionne : if (!visible) begin exit ; end while (place_meeting(x, y)) begin x -= lengthdir_x(1, direction - 180) ; y -= lengthdir_y(1, direction - 180) ; end Notez que l'utilisation de ces mots-clés n'est pas habituelle et qu'ils sont fournis dans le cadre du langage plus pour des raisons d'héritage que pour autre chose, et à tout moment dans le futur, ils peuvent être dépréciés.
