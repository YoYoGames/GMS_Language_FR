begin / end Le GameMaker Language vous permet d'utiliser les mots-clés begin et end au lieu des accolades plus habituelles {} lors de la création de blocs de code. L'exemple de code ci-dessous montre comment cela fonctionne:if (!visible)     begin     exit;     end while (place_meeting(x, y))     begin     x -= lengthdir_x(1, direction - 180);     y -= lengthdir_y(1, direction - 180);     end Notez que l'utilisation de ces mots-clés n'est pas typique et est fournie dans le cadre du langage plus pour le support hérité que pour toute autre chose, et à tout moment dans le futur, ils peuvent être obsolètes.
