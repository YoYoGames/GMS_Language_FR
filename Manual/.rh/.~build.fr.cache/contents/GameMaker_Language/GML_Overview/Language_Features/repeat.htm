repeat Le GameMaker Language a un certain nombre de façons dont vous pouvez effectuer des boucles, c'est-à-dire: faire itérer une ou plusieurs instructions sur elle-même un certain nombre de fois. Le plus simple d'entre eux est le repeat déclaration, qui a la forme:repeat ()     {     ;     ;     ...     } Avec repeat l' instruction donnée est répétée le nombre de fois indiqué par la valeur arrondie de l' expression. Par exemple, ce qui suit crée cinq balles à des positions aléatoires:repeat (5)     {     instance_create_layer(random(400), random(400), "Instances", obj_ball);     } Cela peut être très utile pour éviter de taper plusieurs fois le même code, ou pour utiliser des tableaux, ou pour compter à travers un certain nombre d'opérations, etc. les entre accolades {}. Par exemple:var _x = 32; repeat (global.p_lives)     {     draw_sprite(spr_heart, 0, _x, 32);     _x += sprite_get_width(spr_heart);     } L'exemple ci - dessus reprend les déclarations contenues dans les accolades pour que soit le nombre d' itération de la variable globale « vie » a, et chaque itération dessine l'image - objet cardiaque au _x position, puis déplace la position le long d'un peu en fonction de la largeur du sprite cœur. Quand devriez-vous utiliser un repeat boucle? Chaque fois que vous souhaitez répéter une ou plusieurs instructions un nombre fixe de fois sans qu'il soit nécessaire de maintenir un nombre d'itérations. Il est à noter que vous pouvez utiliser le spécial break et continue instructions dans une boucle de répétition aussi. Utilisant break quittera immédiatement la boucle et passera à tout code qui est dans l'événement ou la fonction après que la boucle aurait dû se terminer, par exemple:var i = 0; var temp = 0; repeat (10)     {     temp += array[i];     if (temp > max_total)         {         break;         }     else         {         i += 1;         }     } Le code ci-dessus parcourt 10 valeurs de tableau et les ajoute à une variable locale. Si le total de la variable locale est supérieur à la valeur donnée pour max_total, la boucle se termine par break, sinon la boucle continuera. Un exemple d'utilisation continue dans un repeat boucle serait:repeat(10)     {        var _x = random(room_width);     var _y = random(room_height);     if (instance_position(_x, y, obj_Enemy)         {         continue;         }     instance_create_layer(_x, _y, "Instances", obj_Enemy);     } Ce code se répétera 10 fois, générant une position de pièce aléatoire puis vérifiant si une instance de l'objet obj_Enemy existe à cette position. Si tel est le cas, l'itération de la boucle actuelle se termine en utilisant continue et une nouvelle itération est lancée, et si ce n'est pas le cas, une instance de l'objet obj_Enemy est créé à la position aléatoire. Pour plus d'exemples de fonctions de boucle, veuillez consulter les sections sur while, do... until, et for.
