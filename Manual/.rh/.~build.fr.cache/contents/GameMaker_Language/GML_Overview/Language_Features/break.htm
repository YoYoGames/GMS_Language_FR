rupture L'instruction break est utilisée pour terminer prématurément une pour, repeat, while ou do / until ou une boucle quelconque, ou pour indiquer à une instruction switch de s'arrêter à ce point, ou pour terminer prématurément un appel avec appel. Veuillez consulter les pages individuelles de ces différentes fonctions pour obtenir une pour obtenir une explication plus approfondie de la façon dont elle peut être utilisée dans chaque circonstance. Notez que si break est utilisé en dehors de l'un de ces contextes, une erreur sera générée. Vous pouvez voir ci-dessous quelques exemples de base de l'utilisation de break, dont la syntaxe est simple : pause ; dans une boucle for: for (var i = 0 ; i < 10 ; i += 1) { if (array[i] == 234) { break ; } } num = i ; dans uneboucle de répétition: var i = 0 ; var temp = 0 ; repeat (10) { temp += array[i] ; if (temp > max_total) { break ; } else { i += 1 ; } } pause dans une boucle while: var i = 0 ; while (!place_free(x, y)) { x = random(room_width) ; y = random(room_height) ; if (i > 50) { break ; } else { i+=1 ; } } break dans une boucle do / until: var _id = noone ; do { _id = list[| 0] ; if instance_exists(_id) { _break ; } ds_list_delete(list, 0) ; } until (ds_list_empty(list)) ; target = _id ; se brisent lorsqu'ils sont utilisés avec: var count = 0 ; with (obj_Enemy) { count++ ; if (count > 10) { break ; } hp = 100 ; } break dans un switch: switch (keyboard_key) { case vk_left : case ord("A") : x -= 4 ; break ; case vk_right : case ord("D") : x += 4 ; break ; case vk_up : case ord("W") : y -= 4 ; break ; case vk_down : case ord("S") : y += 4 ; break ; } Retour : Caractéristiques de la langue Suivant : continuer Copyright YoYo Games Ltd. 2021 Tous droits réservés
