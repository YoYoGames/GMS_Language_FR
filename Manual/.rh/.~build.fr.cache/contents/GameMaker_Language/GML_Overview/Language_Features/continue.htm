continuer L'instruction continue a la syntaxe de base suivante : continuer ; Si elle est utilisée à l'intérieur d'une instruction qui forme une boucle (pour, repeat, while ou do / until), elle met immédiatement fin à l'itération actuelle actuelle et retournera au début de la boucle en commençant une nouvelle itération et en omettant tout le code qui vient après le continue dans la boucle. Il peut également peut également être utilisé dans l'instruction avec où il fera en sorte que le code passe à l'instance suivante et s'exécute à nouveau. Notez que si continue est utilisé en dehors de l'un de ces contextes, une erreur sera générée.
