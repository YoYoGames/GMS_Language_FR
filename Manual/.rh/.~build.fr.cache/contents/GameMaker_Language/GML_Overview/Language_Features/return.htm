retourner L'instruction de retour a la syntaxe suivante : retour () Vous n'utilisez l'instruction return que dans les fonctions et méthodes de script, et elle est utilisée pour renvoyer une valeur de la fonction qui sera utilisée dans d'autres appels de code ou de fonction. de la fonction. Il convient de noter que l'exécution de la fonction se termine à l'instruction return , ce qui signifie que tout code intervenant après l'appel de return ne sera pas exécuté. Voici un court exemple de script appelé sqr_calc qui calcule le carré de n'importe quelle valeur qui lui est passée, et qui inclut un système de détection d'erreur au cas où le paramètre qui lui est passé n'est pas un nombre réel : /// @function sqr_calc(val) ; /// @param {real} val La valeur dont il faut calculer le carré /// @description Calculer le carré de la valeur donnée function sqr_calc(value) { if !is_real(value) { return 0 ; } else { return (valeur * valeur) ; } } Pour appeler une fonction à partir d'un morceau de code, il suffit de l'utiliser de la même manière que pour appeler des fonctions d'exécution, c'est-à-dire d'écrire le nom de la fonction avec les valeurs des paramètres entre parenthèses. Ainsi, la fonction ci-dessus serait appelée de la manière suivante : if keyboard_check_pressed(vk_enter) { val = scr_sqr(amount) ; } Ici, la variable val sera soit 0 (car la variable amount n'est pas un nombre réel), soit le total pour value * value.
