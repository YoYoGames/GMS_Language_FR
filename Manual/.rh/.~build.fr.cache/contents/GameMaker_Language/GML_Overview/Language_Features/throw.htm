lancez L'instruction d'exécution throw vous permet de générer vos propres exceptions d'exécution, en mettant fin au jeu et en affichant un message d'erreur, à l'aide de la syntaxe suivante : throw () ; L'expression utilisée peut être une valeur, une chaîne de caractères ou tout autre type de données. Elle génère alors une erreur d'exception qui est - par défaut - affichée à l'écran et le jeu se termine à la fermeture du message d'erreur. Par exemple, en appelant ceci : throw ("Hello World !") ; entraînera l'affichage de l'erreur d'exception non gérée suivante : Vous pouvez toutefois reprendre ce message d'erreur et utiliser votre propre code de gestion en appelant la la fonction exception_unhandled_handler(). Cette fonction d 'exécution vous permet de fournir une méthode personnalisée qui sera appelée à chaque fois qu'une exception non gérée se produit dans votre jeu.
