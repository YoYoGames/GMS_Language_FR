is_int32 Cette fonction renvoie si une variable donnée est un entier 323 bits ou non. Dans certains cas, vous souhaitez vérifier et voir quel type de données contient une variable dans GameMaker Studio 2 et c'est à ce moment que vous utiliserez cette fonction. Il retourne true ou false selon que la valeur est un int 32 ou non. Syntaxe:is_int32(n); Argument Description n L'argument à vérifier. Retour:Boolean Exemple:if !is_int32(val)     {     show_debug_message("Not a 32 bit integer!");     } Le code ci-dessus vérifie la variable "val" pour voir si elle contient un int32 et si ce n'est pas le cas, il affiche un message dans la console de débogage.
