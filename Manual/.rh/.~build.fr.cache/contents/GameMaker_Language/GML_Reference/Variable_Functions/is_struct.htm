is_struct Cette fonction peut être utilisée pour vérifier et voir si une variable contient une structure (elle retournera true ) ou non (auquel cas il retournera false ). Syntaxe:is_struct(n); Argument Description n La variable à vérifier. Retour:Boolean Exemple:if is_struct(a)     {     delete(a);     } Le code ci-dessus vérifie une variable pour voir s'il s'agit d'une structure et si la fonction retourne true, la structure est supprimée.
