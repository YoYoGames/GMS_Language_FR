variable_instance_existante Cette fonction vous permet de vérifier si une variable d'instance existe ou non. Vous fournissez la valeur unique de l'ID de l'instance (qui peut être trouvée dans les propriétés de l'instance dans l'éditeur de pièce, ou qui est retournée lorsque vous appelez la fonction instance_create_layer()) ainsi que le nom de la variable à vérifier sous forme de chaîne (voir l'exemple de code ci-dessous). La fonction retournera true si une variable avec le nom donné existe pour l'instance et false sinon. Syntaxe : variable_instance_exists(instance_id, name) ; Arguments Description identifiant de l'instance La valeur d'identification unique de l'instance à vérifier nom Le nom de la variable à vérifier (sous forme de chaîne) Les retours : Booléen Exemple : if !variable_instance_exists(id, "shields")) { boucliers = 0 ; } Le code ci-dessus va vérifier si la variable appelée "shields" existe dans l'instance qui exécute le code et si ce n'est pas le cas, elle est créée et initialisée à 0.
