is_bool Cette fonction retourne si une variable donnée est un booléen(vrai ou faux) ou non. Dans certains cas, vous souhaitez vérifier si une variable de GameMaker Studio 2 contient une valeur booléenne. et c'est dans ce cas que vous utiliserez cette fonction. Syntaxe : is_bool(n) ; Arguments Description n L'argument à vérifier. Les retours : Booléen Exemple : if is_bool(val) { global.Son = val ; } else { global.Sound = true ; } Le code ci-dessus vérifie la variable "val" pour voir si c'est un nombre réel et si c'est le cas, il l'utilise pour définir une variable globale.
