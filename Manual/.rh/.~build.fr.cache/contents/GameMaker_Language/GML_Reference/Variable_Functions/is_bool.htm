is_bool Cette fonction renvoie si une variable donnée est un booléen (true ior false ) ou non. Dans certains cas, vous souhaitez vérifier et voir si une variable dans GameMaker Studio 2 contient une valeur booléenne, et c'est à ce moment que vous utiliserez cette fonction. Syntaxe:is_bool(n); Argument Description n L'argument à vérifier. Retour:Boolean Exemple:if is_bool(val)     {     global.Sound = val;     } else     {     global.Sound = true;     } Le code ci-dessus vérifie la variable "val" pour voir si c'est un nombre réel et si c'est le cas, il l'utilise alors pour définir une variable globale.
