est_numérique Cette fonction retourne si une variable donnée est une valeur numérique (real, int32, int64 ou boolean) ou non. Dans certains cas, vous souhaitez vérifier si une variable de GameMaker Studio 2 contient une valeur numérique. c'est dans ce cas que vous utiliserez cette fonction. La fonction renverra true si l'entrée donnée est numérique, et false sinon. Syntaxe : est_numérique(n) ; Arguments Description n L'entrée à vérifier. Les retours : Booléen Exemple : if is_numeric(val) { current_val += val ; } Le code ci-dessus vérifie la variable "val" pour voir si c'est une valeur numérique et si c'est le cas, il l'ajoute à la variable donnée.
