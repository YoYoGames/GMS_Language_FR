tableau_suppression Avec cette fonction, vous pouvez supprimer une ou plusieurs valeurs d'un tableau à n'importe quelle position. La fonction vous demande de fournir une variable qui contient le tableau, l'index (position) dans le tableau à supprimer, ainsi que le nombre de valeurs à supprimer. Syntaxe : array_delete(variable, index, nombre) ; Arguments Description variable La variable qui contient le tableau. indice L'index (position) dans le tableau pour supprimer la ou les valeurs. numéro Le nombre de valeurs à supprimer. Les retours : N/A Exemple : array_delete(score_array, 0, 10) ; Le code ci-dessus supprimera les 10 premières valeurs du tableau donné en commençant par l'indice 0 (les indices 0 à 9 seront donc supprimés du tableau).
