array_insert Avec cette fonction, vous pouvez insérer une ou plusieurs valeurs dans un tableau à une position donnée. La fonction nécessite que vous fournissiez une variable qui contient le tableau, l'index (position) dans le tableau à insérer, ainsi qu'au moins une valeur à Insérer, bien que vous puissiez optionnellement fournir d'autres arguments et ils seront tous insérés dans le tableau dans un ordre consécutif à partir de l'index donné. Notez que si l'indice donné est en dehors de la longueur du tableau, les valeurs seront ajoutées à l'indice donné, et tout emplacement vide dans le tableau entre sa dernière valeur et les valeurs insérées sera mis à une valeur par défaut de 0. Syntaxe : array_insert(variable, index, valeur, [valeur], [valeur], [etc...]) ; Arguments Description variable La variable qui contient le tableau. indice L'index (position) dans le tableau pour insérer la ou les valeurs. valeur La valeur à insérer [valeur], [valeur], [etc...]. (Facultatif) Autres valeurs à insérer dans le tableau Les retours : N/A Exemple : array_insert(score_array, score_rank, current_score) ; Le code ci-dessus insère le score actuel dans le tableau des scores, à l'index défini par la variable score_rank.
