array_length_2d(Obsolète) ATTENTION ! Cette fonction est dépréciée car les tableaux ne sont plus limités à 1 ou 2 dimensions, et en tant que telle, cette fonction n'est fournie que pour le support des anciens projets. Tous les nouveaux projets doivent utiliser la méthode actuelle de création et d'accès aux tableaux multidimensionnels (voir ici). multidimensionnels (voir ici pour plus d'informations). Avec cette fonction, vous pouvez obtenir la longueur (nombre d'entrées) de la deuxième dimension d'un tableau. Vous fournissez le numéro d'entrée de la première dimension et la fonction renverra le nombre d'entrées de la deuxième dimension que possède le tableau (pour trouver la longueur de la première dimension, utilisez la fonction "Longueur de la deuxième dimension"). la longueur de la première dimension, utilisez la fonction tableau_hauteur_2D()). La fonction renvoie 0 si la variable donnée n'est pas un tableau ou 1 si la variable est un tableau 1D (car il y a toujours 1 ligne). Syntaxe : array_length_2d(array_index, n) ; Arguments Description tableau_index L'index du tableau à vérifier. n L'entrée du tableau dont on veut obtenir la longueur. Les retours : Real Exemple : for (var i = 0 ; i < array_height_2d(a) ; ++i ;) { for (var j = 0 ; j < array_length_2d(a, i) ; ++j ;) { a[i, j] = -1 ; } } Le code ci-dessus va parcourir en boucle un tableau 2D et donner la valeur -1 à chaque entrée.
