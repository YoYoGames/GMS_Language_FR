variable_struct_names_count Avec cette fonction, vous pouvez trouver le nombre total de variables définies pour une structure. Vous fournissez l'ID de structure à vérifier et la fonction retournera une valeur entière pour le nombre de variables rencontrées, ou (si aucune structure de l'ID donné n'existe) -1. Syntaxe:variable_struct_names_count(struct_id); Argument Description struct_id La valeur d'ID unique de la structure à vérifier. Retour:Integer Exemple:var _num = variable_struct_names_count(mystruct); show_debug_message("Struct Variables = " + string(_num)); Le code ci-dessus récupérera le nombre de variables dans la structure donnée et affichera un message de débogage dans la sortie de la console avec cette valeur.
