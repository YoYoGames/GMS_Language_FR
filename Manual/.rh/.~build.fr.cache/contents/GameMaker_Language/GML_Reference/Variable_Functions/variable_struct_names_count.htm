nombre_de_noms_de_structures_variables Avec cette fonction, vous pouvez trouver le nombre total de variables définies pour une structure. Vous fournissez l'ID de la structure à vérifier, et la fonction retournera une valeur entière pour le nombre de variables rencontrées, ou (si aucune structure de l'ID donné n'existe) -1. Syntaxe : variable_struct_names_count(struct_id) ; Arguments Description struct_id La valeur d'identification unique de la structure à vérifier. Les retours : Entier Exemple : var _num = variable_struct_names_count(mystruct) ; show_debug_message("Struct Variables = " + string(_num)) ; Le code ci-dessus récupère le nombre de variables dans la structure donnée et affiche un message de débogage dans la sortie console avec cette valeur.
