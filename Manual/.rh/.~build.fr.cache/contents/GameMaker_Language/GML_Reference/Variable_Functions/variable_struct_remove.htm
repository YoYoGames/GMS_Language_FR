variable_struct_remove Avec cette fonction, vous pouvez supprimer une variable d'une structure. Vous fournissez l'ID struct pour supprimer la variable et le nom de chaîne de la variable à supprimer. Syntaxe:variable_struct_set(struct, name); Argument Description struct La référence struct pour supprimer la variable de name Le nom de la variable à supprimer (sous forme de chaîne) Retour:N/A Exemple:if variable_struct_exists(mystruct, "shields")    {    variable_struct_remove(mystruct, "shields");    } Le code ci-dessus vérifiera si la variable donnée existe dans la structure donnée et si c'est le cas, elle est supprimée.
