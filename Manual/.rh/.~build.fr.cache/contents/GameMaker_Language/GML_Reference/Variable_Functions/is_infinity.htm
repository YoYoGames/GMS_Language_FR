is_infinity Cette fonction renvoie si une variable donnée est infinity (un nombre infini) ou non, retournant true si c'est le cas, et false si ce n'est pas le cas. Syntaxe:is_infinity(n); Argument Description n L'argument à vérifier. Retour:Boolean Exemple:if is_infinity(global.value)     {     show_debug_message("Value is infinite!");     } Le code ci-dessus vérifie la variable globale "valeur" pour voir si elle est infinie ou non et affiche un message de débogage si c'est le cas.
