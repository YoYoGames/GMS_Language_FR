est_infini Cette fonction indique si une variable donnée est infinie (un nombre infini) ou non, en retournant true si c'est le cas, et false si ce n'est pas le cas. Syntaxe : est_infini(n) ; Arguments Description n L'argument à vérifier. Les retours : Booléen Exemple : if is_infinity(global.value) { show_debug_message("La valeur est infinie !") ; } Le code ci-dessus vérifie la variable globale "value" pour voir si elle est infinie ou non et affiche un message de débogage si c'est le cas.
