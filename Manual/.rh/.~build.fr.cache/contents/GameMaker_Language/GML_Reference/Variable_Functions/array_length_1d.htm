array_length_1d(Obsolète) ATTENTION ! Cette fonction est dépréciée car les tableaux ne sont plus limités à 1 ou 2 dimensions, et en tant que telle, cette fonction n'est fournie que pour le support des anciens projets. Tous les nouveaux projets doivent utiliser la méthode actuelle de création et d'accès aux tableaux multidimensionnels (voir ici). multidimensionnels (voir ici pour plus d'informations). Avec cette fonction, vous pouvez obtenir la longueur (nombre d'entrées) d'un tableau 1D. Pour les tableaux 2D, vous devriez utiliser les fonctions tableau_hauteur_2d et tableau_longueur_2d pour les tableaux 2D. IMPORTANT : : Si le tableau comporte plus de 32 000 entrées, cette fonction renverra une valeur erronée et ne doit pas être utilisée. Syntaxe : array_length_1d(array_index) ; Arguments Description tableau_index L'index du tableau à vérifier. Les retours : Real Exemple : for (var i = array_length_1d(a) - 1 ; i > -1 ; i-- ;) { a[i] = -1 ; } Le code ci-dessus va parcourir un tableau en boucle et donner la valeur -1 à chaque entrée.
