ptr Cette fonction tentera de convertir une valeur donnée en un type de données de type pointeur, où la valeur doit être soit un real, un string, un int64, un int32, ou un ptr. Toute autre valeur entraînera un crash du jeu avec un message d'erreur. Syntaxe : ptr(n) ; Arguments Description n La valeur à convertir. Les retours : Pointeur Exemple : if !is_ptr(val) { val = ptr(application_surface) ; } Le code ci-dessus vérifie la variable "val" pour voir si elle contient un pointeur et si ce n'est pas le cas, un pointeur lui est attribué.
