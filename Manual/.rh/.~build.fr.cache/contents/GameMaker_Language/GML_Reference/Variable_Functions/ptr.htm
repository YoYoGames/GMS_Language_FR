ptr Cette fonction tentera de convertir une valeur donnée en un type de données pointeur, où la valeur doit être soit un real, une string, une int64, une int32, ou un ptr. Toute autre chose provoquera le crash du jeu avec un message d'erreur. Syntaxe:ptr(n); Argument Description n La valeur à convertir. Retour:Pointer Exemple:if !is_ptr(val)    {    val = ptr(application_surface);    } Le code ci-dessus vérifie la variable "val" pour voir si elle contient un pointeur et si ce n'est pas le cas, un lui est assigné.
