is_matrix Cette fonction renvoie si une variable donnée fait référence à une matrice ou non. Dans certains cas, vous souhaitez vérifier et voir quel type de données contient une variable dans GameMaker Studio 2 et c'est à ce moment que vous utiliserez cette fonction. Il retourne true ou false selon que la valeur est une matrice ou non. Syntaxe:is_matrix(n); Argument Description n L'argument à vérifier. Retour:Boolean Exemple:if !is_matrix(val)     {     show_debug_message("Not a valid matrix!");     } Le code ci-dessus vérifie la variable "val" pour voir si elle contient une matrice et si ce n'est pas le cas, il affiche un message dans la console de débogage.
