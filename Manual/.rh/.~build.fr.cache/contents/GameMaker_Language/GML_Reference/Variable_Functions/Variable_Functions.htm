Fonctions variables Les fonctions de cette page sont conçues pour gérer les différentes variables et les différents types de variables lorsque vous utilisez le langage GameMaker dans vos jeux. Pour plus d'informations sur les variables, veuillez consulter la page sur les variables et la portée des variables. Les fonctions variables suivantes sont disponibles : variable_instance_existante variable_instance_get_names nombre_de_noms_d_instances_variables variable_instance_get variable_instance_set variable_globale_existe variable_global_get variable_global_set variable_struct_exists variable_struct_get ensemble_de_structures_variables suppression_de_la_structure_variable variable_struct_get_names nombre_de_noms_de_structures_variables instanceof Il existe également quelques fonctions associées à l'utilisation des tableaux. Elles sont conçues pour vous donner de la flexibilité dans votre code, et vous permettront de créer des tableaux plus fonctionnels et dynamiques lors de la création de vos jeux. Ces fonctions sont les suivantes : tableau_créer tableau_copie tableau_équivalents tableau_get tableau_set array_push tableau_pop array_insert tableau_suppression array_sort longueur du tableau tableau_resize array_length_1d(obsolète) array_length_2d(obsolète) array_height_2d(obsolète) Ces fonctions suivantes sont toutes liées à l'utilisation des variables de méthode: méthode méthode_get_self méthode_get_index Enfin, vous disposez également des fonctions suivantes pour déterminer le type des variables (pour plus d'informations sur les types de données, veuillez consulter ici, et voir ici pour les tables de types) : est_chaîne est_réel est_numérique is_bool is_array is_struct est_méthode est_ptr est_int32 est_int64 is_vec3 is_vec4 is_matrix est_indéfini is_nan est_infini typeof bool ptr int64 Retour : RéférenceGML Suivant : Gestion des actifs Copyright YoYo Games Ltd. 2021 Tous droits réservés
