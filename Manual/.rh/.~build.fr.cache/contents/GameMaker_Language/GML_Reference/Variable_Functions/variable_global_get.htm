variable_global_get Avec cette fonction, vous pouvez obtenir la valeur d'une variable globale nommée donnée. Vous fournissez le nom de la variable globale dont vous voulez obtenir la valeur sous la forme d'une chaîne de caractères (voir l'exemple de code ci-dessous). détenue par la variable globale ou undefined si la variable n'existe pas. IMPORTANT ! Si la variable globale que vous obtenez n'existe pas, la fonction renverra le mot-clé undefined et vous pourrez obtenir des erreurs qui empêcheront le jeu de fonctionner. utilisez d'abord la fonction variable_globale_existante d'abord. Syntaxe : variable_global_get(name) ; Arguments Description nom Le nom de la variable globale à récupérer (sous forme de chaîne) Les retours : Valeur (tout type de données) ou indéfinie (si la variable nommée n'existe pas) Exemple : if variable_global_exists("enemy_num") { show_debug_message("enemy_num = " + string(variable_global_get("enemy_num")) ;) } Le code ci-dessus va vérifier si une variable globale existe et si c'est le cas, elle sera affichée dans la console.
