bool Cette fonction tentera de convertir une valeur donnée en un type de données booléen, où la valeur sera renvoyée sous la forme true s'il est supérieur à 0,5 et false autrement. Syntaxe:bool(n); Argument Description n La valeur à convertir. Retour:Boolean Exemple:if !is_bool(val)    {    val = bool(val);    } Le code ci-dessus vérifie la variable "val" pour voir si c'est un booléen et si ce n'est pas le cas, il est converti en un.
