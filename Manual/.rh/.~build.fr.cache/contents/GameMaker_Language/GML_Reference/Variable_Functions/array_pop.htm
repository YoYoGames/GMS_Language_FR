array_pop Avec cette fonction, vous pouvez récupérer une valeur à la fin d'un tableau sans avoir à connaître la longueur du tableau. Vous fournissez la variable qui contient le tableau et la fonction renverra le contenu de la dernière entrée du tableau. Si le tableau est vide, la fonction retournera undefined. Syntaxe:array_pop(variable); Argument Description variable La variable qui contient le tableau. Retour:Value (any data type) or undefined Exemple:var _lastscore = array_pop(score_array); draw_text(32, 32, "Last Score = " + string(_lastscore)); Le code ci-dessus affichera la dernière valeur du tableau donné, puis la dessinera avec du texte à l'écran.
