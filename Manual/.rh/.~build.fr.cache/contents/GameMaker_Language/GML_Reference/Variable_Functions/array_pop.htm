tableau_pop Avec cette fonction, vous pouvez récupérer une valeur à la fin d'un tableau sans avoir à connaître la longueur du tableau. Vous fournissez la variable qui contient le tableau, et la fonction renvoie le contenu de la dernière entrée du tableau. Si le tableau est vide, la fonction renvoie undefined. Syntaxe : array_pop(variable) ; Arguments Description variable La variable qui contient le tableau. Les retours : Valeur (tout type de données) ou indéfinie Exemple : var _lastscore = array_pop(score_array) ; draw_text(32, 32, "Last Score = " + string(_lastscore)) ; Le code ci-dessus va extraire la dernière valeur du tableau donné et l'afficher à l'écran avec du texte.
