array_length Avec cette fonction, vous pouvez obtenir la longueur (nombre d'entrées) d'une dimension de tableau. Vous fournissez la valeur d'index du tableau et la fonction renverra une valeur entière représentant le nombre d'entrées que le tableau contient. Cette fonction peut également être utilisée pour les tableaux multidimensionnels, à condition de spécifier la dimension dont vous souhaitez obtenir la longueur lorsque vous fournissez l'index du tableau, en suivant ce modèle:// Returns the first dimension of the array val = array_length(my_array); // Returns the second dimension of the array val = array_length(my_array[0]); // Returns the third dimension of the array val = array_length(my_array[0][0]); // etc... Syntaxe:array_length(array_index); Argument Description array_index L'index du tableau à vérifier. Retour:Integer Exemple:for (var i = 0; i < array_length(a); ++i;)     {     a[i] = -1;     } Le code ci-dessus va parcourir un tableau et définir chaque entrée sur -1.
