longueur du tableau Cette fonction permet d'obtenir la longueur (nombre d'entrées) d'une dimension de tableau. Vous fournissez la valeur de l'index du tableau et la fonction renvoie une valeur entière représentant le nombre d'entrées que le tableau contient. Cette fonction peut également être utilisée pour les tableaux multidimensionnels, à condition de spécifier la dimension dont vous souhaitez obtenir la longueur lorsque vous fournissez l'indice du tableau, en suivant ce modèle : // Retourne la première dimension du tableau val = array_length(my_array) ; // Retourne la deuxième dimension du tableau val = array_length(my_array[0]) ; // Retourne la troisième dimension du tableau val = array_length(my_array[0][0]) ; // etc... Syntaxe : longueur_du_tableau(indice_du_tableau) ; Arguments Description tableau_index L'index du tableau à vérifier. Les retours : Entier Exemple : for (var i = 0 ; i < array_length(a) ; ++i ;) { a[i] = -1 ; } Le code ci-dessus va parcourir un tableau en boucle et donner la valeur -1 à chaque entrée.
