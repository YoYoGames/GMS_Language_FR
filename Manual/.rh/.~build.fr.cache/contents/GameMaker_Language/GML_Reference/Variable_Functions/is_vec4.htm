is_vec4 Dans certains cas, vous souhaitez vérifier et voir quel type de données contient une variable dans GameMaker Studio 2 et c'est à ce moment que vous utiliserez cette fonction. Il retourne true ou false selon que la valeur est un vecteur de 4 valeurs ou non. Syntaxe:is_vec4(n); Argument Description n L'argument à vérifier. Retour:Boolean Exemple:if !is_vec4(val)     {     show_debug_message("Not a 4 value vector!");     } Le code ci-dessus vérifie la variable "val" pour voir si elle contient un vec4 et si ce n'est pas le cas, il affiche un message dans la console de débogage.
