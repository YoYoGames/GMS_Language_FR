tableau_copie Avec cette fonction, vous pouvez copier tout ou partie d'un tableau dans un autre tableau à n'importe quelle position. Vous devez fournir les tableaux source et destination (les deux doivent avoir été créés précédemment), ainsi qu'une position dans le tableau source à copier et une position dans le tableau destination à copier. et une position dans le tableau de destination vers laquelle copier. Enfin, vous devez spécifier la longueur du tableau (ou la longueur de la partie que vous voulez) à copier. Si les données copiées dépassent la longueur du tableau de destination, le tableau sera étendu pour accepter les données. le tableau sera étendu pour accepter les données. Cette fonction peut également être utilisée pour des tableaux multidimensionnels, à condition de spécifier quelle dimension vous souhaitez copier lorsque vous fournissez l'index du tableau, en suivant ce modèle : // Copie de la première dimension d'un tableau // vers la deuxième dimension d'un tableau array_copy(item_array, 0, inventory_array[0], 0, len) ; // Copie de la troisième dimension d'un tableau // vers la première dimension d'un tableau array_copy(item_array[0][0], 0, inventory_array, 0, len) ; // etc... Syntaxe : array_copy(dest, dest_index, src, src_index, length) ; Arguments Description dest L'ID du tableau à copier. dest_index L'index dans le tableau à copier. src L'ID du tableau à copier. src_index L'index du tableau à partir duquel la copie doit commencer. longueur La longueur (nombre d'indices du tableau) à copier. Les retours : N/A Exemple : if !array_equals(inventory_array, item_array) { var len = array_length(inventory_array) ; array_copy(item_array, 0, inventory_array, 0, len) ; } Le code ci-dessus va vérifier deux tableaux pour voir s'ils contiennent des valeurs équivalentes, et si ce n'est pas le cas, le code va copier tout le contenu du tableau "inventory_array" vers le tableau "item_array".
