tableau_set Avec cette fonction, vous pouvez définir la valeur d'un index dans un tableau à une valeur. La fonction exige que vous fournissiez une variable qui contient le tableau ainsi que l'indice à définir et la valeur à définir. Cette fonction peut aussi être utilisée pour les tableaux multidimensionnels multidimensionnels, à condition de spécifier quelle dimension vous souhaitez définir lorsque vous fournissez l'indice du tableau, en suivant ce modèle : // Tableau 1D array_set(my_array, 0, 100) ; // Tableau 2D array_set(my_array[0], 0, 100) ; // Tableau 3D array_set(my_array[0][0], 0, 100) ; // etc... Syntaxe : array_set(variable, index, valeur) ; Arguments Description variable La variable qui contient le tableau. indice L'index du tableau pour lequel il faut définir la valeur. valeur La valeur à définir. Les retours : N/A Exemple : for (var i = 0 ; i < 10 ; ++i ;) { array_set(score_array, i, i*100)) ; } Le code ci-dessus va définir les 10 premiers éléments du tableau donné à une valeur.
