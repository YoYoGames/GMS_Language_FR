tableau_équivalents Avec cette fonction, vous pouvez vérifier si deux tableaux sont égaux (équivalents ou identiques). Vous donnez les deux tableaux à vérifier, et la fonction renvoie vrai s'ils sont égaux (équivalents ou identiques) ou faux s'ils ne le sont pas. ils ne le sont pas. Notez que ce n' est pas la même chose que de vérifier si deux tableaux sont identiques en utilisant ==, qui ne vérifiera pas si les deux tableaux contiennent des valeurs équivalentes, mais seulement si les tableaux font référence au même tableau initial. même tableau initial. Par exemple : var a = [1,2,3,4] ; var b = [1,2,3,4] ; if (a == b) { show_debug_message( "This will never fire, as a and b do not reference the SAME array" ) ; } if (array_equals(a, b)) { show_debug_message("Ceci se produira maintenant" ) ; } Syntaxe : array_equals(var1, var2) ; Arguments Description var1 L'index du premier tableau. var2 L'index du deuxième tableau. Les retours : Booléen Exemple : if !array_equals(inventory_array, item_array) { var len = array_length(inventory_array) ; array_copy(item_array, 0, inventory_array, 0, len) ; } Le code ci-dessus va vérifier deux tableaux pour voir s'ils contiennent des valeurs équivalentes, et si ce n'est pas le cas, le code va copier tout le contenu du tableau "inventory_array" vers le tableau "item_array".
