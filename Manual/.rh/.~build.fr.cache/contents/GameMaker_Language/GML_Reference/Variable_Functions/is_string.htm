is_string Cette fonction renvoie si une variable donnée est un numéro de chaîne ou non. Dans certains cas, vous souhaitez vérifier et voir si une variable dans GameMaker Studio 2 contient une chaîne et non un réel et c'est à ce moment que vous utiliserez cette fonction. Il ne renvoie pas la chaîne mais plutôt true ou false, donc une valeur de, par exemple, "poisson" pour n renverra true, mais une valeur de 200 pour n renverra false. Syntaxe:is_string(n); Argument Description n L'argument à vérifier. Retour:Boolean Exemple:if is_string(val)     {     name = "Player: " + val;     } Le code ci-dessus vérifie la variable "val" pour voir si elle contient une chaîne et si c'est le cas, il l'ajoute dans une autre chaîne qui est ensuite affectée à la variable "nom".
