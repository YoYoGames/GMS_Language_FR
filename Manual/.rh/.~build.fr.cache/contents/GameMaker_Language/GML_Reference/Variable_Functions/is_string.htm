est_chaîne Cette fonction renvoie si une variable donnée est une chaîne de caractères ou non. Dans certains cas, vous souhaitez vérifier si une variable de GameMaker Studio 2 contient une chaîne de caractères et non un réel. cette fonction. Elle ne renvoie pas la chaîne de caractères mais plutôt true ou false, donc une valeur de, par exemple, "fish" pour n renverra true, mais une valeur de 200 pour n renverra faux. Syntaxe : is_string(n) ; Arguments Description n L'argument à vérifier. Les retours : Booléen Exemple : if is_string(val) { name = "Player : " + val ; } Le code ci-dessus vérifie la variable "val" pour voir si elle contient une chaîne de caractères et si c'est le cas, il l'ajoute à une autre chaîne de caractères qui est ensuite assignée à la variable "name".
