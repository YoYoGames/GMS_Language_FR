is_array Cette fonction peut être utilisée pour vérifier et voir si une variable contient un tableau (elle retournera true ) ou non (auquel cas il retournera false ). Syntaxe:is_array(n); Argument Description n La variable à vérifier. Retour:Boolean Exemple:if is_array(a)     {     a = -1;     } Le code ci-dessus vérifie une variable pour voir s'il s'agit d'un tableau et si la fonction retourne true, le tableau est supprimé en définissant la variable sur -1.
