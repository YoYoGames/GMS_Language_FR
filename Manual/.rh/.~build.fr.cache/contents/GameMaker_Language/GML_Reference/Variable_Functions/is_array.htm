is_array Cette fonction peut être utilisée pour vérifier et voir si une variable contient un tableau (elle renverra true) ou non (dans ce cas, elle renverra false). Syntaxe : is_array(n) ; Arguments Description n La variable à vérifier. Les retours : Booléen Exemple : if is_array(a) { a = -1 ; } Le code ci-dessus vérifie une variable pour voir s'il s'agit d'un tableau, et si la fonction renvoie vrai, le tableau est supprimé en mettant la variable à -1.
