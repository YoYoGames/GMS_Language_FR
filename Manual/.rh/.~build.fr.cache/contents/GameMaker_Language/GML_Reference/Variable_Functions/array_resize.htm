tableau_resize Avec cette fonction, vous pouvez redimensionner un tableau existant pour lui donner une nouvelle dimension. Vous fournissez le tableau à redimensionner ainsi que le nouveau nombre d'indices pour le tableau, et la fonction redimensionnera ce tableau. Notez que cette fonction est conçue pour redimensionner un tableau à une longueur plus petite, car vous pouvez le redimensionner en fixant simplement un nouvel index dans le tableau. Cela dit, si vous l'utilisez pour redimensionner un tableau, tous les nouveaux indices seront fixés à la valeur par défaut de 0. Cette fonction peut également être utilisée pour des tableaux multidimensionnels, à condition de préciser quelle dimension vous souhaitez redimensionner lorsque vous fournissez l'indice du tableau, en suivant ce modèle : // Redimensionne la première dimension du tableau array_resize(my_array, 10) ; // Redimensionne la deuxième dimension du tableau array_resize(my_array[0], 10) ; // Redimensionne la troisième dimension du tableau array_resize(my_array[0][0], 10) ; // etc... Syntaxe : array_resize(array_index, new_size) ; Arguments Description tableau_index L'index du tableau à redimensionner. nouvelle_taille La nouvelle taille du tableau (un nombre entier, commençant à 0). Les retours : N/A Exemple : si array_length(a) > 10 { array_resize(a, 10) ; } Le code ci-dessus vérifie la longueur d'un tableau et s'il a plus de 10 indices, il est redimensionné.
