int64 Cette fonction tente de convertir une valeur donnée en un entier 64 bits, la valeur devant être un réel, une chaîne de caractères, un int64, un int32 ou un ptr. Toute autre valeur Toute autre valeur entraînera un crash du jeu avec un message d'erreur. Vous pouvez vérifier si une variable contient un int64 en utilisant la fonction is_int64(). Syntaxe : int64(val) ; Arguments Description val La valeur à convertir. Les retours : Entier 64 bits Exemple : steam_handle= int64(global.fileReadString) ; Le code ci-dessus convertit la valeur contenue dans la variable globale en un entier de 64 bits.
