variable_globale_existe Avec cette fonction, vous pouvez vérifier si une variable de portée globale existe ou non. Vous fournissez le nom de la variable globale à vérifier sous forme de chaîne de caractères (voir l'exemple de code ci-dessous) et la fonction retournera true si une variable globale avec le nom donné existe ou faux sinon. Syntaxe : variable_global_exists(name) ; Arguments Description nom Le nom de la variable globale à vérifier (sous forme de chaîne). Les retours : Booléen Exemple : if !variable_global_exists("enemy_num")) { global.enemy_number = instance_number(obj_Enemey_Parent) ; } Le code ci-dessus vérifiera si la variable globale appelée "enemy_num" existe et si ce n'est pas le cas, elle sera créée et une valeur lui sera attribuée.
