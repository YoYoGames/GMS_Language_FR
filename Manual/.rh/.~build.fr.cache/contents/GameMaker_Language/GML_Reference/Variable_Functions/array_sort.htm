array_sort Avec cette fonction, vous pouvez trier un tableau par ordre croissant ou décroissant ou en utilisant une fonction personnalisée pour définir l'ordre de tri. La fonction vous oblige à fournir une variable contenant le tableau, puis soit: Les constantes true ou false pour indiquer ascendant (true ) ou décroissant (false ) ordre de tri Une fonction pour définir l'ordre de tri des tableaux Si vous utilisez une fonction pour créer l'ordre de tri, elle doit prendre 2 arguments (ceux-ci correspondront à deux éléments du tableau), et la fonction doit renvoyer 0 pour l'égalité, inférieure à 0 ou supérieure à 0. La valeur de retour sera puis être utilisé pour déterminer l'ordre de tri (voir l'exemple ci-dessous). Notez que si le tableau contient un ensemble de chaînes, les chaînes seront triées par ordre alphabétique en fonction de l'alphabet anglais lors de l'utilisation du type de tri croissant / décroissant par défaut. Tous les autres types de données seront triés en fonction de la valeur numérique, dont les valeurs exactes dépendront du type de données lui-même (par exemple, un tableau de tampons serait trié en fonction de la valeur numérique convertie de leur adresse mémoire). Syntaxe:array_sort(variable, sorttype_or_function); Argument Description variable La variable qui contient le tableau. sorttype_or_function Le type de tri ( truepour monter ou false pour décroissant) ou une référence de fonction à utiliser pour le tri.
