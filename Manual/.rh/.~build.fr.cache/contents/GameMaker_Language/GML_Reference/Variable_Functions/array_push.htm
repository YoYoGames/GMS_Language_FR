array_push Avec cette fonction, vous pouvez pousser une valeur (ou des valeurs) à la fin d'un tableau sans avoir à connaître la longueur du tableau. La fonction vous oblige à fournir une variable qui contient le tableau ainsi qu'au moins une valeur à pousser, bien que vous puissiez éventuellement fournir d'autres arguments et ils seront tous poussés sur le tableau dans un ordre consécutif. Syntaxe:array_push(variable, value, [value], [value], [etc...]); Argument Description variable La variable qui contient le tableau. value La valeur à pousser à la fin du tableau [value], [value], [etc...] OPTIONNEL! Autres valeurs à pousser sur le tableau Retour:N/A Exemple:array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr); Le code ci-dessus poussera quatre valeurs à la fin du tableau donné.
