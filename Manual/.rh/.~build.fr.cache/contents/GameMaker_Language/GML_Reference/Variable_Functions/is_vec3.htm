is_vec3 Dans certains cas, vous souhaitez vérifier le type de données d'une variable dans GameMaker Studio 2 et c'est dans ce cas que vous utilisez cette fonction. Elle renvoie vrai ou faux selon selon que la valeur est un vecteur à 3 valeurs ou non. Syntaxe : est_vec3(n) ; Arguments Description n L'argument à vérifier. Les retours : Booléen Exemple : if !is_vec3(val) { show_debug_message("Not a 3 value vector !") ; } Le code ci-dessus vérifie la variable "val" pour voir si elle contient un vec3 et si ce n'est pas le cas, il affiche un message dans la console de débogage.
