tableau_créer Avec cette fonction, vous pouvez créer un tableau 1D d'une taille donnée. Vous indiquez à la fonction la longueur du tableau à créer, et elle renvoie le "handle" du tableau que vous pouvez ensuite affecter à une variable. Les tableaux créés de cette manière auront chaque entrée sera initialisée à 0, sauf si vous spécifiez une valeur d'initialisation (facultative). Si vous fournissez une valeur supplémentaire pour l'initialisation du tableau, tous les indices du nouveau tableau seront initialisés à cette valeur au lieu de 0. fonction aura une plus grande surcharge de performance dans ce cas. Syntaxe : array_create(size, [value]) ; Arguments Description taille La taille du tableau à créer. valeur La valeur à utiliser pour initialiser tous les indices du tableau (facultatif). Les retours : Référence au tableau Exemple : tableau_instance = tableau_créé(100, personne) ; Le code ci-dessus créera un nouveau tableau de 100 entrées, chacune étant définie par le mot clé noone, puis l'affectera à la variable "instance_array".
