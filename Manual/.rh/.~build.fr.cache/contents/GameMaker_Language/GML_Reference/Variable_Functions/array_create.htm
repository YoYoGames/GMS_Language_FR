array_create Avec cette fonction, vous pouvez créer un tableau 1D d'une taille donnée. Vous indiquez à la fonction la longueur du tableau à créer, et elle renverra le "handle" du tableau que vous pourrez ensuite affecter à une variable. Les tableaux créés de cette manière auront chaque entrée initialisée à 0, sauf si vous spécifiez une valeur d'initialisation (facultative). Si vous fournissez la valeur supplémentaire pour initialiser le tableau, tous les indices du nouveau tableau seront définis sur cette valeur au lieu de 0, mais notez que la fonction aura une plus grande surcharge de performances dans ce cas. Syntaxe:array_create(size, [value]); Argument Description size La taille du tableau à créer. value La valeur à utiliser pour initialiser tous les indices de tableau (facultatif). Retour:Array Reference Exemple:instance_array = array_create(100, noone); Le code ci-dessus créera un nouveau tableau de 100 entrées, chacune définie sur le mot-clé noone, puis affectez-le à la variable "instance_array".
