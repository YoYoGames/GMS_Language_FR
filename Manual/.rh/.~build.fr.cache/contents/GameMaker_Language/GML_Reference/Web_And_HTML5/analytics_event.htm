analytics_event Cette fonction enverra le texte spécifié au fournisseur d'analyse que vous avez configuré via les options de jeu HTML5. Cette fonction peut être utilisée pour créer une analyse personnalisée pour suivre quelque chose en dehors de la portée du fournisseur utilisé. Syntaxe:analytics_event(string); Argument Description string Une chaîne à envoyer au fournisseur Retour:N/A Exemple:if hs_new     {     analytics_event("New hiscore of " + string (score));     } Le code ci-dessus vérifiera une variable pour voir si elle est true et si c'est le cas, un événement d'analyse spécial sera déclenché avec la chaîne spécifiée.
