clickable_delete Cette fonction doit être utilisée pour supprimer une icône cliquable créée précédemment avec clickable_add() depuis la fenêtre de jeu. Syntaxe:clickable_delete(index); Argument Description index Index de l'icône cliquable à supprimer. Retour:N/A Exemple:if clickable_exists(global.Help_Icon)     {     clickable_delete(global.Help_Icon);     } Le code ci-dessus supprime l'icône cliquable indexée dans la variable globale "Help_Icon" de la fenêtre du jeu, si elle existe.
