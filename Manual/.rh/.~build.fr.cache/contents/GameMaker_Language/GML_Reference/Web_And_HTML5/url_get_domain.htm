url_get_domain Vous pouvez l'utiliser pour obtenir le domaine où votre jeu HTML5 est joué. Syntaxe : url_get_domain() ; Les retours : chaîne de caractères Exemple : dom_name=url_get_domain() ; Cela permettra de stocker le domaine de votre jeu dans la variable "dom_name".
