url_get_domain Vous pouvez l'utiliser pour obtenir le domaine dans lequel votre jeu HTML5 est joué. Syntaxe:url_get_domain(); Retour:string Exemple:dom_name=url_get_domain(); Cela stockera le domaine de votre jeu dans la variable "nom_domaine".
