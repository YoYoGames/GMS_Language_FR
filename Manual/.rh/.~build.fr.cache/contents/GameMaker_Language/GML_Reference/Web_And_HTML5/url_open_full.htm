url_open_full Cela ouvrira l'URL spécifiée sur le navigateur de l'appareil cible choisi ou, si vous utilisez le module HTML5, dans le navigateur actuellement ouvert. Le paramètre " target " que vous spécifiez est le même que la valeur standard JavaScript " name " lorsque vous utilisez la méthode open(). lorsque vous utilisez la méthode open() (sachez que tout autre paramètre que " _self" peut entraîner un blocage du navigateur ou demander à l'utilisateur s'il souhaite l'autoriser) et le paramètre "options" est identique au paramètre JavaScript "specs". pour contrôler les propriétés que la nouvelle fenêtre/onglet doit afficher (tous les navigateurs ne prennent pas forcément en charge toutes les fonctionnalités). Les cibles valides sont : Cible Description _blank Ouvre le document lié dans une nouvelle fenêtre ou un nouvel onglet (cela ne fonctionnera pas si les fenêtres pop-up sont bloquées par l'utilisateur, auquel cas vous pouvez utiliser les fonctions cliquable_* à la place). _self Ouvre le document lié dans le même cadre que celui dans lequel il a été cliqué (c'est la valeur par défaut). _parent Ouvre le document lié dans le cadre parent. _haut Ouvre le document lié dans le corps complet de la fenêtre. Les options valides sont : Paramètres Description height=[px] La hauteur de la fenêtre, la valeur minimale étant de 100. width=[px] La largeur de la fenêtre, la valeur minimale étant de 100. left=[px] La position gauche de la fenêtre. top=[px] La position supérieure de la fenêtre (IE uniquement). location=[booléen] Affichage ou non du champ d'adresse (1 par défaut). menubar=[booléen]". Affichage ou non de la barre de menu (1 par défaut). resizable=[boolean]". Si la fenêtre est redimensionnable ou non (la valeur par défaut est 1). scrollbars=[boolean]". Affichage ou non des barres de défilement (la valeur par défaut est 1). status=[booléen] Ajout ou non d'une barre d'état (la valeur par défaut est 1). barre de titre = [booléen] ". Affichage ou non de la barre de titre. Cette option est ignorée, sauf si l'application appelante est une application HTML ou une boîte de dialogue sécurisée (la valeur par défaut est 1) ; toolbar=[boolean]". Affichage ou non de la barre d'outils du navigateur (par défaut, oui). Syntaxe : url_open_full(url, cible, options) ; Arguments Description url L'URL (adresse du site web) vers lequel le lien doit être établi. cible Il s'agit de la zone cible dans laquelle il faut ouvrir l'URL (voir la description). options Options standard du navigateur (voir la description). Les retours : N/A Exemple : url_open_full("http://yoyogames.com", "_blank", "resizable=0, height=200, scrollbars=0") ; Cela ouvrirait la page d'accueil de YoYo Games dans une nouvelle fenêtre qui ne peut pas être redimensionnée, qui a une hauteur de 200 pixels et qui ne comporte pas de barres de défilement.
