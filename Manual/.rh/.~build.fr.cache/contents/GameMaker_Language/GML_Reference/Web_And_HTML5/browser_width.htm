browser_width Cette variable contient la largeur (en pixels) du navigateur dans lequel le jeu est exécuté. Si aucun navigateur n'est présent, la taille de la fenêtre est renvoyée. l'image suivante illustre la relation entre cette variable et le reste des fonctions de positionnement disponibles via les fonctions de fenêtre. Syntaxe:browser_width Retour:Real Exemple:if (browser_height > window_get_height()) || (browser_width > window_get_width())     {     var xx, yy;     if (browser_width > window_get_width())         {         xx = (browser_width - window_get_width()) / 2;         }     else         {         xx = 0;         }     if (browser_height > window_get_height())         {         yy = (browser_height - window_get_height()) / 2;         }     else         {         yy = 0;         }     window_set_position(xx, yy);     } Le code ci-dessus vérifie la largeur et la hauteur du navigateur pour voir si l'une ou l'autre est plus grande que la largeur et la hauteur actuelles de la fenêtre. Si l'un d'entre eux le sont, il calcule alors une nouvelle position pour la fenêtre de jeu dans le navigateur.
