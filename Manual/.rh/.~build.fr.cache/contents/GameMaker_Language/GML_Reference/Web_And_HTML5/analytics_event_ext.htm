analytics_event_ext Cette fonction enverra le texte spécifié au fournisseur de données analytiques que vous avez configuré dans les options de jeu HTML5. Cette fonction peut être utilisée pour créer une analyse personnalisée afin de suivre quelque chose en dehors de la portée du fournisseur utilisé. Cette fonction peut être utilisée pour créer une analyse personnalisée afin de suivre quelque chose en dehors de la portée du fournisseur utilisé. Elle acceptera également des paires paramètre/valeur personnalisées, où le paramètre est une chaîne et la valeur un nombre réel. Pour Google Analytics, vous ne pouvez ajouter qu'une seule paire supplémentaire, tandis que Flurry accepte jusqu'à 7. Syntaxe : analytics_event_ext(string, string_param, value) ; Arguments Description chaîne de caractères Une chaîne de caractères à envoyer au fournisseur. string_param [0 -9] Le paramètre à envoyer (une chaîne de caractères). valeur [0 - 9] La valeur du paramètre (un nombre réel). Les retours : N/A Exemple : var num, time ; ini_open("Options.ini") ; num = ini_read_real("Data", "Plays", 0) ; num += 1 ; time = current_time ; analytics_event_ext(GAME_NAME, "Plays", num, "Time", time) ; ini_write_real("Data", "Plays", num) ; ini_close() ; Le code ci-dessus récupérera les informations de lecture d'un fichier ini ainsi que l'heure actuelle et enverra ensuite ces détails au fournisseur d'analyses.
