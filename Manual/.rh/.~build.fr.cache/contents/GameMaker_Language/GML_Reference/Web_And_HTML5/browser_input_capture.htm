browser_input_capture Avec cette fonction, vous pouvez définir si la fenêtre du navigateur doit capturer toutes les entrées (définissez-la sur false ) ou si le jeu doit capturer l'entrée (réglez-le sur true ). Notez que cette fonction est à utiliser uniquement avec le module HTML5. Syntaxe:browser_input_capture(enable) Retour:N/A Exemple:browser_input_capture(true); Le code ci-dessus configure le jeu pour capturer toutes les entrées du navigateur sur la plate-forme cible HTML5.
