capture d'entrée du navigateur Cette fonction vous permet de définir si la fenêtre du navigateur doit capturer toutes les entrées (en lui attribuant la valeur false) ou si le jeu doit capturer les entrées (en lui attribuant la valeur true). Notez que cette fonction ne peut être utilisée qu'avec le module HTML5. Syntaxe : browser_input_capture(enable) Les retours : N/A Exemple : browser_input_capture(true) ; Le code ci-dessus configure le jeu pour capturer toutes les entrées du navigateur sur la plate-forme cible HTML5.
