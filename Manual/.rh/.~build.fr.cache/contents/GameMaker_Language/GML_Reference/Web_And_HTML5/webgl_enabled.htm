webgl_enabled Cette variable en lecture seule retournera si WebGL est activé (true ) ou non (false ) pour votre jeu. Cela ne fonctionnera que pour les jeux exécutés via un navigateur (par exemple: HTML5), et pour toutes les autres plates-formes, il retournera true. Syntaxe:webgl_enabled Retour:Boolean Exemple:if webgl_enabled     {     global.quality = 1;     } else global.quality = 0; Le code ci-dessus vérifie l'indicateur WebGL et définit ensuite la variable globale «qualité» en conséquence.
