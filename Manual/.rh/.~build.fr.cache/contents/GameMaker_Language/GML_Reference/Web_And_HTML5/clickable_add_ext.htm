Ajout d'un texte cliquable Avec cette fonction, vous pouvez créer une icône personnalisée et cliquable qui agit comme un "bouton" en dehors du canevas de jeu, mais dans la fenêtre du navigateur elle-même. La fonction renvoie une valeur d'identification qui doit être stockée dans une variable qui sera utilisée dans toutes les autres fonctions de bouton faisant référence à la nouvelle icône. fonctions de bouton ultérieures qui font référence à la nouvelle icône. Cette fonction est très utile pour éviter que le navigateur ne crée une fenêtre pop-up lorsqu'on clique sur le bouton, car tout est fait par GameMaker Studio 2 lui-même. Cette fonction fonctionne exactement de la même manière que la fonction clickable_add() où vous devez définir la position relative au (0,0) de la fenêtre du navigateur, donner l'entrée de la page de texture pour le sprite que vous souhaitez donner à l'icône du bouton (voir : sprite_get_tpe()), spécifier soit une URL à ouvrir, soit un script (à partir du navigateur d'actifs de GameMaker Studio 2) à ouvrir ou à exécuter puis définissez la cible et les paramètres à utiliser. Cependant, cette fonction possède également deux options supplémentaires qui vous permettent de définir l'alpha de l'icône, de 0 (totalement transparent) à 1 (totalement opaque), ainsi que l'échelle de l'icône, qui est une valeur multiplicatrice appliquée au sprite sprite utilisé. Ainsi, une valeur de 1 correspond à l'échelle par défaut, tandis que (par exemple) 0,5 correspond à la moitié de l'échelle et 2 au double. Il convient également de noter que si l'argument URL que vous transmettez est le nom d'une fonction de script (c'est-à-dire qu'il ne commence pas par "http://") ET que le script est appelé gmcallback_* alors vous pouvez obtenir des flux directs dans un événement html5. Les noms des fonctions de script de GameMaker Studio 2 ne sont pas obscurcis s'ils commencent par gmcallback_. Syntaxe : clickable_add_ext(x, y, tpe, url, target, params, alpha, scale) ; Arguments Description x La position x dans la fenêtre. y La position y dans la fenêtre. tpe L'entrée de la page de texture pour le sprite à utiliser. url L'URL (adresse du site Web) vers lequel le lien doit être établi, ou le script à exécuter. cible Il s'agit de la zone cible dans laquelle il faut ouvrir l'URL (voir la description). paramètres Les différents paramètres de l'icône (voir description). alpha L'alpha de l'image de l'icône (par défaut 1). échelle L'échelle de l'icône (par défaut 1). Les retours : Real Exemple : home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2) ; Le code ci-dessus crée une icône DOM cliquable à la position (32, 32) de la page sur laquelle le canevas de jeu est exécuté. L'icône utilise le sprite référencé à partir de la page de texture comme "spr_MS_Home" et, lorsqu'on clique sur l'icône, une nouvelle fenêtre s'ouvre pour l'URL spécifiée et avec les propriétés définies pour cette fenêtre. Le bouton sera également mis à l'échelle à deux fois la taille du sprite original et à un alpha d'image de 1.
