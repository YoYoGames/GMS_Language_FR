clickable_add_ext Avec cette fonction, vous pouvez créer une icône personnalisée, cliquable, qui agit comme un "bouton" en dehors du canevas du jeu, mais dans la fenêtre du navigateur elle-même. La fonction renverra une valeur d'identifiant qui doit être stockée dans une variable pour être utilisée dans toutes les autres fonctions de bouton faisant référence à la nouvelle icône. Cette fonction est très utile pour empêcher le navigateur de créer une fenêtre contextuelle en cliquant sur le bouton car tout est fait via GameMaker Studio 2 lui-même. Cette fonction fonctionne exactement de la même manière que le clickable_add(), où vous devez définir la position par rapport au (0,0) de la fenêtre du navigateur, donnez l'entrée de page de texture pour le sprite que vous souhaitez que l'icône du bouton ait (voir:sprite_get_tpe() ), spécifiez soit une URL à ouvrir, soit un script (à partir du navigateur de GameMaker Studio 2 ) à ouvrir ou à exécuter, puis définissez la cible et les paramètres à utiliser. Cependant, cette fonction a également deux options supplémentaires qui vous permettent de définir l'alpha de l'icône de 0 (entièrement transparent) à 1 (entièrement opaque) et l'échelle de l'icône, qui est une valeur multiplicatrice appliquée au sprite utilisé. Ainsi, une valeur de 1 est l'échelle par défaut, tandis que (par exemple) 0,5 serait la moitié de l'échelle et 2 serait le double. Il est également intéressant de noter que si l'argument URL que vous passez est le nom d'une fonction de script (c'est-à-dire qu'il ne commence pas par "http: //") ET que le script est appelé gmcallback_* alors vous pouvez obtenir des flux directs dans un événement html5. Les noms des fonctions de script GameMaker Studio 2 ne sont pas masqués s'ils commencent par gmcallback_. Syntaxe:clickable_add_ext(x, y, tpe, url, target, params, alpha, scale); Argument Description x La position x dans la fenêtre. y La position y dans la fenêtre. tpe L'entrée de la page de texture pour le sprite à utiliser. url L'URL (adresse du site Web) vers laquelle créer un lien ou le script à exécuter. target Il s'agit de la zone cible dans laquelle ouvrir l'URL (voir description). params Les différents paramètres de l'icône (voir description). alpha L'image alpha de l'icône (par défaut 1). scale L'échelle de l'icône (par défaut 1). Retour:Real Exemple:home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2); Le code ci-dessus crée une icône DOM cliquable à la position (32, 32) de la page sur laquelle le canevas de jeu s'exécute. L'icône utilise le sprite référencé à partir de la page de texture comme "spr_MS_Home" et lorsque vous cliquez dessus, l'icône ouvrira une nouvelle fenêtre pour l'URL spécifiée et avec les propriétés définies pour cette fenêtre. le bouton sera également redimensionné à deux fois la taille originale du sprite et à la moitié d'une image alpha de 1.
