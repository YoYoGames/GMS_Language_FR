clickable_change_ext Avec cette fonction, vous pouvez changer le sprite et la position d'une icône cliquable précédemment créée avec clickable_add(). Gardez à l'esprit que la position est basée sur la fenêtre, pas sur le canevas, (0,0) position et que le sprite doit être référencé directement à partir de la page de texture (voir:sprite_get_tpe() ). Cette fonction vous permet également de changer l'échelle du sprite utilisé (comme un multiplicateur de sorte que 1 soit la valeur par défaut, 0,5 serait la moitié et 2 serait le double) et la valeur alpha de 0 (entièrement transparent) à 1 (entièrement opaque) pour l'icône finale à l'écran. Syntaxe:clickable_change_ext(index, tpe, x, y, alpha, scale) Argument Description index L'index de l'icône cliquable à modifier. tpe L'entrée de page de texture pour le sprite à utiliser x La nouvelle position x dans la fenêtre. y La nouvelle position y dans la fenêtre. scale L'échelle de l'icône (par défaut 1). alpha L'image alpha de l'icône (par défaut 1). Retour:N/A Exemple:switch (room)     {     case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;     case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;     } Le code ci-dessus changera l'index de l'image et la position de l'icône cliquable indexée dans la variable "global.Help" en fonction de la pièce actuelle, en changeant également l'échelle et l'alpha de l'icône.
