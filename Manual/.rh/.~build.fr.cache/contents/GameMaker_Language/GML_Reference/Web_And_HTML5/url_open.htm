url_open Cela ouvrira l'URL spécifiée sur le navigateur de l'appareil cible choisi ou, si vous utilisez le module HTML5, dans le navigateur actuellement ouvert. Syntaxe:url_open(url); Argument Description url L'URL (adresse du site Web) vers laquelle créer un lien. Retour:N/A Exemple:url_open("http://yoyogames.com"); Cela ouvrirait la YoYo Games dans la fenêtre actuelle.
