GM_runtime_version Cette constante contient le numéro de version du runtime défini dans les Préférences comme le runtime utilisé pour construire le projet. La valeur est stockée sous forme de une chaîne de caractères. Syntaxe : GM_runtime_version ; Les retours : Chaîne de caractères Exemple : draw_text(32, 32, date_time_string(GM_build_date)) ; draw_text(32, 64, "v" + GM_version) ; draw_text(32, 96, " Runtime " + GM_runtime_version) ; Le code ci-dessus prend les constantes GM date, version et runtime et les affiche à l'écran.
