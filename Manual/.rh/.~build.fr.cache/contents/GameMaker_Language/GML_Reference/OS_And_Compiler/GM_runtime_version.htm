_Version d' runtime GM_ Cette constante contient le runtime tel que défini dans les préférences en tant que runtime utilisé pour générer le projet. La valeur est stockée sous forme de chaîne. Syntaxe:GM_runtime_version; Retour:String Exemple:draw_text(32, 32, date_time_string(GM_build_date)); draw_text(32, 64, "v" + GM_version); draw_text(32, 96, "Runtime " + GM_runtime_version); Le code ci-dessus prend la date GM, la version et runtime constantes d'exécution et les dessine à l'écran.
