os_check_permission Avec cette fonction, vous pouvez demander une permission spécifique. Vous fournissez la permission à demander sous forme de chaîne de caractères en utilisant le format " android.permission. ", donc pour demander la permission RECORD_AUDIO (par exemple) vous devez appeler os_request_permission("android.permission.RECORD_AUDIO") ; Le retour de la fonction déclenchera un événement asynchrone du système où la carte intégrée de async_load DS map contiendra les paires clé/valeur suivantes : "Type": Il s'agit du type d'événement qui a été déclenché et la valeur sera la chaîne "permission_request_result ". " ": Ceci sera la permission demandée par la fonction sous la forme d'une chaîne de caractères, qui contiendra l'une des constantes suivantes : Constant Description os_permission_granted Indique que la permission a été accordée os_permission_denied Indique que l'autorisation n'a pas été accordée Il convient de noter que les autorisations suivantes sont prises en charge nativement par GameMaker Studio 2, mais sont considérées comme "dangereuses" par Google et doivent donc être explicitement demandées (notez également que certaines autorisations peuvent être demandées à l'aide de la fonction les options de jeu d 'Android sans avoir besoin de cette fonction) : android.permission.WRITE_EXTERNAL_STORAGE android.permission.READ_PHONE_STATE android.permission.RECORD_AUDIO Pour plus d'informations sur les autorisations des applications, veuillez consulter la documentationAndroid. IMPORTANT ! Cette fonction est destinée à la Android cible uniquement. Syntaxe : os_request_permission(permission) Arguments Description autorisation L'autorisation à demander (une chaîne de caractères) Les retours : N/A Exemple : if (os_type == os_android) { if (os_check_permission("android.permission.INTERNET") == os_permission_denied) { os_request_permission("android.permission.INTERNET") ; } } Le code ci-dessus vérifie le type de système d'exploitation et s'il s'agit d'Android, il effectue une vérification des permissions et si la permission "INTERNET" n'a pas encore été accordée, il la demande.
