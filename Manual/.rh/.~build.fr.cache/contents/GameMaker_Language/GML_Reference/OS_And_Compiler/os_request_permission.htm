os_check_permission Avec cette fonction, vous pouvez demander une autorisation spécifique. Vous fournissez l'autorisation de demander sous forme de chaîne en utilisant le format "android.permission. ", afin de demander le RECORD_AUDIO permission (par exemple) que vous appelleriezos_request_permission("android.permission.RECORD_AUDIO"); La fonction retournera déclencher un événement asynchrone système où le async_load La carte DS contiendra les paires clé / valeur suivantes: " type ": il s'agit du type d'événement qui a été déclenché et la valeur sera la chaîne "permission_request_result ". " ": Cela donnera l'autorisation demandée par la fonction sous forme de chaîne, qui contiendra l'une des constantes suivantes: Constant Description os_permission_granted Cela indique que l'autorisation a été accordée os_permission_denied Cela indique que l'autorisation n'a pas été accordée Il convient de noter que les autorisations suivantes sont prises en charge nativement par GameMaker Studio 2, mais sont considérées comme «dangereuses» par Google et doivent donc être explicitement demandées (notez également que certaines autorisations peuvent être demandées à l'aide des options de jeu Android sans avoir besoin de cette fonction):android.permission.WRITE_EXTERNAL_STORAGE android.permission.READ_PHONE_STATE android.permission.RECORD_AUDIO Pour plus d'informations sur les autorisations des applications, veuillez consulter la documentation Android. IMPORTANT! Cette fonction est uniquement Android Syntaxe:os_request_permission(permission) Argument Description permission La permission de demander (une chaîne) Retour:N/A Exemple:if (os_type == os_android)     {     if (os_check_permission("android.permission.INTERNET") == os_permission_denied)         {         os_request_permission("android.permission.INTERNET");         }     } Le code ci-dessus vérifie le type de système d'exploitation et s'il s'agit d' Android, il effectue une vérification des autorisations et si l'autorisation "INTERNET" n'a pas encore été accordée, il la demande.
