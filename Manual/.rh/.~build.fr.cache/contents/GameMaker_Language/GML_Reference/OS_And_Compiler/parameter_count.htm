parameter_count Les paramètres de ligne de commande sont ces commandes supplémentaires que vous pouvez ajouter à un exe pour modifier la façon dont le programme est exécuté. Vous pouvez trouver le nombre de paramètres pour le jeu en cours en utilisant cette fonction, où le premier paramètre a l'index 1 et le dernier a l'index retourné par la fonction (une valeur de 0 est spéciale car c'est le nom de fichier de l'exécutable du jeu, y compris le chemin). Il est à noter que cette fonction fonctionnera sur la plate-forme HTML5, en récupérant les paramètres d'url. Syntaxe:parameter_count(); Retour:Real Exemple:p_num = parameter_count(); Le code ci-dessus stockera le nombre de paramètres de ligne de commande qui ont été utilisés pour le jeu dans la variable "p_num".
