nombre de paramètres Les paramètres de ligne de commande sont les commandes supplémentaires que vous pouvez ajouter à un exe pour modifier la façon dont le programme est exécuté. Vous pouvez trouver le nombre de paramètres pour le jeu actuel en utilisant cette fonction, où le premier paramètre a l'indice 1 et le dernier a l'indice renvoyé par la fonction (une valeur de 0 est spéciale car il s'agit du nom de fichier de l'exécutable du jeu, y compris le chemin). Il est à noter que cette fonction fonctionnera pour la plateforme HTML5, en récupérant les paramètres de l'url. Syntaxe : paramètre_count() ; Les retours : Real Exemple : p_num = parameter_count() ; Le code ci-dessus stockera le nombre de paramètres de ligne de commande qui ont été utilisés pour le jeu dans la variable "p_num".
