appel_externe Si vous avez créé un appel de fonction externe à une dll ou une dylib en utilisant external_define()vous pouvez utiliser cette fonction pour l'appeler. Vous fournissez le nom de la fonction précédemment définie ainsi que chacun des arguments qu'elle requiert (chaque argument doit être d'un type différent). ainsi que chacun des arguments qu'elle requiert (chaque argument doit être du bon type, réel ou chaîne) et la fonction renvoie le résultat de l'appel externe. Syntaxe : appel_externe(id, args[0...15]) ; Arguments Description id Le nom de la fonction que vous voulez appeler args[0...10] Les différents arguments que vous voulez faire passer à la fonction externe Les retours : Valeur (le type de valeur renvoyée dépendra de la fonction définie) Exemple : ma_fonction = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real) ; var _a = external_call(ma_fonction, x, y) ; L'exemple de code ci-dessus appelle une fonction externe préalablement définie et stocke la valeur renvoyée dans une variable locale.
