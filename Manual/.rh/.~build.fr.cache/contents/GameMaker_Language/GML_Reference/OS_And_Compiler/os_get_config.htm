os_get_config Cette fonction renvoie le nom (sous forme de chaîne) de la configuration actuellement sélectionnée pour votre jeu. Pour plus d'informations sur les configurations, veuillez consulter la section Configurations. Syntaxe:os_get_config() Retour:String Exemple:if os_get_config() = "Free_Version"    {    global.Ads = true;    } else global.Ads = false; Le code ci-dessus vérifiera quelle configuration est utilisée et s'il s'agit de celle appelée "Free_Version", les publicités seront activées dans le jeu.
