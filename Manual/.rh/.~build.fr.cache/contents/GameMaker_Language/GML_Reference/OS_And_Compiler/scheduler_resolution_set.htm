scheduler_resolution_set Cette fonction est utilisée pour modifier la résolution du planificateur de threads de Windows. La valeur de la résolution fournie dans l'argument doit être en millisecondes. Par défaut, GameMaker Studio 2 utilise une résolution de 1 milliseconde, qui est la plus petite résolution autorisée et donne les meilleures performances. La modification de la résolution à une valeur plus élevée peut affecter négativement les performances, mais cela peut également réduire la consommation d'énergie. Vous pouvez fournir -1 comme argument pour rétablir la résolution à sa valeur par défaut (telle que définie par Windows). Notez que la modification de la résolution du programmateur aura un impact sur tous les processus exécutés en même temps que votre jeu. NOTE: Cette fonction est destinée à Windows seulement. Syntaxe : scheduler_resolution_set(millisecondes) ; Arguments Description millisecondes La nouvelle valeur de résolution (en millisecondes) ou -1 pour la valeur par défaut. Les retours : N/A Exemple : scheduler_resolution_set(2) ; Cet exemple définit la résolution du planificateur de threads de Windows à 2 millisecondes.
