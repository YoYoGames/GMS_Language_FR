scheduler_resolution_set Cette fonction est utilisée pour changer la résolution du planificateur de thread Windows La valeur de résolution fournie dans l'argument doit être exprimée en millisecondes. Par défaut, GameMaker Studio 2 utilise une résolution de 1 milliseconde, qui est la plus petite résolution autorisée et donne les meilleures performances. Changer la résolution à une valeur plus élevée peut affecter négativement les performances, mais cela peut également réduire la consommation d'énergie. Vous pouvez fournir -1 comme argument pour rétablir la résolution à sa valeur par défaut (telle que définie par Windows ). Notez que la modification de la résolution du planificateur aura un impact sur tous les processus exécutés en même temps que votre jeu. REMARQUE: cette fonction est pour Windows uniquement. Syntaxe:scheduler_resolution_set(milliseconds); Argument Description milliseconds La nouvelle valeur de résolution (en millisecondes) ou -1 par défaut Retour:N/A Exemple:scheduler_resolution_set(2); Cet exemple définit la résolution du Windows sur 2 millisecondes.
