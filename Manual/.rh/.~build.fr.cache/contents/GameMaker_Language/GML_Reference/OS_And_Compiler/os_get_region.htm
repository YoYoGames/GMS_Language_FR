os_get_region Cette fonction renvoie une chaîne contenant le code régional à deux ou trois lettres du système d'exploitation qui exécute le jeu, tel qu'il est défini par la norme ISO3166-1. Si cette information n'est pas disponible elle contiendra simplement une chaîne vide "". REMARQUE: Ce n'est pas le code régional de l'emplacement qui est renvoyé, mais le code de la langue régionale du système d'exploitation. Syntaxe : os_get_region() Les retours : Chaîne de caractères Exemple : switch (os_get_language()) { case "zh" : var region = os_get_region() ; if (region == "HK" || region == "MO" || region == "TW") { ini_open("chinese_traditional.ini") ; } else { ini_open("chinese_simplified.ini") ; } break ; case "fr" : ini_open("french.ini") ; break ; case "it" : ini_open("italian.ini") ; pause ; par défaut : ini_open("english.ini") ; pause ; } Le code ci-dessus vérifie la langue du système d'exploitation et s'il s'agit du chinois, il vérifie ensuite la région du système d'exploitation, ouvrant un fichier *.ini différent en fonction des valeurs retournées.
