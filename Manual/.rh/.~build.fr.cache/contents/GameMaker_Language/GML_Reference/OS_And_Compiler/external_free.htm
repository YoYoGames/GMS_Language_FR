external_free Cette fonction libère la mémoire associée au dll ou dylib avec le prénom. Cela devrait être fait chaque fois que le fichier en question n'est plus nécessaire dans le jeu, normalement (par exemple) dans un événement de fin de partie. Syntaxe:external_free(id); Argument Description id Le nom du dll ou dylib que tu veux libérer Retour:N/A Exemple:external_free("MyDLL.dll"); L'exemple de code ci-dessus libérera la mémoire associée à la dll donnée.
