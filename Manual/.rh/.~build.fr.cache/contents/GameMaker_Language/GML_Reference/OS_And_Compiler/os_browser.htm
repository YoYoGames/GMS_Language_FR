os_browser Cette variable en lecture seule contient l'une des différentes constantes (répertoriées ci-dessous) que GameMaker Studio 2 doit vous indiquer dans quel navigateur vous exécutez actuellement le jeu (le cas échéant). Syntaxe:os_browser Retour:Constant Constant Description browser_not_a_browser Le jeu n'est pas joué dans un navigateur browser_unknown Navigateur inconnu browser_ie Internet Explorer browser_ie_mobile Internet Explorer sur un appareil mobile browser_firefox Mozilla Firefox browser_chrome Google Chrome browser_safari Safari browser_safari_mobile Safari sur un appareil mobile browser_opera Opéra browser_tizen Navigateur d'appareil mobile Tizen browser_windows_store Application Windows Exemple:if (os_browser == browser_not_a_browser)    {    global.Config = 0;    } else    {    global.Config = 1;    } Le code ci-dessus vérifie si le jeu fonctionne dans un navigateur ou non et définit une variable globale sur une valeur en fonction du résultat de la vérification.
