scheduler_resolution_get Cette fonction est utilisée pour récupérer la résolution du planificateur de threads de Windows en millisecondes. Si la résolution de l'ordonnanceur est définie à la valeur par défaut (telle que définie par Windows), la fonction renverra -1. Pour plus d'informations sur le planificateur de threads, voir la page pour scheduler_resolution_set(). NOTE: Cette fonction est destinée à Windows seulement. Syntaxe : scheduler_resolution_get() ; Les retours : Réel (ou -1 pour la valeur par défaut) Exemple : scheduler_resolution = scheduler_resolution_get() ; Cet exemple récupère la résolution du planificateur de threads de Windows, et la stocke dans la variable "scheduler_resolution".
