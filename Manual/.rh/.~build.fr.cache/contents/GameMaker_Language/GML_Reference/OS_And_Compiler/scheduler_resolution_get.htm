scheduler_resolution_get Cette fonction est utilisée pour récupérer la résolution du planificateur de thread Windows Si la résolution du planificateur est définie sur la valeur par défaut (telle que définie par Windows ), la fonction renverra -1. Pour plus d'informations sur le planificateur de threads, reportez-vous à la pagescheduler_resolution_set(). REMARQUE: cette fonction est pour Windows uniquement. Syntaxe:scheduler_resolution_get(); Retour:Real (or -1 for default) Exemple:scheduler_resolution = scheduler_resolution_get(); Cet exemple récupère la résolution du Windows et la stocke dans le"scheduler_resolution" variable.
