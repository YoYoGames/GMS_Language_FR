GM_build_date Cette constante contient la date et l'heure auxquelles l'exécutable en cours d'exécution a été construit par GameMaker Studio 2. La valeur peut être analysée à l'aide des fonctions de date et d'heure. de la valeur UTC du système au moment de la compilation. Syntaxe : GM_build_date ; Les retours : Real Exemple : draw_text(32, 32, date_time_string(GM_build_date)) ; draw_text(32, 64, "v" + GM_version) ; Le code ci-dessus affichera le numéro de version du jeu ainsi que la date et l'heure auxquelles il a été compilé.
