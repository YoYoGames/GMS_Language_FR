environnement_get_variable Cette fonction renvoie la valeur (une chaîne) de la variable d'environnement portant le nom donné (également une chaîne). Vous pouvez obtenir les variables d'environnement disponibles sur macOS et Ubuntu(Linux) en tapant " env " dans l'application terminal. informations sur les variables d'environnement Windows, si vous utilisez l'invite de commande, tapez " echo%PATH%", et avec PowerShell, c'est " ls env". Notez que sur macOS et Ubuntu(Linux), la variable d'environnement " HOME " renverra à la variable " ~/compartiment ". renverra le chemin " ~/ " qui correspond à " /Users/" sous macOS et " /home/ " sous Ubuntu (Linux). REMARQUE: cette fonction est réservée à Windows, macOS et Ubuntu(Linux). Syntaxe : environnement_get_variable(nom) ; Arguments Description nom Le nom (une chaîne) de la variable d'environnement à vérifier. Les retours : Chaîne de caractères Exemple : e_str = environment_get_variable("APPDATA") ; Le code ci-dessus renvoie le chemin complet de la variable d'environnement " %appdata%", qui est normalement " C:\Users\{username}\AppData\Roaming".
