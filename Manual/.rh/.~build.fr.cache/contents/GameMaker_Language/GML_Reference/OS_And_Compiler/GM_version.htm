GM_version Cette constante contient le numéro de version tel que défini dans les options du jeu pour chaque plate-forme cible. La valeur est stockée sous forme de chaîne. Syntaxe : GM_version ; Les retours : Chaîne de caractères Exemple : draw_text(32, 32, date_time_string(GM_build_date)) ; draw_text(32, 64, "v" + GM_version) ; Le code ci-dessus prend les constantes date et version de la GM et les affiche à l'écran.
