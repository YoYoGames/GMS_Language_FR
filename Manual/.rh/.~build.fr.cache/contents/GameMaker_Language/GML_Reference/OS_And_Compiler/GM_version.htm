GM_version Cette constante contient le numéro de version défini dans les options de jeu pour chaque plate-forme cible. La valeur est stockée sous forme de chaîne. Syntaxe:GM_version; Retour:String Exemple:draw_text(32, 32, date_time_string(GM_build_date)); draw_text(32, 64, "v" + GM_version); Le code ci-dessus prend la date et les constantes de version GM et les dessine à l'écran.
