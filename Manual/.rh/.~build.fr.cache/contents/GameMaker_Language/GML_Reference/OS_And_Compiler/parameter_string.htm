chaîne_de_paramètre Les paramètres de ligne de commande sont ces commandes supplémentaires que vous pouvez ajouter à un exe pour modifier la façon dont le programme est exécuté. Avec cette fonction, vous pouvez obtenir le paramètre de ligne de commande choisi sous forme de chaîne. Vous pouvez trouver le nombre de paramètres pour le jeu actuel en utilisant la fonction parameter_count()où le premier paramètre a l'indice 1 et le dernier a l'indice renvoyé par la fonction (une valeur de 0 est spéciale car il s'agit du nom de fichier de l'exécutable du jeu, y compris le chemin d'accès). Il est à noter que cette fonction fonctionne pour la plateforme HTML5, en récupérant les paramètres de l'url. Syntaxe : paramètre_chaîne(n) ; Les retours : Chaîne de caractères Exemple : var p_num ; p_num = parameter_count() ; if p_num > 0 { var i ; for (i = 0 ; i < p_num ; i += 1) { p_string[i] = parameter_string(i + 1) ; } } Le code ci-dessus récupère le nombre de paramètres de la ligne de commande, et s'il y en a 1 ou plus, il les parcourt en boucle et les stocke sous forme de chaînes dans un tableau.
