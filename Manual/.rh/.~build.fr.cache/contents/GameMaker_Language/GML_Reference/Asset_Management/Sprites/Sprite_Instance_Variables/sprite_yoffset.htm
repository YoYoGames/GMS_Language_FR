sprite_yoffset Cette variable en lecture seule renvoie le yoffset local (le composant y de l'origine tel que défini dans l'éditeur de sprite) du sprite qui a été affecté à l'instance. Ce décalage y est renvoyé en pixels et dépendra de la image_yscale, de sorte que (par exemple) si un sprite carré 32x32 est défini comme ayant l'origine y à 16, lorsque ce sprite est mis à l'échelle par 2, cette variable retournera 32. Si vous avez besoin du yoffset non mis à l'échelle, vous devez utiliser sprite_get_yoffset(). Syntaxe:sprite_yoffset; Retour:Real (single precision floating point value) Exemple:if y - sprite_yoffset < 0     {     y = sprite_yoffset;     } Le code ci-dessus vérifie si le bord supérieur de l'image-objet de l'instance est à l'extérieur de la pièce et si c'est le cas, il déplace l'instance afin que l'ensemble du sprite soit visible.
