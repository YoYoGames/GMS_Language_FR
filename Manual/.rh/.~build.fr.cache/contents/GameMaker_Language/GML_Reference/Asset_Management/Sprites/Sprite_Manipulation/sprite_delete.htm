sprite_delete Cette fonction supprime un sprite du jeu, libérant ainsi toute la mémoire qui lui était réservée. C'est une fonction très importante pour les moments où vous devez créer et modifier des sprites à partir de sources externes (comme charger un sprite à partir d'un fichier avec sprite_add() , ou la duplication d'un sprite à l'aide de sprite_duplicate()) et devrait toujours être utilisée pour supprimer les actifs dont le jeu n'a plus besoin, ou pour vider la mémoire. plus nécessaires au jeu, ou pour effacer un actif indexé d'une variable avant de réassigner un autre actif à cette variable. Cette fonction ne peut pas être utilisée pour supprimer les sprites qui sont inclus dans le jeu comme faisant partie des actifs dans le navigateur d'actifs. Syntaxe : sprite_delete(index) ; Arguments Description indice L'index du sprite à supprimer. Renvoie à N/A Exemple : var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16) ; sprite_merge(spr_Player, spr) ; sprite_delete(spr) ; Le code ci-dessus crée une variable locale, puis stocke l'index du sprite créé à partir de la surface de l'application. Ce sprite est ensuite fusionné avec l'actif indexé dans la variable "spr_Player" avant d'être à nouveau retiré de la mémoire.
