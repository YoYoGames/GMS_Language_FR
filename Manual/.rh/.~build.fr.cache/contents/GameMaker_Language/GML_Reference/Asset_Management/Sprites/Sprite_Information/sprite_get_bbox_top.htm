sprite_get_bbox_top Cette fonction renvoie la position relative du haut de la boîte englobante de sprite. Cette valeur est donnée en tant que valeur relative basée sur le coin supérieur gauche de l'actif de sprite de base étant (0,0). c'est la même valeur que celle trouvée dans l'éditeur de sprite pour les propriétés du masque de collision. L'image ci-dessous montre comment il est calculé: Syntaxe:sprite_get_bbox_top(ind); Argument Description ind L'index du sprite à vérifier. RetourReal Exemple:var ww, hh; ww = sprite_get_bbox_left(sprite_index) - sprite_get_bbox_right(sprite_index); hh = sprite_get_bbox_bottom(sprite_index) - sprite_get_bbox_top(sprite_index); Le code ci-dessus calcule la largeur et la hauteur du masque de collision en fonction des positions latérales relatives de la zone de délimitation.
