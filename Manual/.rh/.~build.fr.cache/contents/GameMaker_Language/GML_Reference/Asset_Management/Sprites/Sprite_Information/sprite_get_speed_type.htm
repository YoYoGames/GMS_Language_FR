sprite_get_speed_type Cette fonction peut être utilisée pour récupérer le type d'animation de sprite tel que défini pour la ressource de sprite dans l' éditeur de sprite. La valeur renvoyée sera l'une des constantes suivantes: Constant Descriptionspritespeed_framespersecond Le sprite a été défini avec une animation en images par seconde.spritespeed_framespergameframe Le sprite a été défini avec une animation en images par image de jeu. Les deux exemples suivants illustrent la différence que cela peut faire lors du calcul des vitesses d'animation pour le sprite: Si vous avez un sprite qui dessine 1 image par seconde et définissez la vitesse de l'image à 0,5, il dessinera à 0,5 image par seconde. Si la fréquence d'images de votre jeu est de 60 images par seconde, le sprite dessinera 1 image pour 120 images de jeu. Si vous avez un sprite qui dessine 1 image par image de jeu et définissez la vitesse d'image sur 0,5, il dessinera 0,5 image par image de jeu. Si la fréquence d'images de votre jeu est de 60 images par seconde, alors le sprite dessinera 30 images pour 60 images de jeu. Vous pouvez savoir quelle était la valeur de vitesse utilisée pour l'animation de sprite en utilisant la fonction sprite_get_speed(), et vous pouvez définir la vitesse et le type d'animation à l'aide de la fonction sprite_set_speed(). Syntaxe:sprite_get_speed_type(index) Argument Description index L'index du sprite pour obtenir le type d'animation de Retour:Constant (see above) Exemple:s_speed = sprite_get_speed(sprite_index); s_type = sprite_get_speed_type(sprite_index); Le code ci-dessus obtient la vitesse du sprite et le type d'animation du sprite et les stocke dans des variables d'instance pour une utilisation future.
