attachement_squelette_get Un sprite d'animation squelettique peut avoir d'autres sprites ajoutés en tant qu'attachements, ces sprites étant ajoutés à un slot nommé (le nom est donné lorsque vous créez le slot d'attachement dans votre programme d'animation) et ils seront dessinés avec l'animation du sprite actuel. Avec cette fonction, vous pouvez obtenir le nom (sous forme de chaîne) de l'attachement pour le slot donné du sprite actuellement assigné. Notez que les sprites attachés sont référencés par leur chaîne de nom telle qu'assignée dans Spine, ou lorsque vous avez appelé skeleton_attachment_create(). Syntaxe : skeleton_attachment_get(slot) ; Arguments Description fente Le nom du slot (une chaîne de caractères) pour obtenir l'attachement. Les retours : Chaîne de caractères Exemple : if skeleton_attachment_get("slot_leftHand") == "" { skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife")) ; } Le code ci-dessus vérifie le nom d'attachement actuellement attribué au slot nommé "slot_leftHand" et si une chaîne vide est retournée, un nouveau sprite est attaché.
