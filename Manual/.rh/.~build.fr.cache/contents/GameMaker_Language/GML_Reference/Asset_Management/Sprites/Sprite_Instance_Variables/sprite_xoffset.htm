sprite_xoffset Cette variable en lecture seule renvoie le décalage x local (la composante x de l'origine telle que définie dans l'éditeur de sprite) du sprite qui a été attribué à l'instance. Ce xoffset est retourné en pixels et dépendra de la valeur de l'élément d'origine. image_xscale, de sorte que (par exemple) si un sprite carré de 32x32 est défini comme ayant une origine x à 16, lorsque ce sprite est mis à l'échelle par 2, cette variable retournera 32. Si vous avez besoin du xoffset non mis à l'échelle, vous devez utiliser sprite_get_xoffset(). Syntaxe : sprite_xoffset ; Les retours : Réel (valeur à virgule flottante de simple précision) Exemple : si x - sprite_xoffset < 0 { x = sprite_xoffset ; } Le code ci-dessus vérifie si le bord gauche du sprite de l'instance est à l'extérieur de la pièce et si c'est le cas, il déplace l'instance afin que le sprite entier soit visible.
