index_spirit Cette variable retourne l'index du sprite actuel de l'instance, ou -1 si l'instance n'a pas de sprite associé. Vous pouvez la modifier pour donner à l'instance un sprite différent en lui donnant le nom d'un sprite de l'arbre des ressources ou en utilisant une variable dans laquelle est indexé un sprite chargé en externe. Changer le sprite ne change pas l'index de la frame actuellement visible, donc si vous changez le sprite de la frame numéro 3, le nouveau sprite sera dessiné avec cette frame visible. (en supposant qu'il ait le même nombre de frames). Syntaxe : sprite_index ; Les retours : Real (nombre entier - valeur de l'index de la ressource) Exemple : with (obj_Check) { if !collision_line(x, y, other.x, other.y, obj_Wall, false, true) { sprite_index = spr_spotted ; } else { sprite_index = spr_clear ; } } Le code ci-dessus passera en boucle toutes les instances de "obj_Check" pour vérifier s'il existe une ligne de collision entre elles et l'instance qui exécute le code. Le sprite de ces instances sera modifié en fonction de la valeur de retour (vrai ou faux) de la ligne de collision. ligne de collision.
