skeleton_bone_list Avec cette fonction, vous pouvez remplir une liste DS (pré-créée) avec tous les noms des structures utilisées dans le cadre du sprite d'animation squelettique. Les noms seront des chaînes et pourront ensuite être utilisés dans les autres fonctions osseuses d'animation squelette pour ces types de sprite. Syntaxe:skeleton_bone_list(sprite, list); Argument Description sprite Index de sprite de l'animation squelettique Spine à partir de laquelle obtenir la liste. list L'ID de la liste DS à renseigner avec les noms d'os. Retour:N/A Exemple:bone_list = ds_list_create(); skeleton_bone_list(sprite_index, bone_list); Le code ci-dessus crée une liste DS, puis la remplit avec les noms d'os pour une utilisation ultérieure.
