image_speed Cette variable détermine la vitesse à laquelle GameMaker Studio 2 les sous-images de l'image-objet de l'instance actuelle. La valeur de vitesse donnée est un multiplicateur, 1 étant la valeur par défaut, et la définir sur 0,5 réduira la moitié de la vitesse de l'animation - telle que définie dans l' éditeur de sprite ou l' éditeur d'images - tandis que la définir sur 2 la doublera. Si le sprite utilisé n'a pas de sous-images, cette variable n'aura aucun effet. Syntaxe:image_speed; Retour:Real (single precision floating point value) Exemple:with (instance_create_layer(x, y, "Effects", obj_Explosion))     {     image_speed = 0.5;     } Le code ci-dessus crée une instance de l'objet "obj_Explosion" et définit son image_speed sur 0,5 (réduisant de moitié la vitesse de défilement).
