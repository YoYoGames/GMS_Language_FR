sprite_get_bbox_right Cette fonction renvoie la position relative de la droite de la boîte de délimitation du sprite. Cette valeur est donnée sous la forme d'une valeur relative basée sur le coin supérieur gauche du sprite de base (0,0). pour les propriétés du masque de collision. L'image ci-dessous montre comment elle est calculée : Syntaxe : sprite_get_bbox_right(ind) ; Arguments Description ind L'index du sprite à vérifier. Renvoie à Real Exemple : var ww, hh ; ww = sprite_get_bbox_left(sprite_index) - sprite_get_bbox_right(sprite_index) ; hh = sprite_get_bbox_bottom(sprite_index) - sprite_get_bbox_top(sprite_index) ; Le code ci-dessus calcule la largeur et la hauteur du masque de collision en fonction des positions latérales relatives de la boîte englobante.
