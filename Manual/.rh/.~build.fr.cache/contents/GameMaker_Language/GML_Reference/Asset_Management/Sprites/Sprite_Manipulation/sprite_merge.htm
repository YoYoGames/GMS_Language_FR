fusion de sprites Cette fonction fusionnera le sprite indexé dans l'argument 1 ("ind2") avec celui qui est indexé dans l'argument 0 ("ind1"). Les images elles-mêmes ne sont PAS fusionnées, mais plutôt les indices des images sont fusionnées, avec les sous-images du sprite "ind2" ajoutées à celles du sprite "ind1", c'est-à-dire qu'elles sont ajoutées à la fin. Notez que si les sprites sont de tailles différentes, alors les sprites ajoutés sont étirés pour s'adapter à la taille de l'image pour "ind1". Ce changement est permanent, et à partir du moment où vous utilisez cette fonction jusqu'à la fermeture du jeu ou la suppression du sprite, le sprite dans lequel il est fusionné sera modifié, mais le sprite avec lequel il est fusionné restera le même. REMARQUE: ni l'image qui fait l'objet de la fusion, ni l'image qui fait l'objet de la fusion, ne peuvent faire partie des ressources originales du jeu. Vous devez utiliser sprite_duplicate() premièrement pour créer deux nouvelles ressources (si vous fusionnez des ressources de jeu incluses), puis les fusionner. REMARQUE: Cette fonction n'est utile que pour les sprites bitmap et ne fonctionne pas avec les sprites SWF ou JSON (Spine). Syntaxe : sprite_merge(ind1, ind2) ; Arguments Description ind1 L'index du sprite à fusionner. ind2 L'index du sprite avec lequel ind1 doit être fusionné. Renvoie à N/A Exemple : var tspr ; tspr[0] = spr_add(working_directory + "Sprite Assets\explode1.png", 16, true, true, 0, 0) ; tspr[1] = sprite_duplicate(spr_Explosion) ; sprite_merge(tspr[0], tspr[1]) ; sprite_index = t_spr[0] ; sprite_delete(tspr[1]) ; Le code ci-dessus charge un sprite dans une variable locale puis le fusionne avec un autre sprite qui a été dupliqué. Enfin, le sprite fusionné est assigné à l'instance sprite_index tandis que le sprite chargé est supprimé pour libérer la mémoire qu'il utilise. pour libérer la mémoire qu'il utilise. Notez qu'à un moment donné après cette opération, comme dans les événements Destroy ou Room End, vous devrez également utiliser la fonction sprite_delete() pour supprimer le sprite fusionné de la mémoire également.
