animation_squelette_set_frame Cette fonction va définir l'animation assignée à la piste donnée à l'image que vous fournissez. L'indice de trame doit être une valeur réelle comprise entre 0 et le nombre de trames de l'animation (que vous pouvez obtenir en utilisant la fonction animation_squelette_get_frames). Si vous fournissez une valeur en dehors de cette plage, l'animation sera mise en boucle pour compenser la différence, par exemple si l'animation a 5 images et que vous définissez l'image à 8, l'image réelle affichée sera 3. Syntaxe : skeleton_animation_set_frame(piste, index) ; Arguments Description piste La piste d'animation pour laquelle il faut définir le cadre. indice Le nouvel index pour le cadre d'animation. Les retours : N/A Exemple : if global.pause { skeleton_animation_set_frame(0, -image_speed) ; } Le code ci-dessus définira l'indice de l'image pour l'animation dans la piste 0 à la vitesse négative de l'image lorsque la variable globale "pause" est vraie (essentiellement en mettant l'animation en pause).
