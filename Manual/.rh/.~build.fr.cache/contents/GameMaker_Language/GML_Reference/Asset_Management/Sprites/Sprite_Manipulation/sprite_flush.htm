sprite_flush Cette fonction permet de supprimer de la mémoire de texture la page de texture donnée pour le sprite donné. Vous fournissez l'index du sprite (tel que défini lors de la création de la ressource sprite) et la page de texture à laquelle il est assigné sera supprimée de la mémoire de texture. Notez que si la page de texture est utilisée ailleurs dans la room (par un autre sprite d'instance ou un arrière-plan, etc...), vous pouvez avoir un léger impact sur les performances car la page est immédiatement rechargée, donc il faut faire attention en utilisant cette fonction. Notez que la fonction renvoie -1 si le flush n'est pas supporté par la ressource choisie, ou 0 si tout a fonctionné correctement. Syntaxe : sprite_flush(ind) Arguments Description ind L'index (nom de la ressource) du sprite à purger. Les retours : Int (-1 ou 0) Exemple : sprite_flush(spr_Player_Aura) ; Le code ci-dessus vide le sprite "spr_Player_Aura" de la mémoire.
