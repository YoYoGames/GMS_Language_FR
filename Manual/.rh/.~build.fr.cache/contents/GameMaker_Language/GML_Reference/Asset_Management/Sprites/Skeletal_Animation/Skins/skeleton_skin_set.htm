squelette_skin_set Un sprite d'animation squelettique peut être dessiné avec une variété de textures (appelées "peaux"), ce qui vous permet d'utiliser une seule animation à des fins multiples. Par exemple, vous pouvez avoir une animation de marche générique qui peut être utilisée par vos personnages PNJ, vos ennemis, etc. personnages PNJ, ennemis, etc... dans un jeu en changeant simplement la peau avec laquelle elle est dessinée. Ces habillages doivent avoir été préalablement créés et nommés dans votre programme d'animation, et ce nom est passé dans la fonction (sous forme de chaîne) pour que le sprite à utiliser cet habillage lorsqu'il est dessiné. Syntaxe : skeleton_skin_set(skinname) ; Arguments Description nom de peau Le nom (une chaîne) de la peau à utiliser. Les retours : N/A Exemple : if skeleton_skin_get() == "skin_Enemy1" { skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3") ; } Le code ci-dessus vérifie le skin actuellement attribué à l'animation et s'il s'agit de "skin_Enemy1", il choisit et définit un nouveau skin parmi trois options.
