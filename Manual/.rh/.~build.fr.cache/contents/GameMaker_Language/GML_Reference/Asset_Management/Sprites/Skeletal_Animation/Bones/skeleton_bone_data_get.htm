données de l'ossature squelette_get Votre animation squelettique est composée d'un certain nombre d'"os", que vous aurez définis et auxquels vous aurez donné des noms dans votre programme d'animation. Cette fonction peut être utilisée pour obtenir des informations sur l'os nommé à n'importe quel moment et stocker les données dans une map DS map vide, que vous aurez préalablement créée. Notez que ces données font référence à la pose par défaut du squelette, et non à la pose actuelle qui est dessinée. Si vous avez besoin des données de la pose actuelle, utilisez skeleton_bone_state_get(). Lorsque vous appelez cette fonction, la carte sera remplie avec les clés suivantes : "x" : La position x locale de l'os par rapport à l'os parent. "y" : La position locale y de l'os par rapport à l'os parent. "angle" : La rotation locale de l'os par rapport à l'os parent. "xscale" : L'échelle horizontale locale de l'os, en référence à l'os parent. "yscale" : L'échelle verticale locale de l'os, en référence à l'os parent. "parent" : Le nom (une chaîne de caractères) de l'os parent. Syntaxe : skeleton_bone_data_get(bone, map) ; Arguments Description os Le nom (sous forme de chaîne) de l'os. carte La carte DS (précédemment créée) qui stocke les données relatives aux os. Les retours : N/A Exemple : bone_map = ds_map_create() ; skeleton_bone_data_get("leftarm", bone_map) ; Le code ci-dessus crée une carte DS et la remplit ensuite avec les données de l'os nommé "bras gauche".
