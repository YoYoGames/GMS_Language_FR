bbox_bottom Cette variable en lecture seule renvoie la position y (dans la room) du bas de la boîte de délimitation de l'instance, la boîte de délimitation étant définie par la largeur et la hauteur maximales du masque de l'instance (telles que définies par définies par le site sprite_index ou par le site mask_index). Même lorsqu'un sprite possède un masque de collision précis, la boîte de délimitation existe et est utilisée pour certaines choses. pour certaines choses, et vous pouvez donc utiliser cette variable pour la trouver. Veuillez noter que lorsque l'instance n'a pas de sprite assigné, la valeur retournée sera la même que la position Y de l'instance. Syntaxe : bbox_bottom ; Les retours : Réel (nombre entier) Exemple : si bbox_bottom > room_height { y = room_height - sprite_yoffset ; } Le code ci-dessus vérifie si la boîte englobante de l'instance est en dehors de la pièce et si c'est le cas, il place l'instance à une nouvelle position.
