liste_slot_squelette Avec cette fonction, vous pouvez remplir une liste DS (pré-créée) avec tous les noms des slots créés dans le cadre du sprite d'animation squelettique. Les noms seront des chaînes de caractères et pourront ensuite être utilisés dans les autres fonctions de slot d'animation squelettique pour ces types de sprites. Syntaxe : skeleton_slot_list(sprite, liste) ; Arguments Description sprite L'index du sprite de l'animation squelettique Spine pour obtenir la liste. liste L'ID de la liste DS à remplir avec les noms des os. Les retours : N/A Exemple : bone_list = ds_list_create() ; skeleton_slot_list(sprite_index, bone_list) ; Le code ci-dessus crée une liste de DS puis la remplit avec les noms de slot pour une utilisation ultérieure.
