skeleton_bone_data_set Votre animation squelettique est composée d'un certain nombre de "structures", que vous aurez défini et donné des noms dans votre programme d'animation, et cette fonction peut être utilisée pour définir certaines données pour l'os nommé à tout moment. Notez que ces données se réfèrent à la pose par défaut du squelette, et non à la pose actuelle qui est dessinée (pour cela, utilisez la fonction skeleton_bone_state_set() ), et doit être définie à partir d'une carte DS créée précédemment, qui doit avoir les clés suivantes et leurs valeurs équivalentes: "x": la position x locale de l'os par rapport à l'os parent. "y": la position y locale de l'os par rapport à l'os parent. "angle": La rotation locale de l'os par rapport à l'os parent. "xscale": l'échelle horizontale locale de l'os. "yscale": l'échelle verticale locale de l'os. "parent": le nom (une chaîne) de l'os parent. Syntaxe:skeleton_bone_data_set(bone, map); Argument Description bone Le nom (sous forme de chaîne) de l'os. map La carte DS (créée précédemment) qui stocke les données d'os. Retour:N/A Exemple:var bone_map = ds_map_create(); skeleton_bone_data_get("head", bone_map); ds_map_replace(bone_map, "parent", "body"); skeleton_bone_data_set("head", bone_map); ds_map_destroy(bone_map); Le code ci-dessus crée une carte DS, puis la remplit avec les données de l'os nommé "tête". Il remplace ensuite la clé d'os «parent» dans la carte par une nouvelle valeur et définit à nouveau l'os «de tête» avec le nouvel ensemble de données.
