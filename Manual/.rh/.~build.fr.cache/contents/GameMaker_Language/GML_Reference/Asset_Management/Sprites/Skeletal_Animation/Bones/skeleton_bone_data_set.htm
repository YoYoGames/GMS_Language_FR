données_squelettes Votre animation squelettique est composée d'un certain nombre d'"os", que vous aurez définis et auxquels vous aurez donné des noms dans votre programme d'animation. Cette fonction peut être utilisée pour définir certaines données pour l'os nommé à tout moment. Notez que ces données font référence à la pose par défaut du squelette, et non à la pose actuelle qui est dessinée (pour cela, utilisez la fonction skeleton_bone_state_set()), et doivent être définies à partir d'une carte DS précédemment créée. DS map, qui doit avoir les clés suivantes et leurs valeurs équivalentes : "x" : La position x locale de l'os par rapport à l'os parent. "y" : La position locale y de l'os par rapport à l'os parent. "angle" : La rotation locale de l'os par rapport à l'os parent. "xscale" : L'échelle horizontale locale de l'os. "yscale" : L'échelle verticale locale de l'os. "parent" : Le nom (une chaîne de caractères) de l'os parent. Syntaxe : skeleton_bone_data_set(bone, map) ; Arguments Description os Le nom (sous forme de chaîne) de l'os. carte La carte DS (précédemment créée) qui stocke les données relatives aux os. Les retours : N/A Exemple : var bone_map = ds_map_create() ; skeleton_bone_data_get("head", bone_map) ; ds_map_replace(bone_map, "parent", "body") ; skeleton_bone_data_set("head", bone_map) ; ds_map_destroy(bone_map) ; Le code ci-dessus crée une carte DS, puis la remplit avec les données de l'os nommé "head". Il remplace ensuite la clé de l'os "parent" dans la carte par une nouvelle valeur et définit à nouveau l'os "head" avec le nouvel ensemble de données.
