sprite_get_bbox_mode Cette fonction renvoie le "mode" actuel pour les calculs de la boîte englobante. Vous fournissez l'index du sprite à vérifier et la fonction renverra l'une des constantes indiquées ci-dessous. Syntaxe : sprite_get_bbox_mode(ind) ; Arguments Description ind L'index du sprite à vérifier. Renvoie à Constant Constant Description bboxmode_automatic Automatique - La boîte de délimitation sera calculée automatiquement, sur la base du paramètre de tolérance du sprite. bboxmode_fullimage Image complète - La boîte de délimitation sera définie pour utiliser la largeur et la hauteur complètes du sprite, sans tenir compte de la tolérance et des pixels "vides". bboxmode_manual Manuel - Le rectangle de délimitation a été réglé manuellement sur des valeurs définies par l'utilisateur (soit dans l'éditeur de sprites, soit à l'aide de la fonction sprite_set_bbox()) Exemple : if sprite_get_bbox_mode(sprite_index) != 0 { sprite_set_bbox_mode(sprite_index, bboxmode_automatic) ; } La commande ci-dessus vérifie le mode bbox du sprite actuel et s'il n'est pas automatique, il le définit à cette valeur.
