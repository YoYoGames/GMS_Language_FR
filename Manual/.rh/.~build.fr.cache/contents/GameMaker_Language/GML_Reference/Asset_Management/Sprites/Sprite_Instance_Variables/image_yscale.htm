image_yscale Cette valeur définit la mise à l'échelle verticale (selon l'axe des y) appliquée au sprite qui a été affecté à l'instance actuelle. Une échelle de 1 indique qu'il n'y a pas de mise à l'échelle (1:1), des valeurs plus petites réduiront l'échelle (0.5, par exemple, réduira de moitié la hauteur du sprite), des valeurs plus grandes augmenteront l'échelle et des valeurs négatives reflèteront le sprite et le mettront à l'échelle, sauf si la valeur utilisée est exactement -1 (dans ce cas, le sprite est simplement reflété le long de l'axe des y, sans mise à l'échelle). Syntaxe : image_yscale ; Les retours : Réel (valeur à virgule flottante de précision simple) Exemple : if image_xscale < 5 { image_xscale += 0.2 ; image_yscale = image_xscale ; } else { instance_create_layer(x, y, "Effects", obj_Explosion) ; instance_destroy() ; } Le code ci-dessus met à l'échelle le sprite et une fois qu'il a atteint 5 fois sa taille d'origine, une nouvelle instance d'un autre objet est créée et l'instance est détruite.
