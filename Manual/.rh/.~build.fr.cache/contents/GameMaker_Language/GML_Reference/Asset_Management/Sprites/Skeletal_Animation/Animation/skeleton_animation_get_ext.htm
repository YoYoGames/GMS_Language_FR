skeleton_animation_get_ext Un seul sprite d'animation squelettique peut avoir différents ensembles d'animations, et ces ensembles peuvent se voir attribuer différentes pistes afin que vous puissiez «mélanger et assortir» des ensembles d'animations. Cette fonction renverra le nom du jeu d'animation actuellement utilisé par le numéro de piste donné (tel que défini par la fonction skeleton_animation_set_ext ). Syntaxe:skeleton_animation_get_ext(track); Argument Description track Numéro de piste pour lequel obtenir le nom du jeu d'animations. Retour:String Exemple:if skeleton_animation_get_ext(1) != "bodyfight"     {     skeleton_animation_set_ext("bodyfight", 1);     } Le code ci-dessus changera le jeu d'animations utilisé par la piste 1 en "bodyfight" si ce n'est déjà fait.
