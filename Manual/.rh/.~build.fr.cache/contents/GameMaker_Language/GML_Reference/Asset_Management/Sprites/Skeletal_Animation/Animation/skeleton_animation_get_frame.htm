skeleton_animation_get_frame Cette fonction renverra le numéro d'image de l'animation sur la piste d'animation spécifiée. La fonction retournera -1 si aucune animation n'est affectée à la piste spécifique donnée. Syntaxe:skeleton_animation_get_frame(track); Argument Description track La piste d'animation dont vous souhaitez obtenir l'image. Retour:Integer Exemple:var frame = skeleton_animation_get_frame(0); if frame > 0 && frame < 2     {     var box = skeleton_get_minmax();     var ww = (box[2] - box[0]) / 2;     var hh = (box[3] - box[1]) / 2;     part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);     } Le code ci-dessus vérifiera l'image actuelle de l'animation affectée à la piste 0, puis éclatera certaines particules à partir d'un point situé au milieu de la boîte englobante totale du sprite.
