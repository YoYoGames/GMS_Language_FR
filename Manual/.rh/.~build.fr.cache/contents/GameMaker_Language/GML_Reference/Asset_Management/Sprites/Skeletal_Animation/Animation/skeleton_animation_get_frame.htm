animation_squelette_get_frame Cette fonction renvoie le numéro d'image de l'animation sur la piste d'animation spécifiée. La fonction renverra -1 si aucune animation n'est assignée à la piste spécifique donnée. Syntaxe : skeleton_animation_get_frame(track) ; Arguments Description piste La piste d'animation dont on veut obtenir l'image. Les retours : Entier Exemple : var frame = skeleton_animation_get_frame(0) ; if frame > 0 && frame < 2 { var box = skeleton_get_minmax() ; var ww = (box[2] - box[0]) / 2 ; var hh = (box[3] - box[1]) / 2 ; part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10) ; } Le code ci-dessus vérifiera l'image actuelle de l'animation assignée à la piste 0, puis fera éclater des particules à partir d'un point situé au milieu de la boîte de délimitation totale du sprite.
