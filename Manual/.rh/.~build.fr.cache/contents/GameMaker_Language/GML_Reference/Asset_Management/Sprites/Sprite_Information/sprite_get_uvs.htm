sprite_get_uvs Cette fonction renvoie un tableau avec les coordonnées UV et d'autres données pour la texture de la sous-image de sprite sur la page de texture. La fonction renvoie un tableau avec les 8 éléments suivants: [0] = gauche [1] = haut [2] = droite [3] = bas [4] = quantité de pixels que le compilateur d'éléments a supprimés du côté gauche des images-objets [5] = quantité de pixels que le compilateur d'éléments a supprimés de la partie supérieure des images-objets [6] = pourcentage normalisé de données de pixels par rapport à la largeur des sprites d'origine qui a été enregistrée sur la page de texture [7] = pourcentage normalisé de données de pixels par rapport à la hauteur des sprites d'origine qui a été enregistrée sur la page de texture Ce tableau peut ensuite être utilisé dans d'autres fonctions de dessin, en particulier lorsque vous travaillez en 3D ou que vous utilisez les fonctions primitives 2D, ainsi que lorsque vous travaillez avec les fonctions Shader REMARQUE: cette fonction ne fonctionnera pas avec les sprites vectoriels ou les sprites d'animation squelette. Syntaxe:sprite_get_uvs(sprite, subimage); Argument Description sprite L'index du sprite à utiliser. subimage La sous-image du sprite à utiliser. RetourArray (1D, 8 elements) Exemple:var tex = sprite_get_uvs(sprite, 0); tex_left = tex[0]; tex_top = tex[1]; tex_right = tex[2]; tex_bottom = tex[3]; Le code ci-dessus stockera les coordonnées UV du sprite donné dans un tableau local, puis affectera les valeurs aux variables d'instance.
