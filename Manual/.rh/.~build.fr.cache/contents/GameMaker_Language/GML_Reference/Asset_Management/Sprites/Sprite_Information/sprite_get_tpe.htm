sprite_get_tpe Cette fonction renvoie une valeur spéciale qui correspond à la position du sprite et de la sous-image donnés sur la page de texture utilisée par votre jeu. De cette façon, vous pouvez transmettre la valeur à des extensions pour une utilisation en dehors du jeu compilé, en contournant efficacement les fonctions de dessin de GameMaker Studio 2. les fonctions de dessin de GameMaker Studio 2 et permettre au sprite d'être utilisé dans le contenu DOM qui peut ensuite être dessiné n'importe où dans la fenêtre qui contient le canevas du jeu. Cette fonction est particulièrement intéressante pour ceux qui souhaitent créer des boutons et d'autres médias interactifs en dehors de l'élément de canevas de GameMaker Studio 2 sur leur page hôte à l'aide de la fonction clickable_add(). REMARQUE: Cette fonction est réservée au HTML5. Syntaxe : sprite_get_tpe(sprite, index) ; Arguments Description sprite L'index du sprite pour trouver l'entrée de la page de texture. indice L'index de l'image du sprite. Renvoie à Real Exemple : home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0") ; Le code ci-dessus crée un bouton DOM à la position (32, 32) de la page sur laquelle le canevas de jeu est exécuté. Le bouton utilise le sprite référencé dans la page de texture en tant que "spr_MS_Home" et, lorsqu'il est cliqué, le bouton ouvre une nouvelle fenêtre à l'adresse suivante l'URL spécifiée et avec les propriétés définies pour cette fenêtre.
