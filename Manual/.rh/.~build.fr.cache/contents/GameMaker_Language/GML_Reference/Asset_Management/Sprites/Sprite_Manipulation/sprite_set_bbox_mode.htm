sprite_set_bbox_mode Cette fonction peut être utilisée pour définir le mode de la boîte englobante d'un sprite. Vous fournissez l'index du sprite et le mode à utiliser, qui doit être l'une des constantes suivantes : Constant Description bboxmode_automatic Automatique - La boîte de délimitation sera calculée automatiquement, sur la base du paramètre de tolérance du sprite. bboxmode_fullimage Image complète - La boîte de délimitation sera définie pour utiliser la largeur et la hauteur complètes du sprite, sans tenir compte de la tolérance et des pixels "vides". bboxmode_manual Manuel - La boîte englobante est définie manuellement sur des valeurs définies par l'utilisateur, ce qui nécessite l'utilisation de la fonction sprite_set_bbox() REMARQUE: cette fonction affecte l'actif du sprite de sorte que toutes les autres instances de ce sprite auront le même mode de boîte englobante. Syntaxe : sprite_set_bbox_mode(ind, mode) ; Arguments Description ind L'index du sprite dont il faut changer le mode. mode Le mode à définir (une constante). Renvoie à N/A Exemple : if sprite_get_bbox_mode(sprite_index) != 0 { sprite_set_bbox_mode(sprite_index, bboxmode_automatic) ; } L'opération ci-dessus vérifie le mode bbox du sprite actuel et s'il n'est pas automatique, il est défini à cette valeur.
