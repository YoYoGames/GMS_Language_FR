Des os Comme son nom l'indique, les sprites d'animation squelette sont constitués d'une série de "bones", sur lesquels un skin est ajouté pour créer l'animation finale du personnage. Normalement, vous voudrez configurer ces os dans le logiciel d'animation (Spine), puis exporter le sprite avec un ensemble d'animations déjà prédéfini afin que vous puissiez simplement basculer entre eux si nécessaire. Cependant, il peut arriver que vous souhaitiez intercepter les données osseuses et les manipuler manuellement ou obtenir des informations sur la position d'os spécifiques, et dans ces cas, vous pouvez utiliser les fonctions suivantes: skeleton_bone_data_getskeleton_bone_data_setskeleton_bone_state_getskeleton_bone_state_setskeleton_bone_list Retour: Sprites d'animation squelettique Suivant: Machines à sous © Copyright YoYo Games Ltd. 2021 Tous droits réservés
