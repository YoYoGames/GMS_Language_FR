sprite_get_width Avec cette fonction, vous pouvez trouver la largeur de l'image-objet de ressource de base, sans transformation, en pixels. Syntaxe:sprite_get_width(index); Argument Description index L'index du sprite pour trouver la largeur de. RetourReal Exemple:if sprite_width != sprite_get_width(sprite_index)     {     image_xscale = 1;     } Le code ci-dessus vérifie la largeur du sprite tel qu'il est dans l'instance actuelle et s'il y a une différence entre cela et le sprite de base d'origine, il réinitialise l'échelle de l'axe x.
