sprite_get_width Avec cette fonction, vous pouvez trouver la largeur du sprite de la ressource de base, sans transformations, en pixels. Syntaxe : sprite_get_width(index) ; Arguments Description indice L'index du sprite dont il faut trouver la largeur. Renvoie à Real Exemple : if sprite_width != sprite_get_width(sprite_index) { image_xscale = 1 ; } Le code ci-dessus vérifie la largeur du sprite tel qu'il est dans l'instance actuelle et s'il y a une différence entre celle-ci et le sprite de base original, il réinitialise l'échelle de l'axe x.
