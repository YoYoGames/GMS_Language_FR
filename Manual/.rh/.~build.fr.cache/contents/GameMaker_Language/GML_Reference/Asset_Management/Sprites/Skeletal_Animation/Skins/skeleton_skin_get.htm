peau_squelette_get Avec les sprites d'animation squelettique, vous pouvez attribuer des textures distinctes (appelées "skins") à l'animation, ce qui permet d'utiliser une animation pour plusieurs choses différentes. Cette fonction renverra le nom de la peau (sous forme de chaîne) qui est actuellement assigné au sprite d'animation squelettique que votre instance utilise. Le nom renvoyé est celui que vous avez défini lorsque vous avez créé le sprite dans votre programme d'animation. Syntaxe : skeleton_skin_get() ; Les retours : Chaîne de caractères Exemple : if skeleton_skin_get() == "skin_Enemy1" { skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3") ; } Le code ci-dessus vérifie le skin actuellement attribué à l'animation et s'il s'agit de "skin_Enemy1", il choisit et définit un nouveau skin parmi trois options.
