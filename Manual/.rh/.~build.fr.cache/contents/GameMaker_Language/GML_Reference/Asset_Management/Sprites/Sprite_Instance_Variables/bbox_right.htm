bbox_right Cette variable en lecture seule renvoie la position dans la room (le long de l'axe des x) du côté droit de la boîte de délimitation de l'instance, la boîte de délimitation étant définie par la largeur et la hauteur maximales du masque de l'instance (telles que définies par sprite_index ou par mask_index). Même lorsqu'un sprite dispose d'un masque de collision précis, la boîte englobante existe et est utilisée pour certaines choses. existe et est utilisée pour certaines choses, et vous pouvez donc utiliser cette variable pour la trouver. Veuillez noter que lorsque l'instance n'a pas de sprite assigné, la valeur retournée sera la même que la position X de l'instance. Syntaxe : bbox_right ; Les retours : Réel (nombre entier) Exemple : si bbox_right > room_width { x = room_width - sprite_xoffset ; } Le code ci-dessus vérifie si la boîte englobante de l'instance est en dehors de la pièce et si c'est le cas, il place l'instance à une nouvelle position.
