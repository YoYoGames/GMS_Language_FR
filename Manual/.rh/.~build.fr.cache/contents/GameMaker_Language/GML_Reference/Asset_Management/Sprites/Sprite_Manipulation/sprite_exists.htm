sprite_exists Cette fonction renvoie si un sprite avec l'index spécifié existe ou non dans le projet en cours d'exécution. Syntaxe:sprite_exists(index); Argument Description index L'index du sprite à vérifier. RetourN/A Exemple:if sprite_exists(spr_array[0])     {     sprite_index = spr_array[0];     } else     {     sprite_index = spr_BaseSprite;     } Le code ci-dessus vérifie un tableau pour voir s'il contient un index de sprite valide et si c'est le cas, il affecte ce sprite à l'instance, mais sinon, il affecte un sprite à partir des ressources incluses.
