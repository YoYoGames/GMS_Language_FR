sprite_existe Cette fonction retourne si un sprite avec l'index spécifié existe ou non dans le projet en cours d'exécution. Syntaxe : sprite_exists(index) ; Arguments Description indice L'index du sprite à vérifier. Renvoie à N/A Exemple : if sprite_exists(spr_array[0]) { sprite_index = spr_array[0] ; } else { sprite_index = spr_BaseSprite ; } Le code ci-dessus vérifie un tableau pour voir s'il contient un index de sprite valide et si c'est le cas, il attribue ce sprite à l'instance, mais si ce n'est pas le cas, il attribue un sprite à partir des ressources incluses.
