animation_squelette_claire Cette fonction vide la piste d'animation spécifiée de toutes les animations, prêtes à être réaffectées. Syntaxe : skeleton_animation_clear(track) ; Arguments Description piste La piste d'animation s'efface. Les retours : N/A Exemple : if mouse_check_button(mb_right) { skeleton_animation_clear(1) ; } Le code ci-dessus effacera la piste 1 de l'animation si le bouton droit de la souris est enfoncé.
