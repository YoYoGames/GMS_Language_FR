skeleton_animation_clear Cette fonction effacera la piste d'animation spécifiée de toutes les animations, prêtes à être réaffectées. Syntaxe:skeleton_animation_clear(track); Argument Description track La piste d'animation à effacer. Retour:N/A Exemple:if mouse_check_button(mb_right)     {     skeleton_animation_clear(1);     } Le code ci-dessus effacera la piste d'animation 1 si le bouton droit de la souris est enfoncé.
