image_alpha Cette variable est utilisée pour obtenir ou définir la valeur alpha du sprite. Alpha est toujours calculé comme une valeur entre 0 et 1 où 0 est complètement transparent et 1 est complètement opaque. Veuillez noter que pour que les modifications apportées à cette variable soient visibles, l'instance doit avoir soit aucun événement de dessin (et donc GameMaker Studio 2 dessine par défaut le sprite), soit être dessinée à l'aide de l'une des fonctions de dessin étendues comme draw_self() ou draw_sprite_ext(). Syntaxe:image_alpha; Retour:Real (between 0 and 1 - single precision floating point value) Exemple:image_alpha = clamp(image_alpha - 0.01, 0, 1); Le code ci-dessus réduira lentement image_alpha jusqu'à ce qu'il atteigne 0.
