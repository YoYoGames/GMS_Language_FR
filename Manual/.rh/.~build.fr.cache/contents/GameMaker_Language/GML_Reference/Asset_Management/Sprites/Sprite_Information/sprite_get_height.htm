sprite_get_height Avec cette fonction, vous pouvez trouver la hauteur de l'actif de sprite de base, sans transformation, en pixels. Syntaxe:sprite_get_height(index); Argument Description index L'index du sprite pour trouver la hauteur de. RetourReal Exemple:if sprite_height != sprite_get_height(sprite_index)     {     image_yscale = 1;     } Le code ci-dessus vérifie la hauteur du sprite tel qu'il est dans l'instance actuelle et s'il existe une différence entre celui-ci et le sprite de base d'origine, il réinitialise l'échelle de l'axe y.
