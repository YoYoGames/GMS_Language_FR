sprite_get_height Avec cette fonction, vous pouvez trouver la hauteur du sprite de base, sans transformation, en pixels. Syntaxe : sprite_get_height(index) ; Arguments Description indice L'index du sprite dont il faut trouver la hauteur. Renvoie à Real Exemple : if sprite_height != sprite_get_height(sprite_index) { image_yscale = 1 ; } Le code ci-dessus vérifie la hauteur du sprite tel qu'il est dans l'instance actuelle et s'il existe une différence entre celle-ci et le sprite de base original, il réinitialise l'échelle de l'axe des y.
