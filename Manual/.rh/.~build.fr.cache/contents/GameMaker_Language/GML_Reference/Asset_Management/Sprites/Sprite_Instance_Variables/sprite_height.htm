hauteur du sprite Cette variable en lecture seule renvoie la hauteur du sprite qui a été assigné à l'instance. Cette hauteur est retournée en pixels et dépendra de la valeur de la variable image_yscale. Si vous avez besoin de la hauteur non mise à l'échelle, vous devez utiliser la fonction sprite_get_height(). Syntaxe : hauteur du sprite ; Les retours : Réel (valeur à virgule flottante de simple précision) Exemple : if sprite_height != sprite_get_height(sprite_index) { image_yscale = 1 ; } Le code ci-dessus vérifie la hauteur du sprite attribué à l'instance qui exécute le code par rapport à la hauteur de la ressource du sprite et si ce n'est pas la même chose, il réinitialise l'image_yscale à 1, Retour : Variables d'instance de sprite Suivant : sprite_xoffset Copyright YoYo Games Ltd. 2021 Tous droits réservés
