Neuf fonctions de tranche En plus de la section Nine Slice de l'éditeur de sprites, GameMaker Studio 2 fournit des fonctions permettant de définir les propriétés Nine Slice pour les sprites au moment de l'exécution. Avant de prendre connaissance de ces fonctions, il est recommandé de lire cette page pour comprendre la technique du Nine Slice. Les fonctions suivantes peuvent être utilisées pour récupérer et modifier les propriétés de Nine Slice pour un actif sprite au moment de l'exécution: sprite_nineslice_create sprite_get_nineslice sprite_set_nineslice Ces fonctions utilisent une structure Nine Slice, dont vous pouvez lire le contenu sur cette page.
