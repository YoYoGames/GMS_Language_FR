Neuf fonctions de tranche En plus de la section Nine Slice de l' éditeur de sprites, GameMaker Studio 2 fournit des fonctions permettant de définir les propriétés de Nine Slice pour les sprites lors de l' runtime. Avant de lire sur ces fonctions, il est recommandé de lire cette page pour comprendre la technique Nine Slice. Les fonctions suivantes peuvent être utilisées pour récupérer et modifier les propriétés de Nine Slice pour une ressource de sprite au runtime de l'exécution: sprite_nineslice_create sprite_get_nineslice sprite_set_nineslice Ces fonctions utilisent une structure Nine Slice, et vous pouvez lire le contenu de cette structure sur cette page.
