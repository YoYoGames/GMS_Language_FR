sprite_get_number Un sprite doit avoir au moins une sous-image et cette fonction renvoie le nombre de sous-images qu'il possède. Rappelez-vous qu'une sous-image de sprite commence à être numérotée à partir de 0, donc si cette fonction renvoie 3 (le sprite vérifié possède 3 sous-images), elles seront numérotées 0, 1 et 2. Syntaxe : sprite_get_number(index) ; Arguments Description indice L'index du sprite. Renvoie à Entier Exemple : image_index = floor(random(sprite_get_number(sprite_index))) ; Le code ci-dessus définira l'index de l'image à un cadre aléatoire basé sur le nombre de sous-images que le sprite possède.
