sprite_get_number Un sprite doit avoir au moins une sous-image et cette fonction retournera juste combien il en a. N'oubliez pas qu'une sous-image de sprite commence à être numérotée à partir de 0, donc si cette fonction renvoie 3 (le sprite en cours de vérification a 3 sous-images), elles seront numérotées 0, 1 et 2. Syntaxe:sprite_get_number(index); Argument Description index L'index du sprite. RetourInteger Exemple:image_index = floor(random(sprite_get_number(sprite_index))); Le code ci-dessus définira l'index de l'image sur une image aléatoire en fonction du nombre de sous-images que possède le sprite.
