largeur de sprite Cette variable en lecture seule renvoie la largeur du sprite qui a été assigné à l'instance. Cette largeur est renvoyée en pixels et dépendra de la variable image_xscale. Si vous avez besoin de la largeur non mise à l'échelle vous devez utiliser sprite_get_width(). Syntaxe : largeur du sprite ; Les retours : Réel (valeur à virgule flottante de précision simple) Exemple : if sprite_width != sprite_get_width(sprite_index) { image_xscale = 1 ; } Le code ci-dessus vérifie la largeur du sprite attribué à l'instance qui exécute le code par rapport à la largeur de la ressource du sprite et si elle n'est pas la même, il réinitialise l'image_xscale à 1, Retour : Variables d'instance de sprite Suivant : sprite_height Copyright YoYo Games Ltd. 2021 Tous droits réservés
