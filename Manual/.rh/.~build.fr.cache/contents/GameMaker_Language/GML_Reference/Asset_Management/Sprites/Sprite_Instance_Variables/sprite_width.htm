sprite_width Cette variable en lecture seule renvoie la largeur de l'image-objet qui a été affectée à l'instance. Cette largeur est renvoyée en pixels et dépendra de la image_xscale. Si vous avez besoin de la largeur non mise à l'échelle, vous devez utiliser sprite_get_width(). Syntaxe:sprite_width; Retour:Real (single precision floating point value) Exemple:if sprite_width != sprite_get_width(sprite_index)     {     image_xscale = 1;     } Le code ci-dessus vérifie la largeur du sprite affecté à l'instance exécutant le code par rapport à la largeur de la ressource sprite et si ce n'est pas le même, il réinitialise le image_xscale à 1, Retour: Variables d'instance de sprite Suivant:sprite_height © Copyright YoYo Games Ltd. 2021 Tous droits réservés
