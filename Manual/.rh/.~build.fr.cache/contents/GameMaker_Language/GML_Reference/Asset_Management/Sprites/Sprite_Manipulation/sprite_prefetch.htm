sprite_prefetch Cette fonction peut être utilisée pour préextraire (placer dans la mémoire de texture) une page de texture avec le sprite donné. Vous fournissez l'index du sprite tel que défini lors de la création du sprite, et la page de texture sur laquelle il se trouve sera chargée en mémoire. Notez que la fonction renvoie -1 si la préextraction n'est pas prise en charge par la ressource choisie ou si la plate-forme cible est HTML5, ou elle renvoie 0 si tout a fonctionné correctement. REMARQUE: Il y a un impact sur les performances lorsque la texture est téléchargée dans la mémoire de texture sur la plupart des périphériques, il est donc recommandé d'appeler sprite_prefetch sur tout graphique requis au début d'un niveau pour éviter tout blocage. Syntaxe : sprite_prefetch(ind) Arguments Description ind L'index du sprite à récupérer Les retours : Int (-1 ou 0) Exemple : sprite_prefetch(spr_Player_Aura) ; Le code ci-dessus placera le sprite référencé dans la mémoire de texture, prêt à être utilisé.
