image_angle Cette valeur définit l'angle (rotation) de l'image-objet et est mesurée en degrés, la droite étant 0 °, le haut étant 90 °, la gauche 180 ° et le bas 270 °. Définissez cette variable sur 0 pour réinitialiser le sprite à dessiner comme cela a été défini dans l'éditeur de sprite. Veuillez noter que pour que les modifications apportées à cette variable soient visibles, l'instance doit avoir soit aucun événement de dessin (et donc GameMaker Studio 2 dessine par défaut le sprite), soit être dessinée à l'aide de l'une des fonctions de dessin étendues comme draw_self() ou draw_sprite_ext(). Syntaxe:image_angle; Retour:Real (single precision floating point value) Exemple:image_angle = point_direction(x, y, mouse_x, mouse_y); Le code ci-dessus fera pivoter le sprite de l'instance pour toujours pointer vers la position de la souris.
