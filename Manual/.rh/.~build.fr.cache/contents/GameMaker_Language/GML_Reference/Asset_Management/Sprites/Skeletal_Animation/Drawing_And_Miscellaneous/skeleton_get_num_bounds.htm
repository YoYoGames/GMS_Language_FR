squelette_get_num_bounds Cette fonction renvoie le nombre de boîtes de délimitation associées au sprite d'animation du squelette attribué à l'instance qui exécute le code. Elle peut ensuite être utilisée avec la fonction skeleton_get_bounds() pour récupérer des données sur chacune des boîtes englobantes. Syntaxe : skeleton_get_num_bounds() ; Les retours : Entier Exemple : var num = skeleton_get_num_bounds() ; var yy = 60 ; for(var i = 0 ; i < num ; i++ ;) { var b_info = skeleton_get_bounds(i) ; if b_info[0] > 0 { var data = b_info[1] + " :"; for(var j = 0 ; j < b_info[0] ; j++ ;) { data += " (" + string(b_info[(j * 2) + 2])) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")" ; } draw_text(20, yy, data) ; yy += 20 ; } } Le code ci-dessus parcourt en boucle chacune des boîtes englobantes associées au sprite actuellement attribué, puis affiche ces données à l'écran sous forme de chaîne.
