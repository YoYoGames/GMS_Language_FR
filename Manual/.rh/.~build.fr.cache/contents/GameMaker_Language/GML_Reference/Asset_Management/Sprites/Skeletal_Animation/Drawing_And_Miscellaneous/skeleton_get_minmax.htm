squelette_get_minmax Cette fonction renvoie un tableau contenant la valeur totale de la boîte englobante pour toutes les boîtes englobantes individuelles attribuées à un sprite d'animation de squelette. Le tableau retourné comportera 4 éléments : 0] - la position x minimale pour toutes les boîtes englobantes. 1] - la position y minimale pour toutes les boîtes englobantes. 2] - la position x maximale pour toutes les boîtes englobantes. 3] - la position maximale en y pour toutes les boîtes englobantes. Syntaxe : skeleton_get_minmax() ; Les retours : Tableau (4 éléments : xMin, yMin, xMax, yMax) Exemple : var frame = skeleton_animation_get_frame(0) ; if frame > 0 && frame < 2 { var box = skeleton_get_minmax() ; var ww = (box[2] - box[0]) / 2 ; var hh = (box[3] - box[1]) / 2 ; part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10) ; } Le code ci-dessus vérifiera l'image actuelle de l'animation assignée à la piste 0, puis fera éclater des particules à partir d'un point situé au milieu de la boîte de délimitation totale du sprite.
