dessiner_squelette_collision Cette fonction va dessiner les masques de collision associés à l'animation squelettique donnée. Vous fournissez le sprite de base, le jeu d'animation à utiliser et l'image à partir de laquelle obtenir les informations, et vous pouvez également définir les propriétés de transformation en conséquence. Syntaxe : draw_skeleton_collision(sprite, animname, frame, x, y, xscale, yscale, rot, colour) Arguments Description sprite L'index du sprite à dessiner. nom de l'animateur Le nom de l'animation dont il faut récupérer l'image (une chaîne). cadre Le cadre d'animation à dessiner (de 0 à nombre_d'images - 1). x La coordonnée x de l'endroit où dessiner le sprite. y La coordonnée y de l'endroit où dessiner le sprite. xscale La mise à l'échelle horizontale du sprite, sous forme de multiplicateur : 1 = mise à l'échelle normale, 0.5 est la moitié etc... yscale La mise à l'échelle verticale du sprite, sous forme de multiplicateur : 1 = mise à l'échelle normale, 0.5 est la moitié etc... pourriture La rotation du sprite. 0=normal, 90=tourné de 90 degrés dans le sens inverse des aiguilles d'une montre, etc. couleur La couleur avec laquelle le sprite doit être mélangé. Les retours : N/A Exemple : draw_skeleton_collision(sprite_index, "jump", image_index, x, y, image_xscale, image_yscale, image_angle, c_white) ; Le code ci-dessus dessine les données du masque de collision pour le sprite actuel, en utilisant les transformations actuelles, pour l'animation "jump".
