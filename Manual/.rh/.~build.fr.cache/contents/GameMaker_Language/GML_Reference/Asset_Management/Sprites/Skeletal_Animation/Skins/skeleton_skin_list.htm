liste_de_skin_squelette Avec cette fonction, vous pouvez remplir une liste DS (pré-créée) avec tous les noms des skins inclus dans le sprite de l'animation squelettique. Les noms seront des chaînes de caractères et peuvent ensuite être être utilisés dans les autres fonctions de skin d'animation squelettique pour ces types de sprites. Syntaxe : squelette_skin_list(sprite, liste) ; Arguments Description sprite L'index du sprite de l'animation squelettique Spine pour obtenir la liste. liste L'ID de la liste DS à remplir avec les noms des animations. Les retours : N/A Exemple : var list = ds_list_create() ; skeleton_skin_list(sprite_index, list) ; var num = ds_list_size(list) ; skeleton_skin_set(list[| irandom(num - 1)) ; ds_list_destroy(list) ; Le code ci-dessus crée une liste DS puis la remplit avec les noms de skin. Une peau aléatoire est ensuite choisie et appliquée à l'instance avant que la liste ne soit détruite.
