sprite_get_xoffset Lorsque vous définissez un sprite dans l'éditeur de sprite, vous devez définir l' origine de ce sprite. C'est le point auquel le sprite sera "attaché" ou "dessiné" lorsqu'il est utilisé avec une instance. Cette fonction renvoie le décalage relatif pour l'axe des x de l'origine en fonction du coin supérieur gauche étant la position 0,0, avec un + x à droite et un -x à gauche. L'image suivante illustre cela: Syntaxe:sprite_get_xoffset(index); Argument Description index L'index du sprite pour trouver le xoffset de. RetourReal Exemple:if x - sprite_get_xoffset(sprite_index) < 0     {     x = sprite_get_xoffset(sprite_index);     } Le code ci-dessus garantira qu'une instance est maintenue dans la salle en fonction du sprite (afin que le sprite soit toujours visible).
