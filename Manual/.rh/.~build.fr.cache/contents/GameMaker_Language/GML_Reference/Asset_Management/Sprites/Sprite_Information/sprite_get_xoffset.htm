sprite_get_xoffset Lorsque vous définissez un sprite dans l'éditeur de sprites, vous devez définir l'origine de ce sprite. Il s'agit du point auquel le sprite sera "attaché" ou "dessiné" lorsqu'il sera utilisé avec une instance. Cette fonction renvoie le décalage relatif pour l'axe x de l'origine, en se basant sur le coin supérieur gauche qui est la position 0,0, avec un +x à droite et un -x à gauche de celle-ci. L'image suivante illustre cela : Syntaxe : sprite_get_xoffset(index) ; Arguments Description indice L'index du sprite dont il faut trouver le décalage en x. Renvoie à Real Exemple : si x - sprite_get_xoffset(sprite_index) < 0 { x = sprite_get_xoffset(sprite_index) ; } Le code ci-dessus garantit qu'une instance est maintenue dans la pièce en fonction du sprite (de sorte que le sprite soit toujours visible).
