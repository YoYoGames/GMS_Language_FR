skeleton_get_bounds Cette fonction renverra un tableau de valeurs associées à tout cadre de délimitation donné pour le sprite d'animation squelette actuellement affecté. Vous fournissez le numéro d'index de la boîte englobante pour obtenir les détails de (vous pouvez récupérer le nombre total de boîtes englobantes pour l'image-objet à l'aide de la fonction skeleton_get_num_bounds() ) et la fonction renverra un tableau avec les éléments suivants: [0] - le nombre de points sur la boîte englobante (un réel) [1] - le nom de la boîte englobante (une chaîne) [2] - la position x du premier point [3] - la position y du premier point [etc...] - autres données de position x / y jusqu'au nombre de points sur la boîte englobante Syntaxe:skeleton_get_bounds(index); Argument Description index Index de la boîte englobante pour obtenir les valeurs de. Retour:Array (minimum 2 elements: numPoints, name [, xPos, yPos, etc...]) Exemple:var num = skeleton_get_num_bounds(); var yy = 60; for(var i = 0; i < num; i++;)     {     var b_info = skeleton_get_bounds(i);     if b_info[0] > 0         {         var data = b_info[1] + ":";         for(var j = 0; j < b_info[0]; j++;)             {             data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";             }         draw_text(20, yy, data);         yy += 20;         }     } Le code ci-dessus parcourra chacune des boîtes englobantes associées au sprite actuellement attribué, puis dessinera ces données sous forme de chaîne à l'écran.
