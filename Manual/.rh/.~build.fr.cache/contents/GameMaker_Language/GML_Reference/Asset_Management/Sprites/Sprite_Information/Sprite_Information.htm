Informations sur les sprites Les fonctions suivantes peuvent être utilisées pour obtenir des informations sur un actif sprite. Veuillez noter que ces fonctions obtiennent des informations à partir du sprite de base (c'est-à-dire le sprite asset, tel que créé dans l'Asset Browser) et non du sprite tel qu'il est utilisé dans une instance. Par exemple, vous pouvez utiliser la sprite variable d'instance largeur_spirit pour obtenir la largeur du sprite qu'une instance utilise et cette largeur prendra en compte toute mise à l'échelle qui aurait pu être appliquée. Cependant, si vous avez besoin de connaître la largeur d'un sprite tel qu'il serait normalement, sans transformation appliquée, vous devrez utiliser la fonction sprite_get_width() ci-dessous.
