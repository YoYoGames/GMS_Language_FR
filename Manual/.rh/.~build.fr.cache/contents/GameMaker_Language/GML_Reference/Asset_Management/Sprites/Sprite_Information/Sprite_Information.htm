Informations sur les sprites Les fonctions suivantes peuvent être utilisées pour obtenir des informations sur une ressource de sprite. Veuillez noter que ces fonctions obtiennent les informations du sprite de base (c.-à-d. L'actif du sprite, tel que créé dans le navigateur de ressources ) et non du sprite tel qu'il est utilisé dans une instance. Par exemple, vous pouvez utiliser la variable d'instance de sprite sprite_width pour obtenir la largeur du sprite qu'une instance utilise et cette largeur prendra en compte toute mise à l'échelle qui peut avoir été appliquée. Cependant, si vous avez besoin de connaître la largeur d'un sprite comme elle le serait normalement, sans aucune transformation appliquée, vous devrez utiliser le sprite_get_width() fonction ci-dessous.
