draw_skeleton_time Cette fonction va dessiner l'animation donnée en utilisant le skin donné à un moment précis de l'animation. La valeur du temps doit être comprise entre 0 (le début) et la durée finale de l'animation, que vous pouvez trouver en utilisant la fonction skeleton_animation_get_duration(). Vous pouvez définir la valeur de temps à des valeurs supérieures à la durée totale de l'animation et l'animation reviendra en boucle au début, mais vous courez le risque de perdre la précision de la virgule flottante à mesure que le temps cumulé accumulé s'accroît. Syntaxe : draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, colour) Arguments Description sprite L'index du sprite à dessiner. nom de l'animateur Le nom de l'animation dont il faut récupérer l'image (une chaîne). nom de peau Le nom de la peau à utiliser pour le dessin (une chaîne). temps L'intervalle de temps pour dessiner l'animation (de 0 à la durée finale, en secondes). x La coordonnée x de l'endroit où dessiner le sprite. y La coordonnée y de l'endroit où dessiner le sprite. xscale La mise à l'échelle horizontale du sprite, sous forme de multiplicateur : 1 = mise à l'échelle normale, 0.5 est la moitié etc... yscale La mise à l'échelle verticale du sprite, sous forme de multiplicateur : 1 = mise à l'échelle normale, 0.5 est la moitié etc... pourriture La rotation du sprite. 0=normal, 90=tourné de 90 degrés dans le sens inverse des aiguilles d'une montre, etc. couleur La couleur avec laquelle le sprite doit être mélangé. Les retours : N/A Exemple : var time += delta_time / 1000000 ; var d = skeleton_animation_get_duration("walk") ; if time > d time -= d ; draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_blanc) ; Le code ci-dessus dessine le sprite d'animation squelettique donné en utilisant le delta-time pour définir l'image à dessiner.
