squelette_attachement_créer Lorsque vous créez votre sprite d'animation squelettique, vous pouvez lui attribuer des emplacements d'attachement et des pièces jointes. Il s'agit simplement d'images qui ne font pas partie de l'animation mais qui, lorsqu'elles sont attachées, se déplacent avec l'os auquel elles sont attachées. se déplaceront en même temps que l'os auquel elles sont attachées. Normalement, vous devez attribuer ces pièces jointes dans votre programme d'animation (Spine), mais avec cette fonction, vous pouvez créer les vôtres au moment de l'exécution en utilisant un sprite de votre jeu. que vous donniez un nom à la pièce jointe (sous la forme d'une chaîne), puis que vous définissiez le sprite_index et l'image_index à utiliser, ainsi que l'origine x et y (qui peut être différente de celle définie par le sprite dans les propriétés du sprite). que vous souhaitez appliquer à l'image lorsqu'elle est jointe. Si la pièce jointe a été créée avec succès, la fonction renvoie 1, sinon (vous avez fourni un index de sprite invalide, ou le sprite de base n'est pas un sprite Spine ), elle renvoie -1. Syntaxe : skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot) ; Arguments Description nom Le nom (sous forme de chaîne) de la pièce jointe à créer. sprite Le sprite_index pour obtenir l'image de la pièce jointe. ind L'index d'image pour obtenir l'image de la pièce jointe. xorigine L'origine x de l'image utilisée. yorigine L'origine y de l'image utilisée. xscale La mise à l'échelle horizontale de l'image, sous forme de multiplicateur : 1 = mise à l'échelle normale, 0.5 est la moitié etc... yscale La mise à l'échelle verticale de l'image, sous forme de multiplicateur : 1 = mise à l'échelle normale, 0.5 est la moitié etc... pourriture La rotation de l'image. 0=normal, 90=tourné de 90 degrés dans le sens inverse des aiguilles d'une montre, etc. Les retours : 1 en cas de succès, -1 en cas d'échec Exemple : skeleton_attachment_create("sword", spr_Weapons, 0, 0, 80, 1, 1, 90) ; skeleton_attachment_create("knife", spr_Weapons, 1, 0, 45, 1, 1, 90) ; skeleton_attachment_create("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0) ; skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow")) ; Le code ci-dessus vérifie le nom d'attachement actuellement attribué au slot nommé "slot_leftHand" et si une chaîne vide est retournée, un nouveau sprite est attaché.
