skeleton_attachment_create Lorsque vous créez votre sprite d'animation squelettique, vous pouvez attribuer des emplacements de pièce jointe et des pièces jointes pour y accéder. Ce sont simplement des images séparées de l'animation, mais lorsqu'elles sont attachées, elles se déplacent avec l'os auquel elles sont attachées. Normalement, vous attribuez ces pièces jointes dans votre programme d'animation (Spine), mais avec cette fonction, vous pouvez créer les vôtres au moment de l'exécution en utilisant un élément de sprite de votre jeu. la fonction nécessite que vous donniez un nom à la pièce jointe (sous forme de chaîne), puis que vous définissiez le sprite_index et l'image_index à utiliser, ainsi que l'origine x et y (qui peut être différente de celle définie par le sprite dans les propriétés du sprite), et vous pouvez également définir toutes les transformations que vous souhaitez appliquer à l'image une fois attachée. Si la pièce jointe a été créée avec succès, la fonction retournera 1 et si ce n'était pas le cas (vous avez fourni un index de sprite invalide, ou le sprite de base n'est pas un sprite Spine) alors il retournera -1. Syntaxe:skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot); Argument Description name Le nom (sous forme de chaîne) de la pièce jointe à créer. sprite Le sprite_index à partir duquel obtenir l'image de la pièce jointe. ind Image_index à partir de laquelle obtenir l'image de la pièce jointe. xorigin L'origine x de l'image utilisée. yorigin L'origine y de l'image utilisée. xscale La mise à l'échelle horizontale de l'image, sous forme de multiplicateur: 1 = mise à l'échelle normale, 0,5 est la moitié etc... yscale La mise à l'échelle verticale de l'image, sous forme de multiplicateur: 1 = mise à l'échelle normale, 0,5 est la moitié etc... rot La rotation de l'image. 0 = normal, 90 = tourné de 90 degrés dans le sens inverse des aiguilles d'une montre, etc. Retour:1 if successful, -1 if not Exemple:skeleton_attachment_create("sword", spr_Weapons, 0, 0, 80, 1, 1, 90); skeleton_attachment_create("knife", spr_Weapons, 1, 0, 45, 1, 1, 90); skeleton_attachment_create("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0); skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow")); Le code ci-dessus vérifierait le nom de pièce jointe actuellement attribué pour l'emplacement nommé "slot_leftHand" et si une chaîne vide est renvoyée, un nouveau sprite est attaché.
