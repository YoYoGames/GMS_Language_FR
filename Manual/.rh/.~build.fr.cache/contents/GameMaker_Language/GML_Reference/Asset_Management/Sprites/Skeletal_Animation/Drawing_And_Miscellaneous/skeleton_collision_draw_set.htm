skeleton_collision_draw_set Avec cette fonction, vous pouvez activer (true ) ou désactivé (false ) dessinant les données de collision pour l'image-objet d'animation squelettique actuelle utilisée par l'instance. Si cette option est activée, la boîte englobante et le masque de collision précis seront dessinés sous forme de contours autour de l'image-objet. Syntaxe:skeleton_collision_draw_set(flag); Argument Description flag Mis à true pour activer le dessin, et false pour l'éteindre. Retour:N/A Exemple:if debug_mode == true     {     skeleton_collision_draw_set(true);     } Le code ci-dessus vérifie si le jeu est exécuté en mode débogage et si tel est le cas, il bascule les données de collision squelettique pour l'instance à afficher.
