timeline_exists Avec cette fonction, vous pouvez vérifier et voir si une ligne de temps existe (renvoie true ) ou non (retourne false ). Ceci est particulièrement utile lors de la création dynamique de timelines à l'aide de timeline_add(), mais vous devez noter que si vous recherchez une variable non initialisée (qui serait autrement affectée à l'index d'une ligne de temps), une erreur sera générée. Syntaxe:timeline_exists(ind); Argument Description ind L'index de la chronologie à vérifier. Retour:Boolean Exemple:if timeline_exists(global.tl)     {     timeline_delete(global.tl);     } Le code ci-dessus vérifie si une ligne de temps existe et est indexée dans la variable globale "tl" et si c'est le cas, il supprime cette ligne de temps.
