Délais Les chronologies sont un mécanisme puissant pour contrôler les événements de votre jeu et sont basées sur des «moments», où un «moment» est simplement une seule étape de jeu. Ces chronologies sont généralement créées à partir du navigateur de ressources en utilisant l' éditeur de chronologie de la même manière qu'un objet, c'est-à-dire: vous créez une chronologie, puis à chaque moment où vous en avez besoin pour effectuer une action, vous ajoutez du code ou une DnD™ pour il pour effectuer. Une fois créé, vous pouvez alors arrêter, démarrer et manipuler les chronologies via le code. Vous pouvez même modifier des moments individuels en définissant une fonction de script et en l'ajoutant dynamiquement à la chronologie à partir d'un objet contrôleur (par exemple), ce qui en fait un outil très puissant et polyvalent lors de la création de vos jeux. Les fonctions suivantes permettent d'ajouter et de manipuler des chronologies à partir de votre jeu: timeline_existstimeline_get_nametimeline_addtimeline_deletetimeline_moment_add_scripttimeline_moment_cleartimeline_cleartimeline_sizetimeline_max_moment Les variables suivantes sont "intégrées" à toutes les instances pour faciliter l'utilisation des timelines et vous permettre de les contrôler: timeline_indextimeline_runningtimeline_speedtimeline_positiontimeline_loop Notez que vous pouvez trouver plus d'informations sur les variables d'instance ici: Variables d'instance.
