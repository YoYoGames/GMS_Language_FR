timeline_clear Avec cette fonction, vous pouvez effacer une ligne de temps spécifique de "moments", en supprimant tous les codes et actions pour cette ligne de temps et en la laissant vide. Syntaxe:timeline_clear(ind); Argument Description ind L'index de la ligne de temps à effacer. Retour:N/A Exemple:if timeline_position > 200     {     timeline_clear(global.tl);     timeline_index = -1;     } Le code ci-dessus effacera la ligne de temps spécifiée indexée par la variable "global.tl" de tous les moments où un moment spécifique a été passé.
