position_temporaire Cette variable contient la position actuelle (moment) d'une ligne de temps. Vous pouvez modifier cette valeur pour sauter des parties de la ligne de temps, pour répéter des parties ou pour recommencer la ligne de temps depuis le début. Syntaxe : timeline_position ; Les retours : Réel (valeur à virgule flottante de simple précision) Exemple : if !timeline_running { timeline_position = 0 ; timeline_running = true ; } Le code ci-dessus vérifiera si l'instance exécute une ligne de temps, et si ce n'est pas le cas, il réinitialisera la ligne de temps assignée pour qu'elle démarre au premier moment, puis la lancera.
