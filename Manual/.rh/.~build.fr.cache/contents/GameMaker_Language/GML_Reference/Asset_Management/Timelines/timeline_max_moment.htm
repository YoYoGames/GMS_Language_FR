délai_maximum_moment Cette fonction renvoie la valeur du moment où la timeline effectue sa dernière action. Ainsi, si vous avez une ligne de temps avec 8 actions différentes placées à 20 moments d'intervalle, cette fonction renverra la valeur du moment de la dernière action dans cette ligne de temps, qui serait de 160. Cette fonction est un bon moyen de tester si une ligne de temps a dépassé le dernier moment actif lors de son exécution, puisque la position de la ligne de temps continuera à s'incrémenter à chaque étape du jeu, qu'il y ait d'autres actions ou non. Syntaxe : timeline_max_moment(ind) ; Arguments Description ind L'index de la ligne de temps pour obtenir le dernier moment. Les retours : N/A Exemple : si timeline_position > timeline_max_moment(timeline_index) { timeline_running = false ; } Le code ci-dessus vérifiera la position actuelle de la timeline par rapport au moment actif maximum, et si elle est supérieure, la timeline sera arrêtée.
