timeline_speed Normalement, à chaque étape, la position dans la ligne de temps est augmentée de 1, mais vous pouvez modifier ce montant en définissant cette variable sur une valeur différente. Vous pouvez utiliser des nombres réels (comme 0,5 ou 2,4 par exemple) et si la valeur est supérieure à un, plusieurs moments peuvent se produire dans le même pas de temps (ils seront tous exécutés dans le même ordre que celui défini pour la ligne de temps, donc aucune action ne sera ignorée). Syntaxe:timeline_speed; Retour:Real (single precision floating point value) Exemple:if !timeline_running     {     timeline_running = true;     timeline_speed = 0.5;     } Le code ci-dessus vérifiera et verra si l'instance exécute sa chronologie associée et si ce n'est pas le cas, elle démarrera la chronologie à demi-vitesse.
