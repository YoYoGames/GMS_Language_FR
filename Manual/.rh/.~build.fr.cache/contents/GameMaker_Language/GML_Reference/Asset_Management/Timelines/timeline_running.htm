timeline_running Cette variable contient l'état actuel de la ligne de temps assignée et retournera true s'il est en cours d'exécution et false dans le cas contraire. Vous pouvez également définir cette variable sur true ou false pour démarrer et arrêter la chronologie à tout moment. il convient de noter qu'une ligne de temps arrêtée n'est pas réinitialisée, et donc la redémarrer ultérieurement la redémarrera à partir du moment exact où elle a été arrêtée. Syntaxe:timeline_running; Retour:Boolean Exemple:if !timeline_running     {     timeline_position = 0;     timeline_running = true;     } Le code ci-dessus vérifiera si l'instance exécute une ligne de temps, et si ce n'est pas le cas, il réinitialise la ligne de temps assignée pour démarrer au premier moment, puis la démarre.
