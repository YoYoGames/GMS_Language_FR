taille de la ligne de temps Avec cette fonction, vous pouvez obtenir le nombre total de moments actifs pour une ligne de temps (un moment "actif" est un moment auquel on a ajouté ducode ou des DnD™). Cela peut être pratique lors de la création de lignes de temps dynamiques en utilisant les fonctions timeline_moment_add_script() et timeline_moment_clear() car vous pouvez vérifier si une timeline donnée a le nombre correct de moments actifs ou aucun. Syntaxe : timeline_size(ind) ; Arguments Description ind L'index de la ligne de temps pour obtenir l'information. Les retours : N/A Exemple : if timeline_size(global.tl) == 0 { timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy) ; } Le code ci-dessus vérifie la taille de la timeline donnée, et s'il retourne 0 (c'est-à-dire que la timeline n'a pas de moments actifs), il ajoutera une fonction de script à utiliser à un moment aléatoire dans la timeline indexée dans la variable globale "tl".
