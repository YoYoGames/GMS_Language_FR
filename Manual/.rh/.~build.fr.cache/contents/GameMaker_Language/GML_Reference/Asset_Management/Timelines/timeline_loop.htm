boucle de temps Cette variable renvoie si la ligne de temps est en boucle (true) ou non (false). Vous pouvez changer cette variable pour activer ou désactiver le bouclage et cela fonctionne avec une vitesse de ligne de temps négative (si la position de la ligne de temps descend en dessous de 0, elle recommencera au dernier moment défini). dernier moment défini). Syntaxe : timeline_loop ; Les retours : Booléen Exemple : if !timeline_loop { timeline_loop = true ; } Le code ci-dessus vérifie si la ligne de temps actuellement assignée à une instance va boucler ou non et si ce n'est pas le cas, il la met à true.
