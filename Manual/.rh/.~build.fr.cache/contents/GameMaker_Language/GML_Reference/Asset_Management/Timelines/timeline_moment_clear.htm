timeline_moment_clear Avec cette fonction, vous pouvez effacer un moment spécifique de toute ligne de temps précédemment définie de tous les codes et actions. Syntaxe:timeline_moment_clear(ind, step); Argument Description ind L'index de la ligne de temps à effacer. step Le moment de clarifier. Retour:N/A Exemple:timeline_moment_clear(global.tl, room_speed * 30); Le code ci-dessus effacera le moment spécifié de la ligne de temps indexée par la variable "global.tl".
