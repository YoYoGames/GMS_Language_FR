layer_sprite_get_index Cette fonction peut être utilisée pour obtenir la valeur d'index d'image actuelle de l'élément sprite. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), et la fonction retournera une valeur réelle qui représente l'index d'image affiché pour le sprite. La fonction retournera -1 si l'élément de sprite n'existe pas ou si l'élément n'a pas de sprite valide qui lui est affecté. Syntaxe:layer_sprite_get_index(sprite_element_id); Argument Description sprite_element_id La valeur d'ID unique de l'élément de sprite à partir duquel obtenir les informations Retour:Real (the current sprite image index or -1) Exemple:var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_index(spr_id) < 4     {     layer_sprite_index(spr_id, 4);     } Le code ci-dessus obtiendra l'ID de la couche nommée "sprite_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément sprite sur cette couche. Cet ID est ensuite utilisé pour vérifier si l'index d'image de l'élément est inférieur à 4, et si tel est le cas, il est défini sur 4.
