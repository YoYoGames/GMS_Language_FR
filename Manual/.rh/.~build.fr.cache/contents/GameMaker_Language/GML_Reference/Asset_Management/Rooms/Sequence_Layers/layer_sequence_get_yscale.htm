layer_sequence_get_yscale Avec cette fonction, vous fournissez l'ID d'élément de séquence - tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et il renverra l'échelle actuelle le long de l'axe Y de l'élément de séquence dans la salle de jeux. Syntaxe:layer_sequence_get_yscale(sequence_element_id) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler Retour:Real Exemple:var _ys = layer_sequence_get_yscale(title_sequence) if _ys < 1     {     _ys += 0.01;     layer_sequence_yscale(title_sequence, _ys);     } Le code ci-dessus récupère l'échelle actuelle le long de l'axe Y de l'élément de séquence avec l'ID stocké dans la variable "title_sequence", et s'il est inférieur à 1, alors 0,01 est ajouté à l'échelle Y actuelle.
