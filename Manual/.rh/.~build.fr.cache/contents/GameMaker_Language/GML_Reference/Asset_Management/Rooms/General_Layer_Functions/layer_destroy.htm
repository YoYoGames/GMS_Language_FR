couche_destruction Cette fonction va détruire la couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create()) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et cette fonction le supprimera de la pièce actuelle. Si le calque a été conçu dans l'éditeur de pièce, la prochaine fois que vous quitterez la pièce et y reviendrez, le calque sera recréé avec son contenu original, la couche sera supprimée à moins que la persistance de la pièce ne soit à nouveau désactivée. Lorsque vous détruisez un calque de cette manière, tout son contenu sera également supprimé, donc tout ID de référence pour les arrière-plans ou les cartes de tuiles, etc... ne seront plus valides et toute instance assignées à la couche seront détruites (en exécutant leur événement Destroy en même temps, si elles en ont un, ainsi que l'événement Clean Up). Syntaxe : layer_destroy(layer_id) Arguments Description identifiant de la couche La valeur d'identification unique de la couche à détruire (ou le nom de la couche sous forme de chaîne). Les retours : N/A Exemple : if !instance_exists(obj_Bullet_Parent) { layer_destroy(global.Bullet_Layer) ; } Le code ci-dessus vérifie si des instances de l'objet "obj_Bullet_Parent" existent, et si ce n'est pas le cas, il détruit le calque avec l'ID stocké dans la variable globale.
