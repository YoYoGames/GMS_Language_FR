couche_arrière-plan_get_speed Cette fonction peut être utilisée pour obtenir la valeur actuelle du multiplicateur de vitesse de l'élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), et la fonction renvoie une valeur réelle qui représente le multiplicateur de vitesse utilisé pour animer le sprite. La valeur par défaut est 1. Syntaxe : layer_background_get_speed(background_element_id) ; Arguments Description id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à partir duquel les informations doivent être obtenues. Les retours : Real Exemple : var lay_id = layer_get_id("Background_sky") ; var back_id = layer_background_get_id(lay_id) ; if layer_background_get_speed(back_id) > 0 { layer_background_speed(back_id, 0) ; } Le code ci-dessus récupère l'ID du calque nommé "Background_sky" et l'utilise ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier la vitesse d'animation de l'élément et s'il est supérieur à 0, il est réglé sur 0.
