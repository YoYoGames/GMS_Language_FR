layer_sequence_destroy Avec cette fonction, vous pouvez détruire (retirer de la pièce) un élément de séquence. Vous fournissez l'ID d'élément de séquence renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche et la séquence sera détruite. Syntaxe:layer_sequence_destroy(sequence_element_id) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler Retour:N/A Exemple:var a = layer_get_all_elements(layer); for (var i = 0; i < array_length(a); i++;)     {     if layer_get_element_type(a[i]) == layerelementtype_sequence         {         layer_sequence_destroy(a[i])         }     } Le code ci-dessus obtient les ID de tous les éléments affectés à la couche de l'instance exécutant le code. Le code vérifie ensuite si l'un des éléments retournés est un actif de séquence et s'il est ensuite détruit.
