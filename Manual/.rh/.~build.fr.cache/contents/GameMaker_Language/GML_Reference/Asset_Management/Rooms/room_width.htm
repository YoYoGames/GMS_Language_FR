room_width Cette variable contient la largeur de la pièce actuelle en pixels. Vous pouvez modifier cette variable pour modifier la largeur de la pièce à tout moment. Syntaxe:room_width; Retour:Real Exemple:if bbox_right > room_width     {     x += room_width - bbox_right;     } Le code ci-dessus vérifie si le cadre de délimitation du sprite de l'instance actuelle est supérieur à la largeur de la pièce, et si c'est le cas, il déplace l'instance vers le haut.
