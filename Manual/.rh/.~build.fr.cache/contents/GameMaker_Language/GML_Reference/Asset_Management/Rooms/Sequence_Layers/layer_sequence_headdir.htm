layer_sequence_headdir Avec cette fonction, vous pouvez définir la direction de la tête de lecture de séquence donnée. Vous fournissez l'ID d'élément de séquence renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de calque, puis indiquez la direction de la tête de lecture qui doit être l'une des constantes suivantes: Constant Description Évaluer seq_dir_right La séquence lira les images dans un ordre incrémentiel de gauche à droite 1 seq_dir_left La séquence jouera les images dans un ordre décrémentiel de droite à gauche -1 Syntaxe:layer_sequence_headdir(sequence_element_id, direction) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler direction La direction de la tête de lecture, une constante, répertoriée ci-dessus Retour:N/A Exemple:var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); layer_sequence_headdir(_seq, seq_dir_left); Le code ci-dessus crée une nouvelle séquence et stocke son ID dans une variable locale. Cet ID est ensuite utilisé pour définir la direction de la tête de lecture pour décrémenter les images (lecture de droite à gauche).
