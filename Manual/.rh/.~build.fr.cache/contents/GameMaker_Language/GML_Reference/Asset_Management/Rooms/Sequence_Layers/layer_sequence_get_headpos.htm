layer_sequence_get_headpos Avec cette fonction, vous fournissez l'ID d'élément de séquence - tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de calque - et il renverra la position actuelle de la tête de lecture (l'image actuelle sur laquelle se trouve la tête de lecture). Syntaxe:layer_sequence_get_headpos(sequence_element_id) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler Retour:Real Exemple:if layer_sequence_get_headpos(title_sequence) != 0     {     layer_sequence_headpos(title_sequence, 0);     } Le code ci-dessus vérifie la position actuelle de la tête de lecture de l'ID d'élément de séquence stocké dans la variable "title_sequence", et s'il n'est pas défini sur 0, il est défini sur cette valeur.
