couche_instance_get_instance Cette fonction peut être utilisée pour obtenir l'ID d'instance unique de l'élément d'instance donné. Vous donnez l'ID de l'élément d' instance (voir l'exemple de code ci-dessous pour savoir comment l'obtenir), et la fonction renvoie une valeur réelle qui représente l'unique unique de l'élément. Si l'élément n'est pas une instance, la fonction renvoie -1. Syntaxe : layer_instance_get_instance(element_id) Arguments Description identifiant de l'élément La valeur d'ID unique de l'élément d'instance dont on veut obtenir l'ID Les retours : Real (valeur d'identification unique pour une instance) Exemple : elements = layer_get_all_elements("Instances") ; for (var i = 0 ; i < array_length(elements) ; i++) { si (layer_get_element_type(elements[i]) == layerelementtype_instance) { var layerelement = elements[i] ; var inst = layer_instance_get_instance(layerelement) ; inst.x = inst.x + 10 ; } } Le code ci-dessus vérifie tous les éléments d'instance sur un calque, puis récupère leur valeur d'identification unique et l'utilise pour les déplacer de 10px vers la droite.
