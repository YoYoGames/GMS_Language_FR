couche_sprite_get_angle Cette fonction peut être utilisée pour obtenir l'angle actuel de l'élément sprite. Vous donnez l'ID de l'élément sprite (obtenu lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction fonction layer_sprite_get_id()), et la fonction renvoie une valeur réelle pour l'angle. Cette valeur sera comprise entre 0 et 360, où 0 représente la droite, 90 le haut, 180 la gauche et 270 le bas. Syntaxe : layer_sprite_get_angle(sprite_element_id) ; Arguments Description sprite_element_id La valeur d'identification unique de l'élément de sprite à partir duquel les informations doivent être obtenues. Les retours : Real Exemple : var lay_id = layer_get_id("sprite_sky") ; var spr_id = layer_sprite_get_id(lay_id, "Clouds") ; if layer_sprite_get_angle(spr_id) != 0 { layer_sprite_yscale(spr_id, 0) ; } Le code ci-dessus récupère l'ID du calque nommé "sprite_sky" et l'utilise pour obtenir l'ID de l'élément sprite sur ce calque. Cet ID est ensuite utilisé pour vérifier l'angle et s'il est différent de 0, il est mis à 0.
