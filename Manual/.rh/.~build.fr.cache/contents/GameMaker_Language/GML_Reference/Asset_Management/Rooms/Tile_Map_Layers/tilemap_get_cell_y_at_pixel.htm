tilemap_get_cell_y_at_pixel En utilisant cette fonction, vous pouvez récupérer la position de l'axe y d'une cellule individuelle de la carte de tuiles en donnant la position relative de l'axe y dans la pièce. Vous donnez l'ID d'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id() ), ainsi que les positions x et y dans la pièce et la fonction renverra la position y de la cellule dans la carte de tuiles pour ce point. Notez que si la valeur est en dehors de la zone de la carte de tuiles et qu'aucune cellule n'est disponible, elle renverra -1. Syntaxe:tilemap_get_cell_y_at_pixel(tilemap_element_id, x, y); Argument Description tilemap_element_id La valeur d'ID unique de l'élément de mappage de tuiles pour obtenir la position y de la cellule x La position x dans la pièce à utiliser pour obtenir la cellule y La position y dans la pièce à utiliser pour obtenir la cellule Retour:Integer (y-axis cell position or -1 if there is an error) Exemple:var lay_id = layer_get_id("Tiles_Walls"); var map_id = layer_tilemap_get_id(lay_id); var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); tiledata = tilemap_get(map_id, _x, _y); Le code ci-dessus utilise l'ID de mappage de tuiles récupéré pour obtenir la position de cellule à partir d'un ensemble de coordonnées de pièce, puis stocker les données de toute tuile qui s'y trouve dans une variable d'instance.
