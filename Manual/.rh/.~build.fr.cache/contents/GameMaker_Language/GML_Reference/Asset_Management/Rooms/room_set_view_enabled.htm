room_set_view_enabled Avec cette fonction, vous pouvez activer (true ) ou désactiver (false ) la vue de n'importe quelle pièce de votre jeu, à l'exception de celle actuelle. Syntaxe:room_set_view_enabled(index, val); Argument Description index L'index de la pièce à définir. val Activer ou non (true ) ou désactiver (false ) vues dans la pièce donnée. Retour:N/A Exemple:global.myroom = room_add(); room_set_view_enabled(global.myroom, true); Cela activera les vues dans la salle indexée dans "global.myroom".
