room_set_view_enabled Cette fonction vous permet d'activer(true) ou de désactiver(false) l'affichage de n'importe quelle pièce de votre jeu , sauf la pièce actuelle. Syntaxe : room_set_view_enabled(index, val) ; Arguments Description indice L'index de la pièce à définir. val Permet d'activer(true) ou de désactiver(false) les vues dans la pièce donnée. Les retours : N/A Exemple : global.myroom = room_add() ; room_set_view_enabled(global.myroom, true) ; Cela permettra d'afficher des vues dans la salle indexée dans "global.myroom".
