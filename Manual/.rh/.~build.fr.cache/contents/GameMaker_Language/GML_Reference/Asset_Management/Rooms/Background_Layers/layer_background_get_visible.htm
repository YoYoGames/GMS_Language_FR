couche_arrière-plan_get_visible Cette fonction peut être utilisée pour obtenir l'état visible de l'élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), et la fonction renvoie soit true si l'élément est actuellement visible, soit false s'il ne l'est pas. Notez que cette valeur de retour n' est pas affectée par le fait que le calque sur lequel se trouve l'élément soit visible ou non. Syntaxe : layer_background_get_visible(background_element_id) ; Arguments Description id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à partir duquel les informations doivent être obtenues. Les retours : Booléen Exemple : var lay_id = layer_get_id("Background_sky") ; var back_id = layer_background_get_id(lay_id) ; if layer_background_get_visible(back_id) { layer_background_visible(back_id, false) ; } Le code ci-dessus obtiendra l'ID du calque nommé "Background_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier la visibilité de l'élément et s'il est visible, alors cette propriété est définie comme fausse.
