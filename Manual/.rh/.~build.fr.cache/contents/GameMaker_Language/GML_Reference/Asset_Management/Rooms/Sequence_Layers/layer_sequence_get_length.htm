séquence_de_couche_get_length Avec cette fonction, vous fournissez l'ID de l'élément de la séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle renvoie la longueur de la séquence. Il s'agit du nombre d'images pendant lesquelles la séquence sera exécutée. Syntaxe : layer_sequence_get_length(sequence_element_id) Arguments Description id_élément_de_séquence La valeur d'identification unique de l'élément de séquence à cibler. Les retours : Entier Exemple : var _frames = layer_sequence_get_length(my_seq) ; alarm[0] = frames ; Le code ci-dessus récupère le nombre d'images qu'une séquence va exécuter puis utilise cette valeur pour déclencher une alarme.
