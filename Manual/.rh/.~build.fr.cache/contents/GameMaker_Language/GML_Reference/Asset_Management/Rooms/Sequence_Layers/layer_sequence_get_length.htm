layer_sequence_get_length Avec cette fonction, vous fournissez l'ID d'élément de séquence - tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et il renverra la longueur de la séquence. Il s'agit du nombre d'images pendant lesquelles la séquence s'exécutera. Syntaxe:layer_sequence_get_length(sequence_element_id) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler Retour:Integer Exemple:var _frames = layer_sequence_get_length(my_seq); alarm[0] = frames; Le code ci-dessus récupère le nombre d'images qu'une séquence exécutera puis utilise cette valeur pour définir une alarme.
