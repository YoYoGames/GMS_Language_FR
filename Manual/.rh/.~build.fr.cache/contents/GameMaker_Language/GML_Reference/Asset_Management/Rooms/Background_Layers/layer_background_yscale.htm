couche_arrière-plan_échelle Cette fonction peut être utilisée pour définir l'échelle le long de l'axe des y d'un élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis vous définissez la valeur de l'échelle. La valeur d'échelle que vous donnez est un multiplicateur qui sera utilisé pour modifier la façon dont l'élément d'arrière-plan est affiché, où une valeur de 0,5 affiche l'élément à la moitié de son échelle, et une valeur de 2 l'affiche à l'échelle double. Notez que les valeurs négatives sont valables, et qu'elles " retourneront " l'élément autour de la position (0,0), donc une échelle x de -1 affichera l'image inversée. l'image inversée. Syntaxe : layer_background_yscale(background_element_id, yscale) ; Arguments Description id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à modifier. vitesse Le multiplicateur/valeur de vitesse à utiliser pour l'élément. Les retours : N/A Exemple : var lay_id = layer_get_id("Background_water") ; var back_id = layer_background_get_id(lay_id) ; layer_background_yscale(back_id, -1) ; Le code ci-dessus obtiendra l'ID du calque nommé "Background_water" et l'utilisera ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque.
