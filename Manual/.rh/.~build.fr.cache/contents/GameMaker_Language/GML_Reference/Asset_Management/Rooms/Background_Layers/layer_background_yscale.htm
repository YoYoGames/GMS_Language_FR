layer_background_yscale Cette fonction peut être utilisée pour définir l'échelle le long de l'axe y d'un élément d'arrière-plan. Vous donnez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), puis définissez la valeur d'échelle. La valeur d'échelle que vous donnez est un multiplicateur qui sera utilisé pour changer la façon dont l'élément d'arrière-plan est affiché, où une valeur de 0,5 afficherait l'élément à demi-échelle et une valeur de 2 s'afficherait à double échelle. Notez que les valeurs négatives sont valides et "retournent" l'élément autour de la position (0,0), donc une échelle x de -1 afficherait l'image inversée. Syntaxe:layer_background_yscale(background_element_id, yscale); Argument Description background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à modifier speed Le multiplicateur de vitesse / valeur à utiliser pour l'élément Retour:N/A Exemple:var lay_id = layer_get_id("Background_water"); var back_id = layer_background_get_id(lay_id); layer_background_yscale(back_id, -1); Le code ci-dessus obtiendra l'ID de la couche nommée "Background_water", puis l'utilisera pour obtenir l'ID de l'élément d'arrière-plan sur cette couche.
