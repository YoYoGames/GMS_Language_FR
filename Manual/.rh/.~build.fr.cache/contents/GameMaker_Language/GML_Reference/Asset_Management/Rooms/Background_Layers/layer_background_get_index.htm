layer_background_get_index Cette fonction peut être utilisée pour obtenir la valeur d'index d'image actuelle de l'élément d'arrière-plan. Vous donnez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), et la fonction retournera une valeur réelle qui représente l'index d'image affiché pour le sprite. La fonction retournera -1 si l'élément d'arrière-plan n'existe pas ou si l'élément n'a pas de sprite valide qui lui est affecté. Syntaxe:layer_background_get_index(background_element_id); Argument Description background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à partir duquel obtenir les informations Retour:Real (the current sprite image index or -1) Exemple:var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_index(back_id) < 4     {     layer_background_index(back_id, 4);     } Le code ci-dessus obtiendra l'ID de la couche nommée "Background_sky", puis l'utilisera pour obtenir l'ID de l'élément d'arrière-plan sur cette couche. Cet ID est ensuite utilisé pour vérifier si l'index d'image de l'élément est inférieur à 4, et si tel est le cas, il est défini sur 4.
