chambre_get_viewport Avec cette fonction, vous pouvez récupérer les détails d'un hublot dans une pièce autre que la pièce actuelle. Vous donnez l'ID de la pièce et l'index du hublot à récupérer (de 0 à 7) et la fonction renvoie un tableau de 5 index, où.. : 0] = visible (booléen : vrai / faux) [1] = position x (réelle) [2] = position y (réelle) [3] = largeur (réelle) 4] = hauteur (réelle) Syntaxe : room_get_viewport(rm, vind) ; Arguments Description rm L'index de la pièce pour obtenir les données de la fenêtre d'affichage. vind L'index du port de vue à obtenir Les retours : Tableau (5 éléments : visible, x, y, largeur, hauteur) Exemple : v_vals = room_get_viewport(rm_Game, 0) ; if v_vals[0] == false { room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]) ; } Le code ci-dessus récupère les données du port d'observation pour la pièce donnée, puis vérifie si le port est marqué comme visible. S'il ne l'est pas, les données du port d'observation sont définies pour le rendre visible. Retour : Pièces Suivant : room_set_camera Copyright YoYo Games Ltd. 2021 Tous droits réservés
