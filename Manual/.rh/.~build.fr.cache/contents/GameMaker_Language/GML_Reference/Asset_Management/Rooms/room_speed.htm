room_speed (obsolète) Cette variable contient la vitesse d'exécution de toutes les pièces (et du jeu) en images par seconde. Notez qu'il ne s'agit PAS du FPS (images par seconde), mais plutôt du nombre d'étapes de jeu que GameMaker Studio 2 essaiera de maintenir à chaque seconde. seconde. IMPORTANT ! Cette variable est maintenue pour le support Legacy uniquement, et ne doit pas être utilisée car elle ne définit plus la vitesse pour une seule pièce, mais pour TOUTES les pièces du jeu. Pour changer la vitesse du jeu, vous devriez plutôt utiliser la fonction game_set_speed(). Syntaxe : Vitesse de la pièce ; Les retours : Real Exemple : alarme[0] = room_speed * 10 ; Le code ci-dessus utilise room_speed pour régler une alarme sur 10 secondes. En utilisant la variable de cette manière, l'alarme fonctionnera pendant 10 secondes, quelle que soit la vitesse définie pour la pièce.
