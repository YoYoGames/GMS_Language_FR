layer_sprite_index Cette fonction peut être utilisée pour définir l'index d'image d'un élément de sprite qui a plusieurs sous-images sur un calque. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), puis définissez l'index d'image à utiliser. Si vous définissez une valeur en dehors de la plage des sous-images, l'index de l'image sera en boucle. Syntaxe:layer_sprite_index(sprite_element_id, image_index); Argument Description sprite_element_id La valeur d'ID unique de l'élément de sprite à définir index L'index d'image à utiliser pour le sprite Retour:N/A Exemple:var lay_id = layer_get_id("sprite_trees"); var spr_id = layer_sprite_get_id(lay_id, "gfc_trees"); layer_sprite_index(spr_id, 1); Le code ci-dessus obtiendra l'ID de la couche nommée "sprite_trees" et l'utilisera ensuite pour obtenir l'ID de l'élément de sprite donné sur cette couche. Cet identifiant est ensuite utilisé pour modifier l'index d'image de l'élément. Retour: Couches de sprite Suivant:layer_sprite_speed © Copyright YoYo Games Ltd. 2021 Tous droits réservés
