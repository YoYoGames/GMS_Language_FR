layer_background_get_yscale Cette fonction peut être utilisée pour obtenir la valeur actuelle du multiplicateur d'échelle de l'élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), et la fonction renvoie une valeur réelle qui représente le multiplicateur d'échelle utilisé pour dessiner le sprite. La valeur par défaut est 1. Syntaxe : layer_background_get_yscale(background_element_id) ; Arguments Description id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à partir duquel les informations doivent être obtenues. Les retours : Real Exemple : var lay_id = layer_get_id("Background_sky") ; var back_id = layer_background_get_id(lay_id) ; if layer_background_get_yscale(back_id) != -1 { layer_background_yscale(back_id, -1) ; } Le code ci-dessus récupère l'ID du calque nommé "Background_sky" et l'utilise ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier l'échelle y de l'élément et si elle n'est pas égale à -1, elle est fixée à -1.
