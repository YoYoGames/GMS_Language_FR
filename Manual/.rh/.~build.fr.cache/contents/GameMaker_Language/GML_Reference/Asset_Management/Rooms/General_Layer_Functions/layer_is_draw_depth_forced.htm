layer_is_draw_depth_forced Cette fonction peut être utilisée pour vérifier et voir si la profondeur Z forcée est activée pour le rendu des couches dans le projet. Voir layer_force_draw_depth() pour plus d'informations. Syntaxe:layer_is_draw_depth_forced() Retour:Boolean Exemple:if !layer_is_draw_depth_forced()    {    layer_force_draw_depth(true, 0);    } Le code ci-dessus vérifie si la profondeur Z de la couche est forcée ou non et si ce n'est pas le cas, il définit la profondeur Z sur 0 et l'active.
