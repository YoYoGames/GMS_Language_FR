tilemap_get_tileset Cette fonction vous permet de récupérer la valeur de l'index de la ressource de l'ensemble de tuiles affectée à un élément de carte de tuiles donné sur une couche. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la fonction renvoie la valeur de l'index du jeu de tuiles. Syntaxe : tilemap_get_tileset(tilemap_element_id) ; Arguments Description tilemap_element_id La valeur d'identification unique de l'élément de la carte des tuiles pour obtenir l'ensemble de tuiles à partir de Les retours : Valeur de l'index de la tuile (Real) Exemple : var lay_id = layer_get_id("Tiles_trees") ; var map_id = layer_tilemap_get_id(lay_id) ; if tilemap_get_tileset(map_id) != ts_Nighttime { tilemap_tileset(map_id, ts_Nighttime) ; } Le code ci-dessus vérifie le jeu de tuiles actuel attribué à la couche "Tiles_trees" et si ce n'est pas "ts_Nighttime", alors ce jeu de tuiles est attribué à la carte des tuiles.
