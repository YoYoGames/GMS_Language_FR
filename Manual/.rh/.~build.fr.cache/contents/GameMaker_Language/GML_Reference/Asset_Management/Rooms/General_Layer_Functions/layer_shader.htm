layer_shader Avec cette fonction, vous pouvez affecter une ressource de shaders à un calque donné et le calque sera alors rendu en utilisant ce shader. Vous fournissez soit la valeur d'identification unique de la couche, soit le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) ainsi que l'ID du shader à utiliser. Le shader doit avoir été créé précédemment dans l'Asset Browser et l'index du shader (le nom de la ressource du shader ) est ensuite transmis à cette fonction. Si des instances ont été ajoutées à la couche attribuée, alors le shader sera appliqué à tous les événements de dessin que l'instance utilise - par exemple, si l'instance a un événement Draw GUI Begin, le shader lui sera appliqué automatiquement. Le shader affectera également tout autre élément graphique dessiné sur cette couche, comme les sprites. sur cette couche, comme les sprites ou les cartes de tuiles. Notez que la fonction n'est pas censée être appelée dans des événements de dessin ou des événements d'étape, mais qu'elle ne doit être appelée qu'au début de la pièce dans le code de création de pièce ou dans l'événement de création / de début de pièce d'une instance. Syntaxe : layer_shader(layer_id, shader) Arguments Description identifiant de la couche La valeur d'identification unique de la couche à cibler (ou le nom de la couche sous forme de chaîne). nuanceur L'index du shader à affecter à la couche Les retours : N/A Exemple : var lay_id = layer_get_id("Instances") ; layer_shader(lay_id, shd_Sepia) ; Le code ci-dessus assignera la ressource shader "shd_Sepia" à la couche donnée pour tous les dessins.
