tuile_set_flip Cette fonction peut être utilisée pour définir un ensemble donné de données de tuiles pour retourner la tuile ou non. Vous donnez les données des tuiles, qui peuvent être récupérées en utilisant la fonction tilemap_get()puis définissez l'argument flip à true (vrai) si vous voulez que la tuile soit retournée, ou false (faux) si vous voulez que la tuile soit dans son état par défaut, non retournée. La fonction renvoie un ensemble de données de tuiles modifié qui peut ensuite être appliqué à l'aide de la fonction tilemap_set() fonction. Syntaxe : tile_set_flip(tiledata, flip) Arguments Description données en mosaïque les données de la tuile à définir flip Définissez-le à true pour le retourner et false pour le laisser tel quel. Les retours : Réel (tiledata) Exemple : var lay_id = layer_get_id("Tiles_sky") ; var map_id = layer_tilemap_get_id(lay_id) ; var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y) ; var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y) ; var data = tilemap_get(map_id, mx, my) ; var bool = !tile_get_flip(data) ; data = tile_set_flip(data, bool) ; tilemap_set(map_id, data, mx, my) ; Le code ci-dessus récupère l'ID de la carte des tuiles à partir de la couche donnée, puis obtient la position x et y de la cellule de la tuile sous la souris. Cette position est ensuite utilisée pour obtenir les données de la tuile qui sont retournées et ensuite utilisées pour définir à nouveau la tuile.
