layer_background_speed Cette fonction peut être utilisée pour définir la vitesse d'un sprite animé qui a été affecté à un élément d'arrière-plan. Vous donnez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), puis définissez la vitesse de l'animation. Si l'EDI a défini la vitesse d'animation pour le sprite utilisée pour être calculée comme des images par image de jeu, alors la valeur de vitesse donnée est un multiplicateur, 1 étant la valeur par défaut, et le définir sur 0,5 entraînera la moitié de la vitesse d'animation, lors du réglage il à 2 le doublera. Cependant, si l'EDI a défini la vitesse d'animation pour le sprite utilisée pour être calculée en images par seconde, alors la valeur de vitesse donnée est une valeur littérale d'images par seconde, donc le définir sur 5, par exemple, mettra l'animation en boucle 5 images par seconde que le jeu s'exécute. Si l'élément d'arrière-plan n'a pas de sprite attaché ou si le sprite utilisé n'a pas de sous-images, cette fonction n'aura aucun effet. Syntaxe:layer_background_speed(background_element_id, speed); Argument Description background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à modifier speed Le multiplicateur de vitesse / valeur à utiliser pour l'élément Retour:N/A Exemple:var lay_id = layer_get_id("Background_water"); var back_id = layer_background_get_id(lay_id); layer_background_speed(back_id, 0.5); Le code ci-dessus obtiendra l'ID de la couche nommée "Background_water", puis l'utilisera pour obtenir l'ID de l'élément d'arrière-plan sur cette couche. Cet identifiant est ensuite utilisé pour modifier la vitesse d'image de l'élément. Retour: Calques d'arrière-plan Suivant:layer_background_xscale © Copyright YoYo Games Ltd. 2021 Tous droits réservés
