layer_background_vtiled En utilisant cette fonction, vous pouvez changer si l'élément d'arrière-plan donné sur une couche doit être empilé verticalement ou non. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis définissez la valeur du tuilage. Si la valeur true est définie, l'élément sera affiché en mosaïque verticale dans la pièce, et si la valeur false est définie, l'élément ne sera pas disposé en mosaïque. Cette fonction est réservée aux projets 2D et ne fonctionnera pas correctement en cas de projection avec une caméra 3D. Syntaxe : layer_background_vtiled(background_element_id, tiled) ; Arguments Description id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à modifier. visible La bascule de l'inclinaison verticale, qui peut être vraie ou fausse. Les retours : N/A Exemple : var back = layer_background_get_id(layer) ; if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back) { layer_background_htiled(back, true) ; layer_background_vtiled(back, true) ; } Le code ci-dessus vérifiera l'élément d'arrière-plan attribué au calque sur lequel se trouve l'instance qui exécute le code et s'il n'est pas tuilé dans un sens ou dans l'autre, il le configurera pour qu'il soit tuilé verticalement et horizontalement.
