layer_sequence_get_xscale Avec cette fonction, vous fournissez l'ID d'élément de séquence - tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et il renverra l'échelle actuelle le long de l'axe X de l'élément de séquence dans la salle de jeux. Syntaxe:layer_sequence_get_xscale(sequence_element_id) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler Retour:Real Exemple:var _xs = layer_sequence_get_xscale(title_sequence) if _xs < 1     {     _xs += 0.01;     layer_sequence_xscale(title_sequence, _xs);     } Le code ci-dessus récupère l'échelle actuelle le long de l'axe X de l'élément de séquence avec l'ID stocké dans la variable "title_sequence", et s'il est inférieur à 1, alors 0,01 est ajouté à l'échelle X actuelle.
