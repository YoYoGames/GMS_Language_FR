Couches de séquence Dans cette section, nous avons les fonctions qui sont utilisées pour ajouter, supprimer et modifier le comportement des séquences dans une salle de jeu par rapport au calque sur lequel elles se trouvent. Il convient de noter que les séquences sont similaires aux objets, en ce que lorsqu'une séquence est créée sur un calque dans une pièce, elle est considérée comme une instance de l'objet séquence principal à partir du navigateur de ressources. Ainsi, lorsque vous ajoutez une séquence à un calque dans une pièce, vous créez une instance de l'objet de séquence de base, et cette instance de séquence est ajoutée en tant qu'élément dans un calque de pièce. L'instance de séquence est ce qui contrôle des choses comme la vitesse et la direction de lecture, et contiendra elle-même une autre structure de données de séquence qui contient les données de piste de séquence réelles. Ceci est important, car les fonctions répertoriées ci-dessous référenceront l' ID d'élément de séquence (l'ID de la séquence sur la couche ) ainsi que l' ID d'instance de séquence (l'ID réel de la séquence référencée par l'élément) et la séquence data struct (qui contient toutes les données de séquence).
