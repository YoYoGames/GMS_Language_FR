room_set_persistent Avec cette fonction, vous pouvez modifier (ou définir) la persistance de n'importe quelle pièce de votre jeu à l'exception de celle actuelle. Une salle avec persistance marquée comme true il maintiendra l'état de toutes les instances dans cette salle si le joueur quitte puis revient, alors que si la persistance est marquée comme false il sera remis à l'état initial à chaque fois. Vous devez noter qu'une salle persistante utilise beaucoup plus de mémoire qu'une salle normale et il n'est pas recommandé d'en avoir trop dans votre jeu. REMARQUE: Cette fonction ne fonctionnera PAS pour désactiver la persistance si la pièce a déjà été visitée! Une salle persistante, une fois visitée, est conservée en mémoire et pour désactiver la persistance, vous devez vous rendre dans cette pièce et définir la variable room_persistent sur false puis quittez à nouveau la pièce. Syntaxe:room_set_persistent(index, val); Argument Description index L'index de la pièce dont vous souhaitez définir la persistance. val Si la pièce doit être persistante (true ) ou non (false ). Retour:N/A Exemple:global.myroom = room_add(); room_set_width(global.myroom, 640); room_set_height(global.myroom, 480); room_set_persistent(global.myroom, true); Cela créera une nouvelle salle et stockera son index dans la variable "global.myroom". Il fixera alors sa largeur à 640 pixels, sa hauteur à 480 pixels, sa légende à 'Game Room' et sa persistance à 'true'.
