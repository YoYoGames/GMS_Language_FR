layer_sprite_get_yscale Cette fonction peut être utilisée pour obtenir le multiplicateur d'échelle actuel sur l'axe des y pour l'élément sprite. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), et la fonction renvoie une valeur réelle pour l'échelle y, où 1 correspond à une absence d'échelle. Syntaxe : layer_sprite_get_yscale(sprite_element_id) ; Arguments Description sprite_element_id La valeur d'identification unique de l'élément de sprite à partir duquel les informations doivent être obtenues. Les retours : Real Exemple : var lay_id = layer_get_id("sprite_sky") ; var spr_id = layer_sprite_get_id(lay_id, "Clouds") ; if layer_sprite_get_yscale(spr_id) != 1 { layer_sprite_yscale(spr_id, 1) ; } Le code ci-dessus récupère l'ID du calque nommé "sprite_sky" et l'utilise pour obtenir l'ID de l'élément sprite sur ce calque. Cet ID est ensuite utilisé pour vérifier la valeur yscale et si elle est différente de 1, elle est fixée à 1.
