tuile_set_empty Cette fonction peut être utilisée pour définir un ensemble donné de données de tuiles comme étant une tuile vide. Vous donnez les données des tuiles, qui peuvent être récupérées en utilisant la fonction tilemap_get()et la fonction retournera true si la est vide, ou faux s'il y a un index de tuile. La fonction renvoie un ensemble modifié de données de tuiles qui peut ensuite être appliqué à l'aide de la fonction tilemap_set() fonction. Syntaxe : tile_set_empty(tiledata) Arguments Description données en mosaïque les données de la tuile à définir Les retours : Réel (tiledata) Exemple : var lay_id = layer_get_id("Tiles_sky") ; var map_id = layer_tilemap_get_id(lay_id) ; for (var i = 0 ; i < tilemap_get_width(map_id) ; i++ ;) { for (var j = 0 ; j < tilemap_get_height(map_id) ; j++ ;) { var data = tilemap_get(map_id, i, j) ; if !tile_get_empty(data) { data = tile_set_empty(data) tilemap_set(map_id, data, i, j) ; } } } Le code ci-dessus récupère l'ID de la carte des tuiles à partir de la couche donnée, puis vérifie chaque cellule de la carte pour voir si elle contient des données ou non. Si c'est le cas, la tuile est définie comme vide. Retour : Carrelage des couches de la carte Suivant : tile_set_index Copyright YoYo Games Ltd. 2021 Tous droits réservés
