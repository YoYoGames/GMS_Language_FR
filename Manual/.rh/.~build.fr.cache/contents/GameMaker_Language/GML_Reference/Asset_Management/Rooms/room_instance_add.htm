room_instance_add Avec cette fonction, vous pouvez ajouter une instance dans n'importe quelle pièce autre que celle actuelle et à n'importe quelle position dans cette pièce. La fonction renvoie l'unique id de l'instance qui peut ensuite être utilisée dans d'autres appels de fonction pour définir les propriétés, etc. de cette instance, mais seulement une fois que le jeu est entré dans la salle spécifiée. Si vous souhaitez créer une instance dans la salle actuelle, vous devez utiliser la fonction instance_create_layer(). Notez que l'appel de cette fonction sur un élément de salle créé dans le navigateur d'actifs ajoutera de façon permanente l'instance à la salle, et même en appelant game_restart() ne ramènera pas la salle à son état d'origine (le seul fait de terminer le jeu et de l'ouvrir à nouveau recommencera avec la salle dans son état d'origine). Syntaxe:room_instance_add(index, x, y, obj); Argument Description index L'index de la salle à laquelle ajouter une instance d'objet. x La position x de la nouvelle instance. y La position y de la nouvelle instance. obj L'objet auquel ajouter une instance. Retour:Instance ID Exemple:global.rm = room_add(); room_assign(rm_Base, global.rm); room_instance_add(global.rm, 100, 100, obj_Player); Le code ci-dessus ajoutera une nouvelle salle au jeu et y copiera ensuite le contenu de la salle indexée comme "rm_Base". Il ajoutera ensuite une instance de l'objet "obj_player" à la position 100,100 de cette nouvelle salle.
