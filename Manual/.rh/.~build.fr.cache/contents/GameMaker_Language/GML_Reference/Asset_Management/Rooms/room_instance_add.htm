chambre_instance_add Avec cette fonction, vous pouvez ajouter une instance dans n'importe quelle pièce autre que la pièce actuelle et à n'importe quelle position dans cette pièce. La fonction renvoie l'unique id unique de l'instance qui peut alors être utilisé dans d'autres appels de fonction pour définir les propriétés etc... de cette instance, mais seulement une fois que le jeu est entré dans la room spécifiée. Si vous souhaitez créer une instance dans le room actuel, vous devez utiliser la fonction instance_create_layer(). Notez que l'appel de cette fonction sur un asset de room créé dans l'Asset Browser ajoutera de façon permanente l'instance à la room, et que même l'appel de game_restart() ne ramènera pas la room à son état d'origine. (seule la fin du jeu et sa réouverture permettront de remettre la room dans son état d'origine). Syntaxe : room_instance_add(index, x, y, obj) ; Arguments Description indice L'index de la pièce à laquelle ajouter une instance d'objet. x La position x de la nouvelle instance. y La position y de la nouvelle instance. obj. L'objet dont il faut ajouter une instance. Les retours : ID de l'instance Exemple : global.rm = room_add() ; room_assign(rm_Base, global.rm) ; room_instance_add(global.rm, 100, 100, obj_Player) ; Le code ci-dessus va ajouter une nouvelle pièce au jeu et y copier le contenu de la pièce indexée comme "rm_Base". Il ajoutera ensuite une instance de l'objet "obj_player" à la position 100,100 de cette nouvelle pièce.
