room_set_camera Cette fonction permet d'affecter une caméra à une fenêtre spécifique dans une pièce autre que la pièce actuelle. Vous fournissez l'index de la pièce, l'index de la vue (de 0 à 7) et ensuite l'index de la caméra à utiliser. Syntaxe : room_set_camera(rm, vind, camera) ; Arguments Description rm L'index de la pièce dans laquelle la caméra doit être installée. vind L'index du port de vue auquel la caméra doit être assignée. caméra L'index de la caméra à affecter Les retours : N/A Exemple : global.myroom = room_add() ; room_set_camera(global.myroom, 0, global.MainCam) ; Le code ci-dessus attribue une caméra dans une pièce nouvellement créée au port de vue [0].
