room_set_camera Avec cette fonction, vous pouvez affecter une caméra à une fenêtre spécifique dans une pièce autre que celle actuelle. Vous fournissez l'index de la pièce, l'index de vue (de 0 à 7) puis l'index de la caméra à utiliser. Syntaxe:room_set_camera(rm, vind, camera); Argument Description rm L'index de la pièce pour régler la caméra de vue de vind L'index du port de vue auquel attribuer la caméra camera L'index de la caméra à attribuer Retour:N/A Exemple:global.myroom = room_add(); room_set_camera(global.myroom, 0, global.MainCam); Le code ci-dessus affecte une caméra dans une pièce nouvellement créée pour afficher le port [0].
