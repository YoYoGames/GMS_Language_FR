layer_sprite_blend Cette fonction contrôle le mélange (ou "teinte") du sprite sprite et la valeur par défaut est -1 (qui représente la constante c_white, qui peut également être utilisé). Toute autre valeur (y compris les constantes de couleur internes comme c_red, ou c_aqua ) mélangera la couleur spécifiée avec le sprite d'origine. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), puis définissez la couleur de fusion à utiliser. Ci-dessous, vous pouvez voir un exemple de sprite qui a été mélangé avec différentes couleurs: Veuillez noter que vous devriez essayer de limiter le mélange sur la plate-forme HTML5 (sauf si vous utilisez WebGL), car chaque sprite mélangé doit être mis en cache séparément et ainsi avoir de nombreux sprites mélangés peut affecter les performances (vous pouvez également définir la taille du cache à l'aide de la fonction sprite_set_cache_size() ). Syntaxe:layer_sprite_blend(sprite_element_id, blend); Argument Description sprite_element_id La valeur d'ID unique de l'élément de sprite à modifier blend La couleur à mélanger avec l'image-objet (la valeur par défaut est c_white ) Retour:N/A Exemple:var lay_id = layer_get_id("Asset_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); layer_sprite_blend(spr_id, c_gray); Le code ci-dessus obtient la valeur d'ID du sprite appelé "Clouds" assigné au calque "Asset_sky", puis la teinte en une couleur.
