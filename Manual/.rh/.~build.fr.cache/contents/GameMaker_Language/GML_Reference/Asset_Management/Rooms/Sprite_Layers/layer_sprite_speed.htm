layer_sprite_speed Cette fonction peut être utilisée pour définir la vitesse d'une image animée qui a été affectée à un élément de sprite. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), puis définissez la vitesse de l'animation. Si l'EDI a défini la vitesse d'animation pour le sprite utilisée pour être calculée comme des images par image de jeu, alors la valeur de vitesse donnée est un multiplicateur, 1 étant la valeur par défaut, et le définir sur 0,5 entraînera la moitié de la vitesse d'animation, lors du réglage il à 1,5 le doublera. Cependant, si l'EDI a défini la vitesse d'animation pour le sprite utilisée pour être calculée en images par seconde, alors la valeur de vitesse donnée est une valeur littérale d'images par seconde, donc le définir sur 5, par exemple, mettra l'animation en boucle 5 images par seconde que le jeu s'exécute. Si le sprite utilisé n'a pas de sous-images, cette fonction n'aura aucun effet. Syntaxe:layer_sprite_speed(sprite_element_id, speed); Argument Description sprite_element_id La valeur d'ID unique de l'élément de sprite à modifier speed Le multiplicateur de vitesse / valeur à utiliser pour l'élément Retour:N/A Exemple:var lay_id = layer_get_id("Assets_water"); var spr_id = layer_sprite_get_id(lay_id, "gfc_water"); layer_sprite_speed(spr_id, 0.5); Le code ci-dessus obtiendra l'ID de la couche nommée "Assets_water" et l'utilisera ensuite pour obtenir l'ID de l'élément de sprite donné sur cette couche.
