couche_sprite_get_alpha Cette fonction peut être utilisée pour obtenir la valeur alpha de l'élément sprite. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), et la fonction renvoie une valeur comprise entre 0 (totalement transparent) et 1 (totalement opaque). Syntaxe : layer_sprite_get_alpha(sprite_element_id) ; Arguments Description sprite_element_id La valeur d'identification unique de l'élément de sprite à partir duquel les informations doivent être obtenues. Les retours : Réel (de 0 à 1) Exemple : var lay_id = layer_get_id("sprite_sky") ; var spr_id = layer_sprite_get_id(lay_id, "Clouds") ; if layer_sprite_get_alpha(spr_id) < 0.1 { layer_sprite_destroy(spr_id) ; } Le code ci-dessus obtiendra l'ID du calque nommé "sprite_sky" et l'utilisera pour obtenir l'ID de l'élément sprite sur ce calque. Cet ID est ensuite utilisé pour vérifier l'alpha de l'élément et s'il est inférieur à 0,1, alors l'élément du calque est détruit.
