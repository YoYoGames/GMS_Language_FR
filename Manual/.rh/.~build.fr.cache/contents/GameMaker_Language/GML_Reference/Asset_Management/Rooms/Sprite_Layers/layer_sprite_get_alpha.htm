layer_sprite_get_alpha Cette fonction peut être utilisée pour obtenir la valeur alpha de l'élément sprite. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), et la fonction renverra une valeur comprise entre 0 (totalement transparent) et 1 (totalement opaque). Syntaxe:layer_sprite_get_alpha(sprite_element_id); Argument Description sprite_element_id La valeur d'ID unique de l'élément de sprite à partir duquel obtenir les informations Retour:Real (from 0 to 1) Exemple:var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_alpha(spr_id) < 0.1     {     layer_sprite_destroy(spr_id);     } Le code ci-dessus obtiendra l'ID de la couche nommée "sprite_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément sprite sur cette couche. Cet ID est ensuite utilisé pour vérifier l'élément alpha et s'il est inférieur à 0,1, alors l'élément de couche est détruit.
