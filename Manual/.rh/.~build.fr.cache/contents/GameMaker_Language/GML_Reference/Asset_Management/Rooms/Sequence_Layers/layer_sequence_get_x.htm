séquence_couche_get_x Avec cette fonction, vous fournissez l'ID de l'élément de la séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle renverra la position X actuelle dans la salle de jeu pour la séquence. Syntaxe : layer_sequence_get_x(sequence_element_id) Arguments Description id_élément_de_séquence La valeur d'identification unique de l'élément de séquence à cibler. Les retours : Real Exemple : if layer_sequence_get_x(séquence_titre) != largeur_de_la_salle / 2 { layer_sequence_x(séquence_titre, largeur_de_la_salle / 2) ; } Le code ci-dessus vérifie la position X de l'ID de l'élément de séquence stocké dans la variable "title_sequence", et s'il n'est pas défini à la moitié de la largeur de la pièce, il est défini à cette valeur.
