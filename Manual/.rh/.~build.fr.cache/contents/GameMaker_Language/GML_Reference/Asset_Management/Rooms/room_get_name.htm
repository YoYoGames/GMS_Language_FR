chambre_get_name Cette fonction peut être utilisée pour retourner le nom de la pièce spécifiée sous forme de chaîne de caractères. Veuillez noter qu'il s'agit uniquement d'une chaîne de caractères et qu'elle ne peut pas être utilisée pour référencer directement la salle - pour cela, vous aurez besoin de l'index de la salle. Vous pouvez toutefois utiliser cette chaîne pour obtenir l'index de la pièce en utilisant la chaîne retournée avec la fonction asset_get_index(). Syntaxe : chambre_get_name(index) ; Arguments Description indice L'index de la pièce dont il faut vérifier le nom. Les retours : Chaîne de caractères Exemple : var roomname = room_get_name(room) ; draw_text(32, 32, roomname) ; Le code ci-dessus récupère le nom de la pièce actuelle et l'affiche à l'écran.
