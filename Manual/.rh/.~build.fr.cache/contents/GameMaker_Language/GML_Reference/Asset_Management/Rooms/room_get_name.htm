room_get_name Cette fonction peut être utilisée pour renvoyer le nom de la pièce spécifiée sous forme de chaîne. Veuillez noter qu'il ne s'agit que d' une chaîne et ne peut pas être utilisé pour référencer directement la salle - pour cela, vous aurez besoin de l' index de la salle. Vous pouvez cependant utiliser cette chaîne pour obtenir l' index de la salle à l'aide de la chaîne renvoyée avec la fonction asset_get_index(). Syntaxe:room_get_name(index); Argument Description index L'index de la pièce dont le nom doit être vérifié. Retour:String Exemple:var roomname = room_get_name(room); draw_text(32, 32, roomname); Le code ci-dessus obtiendra le nom de la pièce actuelle et le dessinera à l'écran.
