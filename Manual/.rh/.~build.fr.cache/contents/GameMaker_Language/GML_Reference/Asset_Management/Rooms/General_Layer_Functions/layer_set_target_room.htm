layer_set_target_room Lorsque vous appelez cette fonction, vous GameMaker Studio 2 que toutes les autres fonctions de calque doivent être appliquées à la pièce donnée. De cette manière, vous pouvez modifier ou générer des calques et des contenus de calque de manière procédurale dans une pièce qui n'est pas la pièce actuelle. Lorsque vous avez terminé d'ajouter des calques ou des éléments de calque à une pièce, appelez la fonction layer_reset_target_room() pour réinitialiser la cible de la pièce (ou appeler à nouveau cette fonction avec un argument de salle de -1). Notez que cette fonction ne peut être utilisée que sur des pièces autres que la pièce actuelle et est conçue pour que vous puissiez ajouter / supprimer des calques et des éléments de calque dans des pièces autres que la pièce en cours d'exécution. ATTENTION! Lorsque vous ciblez une autre pièce, vous pouvez utiliser toutes les fonctions de couche normales, sauf que vous ne pouvez pas créer d'instances à l'aide de instance_create_layer () ou instance_create_depth (), et la fonction de couche layer_add_instance () ne sera pas non plus disponible. Syntaxe:layer_set_target_room(room) Argument Description room La pièce à cibler pour toutes les autres fonctions de couche Retour:N/A Exemple:layer_set_target_room(rm_Game); var l = layer_get_id("SpriteAssets"); repeat(50)     {     var _x = random(room_get_width(rm_Game));     var _y = random(room_get_height(rm_Game));     layer_sprite_create(l, _x, _y, spr_Trees);     } layer_reset_target_room(); Le code ci-dessus définit la salle cible sur la salle "rm_Game", puis obtient l'ID de couche pour la couche appelée "SpriteAssets" dans cette salle. Cet ID de couche est ensuite utilisé pour ajouter 50 ressources de sprite aléatoires à la couche, avant que la cible de la couche ne soit réinitialisée dans la pièce actuelle.
