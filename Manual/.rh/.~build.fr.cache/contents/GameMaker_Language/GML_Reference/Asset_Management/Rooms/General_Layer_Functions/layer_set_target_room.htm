layer_set_target_room Lorsque vous appelez cette fonction, vous indiquez à GameMaker Studio 2 que toutes les autres fonctions de calque doivent être appliquées à la pièce donnée. De cette façon, vous pouvez modifier ou générer de manière procédurale des calques et leur contenu dans une pièce qui n'est pas la pièce actuelle. pièce actuelle. Lorsque vous avez fini d'ajouter des calques ou des éléments de calque à une pièce, appelez la fonction layer_reset_target_room() pour réinitialiser la cible de la pièce (ou appelez à nouveau cette fonction avec un argument de pièce de -1). Notez que cette fonction ne peut être utilisée que sur des pièces autres que la pièce actuelle, et qu'elle est conçue pour que vous puissiez ajouter/supprimer des calques et des éléments de calque dans des pièces autres que la pièce en cours d'exécution. ATTENTION ! Lorsque vous ciblez une autre pièce, vous pouvez utiliser toutes les fonctions normales des couches , sauf que vous ne pouvez pas créer d'instance en utilisant instance_create_layer() ou instance_create_depth()et la fonction layer layer_add_instance() ne sera pas disponible. Syntaxe : layer_set_target_room(room) Arguments Description chambre La pièce à cibler pour toutes les autres fonctions de la couche Les retours : N/A Exemple : layer_set_target_room(rm_Game) ; var l = layer_get_id("SpriteAssets") ; repeat(50) { var _x = random(room_get_width(rm_Game)) ; var _y = random(room_get_height(rm_Game)) ; layer_sprite_create(l, _x, _y, spr_Trees) ; } layer_reset_target_room() ; Le code ci-dessus définit la pièce cible comme étant la pièce "rm_Game" et obtient ensuite l'ID du calque appelé "SpriteAssets" dans cette pièce. Cet ID de couche est ensuite utilisé pour ajouter 50 sprites aléatoires à la couche, avant que la cible de la couche ne soit réinitialisée à la pièce actuelle. soit réinitialisée à la pièce actuelle.
