chambre Cette variable contient l'index de la salle actuelle dans laquelle votre jeu est en cours. Ce n' est pas une variable en lecture seule, cependant changer cette variable ne changera pas la valeur de l'index de la salle actuelle, mais changera plutôt la salle pour qu'elle corresponde à l'index que vous avez défini. que vous avez défini pour la variable. Il convient d'être prudent car si l'indice que vous modifiez n'est pas valide, le jeu affichera une erreur et se fermera. En général, il est préférable d'utiliser room_goto() pour changer de pièce. Remarque: Les identifiants de salle ne sont pas basés sur leur ordre dans l'Asset Browser ou le Room Manager, et vous devez donc éviter de fournir directement une valeur numérique. Utilisez plutôt la constante de salle pour l'actif que vous souhaitez référencer (qui est simplement son nom) ou récupérez-le par le biais d'une fonction. Syntaxe : chambre ; Les retours : Real (valeur d'index unique) Exemple : if (room == rm_level1) { audio_play_sound(snd_level1, 1, 1) ; } Le code ci-dessus vérifiera si la pièce actuelle est le premier niveau, et dans ce cas, jouera la musique de ce niveau.
