couche_sprite_x Cette fonction contrôle la position le long de l'axe x de la pièce de l'élément sprite de l'actif sur la couche. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur x à utiliser (basée sur les coordonnées de la pièce). Syntaxe : layer_sprite_x(sprite_element_id, x) ; Arguments Description sprite_element_id La valeur d'identification unique de l'élément du sprite à modifier. x La position x du sprite Les retours : N/A Exemple : var lay_id = layer_get_id("Asset_sky") ; var spr_id = layer_sprite_get_id(lay_id, "Clouds") ; layer_sprite_x(spr_id, irandom(room_width)) ; Le code ci-dessus récupère la valeur ID du sprite "Nuages" affecté au calque "Asset_sky", puis définit sa position x à une valeur aléatoire comprise entre 0 et la largeur de la pièce.
