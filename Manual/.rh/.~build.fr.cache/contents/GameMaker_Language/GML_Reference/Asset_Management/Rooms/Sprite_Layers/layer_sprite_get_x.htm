layer_sprite_get_x Cette fonction peut être utilisée pour obtenir la position x de l'élément sprite dans la pièce. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), et la fonction renverra la valeur de position x. Syntaxe:layer_sprite_get_x(sprite_element_id); Argument Description sprite_element_id La valeur d'ID unique de l'élément de sprite à partir duquel obtenir les informations Retour:Real Exemple:var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_x(spr_id) < 0     {     layer_sprite_x(spr_id, room_width);     } Le code ci-dessus obtiendra l'ID de la couche nommée "sprite_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément sprite sur cette couche. Cet ID est ensuite utilisé pour vérifier la position x de l'élément et si elle est inférieure à 0, alors l'élément de calque est déplacé vers une position x différente.
