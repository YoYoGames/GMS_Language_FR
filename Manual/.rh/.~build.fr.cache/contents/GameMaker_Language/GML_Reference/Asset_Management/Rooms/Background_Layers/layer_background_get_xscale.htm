layer_background_get_xscale Cette fonction peut être utilisée pour obtenir la valeur actuelle du multiplicateur d'échelle de l'élément d'arrière-plan. Vous donnez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), et la fonction retournera une valeur réelle qui représente le multiplicateur d'échelle utilisé pour dessiner le sprite. La valeur par défaut est 1. Syntaxe:layer_background_get_xscale(background_element_id); Argument Description background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à partir duquel obtenir les informations Retour:Real Exemple:var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_xscale(back_id) != -1     {     layer_background_xscale(back_id, -1);     } Le code ci-dessus obtiendra l'ID de la couche nommée "Background_sky", puis l'utilisera pour obtenir l'ID de l'élément d'arrière-plan sur cette couche. Cet ID est ensuite utilisé pour vérifier l'échelle x de l'élément et s'il n'est pas -1, il est mis à -1.
