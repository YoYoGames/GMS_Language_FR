couche_get_tout_élément Vous pouvez utiliser cette fonction pour obtenir les ID des éléments de la couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create()) ou le nom de la couche (sous la forme d'une chaîne - cela aura un impact sur les performances) et la fonction renverra un tableau d'ID, où chaque entrée du tableau est un ID unique pour un élément de cette couche. Par exemple, si la couche est une couche d'actifs, le tableau sera rempli avec les valeurs d'ID de chaque actif sprite qui est affecté à la couche. Notez que l'utilisation du code pour travailler avec des couches signifie que vous pouvez affecter différents types d'éléments à la même couche - vous pouvez donc avoir des sprites en plus des instances, par exemple - dans ce cas, vous pouvez utiliser la fonction layer_get_element_type() pour obtenir le type d'élément auquel l'ID se rapporte. Syntaxe : layer_get_all_elements(layer_id) Arguments Description identifiant de la couche La valeur d'identification unique de la couche pour obtenir les éléments de (ou le nom de la couche comme une chaîne) Les retours : Tableau (1D, rempli d'ID d'éléments) Exemple : var a = layer_get_all_elements(layer) ; for (var i = 0 ; i < array_length(a) ; i++ ;) { if layer_get_element_type(a[i]) == layerelementtype_sprite { layer_sprite_destroy(a[i]) } } Le code ci-dessus récupère les ID de tous les éléments d'instance affectés au layer de l'instance qui exécute le code. Le code vérifie ensuite si l'un des éléments retournés est un sprite et s'il l'est, il est détruit.
