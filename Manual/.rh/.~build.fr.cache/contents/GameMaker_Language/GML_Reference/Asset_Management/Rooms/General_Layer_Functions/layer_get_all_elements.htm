layer_get_all_elements Vous pouvez utiliser cette fonction pour obtenir les ID d'élément de la couche donnée. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et la fonction renverra un tableau d'ID, où chaque entrée du tableau est un ID unique pour un élément de cette couche. Par exemple, si la couche est une couche d'actif, le tableau sera rempli avec les valeurs d'ID de chaque ressource d'image-objet affectée à la couche. Notez que l'utilisation de code pour travailler avec des couches signifie que vous pouvez attribuer différents types d'élément à la même couche - vous pouvez donc avoir des ressources de sprite avec des instances, par exemple - auquel cas vous pouvez ensuite utiliser la fonction layer_get_element_type() pour obtenir le type d'élément auquel l'ID se rapporte. Syntaxe:layer_get_all_elements(layer_id) Argument Description layer_id La valeur d'identifiant unique du calque à partir duquel obtenir les éléments (ou le nom du calque sous forme de chaîne) Retour:Array (1D, populated with Element IDs) Exemple:var a = layer_get_all_elements(layer); for (var i = 0; i < array_length(a); i++;)    {    if layer_get_element_type(a[i]) == layerelementtype_sprite       {       layer_sprite_destroy(a[i])       }    } Le code ci-dessus obtient les ID de tous les éléments d'instance affectés à la couche de l'instance exécutant le code. Le code vérifie ensuite si l'un des éléments renvoyés est un actif de sprite et s'il est détruit.
