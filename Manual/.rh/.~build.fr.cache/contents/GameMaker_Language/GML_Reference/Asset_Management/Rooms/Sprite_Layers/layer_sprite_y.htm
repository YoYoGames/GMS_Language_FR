couche_sprite_y Cette fonction contrôle la position sur l'axe des y de la pièce de l'élément sprite de l'actif sur le calque. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur y à utiliser (en fonction des coordonnées de la pièce). Syntaxe : layer_sprite_y(sprite_element_id, y) ; Arguments Description sprite_element_id La valeur d'identification unique de l'élément du sprite à modifier. y La position y du sprite Les retours : N/A Exemple : var lay_id = layer_get_id("Asset_sky") ; var spr_id = layer_sprite_get_id(lay_id, "Clouds") ; layer_sprite_y(spr_id, 50 + irandom(room_height - 200)) ; Le code ci-dessus récupère la valeur ID du sprite "Nuages" affecté au calque "Asset_sky", puis définit sa position y à une valeur aléatoire comprise entre 50 et la hauteur de la pièce moins 200.
