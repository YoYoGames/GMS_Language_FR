layer_sequence_get_y Avec cette fonction, vous fournissez l'ID d'élément de séquence - tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et il renverra la position Y actuelle dans la salle de jeu pour la séquence. Syntaxe:layer_sequence_get_y(sequence_element_id) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler Retour:Real Exemple:if layer_sequence_get_y(title_sequence) != room_height / 2     {     layer_sequence_y(title_sequence, room_height / 2);     } Le code ci-dessus vérifie la position Y de l'ID d'élément de séquence stocké dans la variable "title_sequence", et s'il n'est pas défini sur la moitié de la hauteur de la pièce, il est défini sur cette valeur.
