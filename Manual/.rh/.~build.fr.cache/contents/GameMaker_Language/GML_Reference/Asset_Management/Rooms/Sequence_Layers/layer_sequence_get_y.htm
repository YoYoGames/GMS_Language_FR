séquence_couche_get_y Avec cette fonction, vous fournissez l'ID de l'élément de la séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle renverra la position Y actuelle dans la salle de jeu pour la séquence. Syntaxe : layer_sequence_get_y(sequence_element_id) Arguments Description id_élément_de_séquence La valeur d'identification unique de l'élément de séquence à cibler. Les retours : Real Exemple : if layer_sequence_get_y(séquence_titre) != room_height / 2 { layer_sequence_y(séquence_titre, hauteur_de_la_salle / 2) ; } Le code ci-dessus vérifie la position Y de l'ID de l'élément de séquence stocké dans la variable "title_sequence", et s'il n'est pas défini à la moitié de la hauteur de la pièce, il est défini à cette valeur.
