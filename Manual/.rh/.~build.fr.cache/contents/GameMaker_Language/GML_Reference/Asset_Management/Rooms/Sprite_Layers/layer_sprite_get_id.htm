layer_sprite_get_id Cette fonction peut être utilisée pour récupérer la valeur d'ID unique de l'élément sprite sur un calque. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ou lorsque vous utilisez le nom du calque avec layer_get_id() ) et le nom de l'actif tel que défini dans les propriétés de l'actif IDE. La fonction retournera la valeur d'ID associée à cet élément de sprite sur le calque. Notez que cette fonction est spécialement conçue pour être utilisée avec des sprites qui ont été ajoutés dans l'EDI sur une couche d'actifs, comme si vous ajoutiez un sprite à un calque via du code à l'aide de la fonction layer_sprite_create(), puis il renverra l'ID unique de l'élément sprite ajouté (et le nouvel élément n'aura pas de nom). Syntaxe:layer_sprite_get_id(layer_id, asset_name) Argument Description layer_id La valeur d'ID unique du calque à cibler asset_name Le nom unique de l'actif sur le calque tel que défini dans l'éditeur de pièce Retour:Sprite element ID value (Real) Exemple:var lay_id = layer_get_id("Assets_trees"); var back_id = layer_sprite_get_id(lay_id, "gfc_Trees"); layer_sprite_change(back_id, spr_Trees_Winter); Le code ci-dessus obtiendra l'ID de couche pour la couche nommée "Assets_trees" et l'utilisera ensuite pour obtenir l'ID de l'élément de sprite "gfc_Trees" sur cette couche. Cet identifiant est ensuite utilisé pour changer l'élément sprite. Retour: Couches de sprite Suivant:layer_sprite_exists © Copyright YoYo Games Ltd. 2021 Tous droits réservés
