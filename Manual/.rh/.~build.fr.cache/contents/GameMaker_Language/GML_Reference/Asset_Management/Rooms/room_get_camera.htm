chambre_get_caméra Cette fonction vous permet d'obtenir l'index unique de la caméra affectée à une vue spécifique dans une pièce autre que la pièce actuelle. Vous indiquez la salle à utiliser, le port de vue à utiliser (de 0 à 7) et la fonction renvoie un index de caméra. Syntaxe : room_get_camera(rm, vind) ; Arguments Description rm L'index de la salle pour obtenir la caméra de vue de vind L'index du port de vue pour obtenir la caméra de Les retours : Real (indice de caméra) Exemple : var cam = room_get_camera(rm_Game, 0) ; if cam != global.MainCam { room_set_camera(rm_Game, 0, global.MainCam) ; } Le code ci-dessus attribue une caméra dans une pièce nouvellement créée au port de vue [0].
