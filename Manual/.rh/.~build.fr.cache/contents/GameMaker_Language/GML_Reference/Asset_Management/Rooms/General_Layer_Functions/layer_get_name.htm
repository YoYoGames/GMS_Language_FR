couche_get_name Vous pouvez utiliser cette fonction pour obtenir le nom de la couche donnée. Vous fournissez la valeur unique de l'ID de la couche et si la couche est l'une des couches nommées créées dans l'éditeur de pièce, la fonction renverra une chaîne avec le nom de la couche. Si la couche n'est pas l'une de celles de l'éditeur de pièce (c'est-à-dire qu'elle a été créée à l'aide de la fonction layer_create()), une chaîne vide sera retournée. Syntaxe : layer_get_name(layer_id) Arguments Description identifiant de la couche La valeur d'identification unique de la couche dont on veut obtenir le nom. Les retours : Chaîne de caractères Exemple : var a = layer_get_all() ; var layer_list = ds_list_create() ; for (var i = 0 ; i Alt ; array_length(a) ; i++ ;) { if layer_get_name(a[i]) != "" { ds_list_add(layer_list, a[i]) } } Le code ci-dessus récupère les ID de tous les calques de la pièce et les parcourt en boucle pour vérifier si certains sont des calques nommés. S'ils le sont, leur ID est ajouté à une liste. Retour : Fonctions générales des couches Suivant : layer_get_all Copyright YoYo Games Ltd. 2021 Tous droits réservés
