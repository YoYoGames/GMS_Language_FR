séquence_couche_x Avec cette fonction, vous pouvez définir la position le long de l'axe X (horizontal) de la pièce pour l'élément de séquence donné. Vous fournissez l'ID de l'élément de séquence tel que retourné par layer_sequence_create() ou par une des fonctions d'élément de couche avec la position X à définir et la séquence sera déplacée vers la nouvelle position. Syntaxe : layer_sequence_x(sequence_element_id, pos_x) Arguments Description id_élément_de_séquence La valeur d'identification unique de l'élément de séquence à cibler. pos_x La position X à laquelle déplacer l'élément de la séquence Les retours : N/A Exemple : if layer_sequence_exists(my_seq) { layer_sequence_x(ma_seq, x) ; layer_sequence_y(ma_seq, y) ; } Le code ci-dessus vérifie si l'élément de séquence référencé dans la variable "my_seq" existe, et si c'est le cas, il définit la position x/y à celle de l'instance qui exécute le code.
