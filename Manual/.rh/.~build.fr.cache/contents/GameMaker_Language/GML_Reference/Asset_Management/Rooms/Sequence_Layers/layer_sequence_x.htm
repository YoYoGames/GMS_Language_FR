layer_sequence_x Avec cette fonction, vous pouvez définir la position le long de l'axe X (horizontal) de la pièce pour l'élément de séquence donné. Vous fournissez l'ID d'élément de séquence renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de calque avec la position X à définir et la séquence sera déplacée vers la nouvelle position. Syntaxe:layer_sequence_x(sequence_element_id, pos_x) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler pos_x La position X vers laquelle déplacer l'élément de séquence Retour:N/A Exemple:if layer_sequence_exists(my_seq)     {     layer_sequence_x(my_seq, x);     layer_sequence_y(my_seq, y);     } Le code ci-dessus vérifie si l'élément de séquence référencé dans la variable "my_seq" existe, et s'il le fait, il définit la position x / y sur celle de l'instance exécutant le code.
