layer_tilemap_exists Vous pouvez utiliser cette fonction pour vérifier et voir si un élément de mosaïque existe sur une couche donnée. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et l'ID d'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id() ) et la fonction retournera soit true si l'élément existe, ou false si ça ne fait pas. NOTE: Cette fonction fonctionne dans le cadre de la salle cible actuelle - par défaut la salle dans laquelle la fonction est appelée - qui peut être définie à l'aide de la fonction layer_set_target_room (). Syntaxe:layer_tilemap_exists(layer_id, tilemap_element_id) Argument Description layer_id La valeur d'ID unique du calque à cibler (ou le nom du calque sous forme de chaîne) tilemap_element_id La valeur d'ID unique de l'élément de mappage de tuiles à vérifier Retour:Boolean Exemple:var lay_id = layer_get_id("tilemap_trees"); if layer_tilemap_exists(lay_id, global.Treestilemap)     {     layer_tilemap_destroy(lay_id, global.Treestilemap);     } Le code ci-dessus vérifie la couche "tilemap_trees" pour voir si l'élément de carte de tuile donné existe et si c'est le cas, alors il est détruit (mais pas la couche).
