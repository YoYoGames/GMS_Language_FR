séquence_de_couche_get_séquence Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle renverra la structure de l'objet de la séquence. Cette fonction évite d'avoir à obtenir d'abord la structure d'instance de séquence et vous permet d'accéder directement aux données de la séquence. Vous pouvez en savoir plus sur le format de la structure d'objet de séquence sur cette page. Syntaxe : layer_sequence_get_sequence(sequence_element_id) Arguments Description id_élément_de_séquence La valeur d'identification unique de l'élément de séquence à cibler. Les retours : Structure Exemple : var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground) ; var _struct = layer_sequence_get_sequence(_seq) ; _struct.playbackSpeedType = spritespeed_framespersecond ; _struct.playbackSpeed = 30 ; Le code ci-dessus crée un nouvel élément de séquence et récupère ensuite sa structure de données de séquence. Celle-ci est ensuite utilisée pour modifier la vitesse et le type de lecture de la séquence.
