layer_sequence_get_sequence Avec cette fonction, vous fournissez l'ID d'élément de séquence - tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et il retournera la structure de séquence. Cette fonction évite la nécessité d'obtenir d'abord la structure d'instance de séquence et vous permet d'accéder directement aux données de séquence. Vous pouvez en savoir plus sur le format de la structure de séquence dans la section sur les fonctions de séquence. Syntaxe:layer_sequence_get_sequence(sequence_element_id) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler Retour:struct Exemple:var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); var _struct = layer_sequence_get_sequence(_seq); _struct.playbackSpeedType = spritespeed_framespersecond; _struct.playbackSpeed = 30; Le code ci-dessus crée un nouvel élément de séquence, puis récupère la structure de données de séquence. Ceci est ensuite utilisé pour définir la vitesse de lecture et le type de la séquence.
