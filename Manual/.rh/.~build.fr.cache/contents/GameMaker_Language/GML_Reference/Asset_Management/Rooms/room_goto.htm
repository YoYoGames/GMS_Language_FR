chambre_goto Cette fonction vous permet d'accéder à n'importe quelle pièce de votre projet de jeu, qu'elle ait été créée à l'aide du code ou dans le navigateur d'actifs. Vous fournissez l'index de la pièce (stocké dans la variable correspondant au nom de la pièce, ou comme une variable renvoyée par la fonction room_add()). Notez que la room ne changera pas jusqu'à la fin de l'événement où la fonction a été appelée, donc tout code après l'appel de la fonction sera toujours exécuté s'il se trouve dans le même événement. Le problème est que tout code qui crée des instances et qui est appelé après cette fonction ne créera pas l'instance et ne sera pas exécuté. après cette fonction, ne créera pas l'instance et doit donc être appelé avant de changer la salle. Cette fonction déclenchera également l'événement Room End. Remarque: Les identifiants de salle ne sont pas basés sur leur ordre dans l'Asset Browser ou le Room Manager, et vous devez donc éviter de fournir directement une valeur numérique. Utilisez plutôt la constante de salle pour l'actif que vous souhaitez référencer (qui est simplement son nom) ou récupérez-le par le biais d'une fonction. Syntaxe : chambre_goto(index) ; Arguments Description indice L'index de la pièce où se rendre. Les retours : N/A Exemple : switch (global.level) { cas 0 : room_goto(rm_level1) ; pause ; cas 1 : room_goto(rm_level2) ; pause ; cas 2 : room_goto(rm_level3) ; pause ; } Le code ci-dessus vérifiera une variable globale et changera de pièce en fonction de la valeur qu'elle contient.
