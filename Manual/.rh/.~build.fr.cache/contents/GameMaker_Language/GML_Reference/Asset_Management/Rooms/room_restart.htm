room_restart Cette fonction redémarrera la pièce actuelle, comme si elle venait d'être entrée. Notez que la salle ne redémarrera pas avant la fin de l'événement où la fonction a été appelée, donc tout code après l'appel sera toujours exécuté s'il se trouve dans le même événement. La mise en garde à ceci est que tout code qui crée des instances qui est appelé après cela, ne créera pas l'instance et devrait donc être appelé avant de changer de salle. Cette fonction déclenchera également l'événement de fin de pièce. Syntaxe:room_restart(); Retour:N/A Exemple:if lives < 1 room_restart(); Le code ci-dessus vérifie si la variable "lives" est inférieure à 1 et si c'est le cas, il redémarrera la salle.
