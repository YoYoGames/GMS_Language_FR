couche_get_élément_couche Vous pouvez utiliser cette fonction pour obtenir l'ID du calque sur lequel se trouve l'élément donné. Vous fournissez la valeur unique de l'ID de l'élément (par exemple, telle que renvoyée par la fonction qui a créé l'élément ou par l'éditeur de pièce) et la fonction renvoie l'ID unique de la couche sur laquelle se trouve l'élément. de la couche sur laquelle se trouve l'élément. Si l'ID de l'élément donné n'est pas valide, la fonction renvoie -1. Syntaxe : layer_get_element_layer(element_id) Arguments Description identifiant de l'élément La valeur d'identification unique de l'élément dont on veut obtenir la couche. Les retours : ID de la couche ou -1 si l'élément n'est pas valide (c'est-à-dire s'il n'existe pas). Exemple : element = layer_get_element_layer(asset_1) ; Le code ci-dessus obtient l'ID de la couche pour l'élément avec l'ID stocké dans la variable "asset_1" et le stocke dans une variable.
