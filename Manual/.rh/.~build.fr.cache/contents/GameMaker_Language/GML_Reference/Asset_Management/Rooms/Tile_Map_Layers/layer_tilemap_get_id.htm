layer_tilemap_get_id Cette fonction peut être utilisée pour récupérer la valeur d'identification unique de l'élément de carte de carreaux sur une couche. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous la forme d'une chaîne de caractères - cela aura un impact sur les performances) et la fonction renverra la valeur d'identification associée à l'élément de carte de tuiles sur la couche. Notez que cette fonction est spécifiquement conçue pour être utilisée avec des cartes de tuiles qui ont été ajoutées dans l'IDE. En effet, si vous ajoutez une carte de tuiles à une couche par le biais d'un code utilisant la fonction layer_tilemap_create()alors elle renverra l'ID unique de l'élément de carte de carreaux ajouté. Si l'ID de la carte de tuiles est incorrecte ou que la carte de tuiles n'existe pas, la fonction renvoie -1. Syntaxe : layer_tilemap_get_id(layer_id) Arguments Description identifiant de la couche La valeur d'identification unique de la couche à cibler (ou le nom de la couche sous forme de chaîne). Les retours : valeur de l'ID de l'élément de la carte des tuiles (réel) ou -1 Exemple : var lay_id = layer_get_id("tilemap_trees") ; var tile_id = layer_tilemap_get_id(lay_id) ; layer_tilemap_destroy(tile_id) ; Le code ci-dessus obtiendra l'ID de la couche nommée "tilemap_trees", puis l'utilisera pour obtenir l'ID de l'élément de carte de carreaux sur cette couche. Cet ID est ensuite utilisé pour supprimer la carte de tuiles de la couche. Retour : Carrelage des couches de la carte Suivant : layer_tilemap_exists Copyright YoYo Games Ltd. 2021 Tous droits réservés
