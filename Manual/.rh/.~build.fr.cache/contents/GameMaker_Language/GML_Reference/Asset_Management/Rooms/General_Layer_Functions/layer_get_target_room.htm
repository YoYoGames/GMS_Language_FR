layer_get_target_room Cette fonction renvoie la pièce actuelle ciblée par les fonctions de la couche. Voir layer_set_target_room() pour plus d'informations. Syntaxe : layer_get_target_room() Les retours : ID de la chambre (réel) Exemple : si layer_get_target_room() != room { layer_reset_target_room() ; } Le code ci-dessus vérifie la pièce cible actuelle et s'il ne s'agit pas de la pièce actuelle, la pièce cible de la couche est réinitialisée.
