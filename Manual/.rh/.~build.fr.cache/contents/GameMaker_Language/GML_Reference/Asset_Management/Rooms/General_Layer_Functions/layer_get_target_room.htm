layer_get_target_room Cette fonction renverra la pièce actuelle ciblée par les fonctions de calque. Voir layer_set_target_room() pour plus d'informations. Syntaxe:layer_get_target_room() Retour:Room ID (real) Exemple:if layer_get_target_room() != room    {    layer_reset_target_room();    } Le code ci-dessus vérifie la pièce cible actuelle et si ce n'est pas la pièce actuelle, la pièce cible de la couche est réinitialisée.
