chambre_add Cette fonction crée une nouvelle pièce, vide, et l'ajoute à votre jeu, en renvoyant son index qui sera stocké dans une variable pour tous les codes ultérieurs qui traitent de cette pièce. Notez que chaque pièce est ajoutée de façon permanente au jeu jusqu'à ce que l'exécutable soit fermé, c'est-à-dire que les pièces ajoutées par code ne peuvent plus être supprimées. Ceci a des implications importantes pour l'utilisation de la mémoire et vous devez donc utiliser cette fonction avec précaution. NOTE: Les nouvelles pièces ne font pas partie de l'ordre habituel des pièces, elles n'ont donc pas de pièce "précédente" ou "suivante" (ce qui signifie que les fonctions chambre_suivante() et room_previous() ne fonctionneront pas). Pour passer de la pièce ajoutée à une autre, vous devez utiliser l'index de la pièce elle-même. Syntaxe : room_add() ; Les retours : Index Exemple : global.myroom = room_add() ; room_set_width(global.myroom, 640) ; room_set_height(global.myroom, 480) ; room_set_persistent(global.myroom, false) ; Cette opération crée une nouvelle pièce et enregistre son index dans la variable "global.myroom".
