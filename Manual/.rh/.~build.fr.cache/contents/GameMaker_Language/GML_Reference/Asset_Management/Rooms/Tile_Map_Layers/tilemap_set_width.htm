tilemap_set_width Cette fonction peut être utilisée pour redimensionner un élément de carte de tuiles. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la nouvelle largeur de la carte en cellules de tuiles. Syntaxe : tilemap_set_width(tilemap_element_id, width) Arguments Description tilemap_element_id La valeur d'identification unique de l'élément de la carte des tuiles dont il faut définir la largeur. largeur La valeur de la largeur (en "cellules") Les retours : N/A Exemple : var lay_id = layer_get_id("Tiles_Walls") ; var map_id = layer_tilemap_get_id(lay_id) ; if tilemap_get_width(map_id) != room_width div 16 { tilemap_set_width(map_id, room_width div 16) ; } Le code ci-dessus vérifie la largeur d'une carte de tuiles spécifique et si elle n'est pas de la bonne taille, la largeur est définie.
