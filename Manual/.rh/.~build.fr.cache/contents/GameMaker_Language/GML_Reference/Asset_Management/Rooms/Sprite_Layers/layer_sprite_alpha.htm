couche_sprite_alpha Cette fonction contrôle l'alpha (la transparence) du sprite sur la couche de l'actif. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur alpha à utiliser. La valeur alpha peut être comprise entre 0 (totalement transparent) et 1 (totalement opaque), la valeur alpha par défaut de l'élément sprite étant de 1. Notez que si le calque auquel l'élément sprite a été assigné n'est pas visible - ou si l'élément lui-même a été rendu invisible - vous ne verrez aucune différence avec cette fonction jusqu'à ce que le calque ou l'élément soit rendu visible à nouveau. Syntaxe : layer_sprite_alpha(sprite_element_id, alpha) ; Arguments Description sprite_element_id La valeur d'identification unique de l'élément du sprite à modifier. alpha L'alpha pour le sprite, de 0 à 1 (la valeur par défaut est 1) Les retours : N/A Exemple : var lay_id = layer_get_id("Asset_sky") ; var spr_id = layer_sprite_get_id(lay_id, "Clouds") ; layer_sprite_alpha(spr_id, random(1)) ; Le code ci-dessus récupère la valeur d'identification du sprite nommé "Nuages" affecté au calque "Asset_sky", puis définit son alpha sur une valeur aléatoire comprise entre 0 et 1.
