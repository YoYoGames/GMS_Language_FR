layer_sprite_alpha Cette fonction contrôle l'alpha (transparence) de l'image-objet sur le calque d'actif. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), puis définissez la valeur alpha à utiliser. Alpha peut être compris entre 0 (totalement transparent) et 1 (totalement opaque), la valeur alpha par défaut de l'élément sprite étant 1. Notez que si le calque auquel l'élément sprite a été affecté n'est pas visible - ou l'élément lui-même a été créé invisible - vous ne verrez aucune différence avec cette fonction tant que le calque ou l'élément n'a pas été rendu visible à nouveau. Syntaxe:layer_sprite_alpha(sprite_element_id, alpha); Argument Description sprite_element_id La valeur d'ID unique de l'élément de sprite à modifier alpha L'alpha du sprite sprite, de 0 à 1 (la valeur par défaut est 1) Retour:N/A Exemple:var lay_id = layer_get_id("Asset_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); layer_sprite_alpha(spr_id, random(1)); Le code ci-dessus obtient la valeur d'ID de l'élément de sprite nommé "Clouds" attribué à la couche "Asset_sky", puis définit son alpha sur une valeur aléatoire entre 0 et 1.
