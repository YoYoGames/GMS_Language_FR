couche_get_vspeed Vous pouvez utiliser cette fonction pour récupérer la vitesse verticale (en pixels par image de jeu) de la couche dans la pièce actuellement analysée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom du calque (sous forme de chaîne de caractères - cela aura un impact sur les performances) et la fonction renvoie un nombre réel pour la vitesse verticale, où une valeur positive correspond à une descente et une valeur négative à une montée. La valeur par défaut est 0 (sauf si elle est définie dans l'éditeur de pièce). Syntaxe : layer_get_vspeed(layer_id) Arguments Description identifiant de la couche La valeur d'identification unique de la couche pour obtenir la vitesse verticale à partir de Les retours : N/A Exemple : var lay_id = layer_get_id("Sprites") ; if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0 { layer_hspeed(lay_id, 0) ; layer_vspeed(lay_id, 0) ; } Le code ci-dessus vérifie les vitesses horizontale et verticale de la couche donnée et si elles ne sont pas toutes deux à 0, il les met à 0.
