layer_background_destroy Cette fonction va détruire l'élément d'arrière-plan donné. Vous fournissez l'ID de l'arrière-plan (que vous obtenez lorsque vous créez l'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez l'ID de la couche avec avec layer_get_background_id()) et cette fonction le supprimera. Notez que cela ne supprime pas le calque, mais seulement l'arrière-plan, et si l'arrière-plan a été ajouté dans l'éditeur room la prochaine fois que vous quitterez la pièce et y reviendrez, l'arrière-plan sera recréé. Cependant, si la pièce est persistante, l'arrière-plan sera supprimé, à moins que la persistance de la pièce ne soit à nouveau désactivée. Syntaxe : layer_background_destroy(background_element_id) Arguments Description id_élément_fond La valeur d'identification unique de l'arrière-plan à détruire. Les retours : N/A Exemple : var lay_id = layer_get_id("Background_trees") ; var bck_id = layer_background_get_id("Forrest") ; if layer_background_exists(lay_id, bck_id) { layer_background_destroy(bck_id) ; } Le code ci-dessus vérifie la couche "Background_trees" pour voir si l'élément d'arrière-plan donné existe et si c'est le cas, alors il est détruit (mais pas la couche).
