tuile_get_flip Cette fonction peut être utilisée pour vérifier si, dans un ensemble donné de données de tuiles, la tuile a été retournée ou non. Vous donnez les données des tuiles, qui peuvent être récupérées en utilisant la fonction tilemap_get()et la fonction retournera vrai si la tuile est retournée, ou faux si non. Syntaxe : tile_get_flip(tiledata) Arguments Description données en mosaïque les données de la tuile à vérifier Les retours : Booléen Exemple : var lay_id = layer_get_id("Tiles_sky") ; var map_id = layer_tilemap_get_id(lay_id) ; var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y) ; var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y) ; var data = tilemap_get(map_id, mx, my) ; var bool = !tile_get_flip(data) ; data = tile_set_flip(data, bool) ; tilemap_set(map_id, data, mx, my) ; Le code ci-dessus récupère l'ID de la carte des tuiles à partir de la couche donnée, puis obtient la position x et y de la cellule de la tuile sous la souris. Cette position est ensuite utilisée pour obtenir les données de la tuile qui sont retournées et ensuite utilisées pour définir à nouveau la tuile.
