tile_get_flip Cette fonction peut être utilisée pour vérifier si, dans un ensemble donné de données de tuiles, la tuile a été retournée ou non. Vous donnez les données de tuile, qui peuvent être récupérées à l'aide de la fonction tilemap_get(), et la fonction retournera true si la tuile est retournée, ou false sinon. Syntaxe:tile_get_flip(tiledata) Argument Description tiledata les données de tuile à vérifier Retour:Boolean Exemple:var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var bool = !tile_get_flip(data); data = tile_set_flip(data, bool); tilemap_set(map_id, data, mx, my); Le code ci-dessus obtient l'ID de carte de tuiles de la couche donnée, puis obtient la position des cellules x et y pour la tuile sous la souris. Cette position est ensuite utilisée pour obtenir les données de tuile qui sont retournées puis utilisées pour redéfinir la tuile.
