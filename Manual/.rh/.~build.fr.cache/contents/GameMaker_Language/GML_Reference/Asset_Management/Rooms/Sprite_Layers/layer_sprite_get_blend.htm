couche_sprite_get_blend Cette fonction peut être utilisée pour obtenir la couleur de fusion de l'élément sprite. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), et la fonction renverra une valeur réelle qui représente la couleur attribuée. Syntaxe : layer_sprite_get_blend(sprite_element_id) ; Arguments Description sprite_element_id La valeur d'identification unique de l'élément de sprite à partir duquel les informations doivent être obtenues. Les retours : Valeur de la couleur (réel) Exemple : var lay_id = layer_get_id("sprite_sky") ; var back_id = layer_sprite_get_id(lay_id, "Clouds") ; if layer_sprite_get_blend(back_id) == c_white { layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255)) ; } Le code ci-dessus obtiendra l'ID du calque nommé "sprite_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément sprite sur ce calque. Cet ID est ensuite utilisé pour vérifier la couleur de fusion de l'élément et si elle est équivalente à la constante c_white, alors le mélange de l'élément est fixé à une couleur aléatoire.
