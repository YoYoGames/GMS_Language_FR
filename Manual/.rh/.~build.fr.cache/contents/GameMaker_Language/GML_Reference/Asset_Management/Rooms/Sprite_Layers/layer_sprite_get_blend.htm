layer_sprite_get_blend Cette fonction peut être utilisée pour obtenir la couleur de fusion de l'élément sprite. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), et la fonction renverra une valeur réelle qui représente la couleur attribuée. Syntaxe:layer_sprite_get_blend(sprite_element_id); Argument Description sprite_element_id La valeur d'ID unique de l'élément de sprite à partir duquel obtenir les informations Retour:Colour Value (Real) Exemple:var lay_id = layer_get_id("sprite_sky"); var back_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_blend(back_id) == c_white     {     layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255));     } Le code ci-dessus obtiendra l'ID de la couche nommée "sprite_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément sprite sur cette couche. Cet identifiant est ensuite utilisé pour vérifier la couleur de fusion de l'élément et si elle est équivalente à la constante c_white, puis le mélange d'éléments est défini sur une couleur aléatoire.
