layer_get_all Cette fonction renverra un tableau rempli avec les valeurs d'ID uniques de chaque couche de la pièce. Syntaxe:layer_get_all() Retour:Array (1D, populated with Layer IDs) Exemple:var a = layer_get_all(); for (var i = 0; i < array_length(a); i++;)    {    layer_destroy(a[i]);    } Le code ci-dessus récupère toutes les couches d'une pièce et ajoute leurs valeurs d'ID à un tableau.
