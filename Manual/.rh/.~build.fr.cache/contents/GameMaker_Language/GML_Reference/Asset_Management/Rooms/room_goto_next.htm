chambre_goto_suivante Avec cette fonction, vous pouvez faire en sorte que votre jeu passe à la salle suivante, telle qu'elle est répertoriée dans le gestionnaire de salles au moment où le jeu a été compilé. Si cette salle n'existe pas, une erreur sera lancée et le jeu sera forcé de se fermer. le jeu sera forcé de se fermer. Notez que la salle ne changera pas avant la fin de l'événement dans lequel la fonction a été appelée, donc tout code après l'appel de cette fonction sera toujours exécuté s'il se trouve dans le même événement. L'inconvénient est que tout code qui crée des instances qui est appelé après cette fonction, ne créera pas l'instance et doit donc être appelé avant le changement de salle. Cette fonction déclenchera également l'événement Room End. Syntaxe : room_goto_next() ; Les retours : N/A Exemple : if room_exists(room_next(room)) { room_goto_next() ; } Le code ci-dessus vérifiera s'il y a une autre pièce après la pièce actuelle et si c'est le cas, il ira dans cette pièce.
