couche_sprite_get_speed Cette fonction peut être utilisée pour obtenir la valeur actuelle du multiplicateur de vitesse de l'élément sprite. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), et la fonction renvoie une valeur réelle qui représente le multiplicateur de vitesse utilisé pour animer le sprite. La valeur par défaut est 1. Syntaxe : layer_sprite_get_speed(sprite_element_id) ; Arguments Description sprite_element_id La valeur d'identification unique de l'élément de sprite à partir duquel les informations doivent être obtenues. Les retours : Real Exemple : var lay_id = layer_get_id("sprite_sky") ; var spr_id = layer_sprite_get_id(lay_id, "Clouds") ; if layer_sprite_get_speed(spr_id) > 0 { layer_sprite_speed(spr_id, 0) ; } Le code ci-dessus récupère l'ID du calque nommé "sprite_sky" et l'utilise pour obtenir l'ID de l'élément sprite sur ce calque. Cet ID est ensuite utilisé pour vérifier la vitesse d'animation de l'élément et, si elle est supérieure à 0, elle est à 0.
