layer_sprite_get_speed Cette fonction peut être utilisée pour obtenir la valeur actuelle du multiplicateur de vitesse de l'élément sprite. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), et la fonction renverra la valeur réelle qui représente le multiplicateur de vitesse utilisé pour animer le sprite. La valeur par défaut est 1. Syntaxe:layer_sprite_get_speed(sprite_element_id); Argument Description sprite_element_id La valeur d'ID unique de l'élément de sprite à partir duquel obtenir les informations Retour:Real Exemple:var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_speed(spr_id) > 0     {     layer_sprite_speed(spr_id, 0);     } Le code ci-dessus obtiendra l'ID de la couche nommée "sprite_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément sprite sur cette couche. Cet identifiant est ensuite utilisé pour vérifier la vitesse d'animation de l'élément et s'il est supérieur à 0, il est mis à 0.
