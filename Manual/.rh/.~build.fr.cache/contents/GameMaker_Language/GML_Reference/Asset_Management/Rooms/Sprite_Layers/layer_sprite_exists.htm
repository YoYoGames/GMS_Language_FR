couche_sprite_existe Vous pouvez utiliser cette fonction pour vérifier si un élément sprite existe sur un calque donné. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la commande layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()) La fonction renvoie soit true si l'élément existe, soit false s'il n'existe pas. REMARQUE: Cette fonction fonctionne dans le cadre de la pièce cible actuelle - par défaut, la pièce dans laquelle la fonction est appelée - qui peut être définie à l'aide de la fonction layer_set_target_room(). Syntaxe : layer_sprite_exists(layer_id, sprite_element_id) Arguments Description identifiant de la couche La valeur d'identification unique de la couche à cibler (ou le nom de la couche sous forme de chaîne). sprite_element_id La valeur d'identification unique de l'élément du sprite à vérifier. Les retours : Booléen Exemple : var lay_id = layer_get_id("Asset_Trees") ; if layer_sprite_exists(lay_id, global.TreesSprites) { layer_sprite_destroy(lay_id, global.TreesSprites) ; } Le code ci-dessus vérifie le calque "Asset_trees" pour voir si l'élément sprite donné existe et si c'est le cas, alors il est détruit (mais pas le calque).
