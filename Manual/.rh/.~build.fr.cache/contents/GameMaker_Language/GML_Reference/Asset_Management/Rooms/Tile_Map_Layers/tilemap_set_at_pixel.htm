tilemap_set_at_pixel Cette fonction peut être utilisée pour définir une cellule dans l'élément de carte de tuiles sur le calque sur une nouvelle tuile en utilisant la position réelle de la tuile à changer dans la pièce. Vous donnez l'ID d'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id() ), puis les données de mosaïque à définir ainsi que la position dans la pièce. Vous pouvez récupérer des données de mosaïque à l'aide de la fonction tilemap_get_at_pixel() puis utilisez le tile_ fonctions pour le changer avant de le régler à nouveau à l'aide de cette fonction. La fonction retournera true si la tuile a été correctement installée et false s'il y avait un problème et qu'il n'a pas été réglé. Syntaxe:tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y) Argument Description tilemap_element_id La valeur d'identifiant unique de l'élément de mosaïque à modifier tiledata Les données du jeu de tuiles à définir xcell La position x (dans la pièce) ycell La position y (dans la pièce) Retour:Boolean Exemple:var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var data = tilemap_get_at_pixel(map_id, 4, 4); data = tile_set_flip(data, true); tilemap_set_at_pixel(map_id, data, 4, 4); Le code ci-dessus obtient l'ID de la carte de tuiles sur la couche "Tiles_Sky" et l'utilise ensuite pour obtenir les données de la tuile à la position (4, 4). Ces données de tuile sont ensuite retournées avant d'être utilisées pour redéfinir la tuile sur la carte de tuiles.
