tilemap_set_at_pixel Cette fonction peut être utilisée pour définir une cellule dans l'élément de carte de carreaux sur la couche à un nouveau carreau en utilisant la position réelle du carreau à changer dans la pièce. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), puis les données de la tuile à définir ainsi que la position dans la pièce. Vous pouvez consulter le site pouvez récupérer les données des tuiles à l'aide de la fonction tilemap_get_at_pixel() puis utiliser les fonctions tile_ pour les modifier avant de les définir à nouveau à l'aide de cette fonction. La fonction renvoie vrai si la tuile a été définie avec succès et faux s'il y a eu un problème et qu'elle n'a pas été définie. Syntaxe : tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y) Arguments Description tilemap_element_id La valeur d'identification unique de l'élément de la carte des tuiles à modifier. données en mosaïque Les données du jeu de tuiles à définir xcell La position x (dans la pièce) ycell La position y (dans la pièce) Les retours : Booléen Exemple : var lay_id = layer_get_id("Tiles_sky") ; var map_id = layer_tilemap_get_id(lay_id) ; var data = tilemap_get_at_pixel(map_id, 4, 4) ; data = tile_set_flip(data, true) ; tilemap_set_at_pixel(map_id, data, 4, 4) ; Le code ci-dessus obtient l'ID de la carte des tuiles sur la couche "Tiles_Sky" et l'utilise ensuite pour obtenir les données de la tuile à la position (4, 4). Les données de cette tuile sont ensuite retournées avant d'être utilisées pour remettre la tuile sur la carte des tuiles.
