couche_force_draw_depth Cette fonction force tous les calques à être dessinés à la profondeur z spécifiée. Cela ne change pas l'ordre dans lequel les couches sont rendues et elles seront toujours dessinées dans l'ordre de profondeur, cela change simplement la valeur z utilisée. En général, vous n'avez pas mais si vous avez des couches qui ont une profondeur en dehors de la plage légale (-16000 à 16000), elles ne seront pas rendues, donc vous pouvez forcer la profondeur Z à une valeur raisonnable - 0 par exemple - et elles seront toutes rendues correctement. Notez que ceci n'est généralement utilisé que pour les projets hérités de versions antérieures de GameMaker où vous pouviez avoir des profondeurs de dessin supérieures ou inférieures à la plage de couches autorisée. Syntaxe : layer_force_draw_depth(force, profondeur) Arguments Description force S'il faut activer (mettre à true ) ou désactiver (mettre à false) le forçage de la profondeur Z profondeur La nouvelle profondeur Z Les retours : N/A Exemple : if !layer_is_draw_depth_forced() { layer_force_draw_depth(true, 0) ; } Le code ci-dessus vérifie si la profondeur Z de la couche est forcée ou non et si elle ne l'est pas, il fixe la profondeur Z à 0 et l'active.
