layer_sequence_is_finished Avec cette fonction, vous fournissez l'ID d'élément de séquence - tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de calque - et il vérifiera si la séquence est terminée ou non, en retournant true si c'est le cas, et false si ce n'est pas le cas. Notez que cela n'est applicable que lorsque la séquence n'est pas définie sur boucle ou ping-pong en mode de lecture. Syntaxe:layer_sequence_is_finished(sequence_element_id) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler Retour:Boolean Exemple:if layer_sequence_is_finished(title_sequence) != 0     {     alarm[0] = room_speed * 3;     layer_sequence_play(title_sequence);     } Le code ci-dessus vérifie si l'ID d'élément de séquence stocké dans la variable "title_sequence" est terminé, et s'il l'est, il recommence à jouer et déclenche une alarme.
