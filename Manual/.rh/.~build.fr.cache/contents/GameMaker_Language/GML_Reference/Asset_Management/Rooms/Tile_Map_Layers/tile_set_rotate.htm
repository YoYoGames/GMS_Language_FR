tile_set_rotate Cette fonction peut être utilisée pour définir un ensemble donné de données de mosaïque pour faire pivoter la mosaïque de 90 degrés ou non. Vous donnez les données de tuile, qui peuvent être récupérées à l'aide de la fonction tilemap_get(), puis définissez l'argument de rotation sur soit true si vous voulez que la tuile pivote, ou false si vous voulez que la vignette soit dans son état par défaut, sans rotation. La fonction renverra un ensemble de données de mosaïque modifié qui peut ensuite être appliqué à l'aide du tilemap_set() fonction. Syntaxe:tile_set_rotate(tiledata, rotate) Argument Description tiledata les données de tuile à définir rotate Mis à true faire pivoter et false le laisser tel quel Retour:Real Exemple:var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var bool = !tile_get_rotate(data); data = tile_set_rotate(data, bool); tilemap_set(map_id, data, mx, my); Le code ci-dessus obtient l'ID de carte de tuiles de la couche donnée, puis obtient la position des cellules x et y pour la tuile sous la souris. Cette position est ensuite utilisée pour obtenir les données de tuile qui sont tournées puis utilisées pour redéfinir la tuile.
