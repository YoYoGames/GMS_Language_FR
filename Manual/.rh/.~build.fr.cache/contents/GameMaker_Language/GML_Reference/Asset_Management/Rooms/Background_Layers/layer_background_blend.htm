couche_fond_mélange Cette fonction contrôle le mélange (ou "teinte") du sprite d'arrière-plan. La valeur par défaut est -1 (qui représente la constante c_white, qui peut également être utilisée). Toute autre valeur (y compris les constantes de couleur internes comme c_red, ou c_aqua) mélangera la couleur spécifiée avec le sprite d'origine. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis définissez la couleur de mélange à utiliser. Vous pouvez voir ci-dessous un exemple de sprite qui a été mélangé avec différentes couleurs : Veuillez noter que vous devez essayer de limiter les mélanges sur la plate-forme HTML5 (sauf si vous utilisez WebGL), car chaque sprite mélangé doit être mis en cache séparément. mélangé doit être mis en cache séparément et le fait de disposer de nombreux sprites mélangés peut nuire aux performances (vous pouvez également définir la taille du cache à l'aide de la fonction sprite_set_cache_size()). Syntaxe : layer_background_blend(background_element_id, blend) ; Arguments Description id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à modifier. mélange La couleur à mélanger avec le sprite d'arrière-plan (par défaut, c_white). Les retours : N/A Exemple : var lay_id = layer_get_id("Background_sky") ; var back_id = layer_background_get_id(lay_id) ; layer_background_blend(back_id, c_aqua) ; Le code ci-dessus récupère la valeur ID de l'arrière-plan assigné au calque "Background_sky" et le teinte ensuite en une couleur.
