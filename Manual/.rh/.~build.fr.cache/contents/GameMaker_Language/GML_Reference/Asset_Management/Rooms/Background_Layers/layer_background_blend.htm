layer_background_blend Cette fonction contrôle la fusion (ou "teinte") du sprite d'arrière-plan et la valeur par défaut est -1 (qui représente la constante c_white, qui peut également être utilisé). Toute autre valeur (y compris les constantes de couleur internes comme c_red, ou c_aqua ) mélangera la couleur spécifiée avec le sprite d'origine. Vous donnez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), puis définissez la couleur de fusion à utiliser. Ci-dessous, vous pouvez voir un exemple de sprite qui a été mélangé avec différentes couleurs: Veuillez noter que vous devriez essayer de limiter le mélange sur la plate-forme HTML5 (sauf si vous utilisez WebGL), car chaque sprite mélangé doit être mis en cache séparément et ainsi avoir de nombreux sprites mélangés peut affecter les performances (vous pouvez également définir la taille du cache à l'aide de la fonction sprite_set_cache_size() ). Syntaxe:layer_background_blend(background_element_id, blend); Argument Description background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à modifier blend La couleur à mélanger avec l'image-objet d'arrière-plan (la valeur par défaut est c_white ) Retour:N/A Exemple:var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); layer_background_blend(back_id, c_aqua); Le code ci-dessus obtient la valeur d'identification de l'arrière-plan affectée au calque "Background_sky", puis le teinte en une couleur.
