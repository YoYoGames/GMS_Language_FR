tilemap_tileset Cette fonction vous permet de modifier la ressource de l'ensemble de tuiles affectée à un élément de carte de tuiles donné sur une couche. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), puis fournissez un index de jeu de tuiles et la carte de tuiles recevra le nouveau sprite. Syntaxe : tilemap_tileset(tilemap_element_id, tileset_index) Arguments Description tilemap_element_id La valeur d'identification unique de l'élément de la carte des tuiles à modifier. tileset_index Le nouvel index du jeu de tuiles à utiliser Les retours : N/A Exemple : var lay_id = layer_get_id("Tiles_trees") ; var tile_id = layer_tilemap_get_id(lay_id) ; if tilemap_get_tileset(tile_id) != ts_Nighttime { tilemap_tileset(tile_id, ts_Nighttime) ; } Le code ci-dessus vérifie le jeu de tuiles actuel attribué à la carte des tuiles sur la couche "Tiles_trees" et si ce n'est pas "ts_Nighttime", alors ce jeu de tuiles est attribué à la carte des tuiles.
