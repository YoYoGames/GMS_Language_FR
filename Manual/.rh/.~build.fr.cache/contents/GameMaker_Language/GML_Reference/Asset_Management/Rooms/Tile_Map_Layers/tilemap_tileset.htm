tilemap_tileset À l'aide de cette fonction, vous pouvez modifier la ressource d'ensemble de tuiles affectée à un élément de carte de tuiles donné sur une couche. Vous donnez l'ID d'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id() ), puis fournissez un index de jeu de tuiles et la carte de tuiles recevra le nouveau sprite. Syntaxe:tilemap_tileset(tilemap_element_id, tileset_index) Argument Description tilemap_element_id La valeur d'identifiant unique de l'élément de mosaïque à modifier tileset_index Le nouvel index de jeu de tuiles à utiliser Retour:N/A Exemple:var lay_id = layer_get_id("Tiles_trees"); var tile_id = layer_tilemap_get_id(lay_id); if tilemap_get_tileset(tile_id) != ts_Nighttime     {     tilemap_tileset(tile_id, ts_Nighttime);     } Le code ci-dessus vérifie l'ensemble de tuiles actuel affecté à la carte de tuiles sur la couche "Tiles_trees" et si ce n'est pas "ts_Nighttime" alors cet ensemble de tuiles est affecté à la carte de tuiles.
