angle_de_séquence_de_couche Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle renverra l'angle actuel de l'élément de séquence dans la salle de jeu. Notez que les angles sont retournés en degrés, et que 0º est à droite, 90º est en haut, 180º est à gauche et 270º est en bas. Syntaxe : layer_sequence_get_angle(sequence_element_id) Arguments Description id_élément_de_séquence La valeur d'identification unique de l'élément de séquence à cibler. Les retours : Real Exemple : var _ang = layer_sequence_get_angle(title_sequence) if _ang > 0 { _ang -= 1 ; layer_sequence_angle(séquence_titre, _ang) ; } Le code ci-dessus récupère l'angle actuel de l'élément de la séquence avec l'ID stocké dans la variable "title_sequence", et s'il est différent de 0, alors 1 est soustrait de l'angle actuel.
