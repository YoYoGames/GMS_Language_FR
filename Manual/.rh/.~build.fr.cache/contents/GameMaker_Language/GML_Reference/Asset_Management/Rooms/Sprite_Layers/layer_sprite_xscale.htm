layer_sprite_xscale En utilisant cette fonction, vous pouvez changer si l'élément sprite donné sur une couche doit être mis à l'échelle le long de l'axe x ou non. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur de l'échelle. Une échelle de 1 indique qu'il n'y a pas de mise à l'échelle (1:1), les valeurs plus petites seront mises à l'échelle (0,5, par exemple, correspondra à la moitié de la largeur du sprite utilisé), des valeurs plus grandes augmenteront l'échelle, et des valeurs négatives retourneront le sprite et le mettront à l'échelle, sauf si la valeur utilisée est exactement -1 (dans ce cas, le sprite utilisé est simplement retourné de droite à gauche autour de sa position d'origine, sans mise à l'échelle). Syntaxe : layer_sprite_xscale(sprite_element_id, xscale) ; Arguments Description Identifiant de l'élément de sprite La valeur d'identification unique de l'élément du sprite à modifier. xscale La valeur de l'échelle x (1 par défaut) Les retours : N/A Exemple : var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees") ; if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1 { layer_sprite_xscale(asset_sprite, 1) ; layer_sprite_yscale(asset_sprite, 1) ; } Le code ci-dessus vérifiera l'élément sprite attribué au calque sur lequel se trouve l'instance qui exécute le code et s'il est mis à l'échelle dans l'une ou l'autre direction, il définira l'échelle de l'axe des x et l'échelle de l'axe des y à 1.
