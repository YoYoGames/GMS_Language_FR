layer_sequence_play Avec cette fonction, vous pouvez démarrer la lecture de la séquence donnée. Vous fournissez l'ID d'élément de séquence renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche et la fonction jouera la séquence, que vous pouvez ensuite mettre en pause si nécessaire en utilisant la fonction layer_sequence_pause(). IMPORTANT! Si votre séquence contient des instances, ces instances ne doivent pas changer leurimage_xscale /image_yscale / image_angle /x /y variables car elles seront écrasées lorsque la séquence sera mise à jour à chaque étape après le début de la lecture. Syntaxe:layer_sequence_play(sequence_element_id) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler Retour:N/A Exemple:if keyboard_check_pressed(ord("P"))     {     global.Pause = !global.Pause;     var a = layer_get_all_elements(layer);     for (var i = 0; i < array_length(a); i++;)         {         if layer_get_element_type(a[i]) == layerelementtype_sequence             {             if global.Pause                 {                 layer_sequence_pause(a[i]);                 }             else                 {                 layer_sequence_play(a[i]);                 }             }         }     } Le code ci-dessus vérifie si le jeu a été mis en pause ou non lorsqu'une touche est enfoncée. Si le jeu est mis en pause, il parcourt tous les éléments de la séquence sur le calque actuel (le calque de l'instance appelante) et met en pause leur lecture, et si le jeu n'est pas mis en pause, la boucle recommencera sa lecture.
