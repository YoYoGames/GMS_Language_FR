room_set_viewport Avec cette fonction, vous pouvez définir les propriétés du port de vue pour n'importe quelle pièce de votre jeu , à l'exception de la pièce actuelle. Vous fournissez l'index de la pièce dans laquelle le port de vue doit être défini, l'index du port de vue (de 0 à 7) et ensuite si le port de vue doit être visible au début de la pièce ou non (défini à true pour rendre le port visible et false sinon). au début de la pièce ou non (mettez true pour rendre le port visible, false sinon). Ensuite, vous définissez les coordonnées x et y (correspondant à la position supérieure gauche du port de vue), puis la largeur et la hauteur en pixels. Syntaxe : room_set_viewport(rm, vind, vis, xport, yport, wport, hport) ; Arguments Description rm L'index de la pièce à définir vind L'index du port de vue à définir vis La visibilité du port de vue(true est visible, false est invisible) xport La position x pour le port de vue dans la chambre yport La position y du port de vue dans la pièce wport La largeur (en pixels) du port d'affichage hport La hauteur (en pixels) du port de vue Les retours : N/A Exemple : global.myroom = room_add() ; room_set_width(global.myroom, 640) ; room_set_height(global.myroom, 480) ; room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480) ; Cela va définir les propriétés du port de vue [0] dans la pièce indexée dans la variable "global.myroom".
