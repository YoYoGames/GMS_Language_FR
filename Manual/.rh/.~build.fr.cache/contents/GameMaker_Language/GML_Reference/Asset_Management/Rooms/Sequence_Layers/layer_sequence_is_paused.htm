layer_sequence_is_paused Avec cette fonction, vous fournissez l'ID d'élément de séquence - tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et il vérifiera si la séquence est actuellement en pause ou non, en retournant true s'il est mis en pause, et false si ce n'est pas le cas. Syntaxe:layer_sequence_is_paused(sequence_element_id) Argument Description sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler Retour:Boolean Exemple:if layer_sequence_is_paused(title_sequence) != 0     {     layer_sequence_play(title_sequence);     } Le code ci-dessus vérifie si l'ID d'élément de séquence stocké dans la variable "title_sequence" est mis en pause, et s'il l'est, il démarre la lecture.
