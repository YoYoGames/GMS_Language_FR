couche_get_script_end Cette fonction renvoie l'index de la fonction de script de la fonction assignée à exécuter à la fin du rendu de la couche donnée, ou elle renvoie -1 si aucune fonction n'est assignée. Vous fournissez la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom du calque (sous forme de chaîne de caractères - cela aura un impact sur les performances). Vous pouvez attribuer des fonctions de script à une couche avec layer_script_begin() et layer_script_end() . Syntaxe : layer_get_script_end(layer_id) ; Arguments Description identifiant de la couche La valeur d'identification unique de la couche à cibler (ou le nom de la couche sous forme de chaîne). Les retours : Index de la fonction script (réel) ou -1 si aucun script n'est assigné. Exemple : si layer_get_script_end(layer) == -1 { layer_script_end(layer, scr_ResetShaderValues) ; } Le code ci-dessus vérifie si le calque de l'instance qui exécute le code a une fonction de script assignée et si ce n'est pas le cas, une fonction est assignée.
