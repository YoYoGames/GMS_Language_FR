couche_arrière-plan_get_alpha Cette fonction peut être utilisée pour obtenir la valeur alpha de l'élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), et la fonction renvoie une valeur comprise entre 0 (totalement transparent) et 1 (totalement opaque). Syntaxe : layer_background_get_alpha(background_element_id) ; Arguments Description id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à partir duquel les informations doivent être obtenues. Les retours : Réel (de 0 à 1) Exemple : var lay_id = layer_get_id("Background_sky") ; var back_id = layer_background_get_id(lay_id) ; if layer_background_get_alpha(back_id) < 0.1 { layer_background_destroy(back_id) ; } Le code ci-dessus obtiendra l'ID du calque nommé "Background_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier l'alpha de l'élément et s'il est inférieur à 0,1, l'élément de la couche est détruit. est détruit.
