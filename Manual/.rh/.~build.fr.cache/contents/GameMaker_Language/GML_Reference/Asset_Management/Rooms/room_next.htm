room_next Avec cette fonction, vous pouvez récupérer l'index de la pièce après l'entrée de la pièce dans la fonction. Par exemple, vous pouvez utiliser la variable room pour obtenir l'index de la pièce actuelle, puis utilisez cette fonction pour trouver la pièce qui la suit, comme indiqué dans le Gestionnaire de salle. S'il n'y a pas de place après celle que vous avez entrée, -1 est renvoyé. Notez que cette fonction ne reconnaîtra ni ne prendra en compte les pièces qui ont été ajoutées dynamiquement à l'aide deroom_add() ouroom_duplicate(). Syntaxe:room_next(numb); Argument Description numb L'index de la pièce pour obtenir la suivante après. Retour:Index Exemple:if room_next(room) != -1     {     room_goto_next());     } Le code ci-dessus vérifiera si la pièce suivante existe et si c'est le cas, elle y ira.
