chambre_suivante Avec cette fonction, vous pouvez récupérer l'index de la pièce après l'entrée de la pièce dans la fonction. Par exemple, vous pouvez utiliser la variable pièce pour obtenir l'index de la pièce actuelle, puis utiliser cette fonction pour trouver la pièce qui la suit, telle qu'elle est répertoriée dans le Room Manager. S'il n'y a pas de pièce après celle que vous avez saisie, la fonction renvoie -1. Notez que cette fonction ne reconnaîtra pas ou ne prendra pas en compte les salles qui ont été ajoutées dynamiquement à l'aide de room_add() ou room_duplicate(). Syntaxe : chambre_suivante(numb) ; Arguments Description engourdi L'index de la pièce pour obtenir la suivante. Les retours : Index Exemple : if room_next(room) != -1 { room_goto_next()) ; } Le code ci-dessus vérifiera si la pièce suivante existe et si c'est le cas, il s'y rendra.
