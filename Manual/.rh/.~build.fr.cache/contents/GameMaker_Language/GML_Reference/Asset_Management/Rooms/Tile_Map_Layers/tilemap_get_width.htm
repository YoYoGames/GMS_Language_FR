tilemap_get_width Cette fonction vous permet de récupérer la largeur (en cellules) de l'élément de carte de tuiles. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la fonction renvoie la largeur (nombre de cellules). Syntaxe : tilemap_get_width(tilemap_element_id) ; Arguments Description tilemap_element_id La valeur d'identification unique de l'élément de la carte des tuiles dont on veut obtenir la largeur. Les retours : Entier Exemple : var lay_id = layer_get_id("Tiles_Walls") ; var map_id = layer_tilemap_get_id(lay_id) ; var _w = tilemap_get_width(map_id) ; var _h = tilemap_get_height(map_id) ; total_tiles = (_w * _h) ; Le code ci-dessus utilise l'ID de la carte des tuiles récupérée pour obtenir la largeur et la hauteur de la carte des tuiles, puis utilise ces valeurs pour définir une variable d'instance.
