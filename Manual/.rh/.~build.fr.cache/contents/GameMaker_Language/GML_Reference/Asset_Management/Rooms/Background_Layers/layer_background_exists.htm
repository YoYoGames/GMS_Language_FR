layer_background_exists Vous pouvez utiliser cette fonction pour vérifier et voir si un élément d'arrière-plan existe sur un calque donné. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ) ou le nom du calque (sous forme de chaîne - cela aura un impact sur les performances) et l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ) et la fonction retournera soit true si l'élément existe, ou false si ça ne fait pas. NOTE: Cette fonction fonctionne dans le cadre de la salle cible actuelle - par défaut la salle dans laquelle la fonction est appelée - qui peut être définie à l'aide de la fonction layer_set_target_room (). Syntaxe:layer_background_exists(layer_id, background_element_id) Argument Description layer_id La valeur d'ID unique du calque à cibler (ou le nom du calque sous forme de chaîne) background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à vérifier Retour:Boolean Exemple:var lay_id = layer_get_id("Background_trees"); if layer_background_exists(lay_id, global.TreesBackground)     {     layer_background_destroy(lay_id, global.TreesBackground);     } Le code ci-dessus vérifie le calque "Background_trees" pour voir si l'élément d'arrière-plan donné existe et si c'est le cas, il est détruit (mais pas le calque).
