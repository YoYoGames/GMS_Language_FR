couche_arrière-plan_existe Vous pouvez utiliser cette fonction pour vérifier si un élément d'arrière-plan existe sur une couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne de caractères - cela aura un impact sur les performances) et l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la commande layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()) et la fonction renvoie soit true si l'élément existe, soit false s'il n'existe pas. NOTE: Cette fonction fonctionne dans le cadre de la pièce cible actuelle - par défaut la pièce dans laquelle la fonction est appelée - qui peut être définie en utilisant la fonction layer_set_target_room(). Syntaxe : layer_background_exists(layer_id, background_element_id) Arguments Description identifiant de la couche La valeur d'identification unique de la couche à cibler (ou le nom de la couche sous forme de chaîne). id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à vérifier. Les retours : Booléen Exemple : var lay_id = layer_get_id("Background_trees") ; if layer_background_exists(lay_id, global.TreesBackground) { layer_background_destroy(lay_id, global.TreesBackground) ; } Le code ci-dessus vérifie la couche "Background_trees" pour voir si l'élément d'arrière-plan donné existe et si c'est le cas, alors il est détruit (mais pas la couche).
