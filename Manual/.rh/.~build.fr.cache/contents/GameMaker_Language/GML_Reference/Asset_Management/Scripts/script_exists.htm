script_existe Cette fonction retournera vrai ou faux selon que le script ou la fonction de script avec l'index donné existe ou non. Note, qu'il ne s'agit pas d'une chaîne, mais plutôt du nom de l'actif qui contient l'index unique de chaque script (tel qu'il apparaîtrait dans l'IDE) ou de la variable du nom de la fonction de script, telle qu'elle est définie dans l'actif du script (notez que ceci ne fonctionne pas pour les variables de méthode). Pour plus d'informations sur les scripts, voir L'éditeur de script. Syntaxe : script_exists(scr) ; Arguments Description scr Le nom du script que vous voulez vérifier. Les retours : Booléen Exemple : script[0] = -1 ; script[1] = AI_Gauche ; script[2] = AI_Droit ; var script_num = choisir(0, 1, 2) ; if script_exists(script[script_num]) { script_execute(script[script_num]) ; } L'exemple ci-dessus ajoute deux fonctions de script et une valeur dans un tableau, puis procède à l'obtention d'un nombre aléatoire et l'utilise pour choisir une fonction de script à exécuter, sauf si le -1 est choisi, auquel cas rien ne se passera.
