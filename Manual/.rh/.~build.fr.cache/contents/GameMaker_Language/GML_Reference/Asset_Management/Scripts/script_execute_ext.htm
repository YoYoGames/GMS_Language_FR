script_execute_ext Cette fonction fonctionne de la même manière que la fonction script_execute (). Vous seul pouvez fournir un tableau contenant les arguments requis pour la fonction / le script appelé. Vous pouvez également fournir deux arguments facultatifs à la fonction pour spécifier un décalage dans le tableau à partir duquel obtenir les arguments, ainsi que le nombre d'arguments à utiliser à partir du tableau (ce doit être un maximum de array_length - offset). Syntaxe:script_execute(scr, array_args, [offset], [num_args]); Argument Description scr Le nom de la fonction / du script que vous souhaitez appeler array_args Le tableau contenant les arguments de la fonction / du script [offset] OPTIONNEL! Le décalage dans le tableau d'arguments [num_args] OPTIONNEL! Le nombre d'arguments à (à partir du décalage) Retour:Will depend on the return value from the function/script being called Exemple:script_execute_ext(move_inst, move_array, floor(random(4)), 1); L'exemple de code ci-dessus utilisera script_execute_ext pour appeler la fonction donnée, en fournissant un tableau de quatre arguments, mais en n'utilisant qu'un seul d'entre eux au hasard. REMARQUE: Cette fonction ne peut pas être utilisée avec les fonctions intégrées. Retour: Scripts Suivant:script_exists © Copyright YoYo Games Ltd. 2021 Tous droits réservés
