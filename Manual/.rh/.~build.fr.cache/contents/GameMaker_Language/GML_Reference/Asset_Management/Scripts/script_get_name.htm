script_get_name Cette fonction renvoie le nom sous forme de chaîne du script spécifié. Ce nom est celui qui a été spécifié pour le script dans l'Asset Browser de la fenêtre principale de GameMaker Studio 2. Pour plus d'informations sur les scripts, consultez la rubrique L'éditeur de script. Syntaxe : script_get_name(scr) ; Arguments Description scr L'index du script dont vous voulez obtenir le nom. Les retours : Chaîne de caractères Exemple : scr_name = script_get_name(Help_File) ; L'exemple de code ci-dessus stockera le nom de l'index de script indiqué dans la variable "scr_name".
