script_get_name Cette fonction renverra le nom sous forme de chaîne du script spécifié. Ce nom est celui qui a été spécifié pour le script dans le navigateur de ressources de la GameMaker Studio 2. Pour plus d'informations sur les scripts, consultez l' éditeur de scripts. Syntaxe:script_get_name(scr); Argument Description scr L'index du script dont vous souhaitez obtenir le nom. Retour:String Exemple:scr_name = script_get_name(Help_File); L'exemple de code ci-dessus stockera le nom de l'index de script indiqué dans la variable "scr_name".
