Orientation de l'auditeur audio Avec cette fonction, vous pouvez changer l'orientation de l'écouteur donné dans l'espace audio 3D. L'index par défaut de l'écouteur est 0, mais vous pouvez utiliser la fonction audio_get_listener_info() pour pour obtenir les différents indices disponibles pour la plate-forme cible. Le vecteur d'observation et le vecteur haut sont basés sur les valeurs résolues à partir des positions relatives x, y et z données, et ont par défaut la valeur (0, 0, -1) pour le vecteur d'observation et (0, 1, 0) pour le vecteur haut. pour le vecteur haut, comme le montre l'illustration ci-dessous : En changeant l'orientation de l'auditeur avec cette fonction, vous modifiez la façon dont les sons créés par les émetteurs audio autour de la salle de jeu sont perçus par le joueur de votre jeu. Syntaxe : audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z) ; Arguments Description indice L'écouteur dont l'orientation doit être définie. lookat_x Le vecteur d'aspect x (par défaut 0). lookat_y Le vecteur d'aspect y (par défaut 0). lookat_z Le vecteur d'aspect z (par défaut -1). up_x Le vecteur x up (par défaut 0). up_y Le vecteur y up (par défaut 1). up_z Le vecteur z vers le haut (par défaut 0).
