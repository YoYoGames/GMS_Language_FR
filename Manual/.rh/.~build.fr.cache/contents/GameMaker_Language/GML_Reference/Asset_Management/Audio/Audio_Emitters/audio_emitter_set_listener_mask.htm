audio_emitter_set_listener_mask Cette fonction peut être utilisée pour définir le masque binaire d'un emitter afin que tous les sons joués par l'emitter ne soient lus que par les auditeurs spécifiés. Vous pouvez créer un masque binaire en utilisant les fonctions audio_get_listener_info() et puis en utilisant le bitwise ou ("|") pour créer un masque personnalisé avec les auditeurs à partir desquels vous souhaitez que le son soit joué, et enfin en appliquant ce masque personnalisé à l'émetteur. Ce masque remplacera le masque par défaut ou celui que vous avez pu définir à l'aide de la fonction audio_set_listener_mask(). Syntaxe : audio_emitter_set_listener_mask(emitterID, mask) ; Arguments Description emitterID L'identifiant unique de l'émetteur dont il faut définir le masque. masque Les données bitmask à définir pour le son Les retours : N/A Exemple : s_emit = audio_emitter_create() ; audio_emitter_set_listener_mask(snd, global.PlayerMask) ; Le code ci-dessus crée un émetteur audio, puis définit le ou les écouteurs à partir desquels il faut jouer en utilisant les données du masque stockées dans une variable globale.
