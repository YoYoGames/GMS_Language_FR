audio_get_listener_count Certaines plates-formes cibles autorisent plusieurs écouteurs, il est donc important que vous sachiez combien la cible en possède avant de modifier ou d'utiliser différents écouteurs. Cette fonction renverra le nombre d'écouteurs disponibles. Syntaxe:audio_get_listener_count(); Retour:Integer Exemple:global.listener_num= audio_get_listener_count(); Le code ci-dessus obtient le nombre d'écouteurs disponibles et stocke la valeur de retour dans une variable globale.
