audio_sound_set_track_position Cette fonction permet de définir la position (en secondes) du son dans le fichier audio. Le son peut être une instance unique d'un son (l'index des sons individuels joués peut être stocké dans une variable lors de l'utilisation de la fonction audio_play_sound() ou audio_play_sound_at() ) ou un actif sonore. S'il s'agit d'un ID de son unique que vous utilisez, la position changera immédiatement S'il s'agit d'un actif du navigateur d'actifs, toutes les lectures ultérieures du son en question commenceront au nouveau moment. La fonction ignorera toute valeur donnée qui est plus longue que la longueur du son donné pour la valeur "index". valeur. Syntaxe : audio_sound_set_track_position(index, time) ; Arguments Description indice L'indice du son à modifier. temps Le temps (en secondes) sur lequel il faut régler le point de départ. Les retours : N/A Exemple : var snd = audio_play_sound(snd_MainTrack, 0, false) ; audio_sound_set_track_position(snd, 32) ; Le code ci-dessus joue un son et utilise ensuite la valeur ID retournée pour définir la position de départ du son à 32 secondes.
