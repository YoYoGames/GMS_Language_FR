audio_sound_set_listener_mask Cette fonction peut être utilisée pour définir le masque binaire d'un son afin qu'il ne soit lu que par les auditeurs spécifiés. Vous pouvez créer un masque binaire à l'aide des fonctions audio_get_listener_info() et puis en utilisant la fonction bitwise ou ("|") pour créer un masque personnalisé avec les auditeurs à partir desquels vous souhaitez que le son soit lu, puis en appliquant ce masque personnalisé au son après l'avoir lu. Ce masque remplacera le masque par défaut ou celui que vous avez pu définir à l'aide de la fonction audio_set_listener_mask(). REMARQUE : Cette fonction ne peut être utilisée sur un son qu'après qu'il ait été réglé pour être joué et nécessite le soundID tel que renvoyé par la fonction utilisée pour jouer le son. Syntaxe : audio_sound_set_listener_mask(soundID, mask) ; Arguments Description soundID L'ID unique du son dont il faut définir le masque masque Les données bitmask à définir pour le son Les retours : N/A Exemple : var snd = audio_play_sound(snd_PlayerDead, 10, false) ; audio_sound_set_listener_mask(snd, global.PlayerMask) ; Le code ci-dessus joue un son puis définit le(s) écouteur(s) à partir duquel/desquels le son sera joué en utilisant les données du masque stockées dans une variable globale.
