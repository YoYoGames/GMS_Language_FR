audio_group_load Cette fonction chargera en mémoire tous les sons qui sont marqués comme appartenant au groupe audio donné. La fonction retournera true si le chargement est initié et false si le groupe est invalide, ou s'il a déjà été marqué pour le chargement. La fonction est asynchrone, de sorte que votre jeu continuera à fonctionner pendant que l'audio est chargé en arrière-plan. Cela signifie qu'elle déclenchera également un événement Asynchronous Load/Save Event pour vous informer que le groupe entier a été chargé en mémoire et que les sons peuvent maintenant être utilisés. Syntaxe : audio_group_load(groupID) ; Arguments Description groupID L'index du groupe audio à charger (tel que défini dans la fenêtre Groupes audio). Les retours : Booléen Exemple : if !audio_group_is_loaded(audiogroup_level1) { audio_group_loaded(audiogroup_level1) ; } Le code ci-dessus vérifie si un groupe audio a été chargé et si non, il le charge.
