création d'un flux audio Avec cette fonction, vous pouvez créer un nouvel index sonore qui peut ensuite être utilisé dans les fonctions audio ordinaires pour diffuser de l'audio directement à partir d'une source de fichier OGG externe. La fonction requiert le nom du fichier (qui peut être un fichier inclus, par exemple). renvoie le nouvel index sonore à utiliser. Notez que lorsque vous n'avez plus besoin du son, vous devez appeler la fonction audio_destroy_stream() avec l'index du son pour le supprimer de la mémoire, sinon vous risquez d'obtenir une fuite de mémoire qui ralentira et finira par faire planter votre jeu. REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5. Syntaxe : audio_create_stream(nom du fichier) ; Arguments Description nom de fichier Le fichier (OGG uniquement) à partir duquel le flux audio sera diffusé. Les retours : Indice sonore Exemple : snd = audio_create_stream("Music/Track1.ogg") ; audio_play_sound(snd, 0, true) ; Le code ci-dessus crée un nouvel indice sonore dans la variable "snd" à partir du fichier donné, puis joue ce son.
