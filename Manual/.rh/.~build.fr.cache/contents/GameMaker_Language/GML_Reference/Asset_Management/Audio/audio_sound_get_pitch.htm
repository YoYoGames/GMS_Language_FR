audio_sound_get_pitch Cette fonction peut être utilisée pour obtenir le pitch d'un son donné. Le son peut être référencé à partir d'un index pour un son individuel en cours de lecture qui a été stocké dans une variable lors de l'utilisation du audio_play_sound() ou audio_play_sound_at() fonctions, ou une ressource sonore réelle à partir du navigateur de ressources. Syntaxe:audio_sound_get_pitch(index); Argument Description index L'indice du son pour obtenir la hauteur. Retour:Real Exemple:if audio_sound_get_pitch(snd_Explode) != 1    {    audio_sound_pitch(snd_Explode, 1);    } Le code ci-dessus changera la hauteur de l'audio lu à partir du son indexé comme "snd_Explode" si sa hauteur n'est pas égale à 1.
