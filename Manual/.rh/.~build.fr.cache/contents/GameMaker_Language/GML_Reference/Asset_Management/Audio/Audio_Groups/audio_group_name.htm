nom_groupe_audio Cette fonction renvoie une chaîne de caractères contenant le nom du groupe audio donné à afficher ou à vérifier. Lorsque vous définissez un groupe audio dans les Options du jeu, vous lui donnez un "nom" unique qui est en fait une constante à utiliser comme valeur d'ID pour le groupe. Cette fonction ne fait que prendre l'ID et retourner une chaîne de caractères contenant le nom de l'ID que vous avez donné. Syntaxe : audio_group_name(groupID) ; Arguments Description groupID La constante de la valeur d'index du groupe audio à vérifier (comme défini dans la fenêtre Groupes audio). Les retours : Chaîne de caractères Exemple : var name= nom_du_groupe_audio(audiogroup_level1) ; draw_text(32, 32, "Now Playing Group : " + name) ; Le code ci-dessus récupère le nom de la constante de groupe audio donnée et l'affiche à l'écran.
