audio_group_name Cette fonction renverra une chaîne contenant le nom du groupe audio donné pour l'affichage ou la vérification. Lorsque vous définissez un groupe audio dans les options de jeu, vous donnez un « nom » unique qui est vraiment une constante à utiliser comme ID v aleur pour le groupe. Tout ce que cette fonction fait est de prendre l'ID et de renvoyer une chaîne du nom d'ID que vous avez donné. Syntaxe:audio_group_name(groupID); Argument Description groupID La constante de valeur d'index du groupe audio à vérifier (telle que définie dans la fenêtre Groupes audio ) Retour:String Exemple:var name= audio_group_name(audiogroup_level1); draw_text(32, 32, "Now Playing Group: " + name); Le code ci-dessus récupère le nom de la constante de groupe audio donnée et l'affiche à l'écran.
