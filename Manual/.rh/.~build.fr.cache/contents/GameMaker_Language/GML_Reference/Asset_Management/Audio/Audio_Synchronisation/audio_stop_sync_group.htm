audio_stop_sync_group Cette fonction arrêtera le groupe de synchronisation donné s'il est en cours de lecture, l'index du groupe étant la valeur retournée lorsque vous avez créé le groupe à l'aide de la fonction audio_create_sync_group(). REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5. Syntaxe : audio_stop_sync_group(group_index) ; Arguments Description groupe_index L'indice du groupe pour arrêter de jouer. Les retours : N/A Exemple : if mouse_check_button_pressed(mb_left) { audio_stop_sync_group(sg) ; } Le code ci-dessus vérifie la présence d'un clic de souris, et si un clic est détecté, il arrête le groupe de synchronisation indexé dans la variable "sg".
