audio_stop_sync_group Cette fonction arrêtera le groupe de synchronisation donné s'il est en cours de lecture, l'index du groupe étant la valeur renvoyée lorsque vous avez créé le groupe à l'aide de la fonction audio_create_sync_group(). REMARQUE: cette fonctionnalité n'est pas disponible pour la plate-forme cible HTML5. Syntaxe:audio_stop_sync_group(group_index); Argument Description group_index L'index du groupe pour arrêter la lecture. Retour:N/A Exemple:if mouse_check_button_pressed(mb_left)    {    audio_stop_sync_group(sg);    } Le code ci-dessus vérifie un clic de souris, et si un clic est détecté, il arrête le groupe de synchronisation indexé dans la variable "sg".
