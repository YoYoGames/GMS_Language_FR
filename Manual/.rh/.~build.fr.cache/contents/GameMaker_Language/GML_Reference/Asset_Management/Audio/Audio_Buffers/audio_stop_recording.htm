audio_stop_recording Cette fonction arrêtera l'enregistrement sur le canal de l'enregistreur donné (l'index du canal est renvoyé lorsque vous appelez la fonction audio_start_recording()). Lorsque vous arrêtez l'enregistrement, aucun autre événement asynchrone d'enregistrement audio ne sera déclenché pour le canal d'enregistreur donné. événement asynchrone pour vous assurer que vous avez capturé toutes les données. REMARQUE : la plupart des plates-formes , à l'exception de HTML5, prennent en charge l'enregistrement audio sous une forme ou une autre, mais cela ne signifie pas que tous les appareils le permettront, même si la plate-forme le permet. Vous devez donc toujours vérifier que la fonction audio_get_recorder_count() fonction renvoie une valeur supérieure à 0 pour vérifier que les périphériques d'enregistrement sont disponibles avant d'utiliser le reste des fonctions d'enregistrement. Syntaxe : audio_stop_recording(channel_index) ; Arguments Description canal_index L'index du canal de l'enregistreur à arrêter. Les retours : N/A Exemple : audio_stop_recording(audio_channel) ; Le code ci-dessus indique à GameMaker Studio 2 d'arrêter l'enregistrement sur l'index du canal audio donné.
