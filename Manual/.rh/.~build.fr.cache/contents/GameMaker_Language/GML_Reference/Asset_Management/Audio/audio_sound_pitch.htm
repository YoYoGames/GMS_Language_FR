audio_sound_pitch Cette fonction peut être utilisée pour changer la hauteur d'un son donné. Le son peut être référencé à partir d'un index pour un son individuel en cours de lecture qui a été stocké dans une variable lors de l'utilisation du audio_play_sound() ou audio_play_sound_at() fonctions, ou une ressource sonore réelle à partir du navigateur de ressources. S'il s'agit d'un index d'un son en cours de lecture, alors seule cette instance sera modifiée, mais lors de l'utilisation d'un élément audio à partir du navigateur de ressources, toutes les autres instances en cours de lecture de ce son seront modifiées. L'argument de hauteur est un multiplicateur de hauteur, en ce que la valeur d'entrée multiplie la hauteur actuelle par cette quantité, de sorte que la valeur par défaut de 1 n'est pas de changement de hauteur, tandis qu'une valeur inférieure à 1 abaissera la hauteur et supérieure à 1 augmentera la hauteur. terrain. Il est préférable d'utiliser de petits incréments pour cette fonction car toute valeur inférieure à 0 ou supérieure à 5 peut ne pas être audible de toute façon. Il est à noter que le changement de hauteur total autorisé est limité à (1/256) - 256 octaves, donc toute valeur supérieure ou inférieure à celle-ci ne sera pas enregistrée. REMARQUE: La valeur de blocage donnée ci-dessus est celle à laquelle GameMaker Studio 2 tente de limiter la plage, mais cette valeur n'est pas garantie sur toutes les plates-formes cibles. iOS, par exemple, se fixe à (1/256) - 8, vous devrez donc peut-être expérimenter sur chaque plate-forme cible et avoir différentes versions d'une ressource sonore, chacune pré-décalée, si vous avez besoin de valeurs d'octave supérieures ou inférieures. Syntaxe:audio_sound_pitch(index, pitch); Argument Description index L'indice du son à changer. pitch Le multiplicateur de hauteur (par défaut 1). Retour:N/A Exemple:var s_engine = audio_play_sound(snd_CarEngine, 10, false); switch (gear)    {    case 1: audio_sound_pitch(s_engine, 0.8); break;    case 2: audio_sound_pitch(s_engine, 0.9); break;    case 3: audio_sound_pitch(s_engine, 0.95); break;    case 4: audio_sound_pitch(s_engine, 1); break;    case 5: audio_sound_pitch(s_engine, 1.2); break;    } Le code ci-dessus changera la hauteur de l'audio joué à partir du son indexé dans la variable "s_engine" en fonction de la valeur de la variable "gear".
