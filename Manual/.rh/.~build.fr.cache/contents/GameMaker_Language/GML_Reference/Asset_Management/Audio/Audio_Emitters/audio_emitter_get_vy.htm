audio_emitter_get_vy Cette fonction renvoie la vitesse actuelle le long de l'axe y pour l'émetteur audio donné. Syntaxe:audio_emitter_get_vy(emitter); Argument Description emitter L'indice de l'émetteur à utiliser. Retour:Real Exemple:if audio_emitter_get_vy(emitter_player) != 0    {    audio_emitter_velocity(emitter_player, 0, 0, 0);    } Le code ci-dessus vérifie la vitesse y actuelle d'un émetteur donné et si elle n'est pas égale à 0, elle est définie sur 0.
