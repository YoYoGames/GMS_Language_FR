audio_system_is_available Cette fonction peut être utilisée pour vérifier et voir si le système audio a été initialisé ou si le contexte audio est en cours d'exécution. Sur toutes les plates-formes, cette fonction retournera true immédiatement après le démarrage du jeu lorsque le moteur audio est initialisé, sauf sur la cible HTML5. Sur HTML5, l'état du contexte audio peut changer à tout moment en fonction de l'entrée de l'utilisateur, du navigateur utilisé et de sa version, cette fonction peut donc être utilisée pour vérifier et voir si l'audio peut être lu ou non. Si la fonction retourne false (c'est-à-dire: l'état du contexte audio n'est pas en cours d'exécution), alors seuls les sons non diffusés peuvent être lus (mais ce n'est pas garanti, et vous devez supposer qu'aucun audio ne peut être lu), tandis que s'il revient true alors tout l'audio sera lu. Syntaxe:audio_system_is_available(); Retour:Boolean Exemple:if audio_system_is_available()     {     if audio_is_paused(global.Music)         {         audio_resume_sound(global.Music)         }     else         {         if !audio_is_playing(global.Music)             {             global.Music = audio_play_sound(snd_Music, 0, true);             }         }     } else     {     if audio_is_playing(global.Music)         {         audio_pause_sound(global.Music);         }     } Le code ci-dessus mettra en pause / réactivera une piste audio selon que le système audio est initialisé et disponible ou non.
