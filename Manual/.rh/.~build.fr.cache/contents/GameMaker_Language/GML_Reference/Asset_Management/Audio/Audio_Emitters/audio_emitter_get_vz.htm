audio_emitter_get_vz Cette fonction renvoie la vitesse actuelle le long de l'axe z pour l'émetteur audio donné. Syntaxe:audio_emitter_get_vz(emitter); Argument Description emitter L'indice de l'émetteur à utiliser. Retour:Real Exemple:if audio_emitter_get_vz(emitter_player) != 0    {    audio_emitter_velocity(emitter_player, 0, 0, 0);    } Le code ci-dessus vérifie la vitesse z actuelle d'un émetteur donné et si elle n'est pas égale à 0, elle est mise à 0.
