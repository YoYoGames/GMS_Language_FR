audio_emitter_get_listener_mask Cette fonction renvoie les données de masque de bits qui définissent les écouteurs audio à partir desquels un émetteur doit jouer des sons. Pour plus d'informations, consultez la section sur les écouteurs audio. Syntaxe : audio_emitter_get_listener_mask(emitterID) ; Arguments Description emitterID L'ID unique de l'émetteur dont on veut obtenir le masque. Les retours : Real Exemple : s_emit = audio_emitter_create() ; if audio_emitter_get_listener_mask(s_emit) != global.PlayerMask { audio_emitter_set_listener_mask(snd, global.PlayerMask) ; } Le code ci-dessus crée un émetteur, puis vérifie les données de masque de l'écouteur pour celui-ci, et si elles ne sont pas les mêmes que celles stockées dans une variable globale, il définit le(s) écouteur(s) à jouer en utilisant les données de masque stockées dans la variable globale.
