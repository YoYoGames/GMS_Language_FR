audio_sound_get_track_position Cette fonction obtiendra la position (en secondes) dans le fichier son pour la lecture du son. Le son ne peut être qu'une seule instance d'un son (l'index des sons individuels en cours de lecture peut être stocké dans une variable lors de l'utilisation du audio_play_sound() ou audio_play_sound_at() les fonctions). Syntaxe:audio_sound_get_track_position(index); Argument Description index L'indice du son pour obtenir la position de lecture. Retour:Real Exemple:if audio_sound_get_track_position(global.Music) != 0    {    audio_sound_set_track_position(global.Music, 0);    } Le code ci-dessus vérifie une piste pour obtenir sa position de départ et si ce n'est pas 0 seconde, il la définit sur 0 seconde.
