audio_get_type Lorsque vous ajoutez de l'audio à GameMaker Studio 2 il peut être diffusé en continu ou en mémoire. Si vous avez besoin de savoir si un index sonore donné est pour l'audio diffusé ou non, vous pouvez utiliser cette fonction qui renverra 1 pour le streaming, 0 pour le son en mémoire et -1 s'il y a une erreur ou que l'index ne pointe pas vers un ressource sonore valide. Syntaxe:audio_get_type(index); Argument Description index L'index du son à vérifier. Retour:Real Exemple:type = audio_get_type(snd_Music_1); Le code ci-dessus vérifie le type d'audio indexé dans la variable "snd_Music_1" et stocke la valeur renvoyée dans la variable "type".
