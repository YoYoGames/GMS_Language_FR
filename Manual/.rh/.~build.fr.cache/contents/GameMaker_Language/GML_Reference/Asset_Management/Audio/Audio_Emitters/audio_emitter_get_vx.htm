audio_emitter_get_vx Cette fonction renvoie la vitesse actuelle le long de l'axe x pour l'émetteur audio donné. Syntaxe : audio_emitter_get_vx(emitter) ; Arguments Description émetteur L'indice de l'émetteur à utiliser. Les retours : Real Exemple : if audio_emitter_get_vx(emitter_player) != 0 { audio_emitter_velocity(emitter_player, 0, 0, 0) ; } Le code ci-dessus vérifie la vitesse x actuelle d'un émetteur donné et si elle n'est pas égale à 0, elle est mise à 0.
