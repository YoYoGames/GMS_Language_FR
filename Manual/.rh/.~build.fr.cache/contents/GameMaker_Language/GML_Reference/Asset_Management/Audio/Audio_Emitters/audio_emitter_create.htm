création d'un émetteur audio Cette fonction crée un nouvel émetteur audio et renvoie son index. Cet index doit être stocké dans une variable pour toutes les fonctions ultérieures qui se rapportent à cet émetteur, et lorsqu'il n'est plus nécessaire, il doit être supprimé de la mémoire en utilisant la fonction fonction audio_emitter_free() pour éviter les fuites de mémoire qui peuvent éventuellement faire planter votre jeu. Syntaxe : audio_emitter_create() ; Les retours : Entier (ID de l'émetteur) Exemple : s_emit = audio_emitter_create() ; Le code ci-dessus crée un nouvel émetteur audio et assigne son index à la variable "s_emit".
