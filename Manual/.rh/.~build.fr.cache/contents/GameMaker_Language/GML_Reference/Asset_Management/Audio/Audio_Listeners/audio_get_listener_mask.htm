audio_get_listener_mask Cette fonction renverra les données de masque de bits qui définissent le masque par défaut (global) actuel pour les écouteurs audio. "mask"]);       global.Audio_Mask = global.Audio_Mask | m;       ds_map_destroy(info);       }    audio_set_listener_mask(mask);    } Le code ci-dessus obtient les données de masque d'écoute actuelles et les compare aux données stockées dans une variable globale. S'ils ne sont pas les mêmes, le code vérifie le nombre d'écouteurs disponibles puis les parcourt en boucle et obtient leurs bits de masque, qui sont ensuite combinés pour créer un seul masque qui est appliqué au système audio pour définir les auditeurs globaux.
