audio_master_gain Avec cette fonction, vous pouvez définir la valeur absolue du volume global de tous les sons et musiques. Il est basé sur une échelle linéaire de 0 (silencieux) à toute valeur supérieure à 0, bien que normalement vous considériez le volume maximum comme 1. Tout ce qui dépasse 1 peut être utilisé mais, en fonction du son utilisé et de la plate-forme compilée sur, vous pouvez obtenir une distorsion ou un écrêtage lors de la lecture du son. Cette fonction affectera le volume relatif de tous les sons et musiques joués à partir de votre jeu. Syntaxe:audio_master_gain(gain); Argument Description gain Valeur du volume global (0 à 1). Retour:N/A Exemple:if keyboard_check(vk_up)    {    if vol < 1 vol += 0.05;    audio_master_gain(vol);    } if keyboard_check(vk_down)    {    if vol > 0 vol -= 0.05;    audio_master_gain(vol);    } Le code ci-dessus vérifie les pressions sur les touches fléchées haut et bas, qui augmentent ou diminuent ensuite la variable «vol».
