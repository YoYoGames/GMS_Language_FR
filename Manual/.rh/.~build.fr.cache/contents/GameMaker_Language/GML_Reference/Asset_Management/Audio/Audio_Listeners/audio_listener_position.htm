audio_listener_position Cette fonction vous permet de modifier la position de l'auditeur dans l'espace audio 3D. L'image d'exemple ci-dessous montre la position par défaut de l'auditeur dans l'espace audio : NOTE : si vous avez plusieurs écouteurs, vous devez utiliser la fonction audio_listener_set_position(). Comme vous pouvez le voir, la position par défaut est (0, 0, 0) mais vous utiliserez normalement cette fonction pour déplacer l'écouteur avec l'objet joueur dans votre jeu et ainsi changer la façon dont l'audio créé par les émetteurs est entendu par le joueur. par exemple, dans l'image ci-dessous d'un jeu de haut en bas, l'instance du joueur définit l'écouteur qui fera en sorte que l'audio des différents émetteurs "change" à mesure que le joueur se déplace dans le niveau : Syntaxe : audio_listener_position(x, y, z) ; Arguments Description x La position x de l'écouteur (par défaut 0). y La position y de l'écouteur (par défaut 0). z La position z de l'écouteur (par défaut 0). Les retours : N/A Exemple : si vitesse > 0 { audio_listener_position(x, y, 0) ; } Le code ci-dessus vérifie si la vitesse de l'instance du lecteur est supérieure à 0 et si c'est le cas, il met à jour l'écouteur audio à la position x/y actuelle.
