progression du chargement du groupe audio Cette fonction vérifie la progression du chargement d'un groupe audio et renvoie une valeur (approximative) comprise entre 0 et 100. Syntaxe : audio_group_load_progress(groupID) ; Arguments Description groupID L'index du groupe audio à vérifier (tel que défini dans la fenêtre Groupes audio). Les retours : Real Exemple : if load { var pc = audio_group_load_progress(audiogroup_level1) ; draw_text(32, 32, "Loading : " + string(pc)) ; } Le code ci-dessus vérifie une variable et si elle renvoie un résultat positif, un texte sera affiché pour montrer la progression du chargement de l'audio.
