longueur_son_audio Cette fonction renvoie la durée du son donné en secondes. Le son peut être soit un index référencé d'un son individuel en cours de lecture qui a été stocké dans une variable lors de l'utilisation de la fonction audio_play_sound() ou audio_play_sound_at() ou un son réel provenant du navigateur d'actifs. Syntaxe : audio_sound_length(index) ; Arguments Description indice L'index du son à vérifier. Les retours : Real Exemple : var len ; len = audio_sound_length(snd_Beam) ; audio_play_sound(snd_Beam, 1, false) ; alarm[0] = room_speed * len ; Le code ci-dessus récupère la durée (en secondes) du son indexé dans la variable "snd_Beam", puis joue le son et déclenche une alarme à la fin de la lecture du son en utilisant la durée du son pour calculer le temps nécessaire à l'alarme. l'alarme.
