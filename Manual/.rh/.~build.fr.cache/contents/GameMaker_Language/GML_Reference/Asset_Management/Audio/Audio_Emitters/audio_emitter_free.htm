audio_émetteur_libre Avec cette fonction, vous pouvez supprimer l'emitter donné de la mémoire. Cette opération doit toujours être effectuée lorsque l'emitter n'est plus utilisé dans la room ou dans le jeu, c'est-à-dire lors de l'événement Destroy de l'instance l'instance ou dans l'événement de fin de salle, sinon vous pouvez vous retrouver avec une fuite de mémoire qui ralentira et finira par faire planter votre jeu. Syntaxe : audio_emitter_free(emitter) ; Arguments Description émetteur L'indice de l'émetteur à libérer. Les retours : N/A Exemple : si vies = 0 { audio_emitter_free(s_emit) ; room_goto(rm_Menu) ; } Le code ci-dessus vérifie la valeur de la variable globale "lives" et s'il retourne 0, il détruira l'emitter indexé dans la variable "s_emit" et ira ensuite dans la salle indexée dans la variable "rm_Menu".
