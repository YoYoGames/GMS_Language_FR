audio_stop_all Cette fonction arrêtera tous les sons en cours de lecture. Syntaxe:audio_stop_all(); Retour:N/A Exemple:if !global.SFX    {    audio_stop_all();    } Le code ci-dessus vérifie la variable globale "SFX" et si elle renvoie false, il arrêtera tous les sons en cours de lecture.
