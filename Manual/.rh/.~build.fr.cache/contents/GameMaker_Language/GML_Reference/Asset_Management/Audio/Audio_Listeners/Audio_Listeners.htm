Auditeurs audio Lorsque l'on traite des situations audio complexes, il peut être utile de définir l'emplacement de l'auditeur dans l'environnement audio. Lorsque vous travaillez avec les fonctions audio simples, il n'est normalement pas nécessaire que l'auditeur soit modifié de quelque manière que ce soit, Cependant, lorsque vous utilisez des émetteurs pour générer un son réaliste dans l'espace audio 3D, il est important que l'auditeur soit modifié et déplacé correctement, sinon le son ne sera pas dans la bonne position pour la personne qui joue. votre jeu. GameMaker Studio 2 propose les fonctions d'écoute suivantes : audio_listener_position vitesse de l'auditeur audio Orientation de l'auditeur audio_listener_get_data audio_listener_set_position audio_listener_set_velocity Orientation de l'auditeur audio Certaines plates-formes cibles peuvent autoriser plus d'un auditeur, chacun correspondant à un périphérique de sortie différent. Par exemple, votre plate-forme cible pourrait autoriser deux auditeurs, un pour le casque et un pour le moniteur. C'est pourquoi GameMaker Studio 2 vous fournit un certain nombre de fonctions supplémentaires qui vous permettent de trouver les écouteurs disponibles et de les définir pour les sons joués : audio_get_listener_count audio_get_listener_info audio_get_listener_mask audio_set_listener_mask Il existe également des fonctions permettant d'obtenir et de régler les données de masque sur les sons individuels : audio_sound_get_listener_mask audio_sound_set_listener_mask Ainsi que pour obtenir et définir les données de masque sur les émetteurs : audio_emitter_get_listener_mask audio_emitter_set_listener_mask Retour : Audio Suivant : Groupes audio Copyright YoYo Games Ltd. 2021 Tous droits réservés
