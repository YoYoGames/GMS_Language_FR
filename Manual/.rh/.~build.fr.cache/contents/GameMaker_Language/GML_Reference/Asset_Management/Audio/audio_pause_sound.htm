audio_pause_sound Cette fonction permet de mettre en pause tout son en cours de lecture. Le son peut être une instance unique d'un son (l'index des sons individuels en cours de lecture peut être stocké dans une variable lors de l'utilisation des fonctions audio_play_sound() ou audio_play_sound_at() ) ou un actif sonore, auquel cas toutes les instances du son en question seront mises en pause. Syntaxe : audio_pause_sound(index) ; Arguments Description indice L'indice du son à mettre en pause. Les retours : N/A Exemple : if keyboard_check_pressed(ord("P")) { global.Pause = !global.Pause ; if global.Pause { audio_pause_sound(snd_Waterfall) ; } else { audio_resume_sound(snd_Waterfall) ; } } Le code ci-dessus vérifie s'il y a une pression sur la touche "P" du clavier et s'il en détecte une, il met la variable globale "Pause" à true ou false et met en pause le son indexé dans la variable "snd_Waterfall". soit il reprend le son à partir de son état de pause.
