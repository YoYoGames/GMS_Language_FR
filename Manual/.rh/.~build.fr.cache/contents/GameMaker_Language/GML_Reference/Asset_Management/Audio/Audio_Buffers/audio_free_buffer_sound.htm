audio_free_buffer_sound Avec cette fonction, vous pouvez libérer la valeur d'index du pointeur associée à l'ID de son. Les sons libérés ne seront pas disponibles pour la lecture, et si plusieurs instances du son sont jouées, elles seront toutes arrêtées. Notez qu'avant de pouvoir supprimer le tampon lui-même, vous devez d'abord libérer tous les identifiants sonores qui lui sont associés. Syntaxe:audio_free_buffer_sound(index); Argument Description index L'indice du son tamponné à libérer. Retour:N/A Exemple:audio_free_buffer_sound(soundID); Le code ci-dessus libère le son tamponné indexé dans la variable "soundID".
