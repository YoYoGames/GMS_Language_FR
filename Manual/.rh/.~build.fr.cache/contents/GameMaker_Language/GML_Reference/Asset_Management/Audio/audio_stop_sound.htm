audio_stop_sound Cette fonction arrêtera le son donné s'il est en cours de lecture. Le son peut être une instance unique d'un son (l'indice des sons individuels en cours de lecture peut être stocké dans une variable lors de l'utilisation des fonctions audio_play_sound() ou audio_play_sound_at() ) ou un actif sonore, auquel cas toutes les instances du son donné seront arrêtées. Syntaxe : audio_stop_sound(index) ; Arguments Description indice L'indice du son à arrêter. Les retours : N/A Exemple : if !global.SFX { audio_stop_sound(snd_Waterfall) ; } else { audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1) ; } Le code ci-dessus vérifie la variable globale "SFX" et si elle retourne false, il arrêtera le son indexé dans la variable "snd_Waterfall" qui est en train de jouer, et si elle retourne true, il mettra le son en boucle.
