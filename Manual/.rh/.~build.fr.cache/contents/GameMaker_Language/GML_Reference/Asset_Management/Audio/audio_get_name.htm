audio_get_name Cette fonction renvoie le nom d'une ressource audio donnée sous forme de chaîne. La valeur "index" peut être celle de la ressource elle-même (telle que vue dans l'Asset Browser) ou la valeur d'identification unique qui est donnée lorsque vous jouez le son en utilisant, par exemple, audio_play_sound(). Notez que la chaîne retournée n' est pas la même que l'ID de la ressource et ne peut pas être utilisée pour accéder à la ressource elle-même, elle ne doit donc être utilisée que pour l'affichage ou la vérification des erreurs. Syntaxe : audio_get_name(index) ; Arguments Description indice L'index du son à vérifier. Les retours : Chaîne de caractères Exemple : var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false) ; var name = audio_get_name(snd) ; show_debug_message("Sound = " + name) ; Le code ci-dessus joue un son aléatoire choisi parmi trois ressources sonores différentes puis affiche un message de débogage avec son nom.
