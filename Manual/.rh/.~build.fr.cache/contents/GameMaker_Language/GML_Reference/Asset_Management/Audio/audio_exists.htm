audio_existe Cette fonction renvoie si un son existe(true) ou non(false). Notez que si l'index recherché n'a pas été initialisé précédemment, cette fonction provoquera une erreur car elle recherche des index de biens inexistants. Le son à vérifier peut être soit une instance unique d'un son (l'index des sons individuels joués peut être stocké dans une variable lors de l'utilisation des fonctions audio_play_sound() ou audio_play_sound_at() ) ou un actif sonore. Syntaxe : audio_exists(index) ; Arguments Description indice L'indice du son dont il faut vérifier l'existence. Les retours : Booléen Exemple : if audio_exists(global.Music) { audio_play_sound(global.Music, 0, true) ; } Le code ci-dessus vérifie si un son existe. Si c'est le cas, il est programmé pour être joué en boucle. Retour : Audio Suivant : audio_get_name Copyright YoYo Games Ltd. 2021 Tous droits réservés
