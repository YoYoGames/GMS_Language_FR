audio_emitter_get_x Cette fonction renvoie la position x actuelle de l'émetteur audio donné. Syntaxe:audio_emitter_get_x(emitter); Argument Description emitter L'indice de l'émetteur à utiliser. Retour:Real Exemple:if audio_emitter_get_x(emitter_player) != x    {    audio_emitter_position(emitter_player, x, y, 0);    } Le code ci-dessus vérifie la position x actuelle d'un émetteur donné et si elle n'est pas égale à la position x de l'instance, elle est définie sur la position de l'instance.
