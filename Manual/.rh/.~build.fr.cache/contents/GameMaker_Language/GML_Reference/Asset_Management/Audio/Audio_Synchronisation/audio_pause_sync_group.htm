audio_pause_sync_group sa fonction mettra en pause le groupe de synchronisation donné s'il est en cours de lecture, l'index du groupe étant la valeur renvoyée lorsque vous avez créé le groupe à l'aide de la fonction audio_create_sync_group(). Cela n'arrête pas le son, et l'appel audio_resume_sync_group(), la lancera à partir de la même position où elle a été interrompue à nouveau. REMARQUE: cette fonctionnalité n'est pas disponible pour la plate-forme cible HTML5. Syntaxe:audio_pause_sync_group(group_index); Argument Description group_index L'index du groupe à mettre en pause. Retour:N/A Exemple:if (keyboard_check_pressed((ord)"P"))    {    global.Pause = !global.Pause    if global.Pause       {       audio_pause_sync_group(sg);       }    else       {       audio_resume_sync_group(sg);       }    } Le code ci-dessus vérifie si une touche est enfoncée sur la touche "P", et si une est détectée, il bascule la variable "global.Pause" puis la vérifie pour mettre en pause ou reprendre le groupe de synchronisation indexé dans la variable "sg".
