audio_emitter_velocity Cette fonction peut être utilisée pour donner des effets Doppler à un émetteur et simuler un mouvement audio basé sur le vecteur qui est résolu à partir des positions relatives x, y et z données. Si l'émetteur lui-même ne bouge jamais, vous n'avez normalement pas besoin de définir ces valeurs, mais, par exemple, si vous créez un jeu de tir à défilement où les ennemis viennent de la droite et défilent vers la gauche, vous définirez ceci. pour avoir une vitesse de (hspeed, 0, 0) dans l'événement de création des ennemis (et mettre à jour l'émetteur de chaque instance dans l'événement d'étape en utilisant audio_emitter_position() ) pour donner à tous les sons émis par les instances ennemies le Doppler correct lorsqu'ils passent l'instance du joueur (qui à son tour utiliserait la fonction audio_listener_position() pour mettre l' auditeur à la bonne position). Syntaxe:audio_emitter_velocity(emitter, vx, vy, vz); Argument Description emitter L'indice de l'émetteur à changer. vx La valeur du vecteur x (0 par défaut). vy La valeur du vecteur y (0 par défaut). vz La valeur du vecteur z (0 par défaut). Retour:N/A Exemple:s_emit = audio_emitter_create(); audio_emitter_position(s_emit, room_width, 0, 0); audio_emitter_velocity(s_emit, -5, 0, 0); Le code ci-dessus crée un émetteur audio et assigne son index à la variable "s_emit". Cet émetteur est ensuite placé à une position dans la pièce et reçoit une vitesse de 5 pixels par pas le long de l'axe x (il effectuera un Doppler correctement par rapport à l'auditeur comme s'il avait une vitesse horizontale de 5 pixels par pas).
