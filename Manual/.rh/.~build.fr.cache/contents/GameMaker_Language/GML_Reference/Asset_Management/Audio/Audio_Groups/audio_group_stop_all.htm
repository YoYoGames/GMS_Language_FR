audio_group_stop_all Cette fonction arrêtera tous les sons du groupe audio donné en cours de lecture. Syntaxe:audio_group_stop_all(groupID); Argument Description groupID L'index du groupe audio à arrêter (tel que défini dans la fenêtre Groupes audio ) Retour:N/A Exemple:if keyboard_check_pressed(vk_space)    {    audio_group_stop_all(audiogroup_level1);    } Le code ci-dessus vérifie la touche «espace», puis arrête toute la lecture audio du groupe donné.
