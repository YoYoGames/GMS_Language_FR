audio_group_stop_all Cette fonction arrête tous les sons du groupe audio donné qui sont en cours de lecture. Syntaxe : audio_groupe_stop_all(groupID) ; Arguments Description groupID L'index du groupe audio à arrêter (tel que défini dans la fenêtre Groupes audio). Les retours : N/A Exemple : if keyboard_check_pressed(vk_space) { audio_group_stop_all(audiogroup_level1) ; } Le code ci-dessus vérifie la présence de la touche "espace", puis arrête la lecture de tous les fichiers audio du groupe donné.
