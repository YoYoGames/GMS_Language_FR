audio_destroy_sync_group Les groupes de synchronisation audio doivent être détruits lorsqu'ils ne sont pas utilisés pour libérer la mémoire et les ressources sonores qui leur sont associées à l'aide de cette fonction. Il prend l'index de groupe comme renvoyé lorsque le groupe a été créé à l'aide de la fonction audio_create_sync_group() et libère toutes les ressources utilisées par le groupe. REMARQUE: cette fonctionnalité n'est pas disponible pour la plate-forme cible HTML5. Syntaxe:audio_destroy_sync_group(group_index); Argument Description group_index L'index de groupe à détruire. Retour:N/A Exemple:audio_destroy_sync_group(sg); Le code ci-dessus détruit le groupe de synchronisation indexé dans la variable "sg", et serait probablement utilisé dans les événements destroy ou Room End.
