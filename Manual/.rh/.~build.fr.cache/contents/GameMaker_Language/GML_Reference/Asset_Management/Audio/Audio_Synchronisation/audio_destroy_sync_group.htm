audio_destroy_sync_group Les groupes de synchronisation audio doivent être détruits lorsqu'ils ne sont pas utilisés afin de libérer la mémoire et les ressources sonores qui leur sont associées en utilisant cette fonction. Elle prend l'index du groupe tel qu'il a été retourné lorsque le groupe a été créé à l'aide de la fonction audio_create_sync_group(), et libère toutes les ressources utilisées par le groupe. REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5. Syntaxe : audio_destroy_sync_group(group_index) ; Arguments Description groupe_index L'indice du groupe à détruire. Les retours : N/A Exemple : audio_destroy_sync_group(sg) ; Le code ci-dessus détruit le groupe de synchronisation indexé dans la variable "sg", et serait probablement utilisé dans les événements destroy ou Room End .
