audio_is_paused Cette fonction vérifiera le son donné pour voir s'il est actuellement en pause. Le son peut être une instance unique d'un son (l'index des sons individuels joués peut être stocké dans une variable lors de l'utilisation des fonctions audio_play_sound() ou audio_play_sound_at() ) ou un actif sonore, auquel cas toutes les instances du son donné seront vérifiées et si l'une d'entre elles est en pause, la fonction renverra true, sinon elle renvoie false. Syntaxe : audio_is_paused(index) ; Arguments Description indice L'index du son à vérifier. Les retours : Booléen Exemple : if audio_is_paused(snd_Waterfall) { audio_resume_sound(snd_Waterfall) ; } Le code ci-dessus vérifie si le son indexé dans la variable "snd_Waterfall" est actuellement en pause et s'il retourne vrai, la lecture du son sera reprise.
