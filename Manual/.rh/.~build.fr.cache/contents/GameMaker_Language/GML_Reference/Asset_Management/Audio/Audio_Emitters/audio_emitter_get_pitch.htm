audio_emitter_get_pitch Cette fonction renvoie la valeur de hauteur actuelle définie pour l'émetteur audio donné. Syntaxe : audio_emitter_get_pitch(emitter) ; Arguments Description émetteur L'indice de l'émetteur à utiliser. Les retours : Real Exemple : if audio_emitter_get_pitch(emitter_player) != 1 { audio_emitter_pitch(emitter_player, 1) ; } Le code ci-dessus vérifie le pitch actuel d'un émetteur donné et s'il n'est pas égal à 1, il est mis à 1.
