audio_start_sync_group Avec cette fonction, vous pouvez commencer à lire un groupe de synchronisation créé précédemment. Vous devez fournir l' index de groupe qui est la valeur renvoyée lorsque vous avez créé le groupe à l'aide de la fonction audio_create_sync_group(). REMARQUE: cette fonctionnalité n'est pas disponible pour la plate-forme cible HTML5. Syntaxe:audio_start_sync_group(group_index); Argument Description group_index L'index de groupe à jouer. Retour:N/A Exemple:sg = audio_create_sync_group(true); audio_play_in_sync_group(sg, sound1); audio_play_in_sync_group(sg, sound2); audio_sound_gain(sound2, 0, 0); audio_play_in_sync_group(sg, sound3); audio_sound_gain(sound3, 0, 0); audio_play_in_sync_group(sg, sound4); audio_sound_gain(sound4, 0, 0); audio_start_sync_group(sg); Ce qui précède crée un nouveau groupe de synchronisation et affecte l'index du groupe à la variable "sg". Quatre sons sont ensuite ajoutés au groupe, le gain de trois d'entre eux étant réglé sur 0. Enfin, le groupe de synchronisation est joué.
