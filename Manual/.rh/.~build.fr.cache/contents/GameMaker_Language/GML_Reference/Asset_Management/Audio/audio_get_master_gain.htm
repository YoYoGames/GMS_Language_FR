audio_get_master_gain Cette fonction permet d'obtenir la valeur absolue du volume global de tous les sons et musiques pour un auditeur spécifique. L'index par défaut de l'auditeur est 0, mais vous pouvez utiliser la fonction audio_get_listener_info() pour obtenir les différents indices disponibles pour la plate-forme cible. La valeur de gain renvoyée est basée sur une échelle linéaire allant de 0 (silencieux) à 1 (plein volume). Notez que sur certaines plateformes, vous pouvez avoir un gain supérieur à 1, bien qu'une valeur de 1 soit considérée comme un "plein volume". considérée comme "plein volume" et toute valeur supérieure peut introduire un écrêtage audio. Syntaxe : audio_get_master_gain(listenerIndex) ; Arguments Description listenerIndex L'index de l'auditeur pour obtenir le gain de. "index"], 1) ; } ds_map_destroy(info) ; } Le code ci-dessus boucle à travers les écouteurs disponibles, vérifiant si leur gain est de 1 ou non, et si ce n'est pas le cas, il le met à 1 pour chacun d'eux.
