audio_resume_sound Cette fonction vous permet de reprendre tout son actuellement en pause (après avoir utilisé la fonction audio_pause_sound()). Le son peut être une instance unique d'un son (l'index des sons individuels joués peut être stocké dans une variable lors de l'utilisation de la fonction ). en cours de lecture peut être stocké dans une variable lorsque vous utilisez la fonction audio_play_sound() ou audio_play_sound_at() ) soit un actif sonore, auquel cas toutes les instances du son donné seront redémarrées. Syntaxe : audio_resume_sound(index) ; Arguments Description indice L'index du son à reprendre. Les retours : N/A Exemple : if keyboard_check_pressed(ord("P")) { global.Pause = !global.Pause ; if global.Pause { audio_pause_sound(snd_Waterfall) ; } else { audio_resume_sound(snd_Waterfall) ; } } Le code ci-dessus vérifie s'il y a une pression sur la touche "P" du clavier et s'il en détecte une, il met la variable globale "Pause" à true ou false et met en pause le son indexé dans la variable "snd_Waterfall". soit il reprend le son à partir de son état de pause.
