shader_set Avec cette fonction, vous pouvez définir la cible de dessin au shader donné et tous les dessins ultérieurs seront effectués en utilisant ce shader. Vous pouvez mettre fin à l'utilisation du shader avec la fonction shader_reset(). REMARQUE: Vous ne pouvez définir un shader que dans l'événement Draw d'une instance. Syntaxe : shader_set(shader) ; Arguments Description nuanceur L'index du shader à utiliser. Les retours : N/A Exemple : shader_set(shader_Glass) ; draw_self() ; shader_reset() ; Le code ci-dessus va définir un shader top à utiliser pour le dessin, puis dessiner le sprite courant utilisé pour l'instance qui l'utilise.
