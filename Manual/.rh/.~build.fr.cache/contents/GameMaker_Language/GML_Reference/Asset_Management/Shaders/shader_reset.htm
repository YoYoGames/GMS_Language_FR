shader _set_uniform_matrix_array Cette fonction réinitialise la cible de dessin et doit être appelée lorsque vous ne souhaitez plus utiliser le shader actuel (défini à l'aide de shader_set() ). Syntaxe:shader_reset(); Retour:N/A Exemple:shader_set(shader_Glass); draw_self(); shader_reset(); Le code ci-dessus définira un shader top à utiliser pour le dessin, puis dessinera le sprite actuel utilisé pour l'instance qui l'utilise.
