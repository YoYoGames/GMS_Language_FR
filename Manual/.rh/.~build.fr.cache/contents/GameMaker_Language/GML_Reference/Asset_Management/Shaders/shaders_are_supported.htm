shaders_sont_supportés Cette fonction vérifie si la plate-forme cible choisie prend en charge les shaders. Elle renvoie un message vrai si c'est le cas, et faux si ce n'est pas le cas. Il est important de noter que sur Android, si le projet n'a pas de ressources de shaders définies, la fonction retournera toujours false, que le périphérique supporte ou non les shaders. Syntaxe : shaders_are_supported() ; Les retours : Booléen Exemple : global.GFX = false ; if shaders_are_supported() { if shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp) { global.GFX = true ; } } Le code ci-dessus définira une variable globale à false, puis, si la plate-forme prend en charge les shaders et que les deux shaders vérifiés ont été compilés correctement, elle sera définie à true.
