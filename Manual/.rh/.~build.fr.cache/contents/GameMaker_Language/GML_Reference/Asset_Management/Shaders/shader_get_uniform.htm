shader_get_uniform Puisque vous ne pouvez pas modifier la valeur d'une constante d'ombrage dans l'ombrage lui-même, vous devez la définir avant d'appeler l'ombrage à l'aide de l'une des fonctions d'uniformisation disponibles. Cependant, pour pouvoir le faire, vous devez d'abord appeler cette fonction pour obtenir le "handle" de la constante du shader que vous souhaitez modifier. NOTE: Bien qu'un shader soit composé de deux programmes discrets (vertex et fragment), cette fonction ne fera pas la différence entre les deux et retournera le handle de la constante du shader de l'un ou l'autre. Syntaxe : shader_get_uniform(shader, uniform) ; Arguments Description nuanceur L'index du shader à utiliser. uniforme La constante du shader pour obtenir le handle de (une chaîne). Les retours : Real (poignée uniforme) Exemple : shader_params= shader_get_uniform(shd_glass, "u_vRefractColour") ; Le code ci-dessus obtiendra le gestionnaire de la constante du shader "u_vRefractColour".
