shader_set_uniform_i Avec cette fonction, vous pouvez définir la valeur (ou les valeurs) d'une constante de shader. Vous devez auparavant avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform()et vous devez savoir de quel type de constante il s'agit pour lui transmettre le nombre correct de valeurs entières, c'est-à-dire que si vous avez un vec2, vous devrez transmettre deux valeurs à la fonction. NOTE: Tous les uniformes doivent être définis après avoir appelé la fonction shader_set()et avant d'appeler shader_reset(). Syntaxe : shader_set_uniform_i(handle, value1 [, value2, value3, value4]) ; Arguments Description poignée Le handle de la constante du shader à définir. valeur1 ... valeur4 La (ou les) valeur(s) entière(s) à laquelle (auxquelles) la constante du shader doit être définie. Les retours : N/A Exemple : shader_set(shader_Glass) ; shader_params= shader_get_uniform(shader_glass, "u_vParams") ; shader_set_uniform_i(shader_params, 0, 65, 255) ; draw_self() ; shader_reset() ; Le code ci-dessus obtiendra le handle de la constante du shader "u_vParams" ( un vec3), puis réglera cette constante sur les valeurs entières données.
