shader_current Cette fonction renvoie la valeur de l'ID de l'index du shader actuellement utilisé pour le rendu, ou elle renvoie -1 si aucun shader n'est utilisé. Syntaxe : shader_current() ; Les retours : Real (valeur d'identification du shader, ou -1 si pas de shader) Exemple : if(shader_current() == -1) { shader_set(sh_warp) } Le code ci-dessus vérifie quel est le shader actuel et s'il renvoie -1 (aucun shader utilisé), un shader est défini.
