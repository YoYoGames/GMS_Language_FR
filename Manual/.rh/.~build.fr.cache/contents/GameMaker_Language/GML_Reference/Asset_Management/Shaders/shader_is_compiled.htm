shader_est_compilé Cette fonction va vérifier un shader au moment de l'exécution pour s'assurer qu'il a été compilé avec succès. Si c'est le cas, la fonction renvoie true, sinon elle renvoie false. Cette fonction doit être utilisée au début du jeu pour s'assurer que la plate-forme qui exécute votre jeu a compilé avec succès les shaders utilisés (en particulier sous Windows où certains ordinateurs peuvent utiliser DX9 avec le niveau de shaders 2.0 et non une version ultérieure utilisant le niveau de shaders 3.0). 3.0). Si votre shader n'a PAS été compilé et que vous appelez shader_set() le jeu se plantera, il vaut donc la peine d'avoir une sorte de vérification chaque fois que vous utilisez autre chose que de simples shaders GLSL ES. autre chose que de simples shaders GLSL ES. Syntaxe : shader_is_compiled(shader) ; Arguments Description nuanceur Le shader à vérifier. Les retours : N/A Exemple : global.GFX = false ; if(shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp)) { global.GFX = true ; } Le code ci-dessus définit une variable globale à false, puis si les deux shaders vérifiés ont été compilés correctement, elle sera définie à true.
