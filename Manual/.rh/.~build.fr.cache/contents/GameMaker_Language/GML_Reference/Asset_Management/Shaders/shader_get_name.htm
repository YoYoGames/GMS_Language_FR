shader_get_name Avec cette fonction, vous pouvez récupérer le nom d'une ressource de shader. Vous fournissez la valeur d'identification unique du shader dont vous voulez obtenir le nom et la fonction renvoie le nom de la ressource sous forme de chaîne. Syntaxe : shader_get_name(shader) ; Arguments Description nuanceur L'index du shader dont on veut obtenir le nom. Les retours : Chaîne de caractères Exemple : var _shader = shader_current() ; var _name = shader_get_name(_shader) ; draw_text(32, 32, "Debug - Currently Rendering = " + _name) ; Le code ci-dessus obtiendra le nom du shader donné et le dessinera à l'écran.
