shader _get_name Avec cette fonction, vous pouvez récupérer le nom d'une ressource de shader Vous fournissez la valeur d'ID unique pour le shader pour obtenir le nom et la fonction renverra le nom de la ressource sous forme de chaîne. Syntaxe:shader_get_name(shader); Argument Description shader L'index du shader pour lequel obtenir le nom. Retour:String Exemple:var _shader = shader_current(); var _name = shader_get_name(_shader); draw_text(32, 32, "Debug - Currently Rendering = " + _name); Le code ci-dessus obtiendra le nom du shader donné et le dessinera à l'écran.
