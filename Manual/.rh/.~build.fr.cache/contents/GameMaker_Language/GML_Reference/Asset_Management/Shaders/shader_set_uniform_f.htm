shader_set_uniform_f Avec cette fonction, vous pouvez définir la valeur (ou les valeurs) d'une constante de shader. Vous devez auparavant avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform()et vous devez savoir de quel type de constante il s'agit pour lui transmettre le nombre correct de valeurs à virgule flottante, c'est-à-dire que si vous avez un vec2, vous devrez transmettre deux valeurs à la fonction. Une exception à la règle ci-dessus est le traitement des valeurs de couleur 32bit. Vous pouvez passer dans le shader jusqu'à huit valeurs de couleur 32bit et elles seront automatiquement converties pour vous en valeurs vec4 de 0 - 1. Cependant pour cela vous devez utiliser le nom de la constante du shader in_Colour, pour une seule couleur, ou in_Colour0, in_Colour1, etc... jusqu'à in_Colour7 pour des valeurs d'entrée multiples. Les exemples suivants exemples suivants montrent comment cela se présente dans le shader lui-même : //Simple couleur attribut vec2 in_Position ; attribut vec4 in_Colour ; attribut vec2 in_TextureCoord ; //Multiple couleur attribut vec2 in_Position ; attribut vec4 in_Colour0 ; attribut vec4 in_Colour1 ; attribut vec2 in_TextureCoord ; Notez également que si vous voulez faire des normales compressées etc... en utilisant un format de couleur de 4 octets, vous devrez également utiliser le même format "in_Colour" comme indiqué ci-dessus. NOTE: Tous les uniformes doivent être définis après avoir appelé la fonction shader_set()et avant d'appeler shader_reset(). Syntaxe : shader_set_uniform_f(handle, value1 [, value2, value3, value4]) ; Arguments Description poignée Le handle de la constante du shader à définir. valeur1 ... valeur4 La (ou les) valeur(s) à virgule flottante(s) à laquelle (auxquelles) la constante du shader doit être définie. Les retours : N/A Exemple : shader_set(shader_Glass) ; shader_params= shader_get_uniform(shader_glass, "u_vParams") ; shader_set_uniform_f(shader_params, 0.1, 0.8, 0.25) ; draw_self() ; shader_reset() ; Le code ci-dessus obtiendra le handle de la constante du shader "u_vParams" ( un vec3), puis réglera cette constante sur les valeurs données.
