shader _set_uniform_f Avec cette fonction, vous pouvez définir la ou les valeurs d'une constante de shader Vous devez au préalable avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform(), et vous devrez savoir de quel type de constante il s'agit pour lui passer le nombre correct de valeurs en virgule flottante, c'est-à-dire: si vous avez un vec2 vous devrez passer deux valeurs à la fonction. Une exception à la règle ci-dessus concerne les valeurs de couleur 32 bits. Vous pouvez passer dans le shader jusqu'à huit valeurs de couleur 32 bits et elles seront automatiquement converties pour vous en valeurs vec4 de 0 à 1. Cependant, pour cela, vous devez utiliser le nom de la constante de shader in_Colour, pour une seule couleur, ou in_Colour0, in_Colour1, etc... jusqu'à in_Colour7 pour plusieurs valeurs d'entrée. Les exemples suivants montrent à quoi cela ressemblerait dans le shader lui-même:    //Single colour     attribute vec2 in_Position;     attribute vec4 in_Colour;     attribute vec2 in_TextureCoord;     //Multiple colours     attribute vec2 in_Position;     attribute vec4 in_Colour0;     attribute vec4 in_Colour1;     attribute vec2 in_TextureCoord; Notez aussi que si vous voulez faire des normales compressées etc... en utilisant un format de couleur de 4 octets, alors vous devrez également utiliser le même format "in_Colour" comme indiqué ci-dessus. REMARQUE: tous les uniformes doivent être définis après avoir appelé la fonction shader _set () et avant d'appeler shader _reset (). Syntaxe:shader_set_uniform_f(handle, value1 [, value2, value3, value4]); Argument Description handle La poignée de la shader à définir. value1... value4 Valeur à virgule flottante (ou valeurs) sur laquelle définir la constante d' shader Retour:N/A Exemple:shader_set(shader_Glass); shader_params = shader_get_uniform(shader_glass, "u_vParams"); shader_set_uniform_f(shader_params, 0.1, 0.8, 0.25); draw_self(); shader_reset(); Le code ci-dessus obtiendra le handle de la shader "u_vParams" (un vec3 ), puis définissez cette constante sur les valeurs données.
