séquence_existe Avec cette fonction, vous pouvez vérifier si un objet séquence existe ou non. Vous fournissez soit la structure de l'objet séquence (telle que retournée par la fonction séquence_créer() ou séquence_get()) ou l'ID de la séquence (tel que retourné par la fonction layer_sequence_get_sequence() ou à partir de la propriété de la structure d'instance de séquence , ou l'index à partir du navigateur d'actifs. ) et la fonction renvoie true si l'objet de la séquence existe ou false si ce n'est pas le cas. Syntaxe : sequence_exists(sequence_struct_or_id) ; Arguments Description séquence_structure_ou_id La structure ou l'ID de l'objet de la séquence à vérifier Les retours : Bool Exemple : if sequence_exists(my_seq) { sequence_destroy(my_seq) ; } Le code ci-dessus vérifie si l'objet séquence donné existe et si c'est le cas, il est détruit.
