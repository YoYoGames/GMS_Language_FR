séquence_créer Avec cette fonction, vous pouvez créer un nouvel objet séquence auquel vous pouvez ensuite ajouter des pistes. La fonction renvoie une structure à laquelle vous pouvez accéder pour configurer la nouvelle séquence que vous avez créée. Le contenu de cette de cette structure est détaillé sur cette page. La structure de l'objet séquence peut ensuite être utilisée pour créer des instances de la séquence sur une couche de salle à l'aide de la fonction layer_sequence_create(). Notez que lorsque vous créez des objets de séquence de cette manière, vous devez les supprimer en appelant la fonction séquence_destroy() lorsqu'ils ne sont plus nécessaires, sinon vous aurez une fuite de mémoire qui qui peut ralentir et éventuellement faire planter votre jeu. Syntaxe : sequence_create() ; Les retours : Structure Exemple : myseq = sequence_create() ; myseq.length = 120 ; myseq.loopmode = seqplay_pingpong ; Le code ci-dessus crée une nouvelle structure d'objet de séquence et définit sa longueur de lecture et son mode de boucle.
