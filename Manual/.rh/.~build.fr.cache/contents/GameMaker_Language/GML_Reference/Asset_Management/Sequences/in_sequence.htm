in_sequence Il s'agit d'une variable intégrée qui fait partie des variables d'instance créées pour chaque instance d'un objet dans votre jeu. Lorsqu'une instance fait partie d'une séquence, cette variable retournera true, et quand ce n'est pas le cas, il reviendra false. Il s'agit d'une variable en lecture seule et ne peut pas être définie. Syntaxe:in_sequence Retour:Boolean Exemple:if (!in_sequence)     {     instance_destroy()     } Le code ci-dessus vérifie le in_sequence variable, et si c'est false (l'instance ne fait pas partie d'une séquence) alors elle détruira l'instance.
