séquence_get_objets Avec cette fonction, vous pouvez récupérer un tableau de tous les indices d'objets qui ont des instances créées dans la séquence donnée. Vous fournissez soit la structure de l'objet de la séquence (telle que retournée par la fonction séquence_créer() ou séquence_get()) ou l'ID de la séquence (tel que retourné par la fonction layer_sequence_get_sequence() ou de l'instance de séquence struct property sequence) et la fonction renverra un tableau, où chaque élément du tableau est un identifiant object_index pour les différents objets utilisés par la séquence pour créer des instances. Syntaxe : sequence_get_objects(sequence_struct_or_id) ; Arguments Description séquence_structure_ou_id La structure ou l'ID de l'objet de la séquence pour obtenir les objets. Les retours : Array Exemple : var _seq = sequence_get(seq_Logo) ; obj_array = sequence_get_objects(_seq) ; Le code ci-dessus obtient la structure d'un objet séquence, puis récupère les objets qu'il utilise et stocke le tableau dans une variable.
