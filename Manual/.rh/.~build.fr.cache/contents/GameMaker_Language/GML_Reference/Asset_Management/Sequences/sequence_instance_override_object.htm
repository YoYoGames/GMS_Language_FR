objet de recouvrement de l'instance de la séquence Avec cette fonction, vous pouvez remplacer toutes les instances d'un objet utilisé dans une séquence par une autre. Vous devez fournir l'ID de structure de l'instance de la séquence (tel que renvoyé lorsque l'instance de la séquence a été créée dans la pièce ou en utilisant l'une des fonctions de calque de pièce voir ici), ainsi que l'index de l'objet (tel que défini dans le navigateur de ressources) pour l'objet que vous souhaitez remplacer. Enfin, vous donnez un index d'objet ou un ID d'instance à utiliser comme l'objet qui va remplacer la séquence (fournir un ID d'instance utilisera simplement l'objet à partir duquel l'instance a été créée comme remplacement). Notez que ceci ne peut être fait que sur les instances de séquence (pas sur les objets de séquence) et doit être fait avant que la séquence ne commence à être lue, sinon cela ne fonctionnera pas. Syntaxe : sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id) ; Arguments Description séquence_instance_struct La structure de l'instance de la séquence à modifier. identifiant de l'objet L'indice de l'objet à remplacer dans la séquence. instance_ou_objet_id L'index de l'objet ou l'ID de l'instance à utiliser pour remplacer les objets de la séquence. Les retours : N/A Exemple : var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground) ; var _seq_inst = layer_sequence_get_instance(_seq) ; sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer) ; Le code ci-dessus crée une nouvelle instance de séquence sur la couche donnée, puis la modifie de sorte que toutes les instances de l'objet "obj_Trees_Winter" soient remplacées par des instances de l'objet "obj_Trees_Summer".
