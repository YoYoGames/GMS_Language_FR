séquence_ keyframe data_new Avec cette fonction, vous pouvez créer une nouvelle structure de données d' keyframe, en fournissant le type de piste à laquelle les keyframe seront appliquées, qui sera l'une des constantes suivantes: Constant Description Évaluer seqtracktype_graphic Il s'agit d'une piste d'actifs graphiques (sprite). 1 seqtracktype_audio Ceci est une piste d'actif audio. 2 seqtracktype_instance Ceci est une piste d'actif d'instance. 14 seqtracktype_sequence Il s'agit d'une piste d'actif de séquence. 7 seqtracktype_clipmask Il s'agit d'une piste d'actif de groupe de masques de clip. 8 seqtracktype_clipmask_mask Il s'agit d'une piste d'actif de sprite de masque de clip utilisée pour générer le masque de clip. 9 seqtracktype_clipmask_subject Il s'agit d'une piste d'actif de sprite de masque de clip qui est masquée. 10 seqtracktype_group Cette piste d'actif de dossier de groupe. 11 seqtracktype_colour Il s'agit d'une piste de paramètres de données de couleur. 4 seqtracktype_real Il s'agit d'une piste de paramètre de valeur de nombre réel. 3 seqtracktype_message Ceci est une piste de message de diffusion. 15 seqtracktype_moment Ceci est une piste d'événement / moment. 16 seqtracktype_bool Non utilisé actuellement. 5 seqtracktype_string Non utilisé actuellement. 6 seqtracktype_spriteframes Non utilisé actuellement. 13 seqtracktype_empty Non utilisé actuellement. 12 La fonction renverra une structure de données d' keyframe piste qui peut ensuite recevoir des valeurs ajoutées avant d'être affectée à une structure d' keyframe. Syntaxe:sequence_keyframedata_new(type); Argument Description type Le type de keyframe à créer, une constante, répertoriée ci-dessus. Retour:Struct Exemple:myseq = sequence_create(); var mytracks = array_create(1); mytracks[0] = sequence_track_new(seqtracktype_graphic); var graphickeys = array_create(1); graphickeys[0] = sequence_keyframe_new(seqtracktype_graphic); graphickeys[0].frame = 0; graphickeys[0].length = 1; graphickeys[0].stretch = true; graphickeys[0].disabled = false; var graphickeydata = array_create(1); graphickeydata[0] = sequence_keyframedata_new(seqtracktype_graphic); graphickeydata[0].spriteIndex = spr_Platform; graphickeydata[0].channel = 0; graphickeys[0].channels = graphickeydata; mytracks[0].name = "TestGraphicTrack"; mytracks[0].keyframes = graphickeys; myseq.tracks = mytracks; Le code ci-dessus crée une nouvelle séquence, puis crée une piste d'actif graphique et définit certaines keyframe sur la piste.
