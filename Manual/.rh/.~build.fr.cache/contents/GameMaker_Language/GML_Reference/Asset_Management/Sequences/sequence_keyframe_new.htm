séquence_cadre_clé_nouveau Avec cette fonction, vous pouvez créer une nouvelle structure d' image clé de piste, en fournissant le type de piste sur lequel l'image clé sera appliquée, qui sera l'une des constantes suivantes : Constant Description Valeur seqtracktype_graphic Il s'agit d'une piste d'actif graphique (sprite). 1 seqtracktype_audio Il s'agit d'une piste d'actif audio. 2 seqtracktype_instance Il s'agit d'une instance de suivi des actifs. 14 seqtracktype_sequence Il s'agit d'une piste d'actifs séquentiels. 7 seqtracktype_clipmask Il s'agit d'une piste d'actifs de groupe de masques de clips. 8 seqtracktype_clipmask_mask Il s'agit d'une piste d'actif de sprite de masque de clip utilisée pour générer le masque de clip. 9 seqtracktype_clipmask_subject Il s'agit d'une piste d'actif sprite de masque de clip qui est masquée. 10 seqtracktype_group Ce groupe de dossiers de suivi des actifs. 11 seqtracktype_colour Il s'agit d'une piste de paramètres de données de couleur. 4 seqtracktype_real Il s'agit d'une piste de paramètres à valeur numérique réelle. 3 seqtracktype_message Il s'agit d'une piste de message de diffusion. 15 seqtracktype_moment Il s'agit d'une piste d'événement/moment. 16 seqtracktype_bool Non utilisé actuellement. 5 seqtracktype_string Non utilisé actuellement. 6 seqtracktype_spriteframes Non utilisé actuellement. 13 seqtracktype_empty Non utilisé actuellement. 12 La fonction renvoie une structure d'image clé de piste à laquelle on peut ajouter des données avant de l'affecter à une structure de piste. Syntaxe : sequence_keyframe_new(type) ; Arguments Description type Le type d'image clé à créer, une constante, listée ci-dessus. Les retours : Structure Exemple : myseq = sequence_create() ; var mytracks = array_create(1) ; mytracks[0] = sequence_track_new(seqtracktype_graphic) ; var graphickeys = array_create(1) ; graphickeys[0] = sequence_keyframe_new(seqtracktype_graphic) ; graphickeys[0].frame = 0 ; graphickeys[0].length = 1 ; graphickeys[0].stretch = true ; graphickeys[0].disabled = false ; var graphickeydata = array_create(1) ; graphickeydata[0] = sequence_keyframedata_new(seqtracktype_graphic) ; graphickeydata[0].spriteIndex = spr_Platform ; graphickeydata[0].channel = 0 ; graphickeys[0].channels = graphickeydata ; mytracks[0].name = "TestGraphicTrack" ; mytracks[0]. keyframes= graphickeys; myseq.tracks = mytracks ; Le code ci-dessus crée une nouvelle séquence, puis crée une piste d'actif graphique et définit des données d'images clés sur la piste.
