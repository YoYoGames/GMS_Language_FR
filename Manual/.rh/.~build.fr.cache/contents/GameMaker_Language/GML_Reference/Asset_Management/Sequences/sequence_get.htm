sequence_get Avec cette fonction, vous pouvez récupérer la structure d' objet de séquence à partir d'un index d'actif de séquence. Vous fournissez l'index d'actif pour la séquence tel que défini dans le navigateur d'actifs, et la fonction renverra la structure d'objet de séquence qui sera alors accessible, ou -1 si la séquence n'existe pas ou si l'index d'actif donné n'est pas une séquence. Syntaxe:sequence_get(sequence_index); Argument Description sequence_index L'index de séquence tel que défini dans le navigateur de ressources. Retour:struct or -1 Exemple:var _seq = sequence_get(seq_Logo); seq.loopmode = seqplay_pingpong; Le code ci-dessus récupère la structure de l'objet séquence à partir de l'objet séquence "seq_Loop", puis définit le mode de boucle pour la séquence (toutes les instances de séquence créées ultérieurement utiliseront désormais ce mode de boucle).
