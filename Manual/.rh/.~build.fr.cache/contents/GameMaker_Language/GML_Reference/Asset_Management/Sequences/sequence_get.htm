séquence_get Cette fonction permet d'extraire la structure de l'objet de la séquence à partir d'un index d'actifs de la séquence. Vous fournissez l'index d'actif de la séquence tel que défini dans le navigateur d'actifs, et la fonction renvoie la structure d'objet de séquence à laquelle il est possible d'accéder, ou -1 si la séquence n'existe pas ou si l'index d'asset donné n'est pas une séquence. Syntaxe : séquence_get(séquence_index) ; Arguments Description séquence_index L'indice de séquence tel que défini dans le navigateur de biens. Les retours : struct ou -1 Exemple : var _seq = sequence_get(seq_Logo) ; seq.loopmode = seqplay_pingpong ; Le code ci-dessus récupère la structure d'objet de séquence de l'objet de séquence "seq_Loop" et définit ensuite le mode de boucle pour la séquence (toutes les instances de séquence créées ultérieurement utiliseront désormais ce mode de boucle).
