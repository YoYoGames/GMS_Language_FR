instance_nearest Cette fonction vérifie toutes les instances de l'objet donné pour voir laquelle est la plus proche du point x / y donné. Toutes les vérifications se feront de la position x / y donnée à l' origine (la position x / y) des instances de l'objet spécifié. Si aucune instance de l'objet n'existe, la fonction renverra le mot-clénoone, mais s'il y a des instances, il renverra le id de l'instance trouvée. Veuillez noter que si l'instance exécutant le code a le même index d'objet que l'objet en cours de vérification, elle sera incluse dans la vérification (cela inclut la vérification des objets parents si l'instance appelante est également un enfant du parent). Syntaxe:instance_nearest(x, y, obj); Argument Description x La position x à partir de laquelle effectuer la vérification. y La position y à partir de laquelle effectuer la vérification. obj Objet pour lequel rechercher des instances. Retour:Real (either an instance ID value or the keyword noone) Exemple:var inst, xx; xx = x; x -= 10000; inst = instance_nearest(xx, y, object_index); if inst != id     {     draw_line(x, y, inst.x, inst.y);     } x += 10000; Le code ci-dessus déplace l'instance actuelle de 10000 pixels, puis vérifie sa position précédente pour trouver l'instance la plus proche du même type d'objet. Si cette instance est elle-même, elle ne fera rien de plus que de revenir à sa position d'origine, mais si l'instance trouvée est différente, elle tracera une ligne entre les deux.
