direction Toutes les instances de GameMaker Studio 2 possèdent certaines propriétés "intégrées" que vous pouvez utiliser et définir pour déterminer leur apparence et leur comportement. La direction est l'une de ces propriétés et peut être utilisée pour définir la direction du mouvement de l'instance lorsque l'instance a une fonction de déplacement. lorsque l'instance a une valeur vitesse différent de 0. Notez que les directions dans GameMaker Studio 2 sont généralement calculées comme suit : 0° pour la droite, 90° pour le haut, 180° pour la gauche et 270° pour le bas. gravity et gravity_direction peuvent modifier la valeur direction lorsqu'elles sont utilisées dans vos jeux. Syntaxe : direction ; Les retours : Réel (valeur à virgule flottante de précision simple) Exemple : si keyboard_check(vk_left) direction += 5 ; si keyboard_check(vk_right) direction -= 5 ; Le code ci-dessus changera la direction du mouvement de l'instance en fonction de la touche pressée (gauche ou droite).
