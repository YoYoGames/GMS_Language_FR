direction Toutes les instances de GameMaker Studio 2 ont certaines propriétés «intégrées» que vous pouvez utiliser et définir pour régir leur apparence et leur comportement.Direction est l'une de ces propriétés et peut être utilisée pour définir la direction du mouvement de l'instance lorsque l'instance a un speed autre que 0. Notez que les directions dans GameMaker Studio 2 sont généralement calculées comme 0 ° étant à droite, 90 ° vers le haut, 180 ° vers la gauche et 270 ° vers le bas, et que legravity etgravity_direction les variables peuvent modifier direction valeur lorsqu'ils sont utilisés dans vos jeux. Syntaxe:direction; Retour:Real (single precision floating point value) Exemple:if keyboard_check(vk_left) direction += 5; if keyboard_check(vk_right) direction -= 5; Le code ci-dessus changera la direction du mouvement de l'instance en fonction de la touche (gauche ou droite) enfoncée.
