instance_deactivate_region Avec cette fonction, vous pouvez définir une région dans la pièce pour désactiver les instances qui ont été précédemment activées. Cette région peut être signalée comme "à l'intérieur" ou "à l'extérieur", comme illustré dans l'image suivante: Vous pouvez voir dans l'image ci-dessus que l'instance "apple" est toujours inactive car, même si le sprite lui-même ne chevauche pas la région, le cadre englobant le fait. Ainsi, les instances sont considérées comme étant dans la région spécifiée lorsque leur boîte englobante la recouvre, et l'état du masque de collision (précis ou non) n'est pas pris en compte. Notez que la désactivation n'est pas instantanée et qu'une instance ainsi désactivée ne sera considérée comme inactive qu'à la fin de l'événement au cours duquel la fonction a été appelée. REMARQUE: Si vous désactivez une instance au démarrage de la salle (c'est-à-dire à partir du code de création de salle, ou à partir d'un événement de création d'instance d'une instance dans la salle), toutes les instances placées dans la salle à partir de l'éditeur de salle continueront d'exécuter leur événement de création. avant d'être désactivé. AVERTISSEMENT: la désactivation des instances sur lesquelles la physique est activée n'empêchera PAS leurs appareils d'interagir dans la simulation physique. Pour cela, vous devez définir leur phy_active variable à true ou false lorsque vous activez / désactivez les instances. Syntaxe:instance_deactivate_region(left, top, width, height, inside, notme); Argument Description left La coordonnée x de la gauche de la région rectangulaire à désactiver. top Coordonnée y du haut de la région rectangulaire à désactiver. width La largeur de la région à désactiver. height La hauteur de la région à désactiver. inside Indique s'il faut désactiver les instances à l'intérieur de la région (vrai) ou à l'extérieur (faux). notme Indique s'il faut exclure l'instance appelante de la désactivation (vrai) ou non (faux). Retour:N/A Exemple:instance_activate_all(); var _vx = camera_get_view_x(view_camera[0]); var _vy = camera_get_view_y(view_camera[0]); var _vw = camera_get_view_width(view_camera[0]); var _vh = camera_get_view_height(view_camera[0]); instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false); Le code ci-dessus active toutes les instances, puis désactive une région dans la salle.
