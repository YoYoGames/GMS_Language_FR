xprécédent Cette variable intégrée renvoie la position x précédente de l'instance. Cette variable sera définie juste avant le début de l'événement begin step mais elle peut aussi être définie par le code à tout moment, ce qui signifie que vous pouvez lui donner votre propre valeur personnalisée si cela s'avère nécessaire. Syntaxe : xprécédent ; Les retours : Réel (valeur à virgule flottante de simple précision) Exemple : if x != xprevious || y != yprevious { déplacé = vrai ; } Le code ci-dessus vérifie s'il existe une différence entre les valeurs xprevious et yprevious et les valeurs x et y actuelles. Si c'est le cas, il définit une variable à true. Retour : Variables d'instance Suivant : yprevious Copyright YoYo Games Ltd. 2021 Tous droits réservés
