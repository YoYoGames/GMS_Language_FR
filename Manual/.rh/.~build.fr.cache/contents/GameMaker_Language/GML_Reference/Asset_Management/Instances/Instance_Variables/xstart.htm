xstart Cette variable stocke la position x initiale de l'instance lorsqu'elle est créée pour la première fois dans la room. Ce n'est pas une variable en lecture seule et elle peut être définie aussi bien que lue. Syntaxe : xdébut ; Les retours : Réel (valeur à virgule flottante de précision simple) Exemple : if place_meeting(x, y, obj_spike) { score -= 100 ; x = xstart ; y = ystart ; } Le code ci-dessus vérifie s'il y a une collision avec une instance de "obj_spike" et s'il y en a une, il déduit 100 du score et déplace l'instance à sa position de départ.
