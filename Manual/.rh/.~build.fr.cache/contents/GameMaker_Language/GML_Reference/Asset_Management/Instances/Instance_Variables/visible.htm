visible Une instance peut être marquée comme visible ou non en donnant à cette variable la valeur true (visible) ou false (invisible). Ceci fonctionne en indiquant à GameMaker Studio 2 d'ignorer l'événement de dessin pour cette instance. lorsque vous l'utilisez, car cela signifie qu'aucun code placé dans l'événement de dessin ne sera exécuté tant que l'instance n'est pas visible. Notez également que si le calque auquel l'instance est affectée est signalé comme invisible, la définition de cette variable à la valeur true n'aura aucun effet jusqu'à ce que le calque lui-même soit également marqué comme visible. Syntaxe : visible ; Les retours : Booléen Exemple : if other.visible = true { x = xprevious ; y = yprevious ; } Le code ci-dessus vérifiera le drapeau visible de l'autre instance lors d'un événement de collision et, s' il est vrai, ramènera l 'instance à sa position précédente.
