alarme_get Cette fonction peut être utilisée pour obtenir la valeur actuelle de l'alarme donnée. Vous fournissez le numéro de l'alarme de 0 à 11 et cette fonction renvoie la valeur actuelle de l'alarme. Il s'agit d'une méthode alternative à l'obtention directe de la valeur du tableau des alarmes. directement. Syntaxe : alarme_get(index) ; Arguments Description indice L'indice d'alarme à obtenir, de 0 à 11. Les retours : Réel (nombre entier) Exemple : for (var i = 0 ; i < 12 ; i++) { if alarm_get(i) > 0 alarm_set(i, -1) ; } Le code ci-dessus vérifie toutes les alarmes de l'instance appelante et si elles sont supérieures à 0, il leur attribue la valeur -1, les empêchant ainsi de poursuivre leur décompte.
