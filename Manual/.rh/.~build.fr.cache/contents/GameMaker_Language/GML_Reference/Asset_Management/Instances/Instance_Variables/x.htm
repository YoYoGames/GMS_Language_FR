x La valeur x d'une instance est la position horizontale dans la pièce actuelle, mesurée en pixels. Cette valeur peut être 0, positive ou négative, où 0 est le côté gauche de la pièce et le déplacement vers la droite augmente x, le déplacement vers la gauche diminue x (une valeur négative pour x signifie que l'instance est sortie du côté gauche de la pièce ). Vous pouvez même donner à x une valeur réelle telle que 12,345, mais comme vous ne pouvez pas obtenir 0,345 de pixel, l'instance peut ne pas sembler avoir bougé, bien que ces valeurs soient conservées et utilisées en interne par GameMaker Studio 2 et soient parfaitement valides. En définissant les valeurs x (et y), vous pouvez faire sauter l'objet dans la pièce à la position de votre choix, ou vous pouvez les ajouter et les soustraire par des quantités plus petites pour donner l'illusion du mouvement sans utiliser réellement le intégré speed et direction les fonctions. Cela doit être pris en compte lors de la création de vos jeux, car souvent un type de mouvement se prête plus à un style de jeu particulier qu'à un autre. Syntaxe:x; Retour:Real (single precision floating point value) Exemple:if keyboard_check(vk_left) x += 5; if keyboard_check(vk_right) x -= 5; Le code ci-dessus utilisera les touches fléchées pour ajouter ou soustraire à la position x et ainsi créer l'illusion du mouvement.
