instance_id Ce tableau en lecture seule contient tous les ids de chaque instance active dans la salle. Cela signifie que si vous avez utilisé l'une des fonctions de désactivation d'instance, ces instances qui ont été désactivées ne seront pas incluses dans ce tableau (si vous avez utilisé une valeur de ce tableau précédemment, il renverra désormais le mot-clénoone ). Syntaxe:instance_id[num]); Retour:Real (instance ID value) Exemple:for (var i = 0; i < instance_count; i ++;)     {     with (instance_id[i]) speed += 0.1;     } Le code ci-dessus parcourra toutes les instances de la salle et ajoutera 0,1 à leur vitesse.
