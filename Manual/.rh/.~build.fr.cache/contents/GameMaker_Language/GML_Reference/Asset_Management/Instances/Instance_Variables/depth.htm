profondeur Lorsque vous créez un objet, vous pouvez lui attribuer une profondeur initiale qui définit la façon dont les instances de cet objet seront dessinées dans la pièce lorsque le jeu est en cours. jeu est en cours. Normalement, vous n'avez pas besoin d'utiliser cette variable, car vous devez régler les instances pour qu'elles soient ajoutées à des couches discrètes, qui elles-mêmes sont réglées sur une profondeur spécifique. d'une instance en utilisant cette valeur, auquel cas une "couche temporaire" sera créée spécifiquement pour l'instance à la profondeur donnée. Notez que si aucune instance ne se trouve à la même profondeur, cette couche temporaire sera supprimée de la mémoire. (contrairement aux couches régulières qui resteront en mémoire même si elles ne contiennent rien). IMPORTANT ! Lorsque vous modifiez la variable de profondeur et que GameMaker Studio 2 gère les calques, la variable incorporée couche renverra -1 et non l'ID du calque, puisque les calques gérés ne peuvent pas être manipulés par le code. Dans GameMaker Studio 2, plus la valeur de profondeur d'une instance est faible, plus l'instance sera dessinée "près de la caméra", tandis qu'une valeur de profondeur plus élevée signifie que l'instance sera dessinée "plus loin de la caméra" : -1000 est dessiné au-dessus de -100, qui est dessiné au-dessus de 0, qui est dessiné au-dessus de 100, qui est dessiné au-dessus de 1000. Notez que les instances d'un objet qui ont la même profondeur peuvent être dessinées au-dessus ou au-dessous les unes des autres, même si elles semblent être placées les unes au-dessus des autres dans l'éditeur de salle. Si vous voulez garantir qu'un objet est dessiné au-dessus ou au-dessous de tout le reste, vous devez toujours définir la profondeur (ou le calque) explicitement. Notez également qu'il existe un minimum (-16000) et maximale (+16000) en dehors desquelles les instances ne seront pas dessinées, bien qu'elles existent toujours et traitent les événements. REMARQUE : Vous ne pouvez pas définir la profondeur d'une instance dans son événement de dessin (tous les autres événements sont acceptés). Syntaxe : profondeur ; Les retours : Réel (valeur à virgule flottante de simple précision) Exemple : si y != yprevious { profondeur = -y ; } Le code ci-dessus vérifiera si la position y a changé et si c'est le cas, la profondeur sera également définie pour y correspondre.
