hspeed hspeed est l'une des propriétés "intégrées" de toutes les instances et définit la vitesse de déplacement horizontal (le long de l'axe x) de l'instance en pixels par pas. Donc, un hspeed de 3 signifie 3 pixels de mouvement vers la droite (+ x) à chaque pas, et un hspeed de -3 signifierait 3 pixels de mouvement vers la gauche (-x) à chaque pas. Notez que si vous définissez le speed et / ou direction variables puis les hspeed La valeur sera mise à jour automatiquement pour refléter ces changements, et de même, en changeant le hspeed la valeur affectera également le speed et direction valeurs. Notez également quegravity,gravity_direction etfriction peuvent tous modifier la valeur de cette variable lorsqu'ils sont utilisés dans vos jeux. Syntaxe:hspeed; Retour:Real (single precision floating point value) Exemple:if keyboard_check(vk_left) hspeed = -5; if keyboard_check(vk_right) hspeed = 5; Le code ci-dessus changera la vitesse horizontale en fonction des touches enfoncées.
