gravity_direction gravity_direction est l'une des propriétés "intégrées" de toutes les instances et peut être utilisée pour définir la direction du mouvement lorsque les instances gravity est supérieur à 0. Notez que les directions dans GameMaker Studio 2 sont généralement calculées comme 0 ° étant à droite, 90 ° vers le haut, 180 ° vers la gauche et 270 ° vers le bas. Syntaxe:gravity_direction; Retour:Real (single precision floating point value) Exemple:if place_meeting(x, y, obj_switch)     {     gravity_direction += 180;     } Le code ci-dessus changera la direction gravity_direction si l'objet joueur rencontre une instance de "obj_switch".
