direction_gravité gravity_direction est l'une des propriétés "intégrées" que possèdent toutes les instances et peut être utilisée pour définir la direction du mouvement lorsque l'instance gravité est supérieure à 0. Notez que les directions dans GameMaker Studio 2 sont généralement calculées comme suit : 0° pour la droite, 90° pour le haut, 180° pour la gauche et 270° pour le bas. Syntaxe : Direction de la gravité ; Les retours : Réel (valeur à virgule flottante de simple précision) Exemple : if place_meeting(x, y, obj_switch) { gravity_direction += 180 ; } Le code ci-dessus changera la direction de la gravité si l'objet du joueur rencontre une instance de "obj_switch".
