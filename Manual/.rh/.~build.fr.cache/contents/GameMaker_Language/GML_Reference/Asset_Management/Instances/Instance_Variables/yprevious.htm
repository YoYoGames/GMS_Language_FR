yprécédent Cette variable intégrée renvoie la position y précédente de l'instance. Cette variable sera définie sur la position x actuelle juste avant le début de l'événement begin step, mais elle peut également être définie par le code à tout moment, ce qui signifie que vous pouvez lui donner votre propre valeur si nécessaire. vous pouvez lui donner votre propre valeur personnalisée si nécessaire. Syntaxe : yprécédent ; Les retours : Réel (valeur à virgule flottante de précision simple) Exemple : if x != xprevious || y != yprevious { déplacé = vrai ; } Le code ci-dessus vérifie s'il existe une différence entre les valeurs xprevious et yprevious et les valeurs x et y actuelles. Si c'est le cas, il définit une variable à true. Retour : Variables d'instance Suivant : id Copyright YoYo Games Ltd. 2021 Tous droits réservés
