instance_place_list Avec cette fonction, vous pouvez vérifier une position pour une collision avec toutes les instances d'un objet en utilisant le masque de collision de l'instance qui exécute le code pour la vérification. Lorsque vous l'utilisez, vous demandez effectivement à GameMaker Studio 2 de déplacer l'instance vers la nouvelle position, de rechercher une collision, puis de revenir en arrière et de vous dire si une collision a été trouvée ou non. Cela fonctionnera pour des collisions précises, mais uniquement si l'instance et les instances de l'objet à vérifier ont des masques de collision précis sélectionnés, sinon seules les collisions de boîte englobante sont appliquées. La fonction remplira une liste DS (créée précédemment) avec l'unique id valeurs de toutes les instances de l'objet considérées comme étant en collision avec l'instance appelante, et vous avez la possibilité de classer la liste en fonction de la distance entre l'origine de l'instance effectuant la vérification et l'origine des instances trouvées en collision. Notez que la fonction accepte également le mot-clé spécial all, auquel cas toutes les instances trouvées en collision seront répertoriées. La fonction renvoie le nombre d'instances trouvées, ou 0 si aucune n'est trouvée. Notez que les coordonnées x / y données seront arrondies à l'entier le plus proche avant que la vérification ne soit effectuée, donc si ce n'est pas ce dont vous avez besoin (ou si vous avez utilisé un ancien GameMaker ), vous devriez floor les coordonnées dans le chèque:instance_place_list(floor(x), floor(y), obj_Enemy, list, true). Syntaxe:instance_place_list(x, y, obj, list, ordered); Argument Description x La position x pour rechercher des instances. y La position y pour rechercher des instances. obj Objet pour lequel rechercher des instances. list Liste DS à utiliser pour stocker les ID des instances en collision. ordered Si la liste doit être triée par distance (true ) ou non (false ). Retour:Int (The number of instances found to be in collision) Exemple:var _list = ds_list_create(); var _num = instance_place_list(x, y, obj_Enemy, _list, false); if _num > 0     {     for (var i = 0; i < _num; ++i;)         {         instance_destroy(_list[| i]);         }     } ds_list_destroy(_list); Le code ci-dessus vérifiera une collision avec toutes les instances trouvées à la position des instances appelantes. Ceux-ci seront ajoutés à une liste DS, qui sera ensuite bouclée pour détruire chacune des instances en collision.
