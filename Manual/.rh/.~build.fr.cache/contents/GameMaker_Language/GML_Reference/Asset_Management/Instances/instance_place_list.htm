liste_de_lieux_instances Avec cette fonction, vous pouvez vérifier une position pour une collision avec toutes les instances d'un objet en utilisant le masque de collision de l'instance qui exécute le code pour la vérification. Lorsque vous utilisez cette fonction, vous demandez à GameMaker Studio 2 de déplacer l'instance à la nouvelle position. vers la nouvelle position, de vérifier s'il y a une collision, puis de revenir en arrière et de vous dire si une collision a été trouvée ou non. Cela fonctionne pour les collisions précises, mais uniquement si l'instance et les instances de l'objet à vérifier ont des masques de collision précis sélectionnés. sont sélectionnés, sinon seules les collisions de la boîte englobante sont appliquées. La fonction remplira une liste DS (précédemment créée) avec les valeurs uniques de toutes les instances de l'objet qui ont été vérifiées. id de toutes les instances de l'objet qui sont considérées comme étant en collision avec l'instance appelante, et vous avez la possibilité d'ordonner la liste en fonction de la distance entre l'origine de l'instance qui effectue la vérification et l'origine des instances qui sont en collision. considérées comme étant en collision. Notez que la fonction accepte également le mot-clé spécial tousdans ce cas, toutes les instances trouvées en collision seront listées. La fonction renvoie le nombre d'instances trouvées, ou 0 si aucune n'est trouvée. Notez que les coordonnées x/y données seront arrondies à l'entier le plus proche avant que la vérification ne soit effectuée, donc si ce n'est pas ce dont vous avez besoin (ou si vous avez utilisé un ancien produit GameMaker ), vous devez plancher les coordonnées dans l'option vérification : instance_place_list(floor(x), floor(y), obj_Enemy, list, true). Syntaxe : instance_place_list(x, y, obj, list, ordered) ; Arguments Description x La position x à vérifier pour les instances. y La position y pour vérifier les instances. obj. L'objet dont il faut rechercher les instances. liste La liste DS à utiliser pour stocker les ID des instances en collision. commandé Si la liste doit être ordonnée par distance(true) ou non(false). Les retours : Int (Le nombre d'instances trouvées en collision) Exemple : var _list = ds_list_create() ; var _num = instance_place_list(x, y, obj_Enemy, _list, false) ; if _num > 0 { for (var i = 0 ; i < _num ; ++i ;) { instance_destroy(_list[| i]) ; } } ds_list_destroy(_list) ; Le code ci-dessus vérifiera la présence d'une collision avec toutes les instances trouvées à la position de l'instance appelante. Celles-ci seront ajoutées à une liste DS, qui sera ensuite parcourue en boucle pour détruire chacune des instances en collision.
