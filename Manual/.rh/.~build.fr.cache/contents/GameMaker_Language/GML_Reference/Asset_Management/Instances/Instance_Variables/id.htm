id Cette variable en lecture seule contient le numéro d'identification unique de l'instance. Chaque instance que vous créez - que ce soit par le biais du code ou en l'ajoutant à une pièce dans l'éditeur de pièce - reçoit un numéro qui est utilisé en interne pour identifier cette instance. et la variable id est ce que vous pouvez utiliser pour la référencer. La variable id est également renvoyée (et peut être stockée dans une variable) lorsqu'une instance est créée à l'aide de la commande instance_create_layer() ou instance_create_depth() , ainsi que d'autres fonctions d'instance. NOTE: La valeur de cette variable n' est pas la même que l'identifiant donné à l'instance dans l'éditeur de salle et est également différente de l'identifiant du instance_id tableau qui contient tous les ids de toutes les instances actuellement actives. Syntaxe : id ; Les retours : Real (nombre entier - valeur de l'ID de l'instance) Exemple : for(var i = 0 ; i < instance_count ; i++ ;) { if instance_id[i] != id { instance_id[i].scr += 5 ; } } Le code ci-dessus ajoute 5 à la variable "scr" pour chaque instance active dans la salle, sauf celle qui exécute le code. Pour ce faire, il parcourt en boucle TOUTES les instances actives (en utilisant le tableau "instance_id" pour retourner l'ID de chaque instance active) et les compare à la variable "id" intégrée.
