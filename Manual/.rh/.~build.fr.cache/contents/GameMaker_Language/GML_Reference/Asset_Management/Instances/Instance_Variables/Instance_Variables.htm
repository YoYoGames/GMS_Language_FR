Variables d'instance Lorsque vous créez un nouvel objet, il est livré avec certaines variables déjà initialisées avec des valeurs par défaut. Ces variables sont appelées variables d'instance, car elles seront créées pour chaque instance que vous placerez de l'objet dans vos salles de jeu, et une fois qu'une instance est créée, les valeurs de ces variables sont uniques à cette instance et uniquement à cette instance. Certaines de ces variables seront très utilisées, comme x et y, tandis que d'autres seront moins nécessaires en fonction de ce que vous voulez faire. Mais en général, elles sont très utiles et, dans la mesure du possible, vous devriez les utiliser plutôt que de créer vos propres variables personnalisées. La plupart des variables d'instance peuvent être définies et lues, ce qui vous permet de modifier les propriétés et le comportement d'une instance en modifiant simplement la valeur d'une certaine variable. visible à false. Vous trouverez ci-dessous les différentes variables qui sont initialisées pour toutes les instances de tous les objets de votre jeu. IMPORTANT ! Si une instance se trouve dans une séquence, certaines de ces variables - comme image_xscale / image_yscale / image_angle / x / y - seront écrasées lors de la mise à jour de la séquence met à jour chaque étape après avoir commencé à être jouée.
