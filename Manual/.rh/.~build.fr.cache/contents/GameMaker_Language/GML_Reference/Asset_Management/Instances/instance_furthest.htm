instance_furthest Cette fonction va vérifier toutes les instances de l'objet donné pour voir laquelle est la plus éloignée du point x/y donné. Toutes les vérifications se feront de la position x/y donnée à l'origine (la position x/y) des instances de l'objet spécifié. Si aucune instance de l'objet n'existe, la fonction renverra le mot-clé noone, mais s'il existe des instances, elle renverra le mot-clé id de l'instance trouvée. Veuillez noter que si l'instance qui exécute le code a le même index d'objet que l'objet vérifié, elle sera incluse dans la vérification (cela inclut la vérification des objets parents si l'instance appelante est également un enfant du parent). Syntaxe : instance_furthest(x, y, obj) ; Arguments Description x La position x pour vérifier les instances éloignées de. y La position y à vérifier pour les instances éloignées de. obj. L'objet dont il faut rechercher les instances. Les retours : Real (soit une valeur d'ID d'instance, soit le mot clé noone) Exemple : var inst ; inst = instance_furthest(x, y, object_index) ; if inst != id { draw_line(x, y, inst.x, inst.y) ; } Le code ci-dessus trouvera l'instance la plus éloignée qui partage le même index d'objet que l'instance qui exécute le code et stockera l'id dans une variable. Cette variable est ensuite vérifiée pour voir si elle est la même que l'id de l'instance appelante et, si ce n'est pas le cas, une ligne est tracée entre les deux instances.
