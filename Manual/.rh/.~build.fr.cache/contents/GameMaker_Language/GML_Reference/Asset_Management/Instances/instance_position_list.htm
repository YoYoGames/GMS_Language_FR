liste_position_instance Cette fonction vous permet de vérifier qu'une position n'entre pas en collision avec toutes les instances d'un autre objet. Lorsque vous utilisez cette fonction, vous vérifiez la présence de toutes les instances d'un objet en un seul point de la pièce. ou par rapport au masque des instances si la fonction de collisions précises est activée pour cet objet. La fonction remplira une liste de DS (précédemment créée) avec les valeurs uniques de toutes les instances de l'objet qui sont considérées comme faisant partie de la liste de DS. id uniques de toutes les instances de l'objet qui sont considérées comme étant en collision avec l'instance appelante, et vous avez la possibilité d'ordonner la liste en fonction de la distance entre la position vérifiée et l'origine des instances considérées comme étant en collision. Notez que la fonction accepte également le mot-clé spécial GML_Overview/Instance_Keywords.htm">tousdans ce cas, toutes les instances trouvées en collision seront listées. La fonction renvoie le nombre d'instances trouvées, ou 0 si aucune n'est trouvée. Syntaxe : instance_position_list(x, y, obj, liste, ordonné) ; Arguments Description x La position x à vérifier pour les instances. y La position y pour vérifier les instances. obj. L'objet dont il faut rechercher les instances. liste La liste DS à utiliser pour stocker les ID des instances en collision. commandé Si la liste doit être ordonnée par distance(true) ou non(false). Les retours : Int (Le nombre d'instances trouvées en collision) Exemple : var _list = ds_list_create() ; var _num = instance_position_list(mouse_x, mouse_y, all, _list, false) ; if _num > 0 { for (var i = 0 ; i < _num ; ++i ;) { instance_destroy(_list[| i]) ; } } ds_list_destroy(_list) ; Le code ci-dessus vérifiera la présence d'une collision avec toutes les instances trouvées à la position de la souris. Celles-ci seront ajoutées à une liste DS, qui sera ensuite parcourue en boucle pour détruire chacune des instances en collision.
