friction Friction est l'une des propriétés "intégrées" que les instances peuvent avoir et peut être utilisée pour ralentir l'instance au fil du temps lorsque le speed est différent de zéro. Cela fonctionne simplement en soustrayant une quantité de la vitesse à chaque étape jusqu'à ce que l'objet ait une vitesse de 0, donc si le frottement est réglé sur, par exemple, 0,1 et la vitesse de l'instance est de 1 (1 pixel par étape), il sera ralentissez et arrêtez après 10 étapes. Notez également que le frottement est appliqué aux vitesses positives et négatives de la même manière, le résultat net étant toujours que l'objet a une vitesse de 0 après un temps donné. Syntaxe:friction; Retour:Real (single precision floating point value) Exemple:if abs(speed) > 0     {     friction = 0.05;     } else     {     friction = 0;     } Le code ci-dessus n'appliquera le frottement que si la vitesse absolue de l'instance est supérieure à 0.
