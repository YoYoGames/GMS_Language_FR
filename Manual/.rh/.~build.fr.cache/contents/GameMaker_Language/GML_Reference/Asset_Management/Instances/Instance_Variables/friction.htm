friction Lafriction est l'une des propriétés "intégrées" que les instances peuvent avoir et peut être utilisée pour ralentir l'instance au fil du temps lorsque le paramètre vitesse est différente de zéro. Elle fonctionne simplement en soustrayant une quantité de la vitesse à chaque pas jusqu'à ce que l'objet ait une vitesse de 0. Ainsi, si la friction est fixée à, par exemple, 0.1 et que la vitesse de l'instance est de 1 (1 pixel par pas), elle ralentira et s'arrêtera après 10 pas. Notez également que la friction est s'applique également aux vitesses positives et négatives, le résultat net étant toujours que l'objet a une vitesse de 0 après un temps donné. Syntaxe : friction ; Les retours : Réel (valeur à virgule flottante de simple précision) Exemple : si abs(vitesse) > 0 { friction = 0.05 ; } else { friction = 0 ; } Le code ci-dessus n'appliquera la friction que si la vitesse absolue de l'instance est supérieure à 0.
