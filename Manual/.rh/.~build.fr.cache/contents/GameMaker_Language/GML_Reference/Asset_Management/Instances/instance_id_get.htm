instance_id_get Avec cette fonction, vous pouvez obtenir la valeur d'identification unique de n'importe quelle instance de la liste d'instances actuellement active. Vous indiquez l'index de la liste d'instances à partir duquel vous souhaitez obtenir l'ID et la fonction renvoie la valeur à stocker dans une variable. Syntaxe : instance_id_get(index) ; Arguments Description indice L'index dans la liste d'instances de 0 à (nombre d'instances - 1). Les retours : Valeur de l'ID de l'instance (réel) Exemple : for (var i = 0 ; i < instance_count ; ++i ;) { var temp_id = instance_id_get(i) ; with (temp_id) { speed += 0.1 ; } } Le code ci-dessus passera en boucle toutes les instances de la pièce et ajoutera 0.1 à leur vitesse.
