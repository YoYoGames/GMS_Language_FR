Les instances Les instances sont créées à partir d' objets et, une fois créées, elles peuvent être modifiées et manipulées via le GML et les DnD™. Les principales fonctions d'instance liées aux instances disponibles et les plus couramment utilisées sont: instance_create_layerinstance_create_depthinstance_destroyinstance_existsinstance_changeinstance_copyinstance_findinstance_furthestinstance_nearestinstance_numberinstance_placeinstance_place_listinstance_positioninstance_position_list Notez qu'il existe deux variables intégrées supplémentaires (ce sont des variables de portée globale ) et une fonction liée à l'utilisation d'instances qui peuvent être utiles pour récupérer certains détails: instance_id_getinstance_idinstance_count Il existe un certain nombre de variables intégrées disponibles dans chaque instance d'un objet qui peuvent être utilisées pour contrôler différents aspects de la façon dont l'instance est affichée, comment elle utilise d'autres actifs et comment elle réagit aux collisions, etc. énumérés ici: Variables d'instance Les instances ont une variable de tableau qui peut être utilisée pour obtenir ou définir des valeurs d'événement d'alarme, mais il existe également deux fonctions qui peuvent être utilisées pour faire la même chose:alarm_getalarm_set Enfin, vous avez également la possibilité de désactiver les instances, ce qui signifie qu'elles ne traiteront aucun de leurs événements, même si elles existeront toujours dans le jeu.
