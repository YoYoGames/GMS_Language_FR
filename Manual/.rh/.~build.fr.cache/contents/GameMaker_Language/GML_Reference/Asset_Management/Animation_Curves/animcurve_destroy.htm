animcurve_destroy Cette fonction peut être utilisée pour détruire toutes les courbes d'animation qui ont été précédemment créées avec la fonction animcurve_create(). L'appel de cette fonction supprimera la courbe d'animation de la mémoire et nettoiera également tous les canaux ou points qu'elle contient (il n'est jamais nécessaire de les nettoyer manuellement). Notez que vous ne pouvez pas détruire les courbes d'animation créées dans le navigateur d'actifs, mais uniquement celles créées au moment de l 'exécution. Syntaxe : animcurve_destroy(curve_struct) ; Arguments Description structure_courbe Le pointeur vers la structure de la courbe à détruire. Les retours : N/A Exemple : animcurve_destroy(my_curve) ; Le code ci-dessus va détruire la structure de courbe d'animation (précédemment créée) indexée dans la variable "my_curve".
