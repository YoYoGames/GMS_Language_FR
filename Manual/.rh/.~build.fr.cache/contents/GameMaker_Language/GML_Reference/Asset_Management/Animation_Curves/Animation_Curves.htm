Courbes d'animation Les courbes d'animation peuvent être configurées dans le navigateur d'actifs en tant qu'actifs à l'aide de l' éditeur de courbes d'animation et elles peuvent ensuite être utilisées dans l' éditeur de séquences pour contrôler les pistes d'une séquence. Cependant, ils peuvent également être utilisés dans votre code pour modifier les valeurs au fil du temps dans vos jeux, des effets d'interpolation sur les boutons au mouvement, en passant par le changement de couleur, etc. Vous pouvez également créer vos propres courbes d'animation au runtime utilisant les fonctions répertoriées ici, ce qui en fait un outil très flexible pour vos jeux.
