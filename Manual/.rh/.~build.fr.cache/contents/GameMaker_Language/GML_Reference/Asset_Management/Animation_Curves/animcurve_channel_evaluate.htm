animcurve_channel_evaluate Cette fonction peut être utilisée pour obtenir la valeur à un moment donné à partir d'une structure de canal. Vous fournissez le pointeur struct pour le canal (comme renvoyé par la fonction animcurve_get_channel(), ou comme renvoyé dans la structure de courbe d'animation de la fonction animcurve_get() ) et la position "x" (temps) le long du canal de la courbe à évaluer. Cette position doit être comprise entre 0 et 1, et la fonction renverra la valeur de la courbe à cette position, ou elle renverra 0 si la structure de canal fournie n'est pas valide. Syntaxe:animcurve_channel_evaluate(channel_struct, posx); Argument Description channel_struct Pointeur struct pour le canal à évaluer. posx La position dans le temps à vérifier (de 0 à 1). Retour:Real Exemple:var _channel = animcurve_get_channel(ac_AlphaCurve, 0); var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000)); image_alpha = _val; Le code ci-dessus obtient la structure de canal pour l'élément de courbe d'animation "ac_AlphaCurve". Il utilise ensuite la valeur d'évaluation renvoyée pour définir l'alpha de l'image pour l'instance.
