object_index Cette variable en lecture seule renvoie l'index de l'objet à partir duquel l'instance a été créée. Ce n'est pas le même que le nom de l'objet, qui est une chaîne et peut être trouvé en utilisant object_get_name(), car cette fonction renvoie le numéro d'index, qui est une valeur unique que GameMaker Studio 2 attribue à chaque objet au moment de la création. Syntaxe:object_index; Retour:Real (integer - resource index value) Exemple:obj_name = object_get_name(object_index); Le code ci-dessus utilisera le object_index pour trouver le nom de l'objet à partir duquel l'instance actuelle a été créée.
