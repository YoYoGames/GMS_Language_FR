objet_index Cette variable en lecture seule renvoie l'index de l'objet à partir duquel l'instance a été créée. Ce n' est pas la même chose que le nom de l'objet, qui est une chaîne de caractères et peut être trouvé en utilisant la commande object_get_name(), car cette fonction renvoie le numéro d'index, qui est une valeur unique que GameMaker Studio 2 attribue à chaque objet au moment de sa création. Syntaxe : object_index ; Les retours : Real (nombre entier - valeur de l'index de la ressource) Exemple : obj_name = object_get_name(object_index) ; Le code ci-dessus utilisera l'object_index pour trouver le nom de l'objet à partir duquel l'instance actuelle a été créée.
