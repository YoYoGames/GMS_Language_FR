événement_performer Cette fonction exécutera l'événement spécifié, avec l'argument désigné, pour l'instance qui exécute le code. Il existe de nombreuses options qui permettent une simulation complète de tous les événements possibles, mais notez que cette fonction ne fait qu'exécuter tout le code de l'événement. dans l'événement et le jeu ne modifiera rien pour qu'il se déclenche manuellement. Par exemple, si vous choisissez d'exécuter un événement de pression du clavier, l'événement sera déclenché mais la touche correspondante ne sera pas reconnue comme ayant été pressée. Ou si vous un événement d'alarme, le compteur d'alarme ne sera pas mis à -1 mais continuera à décompter.
