object_set_sprite Avec cette fonction, vous pouvez définir l'index du sprite d'un objet spécifique. Cela signifie que toutes les instances de cet objet créées après la modification du sprite_index seront créées avec ce nouveau sprite_index, tandis que les instances qui se trouvent déjà dans la salle peuvent ne pas être affectées. Veuillez noter qu'il ne s'agit pas d'une fonction d'instance! Vous pouvez définir l'index de sprite d'instances individuelles à l'aide de la sprite_index variable et ont donc dix instances, toutes avec un sprite différent de l'objet à partir duquel elles ont été créées, et même si vous modifiez l'index de sprite de l'objet à l'aide de cette fonction, toutes les instances qui sont actuellement dans la salle resteront telles qu'elles étaient, et seulement les instances créées après l'appel de la fonction démarreront avec le nouveau sprite. Syntaxe:object_set_sprite( index, spr ); Argument Description index L'index de l'objet à modifier. spr Le sprite à attribuer à l'objet. Retour:N/A Exemple:switch (room)     {     case rm_start: object_set_sprite(obj_Player, spr_uniform); break;     case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;     case rm_end: object_set_sprite(obj_Player, spr_casual); break;     } instance_create_layer(32, 32, "Instances", obj_Player); Le code ci-dessus définira l'index du sprite de l'objet "obj_Player" à des valeurs différentes en fonction de la pièce dans laquelle se trouve actuellement l'instance exécutant le code.
