objet_set_sprite Cette fonction vous permet de définir l'indice de sprite d'un objet spécifique. Cela signifie que toutes les instances de cet objet qui sont créées après que le sprite_index a été modifié seront créées avec ce nouveau sprite_index, tandis que les instances qui sont déjà dans la pièce peuvent ne pas être affectées. Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Vous pouvez définir l'indice de sprite des instances individuelles à l'aide de la variable sprite_index et ainsi avoir dix instances ayant toutes un sprite différent de celui de l'objet à partir duquel elles ont été créées, et même si vous changez l'indice de sprite de l'objet en utilisant cette fonction, toutes les instances qui sont actuellement dans la room resteront telles qu'elles étaient, et seules les instances créées après avoir appelé la fonction commenceront avec le nouveau sprite. Syntaxe : object_set_sprite( index, spr ) ; Arguments Description indice L'indice de l'objet à modifier. spr Le sprite à affecter à l'objet. Les retours : N/A Exemple : switch (room) { case rm_start : object_set_sprite(obj_Player, spr_uniform) ; break ; case rm_middle : object_set_sprite(obj_Player, spr_swimsuit) ; break ; case rm_end : object_set_sprite(obj_Player, spr_casual) ; break ; } instance_create_layer(32, 32, "Instances", obj_Player) ; Le code ci-dessus définit l'indice de sprite de l'objet "obj_Player" à différentes valeurs en fonction de la pièce dans laquelle se trouve l'instance qui exécute le code.
