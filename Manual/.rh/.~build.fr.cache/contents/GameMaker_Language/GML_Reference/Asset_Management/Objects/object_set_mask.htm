object_set_mask Avec cette fonction, vous pouvez définir le mask_index d'un objet spécifique (pour plus d'informations sur les masques, voir l' éditeur d'objets ). Cela signifie que toutes les instances de cet objet créées après la modification du masque seront créées avec ce nouveau masque, tandis que les instances déjà présentes dans la pièce ne seront peut-être pas affectées. Vous pouvez définir ceci sur -1 pour supprimer un sprite de masque et donc par défaut sur le masque défini pour le sprite de l'objet (ou pas de masque si aucun sprite n'a été choisi). Veuillez noter qu'il ne s'agit pas d'une fonction d'instance! Vous pouvez définir l'index de masque des instances individuelles à l'aide de la mask_index variable et ont donc dix instances, toutes avec un masque différent de celui de l'objet à partir duquel elles ont été créées, et même si vous modifiez l'index de masque de l'objet à l'aide de cette fonction, toutes les instances qui se trouvent actuellement dans la pièce resteront telles quelles, et seulement les instances créées après l'appel de la fonction démarreront avec le nouveau masque. Syntaxe:object_set_mask(index, spr); Argument Description index L'index de l'objet à modifier. spr Le nouveau sprite à utiliser comme masque de l'objet. Retour:N/A Exemple:if (global.level == 10)     {     object_set_mask(obj_Platform, spr_mask_10);     } Le code ci-dessus vérifie la valeur de la variable globale et modifie ensuite l'index de masque de l'objet "obj_Platform" s'il est égal à dix.
