objet_set_mask Cette fonction vous permet de définir le mask_index d'un objet spécifique (pour plus d'informations sur les masques, voir L'éditeur d'objets). Cela signifie que toutes les instances de cet objet qui sont créées après la modification du masque seront créées avec ce nouveau masque, tandis que les instances qui sont déjà dans la pièce peuvent ne pas être affectées. Vous pouvez définir cette valeur à -1 pour supprimer un sprite de masque et utiliser par défaut le masque défini pour le sprite de l'objet (ou aucun masque si aucun sprite n'a été choisi). choisi). Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Vous pouvez définir l'indice de masque des instances individuelles en utilisant la variable mask_index et ainsi avoir dix instances ayant toutes un masque différent de celui de l'objet à partir duquel elles sont créées. masque différent de celui de l'objet à partir duquel elles sont créées, et même si vous changez l'indice de masque de l'objet en utilisant cette fonction, toutes les instances qui sont actuellement dans la room resteront telles qu'elles étaient, et seules les instances créées après avoir appelé la fonction commenceront avec le nouveau masque. avec le nouveau masque. Syntaxe : objet_set_mask(index, spr) ; Arguments Description indice L'indice de l'objet à modifier. spr Le nouveau sprite à utiliser comme masque de l'objet. Les retours : N/A Exemple : if (global.level == 10) { object_set_mask(obj_Platform, spr_mask_10) ; } Le code ci-dessus vérifie la valeur de la variable globale et change l'index du masque de l'objet "obj_Platform" s'il est égal à dix.
