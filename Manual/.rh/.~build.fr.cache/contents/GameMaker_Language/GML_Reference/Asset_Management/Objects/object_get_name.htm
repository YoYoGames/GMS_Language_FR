object_get_name Cette fonction renverra le nom sous forme de chaîne de l'objet spécifié. Ce nom est celui qui a été spécifié pour l'objet dans le navigateur de ressources de la GameMaker Studio 2. Veuillez noter qu'il ne s'agit que d' une chaîne et ne peut pas être utilisée pour référencer l'objet directement - pour cela, vous auriez besoin de l' index d'objet. Vous pouvez cependant utiliser cette chaîne pour obtenir l' index de l'objet en utilisant la chaîne retournée avec la fonction asset_get_index(). Syntaxe:object_get_name(obj); Argument Description obj L'index de l'objet à vérifier. Retour:String Exemple:str = object_get_name(object_index); Le code ci-dessus obtiendra le nom de l'index d'objet pour l'instance exécutant le code et stockera la valeur de retour dans la variable "str".
