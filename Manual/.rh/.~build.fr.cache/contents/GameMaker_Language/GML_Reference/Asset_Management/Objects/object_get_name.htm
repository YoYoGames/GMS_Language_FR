objet_get_name Cette fonction renvoie le nom sous forme de chaîne de l'objet spécifié. Ce nom est celui qui a été spécifié pour l'objet dans l'Asset Browser de la fenêtre principale de GameMaker Studio 2. Veuillez noter qu'il s'agit uniquement d'une chaîne de caractères et ne peut pas être utilisée pour référencer l'objet directement - pour cela, vous aurez besoin de l'index de l'objet. Vous pouvez toutefois utiliser cette chaîne pour obtenir l'index de l'objet en utilisant la chaîne retournée avec la fonction asset_get_index(). Syntaxe : objet_get_name(obj) ; Arguments Description obj. L'index de l'objet à vérifier. Les retours : Chaîne de caractères Exemple : str = object_get_name(object_index) ; Le code ci-dessus obtiendra le nom de l'index de l'objet pour l'instance qui exécute le code et stockera la valeur de retour dans la variable "str".
