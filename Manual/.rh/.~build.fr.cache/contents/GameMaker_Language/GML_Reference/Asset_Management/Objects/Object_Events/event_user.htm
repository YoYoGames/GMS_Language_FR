event_user Avec cette fonction, vous indiquez à l'instance d'exécuter les actions ou le code qui ont été placés dans l'un des 16 événements définis par l'utilisateur. Ces événements ne peuvent être appelés que de cette manière, ou en utilisant le event_perform fonction. Syntaxe:event_user(numb); Argument Description numb Le nombre d'événement à appeler, entre 0 et 15. Retour:N/A Exemple:event_user(4); Cela appellerait l'événement 4 défini par l'utilisateur.
