utilisateur de l'événement Avec cette fonction, vous demandez à l'instance d'exécuter les actions ou le code qui ont été placés dans l'un des 16 événements définis par l'utilisateur. Ces événements ne peuvent être appelés que de cette façon, ou en utilisant la fonction event_perform fonction. Syntaxe : event_user(numb) ; Arguments Description engourdi Le numéro de l'événement à appeler, entre 0 et 15. Les retours : N/A Exemple : event_user(4) ; Il s'agit de l'événement 4 défini par l'utilisateur.
