numéro de l'événement Cette variable en lecture seule renvoie le numéro de l'événement en cours d'appel, où le numéro se réfère en fait au "sous-événement" de l'événement, c'est-à-dire que pour l'événement step, le numéro de l'événement pourrait être l'une des constantes ev_step_normal , ev_step_begin, ou ev_step_end. Pour une liste complète des constantes disponibles pour les sous-événements spécifiques, voir event_perform()et si vous avez besoin de connaître l'événement principal lui-même, vous devriez vérifier les constantes event_type. Syntaxe : numéro de l'événement ; Les retours : Real Exemple : switch (event_number) { case ev_step_normal : show_debug_message("Step event !") ; break ; case ev_game_start : show_debug_message("Game Start"") ; break ; case ev_room_start : show_debug_message("Room Start !") ; break ; } Le code ci-dessus pourrait être appelé depuis un script et utilisé pour afficher des messages de débogage vous informant de l'événement en cours de déclenchement.
