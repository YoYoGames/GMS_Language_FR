object_get_persistent Cette fonction vous dira si l'objet que vous vérifiez a été marqué comme "persistant" ou non. Un objet persistant est un objet qui fera en sorte que toutes les instances de celui-ci soient transmises d'une pièce à l'autre à moins qu'elles ne soient explicitement détruites. Veuillez noter qu'il ne s'agit pas d'une fonction d'instance! Ainsi, vous pouvez avoir un objet persistant et une instance non persistante du même objet et vice-versa. Vous pouvez définir un indicateur persistant d'instances individuelles à l'aide de la persistent variable d'instance. Syntaxe:object_get_persistent(obj); Argument Description obj L'index de l'objet à vérifier. Retour:Boolean Exemple:if (!persistent) && (object_get_persistent(object_index))     {     persistent = true;     } Le code ci-dessus vérifiera l'instance qui l'exécute pour voir si elle est persistante ou non et vérifiera l'index d'objet de l'instance pour voir si elle est marquée comme persistante ou non. Si l'instance n'est pas encore persistante, l'index d'objet est marqué comme persistant, il définira "persistent" sur true pour cette instance.
