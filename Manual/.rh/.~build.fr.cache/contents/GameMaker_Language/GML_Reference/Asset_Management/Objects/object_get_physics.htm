objet_get_physique Cette fonction vous dira si l'objet que vous vérifiez a été marqué comme "physics enabled" - dans ce cas, elle renverra true, - ou non - dans ce cas, elle renverra false. Syntaxe : objet_get_physics(obj) ; Arguments Description obj. L'index de l'objet à vérifier. Les retours : Booléen Exemple : if object_get_physics(object_index) { phy_active = true ; } Le code ci-dessus vérifie si l'objet à partir duquel il a été créé est compatible avec la physique et, si c'est le cas, il active la simulation physique pour l'instance.
