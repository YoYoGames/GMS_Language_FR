object_get_physics Cette fonction vous dira si l'objet que vous vérifiez a été marqué comme "physique activé" - auquel cas il retournera true, - ou pas - auquel cas il reviendra false. Syntaxe:object_get_physics(obj); Argument Description obj L'index de l'objet à vérifier. Retour:Boolean Exemple:if object_get_physics(object_index)     {     phy_active = true;     } Le code ci-dessus vérifiera l'instance qui l'exécute pour voir si l'objet à partir duquel il est créé est activé pour la physique, et si c'est le cas, il active la simulation physique pour l'instance.
