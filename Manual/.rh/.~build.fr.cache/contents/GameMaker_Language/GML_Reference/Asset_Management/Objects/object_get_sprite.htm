objet_get_sprite Cette fonction vous dira si l'objet que vous vérifiez a un sprite ou non, et si c'est le cas, elle retournera l'index de ce sprite, ou -1 si ce n'est pas le cas. Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Vous pouvez avoir un objet sans alors qu'une instance de ce même objet peut en avoir un et vice-versa, ou ils peuvent même avoir des sprites différents. Vous pouvez définir le sprite d'une instance individuelle en utilisant la variable sprite_index instance. Syntaxe : objet_get_sprite(obj) ; Arguments Description obj. L'index de l'objet à vérifier Les retours : Real (ID de l'actif du sprite) Exemple : var _spr = object_get_sprite(object_index) ; if (sprite_index != _spr) { sprite_index = _spr ; } L'exemple ci-dessus vérifiera le sprite_index de l'instance par rapport au sprite du object_index de l'instance. S'ils ne sont pas identiques, il attribuera à l'instance le même sprite que celui de l'index de l'objet. l'index de l'objet à l'instance.
