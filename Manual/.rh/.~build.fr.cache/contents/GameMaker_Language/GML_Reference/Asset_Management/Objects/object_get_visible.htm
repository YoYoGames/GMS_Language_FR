objet_get_visible Cette fonction vous dira si l'objet que vous vérifiez a été marqué comme "visible" (exécute son événement de dessin) ou non (n'exécute pas son événement de dessin). Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Ainsi, vous pouvez avoir un objet visible et une instance invisible du même objet et vice-versa. Vous pouvez définir la visibilité d'une instance individuelle en utilisant la variable visible instance. Syntaxe : objet_get_visible(obj) ; Arguments Description obj. L'index de l'objet à vérifier. Les retours : Booléen Exemple : if (!visible) && (object_get_visible(object_index)) { visible = true ; } Le code ci-dessus vérifiera l'instance en cours d'exécution pour voir si elle est visible ou non, ainsi que l'index de l'objet de l'instance pour voir si elle est marquée comme visible ou non. Si l'instance n' est pas encore visible, l'index de l'objet est marqué comme on, il définira "visible" à true pour cette instance.
