font_get_fontname Avec cette fonction, vous pouvez obtenir le nom système réel de la ressource de police sélectionnée. Cette fonction renvoie une chaîne et non un index, et le nom retourné dépendra de la police utilisée. Par exemple, vous pouvez avoir une ressource de police appelée "fnt_Main" dans le GameMaker Studio 2, et la police elle-même peut être la police système Windows Arial. Dans ce cas, la fonction retournera "Arial" car c'est le nom système de la police. Si vous avez besoin du nom tel qu'il apparaît dans le navigateur de ressources, vous devez utiliser font_get_name(). Syntaxe:font_get_fontname(ind); Argument Description ind Index de la police à vérifier. Retour:String Exemple:fnt_Name = font_get_fontname(font0); Le code ci-dessus obtiendra le nom système d'une ressource de police et le stockera sous forme de chaîne dans la variable "fnt_Name".
