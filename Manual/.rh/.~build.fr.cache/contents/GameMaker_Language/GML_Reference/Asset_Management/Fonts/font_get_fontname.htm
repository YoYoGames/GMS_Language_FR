font_get_fontname Cette fonction permet d'obtenir le nom système réel de la ressource de police sélectionnée. Cette fonction renvoie une chaîne et non un index, et le nom renvoyé dépend de la police utilisée. Par exemple, vous pouvez avoir un actif de police appelé "fnt_Main" dans le navigateur d'actifs de GameMaker Studio 2, et la police elle-même peut être la police système Windows Arial. Dans ce cas, la fonction renvoie " Arial ". car c'est le nom système de la police. Si vous avez besoin du nom tel qu'il apparaît dans l'Asset Browser, vous devez utiliser la fonction suivante font_get_name(). Syntaxe : font_get_fontname(ind) ; Arguments Description ind Index de la police à vérifier. Les retours : Chaîne de caractères Exemple : fnt_Name = font_get_fontname(font0) ; Le code ci-dessus récupère le nom système d'une ressource de police et le stocke sous forme de chaîne dans la variable "fnt_Name".
