font_replace_sprite Avec cette fonction, vous pouvez utiliser une «bande de sprite» pour remplacer un actif de police de sprite précédemment créé, où chaque sous-image serait un symbole ou une lettre individuelle. L'index est le numéro de carte UTF8 pour le premier caractère à utiliser et les plages à utiliser sont les mêmes que vous le feriez pour une police normale, commençant par "!" mais rappelez-vous que vous ne devez inclure que ce dont vous avez besoin et que vous devez utiliser le ord() pour obtenir la valeur UTF8 correcte de la lettre de départ. Vous pouvez également définir l'espacement de la police pour qu'il soit proportionnel (true ) ou non (false ), où une police proportionnelle est espacée en fonction de la largeur réelle des lettres (la lettre «i» prend donc moins de place que la lettre «w», par exemple) tandis qu'une police non proportionnelle sera espacée en fonction du sous- largeur de l'image, comme une police à espacement fixe. Enfin, vous pouvez définir l'espace à laisser entre chaque lettre lors de l'écriture, et cela peut être n'importe quelle valeur entière, 0 étant sans espace (les lettres "toucheront" si elles sont proportionnelles). Si la fonction échoue, elle échouera silencieusement sans remplacer la police de sprite donnée. Il est également intéressant de noter qu'il ne peut y avoir d' espaces dans la table de caractères, de sorte que si vous, par exemple, utilisez '.' et '0' mais pas '/', vous devez quand même inclure '/' ou laisser une sous-image vide entre les deux caractères utilisés. L'image ci-dessous montre une disposition de police de sprite typique: Syntaxe:font_replace_sprite(ind, spr, first, prop, sep); Argument Description ind L'index de la police qui doit être remplacée (c'est-à-dire que toutes les références à cette ressource après cette fonction feront référence à la nouvelle police, pas à l'original). spr Le sprite sur lequel ajouter une police. first L'index du premier caractère disponible. prop Si l'espace blanc autour de chaque lettre doit être supprimé (s'il est faux, la police créée sera monospace, sinon ce sera comme la plupart des polices régulières). sep L'espace entre chaque lettre, 0 signifiant la droite d'une lettre touche juste la gauche de la suivante. Retour:N/A Exemple:font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2); Le code ci-dessus remplacera la ressource de police indexée dans la variable globale "Police" par une nouvelle ressource de police à partir d'un fichier image externe.
