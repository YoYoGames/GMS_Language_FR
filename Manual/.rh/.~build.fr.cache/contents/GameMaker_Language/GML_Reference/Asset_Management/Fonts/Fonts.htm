Polices GameMaker Studio 2 dispose d'un certain nombre de fonctions qui peuvent être utilisées pour obtenir des informations sur les ressources de polices incluses, ainsi que d'un autre ensemble de fonctions qui peuvent être utilisées pour modifier, ajouter ou supprimer des polices de votre jeu au moment de l'exécution. En particulier la fonction font_add() peut être utilisé pour ajouter des polices non standard, comme les polices de glyphes asiatiques, dans *.ttf format. Il est à noter que si vous ajoutez une police à votre jeu qui n'a pas les glyphes requis par le texte que vous souhaitez écrire, alors le caractère Unicode 9647 (▯) est utilisé pour remplacer ces glyphes manquants lors du rendu dans le événement de tirage au sort. Donc, si votre police n'a pas, par exemple, le symbole º, alors écrire 90 ° produira en fait 90 °.
