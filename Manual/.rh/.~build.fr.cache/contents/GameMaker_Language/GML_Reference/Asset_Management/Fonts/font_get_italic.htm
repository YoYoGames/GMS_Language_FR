font_get_italic Avec cette fonction, vous pouvez vérifier n'importe quelle ressource de police pour voir si elle a le drapeau italique ou non. Si c'est le cas, la fonction renvoie true, sinon elle renvoie false. Syntaxe : font_get_italic(ind) ; Arguments Description ind Index de la police à vérifier. Les retours : Booléen Exemple : if font_get_italic(fnt_Main) { draw_set_font(fnt_Main) ; } La police de dessin active devient fnt_Main si elle est définie comme italique dans ses propriétés de police.
