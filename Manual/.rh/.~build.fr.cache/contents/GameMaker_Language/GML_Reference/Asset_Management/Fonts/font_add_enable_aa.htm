font_add_enable_aa Cette fonction peut être utilisée pour activer ou désactiver l'anticrénelage (AA) pour les polices ajoutées à l'aide de la fonction font_add(). Cette fonction doit être appelée avant d' ajouter des polices. Cette fonction doit être appelée avant que vous n'ajoutiez des polices . Par défaut, l'AA est activé. Notez que cela n'aura aucun effet sur les polices qui ont été ajoutées avant l'appel de la fonction et que la fonction ne doit être appelée qu'une seule fois lors de l'ajout de la police, et non pas à chaque tirage/pas où la police est utilisée. Syntaxe : font_add_enable_aa(enable) ; Arguments Description activer Activation(true) ou désactivation(false) de l'AA pour les polices ajoutées. Les retours : N/A Exemple : if !font_add_get_enable_aa() { font_add_enable_aa(true) ; } Le code ci-dessus vérifie l'état de l'anticrénelage pour les polices ajoutées et s'il n'est pas activé, alors nous l'activons.
