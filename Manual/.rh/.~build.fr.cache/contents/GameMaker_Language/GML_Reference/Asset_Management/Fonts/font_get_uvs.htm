font_get_uvs Cette fonction renvoie un tableau avec les coordonnées UV pour la texture de la police sur la page de texture, en remplissant le tableau avec les valeurs suivantes : [0] = gauche [1] = haut [2] = droite [3] = fond Cette valeur peut ensuite être utilisée dans d'autres fonctions de dessin, notamment dans le dessin général lorsqu'on utilise des primitives ainsi que les fonctions Shader fonctions. Syntaxe : font_get_uvs(police) ; Arguments Description police L'index de la police à utiliser. Les retours : Array Exemple : var tex = font_get_uvs(fnt_Main) ; tex_left = tex[0] ; tex_top = tex[1] ; tex_right = tex[2] ; tex_left = tex[3] ; Le code ci-dessus stocke les coordonnées UV pour l'arrière-plan donné dans un tableau local, puis affecte les valeurs aux variables d'instance.
