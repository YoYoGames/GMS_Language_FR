font_get_bold Avec cette fonction, vous pouvez vérifier n'importe quelle ressource de police pour voir si elle a le drapeau gras ou non. Si c'est le cas, la fonction retournera true, sinon il reviendra false. Syntaxe:font_get_bold(ind); Argument Description ind Index de la police à vérifier. Retour:Boolean Exemple:if font_get_bold(fnt_Main)    {    draw_set_font(fnt_Main);    } Cela définira la police de dessin active sur fnt_Main s'il est défini en gras dans ses propriétés de police.
