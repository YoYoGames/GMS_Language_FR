font_get_name Cette fonction peut être utilisée pour obtenir le nom (sous forme de chaîne) qui a été donné à la police lorsqu'elle a été ajoutée en tant que ressource au navigateur de ressources GameMaker Studio 2 Veuillez noter qu'il ne s'agit que d' une chaîne et ne peut pas être utilisée pour référencer directement la police - pour cela, vous auriez besoin de l' index de police. Vous pouvez cependant utiliser cette chaîne pour obtenir l' index de la police en utilisant la chaîne renvoyée avec la fonction asset_get_index(). Syntaxe:font_get_name(ind); Argument Description ind Index de la police à vérifier. Retour:String Exemple:fnt_Name = font_get_name(font0); Le code ci-dessus obtiendra le nom d'une ressource de police tel qu'il apparaît dans le navigateur de ressources et le stockera sous forme de chaîne dans la variable "fnt_Name".
