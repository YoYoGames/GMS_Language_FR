font_get_name Cette fonction peut être utilisée pour obtenir le nom (sous forme de chaîne) donné à la police lorsqu'elle a été ajoutée en tant qu'actif dans le navigateur d'actifs de GameMaker Studio 2. Veuillez noter qu'il s'agit uniquement d'une et ne peut pas être utilisé pour référencer directement la police - pour cela, vous aurez besoin de l'index de la police. Vous pouvez toutefois utiliser cette chaîne pour obtenir l'index de la police en utilisant la chaîne renvoyée avec la fonction fonction asset_get_index(). Syntaxe : font_get_name(ind) ; Arguments Description ind Index de la police à vérifier. Les retours : Chaîne de caractères Exemple : fnt_Name = font_get_name(font0) ; Le code ci-dessus récupère le nom d'une ressource de police tel qu'il apparaît dans le navigateur d'actifs et le stocke sous forme de chaîne dans la variable "fnt_Name".
