font_exists Cette fonction indique si une police avec l'index spécifié existe ou non. Vous pouvez vérifier les indices de police tels qu'ils sont définis dans le navigateur d'actifs, ou les polices qui ont été ajoutées à l'aide de fonctions comme font_add(). Syntaxe : font_exists(ind) ; Arguments Description ind Index de la police à vérifier. Les retours : Booléen Exemple : if font_exists(fnt_Main) { draw_set_font(fnt_Main) ; } Ceci définira la police de dessin active à fnt_Main si elle existe.
