font_get_size Avec cette fonction, vous pouvez obtenir la taille de n'importe quelle ressource de police, qui est la valeur en points indiquée par la boîte de dialogue de ressource de police. Syntaxe:font_get_size(ind); Argument Description ind Index de la police pour obtenir la taille de. Retour:Real Exemple:sz = font_get_size(font0); Cela obtiendra la taille de la police indexée par la variable "font0" et la stockera dans la variable "sz".
