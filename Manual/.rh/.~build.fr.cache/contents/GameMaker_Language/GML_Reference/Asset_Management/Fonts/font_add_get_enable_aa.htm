font_add_get_enable_aa Cette fonction peut être utilisée pour vérifier si l'anti-aliasing (AA) est activé pour les polices ajoutées à l'aide de font_add(). La fonction retournera true si AA est activé, et false si ce n'est pas le cas. Notez que AA est activé par défaut, mais vous pouvez modifier l'état AA pour les polices ajoutées à l'aide de la fonction font_add_enable_aa(), tant qu'il est appelé avant d' ajouter la police. Syntaxe:font_add_get_enable_aa(); Retour:Boolean Exemple:if !font_add_get_enable_aa()     {     font_add_enable_aa(true);     } Le code ci-dessus vérifie l'état de l'anti-aliasing pour les polices ajoutées et s'il n'est pas activé, nous l'activons.
