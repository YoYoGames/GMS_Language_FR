font_add_get_enable_aa Cette fonction peut être utilisée pour vérifier si l'anticrénelage (AA) est activé pour les polices ajoutées à l'aide de la fonction font_add(). La fonction renvoie true si AA est activé, et false si il ne l'est pas. Notez que l'anticrénelage est activé par défaut, mais vous pouvez modifier l'état de l'anticrénelage pour les polices ajoutées en utilisant la fonction font_add_enable_aa()à condition qu'elle soit appelée avant d' ajouter la police. Syntaxe : font_add_get_enable_aa() ; Les retours : Booléen Exemple : if !font_add_get_enable_aa() { font_add_enable_aa(true) ; } Le code ci-dessus vérifie l'état de l'anticrénelage pour les polices ajoutées et s'il n'est pas activé, alors nous l'activons.
