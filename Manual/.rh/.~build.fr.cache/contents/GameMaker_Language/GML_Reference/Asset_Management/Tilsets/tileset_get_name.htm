tuiles_get_name Cette fonction renvoie le nom sous forme de chaîne du jeu de tuiles spécifié. Ce nom est celui qui a été spécifié pour le jeu de tuiles dans l'Asset Browser de la fenêtre principale de GameMaker Studio 2. Veuillez noter qu'il s'agit uniquement d'une chaîne et ne peut pas être utilisée pour référencer le jeu de tuiles directement - pour cela, vous aurez besoin de l'index du jeu de tuiles. Vous pouvez toutefois utiliser cette chaîne pour obtenir l'index du jeu de tuiles en utilisant la chaîne retournée avec la fonction asset_get_index(). Syntaxe : tileset_get_name(index) ; Arguments Description indice L'index du jeu de tuiles dont on veut obtenir le nom. Renvoie à Chaîne de caractères Exemple : var _l = layer_get_id("tilemap_trees") ; var _m = layer_tilemap_get_id(_l) ; var _t = tilemap_tileset(_m) ; tileset_name = tileset_get_name(_t) ; Le code ci-dessus obtiendra le nom de l'index du jeu de tuiles pour la couche donnée, en stockant la chaîne de retour dans la variable tileset_name.
