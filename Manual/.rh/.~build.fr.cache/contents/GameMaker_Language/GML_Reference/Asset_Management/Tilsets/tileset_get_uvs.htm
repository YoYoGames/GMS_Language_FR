tileset_get_uvs Cette fonction renvoie un tableau avec les coordonnées UV et d'autres données pour la texture de la tuile donnée définie sur la page de texture. La fonction retourne un tableau avec les 8 éléments suivants : [0] = gauche [1] = haut [2] = droite [3] = fond 4] = nombre de pixels que le compilateur d'actifs a enlevé du côté gauche du jeu de tuiles. 5] = nombre de pixels que le compilateur d'actifs a enlevé du côté supérieur du jeu de tuiles. 6] = pourcentage normalisé de données de pixels de la largeur de l'ensemble de tuiles d'origine qui ont été sauvegardées dans la page de texture. 7] = pourcentage normalisé de données de pixels de la hauteur de l'ensemble de tuiles d'origine qui ont été sauvegardées dans la page de texture. Ce tableau peut ensuite être utilisé dans d'autres fonctions de dessin, en particulier lorsque vous travaillez en 3D ou que vous utilisez les fonctions primitives 2D, ainsi que lorsque vous travaillez avec les fonctions Shader fonctions. Syntaxe : tileset_get_uvs(tileset) ; Arguments Description jeu de tuiles L'index du jeu de tuiles à utiliser. Renvoie à Tableau (1D, 8 éléments) Exemple : var tex = tileset_get_uvs(tl_Grass) ; tex_left = tex[0] ; tex_top = tex[1] ; tex_right = tex[2] ; tex_bottom = tex[3] ; Le code ci-dessus stocke les coordonnées UV pour le jeu de tuiles donné dans un tableau local, puis affecte les valeurs aux variables d'instance.
