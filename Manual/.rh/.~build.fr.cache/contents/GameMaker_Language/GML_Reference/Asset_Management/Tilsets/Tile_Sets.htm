Sets de carreaux Les fonctions suivantes sont disponibles lorsque vous travaillez avec les actifs du jeu de tuiles, tels que créés dans le navigateur d'actifs. Ces fonctions ne servent qu' à traiter l'actif lui-même, et non à travailler avec les tuiles ou les cartes de tuiles lorsqu'elles sont présentes dans votre jeu (pour cela, consultez les fonctions de couche appropriées dans la section sur les pièces).
