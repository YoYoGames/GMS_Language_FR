Ensembles de tuiles Les fonctions suivantes sont disponibles lorsque vous travaillez avec des actifs de jeu de tuiles, tels qu'ils ont été créés dans le navigateur d'actifs. Ces fonctions sont uniquement destinées à traiter l'actif lui-même, et non à travailler avec des tuiles ou des cartes de tuiles lorsqu'elles sont présentes dans votre jeu (pour cela, voir les fonctions de calque appropriées dans la section sur les pièces ).
