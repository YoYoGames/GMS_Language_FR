path_get_x Avec cette fonction, vous pouvez obtenir la coordonnée x d'une position sur n'importe quel chemin donné. 0 est le début du chemin, 1 est la fin du chemin, et tout ce qui se trouve entre les deux équivaut à cette distance à travers le chemin. Cela n'a pas besoin d'être un point déterminant du chemin, cela peut être n'importe où. Syntaxe:path_get_x(ind, pos); Argument Description index L'index du chemin à vérifier. pos À quelle distance du chemin à vérifier. Entre 0 (début) et 1 (fin). Retour:Real Exemple:x = path_get_x(pth_WallWalk, 0.5); Cela définira x de l'instance appelante sur la coordonnée x du point exactement à mi-chemin du chemin donné.
