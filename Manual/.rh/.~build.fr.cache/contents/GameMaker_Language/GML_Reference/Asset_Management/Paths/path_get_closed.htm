path_get_closed Cette fonction peut être utilisée pour retourner si le chemin est marqué comme fermé (vrai) ou ouvert (faux), c'est-à-dire si le chemin fait une boucle ou s'il a un début et une fin définitifs. Syntaxe : path_get_closed(index) ; Arguments Description indice L'index du chemin à vérifier. Les retours : Booléen Exemple : state = path_get_closed(pth_Patrol) ; Ceci mettra "state" à true ou false en fonction de l'état de fermeture du chemin indexé dans "pth_Patrol".
