path_get_speed Cette fonction renvoie le facteur de vitesse de toute position donnée sur un chemin. La position doit être une valeur comprise entre 0 et 1 (vous pouvez utiliser path_position, par exemple, si vous avez besoin de la position actuelle de l'instance), 0 étant la position de départ et 1 la fin. La valeur renvoyée sera le facteur par lequel la valeur réelle path_speed  est en cours de définition. Par exemple, sur un chemin avec seulement deux points, 0 et 1, où le facteur de vitesse du point 0 est 100 et le facteur de vitesse du point 1 est 0, la fonction path_get_speed(path, 0.5) retournera 50. Syntaxe:path_get_speed(ind, pos); Argument Description index L'index du chemin à vérifier. pos À quelle distance du chemin à vérifier. Entre 0 (début) et 1 (fin), peut être décimal. Retour:Real Exemple:speed = path_get_speed(pth_Patrol, 0.5) / 20; Cela définira la vitesse de l'instance appelante sur le facteur de vitesse du point exactement à mi-chemin du chemin donné, divisé par 20. Donc, si le facteur de vitesse trouvé est la valeur par défaut de 100, il définira la vitesse de l'instance appelante sur 5.
