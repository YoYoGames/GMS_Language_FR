path_reverse Avec cette fonction, vous pouvez inverser l'ordre dans lequel les points de chemin individuels sont numérotés, ainsi, par exemple, si le chemin a 5 points, le point 0 deviendrait le point 4, le point 1 serait le point 3 et le point 2 ne serait pas changé. La position réelle des points reste la même, seul l'ordre dans lequel ils sont traités est modifié. Cette fonction modifie l'actif du chemin réel, et affectera donc de manière permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, vous devez utiliser une fonction telle que path_duplicate() pour créer d'abord une copie du chemin, puis appelez cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire). Syntaxe:path_reverse(index); Argument Description index L'index du chemin à changer. Retour:N/A Exemple:path_reverse(pth_AI); Cela inverserait l'ordre dans lequel tous les points du chemin donné sont traités.
