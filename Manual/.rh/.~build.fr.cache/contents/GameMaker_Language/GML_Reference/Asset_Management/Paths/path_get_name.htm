path_get_name Cette fonction renverra le nom du chemin référencé sous forme de chaîne. Le nom est tout ce qui a été utilisé pour définir le chemin dans l'éditeur ou (si le chemin a été créé via une fonction de code) il renverra une chaîne au format " _newpathXX " où "XX "est le numéro du chemin généré, commençant à 0 et incrémenté de un à chaque fois qu'un nouveau chemin est créé. Veuillez noter qu'il ne s'agit que d' une chaîne et ne peut pas être utilisé pour référencer le chemin directement - pour cela, vous auriez besoin de l' index du chemin ID. Vous pouvez toutefois utiliser cette chaîne pour obtenir l' ID d'index du chemin à l'aide de la chaîne renvoyée avec la fonction asset_get_index(). Syntaxe:path_get_name(index); Argument Description index L'index du chemin à vérifier. Retour:String Exemple:path_name = path_get_name(path_array[0]); Cela définira "path_name" sur le nom du chemin indexé dans le tableau donné à la position 0.
