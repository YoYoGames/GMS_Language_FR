chemin_get_name Cette fonction renvoie le nom du chemin référencé sous forme de chaîne. Le nom est celui qui a été utilisé pour définir le chemin dans l'éditeur ou (si le chemin a été créé par une fonction de code) il renverra une chaîne au format " _newpathXX" où "XX" est le numéro du chemin généré, commençant à 0 et s'incrémentant de un à chaque fois qu'un nouveau chemin est créé. Veuillez noter qu'il s'agit uniquement d'une et ne peut pas être utilisée pour référencer le chemin directement - pour cela, vous auriez besoin de l'ID de l'index du chemin. Vous pouvez, cependant, utiliser cette chaîne pour obtenir l'ID de l'index du chemin en utilisant la chaîne retournée avec la fonction avec la fonction asset_get_index(). Syntaxe : path_get_name(index) ; Arguments Description indice L'index du chemin à vérifier. Les retours : Chaîne de caractères Exemple : path_name = path_get_name(path_array[0]) ; Ceci mettra "nom_du_chemin" au nom du chemin indexé dans le tableau donné à la position 0.
