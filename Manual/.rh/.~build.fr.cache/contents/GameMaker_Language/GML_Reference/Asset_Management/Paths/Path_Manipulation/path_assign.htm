path_assign Avec cette fonction, vous pouvez copier les données de chemin d'un chemin à un autre. Le chemin étant copié sera effacé d' abord (il devrait avoir des points de chemin) et être complètement écrasée par le chemin en cours de copie à partir. Aucun des deux chemins n'est supprimé dans le processus et le résultat est deux chemins, avec deux index différents, mais exactement la même forme et les mêmes points. En général, vous souhaitez utiliser cette fonction sur un chemin créé à l'aide de path_add(), car si vous l'utilisez sur un actif de chemin, cela affectera de manière permanente le chemin de toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin de la partie. Syntaxe:path_assign(index, path); Argument Description index L'index du chemin à écraser. path L'index du chemin qui écrasera «index». Retour:N/A Exemple:mypath = path_add(); path_assign(mypath, choose(path_1, path_2, path_3)); Le code ci-dessus créera un nouveau chemin vide indexé dans la variable "mypath", puis copiera les données du chemin à partir de l'une des trois ressources de chemin disponibles.
