Manipulation des chemins Dans GameMaker Studio 2, vous n'êtes pas seulement limité aux chemins que vous créez à l'aide de l'éditeur de chemins. Il existe un certain nombre de fonctions qui peuvent être utilisées pour manipuler et modifier ces chemins lors de l'exécution, ou même pour en créer de nouveaux, puis les modifier et les manipuler.
