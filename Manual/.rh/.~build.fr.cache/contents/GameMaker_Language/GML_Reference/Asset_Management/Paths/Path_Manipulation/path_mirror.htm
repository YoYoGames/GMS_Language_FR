chemin_miroir Cette fonction prend tous les points du chemin et les met en miroir le long de l'axe vertical. Cette fonction modifie l'actif réel du chemin, et affectera donc de façon permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, utilisez une fonction telle que path_duplicate() pour créer d'abord une copie du chemin, puis appelez cette fonction sur l'élément dupliquée (n'oubliez pas d'appeler path_delete() sur la ressource lorsqu'elle n'est plus nécessaire). Syntaxe : path_mirror(index) ; Arguments Description indice L'index du chemin d'accès au miroir. Les retours : N/A Exemple : path_mirror(mypath) ; Cela refléterait le chemin indexé dans la variable "mypath" le long de l'axe vertical.
