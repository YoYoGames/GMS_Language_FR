path_insert_point Avec cette fonction, vous pouvez insérer un nouveau point dans un chemin (le chemin peut avoir été créé dans l'éditeur de chemin ou via du code en utilisant path_add() ). Le point sera ajouté au chemin avant le point "n" spécifié dans la fonction. Syntaxe:path_insert_point(index, n, x, y, speed); Argument Description index L'index du chemin dans lequel insérer le point. n Le point de définition pour insérer le nouveau point AVANT. x La coordonnée x (relative au chemin) du nouveau point. y La coordonnée y (relative au chemin) du nouveau point. speed Le facteur de vitesse du point. Retour:N/A Exemple:path_insert_point(mypath, 0, 50, 50, 100); Ceci insérera un point au tout début du chemin indexé dans la variable "mypath", à (50,50), avec un facteur de vitesse de 100%.
