path_insert_point Avec cette fonction, vous pouvez insérer un nouveau point dans un chemin (le chemin peut avoir été créé dans l'éditeur de chemin ou par le code en utilisant la fonction path_add()). Le point sera ajouté dans le chemin avant le point "n" qui est spécifié dans la fonction. qui est spécifié dans la fonction. Syntaxe : path_insert_point(index, n, x, y, speed) ; Arguments Description indice L'index du chemin dans lequel insérer le point. n Le point de définition pour insérer le nouveau point AVANT. x La coordonnée x (par rapport à la trajectoire) du nouveau point. y La coordonnée y (par rapport à la trajectoire) du nouveau point. vitesse Le facteur de vitesse du point. Les retours : N/A Exemple : path_insert_point(mypath, 0, 50, 50, 100) ; Ceci va insérer un point au tout début du chemin indexé dans la variable "mypath", à (50,50), avec un facteur de vitesse de 100%.
