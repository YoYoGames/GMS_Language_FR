path_delete_point Avec cette fonction, vous pouvez supprimer un point du chemin spécifié. Les points du chemin sont numérotés à partir de 0, donc le premier point est toujours le 0ème et le dernier point du chemin sera le 1. path_get_number() - 1. appelez cette fonction sur un actif de trajectoire, la suppression du point sera permanente et toutes les instances affectées à la trajectoire à l'avenir seront dépourvues du point qui a été supprimé. Si ce n'est pas ce dont vous avez besoin, alors vous devriez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appeler cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire). Syntaxe : path_delete_point(ind, n) ; Arguments Description indice L'index de la trajectoire dont il faut supprimer un point. n Le point de définition à supprimer. Les retours : N/A Exemple : var _col = true ; while (_col) { var _num = path_get_number(mypath) - 1 ; if (_num > 2) { var _x1 = path_get_point_x(my_path, _num) ; var _y1 = path_get_point_y(my_path, _num) ; var _x2 = path_get_point_x(my_path, _num -1) ; var _y2 = path_get_point_y(my_path, _num -1) ; if collision_line(_x1, _y1, _x2, _y2, obj_Wall, true, false) { path_delete_point(mypath, _num) ; _num -= 1 ; } else _col = false ; } else _col = false ; } Le code ci-dessus génère une boucle qui vérifie les points d'un chemin en arrière à partir de la fin pour une collision avec l'objet donné. Si une collision est trouvée, un point du chemin est supprimé, jusqu'à ce qu'il n'y ait plus de collision, ou que le chemin ne fasse plus que 2 points.
