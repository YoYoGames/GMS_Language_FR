chemin_existant Cette fonction renvoie si un chemin avec l'indice donné existe ou non. Notez que si vous vérifiez l'existence d'un chemin par le biais d'une variable qui n'a pas encore été déclarée, cela entraînera une erreur. Syntaxe : path_exists(index) ; Arguments Description indice L'index du chemin à vérifier. Les retours : Booléen Exemple : if (path_exists(my_path)) { path_delete(my_path) ; } Ce code vérifie si la variable donnée stocke un index de chemin et si c'est le cas, alors le chemin est supprimé de la mémoire.
