path_exists sa fonction renvoie si un chemin avec l'index donné existe ou non. Notez que si vous vérifiez l'existence d'un chemin à travers une variable qui n'a pas encore été déclarée, cela générera une erreur. Syntaxe:path_exists(index); Argument Description index L'index du chemin à rechercher. Retour:Boolean Exemple:if (path_exists(my_path))     {     path_delete(my_path);     } Ce code vérifie si la variable donnée stocke un index de chemin et si c'est le cas, le chemin est supprimé de la mémoire.
