chemin_get_numéro Cette fonction peut être utilisée pour retourner le nombre de points sur un chemin. Syntaxe : path_get_number(index) ; Arguments Description indice L'index du chemin à vérifier. Les retours : Real Exemple : if (path_get_number(pth_AI) > 1) { path_start( pth_AI, 4, 3, 0) ; } Le code ci-dessus vérifie si un chemin a plus d'un point et si c'est le cas, il commence à déplacer l'instance le long de ce chemin.
