path_endaction Cette variable peut être utilisée pour obtenir ou modifier la réaction d'une instance lorsqu'elle atteint la fin du chemin actuel. Normalement, vous devriez la définir lorsque vous démarrez le chemin en utilisant path_start() mais vous pouvez souhaiter modifier ce comportement en fonction d'un certain nombre d'événements dans votre jeu. Les valeurs disponibles sont exprimées à l'aide des constantes suivantes : Constant Description path_action_stop arrêter le chemin path_action_restart continuer à partir de la position de départ, en sautant au départ si le chemin n'est pas fermé path_action_continue recommencer le chemin à partir de la position actuelle path_action_reverse inverser la vitesse de la trajectoire (exécuter la trajectoire en sens inverse) Syntaxe : path_endaction ; Les retours : Constante (voir description) Exemple : if (path_endaction == path_action_stop) { path_endaction = path_action_reverse ; } Le code ci-dessus vérifiera l'action de fin de chemin et s'il est réglé sur stop, alors l'action de fin sera changée en marche arrière.
