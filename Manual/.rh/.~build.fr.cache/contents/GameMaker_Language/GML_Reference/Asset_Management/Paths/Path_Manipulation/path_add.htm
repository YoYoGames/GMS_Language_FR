chemin_add Avec cette fonction, vous pouvez créer un chemin dans GameMaker Studio 2 sans utiliser l'éditeur de chemin. Cette fonction renvoie l'index du chemin qui doit être stocké dans une variable et utilisé comme référence pour ce chemin à partir de maintenant. Veuillez noter que que le chemin créé est vide, c'est-à-dire qu'il n'a pas de points définis. Vous devrez donc utiliser les autres fonctions disponibles pour ajouter des points au chemin ou utiliser des grilles MP pour générer le chemin. générer la trajectoire. Une fois que vous avez fini d'utiliser le chemin, ou que vous souhaitez en créer un nouveau et stocker son index dans la même variable, vous devez d'abord effacer l'ancien chemin avec la commande path_delete afin d'éviter les fuites de mémoire qui qui peuvent éventuellement faire planter votre jeu. Syntaxe : path_add() ; Les retours : Real Exemple : global.newpath = path_add() ; Ceci créera un nouveau chemin et assignera son index à global.newpath.
