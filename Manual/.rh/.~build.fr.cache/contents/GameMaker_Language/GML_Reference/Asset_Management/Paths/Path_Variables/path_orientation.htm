orientation_du_chemin Cette variable contient l'orientation actuelle du chemin qui a été attribué à l'instance lorsque la fonction path_start() a été appelée. Lorsqu'un chemin est créé, son orientation est par défaut de 0 degrés, mais vous pouvez définir cette valeur comme vous le souhaitez à l'aide de cette variable. N'oubliez pas que dans GameMaker Studio 2 (sauf si vous utilisez la physique), les angles sont calculés dans le sens inverse des aiguilles d'une montre. à 90° fera pivoter le chemin vers la gauche. Syntaxe : orientation_du_chemin ; Les retours : Real Exemple : mypath = path_duplicate(choose(path_1, path_2, path_3, path_4)) ; path_start(path, 4, path_action_reverse, 0) ; path_orientation = 90 ; Le code ci-dessus duplique un actif de chemin aléatoire et préétabli dans la variable "mypath".
