chemin_index Toutes les ressources de GameMaker Studio 2 possèdent un numéro d'identification unique, et la variable path_index est une variable en lecture seule qui contient la valeur de l'index d'une ressource de chemin donnée, affectée à une instance à l'aide de la fonction . path_index est une variable en lecture seule qui contient la valeur de l'index pour une ressource path donnée qui a été affectée à une instance à l'aide de la fonction path_start(). Si aucun chemin n'a été attribué à l'instance, la variable prend la valeur -1. Syntaxe : path_index ; Les retours : integer (ID du chemin) Exemple : si (path_index == -1) { path_start(pth_enemy3, 4, path_action_reverse, 0) ; } Le code ci-dessus vérifie si un chemin a été assigné à l'instance, et si ce n'est pas le cas, il commence un nouveau chemin (en l'assignant au path_index en même temps).
