path_rescale Cette fonction peut être utilisée pour redimensionner le chemin donné le long de l'axe vertical (ou de l'axe horizontal), en déplaçant chacun des points du chemin vers une nouvelle position correspondant à cette échelle autour du centre du chemin. Cette fonction modifie l'actif réel du chemin, et affectera donc de façon permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, vous devriez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appeler cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire). Syntaxe : path_rescale(index, xscale, yscale) ; Arguments Description indice L'index du chemin à mettre à l'échelle. xscale Par quoi multiplier l'échelle horizontale actuelle. L'échelle par défaut est 1. yscale Par quoi multiplier l'échelle verticale actuelle. L'échelle par défaut est 1. Les retours : N/A Exemple : path_rescale(pth_AI, 3, 3) ; Cela redimensionnerait le chemin donné à trois fois sa taille d'origine.
