path_rescale Cette fonction peut être utilisée pour redimensionner le chemin donné le long de l'axe vertical et horizontal (ou de l'un ou l'autre), en déplaçant fondamentalement chacun des points de chemin vers une nouvelle position correspondant à cette échelle autour du centre du chemin. Cette fonction modifie l'actif du chemin réel, et affectera donc de manière permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, vous devez utiliser une fonction telle que path_duplicate() pour créer d'abord une copie du chemin, puis appelez cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire). Syntaxe:path_rescale(index, xscale, yscale); Argument Description index L'index du chemin à l'échelle. xscale Par quoi multiplier l'échelle horizontale actuelle. L'échelle par défaut est 1. yscale Par quoi multiplier l'échelle verticale actuelle. L'échelle par défaut est 1. Retour:N/A Exemple:path_rescale(pth_AI, 3, 3); Cela redimensionnerait le chemin donné à trois fois sa taille d'origine.
