chemin_get_longueur Vous pouvez utiliser cette fonction pour obtenir la longueur exacte d'un chemin en pixels. Il ne s'agit pas d'une longueur approximative d'un point à un autre, mais plutôt d'une longueur exacte le long de la forme du chemin, même lorsque le chemin est lisse avec une grande précision de courbure. Syntaxe : path_get_length(index) ; Arguments Description indice L'indice du chemin à mesurer. Les retours : Real Exemple : path_len = path_get_length(pth_AI) ; Ceci définit "path_len" comme la longueur totale du chemin indexé dans "pth_AI" en pixels.
