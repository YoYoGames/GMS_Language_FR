path_get_length Vous pouvez utiliser cette fonction pour obtenir la longueur exacte d'un chemin en pixels. ce n'est pas une longueur approximative d'un point à un autre, mais plutôt une longueur exacte le long de la forme du chemin, même lorsque le chemin est lisse avec une haute précision de courbure. Syntaxe:path_get_length(index); Argument Description index L'index du chemin à mesurer. Retour:Real Exemple:path_len = path_get_length(pth_AI); Cela définira "path_len" à la longueur totale du chemin indexé dans "pth_AI" en pixels.
