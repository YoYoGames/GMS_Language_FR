Variables de chemin Les variables suivantes sont intégrées à toutes les instances lors de leur création et peuvent être utilisées pour obtenir des informations sur un chemin, ou pour modifier certaines propriétés : chemin_index path_position path_positionprevious vitesse du chemin path_scale orientation_du_chemin path_endaction Notez que vous pouvez trouver de plus amples informations sur les variables d'instance ici : Variables d'instance.
