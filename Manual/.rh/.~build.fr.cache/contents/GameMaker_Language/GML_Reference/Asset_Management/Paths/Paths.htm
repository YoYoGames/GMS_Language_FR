Chemins d'accès GameMaker Studio 2 dispose d'une vaste collection de fonctions et de variables intégrées permettant d'utiliser les chemins dans votre jeu. Cette section traite des fonctions de base utilisées pour traiter les chemins qui ont été précédemment créés dans l'éditeur de chemins, Mais il existe d'autres fonctions qui vous permettent de créer vos propres chemins en utilisant du code et de les manipuler de diverses manières. Les fonctions de base suivantes traitent des chemins : Début du chemin chemin_fin path_get_closed path_get_kind chemin_get_longueur chemin_get_name chemin_get_numéro path_get_point_speed chemin_get_point_x chemin_get_point_y path_get_precision chemin_get_speed chemin_get_x chemin_get_y Pour faciliter le débogage des chemins, vous disposez également d'une fonction permettant de les dessiner à l'écran : dessiner_chemin Chaque instance d'un objet possède également une série de variables intégrées conçues pour faciliter l'utilisation des chemins. Ces variables sont énumérées dans la section suivante : Variables de chemin Enfin, il est possible de manipuler le contenu d'un chemin ainsi que de créer de nouveaux chemins lorsque le jeu est en cours.
