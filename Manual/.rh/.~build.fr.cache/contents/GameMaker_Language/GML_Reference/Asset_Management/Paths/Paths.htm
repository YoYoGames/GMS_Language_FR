Les chemins GameMaker Studio 2 possède une vaste collection de fonctions et de variables intégrées pour utiliser les chemins dans votre jeu. Cette section traite des chemins de base qui sont utilisés pour traiter les chemins qui ont été précédemment créés dans l' éditeur de chemin, mais il existe d'autres fonctions qui vous permettent de créer vos propres chemins à l'aide de code et de les manipuler de différentes manières. Il existe les fonctions de base suivantes qui traitent des chemins: path_startpath_endpath_get_closedpath_get_kindpath_get_lengthpath_get_namepath_get_numberpath_get_point_speedpath_get_point_xpath_get_point_ypath_get_precisionpath_get_speedpath_get_xpath_get_y Pour faciliter le débogage des chemins, vous avez également une fonction pour le dessiner à l'écran:draw_path Chaque instance d'un objet possède également une série de variables intégrées conçues pour faciliter l'utilisation des chemins, et ces variables sont répertoriées dans la section suivante: Variables de chemin Enfin, il est possible de manipuler le contenu d'un chemin ainsi que de créer de nouveaux chemins lorsque le jeu est en cours d'exécution.
