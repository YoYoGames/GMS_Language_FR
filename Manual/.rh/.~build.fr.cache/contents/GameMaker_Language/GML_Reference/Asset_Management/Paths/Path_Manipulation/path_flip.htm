path_flip Cette fonction prend tous les points de chemin et les retourne le long de l'axe horizontal. Cette fonction modifie l'actif du chemin réel, et affectera donc de manière permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu.Si ce n'est pas ce dont vous avez besoin, vous devez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appelez cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire). Syntaxe:path_flip(index); Argument Description index L'index du chemin à retourner. Retour:N/A Exemple:path_flip(mypath); Cela inverserait le chemin indexé dans la variable "mypath" le long de l'axe horizontal.
