asset_add_tags Avec cette fonction, vous pouvez ajouter une ou plusieurs chaînes de balises à n'importe quel actif à partir du navigateur d'actifs. Vous fournissez le nom de l'actif (sous forme de chaîne) ou son index d'actif, ainsi qu'une seule chaîne de balise ou un tableau où chaque élément est une seule chaîne de balise. Si vous fournissez une valeur d'index d'actif, vous devrez alors fournir l'argument de type d'actif facultatif (une constante), car les actifs de types différents peuvent avoir le même index, même s'ils ne peuvent pas avoir le même nom. Les types d'actifs disponibles sont répertoriés dans le tableau ci-dessous: Constant Description asset_object Le nom donné fait référence à un objet. asset_sprite Le nom donné fait référence à un sprite. asset_sound Le prénom fait référence à un son. asset_room Le prénom fait référence à unroom. asset_tiles Le nom donné fait référence à un jeu de tuiles. asset_path Le nom donné fait référence à un chemin. asset_script Le prénom fait référence à unscript. asset_font Le prénom fait référence à une police. asset_timeline Le prénom fait référence à une chronologie. asset_ shader Le prénom fait référence à un shader. asset_animationcurve Le nom donné fait référence à une courbe d'animation. asset_sequence Le prénom fait référence à une séquence. Si la fonction réussit à ajouter la ou les balises, elle retournera true sinon il reviendra false. Syntaxe:asset_add_tags(name_or_index, tags, [asset_type]); Argument Description name_or_index Le nom de l'actif (une chaîne) ou sa valeur d'index (un entier). tags Une seule chaîne d'étiquettes d'inventaire ou un tableau avec diverses étiquettes d'inventaire. [asset_type] OPTIONNEL! Type d'actif sur lequel définir les balises, utilisé uniquement lors de la fourniture d'un index pour le premier argument. Retour:Boolean Exemple:var _a = array_create(3); _a[0] = "enemy"; _a[1] = "all_levels"; _a[2] = "boss"; asset_add_tags(obj_Enemy_Boss_Parent, _a, asset_object); Le code ci-dessus créera un tableau de balises, puis les ajoutera à l'objet donné.
