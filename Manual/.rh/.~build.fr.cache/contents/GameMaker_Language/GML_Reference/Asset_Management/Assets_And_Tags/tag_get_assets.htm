tag_get_assets Cette fonction permet de récupérer les noms de tous les actifs auxquels ont été attribuées la ou les balises données. Vous fournissez soit une chaîne de tag unique, soit un tableau, où chaque élément du tableau est une chaîne de tag. La fonction renvoie un tableau dont chaque entrée est le nom (sous forme de chaîne) du bien ayant la balise donnée. Si vous avez besoin de l'index unique du poste, vous pouvez utiliser la fonction asset_get_index() avec avec le nom retourné. S'il n'y a pas de ressources avec la ou les balises données, ou s'il y a une erreur (par exemple, les balises données n'existent pas), un tableau vide sera retourné. Syntaxe : tag_get_assets(tags) ; Arguments Description tags Une seule chaîne de tags d'actifs ou un tableau avec plusieurs tags d'actifs. Les retours : Array Exemple : backgrounds = ds_list_create() ; var _assets = tag_get_assets("background") ; for (var i = 0 ; i< array_length(_assets) ; ++i ;) { if asset_get_type(_assets[i]) == asset_sprite { ds_list_add(backgrounds, asset_get_index(_assets[i])) ; } } Le code ci-dessus crée une liste, puis récupère les noms de toutes les ressources avec la balise "background". Il boucle à travers le tableau des noms retournés, en vérifiant si l'un d'entre eux est un actif sprite, et si c'est le cas, la valeur d'index unique de la ressource est ajoutée à la liste pour une utilisation ultérieure.
