balises_claires_de_l'actif Cette fonction permet d'effacer toutes les balises présentes sur un poste à partir du navigateur de postes. Vous fournissez soit le nom du poste (sous forme de chaîne), soit son index de poste. Si vous fournissez une valeur d'index de poste, vous devrez fournir l'argument facultatif de type de poste (une constante), car des postes de types différents peuvent avoir le même index, même s'ils ne peuvent pas avoir le même nom. (une constante), car des actifs de types différents peuvent avoir le même index, même s'ils ne peuvent pas avoir le même nom. Les types d'actifs disponibles sont répertoriés dans le tableau ci-dessous : Constant Description objet_actif Le nom donné fait référence à un objet. actif_sprite Le nom donné fait référence à un sprite. actif_son Le prénom fait référence à un son. salle des actifs Le prénom fait référence à une pièce. piliers_de_l'actif Le nom donné fait référence à un ensemble de carreaux. chemin_de_l'actif Le nom donné fait référence à un chemin. script_de_l'actif Le nom donné fait référence à un script. asset_font Le nom donné fait référence à une police de caractères. chronologie des actifs Le prénom fait référence à une ligne de temps. asset_shader Le nom donné fait référence à un shader. courbe d'animation de l'actif Le nom donné fait référence à une courbe d'animation. séquence_d'actifs Le nom donné fait référence à une Séquence. Si la fonction réussit à effacer les balises, elle renvoie true, sinon elle renvoie false. Syntaxe : asset_clear_tags(name, [asset_type]) ; Arguments Description nom_ou_index Le nom du poste (une chaîne de caractères) ou sa valeur d'index (un nombre entier). [type d'actif] FACULTATIF ! Le type de bien pour lequel il faut effacer les étiquettes, utilisé uniquement lorsqu'un index est fourni pour le premier argument. Les retours : Booléen Exemple : asset_clear_tags(obj_Enemy_Parent, asset_object) ; Le code ci-dessus effacera toutes les balises de l'objet donné.
