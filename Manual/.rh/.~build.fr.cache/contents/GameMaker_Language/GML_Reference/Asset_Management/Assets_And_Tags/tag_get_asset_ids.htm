tag_get_asset_ids Avec cette fonction, vous pouvez obtenir tous les actifs d'un type donné auxquels les balises données leur sont attribuées. Vous fournissez soit une seule balise (sous forme de chaîne), soit un tableau, où chaque élément du tableau est une balise (sous forme de chaîne), ainsi que le type d'actif à vérifier. le type d'actif doit être l'une des constantes suivantes: Constant Description asset_object Le nom donné fait référence à un objet. asset_sprite Le nom donné fait référence à un sprite. asset_sound Le prénom fait référence à un son. asset_room Le prénom fait référence à unroom. asset_tiles Le nom donné fait référence à un jeu de tuiles. asset_path Le nom donné fait référence à un chemin. asset_script Le prénom fait référence à unscript. asset_font Le prénom fait référence à une police. asset_timeline Le prénom fait référence à une chronologie. asset_ shader Le prénom fait référence à un shader. asset_animationcurve Le nom donné fait référence à une courbe d'animation. asset_sequence Le prénom fait référence à une séquence. La fonction renverra un tableau, où chaque élément du tableau sera une valeur d'index d'actif unique. S'il n'y a pas d'actifs du type qui ont la ou les balises données, un tableau vide sera renvoyé. Syntaxe:tag_get_asset_ids(tags, asset_type); Argument Description tags Une seule chaîne d'étiquettes d'inventaire ou un tableau avec diverses étiquettes d'inventaire. asset_type Une constante de type d'actif (répertoriée ci-dessus) Retour:Array Exemple:var _paths = tag_get_asset_ids("enemy", asset_path); var _num = irandom(array_length(_paths) - 1); path_start(_paths[_num], 1, path_action_reverse, false); Le code ci-dessus utilise la balise «ennemi» pour trouver tous les éléments de chemin avec cette balise, avant d'en choisir un au hasard et de l'attribuer à l'instance exécutant le code.
