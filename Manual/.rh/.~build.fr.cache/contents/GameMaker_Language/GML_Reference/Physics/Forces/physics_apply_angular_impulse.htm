physics_apply_angular_impulse Cette fonction donnera une impulsion angulaire à une instance activée par la physique. Cette impulsion définira la rotation angulaire de la quantité donnée, ignorant le couple actuel, définissant essentiellement la quantité de «rotation» d'un appareil. Si vous souhaitez appliquer une force angulaire à une instance en utilisant le couple, vous devez utiliser la fonction physics_apply_torque(). Syntaxe:physics_apply_angular_impulse(impulse) Argument Description impulse L'impulsion (en Newton mètres) à appliquer Retour:N/A Exemple:if keyboard_check(vk_up)     {     physics_apply_angular_impulse(10);     } Le code ci-dessus donnera au luminaire activé par la physique une impulsion angulaire si une touche est enfoncée.
