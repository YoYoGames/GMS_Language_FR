physics_particle_set_group_flags Avec cette fonction, vous pouvez modifier les indicateurs de groupe pour un groupe de particules. Le group valeur est celle qui a été renvoyée lors de la création du groupe de particules à l'aide de la fonction physics_particle_group_end(), tandis que les indicateurs sont la valeur de retour d'une combinaison des constantes suivantes: Constant Description phy_particle_group_flag_solid Un groupe de particules solides empêche les autres appareils de s'y loger. Si quelque chose y pénètre, le groupe de particules solides repousse l'appareil défectueux vers sa surface, ce qui fait qu'un groupe de particules solides possède une force de répulsion particulièrement forte. phy_particle_group_flag_rigid Les groupes de particules rigides sont ceux dont la forme ne change pas, même lorsqu'ils entrent en collision avec d'autres appareils. Ces indicateurs de groupe utilisent le masquage de bits pour créer une valeur de sortie finale qui est ensuite vérifiée pour définir les différentes propriétés de base du groupe, en dehors de celles définies par les indicateurs de particule. Vous pouvez utiliser aucune, une ou les deux de ces constantes en utilisant le bit au niveau du bit ou "|" pour masquer les bits appropriés (comme indiqué dans l'exemple ci-dessous). Syntaxe:physics_particle_set_group_flags(group, flags) Argument Description group Le groupe de particules à définir. flags Les drapeaux à placer sur la particule. Retour:N/A Exemple:var flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid; physics_particle_set_group_flags(group, flags); Le code ci-dessus créera une variable pour stocker la valeur des indicateurs, puis l'utilisera pour définir les indicateurs de toutes les particules créées dans le groupe indexé par la variable «groupe».
