physics_particle_set_category_flags Cette fonction vous permet de modifier les indicateurs de particules pour une catégorie de particules. La valeur de la catégorie est celle que vous avez définie dans la fonction physics_particle_create() ou (si vous avez créé un groupe) physics_particle_group_begin()tandis que les indicateurs sont la valeur de retour d'une combinaison des constantes suivantes : Constant Description phy_particle_flag_water Les propriétés par défaut d'une particule de corps mou. phy_particle_flag_zombie Une particule zombie est une particule qui sera détruite en une seule étape, toutes les autres étant marquées de cette manière. phy_particle_flag_wall Cela définit la particule comme statique, la créant essentiellement comme un objet inamovible dans la simulation physique, car elle restera dans une position fixe, peu importe ce qui entre en collision avec elle. Vous devriez utiliser cet indicateur plutôt que de définir la densité à 0. phy_particle_flag_spring Les particules à ressort produisent l'effet d'être attachées les unes aux autres, comme par un ressort. Les particules créées avec ce drapeau sont "connectées" par paires, chaque particule étant connectée à celle qui était la plus proche d'elle au moment de la création. Une fois appariées, les particules ne changent pas de "partenaires", et plus une force extérieure les éloigne les unes des autres, plus la puissance avec laquelle elles entreront en collision lorsque cette force extérieure sera supprimée sera grande. Notez que peu importe la distance entre des particules appariées l'une de l'autre, la connexion entre elles ne se rompra pas. phy_particle_flag_elastic Les particules élastiques se déforment et peuvent également rebondir lorsqu'elles entrent en collision avec d'autres corps rigides dans la simulation physique. phy_particle_flag_viscous Une particule visqueuse est une particule qui présente un caractère "collant", comme l'huile. Les particules visqueuses s'agglutinent et collent davantage les unes aux autres. phy_particle_flag_powder (poudre) Les particules de poudre produisent un effet de dispersion tel que celui que l'on peut observer avec le sable ou la poussière. phy_particle_flag_tensile Les particules de tension sont utilisées pour produire l'effet de la tension de surface, ou la courbure tendue à la surface d'un corps liquide. Elles peuvent être utilisées, par exemple, pour créer la tension de surface que l'on observe sur une goutte d'eau. Par conséquent, les particules ont tendance à former des amas lorsqu'elles rebondissent. phy_particule_flag_colourmixing Les particules dont la couleur est mélangée prennent une partie de la couleur des autres particules avec lesquelles elles entrent en collision. Notez que si une seule des deux particules qui entrent en collision est une particule à couleur mixte, l'autre particule conserve sa couleur d'avant la collision. Ces indicateurs utilisent le masquage de bits pour créer une valeur de sortie finale qui est ensuite vérifiée pour définir les différentes propriétés de base de la catégorie de particules (la propriété de base étant toujours celle de phy_particle_flag_water). Par exemple, si vous voulez simuler un liquide visqueux avec une tension de surface, vous utiliserez le bitwise ou "|" pour masquer les bits appropriés comme indiqué dans l'exemple ci-dessous. De cette façon, vous pouvez définir différentes propriétés (autres que les propriétés globales) pour chaque catégorie de particule créée. Syntaxe : physics_particle_set_category_flags(category, flags) Arguments Description catégorie La catégorie de particules à définir. drapeaux Les drapeaux à définir sur la particule. Les retours : N/A Exemple : var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile ; physics_particle_set_category_flags(1, flags) ; Le code ci-dessus crée une variable pour stocker la valeur des drapeaux, puis l'utilise pour définir les drapeaux de toutes les particules créées avec la valeur de catégorie 1.
