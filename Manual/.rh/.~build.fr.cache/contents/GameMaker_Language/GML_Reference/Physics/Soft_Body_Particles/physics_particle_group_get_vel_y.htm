groupe_de_particules_physique_get_vel_y Cette fonction permet de récupérer la vitesse verticale d'un groupe de particules. La valeur du groupe est celle qui a été renvoyée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renvoie une valeur qui est la valeur combinée des drapeaux actuellement définis. Syntaxe : physics_particle_group_get_vel_y(group) Arguments Description groupe Le groupe de particules à obtenir. Les retours : Real Exemple : vx = physics_particle_group_get_vel_x(group1) ; vy = physics_particle_group_get_vel_y(group1) ; Le code ci-dessus va obtenir la vitesse horizontale et verticale du groupe de particules indexé dans la variable "group1" et les stocker dans des variables.
