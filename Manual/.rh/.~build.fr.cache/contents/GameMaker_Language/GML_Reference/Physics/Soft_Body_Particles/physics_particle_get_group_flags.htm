physics_particle_get_group_flags Cette fonction permet de récupérer les indicateurs de groupe pour un groupe de particules. La valeur du groupe est celle qui a été renvoyée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renvoie une valeur qui est la valeur combinée des drapeaux actuellement définis. Syntaxe : physics_particle_get_group_flags(groupe) Arguments Description groupe Le groupe de particules à obtenir. Les retours : Real Exemple : var flags = physics_particle_get_group_flags(group, flags) ; if flags != phy_particle_group_flag_solid | phy_particle_group_flag_rigid { flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid ; physics_particle_set_group_flags(group, flags) ; } Le code ci-dessus crée une variable pour stocker la valeur de flags et l'utilise ensuite pour vérifier les flags du groupe indexé dans la variable "group". S'ils ne sont pas les mêmes, le groupe est défini avec ces drapeaux.
