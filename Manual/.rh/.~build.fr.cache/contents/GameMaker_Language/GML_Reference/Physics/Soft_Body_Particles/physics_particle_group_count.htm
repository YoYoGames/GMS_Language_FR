physics_particle_group_count Cette fonction renverra le nombre de particules actives dans un seul groupe. L'index de groupe (ID) est la valeur renvoyée lorsque vous appelez la fonction physics_particle_group_end(). Syntaxe:physics_particle_group_count(group) Argument Description group L'indice de groupe (ID) du groupe de particules à compter. Retour:Real Exemple:gp_num = physics_particle_group_count(group1); Le code ci-dessus obtiendra le nombre de particules utilisées pour rendre le groupe indexé dans la variable "group1" et stocker la valeur dans une variable.
