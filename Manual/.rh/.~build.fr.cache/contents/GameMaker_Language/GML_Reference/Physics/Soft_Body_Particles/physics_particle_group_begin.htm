physics_particle_group_begin Avec cette fonction, vous pouvez créer un groupe de particules dans une pièce. Un groupe de particules est un groupe dans lequel les particules nécessaires pour créer une forme spécifique sont créées en même temps en tant que groupe, ce qui vous permet de créer des corps mous simulés avec diverses propriétés définies par les indicateurs utilisés. Les "flags" sont la valeur de retour d'une combinaison des constantes de particules (voir la fonction physics_particle_create() pour plus de détails), tandis que les "groupFlags" sont une combinaison des constantes de groupe suivantes (ou 0 pour aucun indicateur): Constant Description phy_particle_group_flag_solid Un groupe de particules solides empêche les autres appareils de s'y loger. Si quelque chose y pénètre, le groupe de particules solides repousse l'appareil défectueux vers sa surface, ce qui fait qu'un groupe de particules solides possède une force de répulsion particulièrement forte. phy_particle_group_flag_rigid Les groupes de particules rigides sont ceux dont la forme ne change pas, même lorsqu'ils entrent en collision avec d'autres appareils. Ces indicateurs de groupe utilisent le masquage de bits pour créer une valeur de sortie finale qui est ensuite vérifiée pour définir les différentes propriétés de base du groupe, en dehors de celles définies par les indicateurs de particule. Vous pouvez utiliser aucune, une ou les deux de ces constantes en utilisant le bit au niveau du bit ou "|" pour masquer les bits appropriés (comme indiqué dans l'exemple ci-dessous). Outre les drapeaux, vous pouvez également définir la position dans la pièce pour créer le groupe de particules (la position donnée sera le centre de masse du groupe fini), la vitesse horizontale et verticale initiale, ainsi que la vitesse angulaire de le groupe, vous permettant de créer des groupes mobiles et rotatifs. Vous pouvez également définir la couleur initiale et l'alpha de toutes les particules du groupe, les valeurs étant utilisées lors du dessin des particules à l'aide de la fonction physics_particle_draw(). Après cela, vous définissez la force de cohésion des particules dans le groupe, ce qui est une autre façon de dire à quel point elles "collent" les unes aux autres. La force peut être une valeur comprise entre 0 et 1, où 0 signifie aucune cohésion et 1 est une cohésion maximale. Enfin, vous devez donner la catégorie de votre groupe de particules. Cette valeur est une valeur entière arbitraire que vous donnez à chaque groupe de particules dont vous souhaitez avoir des propriétés similaires, et elle peut être utilisée pour définir des indicateurs ultérieurement pour cette catégorie spécifique ainsi que pour un certain nombre d'autres choses. REMARQUE: vous pouvez utiliser n'importe quelle valeur entière pour la catégorie sauf 0 qui est réservé par la simulation et est utilisé pour sélectionner toutes les catégories dans d'autres fonctions (comme pour le dessin). Il est important de noter que l'appel de cette fonction ne crée pas le groupe dans la salle de physique. Cette fonction démarre simplement le processus et définit les propriétés des particules de base pour le groupe. Pour créer réellement le groupe de particules, vous devez lui donner une forme (voir l'exemple ci-dessous) puis appeler la fonction physics_particle_group_end(), à quel point il sera créé. REMARQUE: Les indicateurs de type de particule que vous définissez influenceront également la cohésion du groupe de particules, de sorte que si vous marquez les particules comme étant du type d'eau (par exemple), lorsque le groupe entre en collision avec un autre appareil, il se séparera, tandis que si vous marquez la particule comme un type de ressort, elle conservera sa cohésion en cas de collision. Syntaxe:physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category) Argument Description flags Les drapeaux à placer sur la particule. groupflags Les indicateurs de groupe à définir sur la particule (voir Description ci-dessous). x Position x à laquelle créer le groupe de particules. y La position y à laquelle créer le groupe de particules. ang Angle (en degrés) vers lequel faire pivoter le groupe lors de la création. xv La vitesse horizontale initiale. yv La vitesse verticale initiale. ang_velocity La vitesse angulaire initiale. col Couleur de base à utiliser pour le groupe de particules. alpha Alpha de base à utiliser pour le groupe de particules. strength Définissez la force de cohésion entre les particules du groupe. category La catégorie définie par l'utilisateur à laquelle appartient le groupe. Retour:N/A Exemple:var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid; physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2); physics_particle_group_circle(100); mLastGroup = physics_particle_group_end(); Le code ci-dessus stocke les indicateurs pour le type de particule et les propriétés du groupe de particules dans des variables, puis les utilise pour créer un groupe de particules circulaire avec un rayon de 100 px à la position de la souris.
