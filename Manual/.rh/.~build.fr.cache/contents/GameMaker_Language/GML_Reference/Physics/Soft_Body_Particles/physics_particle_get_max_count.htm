physics_particle_get_max_count Avec cette fonction, vous pouvez savoir quelle est la valeur plafond actuelle des particules autorisées dans la simulation physique (vous pouvez définir cette valeur à l'aide de la fonction physics_particle_set_max_count()). Syntaxe : physics_particle_get_max_count() Les retours : Real Exemple : if (physics_particle_count() < physics_particle_get_max_count()) { physics_particle_create(0, x, y, 0, 0, c_white, 1, 1) } Le code ci-dessus vérifiera s'il y a moins que le nombre maximum de particules autorisées dans la pièce, et si c'est le cas, il en créera une de plus.
