physics_particle_get_max_count Avec cette fonction, vous pouvez connaître la valeur actuelle du plafond sur les particules autorisées dans la simulation physique (vous pouvez définir cette valeur en utilisant physics_particle_set_max_count() ). Syntaxe:physics_particle_get_max_count() Retour:Real Exemple:if (physics_particle_count() < physics_particle_get_max_count())     {     physics_particle_create(0, x, y, 0, 0, c_white, 1, 1)     } Le code ci-dessus vérifiera s'il y a moins que le nombre maximum de particules autorisées dans la pièce, et si tel est le cas, en créera une de plus.
