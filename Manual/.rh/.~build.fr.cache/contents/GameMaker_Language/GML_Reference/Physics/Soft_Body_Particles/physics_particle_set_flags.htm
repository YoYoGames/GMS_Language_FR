physics_particle_set_flags Avec cette fonction, vous pouvez modifier les indicateurs de particule pour une particule individuelle. Le index valeur est l'ID de particule renvoyé par la fonction physics_particle_create(), tandis que les indicateurs sont la valeur de retour d'une combinaison des constantes suivantes: Constant Description phy_particle_flag_water Les propriétés par défaut d'une particule de corps mou. phy_particle_flag_zombie Une particule de zombie est une particule qui sera détruite en une seule étape avec toutes les autres marquées de cette manière. phy_particle_flag_wall Cela définit la particule comme statique, la créant essentiellement comme un objet immobile dans la simulation physique, car elles resteront dans une position fixe, peu importe ce qui les heurte. Vous devez utiliser cet indicateur plutôt que de définir la densité sur 0. phy_particle_flag_spring Les particules de ressort produisent l'effet d'être attachées les unes aux autres, comme par un ressort. Les particules créées avec ce drapeau sont "connectées" par paires, chaque particule étant connectée à celle qui en était la plus proche au moment de la création. Une fois appariées, les particules ne changent pas de «partenaires», et plus une force externe les tire les unes des autres, plus la puissance avec laquelle elles entreront en collision lorsque cette force externe est supprimée est grande. Notez que quelle que soit la distance entre les particules appariées, la connexion entre elles ne se cassera pas. phy_particle_flag_elastic Les particules élastiques se déforment et peuvent également rebondir lorsqu'elles entrent en collision avec d'autres corps rigides dans la simulation physique. phy_particle_flag_viscous Une particule visqueuse est une particule qui présente une «adhérence» ou une «adhérence», comme l'huile. Les particules visqueuses s'agglutineront et se colleront davantage. phy_particle_flag_powder Les particules de poudre produisent un effet de dispersion tel que vous pourriez le voir avec du sable ou de la poussière. phy_particle_flag_tensile Les particules de traction sont utilisées pour produire l'effet de la tension superficielle, ou la courbure tendue sur la surface d'un corps de liquide. Ils peuvent être utilisés, par exemple, pour créer la tension superficielle que vous verriez sur une goutte d'eau. Une fois la tension rompue, les particules rebondissent comme si elles étaient élastiques, mais continuent également à s'attirer les unes les autres. En conséquence, les particules ont tendance à former des grappes lorsqu'elles rebondissent. phy_particle_flag_colourmixing Les particules de mélange de couleurs prennent une partie de la couleur des autres particules avec lesquelles elles entrent en collision. Notez que si une seule des deux particules en collision est une particule mélangeant des couleurs, l'autre particule conserve sa couleur d'avant la collision. Ces indicateurs utilisent le masquage de bits pour créer une valeur de sortie finale qui est ensuite vérifiée pour définir les différentes propriétés de base de la particule (la propriété de base étant toujours celle de phy_particle_flag_water ). Par exemple, si vous souhaitez simuler un liquide visqueux avec une tension superficielle, vous utiliserez le bit au niveau du bit ou "|" pour masquer les bits appropriés comme indiqué dans l'exemple ci-dessous. De cette façon, vous pouvez définir différentes propriétés (autres que les propriétés globales) pour chaque particule individuelle créée, si vous le souhaitez. Syntaxe:physics_particle_set_flags(index, flags) Argument Description index L'indice de la particule. flags Les drapeaux à placer sur la particule. Retour:Real Exemple:var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; physics_particle_set_flags(p, flags); Le code ci-dessus créera une variable pour stocker la valeur des indicateurs, puis l'utilisera pour définir les indicateurs sur une particule créée précédemment avec l'index stocké dans la variable «p».
