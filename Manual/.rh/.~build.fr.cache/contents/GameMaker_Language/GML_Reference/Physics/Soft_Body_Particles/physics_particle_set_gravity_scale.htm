physic_particule_set_gravité_échelle Cette fonction vous permet de définir le facteur d'échelle de gravité pour les particules dans la simulation physique. La fonction est conçue pour aider à prévenir l'instabilité dans la simulation physique, en particulier lors de l'utilisation de très petites particules qui peuvent se comporter de manière imprévisible (c'est-à-dire briser la conservation de la quantité de mouvement) dans des scénarios tels que les explosions. (c'est-à-dire briser la conservation de la quantité de mouvement) dans des scénarios tels que les explosions. Le ralentissement de ces particules en réduisant l'échelle de gravité peut stabiliser leur comportement. Cette fonction a une portée globale, en ce sens qu'elle modifiera l'échelle de gravité non seulement pour les nouvelles particules créées après le changement, mais aussi pour celles déjà présentes dans la simulation. NOTE: L'ajustement du nombre d'itérations de mise à jour par étape (en utilisant la fonction physics_world_update_iterations() peut également affecter l'effet de la gravité sur les particules. Des itérations plus grandes confèrent une plus grande résistance à la gravité. Syntaxe : physics_particle_set_gravity_scale(scale) Arguments Description échelle Le facteur d'échelle à appliquer à la gravité pour toutes les particules. Les retours : N/A Exemple : physics_particle_set_radius(15) ; physics_particle_set_density(0.5) ; physics_particle_set_damping(1) ; physics_particle_set_gravity_scale(1) ; Le code ci-dessus définira les propriétés de base pour toutes les particules de la simulation.
