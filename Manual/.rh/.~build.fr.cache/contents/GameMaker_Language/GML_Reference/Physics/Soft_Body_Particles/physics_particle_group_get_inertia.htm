groupe_de_particules_physique_get_inertie Avec cette fonction, vous pouvez récupérer l'inertie d'un groupe entier de particules. La valeur du groupe est celle qui a été retournée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renvoie une valeur qui est la valeur combinée des drapeaux actuellement définis. Syntaxe : physics_particle_group_get_inertia(groupe) Arguments Description groupe Le groupe de particules à obtenir. Les retours : Real Exemple : i = physics_particle_group_get_inertia(group1) ; Le code ci-dessus va récupérer l'inertie du groupe de particules indexé dans la variable "group1" et la stocker dans une variable.
