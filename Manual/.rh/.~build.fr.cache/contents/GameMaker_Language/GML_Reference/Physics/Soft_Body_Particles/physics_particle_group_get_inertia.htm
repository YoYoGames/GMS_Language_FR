physics_particle_group_get_inertia Avec cette fonction, vous pouvez récupérer l'inertie d'un groupe entier de particules. Le group valeur est celle qui a été renvoyée lors de la création du groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renverra une valeur qui est la valeur combinée des indicateurs actuellement définis. Syntaxe:physics_particle_group_get_inertia(group) Argument Description group Le groupe de particules à obtenir. Retour:Real Exemple:i = physics_particle_group_get_inertia(group1); Le code ci-dessus récupérera l'inertie du groupe de particules indexé dans la variable "group1" et le stockera dans une variable.
