physics_particle_group_get_angle Avec cette fonction, vous pouvez récupérer la rotation (angle) dans la pièce d'un groupe de particules. Le group valeur est celle qui a été renvoyée lors de la création du groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renverra une valeur qui est la valeur combinée des indicateurs actuellement définis. Syntaxe:physics_particle_group_get_angle(group) Argument Description group Le groupe de particules à obtenir. Retour:Real Exemple:ang = physics_particle_group_get_angle(group1); Le code ci-dessus obtiendra l'angle du groupe de particules indexé dans la variable "group1" et le stockera dans une variable.
