physics_particle_group_get_angle Avec cette fonction, vous pouvez récupérer la rotation (angle) dans la pièce d'un groupe de particules. La valeur du groupe est celle qui a été retournée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renvoie une valeur qui est la valeur combinée des drapeaux actuellement définis. Syntaxe : physics_particle_group_get_angle(group) Arguments Description groupe Le groupe de particules à obtenir. Les retours : Real Exemple : ang = physics_particle_group_get_angle(group1) ; Le code ci-dessus va obtenir l'angle du groupe de particules indexé dans la variable "group1" et le stocker dans une variable.
