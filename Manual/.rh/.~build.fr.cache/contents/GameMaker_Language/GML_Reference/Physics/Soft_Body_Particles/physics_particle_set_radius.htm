physics_particle_set_radius Avec cette fonction, vous pouvez définir le rayon (en pixels) des particules dans une simulation physique. Cette fonction a une portée globale, en ce sens qu'elle changera le rayon non seulement pour les nouvelles particules créées après le changement, mais aussi pour celles déjà présentes dans la simulation. Syntaxe:physics_particle_set_radius(radius) Argument Description radius Le rayon (en pixels) de la fixation des particules. Retour:N/A Exemple:physics_particle_set_radius(15); physics_particle_set_density(0.5); physics_particle_set_damping(1); physics_particle_set_gravity_scale(1); Le code ci-dessus définira les propriétés de base de toutes les particules de la simulation.
