physics_particle_set_radius Cette fonction vous permet de définir le rayon (en pixels) des particules dans une simulation physique. Cette fonction a une portée globale, en ce sens qu'elle modifie le rayon non seulement pour les nouvelles particules créées après la modification, mais aussi pour celles déjà présentes dans la simulation. déjà présentes dans la simulation. Syntaxe : physics_particle_set_radius(radius) Arguments Description rayon Le rayon (en pixels) de la fixation de la particule. Les retours : N/A Exemple : physics_particle_set_radius(15) ; physics_particle_set_density(0.5) ; physics_particle_set_damping(1) ; physics_particle_set_gravity_scale(1) ; Le code ci-dessus définira les propriétés de base pour toutes les particules de la simulation.
