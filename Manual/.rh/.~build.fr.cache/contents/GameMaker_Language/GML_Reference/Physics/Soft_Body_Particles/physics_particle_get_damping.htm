physics_particle_get_damping Avec cette fonction, vous pouvez savoir quel est l'amortissement linéaire actuel pour les particules dans la simulation physique (vous pouvez définir cette valeur en utilisant physics_particle_set_damping() ). Syntaxe:physics_particle_get_damping() Retour:Real Exemple:if (physics_particle_get_damping() < 1)     {     physics_particle_set_damping(physics_particle_get_damping() + 0.01);     } Le code ci-dessus vérifiera la valeur d'amortissement actuelle pour toutes les particules du système et si elle est inférieure à 1, il y ajoutera 0,01.
