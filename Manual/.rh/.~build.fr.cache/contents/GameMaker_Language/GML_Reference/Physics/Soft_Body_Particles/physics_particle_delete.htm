physics_particle_delete Avec cette fonction, vous pouvez supprimer (supprimer) une particule de la simulation physique dans la pièce actuelle. La fonction prend l'identifiant unique de la particule à supprimer, tel que renvoyé par la fonction physics_particle_create(). Syntaxe:physics_particle_group_delete(ind) Argument Description ind L'index (ID) de la particule à supprimer. Retour:N/A Exemple:physics_particle_delete(part); Le code ci-dessus supprimera la particule avec l'ID stocké dans la variable «partie» de la simulation.
