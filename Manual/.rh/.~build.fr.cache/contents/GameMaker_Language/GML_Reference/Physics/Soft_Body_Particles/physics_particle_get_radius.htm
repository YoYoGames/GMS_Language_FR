physics_particle_get_radius Cette fonction vous permet de connaître le rayon actuel (en pixels) des particules dans la simulation physique (vous pouvez définir cette valeur à l'aide de la fonction physics_particle_set_radius()). Syntaxe : physics_particle_get_radius() Les retours : Real Exemple : if (physics_particle_get_radius() < 32) { physics_particle_set_radius(physics_particle_get_radius() + 1) ; } Le code ci-dessus va vérifier le rayon actuel des particules dans la simulation et s'il est inférieur à 32, il va augmenter leur taille de 1.
