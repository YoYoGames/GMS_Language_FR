physics_particle_get_radius Avec cette fonction, vous pouvez savoir quel est le rayon actuel (en pixels) des particules dans la simulation physique (vous pouvez définir cette valeur en utilisant physics_particle_set_radius() ). Syntaxe:physics_particle_get_radius() Retour:Real Exemple:if (physics_particle_get_radius() < 32)     {     physics_particle_set_radius(physics_particle_get_radius() + 1);     } Le code ci-dessus vérifiera le rayon actuel des particules dans la simulation et s'il est inférieur à 32, il augmentera leur taille de 1.
