physics_particle_group_get_ang_vel Cette fonction permet de récupérer la vitesse angulaire d'un groupe de particules. La valeur du groupe est celle qui a été renvoyée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renvoie une valeur qui est la valeur combinée des drapeaux actuellement définis. Syntaxe : physics_particle_group_get_ang_vel(group) Arguments Description groupe Le groupe de particules à obtenir. Les retours : Real Exemple : ang_v = physics_particle_group_get_ang_vel(group1) ; Le code ci-dessus va obtenir la vitesse angulaire du groupe de particules indexé dans la variable "group1" et la stocker dans une variable.
