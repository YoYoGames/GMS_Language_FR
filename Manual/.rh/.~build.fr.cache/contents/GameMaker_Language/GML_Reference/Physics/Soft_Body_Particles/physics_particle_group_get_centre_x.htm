physics_particle_group_get_centre_x Avec cette fonction, vous pouvez récupérer la composante x du centre de masse d'un groupe entier de particules. Le group valeur est celle qui a été renvoyée lors de la création du groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renverra une valeur qui est la valeur combinée des indicateurs actuellement définis. Syntaxe:physics_particle_group_get_centre_x(group) Argument Description group Le groupe de particules à obtenir. Retour:Real Exemple:xx = physics_particle_group_get_centre_x(group1); yy = physics_particle_group_get_centre_y(group1); Le code ci-dessus obtiendra la position x et y pour le centre de masse du groupe de particules indexé dans la variable "group1" et les stockera dans des variables.
