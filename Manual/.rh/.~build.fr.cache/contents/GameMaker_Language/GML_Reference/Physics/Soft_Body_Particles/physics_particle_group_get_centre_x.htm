groupe_de_particules_physique_get_centre_x Avec cette fonction, vous pouvez récupérer la composante x du centre de masse d'un groupe entier de particules. La valeur du groupe est celle qui a été renvoyée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renvoie une valeur qui est la valeur combinée des indicateurs actuellement définis. Syntaxe : physics_particle_group_get_centre_x(group) Arguments Description groupe Le groupe de particules à obtenir. Les retours : Real Exemple : xx = physics_particle_group_get_centre_x(group1) ; yy = physics_particle_group_get_centre_y(group1) ; Le code ci-dessus va obtenir la position x et y du centre de masse du groupe de particules indexé dans la variable "group1" et les stocker dans des variables.
