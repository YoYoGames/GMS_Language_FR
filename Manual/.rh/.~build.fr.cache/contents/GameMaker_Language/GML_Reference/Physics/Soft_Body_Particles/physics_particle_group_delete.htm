physics_particle_group_delete Avec cette fonction, vous pouvez supprimer (supprimer) un groupe de particules de la simulation physique dans la pièce actuelle. La fonction prend l'ID de groupe unique du groupe à supprimer, tel que renvoyé par la fonction physics_particle_group_end(). Syntaxe:physics_particle_group_delete(ind) Argument Description ind L'index (ID) du groupe de particules à supprimer. Retour:N/A Exemple:physics_particle_group_delete(gp1); Le code ci-dessus supprimera toutes les particules qui composent le groupe avec l'ID stocké dans la variable "gp1" de la simulation.
