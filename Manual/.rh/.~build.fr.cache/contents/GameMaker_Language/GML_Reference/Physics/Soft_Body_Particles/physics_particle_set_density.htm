physics_particle_set_density Cette fonction vous permet de définir la densité des particules dans une simulation physique. La définition de la densité de la particule aura un impact direct sur son inertie et sur sa réaction aux collisions. une densité élevée, elle aura une masse très importante, mais si vous définissez une grande particule avec une faible densité, elle aura une masse beaucoup plus faible. Cette fonction a une portée globale, en ce sens qu'elle modifie la densité non seulement pour les nouvelles particules créées après la modification, mais aussi pour celles déjà existantes. la modification, mais aussi pour celles déjà présentes dans la simulation. Syntaxe : physics_particle_set_density(density) Arguments Description densité La densité de la fixation de la particule. Les retours : N/A Exemple : physics_particle_set_radius(15) ; physics_particle_set_density(0.5) ; physics_particle_set_damping(1) ; physics_particle_set_gravity_scale(1) ; Le code ci-dessus définira les propriétés de base pour toutes les particules de la simulation.
