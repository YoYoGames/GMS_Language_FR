physics_particle_set_density Avec cette fonction, vous pouvez définir la densité des particules dans une simulation physique. Le réglage de la densité de la particule aura un impact direct sur la quantité d'inertie dont elle dispose ainsi que sur la façon dont elle réagit aux collisions, donc si vous créez une petite particule avec une densité élevée, elle aura une très grande masse, mais si vous définissez un grosse particule de faible densité, elle aura une masse beaucoup plus petite. Cette fonction a une portée globale, en ce sens qu'elle modifiera la densité non seulement pour les nouvelles particules créées après le changement, mais aussi pour celles déjà présentes dans la simulation. Syntaxe:physics_particle_set_density(density) Argument Description density La densité de la fixation de particules. Retour:N/A Exemple:physics_particle_set_radius(15); physics_particle_set_density(0.5); physics_particle_set_damping(1); physics_particle_set_gravity_scale(1); Le code ci-dessus définira les propriétés de base de toutes les particules de la simulation.
