physics_particle_get_density Avec cette fonction, vous pouvez savoir quelle est la densité de courant pour les particules dans la simulation physique (vous pouvez définir cette valeur en utilisant physics_particle_set_density() ). Syntaxe:physics_particle_get_damping() Retour:Real Exemple:if (physics_particle_get_density() < 1)    {    physics_particle_set_density(physics_particle_get_density() + 0.01);    } Le code ci-dessus vérifiera la valeur de densité actuelle pour toutes les particules du système et si elle est inférieure à 1, il y ajoutera 0,01.
