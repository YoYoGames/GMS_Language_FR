physics_particle_group_box Cette fonction définira la forme du groupe de particules en cours de création. Vous devez d'abord avoir commencé la définition du groupe à l'aide de la fonction physics_particle_group_begin(), puis vous utiliseriez cette fonction pour définir la demi-largeur et la demi-hauteur approximatives du groupe en pixels - approximatives car la largeur et la hauteur exactes dépendront de la taille des particules de base, comme défini par le physics_particle_set_radius() fonction, car la simulation physique essaie «d'adapter» autant de particules que possible dans la forme définie. Enfin, vous devez appeler physics_particle_group_end() pour créer le groupe de particules dans la pièce. Syntaxe:physics_particle_group_box(halfWidth, halfHeight) Argument Description halfWidth La demi- largeur de la boîte. halfHeight La demi- hauteur de la boîte. Retour:N/A Exemple:var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; var groupflags = phy_particle_group_flag_solid; physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2); physics_particle_group_box(100, 100); mLastGroup = physics_particle_group_end(); Le code ci-dessus stocke les indicateurs pour le type de particule et les propriétés du groupe de particules dans des variables, puis les utilise pour créer un groupe de particules rectangulaire avec des côtés de 200 pixels à la position de la souris.
