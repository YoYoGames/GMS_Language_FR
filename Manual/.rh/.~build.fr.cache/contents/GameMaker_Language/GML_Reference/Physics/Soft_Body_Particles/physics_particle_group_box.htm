boîte_de_groupe_de_particules_physiques Cette fonction va définir la forme du groupe de particules en cours de création. Vous devez d'abord avoir commencé la définition du groupe en utilisant la fonction physics_particle_group_begin()et ensuite vous devez utiliser cette fonction pour définir la demi-largeur et la demi-hauteur approximatives du groupe en pixels - approximatives car la largeur et la hauteur exactes dépendront de la taille des particules de base, telles que définies par la fonction physics_particle_set_radius() car la simulation physique tente de faire entrer le plus grand nombre possible de particules dans la forme définie. Enfin, vous devez appeler physics_particle_group_end() pour créer le groupe de particules dans la pièce. Syntaxe : physics_particle_group_box(halfWidth, halfHeight) Arguments Description demi-largeur La demi-largeur de la boîte. mi-hauteur La demi-hauteur de la boîte. Les retours : N/A Exemple : var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile ; var groupflags = phy_particle_group_flag_solid ; physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2) ; physics_particle_group_box(100, 100) ; mLastGroup = physics_particle_group_end() ; Le code ci-dessus stocke les drapeaux pour le type de particule et les propriétés du groupe de particules dans des variables, puis les utilise pour créer un groupe de particules rectangulaires avec des côtés de 200px à la position de la souris.
