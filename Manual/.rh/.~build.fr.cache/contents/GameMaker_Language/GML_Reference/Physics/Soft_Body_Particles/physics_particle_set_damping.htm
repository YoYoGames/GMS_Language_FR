physics_particle_set_damping Avec cette fonction, vous pouvez définir l'amortissement linéaire des particules dans la simulation. L'amortissement est utilisé pour réduire la vitesse de simulation physique des instances au fil du temps, tout comme la résistance de l'air dans le monde réel. Cette fonction a une portée globale, en ce sens qu'elle modifiera l'amortissement non seulement pour les nouvelles particules créées après le changement, mais aussi pour celles déjà présentes dans la simulation. Syntaxe:physics_particle_set_damping(damping) Argument Description damping L'amortissement linéaire à appliquer à la fixation de particules. Retour:N/A Exemple:physics_particle_set_radius(15); physics_particle_set_density(0.5); physics_particle_set_damping(1); physics_particle_set_gravity_scale(1); Le code ci-dessus définira les propriétés de base de toutes les particules de la simulation.
