nombre_de_particules_physiques Cette fonction renvoie le nombre de particules qui sont actives dans une pièce activée par la physique. Syntaxe : physics_particle_count() Les retours : Real Exemple : if (physics_particle_count() < physics_particle_get_max_count()) { physics_particle_create(0, x, y, 0, 0, c_white, 1, 1) } Le code ci-dessus vérifiera s'il y a moins que le nombre maximum de particules autorisées dans la pièce, et si c'est le cas, il en créera une de plus.
