groupe_de_particules_physique_get_y Avec cette fonction, vous pouvez récupérer la position y dans la pièce d'un groupe de particules. La valeur du groupe est celle qui a été retournée lorsque vous avez créé le groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renvoie une valeur qui est la valeur combinée des drapeaux actuellement définis. Syntaxe : physics_particle_group_get_y(groupe) Arguments Description groupe Le groupe de particules à obtenir. Les retours : Real Exemple : xx = physics_particle_group_get_x(group1) ; yy = physics_particle_group_get_y(group1) ; Le code ci-dessus récupère les positions x et y du groupe de particules indexé dans la variable "group1" et les stocke dans des variables.
