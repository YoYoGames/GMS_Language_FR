physics_particle_group_get_y Avec cette fonction, vous pouvez récupérer la position y dans la pièce d'un groupe de particules. Le group valeur est celle qui a été renvoyée lors de la création du groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renverra une valeur qui est la valeur combinée des indicateurs actuellement définis. Syntaxe:physics_particle_group_get_y(group) Argument Description group Le groupe de particules à obtenir. Retour:Real Exemple:xx = physics_particle_group_get_x(group1); yy = physics_particle_group_get_y(group1); Le code ci-dessus obtiendra les positions x et y du groupe de particules indexées dans la variable "group1" et les stockera dans des variables.
