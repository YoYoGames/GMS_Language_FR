physics_particle_set_max_count Cette fonction définira le nombre total autorisé de particules dans une simulation physique. Si vous définissez cette valeur sur 0, alors il n'y a pas de plafond pour les particules créées, et le définir sur une autre valeur arrêtera la création de toute autre particule lorsque le nombre total de particules est égal à la valeur. Notez que le nombre minimum de particules que vous pouvez créer dans une simulation est de 128, donc la définition de cette valeur plus basse n'aura aucun effet (à moins qu'elle ne soit définie sur 0). Syntaxe:physics_particle_set_max_count(count) Argument Description count Le nombre maximum de particules à autoriser. Retour:N/A Exemple:physics_particle_set_max_count(500); Le code ci-dessus définira le plafond de particules de la simulation physique à 500.
