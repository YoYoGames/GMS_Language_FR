phy_position_y Cette variable peut être utilisée pour obtenir (ou définir) la position y de l'instance dans le monde physique de la salle de jeu. Veuillez noter que le monde physique peut présenter des erreurs lorsque les instances sont déplacées en réglant directement cette variable, car cela interrompra la simulation continue. simulation continue. Cette variable est l'équivalent physique de la variable d'instance y. Syntaxe : phy_position_y ; Les retours : Réel (valeur à virgule flottante de simple précision, ou indéfini si l'instance n'est pas activée pour la physique) Exemple : xx = phy_position_x ; yy = phy_position_y ; Le code ci-dessus stocke la position x et y de l'instance dans deux variables.
