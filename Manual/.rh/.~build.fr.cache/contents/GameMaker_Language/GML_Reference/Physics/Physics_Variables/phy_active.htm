phy_active Cette variable contrôle si l'instance est actuellement "active" ou non. En lui attribuant la valeur false, l'instance ne pourra pas participer au monde de la physique, et en lui attribuant la valeur true, elle participera à nouveau. Veuillez consulter noter que ce n'est pas la même chose que de désactiver l'instance, car l'instance est toujours visible à l'écran et peut toujours être modifiée par le code. cette fonction l'empêche simplement de participer à la simulation physique. Syntaxe : phy_active ; Les retours : Booléen (ou indéfini si l'instance n'est pas activée pour la physique) Exemple : if keyboard_check_pressed(ord "P") { global.Pause = !global.Pause with (obj_Parent) { phy_active = !global.Pause ; } } Le code ci-dessus détectera une pression sur la lettre "P" et fera basculer la variable globale "Pause" de true à false et inversement . Cette variable est ensuite utilisée pour définir si la physique est active ou non dans les instances enfant de l'objet indexé dans la variable "obj_Parent".
