phy_position_xprevious Cette variable peut être utilisée pour obtenir (ou définir) la position x précédente de l'instance dans le monde physique de la salle de jeu. Il s'agit de la position de l'instance dans le monde physique lors de l'étape précédente à l'étape actuelle. Syntaxe : phy_position_xprevious ; Les retours : Réel (valeur à virgule flottante de simple précision, ou indéfini si l'instance n'est pas activée pour la physique) Exemple : xx = phy_position_xprevious ; yy = phy_position_yprevious ; Le code ci-dessus stocke la position x et y précédente de l'instance activée par la physique dans deux variables.
