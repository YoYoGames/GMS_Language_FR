phy_linear_damping Cette variable peut être utilisée pour définir l'amortissement linéaire de l'instance, ou pour obtenir l'amortissement linéaire actuel. L'amortissement est la quantité de "résistance" au mouvement vers l'avant que possède l'instance activée par la physique. permettant à l'instance de se déplacer et d'accélérer plus rapidement et une valeur plus élevée nécessitant une poussée plus forte. Syntaxe : phy_linear_damping ; Les retours : Réel (valeur à virgule flottante de simple précision, ou indéfini si l'instance n'est pas activée pour la physique) Exemple : if place_meeting(phy_position_x, phy_position_y, obj_Water) { phy_linear_damping = 10 ; } else { phy_linear_damping = 3 ; } Le code ci-dessus vérifiera la présence d'une collision entre l'instance appelante et les instances de "obj_Water" et modifiera l'amortissement linéaire en conséquence.
