phy_angular_damping Cette variable peut être utilisée pour définir l'amortissement angulaire de l'instance, ou pour obtenir l'amortissement angulaire actuel. L'amortissement est la quantité de "résistance" à la rotation angulaire que possède l'instance activée par la physique. permettant à l'instance de tourner plus rapidement après une collision (par exemple) et une valeur plus élevée nécessitant une poussée plus forte et une rotation plus lente. Syntaxe : phy_angular_damping ; Les retours : Réel (valeur à virgule flottante de simple précision, ou indéfini si l'instance n'est pas activée pour la physique) Exemple : if place_meeting(phy_position_x, phy_position_y, obj_Water) { phy_angular_damping = 10 ; } else { phy_angular_damping = 3 ; } Le code ci-dessus vérifiera la présence d'une collision entre l'instance appelante et les instances de "obj_Water" et modifiera l'amortissement angulaire en conséquence.
