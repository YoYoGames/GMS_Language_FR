phy_fixed_rotation Cette variable peut être utilisée pour définir si l'instance peut être affectée ou non par des forces de rotation (la valeur par défaut est false). Si cette variable est définie sur true, aucune force externe (qu'elle provienne d'impulsions ou de forces codées, ou de collisions) n'affectera la valeur de rotation de l'instance. la valeur de rotation de l'instance et celle-ci devra être définie manuellement en utilisant la phy_rotation variable Syntaxe : phy_fixed_rotation ; Les retours : Booléen (ou indéfini si l'instance n'est pas activée pour la physique) Exemple : phy_rotation = 0 ; phy_fixed_rotation = true ; Le code ci-dessus fera en sorte que l'instance ait une rotation fixe, puis définira l'angle de rotation.
