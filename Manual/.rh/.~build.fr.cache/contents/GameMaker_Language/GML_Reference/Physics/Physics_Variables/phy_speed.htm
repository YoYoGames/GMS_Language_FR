phy_speed Cette variable en lecture seule renvoie la vitesse sous forme de valeur en pixels par seconde de l'instance activée pour la physique. Si vous devez modifier cette valeur, vous devez le faire en modifiant les vecteurs x et y à l'aide des variables phy_speed_x et phy_speed_y Syntaxe:phy_speed; Retour:Real (single precision floating point value, or undefined if the instance is not physics enabled) Exemple:if phy_speed > 10     {     phy_linear_damping += 0.01;     } else     {     phy_linear_damping = 2;     } Le code ci-dessus vérifie la vitesse de l'instance activée pour la physique, puis modifie l'amortissement linéaire en fonction de la valeur renvoyée.
