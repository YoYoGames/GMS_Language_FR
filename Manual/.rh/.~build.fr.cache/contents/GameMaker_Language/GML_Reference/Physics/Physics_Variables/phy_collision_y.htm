phy_collision_y Ce tableau en lecture seule renvoie la position y de tous les points détectés lors d'une collision entre deux instances activées pour la physique. REMARQUE: cette variable n'est disponible qu'en cas de collision d'une instance activée pour la physique. Syntaxe:phy_collision_y; Retour:Real (single precision floating point value, or undefined if the instance is not physics enabled) Exemple:for(var i = 0; i < phy_collision_points; i += 1;)     {     part_particles_create(global.Sname, phy_collision_x[i], phy_collision_y[i], global.Spark, 5);     } Le code ci-dessus crée des particules à tous les points définis d'une collision entre deux instances activées pour la physique.
