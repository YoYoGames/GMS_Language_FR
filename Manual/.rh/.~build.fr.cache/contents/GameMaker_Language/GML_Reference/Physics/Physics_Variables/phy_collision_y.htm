phy_collision_y Ce tableau en lecture seule renvoie la position y de tous les points détectés dans une collision entre deux instances de physique. REMARQUE: Cette variable n'est disponible que dans l'événement de collision d'une instance activée pour la physique. Syntaxe : phy_collision_y ; Les retours : Réel (valeur à virgule flottante de simple précision, ou indéfini si l'instance n'est pas activée pour la physique) Exemple : for(var i = 0 ; i < phy_collision_points ; i += 1 ;) { part_particles_create(global.Sname, phy_collision_x[i], phy_collision_y[i], global.Spark, 5) ; } Le code ci-dessus crée des particules à tous les points définis d'une collision entre deux instances activées par la physique.
