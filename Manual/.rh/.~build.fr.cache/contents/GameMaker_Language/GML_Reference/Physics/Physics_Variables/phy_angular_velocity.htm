phy_angular_velocity Cette variable peut être utilisée pour définir la vitesse angulaire de l'instance, ou elle peut être utilisée pour obtenir la vitesse angulaire actuelle, en degrés par seconde et la valeur utilisée peut être positive (pour une rotation dans le sens des aiguilles d'une montre) ou négative (pour une rotation dans le sens inverse des aiguilles d'une montre). Si vous définissez ceci sur une instance qui était auparavant statique (c'est-à-dire: elle a une densité de 0), elle deviendra un objet cinématique et commencera à tourner Syntaxe:phy_angular_velocity; Retour:Real (single precision floating point value, or undefined if the instance is not physics enabled) Exemple:if abs(phy_angular_velocity) > 0     {     phy_angular_velocity -= sign(phy_angular_velocity) * 0.01;     } else     {     phy_angular_velocity = 0;     } Le code ci-dessus vérifiera la vitesse angulaire de l'instance et si ce n'est pas 0, il ajoutera (ou soustraira) une petite quantité à chaque pas jusqu'à ce que la valeur soit 0.
