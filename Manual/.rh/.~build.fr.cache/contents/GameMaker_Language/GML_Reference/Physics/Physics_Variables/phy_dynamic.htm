phy_dynamique Une instance dynamique est une instance qui est entièrement simulée dans le monde de la physique. Cette variable en lecture seule renverra un message vrai si l'instance vérifiée est entièrement simulée ou faux si elle ne l'est pas. Syntaxe : phy_dynamique ; Les retours : Booléen (ou indéfini si l'instance n'est pas activée pour la physique) Exemple : if other.phy_dynamic { with (other) { var dir ; dir = point_direction(x, y, other.x, other.y) ; physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir)) ; } } Le code ci-dessus crée une nouvelle instance, puis la définit comme étant une "balle" dans le monde de la physique avant de lui donner une impulsion massive le long de l'axe y.
