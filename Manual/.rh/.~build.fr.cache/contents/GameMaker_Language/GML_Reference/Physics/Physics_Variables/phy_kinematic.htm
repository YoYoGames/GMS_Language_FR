phy_kinematic Cette variable en lecture seule retournera true si l'instance est classée comme étant un objet cinématique, ou false si elle ne l'est pas. Une instance cinématique est une instance qui a une masse infinie (une densité de 0) mais qui peut se déplacer. Ainsi, pour rendre une instance cinématique, vous devez d'abord créer une instance statique, puis définir une ou plusieurs variables d'instance liées au mouvement (par ex : phy_speed_x, phy_speed_y, ou phy_angular_velocity) Syntaxe : phy_kinematic ; Les retours : Booléen (ou indéfini si l'instance n'est pas activée pour la physique) Exemple : if !phy_kinematic { phy_speed_x = 5 ; } Le code ci-dessus vérifie si l'instance est cinématique et si elle ne l'est pas, il fixe la vitesse horizontale à 5.
