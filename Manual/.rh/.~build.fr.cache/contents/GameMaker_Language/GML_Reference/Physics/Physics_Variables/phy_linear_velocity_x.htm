phy_linear_velocity_x Cette variable peut être utilisée pour obtenir ou modifier la composante x du vecteur vitesse linéaire de l'instance et est définie en pixels par seconde (pour les pixels par pas, voir phy_speed_x). La modification de cette variable pour une instance statique (c'est-à-dire une instance avec une densité de 0) la transformera en une instance cinématique. Syntaxe : phy_linear_velocity_x ; Les retours : Réel (valeur à virgule flottante de simple précision, ou indéfini si l'instance n'est pas activée pour la physique) Exemple : if (phy_linear_velocity_x != 0) { phy_linear_velocity_x = 0 ; } Le code ci-dessus vérifiera la composante x du vecteur de vitesse linéaire et si elle est différente de 0, il la mettra à 0.
