phy_speed_y Cette variable peut être utilisée pour obtenir ou modifier la composante y du vecteur de vitesse linéaire des instances et est définie en pixels par pas (pour les pixels par seconde, voir phy_linear_velocity_y ). Modifier ceci pour une instance statique (c'est-à-dire: une instance avec une densité de 0) la transformera en une instance cinématique Syntaxe:phy_speed_y; Retour:Real (single precision floating point value, or undefined if the instance is not physics enabled) Exemple:if phy_speed_y != 0     {     phy_speed_y = 0;     } Le code ci-dessus vérifiera la composante y du vecteur de vitesse linéaire et s'il est autre que 0, il le mettra à 0.
