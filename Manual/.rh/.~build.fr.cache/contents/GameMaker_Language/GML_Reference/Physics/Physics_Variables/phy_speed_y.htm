phy_speed_y Cette variable peut être utilisée pour obtenir ou modifier la composante y du vecteur vitesse linéaire de l'instance et est définie en pixels par pas (pour les pixels par seconde, voir phy_linear_velocity_y). La modification de cette variable pour une instance statique (c'est-à-dire une instance avec une densité de 0) la transformera en une instance cinématique. Syntaxe : phy_speed_y ; Les retours : Réel (valeur à virgule flottante de simple précision, ou indéfini si l'instance n'est pas activée pour la physique) Exemple : if phy_speed_y != 0 { phy_speed_y = 0 ; } Le code ci-dessus vérifiera la composante y du vecteur vitesse linéaire et si elle est différente de 0, il la mettra à 0.
