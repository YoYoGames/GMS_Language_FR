phy_linear_velocity_y Cette variable peut être utilisée pour obtenir ou modifier la composante y du vecteur vitesse linéaire de l'instance et est définie en pixels par seconde (pour les pixels par pas, voir phy_speed_y). Modifier cette variable pour une instance statique (c'est-à-dire une instance avec une densité de 0) la transformera en une instance cinématique. Syntaxe : phy_linear_velocity_y ; Les retours : Réel (valeur à virgule flottante de simple précision, ou indéfini si l'instance n'est pas activée pour la physique) Exemple : if (phy_linear_velocity_y != 0) { phy_linear_velocity_y = 0 ; } Le code ci-dessus vérifiera la composante y du vecteur de vitesse linéaire et s'il est différent de 0, il le mettra à 0.
