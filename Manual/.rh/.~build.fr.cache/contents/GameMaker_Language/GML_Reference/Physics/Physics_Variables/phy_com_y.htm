phy_com_y Cette variable en lecture seule retournera la position y du centre de masse de l'instance. Cette position est calculée automatiquement en fonction de la densité, de l'inertie et de la masse de l'instance, telles que définies par les fonctions appropriées. Syntaxe : phy_com_y ; Les retours : Réel (valeur à virgule flottante de simple précision, ou indéfini si l'instance n'est pas activée pour la physique) Exemple : xx = phy_com_x ; yy = phy_com_y ; Le code ci-dessus définit deux variables pour la position x et y du centre de masse des instances.
