phy_position_x Cette variable peut être utilisée pour obtenir (ou pour définir) la position x de l'instance dans le monde physique de la salle de jeux. Veuillez noter que le monde de la physique peut présenter des erreurs lorsque les instances sont déplacées en définissant directement cette variable car cela interrompra la simulation continue. Cette variable est l'équivalent physique de la variable d'instancex. Syntaxe:phy_position_x; Retour:Real (single precision floating point value, or undefined if the instance is not physics enabled) Exemple:xx = phy_position_x; yy = phy_position_y; Le code ci-dessus stocke la position x et y de l'instance dans deux variables.
