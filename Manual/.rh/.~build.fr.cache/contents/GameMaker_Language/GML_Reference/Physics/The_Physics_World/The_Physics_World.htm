Le monde de la physique Avant que toute réaction physique ne puisse se produire dans GameMaker Studio 2, vous devez d'abord définir le monde physique dans lequel elle va se produire. Si vous ne le faites pas, GameMaker Studio 2 utilisera par défaut les fonctions de collision de base. GameMaker Studio 2 utilisera par défaut les fonctions de collision de base, et tout autre code ou paramètre lié à la physique sera ignoré.
