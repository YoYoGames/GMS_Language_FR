physics_joint_enable_motor Lorsque vous avez une articulation avec un moteur ( prismatique ou révolutionnaire ), vous voudrez peut-être pouvoir allumer ou éteindre le moteur en fonction des variables et des conditions du jeu. Pour cela, vous devez avoir préalablement stocké l'index de l'articulation dans une variable puis vous pouvez allumer ou éteindre le moteur en utilisant cette fonction et en définissant l'argument "moteur" sur true ou false. Syntaxe:physics_joint_enable_motor(joint, motor) Argument Description joint L'articulation sur laquelle vous souhaitez activer ou désactiver le moteur motor Que vous souhaitiez allumer (vrai) ou éteindre le moteur (faux) Retour:N/A Exemple:var t_fix = physics_fixture_create(); physics_fixture_set_circle_shape(t_fix, sprite_get_width(sprite_index) / 2); var o_id=instance_create_layer(x+300, y, "Instances", obj_Door); physics_fixture_bind(t_fix, id); physics_fixture_bind(t_fix, o_id); perma_joint = physics_joint_revolute_create(id, o_id, x+25, y, -90, 90, 1, 10, 2, 0, 0); physics_joint_enable(perma_joint, 1); physics_fixture_delete(t_fix); Le code ci-dessus crée et définit un nouveau luminaire, puis crée une instance de "obj_Door", liant le luminaire créé aux deux instances. Ils sont ensuite joints par un joint tournant sans moteur et les angles limités à une oscillation de +/- 90 degrés, et nous stockons l'indice de joint dans la variable "perma_joint". Nous allumons ensuite le moteur à l'aide de cette variable, avant de supprimer définitivement l'appareil de la mémoire car il n'est plus nécessaire.
