physics_joint_get_value En utilisant une série de constantes prédéfinies, vous pouvez demander à GameMaker Studio 2 de vous dire un certain nombre de choses sur l'état d'une articulation donnée. Ceci est très utile car cela vous donne la possibilité de supprimer des joints ou de modifier le comportement d'une instance en fonction de vos besoins à ce moment-là. Il existe un certain nombre de constantes qui peuvent être utilisées dans cette fonction et elles peuvent être trouvées ici: Physics Joint Constants, mais sachez que des calculs complexes sont effectués lorsque vous les appelez, ils doivent donc être utilisés avec précaution et uniquement lorsque cela est nécessaire et notez que beaucoup sont uniques à un type spécifique de joint. Si la propriété n'existe pas (si, par exemple, vous vérifiez une articulation de poulie pour une translation prismatique), 0 sera la valeur de retour. Syntaxe:physics_joint_get_value(joint, value) Argument Description joint L'index de l'articulation que vous souhaitez tester value La constante de la propriété commune que vous souhaitez tester Retour:Real Exemple:if shipJoint     {     var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);     var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);     var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);     if reactionForce > 2         {         physics_joint_delete(shipJoint);         shipJoint = -1;         }     } Le code ci-dessus vérifie si la variable "shipJoint" contient un index d'articulation et si c'est le cas, il calcule alors la force appliquée à cette articulation en utilisant les deux constantes. Enfin, si la force totale est supérieure à 2, l'articulation est supprimée. Arrière: Joints Suivant:physics_joint_set_value © Copyright YoYo Games Ltd. 2021 Tous droits réservés
