physics_joint_delete Une fois que deux instances avec des représentations physiques ont été liées par un joint, celui-ci peut être supprimé à nouveau à tout moment. Normalement, cela se produit automatiquement lorsque l'une des deux instances est détruite, ou lorsque la room se termine, mais il arrive que vous souhaitiez le faire manuellement. mais il peut arriver que vous souhaitiez le faire manuellement. Dans ce cas, vous devez utiliser cette fonction. Il faut également noter que lorsqu'une instance faisant partie d'un joint d'engrenage est détruite, le joint d'engrenage doit être supprimé à l'aide de cette fonction avant toute instance impliquée dans la formation de l'engrenage (mais les joints restants seront supprimés automatiquement, par exemple dans l'événement destroy de l'instance). Syntaxe : physics_joint_delete(joint) Arguments Description joint L'index du joint que vous souhaitez supprimer Les retours : N/A Exemple : var reactionForceX, reactionForceY, reactionForce ; if shipJoint { reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x) ; reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x) ; reactionForce = sqrt((reactionForceX + reactionForceX) + (reactionForceY + reactionForceY)) ; if reactionForce 2 (reactionForceY + reactionForceY)) ; if reactionForce > 2 { physics_joint_delete(shipJoint) ; shipJoint = -1 ; } } Le code ci-dessus vérifie si la variable "shipJoint" contient un indice d'articulation et si c'est le cas, il calcule la force appliquée à cette articulation à l'aide des deux constantes.
