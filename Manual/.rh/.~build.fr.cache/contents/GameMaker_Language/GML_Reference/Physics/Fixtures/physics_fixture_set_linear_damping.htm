physics_fixture_set_linear_damping L'amortissement est utilisé pour réduire la vitesse des instances dans le monde physique, et diffère de la friction en ce que la friction ne se produit que lorsque deux instances avec des fixtures sont en contact. L'amortissement est également beaucoup moins cher à simuler que la friction, mais veuillez noter que l'amortissement ne remplace pas la friction. ne remplace pas la friction ; les deux effets peuvent, et doivent, être utilisés ensemble. Voici une image pour illustrer l'amortissement : Les paramètres d'amortissement doivent être compris entre 0 et l'infini, 0 signifiant aucun amortissement, et l'infini un amortissement total. Normalement, vous utiliserez une valeur d'amortissement comprise entre 0 et 1, mais vous pouvez utiliser toute valeur non négative si nécessaire. Syntaxe : physics_fixture_set_linear_damping(fixture, damping) Arguments Description L'appareil l'indice du dispositif amortissement l'amortissement du dispositif, généralement entre 0 et 1 Les retours : N/A Exemple : physics_fixture_set_linear_damping(fix_Ball, 0.1) ; Le code ci-dessus fixera l'amortissement linéaire de l'accessoire indexé dans "fix_ball" à 0.1.
