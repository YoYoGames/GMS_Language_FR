physics_remove_fixture Cette fonction supprime (ou "dé-liaison") un dispositif d'une ou plusieurs instances. Elle requiert l'"id" unique du dispositif lié (tel que retourné par la fonction physics_fixture_bind() et elle supprime toutes les propriétés physiques actuellement définies pour l'instance, ce qui vous permet de redéfinir un nouveau dispositif et de le lier à l'instance. De cette façon, vous pouvez modifier les propriétés physiques de l'instance sans avoir à la détruire et à la recréer. l'instance. Syntaxe : physics_remove_fixture(id, fixture) Arguments Description id L'ID de l'instance dont il faut retirer le dispositif de fixation L'appareil L'ID de l'appareil qui doit être retiré de l'instance. Les retours : N/A Exemple : physics_remove_fixture(id, my_fix) ; Le code ci-dessus supprimera de l'instance le fixture avec le "id" stocké dans la variable "my_fix".
