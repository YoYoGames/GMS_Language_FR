physics_remove_fixture Cette fonction supprime (ou "dé-lie") un fixture d'une ou plusieurs instances. Il nécessite le "id" unique du luminaire lié (tel que renvoyé par la fonction physics_fixture_bind() et il supprimera toutes les propriétés physiques actuellement définies pour l'instance, vous permettant de redéfinir un nouvel appareil et de le lier à l'instance. De cette façon, vous pouvez modifier les propriétés physiques des instances sans avoir à les détruire et à les recréer. Syntaxe:physics_remove_fixture(id, fixture) Argument Description id L'ID de l'instance à partir de laquelle supprimer le projecteur fixture L'ID de l'appareil à supprimer de l'instance Retour:N/A Exemple:physics_remove_fixture(id, my_fix); Le code ci-dessus supprimera le projecteur avec "id" stocké dans la variable "my_fix" de l'instance.
