physics_fixture_set_restitution En physique, la restitution est définie comme "le retour d'un objet ou d'un système à son état d'origine après déformation élastique", mais comme les fixations dans GameMaker Studio 2 sont des corps vraiment rigides et ne peuvent pas être déformés, la restitution est vraiment une façon de dire comment "rebondissant" le luminaire est. Ce paramètre affectera la mesure dans laquelle un objet «rebondit» lorsqu'il entre en collision avec d'autres objets et dépend d'autres forces qui agissent sur l'instance comme la gravité et le frottement, et est généralement une valeur comprise entre 0 et 1 (des valeurs plus élevées peuvent être utilisées mais peut donner des résultats imprévisibles). Voici une illustration de son fonctionnement: Syntaxe:physics_fixture_set_restitution(fixture, restitution) Argument Description fixture L'index du luminaire restitution La restitution du luminaire (généralement entre 0 et 1) Retour:N/A Exemple:physics_fixture_set_restitution(fix_Ball, 0.9); Le code ci-dessus définira la restitution de l'appareil indexé dans "fix_ball" à 0.9.
