physics_fixture_set_awake Lorsque vous liez un appareil à une instance en utilisant physics_fixture_bind() cela renvoie un "id" pour le projecteur lié. Vous pouvez utiliser cet identifiant pour obtenir la valeur de friction de l'appareil lié (et non de l'appareil "de base") à tout moment en utilisant cette fonction. Syntaxe:physics_get_friction(fixture) Argument Description fixture l'id de l'appareil lié Retour:Real Exemple:var fric = physics_get_friction(fix_id); physics_set_friction(fix_id, fric + 0.1); Le code ci-dessus obtient la valeur de friction actuelle pour les propriétés physiques liées de l'instance, puis les définit sur une valeur différente.
