physics_set_density Lorsque vous liez un appareil à une instance en utilisant physics_fixture_bind() cela renvoie un "id" pour le projecteur lié. Vous pouvez utiliser cet identifiant pour définir la valeur de densité de l'appareil lié, et non de l'appareil «de base», à tout moment en utilisant cette fonction. Syntaxe:physics_set_density(fixture, density) Argument Description fixture l'id de l'appareil lié density la nouvelle valeur de densité à appliquer Retour:N/A Exemple:var density = physics_get_density(fix_id); physics_set_density(fix_id, density - 0.1); Le code ci-dessus obtient la valeur de densité actuelle pour les propriétés physiques liées de l'instance, puis les définit sur une valeur différente.
