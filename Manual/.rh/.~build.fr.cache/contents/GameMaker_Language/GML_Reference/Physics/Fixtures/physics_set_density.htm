physics_set_density Lorsque vous liez un dispositif à une instance à l'aide de la fonction physics_fixture_bind() cela renvoie un "id" pour le dispositif lié. Vous pouvez utiliser cet identifiant pour définir la valeur de densité du projecteur lié, et non du projecteur "de base", à tout moment en utilisant cette fonction. "pas le dispositif de base, à tout moment en utilisant cette fonction. Syntaxe : physics_set_density(fixture, density) Arguments Description L'appareil l'identifiant du dispositif lié densité la nouvelle valeur de densité à appliquer Les retours : N/A Exemple : var density = physics_get_density(fix_id) ; physics_set_density(fix_id, density - 0.1) ; Le code ci-dessus obtient la valeur de la densité de courant pour les propriétés physiques liées de l'instance, puis les définit à une valeur différente.
