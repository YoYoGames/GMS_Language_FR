physics_fixture_set_box_shape Cette fonction définit une forme de boîte pour votre appareil. Elle prend la moitié de la largeur et de la hauteur car le monde de la physique utilise cette valeur bien plus que les valeurs entières de largeur/hauteur pour déterminer des choses comme les collisions. Syntaxe : physics_fixture_set_box_shape(fixture, halfWidth, halfHeight) Arguments Description L'appareil l'indice du dispositif demi-largeur la demi-largeur de la boîte mi-hauteur la demi-hauteur de la boîte Les retours : N/A Exemple : physics_fixture_set_box_shape(fix_Border, room_width/2, 10) ; Le code ci-dessus appliquera une forme de boîte à la pièce indexée dans la variable "fix_Border" avec une largeur de la pièce et une hauteur de 20 pixels.
