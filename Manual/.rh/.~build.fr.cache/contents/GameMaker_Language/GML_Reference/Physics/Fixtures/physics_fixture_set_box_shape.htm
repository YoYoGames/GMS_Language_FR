physics_fixture_set_box_shape Cette fonction définit une forme de boîte pour votre appareil. Cela prend la moitié de la largeur et de la hauteur car le monde de la physique utilise cette valeur beaucoup plus que les valeurs de largeur / hauteur entières pour déterminer des choses comme les collisions. Syntaxe:physics_fixture_set_box_shape(fixture, halfWidth, halfHeight) Argument Description fixture l'index du luminaire halfWidth la demi- largeur de la boîte halfHeight la demi- hauteur de la boîte Retour:N/A Exemple:physics_fixture_set_box_shape(fix_Border, room_width/2, 10); Le code ci-dessus appliquera une forme de boîte au luminaire indexé dans la variable "fix_Border" avec une largeur de la pièce et une hauteur de 20 pixels.
