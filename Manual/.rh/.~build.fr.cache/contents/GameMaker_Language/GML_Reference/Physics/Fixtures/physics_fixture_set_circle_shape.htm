physics_fixture_set_circle_shape Cette fonction définit une forme de cercle pour votre appareil avec un rayon défini par l'argument "rad". Syntaxe : physics_fixture_set_circle_shape(fixture, rad) Arguments Description L'appareil l'indice du dispositif rad rayon du cercle Les retours : N/A Exemple : physics_fixture_set_circle_shape(fix_Ball, sprite_get_width(spr_Ball) / 2) ; Le code ci-dessus appliquera une forme de cercle à la fixation indexée dans la variable "fix_Ball" avec un rayon identique à celui de la largeur du sprite "spr_Ball" divisé par 2.
