physique_fixture_create La première étape de la mise en place d'un dispositif est de le créer avec cette fonction. L'indice de retour doit être stocké dans une variable qui sera utilisée dans toutes les autres fonctions utilisées pour définir et utiliser ce dispositif. Syntaxe : physics_fixture_create() Les retours : Index de l'appareil Exemple : fix_Ball = physics_fixture_create() ; Le code ci-dessus va créer un accessoire et stocker son index dans la variable "fix_Ball".
