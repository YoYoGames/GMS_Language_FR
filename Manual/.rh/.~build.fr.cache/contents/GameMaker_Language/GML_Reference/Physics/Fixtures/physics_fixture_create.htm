physics_fixture_create La première étape de la configuration d'un appareil consiste à le créer avec cette fonction. L'index renvoyé doit être stocké dans une variable à utiliser dans toutes les autres fonctions utilisées pour définir et utiliser cet appareil. Syntaxe:physics_fixture_create() Retour:Index of the fixture Exemple:fix_Ball = physics_fixture_create(); Le code ci-dessus créera un appareil et stockera son index dans la variable "fix_Ball".
