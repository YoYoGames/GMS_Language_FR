ds_list_mark_as_map Cette fonction va "marquer" (ou "signaler") une position donnée dans une liste de DS précédemment créée comme contenant une carte de DS. Cette fonctionnalité est nécessaire lors de l'encodage des chaînes JSON (que vous pouvez créer en utilisant json_encode()), mais peut également être utile lors de l'imbrication de structures de données, car les éléments marqués de cette manière seront automatiquement récupérés (détruits) lorsque la liste DS parente sera détruite. parent est détruite. Cela signifie que vous n'avez pas besoin de parcourir manuellement le contenu de la liste et de détruire les structures de données marquées individuellement avant de détruire la liste "parent". Cependant, si vous détruisez la position de la liste individuellement, la structure de données qu'elle contenait ne sera pas récupérée et vous devez appeler la fonction de destruction de la carte DS appropriée avant de supprimer la position de la liste parent. Notez également que si vous appelez la fonction ds_list_clear() sur une liste, tous les éléments marqués comme des cartes seront également détruits lorsque la liste sera effacée. Il convient également de noter que si la position qui a été marquée change au sein de la liste, la "marque" se déplacera avec elle. Ainsi, si vous avez marqué la position 3 dans la liste (par exemple) et que vous insérez ensuite 2 éléments supplémentaires sous cette position, de sorte qu'elle passe à la position 5, elle sera toujours marquée comme une carte. 5, il sera toujours marqué comme une carte. Syntaxe : ds_list_mark_as_map(id, pos) ; Arguments Description id L'identifiant de la liste à marquer. pos La position à marquer dans la liste. Les retours : Real (entier - ID de la carte DS si elle est réussie ou -1 si elle échoue) Exemple : var sub_map = ds_map_create() ; ds_map_add(sub_map, "player", player_array) ; ds_map_add(sub_map, "enemy", enemy_array) ; ds_list_add(j_list, sub_map) ; ds_list_mark_as_map(j_list, 0) ; Le code ci-dessus crée une carte DS et la remplit avec deux clés, chacune contenant un tableau de valeurs. Cette carte est ensuite ajoutée à la liste DS donnée, et la position est "marquée" comme telle afin qu'elle puisse être correctement encodée par la suite.
