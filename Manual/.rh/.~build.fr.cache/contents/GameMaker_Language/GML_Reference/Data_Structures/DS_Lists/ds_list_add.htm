ds_list_add Cette fonction peut être utilisée pour ajouter une nouvelle valeur (réelle ou chaîne) à la liste, qui sera ajoutée à la fin. La fonction peut prendre d'autres arguments facultatifs (autant que vous le souhaitez), vous permettant d'ajouter plusieurs valeurs consécutivement à la liste en un seul appel. Syntaxe:ds_list_add(id, val1 [, val2, ... max_val]); Argument Description id L'identifiant de la liste à ajouter. val La valeur à ajouter à la liste. [val2,... max_val] Valeurs facultatives à ajouter à la liste. Retour:N/A Exemple:ds_list_add(sc_list, score); Le code ci-dessus ajoutera la valeur stockée dans la variable "score" dans la liste indexée dans la variable "sc_list".
