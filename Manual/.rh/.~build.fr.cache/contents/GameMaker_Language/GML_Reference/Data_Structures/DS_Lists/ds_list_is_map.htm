ds_list_is_map Cette fonction vous permet de vérifier si une carte DS est stockée à la position donnée dans une liste DS. Si la position donnée contient un ID de carte DS, la fonction renvoie true, sinon elle renvoie false. Syntaxe : ds_list_is_map(id, pos) ; Arguments Description id L'identifiant de la liste à vérifier. pos La position à vérifier dans la liste. Les retours : Booléen Exemple : var size = ds_list_size(ships) ; for (var i = 0 ; i < size ; i++ ;) { if ds_list_is_map(ships, i) { ds_map_destroy(i) ; } } ds_list_destroy(ships) ; Le code ci-dessus boucle à travers une liste DS et vérifie si l'une des entrées contient des ID de carte. Si c'est le cas, ces cartes sont détruites, puis la liste principale est détruite à la fin de la boucle.
