ds_list_find_value Avec cette fonction, vous pouvez vérifier la position de liste donnée et la valeur contenue dans la liste pour cette position sera renvoyée. Notez que si vous donnez une position qui est en dehors de la taille de la liste donnée (c'est-à-dire: la position 11 dans une liste de 10 valeurs), la fonction peut retourner undefined ou 0. En effet, lorsque vous créez la liste, les premières entrées de la liste sont définies sur 0 en interne pour minimiser les problèmes de performances lors de l'ajout initial d'éléments à la liste (bien que le ds_list_size() la fonction retournera toujours 0 sur une liste nouvellement créée). Si vous souhaitez vous assurer que la liste est "vraiment" vide lors de la création, vous devez appeler ds_list_clear() après avoir créé la liste, ce qui signifiera alors que toutes les valeurs renvoyées pour les emplacements de liste non remplis seront undefined, que vous pouvez ensuite vérifier de manière cohérente en utilisant le is_undefined() fonction. Syntaxe:ds_list_find_value(id, pos); Argument Description id L'identifiant de la liste à utiliser. pos La position à regarder, où 0 correspond au tout début de la liste et la position finale est ds_list_size (id) -1. Retour:Real, String or Undefined Exemple:val = ds_list_find_value(list, ds_list_size(list) - 1); Le code ci-dessus vérifie la liste indexée dans la variable "list" à la dernière position de la liste et stocke la valeur renvoyée dans la variable "val".
