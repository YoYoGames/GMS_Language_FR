ds_list_size Cette fonction retournera la "taille" de la liste, c'est-à-dire: le nombre d'éléments qui y ont été ajoutés. Syntaxe:ds_list_size(id); Argument Description id L'identifiant de la structure de données à vérifier. Retour:Real Exemple:if !ds_list_empty(control_list)     {     num = ds_list_size(control_list);     } Le code ci-dessus vérifie une liste DS pour voir si elle est vide ou non, et si ce n'est pas le cas, il obtient le nombre d'éléments qu'elle contient et stocke la valeur dans une variable.
