ds_list_sort Cette fonction vous permet de trier toutes les valeurs d'une liste, dans l'ordre croissant ou décroissant. Si la liste contient des chaînes de caractères, celles-ci seront triées par ordre alphabétique, sur la base de l'alphabet anglais de 26 lettres. Syntaxe : ds_list_sort(id, ascend) ; Arguments Description id L'identifiant de la liste à trier. monter Si les valeurs doivent être classées par ordre croissant(true) ou décroissant(false). Les retours : N/A Exemple : if newgame { ds_list_sort(name_list, true) ; } Le code ci-dessus va trier la liste indexée dans la variable "name_list" si la variable "newgame" est marquée comme vraie.
