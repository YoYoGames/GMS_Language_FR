ds_list_insert Cette fonction ajoutera la valeur donnée dans la liste à la position donnée. si la liste contient plus de valeurs après la position donnée, leur position sera décalée d'une unité vers le haut pour faire de la place en agrandissant la liste d'une unité. Syntaxe:ds_list_insert(id, pos, val); Argument Description id L'identifiant de la liste à ajouter. pos La position pour ajouter la valeur, où 0 correspond au tout début de la liste et la position finale est ds_list_size (id) -1. val La valeur à ajouter à la liste. Retour:N/A Exemple:ds_list_insert(list, 9, score); Le code ci-dessus ajoutera la valeur stockée dans la variable "score" à la 10ème position de la liste indexée par la variable "list" (les listes sont comptées à partir de 0, donc une liste de dix valeurs est numérotée de 0 à 9).
