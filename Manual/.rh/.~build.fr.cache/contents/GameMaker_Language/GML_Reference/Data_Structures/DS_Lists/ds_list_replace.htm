ds_list_replace Cette fonction remplacera la valeur à la position donnée par une autre. Syntaxe:ds_list_replace(id, pos, val); Argument Description id L'identifiant de la liste à modifier. pos La position pour remplacer la valeur, où 0 correspond au tout début de la liste et la position finale est ds_list_size (id) -1. val La nouvelle valeur par laquelle remplacer la valeur donnée. Retour:N/A Exemple:ds_list_replace(n_list, 3, name); Le code ci-dessus remplacera la valeur stockée en position 3 dans la liste par celle stockée dans la variable "nom".
