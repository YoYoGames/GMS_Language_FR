ds_list_replace Cette fonction va remplacer la valeur à la position donnée par une autre. Syntaxe : ds_list_replace(id, pos, val) ; Arguments Description id L'identifiant de la liste à modifier. pos La position pour remplacer la valeur, où 0 correspond au tout début de la liste et la position finale est ds_list_size(id)-1. val La nouvelle valeur avec laquelle remplacer la valeur donnée. Les retours : N/A Exemple : ds_list_replace(n_list, 3, nom) ; Le code ci-dessus remplacera la valeur stockée à la position 3 de la liste par celle stockée dans la variable "nom".
