ds_list_is_list Avec cette fonction, vous pouvez vérifier si une autre liste DS est stockée à la position donnée dans une liste DS. Si la position donnée contient un ID de liste DS, la fonction retournera true sinon il reviendra false. Syntaxe:ds_list_is_list(id, pos); Argument Description id L'identifiant de la liste à vérifier. pos La position dans la liste à vérifier. Retour:Boolean Exemple:var size = ds_list_size(ships) ; for (var i = 0; i < size; i++;)     {     if ds_list_is_list(ships, i)         {         ds_list_destroy(i);         }     } ds_list_destroy(ships); Le code ci-dessus parcourt une liste DS et vérifie si l'une des entrées contient d'autres ID de liste. Si c'est le cas, ces listes sont détruites, puis la liste principale est détruite une fois la boucle terminée.
