ds_list_is_list Cette fonction permet de vérifier si une autre liste de DS est stockée à la position donnée dans une liste de DS. Si la position donnée contient un ID de liste de DS, alors la fonction renverra true, sinon elle renverra false. Syntaxe : ds_list_is_list(id, pos) ; Arguments Description id L'identifiant de la liste à vérifier. pos La position à vérifier dans la liste. Les retours : Booléen Exemple : var size = ds_list_size(ships) ; for (var i = 0 ; i < size ; i++ ;) { if ds_list_is_list(ships, i) { ds_list_destroy(i) ; } } ds_list_destroy(ships) ; Le code ci-dessus boucle sur une liste DS et vérifie si l'une des entrées contient d'autres ID de liste. Si c'est le cas, ces listes sont détruites, puis la liste principale est détruite à la fin de la boucle.
