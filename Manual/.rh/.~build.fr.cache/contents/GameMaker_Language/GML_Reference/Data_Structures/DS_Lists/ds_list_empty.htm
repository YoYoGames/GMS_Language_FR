ds_list_empty Avec cette fonction, vous pouvez vérifier la liste DS donnée pour voir si elle est vide (renvoie true) ou non (renvoie false). Syntaxe : ds_list_empty(id) ; Arguments Description id L'identifiant de la structure de données à vérifier. Les retours : Booléen Exemple : if (count == 15( && (!ds_list_empty(command_list)) { ds_list_clear(command_list) ; alarm[0] = room_speed ; count = 0 ; } Le code ci-dessus vérifie une variable pour voir si elle a atteint une valeur spécifique et si c'est le cas, il efface la liste DS indexée dans la variable "command_list", déclenche une alarme et remet la variable à 0.
