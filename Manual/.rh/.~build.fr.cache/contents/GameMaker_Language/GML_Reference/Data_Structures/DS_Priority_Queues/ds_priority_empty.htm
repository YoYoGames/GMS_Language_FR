ds_priority_empty Avec cette fonction, vous pouvez vérifier la file d'attente prioritaire DS donnée pour voir si elle est vide (renvoie true) ou non (renvoie false). Syntaxe : ds_priority_empty(id) ; Arguments Description id L'identifiant de la structure de données à vérifier. Les retours : Booléen Exemple : if (count == 15) && (!ds_priority_empty(command_queue)) { ds_priority_clear(command_queue) ; alarm[0] = room_speed ; ai_count = 0 ; } Le code ci-dessus vérifie une variable pour voir si elle a atteint une valeur spécifique et si c'est le cas, il efface la file d'attente prioritaire DS indexée dans la variable "command_queue", déclenche une alarme et remet la variable à 0.
