ds_priority_size Cette fonction renverra la "taille" de la file d'attente prioritaire, c'est-à-dire: le nombre d'éléments qui y ont été priorisés. Syntaxe:ds_priority_size(id); Argument Description id L'identifiant de la structure de données à vérifier. Retour:Real Exemple:if !ds_priority_empty(control_priority)     {     num = ds_priority_size(control_priority);     } Le code ci-dessus vérifie une file d'attente de priorité DS pour voir si elle est vide ou non, et si ce n'est pas le cas, il obtient le nombre d'éléments qu'elle contient et stocke la valeur dans une variable.
