ds_priority_size Cette fonction renvoie la "taille" de la file d'attente prioritaire, c'est-à-dire le nombre d'éléments qui y ont été classés par ordre de priorité. Syntaxe : ds_priority_size(id) ; Arguments Description id L'identifiant de la structure de données à vérifier. Les retours : Real Exemple : if !ds_priority_empty(control_priority) { num = ds_priority_size(control_priority) ; } Le code ci-dessus vérifie une file d'attente prioritaire DS pour voir si elle est vide ou non, et si elle ne l'est pas, il obtient le nombre d'éléments qu'elle contient et stocke la valeur dans une variable.
