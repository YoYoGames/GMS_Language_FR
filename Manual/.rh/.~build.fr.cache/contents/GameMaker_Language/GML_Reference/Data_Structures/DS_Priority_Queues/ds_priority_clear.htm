ds_priority_clear Avec cette fonction, vous pouvez effacer toutes les données de la structure de données de la file d'attente prioritaire donnée. Cela ne détruit PAS la structure de données (pour cela vous devez utiliser la fonction ds_priority_destroy()), elle efface seulement toutes les données et renvoie une file prioritaire vide. Syntaxe : ds_priority_clear(id) ; Arguments Description id L'identifiant de la structure de données à effacer. Les retours : N/A Exemple : if (count = 15) && (!ds_priority_empty(command_queue)) { ds_priority_clear(command_queue) ; alarm[0] = room_speed ; ai_count = 0 ; } Le code ci-dessus vérifie une variable pour voir si elle a atteint une valeur spécifique et si c'est le cas, il efface la file d'attente prioritaire DS indexée dans la variable "command_queue", déclenche une alarme et remet la variable à 0.
