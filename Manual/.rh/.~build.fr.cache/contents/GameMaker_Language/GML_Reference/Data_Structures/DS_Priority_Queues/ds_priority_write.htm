ds_priority_write Cette fonction renvoie une chaîne de caractères qui peut ensuite être stockée ou transférée dans une autre structure de données à l'aide de la fonction ds_priority_read() fonction. NOTE: La chaîne retournée n'est pas une chaîne lisible par l'homme, mais plutôt un vidage du contenu de la structure de données. Syntaxe : ds_priority_write(id) ; Arguments Description id L'identifiant de la structure de données à vérifier. Les retours : Chaîne de caractères Exemple : var str ; ini_open("save.ini") ; str =ds_priority_write(p_queue) ; ini_write_string("P_Queues", "0", str) ; ds_priority_clear(p_queue) ; ini_close() ; Le code ci-dessus ouvre un fichier ini et écrit ensuite une chaîne de caractères contenant les informations stockées dans la file d'attente prioritaire DS indexée dans la variable "p_queue". La file prioritaire est ensuite effacée et le fichier ini fermé. Retour : Files d'attente prioritaires DS Suivant : ds_priority_create Copyright YoYo Games Ltd. 2021 Tous droits réservés
