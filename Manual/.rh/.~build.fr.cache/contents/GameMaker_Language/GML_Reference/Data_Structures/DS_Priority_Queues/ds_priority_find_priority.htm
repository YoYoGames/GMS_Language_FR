ds_priority_find_priority Avec cette fonction, vous pouvez récupérer la priorité d'une valeur donnée. Si la valeur n'existe pas dans la file d'attente des priorités, undefined sera retourné. Syntaxe : ds_priority_find_priority(id, val) ; Arguments Description id L'identifiant de la file d'attente prioritaire à utiliser. val La valeur dont il faut trouver la priorité. Les retours : Réel ou indéfini Exemple : p = ds_priority_find_priority(ai_priority, "intelligence") ; Le code ci-dessus stockera la priorité retournée pour la valeur donnée dans la variable d'instance "p".
