ds_priority_delete_value Cette fonction supprimera simplement la valeur donnée, ainsi que sa priorité, de la file d'attente des priorités indexées. Syntaxe:ds_priority_delete_value(id,val); Argument Description id L'ID de la file d'attente prioritaire à utiliser. val La valeur à supprimer de la file d'attente prioritaire. Retour:N/A Exemple:if (ai_move == false)     {     ds_priority_delete_value(ai_priority, AI_Move);     } Le code ci-dessus vérifie une variable d'instance et si elle retourne false il supprimera la fonction de script indexée de la file d'attente prioritaire.
