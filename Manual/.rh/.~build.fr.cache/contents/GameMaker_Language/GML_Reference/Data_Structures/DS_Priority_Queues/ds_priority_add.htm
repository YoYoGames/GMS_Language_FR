ds_priority_add Avec cette fonction, vous pouvez ajouter une valeur (un nombre réel ou une chaîne) à une file d'attente prioritaire, en lui attribuant en même temps une valeur de priorité. Syntaxe:ds_priority_add(id, val, priority); Argument Description id L'ID de la file d'attente prioritaire à laquelle ajouter. val La valeur à ajouter à la file d'attente prioritaire. priority La priorité de la valeur à ajouter. Retour:N/A Exemple:ds_priority_add(ai_priority, AI_Search, 5); Le code ci-dessus ajoute une fonction de script à la file de priorité indexée dans la variable "ai_priority" et lui attribue une priorité de 5.
