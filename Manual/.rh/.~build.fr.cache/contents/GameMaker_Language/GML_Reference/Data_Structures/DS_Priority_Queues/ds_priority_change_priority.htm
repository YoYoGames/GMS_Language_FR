ds_priority_change_priority Cette fonction prend une valeur donnée et change sa priorité dans la file d'attente de priorité référencée. Syntaxe : ds_priority_change_priority(id, val, priority) ; Arguments Description id L'identifiant de la file d'attente prioritaire à modifier. val La valeur dont il faut changer la priorité. priorité La nouvelle priorité de la valeur. Les retours : N/A Exemple : if (global.Game_Time < 1000) { ds_priority_change(ai_priority, AI_Search, 1) ; } Le code ci-dessus vérifie une variable globale et si elle est inférieure à une certaine valeur, il modifie la priorité de l'index de la fonction de script dans la file d'attente des priorités.
