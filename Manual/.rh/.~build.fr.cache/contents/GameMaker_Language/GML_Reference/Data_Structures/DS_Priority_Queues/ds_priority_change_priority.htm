ds_priority_change_priority Cette fonction prendra une valeur donnée et changera sa priorité dans la file d'attente de priorité référencée. Syntaxe:ds_priority_change_priority(id, val, priority); Argument Description id L'identifiant de la file d'attente prioritaire à modifier. val Valeur pour laquelle modifier la priorité. priority La nouvelle priorité de la valeur. Retour:N/A Exemple:if (global.Game_Time < 1000)     {     ds_priority_change(ai_priority, AI_Search, 1);     } Le code ci-dessus vérifie une variable globale et si elle est inférieure à une certaine valeur, il changera alors la priorité de l'index de fonction de script contenu dans la file d'attente de priorité.
