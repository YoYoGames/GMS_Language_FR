ds_priority_find_max Avec cette fonction, vous pouvez trouver la valeur stockée dans la file d'attente prioritaire avec la priorité la plus élevée, et si plusieurs valeurs ont la même priorité, l'une d'entre elles peut être renvoyée dans n'importe quel ordre. Cependant, contrairement à ds_priority_delete_max(), cette fonction ne supprimera pas la valeur de la file d'attente. Syntaxe:ds_priority_find_max(id); Argument Description id L'ID de la file d'attente prioritaire à utiliser. Retour:Real or String Exemple:if ai_move     {     script_execute(ds_priority_find_max(ai_priority));     } Le code ci-dessus vérifie une variable d'instance et si elle retourne true il exécutera une fonction de script indexée dans la file d'attente prioritaire avec la valeur de priorité la plus élevée.
