ds_priority_delete_min Cette fonction renvoie la valeur qui a la priorité la plus basse dans la file d'attente, puis supprime la valeur (et la priorité) de la structure de données. Si plus d'une valeur a la même priorité, alors n'importe laquelle d'entre elles peut être retournée dans n'importe quel ordre, mais toutes les autres valeurs avec la même priorité seront toujours dans la file d'attente. mais toutes les autres valeurs ayant la même priorité seront toujours dans la file d'attente. Syntaxe : ds_priority_delete_min(id) ; Arguments Description id L'identifiant de la file d'attente prioritaire à utiliser. Les retours : Type de données stockées dans la priorité Exemple : if ai_move { script_execute(ds_priority_delete_min(ai_priority)) ; } Le code ci-dessus vérifie une variable d'instance et s'il retourne vrai, il exécute une fonction de script indexée dans la file d'attente des priorités avec la valeur de priorité la plus basse, puis supprime ce script de la file d'attente.
