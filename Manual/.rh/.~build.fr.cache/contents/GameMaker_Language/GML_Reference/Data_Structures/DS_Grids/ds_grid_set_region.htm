ds_grid_set_region Cette fonction peut être utilisée pour définir une région rectangulaire d'une grille donnée à une valeur spécifiée (qui peut être un réel ou une chaîne de caractères) comme l'illustre l'image ci-dessous : Syntaxe : ds_grid_set_region(index, x1, y1, x2, y2, val) ; Arguments Description indice L'index de la grille. x1 La position x de la partie gauche de la région dans la grille. y1 La position y du haut de la région dans la grille. x2 La position x de la droite de la région dans la grille. y2 La position y du bas de la région dans la grille. val La valeur à donner aux cellules de la région. Les retours : N/A Exemple : ds_grid_set_region(grid, 5, 5, 10, 10, 99) Toutes les cellules situées dans la région de la grille indexée dans la variable "grille" de (5,5) à (10,10) seraient ainsi fixées à 99.
