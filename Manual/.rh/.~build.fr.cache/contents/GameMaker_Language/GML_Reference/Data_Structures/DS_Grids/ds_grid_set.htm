ds_grid_set Cette fonction peut être utilisée pour donner à une cellule donnée de la grille DS une valeur quelconque, qui peut être un nombre réel ou une chaîne de caractères. L'image ci-dessous l'illustre : Syntaxe : ds_grid_set(index, x, y, valeur) ; Arguments Description indice Cet indice de la grille. x La position x de la cellule à définir. y La position y de la cellule à définir. valeur La valeur avec laquelle la cellule doit être définie. Les retours : N/A Exemple : grid = ds_grid_create(5, 5) ; var i = 0 ; var j = 0 ; repeat (ds_grid_width(grid)) { repeat (ds_grid_height(grid)) { ds_grid_set(grid, i, j, irandom(9)) ; j += 1 ; } j = 0 ; i += 1 ; } Le code ci-dessus crée une grille et stocke son index dans la variable "grid". Il remplit ensuite cette grille avec des entiers aléatoires de 0 à 9. Retour : Grilles DS Suivant : ds_grid_set_disk Copyright YoYo Games Ltd. 2021 Tous droits réservés
