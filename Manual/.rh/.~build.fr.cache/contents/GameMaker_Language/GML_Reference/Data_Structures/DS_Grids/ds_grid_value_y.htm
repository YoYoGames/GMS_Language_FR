ds_grid_value_y Cette fonction permet d'obtenir la coordonnée y (dans la région de grille rectangulaire donnée) de la valeur recherchée. Vous donnez l'index de la grille DS (tel que retourné par ds_grid_create()) ainsi que les positions x/y pour les coins supérieur gauche et inférieur droit de la région de la grille à rechercher, avant de fournir la valeur à rechercher. Si la valeur recherchée n'existe pas, la fonction renvoie -1. Syntaxe : ds_grid_value_y(index, x1, y1, x2, y2, val) ; Arguments Description indice L'index de la grille. x1 La position x de la partie gauche de la région dans la grille, de 0 à (largeur de la grille - 1). y1 La position y du haut de la région dans la grille, de 0 à (hauteur de la grille - 1). x2 La position x de la droite de la région dans la grille, de 0 à (largeur de la grille - 1). y2 La position y du bas de la région dans la grille, de 0 à (hauteur de la grille - 1) val La valeur à trouver. Les retours : Real Exemple : if ds_grid_value_exists(grid, 0, 1, 5, 6, val) { xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val) ; ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val) ; } Le code ci-dessus vérifie une ds_grid à la recherche d'une valeur spécifique dans une région rectangulaire. Si elle est trouvée, il stocke alors la position x et y de la valeur dans deux variables pour une utilisation ultérieure.
