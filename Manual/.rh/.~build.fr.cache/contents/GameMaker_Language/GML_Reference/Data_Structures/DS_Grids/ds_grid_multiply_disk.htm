ds_grid_multiply_disk Cette fonction prendra toutes les valeurs dans une région en forme de disque donnée de la grille DS, et multipliera chacune par la quantité donnée. REMARQUE: Cette fonction ne fonctionnera qu'avec des nombres réels, pas des chaînes. Syntaxe:ds_grid_multiply_disk(index, xm, ym, r, val); Argument Description index L'index de la grille. xm La position x du disque sur la grille. ym La position y du disque sur la grille. r Le rayon du disque sur la grille. val Valeur avec laquelle multiplier les cellules du disque. Retour:N/A Exemple:ds_grid_multiply_disk(mygrid, 5, 5, 5, 2) Le code ci-dessus prendra toutes les valeurs trouvées dans la zone de grille circulaire et multipliera chacune par 2.
