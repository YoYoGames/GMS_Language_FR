ds_grid_get_disk_max Cette fonction peut être utilisée pour trouver la valeur maximale de toutes les cellules se trouvant dans la zone de disque définie d'une grille, comme le montre l'image ci-dessous : Syntaxe : ds_grid_get_disk_max(index, xm, ym, r) ; Arguments Description indice L'index de la grille. xm La position x du centre du disque dans la grille. ym La position y du centre du disque dans la grille. r Le rayon du disque dans la grille. Les retours : Réel ou Chaîne Exemple : val = ds_grid_get_disk_max(grid, 5, 5, 2) Le code ci-dessus va définir la variable "val" à la valeur maximale contenue dans le disque donné de la grille DS indexée dans la variable "grid".
