ds_grid_get_disk_sum Cette fonction peut être utilisée pour ajouter toutes les valeurs de toutes les cellules trouvées dans la zone de disque définie d'une grille ensemble, comme indiqué dans l'image ci-dessous: Syntaxe:ds_grid_get_disk_sum(index, xm, ym, r); Argument Description index L'index de la grille. xm La position x du disque sur la grille. ym La position y du disque sur la grille. r Le rayon du disque sur la grille. Retour:Real or String Exemple:val = ds_grid_get_disk_sum(grid, 5, 5, 2) Le code ci-dessus définira la variable «val» à la somme de toutes les valeurs contenues dans le disque donné de la grille DS indexée dans la variable «grille».
