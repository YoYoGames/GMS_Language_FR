ds_grid_add_disk Cette fonction peut être utilisée pour ajouter une valeur donnée (réelle ou chaîne de caractères) à toutes les valeurs des cellules situées dans la zone de disque définie d'une grille. La valeur à ajouter doit être du même type que celle contenue dans les cellules de la grille, c'est-à-dire que vous ne pouvez pas ajouter une chaîne de caractères à un réel ou vice-versa. à un réel ou vice-versa, et pour les chaînes de caractères, cela correspond à une concaténation. Syntaxe : ds_grid_add_disk(index, xm, ym, r, val) ; Arguments Description indice L'index de la grille. xm La position x du disque sur la grille. ym La position y du disque sur la grille. r Le rayon du disque sur la grille. val La valeur à ajouter aux cellules du disque. Les retours : N/A Exemple : ds_grid_add_disk(grid, 7, 6, 5, 2) Cela ajouterait 2 à toutes les valeurs contenues dans les cellules de la zone de disque définie de la grille DS référencée par la variable "grid".
