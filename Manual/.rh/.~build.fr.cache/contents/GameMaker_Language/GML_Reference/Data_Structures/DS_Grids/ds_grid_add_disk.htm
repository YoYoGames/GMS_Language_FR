ds_grid_add_disk Cette fonction peut être utilisée pour ajouter une valeur donnée (réelle ou chaîne) à toutes les valeurs des cellules trouvées dans la zone de disque définie d'une grille. La valeur à ajouter doit être du même type que celle contenue dans les cellules de la grille, c'est-à-dire: vous ne pouvez pas ajouter une chaîne à un réel ou vice-versa, et pour les chaînes cela correspond à la concaténation. Syntaxe:ds_grid_add_disk(index, xm, ym, r, val); Argument Description index L'index de la grille. xm La position x du disque sur la grille. ym La position y du disque sur la grille. r Le rayon du disque sur la grille. val Valeur à ajouter aux cellules du disque. Retour:N/A Exemple:ds_grid_add_disk(grid, 7, 6, 5, 2) Cela ajouterait 2 à toutes les valeurs contenues dans les cellules dans la zone de disque définie de la grille DS référencée par la variable «grille».
