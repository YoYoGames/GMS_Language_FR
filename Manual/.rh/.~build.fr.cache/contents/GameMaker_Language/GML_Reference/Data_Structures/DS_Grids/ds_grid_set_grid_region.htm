ds_grid_set_grid_region Cette fonction peut être utilisée pour copier le contenu d'une zone rectangulaire de cellules de grille d'une grille DS (préalablement définie) à une autre, ou pour copier une région à l'intérieur de la même grille. Les images suivantes illustrent les deux façons d'utiliser cette fonction que cette fonction peut être utilisée : Syntaxe : ds_grid_set_grid_region(index, source, x1, y1, x2, y2, xpos, ypos) ; Arguments Description indice L'index de la grille de destination. source L'index de la grille source. x1 La position gauche de la région de cellules à copier depuis la grille source. y1 La position supérieure de la région de cellules à copier à partir de la grille source. x2 La position droite de la région de cellules à copier depuis la grille source. y2 La position inférieure de la région de cellules à copier depuis la grille source. xpos La position x sur la grille de destination à laquelle copier la région source. ypos La position y sur la grille de destination pour copier la région source. Les retours : N/A Exemple : ds_grid_set_grid_region(grid, t_grid, 0, 0, 5, 5, 10, 10) Ceci copierait la région des cellules de (0,0) à (5,5) de la grille indexée dans la variable "t_grid" et les copierait à la position (10,10) dans la grille indexée dans la variable "grid".
