ds_grid_get_mean Cette fonction peut être utilisée pour trouver la valeur moyenne de toutes les cellules trouvées dans la région définie d'une grille (toutes les valeurs des cellules sont additionnées puis divisées par le nombre total de cellules qui composent la région), comme le montre l'image ci-dessous : Syntaxe : ds_grid_get_mean(index, x1, y1, x2, y2) ; Arguments Description indice L'index de la grille. x1 La colonne de cellules de gauche de la région. y1 La rangée de cellules supérieure de la région. x2 La colonne de cellules de droite de la région. y2 La rangée de cellules inférieure de la région. Les retours : Réel ou Chaîne Exemple : val = ds_grid_get_mean(grid, 0, 0, 5, 5) Le code ci-dessus va définir la variable "val" à la valeur moyenne contenue dans la région donnée de la grille DS indexée dans la variable "grid".
