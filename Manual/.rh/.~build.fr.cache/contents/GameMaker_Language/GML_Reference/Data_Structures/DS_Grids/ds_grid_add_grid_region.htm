ds_grid_add_grid_region Cette fonction peut être utilisée pour ajouter toutes les valeurs de toutes les cellules trouvées dans la zone source d'une grille aux valeurs de la grille de destination, comme illustré ci-dessous: REMARQUE: vous pouvez également utiliser cette fonction sur la même grille pour ajouter des valeurs d'une région de la grille à celles stockées dans une autre (voir l'exemple de code ci-dessous). Syntaxe:ds_grid_add_grid_region(index, source, x1, y1, x2, y2, xpos, ypos); Argument Description index L'index de la grille de destination. source L'index de la grille source. x1 La position gauche de la région de cellules à copier à partir de la grille source. y1 La position supérieure de la région de cellules à copier à partir de la grille source. x2 La bonne position de la région de cellules à copier à partir de la grille source. y2 La position inférieure de la région de cellules à copier à partir de la grille source. xpos Position x sur la grille de destination à laquelle ajouter la région source. ypos La position y sur la grille de destination à laquelle ajouter la région source. Retour:N/A Exemple:ds_grid_add_grid_region(grid, grid, 0, 0, 1, 5, 2, 0) Le code ci-dessus copierait la région des cellules de (0,0) à (1,5) de la grille DS indexée dans la variable "grille" et les ajouterait aux cellules de la position (2,0) de la même grille DS.
