ds_grid_get_disk_mean Cette fonction peut être utilisée pour trouver la valeur moyenne de toutes les cellules se trouvant dans la zone du disque défini d'une grille (toutes les valeurs des cellules sont additionnées puis divisées par le nombre total de cellules qui composent le disque), comme le montre l'image ci-dessous : Syntaxe : ds_grid_get_disk_mean(index, xm, ym, r) ; Arguments Description indice L'index de la grille. xm La position x du disque sur la grille. ym La position y du disque sur la grille. r Le rayon du disque sur la grille. Les retours : Réel ou Chaîne Exemple : val = ds_grid_get_disk_mean(grid, 5, 5, 2) Le code ci-dessus va définir la variable "val" à la valeur moyenne contenue dans le disque donné de la grille DS indexée dans la variable "grid".
