ds_grid_shuffle Cette fonction peut être utilisée pour randomiser les positions de toutes les valeurs dans toutes les cellules d'une grille. REMARQUE: Cette fonction mélange les cellules de la grille aux mêmes positions chaque fois que le jeu est réexécuté car GameMaker Studio 2 génère la même graine aléatoire initiale à chaque fois pour rendre le code de débogage beaucoup plus facile. Pour éviter ce comportement, utilisez randomise() au début de votre partie. Cela n'est vrai que lors du test et du débogage du jeu, car le package exécutable final ne montrera pas ce comportement et sera aléatoire à chaque lecture. Syntaxe:ds_grid_shuffle(index); Argument Description index L'index de la grille à mélanger. Retour:N/A Exemple:ds_grid_shuffle(grid) Cela prendrait toutes les valeurs de la grille DS indexées dans la variable «grille» et les mélangerait pour obtenir un ordre différent.
