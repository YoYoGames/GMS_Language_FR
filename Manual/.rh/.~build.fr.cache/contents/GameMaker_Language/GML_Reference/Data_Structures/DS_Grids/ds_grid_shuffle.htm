ds_grid_shuffle Cette fonction peut être utilisée pour rendre aléatoire la position de toutes les valeurs dans toutes les cellules d'une grille. REMARQUE : cette fonction mélangera les cellules de la grille aux mêmes positions à chaque nouvelle exécution du jeu, car GameMaker Studio 2 génère à chaque fois la même graine aléatoire initiale afin de faciliter le débogage du code. Pour éviter ce ce comportement, utilisez randomise() au début de votre jeu. Ceci n'est vrai que lorsque vous testez et déboguez le jeu, car l'exécutable final ne présentera pas ce comportement et sera plus facile à utiliser. final ne présentera pas ce comportement et sera aléatoire à chaque jeu. Syntaxe : ds_grid_shuffle(index) ; Arguments Description indice L'index de la grille à mélanger. Les retours : N/A Exemple : ds_grid_shuffle(grille) Cela prendrait toutes les valeurs de la grille DS indexées dans la variable "grid" et les mélangerait pour obtenir un ordre différent.
