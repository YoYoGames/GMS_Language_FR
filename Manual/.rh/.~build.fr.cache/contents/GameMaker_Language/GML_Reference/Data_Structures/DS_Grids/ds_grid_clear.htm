ds_grid_clear Cette fonction peut être utilisée pour effacer une grille DS donnée à une valeur spécifique. Toutes les cellules de la grille contiendront alors cette valeur, qui peut être un nombre réel ou une chaîne. L'image ci-dessous illustre comment cela fonctionne: Syntaxe:ds_grid_clear(index, val); Argument Description index Cet index de la grille à effacer. val La nouvelle valeur de toutes les cellules de la grille. Retour:N/A Exemple:ds_grid_resize(global.Grid, room_width / 32, room_height / 32); ds_grid_clear(global.Grid, -1) Le code ci-dessus redimensionnera la grille DS indexée dans la variable globale "Grid", puis l'effacera afin que chaque cellule contienne la valeur -1.
