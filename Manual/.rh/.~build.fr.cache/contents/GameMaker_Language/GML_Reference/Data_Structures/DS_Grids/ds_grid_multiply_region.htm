ds_grid_multiply_region Cette fonction permet de spécifier une région de la grille dans laquelle la valeur de chaque cellule sera multipliée par une quantité donnée. REMARQUE : Cette fonction ne fonctionne qu'avec des nombres réels, pas avec des chaînes de caractères. Syntaxe : ds_grid_multiply_region(index, x1, y1, x2, y2, val) ; Arguments Description indice L'index de la grille. x1 La position x de la partie gauche de la région dans la grille. y1 La position y du haut de la région dans la grille. x2 La position x de la droite de la région dans la grille. y2 La position y du bas de la région dans la grille. val La valeur à multiplier avec les cellules de la région. Les retours : N/A Exemple : ds_grid_multiply_region( mygrid, 5, 5, 10, 10, 2 ) Le code ci-dessus prend toutes les valeurs trouvées dans la zone de grille rectangulaire définie et les multiplie par 2.
