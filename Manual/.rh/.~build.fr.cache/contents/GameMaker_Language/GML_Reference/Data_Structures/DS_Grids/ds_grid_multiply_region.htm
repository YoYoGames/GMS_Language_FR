ds_grid_multiply_region Avec cette fonction, vous pouvez spécifier une région de la grille dans laquelle multiplier chaque valeur de cellule par un montant donné. REMARQUE: Cette fonction ne fonctionnera qu'avec des nombres réels, pas des chaînes. Syntaxe:ds_grid_multiply_region(index, x1, y1, x2, y2, val); Argument Description index L'index de la grille. x1 La position x de la gauche de la région dans la grille. y1 La position y du haut de la région dans la grille. x2 La position x de la droite de la région dans la grille. y2 La position y du bas de la région dans la grille. val La valeur à multiplier avec les cellules de la région. Retour:N/A Exemple:ds_grid_multiply_region( mygrid, 5, 5, 10, 10, 2 ) Le code ci-dessus prendra toutes les valeurs trouvées dans la zone de grille rectangulaire définie et les multipliera par 2.
