ds_grid_multiply Cette fonction multipliera la valeur d'une cellule de grille donnée par le montant spécifié. REMARQUE: Cette fonction ne fonctionnera qu'avec des nombres réels, pas des chaînes. Syntaxe:ds_grid_multiply(index, x, y, val); Argument Description index L'index de la grille. x La position x de la cellule dans la grille. y La position y de la cellule dans la grille. val La valeur à multiplier avec la cellule. Retour:N/A Exemple:ds_grid_multiply(mygrid, 5, 5, 2) Le code ci-dessus multipliera par 2 la valeur stockée dans la cellule de grille DS donnée.
