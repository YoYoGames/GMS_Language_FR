ds_grid_multiply Cette fonction multiplie la valeur d'une cellule de la grille donnée par le montant spécifié. REMARQUE : Cette fonction ne fonctionne qu'avec des nombres réels, pas avec des chaînes de caractères. Syntaxe : ds_grid_multiply(index, x, y, val) ; Arguments Description indice L'index de la grille. x La position x de la cellule dans la grille. y La position y de la cellule dans la grille. val La valeur à multiplier avec la cellule. Les retours : N/A Exemple : ds_grid_multiply(mygrid, 5, 5, 2) Le code ci-dessus va multiplier par 2 la valeur stockée dans la cellule de la grille DS.
