ds_grid_create Cette fonction vous permet de créer une nouvelle structure de données DS grid avec la largeur et la hauteur de cellule spécifiées. Cette fonction renvoie un id qui doit être utilisé dans toutes les autres fonctions qui traitent de cette grille DS. IMPORTANT ! Lorsque vous créez une structure de données, la valeur de l'index pour l'identifier est une valeur entière commençant à 0. Cela signifie que des structures de données de différents types peuvent avoir la même valeur d'index, donc en cas de doute, vous devriez utiliser l'attribut ds_exists() avant d'y accéder. Notez également que les indices sont réutilisés, donc une valeur d'indice de structure de données détruite peut être utilisée par une structure nouvellement créée par la suite. Syntaxe : ds_grid_create(w, h) ; Arguments Description w La largeur de la grille à créer. h La hauteur de la grille à créer. Les retours : Real Exemple : mygrid = ds_grid_create(10, 10) Cela crée une grille de 10 cellules de haut et de 10 cellules de large.
