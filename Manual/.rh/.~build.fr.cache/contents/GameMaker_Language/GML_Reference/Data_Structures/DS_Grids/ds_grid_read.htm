ds_grid_read Cette fonction peut être utilisée pour reconvertir une chaîne de caractères créée précédemment par la fonction ds_grid_write() en une grille DS. La grille DS doit avoir été créée précédemment (voir l'exemple ci-dessous). Notez que si la structure de données a été créée avec des versions antérieures de GameMaker vous devez ajouter l'argument optionnel "legacy", en le mettant à true car le format de la chaîne a changé avec cette version. Syntaxe : ds_grid_read(index, string [, legacy]) ; Arguments Description indice L'index de la grille à lire. chaîne de caractères La chaîne à lire dans la grille DS. héritage(facultatif) Peut être soit vrai, soit faux, soit complètement omis. Les retours : N/A Exemple : grid = ds_grid_create(room_width div 32, room_height div 32) ; ini_open("Save.ini") ; ds_grid_read(grid, ini_read_string("Save", "0", "")) ; ini_close() Le code ci-dessus crée une grille DS basée sur la taille de la pièce (chaque carré de pixels 32x32 représente une cellule de la grille) et lit ensuite un ensemble de données de grille précédemment sauvegardées dans un fichier ini dans la nouvelle grille DS.
