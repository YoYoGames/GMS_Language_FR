ds_grid_sort Cette fonction peut être utilisée pour trier une grille DS sur la base des valeurs d'une colonne donnée (de la même manière que vous trieriez des fichiers par date, taille etc... dans l'explorateur de fichiers du système d'exploitation). L'image suivante montre un exemple : Syntaxe : ds_grid_sort(index, colonne, ascendant) ; Arguments Description indice L'index de la grille à trier. colonne La colonne à utiliser pour trier les lignes ascendant Si vous voulez trier du plus bas au plus haut(true), ou du plus haut au plus bas(false). Les retours : N/A Exemple : ds_grid_sort(grid, 3, false) Cela prendrait toutes les valeurs de la grille DS indexées dans la variable "grid" et les trierait en fonction des valeurs trouvées dans la 3ème colonne de la grille (comme indiqué dans l'image ci-dessus).
