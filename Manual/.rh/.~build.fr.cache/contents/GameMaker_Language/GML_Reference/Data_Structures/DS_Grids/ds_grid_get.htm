ds_grid_get Cette fonction peut être utilisée pour obtenir la valeur (un nombre réel ou une chaîne) à partir de n'importe quelle cellule de la grille DS donnée. Si vous transmettez des coordonnées de grille invalides à la fonction, la valeur renvoyée sera undefined et une erreur sera affichée dans la fenêtre de sortie. Syntaxe:ds_grid_get(index, x, y); Argument Description index L'index de la grille. x La position x de la cellule dont vous souhaitez trouver la valeur. y La position y de la cellule dont vous souhaitez trouver la valeur. Retour:Real or String Exemple:var xx = irandom(ds_grid_width(grid) - 1); var yy = irandom(ds_grid_height(grid) - 1); val = ds_grid_get(grid, xx, yy) Le code ci-dessus sélectionne une cellule aléatoire de la grille DS indexée dans la variable «grille» et stocke sa valeur dans la variable «val».
