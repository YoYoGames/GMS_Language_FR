ds_grid_copy Cette fonction permet de copier le contenu d'une grille dans une autre. Les deux grilles doivent avoir été créées au préalable à l'aide de la fonction ds_grid_create() fonction. Syntaxe : ds_grid_copy(destination, source) ; Arguments Description destination Cet index de la grille à copier. source Cet index de la grille à copier. Les retours : N/A Exemple : n_grid = ds_grid_create(ds_grid_width(a_grid), ds_grid_height(a_grid)) ; ds_grid_copy(n_grid, a_grid) ; ds_grid_clear(a_grid, -1) Le code ci-dessus crée une nouvelle grille DS, basée sur la largeur et la hauteur d'une grille précédemment créée, puis copie les informations de la grille précédente vers la nouvelle. Enfin, il efface l'ancienne grille afin que toutes les cellules aient une valeur de -1. Retour : Grilles DS Suivant : ds_grid_read Copyright YoYo Games Ltd. 2021 Tous droits réservés
