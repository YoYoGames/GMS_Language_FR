ds_grid_copy Avec cette fonction, vous pouvez copier le contenu d'une grille dans une autre. Les deux grilles doivent avoir été créées au préalable à l'aide du ds_grid_create() fonction. Syntaxe:ds_grid_copy(destination, source); Argument Description destination Cet index de la grille vers laquelle copier. source Cet index de la grille à partir de laquelle copier. Retour:N/A Exemple:n_grid = ds_grid_create(ds_grid_width(a_grid), ds_grid_height(a_grid)); ds_grid_copy(n_grid, a_grid); ds_grid_clear(a_grid, -1) Le code ci-dessus crée une nouvelle grille DS, basée sur la largeur et la hauteur d'une grille précédemment créée, puis copie les informations de la grille précédente dans la nouvelle.
