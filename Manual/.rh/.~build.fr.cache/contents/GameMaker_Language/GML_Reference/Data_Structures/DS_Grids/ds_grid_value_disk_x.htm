ds_grid_value_disk_x Cette fonction permet d'obtenir la coordonnée x (dans la région en forme de disque de la grille donnée) de la valeur recherchée. Vous donnez l'index de la grille DS (tel que retourné par ds_grid_create()) ainsi que les positions x/y pour la cellule centrale du disque. Ensuite, vous indiquez le rayon (sous forme de valeur entière) autour de la cellule centrale à rechercher, avant de fournir la valeur à rechercher. Si la valeur recherchée n'existe pas, la fonction renvoie -1. Syntaxe : ds_grid_value_disk_x(index, xm, ym, r, val) ; Arguments Description indice L'index de la grille. xm La position x du disque sur la grille. ym La position y du disque sur la grille. r Le rayon du disque sur la grille. val La valeur à trouver. Les retours : Real Exemple : if ds_grid_value_disk_exists(grid, 5, 5, 5, val) { xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val) ; ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val) ; } Le code ci-dessus vérifie une ds_grid pour une valeur spécifique dans une région du disque. Si elle est trouvée, il stocke alors la position x et y de la valeur dans deux variables pour une utilisation ultérieure.
