ds_queue_tail Cette fonction ne lira que la dernière valeur de la queue (celle qui est "à la queue"). Elle ne déchargera pas la valeur de la file, ce qui signifie qu'elle pourra toujours être lue dans le futur par cette fonction ou par la fonction ds_queue_dequeue(). Syntaxe : ds_queue_tail(id) ; Arguments Description id L'identifiant de la structure de données à lire. Les retours : Valeur du type de données stockée dans la file d'attente Exemple : num = ds_queue_tail(control_queue) ; Le code ci-dessus va lire la valeur de la file d'attente indexée dans la variable "control_queue" et stocker la valeur de retour dans la variable "num".
