ds_queue_create Cette fonction crée une nouvelle structure de données de file d'attente et renvoie la valeur de l'index. Cette valeur doit être stockée dans une variable et utilisée dans tous les autres appels de fonction relatifs à la file d'attente. IMPORTANT ! Lorsque vous créez une structure de données, la valeur de l'index pour l'identifier est une valeur entière commençant à 0. Cela signifie que des structures de données de types différents peuvent avoir la même valeur d'index, donc en cas de doute, vous devriez utiliser la fonction ds_exists() avant d'y accéder. Notez également que les indices sont réutilisés, de sorte qu'une valeur d'indice de structure de données détruite peut être utilisée par une structure nouvellement créée par la suite. Syntaxe : ds_queue_create() ; Les retours : Real Exemple : queue = ds_queue_create() ; Cela créera une nouvelle file d'attente et assignera son id d'index à la variable d'instance "queue".
