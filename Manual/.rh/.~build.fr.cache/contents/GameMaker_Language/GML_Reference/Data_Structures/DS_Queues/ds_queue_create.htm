ds_queue_create Cette fonction créera une nouvelle structure de données de file d'attente et retournera la valeur d'index. Cette valeur doit être stockée dans une variable et utilisée dans tous les autres appels de fonction relatifs à la file d'attente. IMPORTANT! Lorsque vous créez une structure de données, la valeur d'index pour l'identifier est une valeur entière commençant à 0. Cela signifie que les structures de données de différents types peuvent avoir la même valeur d'index, donc en cas de doute, vous devriez utiliser la fonction ds_exists () avant y accéder. Notez également que les index sont réutilisés, de sorte qu'une valeur d'index de structure de données détruite peut être utilisée par une valeur nouvellement créée par la suite. Syntaxe:ds_queue_create(); Retour:Real Exemple:queue = ds_queue_create(); Cela créera une nouvelle file d'attente et attribuera son identifiant d'index à la variable d'instance "queue".
