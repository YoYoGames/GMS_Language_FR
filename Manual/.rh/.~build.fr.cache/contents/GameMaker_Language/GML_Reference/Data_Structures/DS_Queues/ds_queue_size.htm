ds_queue_size Cette fonction renverra la "taille" de la file d'attente, c'est-à-dire: le nombre d'éléments qui y ont été mis en file d'attente. Syntaxe:ds_queue_size(id); Argument Description id L'identifiant de la structure de données à vérifier. Retour:Real Exemple:if !ds_queue_empty(control_queue)     {     num = ds_queue_size(control_queue);     } Le code ci-dessus vérifie une file d'attente DS pour voir si elle est vide ou non, et si ce n'est pas le cas, il obtient le nombre d'éléments qu'elle contient et stocke la valeur dans une variable.
