ds_queue_enqueue Cette fonction ajoute une valeur (réelle ou chaîne) à la queue de la file d'attente DS. La fonction peut prendre 14 autres arguments facultatifs (soit un total de 15 ajouts possibles), ce qui vous permet d'ajouter plusieurs valeurs consécutivement à la queue de la file d'attente en un seul appel. valeurs consécutives à la queue de la file d'attente en un seul appel. Syntaxe : ds_queue_enqueue(id, val [, val2, ... val15]) ; Arguments Description id L'identifiant de la file d'attente à ajouter. val La valeur à ajouter à la file d'attente. [val2, ... val15] Valeurs facultatives à ajouter à la file d'attente. Les retours : N/A Exemple : move_queue = ds_queue_create() ; ds_queue_enqueue(move_queue, x + 200) ; ds_queue_enqueue(move_queue, y) ; ds_queue_enqueue(move_queue, x + 200) ; ds_queue_enqueue(move_queue, y + 200) ; ds_queue_enqueue(move_queue, x) ; ds_queue_enqueue(move_queue, y + 200) ; ds_queue_enqueue(move_queue, x) ; ds_queue_enqueue(move_queue, y) ; Le code ci-dessus crée une nouvelle file d'attente DS et stocke son index dans la variable "move_queue". Il pousse ensuite un certain nombre de valeurs dans la file d'attente pour une utilisation future.
