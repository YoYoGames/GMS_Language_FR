ds_stack_size Cette fonction retournera la "taille" de la pile, c'est-à-dire: le nombre d'éléments qui y ont été poussés. Syntaxe:ds_stack_size(id); Argument Description id L'identifiant de la structure de données à vérifier. Retour:Real Exemple:if !ds_stack_empty(control_stack)     {     num = ds_stack_size(control_stack);     } Le code ci-dessus vérifie une pile DS pour voir si elle est vide ou non, et si ce n'est pas le cas, il obtient le nombre d'éléments qu'elle contient et stocke la valeur dans une variable.
