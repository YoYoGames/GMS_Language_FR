ds_stack_pop Cette fonction pop la valeur supérieure de la pile hors DS, retirer de la pile et en retournant la valeur à stocker dans une variable. Si la pile est vide, la fonction retournera la constante undefined, sinon il renverra la valeur réelle ou chaîne contenue dans la pile. Syntaxe:ds_stack_pop(id); Argument Description id L'identifiant de la structure de données à partir de laquelle apparaître. Retour:Data type value that is stored in the stack or undefined Exemple:if !ds_stack_empty(move_stack)     {     var xx = ds_stack_pop(move_stack);     var yy = ds_stack_pop(move_stack);     move_towards_point(xx, yy, 4);     } Le code ci-dessus vérifie la pile DS indexée dans la variable "move_stack" pour voir si elle est vide, et si ce n'est pas le cas, il fait apparaître les deux premières valeurs de la pile et les utilise pour définir une direction pour le mouvement.
