ds_stack_empty Avec cette fonction, vous pouvez vérifier la pile DS donnée pour voir si elle est vide (retourne true ) ou non (retourne false ). Syntaxe:ds_stack_empty(id); Argument Description id L'identifiant de la structure de données à vérifier. Retour:Boolean Exemple:if (ai_count == 15) && (!ds_stack_empty(AI_stack))     {     ds_stack_clear(AI_stack);     alarm[0] = room_speed;     ai_count = 0;     } Le code ci-dessus vérifie une variable pour voir si elle a atteint une valeur spécifique et si elle l'a efface la pile DS indexée dans la variable "AI_stack", définit une alarme et réinitialise la variable à 0.
