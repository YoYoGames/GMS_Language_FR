ds_stack_clear Avec cette fonction, vous pouvez effacer toutes les données de la structure de données de la pile donnée. Cela ne détruit PAS la structure de données (pour cela, vous devez utiliser la fonction ds_stack_destroy()), elle efface seulement toutes les données de celle-ci et renvoie une pile vide. Syntaxe : ds_stack_clear(id) ; Arguments Description id L'identifiant de la structure de données à effacer. Les retours : N/A Exemple : if (ai_count = 15) && (!ds_stack_empty(AI_stack)) { ds_stack_clear(AI_stack) ; alarm[0] = room_speed ; ai_count = 0 ; } Le code ci-dessus vérifie une variable pour voir si elle a atteint une valeur spécifique et si c'est le cas, il efface la pile DS indexée dans la variable "AI_stack", déclenche une alarme et remet la variable à 0.
