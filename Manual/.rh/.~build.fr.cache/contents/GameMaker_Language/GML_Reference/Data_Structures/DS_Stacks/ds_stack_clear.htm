ds_stack_clear Avec cette fonction, vous pouvez effacer toutes les données de la structure de données de pile donnée. Cela ne détruit PAS la structure de données (pour cela, vous devez utiliser ds_stack_destroy() ), il efface uniquement toutes les données et renvoie une pile vide. Syntaxe:ds_stack_clear(id); Argument Description id L'identifiant de la structure de données à effacer. Retour:N/A Exemple:if (ai_count = 15) && (!ds_stack_empty(AI_stack))     {     ds_stack_clear(AI_stack);     alarm[0] = room_speed;     ai_count = 0;     } Le code ci-dessus vérifie une variable pour voir si elle a atteint une valeur spécifique et si elle l'a efface la pile DS indexée dans la variable "AI_stack", définit une alarme et réinitialise la variable à 0.
