ds_stack_push Cette fonction poussera (ajoutera) une valeur, qui peut être de n'importe quel type de données, en haut de la pile. La fonction peut prendre 14 arguments optionnels supplémentaires (soit un total de 15 ajouts possibles), vous permettant de pousser plusieurs valeurs consécutivement dans la pile en un seul appel. Syntaxe:ds_stack_push(id, val [, val2, ... val15]); Argument Description id L'identifiant de la structure de données sur laquelle pousser. val La valeur à pousser sur la pile. [val2,... val13] Valeurs facultatives à ajouter à la pile. Retour:N/A Exemple:move_stack = ds_stack_create(); ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y); Le code ci-dessus crée une nouvelle pile DS et stocke son index dans la variable "move_stack".
