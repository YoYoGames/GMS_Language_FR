ds_stack_write Cette fonction renvoie une chaîne de caractères qui peut ensuite être stockée ou transférée dans une autre structure de données à l'aide de la fonction ds_stack_read() fonction. NOTE: La chaîne retournée n'est pas une chaîne lisible par l'homme, mais plutôt un vidage du contenu de la structure de données. Syntaxe : ds_stack_write(id) ; Arguments Description id L'identifiant de la structure de données à écrire. Les retours : Chaîne de caractères Exemple : ini_open("save.ini") ; var str = ds_stack_write(stack) ; ini_write_string("Stacks", "0", str) ; ds_stack_clear(stack) ; ini_close() ; Le code ci-dessus ouvre un fichier ini puis écrit une chaîne contenant les informations stockées dans la pile DS indexée dans la variable "stack".
