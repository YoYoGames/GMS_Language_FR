ds_map_replace_list Avec cette fonction, vous pouvez remplacer une liste de DS qui a été stockée dans la "clé" donnée par une autre liste qui a été créée précédemment. Cette fonction est conçue pour créer des cartes compatibles avec JSON que que vous encoderez ensuite à l'aide de json_encode() et ne doit être utilisée qu'en conjonction avec cette fonctionnalité. Syntaxe : ds_map_replace_list(id, key, value) Arguments Description id L'identifiant de la carte à utiliser. clé La clé du remplacement. valeur L'id de la ds_list à utiliser pour remplacer celle précédemment stockée dans la clé donnée. Les retours : N/A Exemple : var j_list = ds_list_create() ; ds_list_add(j_list, health) ; ds_list_add(j_list, lives) ; ds_list_add(j_list, score) ; ds_map_replace_list(j_map, "list", j_list) ; var j = json_encode(j_map) ; ds_list_destroy(j_list) ; Le code ci-dessus va créer une liste DS et la remplir avec les valeurs de diverses variables globales avant de remplacer une liste précédemment stockée dans la carte DS "j_map".
