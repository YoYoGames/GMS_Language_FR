ds_map_replace Avec cette fonction, vous pouvez modifier la valeur de la clé donnée dans une carte DS. Vous fournissez l'index à la carte (tel que renvoyé par la fonctionds_map_create() ), puis la clé à remplacer - une chaîne ou un entier - et la valeur par laquelle remplacer la valeur de la clé. Si la clé donnée n'existe pas, elle sera créée pour vous, et si c'est le cas, la valeur actuelle sera remplacée par la nouvelle valeur. La fonction retournera true si la clé existe et que la valeur est remplacée, et false si la clé n'existe pas et qu'une nouvelle clé a été créée avec la valeur. Syntaxe:ds_map_replace( id, key, val ); Argument Description id L'identifiant de la carte à modifier. key La clé avec la valeur qui doit être remplacée par la nouvelle val La nouvelle valeur pour remplacer la valeur donnée par Retour:Boolean Exemple:ds_map_replace(inventory, "torso", 55); Le code ci-dessus recherche la carte DS pour la clé "torse" et quand il la trouve (ou elle est créée si elle n'existe pas), la valeur actuelle est remplacée par celle spécifiée.
