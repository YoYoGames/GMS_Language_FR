ds_map_is_list Avec cette fonction, vous pouvez vérifier si une liste de DS est stockée dans la clé de carte donnée. Si la clé donnée contient un ID de liste de DS, alors la fonction renverra true, sinon elle renverra false. Syntaxe : ds_map_is_list(id, key) Arguments Description id L'id de la ds_map à utiliser. clé La clé pour vérifier. Les retours : Booléen Exemple : var size = ds_map_size(inventory) ; var key = ds_map_find_first(inventory) ; for (var i = 0 ; i < size ; i++ ;) { if ds_map_is_list(inventory, key) { ds_list_destroy(key) ; } key = ds_map_find_next(inventory) ; } ds_map_destroy(inventory) ; Le code ci-dessus boucle sur une carte DS et vérifie si l'une des clés qu'elle contient correspond à une liste DS. Si c'est le cas, alors la liste DS est détruite, et à la fin de la boucle, la carte DS est également détruite.
