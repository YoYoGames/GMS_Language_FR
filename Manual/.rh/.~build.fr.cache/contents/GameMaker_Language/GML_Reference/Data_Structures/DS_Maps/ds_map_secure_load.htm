ds_map_secure_load Cette fonction chargera une carte DS précédemment enregistrée à partir du fichier donné. Ce fichier doit avoir été précédemment créé à l'aide de la fonction ds_map_secure_save() et, une fois chargé, la fonction renverra l'indice de la carte DS qui a été créée à partir des données chargées. Cet index de carte DS doit être stocké dans une variable et utilisé pour tous les autres appels de fonction à cette carte. Notez que si la carte DS en cours de chargement a été sauvegardée avec un tableau comme l'une des valeurs clés, ce tableau aura été enregistré dans la base de données. ce tableau aura été converti en une liste DS lors du chargement. IMPORTANT ! L'une des caractéristiques d'un fichier sauvegardé sécurisé est qu'il est verrouillé sur l'appareil sur lequel il a été créé. Vous ne pouvez donc pas charger un fichier sauvegardé sur un appareil dans un projet exécuté sur un autre appareil. Syntaxe : ds_map_secure_load(nom du fichier) ; Arguments Description nom de fichier Le nom du fichier à partir duquel charger les données cartographiques. Les retours : Entier (index de la carte DS) Exemple : p_map = ds_map_secure_load("p_data.dat") ; Le code ci-dessus chargera une carte DS sauvegardée de manière sécurisée et stockera sa valeur d'index dans une variable pour une utilisation future.
