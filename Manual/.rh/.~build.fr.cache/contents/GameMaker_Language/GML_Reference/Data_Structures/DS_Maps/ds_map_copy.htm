ds_map_copy Vous pouvez utiliser cette fonction pour copier le contenu d'une carte dans une autre que vous avez préalablement créée à l'aide de ds_map_create(). Si la carte DS vers laquelle la copie est effectuée n'est pas vide, cette fonction la vide d'abord avant de la copier. La carte DS d'origine reste inchangée par ce processus. Syntaxe : ds_map_copy(id, source) ; Arguments Description id L'identifiant de la carte que vous copiez . source L'identifiant de la carte à partir de laquellevous copiez. Les retours : N/A Exemple : inventaire_2 = ds_map_create() ; ds_map_copy(inventaire_2, inventaire_1) ; Le code ci-dessus va créer une nouvelle carte et l'affecter à la variable "inventory_2". Il copiera ensuite le contenu de la carte DS indexée dans la variable "inventory_1" dans cette nouvelle carte.
