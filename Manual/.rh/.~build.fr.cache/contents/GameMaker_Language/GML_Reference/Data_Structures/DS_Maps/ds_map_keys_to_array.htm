ds_map_keys_to_array Avec cette fonction, vous pouvez récupérer toutes les clés contenues dans une carte DS. Vous fournissez l'ID de carte DS à partir duquel obtenir les clés (tel que renvoyé pards_map_create() ) et la fonction renverra un tableau GML _Overview / Arrays.htm "> où chaque entrée du tableau est une clé de la carte DS. La fonction a un deuxième argument facultatif dans lequel vous pouvez fournir un tableau que vous avez créé, dans lequel cas les données de la clé de mappage seront ajoutées à toutes les données existantes dans le tableau. Notez que la fonction modifiera le tableau fourni directement, mais retournera également une référence à celui-ci (ou une référence à un nouveau tableau si aucun n'est fourni). Syntaxe:ds_map_keys_to_array(id, [array]) Argument Description id L'identifiant de la carte à utiliser. [array] OPTIONNEL! Tableau auquel ajouter les clés de mappage DS. Retour:Array Exemple:map_keys = ds_map_keys_to_array(inventory); Le code ci-dessus récupère les clés d'une carte DS, puis les stocke sous forme de tableau pour une utilisation future.
