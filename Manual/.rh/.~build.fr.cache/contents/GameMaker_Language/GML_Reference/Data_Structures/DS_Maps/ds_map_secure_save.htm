ds_map_secure_save Cette fonction enregistre le contenu de la carte DS donnée dans un fichier lié à l'appareil sur lequel elle a été créée (ce qui signifie qu'il ne peut pas être lu s'il est transféré vers un autre appareil). Le fichier lui-même peut avoir presque n'importe quelle extension (par exemple, *.dat, *.json, *.bin, etc...) et sera masquée et stockée dans le stockage local sur la plate-forme cible. Vous pouvez ensuite recharger le ds_map en utilisant la fonction ds_map_secure_load(). Notez que si la carte DS en cours d'enregistrement contient un tableau, ce tableau sera converti en liste DS à la place lors de l'enregistrement. IMPORTANT! L'une des caractéristiques d'un fichier enregistré sécurisé est qu'il est verrouillé sur l'appareil sur lequel il a été créé, de sorte que vous ne pouvez pas charger un fichier enregistré sur un appareil dans un projet exécuté sur un autre appareil. Syntaxe:ds_map_secure_save(map, filename); Argument Description map L'identifiant de la structure de données à utiliser filename Le nom du fichier dans lequel enregistrer la carte Retour:Boolean Exemple:ds_map_secure_save(purchase_map, "p_data.dat"); Le code ci-dessus enregistrera la carte DS indexée dans la variable "p_data" dans le fichier donné pour une récupération ultérieure.
