ds_map_secure_save_buffer Cette fonction permet de sauvegarder une carte DS précédemment créée dans un tampon. Vous fournissez la valeur de l'ID de la carte DS (telle que retournée par la fonction ds_map_create()) et l'ID du tampon dans lequel vous souhaitez écrire (tel que renvoyé par la fonction buffer_create()). Notez que si la carte DS à sauvegarder contient un tableau, celui-ci sera converti en une liste DS lors de la sauvegarde. IMPORTANT ! Le fichier de carte DS sauvegardé en toute sécurité ne peut être chargé que sur l'appareil qui l'a créé. Si vous essayez de charger un fichier qui a été sauvegardé sur un autre appareil, il ne fonctionnera pas. Syntaxe : ds_map_secure_save_buffer(filename) ; Arguments Description id La valeur de l'ID de la carte DS. tampon La valeur de l'ID du tampon. Les retours : Entier (index de la carte DS) Exemple : buff = buffer_create(128,buffer_grow,4) ; var map = ds_map_create() ; ds_map_add(map, "bob", "ajob") ; ds_map_add(map, "money",10) ; ds_map_secure_save_buffer(map, buff) ; ds_map_destroy(map) ; Le code ci-dessus va créer un tampon et une carte DS, puis remplir la carte avec des valeurs et les écrire dans le tampon avant de supprimer la carte.
