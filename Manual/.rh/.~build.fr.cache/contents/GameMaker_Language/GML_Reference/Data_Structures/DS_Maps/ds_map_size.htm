ds_map_size Cette fonction vous permet de savoir combien de paires clé/valeur la carte DS (précédemment créée) contient. Syntaxe : ds_map_size(id) ; Arguments Description id L'identifiant de la structure de données à vérifier. Les retours : Real Exemple : si (ds_map_size(inventory) > 49) { full = true ; } Le code ci-dessus vérifie la taille de la carte DS (c'est-à-dire le nombre de paires clé/valeur) et s'il est supérieur à 49, il met la variable "full" à true.
