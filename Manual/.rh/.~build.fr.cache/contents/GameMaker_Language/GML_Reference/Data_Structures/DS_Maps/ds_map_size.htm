ds_map_size Avec cette fonction, vous pouvez trouver le nombre de paires clé / valeur que contient la carte DS (créée précédemment). Syntaxe:ds_map_size(id); Argument Description id L'identifiant de la structure de données à vérifier. Retour:Real Exemple:if (ds_map_size(inventory) > 49)    {    full = true;    } Le code ci-dessus vérifiera la taille de la carte DS (c'est-à-dire le nombre de paires clé / valeur) et si elle est supérieure à 49, il définit la variable "full" sur true.
