ds_map_find_next Cette fonction renvoie la clé suivante stockée dans la carte DS après celle spécifiée dans la fonction. Cela peut être utile si vous devez itérer dans la carte DS à la recherche de quelque chose, mais devrait être évité si possible car cela peut être lent. Si une telle clé n'existe pas, la fonction retournera undefined. Vous devriez toujours vérifier cela en utilisant la fonction is_undefined() fonction. Syntaxe : ds_map_find_next(id, key) ; Arguments Description id L'identifiant de la carte à utiliser. clé La clé pour trouver le prochain. Les retours : Réel, chaîne de caractères ou indéfini Exemple : var size = ds_map_size(inventory) ; var key = ds_map_find_first(inventory) ; for (var i = 0 ; i < size ; i++ ;) { if (key != "gold") { key = ds_map_find_next(inventory, key) ; } else break ; } Le code ci-dessus crée quelques variables temporaires, puis obtient la taille de la carte DS et trouve la première clé stockée par l'ordinateur dans la carte. Il utilise ensuite une boucle for pour itérer dans la carte DS à la recherche de la valeur clé "gold". Si Si elle est trouvée, la boucle est interrompue.
