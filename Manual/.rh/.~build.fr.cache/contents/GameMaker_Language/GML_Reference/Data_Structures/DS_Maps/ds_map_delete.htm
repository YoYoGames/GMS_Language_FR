ds_map_delete Cette fonction vous permet de supprimer une clé donnée (et sa valeur correspondante) de la carte DS donnée, précédemment créée. Syntaxe : ds_map_delete(id, key) ; Arguments Description id L'identifiant de la carte à modifier. clé La clé (ainsi que sa valeur associée) à supprimer. Les retours : N/A Exemple : ds_map_delete(inventory, "shield") ; Le code ci-dessus supprimera la clé "shield" (et la valeur à laquelle elle est associée) de la carte DS (inventaire).
