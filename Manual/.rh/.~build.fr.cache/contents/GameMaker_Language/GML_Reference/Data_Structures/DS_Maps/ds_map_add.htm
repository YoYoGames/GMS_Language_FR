ds_map_add Cette fonction doit être utilisée pour ajouter des ensembles de paires clé/valeur dans la carte DS spécifiée. Vous pouvez vérifier cette fonction pour savoir si elle a réussi ou non (elle renvoie true en cas de succès ou false sinon), car elle peut échouer s'il existe déjà la même clé dans la carte DS ou si vous spécifiez une carte DS inexistante comme ID de la carte à ajouter. Les clés et les valeurs que vous fournissez peuvent être constituées de n'importe quelle combinaison de types de données. plus - sont acceptables (bien que, dans la pratique, vous utilisiez le plus souvent une chaîne de caractères pour la clé) : ds_map_add(map, 5, undefined) ; ds_map_add(map, "level", 1) ; ds_map_add(map, 89.6, "hello world") ; ds_map_add(map, 5, buffer_get_address(buff)) ; Vous pouvez également ajouter des éléments à une carte en utilisant l'accesseur " ?", comme indiqué ci-dessous : map[ ? 5] = undefined ; map[ ? "level"] = 1 ; map[ ? 89.6] = "hello world" ; map[ ? 5] = buffer_get_address(buff) ; REMARQUE: contrairement à d'autres structures de données de GameMaker Studio 2, cette clé n'ira pas au début (ni à la fin) de la carte DS, mais elle ira simplement dans la carte DS quelque part. Syntaxe : ds_map_add(id, key, val) ; Arguments Description id L'identifiant de la carte à ajouter. clé La clé de la valeur à ajouter. val La valeur à ajouter à la carte. Les retours : Booléen Exemple : inventory = ds_map_create() ; ds_map_add(inventory, "hp potion", 1) ; ds_map_add(inventory, "gold", 100) ; Cela va créer une nouvelle carte, en assignant son id à la variable "inventaire". Elle ajoute ensuite deux nouvelles clés à la carte, "hp potion" et "gold", et fixe leurs valeurs initiales à 1 et 100.
