ds_map_values_to_array Avec cette fonction, vous pouvez récupérer toutes les valeurs qu'une carte DS contient pour chaque clé de la carte. Vous fournissez l'ID de carte DS à partir duquel obtenir les clés (tel que renvoyé pards_map_create() ) et la fonction renverra un tableau où chaque entrée du tableau est une valeur de la carte DS (par exemple, si la carte a 10 clés, alors un tableau de 10 entrées sera renvoyé avec chaque entrée correspondant à une valeur pour un clé). La fonction a un deuxième argument facultatif dans lequel vous pouvez fournir un tableau que vous avez créé, auquel cas les données de valeur de clé de mappage seront ajoutées à toutes les données existantes dans le tableau. Notez que la fonction modifiera directement le tableau fourni, mais retournera également une référence à celui-ci (ou une référence à un nouveau tableau si aucun n'est fourni). Syntaxe:ds_map_values_to_array(id, [array]) Argument Description id L'identifiant de la carte à utiliser. [array] OPTIONNEL! Tableau auquel ajouter les valeurs de mappage DS. Retour:Array Exemple:var _values = ds_map_values_to_array(lvl_score); var _total = 0; for (var i = 0; i < array_lengh(values); ++i;)     {     _total += _values[i];     } draw_text(32, 32, "Total score for all levels = " + string(_total")); Ce qui précède récupère toutes les valeurs d'une carte DS pour chaque clé de la carte et les stocke dans une variable locale sous forme de tableau. Il parcourt ensuite le tableau et ajoute chacune des valeurs à une variable locale qui est ensuite utilisée pour afficher le total de toutes les valeurs.
