ds_map_set Cette fonction permet de définir la valeur d'une clé dans une carte DS donnée. Vous fournissez la valeur de l'ID de la carte DS (telle que retournée par la fonction ds_map_create()), puis vous donnez la clé que vous voulez définir et la valeur à laquelle la définir valeur à définir. Les clés peuvent être des entiers ou des chaînes de caractères, et si la clé donnée n'existe pas, elle sera créée pour vous et définie par la valeur. Cette fonction est identique à l'utilisation de l'accesseur DS map pour définir/créer une paire clé/valeur de map couple clé/valeur. La fonction ne renvoie rien, donc si vous devez vérifier si la valeur de la clé a été remplacée ou si une nouvelle clé a été créée, vous devez utiliser la fonction ds_map_replace(). Syntaxe : ds_map_set(id, key, value) Arguments Description id L'identifiant de la carte à utiliser. clé La clé du jeu. valeur La valeur à donner à la clé. Les retours : N/A Exemple : if is_undefined(ds_map_find_value(map, "score")) { ds_map_set(map, "score", 0) ; } Le code ci-dessus vérifiera si la clé donnée existe et si elle n'existe pas, elle sera créée et définie.
