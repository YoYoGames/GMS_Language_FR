ds_map_clear Cette fonction effacera la carte DS (précédemment créée) de toutes les paires clé/valeur, mais elle ne supprimera pas la carte DS elle-même de la mémoire. Pour cela, vous devez utiliser la fonction ds_map_destroy(). Notez que l'effacement une carte déréférencera toutes les structures de données stockées dans la carte, ce qui entraînera une fuite de mémoire. Vous devrez donc passer en revue la carte et détruire manuellement tous les éléments de la structure de données avant d'effacer la carte pour éviter cela. La seule fois où cela n'est pas nécessaire est lorsque vous avez marqué des éléments de la carte comme étant une liste DS ou une autre carte DS, auquel cas ces éléments seront détruits (pas effacés !) et leur mémoire nettoyée automatiquement lorsque la carte parent est effacée. carte parent est effacée. Syntaxe : ds_map_clear(id) ; Arguments Description id L'identifiant de la structure de données à effacer. Les retours : N/A Exemple : if global.new_game { ds_map_clear(inventory) ; } La fonction ci-dessus vérifie une variable globale pour voir si elle est vraie et, si c'est le cas, la carte DS indexée dans la variable d'instance "inventaire" sera vidée de toutes les paires clé/valeur (mais elle ne sera pas détruite ).
