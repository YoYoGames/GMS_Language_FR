ds_map_secure_load_buffer Cette fonction permet de charger une carte DS sécurisée et sauvegardée à partir d'un tampon. Vous devez avoir préalablement chargé le tampon en mémoire (en utilisant la fonction buffer_load()), puis le passer à cette fonction renverra une carte DS remplie avec le contenu du tampon. Notez que le tampon doit avoir été créé à l'aide de la fonction ds_map_secure_save_buffer()pour que cela fonctionne correctement, et notez également que si la carte DS chargée contient un tableau, celui-ci sera converti en une liste DS lors du chargement. IMPORTANT ! L'une des caractéristiques d'un fichier sauvegardé sécurisé est qu'il est verrouillé sur l'appareil sur lequel il a été créé. Vous ne pouvez donc pas charger un fichier sauvegardé sur un appareil dans un projet exécuté sur un autre appareil. Syntaxe : ds_map_secure_load_buffer(buffer) ; Arguments Description nom de fichier L'ID du tampon à partir duquel charger les données cartographiques. Les retours : Entier (index de la carte DS) Exemple : var buff = buffer_load("save.dat") ; map = ds_map_secure_load_buffer(buff) ; buffer_delete(buff) ; Le code ci-dessus chargera une carte DS sauvegardée de manière sécurisée à partir d'un tampon et stockera sa valeur d'index dans une variable pour une utilisation future.
