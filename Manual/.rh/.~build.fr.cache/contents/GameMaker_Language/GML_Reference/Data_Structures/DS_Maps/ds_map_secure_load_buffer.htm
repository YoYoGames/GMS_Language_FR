ds_map_secure_load_buffer Cette fonction chargera une carte DS sauvegardée sécurisée à partir d'un tampon. Vous devez au préalable avoir chargé le tampon en mémoire (en utilisant buffer_load() ), puis en le passant dans cette fonction, vous retournerez une carte DS remplie avec le contenu du tampon. Notez que le tampon doit avoir été créé à l'aide de la fonction ds_map_secure_save_buffer() pour que cela fonctionne correctement, et notez également que si la carte DS en cours de chargement contenait un tableau, celui-ci sera converti en liste DS à la place lors du chargement. IMPORTANT! L'une des caractéristiques d'un fichier enregistré sécurisé est qu'il est verrouillé sur l'appareil sur lequel il a été créé, de sorte que vous ne pouvez pas charger un fichier enregistré sur un appareil dans un projet exécuté sur un autre appareil. Syntaxe:ds_map_secure_load_buffer(buffer); Argument Description filename ID de tampon du tampon à partir duquel charger les données cartographiques. Retour:Integer (DS map index) Exemple:var buff = buffer_load("save.dat"); map = ds_map_secure_load_buffer(buff); buffer_delete(buff); Le code ci-dessus chargera une carte DS enregistrée en toute sécurité à partir d'un tampon et stockera sa valeur d'index dans une variable pour une utilisation future.
