ds_map_add_list Avec cette fonction, vous pouvez attribuer une liste DS (créée précédemment) à une touche dans la carte DS donnée. Cette fonction est conçue pour créer des cartes compatibles JSON que vous encoderiez ensuite en utilisant json_encode() et ne doit être utilisé qu'en conjonction avec cette fonctionnalité. Si une carte DS a une liste ajoutée de cette manière, la destruction de la carte parente détruira également les listes contenues et libèrera leur mémoire, et appelera ds_map_clear() sur la carte parente détruira et nettoiera également toutes les listes marquées. Syntaxe:ds_map_add_list(id, key, value) Argument Description id L'identifiant de la carte à utiliser. key La clé de la liste ajoutée. value L'identifiant de la liste à ajouter. Retour:N/A Exemple:var j_list = ds_list_create(); ds_list_add(j_list, health); ds_list_add(j_list, lives); ds_list_add(j_list, score); var j_map = ds_map_create(); ds_map_add_list(j_map, "list", j_list); var j = json_encode(j_map); ds_map_destroy(j_map); Le code ci-dessus créera une liste et la remplira avec les différentes valeurs des variables globales. Cette liste est ensuite "imbriquée" dans une carte DS, et la carte est ensuite codée dans une chaîne JSON, avant que la carte ne soit détruite, en la supprimant de la mémoire, ainsi que toutes les listes qu'elle contient.
