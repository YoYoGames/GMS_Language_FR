ds_map_add_list Avec cette fonction, vous pouvez affecter une liste de DS (précédemment créée) à une clé dans la carte de DS donnée. Cette fonction est conçue pour créer des cartes compatibles JSON que vous encoderez ensuite en utilisant la fonction json_encode() et ne doit être utilisée qu'en conjonction avec cette fonctionnalité. Si une liste est ajoutée à une carte DS de cette manière, la destruction de la carte parente détruira également les listes contenues et libérera leur mémoire. ds_map_clear() sur détruira et nettoiera également toutes les listes signalées. Syntaxe : ds_map_add_list(id, key, value) Arguments Description id L'identifiant de la carte à utiliser. clé La clé pour la liste ajoutée. valeur L'identifiant de la liste à ajouter. Les retours : N/A Exemple : var j_list = ds_list_create() ; ds_list_add(j_list, health) ; ds_list_add(j_list, lives) ; ds_list_add(j_list, score) ; var j_map = ds_map_create() ; ds_map_add_list(j_map, "list", j_list) ; var j = json_encode(j_map) ; ds_map_destroy(j_map) ; Le code ci-dessus va créer une liste et la remplir avec les différentes valeurs des variables globales. Cette liste est ensuite "imbriquée" dans une map DS, et la map est ensuite encodée en une chaîne JSON, avant que la map ne soit détruite, la supprimant, ainsi que toutes les listes qu'elle contient, de la mémoire.
