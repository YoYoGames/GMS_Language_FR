ds_map_exists Cette fonction retournera true si la clé spécifiée existe dans la carte DS (créée précédemment), et false si ça ne fait pas. Syntaxe:ds_map_exists(id, key); Argument Description id l'id de la structure de données à vérifier key la clé à vérifier Retour:Boolean Exemple:if !ds_map_exists(inventory, "potions")    {    ds_map_add(inventory, "potions", 1);    } Le code ci-dessus vérifiera la carte DS indexée dans la variable "inventaire" pour la clé "potions" et si elle n'existe pas, il l'ajoutera à la carte.
