ds_map_exists Cette fonction renvoie true si la clé spécifiée existe dans la carte DS (précédemment créée), et false si ce n'est pas le cas. Syntaxe : ds_map_exists(id, key) ; Arguments Description id l'identifiant de la structure de données à vérifier clé la clé pour vérifier Les retours : Booléen Exemple : if !ds_map_exists(inventory, "potions") { ds_map_add(inventory, "potions", 1) ; } Le code ci-dessus vérifiera la carte DS indexée dans la variable "inventaire" pour la clé "potions" et si elle n'existe pas, il l'ajoutera à la carte.
