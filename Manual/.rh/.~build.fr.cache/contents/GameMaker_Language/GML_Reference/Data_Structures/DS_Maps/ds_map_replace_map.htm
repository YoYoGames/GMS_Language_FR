ds_map_replace_map Avec cette fonction, vous pouvez remplacer une carte DS qui a été stockée dans la "clé" donnée par une autre carte qui a été créée précédemment. Cette fonction est conçue pour créer des cartes compatibles JSON que vous encoderez ensuite à l'aide de la fonction json_encode() et ne doit être utilisée qu'en conjonction avec cette fonctionnalité. Syntaxe : ds_map_replace_map(id, key, value) Arguments Description id L'id de la ds_map à utiliser. clé La clé du remplacement. valeur L'id de la ds_map à utiliser pour remplacer celle précédemment stockée dans la clé donnée. Les retours : N/A Exemple : var temp_map = ds_map_create() ; ds_map_add_list(temp_map, "list", j_list) ; ds_map_add(temp_map, "array", j_array) ; ds_map_replace_map(j_map, "maps", temp_map) ; var j = json_encode(j_map) ; ds_map_destroy(temp_map) ; Le code ci-dessus va créer une DS Map et la remplir d'un tableau et d'une DS List avant de remplacer une carte précédemment stockée dans la DS Map "j_map".
