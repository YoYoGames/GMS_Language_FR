ds_map_empty Cette fonction retournera simplement false si la mappe DS spécifiée (créée précédemment) contient des paires clé / valeur, ou true si ça ne fait pas. Syntaxe:ds_map_empty(id); Argument Description id L'identifiant de la structure de données à vérifier. Retour:Boolean Exemple:if ds_map_empty(inventory)    {    weight = 0;    } Le code ci-dessus vérifie si la carte DS indexée dans la variable "inventaire" a des paires clé / valeur et si ce n'est pas le cas, il définit la variable "poids" sur 0.
