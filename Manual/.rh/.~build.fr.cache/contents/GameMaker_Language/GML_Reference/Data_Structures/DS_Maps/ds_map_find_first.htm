ds_map_find_first Cette fonction renvoie la première clé stockée dans la carte DS. Ce n'est pas la première clé dans l'ordre où vous les avez ajoutées ! Les cartes DS ne sont pas stockées sous une forme linéaire, pour cela utilisez DS list, donc tout ce que fait cette fonction est de trouver la première clé telle qu'elle est stockée par l'ordinateur. Cela peut être utile si vous devez itérer dans la carte DS à la recherche de quelque chose, mais devrait être évité si possible car cela peut être lent. Notez que cette fonction retournera undefined si la carte DS donnée est vide. Syntaxe : ds_map_find_first(id) ; Arguments Description id L'identifiant de la carte à utiliser. Les retours : Réel, chaîne de caractères ou indéfini Exemple : var size = ds_map_size(inventory) ; var key = ds_map_find_first(inventory) ; for (var i = 0 ; i < size ; i++ ;) { if key != "gold" { key = ds_map_find_next(inventory, key) ; } else break ; } Le code ci-dessus crée quelques variables temporaires, puis obtient la taille de la carte DS et trouve la première clé stockée par l'ordinateur dans la carte. Il utilise ensuite une boucle for pour itérer dans la carte DS à la recherche de la valeur clé "gold". Si Si elle est trouvée, la boucle est interrompue.
