debug_mode Cette variable en lecture seule renvoie un résultat vrai lorsque le jeu est joué en mode débogage et faux lorsqu'il est joué normalement. Syntaxe : debug_mode Les retours : Booléen Exemple : if debug_mode { ini_open("Cheats.ini") ; } else { ini_open("Game.ini") ; } Le code ci-dessus ouvre un fichier ini différent selon la valeur de la variable en lecture seule debug_mode.
