debug_mode Cette variable en lecture seule renvoie true lorsque le jeu est joué en mode débogage et false lorsqu'il est joué normalement. Syntaxe:debug_mode Retour:Boolean Exemple:if debug_mode    {    ini_open("Cheats.ini");    } else    {    ini_open("Game.ini");    } Le code ci-dessus ouvre un fichier ini différent en fonction de la valeur de la variable en lecture seule debug_mode.
