show_message Cette fonction crée une boîte de message pop-up qui affiche la chaîne définie dans le code et un bouton marqué "Ok" pour le fermer. REMARQUE: cette fonction est destinée au débogage uniquement sur les cibles Desktop et UWP, mais est déconseillée sur toutes les autres cibles. Syntaxe:show_message(str); Argument Description str La chaîne à afficher dans le message contextuel. Retour:N/A Exemple:var tot = 0; for (var i = 0; i < 10; i += 1)    {    tot += inv[i];    } show_message("Total = " + string(tot)); Le code ci-dessus va parcourir les valeurs stockées dans le tableau "inv" et les ajouter à la variable "tot" avant d'afficher un message avec le total.
