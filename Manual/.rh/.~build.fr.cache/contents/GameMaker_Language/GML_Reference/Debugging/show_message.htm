montrer_message Cette fonction crée une boîte de message contextuelle qui affiche la chaîne de caractères donnée et un bouton marqué "Ok" pour la fermer. REMARQUE : Cette fonction est destinée à être utilisée pour le débogage uniquement sur les cibles Desktop et UWP, mais est dépréciée sur toutes les autres cibles. Syntaxe : show_message(str) ; Arguments Description str La chaîne à afficher dans le message pop-up. Les retours : N/A Exemple : var tot = 0 ; for (var i = 0 ; i < 10 ; i += 1) { tot += inv[i] ; } show_message("Total = " + string(tot)) ; Le code ci-dessus va parcourir en boucle les valeurs stockées dans le tableau "inv" et les ajouter à la variable "tot" avant d'afficher un message avec le total.
