get_string Cette fonction crée une fenêtre contextuelle affichant un message standard, avec un bouton intitulé "Ok", qui invite l'utilisateur à saisir une chaîne de caractères. La fonction renvoie la chaîne de caractères saisie, ou la valeur par défaut si rien n'a été saisi. REMARQUE : Cette fonction est destinée à être utilisée pour le débogage uniquement sur les cibles Desktop, mais est dépréciée sur toutes les autres cibles. Syntaxe : get_string(str, def) ; Arguments Description str La chaîne à afficher dans le message pop-up. def La chaîne par défaut dans la zone de texte. Les retours : Chaîne de caractères Exemple : global.test_name = get_string("Test highscore name :", "Anonymous") ; Le code ci-dessus demandera à l'utilisateur de donner un nom qui sera ensuite stocké dans la variable globale "test_name". Si rien n'est saisi et que l'utilisateur appuie simplement sur "Ok", la valeur par défaut, "Anonyme", sera renvoyée.
