get_string Cela crée une fenêtre contextuelle affichant un message standard, avec un bouton intitulé "Ok", qui invite l'utilisateur à saisir une chaîne. La fonction renverra la chaîne d'entrée, ou la valeur par défaut si rien n'a été entré. REMARQUE: cette fonction est destinée au débogage uniquement sur les cibles Desktop, mais est déconseillée sur toutes les autres cibles. Syntaxe:get_string(str, def); Argument Description str La chaîne à afficher dans le message contextuel. def La chaîne par défaut dans la zone de texte. Retour:String Exemple:global.test_name = get_string("Test highscore name:", "Anonymous"); Le code ci-dessus demandera à l'utilisateur de donner un nom qui sera ensuite stocké dans la variable globale "test_name". Si rien n'est entré et que l'utilisateur appuie simplement sur "Ok", la valeur par défaut, "Anonymous", sera renvoyée.
