show_question Cette fonction crée une boîte de message contextuelle avec deux boutons pour "Oui" et "Non". Il retourne true ou false selon l'un des deux boutons sur lequel l'utilisateur appuie. REMARQUE: cette fonction est destinée au débogage uniquement sur les cibles Desktop, mais est déconseillée sur toutes les autres cibles. Syntaxe:show_question(str); Argument Description str La chaîne à afficher dans la question contextuelle. Retour:Boolean Exemple:if (score > 500) && debug_mode    {    if show_question("Continue to next room?")       {       room_goto(rm_Level2);       }    else game_end();    } Le code ci-dessus vérifiera le score et s'il est supérieur à 500, il demandera à l'utilisateur s'il souhaite continuer ou non et si le bouton "oui" est cliqué il ira dans une autre salle, mais si le bouton "non" est sélectionné, cela mettra fin au jeu.
