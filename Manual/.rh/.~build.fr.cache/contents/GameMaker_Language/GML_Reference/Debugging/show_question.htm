show_question Cette fonction crée une boîte de message contextuelle avec deux boutons pour "Oui" et "Non". Elle renvoie un message vrai ou faux en fonction du bouton sur lequel l'utilisateur appuie. REMARQUE : Cette fonction est destinée à être utilisée pour le débogage uniquement sur les cibles Desktop, mais est dépréciée sur toutes les autres cibles. Syntaxe : show_question(str) ; Arguments Description str La chaîne à afficher dans la question pop-up. Les retours : Booléen Exemple : if (score > 500) && debug_mode { if show_question("Continue to next room ?") { room_goto(rm_Level2) ; } else game_end() ; } Le code ci-dessus vérifiera le score et s'il est supérieur à 500, il demandera à l'utilisateur s'il souhaite continuer ou non et si le bouton "oui" est cliqué, il passera à une autre pièce, mais si le bouton "non" est sélectionné, il mettra fin au jeu.
