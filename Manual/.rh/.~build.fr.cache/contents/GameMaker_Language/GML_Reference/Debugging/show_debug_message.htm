montrer_debug_message Cette fonction va créer un message de débogage personnalisé qui sera affiché dans la fenêtre du compilateur au moment de l'exécution. Notez que le message peut être soit une chaîne de caractères, soit un nombre réel, mais si vous avez besoin des deux, le nombre devra être converti en chaîne de caractères en utilisant la fonction la fonction chaîne() (voir l'exemple ci-dessous) et que si le nombre a plus de deux décimales, vous devez utiliser la fonction string_format() pour les afficher car par défaut, les décimales sont arrondies aux deux décimales les plus proches (ainsi, "1,2468" s'afficherait comme "1,25" dans la fenêtre de sortie). Les messages de débogage affichés avec cette fonction seront affichés dans la fenêtre de sortie du compilateur en bas de l'IDE ainsi que dans la vue graphique du débogueur lorsque le jeu est en mode débogage. graphique du débogueur lorsque le jeu est en mode débogage. Si vous ne voulez voir les messages qu'en mode débogage, vous devriez probablement utiliser la fonction debug_event() à la place. Syntaxe : show_debug_message(string) ; Arguments Description chaîne de caractères Le message de débogage personnalisé à afficher. Les retours : N/A Exemple : if !instance_exists(obj_Exit) { show_debug_message("Exit not created !") ; show_debug_message("Error Num : " + string(global.error)) ; game_end() ; } Le code ci-dessus vérifie si une instance existe et si ce n'est pas le cas, un message de débogage est affiché dans la fenêtre de compilation et le jeu est terminé.
