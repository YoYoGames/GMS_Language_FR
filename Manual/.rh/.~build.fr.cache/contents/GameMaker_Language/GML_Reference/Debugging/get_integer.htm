get_integer Cette fonction crée une fenêtre contextuelle affichant un message personnalisé, avec un bouton intitulé "Ok", et invite l'utilisateur à saisir une valeur entière. La fonction renvoie le nombre entier saisi, ou la valeur par défaut si rien n'a été saisi. REMARQUE : Cette fonction est destinée à être utilisée pour le débogage uniquement sur les cibles Desktop, mais est dépréciée sur toutes les autres cibles. Syntaxe : get_integer(str, def) ; Arguments Description str La chaîne à afficher dans le message pop-up. def La valeur par défaut dans la zone de texte. Les retours : Real Exemple : global.level = get_integer("Niveau à tester ?", 1) ; Le code ci-dessus affichera un message invitant l'utilisateur à sélectionner un niveau à tester.
