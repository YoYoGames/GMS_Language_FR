get_integer Cela crée une fenêtre contextuelle affichant un message personnalisé, avec un bouton intitulé «Ok», et invite l'utilisateur à saisir une valeur entière. La fonction retournera l'entier tapé, ou la valeur par défaut si rien n'a été entré. REMARQUE: cette fonction est destinée au débogage uniquement sur les cibles Desktop, mais est déconseillée sur toutes les autres cibles. Syntaxe:get_integer(str, def); Argument Description str La chaîne à afficher dans le message contextuel. def La valeur par défaut dans la zone de texte. Retour:Real Exemple:global.level = get_integer("Level to test?", 1); Le code ci-dessus affichera un message invitant l'utilisateur à sélectionner un niveau pour le test.
