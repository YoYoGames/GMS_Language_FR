montrer_erreur Cette fonction affichera une chaîne de caractères personnalisée comme message d'erreur et donnera la possibilité d'abandonner le jeu par la suite si nécessaire. REMARQUE : Cette fonction est réservée à un usage de débogage. Syntaxe : show_error(str, abort) ; Arguments Description str La chaîne à afficher dans le message pop-up. abandonner S'il faut avorter (true) ou non (false). Les retours : N/A Exemple : if (room != rm_Dungeon) { show_error("Error : Went to wrong area. Aborting game.", true) ; } Le code ci-dessus vérifie si la pièce dans laquelle se trouve le joueur est "rm_Dungeon" et si ce n'est pas le cas, il affiche un message d'erreur et force l'utilisateur à abandonner le jeu.
