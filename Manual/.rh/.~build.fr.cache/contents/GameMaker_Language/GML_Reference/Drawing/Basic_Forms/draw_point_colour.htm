draw_point_colour Avec cette fonction, vous pouvez dessiner un seul pixel n'importe où sur l'écran avec une couleur que vous définissez. Les paramètres de couleur remplaceront la couleur de base définie avec la fonction draw_set_colour(). REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises, et le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous devez dessiner des formes de cette manière, le shader doit être personnalisé dans cet esprit. Syntaxe:draw_point_colour(x, y, col1); Argument Description x La coordonnée x du point. y La coordonnée y du point. col1 La couleur du point. Retour:N/A Exemple:draw_point_colour(50, 50, c_red); Cela dessinerait un pixel rouge à (50,50). Retour: Formulaires de base Suivant:draw_rectangle © Copyright YoYo Games Ltd. 2021 Tous droits réservés
