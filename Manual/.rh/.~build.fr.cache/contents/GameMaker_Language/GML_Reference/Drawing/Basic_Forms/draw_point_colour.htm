dessiner_point_couleur Avec cette fonction, vous pouvez dessiner un pixel n'importe où sur l'écran avec une couleur que vous définissez. Les paramètres de couleur remplaceront la couleur de base définie avec la fonction draw_set_colour(). NOTE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence. Syntaxe : draw_point_colour(x, y, col1) ; Arguments Description x La coordonnée x du point. y La coordonnée y du point. col1 La couleur du point. Les retours : N/A Exemple : draw_point_colour(50, 50, c_red) ; Cela dessinerait un pixel rouge à (50,50).
