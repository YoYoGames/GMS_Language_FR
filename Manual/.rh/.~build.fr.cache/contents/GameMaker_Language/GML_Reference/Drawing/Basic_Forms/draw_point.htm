dessiner_point Avec cette fonction, vous pouvez dessiner un pixel unique n'importe où sur l'écran, en utilisant la couleur de dessin et l'alpha actuellement définis. Syntaxe : draw_point(x, y) ; Arguments Description x La coordonnée x du point à dessiner. y La coordonnée y du point à dessiner. Les retours : N/A Exemple : draw_set_colour(c_yellow) ; draw_point(100,100) ; Cela va dessiner un pixel jaune à la position (100,100).
