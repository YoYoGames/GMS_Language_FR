dessiner_ligne Avec cette fonction, vous pouvez dessiner une ligne de 1 pixel de large entre deux points quelconques de la salle de jeu. Veuillez noter que la ligne dessinée peut nécessiter des valeurs différentes (+/-1 sur les x, y) pour être dessinée avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge. plateformes supportées. REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence. Syntaxe : draw_line(x1, y1, x2, y2) ; Arguments Description x1 La coordonnée x du début de la ligne. y1 La coordonnée y du début de la ligne. x2 La coordonnée x de la fin de la ligne. y2 La coordonnée y de la fin de la ligne. Les retours : N/A Exemple : draw_set_colour(c_lime) ; draw_line(50,50,150,50) ; Cela va dessiner une ligne horizontale vert clair du point (50,50) au point (150,50).
