dessiner_ellipse_couleur Avec cette fonction, vous pouvez dessiner le contour d'une ellipse ou une ellipse remplie en définissant une zone rectangulaire dans laquelle l'ellipse sera créée. Si elle est remplie, vous pouvez définir les couleurs de remplissage intérieure et extérieure. Si ces couleurs ne sont pas les mêmes, vous obtiendrez un effet de gradient de l'une à l'autre (les paramètres de couleur remplaceront la couleur de base définie avec la fonction draw_set_colour()). Vous pouvez définir le degré de la précision du dessin à l'aide de la fonction draw_set_circle_precision(). NOTE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence. Syntaxe : draw_ellipse_colour(x1, y1, x2, y2, col1, col2, outline) ; Arguments Description x1 La coordonnée x de la gauche de l'ellipse. y1 La coordonnée y du sommet de l'ellipse. x2 La coordonnée x de la droite de l'ellipse. y2 La coordonnée y du bas de l'ellipse. col1 La couleur du centre. col2 La couleur du bord. aperçu Indique si l'ellipse est un contour (true) ou si elle est remplie (false). Les retours : N/A Exemple : draw_ellipse_colour(50, 50, 200, 100, c_black, c_white, false) ; Cela dessine une ellipse remplie, étirée horizontalement, avec un centre noir et un bord blanc.
