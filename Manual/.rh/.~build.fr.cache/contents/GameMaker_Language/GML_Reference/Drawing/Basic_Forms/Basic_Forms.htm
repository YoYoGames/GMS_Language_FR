Formes de base GameMaker Studio 2 dispose de toute une série de fonctions permettant de dessiner différentes formes. Ces fonctions ne doivent être utilisées que dans l'événement draw d'une instance (vous pouvez les utiliser ailleurs mais vous ne verrez rien ! ailleurs mais vous ne verrez rien !) et créent des effets purement graphiques, ce qui signifie qu'elles ne génèrent pas d'événements de collision car ceux-ci ne peuvent être réalisés qu'à travers le sprite ou le masque de collision qui a été assigné à une instance. Toutes ces fonctions sont affectées par la couleur de dessin, l'alpha et le mode de fusion actuels (voir la section sur la couleur). IMPORTANT ! Les lignes, points et formes sont toujours dessinés à 1:1 avec la résolution de l'écran, quel que soit l'événement dans lequel ils sont dessinés. IMPORTANT ! Ces fonctions vont casser le vertex batching et augmenter le nombre de changements de texture dans votre jeu, donc avoir plusieurs appels à l'une d'entre elles dans une seule image de dessin peut conduire à des problèmes de performance. Les fonctions de dessin suivantes permettent de dessiner des formes de base.
