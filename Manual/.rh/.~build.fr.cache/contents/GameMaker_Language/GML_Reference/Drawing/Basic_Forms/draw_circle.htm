draw_circle Avec cette fonction, vous pouvez dessiner le contour d'un cercle ou un cercle plein. Vous pouvez définir la précision du dessin avec la fonction draw_set_circle_precision(). REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises, et le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous devez dessiner des formes de cette manière, le shader doit être personnalisé dans cet esprit. Syntaxe:draw_circle(x, y, r, outline); Argument Description x La coordonnée x du centre du cercle. y La coordonnée y du centre du cercle. r Le rayon du cercle (longueur de son centre à son bord) outline Si le cercle est dessiné rempli (false ) ou sous la forme d'un contour large d'un pixel (true ). Retour:N/A Exemple:draw_set_colour(c_white); draw_circle(100, 100, 50, true); Cela dessinera un contour de cercle blanc d'un pixel de large avec un rayon de 50 pixels.
