dessiner_cercle Avec cette fonction, vous pouvez dessiner soit le contour d'un cercle, soit un cercle rempli. Vous pouvez définir la précision du dessin à l'aide de la fonction draw_set_circle_precision(). REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence. Syntaxe : draw_circle(x, y, r, outline) ; Arguments Description x La coordonnée x du centre du cercle. y La coordonnée y du centre du cercle. r Le rayon du cercle (longueur du centre au bord) aperçu Si le cercle est dessiné rempli(faux) ou comme un contour d'un pixel de large(vrai). Les retours : N/A Exemple : draw_set_colour(c_white) ; draw_circle(100, 100, 50, true) ; Ceci dessinera le contour d'un cercle blanc d'un pixel de large et d'un rayon de 50 pixels.
