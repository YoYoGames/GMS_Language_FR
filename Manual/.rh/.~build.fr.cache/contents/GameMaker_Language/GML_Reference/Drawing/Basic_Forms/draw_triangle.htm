dessiner_triangle Avec cette fonction, vous pouvez dessiner soit le contour d'un triangle, soit un triangle rempli. NOTE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence. Syntaxe : draw_triangle(x1, y1, x2, y2, x3, y3, outline) ; Arguments Description x1 La coordonnée x du premier coin du triangle. y1 La coordonnée y du premier coin du triangle. x2 La coordonnée x du deuxième coin du triangle. y2 La coordonnée y du deuxième coin du triangle. x3 La coordonnée x du troisième coin du triangle. y3 La coordonnée y du troisième coin du triangle. aperçu Si le triangle est dessiné rempli (false) ou comme un contour d'un pixel de large (true). Les retours : N/A Exemple : draw_set_colour(c_aqua) ; draw_triangle(50, 50, 200, 50, 50, 200, 0) ; Cela dessinera un triangle rectangle isocèle rempli de couleur aigue-marine, dont le premier angle est situé à (50,50), le deuxième à (200,50) et le troisième à (50,200).
