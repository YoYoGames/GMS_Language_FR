dessiner_triangle_couleur Avec cette fonction, vous pouvez dessiner soit le contour d'un triangle, soit un triangle rempli. S'il est rempli, vous pouvez définir les couleurs individuelles pour chaque point d'angle et si ces couleurs ne sont pas les mêmes, vous obtiendrez un effet de gradient de l'un à l'autre. (les paramètres de couleur remplaceront la couleur de base définie avec la fonction draw_set_colour()). REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence. Syntaxe : draw_triangle_colour(x1, y1, x2, y2, x3, y3, col1, col2, col3, outline) ; Arguments Description x1 La coordonnée x du premier coin du triangle. y1 La coordonnée y du premier coin du triangle. x2 La coordonnée x du deuxième coin du triangle. y2 La coordonnée y du deuxième coin du triangle. x3 La coordonnée x du troisième coin du triangle. y3 La coordonnée y du troisième coin du triangle. col1 La couleur du premier coin. col2 La couleur du deuxième coin. col3 La couleur du troisième coin. aperçu Indique si le triangle est un contour (vrai) ou rempli (faux). Les retours : N/A Exemple : draw_triangle_colour(200, 200, 300, 200, 200, 300, c_red, c_blue, c_blue, false) ; Cela dessinerait un triangle rectangle isocèle rempli avec du rouge à l'angle droit, du bleu aux deux autres coins.
