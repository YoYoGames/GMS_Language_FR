draw_rectangle Avec cette fonction, vous pouvez dessiner le contour d'un rectangle ou un rectangle rempli où la position (x1, y1) est le coin supérieur gauche et la position (x2, y2) est le coin inférieur droit. Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/- 1 sur le x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les différentes plates-formes prises en charge. REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises, et le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous devez dessiner des formes de cette manière, le shader doit être personnalisé dans cet esprit. Syntaxe:draw_rectangle(x1, y1, x2, y2, outline); Argument Description x1 La coordonnée x du coin supérieur gauche du rectangle. y1 Coordonnée y du coin supérieur gauche du rectangle. x2 La coordonnée x du coin inférieur droit du rectangle. y2 Coordonnée y du coin inférieur droit du rectangle. outline Si le rectangle est dessiné rempli (faux) ou sous la forme d'un contour large d'un pixel (vrai). Retour:N/A Exemple:draw_set_colour(c_yellow); draw_rectangle(100, 100, 300, 200, true); Cela dessinera un contour rectangle, avec son coin supérieur gauche à (100,100) et son coin inférieur droit à (300,200).
