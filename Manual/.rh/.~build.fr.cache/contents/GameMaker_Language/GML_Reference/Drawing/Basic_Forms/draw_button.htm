draw_button Cette fonction dessinera un "bouton" rectangulaire très simple en utilisant la couleur de dessin et l'alpha actuellement sélectionnés où l' argument haut définit l'aspect de l'effet de bord biseauté, comme indiqué dans l'image ci-dessous: REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises, et le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous devez dessiner des formes de cette manière, le shader doit être personnalisé dans cet esprit. Syntaxe:draw_button(x1, y1, x2, y2, up); Argument Description x1 La coordonnée x de la gauche du bouton y1 La coordonnée y du haut du bouton x2 La coordonnée x de la droite du bouton y2 La coordonnée y du bas du bouton up Si le bouton est en place (true ) ou vers le bas (false ) Retour:N/A Exemple:draw_button(100, 100, 200, 150, !mouse_check_button(mb_left)); Cela dessinera un bouton qui apparaîtra enfoncé si le bouton gauche de la souris est maintenu enfoncé.
