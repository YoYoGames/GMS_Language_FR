bouton de dessin Cette fonction dessine un "bouton" rectangulaire très simple en utilisant la couleur de dessin et l'alpha actuellement sélectionnés, l'argument "up" définissant l'effet de bord biseauté, comme le montre l'image ci-dessous : NOTE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur, etc. les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, alors le shader doit être personnalisé dans cette optique. Syntaxe : draw_button(x1, y1, x2, y2, up) ; Arguments Description x1 La coordonnée x de la gauche du bouton y1 La coordonnée y du haut du bouton x2 La coordonnée x de la droite du bouton y2 La coordonnée y du bas du bouton en haut Si le bouton est en haut(vrai) ou en bas(faux). Les retours : N/A Exemple : draw_button(100, 100, 200, 150, !mouse_check_button(mb_left)) ; Ceci dessinera un bouton qui apparaîtra enfoncé si le bouton gauche de la souris est maintenu enfoncé.
