draw_set_circle_precision Pour optimiser les performances lors du dessin de cercles, GameMaker Studio 2 dessine essentiellement une forme de polygone avec suffisamment de côtés pour lui donner une apparence circulaire. Cependant, selon la taille de vos cercles, vous constaterez peut-être que la modification de cette valeur peut vous aider à améliorer les performances de votre jeu ou à améliorer l'apparence des cercles. La valeur de précision que vous pouvez entrer dans cette fonction est le nombre de côtés du polygone circulaire, et ce nombre peut être compris entre 4 et 64, mais doit être un nombre divisible par 4, avec une valeur par défaut de 24. REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises, et le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous devez dessiner des formes de cette manière, le shader doit être personnalisé dans cet esprit. Syntaxe:draw_set_circle_precision(precision); Argument Description precision La précision de tous les cercles dessinés après cette fonction est appelée. Retour:N/A Exemple:draw_set_circle_precision(12); Le code ci-dessus dessinera tous les cercles en utilisant un polygone à 12 côtés.
