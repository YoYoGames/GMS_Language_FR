gpu_set_tex_max_aniso_ext Avec cette fonction, vous pouvez définir le niveau maximal d'anisotropie lorsque vous utilisez le mode de filtre tf_anisotropique (voir gpu_get_tex_mip_filter() pour plus d'informations) sur un échantillonneur de shader. Vous fournissez la valeur pour l'échantillonneur d'ombrage (tel que renvoyé par la fonction shader_get_sampler_index()), puis une valeur comprise entre 1 et 16 pour définir le niveau. Syntaxe : gpu_set_tex_max_aniso_ext(sampler_index, maxaniso) ; Arguments Description sampler_index L'index de l'échantillonneur de shader à obtenir maxaniso Le niveau d'anisotropie maximum à utiliser (par défaut : 16). Les retours : N/A Exemple : var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background") ; var _spriteTex = sprite_get_texture(sprite_index, 0) ; shader_set (shd_Glass) ; if gpu_get_tex_max_aniso_ext(_sampleIndex) != 8 { gpu_set_tex_max_aniso_ext(_sampleIndex, 8) ; } texture_set_stage(_sampleIndex , _spriteTex) ; shader_reset() ; Le code ci-dessus définit le niveau maximum d'anisotropie à 8 pour l'échantillonneur de texture du shader donné s'il n'a pas déjà été défini à 8.
