Mipmapping Cette section contient toutes les fonctions liées à l'utilisation des fonctions mipmapping. Avant d'utiliser ces fonctions, vous devriez avoir activé le mipmapping pour les pages de dans l'éditeur de groupes de textures et/ou activer le mipmapping pour les sprites qui ont été configurés pour n'utiliser qu'une seule page de texture dans les options générales du jeu. L'image ci-dessous montre la différence que le mipmapping peut faire lors du rendu de votre projet : Une fois que vous avez activé le mipmapping pour le projet ou une page de texture, vous pouvez alors utiliser les fonctions suivantes pour obtenir et définir des valeurs qui modifieront l'aspect de votre projet lors de son exécution.
