Mipmapping Cette section contient toutes les fonctions liées à l'utilisation des fonctions de mipmapping. Avant d'utiliser ces fonctions, vous devez avoir activé le mipmapping pour les pages de texture requises dans l' éditeur de groupe de textures et / ou activé le mipmapping pour les sprites qui ont été configurés pour n'utiliser qu'une seule page de texture à partir des Options générales du jeu.
