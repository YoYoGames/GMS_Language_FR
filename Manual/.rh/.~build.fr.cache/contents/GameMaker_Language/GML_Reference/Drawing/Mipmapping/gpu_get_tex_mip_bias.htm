gpu_get_tex_mip_bias Avec cette fonction, vous pouvez récupérer la valeur de biais mipmap, où une valeur de 0 est égal à aucun biais, 1 est égal au premier mipmap, 2 est égal à la deuxième mipmap etc... Cela contrôle la vitesse à laquelle la carte mip est permutée et fera généralement les textures rendues plus floues plus la valeur est élevée et plus la "distance" visualisée est grande. Notez que cela peut également renvoyer des valeurs négatives, auquel cas les textures rendues seront plus nettes sur une plus grande distance, plus la valeur sera basse. Syntaxe:gpu_get_tex_mip_bias(); Retour:Real (default: 0) Exemple:if gpu_get_tex_mip_bias() != 0     {     gpu_set_tex_mip_bias(0);     } Le code ci-dessus vérifiera le biais actuel de mipmap et s'il n'est pas 0, il est mis à 0.
