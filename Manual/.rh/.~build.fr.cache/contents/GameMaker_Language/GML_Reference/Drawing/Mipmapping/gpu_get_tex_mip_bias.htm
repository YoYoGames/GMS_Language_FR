gpu_get_tex_mip_bias Avec cette fonction, vous pouvez récupérer la valeur du biais de la mipmap, où une valeur de 0 équivaut à aucun biais, 1 équivaut à la première mipmap, 2 équivaut à la seconde mipmap etc.... Ceci contrôle la vitesse à laquelle la mipmap est échangée et rendra généralement les textures rendues plus floues. plus floues, plus la valeur est élevée et plus la "distance" vue est grande. Notez que cette option peut également renvoyer des valeurs négatives, dans ce cas les textures rendues seront plus nettes sur une plus grande distance si la valeur est faible. Syntaxe : gpu_get_tex_mip_bias() ; Les retours : Réel (par défaut : 0) Exemple : if gpu_get_tex_mip_bias() != 0 { gpu_set_tex_mip_bias(0) ; } Le code ci-dessus vérifiera le biais actuel de la mipmap et s'il n'est pas égal à 0, il sera mis à 0.
