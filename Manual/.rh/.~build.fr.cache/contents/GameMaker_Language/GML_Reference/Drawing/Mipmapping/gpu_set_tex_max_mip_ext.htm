gpu_set_tex_max_mip_ext Avec cette fonction, vous pouvez obtenir le niveau maximum de mipmap actuellement défini qui doit être utilisé pour un échantillonneur de shader donné. Vous fournissez la valeur de l'index de l'échantillonneur d'ombrage (tel que retourné par la fonction shader_get_sampler_index(), et la fonction renverra le niveau maximum actuel de mipmap, où une valeur de 0 est la plus haute résolution, 1 est d'utiliser le premier mipmap, 2 est le second etc.... Notez que cela peut être très utile pour éviter les artefacts de saignement lors du rendu des textures, Par exemple, en définissant la bordure de la page de texture à 8px et en définissant le niveau de mipmap maximum à 3, vous serez sûr de ne pas avoir de problèmes de saignement à des distances de rendu plus importantes. Syntaxe : gpu_set_tex_max_mip_ext(sampler_index, maxmip) ; Arguments Description sampler_index L'index de l'échantillonneur de shader à obtenir maxmip La valeur maximale du niveau mipmap à utiliser (par défaut : 16) Les retours : N/A Exemple : var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background") ; var _spriteTex = sprite_get_texture(sprite_index, 0) ; shader_set (shd_Glass) ; if gpu_get_tex_max_mip_ext(_sampleIndex) != 4 { gpu_set_tex_max_mip_ext(_sampleIndex, 4) ; } texture_set_stage(_sampleIndex , _spriteTex) ; shader_reset() ; Le code ci-dessus définit le niveau maximum de mipmap à 4 pour l'échantillonneur de texture de shader donné s'il n'a pas déjà été défini à 4.
