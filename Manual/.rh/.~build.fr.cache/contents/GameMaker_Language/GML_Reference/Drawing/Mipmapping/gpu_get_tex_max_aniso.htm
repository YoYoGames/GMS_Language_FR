gpu_get_tex_max_aniso Avec cette fonction, vous pouvez obtenir le niveau maximum actuel d'anisotropie lorsque vous utilisez le mode de filtre tf_anisotropique (voir gpu_get_tex_mip_filter() pour plus d'informations). La valeur renvoyée sera sera comprise entre 1 et 16. Syntaxe : gpu_get_tex_max_aniso() ; Les retours : Real (par défaut : 16) Exemple : if gpu_get_tex_max_aniso() != 8 { gpu_set_tex_max_aniso(8) ; } Le code ci-dessus vérifie le niveau maximum actuel du filtrage anisotrope et s'il n'est pas égal à 8, il est fixé à 8.
