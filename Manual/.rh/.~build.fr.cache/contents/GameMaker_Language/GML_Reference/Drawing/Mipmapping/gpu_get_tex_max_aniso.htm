gpu_get_tex_max_aniso Avec cette fonction, vous pouvez obtenir le niveau maximum actuel d'anisotropie lors de l'utilisation du tf_anisotropic mode de filtrage (voir gpu_get_tex_mip_filter() pour plus d'informations). La valeur renvoyée sera comprise entre 1 et 16. Syntaxe:gpu_get_tex_max_aniso(); Retour:Real (default: 16) Exemple:if gpu_get_tex_max_aniso() != 8     {     gpu_set_tex_max_aniso(8);     } Le code ci-dessus vérifiera le niveau de filtrage anisotrope maximum actuel et s'il n'est pas 8, il est mis à 8.
