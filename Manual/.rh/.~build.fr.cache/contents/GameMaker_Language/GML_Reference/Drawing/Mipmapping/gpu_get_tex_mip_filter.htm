gpu_get_tex_mip_filter Avec cette fonction, vous pouvez obtenir le mode actuel du filtre mip. La valeur du mode retournée par la fonction sera l'une des constantes listées ci-dessous. Syntaxe : gpu_get_tex_mip_filter() ; Les retours : Constantes (énumérées ci-dessous) : Constant Description tf_point Cela signifie que le mélange entre les niveaux de mipmap est désactivé, ce qui peut provoquer des transitions de texture visibles, mais donne les meilleures performances. tf_linear Cela signifie que le mélange entre les niveaux de mipmap est activé (c'est également connu sous le nom de filtrage trilinéaire), ce qui lisse les transitions de texture, mais entraîne une légère baisse des performances. tf_anisotropique Cela signifie que le filtrage anisotropique est activé, ce qui améliore considérablement la qualité de la transition des textures et peut réduire le flou visible avec d'autres modes de filtrage, mais il a le plus grand impact sur les performances. Exemple : if keyboard_check(ck_enter) { switch(gpu_get_tex_mip_filter()) { case tf_point : gpu_set_tex_mip_filter(tf_linear) ; break ; case tf_linear : gpu_set_tex_mip_filter(tf_anisotropic) ; break ; case tf_anisotropic : gpu_set_tex_mip_filter(tf_point) ; break ; } } Le code ci-dessus vérifie le clavier et si la touche spécifiée est pressée, il récupère le filtre mip actuel et passe à la valeur suivante, en passant par les différents modes.
