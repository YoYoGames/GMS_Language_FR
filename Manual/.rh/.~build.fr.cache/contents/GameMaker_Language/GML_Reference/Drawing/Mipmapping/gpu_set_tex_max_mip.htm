gpu_set_tex_max_mip Avec cette fonction, vous pouvez définir le niveau maximum de mipmap qui doit être utilisé, où une valeur de 0 est la résolution la plus élevée, 1 est d'utiliser la première mipmap, 2 est la deuxième etc.... Notez que cela peut être très utile pour éviter les artefacts de saignement lors du rendu des textures. lors du rendu de textures, par exemple, en définissant la bordure de la page de texture à 8px et en définissant le niveau de mipmap maximum à 3, vous serez sûr de ne pas avoir de problèmes de saignement à des distances de rendu plus importantes. Syntaxe : gpu_set_tex_max_mip(maxmip) ; Arguments Description maxmip Le niveau maximum de mipmap (par défaut : 16) Les retours : N/A Exemple : if gpu_get_tex_max_mip() != 4 { gpu_set_tex_max_mip(4) ; } Le code ci-dessus vérifie le niveau maximum actuel de mipmap et s'il n'est pas égal à 4, il est fixé à 4.
