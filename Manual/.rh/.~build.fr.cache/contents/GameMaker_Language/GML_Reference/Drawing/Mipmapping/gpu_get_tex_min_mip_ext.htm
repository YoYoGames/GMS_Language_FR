gpu_get_tex_min_mip_ext Avec cette fonction, vous pouvez obtenir le niveau minimum de mipmap qui est actuellement utilisé pour un échantillonneur de shader donné. Vous fournissez la valeur de l'index de l'échantillonneur de nuanceur (tel que retourné par la fonction shader_get_sampler_index(), et la fonction retournera une valeur de 0 pour la résolution complète, 1 pour le premier mipmap, 2 pour le second etc... Syntaxe : gpu_get_tex_min_mip_ext(sampler_index) ; Arguments Description sampler_index L'index de l'échantillonneur de shader à obtenir Les retours : Réel (par défaut : 0) Exemple : var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background") ; var _spriteTex = sprite_get_texture(sprite_index, 0) ; shader_set (shd_Glass) ; if gpu_get_tex_min_mip_ext(_sampleIndex) != 0 { gpu_set_tex_min_mip_ext(_sampleIndex, 0) ; } texture_set_stage(_sampleIndex , _spriteTex) ; shader_reset() ; Le code ci-dessus définit le niveau minimum de mipmap à 0 pour l'échantillonneur de texture de shader donné s'il n'a pas déjà été défini à 0.
