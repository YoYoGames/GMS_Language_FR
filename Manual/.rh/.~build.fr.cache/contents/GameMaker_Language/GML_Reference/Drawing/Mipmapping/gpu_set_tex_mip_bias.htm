gpu_set_tex_mip_bias Avec cette fonction, vous pouvez définir la valeur du biais de la mipmap, où une valeur de 0 équivaut à aucun biais, 1 équivaut à la première mipmap, 2 équivaut à la seconde mipmap etc.... Ceci contrôle la vitesse à laquelle la mipmap est échangée et rendra généralement les textures rendues plus floues. plus la valeur est élevée et plus la "distance" vue est grande. Notez que cette fonction peut aussi prendre des valeurs négatives, dans ce cas les textures rendues seront plus nettes sur une plus grande distance plus la valeur est faible. Syntaxe : gpu_set_tex_mip_bias(bias) ; Arguments Description biais La valeur du biais de mipmap à utiliser (par défaut : 0) Les retours : N/A Exemple : if gpu_get_tex_mip_bias() != 0 { gpu_set_tex_mip_bias(0) ; } Le code ci-dessus vérifiera le biais actuel de la mipmap et s'il n'est pas égal à 0, il sera mis à 0.
