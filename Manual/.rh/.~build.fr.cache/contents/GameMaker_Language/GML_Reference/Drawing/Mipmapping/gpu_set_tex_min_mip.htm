gpu_set_tex_min_mip Avec cette fonction, vous pouvez définir le niveau minimum de mipmap qui est actuellement utilisé, où une valeur de 0 est la résolution la plus élevée, 1 est d'utiliser le premier mipmap, 2 est le second etc... Syntaxe:gpu_set_tex_min_mip(minmip); Argument Description minmip Le niveau minimum de mipmap à utiliser Retour:N/A Exemple:if gpu_get_tex_min_mip() != 0     {     gpu_set_tex_min_mip(0);     } Le code ci-dessus vérifiera le niveau mipmap minimum actuel et s'il n'est pas 0, il est mis à 0.
