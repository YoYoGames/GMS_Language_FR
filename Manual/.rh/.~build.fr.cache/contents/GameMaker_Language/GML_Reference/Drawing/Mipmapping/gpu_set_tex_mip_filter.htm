gpu_set_tex_mip_filter Avec cette fonction, vous pouvez définir le mode de filtre mip actuel sur l'un des trois types pris en charge. Vous donnez la constante qui fait référence au mode de filtrage mip dont vous avez besoin, où vous pouvez choisir entre le filtrage par points (réglage par défaut, ce qui signifie pas de filtrage), le filtrage linéaire (également appelé filtrage trilinéaire ) ou le filtrage aniostropique (voir le tableau des constantes en dessous de). Constant Description tf_point Cela désactivera le filtrage entre les niveaux de mipmap, ce qui peut provoquer des transitions de texture visibles, mais donne les meilleures performances. tf_linear Cela permettra un filtrage linéaire entre les niveaux mipmap (également connu sous le nom de filtrage trilinéaire ), qui lisse les transitions de texture, mais cela donnera un impact mineur aux performances. tf_anisotropic Cela permettra un filtrage anisotrope entre les niveaux de mipmap, ce qui améliore considérablement la qualité de la transition de texture et peut réduire le flou visible avec d'autres modes de filtrage, mais il a le plus grand impact sur les performances. Syntaxe:gpu_get_tex_mip_filter(filter); Argument Description filter Le mode de filtre mip à utiliser (une constante, par défaut:tf_point ). Retour:N/A Exemple:if keyboard_check(ck_enter)     {     switch(gpu_get_tex_mip_filter())         {         case tf_point: gpu_set_tex_mip_filter(tf_linear); break;         case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;         case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;         }     } Le code ci-dessus vérifie le clavier et si la touche spécifiée est enfoncée, il obtiendra le filtre mip actuel et basculera la valeur sur la suivante, en parcourant les différents modes.
