gpu_set_tex_mip_enable_ext Avec cette fonction, vous pouvez définir si le mipmapping est désactivé, activé pour tout ou activé uniquement pour les groupes de textures sélectionnés dans le gestionnaire de groupes de textures sur un échantillonneur de shader. Vous devez fournissez la valeur de l'index de l'échantillonneur de nuanceurs (tel que renvoyé par la fonction shader_get_sampler_index()et ensuite une des constantes listées ci-dessous. Constant Description mip_off Le Mipmapping est désactivé. mip_on Le Mipmapping pour toutes les textures est activé. mip_markedonly Le Mipmapping est activé pour les textures qui l'ont activé dans les options du groupe de textures (par défaut). Syntaxe : gpu_set_tex_mip_enable_ext(sampler_index, setting) ; Arguments Description sampler_index L'index de l'échantillonneur de shader à obtenir paramètre Le paramètre mipmap (une constante, par défaut : mip_markedonly) Les retours : N/A Exemple : var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background") ; var _spriteTex = sprite_get_texture(sprite_index, 0) ; shader_set (shd_Glass) ; if gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on { gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on) ; } texture_set_stage(_sampleIndex , _spriteTex) ; shader_reset() ; Le code ci-dessus active le mipmapping pour l'échantillonneur de texture du shader donné s'il n'a pas déjà été activé.
