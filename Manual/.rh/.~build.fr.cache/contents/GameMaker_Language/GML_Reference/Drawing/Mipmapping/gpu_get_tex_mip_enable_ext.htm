gpu_get_tex_mip_enable_ext Cette fonction vous permet de savoir si le mipmapping est désactivé, activé pour tout ou activé uniquement pour les groupes de textures sélectionnés dans le gestionnaire de groupes de textures sur un échantillonneur de shaders. Vous devez Vous fournissez la valeur de l'index de l'échantillonneur d'ombrage (tel que renvoyé par la fonction shader_get_sampler_index()) et la fonction renverra l'une des constantes listées ci-dessous. Syntaxe : gpu_get_tex_mip_enable_ext(sampler_index) ; Arguments Description sampler_index L'index de l'échantillonneur de shader à obtenir Les retours : Constant : Constant Description mip_off Le Mipmapping est désactivé. mip_on Le Mipmapping pour toutes les textures est activé. mip_markedonly Le Mipmapping est activé pour les textures qui l'ont activé dans les options du groupe de textures. Exemple : var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background") ; var _spriteTex = sprite_get_texture(sprite_index, 0) ; shader_set (shd_Glass) ; if gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on { gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on) ; } texture_set_stage(_sampleIndex , _spriteTex) ; shader_reset() ; Le code ci-dessus active le mipmapping pour l'échantillonneur de texture du shader donné s'il n'a pas déjà été activé.
