gpu_set_state Cette fonction va définir l'état actuel du GPU en utilisant la carte DS fournie. La carte fournie peut être créée à l'aide de la fonction gpu_get_state(). Syntaxe : gpu_set_state(ds_map) ; Arguments Description ds_map L'état du GPU à définir comme une carte DS. Les retours : N/A Exemple : gpu_set_state(gpu_map) ; Le code ci-dessus définit l'état du GPU en utilisant la carte fournie dans la variable "gpu_map".
