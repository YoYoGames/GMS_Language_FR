gpu_get_blendmode_ext_sepalpha Cette fonction peut être utilisée pour récupérer le mode de fusion étendu actuel utilisé pour le dessin, en séparant les modes de fusion en deux parties : la composante RVB et la composante Alpha. La fonction renvoie un tableau 1D à 4 éléments contenant les éléments suivants suivants : 0] = Mode de fusion source moins la composante alpha (par défaut, bm_src_alpha (As, As, As)). 1] = Mode de fusion de destination moins la composante alpha (la valeur par défaut est bm_inv_src_alpha (1-As, 1-As, 1-As)). 2] = Composante alpha de la source (la valeur par défaut est bm_src_alpha (As)) 3] = Composante alpha de destination (la valeur par défaut est bm_inv_src_alpha (1-As)) Les valeurs du tableau seront l'une des constantes suivantes ("s" désigne une valeur provenant de la source tandis qu'un "d" désigne une valeur provenant de la destination) : Constant Facteur de mélange (rouge, vert, bleu, alpha) bm_zero (0, 0, 0, 0) bm_one (1, 1, 1, 1) bm_src_colour (Rs, Gs, Bs, As) bm_inv_src_colour (1-R, 1-G, 1-B, 1-A) bm_src_alpha (As, As, As, As) bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As, 1-As) bm_dest_alpha (Ad, Ad, Ad, Ad, Ad) bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad) bm_dest_colour (Rd, Gd, Bd, Ad) bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad) bm_src_alpha_sat (f, f, f, 1) où f = min(As, 1-Ad) Lorsque vous créez les modes de fusion étendus de cette manière, les composants RVB seront tirés d'un mode de fusion et le composant alpha sera tiré d'un autre, de sorte que - par exemple - si vous utilisez bm_zero (0, 0, 0, 0) comme mode de fusion RVB et bm_one comme mode de fusion alpha, vous obtiendrez un mélange composite de (0,0,0,1). Notez que vous pouvez modifier ces valeurs directement dans le tableau et passer le tableau complet à la fonction comme un seul argument (comme indiqué dans l'exemple ci-dessous). Syntaxe : gpu_get_blendmode_ext_sepalpha() ; Les retours : Tableau 1D à 4 éléments (voir ci-dessus pour les constantes) Exemple : var bm = gpu_get_blendmode_ext_sepalpha() ; bm[0] = bm_src_alpha ; bm[1] = bm_inv_src_alpha ; gpu_set_blendmode_ext_sepalpha(bm) ; Le code ci-dessus crée un tableau local et obtient le mode de fusion étendu actuel avec des valeurs alpha séparées.
