gpu_get_texfilter_ext Avec cette fonction, vous pouvez vérifier si l'interpolation de texture (interpolation linéaire) est activée (renvoie true ) ou non (retourne false ) pour une texture d'échantillonneur de shader REMARQUE: sur la cible HTML5, cette fonction ne fonctionnera qu'avec WebGL activé. Syntaxe:gpu_get_texfilter_ext(sampler_id); Argument Description sampler_id L'identifiant de l'échantillonneur du shader. Retour:Boolean Exemple:var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler"); if gpu_get_texfilter_ext(s_tex)     {     gpu_set_texfilter_ext(s_tex, false);     } else     {     gpu_set_texfilter_ext(s_tex, true);     } Le code ci-dessus vérifie si le filtrage de texture est activé ou désactivé pour un ID d'échantillonneur spécifique (stocké dans une variable locale) et le change en conséquence.
