gpu_get_texrepeat_ext Avec cette fonction, vous pouvez vérifier si la répétition de la texture est activée (renvoie true) ou non (renvoie false) pour une texture d'échantillonneur de shader donnée. Syntaxe : gpu_get_texrepeat_ext(sampler_id) ; Arguments Description sampler_id L'identifiant de l'échantillonneur du shader. Les retours : Booléen Exemple : var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler") ; if !gpu_get_texrepeat_ext(s_tex) { gpu_set_texrepeat_ext(s_tex, true) ; } Le code ci-dessus vérifie si le filtrage de la texture est désactivé pour un ID d'échantillonneur spécifique (stocké dans une variable locale) et l'active si ce n'est pas le cas.
