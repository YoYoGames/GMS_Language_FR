gpu_get_blendmode Cette fonction peut être utilisée pour récupérer le mode de fusion actuel utilisé pour le dessin. La valeur retournée sera l'une des constantes suivantes (la valeur par défaut est bm_normal) : Constant Description Mode de mélange étendu bm_normal Mélange normal (le mode de mélange par défaut). (bm_src_alpha, bm_inv_src_alpha) bm_add Mélange additif. Les valeurs de luminosité des zones claires sont ajoutées. (bm_src_alpha, bm_one) bm_subtract Mélange soustractif. Les valeurs de luminosité des zones claires sont soustraites. (bm_zero, bm_inv_src_colour) bmax Mélange maximal. Similaire au mélange additif. (bm_src_alpha, bm_inv_src_colour) Syntaxe : gpu_get_blendmode() ; Les retours : Constante (voir ci-dessus pour les constantes) Exemple : if gpu_get_blendmode() != bm_normal { gpu_set_blendmode(bm_normal) ; } Le code ci-dessus récupère le mode de fusion actuel et s'il n'est pas bm_normal, il est réglé sur cette constante.
