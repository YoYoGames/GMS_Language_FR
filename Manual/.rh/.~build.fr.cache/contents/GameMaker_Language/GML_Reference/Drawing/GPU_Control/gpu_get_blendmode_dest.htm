gpu_get_blendmode_dest Cette fonction peut être utilisée pour récupérer le facteur de mode de fusion étendu de destination actuel utilisé pour le dessin. La valeur renvoyée sera l'une des constantes suivantes ("s" désigne une valeur provenant de la source, tandis que "d" désigne une valeur provenant de la destination) indique une valeur provenant de la destination) : Constant Facteur de mélange (rouge, vert, bleu, alpha) bm_zero (0, 0, 0, 0) bm_one (1, 1, 1, 1) bm_src_colour (Rs, Gs, Bs, As) bm_inv_src_colour (1-R, 1-G, 1-B, 1-A) bm_src_alpha (As, As, As, As) bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As, 1-As) bm_dest_alpha (Ad, Ad, Ad, Ad, Ad) bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad) bm_dest_colour (Rd, Gd, Bd, Ad) bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad) bm_src_alpha_sat (f, f, f, 1) où f = min(As, 1-Ad) Syntaxe : gpu_get_blendmode_dest() ; Les retours : Constante (voir tableau ci-dessus) Exemple : var bm ; bm[0] = gpu_get_blendmode_src() ; bm[1] = gpu_get_blendmode_dest() ; gpu_set_blendmode_ext_sepalpha(bm[0], bm[1], bm_inv_src_alpha, bm_inv_dest_colour) ; Le code ci-dessus crée un tableau local et obtient les facteurs de mélange source et destination actuels. Ce tableau est ensuite utilisé pour manipuler la composante alpha des facteurs de mélange.
