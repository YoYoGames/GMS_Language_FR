gpu_pop_state Cette fonction extrait l'état précédent du GPU de la pile et l'applique. Voir gpu_push_state() pour plus d'informations. Syntaxe : gpu_pop_state() ; Les retours : N/A Exemple : gpu_push_state() ; gpu_set_blendmode(bm_add) ; gpu_set_blendenable(false) ; gpu_set_cullmode(true) ; with (obj_Effect_Parent) { draw_self() ; } gpu_pop_state() ; Le code ci-dessus va "enregistrer" l'état actuel du GPU sur la pile, puis modifier certains paramètres du GPU et dessiner un groupe d'instances avant de réinitialiser l'état du GPU à ce qu'il était auparavant.
