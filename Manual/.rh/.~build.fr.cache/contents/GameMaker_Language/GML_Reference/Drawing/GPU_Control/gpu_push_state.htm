gpu_push_state Avec cette fonction, vous pouvez pousser l'état actuel du GPU sur une pile pour l'utiliser plus tard. Vous utiliserez généralement cette fonction si vous souhaitez "sauvegarder" l'état actuel du GPU (des éléments tels que le mode de fusion, l'écriture alpha, le culling, etc... seront tous poussés vers la pile ), puis dessiner quelque chose avec des paramètres différents, et ensuite réinitialiser la pile GPU à ce qu'elle était auparavant (en appelant gpu_pop_state()). Syntaxe : gpu_push_state() ; Les retours : N/A Exemple : gpu_push_state() ; gpu_set_blendmode(bm_add) ; gpu_set_blendenable(false) ; gpu_set_cullmode(true) ; with (obj_Effect_Parent) { draw_self() ; } gpu_pop_state() ; Le code ci-dessus va "enregistrer" l'état actuel du GPU sur la pile, puis modifier certains paramètres du GPU et dessiner un groupe d'instances avant de réinitialiser l'état du GPU à ce qu'il était auparavant.
