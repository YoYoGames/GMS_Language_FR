gpu_set_alphatestenable Cette fonction permet d'activer ou de désactiver le test alpha pour votre jeu (par défaut, il est désactivé). En activant le test alpha, vous pouvez alors utiliser la fonction complémentaire gpu_set_alphatestref() pour définir la valeur de "cut-off" à laquelle toutes les valeurs alpha seront mises à 0. L'image ci-dessous montre la différence que l'on peut observer lorsque le test d'alpha est activé ou désactivé : REMARQUE: cette fonction peut affecter négativement les performances des appareils iOS et Android. Syntaxe : gpu_set_alphatestenable(enable) ; Arguments Description activer Activer ou désactiver le test alpha(true / false) Les retours : N/A Exemple : if !gpu_get_alphatestenable() { gpu_set_alphatestenable(true) ; gpu_set_alphatestref(128) ; } Le code ci-dessus vérifie si le test d'alpha est activé et, si ce n'est pas le cas, il active le test d'alpha et fixe le seuil de test à 128 (seuls les pixels dont l'alpha est supérieur à 0,5 seront dessinés).
