gpu_get_alphatestref Vous pouvez utiliser cette fonction pour trouver la valeur actuelle de la référence de test alpha (la valeur par défaut est 0, mais vous pouvez utiliser gpu_set_alphatestref() pour définir cette valeur sur autre chose). Syntaxe:gpu_get_alphatestref(); Retour:Real (from 0 - 255) Exemple:if gpu_get_alphatestenable()     {     if gpu_get_alphatestref() < 254         {         gpu_set_alphatestref(254);         }     } Le code ci-dessus vérifie si le test alpha est activé, et si tel est le cas, il vérifie la valeur de référence du test alpha actuelle et la définit si elle est inférieure à 254.
