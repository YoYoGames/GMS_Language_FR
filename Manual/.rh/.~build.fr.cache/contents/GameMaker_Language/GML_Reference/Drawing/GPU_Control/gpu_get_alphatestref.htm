gpu_get_alphatestref Vous pouvez utiliser cette fonction pour trouver la valeur actuelle de la référence du test alpha (la valeur par défaut est 0, mais vous pouvez utiliser la fonction gpu_set_alphatestref() pour définir cette valeur à une autre valeur). Syntaxe : gpu_get_alphatestref() ; Les retours : Réel (de 0 à 255) Exemple : if gpu_get_alphatestenable() { if gpu_get_alphatestref() < 254 { gpu_set_alphatestref(254) ; } } Le code ci-dessus vérifie si le test alpha est activé, et si c'est le cas, il vérifie la valeur de référence actuelle du test alpha et la définit si elle est inférieure à 254.
