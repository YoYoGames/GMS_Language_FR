gpu_get_ztestenable Cette fonction peut être utilisée pour savoir si le test z est activé (la fonction renvoie true ) ou non (la fonction renvoie false ). La valeur par défaut est que z-testing est désactivé, donc la fonction retournera false. Syntaxe:gpu_get_ztestenable(); Retour:Boolean Exemple:if gpu_get_ztestenable() == false     {     gpu_set_ztestenable(true);     } Le code ci-dessus vérifie si le test z est activé ou non et s'il est désactivé, il est ensuite à nouveau activé.
