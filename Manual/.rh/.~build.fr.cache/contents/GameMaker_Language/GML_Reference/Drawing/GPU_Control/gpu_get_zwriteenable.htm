gpu_get_zwriteenable Cette fonction peut être utilisée pour savoir si l'écriture en z est activée (la fonction renvoie true) ou non (la fonction renvoie false). La valeur par défaut est que l'écriture en z est activée, la fonction renvoie donc true. Syntaxe : gpu_get_zwriteenable() ; Les retours : Booléen Exemple : if gpu_get_zwriteenable() == false { gpu_set_zwriteenable(true) ; } Le code ci-dessus vérifie si l'écriture en z est activée ou non et si elle est désactivée, elle est alors réactivée.
