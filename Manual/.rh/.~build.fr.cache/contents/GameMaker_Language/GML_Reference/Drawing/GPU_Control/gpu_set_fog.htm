gpu_set_fog Cette fonction peut être utilisée pour activer ou désactiver le dessin du brouillard. Le brouillard peut être utilisé dans les jeux 3D pour rendre les instances éloignées floues ou même disparaître, ce qui permet de créer une atmosphère et de masquer le fait que vous ne dessinez pas les instances éloignées. qui sont éloignées. Vous définissez s'il est activé(true) ou désactivé(false), la couleur que le brouillard doit utiliser pour le mélange, ainsi que les distances de dessin de début et de fin. Notez que le début du brouillard indique le rayon relatif à la caméra où le brouillard commence. où le brouillard commence, et la fin du brouillard indique à quel rayon (également relatif) les instances seront complètement invisibles. La fonction peut prendre quatre arguments individuels (donnés ci-dessus) ou un tableau avec la structure suivante (l'exemple de code ci-dessous montre cette méthode) : 0] = activation de l'interrupteur à bascule (un booléen, soit vrai ou faux), par défaut faux. [1] = couleur (réelle), par défaut c_black [2] = distance de départ (réelle), par défaut 0 [3] = distance finale (réelle), par défaut 1 Syntaxe : gpu_set_fog(enable, colour, start, end) ; Arguments Description activer Activer ou désactiver le brouillard couleur La couleur du brouillard commencer La distance à partir de laquelle commencer à appliquer le brouillard fin La distance pour mettre fin au brouillard Les retours : N/A Exemple : var fog_a = gpu_get_fog() ; fog_a[1] = c_red ; gpu_set_fog(fog_a) ; Le code ci-dessus récupère les paramètres actuels du brouillard et définit ensuite l'élément couleur du tableau à c_red avant de définir à nouveau le brouillard en utilisant le tableau modifié.
