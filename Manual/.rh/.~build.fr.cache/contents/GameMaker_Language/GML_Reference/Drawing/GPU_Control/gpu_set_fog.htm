gpu_set_fog Cette fonction peut être utilisée pour activer ou désactiver le dessin de brouillard. Le brouillard peut être utilisé dans les jeux 3D pour rendre les instances éloignées floues ou même disparaître, ce qui aide à créer une atmosphère et à masquer le fait que vous ne dessinez pas des instances éloignées. Vous définissez s'il est activé (true ) ou désactivé (false ), la couleur que le brouillard doit utiliser pour le mélange, ainsi que les distances de dessin de début et de fin. Notez que le début du brouillard indique le rayon par rapport à la caméra de vue où le brouillard commence, et la fin du brouillard indique à quel rayon (également relatif) les instances seront complètement invisibles. La fonction peut prendre quatre arguments individuels (donnés ci-dessus) ou un tableau avec la structure suivante (l'exemple de code ci-dessous montre cette méthode): [0] = bascule activée (une valeur booléenne, soit true ou false ), défaut false [1] = couleur (réel), par défaut c_black [2] = distance de départ (réelle), par défaut 0 [3] = distance finale (réelle), par défaut 1 Syntaxe:gpu_set_fog(enable, colour, start, end); Argument Description enable Activer ou désactiver le brouillard colour La couleur du brouillard start La distance à partir de laquelle commencer à appliquer le brouillard end La distance pour mettre fin au brouillard Retour:N/A Exemple:var fog_a = gpu_get_fog(); fog_a[1] = c_red; gpu_set_fog(fog_a); Le code ci-dessus obtient les paramètres de brouillard actuels, puis définit l'élément de couleur du tableau sur c_red avant de régler à nouveau le brouillard en utilisant la matrice modifiée.
