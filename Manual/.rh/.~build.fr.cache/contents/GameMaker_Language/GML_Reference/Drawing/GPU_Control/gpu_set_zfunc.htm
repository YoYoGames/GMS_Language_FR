gpu_set_zfunc Cette fonction peut être utilisée pour définir le mode de comparaison des tests du tampon zéro (voir gpu_set_ztestenable() pour plus d'informations). Les valeurs disponibles sont n'importe laquelle des constantes suivantes (la valeur par défaut est est cmpfunc_lessequal) : cmpfunc_never cmpfunc_less cmpfunc_equal cmpfunc_lessequal cmpfunc_greater cmpfunc_notequal cmpfunc_greaterequal cmpfunc_always Syntaxe : gpu_set_zfunc(cmp_func) ; Arguments Description cmp_func Le mode de comparaison à utiliser (voir liste ci-dessus) Les retours : N/A Exemple : gpu_set_ztestenable(true) ; gpu_set_zfunc(cmpfunc_always) ; draw_sprite(spr_Background, 0, 0, 0) ; gpu_set_ztestenable(false) ; Le code ci-dessus active le test du z-buffer et définit son mode de comparaison avant de dessiner un sprite d'arrière-plan, puis le désactive à nouveau pour poursuivre le dessin.
