gpu_set_texfilter_ext Cette fonction peut être utilisée pour définir l'interpolation linéaire pour un seul "slot" d'échantillonneur lors de l'utilisation de Shaders dans GameMaker Studio 2. Lorsque cette fonction est activée(true), la texture de l'échantillonneur sera lissée. sera lissée et si elle est désactivée(false), les images seront dessinées sur la base du pixel le plus proche. La valeur par défaut est celle définie par les options globales de votre jeu, ou celle définie à l'aide de la fonction gpu_set_texfilter(). NOTE: Ce paramètre sera remplacé par la valeur définie lors de l'appel de la fonction gpu_set_texfilter(). REMARQUE: sur la cible HTML5, cette fonction ne fonctionnera que si WebGL est activé. Syntaxe : gpu_set_texfilter_ext(sampler_id, enable) ; Arguments Description sampler_id L'identifiant de l'échantillonneur du shader. activer Activer ou désactiver le filtrage des textures(true / false) Les retours : N/A Exemple : var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler") ; if gpu_get_texfilter_ext(s_tex) { gpu_set_texfilter_ext(s_tex, false) ; } else { gpu_set_texfilter_ext(s_tex, true) ; } Le code ci-dessus vérifie si le filtrage de texture est activé ou non pour un ID d'échantillonneur spécifique (stocké dans une variable locale) et l'active en conséquence.
