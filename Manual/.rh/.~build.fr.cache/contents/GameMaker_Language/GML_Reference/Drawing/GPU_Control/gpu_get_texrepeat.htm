gpu_get_texrepeat Cette fonction permet de vérifier si la répétition de la texture est activée (renvoie true) ou non (renvoie false). Pour plus d'informations sur la répétition des textures, voir la fonction gpu_set_texrepeat(). Syntaxe : gpu_get_texrepeat() ; Les retours : Booléen Exemple : if gpu_get_texrepeat() { gpu_set_texrepeat(false) ; } else { gpu_set_texrepeat(true) ; } Le code ci-dessus vérifie si la répétition de la texture est activée ou non et l'active en conséquence.
