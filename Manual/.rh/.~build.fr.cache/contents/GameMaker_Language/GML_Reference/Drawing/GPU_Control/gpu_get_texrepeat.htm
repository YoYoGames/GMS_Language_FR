gpu_get_texrepeat Avec cette fonction, vous pouvez vérifier si la répétition de texture est activée (renvoie true ) ou non (retourne false ). Pour plus d'informations sur la répétition de texture, consultez la fonction gpu_set_texrepeat(). Syntaxe:gpu_get_texrepeat(); Retour:Boolean Exemple:if gpu_get_texrepeat()     {     gpu_set_texrepeat(false);     } else     {     gpu_set_texrepeat(true);     } Le code ci-dessus vérifie si la répétition de texture est activée ou désactivée et la modifie en conséquence.
