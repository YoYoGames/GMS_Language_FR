gpu_get_blendenable Cette fonction peut être utilisée pour récupérer l'état de fusion alpha. S'il revient true alors l'alpha-blending est activé, et s'il retourne false il est désactivé. Par défaut, il est activé et donc la fonction retournera true Syntaxe:gpu_get_blendenable(); Retour:Boolean Exemple:if gpu_get_blendenable() == false     {     gpu_set_blendenable(true);     } Le code ci-dessus vérifie l'état du mélange alpha et s'il est désactivé, il est à nouveau activé.
