gpu_get_colourwriteenable Cette fonction peut être utilisée pour récupérer les valeurs actuelles d'activation d'écriture des couleurs. La fonction renvoie un tableau 1D à 4 éléments avec les éléments suivants qui seront soit vrais (activés) soit faux (désactivés). Par défaut, l'écriture de toutes les couleurs par défaut, l'écriture de toutes les couleurs est réglée sur true: 0] = Canal rouge activé/désactivé 1] = Canal vert activé/désactivé [2] = Canal bleu activé/désactivé 3] = Canal alpha activé/désactivé Syntaxe : gpu_get_colorwriteenable() ; Les retours : Réseau 1D à 4 éléments (voir ci-dessus pour les détails) Exemple : var cw = gpu_get_colorwriteenable() ; cw[3] = false ; gpu_set_colorwriteenable(cw) ; Le code ci-dessus récupère les valeurs d'écriture de la couleur actuelle, puis définit la composante alpha à false.
