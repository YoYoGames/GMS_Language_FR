gpu_set_blendmode_ext_sepalpha Lorsque GameMaker Studio 2 dessine un pixel, il y a une couleur source (la couleur du pixel que nous allons dessiner) et une couleur de destination (la couleur qui se trouve déjà dans le pixel que nous dessinons). le pixel, les couleurs source et destination sont calculées en fonction du mode de fusion choisi. Chaque composante des couleurs est stockée sous la forme d'une valeur à virgule flottante comprise entre 0 et 1, et la nouvelle couleur est calculée en multipliant chaque composante de la couleur source par un facteur et en multipliant chaque composante de la couleur de destination par un autre facteur. source par un certain facteur et en multipliant chaque composante de la couleur de destination par un autre facteur, puis en additionnant les résultats composante par composante. Cette fonction vous permet de définir les différentes parties de composants qui doivent être pondérées ensemble pour créer un mode de fusion personnalisé, en sélectionnant différentes parties de différents facteurs de mode de fusion pour les composants RVB et le composant Alpha. Ainsi, les arguments "src" et "dest" utiliseront les 3 premières valeurs du mode de fusion (RGB) et les arguments "alphasrc" et "alphadest" prendront la dernière partie du facteur de mode de fusion. Par exemple, si vous avez la source RGB comme bm_zero, alors (0, 0, 0) sera utilisé. Si vous définissez ensuite la source Alpha comme bm_one, alors (1) sera utilisé, donnant un mode de fusion final de la source de (0, 0, 0, 1). Le tableau suivant montre les valeurs RGBA pour chaque mode de fusion, où (Rs, Gs, Bs, As) sont considérées comme les valeurs de source et (Rd, Gd, Bd, Ad) comme les valeurs de destination. Les onze constantes disponibles peuvent être utilisées comme source ou comme destination (ou les deux). (ou les deux) lorsqu'elles sont utilisées dans cette fonction. Constant Facteur de mélange (rouge, vert, bleu, alpha) bm_zero (0, 0, 0, 0) bm_one (1, 1, 1, 1) bm_src_colour (Rs, Gs, Bs, As) bm_inv_src_colour (1-R, 1-G, 1-B, 1-A) bm_src_alpha (As, As, As, As) bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As, 1-As) bm_dest_alpha (Ad, Ad, Ad, Ad, Ad) bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad) bm_dest_colour (Rd, Gd, Bd, Ad) bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad) bm_src_alpha_sat (f, f, f, 1) où f = min(As, 1-Ad) IMPORTANT ! HTML5 sans WebGL activé ne sera pas en mesure d'afficher correctement les modes suivants : bm_src_colour bm_inv_src_colour bm_dest_colour bm_inv_dest_colour bm_src_alpha_sat Notez que vous pouvez soit fournir quatre arguments individuels à cette fonction, soit fournir un tableau d'arguments (tel que retourné par la fonction gpu_get_blendmode_ext_sepalpha() par exemple). Si vous fournissez un tableau, il doit contenir les quatre éléments suivants : 0] = Mode de fusion source moins la composante alpha (par défaut, bm_src_alpha (As, As, As)). 1] = Mode de fusion de destination moins la composante alpha (la valeur par défaut est bm_inv_src_alpha (1-As, 1-As, 1-As)). 2] = Composante alpha de la source (la valeur par défaut est bm_src_alpha (As)) 3] = Composante alpha de destination (la valeur par défaut est bm_inv_src_alpha (1-As)) Pour vous aider à tirer le meilleur parti des modes de fusion et à comprendre comment ils fonctionnent et comment ils affectent l'image finale dessinée à l'écran, nous vous recommandons de lire le guide suivant : Guide d'utilisation des Blendmodes Syntaxe : gpu_set_blendmode(src, dest, alphasrc, alphadest) ; Arguments Description src Facteur du mode de fusion source pour RGB (voir les constantes ci-dessus). dest Facteur du mode de fusion de destination pour RGB (voir les constantes ci-dessus) alphasrc Facteur du mode de fusion source Alpha (voir les constantes ci-dessus). alphadest Facteur du mode de fusion de destination Alpha (voir les constantes ci-dessus) Les retours : N/A Exemple : var bm = gpu_get_blendmode_ext_sepalpha() ; bm[0] = bm_src_alpha ; bm[1] = bm_inv_src_alpha ; gpu_set_blendmode_ext_sepalpha(bm) ; Le code ci-dessus crée un tableau local et obtient le mode de fusion étendu actuel avec des valeurs alpha séparées.
