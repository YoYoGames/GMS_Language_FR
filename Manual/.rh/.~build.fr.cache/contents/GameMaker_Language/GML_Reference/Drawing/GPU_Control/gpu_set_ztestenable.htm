gpu_set_ztestenable Cette fonction peut être utilisée pour activer ou désactiver le test du z-buffer, ce qui affecte la façon dont les objets seront dessinés (en général, cette fonction n'est utile que pour les projets 3D). Essentiellement, par défaut, lorsque le test du z est désactivé et que vous avez deux objets dessinés dans le même espace, les deux objets seront rendus sans tenir compte du fait que l'un surchargera l'autre, ce qui entraînera des appels de dessin inutiles. Si vous l'activez, le z-buffer est testé pour voir si un objet est "visible" et n'est pas dessiné s'il ne l'est pas. pas. Notez que c'est le comportement par défaut, mais vous pouvez le modifier en changeant le type de comparaison utilisé pour le test du z-buffer (voir la fonction gpu_set_zfunc(). Par défaut, le test du z-buffer est désactivé(faux). Syntaxe : gpu_set_ztestenable(enable) ; Arguments Description activer Active ou désactive le test du z-buffer(true ou false). Les retours : N/A Exemple : gpu_set_ztestenable(true) ; draw_sprite(spr_Background, 0, 0, 0) ; gpu_set_ztestenable(false) ; Le code ci-dessus active le test du z-buffer pour dessiner un sprite d'arrière-plan, puis le désactive à nouveau pour continuer à dessiner.
