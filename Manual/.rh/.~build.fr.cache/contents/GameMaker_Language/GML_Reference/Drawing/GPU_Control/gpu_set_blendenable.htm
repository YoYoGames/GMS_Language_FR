gpu_set_blendenable Cette fonction peut être utilisée pour activer ou désactiver le mélange alpha. En gros, si vous mettez cette fonction à false, toutes les images dessinées seront 100% opaques, ce qui signifie que toutes les zones transparentes ou semi-transparentes d'un sprite ou d'un arrière-plan seront visibles en tant que couleur solide. seront visibles sous la forme d'une couleur unie. Il est conseillé de désactiver le mélange alpha dans la mesure du possible (en particulier lors du développement pour les appareils mobiles), car cela augmente considérablement la vitesse d'affichage. Syntaxe : gpu_set_blendenable(enable) ; Arguments Description activer Active ou désactive la valeur du mélange alpha(true ou false). Les retours : N/A Exemple : gpu_set_blendenable(false) ; draw_sprite(spr_Background, 0, 0, 0) ; gpu_set_blendenable(true) ; Le code ci-dessus désactive le mélange alpha pour dessiner un sprite d'arrière-plan, puis le réactive pour continuer à dessiner.
