gpu_set_texrepeat Cette fonction peut être utilisée pour indiquer à GameMaker Studio 2 si les textures doivent se répéter (true ) ou non (false ) lorsqu'il est utilisé pour les modèles et les primitives. Normalement, les largeurs et hauteurs de texture sont comprises entre 0 et 1, mais si vous spécifiez une valeur supérieure à 1 par défaut, le reste n'est pas dessiné (toute la texture sera dessinée avec une section "vide" correspondant au pourcentage sur 1 qui est Ainsi, une valeur de 2 pour une largeur ou une hauteur de texture dessinerait la texture dans la moitié de l'espace et laisserait l'autre moitié vide). Cependant, en utilisant cette fonction pour définir la répétition sur vrai, la texture est répétée automatiquement sur tous les modèles et primitives. Le cas d'utilisation probable de ces fonctions est de répéter une texture en 3D, mais pour que cela fonctionne et ne pas extraire des images du reste de la page de texture, le sprite utilisé devra être marqué comme étant sur une "Page de texture séparée" dans l'éditeur de sprites. REMARQUE: les sprites et les arrière-plans sont toujours dessinés sans se répéter, donc une fois que vous dessinez un sprite ou un arrière-plan, cette valeur est réinitialisée à false. Ce comportement par défaut n'affectera pas les paramètres de l'étape de texture (c'est-à-dire: l'appel de la fonction le sera, mais le paramètre interne sur false ne le sera pas). REMARQUE: ce paramètre remplacera toutes les répétitions d'étape de texture définies pour les shaders lors de l'utilisation de la fonction gpu_set_texrepeat_ext(). Syntaxe:gpu_set_texrepeat(enable); Argument Description enable Activer ou désactiver le filtrage des textures (true /false ) Retour:N/A Exemple:if gpu_get_texrepeat()     {     gpu_set_texrepeat(false);     } else     {     gpu_set_texrepeat(true);     } Le code ci-dessus vérifie si la répétition de texture est activée ou désactivée et la modifie en conséquence.
