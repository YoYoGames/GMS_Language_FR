gpu_get_texfilter Avec cette fonction, vous pouvez vérifier si le filtrage de texture (interpolation linéaire) est activé (renvoie true ) ou non (retourne false ). Pour plus d'informations sur le filtrage des textures, consultez la fonction gpu_set_texfilter(). REMARQUE: sur la cible HTML5, cette fonction ne fonctionnera qu'avec WebGL activé. Syntaxe:gpu_get_texfilter(); Retour:Boolean Exemple:if gpu_get_texfilter()     {     gpu_set_texfilter(false);     } else     {     gpu_set_texfilter(true);     } Le code ci-dessus vérifie si le filtrage de texture est activé ou désactivé et le règle en conséquence.
