gpu_get_texfilter Cette fonction permet de vérifier si le filtrage des textures (interpolation linéaire) est activé (renvoie true) ou non (renvoie false). Pour plus d'informations sur le filtrage des textures, voir la fonction gpu_set_texfilter(). REMARQUE: sur la cible HTML5, cette fonction ne fonctionnera que si WebGL est activé. Syntaxe : gpu_get_texfilter() ; Les retours : Booléen Exemple : if gpu_get_texfilter() { gpu_set_texfilter(false) ; } else { gpu_set_texfilter(true) ; } Le code ci-dessus vérifie si le filtrage des textures est activé ou non et l'active en conséquence.
