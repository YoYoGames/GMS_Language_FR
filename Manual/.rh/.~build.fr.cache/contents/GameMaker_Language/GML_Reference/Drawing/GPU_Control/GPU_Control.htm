Contrôle GPU Lorsque vous dessinez des objets dans une pièce, vous aurez souvent besoin d'un contrôle plus granulaire sur la manière exacte dont le dessin sera fait. Vous souhaiterez peut-être qu'il n'y ait pas de fusion alpha, ou vous pourriez avoir besoin d'une élimination, ou vous devrez peut-être activer ou désactiver l'interpolation de texture. Tout cela est possible - et bien plus encore! - en utilisant les fonctions du contrôleur GPU répertoriées ci-dessous. Notez que la plupart de ces fonctions ont une portée globale et qu'une fois définies ou modifiées, elles affecteront la façon dont tout est dessiné à partir de là.
