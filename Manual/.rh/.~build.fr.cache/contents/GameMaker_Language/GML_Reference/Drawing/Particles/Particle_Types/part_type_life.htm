part_type_life C'est la fonction qui régit la durée pendant laquelle chaque particule individuelle du type indiqué reste à l'écran. Comme pour les autres fonctions de particules, vous fournissez une valeur minimale et une valeur maximale (en étapes) et la durée de vie de chaque particule sera un nombre aléatoire d'étapes dans la plage spécifiée. Pour que toutes les particules aient la même durée de vie, définissez les deux valeurs pour qu'elles soient identiques. Syntaxe:part_type_life(ind, life_min, life_max); Argument Description ind L'indice du type de particule à modifier. life_min La durée de vie minimale des particules. life_max La durée de vie maximale des particules. Retour:N/A Exemple:part_type_life(part_Flare, room_speed, room_speed * 2); Le code ci-dessus définira toutes les particules créées du type de particule indexé dans la variable "part_Flare" pour avoir une durée de vie comprise entre 1 et 2 secondes, en fonction de la vitesse de la pièce actuelle.
