part_type_colour_mix Avec cette fonction, vous pouvez définir le type de particule donné comme un mélange aléatoire de deux couleurs. Syntaxe:part_type_colour_mix(ind, colour1, colour2); Argument Description ind L'indice du type de particule à modifier. colour1 La première couleur à partir de laquelle le mélange prendra. colour2 La deuxième couleur à partir de laquelle le mélange prendra. Retour:N/A Exemple:part_type_colour_mix(global.P_Damage, c_red, c_yellow); Le code ci-dessus définira la couleur de chaque particule émise du type de particule indexé dans la variable globale "P_Damage" pour être un mélange aléatoire entre les couleurs rouge et jaune.
