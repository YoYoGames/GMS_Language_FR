part_type_shape Avec cette fonction, vous pouvez définir la forme de sprite à utiliser pour le type de particule, les constantes suivantes étant acceptables: Constant Description pt_shape_pixel Un pixel 1x1. (Ce sont les paramètres par défauts.) pt_shape_disk Un cercle plein. pt_shape_square Un carré plein. pt_shape_line Une ligne horizontale de 8 pixels de large. pt_shape_star Une étoile remplie à cinq branches. pt_shape_circle Un cercle entouré de 3 pixels. pt_shape_ring Un cercle avec une lueur intérieure (ressemble à une bulle). pt_shape_sphere Un cercle avec une lueur vers l'extérieur «solide au milieu, brillant vers l'extérieur. pt_shape_flare Un point durement brillant (qui ressemble à une vraie étoile dans la nuit). pt_shape_spark Un effet d'étincelle 'comme une étoile avec plusieurs points qui disparaissent. pt_shape_explosion Un nuage de fumée carré »nécessite plusieurs couleurs pour ressembler à une explosion. pt_shape_cloud Un nuage mince nécessite une mise à l'échelle et plusieurs particules pour ressembler à un nuage. pt_shape_smoke Une version douce de l'effet d'explosion. Utilisez plusieurs pour créer un nuage de fumée. pt_shape_snow Une forme générique de flocon de neige. Syntaxe:part_type_shape(ind, shape); Argument Description ind L'indice du type de particule à définir. shape La forme à choisir. Retour:N/A Exemple:global.p1 = part_type_create(); part_type_shape(global.p1, pt_shape_square); part_type_size(global.p1, 1, 3, 0, 0); part_type_scale(global.p1, 1, 1); part_type_colour1(global.p1, c_white); part_type_alpha2(global.p1, 1, 0); part_type_speed(global.p1, 2, 4, 0, 0); part_type_direction(global.p1, 0, 180, 0, 0); part_type_gravity(global.p1, 0.20, 270); part_type_orientation(global.p1, 0, 0, 0, 0, 1); part_type_blend(global.p1, 1); part_type_life(global.p1, 15, 60); Le code ci-dessus définit le type de particule indexé dans la variable globale "p1" sur la forme d'un carré, puis définit diverses autres propriétés de particule.
