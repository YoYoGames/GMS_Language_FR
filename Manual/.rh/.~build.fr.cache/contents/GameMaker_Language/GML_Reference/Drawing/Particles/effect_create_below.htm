effet_créer_dessous Avec cette fonction, vous pouvez créer un effet simple sous toutes les instances de votre pièce (il est en fait créé à une profondeur de 100000). Si l'effet est autre que ef_rain ou ef_snow, vous pouvez définir une position x/y pour créer l'effet. et la taille peut être une valeur de 0, 1 ou 2, où 0 est petit, 1 est moyen et 2 est grand. Il est intéressant de noter que ces effets peuvent être activés et désactivés, et que leur dessin peut être mis en pause, en utilisant les fonctions part_system_automatic_draw() et part_system_automatic_update() avec la valeur appropriée pour l'indice du système de particules (où 0 correspond aux effets inférieurs et 1 aux effets supérieurs). Les constantes disponibles pour les différents types de particules sont les suivantes : Constant Exemple Description ef_cloud Un effet qui crée des particules de nuage aléatoires de différentes tailles. ef_ellipse Un effet qui crée des ellipses en expansion ef_explosion Un effet qui crée des explosions en expansion qui s'estompent ef_firework Un effet qui crée de multiples petites particules pour générer une explosion de feu d'artifice. ef_flare Un effet qui génère un point brillant qui s'allume et s'éteint. ef_rain Un effet qui génère des particules de pluie descendant du haut de l'écran. ef_ring Un effet qui génère des cercles en expansion et en décroissance ef_smoke Un effet qui génère de petites bouffées de fumée. ef_smokeup Un effet qui crée un panache de fumée qui monte à l'écran. ef_snow Un effet qui génère de multiples particules de neige tombant sur l'écran. ef_spark Un effet qui génère une petite étincelle ef_star Un effet qui génère des particules d'étoiles Syntaxe : effet_create_below(kind, x, y, size, color) ; Arguments Description genre Le type d'effet (utilisez l'une des constantes énumérées ci-dessus). x Le positionnement x de l'effet, le cas échéant. y Le positionnement y de l'effet, le cas échéant. taille L'ampleur de l'effet. couleur La couleur de l'effet. Les retours : N/A Exemple : si vitesse > 0 { effect_create_below(ef_smoke, x, y, 0, c_gray) ; } Le code ci-dessus créera une petite bouffée de fumée grise à chaque fois que la vitesse de l'instance sera supérieure à 0 aux coordonnées x,y de l'instance.
