partie_type_couleur2 Cette fonction peut être utilisée pour définir un gradient de deux couleurs pour chaque particule créée du type donné. La première couleur est celle avec laquelle toutes les particules commenceront, et la deuxième couleur est celle avec laquelle la particule finira. d'une couleur à l'autre au cours de la vie de la particule. Syntaxe : partie_type_couleur2(ind, couleur1, couleur2) ; Arguments Description ind L'indice du type de particule à modifier. couleur1 La couleur de départ de la particule. couleur2 La couleur à laquelle la particule se terminera. Les retours : N/A Exemple : part_type_colour2( part_Health, c_red, c_white) ; Le code ci-dessus fera en sorte que toutes les particules créées du type indexé dans la variable "part_Health" aient un mélange de couleurs du rouge au blanc pendant leur durée de vie.
