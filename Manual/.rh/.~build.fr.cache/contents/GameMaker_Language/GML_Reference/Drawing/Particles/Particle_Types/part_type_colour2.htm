part_type_colour2 Cette fonction peut être utilisée pour définir un dégradé de deux couleurs pour chaque particule créée du type donné. La première couleur est celle avec laquelle toutes les particules commenceront, et la deuxième couleur est celle sur laquelle la particule se terminera, et un changement de gradient progressif se produira dans la couleur au cours de la durée de vie des particules d'une couleur à l'autre. Syntaxe:part_type_colour2(ind, colour1, colour2); Argument Description ind L'indice du type de particule à modifier. colour1 La couleur à laquelle la particule commencera. colour2 La couleur à laquelle la particule se terminera. Retour:N/A Exemple:part_type_colour2( part_Health, c_red, c_white); Le code ci-dessus fera que toutes les particules créées du type indexé dans la variable "part_Health" auront un mélange de couleur du rouge au blanc pendant leur durée de vie.
