vitesse_type_partie Cette fonction est utilisée pour déterminer la vitesse de la particule lors de sa création et peut également être utilisée pour que la particule augmente ou diminue sa vitesse au cours de sa vie. Les valeurs de vitesse minimale et maximale sont fixées par défaut à 1, ce qui signifie que la particule se déplacera d'un pixel à chaque pas. Si elles sont définies sur la même valeur, la vitesse initiale de toutes les particules sera la même, mais si elles sont définies sur des valeurs différentes, chaque particule commencera avec une vitesse aléatoire comprise dans la plage donnée. Par exemple, si vous avez un minimum de 3 et un maximum de 6, toutes les particules seront créées avec une vitesse comprise entre trois et six pixels par pas. Vous pouvez également définir une valeur d'incrémentation qui fera en sorte que la particule se déplace plus rapidement (s'il s'agit d'un nombre positif) ou plus lentement (s'il s'agit d'un nombre négatif) au cours de sa durée de vie, le montant donné étant ajouté à la vitesse de la particule à chaque étape. Cette valeur peut être un minimum de (+/-)0,01. Enfin, vous pouvez définir le facteur "wiggle". Il s'agit d'une valeur qui sera ajoutée ou soustraite aléatoirement à la vitesse à chaque étape de la vie des particules. Évidemment, les grandes valeurs sont plus prononcées que les petites, et cette valeur peut même être négative. négative, la plage maximale étant comprise entre -20 et 20. Syntaxe : part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle) ; Arguments Description ind L'indice du type de particule à modifier. vitesse_min La vitesse minimale à laquelle la particule peut démarrer. vitesse_max La vitesse maximale à laquelle la particule peut démarrer. vitesse_incr De combien la vitesse des particules doit augmenter ou diminuer par étape. speed_wiggle Quelle quantité doit être ajoutée ou soustraite aléatoirement de la vitesse de la particule par étape. Les retours : N/A Exemple : part_type_shape(particle2, pt_shape_flare) ; part_type_size(particle2, 0.01, 0.50, 0.01, 20) ; part_type_scale(particle2, 0.10, 0.10) ; part_type_colour3(particle2, 4210816, 4227327, 8240895) ; part_type_alpha3(particle2, 0.20, 0.80, 0) ; part_type_speed(particle2, 5, 10, -0.20, 20) ; part_type_direction(particle2, 0, 359, -1, 20) ; part_type_gravity(particle2, 0, 270) ; part_type_orientation(particle2, 0, 359, 0, 20, 1) ; part_type_blend(particle2, 1) ; part_type_life(particle2, 1, 20) ; Le code ci-dessus définira diverses valeurs de particules, y compris la vitesse qui fixe une vitesse minimale de cinq et une vitesse maximale de dix, avec un ralentissement de la vitesse d'une valeur de -0,2 à chaque étape.
