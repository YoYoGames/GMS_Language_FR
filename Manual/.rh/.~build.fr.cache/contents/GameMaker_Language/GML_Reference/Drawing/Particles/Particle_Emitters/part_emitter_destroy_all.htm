partie_émetteur_destroy_all Cette fonction supprimera tous les émetteurs définis du système donné et les effacera de la mémoire (cela empêchera également toute particule d'être produite par l'émetteur donné, mais ne les supprimera PAS de la pièce). Cette fonction devrait toujours être appelée lorsque les émetteurs ne sont plus nécessaires pour le système afin d'éviter les fuites de mémoire et les erreurs. Syntaxe : part_emitter_destroy_all( ps ) ; Arguments Description ps Le système de particules dont il faut détruire tous les émetteurs. Les retours : N/A Exemple : si vies = 0 { part_emitter_destroy_all(global.Sname) ; room_goto(rm_Menu) ; } Le code ci-dessus vérifie la variable globale intégrée "lives" et si elle est égale à 0, il détruit tous les émetteurs de particules et change de pièce.
