partie_type_couleur_hsv Avec cette fonction, vous pouvez définir une gamme de teintes, de saturations et de valeurs pour toutes les particules du type donné. Vous fournissez une valeur minimale et une valeur maximale pour chacun des trois composants et les particules créées auront une couleur aléatoire basée sur la gamme de paramètres donnée. gamme de paramètres donnée. De cette façon, vous pouvez créer des particules de la même teinte mais de saturations différentes, ou de teintes différentes mais de la même valeur (luminosité) etc... Toutes les valeurs doivent être comprises entre 0 et 255. Syntaxe : part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax) ; Arguments Description ind L'indice du type de particule à modifier. hmin Le minimum que peut avoir la composante de teinte de la couleur finale. hmax Le maximum que peut atteindre la composante de teinte de la couleur finale. smin Le minimum que peut atteindre la composante de saturation de la couleur finale. smax La composante maximale de la saturation de la couleur finale. vmin Le minimum que peut avoir la composante de valeur de la couleur finale. vmax Le maximum que peut atteindre la composante de valeur de la couleur finale. Les retours : N/A Exemple : part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 ) ; Le code ci-dessus fait en sorte que chaque particule émise du type de particule indexé dans la variable globale "Stars" ait des couleurs et des saturations différentes, mais la même valeur (luminosité).
