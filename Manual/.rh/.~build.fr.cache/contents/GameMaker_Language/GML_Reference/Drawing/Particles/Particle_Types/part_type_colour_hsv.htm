part_type_colour_hsv Avec cette fonction, vous pouvez définir une teinte, une saturation et une plage de valeurs pour toutes les particules du type donné. Vous fournissez une valeur minimale et une valeur maximale pour chacun des trois composants et les particules créées auront une couleur aléatoire basée sur la plage de paramètres donnée. De cette façon, vous pouvez créer des particules de même teinte mais de saturations différentes, ou de teintes différentes mais de même valeur (luminosité) etc... Toutes les valeurs doivent être comprises entre 0 et 255. Syntaxe:part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax); Argument Description ind L'indice du type de particule à modifier. hmin Le minimum de la composante de teinte de la couleur finale peut être. hmax Le maximum de la composante de teinte de la couleur finale peut être. smin Le minimum de la composante de saturation de la couleur finale peut être. smax Le maximum de la composante de saturation de la couleur finale peut être. vmin La valeur minimale de la valeur de la couleur finale peut être. vmax La valeur maximale de la composante de valeur de la couleur finale peut être. Retour:N/A Exemple:part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 ); Le code ci-dessus définit chaque particule émise du type de particule indexé dans la variable globale "Stars" pour avoir des couleurs et des saturations différentes, mais la même valeur (luminosité).
