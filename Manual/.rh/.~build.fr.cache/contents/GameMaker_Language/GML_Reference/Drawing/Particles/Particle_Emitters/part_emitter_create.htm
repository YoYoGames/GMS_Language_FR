création d'un émetteur partiel Cette fonction doit être utilisée pour créer un nouvel émetteur et l'affecter à un système de particules donné. La fonction renvoie la valeur de l'index du nouvel émetteur qui doit être stockée dans une variable et utilisée dans toutes les autres fonctions qui font référence à l'émetteur, et l'émetteur lui-même doit être détruit lorsqu'il n'est plus utilisé afin d'éviter les fuites de mémoire (ceci peut être réalisé en utilisant les fonctions spécifiques de destruction de l'émetteur ou en détruisant l'ensemble du système de particules auquel l'émetteur appartient). Syntaxe : partie_émetteur_créer(ps) ; Arguments Description ps Le système de particules dans lequel l'émetteur doit être créé. Les retours : Real Exemple : p_emit1 = part_emitter_create(Sname) ; Cela créera un nouvel émetteur de particules et stockera son indice dans la variable "p_emit".
