partie_système_existe Avec cette fonction, vous pouvez vérifier si le système de particules indexé existe ou non dans le jeu. Notez que si la variable vérifiée est une variable non initialisée (à laquelle l'index d'un système de particules aurait été assigné), une erreur sera générée. une erreur. Syntaxe : part_system_exists(ind) ; Arguments Description ind L'indice du système de particules à vérifier. Les retours : Booléen Exemple : if !part_system_exists(global.Sname) { global.Sname = part_system_create() ; } Le code ci-dessus vérifie si le système de particules référencé dans la variable globale existe et s'il n'existe pas, il est créé.
