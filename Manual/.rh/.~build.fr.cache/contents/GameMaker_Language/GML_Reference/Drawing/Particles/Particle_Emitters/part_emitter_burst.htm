partie_emitter_burst Cette fonction vous permet de configurer un émetteur pour qu'il émette un type spécifique de particule. Elle est généralement utilisée dans les événements d'alarme et de destruction car il s'agit d'un code unique qui crée le nombre de particules spécifié en une seule fois en suivant la distribution, la forme et la position définies par la fonction. position définies par la fonction part_emitter_region(). Si vous avez besoin que les particules apparaissent à chaque étape, vous devez utiliser la fonction part_emitter_stream() plutôt que d'appeler cette fonction à chaque étape. Syntaxe : Part_emitter_burst(ps, ind, parttype, number) ; Arguments Description ps Le système de particules dans lequel se trouve l'émetteur. ind L'indice de l'émetteur à éclater. type de pièce L'index (type) des particules à créer. numéro Le nombre de particules à créer. Les retours : N/A Exemple : part_emitter_burst(global. Sname, p_emit1, p1, 30 + irandom(30)) ; Le code ci-dessus fera éclater un nombre aléatoire de particules entre 30 et 60 à partir de l'émetteur choisi.
