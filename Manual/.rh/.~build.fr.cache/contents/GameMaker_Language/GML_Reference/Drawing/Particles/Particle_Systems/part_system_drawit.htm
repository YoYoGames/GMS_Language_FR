part_system_update Avec cette fonction, vous pouvez forcer un système de particules à dessiner toutes les particules actuelles sur l'écran. Si part_system_automatic_draw() est désactivée, cette fonction affichera les particules lorsqu'elle est utilisée dans l'événement de dessin d'une occurrence, ou elle peut être utilisée lorsque la cible de dessin est définie sur une surface (en utilisant surface_set_target() ) pour attirer les particules du système vers cette surface. Syntaxe:part_system_drawit(ind); Argument Description ind L'indice du système de particules pour dessiner les particules. Retour:N/A Exemple:if surface_exists(surf)     {     surface_set_target(surf);     part_system_drawit(global.psys);     surface_reset_target();     } Le code ci-dessus vérifie si la surface indexée dans la variable "surf" existe, et si c'est le cas, définit la cible du dessin sur la surface, dessine le système de particules avec l'ID stocké dans la variable globale, puis réinitialise le dessin cible pour que tous les dessins normaux apparaissent à nouveau à l’écran.
