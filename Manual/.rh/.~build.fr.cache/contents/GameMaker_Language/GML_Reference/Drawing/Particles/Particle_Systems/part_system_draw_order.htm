part_system_draw_order Avec cette fonction, vous pouvez définir la manière dont les particules sont dessinées lorsqu'elles sont créées à l'écran. Le système par défaut utilise un ancien> nouveau look (la fonction est définie sur true ), où les anciennes particules sont dessinées à une profondeur plus élevée que les plus récentes et apparaissent ainsi "sous" les nouvelles particules, mais en définissant cette fonction sur false vous pouvez inverser cet ordre et faire dessiner les nouvelles particules plus haut et ainsi apparaître «sous» les plus anciennes. Les images ci-dessous illustrent cela, l'image de gauche étant la valeur par défaut de true et l'image de droite false: REMARQUE: lorsque les particules sont dessinées avec un mode de fusion additif, l'effet de cette fonction peut ne pas toujours être évident. Syntaxe:part_system_draw_order(ind, oldtonew); Argument Description ind L'indice du système de particules à changer. oldtonew Si les anciennes particules doivent être dessinées derrière les plus récentes (vrai) ou vice versa (faux). Retour:N/A Exemple:mysystem = part_system_create(); part_system_draw_order(mysystem, true); Cela créera un nouveau système de particules avec l'index "mysystem ", puis il définit les particules pour attirer des particules plus récentes sur les plus anciennes.
