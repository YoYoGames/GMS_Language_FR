effet_clair Cela effacera toutes les particules actuellement visibles dans la pièce qui ont été créées en utilisant l'une des méthodes suivantes effet_create_above() ou effet_create_below(). Syntaxe : effect_clear() ; Les retours : N/A Exemple : effect_clear() ; En appelant cette fonction, la pièce sera débarrassée de toutes les particules créées avec les fonctions effect_create_*.
