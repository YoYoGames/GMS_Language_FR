part_emitter_clear Avec cette fonction, vous pouvez réinitialiser l'émetteur donné du système de particules spécifié à son état par défaut. Cela arrêtera également toutes les particules qui sont diffusées par l'émetteur à ce moment-là, et si vous souhaitez utiliser à nouveau l'émetteur, vous devrez définir la position de la région et le type de particule à l'aide du bouton part_emitter_region() fonction. Syntaxe:part_emitter_clear(ps, ind); Argument Description ps Le système de particules dans lequel se trouve l'émetteur. ind L'indice de l'émetteur à effacer. Retour:N/A Exemple:part_emitter_clear(Sname, p_emit1 ); Le code ci-dessus effacera l'émetteur de particules indexé dans la variable "p_emit1".
