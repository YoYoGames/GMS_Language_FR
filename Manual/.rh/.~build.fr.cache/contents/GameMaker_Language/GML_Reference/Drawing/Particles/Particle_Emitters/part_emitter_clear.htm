part_emitter_clear Avec cette fonction, vous pouvez remettre l'emitter donné du système de particules spécifié dans son état par défaut. Cela arrêtera également toutes les particules qui sont diffusées par l'émetteur à ce moment-là. vous devrez définir la position de la région et le type de particule à l'aide de la fonction part_emitter_region() fonction. Syntaxe : part_emitter_clear(ps, ind) ; Arguments Description ps Le système de particules dans lequel se trouve l'émetteur. ind L'index de l'émetteur à effacer. Les retours : N/A Exemple : part_emitter_clear(Sname, p_emit1 ) ; Le code ci-dessus va effacer l'émetteur de particules indexé dans la variable "p_emit1".
