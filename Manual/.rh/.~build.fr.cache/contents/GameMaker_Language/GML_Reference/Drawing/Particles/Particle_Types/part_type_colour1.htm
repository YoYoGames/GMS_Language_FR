type_de_pièce_couleur1 Cette fonction est utilisée pour définir un type de particule comme étant d'une seule couleur pour la durée totale de la vie de chaque particule individuelle. Syntaxe : partie_type_couleur1(ind, couleur1) ; Arguments Description ind L'indice du type de particule à modifier. couleur1 La couleur unique pour faire le type de particule. Les retours : N/A Exemple : part_type_colour1(global.Snow_Part, c_white) ; Le code ci-dessus fera en sorte que toutes les particules créées du type de particule indexé dans la variable globale "Snow_Part" soient uniquement blanches.
