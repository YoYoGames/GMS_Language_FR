part_type_clear Avec cette fonction, vous pouvez "réinitialiser" une particule, renvoyant toutes les valeurs de chacune des fonctions relatives à la particule (vie, couleur, alpha, orientation etc...) à leurs valeurs par défaut. Notez que cette fonction ne supprime pas les particules actuellement visibles dans la pièce de l'écran, pour cela vous devriez utiliser part_particles_clear(). Syntaxe:part_type_clear(ind); Argument Description ind L'indice du type de particule à effacer. Retour:N/A Exemple:part_type_clear(global.explode_part); Le code ci-dessus effacera le type de particule indexé dans la variable globale "explode_part" à ses valeurs par défaut.
