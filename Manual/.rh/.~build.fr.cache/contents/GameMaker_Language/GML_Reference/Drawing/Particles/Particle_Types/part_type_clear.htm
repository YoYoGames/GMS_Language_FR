type_de_pièce Avec cette fonction vous pouvez "réinitialiser" une particule, en retournant toutes les valeurs de chacune des fonctions relatives à la particule (vie, couleur, alpha, orientation etc...) à leurs valeurs par défaut. Notez que cette fonction ne supprime pas de l'écran les particules actuellement visibles dans la pièce à l'écran, pour cela vous devez utiliser la fonction part_particles_clear(). Syntaxe : part_type_clear(ind) ; Arguments Description ind L'index du type de particule à effacer. Les retours : N/A Exemple : part_type_clear(global.explode_part) ; Le code ci-dessus remet le type de particule indexé dans la variable globale "explode_part" à ses valeurs par défaut.
