Émetteurs de particules Les émetteurs de particules sont utilisés par GameMaker Studio 2 pour émettre des particules sur une zone de l'écran, qui peuvent avoir différentes formes et distributions. Ils peuvent également créer un flux continu de particules soit faire éclater un certain nombre de particules d'un seul coup, selon la façon dont les fonctions sont utilisées. Puisqu'un émetteur de particules est une ressource créée dynamiquement, vous devez le créer et stocker l'index retourné dans une variable pour référencer l'émetteur dans tous les appels de fonction ultérieurs. détruit l'émetteur lorsqu'il n'est plus nécessaire, sinon vous aurez une fuite de mémoire qui ralentira et finira par faire planter votre jeu. Il convient également de noter que les émetteurs de particules sont conservés pour toujours après leur création, même si l'indice n'est plus stocké. Ainsi, même si vous changez de pièce ou redémarrez le jeu, les systèmes et les particules resteront, et seront visibles, dans toutes les pièces suivantes ; veillez donc à les détruire dès que vous n'en avez plus besoin. Les fonctions suivantes sont disponibles pour paramétrer les émetteurs et leur permettre de créer des particules.
