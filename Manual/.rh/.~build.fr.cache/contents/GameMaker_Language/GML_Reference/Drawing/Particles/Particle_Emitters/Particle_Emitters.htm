Émetteurs de particules Les émetteurs de particules sont utilisés par GameMaker Studio 2 pour émettre des particules sur une zone de l'écran qui peut avoir différentes formes et distributions. Ils peuvent également créer un flux continu de particules ou éclater un certain nombre de particules en même temps, selon la manière dont les fonctions sont utilisées. Puisqu'un émetteur de particules est une ressource créée dynamiquement, vous devez le créer et stocker l'index renvoyé dans une variable pour référencer l'émetteur dans tous les appels de fonction ultérieurs, et il est très important que vous détruisez également l'émetteur lorsqu'il n'est plus nécessaire ou sinon, vous aurez une fuite de mémoire qui ralentira et finira par planter votre jeu. Il est également intéressant de noter que les émetteurs de particules vivront éternellement après leur création, même si l'indice n'est plus stocké. Ainsi, même si vous changez de pièce ou redémarrez le jeu, les systèmes et les particules resteront et seront visibles dans toutes les autres pièces.Vous devez donc vous assurer de les détruire une fois que vous n'en aurez plus besoin.
