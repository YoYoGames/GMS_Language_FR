partie_particules_créer Cette fonction est idéale pour les effets qui ne nécessitent aucune des fonctionnalités offertes par les émetteurs de particules (par exemple, pour créer de la fumée à partir d'un missile, ou un simple effet d'explosion) car elle vous permet de créer rapidement et facilement des particules à n'importe quel endroit de la salle de jeu. Notez que vous devez avoir créé le système de particules et le type de particule précédemment pour que cette fonction puisse être utilisée. Syntaxe : part_particles_create(ind, x, y, parttype, number) ; Arguments Description ind L'indice du système de particules. x La coordonnée x de l'endroit où créer les particules. y La coordonnée y de l'endroit où créer les particules. type de pièce L'index (type) des particules à créer. numéro Le nombre de particules à créer. Les retours : N/A Exemple : if mouse_check_button(mb_left) { part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5) ; } Le code ci-dessus vérifie si le bouton de la souris a été pressé et s'il retourne vrai, il génère 5 particules à la position de la souris.
