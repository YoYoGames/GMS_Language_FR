partie_émetteur_existe Avec cette fonction, vous pouvez vérifier si l'émetteur de particules indexé existe ou non dans le système donné. Notez que si la variable vérifiée est une variable non initialisée (à laquelle un émetteur de particules aurait autrement été assigné) cela entraînera une erreur. Syntaxe : partie_émetteur_existe(ps, ind) ; Arguments Description ps Le système de particules à vérifier pour un émetteur. ind L'index de l'émetteur à rechercher. Les retours : Booléen Exemple : if part_emitter_exists(sname, p_emit1) { part_emitter_burst(sname, p_emit1, part_1, 30) ; } Le code ci-dessus va rechercher l'émetteur indexé dans la variable "permit" et s'il existe, il va éclater 30 particules à partir de celui-ci.
