partie_particules_count Avec cette fonction, vous pouvez vérifier si un système de particules a actuellement des particules créées dans la pièce, et elle renverra également le nombre de celles-ci. Syntaxe : part_particles_count(ind) ; Arguments Description ind L'indice du système de particules. Les retours : Real Exemple : if (part_particles_count(Sname) == 0) { part_system_destroy(Sname) ; instance_destroy() ; } Le code ci-dessus va vérifier le nombre de particules dans le système de particules local indexé dans la variable "Sname" et s'il n'y en a pas, il va détruire le système et ensuite l'instance.
