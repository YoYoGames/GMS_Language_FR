part_system_depth Avec cette fonction, vous pouvez définir la profondeur de dessin pour le système de particules, de la même manière que vous pouvez définir la profondeur de rendu des différentes couches de la pièce, où une faible profondeur de dessin signifie qu'il apparaîtra au-dessus de tout ce qui est dessiné avec un plus haut profondeur et une profondeur de tirage élevée en le plaçant en dessous de tout avec une profondeur de tirage inférieure. Syntaxe:part_system_depth( ind, depth ); Argument Description ind L'indice du système de particules à changer. depth La profondeur à laquelle définir le système de particules. Retour:N/A Exemple:global.Sname = part_system_create(); part_system_depth(global.Sname, -1000 ); Le code ci-dessus créera un système de particules et stockera son index dans la variable globale "Sname". ce système reçoit alors une faible profondeur de -1000, ce qui signifie qu'il apparaîtra au-dessus de tout avec une profondeur de tirage plus élevée.
