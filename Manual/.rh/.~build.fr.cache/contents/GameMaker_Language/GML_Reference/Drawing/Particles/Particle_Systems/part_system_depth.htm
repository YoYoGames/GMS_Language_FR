profondeur_système_partiel Avec cette fonction, vous pouvez définir la profondeur de dessin pour le système de particules, de la même manière que vous pouvez définir la profondeur de rendu des différentes couches dans la pièce, où une faible profondeur de dessin signifie qu'il apparaîtra au-dessus de toutes les choses dessinées avec une profondeur plus élevée, et une profondeur de dessin élevée le plaçant sous tout ce qui a une profondeur de dessin plus faible. plus élevée, et une profondeur de dessin élevée le plaçant sous tout ce qui a une profondeur de dessin plus faible. Syntaxe : part_system_depth( ind, depth ) ; Arguments Description ind L'indice du système de particules à modifier. profondeur La profondeur à laquelle il faut régler le système de particules. Les retours : N/A Exemple : global.Sname = part_system_create() ; part_system_depth(global.Sname, -1000 ) ; Le code ci-dessus crée un système de particules et stocke son index dans la variable globale "Sname". Ce système reçoit ensuite une faible profondeur de -1000, ce qui signifie qu'il apparaîtra au-dessus de tout ce qui a une profondeur de dessin plus élevée.
