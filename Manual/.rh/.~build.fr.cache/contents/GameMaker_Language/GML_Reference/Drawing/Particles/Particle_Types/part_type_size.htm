taille_type_partie Cette fonction est utilisée pour déterminer la taille de la particule lors de sa création et peut également être utilisée pour faire augmenter ou diminuer la taille de la particule au cours de sa vie. Les valeurs de taille minimale et maximale sont par défaut égales à 1 (ce qui signifie que la taille a le même rapport que la particule de base après le traitement de la particule). ratio que la particule de base après que l'échelle part_type_scale() ) et si elles sont réglées sur la même valeur, la taille initiale de toutes les particules sera la même, mais si elles sont réglées sur des valeurs différentes, chaque particule commencera avec une taille aléatoire. Par exemple, si vous avez un minimum de 0,5 et un maximum de 2, toutes les particules seront créées avec une taille comprise entre la moitié et le double de l'échelle de base. Vous pouvez également définir une valeur d'incrémentation qui fera croître (s'il s'agit d'un nombre positif) ou décroître (s'il s'agit d'un nombre négatif) le sprite de la particule pendant sa durée de vie, la quantité donnée étant ajoutée à la taille de la particule à chaque étape. Cette valeur peut être un minimum de (+/-)0,01. Enfin, vous pouvez définir le facteur "wiggle". Il s'agit d'une valeur qui sera ajoutée ou soustraite aléatoirement à la taille à chaque étape de la vie des particules. Évidemment, les grandes valeurs sont plus prononcées que les petites, et cette valeur peut même être négative. négative, la plage maximale étant comprise entre -20 et 20. Syntaxe : part_type_size(ind, size_min, size_max, size_incr, size_wiggle) ; Arguments Description ind L'indice du type de particule à modifier. taille_min La taille minimale à laquelle la particule peut commencer. taille_max La taille maximale à laquelle la particule peut commencer. taille_incr De combien la particule doit augmenter ou diminuer par étape. taille_wiggle Quelle quantité doit être ajoutée ou soustraite aléatoirement de la taille des particules par étape. Les retours : N/A Exemple : part_type_shape(particle1, pt_shape_explosion) ; part_type_size(particle1, 2, 2, 0.10, 0.01) ; part_type_scale(particle1, 1, 1) ; part_type_colour2(particle1, 16777215, 12632256) ; part_type_alpha3(particle1, 0, 0.20, 0) ; part_type_speed(particle1, 0.20, 0.20, 0, 0) ; part_type_direction(particle1, 0, 359, 0, 20) ; part_type_orientation(particle1, 0, 359, 0, 0, 1) ; part_type_blend(particle1, 0) ; part_type_life(particle1, 40, 60) ; Le code ci-dessus définira diverses valeurs de particules, y compris la taille qui est fixée au double de la taille lors de la création de chaque particule, puis incrémentera la taille de 0,01 à chaque étape avec une fluctuation de 0,1 également.
