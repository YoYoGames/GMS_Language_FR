direction_type_partie Cette fonction est utilisée pour déterminer la direction de la particule lorsqu'elle est créée et peut également être utilisée pour que la particule augmente ou diminue sa direction au cours de sa vie. Les valeurs de direction minimale et maximale sont par défaut égales à 0, ce qui signifie que la particule se déplacera vers la droite sans variation, mais ces valeurs peuvent être modifiées pour limiter la direction du mouvement en suivant les directions standard de GameMaker Studio 2, à savoir 0 pour la droite, 90 pour le haut, 180 pour la gauche et 270 pour le bas. Si vous les réglez sur la même valeur, les particules ne se déplaceront que dans une seule direction. Vous pouvez également définir une valeur d'incrémentation qui ajoutera (s'il s'agit d'un nombre positif) ou soustraira (s'il s'agit d'un nombre négatif) une quantité de degrés à la direction pendant sa durée de vie. Cette valeur peut être au minimum de (+/-) 0,01. Enfin, vous pouvez définir le facteur "wiggle". Il s'agit d'une valeur qui sera ajoutée ou soustraite aléatoirement à la direction à chaque étape de la vie des particules. Évidemment, les grandes valeurs sont plus prononcées que les petites, et cette valeur peut même être négative. Cette valeur peut même être négative, la plage maximale étant comprise entre -20 et 20. Syntaxe : part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle) ; Arguments Description ind L'indice du type de particule à modifier. dir_min La direction minimale à laquelle la particule peut commencer. dir_max La direction maximale à laquelle la particule peut commencer. dir_incr De combien la direction de la particule doit augmenter ou diminuer par étape. dir_wiggle Quelle quantité doit être ajoutée ou soustraite aléatoirement de la direction de la particule par étape. Les retours : N/A Exemple : part_type_shape(particle1, pt_shape_pixel) ; part_type_size(particle1, 0.19, 0.79, 0.01, 0) ; part_type_scale(particle1, 5, 5) ; part_type_colour3(particle1, 32768, 8454016, 8454016) ; part_type_alpha3(particle1, 0, 0.80, 0) ; part_type_speed(particle1, 8, 10, -0.50, 1) ; part_type_direction(particle1, 0, 359, 1, 20) ; part_type_gravity(particle1, 0, 270) ; part_type_orientation(particle1, 0, 359, 0, 0, 1) ; part_type_blend(particle1, 1) ; part_type_life(particle1, 10, 50) ; Le code ci-dessus définira diverses valeurs de particules, y compris la direction qui sera une valeur aléatoire entre 0 et 359 degrés. Elle augmentera également d'un degré à chaque étape et une quantité aléatoire sera ajoutée à chaque étape, entre 0 et 20.
