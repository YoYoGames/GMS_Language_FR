partie_type_alpha1 Cette fonction est utilisée pour définir un type de particule pour qu'il ait une seule valeur alpha (transparence) pour la durée totale de la vie de chaque particule individuelle, et cela peut être de 0 (transparent) à 1 (opaque). Syntaxe : partie_type_alpha1(ind, alpha1) ; Arguments Description ind L'indice du type de particule à modifier. alpha1 L'alpha de la particule. Les retours : N/A Exemple : part_type_alpha1(global.Snow_Part, 0.5) ; Le code ci-dessus donnera à toutes les particules créées du type de particule indexé dans la variable globale "Snow_Part" une valeur alpha de 0,5 (semi-transparent).
