part_type_alpha1 Cette fonction est utilisée pour définir un type de particule pour avoir une valeur alpha unique (transparence) pour la durée totale de la vie de chaque particule individuelle, et cela peut être de 0 (transparent) à 1 (opaque). Syntaxe:part_type_alpha1(ind, alpha1); Argument Description ind L'indice du type de particule à modifier. alpha1 L'alpha de la particule. Retour:N/A Exemple:part_type_alpha1(global.Snow_Part, 0.5); Le code ci-dessus définira toutes les particules créées du type de particule indexé dans la variable globale "Snow_Part" pour avoir une valeur alpha de 0,5 (semi-transparent).
