partie_type_destroy Avec cette fonction, vous pouvez supprimer du jeu le type de particule spécifié. Lorsque vous utilisez cette fonction, toutes les particules du type donné disparaissent de la pièce et la particule elle-même est supprimée de la mémoire, donc cette fonction doit être utilisée uniquement que lorsque vous n'avez plus besoin de la particule. Syntaxe : partie_type_destroy(ind) ; Arguments Description ind L'indice du type de particule à détruire. Les retours : N/A Exemple : if part_particles_count(smoke_sys) = 0 { part_type_destroy(smoke_part) ; part_system_destroy(smoke_sys) ; instance_destroy() ; } else alarme[0] = 1 ; Le code ci-dessus vérifie un système de particules pour voir si des particules sont actuellement visibles dans la pièce et si ce n'est pas le cas, les particules, le système et l'instance sont détruits.
