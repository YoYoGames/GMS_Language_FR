Part_emitter_stream Cette fonction vous permet de configurer un émetteur pour qu'il diffuse un type de particule spécifique. Elle est généralement utilisée dans l'événement de création car il s'agit d'un code unique qui configure l'émetteur pour qu'il génère le nombre de particules spécifié à chaque étape du jeu. Les particules de sont émises selon la distribution, la forme et la position définies par la fonction partie_émetteur_région(). Vous pouvez définir le nombre de particules émises à zéro et appeler à nouveau cette fonction pour "désactiver" le flux de particules. La fonction acceptera également des nombres négatifs pour la quantité, auquel cas l'émetteur produira des particules sur la base du hasard. Par exemple, si le nombre de particules est fixé à -5, il y a une chance sur cinq qu'une particule soit produite. 5, il y a 1 chance sur 5 qu'une particule soit produite à chaque étape. Si vous souhaitez que les particules apparaissent en une seule fois plutôt qu'à chaque étape, vous devez utiliser la fonction part_emitter_burst(). Syntaxe : Part_emitter_stream(ps, ind, parttype, number) ; Arguments Description ps Le système de particules dans lequel se trouve l'émetteur. ind L'indice de l'émetteur à éclater. type de pièce L'index (type) des particules à créer. numéro Le nombre de particules à créer par étape. Les retours : N/A Exemple : part_emitter_stream(global. Sname, p_emit1, p1, 1) ; Le code ci-dessus va diffuser une particule à chaque étape du jeu jusqu'à ce que l'émetteur soit détruit ou que le flux soit mis à 0.
