partie_system_get_layer Cette fonction peut être utilisée pour récupérer la valeur d'identification unique de la couche pour le système de particules donné. Vous fournissez la valeur d'identification du système de particules (telle que renvoyée par la fonction part_system_create_layer()) et la fonction renvoie la valeur d'identification de la couche. Syntaxe : part_system_get_layer(ind) ; Arguments Description ind La valeur de l'ID du système de particules pour obtenir l'ID de la couche de Les retours : Réel (valeur d'identification unique de la couche ou 0 si la couche est gérée en interne) Exemple : if (part_system_get_layer(global.p_sys) != "effects_layer") { part_system_layer(global.p_sys, "effects_layer" ; } Le code ci-dessus vérifiera un système de particules pour voir sur quelle couche il se trouve et s'il n'est pas sur la couche donnée, il sera changé.
