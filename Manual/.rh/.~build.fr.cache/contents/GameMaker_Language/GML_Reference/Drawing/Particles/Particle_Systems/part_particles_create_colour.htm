partie_particules_créer Cette fonction est idéale pour les effets qui ne nécessitent aucune des fonctionnalités offertes par les émetteurs de particules (par exemple, pour créer de la fumée à partir d'un missile, ou un simple effet d'explosion) car elle vous permet de créer rapidement et facilement des particules à n'importe quel endroit de la salle de jeu. Vous pouvez également colorer les particules "à la volée" lorsqu'elles sont créées avec cette fonction, et cette couleur remplacera la couleur prédéfinie de la particule, mais cette couleur ne se fond pas dans la durée de vie de la particule. Notez que vous devez avoir créé le système de particules et le type de particule au préalable pour que cette fonction puisse être utilisée. Syntaxe : part_particles_create_colour(ind, x, y, parttype, colour, number) ; Arguments Description ind L'indice du système de particules. x La coordonnée x de l'endroit où créer les particules. y La coordonnée y de l'endroit où créer les particules. type de pièce L'index (type) des particules à créer. couleur Le mélange de couleurs pour les particules. numéro Le nombre de particules à créer. Les retours : N/A Exemple : if (speed > 1) { var _c = choisir(c_aqua, c_lime, c_fuschia, c_orange) ; part_particles_create_colour(sname, x, y, p_Smoke, _c, 1) ; } Le code ci-dessus générera une seule particule à chaque pas où l'instance avec le code a une vitesse supérieure à un.
