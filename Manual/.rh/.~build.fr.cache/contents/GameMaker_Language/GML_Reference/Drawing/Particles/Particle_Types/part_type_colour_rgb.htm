part_type_colour_rgb Avec cette fonction, vous pouvez définir le mélange de couleurs rouge, verte et bleue pour toutes les particules créées du type donné. Vous fournissez une valeur minimale et une valeur maximale pour chacun des trois composants et les particules créées auront une couleur aléatoire basée sur la plage de paramètres donnée. Toutes les valeurs doivent être comprises entre 0 et 255. Syntaxe:part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p> Argument Description ind L'indice du type de particule à modifier. rmin Le minimum de la composante rouge de la couleur finale peut être. rmax Le maximum de la composante rouge de la couleur finale peut être. gmin Le minimum de la composante verte de la couleur finale peut être. gmax Le maximum de la composante verte de la couleur finale peut être. bmin Le minimum de la composante bleue de la couleur finale peut être. bmax Le maximum de la composante bleue de la couleur finale peut être. Retour:N/A Exemple:part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0); Le code ci-dessus définit chaque particule émise du type de particule indexé dans la variable globale "Blood_Part" pour n'être que des nuances de rouge différentes.
