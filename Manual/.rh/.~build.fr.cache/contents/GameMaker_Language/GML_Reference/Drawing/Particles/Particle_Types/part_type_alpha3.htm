part_type_alpha3 Cette fonction peut être utilisée pour définir un gradient de trois valeurs alpha (transparence) pour chaque particule créée du type donné. Le premier alpha est celui avec lequel toutes les particules commenceront, et le deuxième alpha est celui qui sera mélangé à la moitié de sa durée de vie et le troisième alpha est celui avec lequel la particule se terminera. Un changement de gradient progressif se produira à travers les alphas au cours de la durée de vie des particules de l'un à l'autre. Syntaxe:part_type_alpha3(ind, alpha1, alpha2, alpha3); Argument Description ind L'indice du type de particule à modifier. alpha1 L'alpha de départ de la particule. alpha2 Le point alpha à mi-chemin de la particule. alpha3 L'alpha de fin de la particule. Retour:N/A Exemple:part_type_alpha3( part_Health, 0.5, 1, 0); Le code ci-dessus fera que toutes les particules créées du type indexé dans la variable "part_Health" auront un mélange alpha de 0,5 à 1 à 0 sur leur durée de vie.
