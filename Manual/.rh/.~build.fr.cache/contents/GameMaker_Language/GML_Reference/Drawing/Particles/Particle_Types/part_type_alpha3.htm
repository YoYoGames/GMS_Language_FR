partie_type_alpha3 Cette fonction peut être utilisée pour définir un gradient de trois valeurs alpha (transparence) pour chaque particule créée du type donné. Le premier alpha est celui avec lequel toutes les particules commenceront, le second alpha est celui qui sera mélangé à la moitié de sa vie et le troisième alpha est celui avec lequel la particule finira. sa durée de vie et le troisième alpha est celui avec lequel la particule se terminera. Un changement progressif de gradient se produira à travers les alphas pendant la durée de vie de la particule, de l'un à l'autre. Syntaxe : partie_type_alpha3(ind, alpha1, alpha2, alpha3) ; Arguments Description ind L'indice du type de particule à modifier. alpha1 L'alpha de départ de la particule. alpha2 Le point médian alpha de la particule. alpha3 L'alpha final de la particule. Les retours : N/A Exemple : part_type_alpha3( part_Health, 0.5, 1, 0) ; Le code ci-dessus fera en sorte que toutes les particules créées du type indexé dans la variable "part_Health" aient un mélange alpha de 0,5 à 1 à 0 au cours de leur vie.
