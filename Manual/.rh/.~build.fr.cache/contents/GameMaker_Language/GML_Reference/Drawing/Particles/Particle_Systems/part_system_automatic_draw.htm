système_de_pièce_dessin_automatique Cette fonction peut être utilisée pour désactiver le dessin d'un système de particules afin que toute mise à jour du système (automatique ou non) ne soit pas visible. Il s'agit d'une option purement visuelle et lorsqu'elle est définie sur false, vous ne pourrez pas voir les particules car elles ne sont pas dessinées, mais elles existent toujours et changent de position, de couleur etc... comme elles le feraient normalement. Lorsque le dessin automatique est désactivé, vous pouvez demander explicitement à GameMaker Studio 2 de dessiner l'état actuel du système de particules à l'aide de la fonction en utilisant la fonction part_system_drawit()Si vous redéfinissez cette fonction à true, vous pouvez réactiver le dessin automatique. Une chose à noter est que si vous utilisez les effets simples créés par les fonctions effet_créer_supérieur() ou effet_create_below() alors vous pouvez utiliser les valeurs 0 (pour les effets inférieurs) ou 1 (pour les effets supérieurs) comme index du système de particules et ainsi basculer le dessin automatique pour ceux-ci aussi (cela fonctionnera aussi pour basculer le dessin pour les effets de particules DnD™). Syntaxe : part_system_automatic_draw(ind, automatic) ; Arguments Description ind L'indice du système de particules à modifier. automatique Indique si le dessin automatique est activé (true) ou non (false). Les retours : N/A Exemple : part_system_automatic_draw(global.Sname, false) ; Le code ci-dessus désactive le dessin automatique pour le système de particules indexé dans la variable globale "Sname".
