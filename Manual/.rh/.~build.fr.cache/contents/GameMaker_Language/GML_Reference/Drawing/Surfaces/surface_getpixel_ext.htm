surface_getpixel_ext Avec cette fonction, vous pouvez obtenir la valeur complète abgr 32bit de n'importe quel pixel d'une surface (préalablement créée). REMARQUE: Cette fonction a un impact considérable sur les performances et ne doit donc être utilisée qu'en cas de nécessité absolue. Syntaxe : surface_getpixel_ext(surface_id, x, y) ; Arguments Description surface_id L'ID de la surface à utiliser. x La coordonnée x du pixel à vérifier y La coordonnée y du pixel à vérifier Les retours : Real Exemple : col = surface_getpixel_ext(mouse_x, mouse_y) ; alpha = (col >> 24) & 255 ; blue = (col >> 16) & 255 ; green = (col >> 8) & 255 ; red = col & 255 ; Le code ci-dessus récupère la valeur de la couleur 32 bits à la position de la souris, puis la divise en ses valeurs composantes et les stocke dans des variables.
