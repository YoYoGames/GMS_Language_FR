surface_set_target_ext Cette fonction est à utiliser avec les fonctions deshaders et définit le MRT (0 - 3) pour les shaders de niveau natif (DX9, DX11, OpenGL). REMARQUE: Lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment, car elles sont stockées dans la mémoire des textures. Vous devez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists() avant d'y faire référence directement. Syntaxe : surface_set_target_ext(index, surface_id) ; Arguments Description indice L'index de la cible de rendu à utiliser (de 0 à 3). surface_id L'ID de la surface à utiliser. Les retours : Booléen Exemple : surface_set_target_ext(0, surf) ; Le code ci-dessus va définir la cible de rendu 0 à l'ID de la surface indexée dans la variable "surf".
