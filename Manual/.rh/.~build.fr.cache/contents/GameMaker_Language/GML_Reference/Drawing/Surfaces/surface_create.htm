surface_create Cette fonction est utilisée pour créer une surface et renvoie l'index de la surface qui doit être stocké dans une variable pour les appels de fonction futurs. Lorsque la surface est créée pour la première fois, elle peut contenir du "bruit", car il s'agit simplement d'une zone de mémoire mise de côté à cet effet. qui est mise de côté dans ce but (et cette mémoire peut encore contenir des informations), vous pouvez donc vouloir effacer la surface avant de l'utiliser avec une fonction comme draw_clear_alpha(). Il est fortement recommandé de créer toutes les surfaces avec une taille correspondant à une puissance de 2, c'est-à-dire une taille de 16, 128, 512 ou 1024 pixels, par exemple. Ce n'est pas vraiment nécessaire sur certaines plateformes (comme Windows et MacOS), mais cela augmentera certainement la compatibilité sur ces cibles. compatibilité sur ces cibles, tandis que pour HTML5 et les appareils, c'est essentiel et il est très important que vous vous en souveniez, sinon vous risquez de rencontrer des problèmes par la suite. Syntaxe : surface_create(w, h) ; Arguments Description w La largeur de la surface à créer. h La hauteur de la surface à créer. Les retours : Real Exemple : if !surface_exists(surf) { surf = surface_create(1024, 1024) ; surface_set_target(surf) ; draw_clear_alpha(c_black, 0) ; surface_reset_target() ; view_surface_id[0] = surf ; } Le code ci-dessus vérifie si une surface existe et si ce n'est pas le cas, il crée une surface de 1024 pixels de large et 1024 pixels de haut, en affectant l'index à la variable "surf". La cible du dessin est ensuite définie sur la nouvelle surface, qui est effacée et rendue transparente avant d'être utilisée. est effacée et rendue transparente avant que la cible de dessin ne soit réinitialisée à l'affichage.
