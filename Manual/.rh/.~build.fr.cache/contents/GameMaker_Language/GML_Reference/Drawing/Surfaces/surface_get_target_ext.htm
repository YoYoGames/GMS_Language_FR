surface_get_target_ext Cette fonction récupère l'ID de la surface assignée à l'une des 4 cibles de rendu disponibles pour les surfaces. Vous fournissez l'index de la cible de rendu à vérifier, et la fonction renverra -1 si aucune surface n'est assignée, ou une valeur entière >= 0, représentant l'ID de la surface assignée (tel que renvoyé par la fonction surface_create()). Syntaxe : surface_get_target_ext(index) ; Arguments Description indice L'index de la cible de rendu à vérifier (de 0 à 3). Les retours : Nombre entier (-1 pour aucune cible de surface, ou une valeur d'identification de surface) Exemple : si surface_get_target_ext(0) == -1 { surface_set_target_ext(0, global.Surf) ; } Le code ci-dessus va d'abord vérifier si la cible de rendu 0 du shader a été définie sur une surface, et si ce n'est pas le cas, une surface est assignée.
