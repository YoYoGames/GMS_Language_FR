surface_save Cette fonction permet de sauvegarder une surface sur disque en utilisant le nom de fichier donné. La surface doit être sauvegardée comme un fichier au format *.png. Syntaxe : surface_save(surface_id, fname) ; Arguments Description surface_id L'ID de la surface à définir comme cible du dessin. nom de famille Le nom du fichier image enregistré. Les retours : N/A Exemple : if keyboard_check_pressed(ord("S") { surface_save(surf, "test.png") ; } Le code ci-dessus vérifie si l'utilisateur appuie sur la touche "S" du clavier et si c'est le cas, il enregistre la surface indexée dans la variable "surf" sur le disque.
