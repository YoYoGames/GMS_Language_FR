surface_save Cette fonction enregistre une surface sur le disque en utilisant le nom de fichier donné. La surface doit être enregistrée en tant que *.png fichier de format. Syntaxe:surface_save(surface_id, fname); Argument Description surface_id ID de la surface à définir comme cible de dessin. fname Le nom du fichier image enregistré. Retour:N/A Exemple:if keyboard_check_pressed(ord("S")     {     surface_save(surf, "test.png");     } Le code ci-dessus vérifiera si l'utilisateur appuie sur la touche "S" du clavier et s'il le fait, il sauvegardera la surface indexée dans la variable "surf" sur le disque.
