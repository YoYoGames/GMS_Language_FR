surface_getpixel Cette fonction peut être utilisée pour obtenir la couleur d'un pixel spécifique d'une surface, en utilisant les coordonnées locales de la surface où (0,0) est le coin supérieur gauche. Cette fonction ne doit pas être utilisée très souvent car elle est extrêmement lente et peut provoquer une pause dans votre jeu. pause dans votre jeu. REMARQUE: Lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment, car elles sont stockées dans la mémoire de texture. Vous devriez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists() avant d'y faire référence directement. Syntaxe : surface_getpixel(surface_id, x, y) ; Arguments Description surface_id L'ID de la surface. x La position x sur la surface à partir de laquelle il faut obtenir le pixel. y La position y sur la surface à partir de laquelle il faut obtenir le pixel. Les retours : Real Exemple : col = surface_getpixel(surf, 56, 78 ) ; Ceci retournera la couleur du pixel aux coordonnées (56,78) de la surface indexée dans la variable "surf".
