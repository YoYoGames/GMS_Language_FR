application_surface Cette variable intégrée de portée globale peut être utilisée pour accéder à la surface de l'application à l'aide de l'une des fonctions de surface. Cette surface est disponible en permanence et c'est là que l'essentiel du dessin est réalisé par GameMaker Studio 2. Syntaxe : application_surface ; Les retours : Pointeur Exemple : surface_resize(application_surface, display_get_gui_width(), display_get_gui_height()) Le code ci-dessus va redimensionner la surface de l'application pour avoir un rapport 1:1 avec la couche GUI.
