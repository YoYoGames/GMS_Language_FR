surface_create_ext Cette fonction vous permet d'attacher une surface à un élément de canevas qui existe déjà dans votre page Web, ce qui signifie que vous pouvez efficacement diviser des parties de votre jeu à dessiner à différents endroits de la page. Pour cela, vous devez avoir défini correctement l'élément canvas dans le *.html page de votre jeu en utilisant les tailles et les noms corrects qui correspondent aux surfaces que vous souhaitez créer. Ainsi, vous auriez votre canevas "principal", puis vos éléments de canevas de surface secondaires, qui seront affectés en utilisant cette fonction aux bonnes surfaces. L'image suivante est un exemple de configuration d'une page avec trois éléments de canevas: La mise en page peut être délicate, surtout si vous souhaitez que tous les éléments s'alignent correctement, mais une fois la tâche difficile de création de la mise en page terminée, vous pouvez alors ajouter ceci html comme fichier de page par défaut pour le jeu à l'aide de l' onglet HTML5 des options de jeu. La prochaine chose que vous devez faire est de configurer votre pièce et vos vues, car chaque surface devra être associée à une vue spécifique pour «capturer» les images du jeu (voir la variable de vue view_surface_id[0...7] ). L'image ci-dessous montre comment la salle de jeu de l'exemple de toile ci-dessus serait présentée: Enfin, vous utiliserez ensuite cette fonction pour créer les surfaces, le nom étant le même que celui utilisé pour les éléments du canevas et la taille correspondant à la taille de ce même canevas. La fonction retournera l'index de la surface qui devrait être stockée dans une variable pour les futurs appels de fonction. Lorsque la surface est créée pour la première fois, elle peut contenir du «bruit» car il s’agit essentiellement d’une zone de mémoire réservée à cet effet (et cette mémoire peut encore contenir des informations), vous pouvez donc effacer la surface avant de l’utiliser avec une fonction comme draw_clear_alpha(). REMARQUE: cette fonction n'est disponible que pour une utilisation avec le module HTML5. Syntaxe:surface_create_ext(name, w, h); Argument Description name Le nom de l'élément de canevas auquel lier la surface. w La largeur de la surface à créer. h La hauteur de la surface à créer. Retour:Real Exemple:s1 = surface_create_ext("surface1", 192, 550); s2 = surface_create_ext("surface2", 608, 186); view_surface_id[1] = s1; view_surface_id[2] = s2; Le code ci-dessus crée deux surfaces de tailles différentes, en attribuant chacune à un élément de canevas différent, puis ces surfaces sont affectées à deux vues afin que la partie correcte de la pièce soit capturée.
