surface_get_width Cette fonction renvoie simplement la largeur, en pixels, de la surface indexée. Il est à noter que si vous appelez cette fonction pour vérifier la surface de l'application immédiatement après avoir modifié sa taille à l'aide de la fonction surface_resize() elle ne renverra pas la nouvelle valeur car le changement nécessite une étape ou deux pour être entièrement traité. Après avoir attendu un pas, il devrait retourner la nouvelle taille correctement. NOTE: Lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment, car elles sont stockées dans la mémoire des textures. Vous devez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists() avant de les référencer directement. Syntaxe : surface_get_width(surface_id) ; Arguments Description surface_id L'ID de la surface dont on veut obtenir la largeur. Les retours : Real Exemple : sw = surface_get_width(surf) ; Le code ci-dessus va stocker la largeur de la surface indexée dans la variable "surf" dans la variable "sw".
