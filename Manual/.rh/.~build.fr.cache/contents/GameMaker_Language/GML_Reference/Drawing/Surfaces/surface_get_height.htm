surface_get_height Cette fonction renvoie simplement la hauteur, en pixels, de la surface indexée. Il est à noter que si vous l'appelez pour vérifier la surface de l'application immédiatement après avoir modifié sa taille à l'aide de la fonction surface_resize() elle ne renverra pas la nouvelle valeur car le changement nécessite une étape ou deux pour être entièrement traité. Après avoir attendu une étape, il devrait retourner la nouvelle taille correctement. NOTE: Lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment, car elles sont stockées dans la mémoire de texture. Vous devez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists() avant d'y faire référence directement. Syntaxe : surface_get_height(surface_id) ; Arguments Description surface_id L'ID de la surface dont on veut obtenir la hauteur. Les retours : Real Exemple : sh = surface_get_height(surf) ; Le code ci-dessus va stocker la hauteur de la surface indexée dans la variable "surf" dans la variable "sh".
