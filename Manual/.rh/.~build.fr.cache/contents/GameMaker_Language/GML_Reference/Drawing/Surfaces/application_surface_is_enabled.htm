application_surface_is_enabled Cette fonction retournera true si la surface d'application est utilisée pour le dessin, ou false si le tampon d'écran est utilisé. Syntaxe:application_surface_is_enabled(); Retour:Boolean Exemple:if keyboard_check_pressed(vk_space)     {     if application_surface_is_enabled()         {         application_surface_enable(false);         }     else         {         application_surface_enable(true);         }     } Le code ci-dessus vérifie la pression d'une touche et active ou désactive la surface de l'application en fonction de son état (comme dans un menu d'options).
