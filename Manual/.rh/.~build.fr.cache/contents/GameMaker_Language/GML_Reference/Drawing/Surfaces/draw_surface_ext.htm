draw_surface_ext Cette fonction dessinera la surface donnée comme dans la fonction draw_surface() mais avec des options supplémentaires pour modifier l'échelle, la fusion, la rotation et l'alpha de la surface dessinée. La modification de ces valeurs ne modifie pas la ressource de quelque manière que (seulement comment elle est). REMARQUE: lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment car elles sont stockées dans la mémoire de texture. Vous devez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists () avant de les référencer directement. Syntaxe:draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha); Argument Description id La valeur d'identifiant unique de la surface à dessiner. x La position x de l'endroit où dessiner la surface. y La position y de l'endroit où dessiner la surface. xscale L'échelle horizontale. yscale L'échelle verticale. rot La rotation ou l'angle pour dessiner la surface. col La couleur avec laquelle fondre la surface. alpha La transparence alpha pour dessiner la surface. Retour:N/A Exemple:draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5); Le code ci-dessus dessinera une surface indexée dans la variable "surf" à la position (0,0) dans la pièce, avec deux fois l'échelle originale, mélangée rouge et semi transparente.
