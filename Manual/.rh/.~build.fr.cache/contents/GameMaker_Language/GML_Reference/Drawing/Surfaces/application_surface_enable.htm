application_surface_enable Cette fonction permet d'activer ou de désactiver la surface d'application. Par défaut, la surface d'application est activée et tous les dessins pour les événements Draw seront effectués sur cette surface et la surface sera dessinée à l'écran. Cependant, sur certains appareils plus anciens et certains chipsets spécifiques, cela peut entraîner une baisse des performances. Dans ce cas, vous pouvez désactiver le dessin sur la surface de l'application et tous les événements de dessin seront alors dessinés directement dans le tampon de l'écran. Vous pouvez utiliser cette fonction pour activer ou désactiver la surface d'application à tout moment dans votre jeu, mais vous devez noter qu'il faut au moins une image (étape) pour que les changements prennent effet. Désactiverla surface d'application Désactivera toute correction du rapport hauteur/largeur, tel que défini dans les options de jeu pour la plate-forme cible. Cela signifie que vous devez disposer d'une méthode permettant de mettre le jeu à l'échelle de l'appareil, de la fenêtre ou de l'écran requis. Si vous l'activez à nouveau, les options du jeu prendront effet à l'étape suivante. Il est recommandé d'utiliser cette fonction avec précaution car elle aura un impact sur la façon dont tout est dessiné à l'écran. En général, il est préférable de l'utiliser comme option dans un menu de votre jeu, ou de ne l'appeler qu'une seule fois au début de votre jeu et d'écrire votre code en fonction de son activation ou de sa désactivation. code pour savoir si vous l'avez activée ou désactivée. Syntaxe : application_surface_enable(enable) ; Arguments Description activer Défini pour activer(true) ou désactiver(false) la surface d'application. Les retours : N/A Exemple : if keyboard_check_pressed(vk_space) { if application_surface_is_enabled() { application_surface_enable(false) ; } else { application_surface_enable(true) ; } } Le code ci-dessus vérifie l'appui sur une touche et active ou désactive la surface de l'application en fonction de son état (comme dans un menu d'options).
