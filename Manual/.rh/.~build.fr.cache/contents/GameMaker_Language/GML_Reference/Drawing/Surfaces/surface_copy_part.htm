surface_copy_part Cette fonction prend simplement l'image d'une surface et la copie sur une autre à la position locale spécifiée dans cette surface (où la position (0,0) est le coin supérieur gauche de la surface de destination). Vous pouvez spécifier une position x et y locale à partir de laquelle copier, ainsi que la largeur et la hauteur de la section. Veuillez noter que ce sont des coordonnées basées sur la taille de la surface et non sur la position à laquelle la surface est dessinée dans la pièce. Si la surface de destination possède déjà des informations ce sera remplacée par la copie et la fonction ne change pas la surface de la source de quelque façon. REMARQUE: lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment car elles sont stockées dans la mémoire de texture. Vous devez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists () avant de les référencer directement. Syntaxe:surface_copy_part(destination, x, y, source, xs, ys, ws, hs); Argument Description destination ID de la surface sur laquelle copier l'autre surface. x La position x vers laquelle copier. y La position y vers laquelle copier. source L'ID de la surface à copier. xs Position x dans la surface source à partir de laquelle effectuer la copie. ys La position y dans la surface source à partir de laquelle effectuer la copie. ws Largeur de la zone de la surface source à partir de laquelle effectuer la copie. hs Hauteur de la zone de la surface source à partir de laquelle effectuer la copie. Retour:N/A Exemple:if view_current == 0     {     surface_copy_part(surf, 0, 0, temp_surf, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y);     } else     {     draw_surface(surf, 0, 0);     } Le code ci-dessus vérifiera la vue en cours de dessin et s'il s'agit de la vue [0], il copie la surface indexée dans la variable "temp_surf" sur la surface indexée dans la variable "surf". La zone copiée correspond à un rectangle formé par la position relative de la souris dans la surface telle qu'elle serait dessinée dans la vue [1]. Si la vue actuelle est autre chose que la vue [0], la surface "surf" est dessinée à l'écran. Retour: Surfaces Suivant:surface_depth_disable © Copyright YoYo Games Ltd. 2021 Tous droits réservés
