surface_reset_target Avec cette fonction, vous réinitialisez tous les autres dessins de la surface cible à l'écran. Veuillez noter que pour commencer à dessiner sur une surface, vous devez d'abord avoir appelé la fonction surface_set_target() puis celui-ci une fois que vous avez terminé, pour chaque cible de surface que vous avez définie, sinon rien ne sera dessiné à l'écran car tout autre dessin (même dans d'autres cas) sera effectué sur la surface. Vous devez également vous rendre compte que rien ne sera vu si la surface elle-même n'est pas dessinée à l'écran lors de l'événement de dessin d'une instance. REMARQUE: Si vous n'avez pas précédemment défini de cible de rendu avec la fonction surface_set_target (), l'utilisation de cette fonction mettra fin en silence (c'est-à-dire sans aucun message d'erreur) à toute exécution de code supplémentaire pour l'événement. Syntaxe:surface_reset_target(); Retour:N/A Exemple:if view_current = 0     {     surface_set_target(surf);     with (obj_Effect)         {         draw_self();         }     surface_reset_target();     } else     {     draw_surface(surf, 0, 0);     } Le code ci-dessus vérifiera quelle vue est actuellement dessinée, et si c'est view [0], il définit la cible de dessin sur une surface et dessine toutes les instances de l'objet "obj_Effect" avant de réinitialiser à nouveau la cible de dessin. Si la vue n'est pas vue [0], la surface est dessinée sur l'écran. Retour: Surfaces Suivant:surface_copy © Copyright YoYo Games Ltd. 2021 Tous droits réservés
