surface_depth_disable Cette fonction désactive la génération automatique de tampons de profondeur pour toutes les surfaces qui sont créées après qu'elle ait été désactivée. Normalement, toutes les surfaces ont des tampons de profondeur, de sorte que si vous dessinez des objets 3D sur celles-ci, ils seront triés correctement par profondeur afin de maintenir la parité fonctionnelle entre le rendu sur les surfaces et le rendu sur le tampon arrière. Cependant, l'allocation de tampons de profondeur double essentiellement la quantité de mémoire requise pour les surfaces, ce qui peut constituer une surcharge excessive et inutile, en particulier si votre jeu est très gourmand en mémoire. surtout si votre jeu est très gourmand en mémoire, et pour les jeux 2D, ils ne sont généralement pas nécessaires. Pour utiliser cette fonction, vous devez l'appeler avant de créer une surface, puis vous pouvez l'activer à nouveau après (si nécessaire). Toutes les surfaces créées lorsque le tampon de profondeur est désactivé n'utiliseront plus de tampon de profondeur lorsqu'elles seront dessinées. les objets dessinés sur la surface ont une parité de profondeur. Les images ci-dessous illustrent cela en pratique : Dans l'image ci-dessus, nous dessinons un rectangle et un triangle en utilisant un tampon de sommets en 3D. Les coordonnées de pour le triangle rouge le place derrière le rectangle vert, mais le code pour dessiner le triangle est appelé après le code pour le rectangle, et lorsque le tampon de profondeur est activé, le triangle est toujours dessiné derrière le rectangle car la profondeur 3D est respectée. car la profondeur 3D est respectée. Cependant, lorsque nous désactivons le tampon de profondeur pour la surface, seul l'ordre du code est respecté et le triangle est donc dessiné au-dessus du rectangle. REMARQUE: Pour préserver correctement la profondeur en 3D lors de l'utilisation de surfaces, vous devez non seulement activer le tampon de profondeur, mais aussi activer l'écriture et le test de l'axe des z. Voir les fonctions gpu_set_ztestenable() et gpu_set_zwriteenable() pour plus d'informations. REMARQUE: si votre jeu est 100 % 2D ou si vous savez que vous n'avez pas besoin d'un tampon de profondeur de surface, vous pouvez simplement appeler cette fonction une fois au début du jeu pour désactiver le tampon. Syntaxe : surface_depth_disable(disable) ; Arguments Description désactiver S'il est défini à true, le tampon de profondeur sera désactivé pour les surfaces créées. Les retours : N/A Exemple : var _depth_setting = surface_get_depth_disable() ; surface_depth_disable(false) ; surface_with_depth = surface_create(room_width, room_height) ; surface_depth_disable(true) ; surface_without_depth = surface_create(room_width, room_height) ; surface_depth_disable(_depth_setting) ; Le code ci-dessus stocke le paramètre actuel de la mémoire tampon de profondeur dans une variable, puis crée deux surfaces, l'une avec la mémoire tampon de profondeur activée et l'autre sans, avant de restaurer à nouveau le paramètre original.
