draw_enable_swf_aa Cette fonction vous permet d'activer ou de désactiver l'anticrénelage (AA) pour les sprites vectoriels au format SWF. L'AA lisse simplement les bords des images vectorielles pour leur donner un aspect plus agréable. La quantité de AA utilisée dépendra de la valeur définie à l'aide de la fonction draw_set_swf_aa_level(). Par défaut, cette fonction est désactivée. Syntaxe : draw_enable_swf_aa(enable) ; Arguments Description activer Active(true) ou désactive(false) l'AA pour tous les sprites SWF. Les retours : N/A Exemple : si draw_get_swf_aa_level() == 0 { draw_enable_swf_aa(true) ; draw_set_swf_aa_level(0.5) ; } Le code ci-dessus vérifie la valeur AA pour les sprites au format SWF, et si elle est égale à 0, il active AA et fixe la valeur à 0,5.
