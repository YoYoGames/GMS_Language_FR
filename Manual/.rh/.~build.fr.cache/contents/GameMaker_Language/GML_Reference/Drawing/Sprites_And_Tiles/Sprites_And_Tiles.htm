Sprites et tuiles Bien que vous puissiez configurer des arrière-plans dans l'éditeur de salle et que vous puissiez assigner des sprites aux instances, souvent cela n'est pas assez flexible pour le jeu que vous créez. Par exemple, si vous avez une instance qui doit dessiner un menu, vous pouvez vouloir afficher plusieurs sprites de boutons à la fois. ou si vous avez un jeu avec des arrière-plans défilants en parallaxe, vous devrez être en mesure de les définir par le biais du code d'une instance de contrôleur quelconque. Dans ces situations, vous devriez utiliser les fonctions suivantes qui peuvent dessiner des sprites et des arrière-plans n'importe où dans la pièce. Certaines de ces fonctions vous permettent également de manipuler la forme, la taille, la couleur et l'alpha du sprite pour obtenir des effets spéciaux. Vous pouvez affecter un sprite à une instance et, par défaut, GameMaker Studio 2 le dessinera avec les propriétés données par les différentes variables du sprite, Cependant, si vous ajoutez quoi que ce soit dans l'événement draw, vous "prenez le contrôle" et devez indiquer à GameMaker Studio 2 ce qu'il faut dessiner exactement.
