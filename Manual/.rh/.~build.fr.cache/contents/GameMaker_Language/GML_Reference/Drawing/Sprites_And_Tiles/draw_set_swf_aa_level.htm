draw_set_swf_aa_level Cette fonction peut être utilisée pour définir le niveau d'anticrénelage (AA) pour les sprites vectoriels au format SWF. Il peut s'agir d'une valeur réelle comprise entre 0 et 1 qui " lisse " les bords de ces sprites. Notez que pour voir cet effet, vous devez d'abord avoir activé l'AA en utilisant la fonction draw_enable_swf_aa(). Syntaxe : draw_set_swf_aa_level(AA) ; Arguments Description AA La valeur d'anti-crénelage à utiliser entre 0 et 1. Les retours : N/A Exemple : si draw_get_swf_aa_level() == 0 { draw_enable_swf_aa(true) ; draw_set_swf_aa_level(0.5) ; } Le code ci-dessus vérifie la valeur AA pour les sprites au format SWF, et si elle est égale à 0, il active AA et fixe la valeur à 0,5.
