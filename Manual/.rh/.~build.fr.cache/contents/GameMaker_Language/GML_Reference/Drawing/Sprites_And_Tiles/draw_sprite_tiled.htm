dessiner_sprite_inclinaison Cette fonction prendra un sprite et le mettra en mosaïque de manière répétée sur toute la vue (ou la pièce si aucune vue n'est définie), à partir des coordonnées que vous donnez dans la fonction. Le tuilage est basé sur la largeur et la hauteur du sprite telles que définies par les variables variables sprite de l'instance qui exécute le code. Cette fonction est destinée aux projections 2D (orthographiques) uniquement, et ne fonctionnera pas correctement lorsqu'une projection 3D est utilisée. Syntaxe : draw_sprite_tiled(sprite, subimg, x, y) ; Arguments Description sprite L'index du sprite à dessiner. sous-image La sous-image (frame) du sprite à dessiner (image_index ou -1 correspondent à la frame d'animation actuelle de l'objet). x La coordonnée x de l'endroit où dessiner le sprite. y La coordonnée y de l'endroit où dessiner le sprite. Les retours : N/A Exemple : draw_sprite_tiled(sprite_index, image_index, x, y) ; Ceci dessinera le sprite assigné à l'instance (sprite_index) et son image d'animation actuelle (image_index) à la position x et y de l'instance, et en mosaïque horizontalement et verticalement dans la vue.
