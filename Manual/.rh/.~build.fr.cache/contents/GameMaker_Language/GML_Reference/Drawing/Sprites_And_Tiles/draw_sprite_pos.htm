draw_sprite_pos Avec cette fonction, vous pouvez dessiner un sprite déformé sur la zone définie par les quatre coordonnées d'angle. Les deux premiers arguments sont le sprite à dessiner et la sous-image du sprite (le même que dans la fonction draw_sprite() ) mais les suivants sont ceux qui définissent la position de chacun des quatre coins de la boîte englobante du sprite donné. Celles-ci doivent être données dans le sens des aiguilles d'une montre, de sorte que la première coordonnée est en haut à gauche, puis en haut à droite, puis en bas à droite et enfin en bas à gauche. Vous pouvez également définir une valeur pour l'alpha du sprite pour le dessiner avec transparence. L'image ci-dessous illustre le fonctionnement de cette fonction: ATTENTION! L'image ci-dessus est uniquement à des fins d'illustration, et si vous utilisez cette fonction sur un sprite, vous obtiendrez des résultats différents et pourrez rencontrer un «cisaillement» de texture en raison de la façon dont un sprite est construit à partir d'un quad de primitives. REMARQUE: Cette fonction n'est utile que pour les sprites bitmap et ne fonctionnera pas avec les sprites SWF ou JSON (Spine). REMARQUE: lorsque vous utilisez cette fonction, vous devez avoir le paramètre de recadrage automatique désactivé pour les pages de texture. Syntaxe:draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha); Argument Description sprite L'index du sprite à dessiner. subimg Le subimg (frame) du sprite à dessiner (image_index ou -1 en corrélation avec l'image courante de l'animation dans l'objet). x1 La première coordonnée x. y1 La première coordonnée y. x2 La deuxième coordonnée x. y2 La deuxième coordonnée y. x3 La troisième coordonnée x. y3 La troisième coordonnée y. x4 La quatrième coordonnée x. y4 La quatrième coordonnée y. alpha L'alpha du sprite (de 0 à 1 où 0 est transparent et 1 opaque). Retour:N/A Exemple:draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1); Le code ci-dessus dessinera le sprite associé à l'instance exécutant le code déformé autour de la position x / y de l'instance et avec un alpha complètement opaque.
