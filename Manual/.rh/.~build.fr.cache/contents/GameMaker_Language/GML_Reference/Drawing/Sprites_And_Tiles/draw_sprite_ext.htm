draw_sprite_ext Cette fonction va dessiner le sprite donné comme dans la fonction draw_sprite() mais avec des options supplémentaires pour changer l'échelle, le mélange, la rotation et l'alpha du sprite dessiné. Le changement de ces valeurs ne modifie en rien la ressource (seulement la façon dont elle est dessinée), et vous pouvez utiliser n'importe quelle variable de sprite disponible au lieu de valeurs directes pour tous les arguments de la fonction. dans la fonction. L'image ci-dessous illustre comment différentes valeurs affectent le dessin du sprite : REMARQUE: le mélange des couleurs n'est recommandé pour la cible HTML5 que lorsque WebGL est activé, bien que vous puissiez toujours définir la couleur de mélange si elle n'est pas activée et le sprite sera mélangé normalement. Cependant, tout mélange de cette manière crée un sprite dupliqué qui est ensuite stocké dans la base de données. sprite qui est ensuite stocké dans le cache et utilisé au besoin. C'est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de votre jeu, à moins que vous n'activiez WebGL. Si vous ne souhaitez pas utiliser WebGL, vous pouvez définir la taille du cache des polices pour essayer de limiter les effets de la taille de la police. la taille du cache des polices pour essayer de limiter ce phénomène si nécessaire en utilisant la fonction sprite_set_cache_size(). REMARQUE: cette fonction peut ne pas fonctionner comme prévu lorsque vous utilisez des sprites d'animation de squelette, et vous pouvez constater que la fonction ne dessine que la première image de la pose par défaut. Vous devriez utiliser les fonctions draw_skeleton_* à la place. Syntaxe : draw_sprite_ext( sprite, subimg, x, y, xscale, yscale, rot, colour, alpha ) ; Arguments Description sprite L'index du sprite à dessiner. sous-image La sous-image (frame) du sprite à dessiner (image_index ou -1 correspondent à la frame d'animation actuelle de l'objet). x La coordonnée x de l'endroit où dessiner le sprite. y La coordonnée y de l'endroit où dessiner le sprite. xscale La mise à l'échelle horizontale du sprite, sous forme de multiplicateur : 1 = mise à l'échelle normale, 0.5 est la moitié etc... yscale La mise à l'échelle verticale du sprite sous forme de multiplicateur : 1 = mise à l'échelle normale, 0.5 est la moitié etc... pourriture La rotation du sprite. 0=droite vers le haut, 90=rotation de 90 degrés dans le sens inverse des aiguilles d'une montre etc... couleur La couleur avec laquelle le sprite doit être mélangé. c_white permet de l'afficher normalement. alpha L'alpha du sprite (de 0 à 1 où 0 est transparent et 1 opaque). Les retours : N/A Exemple : draw_sprite_ext(sprite_index, image_index, x, y, image_xscale, image_yscale, image_angle, image_blend, image_alpha) ; Ceci dessinera le sprite assigné à l'instance avec toutes ses valeurs par défaut (essentiellement la même chose que l'utilisation de draw_self).
