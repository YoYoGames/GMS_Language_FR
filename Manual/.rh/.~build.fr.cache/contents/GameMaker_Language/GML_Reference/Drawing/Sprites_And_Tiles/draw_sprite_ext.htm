draw_sprite_ext Cette fonction dessinera le sprite donné comme dans la fonction draw_sprite() mais avec des options supplémentaires pour changer l'échelle, le mélange, la rotation et l'alpha du sprite dessiné. La modification de ces valeurs ne modifie pas la ressource de quelque manière que (seulement comment elle est), et vous pouvez utiliser l' une des disponibles des variables sprites au lieu des valeurs directes pour tous les arguments de la fonction. L'image ci-dessous illustre comment différentes valeurs affectent le dessin du sprite: REMARQUE: La fusion des couleurs n'est recommandée pour la cible HTML5 que lorsque WebGL est activé, bien que vous puissiez toujours définir la couleur de fusion si elle n'est pas activée et elle mélangera l'image-objet comme d'habitude. Cependant, tout mélange de cette manière crée un sprite en double qui est ensuite stocké dans le cache et utilisé lorsque cela est nécessaire. Ceci est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de vos jeux à moins que vous n'activiez WebGL. si vous ne souhaitez pas utiliser WebGL, vous pouvez définir la taille du cache de polices pour essayer de limiter cela si nécessaire en utilisant la fonction sprite_set_cache_size (). REMARQUE: cette fonction peut ne pas fonctionner comme prévu lors de l'utilisation de sprites d'animation squelette et vous pouvez constater que la fonction dessine uniquement la première image de la pose par défaut. Vous devriez utiliser le draw_skeleton_* fonctionne à la place. Syntaxe:draw_sprite_ext( sprite, subimg, x, y, xscale, yscale, rot, colour, alpha ); Argument Description sprite L'index du sprite à dessiner. subimg Le subimg (frame) du sprite à dessiner (image_index ou -1 en corrélation avec l'image courante de l'animation dans l'objet). x La coordonnée x de l'endroit où dessiner le sprite. y La coordonnée y de l'endroit où dessiner le sprite. xscale La mise à l'échelle horizontale du sprite, sous forme de multiplicateur: 1 = mise à l'échelle normale, 0,5 est la moitié etc... yscale La mise à l'échelle verticale du sprite en tant que multiplicateur: 1 = mise à l'échelle normale, 0,5 est la moitié etc... rot La rotation du sprite. 0 = vers le haut, 90 = rotation de 90 degrés dans le sens inverse des aiguilles d'une montre, etc. colour La couleur avec laquelle mélanger le sprite.c_white est de l'afficher normalement. alpha L'alpha du sprite (de 0 à 1 où 0 est transparent et 1 opaque). Retour:N/A Exemple:draw_sprite_ext(sprite_index, image_index, x, y, image_xscale, image_yscale, image_angle, image_blend, image_alpha); Cela dessinera les instances assignées au sprite avec toutes ses valeurs par défaut (essentiellement la même chose que l'utilisation draw_self ).
