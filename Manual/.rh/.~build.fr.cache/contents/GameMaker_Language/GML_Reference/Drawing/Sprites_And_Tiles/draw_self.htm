draw_self Cette fonction dessine le sprite affecté à l'instance exactement comme il serait dessiné si l'événement draw ne contenait ni code ni actions, et reflétera les modifications qui ont été apportées aux variables de sprite dans d'autres événements. REMARQUE: si aucun sprite n'est affecté à l'instance, cette fonction lèvera une erreur! Syntaxe:draw_self(); Retour:N/A Exemple:draw_self(); Cela fait que l'instance se dessine elle-même avec les propriétés définies par les variables de sprite intégrées.
