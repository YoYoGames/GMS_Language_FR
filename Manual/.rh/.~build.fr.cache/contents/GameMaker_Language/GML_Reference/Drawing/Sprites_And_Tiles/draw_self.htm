dessiner_soi Cette fonction dessine le sprite affecté à l'instance exactement comme il serait dessiné si l'événement draw ne contenait aucun code ou action, et reflète les modifications apportées aux variables du sprite dans d'autres événements. REMARQUE: Si aucun sprite n'est attribué à l'instance, cette fonction émettra une erreur ! Syntaxe : draw_self() ; Les retours : N/A Exemple : draw_self() ; L'instance se dessine alors avec les propriétés définies par les variables de sprite intégrées.
