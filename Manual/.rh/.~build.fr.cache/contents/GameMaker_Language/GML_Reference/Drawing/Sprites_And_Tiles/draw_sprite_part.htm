draw_sprite_part Avec cette fonction, vous pouvez dessiner une partie de n'importe quel sprite à une position donnée dans la pièce. Comme avec draw_sprite() vous pouvez spécifier un sprite et une sous-image pour le dessin, puis vous devez donner les coordonnées relatives dans le sprite de la zone à sélectionner pour le dessin. Cela signifie qu'une position gauche de 0 et une position haute de 0 serait le coin supérieur gauche du sprite et toutes les autres coordonnées devraient être prises à partir de cette position. L'image ci-dessous montre un exemple de la façon dont cela fonctionne: Vous devez noter que si la page de texture autorise le recadrage automatique, cette fonction peut ne pas fonctionner comme prévu, car l'espace "vide" supplémentaire autour du sprite aura été supprimé pour créer la page de texture. Pour résoudre ce problème, vous devrez définir les paramètres de la page de texture (dans l' éditeur de groupe de texture ) pour désactiver l'option Recadrage automatique. REMARQUE: lorsque vous dessinez avec cette fonction, le décalage x et le décalage y du sprite sont ignorés et la partie du sprite sera dessinée avec le coin supérieur gauche à la position x / y spécifiée dans la pièce. REMARQUE: Cette fonction n'est utile que pour les sprites bitmap et ne fonctionnera pas avec les sprites SWF ou JSON (Spine). Syntaxe:draw_sprite_part(sprite, subimg, left, top, width, height, x, y); Argument Description sprite L'index du sprite à dessiner. subimg Le subimg (frame) du sprite à dessiner (image_index ou -1 en corrélation avec l'image courante de l'animation dans l'objet). left La position x sur le sprite du coin supérieur gauche de la zone à dessiner. top La position y sur le sprite du coin supérieur gauche de la zone à dessiner. width La largeur de la zone à dessiner. height La hauteur de la zone à dessiner. x La coordonnée x de l'endroit où dessiner le sprite. y La coordonnée y de l'endroit où dessiner le sprite. Retour:N/A Exemple:draw_sprite_part(sprite_index, image_index, 4, 0, sprite_width-16, sprite_height-16, x, y ); Cela dessinera les instances assignées au sprite (sprite_index) et son image d'animation actuelle (image_index), mais cela réduira une marge de 4px sur la largeur des deux côtés et une marge de 8 pixels sur la hauteur à partir du bas du pixel original de 24x24 lutin.
