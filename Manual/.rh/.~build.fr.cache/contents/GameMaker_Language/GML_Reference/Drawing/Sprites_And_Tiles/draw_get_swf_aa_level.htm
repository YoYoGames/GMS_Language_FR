draw_get_swf_aa_level Cette fonction peut être utilisée pour obtenir le niveau d'anticrénelage (AA) pour les sprites vectoriels au format SWF. La valeur de retour est comprise entre 0 et 1 et indique à quel point les bords de ces sprites seront dessinés de manière "lisse". Vous pouvez définir le niveau AA à l'aide de la fonction draw_set_swf_aa_level(). Syntaxe : draw_get_swf_aa_level() ; Les retours : Real Exemple : si draw_get_swf_aa_level() == 0 { draw_enable_swf_aa(true) ; draw_set_swf_aa_level(0.5) ; } Le code ci-dessus vérifie la valeur AA pour les sprites au format SWF, et si elle est égale à 0, il active AA et fixe la valeur à 0,5.
