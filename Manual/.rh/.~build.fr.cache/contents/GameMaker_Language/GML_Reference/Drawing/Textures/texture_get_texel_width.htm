largeur de texture_get_texel Un texel, ou élément de texture, est l'unité fondamentale de l'espace de texture utilisé en infographie. Les textures sont représentées par des tableaux de texels, tout comme les images sont représentées par des tableaux de pixels, et cette fonction renvoie la largeur d'un seul texel de la page de texture de l'image utilisée. Syntaxe : texture_get_texel_width(tex) ; Arguments Description tex Le pointeur de ressource de la page de texture à utiliser Les retours : Real Exemple : var tex = sprite_get_texture(sprite_index, 0) ; tex_w = texture_get_texel_width(tex) ; tex_h = texture_get_texel_height(tex) ; Le code ci-dessus permet d'obtenir la largeur et la hauteur du texel de la texture provenant d'un actif de sprite.
