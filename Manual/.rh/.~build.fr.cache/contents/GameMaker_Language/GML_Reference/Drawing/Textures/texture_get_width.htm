texture_get_width Renvoie la largeur de la texture avec l'identifiant donné, qui est toujours une valeur comprise entre 0 et 1. Cela peut ensuite être utilisé lors du mappage de textures sur des modèles ou des primitives. Syntaxe:texture_get_width(tex); Argument Description tex Le pointeur de ressource de page de texture à utiliser Retour:Real Exemple:tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0)); Le code ci-dessus obtiendra la largeur de la texture tirée d'un élément de sprite.
