texture_get_uvs Cette fonction renvoie un tableau 1D avec 4 éléments représentant les coordonnées UV de l'image sur la page de texture, en remplissant le tableau avec les valeurs suivantes : [0] = gauche [1] = haut [2] = droite [3] = fond 4] = nombre de pixels que le compilateur d'actifs a enlevé du côté gauche des sprites (actifs de sprites uniquement). 5] = nombre de pixels que le compilateur d'actifs a enlevé du côté supérieur des sprites (actifs de sprites uniquement) 6] = pourcentage normalisé de données de pixels de la largeur du sprite original qui ont été sauvegardées dans la page de texture (actifs de sprites uniquement). 7] = pourcentage normalisé des données de pixels de la hauteur du sprite original qui ont été sauvegardées dans la page de texture (actifs de sprites uniquement). Cette valeur peut ensuite être utilisée dans d'autres fonctions de dessin, en particulier dans les fonctions générales 3D et certaines des fonctions primitives 2D, ainsi que dans les fonctions Shader. Si vous avez besoin de l'UVS pour un sprite, vous pouvez utiliser les fonctions sprite_get_uvs() et pour une police, font_get_uvs(). Syntaxe : texture_get_uvs(texid) Arguments Description texid L'actif de l'image pour obtenir la texture de Les retours : Matrice 1D (4 - 8 éléments) Exemple : var tex = texture_get_uvs(surf_back) ; tex_left = tex[0] ; tex_top = tex[1] ; tex_right = tex[2] ; tex_left = tex[3] ; Le code ci-dessus stocke les coordonnées UV pour la texture de surface donnée dans un tableau local, puis affecte les valeurs aux variables d'instance.
