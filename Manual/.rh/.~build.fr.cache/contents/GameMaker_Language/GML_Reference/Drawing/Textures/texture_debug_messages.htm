texture_debug_messages Cette fonction peut être utilisée pour activer ou désactiver les messages de débogage de texture. Lorsqu'elle est activée (définie sur true ), des informations supplémentaires sur l'utilisation de la page de texture seront envoyées à la fenêtre de sortie. Mis à false pour désactiver à nouveau cette sortie. Syntaxe:texture_debug_messages(enable); Argument Description enable Activer ou désactiver les messages de débogage de texture Retour:N/A Exemple:if global.debug     {     texture_debug_messages(true);     } Le code ci-dessus vérifiera la valeur d'une variable globale et si elle est définie sur true puis les messages de débogage de texture sont activés.
