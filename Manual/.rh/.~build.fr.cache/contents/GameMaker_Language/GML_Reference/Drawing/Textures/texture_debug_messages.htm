textures_debug_messages Cette fonction peut être utilisée pour activer ou désactiver les messages de débogage de la texture. Lorsqu'elle est activée (réglée sur true), des informations supplémentaires sur l'utilisation des pages de texture seront envoyées dans la fenêtre de sortie. Définissez-la à false pour désactiver à nouveau cette sortie. Syntaxe : texture_debug_messages(enable) ; Arguments Description activer Activez ou désactivez les messages de débogage de la texture. Les retours : N/A Exemple : if global.debug { texture_debug_messages(true) ; } Le code ci-dessus vérifiera la valeur d'une variable globale et si elle est définie comme vraie, les messages de débogage de la texture seront activés.
