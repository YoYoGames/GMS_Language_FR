Texte Vous disposez d'un certain nombre de fonctions pour dessiner du texte sous diverses formes. Il convient de noter que certaines de ces fonctions modifient les couleurs et les tailles du texte de manière dynamique lors du dessin, mais lorsque vous travaillez avec HTML5 ou des appareils, ces fonctions peuvent ralentir votre jeu ou paraître médiocres et il peut donc être préférable d'avoir divers des ressources de polices dimensionnées plutôt que d'utiliser un dessin à l'échelle, ou avoir de grandes ressources de polices, puis les dessiner à l'échelle vers le bas plutôt que vers le haut. De plus, si vous avez besoin de colorer de grands blocs de texte, il est préférable d'utiliser le draw_set_colour() et draw_set_alpha() fonctionne d'abord avec le simple draw_text fonction, plutôt que les fonctions étendues qui ont une composante de couleur.
