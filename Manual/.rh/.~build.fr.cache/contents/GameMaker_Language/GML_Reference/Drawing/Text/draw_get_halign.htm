draw_get_halign Cette fonction est utilisée pour obtenir le paramètre d'alignement du texte sur l'axe horizontal, et renvoie l'une des constantes énumérées ci-dessous. Syntaxe : draw_get_halign() ; Les retours : Constant Constant Alignement fa_gauche fa_centre fa_right Exemple : if draw_get_halign() != fa_left { draw_set_halign(fa_left) ; } draw_text(100, 32, "Score : " + string(score)) ; Le code ci-dessus vérifie si l'alignement actuel du texte est aligné à gauche, et s'il ne l'est pas, il est défini, puis du texte est dessiné.
