draw_get_font Cette fonction permet d'obtenir la police actuellement attribuée pour le dessin du texte. La fonction renvoie -1 si aucune police n'est définie, ou la valeur ID (un entier positif) pour la ressource de police assignée. Syntaxe : draw_get_font() ; Les retours : Entier (-1 ou valeur d'identification de la police) Exemple : if draw_get_font() != fnt_MainMenu { draw_set_colour(c_blue) ; draw_set_font(fnt_MainMenu) ; } draw_text(room_width / 2, 200, "MENU" ) ; Le code ci-dessus vérifie quelle police est utilisée pour le dessin et si ce n'est pas fnt_MainMenu , la police et la couleur de dessin sont définies avant que le texte ne soit affiché à l'écran.
