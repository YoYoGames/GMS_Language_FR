draw_get_valign Cette fonction est utilisée pour obtenir le paramètre d'alignement du texte le long de l'axe vertical, et renvoie l'une des constantes énumérées ci-dessous. Syntaxe : draw_get_valign() ; Les retours : Constant Constant Alignement fa_top fa_middle fa_bottom Exemple : if draw_get_valign() != fa_top { draw_set_valign(fa_top) ; } draw_text(100, 32, "Score : " + string(score)) ; Le code ci-dessus vérifie si l'alignement actuel du texte est aligné sur le haut, et s'il ne l'est pas, il est activé et du texte est dessiné.
