draw_text_ext_transformed Cette fonction est une combinaison de la base draw_text () fonction avec le draw_text_ext () et draw_text_transformed () fonctions, vous permettant de mettre à l'échelle et de faire pivoter le texte tout en conservant un espacement de ligne spécifique et une largeur maximale par ligne. Notez que l'argument "largeur" est basé sur une échelle de 1, donc si l'échelle est différente, cette valeur doit être modifiée proportionnellement. Par exemple, si la largeur de base d'un saut de ligne est de 300 et que vous définissez l'échelle sur 2, le texte apparaîtra incorrect, dépassant la largeur donnée. Au lieu de cela, vous devriez avoir défini la largeur sur 150 pour compenser la mise à l'échelle. Syntaxe:draw_text_ext_transformed(x, y, string, sep, w, xscale, yscale, angle); Argument Description x La coordonnée x de la chaîne dessinée. y La coordonnée y de la chaîne dessinée. string La chaîne à dessiner. sep La distance en pixels entre les lignes de texte. w La largeur maximale en pixels de la chaîne avant un saut de ligne. xscale L'échelle horizontale. yscale L'échelle verticale. angle L'angle du texte. Retour:N/A Exemple:draw_set_halign(fa_center); draw_set_valign(fa_middle); image_angle += 1; draw_text_ext_transformed(room_width / 2, room_height / 2, keyboard_string, 10, 300, 2, 2, image_angle); Le code ci-dessus dessinera le texte donné au milieu de la pièce, avec une longueur de chaîne maximale de 300 pixels, un espacement entre chaque ligne de 10 pixels, tournant et redimensionné à deux fois sa taille d'origine.
