draw_set_valign Cette fonction est utilisée pour aligner le texte le long de l'axe vertical et la modification de l'alignement vertical changera la position et la direction dans lesquelles tout le texte est dessiné, la valeur par défaut étant fa_top. Les constantes suivantes sont acceptées: Constant Alignement fa_top fa_middle fa_bottom Syntaxe:draw_set_valign(valign); Argument Description valign Alignement vertical. Retour:N/A Exemple:draw_set_halign(fa_center); draw_set_valign(fa_middle); draw_text(100, 32, "Score: " + string(score)); Le code ci-dessus dessinera la partition centrée autour du centre même du texte.
