draw_set_halign Cette fonction est utilisée pour aligner le texte le long de l'axe horizontal et changer l'alignement horizontal changera la position et la direction dans lesquelles tout le texte est dessiné avec la valeur par défaut étant fa_left. Les constantes suivantes sont acceptées: Constant Alignement fa_left fa_center fa_right Syntaxe:draw_set_halign(halign); Argument Description halign Alignement horizontal. Retour:N/A Exemple:draw_set_halign(fa_left); draw_text(100, 32, "Score: " + string(score)); draw_set_halign(fa_right); draw_text(room_width - 100, 32, "Health: " + string(health)); Le code ci-dessus dessinera deux chaînes sur la même ligne, le score étant aligné à gauche et la santé alignée à droite.
