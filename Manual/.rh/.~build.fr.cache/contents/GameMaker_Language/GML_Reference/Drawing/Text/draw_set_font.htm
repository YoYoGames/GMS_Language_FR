draw_set_font Cette fonction définira la police à utiliser pour tous les autres dessins de texte. Cette police doit avoir été ajoutée aux éléments de police du jeu ou avoir été créée en utilisant soit le font_add () ou font_add_sprite (). Syntaxe:draw_set_font(font); Argument Description font Le nom de la police à utiliser. Retour:N/A Exemple:draw_set_colour(c_blue); draw_set_font(fnt_Game); draw_text(200, 200, "Hello World" ); Le code ci-dessus dessinera le texte donné en utilisant la police indexée dans la variable "fnt_Game" et colorée en bleu.
