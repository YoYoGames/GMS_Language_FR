draw_text_ext_colour Cette fonction est une combinaison de la base draw_text () fonction avec le draw_text_ext () et draw_text_colour () fonctions, vous permettant de définir des couleurs de dégradé pour le texte ainsi que l'interligne et la largeur maximale par ligne tous ensemble. REMARQUE: la fusion de dégradé n'est pas disponible pour la cible HTML5 sauf si WebGL est activé, bien que vous puissiez toujours définir les couleurs de fusion et il fusionnera la police avec la première couleur donnée. Cependant, tout mélange de cette manière crée une police en double qui est ensuite stockée dans le cache et utilisée si nécessaire, ce qui est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de vos jeux. Vous pouvez définir la taille du cache de polices pour essayer de limiter cela si nécessaire en utilisant la fonction font_set_cache_size (). Syntaxe:draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha); Argument Description x La coordonnée x de la chaîne dessinée. y La coordonnée y de la chaîne dessinée. string La chaîne à dessiner. sep La distance en pixels entre les lignes de texte. w La largeur maximale en pixels de la chaîne avant un saut de ligne. c1 La couleur du coin supérieur gauche du texte dessiné. c2 La couleur du coin supérieur droit du texte dessiné. c3 La couleur du coin inférieur droit du texte dessiné. c4 La couleur du coin inférieur gauche du texte dessiné. alpha L'alpha du texte. Retour:N/A Exemple:draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1); Le code ci-dessus dessinera le texte que l'utilisateur tape sur le clavier, le divisant en nouvelles lignes chaque fois que la longueur de la chaîne pour cette ligne dépasse 300 pixels. le code maintiendra également une séparation de 5 pixels entre les lignes si cela se produit. Chaque ligne du texte sera colorée en utilisant un dégradé de bleu, avec du bleu clair en haut et du bleu foncé en bas.
