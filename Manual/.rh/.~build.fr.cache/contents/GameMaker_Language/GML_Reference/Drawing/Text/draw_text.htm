draw_text Avec cette fonction, vous pouvez dessiner n'importe quelle chaîne à n'importe quelle position dans la pièce (pour dessiner des nombres réels, vous devez utiliser le string () fonction pour les convertir en texte). Pour combiner des chaînes, vous pouvez utiliser + (voir l'exemple ci-dessous) et vous pouvez également utiliser \ n dans une chaîne pour ajouter un saut de ligne afin qu'il soit dessiné sur plusieurs lignes (pour savoir comment formater correctement une chaîne et quels caractères d'échappement vous pouvez utilisation, veuillez voir ici ). La couleur du texte et l'alpha sont régis par l'alpha de base actuel et les valeurs de couleur définies par draw_set_alpha () et draw_set_colour (). REMARQUE: la position réelle du texte sera influencée par les valeurs d'alignement définies par draw_set_halign () et draw_set_valign (). Syntaxe:draw_text(x, y, string); Argument Description x La coordonnée x de la chaîne dessinée. y La coordonnée y de la chaîne dessinée. string La chaîne à dessiner. Retour:N/A Exemple:draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!"); Le code ci-dessus dessinera une chaîne à la position x / y de l'instance, qui utilisera la chaîne stockée dans la variable globale "Nom" et la divisera sur deux lignes.
