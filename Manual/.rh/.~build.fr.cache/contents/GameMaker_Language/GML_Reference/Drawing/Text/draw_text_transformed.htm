draw_text_transformed Cette fonction permet de dessiner du texte de manière similaire à la fonction dessiner_texte() sauf que vous pouvez maintenant choisir de mettre le texte à l'échelle le long de l'axe horizontal ou vertical (ce qui a pour effet de l'étirer ou de le rétrécir) et aussi de faire en sorte que GameMaker Studio 2 de le dessiner selon un angle (où 0 est la valeur normale et chaque degré au-dessus de 0 fait tourner le texte dans le sens inverse des aiguilles d'une montre). Syntaxe : draw_text_transformed(x, y, string, xscale, yscale, angle) ; Arguments Description x La coordonnée x de la chaîne dessinée. y La coordonnée y de la chaîne dessinée. chaîne de caractères La ficelle à tirer. xscale L'échelle horizontale (par défaut 1). yscale L'échelle verticale (par défaut 1). angle L'angle du texte. Les retours : N/A Exemple : draw_set_halign(fa_center) ; draw_set_valign(fa_middle) ; image_angle += 1 ; draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER !", 2, 2, image_angle) ; Le code ci-dessus dessinera le texte donné au milieu de la pièce, en le faisant tourner et en le mettant à l'échelle au double de sa taille originale.
