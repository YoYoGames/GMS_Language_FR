draw_enable_drawevent Avec cette fonction, vous pouvez choisir d'activer(true) ou de désactiver(false) l'événement de dessin pour toutes les instances du jeu, ce qui vous permet de contrôler comment et quand les choses sont dessinées. technique de "saut de trame". Notez que cela n'empêche pas seulement les instances de s'afficher à l'écran, mais supprime complètement l'événement draw, ce qui signifie qu'il faut faire attention car toute logique de jeu présente dans cet événement ne sera pas exécutée non plus. non plus. Une chose importante à comprendre à propos de cette fonction est que si vous l'appelez au début du jeu, avant le rendu de l'image initiale (c'est-à-dire l'événement Create du premier objet dans la première pièce du jeu), la fenêtre du jeu ne sera pas rendue. Cela peut être utile pour les projets en ligne qui nécessitent un serveur qui n'a pas besoin d'effectuer de rendu. Syntaxe : draw_enable_drawevent(enable) ; Arguments Description Activer Défini à true ou false. Les retours : N/A Exemple : frame_skip ++ ; if (frame_skip mod 5) == 0 { draw_enable_drawevent(true) ; } else { draw_enable_drawevent(false) ; } Le code ci-dessus vérifie une variable et s'il retourne vrai, il active l'événement de dessin, sinon l'événement de dessin est désactivé.
