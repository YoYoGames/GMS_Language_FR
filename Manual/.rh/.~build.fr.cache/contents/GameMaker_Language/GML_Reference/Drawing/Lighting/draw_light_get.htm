draw_light_get Cette fonction obtiendra les paramètres de lumière spécifiés sous forme de tableau avec les 6 éléments suivants - 0] = activé / désactivé(vrai / faux) [1] = type de lumière (voir les constantes ci-dessous) [2] = position x [3] = position y [4] = position z 5] = rayon de la lumière (uniquement pour les lumières ponctuelles, 0 pour les lumières directionnelles) [6] = couleur claire (un vrai) Le type de lumière peut être l'une des deux constantes suivantes : Constant Description lighttype_dir La lumière est une lumière directionnelle point_type_lumière La lumière est une lumière ponctuelle Syntaxe : draw_light_get(ind) ; Arguments Description ind Le numéro d'index du feu (de 0 à 7) Les retours : Real Exemple : light_a = draw_light_get(1) ; if light_a[5] < 200 { light_a[5] += 5 ; draw_light_define_point(1, 200, 123, 50, light_a[5], c_white) ; } Le code ci-dessus obtiendra les valeurs utilisées pour définir la lumière indexée comme "1", puis vérifiera le rayon et s'il est inférieur à 200, il sera augmenté et le rayon de la lumière sera fixé à la nouvelle valeur.
