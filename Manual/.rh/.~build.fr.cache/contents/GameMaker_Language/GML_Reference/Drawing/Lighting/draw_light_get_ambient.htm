draw_light_get_ambient Cette fonction renvoie la couleur actuelle utilisée pour l'éclairage ambiant. Syntaxe : draw_light_get_ambient() Les retours : Real Exemple : light_col = draw_light_get_ambient() ; Le code ci-dessus récupère la couleur actuelle de la lumière ambiante et la stocke dans une variable.
