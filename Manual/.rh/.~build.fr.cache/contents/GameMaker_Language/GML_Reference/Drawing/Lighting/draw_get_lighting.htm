draw_get_lighting Cette fonction retournera si l'éclairage est activé ( true ) ou non (false ) pour toute la scène. Syntaxe:draw_get_lighting() Retour:Boolean Exemple:if !draw_get_lighting()     {     draw_set_lighting(true);     } Le code ci-dessus vérifiera si l'éclairage est activé pour la scène, et sinon, il l'active.
