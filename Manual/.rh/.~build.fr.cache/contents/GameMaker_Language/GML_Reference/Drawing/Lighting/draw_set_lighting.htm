draw_set_lighting Cette fonction est utilisée pour activer tous les effets de lumière. Par défaut, elle est désactivée(false). Syntaxe : draw_set_lighting(enable) ; Arguments Description activer activer ou désactiver tous les éclairages(vrai ou faux) Les retours : N/A Exemple : draw_set_lighting(true) ; draw_light_define_direction(1, 0, 1, 0, c_white) ; draw_light_enable(1, true) ; Le code ci-dessus va activer l'éclairage de l'ensemble de la scène, puis définir une lumière blanche directionnelle dans l'espace de la pièce, et enfin allumer cette lumière.
