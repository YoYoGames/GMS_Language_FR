draw_light_define_ambient Cette fonction est utilisée pour contrôler la lumière ambiante d'une scène, qui est la lumière que vous avez dans une scène même sans avoir défini de sources lumineuses ponctuelles ou directionnelles. Il s'agit en fait de la couleur et de la luminosité générales (ou de l'obscurité) d'une scène. La couleur par défaut est c_black. Syntaxe:draw_light_define_ambient(col); Argument Description col La couleur à utiliser (une valeur constante, réelle ou hexadécimale). Retour:N/A Exemple:draw_light_define_ambient(c_white); Le code ci-dessus définira l'éclairage ambiant comme étant blanc.
