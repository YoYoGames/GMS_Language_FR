colour_get_value Cette fonction renverra la valeur (luminosité) de la couleur donnée. Il s'agit de la quantité de «lumière» qui est mélangée à la couleur finale et fait partie de la méthode de teinte, de saturation et de valeur pour définir une couleur. L'image suivante montre comment cette valeur correspond à l'échelle de couleur HSV: Syntaxe:colour_get_value(col); Argument Description col La couleur à vérifier Retour:Int Exemple:col = make_colour_hsv(random(255), 255, colour_get_value(c_teal)); Le code ci-dessus obtient la valeur utilisée pour rendre la couleur constante "c_teal" et utilise ensuite cette valeur pour définir une couleur aléatoire pour avoir la même luminosité, en stockant la couleur résultante dans la variable "col".
