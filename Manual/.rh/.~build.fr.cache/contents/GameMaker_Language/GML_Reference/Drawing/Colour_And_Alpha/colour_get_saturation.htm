colour_get_saturation Cette fonction retournera la saturation de la couleur donnée. Il s'agit de la quantité de teinte de couleur mélangée à la couleur finale et qui fait partie de la méthode de teinte, de saturation et de valeur (luminosité) pour définir une couleur. L'image suivante montre comment cette valeur correspond à l'échelle de couleur HSV: Syntaxe:colour_get_saturation(col); Argument Description col La couleur à vérifier Retour:Int Exemple:col = make_colour_hsv(random(255), colour_get_sat(c_teal), 255); Le code ci-dessus obtient la saturation utilisée pour rendre la couleur constante "c_teal" et utilise ensuite cette valeur pour définir une couleur aléatoire pour avoir la même saturation, en stockant la couleur résultante dans la variable "col".
