couleur_get_hue Cette fonction renvoie la teinte de la couleur donnée. Il s'agit de la teinte "pure" qui fait partie de la méthode de définition d'une couleur par la teinte, la saturation et la valeur (luminosité). L'image suivante illustre comment cette valeur correspond à l'échelle de couleur échelle de couleurs HSV : Syntaxe : couleur_get_hue(col) ; Arguments Description col La couleur à vérifier Les retours : Int Exemple : col = merge_colour(colour_get_hue(c_teal), c_white, 0.5) ; Le code ci-dessus récupère la teinte utilisée pour créer la constante de couleur "c_teal" et la fusionne ensuite avec le blanc dans un rapport 50:50, en stockant la couleur résultante dans la variable "col".
