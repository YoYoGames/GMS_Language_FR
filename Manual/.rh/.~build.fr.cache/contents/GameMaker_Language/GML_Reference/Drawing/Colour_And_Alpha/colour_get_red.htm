colour_get_red Cette fonction renvoie la quantité de rouge utilisée pour créer la couleur donnée, avec une valeur comprise entre 0 et 255, où 0 signifie pas de rouge et 255 est tout rouge. L'image suivante illustre cela: Syntaxe:colour_get_red(col); Argument Description col La couleur à vérifier Retour:Int Exemple:r_comp = colour_get_red(c_teal); Le code ci-dessus obtiendra la composante rouge de la constante de couleur "c_teal" et stockera la valeur dans la variable "r_comp".
