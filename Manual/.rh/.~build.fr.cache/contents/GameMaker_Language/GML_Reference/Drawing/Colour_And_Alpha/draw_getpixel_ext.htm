draw_getpixel_ext Avec cette fonction, vous pouvez obtenir la valeur complète abgr 32bit de n'importe quel pixel qui est dessiné sur la cible de rendu actuelle. Cela signifie que les résultats dépendront de l'événement dans lequel la fonction est appelée, et aussi de la surface cible utilisée. utilisée. REMARQUE: Cette fonction entraîne une baisse considérable des performances et ne doit donc être utilisée qu'en cas de nécessité absolue. Syntaxe : draw_getpixel_ext(x, y) ; Arguments Description x La coordonnée x du pixel à vérifier y La coordonnée y du pixel à vérifier Les retours : Real Exemple : col = draw_getpixel_ext(mouse_x, mouse_y) ; alpha = (col >> 24) & 255 ; blue = (col >> 16) & 255 ; green = (col >> 8) & 255 ; red = col & 255 ; Le code ci-dessus récupère la valeur de la couleur 32 bits à la position de la souris, puis la divise en ses valeurs composantes et les stocke dans des variables.
