draw_getpixel Avec cette fonction, vous pouvez obtenir la valeur de la couleur de n'importe quel pixel qui est dessiné sur la cible de rendu actuelle. Cela signifie que les résultats dépendront de l'événement dans lequel la fonction est appelée, et aussi de la surface cible utilisée. Notez que cette fonction ne renvoie pas de valeurs alpha, pour cela vous devez utiliser draw_getpixel_ext(). REMARQUE: Cette fonction entraîne une baisse considérable des performances et ne doit donc être utilisée qu'en cas de nécessité absolue. Syntaxe : draw_getpixel(x, y) ; Arguments Description x La coordonnée x du pixel à vérifier y La coordonnée y du pixel à vérifier Les retours : Real Exemple : draw_set_colour(draw_getpixel(500,200)) ; Le code ci-dessus définit la couleur de dessin de tous les textes, formulaires et primitives à la couleur du pixel à (500,200) sur l'écran du jeu.
