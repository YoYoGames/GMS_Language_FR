make_colour_hsv GameMaker Studio 2 fournit cette fonction (ainsi que d'autres) pour permettre à l'utilisateur de créer ses propres couleurs. Cette fonction particulière prend trois éléments, la teinte, la saturation et la valeur (également appelée «luminosité») pour créer la couleur souhaitée. Ces valeurs sont comprises entre 0 et 255, vous pouvez donc créer 16 777 216 (256 * 256 * 256) couleurs avec ceci! Ci-dessous, vous pouvez voir une image de l'apparence de ces composants lorsqu'ils sont séparés: L'image de gauche est une ventilation des composants individuels de la fonction, puis sur la droite est une illustration de la façon dont le changement de ces composants affecte la couleur de fin. Comme vous pouvez le voir, même sans changer la teinte (elle reste à 0, rouge), nous pouvons changer radicalement la couleur avec les deux autres paramètres, la saturation faisant passer le ton de couleur du gris (0) à un ton pur (255), et avec la valeur le faisant passer du plus sombre (0) au plus clair (255). Syntaxe:make_colour_hsv(hue, sat, val); Argument Description hue La teinte de la couleur sat À quel point la couleur est saturée val Quelle est la couleur foncée Retour:Integer Exemple:col = make_colour_hsv(100, 145, 255); Le code ci-dessus utilise la fonction pour créer une couleur et stocker sa valeur dans la variable "col" pour une utilisation ultérieure.
