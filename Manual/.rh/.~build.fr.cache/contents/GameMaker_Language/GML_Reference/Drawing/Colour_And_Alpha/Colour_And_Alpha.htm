Couleur et alpha GameMaker Studio 2 propose une sélection complète de fonctions permettant de manipuler la couleur et l'alpha de ce qui est dessiné à l'écran (pour mélanger des images à l'aide de modes de fusion spéciaux et d'autres fonctions spécifiques à PGU, consultez la section sur le contrôle GPU. En utilisant ces fonctions, vous pouvez séparer les couleurs. dans leurs composants, créez de nouvelles couleurs et manipulez l'alpha de dessin pour changer la façon dont les différents éléments de votre jeu sont dessinés ensemble. Outre ces fonctions, il existe également un certain nombre de constantes qui peuvent être utilisées comme couleurs de base dans ces (et d'autres) les fonctions: Constant Couleur Évaluer c_aqua 16776960 c_black 0 c_blue 16711680 c_dkgray 4210752 c_fuchsia 16711935 c_gray 8421504 c_green 32768 c_lime 65280 c_ltgray 12632256 c_maroon 128 c_navy 8388608 c_olive 32896 c_orange 4235519 c_purple 8388736 c_red 255 c_silver 12632256 c_teal 8421376 c_white 16777215 c_yellow 65535 Notez que vous pouvez également créer des couleurs à partir de leur valeur hexadécimale en utilisant au préalable le symbole "$", et ces valeurs peuvent également être utilisées dans les fonctions de couleur pour obtenir les teintes des composants ou pour mélanger, etc... La valeur hexadécimale de ces couleurs est fractionnée en trois parties au format Bleu / Vert / Rouge donc, par exemple, pour définir une couleur violette vous pouvez utiliser:col = $983c95 Où 98 est le composant bleu, 3c est le composant vert et 95 est le composant rouge.
