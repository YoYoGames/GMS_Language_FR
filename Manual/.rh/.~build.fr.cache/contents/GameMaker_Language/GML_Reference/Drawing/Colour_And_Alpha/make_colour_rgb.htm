make_colour_rgb GameMaker Studio 2 fournit cette fonction (ainsi que d'autres) pour permettre à l'utilisateur de créer ses propres couleurs. Cette fonction particulière prend trois composants, le rouge, le vert et le bleu de la couleur que vous souhaitez créer. Ces valeurs sont comprises entre 0 et 255, vous pouvez donc créer 16 777 216 (256 * 256 * 256) couleurs avec ceci! Ci-dessous, vous pouvez voir une image de l'apparence de ces composants lorsqu'ils sont séparés: L'image de gauche est une ventilation des composants individuels de la fonction, puis sur la droite est une illustration de la façon dont le changement de ces composants affecte la couleur de fin. Syntaxe:make_colour_rgb(red, green, blue); Argument Description red La composante rouge de la couleur green La composante verte de la couleur blue La composante bleue de la couleur Retour:Int Exemple:col = make_colour_rgb(100, 145, 255); Le code ci-dessus utilise la fonction pour créer une couleur et stocker sa valeur dans la variable "col" pour une utilisation ultérieure.
