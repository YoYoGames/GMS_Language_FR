fusionner_couleur Avec cette fonction, vous pouvez prendre deux couleurs et les fusionner pour obtenir une nouvelle couleur. La quantité de chacune des couleurs composantes peut être définie en modifiant l'argument "amount", où une valeur de 0 renverra la première couleur (col1), une valeur de 1 renvoie la deuxième couleur (col2) et une valeur intermédiaire renvoie le mélange correspondant. Par exemple, une valeur de 0,5 mélangera les deux couleurs de manière égale. L'image suivante illustre comment cela fonctionne en fusionnant les couleurs rouge et bleu ensemble : Syntaxe : merge_colour(col1, col2, amount) ; Arguments Description col1 La première couleur à fusionner col2 La deuxième couleur à fusionner montant Quelle quantité de chaque couleur doit être fusionnée. Par exemple, 0 renverra la couleur 1, 1 renverra la couleur 2, et 0.5 renverra une fusion des deux couleurs à parts égales. Les retours : Int Exemple : col = merge_colour(c_lime, c_orange, 0.5) ; Le code ci-dessus utilise la fonction pour créer une couleur en fusionnant le citron vert et l'orange à 50/50, puis stocke sa valeur dans la variable "col" pour une utilisation ultérieure.
