draw_get_colour Cette fonction renvoie la couleur de dessin actuelle qui est utilisée pour dessiner les formes, le texte, les primitives et les modèles 3D non texturés. Elle peut être définie avec la fonction draw_set_colour() fonction. Syntaxe : draw_get_colour() Les retours : Real Exemple : if draw_get_colour() != c_black { draw_set_colour(c_black) ; } Le code ci-dessus vérifie d'abord la couleur de dessin actuelle et, si elle n'est pas définie sur c_black, il la définit sur c_black.
