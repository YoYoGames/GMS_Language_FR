draw_clear Cette fonction peut être utilisée pour effacer tout l'écran (sans mélange alpha) à la couleur donnée, et n'est à utiliser que dans l'événement de dessin d'une instance (elle ne s'affichera pas si elle est utilisée dans un autre événement). Cela peut également être utile pour nettoyer les surfaces lorsqu'elles sont nouvellement créées. Syntaxe:draw_clear(col); Argument Description col La couleur avec laquelle l'écran sera effacé Retour:N/A Exemple:draw_clear(c_blue); Cela effacera l'écran avec la couleur bleue.
