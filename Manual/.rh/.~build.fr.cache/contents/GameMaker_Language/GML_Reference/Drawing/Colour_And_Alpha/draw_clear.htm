dessiner_clair Cette fonction peut être utilisée pour effacer la totalité de l'écran (sans mélange alpha) à la couleur donnée, et est uniquement destinée à être utilisée dans l'événement draw d'une instance (elle ne s'affichera pas si elle est utilisée dans un autre événement). Elle peut également être utile pour effacer les surfaces lorsqu'elles sont nouvellement créées. Syntaxe : draw_clear(col) ; Arguments Description col La couleur avec laquelle l'écran sera effacé Les retours : N/A Exemple : draw_clear(c_blue) ; Cela effacera l'écran avec la couleur bleue.
