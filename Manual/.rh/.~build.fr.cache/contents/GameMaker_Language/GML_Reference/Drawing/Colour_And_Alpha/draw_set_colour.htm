draw_set_colour Avec cette fonction, vous pouvez définir la couleur de dessin de base pour le jeu. Cette valeur affectera tous les autres dessins, le cas échéant, y compris les polices, les formes, les primitives et la 3D. Si l'un de ces éléments est dessiné avec sa propre valeur de couleur modifiée, cette valeur sera ignorée. Syntaxe:draw_set_colour(col); Argument Description col La couleur à définir pour le dessin. Retour:N/A Exemple:draw_set_alpha(0.5); draw_set_colour(c_black); draw_text(x+5, y+5, "LEVEL 1"); draw_set_alpha(1); draw_set_colour(c_white); draw_text(x, y, "LEVEL 1"); Le code ci-dessus dessinera du texte à la position spécifiée avec un effet d'ombre créé par l'alpha et la couleur de dessin modifiés.
