draw_get_alpha Cette fonction renvoie la valeur actuelle de l'alpha du dessin, qui est comprise entre 0 (totalement transparent) et 1 (totalement opaque). L'alpha du dessin affecte la transparence de toutes les fonctions de dessin, et peut être défini avec la fonction draw_set_alpha() fonction. Syntaxe : draw_get_alpha() Les retours : Real Exemple : if draw_get_alpha() != 1 { draw_set_alpha(1) ; } Le code ci-dessus vérifie d'abord l'alpha du dessin actuel et s'il n'est pas égal à 1, il le met à 1.
