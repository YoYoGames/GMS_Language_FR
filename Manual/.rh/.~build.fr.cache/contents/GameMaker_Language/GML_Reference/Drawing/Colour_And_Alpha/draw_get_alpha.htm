draw_get_alpha Cette fonction renvoie la valeur actuelle de l'alpha de tirage, qui sera comprise entre 0 (entièrement transparent) et 1 (entièrement opaque). L'alpha de dessin affecte la transparence de toutes les fonctions de dessin et peut être défini avec draw_set_alpha() fonction. Syntaxe:draw_get_alpha() Retour:Real Exemple:if draw_get_alpha() != 1    {    draw_set_alpha(1);    } Le code ci-dessus vérifie d'abord l'alpha de tirage actuel, puis s'il n'est pas défini sur 1, il le définit sur 1.
