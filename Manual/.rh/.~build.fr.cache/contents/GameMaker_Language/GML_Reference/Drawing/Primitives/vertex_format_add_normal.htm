format_de_verre_add_normal Dites à GameMaker Studio 2 d'accepter les données de normales de surface (nx, ny et nz) dans le cadre du nouveau format de vertex en cours de création. Syntaxe : vertex_format_add_normal() ; Les retours : N/A Exemple : vertex_format_begin() ; vertex_format_add_textcoord() ; vertex_format_add_normal() ; mon_format = vertex_format_end() ; Le code ci-dessus créera un nouveau format de vertex avec seulement les valeurs de texture et de normale de surface, puis stockera l'identifiant du format dans la variable "my_format".
