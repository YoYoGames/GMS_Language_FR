Création de tampon de sommet à partir d'un tampon Comme avec la fonction vertex_create_buffer()cette fonction va créer un nouveau tampon de sommets, mais les données de sommets qu'il stocke sont copiées à partir du tampon normal spécifié comme source. Le tampon créé est un tampon de croissance spécial qui est préformaté avec le format de sommet pour construire des primitives à utiliser avec (par exemple) des shaders. Cette fonction nécessite que vous fournissiez le pointeur d'un tampon régulier précédemment créé, et un format de vertex qui doit être appliqué aux données copiées. Syntaxe : vertex_create_buffer_from_buffer(buffer, format) ; Arguments Description tampon Le tampon à partir duquel créer le tampon des sommets. format Le format de sommet primitif à utiliser. Les retours : Pointeur Exemple : vertex_format_begin() ; vertex_format_add_position_3d() ; vertex_format_add_colour() ; vertex_format_add_textcoord() ; var my_format = vertex_format_end() ; v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat) ; Le code ci-dessus crée un nouveau format de vertex puis crée un nouveau tampon de vertex à partir d'un tampon régulier précédemment créé, en lui appliquant le format de vertex personnalisé.
