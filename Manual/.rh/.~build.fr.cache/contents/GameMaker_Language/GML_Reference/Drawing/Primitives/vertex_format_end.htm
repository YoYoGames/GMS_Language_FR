vertex_format_end Cette fonction doit être appelée après avoir défini tout nouveau format de sommet. Il renvoie le nouveau format "handle" (index) qui doit être utilisé dans toutes les autres fonctions de sommet faisant référence à ce nouveau format. Syntaxe:vertex_format_end(); Retour:Real Exemple:vertex_format_begin(); vertex_format_add_colour(); vertex_format_add_position(); my_format = vertex_format_end(); Le code ci-dessus créera un nouveau format de sommet avec juste des valeurs de couleur et de position, puis stockera l'identifiant du format dans la variable "my_format".
