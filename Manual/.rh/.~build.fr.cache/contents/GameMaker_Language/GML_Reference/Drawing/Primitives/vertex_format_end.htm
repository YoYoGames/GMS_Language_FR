Fin du formatage des sommets Cette fonction doit être appelée après avoir défini un nouveau format de sommet. Elle renvoie le "handle" (index) du nouveau format qui doit être utilisé dans toutes les autres fonctions de sommet qui font référence à ce nouveau format. Syntaxe : vertex_format_end() ; Les retours : Real Exemple : vertex_format_begin() ; vertex_format_add_colour() ; vertex_format_add_position() ; mon_format = vertex_format_end() ; Le code ci-dessus créera un nouveau format de sommet avec seulement des valeurs de couleur et de position et stockera ensuite l'identifiant du format dans la variable "my_format".
