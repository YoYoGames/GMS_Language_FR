vertex_get_number Avec cette fonction, vous pouvez connaître le nombre de sommets individuels définis dans un tampon de sommets donné. Syntaxe:vertex_get_number(buffer); Argument Description buffer Le tampon de vertex à vérifier. Retour:Real Exemple:v_num = vertex_get_number(v_buffer); Le code ci-dessus stockera le nombre de sommets stockés dans le tampon de sommets donné dans une variable.
