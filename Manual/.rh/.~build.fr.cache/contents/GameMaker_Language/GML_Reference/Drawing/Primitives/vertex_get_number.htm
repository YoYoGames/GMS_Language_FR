vertex_get_number Cette fonction permet de connaître le nombre de sommets individuels définis dans un tampon de sommets donné. Syntaxe : vertex_get_number(buffer) ; Arguments Description tampon Le tampon de sommets à vérifier. Les retours : Real Exemple : v_num = vertex_get_number(v_buffer) ; Le code ci-dessus stockera le nombre de sommets stockés dans le tampon de sommets donné dans une variable.
