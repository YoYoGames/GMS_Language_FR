vertex_position Cette fonction ajoutera des données de position 2D au sommet actuellement défini pour la primitive personnalisée. Vous fournissez le tampon dans lequel écrire les données ainsi que la position x et y pour le dessin. Syntaxe : vertex_position(buffer, x, y) ; Arguments Description tampon Le tampon dans lequel écrire l'information. x La position x. y La position y. Les retours : N/A Exemple : vertex_position(buff, x - 100, y - 125) ; Le code ci-dessus définira la position du sommet en cours de définition.
