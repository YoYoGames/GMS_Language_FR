vertex_position Cette fonction ajoutera des données de position 2D au sommet en cours de définition pour la primitive personnalisée. Vous fournissez le tampon dans lequel écrire les données ainsi que les positions x et y pour le dessin. Syntaxe:vertex_position(buffer, x, y); Argument Description buffer Le tampon dans lequel écrire les informations. x La position x. y La position y. Retour:N/A Exemple:vertex_position(buff, x - 100, y - 125); Le code ci-dessus définira la position du sommet en cours de définition.
