vertex_float3 Cette fonction ajoutera trois valeurs en virgule flottante aux données de sommet. Le sommet doit avoir été formaté correctement pour accepter cela en utilisant le vertex_format_add_custom() fonction. Syntaxe:vertex_float3(buffer, float, float, float); Argument Description buffer Le tampon dans lequel écrire les informations. float La première valeur d'entrée. float La deuxième valeur d'entrée. float La troisième valeur d'entrée. Retour:N/A Exemple:vertex_float3(buff, 0.05, 0.01, room_width / x); Le code ci-dessus ajoutera trois valeurs en virgule flottante aux données de sommet en cours de définition.
