vertex_delete_buffer Cette fonction peut être utilisée pour supprimer un tampon de sommets précédemment créé (voir vertex_create_buffer() ) de la mémoire système. Cela supprimera également toutes les données de sommet pour toutes les primitives personnalisées qu'elle contenait. Syntaxe:vertex_delete_buffer(buffer); Argument Description buffer Le tampon à supprimer. Retour:N/A Exemple:vertex_delete_buffer(v_buff); Le code ci-dessus supprimera de la mémoire le tampon indexé dans la variable "v_buff".
