tampon_suppression_dex Cette fonction peut être utilisée pour supprimer de la mémoire système un tampon de sommets précédemment créé (voir vertex_create_buffer()) de la mémoire système. Cela supprimera également toutes les données de sommets pour toutes les primitives personnalisées qu'il contenait. Syntaxe : vertex_delete_buffer(buffer) ; Arguments Description tampon Le tampon à supprimer. Les retours : N/A Exemple : vertex_delete_buffer(v_buff) ; Le code ci-dessus va supprimer de la mémoire le tampon indexé dans la variable "v_buff".
