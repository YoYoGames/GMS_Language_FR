draw_vertex_texture_colour Cette fonction définit la position d'un sommet texturé pour une primitive. L'aspect final de la primitive dépendra du type primitif choisi pour dessiner (voir draw_primitive_begin() pour plus d'informations), l'ordre dans lequel vous ajoutez les sommets, la position des points de début et de fin que vous donnez pour l'échantillon de texture et les valeurs de couleur et alpha que vous avez définies. Pour conserver l'apparence de la texture tout en ne modifiant que l'alpha, une valeur de -1 (ou c_white ) peut être utilisé pour l'argument couleur. Pour terminer et dessiner la primitive, vous devez appeler draw_primitive_end(). NOTE: Pour qu'une texture se répète, elle doit avoir une puissance de deux, c'est-à-dire: 32x32, 128x128, etc... Syntaxe:draw_vertex_texture_colour(x, y, xtex, ytex, col, alpha) Argument Description x La coordonnée x du sommet. y La coordonnée y du sommet. xtex La coordonnée x dans la texture. ytex La coordonnée y dans la texture. col La couleur à mélanger avec la texture à ce sommet (-1 ou c_white pour aucun mélange). alpha L'alpha pour dessiner ce sommet avec (0-1). Retour:N/A Exemple:draw_set_colour(c_white); var tex = sprite_get_texture(spr_Background, 0); draw_primitive_begin_texture(pr_trianglestrip, tex); draw_vertex_texture_colour(0, 0, 0, 0, c_fuchsia, 1); draw_vertex_texture_colour(640, 0, 1, 0, c_yellow, 1); draw_vertex_texture_colour(0, 480, 0, 1, c_aqua, 1); draw_vertex_texture_colour(640, 480, 1, 1, c_lime, 1); draw_primitive_end(); Le code ci-dessus dessinera une bande de triangle à 4 sommets (faisant un rectangle) texturée avec la texture contenue dans la variable "tex", et la texture entière sera utilisée pour couvrir la primitive terminée, et elle sera mélangée avec quatre couleurs différentes.
