vertex_position_3d Cette fonction ajoutera des données de position 3D au sommet actuellement défini pour la primitive personnalisée. Vous fournissez le tampon dans lequel écrire les données ainsi que la position x, y et z pour le dessin. Syntaxe : vertex_position_3d(buffer, x, y, z) ; Arguments Description tampon Le tampon dans lequel écrire l'information. x La position x. y La position y. z La position z. Les retours : N/A Exemple : vertex_position_3d(buff, x - 100, y - 125, 0) ; Le code ci-dessus définira la position du sommet en cours de définition.
