taille_du_buffer_des_verrex Cette fonction permet d'obtenir la taille du tampon de sommets en octets. Syntaxe : vertex_get_buffer_size(buffer) ; Arguments Description tampon Le tampon dont on veut obtenir la taille. Les retours : Real Exemple : bufferBytes = vertex_get_buffer_size(buff) ; Le code ci-dessus obtiendra le nombre d'octets utilisés par le tampon de sommets donné et stockera la valeur dans la variable "bufferBytes".
