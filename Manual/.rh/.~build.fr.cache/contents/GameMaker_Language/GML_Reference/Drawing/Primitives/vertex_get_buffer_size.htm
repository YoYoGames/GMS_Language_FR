vertex_get_buffer_size Avec cette fonction, vous pouvez obtenir la taille du tampon de vertex donné en octets. Syntaxe:vertex_get_buffer_size(buffer); Argument Description buffer Le tampon pour obtenir la taille de. Retour:Real Exemple:bufferBytes = vertex_get_buffer_size(buff); Le code ci-dessus obtiendra le nombre d'octets utilisés par le tampon de vertex donné et stockera la valeur dans la variable "bufferBytes".
