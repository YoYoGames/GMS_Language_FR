vertex_colour Si votre format de sommet défini prend une valeur de couleur, vous pouvez utiliser cette fonction pour ajouter ces données au sommet défini pour la primitive actuelle. La fonction a besoin d'un tampon pour stocker les données et prendra soit une constante de couleur, ou une valeur hexadécimale (en utilisant le format standard BGR de GameMaker Studio 2, par exemple : $FF0000 pour le bleu) ainsi qu'une valeur alpha comprise entre 0 (transparent) et 1 (totalement opaque). Syntaxe : vertex_colour(buffer, colour, alpha) ; Arguments Description tampon Le tampon dans lequel écrire l'information. couleur La couleur de ce sommet (peut être une constante ou une valeur hexadécimale). alpha La valeur alpha du sommet (de 0 à 1). Les retours : N/A Exemple : vertex_colour(b, c_white, 1) ; Le code ci-dessus définira le sommet défini comme étant blanc avec une valeur alpha de 1.
