vertex_ubyte4 Cette fonction ajoutera quatre valeurs d'octet non signées (0 - 255) aux données du sommet. Le sommet doit avoir été formaté correctement pour accepter cette fonction. vertex_format_add_custom() fonction. Syntaxe : vertex_ubyte4(buffer, byte, byte, byte, byte) ; Arguments Description tampon Le tampon dans lequel écrire l'information. octet La première valeur d'entrée. octet La deuxième valeur d'entrée. octet La troisième valeur d'entrée. octet La quatrième valeur d'entrée. Les retours : N/A Exemple : vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), irandom(255), 127) ; Le code ci-dessus ajoutera quatre valeurs aux données de vertex en cours de définition.
