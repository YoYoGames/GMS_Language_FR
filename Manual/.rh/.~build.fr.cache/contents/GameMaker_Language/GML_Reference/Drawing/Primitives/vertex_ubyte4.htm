vertex_ubyte4 Cette fonction ajoutera quatre valeurs d'octet non signées (0 à 255) aux données de sommet. Le sommet doit avoir été formaté correctement pour accepter cela en utilisant le vertex_format_add_custom() fonction. Syntaxe:vertex_ubyte4(buffer, byte, byte, byte, byte); Argument Description buffer Le tampon dans lequel écrire les informations. byte La première valeur d'entrée. byte La deuxième valeur d'entrée. byte La troisième valeur d'entrée. byte La quatrième valeur d'entrée. Retour:N/A Exemple:vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127); Le code ci-dessus ajoutera quatre valeurs aux données de sommet en cours de définition.
