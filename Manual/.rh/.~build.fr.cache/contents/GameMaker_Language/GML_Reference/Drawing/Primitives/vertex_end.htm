vertex_end Avec cette fonction, vous terminez la construction de la primitive personnalisée. Une fois que vous appelez cette commande, la primitive peut être utilisée dans le vertex_submit() à utiliser dans un shader ou vous pouvez geler le tampon (rendant le tampon de vertex utilisé en lecture seule et beaucoup plus rapide) en utilisant vertex_freeze(). Syntaxe:vertex_end(buffer); Argument Description buffer Le tampon dans lequel terminer l'écriture. Retour:N/A Exemple:vertex_end(buff); Le code ci-dessus mettra fin à la définition d'une primitive personnalisée.
