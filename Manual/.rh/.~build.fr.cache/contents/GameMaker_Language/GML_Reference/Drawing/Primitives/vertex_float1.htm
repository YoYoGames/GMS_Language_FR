vertex_float1 Cette fonction ajoutera une valeur à virgule flottante aux données du sommet. Le vertex doit avoir été formaté correctement pour accepter cette fonction. vertex_format_add_custom() fonction. Syntaxe : vertex_float1(buffer, float) ; Arguments Description tampon Le tampon dans lequel écrire l'information. float La valeur d'entrée. Les retours : N/A Exemple : vertex_float1(buff, 0.05) ; Le code ci-dessus ajoutera une valeur à virgule flottante aux données de sommet en cours de définition.
