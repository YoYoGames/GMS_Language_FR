vertex_float1 Cette fonction ajoutera une valeur en virgule flottante aux données de sommet. Le sommet doit avoir été formaté correctement pour accepter cela en utilisant le vertex_format_add_custom() fonction. Syntaxe:vertex_float1(buffer, float); Argument Description buffer Le tampon dans lequel écrire les informations. float La valeur d'entrée. Retour:N/A Exemple:vertex_float1(buff, 0.05); Le code ci-dessus ajoutera une valeur en virgule flottante aux données de sommet en cours de définition.
