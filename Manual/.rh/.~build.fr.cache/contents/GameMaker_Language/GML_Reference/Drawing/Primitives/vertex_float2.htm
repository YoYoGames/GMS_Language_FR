vertex_float2 Cette fonction ajoute deux valeurs à virgule flottante aux données du sommet. Le vertex doit avoir été formaté correctement pour accepter ceci en utilisant la fonction vertex_format_add_custom() fonction. Syntaxe : vertex_float2(buffer, float, float) ; Arguments Description tampon Le tampon dans lequel écrire l'information. float La première valeur d'entrée. float La deuxième valeur d'entrée. Les retours : N/A Exemple : vertex_float2(buff, 0.05, 0.01) ; Le code ci-dessus ajoutera deux valeurs à virgule flottante aux données de sommet en cours de définition.
