Primitives et formats de sommets Une primitive est un type de dessin réalisé à l'aide de points pour définir ce qui est affiché à l'écran. Ils peuvent être constitués de points uniques, de lignes ou de triangles et peuvent être colorés, avoir un mélange alpha et même recevoir une texture, ce qui signifie qu'ils peuvent être utilisés pour créer des effets très excitants dans un jeu 2D! Par exemple, vous pouvez créer un rectangle en utilisant plusieurs primitives de triangle, puis le texturer avec un sprite drapeau et dans l'événement de dessin d'une instance, avoir ce drapeau "vague" dans le vent et déformer le sprite que vous avez utilisé en modifiant simplement les points qui forment le primitif. ATTENTION! Ces fonctions ne fonctionnent pas avec le module HTML5 sauf si vous avez activé OpenGL dans les options de jeu. Les primitives que vous créez doivent suivre le format que vous avez défini à l'aide des fonctions de format de sommet, donc si vous avez défini un format de sommet avec uniquement des données de position, il est inutile de construire votre primitive avec des données de couleur. Vous devez noter que l'ordre dans lequel vous ajoutez des propriétés à la primitive que vous construisez est défini par l'ordre dans lequel vous avez ajouté ces propriétés lors de la création du format de sommet, donc si vous avez défini le format vectoriel avec la position, la couleur et la texture de l'ordre coordonnée, vous devez ajouter ces propriétés à la primitive en cours de construction dans le même ordre, sinon vous obtiendrez des erreurs.
