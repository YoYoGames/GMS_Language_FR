Primitives et formats de sommets Une primitive est un type de dessin réalisé à l'aide de points pour définir ce qui est affiché à l'écran. Elles peuvent être composées de points, de lignes ou de triangles et peuvent être colorées, avoir un mélange alpha et même une texture, ce qui signifie qu'elles peuvent être utilisées pour créer des effets très intéressants dans un jeu 2D ! pour créer des effets très intéressants dans un jeu en 2D ! Par exemple, vous pouvez créer un rectangle à l'aide de plusieurs primitives triangulaires, puis le texturer avec un sprite de drapeau et, dans l'événement de dessin d'une instance, faire "onduler" ce drapeau dans le vent et déformer le sprite que vous avez utilisé en modifiant simplement les points qui forment la primitive. ATTENTION ! Ces fonctions ne fonctionnent pas avec le module HTML5 à moins que vous n'ayez activé OpenGL dans les options du jeu.
