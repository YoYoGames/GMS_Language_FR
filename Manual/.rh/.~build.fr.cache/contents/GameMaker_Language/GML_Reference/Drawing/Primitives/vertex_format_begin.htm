vertex_format_begin Avant de pouvoir définir votre nouveau format de vertex, vous devez indiquer à GameMaker Studio 2 que vous le faites à l'aide de cette fonction. Vous devez d'abord l'appeler, puis définir les valeurs de format à l'aide des fonctions appropriées, et enfin appeler vertex_format_end() pour terminer la définition et retourner le nouveau format "handle". Syntaxe:vertex_format_begin(); Retour:N/A Exemple:vertex_format_begin(); vertex_format_add_colour(); vertex_format_add_position(); my_format = vertex_format_end(); Le code ci-dessus créera un nouveau format de sommet avec juste des valeurs de couleur et de position, puis stockera l'identifiant du format dans la variable "my_format".
