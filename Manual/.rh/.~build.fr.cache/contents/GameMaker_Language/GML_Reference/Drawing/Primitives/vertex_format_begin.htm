format_de_verre_début Avant de pouvoir définir votre nouveau format de vertex, vous devez indiquer à GameMaker Studio 2 que vous le faites à l'aide de cette fonction. Vous devez l'appeler en premier, puis définir les valeurs du format à l'aide des fonctions appropriées, et enfin appeler vertex_format_end() pour terminer la définition et renvoyer le "handle" du nouveau format. Syntaxe : vertex_format_begin() ; Les retours : N/A Exemple : vertex_format_begin() ; vertex_format_add_colour() ; vertex_format_add_position() ; mon_format = vertex_format_end() ; Le code ci-dessus créera un nouveau format de sommet avec seulement des valeurs de couleur et de position et stockera ensuite l'identifiant du format dans la variable "my_format".
