vertex_normal Cette fonction ajoutera des données normales de surface au sommet en cours de définition pour la primitive personnalisée. Vous fournissez le tampon dans lequel écrire les données ainsi que les composants x, y et z de la normale. Syntaxe:vertex_normal(buffer, nx, ny, nz); Argument Description buffer Le tampon dans lequel écrire les informations. nx La composante x de la normale à la surface. ny La composante y de la normale à la surface. nz La composante z de la normale à la surface. Retour:N/A Exemple:vertex_normal(buff, 0, 1, 1); Le code ci-dessus définira la normale à la surface du sommet en cours de définition.
