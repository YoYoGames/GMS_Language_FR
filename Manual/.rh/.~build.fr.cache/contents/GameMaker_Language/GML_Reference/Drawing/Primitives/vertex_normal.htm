vertex_normal Cette fonction ajoutera des données de normales de surface au sommet actuellement défini pour la primitive personnalisée. Vous fournissez le tampon dans lequel écrire les données ainsi que les composantes x, y et z de la normale. Syntaxe : vertex_normal(buffer, nx, ny, nz) ; Arguments Description tampon Le tampon dans lequel écrire l'information. nx La composante x de la normale à la surface. ny La composante y de la normale à la surface. nz La composante z de la normale à la surface. Les retours : N/A Exemple : vertex_normal(buff, 0, 1, 1) ; Le code ci-dessus définit la normale à la surface du sommet en cours de définition.
