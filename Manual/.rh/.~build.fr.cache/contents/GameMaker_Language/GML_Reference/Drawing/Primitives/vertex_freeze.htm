vertex_freeze Cette fonction peut être utilisée pour "geler" un tampon de sommets. ce tampon devient en lecture seule, ce qui signifie que si vous avez besoin de le changer, vous devrez supprimer tout le tampon et le recréer. Un tampon gelé peut être soumis au shader plus rapidement qu'un tampon dynamique normal et est recommandé pour les effets qui nécessitent une primitive personnalisée inchangée pour la durée d'un niveau ou du jeu. Syntaxe:vertex_freeze(buffer); Argument Description buffer Le tampon à geler. Retour:N/A Exemple:vertex_freeze(buff); Le code ci-dessus gèlera le tampon de vertex donné.
