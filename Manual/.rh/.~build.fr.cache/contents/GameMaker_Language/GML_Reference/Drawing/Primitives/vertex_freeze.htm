vertex_freeze Cette fonction peut être utilisée pour "geler" un tampon de sommets. Ce tampon devient en lecture seule, ce qui signifie que si vous devez le modifier, vous devrez supprimer tout le tampon et le recréer. Un tampon gelé peut être soumis au shaderplus rapidement qu'un tampon normal et dynamique. plus rapidement qu'un tampon normal et dynamique et est recommandé pour les effets qui nécessitent une primitive personnalisée immuable pendant la durée d'un niveau ou du jeu. Syntaxe : vertex_freeze(buffer) ; Arguments Description tampon Le tampon à congeler. Les retours : N/A Exemple : vertex_freeze(buff) ; Le code ci-dessus va geler le tampon de sommets donné.
