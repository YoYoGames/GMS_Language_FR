draw_primitive_begin_texture Cette fonction doit être appelée avant de définir les sommets d'une primitive texturée. Vous devez donner le type de primitif à utiliser (voir draw_primitive_begin() pour plus d'informations) et l' identifiant d'une texture à utiliser, qui peut être un sprite ou une image d'arrière-plan. Cet identifiant d' actif peut être obtenu à partir des fonctions sprite_get_texture(), par exemple (utilisez -1 pour aucune texture). NOTE: Pour qu'une texture se répète, elle doit avoir une puissance de deux, c'est-à-dire: 32x32, 128x128, etc... Syntaxe:draw_primitive_begin_texture(kind, tex) Argument Description kind Le genre de primitif que vous allez dessiner. tex La texture à utiliser avec le primitif. Retour:N/A Exemple:draw_set_colour(c_white); var tex = sprite_get_texture(spr_Background, 0); draw_primitive_begin_texture(pr_trianglestrip, tex); draw_vertex_texture(0, 0, 0, 0); draw_vertex_texture(640, 0, 1, 0); draw_vertex_texture(0, 480, 0, 1); draw_vertex_texture(640, 480, 1, 1); draw_primitive_end(); Le code ci-dessus dessinera une bande de triangle de 4 sommets (faisant un rectangle) texturée avec la texture contenue dans la variable "tex", et la texture entière sera utilisée pour couvrir la primitive terminée.
