draw_primitive_begin_texture Cette fonction doit être appelée avant de définir les sommets d'une primitive texturée. Vous devez donner le type de primitive à utiliser (voir draw_primitive_begin() pour plus d'informations) et l'identifiant d'une texture à utiliser, qui peut être un sprite ou une image de fond. Cet identifiant d' actif peut être obtenu à partir des fonctions sprite_get_texture()par exemple (utilisez -1 pour ne pas utiliser de texture). NOTE : Pour qu'une texture soit répétée, sa taille doit être une puissance de deux, c'est-à-dire : 32x32, 128x128, etc... Syntaxe : draw_primitive_begin_texture(kind, tex) Arguments Description genre Le type de primitif que vous allez dessiner. tex La texture à utiliser avec la primitive. Les retours : N/A Exemple : draw_set_colour(c_white) ; var tex = sprite_get_texture(spr_Background, 0) ; draw_primitive_begin_texture(pr_trianglestrip, tex) ; draw_vertex_texture(0, 0, 0, 0) ; draw_vertex_texture(640, 0, 1, 0) ; draw_vertex_texture(0, 480, 0, 1) ; draw_vertex_texture(640, 480, 1, 1) ; draw_primitive_end() ; Le code ci-dessus va dessiner une bande de triangle à 4 sommets (formant un rectangle) texturée avec la texture contenue dans la variable "tex", et la texture entière sera utilisée pour couvrir la primitive terminée.
