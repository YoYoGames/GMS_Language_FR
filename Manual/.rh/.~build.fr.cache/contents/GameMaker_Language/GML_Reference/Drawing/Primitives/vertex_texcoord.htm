vertex_texcoord Cette fonction définira les coordonnées de texture à utiliser pour le sommet en cours de définition pour la primitive personnalisée. Vous fournissez le tampon dans lequel écrire les données ainsi que la position UV dans la texture à utiliser. Syntaxe:vertex_texcoord(buffer, u, v); Argument Description buffer Le tampon dans lequel écrire les informations. u La première coordonnée de texture à utiliser (0 - 1). v Deuxième coordonnée de texture à utiliser (0 - 1). Retour:N/A Exemple:vertex_texcoord(buff, 0, 0); Le code ci-dessus définira les valeurs UV du sommet en cours de définition.
