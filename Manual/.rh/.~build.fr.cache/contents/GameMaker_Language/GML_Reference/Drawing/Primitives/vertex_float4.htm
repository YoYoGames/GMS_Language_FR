vertex_float4 Cette fonction ajoute quatre valeurs à virgule flottante aux données du sommet. Le vertex doit avoir été formaté correctement pour accepter cette fonction. vertex_format_add_custom() fonction. Syntaxe : vertex_float4(buffer, float, float, float, float, float) ; Arguments Description tampon Le tampon dans lequel écrire l'information. float La première valeur d'entrée. float La deuxième valeur d'entrée. float La troisième valeur d'entrée. float La quatrième valeur d'entrée. Les retours : N/A Exemple : vertex_float4(buff, 0.05, 0.01, room_width / x, room_height / y) ; Le code ci-dessus ajoutera quatre valeurs à virgule flottante aux données de sommet en cours de définition.
