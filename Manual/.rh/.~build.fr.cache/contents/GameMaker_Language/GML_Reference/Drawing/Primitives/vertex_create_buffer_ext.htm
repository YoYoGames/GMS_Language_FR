création_de_tampon_de_vertueux Comme avec la fonction vertex_create_buffer()cette fonction va créer un nouveau tampon de sommets. Il s'agit d'un tampon de croissance spécial créé par GameMaker Studio 2, préformaté pour être utilisé lors de la construction de primitives à utiliser avec les shaders. construction de primitives à utiliser avec les shaders. Vous pouvez spécifier une taille initiale de départ pour le tampon (en octets) et cette fonction renverra un "handle" (index) pour le tampon qui devra être utilisé dans tous les appels ultérieurs au tampon. Syntaxe : vertex_create_buffer_ext(size) ; Arguments Description taille La taille initiale du tampon (en octets). Les retours : Pointeur Exemple : v_buff = vertex_create_buffer_ext(1024 * 1024) ; Le code ci-dessus crée un nouveau tampon de sommets, d'une taille initiale de 1 Mo, et stocke son identifiant dans la variable "v_buff".
