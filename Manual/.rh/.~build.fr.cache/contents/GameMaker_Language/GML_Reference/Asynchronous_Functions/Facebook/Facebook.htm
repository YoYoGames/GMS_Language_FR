Facebook Facebook est l'un des plus importants sites de médias sociaux sur Internet et de nombreux jeux et développeurs ont actuellement leurs propres pages pour publier des jeux et des médias. GameMaker Studio 2 possède une extension qui vous permet de vous connecter à Facebook et d'effectuer certaines actions et tâches que leur API permet. Il est recommandé de visiter la page des développeurs Facebook avant de commencer à utiliser l'extension, afin de bien comprendre le fonctionnement du graphique Facebook et les appels et tâches disponibles. Facebook Graph et des appels et autorisations disponibles est essentielle pour que vous puissiez tirer le meilleur parti de ces fonctions. Vous pouvez découvrir comment configurer et utiliser l'extension Facebook pour toutes les plateformes disponibles à partir du lien suivant : Android, iOS et HTML5 : Intégration de Facebook Les fonctions suivantes sont obsolètes et n'étaient valables que pour l'ancienne API Facebook. En tant que telles, vous ne pouvez plus utiliser ces fonctions dans vos projets. Elles sont répertoriées ici au cas où vous auriez importé un ancien projet les contenant. Pour la fonctionnalité Facebook actuelle, vous devez avoir activé l'API Facebook dans les options de jeu appropriées pour la plate-forme cible, ce qui vous invitera à télécharger et à installer une extension. L'extension est accompagnée d'un "mini-manuel" au format *.pdf expliquant comment l'utiliser et montrant toutes les définitions de fonctions et les exemples. Toutes les références aux anciennes fonctions facebook_*() devront être remplacées (et mises à jour) pour utiliser l'extension fb. mises à jour) pour utiliser les équivalents de l'extension fb_*().
