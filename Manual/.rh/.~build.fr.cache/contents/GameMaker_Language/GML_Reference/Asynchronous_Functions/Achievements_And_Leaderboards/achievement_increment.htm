achievement_increment Les jeux peuvent avoir des succès sans valeur d'achèvement et vous pouvez donc utiliser cette fonction pour incrémenter ces succès d'un montant donné. Sur Android, vous envoyez l'ID de la réussite sous forme de chaîne (il s'agit de l'ID de réussite unique qui a été attribué lorsque vous configurez la réussite), tandis que sur toutes les autres plates-formes, vous indiquez le nom de la réussite défini sous forme de chaîne, puis la valeur de réalisation réelle à incrémenter. La fonction déclenchera un événement asynchrone social où le retour async_load DS map aura les paires clé / valeur suivantes: " type " - C'est le type d'événement qui a été déclenché, qui sera la chaîne "achievement_increment_result "pour cette fonction. " id " - Il s'agit de l'ID de l'événement et, pour cette fonction, il renverra une constante GML GooglePlayServices_EVENT_ID_INCREMENT_ACHIEVEMENT. " achievement_id " - Il s'agit de l'ID unique du succès qui a été incrémenté. " status " - Ce sera 1 si la réussite a été incrémentée avec succès, ou 0 si la demande a échoué " complete " - Ce sera 1 si la réussite a été achevée par cet incrément, ou 0 si la demande a échoué Syntaxe:achievement_increment(name, value) Argument Description name Le nom / identifiant de la réalisation. value La valeur à envoyer sous forme d'incrément. Retour:N/A Exemple:achievement_increment(global.Achievement[0], score); Le code ci-dessus ajoute le score actuel du joueur à la réussite avec l'ID stocké dans le tableau global.
