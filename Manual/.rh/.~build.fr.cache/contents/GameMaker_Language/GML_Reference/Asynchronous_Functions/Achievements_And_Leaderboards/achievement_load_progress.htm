progression de la charge de travail Cette fonction enverra une requête au serveur pour obtenir des informations sur toutes les réalisations disponibles. Elle déclenchera un callback Social Asynchronous Event qui contient le site async_load map remplie avec les paires clé/valeur pertinentes. La clé id de cette carte DS est utilisée pour identifier le callback correct (il peut y avoir plus d'une fonction de déclenchement pour un événement social asynchrone). (il peut y avoir plus d'une fonction de déclenchement pour un événement asynchrone donné), et sera associée à la constante achievement_achievement_info ainsi qu'à un certain nombre d'autres paires clé/valeur pour chaque joueur. Le contenu exact de la carte est présenté ci-dessous : REMARQUE : Cette fonction est destinée à iOS seulement. "id" - Pour cette fonction, ce devrait être achievement_achievement_info. " numentries" - Le nombre de réalisations dans la liste. "AchN"- Le nom de la réalisation, où "N" est une valeur entière correspondant à sa place dans la liste des entrées. "AchCompN" - Le degré d'achèvement de la réalisation "N" en tant que valeur de pourcentage de 0 à 100 (une chaîne). Syntaxe : achievement_load_progress() Les retours : N/A Exemple étendu : Le code suivant demanderait la progression de la réalisation lorsque l'utilisateur se trouve sur un appareil iOS: if os_type == os_ios { achievement_load_progress() ; } Cela enverra une demande d'informations sur les réalisations et générera un callback asynchrone avec la carte DS spéciale async_load contenant les données suivantes : var ident = ds_map_find_value(async_load, "id") ; if ident == achievement_achievement_info { var numentries = ds_map_find_value(async_load, "numentries") ; for(var i = 0 ; i < numentries ; i++ ;) { ach_id[i] = ds_map_find_value(async_load, "Ach" + string(i)) ; comp[i] = ds_map_find_value(async_load, "AchComp" + string(i)) ; } } Le code ci-dessus vérifie la carte DS retournée dans l'événement social asynchrone et si son "id" correspond à la constante vérifiée, il parcourt ensuite les entrées de la carte pour chacune des réalisations et enregistre leur nom et le pourcentage de réussite.
