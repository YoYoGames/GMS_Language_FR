achievement_available Cette fonction retournera true si l'utilisateur est actuellement connecté à Internet et que le classement et le système de réussite choisis sont disponibles, sinon il reviendra false. Syntaxe:achievement_available() Retour:Boolean Exemple:if achievement_available()     {     achievement_post("ach_Level1", 100);     } Le code ci-dessus vérifiera que le service de classement est disponible et si tel est le cas, publiez-y.
