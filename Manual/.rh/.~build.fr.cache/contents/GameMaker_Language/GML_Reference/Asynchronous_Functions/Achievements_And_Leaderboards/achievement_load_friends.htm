achievement_load_friends Cette fonction enverra une demande au serveur pour obtenir des informations sur tous les amis des utilisateurs connectés et déclenchera un événement social asynchrone de rappel qui contient le async_load carte remplie avec les paires clé / valeur pertinentes. La clé d' identification de ce DS Map est utilisé pour identifier le rappel correct (il peut y avoir plus d'une fonction de déclenchement pour un événement asynchrone donné), et sera associé à la constante achievement_friends_info ainsi qu'un certain nombre d'autres paires clé / valeur pour chaque ami. Le contenu exact de la carte est indiqué ci-dessous: REMARQUE: cette fonction est pour iOS uniquement. " id " - Pour cette fonction, il devrait être achievement_friends_info " FriendN " - Le nom de l'ami, où "N" est une valeur entière correspondant à sa position dans la liste d'amis. " FriendidN " - L'ID utilisateur unique de l'ami, "N". Syntaxe:achievement_load_friends() Retour:N/A Exemple étendu: Le code suivant sera probablement appelé après que le joueur se soit connecté à son compte de jeu en utilisant achievement_login() pour obtenir une liste de tous les amis des utilisateurs:achievement_friends_info(); Cela enverra une demande d'informations sur les amis des utilisateurs et générera un rappel asynchrone avec le spécial async_load Carte DS contenant les données suivantes:var ident = ds_map_find_value(async_load, "id"); if ident == achievement_friends_info     {     var numfriends = ds_map_find_value(async_load, "numfriends");     global.numfriends = numfriends;     for(var i=0; i < numfriends; i++;)         {         global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));         global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));         achievement_get_pic(global.friendid[i]);         }     } Le code ci-dessus vérifie le retour DS Map dans l' événement social asynchrone et si son "id" correspond à la constante vérifiée, il parcourt ensuite la carte en stockant toutes les différentes valeurs dans un certain nombre de tableaux et demande une image pour chacune des entrées de la carte (voir achievement_get_pic() ).
