achievement_reset Cette fonction réinitialisera toutes les réalisations à leurs valeurs initiales pour le jeu. Cette fonction est fournie en tant que fonction de débogage et il n'est pas recommandé d'autoriser l'utilisateur final à le faire dans vos jeux. Syntaxe:achievement_reset() Retour:N/A Exemple:if debug_mode     {     if keyboard_check_pressed(vk_space)         {         achievement_reset()         }     } Le code ci-dessus vérifie que le jeu est en mode débogage, et s'il appuie sur «espace», il réinitialisera toutes les réalisations.
