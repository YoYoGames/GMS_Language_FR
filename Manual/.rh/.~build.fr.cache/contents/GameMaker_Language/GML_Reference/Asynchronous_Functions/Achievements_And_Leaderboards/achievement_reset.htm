remise à zéro Cette fonction réinitialise tous les succès à leur valeur initiale pour le jeu. Cette fonction est fournie en tant que fonction de débogage et il n'est pas recommandé de permettre à l'utilisateur final de faire cela dans vos jeux. Syntaxe : achievement_reset() Les retours : N/A Exemple : if debug_mode { if keyboard_check_pressed(vk_space) { achievement_reset() } } Le code ci-dessus vérifie que le jeu est en mode débogage, et si c'est le cas, appuyer sur "espace" réinitialisera toutes les réalisations.
