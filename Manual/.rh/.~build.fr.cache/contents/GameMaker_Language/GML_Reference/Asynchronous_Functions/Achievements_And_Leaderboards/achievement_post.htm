achievement_post Vous pouvez utiliser cette fonction pour envoyer vos réalisations au service de classement et de réalisation choisi. Sur Android, vous aurez besoin de l'ID de la réalisation sous forme de chaîne (il s'agit de l'ID unique de la réalisation qui a été attribué lorsque vous avez configuré la réalisation), Sur toutes les autres plateformes, vous devez fournir le nom de la réalisation sous forme de chaîne, puis indiquer le pourcentage que vous avez atteint pour obtenir la réalisation (0 - aucun, 100 - terminé). La fonction déclenchera un événement social asynchrone dans lequel la carte DS renvoyée aura les clés/éléments suivants async_load retournée contiendra les paires clé/valeur suivantes : "Type"- Il s'agit du type d'événement qui a été déclenché, qui sera la chaîne "achievement_post_result" pour cette fonction. "id" - Il s'agit de l'ID de l'événement et, pour cette fonction, elle renverra une constante GML GooglePlayServices_EVENT_ID_POST_ACHIEVEMENT . "achievement_id" - Il s'agit de l'ID unique de la réalisation qui a été mise à jour. "status" - Ce sera 1 si la réalisation a été mise à jour avec succès, ou 0 si la demande a échoué. Syntaxe : achievement_post(achievement, pourcentage) Arguments Description réalisation Le nom/ID de la réalisation. pour cent Le pourcentage de la réalisation effectuée. Les retours : N/A Exemple : if achievement_available() { achievement_post(global.Achievement[0], 100) ; } Le code ci-dessus vérifie si le service de classement et de réussite choisi est disponible et, si c'est le cas, il affiche une réussite.
