achievement_logout Cette fonction déconnecte l'utilisateur du service de classement et de réalisation choisi. Cela empêchera l'enregistrement de tous les autres succès et scores. Syntaxe : achievement_logout() Les retours : N/A Exemple : achievement_logout() ; Le code ci-dessus vous déconnectera du service de classement et d'accomplissement choisi.
