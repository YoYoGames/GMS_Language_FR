achievement_show_leaderboards Cette fonction ouvrira la page des classements pour la plateforme choisie. Veuillez noter qu'il s'agit d'une fonction asynchrone, c'est-à-dire que votre jeu continuera à fonctionner en arrière-plan pendant que la page des classements est affichée. En tant que tel, vous devez faire attention à l'endroit où vous l'utilisez et assurez-vous de mettre le jeu en pause ou de ne permettre son affichage que dans les zones de votre jeu où il n'interférera pas avec le jeu. Syntaxe:achievement_show_leaderboards() Retour:N/A Exemple:if achievement_available()     {     global.Pause = true;     instance_create_layer(0, 0, "Controllers", obj_Pause);     achievement_show_leaderboards();     } Le code ci-dessus vérifie si le service de classement et de réussite choisi est disponible et si c'est le cas, il définit une variable globale et crée une instance de "pause" du contrôleur avant d'appeler la fonction pour afficher les classements à l'écran.
