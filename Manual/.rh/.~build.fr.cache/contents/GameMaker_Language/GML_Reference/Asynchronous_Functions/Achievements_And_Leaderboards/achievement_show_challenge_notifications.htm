notifications_de_défis_réalisés Cette fonction vous permet d'afficher ou de supprimer les différentes notifications pop-up "toast" relatives aux défis. Lorsque les arguments sont définis sur true (vrai), ces messages apparaîtront, informant le joueur de tous les défis locaux ou distants reçus ainsi que des défis terminés. Lorsque les arguments sont réglés sur vrai, ces messages apparaîtront, informant le joueur de tout défi local ou distant reçu ainsi que des défis qui ont été complétés, tandis que le réglage sur faux supprimera ces notifications. Syntaxe : achievement_show_challenge_notifications(receive, local, remote) Arguments Description recevoir Afficher les défis reçus dans une fenêtre pop-up(true) ou non(false). local Afficher les défis locaux terminés dans une fenêtre pop-up(true) ou non(false). à distance Afficher la fenêtre contextuelle des défis à distance terminés(vrai) ou non(faux). Les retours : N/A Exemple : achievement_show_challenge_notifications(true, false, false) ; Le code ci-dessus permet d'afficher les notifications de toast de défi uniquement pour les défis reçus.
