shop_leave_rating Cette fonction ouvre une boîte de dialogue dépendante du système d'exploitation dans laquelle vous pouvez demander à l'utilisateur de publier une note ou un commentaire sur une page particulière. Vous pouvez définir le texte qui doit apparaître dans la boîte de dialogue, ainsi que le texte que vous souhaitez voir apparaître sur les deux boutons et l'URL où le commentaire doit être publié. REMARQUE: cette fonction est uniquement destinée aux cibles Android et iOS IMPORTANT! Sur iOS, lors de l'utilisation de TestFlight, lors du développement, l'invite sera affichée mais vous ne pourrez rien faire. Une fois que votre application est dans l'App Store, ce bouton devient actif et l'utilisateur a la possibilité d'écrire un avis ou de laisser une note. Notez que si vous distribuez le jeu via TestFlight, cette fenêtre peut même ne pas s'afficher. Syntaxe:shop_leave_rating(text, yes_string, no_string, url) Argument Description text Texte qui apparaît dans la boîte de dialogue. yes_string Texte qui apparaît sur le bouton "oui". no_string Texte qui apparaît sur le bouton «non». url L'URL complète à laquelle le commentaire doit être envoyé. Retour:N/A Exemple:if timer <= 0     {     shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");     timer = 10000000;     } else     {     timer -= 1;     } Le code ci-dessus demandera à l'utilisateur de laisser un commentaire si la variable "timer" a décompté jusqu'à 0.
