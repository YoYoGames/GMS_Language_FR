http_get_request_crossorigin Cette fonction peut être utilisée pour obtenir le type d'origine croisée défini pour les jeux HTML5 et renvoie une chaîne (sur toutes les autres plateformes, une chaîne vide "" sera renvoyée). Syntaxe : http_get_request_crossorigin() ; Les retours : Chaîne de caractères Exemple : if string_lower(http_get_request_crossorigin()) != "use-credentials" { http_set_request_crossorigin("use-credentials") ; } sprite_add("sprites/player_outfit_1.png", 0, false, false, 0, 0) ; Le code ci-dessus vérifie d'abord le type d'origine croisée actuellement défini et s'il n'est pas "use-credentials", il est défini comme "use-credentials", puis un sprite est ajouté à partir d'un fichier.
