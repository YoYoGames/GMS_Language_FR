Économie dans le cloud Lors de la création de jeux, il est souvent nécessaire de stocker des informations sur l'état du jeu dans un fichier d'un certain type, mais stocker ces informations sur l'appareil n'est pas toujours la meilleure option car, si le joueur supprimait le jeu puis le réinstalle, cela des informations peuvent être perdues. Pour éviter cela, vous pouvez utiliser différents services cloud, qui offrent un stockage de données sur Internet pour la récupération et la modification à tout moment. Notez que cette fonction est limitée à un seul "blob" de données par jeu, donc chaque fois que vous envoyez une nouvelle sauvegarde au service cloud, qu'il s'agisse d'une chaîne ou d'un fichier, elle écrasera toutes les données précédemment stockées. REMARQUE: actuellement, seule la Android utilise ces fonctions et vous devez cocher la case Activer Google Cloud Saving dans la section Social des options de jeu Android. Cela vous invitera à télécharger l' extension Google Services, qui est requise pour que l'enregistrement dans le cloud sur Android fonctionne. Pour plus d'informations sur Cloud Saving, veuillez consulter l'article suivant du service d'assistance: Android Cloud Saving GameMaker Studio 2 prend en charge cela avec quelques fonctions simples qui fonctionnent en conjonction avec l' événement Asynchronous Cloud.
