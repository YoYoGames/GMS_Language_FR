Sauvegarde du fichier dans le nuage Cette fonction va livrer un fichier au service en nuage choisi pour le stockage. La fonction renvoie une valeur d'identification unique qui doit ensuite être utilisée dans l'événement asynchrone approprié pour identifier la carte DS qui est renvoyée en tant que "call back". du service en nuage. Le fichier doit contenir toutes les informations que vous devez sauvegarder pour votre jeu, car vous ne pouvez stocker qu'un seul "blob de données" dans le nuage. (tout comme l'utilisation de la fonction cloud_string_save() ). La description doit être une courte chaîne d'informations décrivant la sauvegarde, par exemple : "Level2, Stage2". Pour plus d'informations sur les données asynchrones renvoyées, veuillez consulter la fonction cloud_synchronise(). Syntaxe : cloud_file_save(fichier, description) ; Arguments Description chaîne de caractères Le fichier à télécharger (sous forme de chaîne). description Une brève description des données stockées. Les retours : Real Exemple : var t_str = "" ; for (var i = 0 ; i < 10 ; i++ ;) { t_str += string(global.Highscore[i]) + "|" } } var file = file_text_open_write("Highscores.txt") ; file_text_write_string(file, t_str) ; file_text_close(file) ; save_check = cloud_file_save("Highscores.txt", "Current Highscores") ; Le code ci-dessus crée une chaîne à partir des valeurs stockées dans le tableau global "Highscores", puis écrit cette chaîne dans un fichier pour le stockage local.
