steam_is_cloud_enabled_for_app Cette fonction vous permet de vérifier que le service Steam Cloud est activé pour votre jeu. Elle renvoie true si c'est le cas et false sinon. Notez que cela ne signifie pas automatiquement que vous pouvez utiliser les fonctions Cloud car l'utilisateur peut désactiver la synchronisation du nuage à partir de son client Steam. Vous pouvez le vérifier à l'aide de la fonction steam_is_cloud_enabled_for_account()mais, même si elle est désactivée pour l'utilisateur (et activée pour le jeu), la fonction (et activée pour le jeu), les fonctions fonctionneront toujours pour stocker et récupérer des données à partir d'une copie locale de tous les fichiers, mais elles ne les téléchargeront pas dans le nuage à la fin du jeu et ne les synchroniseront pas au démarrage du jeu. Syntaxe : steam_is_cloud_enabled_for_app() ; Les retours : Booléen Exemple : if steam_is_cloud_enabled_for_app() { quota = steam_get_quota_total() ; } Le code ci-dessus vérifie si le nuage de vapeur est activé pour le jeu et si oui, il obtient la taille du quota de stockage et la stocke dans une variable.
