steam _file_write_file Avec cette fonction, vous pouvez copier le contenu d'un fichier enregistré localement dans un fichier synchronisé avec le Steam Cloud. Le fichier local doit exister avant d'utiliser cette fonction et il renverra une valeur de 0 s'il échoue pour une raison quelconque et une valeur supérieure à 0 s'il réussit. Syntaxe:steam_file_write_file(steam_filename, local_filename); Argument Description steam _filename Le Steam Cloud à copier. local_filename Le fichier local à utiliser pour copier. Retour:real Exemple:steam_file_write_file("rm_koala.png", "Koala2.png"); Le code ci-dessus copiera le contenu du fichier "Koala2.png" dans le Steam Cloud "rm_koala.png".
