Nuage de Steam Le Steam Cloud fournit un système de stockage de fichiers à distance simple et transparent pour votre jeu. Tous les fichiers écrits sur le disque à l'aide des GameMaker Studio 2 (répertoriées ci-dessous) seront répliqués sur les Steam fois le jeu terminé. Si l'utilisateur change ensuite d'ordinateur, les fichiers seront ensuite téléchargés sur le nouvel ordinateur avant le lancement du jeu, ce qui signifie que le jeu peut ensuite accéder aux fichiers en les lisant à l'aide des fonctions Steam Le Steam veille à ce que les fichiers restent synchronisés sur tous les ordinateurs auxquels l'utilisateur peut accéder. REMARQUE: Par défaut, le Cloud n'est pas activé pour un jeu sur Steam fonctionne. il doit être activé au préalable à partir de l'onglet `` Cloud '' de l' Steam Works, où vous devez définir le quota d'octet et de fichier. La prochaine fois que vous publierez la Steam Works, le stockage Cloud sera prêt à être utilisé.
