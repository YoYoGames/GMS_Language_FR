steam _get_quota_free Avec cette fonction, vous pouvez savoir combien d'espace libre reste à l'utilisateur du Steam Cloud. La valeur renvoyée est en octets. Syntaxe:steam_get_quota_free(); Retour:Real Exemple:if steam_is_cloud_enabled_for_app()     {     quota = steam_get_quota_free();     } Le code ci-dessus vérifie si le steam est activé pour le jeu et si c'est le cas, il obtient la taille de l'espace de stockage gratuit et le stocke dans une variable.
