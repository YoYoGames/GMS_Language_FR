steam_get_quota_free Cette fonction vous permet de savoir combien d'espace libre il reste pour l'utilisateur du quota Steam Cloud. La valeur retournée est en octets. Syntaxe : steam_get_quota_free() ; Les retours : Real Exemple : if steam_is_cloud_enabled_for_app() { quota = steam_get_quota_free() ; } Le code ci-dessus vérifie si le nuage de vapeur est activé pour le jeu et si oui, il obtient la taille de l'espace de stockage libre et la stocke dans une variable.
