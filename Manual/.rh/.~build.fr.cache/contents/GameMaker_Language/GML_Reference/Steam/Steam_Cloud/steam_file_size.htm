steam _file_size Avec cette fonction, vous pouvez vérifier la taille d'un fichier stocké sur le Steam Cloud. Le nombre réel renvoyé est la taille, en octets, du fichier. Syntaxe:steam_file_size(filename); Argument Description filename Le nom du fichier dont la taille doit être vérifiée. Retour:Real Exemple:file_bytes = steam_file_size("Save.txt"); Le code ci-dessus stocke la taille d'un fichier du Steam Cloud dans la variable "file_bytes".
