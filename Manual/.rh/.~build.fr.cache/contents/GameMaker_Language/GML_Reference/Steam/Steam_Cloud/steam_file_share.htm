partage de fichier vapeur Cette fonction vous permet de forcer votre jeu à synchroniser le fichier donné avec le Steam Cloud. Cette opération n'est normalement pas nécessaire, car le jeu se synchronise automatiquement à la fin de la session du joueur, et elle n'est pas recommandée par Steam. par Steam, mais cela peut être utile pour s'assurer que les informations sensibles sont synchronisées immédiatement. La fonction renvoie une valeur de 0 si elle échoue pour une raison quelconque et une valeur supérieure à 0 si elle réussit. Syntaxe : steam_file_share(nom du fichier) ; Arguments Description nom de fichier Le nom du fichier à synchroniser. Les retours : Real Exemple : if !steam_file_persisted("Save.txt") { steam_file_share("Save.txt") ; } Le code ci-dessus vérifiera si un fichier a été stocké dans le Steam Cloud, et s'il ne l'a pas été, il le synchronisera.
