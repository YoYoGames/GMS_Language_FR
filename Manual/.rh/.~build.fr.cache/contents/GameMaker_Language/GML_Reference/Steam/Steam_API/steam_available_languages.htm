steam_available_languages Cette fonction peut être utilisée pour récupérer une liste de toutes les langues disponibles pour Steam. La valeur retournée est simplement une liste de langues séparées par des virgules. Syntaxe : steam_available_languages() ; Les retours : Chaîne de caractères Exemple : langue = steam_available_languages() ; L'exemple ci-dessus récupère les langues disponibles pour Steam sous forme de chaîne et les stocke dans une variable.
