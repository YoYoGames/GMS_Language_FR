steam _langues_disponibles Cette fonction peut être utilisée pour récupérer une liste de toutes les langues disponibles pour Steam. La valeur renvoyée est simplement une liste de langues séparées par des virgules. Syntaxe:steam_available_languages(); Retour:String Exemple:language = steam_available_languages(); Ce qui précède obtient les langues disponibles pour Steam sous forme de chaîne et les stocke dans une variable.
