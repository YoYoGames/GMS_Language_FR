steam _is_user_logged_on Cette fonction retournera true si le Steam a actuellement une connexion en direct aux serveurs Steam S'il revient false, cela signifie qu'il n'y a pas de connexion active en raison d'un problème de réseau sur la machine locale ou du Steam en panne ou occupé. Syntaxe:steam_is_user_logged_on(); Retour:String Exemple:if steam_is_user_logged_on()     {     global.user_id = steam_get_user_account_id();     } Le code ci-dessus vérifiera si l'utilisateur est connecté au Steam et s'il stocke l'ID utilisateur dans une variable globale.
