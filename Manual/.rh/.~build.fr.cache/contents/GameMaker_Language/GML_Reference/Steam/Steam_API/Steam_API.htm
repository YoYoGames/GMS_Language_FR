API général de la vapeur L'ensemble des fonctions suivantes servent à vérifier la disponibilité de certains aspects de l'API du client ou du serveur Steam, et renvoient soit vrai, soit faux, en fonction de facteurs comme l'occupation du serveur, la connexion de l'utilisateur, etc. comme le serveur étant occupé ou l'utilisateur étant connecté etc...
