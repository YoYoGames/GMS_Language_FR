steam _get_app_id Cette fonction est utilisée pour récupérer l'ID d'application unique que Steam attribue à votre jeu, ce qui est nécessaire pour utiliser certaines des fonctions de contenu généré par l'utilisateur. Syntaxe:steam_get_app_id(); Retour:Real Exemple:global.app_id = steam_get_app_id(); Le code ci-dessus obtient l'ID d'application unique de votre jeu sur Steam et le stocke dans une variable globale.
