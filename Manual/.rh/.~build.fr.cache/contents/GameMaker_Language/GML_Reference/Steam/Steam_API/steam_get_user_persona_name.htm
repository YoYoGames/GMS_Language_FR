steam _get_user_persona_name Cette fonction peut être utilisée pour récupérer le nom d'utilisateur (nom d'écran) pour tout ID utilisateur spécifique. Vous donnez l'ID utilisateur, puis la fonction renvoie une valeur d'ID asynchrone unique qui peut être utilisée pour vérifier correctement les détails dans l' événement Steam Async. Cet événement aura le async_load Carte DS qui sera remplie avec les paires clé / valeur suivantes: " id " - L'ID asynchrone renvoyé par la fonction qui a déclenché l'événement " event_type " - La chaîne " user_persona_name " " steam id" "- La Steam utilisée pour obtenir le nom d'utilisateur. " persona_name " - Le nom d'utilisateur pour l'identifiant Steam Syntaxe:steam_get_user_persona_name(steamID); Argument Description steam ID L' Steam unique pour un utilisateur. Retour:Real Exemple:request = steam_get_user_persona_name(global.UGC_UserID); Le code ci-dessus demandera le nom d'utilisateur de l'ID utilisateur stocké dans la variable globale "UGC_UserID", en stockant la valeur renvoyée dans une variable pour l'analyse dans l'événement Async.
