steam _activate_overlay La Steam est un élément de l' Steam qui peut être activé par-dessus presque tous les jeux lancés via Steam. Il permet à l'utilisateur d'accéder à sa liste d'amis, à son navigateur Web, au chat et à l'achat de DLC en jeu. La touche par défaut permettant à un utilisateur d'accéder à la superposition dans un jeu est MAJ + TAB, mais vous pouvez également afficher n'importe quelle page de la superposition à l'aide de cette fonction. Il faut l'une des six constantes répertoriées ci-dessous: Constant Description ov_friends La page d'amis de l'utilisateur actuel ov_community La page communautaire de votre jeu ov_players La page montrant les autres joueurs qui jouent au même jeu ou avec lesquels vous avez récemment joué ov_settings Les paramètres de superposition du client Steam ov_gamegroup Ouvre le Steam au groupe de jeu officiel de ce jeu ov_achievements La page des succès de votre jeu Syntaxe:steam_activate_overlay(overlayindex); Argument Description overlayindex L'index de page de l' Steam à afficher (voir les constantes ci-dessous). Retour:N/A Exemple:var key = keyboard_lastkey; switch (key)     {     case vk_f1: steam_activate_overlay(ov_friends); break;     case vk_f2: steam_activate_overlay(ov_community); break;     case vk_f3: steam_activate_overlay(ov_players); break;     case vk_f4: steam_activate_overlay(ov_settings); break;     case vk_f5: steam_activate_overlay(ov_gamegroup); break;     case vk_f6: steam_activate_overlay(ov_achievements); break;     } Le code ci-dessus interroge la dernière touche du clavier enfoncée et s'il s'agit de l'une des touches de fonction de 1 à 6, il ouvrira la page correspondante de la superposition Steam Retour: Superposition de Steam Suivant: steam _activate_overlay_browser © Copyright YoYo Games Ltd. 2021 Tous droits réservés
