steam _activate_overlay_store Avec cette fonction, vous pouvez ouvrir la Steam sur la page du magasin pour un jeu afin que les utilisateurs puissent acheter ou télécharger des DLC (par exemple). Vous devez fournir l'ID d'application unique pour le jeu ou le DLC que vous obtiendriez à partir du Steam lorsque vous le configurez. Syntaxe:steam_activate_overlay_store(app_id); Argument Description app_id L'ID d'application unique pour votre jeu. Retour:N/A Exemple:if keyboard_check_pressed(vk_f1)     {     steam_activate_overlay_store(global.DLC_id);     } Le code ci-dessus interroge le clavier pour la touche F1 et si c'est le cas, la Steam sera ouverte sur la page du contenu du jeu à l'aide de l'ID d'application stocké dans la variable globale.
