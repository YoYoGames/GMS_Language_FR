steam _is_overlay_activated Cette fonction peut être utilisée pour savoir si l'utilisateur a activé ou non la superposition Steam Si la superposition est active et visible pour l'utilisateur, la fonction retournera true, et si ce n'est pas le cas, il reviendra false. Un exemple d'utilisation de cette fonction serait d'interroger l' Steam pour la superposition afin que vous puissiez mettre votre jeu en pause pendant que la superposition est affichée. Syntaxe:steam_is_overlay_activated(); Retour:Boolean Exemple:if steam_is_overlay_activated()     {     global.Pause = true;     } Le code ci-dessus vérifiera si la Steam est active et si c'est le cas, il définira la variable globale "Pause" sur true.
