steam_is_overlay_activated Cette fonction peut être utilisée pour savoir si la superposition de la vapeur est active ou non pour l'utilisateur. Si la superposition est active et visible pour l'utilisateur, la fonction renverra true, et si elle ne l'est pas, elle renverra false. Un exemple de ce à quoi cette fonction peut être utilisée pour interroger l'API Steam pour la superposition afin que vous puissiez mettre votre jeu en pause pendant que la superposition est affichée. Syntaxe : steam_is_overlay_activated() ; Les retours : Booléen Exemple : if steam_is_overlay_activated() { global.Pause = true ; } Le code ci-dessus vérifie si la superposition Steam est active et, si c'est le cas, il définit la variable globale "Pause" sur true.
