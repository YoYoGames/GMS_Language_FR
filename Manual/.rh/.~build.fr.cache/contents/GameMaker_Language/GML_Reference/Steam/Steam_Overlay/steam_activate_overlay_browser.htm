steam_activate_overlay_browser Avec cette fonction, vous pouvez ouvrir la superposition de jeu Steam dans son navigateur Web et lui faire charger l'URL spécifiée. Vous devez utiliser l'URL complète sous forme de chaîne pour que la résolution soit correcte, par exemple : "http://www. steamgames.com/". Syntaxe : steam_activate_overlay(url) ; Arguments Description url L'URL (complète) de la superposition à ouvrir. Les retours : N/A Exemple : if keyboard_check_pressed(vk_f1) { steam_activate_overlay_browser("http://www. steamgames.com/") ; } Le code ci-dessus interroge le clavier à la recherche de la touche F1 et si c'est le cas, la superposition Steam s'ouvre et se résout à l'URL donnée.
