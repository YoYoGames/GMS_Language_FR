steam _activate_overlay_browser Avec cette fonction, vous pouvez ouvrir la Steam sur son navigateur Web, puis lui demander de charger l'URL spécifiée. vous devez utiliser l' URL complète comme chaîne pour que cela soit résolu correctement, par exemple: "http: // www. steam games.com/". Syntaxe:steam_activate_overlay(url); Argument Description url L'URL (complète) de la superposition à ouvrir. Retour:N/A Exemple:if keyboard_check_pressed(vk_f1)     {     steam_activate_overlay_browser("http://www.steamgames.com/");     } Le code ci-dessus interroge le clavier pour la touche F1 et si c'est le cas, la Steam sera ouverte et résolue vers l'URL donnée.
