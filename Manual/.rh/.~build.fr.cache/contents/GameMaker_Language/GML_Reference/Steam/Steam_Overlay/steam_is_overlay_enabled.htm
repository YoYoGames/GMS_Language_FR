steam _is_overlay_enabled Lors de l'utilisation de l' Steam, cette fonction peut être appelée pour vérifier que l' Steam a activé la fonctionnalité de superposition. Syntaxe:steam_is_overlay_enabled(); Retour:Boolean Exemple:global.steam_api = false; if steam_initialised()     {     if steam_stats_ready() && steam_is_overlay_enabled()        {        global.steamapi = true;        }     } Le code ci-dessus définira une variable globale sur true si l' Steam est correctement initialisée, avec les Steam et la fonctionnalité de superposition, ou elle définira la variable sur false autrement.
