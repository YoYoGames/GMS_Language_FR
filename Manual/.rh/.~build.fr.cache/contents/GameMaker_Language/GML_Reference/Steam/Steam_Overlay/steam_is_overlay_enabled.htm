steam_is_overlay_enabled Lors de l'utilisation de l'API Steam, cette fonction peut être appelée pour vérifier que l'API du client Steam a la fonctionnalité de superposition activée. Syntaxe : steam_is_overlay_enabled() ; Les retours : Booléen Exemple : global.steam_api= false ; if steam_initialised() { if steam_stats_ready() && steam_is_overlay_enabled() { global.steamapi= true ; } } Le code ci-dessus définira une variable globale à true si l'API client Steam est correctement initialisée, ainsi que les statistiques Steam et la fonctionnalité de superposition, ou il définira la variable à false sinon.
