DLC Steam prend en charge le contenu téléchargeable (DLC) gratuit et payant. Dans le client Steam, un jeu avec contenu téléchargeable apparaît comme une seule application dans la liste des jeux de l'utilisateur, le contenu téléchargeable étant visible dans la boîte de dialogue des propriétés du jeu. Une fois possédé, le contenu téléchargeable est traité comme une partie intégrante du jeu et Steam met automatiquement à jour le contenu lorsqu'un patch est disponible et installe le contenu lorsque l'utilisateur installe le jeu.
