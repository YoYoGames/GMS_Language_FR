steam_user_installed_dlc Si votre jeu possède un DLC créé pour lui, vous pouvez utiliser cette fonction pour vérifier si l'utilisateur l'a installé avant d'accéder aux fichiers qui lui sont associés. La fonction renvoie true si le joueur a installé le contenu, et false si l'utilisateur ne l'a pas fait, mais notez que l'utilisateur doit également posséder le DLC, donc vous devriez utiliser la fonction supplémentaire de steam_user_owns_dlc() pour vérifier qu'il est également possédé avant de l'utiliser. Notez que les ID Steam peuvent être des nombres importants. Vous devrez donc peut-être convertir votre valeur d'ID en tant que int64() avant de la fournir à la fonction. Syntaxe : steam_user_installed_dlc(dlc_id) ; Arguments Description dlc_id L'identifiant unique du DLC à vérifier (un nombre entier). Les retours : Booléen Exemple : global.Level_Max = 100 ; if steam_user_owns_dlc(10354) { if steam_user_installed_dlc(10354) { global.Level_max = 200 ; } } Le code ci-dessus vérifie si l'utilisateur a acheté et installé le DLC avec l'identifiant 10354, et si c'est le cas, il définit une variable globale à une valeur différente.
