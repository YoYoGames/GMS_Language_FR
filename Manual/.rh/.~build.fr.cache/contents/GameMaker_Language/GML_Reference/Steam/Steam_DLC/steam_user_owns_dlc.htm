steam_user_owns_dlc Si votre jeu possède un DLC créé pour lui, vous pouvez utiliser cette fonction pour vérifier si l'utilisateur l'a acheté avant d'accéder aux fichiers qui lui sont associés. La fonction renvoie true si le joueur possède le contenu, et false si l'utilisateur ne l'a pas acheté. mais notez que même si l'utilisateur possède le DLC, il se peut qu'il ne l'ait pas installé, vous devez donc utiliser la fonction supplémentaire de steam_user_installed_dlc() pour vérifier qu'il est installé avant de l'utiliser. Notez que les identifiants Steam peuvent être des nombres importants et que vous devrez peut-être convertir la valeur de votre identifiant en un numéro int64() avant de la fournir à la fonction. Syntaxe : steam_user_owns_dlc(dlc_id) ; Arguments Description dlc_id L'identifiant unique du DLC à vérifier (un nombre entier int64). Les retours : Booléen Exemple : global.Level_Max = 100 ; if steam_user_owns_dlc(10354) { if steam_user_installed_dlc(10354) { global.Level_max = 200 ; } } Le code ci-dessus vérifie si l'utilisateur a acheté et installé le DLC avec l'identifiant 10354, et si c'est le cas, il définit une variable globale à une valeur différente.
