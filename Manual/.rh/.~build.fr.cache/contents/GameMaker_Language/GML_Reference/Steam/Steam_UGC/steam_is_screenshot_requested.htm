steam_is_screenshot_requested Cette fonction interroge l'API Steam pour savoir si la touche permettant de prendre une capture d'écran du jeu a été enfoncée. La fonction ne retournera vrai que pour une étape (tick du jeu) si la touche a été pressée, et retournera faux à tous les autres moments. autres moments. REMARQUE: cela ne prend pas de capture d'écran pour vous. Cela signale seulement que la touche a été pressée et que vous devez utiliser les fonctions de GameMaker Studio 2 screen_save() ou screen_save_part() pour enregistrer une copie locale du fichier à télécharger. Syntaxe : steam_is_screenshot_requested() ; Les retours : Booléen Exemple : if steam_is_screenshot_requested() { var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png") ; screen_save(file) steam_send_screenshot(file, window_get_width(), window_get_height()) ; global.scrn_num += 1 ; } Le code ci-dessus interroge l'API Steam pour une demande de capture d'écran et, si c'est le cas, un nom unique pour le fichier image est généré, une capture d'écran est effectuée et le fichier est envoyé à la page de la communauté Steam pour l'utilisateur.
