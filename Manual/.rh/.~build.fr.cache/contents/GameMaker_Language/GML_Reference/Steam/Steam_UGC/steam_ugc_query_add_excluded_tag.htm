steam _ugc_query_add_excluded_tag Cette fonction peut être utilisée pour filtrer davantage toute requête UGC donnée, en particulier pour exclure un UGC donné de la demande de requête. Le descripteur de requête est la valeur renvoyée lorsque vous avez créé la requête (en utilisant, par exemple, steam_ugc_create_query_user() ) et le deuxième argument est une chaîne que vous souhaitez utiliser comme balise à exclure. La fonction retournera true si le filtre de requête a été correctement défini, ou false autrement. Syntaxe:steam_ugc_query_add_excluded_tag(ugc_query_handle , tag_name); Argument Description ugc_query_handle Le handle de requête à utiliser. tag_name Le nom de la balise à exclure (une chaîne). Retour:Boolean Exemple:var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1); steam_ugc_query_add_excluded_tag(query_handle, "walking simulator"); steam_ugc_query_set_return_long_description(query_handle, true); steam_ugc_query_set_allow_cached_response(query_handle, true); query_ID = steam_ugc_send_query(query_handle); Le code ci-dessus crée une demande de requête et stocke son handle dans une variable locale pour une utilisation future dans le reste des fonctions qui définissent davantage la demande de requête avant d'envoyer la requête.
