steam _ugc_get_subscribed_items Cette fonction remplira une liste DS avec tous les ID de fichiers publiés pour les éléments auxquels l'utilisateur est actuellement abonné. Vous devez d'abord créer la liste et stocker l'index dans une variable, puis la transmettre à la fonction. La fonction retournera true si tout est correct et que l' Steam est initialisée, ou false s'il y a une erreur. Syntaxe:steam_ugc_get_subscribed_items(item_list); Argument Description item_list Un index de liste DS (précédemment créé). Retour:Boolean Exemple:steam_list = ds_list_create(); steam_ugc_get_subscribed_items(steam_list); Le code ci-dessus créera une liste DS vide, puis la remplira avec les ID de fichier pour tous les éléments abonnés pour l'utilisateur.
