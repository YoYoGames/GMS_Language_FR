steam_ugc_query_set_search_text Cette fonction peut être utilisée pour filtrer davantage une requête de CGU donnée, notamment pour rechercher la chaîne donnée dans le titre et la description des éléments de CGU. L'identifiant de la requête est la valeur renvoyée lorsque vous avez créé la requête (en utilisant, par exemple, steam_ugc_create_query_user()) et le deuxième argument est une chaîne que vous souhaitez utiliser comme terme de recherche. La fonction renvoie un message vrai si le filtre de la requête a été correctement défini, ou faux dans le cas contraire. Syntaxe : steam_ugc_query_set_search_text(ugc_query_handle , search_text) ; Arguments Description ugc_query_handle La poignée de requête à utiliser. texte de recherche Le texte de recherche à utiliser (une chaîne) pour la requête. Les retours : Booléen Exemple : var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1) ; steam_ugc_query_set_search_text(query_handle, "texture") ; steam_ugc_query_set_return_long_description(query_handle, true) ; steam_ugc_query_set_allow_cached_response(query_handle, true) ; query_ID = steam_ugc_send_query(query_handle) ; Le code ci-dessus crée une demande de requête et stocke son identifiant dans une variable locale pour une utilisation future dans le reste des fonctions qui définissent plus précisément la demande de requête avant d'envoyer la requête.
