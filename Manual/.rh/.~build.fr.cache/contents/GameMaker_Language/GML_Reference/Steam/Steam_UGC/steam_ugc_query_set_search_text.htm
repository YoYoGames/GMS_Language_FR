steam _ugc_query_set_search_text Cette fonction peut être utilisée pour filtrer davantage toute requête UGC donnée, en particulier pour rechercher la chaîne donnée dans le titre et la description des éléments UGC. Le descripteur de requête est la valeur renvoyée lorsque vous avez créé la requête (en utilisant, par exemple, steam_ugc_create_query_user() ) et le deuxième argument est une chaîne que vous souhaitez utiliser comme terme de recherche. La fonction retournera true si le filtre de requête a été correctement défini, ou false autrement. Syntaxe:steam_ugc_query_set_search_text(ugc_query_handle , search_text); Argument Description ugc_query_handle Le handle de requête à utiliser. search_text Le texte de recherche à utiliser (une chaîne) pour la requête. Retour:Boolean Exemple:var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1); steam_ugc_query_set_search_text(query_handle, "texture"); steam_ugc_query_set_return_long_description(query_handle, true); steam_ugc_query_set_allow_cached_response(query_handle, true); query_ID = steam_ugc_send_query(query_handle); Le code ci-dessus crée une demande de requête et stocke son handle dans une variable locale pour une utilisation future dans le reste des fonctions qui définissent davantage la demande de requête avant d'envoyer la requête.
