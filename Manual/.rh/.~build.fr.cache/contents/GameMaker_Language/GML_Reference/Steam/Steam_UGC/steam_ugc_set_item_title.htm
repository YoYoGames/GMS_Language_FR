steam_ugc_set_item_title Cette fonction va définir le titre à utiliser pour l'élément donné. Le titre doit être une chaîne de caractères et le handle de mise à jour est la valeur retournée lorsque vous avez appelé la fonction steam_ugc_start_item_update(). La fonction renvoie true si l'accès à l'API s'est déroulé avec succès et false s'il y a eu un problème. Syntaxe : steam_ugc_set_item_title(ugc_update_handle, title) ; Arguments Description ugc_update_handle L'identifiant unique de l'UGC à mettre à jour. titre Le titre (une chaîne - 128 caractères maximum) à utiliser pour l'élément. Les retours : Booléen Exemple : var app_id = steam_get_app_id() ; var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID) ; steam_ugc_set_item_title(updateHandle, "Mon article d'atelier(3) !") ; steam_ugc_set_item_description( updateHandle, "testant atelier...") ; steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public) ; var tagArray ; tagArray[0] = "Test" ; tagArray[1] = "New" ; steam_ugc_set_item_tags( updateHandle, tagArray) ; steam_ugc_set_item_preview (updateHandle, "promo.jpg") ; steam_ugc_set_item_content(updateHandle, "WorkshopContent1") ; requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2") ; Le code ci-dessus récupère l'ID du jeu, puis l'utilise avec l'ID du fichier publié précédemment stocké pour générer un identifiant de mise à jour pour l'élément. Ce gestionnaire est ensuite utilisé pour mettre à jour divers éléments d'information avant que la mise à jour ne soit envoyée aux serveurs de l'atelier. des serveurs.
