steam_ugc_query_set_return_long_description Cette fonction peut être utilisée pour filtrer davantage une requête UGC donnée, en particulier pour récupérer le texte de la description longue dans l'événement de rappel déclenché lorsque la requête a été envoyée. L'identifiant de la requête est la valeur renvoyée lorsque vous avez créé la requête (en utilisant, par exemple, la fonction par exemple, steam_ugc_create_query_user()) et le second argument est soit true soit false. La fonction renvoie true si le filtre de la requête a été correctement défini, ou false sinon. Syntaxe : steam_ugc_query_set_return_long_description(ugc_query_handle , long_description) ; Arguments Description ugc_query_handle La poignée de requête à utiliser. long_description Si la requête doit retourner le texte de la description longue. Les retours : Booléen Exemple : var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1) ; steam_ugc_query_set_return_long_description(query_handle, true) ; steam_ugc_query_set_allow_cached_response(query_handle, true) ; query_ID = steam_ugc_send_query(query_handle) ; Le code ci-dessus crée une demande de requête et stocke son identifiant dans une variable locale pour une utilisation future dans le reste des fonctions qui définissent plus précisément la demande de requête avant d'envoyer la requête.
