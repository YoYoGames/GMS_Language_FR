steam_ugc_get_item_update_info Cette fonction peut être utilisée pour récupérer des informations sur l'état actuel du téléchargement pour l'ID de fichier donné. Vous donnez l'ID de l'élément et fournissez l'index d'une carte DS vide qui sera alors remplie avec les paires clé/valeur suivantes si l'élément existe : "needs_update" - renvoie true ou false selon que l'élément doit être mis à jour ou non. "is_downloading" - renvoie true ou false selon que l'élément est en cours de téléchargement ou non. "bytes_downloaded" - Le nombre d'octets qui ont été téléchargés. " bytes_total" - La taille totale (nombre d'octets) requise pour l'élément sur le disque. Si l'élément existe, la fonction renvoie true et remplit la carte, sinon elle renvoie false et la carte reste vide. Syntaxe : steam_ugc_get_item_update_info(published_file_id, info_map) ; Arguments Description fichier_publié_id L'ID de fichier unique pour l'UGC à vérifier. info_map Un index de carte DS (précédemment créé). Les retours : Booléen Exemple : var info_map = ds_map_create() ; var info = steam_ugc_get_item_update_info(global.fileID, info_map) ; if info { draw_text(32, 15, "needs_update : " + string(info_map[ ? "needs_update"])) ; draw_text(32, 30, "is_downloading : " + string(info_map[ ? "is_downloading"])) ; draw_text(32, 45, "bytes_downloaded : " + string(info_map[ ? "bytes_downloaded"])) ; draw_text(32, 60, "bytes_total : " + string(info_map[ ? "bytes_total"])) ; } Le code ci-dessus interroge l'état de téléchargement de l'élément indexé dans la variable globale "fileID", en utilisant un DS Map pour stocker l'information.
