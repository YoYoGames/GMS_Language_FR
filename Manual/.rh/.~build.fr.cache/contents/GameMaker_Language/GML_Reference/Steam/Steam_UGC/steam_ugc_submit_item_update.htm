steam_ugc_submit_item_update Cette fonction soumet l'élément UGC indexé par le handle donné aux serveurs Steam Workshop, en ajoutant les notes de modification à utiliser pour l'élément donné. Les notes de modification doivent être une chaîne de caractères et le gestionnaire de mise à jour est la valeur renvoyée lorsque vous avez appelé la fonction fonction steam_ugc_start_item_update(). Lorsque vous utilisez cette fonction, elle renvoie une valeur d'identification asynchrone qui peut ensuite être analysée lorsque l'événement asynchroneSteam est déclenché pour signaler la création de l'élément. L'événement contiendra les valeurs de clé/map suivantes dans le fichier async_load carte DS : "id" - L'identifiant asynchrone renvoyé par la fonction appelante. "resultat" - Le résultat de l'opération (une valeur réelle). Ce sera soit la constante GML ugc_result_success ou un autre nombre réel. Vous devez donc vérifier cette constante pour vous assurer que l'appel a réussi, et si sinon, quelque chose n'a pas fonctionné correctement. Le reste des valeurs possibles retournées sont affichées comme le résultat de la valeur "EResult" de Steam et vous devriez voir steamclientpublic .h dans les en-têtes du SDK pour les 89 valeurs possibles. "event_type" - Cette clé contient la valeur "ugc_update_item". " legal_agreement_required" - Sera vrai ou faux (voir la docs Steam pour plus de détails) Syntaxe : steam_ugc_submit_item_update(ugc_update_handle, change_note) ; Arguments Description ugc_update_handle L'identifiant unique de l'UGC à mettre à jour. change_note Les notes de modification (une chaîne) à utiliser pour l'élément. Les retours : ID asynchrone Exemple : var app_id = steam_get_app_id() ; var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID) ; steam_ugc_set_item_title(updateHandle, "Mon article d'atelier(3) !") ; steam_ugc_set_item_description( updateHandle, "testant atelier...") ; steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public) ; var tagArray ; tagArray[0] = "Test" ; tagArray[1] = "New" ; steam_ugc_set_item_tags( updateHandle, tagArray) ; steam_ugc_set_item_preview (updateHandle, "promo.jpg") ; steam_ugc_set_item_content(updateHandle, "WorkshopContent1") ; requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2") ; Le code ci-dessus récupère l'ID du jeu, puis l'utilise avec l'ID du fichier publié précédemment stocké pour générer un identifiant de mise à jour pour l'élément. Ce gestionnaire est ensuite utilisé pour mettre à jour divers éléments d'information avant que la mise à jour ne soit envoyée aux serveurs de l'atelier. des serveurs.
