steam_ugc_set_item_visibility Cette fonction va définir la visibilité de l'élément donné, défini à l'aide de l'une des constantes suivantes : Constant Description ugc_visibilité_public Définir l'élément pour qu'il soit visible publiquement ugc_visibilité_amis_seulement Configurer l'élément pour qu'il soit visible uniquement par les personnes figurant dans la liste d'amis de l'utilisateur. ugc_visibilité_privée Définir l'élément comme étant privé Le handle de mise à jour est la valeur renvoyée lorsque vous avez appelé la fonction steam_ugc_start_item_update()La fonction renverra un message vrai si l'accès à l'API a été effectué avec succès et un message faux si un problème est survenu. Syntaxe : steam_ugc_set_item_visibility(ugc_update_handle, visibility) ; Arguments Description ugc_update_handle L'identifiant unique de l'UGC à mettre à jour. visibilité La visibilité (une constante, listée ci-dessous) à utiliser pour l'élément. Les retours : Booléen Exemple : var app_id = steam_get_app_id() ; var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID) ; steam_ugc_set_item_title(updateHandle, "Mon article d'atelier(3) !") ; steam_ugc_set_item_description( updateHandle, "testant atelier...") ; steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public) ; var tagArray ; tagArray[0] = "Test" ; tagArray[1] = "New" ; steam_ugc_set_item_tags( updateHandle, tagArray) ; steam_ugc_set_item_preview (updateHandle, "promo.jpg") ; steam_ugc_set_item_content(updateHandle, "WorkshopContent1") ; requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2") ; Le code ci-dessus récupère l'ID du jeu, puis l'utilise avec l'ID du fichier publié précédemment stocké pour générer un identifiant de mise à jour pour l'élément. Ce gestionnaire est ensuite utilisé pour mettre à jour divers éléments d'information avant que la mise à jour ne soit envoyée aux serveurs de l'atelier. des serveurs.
