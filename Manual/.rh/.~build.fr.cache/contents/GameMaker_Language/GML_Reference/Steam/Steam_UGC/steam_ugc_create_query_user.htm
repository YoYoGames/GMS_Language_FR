steam_ugc_create_query_user Cette fonction peut être utilisée pour interroger la base de données UGC. La fonction utilise automatiquement l'identifiant par défaut de l'application, l'utilisateur, et suppose que la requête est effectuée par le consommateur (plutôt que par le créateur). La fonction exige que vous utilisiez les éléments suivants pour le type de données à interroger, le type d'élément à interroger et l'ordre dans lequel les éléments retournés seront triés, ainsi qu'un numéro de page - notez qu'une requête retournera un nombre maximum de 50 éléments. Type de liste Constante Description ugc_list_Published Voir la documentation deSteam. ugc_list_VotedOn - ugc_list_VotedUp - ugc_list_VotedDown - ugc_list_WillVoteLater - ugc_list_Favorite - ugc_list_Subscribed - ugc_list_UsedOrPlayed - ugc_list_Followed - Type de correspondance Constant Description ugc_match_Items Voir la documentation deSteam. ugc_match_Items_Mtx - ugc_match_Items_ReadyToUse - ugc_match_Collections - ugc_match_Artwork - ugc_match_Videos - ugc_match_Screenshots (captures d'écran) - ugc_match_AllGuides - ugc_match_WebGuides - ugc_match_Guides intégrés - ugc_match_UsableInGame - ugc_match_ControllerBindings (liens avec les contrôleurs) - Ordre de tri constant Description ugc_sortorder_CreationOrderDesc Voir la documentation deSteam. ugc_sortorder_CreationOrderAsc - ugc_sortorder_TitreAsc - ugc_sortorder_LastUpdatedDesc - ugc_sortorder_SubscriptionDateDesc - ugc_sortorder_VoteScoreDesc - ugc_sortorder_ForModeration - La fonction renvoie une valeur unique de poignée de requête qui doit être stockée dans une variable pour être utilisée dans les autres fonctions de requête. Notez que cette fonction ne fait que préparer la requête mais ne l'envoie pas réellement - pour cela, vous devez appeler la fonction steam_ugc_send_query.htm">steam_ugc_send_query() - et vous pouvez utiliser d'autres fonctions steam_ugc_query_* () pour affiner la demande de recherche avant qu'elle ne soit réellement envoyée. Syntaxe : steam_ugc_create_query_user(list_type, match_type, sort_order, page) ; Arguments Description Type de liste Le type de liste de données à créer (voir les constantes listées ci-dessous). type de correspondance Le type d'éléments UGC à interroger (voir les constantes énumérées ci-dessous). ordre de tri La manière dont les données doivent être ordonnées (voir les constantes énumérées ci-dessous). page Le numéro de la page à interroger. Les retours : Traitement des requêtes Exemple : query_handle = steam_ugc_create_query_user(ugc_list_Published, ugc_match_Items, ugc_sortorder_TitleAsc, 1) ; Le code ci-dessus crée une demande de requête et stocke son handle dans une variable pour une utilisation future.
