steam _ugc_create_query_user Cette fonction peut être utilisée pour interroger la base de données UGC. La fonction utilise automatiquement l'ID par défaut de l'application, de l'utilisateur et suppose que la requête est effectuée par le consommateur (plutôt que par le créateur). La fonction vous oblige à utiliser les constantes suivantes pour le type de données à interroger, le type d'élément à interroger et l'ordre dans lequel les éléments retournés seront triés, ainsi qu'un numéro de page - notez qu'une requête renverra un maximum nombre de 50 articles. Constante de type de liste Description ugc_list_Published Consultez la documentation Steam. ugc_list_VotedOn - ugc_list_VotedUp - ugc_list_VotedDown - ugc_list_WillVoteLater - ugc_list_Favorited - ugc_list_Subscribe - ugc_list_UsedOrPlayed - ugc_list_Followed - Constante de type de correspondance Description ugc_match_Items Consultez la documentation Steam. ugc_match_Items_Mtx - ugc_match_Items_ReadyToUse - ugc_match_Collections - ugc_match_Artwork - ugc_match_Videos - ugc_match_Screenshots - ugc_match_AllGuides - ugc_match_WebGuides - ugc_match_IntegratedGuides - ugc_match_UsableInGame - ugc_match_ControllerBindings - Ordre de tri constant Description ugc_sortorder_CreationOrderDesc Consultez la documentation Steam. ugc_sortorder_CreationOrderAsc - ugc_sortorder_TitleAsc - ugc_sortorder_LastUpdatedDesc - ugc_sortorder_SubscriptionDateDesc - ugc_sortorder_VoteScoreDesc - ugc_sortorder_ForModeration - La fonction renvoie une valeur de descripteur de requête unique qui doit être stockée dans une variable pour être utilisée dans les autres fonctions de requête. Notez que cette fonction prépare uniquement la requête mais ne l'envoie pas réellement - pour cela, vous devez appeler la fonction steam _ugc_send_query.htm ">steam_ugc_send_query() - et vous pouvez utiliser davantage steam_ugc_query_*() fonctions pour affiner la demande de recherche avant qu'elle ne soit réellement envoyée. Syntaxe:steam_ugc_create_query_user(list_type, match_type, sort_order, page); Argument Description list_type Le type de liste de données à créer (voir les constantes ci-dessous). match_type Le type d'éléments UGC à interroger (voir les constantes ci-dessous). sort_order La façon dont les données doivent être ordonnées (voir les constantes ci-dessous). page Numéro de page à interroger. Retour:Query Handle Exemple:query_handle = steam_ugc_create_query_user(ugc_list_Published, ugc_match_Items, ugc_sortorder_TitleAsc, 1); Le code ci-dessus crée une requête de requête et stocke son handle dans une variable pour une utilisation future.
