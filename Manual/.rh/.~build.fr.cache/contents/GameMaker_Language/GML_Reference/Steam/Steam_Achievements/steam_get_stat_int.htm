steam_get_stat_int Cette fonction vous permet d'obtenir la valeur d'une statistique entière signée spécifique. La statistique doit avoir été préalablement définie sur la page des comptes du panneau de contrôle Steamworkspour votre jeu et la chaîne transmise à la fonction doit correspondre à celle utilisée comme nom d'API sur le panneau de contrôle. correspondre à celle utilisée comme nom d'API sur le panneau de contrôle. Syntaxe : steam_get_stat_int(nom_de_stat) ; Arguments Description nom_statut Le nom de la statistique à obtenir (une chaîne de caractères). Les retours : Real Exemple : xp += 100 ; steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100) ; if steam_get_stat_int( "Total_XP") > 1000 { if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP") ; } Le code ci-dessus définit une statistique et vérifie ensuite la valeur finale pour décider de l'attribution d'une réussite ou non.
