steam_reset_all_stats Cette fonction vous permet de réinitialiser toutes les statistiques de l'utilisateur actuel à leurs valeurs par défaut (telles que définies dans le panneau de configuration Steamworksde votre jeu). Il est recommandé de n'utiliser cette fonction que comme outil de débogage lors du développement de votre jeu. votre jeu. Syntaxe : steam_reset_all_stats() ; Les retours : N/A Exemple : ini_open("Save.ini") ; if global.Version != ini_read_real("Data", "Version", 0) { ini_write_real("Data", "Version", global.Version) ; steam_reset_all_stats () ; } ini_close() ; Le code ci-dessus vérifie une valeur stockée dans un fichier ini par rapport à celle d'une variable globale et si elles sont différentes, il réinitialise les statistiques du jeu.
