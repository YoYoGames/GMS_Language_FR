steam_reset_all_stats_achievements Cette fonction vous permet de réinitialiser toutes les statistiques et réalisations de l'utilisateur actuel à leurs valeurs par défaut (telles que définies dans le panneau de configuration Steamworksde votre jeu). Il est recommandé d'utiliser cette fonction uniquement comme outil de débogage lors du développement de votre jeu. Syntaxe : steam_reset_all_stats_achievements() ; Les retours : N/A Exemple : ini_open("Save.ini") ; if global.Version != ini_read_real("Data", "Version", 0) { ini_write_real("Data", "Version", global.Version) ; steam_reset_all_stats_achievements() ; } ini_close() ; Le code ci-dessus vérifie une valeur stockée dans un fichier ini par rapport à celle d'une variable globale et si elles sont différentes, il réinitialise les statistiques et les réalisations du jeu.
