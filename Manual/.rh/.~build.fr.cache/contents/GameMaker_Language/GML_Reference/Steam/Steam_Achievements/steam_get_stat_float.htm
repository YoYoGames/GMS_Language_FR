steam_get_stat_float Cette fonction vous permet d'obtenir la valeur d'une statistique à virgule flottante spécifique. La statistique doit avoir été préalablement définie sur la page des comptes du panneau de contrôle Steamworkspour votre jeu et la chaîne passée à la fonction doit correspondre à celle utilisée comme nom d'API sur le panneau de contrôle. correspondre à celle utilisée comme nom d'API sur le panneau de contrôle. Syntaxe : steam_get_stat_float(nom_de_stat) ; Arguments Description nom_statut Le nom de la statistique à obtenir (une chaîne de caractères). Les retours : Real Exemple : var dist_pc = (dist / dist_max) * 100 ; if steam_get_stat_float("Travelled") < dist_pc { steam_set_stat_int("Travelled", dist_pc) ; } Le code ci-dessus calcule un pourcentage basé sur la variable de la distance parcourue "dist" et la distance maximale que vous pouvez parcourir "dist_max". Il interroge ensuite la valeur actuelle de la statistique "Travelled" et si elle est inférieure à la valeur calculée, il réinitialise la statistique. inférieure à la valeur calculée, il redéfinit la statistique.
