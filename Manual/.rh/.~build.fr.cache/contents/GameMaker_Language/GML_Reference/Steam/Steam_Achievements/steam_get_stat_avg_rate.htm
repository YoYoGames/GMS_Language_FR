steam _get_stat_avg_rate Avec cette fonction, vous pouvez obtenir la valeur d'une statistique moyenne spécifique. La statistique doit avoir été préalablement définie sur la Steam Works pour votre jeu et la chaîne transmise à la fonction doit correspondre à celle utilisée comme nom d'API sur le panneau de configuration. Syntaxe:steam_get_stat_avg_rate(stat_name); Argument Description stat_name Le nom de la statistique à obtenir (une chaîne). Retour:Real Exemple:var avg = steam_get_stat_avg_rate("PointsPerHour"); draw_text(8, 8, "PPH = " + string(avg); Le code ci-dessus obtient la valeur actuelle de la statistique moyenne "PointsPerHour" et la dessine à l'écran.
