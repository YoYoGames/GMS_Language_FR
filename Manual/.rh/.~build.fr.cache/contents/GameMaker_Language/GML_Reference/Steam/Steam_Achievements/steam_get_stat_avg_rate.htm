steam_get_stat_avg_rate Cette fonction vous permet d'obtenir la valeur d'une statistique moyenne spécifique. La statistique doit avoir été définie au préalable sur la page des comptes du panneau de configuration Steamworkspour votre jeu et la chaîne transmise à la fonction doit correspondre à celle utilisée comme nom d'API sur le panneau de contrôle. Syntaxe : steam_get_stat_avg_rate(stat_name) ; Arguments Description nom_statut Le nom de la statistique à obtenir (une chaîne de caractères). Les retours : Real Exemple : var avg = steam_get_stat_avg_rate("PointsPerHour") ; draw_text(8, 8, "PPH = " + string(avg) ; Le code ci-dessus récupère la valeur actuelle de la statistique moyenne "PointsParHeure" et l'affiche à l'écran.
