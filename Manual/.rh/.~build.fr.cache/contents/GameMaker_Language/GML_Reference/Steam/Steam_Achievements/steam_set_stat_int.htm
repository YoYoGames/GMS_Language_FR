steam_set_stat_int Cette fonction vous permet de définir une statistique spécifique à une nouvelle valeur entière signée. La statistique doit avoir été définie au préalable sur la page des comptes du panneau de configuration Steamworkspour votre jeu et la chaîne de caractères transmise à la fonction doit correspondre à celle utilisée comme nom d'API sur le panneau de contrôle. Vous pouvez par exemple utiliser cette fonction pour savoir combien de fois le joueur meurt ou pour suivre la progression vers un succès. Syntaxe : steam_set_stat_int(nom_de_stat, valeur) ; Arguments Description nom_statut Le nom de la statistique à définir (une chaîne de caractères). valeur La valeur (un entier 32bit) pour définir le statut. Les retours : N/A Exemple : xp += 100 ; steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100) ; if steam_get_stat_int( "Total_XP") > 1000 { if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP") ; } Le code ci-dessus définit une statistique et vérifie ensuite la valeur finale pour décider de l'attribution d'une réussite ou non.
