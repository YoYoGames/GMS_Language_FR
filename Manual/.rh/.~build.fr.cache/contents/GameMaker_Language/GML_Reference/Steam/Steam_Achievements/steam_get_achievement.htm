steam_get_achievement Avec cette fonction, vous pouvez vérifier l'API Steam pour voir si un succès spécifique a été attribué. La réalisation doit avoir été préalablement définie sur la page des comptes du panneau de configuration Steamworkspour votre jeu et la chaîne transmise à la fonction doit correspondre à celle utilisée comme nom d'API sur le panneau de configuration. Syntaxe : steam_get_achievement(ach_name) ; Arguments Description ach_name Le nom de la réalisation à obtenir (chaîne de caractères). Les retours : Booléen Exemple : si hp <= 0 { global.Deaths += 1 ; if global.Deaths == 10 { if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times") ; } } Le code ci-dessus récompensera le joueur d'un succès si la variable globale "Deaths" est égale à 10 et si le succès n'a pas déjà été attribué.
