steam _download_scores_around_user Cette fonction est utilisée pour récupérer les entrées du classement par rapport à l'entrée actuelle des utilisateurs. Le range_start est le nombre d'entrées à récupérer avant l'entrée de l'utilisateur actuel, et le paramètre range_end Le paramètre est le nombre d'entrées après l'entrée de l'utilisateur actuel, et l'entrée de l'utilisateur actuel est toujours incluse dans les résultats. Par exemple, si l'utilisateur actuel est le numéro 5 dans un classement donné, définir la plage de départ sur -2 et la plage de fin sur 2 renverra 5 entrées: 3 à 7. S'il n'y a pas assez d'entrées dans le classement avant ou après l'entrée de l'utilisateur, Steam ajustera la plage pour essayer de renvoyer le nombre d'entrées demandées. Par exemple, si l'utilisateur est n ° 1 dans le classement, le début est défini sur -2 et la fin est défini sur 2, Steam renverra les 5 premières entrées du classement. Syntaxe:steam_download_scores_around_user(lb_name, range_start, range_end); Argument Description lb_name Le nom du classement à partir duquel vous téléchargez les scores (une chaîne). range_start La position de départ dans le classement. range_end La position finale dans le classement. Retour:Real Exemple:score_get = steam_download_scores_around_user("Game Scores", -4, 5); Cela enverra une requête au Steam pour une plage de 10 scores du classement "Game Scores", centrée sur le joueur et stockera l' identifiant asynchrone de la requête dans la variable "score_get". Cela sera ensuite géré dans l' événement Steam Async, comme indiqué dans l'exemple étendu pour steam_download_scores().
