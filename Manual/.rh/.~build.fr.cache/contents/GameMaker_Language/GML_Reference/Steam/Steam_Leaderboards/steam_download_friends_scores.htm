steam_download_friends_scores Cette fonction vous permet de récupérer uniquement les scores du classement qui appartiennent aux personnes marquées comme "amis" dans le client Steam . Ainsi, si votre classement compte 200 entrées, et que 50 d'entre elles sont vos amis, cette fonction ne récupérera que ces 50 résultats. Le nom du classement est une chaîne définie lorsque vous avez créé le classement à l'aide de la fonction steam_create_leaderboard()et la fonction renvoie une valeur qui peut ensuite être utilisée pour identifier le rappel dans l'événement asynchroneSteam, ou elle renvoie -1 si elle a échoué. Syntaxe : steam_download_friends_scores(lb_name) ; Arguments Description lb_name Le nom du classement à partir duquel vous téléchargez les scores (une chaîne). Les retours : Real Exemple : score_get = steam_download_friends_scores("Game Scores") ; Cela enverra une requête au serveur Steam pour les scores des amis de l'utilisateur dans le classement donné et stockera l'identifiant asynchrone de la requête dans la variable "score_get". Cette requête sera ensuite traitée dans l'événement asynchroneSteam, comme indiqué dans l'exemple étendu de steam_download_scores().
