steam _download_friends_scores Avec cette fonction, vous ne pouvez récupérer que les scores du classement qui appartiennent aux personnes marquées comme "amis" dans le client Steam Donc, si votre classement compte 200 entrées et que 50 d'entre eux sont vos amis, cette fonction ne récupérera que ces 50 résultats. Le nom du classement est une chaîne qui a été définie lorsque vous avez créé le classement à l'aide de la fonction steam_create_leaderboard(), et la fonction renverra une valeur qui peut ensuite être utilisée pour identifier le rappel dans l' événement Steam Async, ou elle renverra -1 en cas d'échec. Syntaxe:steam_download_friends_scores(lb_name); Argument Description lb_name Le nom du classement à partir duquel vous téléchargez les scores (une chaîne). Retour:Real Exemple:score_get = steam_download_friends_scores("Game Scores"); Cela enverra une demande au Steam pour les scores des amis des utilisateurs du classement donné et stockera l' identifiant asynchrone de la demande dans la variable "score_get". Cela sera ensuite géré dans l' événement Steam Async, comme indiqué dans l'exemple étendu pour steam_download_scores().
