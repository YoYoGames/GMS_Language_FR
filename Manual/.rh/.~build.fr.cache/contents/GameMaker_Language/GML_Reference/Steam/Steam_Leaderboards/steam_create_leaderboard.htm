steam _create_leaderboard Avec cette fonction, vous pouvez créer un nouveau classement pour votre jeu. Le premier argument est une chaîne qui définit le nom de votre classement, et ce nom doit être utilisé dans tout autre appel de fonction lié au classement en cours de création. Vous pouvez ensuite définir la méthode de tri (voir les constantes répertoriées ci-dessous) ainsi que la manière dont les informations sont affichées (encore une fois, l'une des constantes répertoriées ci-dessous). La fonction renverra une valeur de -1 si elle ne parvient pas à créer le classement ou une valeur supérieure à -1 si elle a réussi. REMARQUE: Si vous avez déjà créé un classement avec le même nom (soit par code, soit via votre Steam pour le jeu), cette fonction sera ignorée! Constante de tri Description lb_sort_none Pas de tri. Les informations seront affichées «telles quelles». lb_sort_ascending Triez le classement par ordre croissant. lb_sort_descending Triez le classement par ordre décroissant. Afficher la constante Description lb_disp_none Afficher le classement "tel quel". lb_disp_numeric Afficher le classement sous forme d'affichage numérique. lb_disp_time_sec Afficher les valeurs du classement sous forme d'heures, la valeur de base étant en secondes. lb_disp_time_ms Afficher les valeurs du classement sous forme de fois, la valeur de base étant en milli-secondes. Syntaxe:steam_create_leaderboard(lb_name, sort_method, display_type); Argument Description lb_name Le nom du classement que vous créez (une chaîne). sort_method La méthode de tri des entrées du classement (voir les constantes ci-dessous). display_type La façon d'afficher le classement à l'utilisateur (voir les constantes ci-dessous). Retour:Real Exemple:steam_create_leaderboard("Game Times", lb_sort_ascending, lb_disp_time_sec); Le code ci-dessus créera un classement appelé "Game Times", et le définira pour afficher les résultats par ordre croissant et avec un affichage en secondes.
