steam_create_leaderboard Cette fonction vous permet de créer un nouveau classement pour votre jeu. Le premier argument est une chaîne de caractères qui définit le nom de votre classement. Ce nom doit être utilisé dans tous les autres appels de fonction relatifs au classement en cours de création. Vous pouvez ensuite ensuite définir la méthode de tri (voir les constantes listées ci-dessous) ainsi que la manière dont les informations sont affichées (encore une fois, une des constantes listées ci-dessous). La fonction renvoie la valeur -1 si elle ne parvient pas à créer le classement ou une valeur supérieure à -1 si elle a réussi. supérieure à -1 si elle a réussi. REMARQUE: Si vous avez déjà créé un classement portant le même nom (soit par le code, soit par votre page Steam pour le jeu), cette fonction sera ignorée ! Tri constant Description lb_sort_none Pas de tri. Les informations seront affichées "telles quelles". lb_tri_ascendant Trier le classement dans l'ordre croissant. lb_tri_décroissant Trier le classement par ordre décroissant. Affichage constant Description lb_disp_none Afficher le classement "tel quel". lb_disp_numeric Afficher le classement sous forme numérique. lb_disp_time_sec Afficher les valeurs du classement sous forme de temps, la valeur de base étant les secondes. lb_disp_time_ms Afficher les valeurs du classement sous forme de temps, la valeur de base étant les milli-secondes. Syntaxe : steam_create_leaderboard(lb_name, sort_method, display_type) ; Arguments Description lb_name Le nom du classement que vous créez (une chaîne). méthode de tri La méthode de tri des entrées du classement (voir les constantes énumérées ci-dessous). type d'affichage La façon d'afficher le classement à l'utilisateur (voir les constantes ci-dessous). Les retours : Real Exemple : steam_create_leaderboard("Game Times", lb_sort_ascending, lb_disp_time_sec) ; Le code ci-dessus va créer un classement appelé "Game Times", et le configurer pour qu'il affiche les résultats dans l'ordre croissant et avec un affichage en secondes.
