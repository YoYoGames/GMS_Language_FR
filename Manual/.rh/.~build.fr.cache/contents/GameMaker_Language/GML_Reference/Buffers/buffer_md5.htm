tampon_md5 En cryptographie, MD5 (Message-Digest algorithm 5) est une fonction de hachage cryptographique largement utilisée avec une valeur de hachage de 128 bits et a été employée dans une grande variété d'applications de sécurité. Elle est également couramment utilisée pour vérifier l'intégrité des données. Cette fonction prend les données d'entrée d'un tampon donné (ou d'une partie du tampon) et renvoie le hachage MD5 hexadécimal de 32 caractères qui est unique à ces données. De cette façon, vous pouvez générer une clé sécurisée qui peut être stockée et utilisée pour vérifier l'intégrité des informations envoyées à (ou reçues de) un serveur externe (par exemple). Lorsque vous appliquez cette fonction à des tampons, vous devez spécifier l'identifiant du tampon à utiliser, puis une valeur de décalage (en pourcentage). utiliser, puis une valeur de décalage (en octets) pour savoir où commencer, et enfin une taille (toujours en octets) pour la région à hacher. REMARQUE: MD5 n'est pas complètement sécurisé et peut être cassé. Voir cette page pour plus d'informations. Syntaxe : buffer_md5(buffer, offset, size) ; Arguments Description tampon L'index du tampon à utiliser. décalage La valeur du décalage des données. taille La taille du tampon. Les retours : Chaîne de caractères Exemple : check_string = buffer_md5(buff, 0, buffer_get_size(buff)) ; Le code ci-dessus va créer un hachage md5 pour les données complètes stockées dans le tampon indexé par la variable "buff", et stocker le hachage renvoyé dans la variable "check_string".
