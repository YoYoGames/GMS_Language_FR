buffer_md5 En cryptographie, MD5 (Message-Digest algorithme 5) est une fonction de hachage cryptographique largement utilisée avec une valeur de hachage de 128 bits et a été utilisée dans une grande variété d'applications de sécurité. Il est également couramment utilisé pour vérifier l'intégrité des données. Cette fonction prend les données d'entrée d'un tampon donné (ou une partie du tampon) et renvoie le hachage MD5 hexadécimal de 32 caractères qui est unique à ces données. De cette manière, vous pouvez générer une clé sécurisée qui peut être stockée et utilisée pour vérifier l'intégrité des informations envoyées (ou reçues depuis) un serveur externe (par exemple). Lorsque vous appliquez cela aux tampons à l'aide de cette fonction, vous devez spécifier l'ID de tampon à utiliser, puis une valeur de décalage (en octets) pour savoir par où commencer, puis une taille (à nouveau en octets) pour la région à hacher. Syntaxe:buffer_md5(buffer, offset, size); Argument Description buffer L'index du tampon à utiliser. offset La valeur de décalage des données. size La taille du tampon. Retour:String Exemple:check_string = buffer_md5(buff, 0, buffer_get_size(buff)); Le code ci-dessus créera un hachage md5 pour toutes les données stockées dans le tampon indexé par la variable "buff", et stockera le hachage retourné dans la variable "check_string".
