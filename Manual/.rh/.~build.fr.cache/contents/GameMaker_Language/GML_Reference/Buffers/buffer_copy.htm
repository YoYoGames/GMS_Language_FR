buffer_copy Cette fonction peut être utilisée pour copier un segment (ou la totalité) des données stockées dans un tampon vers un autre. Lorsque vous utilisez deux tampons et que vous copiez de l'un à l'autre, les deux tampons doivent avoir été créés au préalable à l'aide de la fonction buffer_create() (par exemple), et vous pouvez spécifier un décalage de données (en octets) pour le point de départ des données à copier à partir du tampon source par rapport au début du tampon, ainsi qu'un autre décalage de données pour définir la position où copier les données vers dans le tampon de destination. REMARQUE: Vous ne pouvez pas copier dans le même tampon. Syntaxe : buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset) ; Arguments Description src_buffer L'index du tampon à copier. src_offset Le décalage des données pour commencer la copie (en octets). taille La taille des données à copier (en octets). dest_buffer L'index du tampon à copier. dest_offset La position de décalage pour copier les données (en octets). Les retours : N/A Exemple : buffer_copy(buff1, 0, 2048, buff2, 2048) ; Le code ci-dessus copiera les données stockées dans le tampon indexé dans la variable "buff1", puis les collera dans le tampon indexé dans la variable "buff2", mais décalé de 2048 octets.
