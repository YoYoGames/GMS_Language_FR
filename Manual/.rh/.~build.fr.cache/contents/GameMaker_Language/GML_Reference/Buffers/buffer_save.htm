Sauvegarde de la mémoire tampon Cette fonction permet de sauvegarder le contenu d'un tampon dans un fichier, prêt à être relu en mémoire à l'aide de la fonction buffer_load() fonction. REMARQUE: Sous HTML5, le contenu du tampon sera enregistré sous forme de chaînes codées en base64 lorsque cette fonction est utilisée. Syntaxe : buffer_save(buffer, filename) ; Arguments Description tampon L'index du tampon à sauvegarder. nom de fichier Le nom du fichier à enregistrer sous. Les retours : N/A Exemple : buffer_save(buff, "Player_Save.sav") ; Sauvegarde le contenu actuel du tampon avec l'id stocké dans la variable "buff" dans un fichier.
