buffer_write Cette fonction peut être utilisée pour écrire des données dans un tampon créé précédemment. Les données que vous écrivez doivent être en accord avec l'argument "type" de cette fonction, ce qui signifie que vous ne pouvez pas essayer d'écrire une chaîne sous forme d'entier 16 bits non signé, par exemple. Les constantes suivantes peuvent être utilisées pour définir le type de données: Constant Description buffer_u8 Un entier 8 bits non signé. Il s'agit d'une valeur positive comprise entre 0 et 255. buffer_s8 Un entier signé de 8 bits. Cela peut être une valeur positive ou négative de -128 à 127 (0 est classé comme positif). buffer_u16 Un entier 16 bits non signé. Il s'agit d'une valeur positive comprise entre 0 et 65 535. buffer_s16 Un entier 16 bits signé. Cela peut être une valeur positive ou négative de -32 768 à 32 767 (0 est considéré comme positif). buffer_u32 Un entier 32 bits non signé. Il s'agit d'une valeur positive comprise entre 0 et 4 294 967 295. buffer_s32 Un entier 32 bits signé. Cela peut être une valeur positive ou négative de -2 147 483 648 à 2 147 483 647 (0 est considéré comme positif). buffer_u64 Un entier 64 bits non signé. buffer_f16 Un flotteur 16 bits. Il peut s'agir d'une valeur positive ou négative dans la même plage qu'un entier signé 16 bits. buffer_f32 Un flotteur 32 bits. Il peut s'agir d'une valeur positive ou négative dans la même plage qu'un entier signé de 32 bits. buffer_f64 Un flotteur 64 bits. buffer_bool Une valeur booléenne. Ne peut être que 1 ou 0 (true ou false ) buffer_string Une chaîne de toute taille, finalisée avec un caractère de fin nul. buffer_text Une chaîne de n'importe quelle taille, sans le dernier caractère de fin nul. La fonction renverra 0 si elle réussit ou -1 si elle échoue. Syntaxe:buffer_write(buffer, type, value) Argument Description buffer L'index du tampon dans lequel écrire. type Le type de données à écrire dans le tampon (voir la liste des constantes ci-dessus). value Les données à écrire. Retour:0 if success, or -1 if it fails Exemple:buffer_seek(buff, buffer_seek_start, 0); buffer_write(buff, buffer_s16, 0); buffer_write(buff, buffer_s16, x); buffer_write(buff, buffer_s16, y); Le code ci-dessus trouve le début du tampon avec l'ID stocké dans la variable "buff" et y écrit une série de valeurs entières 16 bits signées.
