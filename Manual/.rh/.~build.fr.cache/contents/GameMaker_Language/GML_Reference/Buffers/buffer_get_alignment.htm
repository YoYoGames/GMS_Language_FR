tampon_get_alignement Avec cette fonction, vous pouvez obtenir l'alignement d' un octet pour l'ID du tampon donné. Syntaxe : buffer_get_alignment(buffer) ; Arguments Description tampon L'index du tampon à vérifier. Les retours : Real Exemple : alignement = buffer_get_alignment(buff) ; Le code ci-dessus récupère l'alignement du tampon à partir de la valeur indexée dans la variable "buff" et le stocke dans une variable.
