buffer_get_alignment Avec cette fonction, vous pouvez obtenir l' alignement d'un octet pour l'ID de tampon donné. Syntaxe:buffer_get_alignment(buffer); Argument Description buffer L'index du tampon à vérifier. Retour:Real Exemple:alignment = buffer_get_alignment(buff); Le code ci-dessus obtiendra l'alignement du tampon à partir de la valeur indexée dans la variable "buff" et le stockera dans une variable.
