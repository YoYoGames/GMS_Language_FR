buffer_sha1 En cryptographie, SHA-1 est une fonction de hachage cryptographique conçue par la National Security Agency des États-Unis et est utilisée dans plusieurs applications et protocoles largement utilisés comme le populaire Git où il est utilisé pour vérifier les modifications de fichiers. Cette fonction prend les données d'entrée d'un tampon et renvoie un résumé de message de 160 bits au format ASCII. De cette manière, vous pouvez générer une clé sécurisée qui peut être stockée et utilisée pour vérifier l'intégrité des informations envoyées (ou reçues depuis) un serveur externe (par exemple). Lorsque vous appliquez cela aux tampons à l'aide de cette fonction, vous devez spécifier l'ID de tampon à utiliser, puis une valeur de décalage (en octets) pour savoir par où commencer, puis une taille (à nouveau en octets) pour la région à hacher. Syntaxe:buffer_sha1(buffer, offset, size); Argument Description buffer L'index du tampon à utiliser. offset La valeur de décalage des données. size La taille du tampon. Retour:String Exemple:check_string = buffer_sha1(buff, 0, buffer_get_size(buff)); Le code ci-dessus créera un hachage sha1 pour toutes les données stockées dans le tampon indexé par la variable "buff", et stockera le hachage retourné dans la variable "check_string".
