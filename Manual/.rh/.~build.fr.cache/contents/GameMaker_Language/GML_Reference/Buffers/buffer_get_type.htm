buffer_get_type Avec cette fonction, vous récupérez le type de tampon qui est référencé par une valeur d'ID de tampon. La valeur de retour sera l'une des constantes suivantes: Constant Description buffer_fixed Un tampon de taille fixe. buffer_grow Un tampon qui «grandira» dynamiquement à mesure que des données sont ajoutées buffer_wrap Un tampon dans lequel les données seront «enveloppées». Lorsque les données ajoutées atteignent la limite de la taille de la mémoire tampon, l'écrasement sera replacé au début de la mémoire tampon et une nouvelle écriture se poursuivra à partir de ce point. buffer_fast Tampon spécial "dépouillé" extrêmement rapide en lecture / écriture. Ne peut être utilisé qu'avec buffer_u8 types de données et doit être aligné sur 1 octet. buffer_vbuffer Un tampon de sommets Syntaxe:buffer_get_type(buffer); Argument Description buffer L'index du tampon à vérifier. Retour:Constant Exemple:type = buffer_get_type(buff); Le code ci-dessus obtiendra le type de tampon à partir de la valeur indexée dans la variable "buff" et le stockera dans une variable.
