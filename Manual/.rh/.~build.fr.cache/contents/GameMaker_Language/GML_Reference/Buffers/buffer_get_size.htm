buffer_get_size Avec cette fonction, vous pouvez obtenir la taille du tampon donné en octets. Syntaxe:buffer_get_size(index); Argument Description index L'index du tampon pour obtenir la taille de. Retour:Real Exemple:var _size = buffer_get_size(player_data); var _temp = buffer_create(_size, buffer_fixed, 0); Le code ci-dessus va créer un nouveau tampon et stocker son index dans la variable locale "_temp", la taille de ce nouveau tampon étant la même que celle du tampon indexé dans la variable "player_data".
