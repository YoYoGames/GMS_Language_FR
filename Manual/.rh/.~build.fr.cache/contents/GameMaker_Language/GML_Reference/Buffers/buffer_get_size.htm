tampon_get_taille Avec cette fonction, vous pouvez obtenir la taille du tampon donné en octets. Syntaxe : tampon_get_size(index) ; Arguments Description indice L'index du tampon dont on veut obtenir la taille. Les retours : Real Exemple : var _size = buffer_get_size(player_data) ; var _temp = buffer_create(_size, buffer_fixed, 0) ; Le code ci-dessus va créer un nouveau tampon et stocker son index dans la variable locale "_temp", la taille de ce nouveau tampon étant la même que celle du tampon indexé dans la variable "player_data".
