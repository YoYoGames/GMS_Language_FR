buffer_sizeof Cette fonction renverra la taille (en octets) de l'une des constantes de données données (répertoriées ici ). Syntaxe:buffer_sizeof(type); Argument Description type Le type de données à vérifier (voir la liste des constantes ici ). Retour:N/A Exemple:var b = 12 * buffer_sizeof(buffer_u8); buff = buffer_create(b, buffer_fixed, 1); Le code ci-dessus calcule d'abord la taille du tampon à créer en multipliant le type de données 8 bits non signé par 12 (puisque nous utiliserons le tampon pour contenir 12 éléments de données), puis utilise cette valeur pour définir un tampon fixe sur le bon Taille.
