Taille de la mémoire tampon Cette fonction renvoie la taille (en octets) de l'une des constantes de données données données (listées ici). Syntaxe : tampon_sizeof(type) ; Arguments Description type Le type de données qui doit être vérifié (voir la liste des constantes ici). Les retours : N/A Exemple : var b = 12 * buffer_sizeof(buffer_u8) ; buff = buffer_create(b, buffer_fixed, 1) ; Le code ci-dessus calcule d'abord la taille du tampon à créer en multipliant le type de données 8 bits non signées par 12 (puisque nous utiliserons le tampon pour contenir 12 données), puis il utilise cette valeur pour définir un tampon fixe à la taille correcte.
