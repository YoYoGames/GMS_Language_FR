buffer_base64_encode Cette fonction convertira les données du tampon donné en une chaîne codée au format base64. Il s'agit d'un schéma de codage couramment utilisé qui est souvent utilisé pour tout média devant être stocké ou transféré sur Internet sous forme de texte, et rend la sortie illisible à l'œil humain. Pour l'utiliser, vous devez spécifier un tampon déjà créé, la valeur de décalage (qui est le point dans le tampon auquel vous souhaitez commencer l'encodage) ainsi que la taille, en octets, de la mémoire tampon à encoder. Syntaxe:buffer_base64_encode(buffer, offset, size); Argument Description buffer L'index du tampon à utiliser. offset La valeur de décalage des données. size La taille du tampon. Retour:String Exemple:var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff)) Le code ci-dessus créera un encodage des données complètes stockées dans le tampon indexé par la variable "buff", et stockera la chaîne retournée dans la variable locale "b_str".
