buffer_load_ext Cette fonction va charger les données de la mémoire tampon qui ont été précédemment sauvegardées à l'aide des fonctions buffer_save() dans un tampon déjà créé. Vous donnez l'identifiant du tampon précédemment créé dans lequel vous voulez charger, puis le fichier de à charger, et enfin le décalage du début du tampon (en octets) vers lequel vous souhaitez charger les données. Syntaxe : buffer_load_ext(buffer, filename, offset) ; Arguments Description tampon L'index du tampon à charger. nom de fichier Le nom du fichier à charger. décalage Le décalage dans le tampon à charger (en octets). Les retours : N/A Exemple : var pos = buffer_seek(player_buffer, buffer_seek_end, 0) ; buffer_load(player_buffer, "Data_Save.sav", pos) ; Le code ci-dessus récupère d'abord la position de la fin de la mémoire tampon indexée dans la variable "player_buffer", puis charge les données de la mémoire donnée à cette position (notez que cet exemple ne fonctionne qu'avec les types de mémoire tampon "grow" ou "wrap"). ).
