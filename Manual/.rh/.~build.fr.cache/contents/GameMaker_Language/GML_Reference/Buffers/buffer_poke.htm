buffer_poke Avec le buffer_write(), vous pouvez écrire des données dans le tampon donné à la position actuelle de «recherche», chaque élément de données avançant cette position par les octets en cours d'écriture ou de lecture. Cependant, il peut être nécessaire pour vous de modifier une donnée donnée sans vouloir changer la position de recherche actuelle, et c'est à ce moment-là que vous utiliserez cette fonction. Il vous suffit de fournir à la fonction un index de tampon, puis la position de décalage à partir du début du tampon (en octets) dans ce tampon dans lequel écrire, ainsi que le type de données et la valeur à écrire. Syntaxe:buffer_poke(buffer, offset, type, value); Argument Description buffer L'index du tampon à utiliser. offset La position de décalage (en octets) dans la mémoire tampon dans laquelle écrire les données données. type Le type de données à écrire dans le tampon (voir la liste des constantes ici ). value Les données à ajouter au tampon, conformément au type spécifié. Retour:N/A Exemple:buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend)); Le code ci-dessus ajoutera la valeur de la composante bleue de la couleur utilisée pour le mélange d'image dans le tampon indexé dans la variable "buff", à la troisième position dans le tampon et en tant que valeur 8 bits non signée.
