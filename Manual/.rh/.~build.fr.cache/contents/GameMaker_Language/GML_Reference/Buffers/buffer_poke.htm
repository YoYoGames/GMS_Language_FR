tampon_poke Avec la fonction buffer_write() vous pouvez écrire des données dans le tampon donné à la position actuelle de "recherche", chaque donnée faisant avancer cette position des octets écrits ou lus. Cependant, il Cependant, il peut être nécessaire de modifier un élément de données donné sans vouloir changer la position de recherche actuelle, et c'est dans ce cas que vous utiliserez cette fonction. Il suffit de fournir à la fonction un index de tampon, puis la position de décalage par rapport au début du tampon (en octets) à l'intérieur du tampon. début du tampon (en octets) à l'intérieur de ce tampon, ainsi que le type de données et la valeur à écrire. Syntaxe : buffer_poke(buffer, offset, type, valeur) ; Arguments Description tampon L'index du tampon à utiliser. décalage La position de décalage (en octets) dans le tampon pour écrire les données données. type Le type de données qui doit être écrit dans le tampon (voir la liste des constantes ici). valeur Les données à ajouter au tampon, conformément au type spécifié. Les retours : N/A Exemple : buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend)) ; Le code ci-dessus ajoutera la valeur de la composante bleue de la couleur utilisée pour le mélange d'images dans le tampon indexé dans la variable "buff", à la troisième position dans le tampon et en tant que valeur 8 bits non signée.
