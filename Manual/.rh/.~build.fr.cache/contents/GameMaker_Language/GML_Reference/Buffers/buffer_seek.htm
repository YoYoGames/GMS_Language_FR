recherche de tampon Cette fonction peut être utilisée pour se déplacer dans un tampon, en trouvant le début, la fin, ou une position relative à celle qui a été utilisée en dernier lors de la lecture ou de l'écriture des données. La valeur "offset" est le décalage (en octets) à ajouter à la position de recherche donnée, par exemple, si la base est relative et le décalage est de 4, alors la position du tampon se déplacera de 4 octets à partir de sa position actuelle. Veuillez noter ce qui suit : Vous pouvez utiliser des valeurs négatives pour le décalage à rechercher dans le tampon, ainsi que des valeurs positives. Si le tampon est de type "wrap" et que vous décalez au-delà de la fin du tampon, la position de recherche sera également décalée. Si le tampon n'est pas de type "wrap", la recherche se fera au début ou à la fin du tampon, même si le décalage fait sortir la recherche des limites du tampon. Les constantes suivantes sont acceptées comme argument "de base" pour chercher à : Constant Description Début de recherche de tampon Le début du tampon tampon_recherche_relatif Une position relative à la position actuelle de lecture/écriture. fin de recherche de tampon La fin du tampon Syntaxe : buffer_seek(buffer, base, offset) ; Arguments Description tampon L'index du tampon à utiliser. base La position de base à rechercher. décalage La valeur du décalage des données. Les retours : N/A Exemple : buffer_seek(buff, buffer_seek_start, 0) ; buffer_write(buff, buffer_s16, 0) ; buffer_write(buff, buffer_s16, x) ; buffer_write(buff, buffer_s16, y) ; Le code ci-dessus trouve le début du tampon avec l'identifiant stocké dans la variable "buff" et y écrit une série de valeurs entières signées de 16 bits.
