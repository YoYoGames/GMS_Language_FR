buffer_seek Cette fonction peut être utilisée pour se déplacer dans un tampon, trouver le début, la fin ou une position par rapport à celle qui a été utilisée en dernier lors de la lecture ou de l'écriture de données. La valeur "offset" est le décalage (en octets) à ajouter à la position de recherche donnée, par exemple, si la base est relative et que le décalage est de 4, alors la position du tampon se déplacera le long de 4 octets à partir de sa position actuelle. Veuillez noter ce qui suit: Vous pouvez utiliser des valeurs négatives pour le décalage pour rechercher dans le tampon ainsi que des valeurs positives. Si le tampon est du type «wrap» et que vous dépassez la fin du tampon, la position de recherche sera également enveloppée. Si le tampon n'est pas du type "wrap", la recherche se bloquera au début ou à la fin du tampon, même lorsque le décalage prendrait la recherche en dehors des limites du tampon. Les constantes suivantes sont acceptées comme argument "de base" pour chercher à: Constant Description buffer_seek_start Le début du tampon buffer_seek_relative Une position par rapport à la position actuelle de lecture / écriture buffer_seek_end La fin du tampon Syntaxe:buffer_seek(buffer, base, offset); Argument Description buffer L'index du tampon à utiliser. base La position de base à rechercher. offset La valeur de décalage des données. Retour:N/A Exemple:buffer_seek(buff, buffer_seek_start, 0); buffer_write(buff, buffer_s16, 0); buffer_write(buff, buffer_s16, x); buffer_write(buff, buffer_s16, y); Le code ci-dessus trouve le début du tampon avec l'ID stocké dans la variable "buff" et y écrit une série de valeurs entières 16 bits signées.
