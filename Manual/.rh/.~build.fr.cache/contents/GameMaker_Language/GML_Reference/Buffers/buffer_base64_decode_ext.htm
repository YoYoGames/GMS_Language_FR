tampon_base64_decode_ext Avec cette fonction, vous pouvez décoder une chaîne de caractères encodée en base64 (créée à l'aide de la fonction tampon_base64_encode() ) dans un tampon. Contrairement à la fonction tampon_base64_decode(), cette fonction ne créera pas de tampon pour vous, mais vous devriez déjà avoir créé le tampon (voir buffer_create()), dont vous utiliserez l'identifiant avec cette fonction. Le "offset" est la position dans le tampon pour décoder la chaîne donnée (en octets). Syntaxe : buffer_base64_decode_ext(buffer, string, offset) ; Arguments Description tampon L'index du tampon dans lequel la chaîne de caractères doit être décodée. chaîne de caractères La chaîne encodée en base64 à décoder. décalage La valeur du décalage des données. Les retours : N/A Exemple : buff = buffer_create(16384, buffer_grow, 2) ; ini_open("Save.ini") ; var str = ini_read_string("Save", "Slot1", "") ; buffer_base64_decode_ext(buff, str, 0) ; ini_close() ; Le code ci-dessus va créer un tampon et stocker son identifiant unique dans la variable "buff", puis ouvrir un fichier ini et lire une chaîne de caractères dans la variable locale "str". Cette chaîne est ensuite décodée dans le tampon nouvellement créé avant de refermer le fichier ini. avant de refermer le fichier ini.
