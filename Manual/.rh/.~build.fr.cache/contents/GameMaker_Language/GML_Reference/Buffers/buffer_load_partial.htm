Chargement partiel de la mémoire tampon Cette fonction chargera certaines des données de la mémoire tampon qui ont été précédemment sauvegardées à l'aide de la méthode buffer_save() dans un tampon déjà créé. Vous indiquez l'identifiant du tampon précédemment créé dans lequel vous souhaitez charger, puis le fichier tampon sauvegardé à charger, et enfin le décalage par rapport au début du tampon (en octets) à partir duquel vous souhaitez charger les données. Les arguments suivants permettent de définir la longueur des données de la mémoire tampon (en octets) à partir du point de décalage initial que vous souhaitez charger et le point de décalage auquel charger les données dans la mémoire tampon (à nouveau, en octets). Syntaxe : buffer_load_partial(buffer, filename, offset, src_len, dest_offset) ; Arguments Description tampon L'index du tampon à charger. nom de fichier Le nom du fichier à charger. décalage Le décalage dans le tampon à charger (en octets). src_len La longueur de la partie du tampon à charger (en octets). dest_offset Le décalage où commencer à placer les données partielles dans le nouveau tampon (en octets). Les retours : Real Exemple : buff = buffer_create(256, buffer_grow, 1) ; var _file = "save.dat" ; var _so = 6 ; var _sl = 5 ; var _do= 0 ; buffer_load_partial(buff, _file, _so, _sl, _do) ; Le code ci-dessus va créer un nouveau tampon "grow" et y charger une partie des données enregistrées dans le fichier "save.dat".
