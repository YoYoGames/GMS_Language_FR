remplissage de la mémoire tampon Cette fonction peut être utilisée pour remplir un tampon précédemment créé avec un type de données et une valeur donnés. Les données avec lesquelles vous remplissez le tampon doivent être en accord avec l'argument "type" de cette fonction, ce qui signifie que vous ne pouvez pas essayer de remplir avec une chaîne de caractères et utiliser le type entier 16 bits non signé, par exemple. chaîne de caractères et utiliser le type "unsigned 16bit integer", par exemple. Les constantes de type sont les mêmes que celles utilisées par les fonctions buffer_read() et buffer_write() . Le paramètre "size est la taille du tampon (en octets) que vous souhaitez remplir, tandis que le décalage est la valeur du décalage (également en octets) à partir du début du tampon pour commencer le remplissage. Syntaxe : buffer_fill(buffer, offset, type, valeur, taille) ; Arguments Description tampon L'index du tampon à remplir. décalage La valeur du décalage des données (en octets). type Le type de données qui doit être écrit dans le tampon (voir la liste des constantes ici). valeur Les données à écrire. taille La taille du tampon (en octets) que vous souhaitez remplir. Les retours : N/A Exemple : map_buffer = buffer_create(16384, buffer_fixed, 0) ; buffer_fill(map_buffer, 0, buffer_u16, 0, 16384) ; Le code ci-dessus trouve le début du tampon avec l'identifiant stocké dans la variable "buff" et y écrit une série de valeurs entières signées de 16 bits.
