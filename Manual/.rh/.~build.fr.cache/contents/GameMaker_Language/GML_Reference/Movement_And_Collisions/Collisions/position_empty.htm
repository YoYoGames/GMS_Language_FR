position_vide Cette fonction vérifie si une position donnée entre en collision avec une instance ayant un masque de collision valide à la position donnée. Syntaxe : position_empty(x, y) ; Arguments Description x La position x à vérifier. y La position y à vérifier. Les retours : Booléen Exemple : var xx, yy ; xx = random(room_width) ; yy = random(room_height) ; if position_empty(xx, yy) { instance_create_layer(xx, yy, "Bullets", obj_Bomb) ; } Cela va vérifier une position aléatoire quelque part dans la pièce pour une collision et s'il n'y en a pas, cela va créer une instance de obj_Bomb.
