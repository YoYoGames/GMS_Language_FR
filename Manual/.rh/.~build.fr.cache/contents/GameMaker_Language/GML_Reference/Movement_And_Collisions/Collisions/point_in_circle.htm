point_in_circle Lorsque vous utilisez cette fonction, vous définissez une zone circulaire et GameMaker Studio 2 déterminera si le point donné se situe dans ses limites ou non. Si le point tombe dans le cercle défini, la fonction retournera true sinon la fonction retournera false. Syntaxe:point_in_circle(px, py, x1, y1, rad); Argument Description px La coordonnée x du point à vérifier. py La coordonnée y du point à vérifier. x1 La coordonnée x du centre du cercle. y1 La coordonnée y du centre du cercle. rad Le rayon du cercle. Retour:Boolean Exemple:if point_in_circle(mouse_x, mouse_y, x, y, 16)    {    over = true;    } else    {    over = false;    } Le code ci-dessus utilise le point_in_circle pour vérifier si la position de la souris se situe dans la zone circulaire définie, en définissant la variable "over" sur true si c'est le cas, ou false autrement.
