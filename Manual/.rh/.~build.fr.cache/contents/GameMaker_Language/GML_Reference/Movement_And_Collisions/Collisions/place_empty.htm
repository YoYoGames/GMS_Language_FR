place_vide Vous pouvez utiliser cette fonction pour vérifier si l'instance appelante entre en collision avec une autre instance d'un objet ou avec toutes les instances de votre jeu. Pour que cette collision soit résolue correctement, l'instance qui exécute le code doit avoir un masque de collision valide (soit pour le sprite lui-même, soit par le biais d'un masque de collision). valide (soit pour le sprite lui-même, soit par le biais de l'option mask_index) et elle n'enregistrera les collisions qu'avec les instances qui ont également un masque valide. La fonction teste s'il n'y a pas de collisions si l'instance appelante est placée à une position spécifique, et vous pouvez fournir un argument facultatif pour affiner la vérification et ne vérifier que si une position est exempte de collisions avec des instances du type donné. donné. Notez que si aucun ID d'objet optionnel n'est fourni, la vérification sera effectuée sur toutes les instances de la pièce. La vérification des collisions sera soit précise, soit basée sur la boîte englobante de l'instance, en fonction du type de masque de collision sélectionné. mais pour que les collisions précises fonctionnent correctement, toutes les instances de la vérification doivent avoir des masques de collision précis. Notez que les coordonnées x/y données seront ramenées à l'entier le plus proche avant que la vérification ne soit effectuée. Syntaxe : place_empty(x, y, [object_id]) ; Arguments Description x La position x à vérifier. y La position y à vérifier. [objet_id] Facultatif ! L'objet à vérifier. Les retours : Booléen Exemple : if place_empty(mouse_x, mouse_y, obj_Enemy) { x = mouse_x ; y = mouse_y ; } Le code ci-dessus vérifiera la présence d'une collision avec toute autre instance de l'objet "obj_Enemy", comme si l'instance appelante devait être placée à la même position que la souris. Si aucune collision n'est détectée, les coordonnées x/y de l'instance sont alors à celles de la souris. Retour : Collisions Suivant : place_free Copyright YoYo Games Ltd. 2021 Tous droits réservés
