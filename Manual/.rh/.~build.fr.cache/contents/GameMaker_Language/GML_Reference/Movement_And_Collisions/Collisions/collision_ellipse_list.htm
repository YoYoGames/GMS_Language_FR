collision_ellipse_list Cette fonction est la même que la collision_ellipse(), mais au lieu de détecter une seule instance en collision à la fois, il détectera plusieurs instances. Vous fournissez les x / positions y du haut à gauche et en bas à droite de la zone elliptique pour vérifier avec l'objet de vérifier, et vous pouvez définir le contrôle pour être précis (dans ce cas, toutes les instances en cours de vérification devront avoir précises masques de collision) et si la vérification doit inclure ou non l'instance appelante. Vous fournissez également une liste DS qui sera remplie avec l'unique id valeurs de toutes les instances de l'objet considérées comme en collision avec l'instance appelante, et vous avez la possibilité de classer la liste en fonction de la distance entre le centre de la zone elliptique et l'origine des instances trouvées en collision. La fonction renvoie le nombre d'instances trouvées, ou 0 si aucune n'est trouvée. Notez qu'au lieu d'un index d'objet, vous pouvez fournir le mot-clé instanceall, pour rechercher toutes les instances de la salle actuelle, qui peuvent inclure l'instance exécutant le code (en fonction de la notme argument). Syntaxe:collision_ellipse_list(x1, y1, x2, y2, obj, prec, notme, list, ordered); Argument Description x1 La coordonnée x du côté gauche de l'ellipse à vérifier. y1 Coordonnée y du côté supérieur de l'ellipse à vérifier. x2 La coordonnée x du côté droit de l'ellipse à vérifier. y2 La coordonnée y du côté inférieur de l'ellipse à vérifier. obj L'objet à vérifier par exemple les collisions. prec Si la vérification est basée sur des collisions précises (true, qui est plus lent) ou sa boîte englobante en général (false, plus rapide). notme Si l'instance appelante, le cas échéant, doit être exclue (true ) ou non (false ). list Liste DS à utiliser pour stocker les ID des instances en collision. ordered Si la liste doit être triée par distance (true ) ou non (false ). Retour:Int (The number of instances found to be in collision) Exemple:var _list = ds_list_create(); var _num = collision_ellipse_list(x - 100, y - 100, x + 100, y + 100, obj_Enemy, false, true, _list, false); if _num > 0     {     for (var i = 0; i < _num; ++i;)         {         instance_destroy(_list[| i]);         }     } ds_list_destroy(_list); Le code ci-dessus vérifiera une zone elliptique de 100 pixels autour de la position de l'instance appelante pour les collisions avec les instances de "obj_Enemy". S'il y a des collisions, la liste pré-créée est bouclée et chaque instance qui était dans la collision est détruite.
