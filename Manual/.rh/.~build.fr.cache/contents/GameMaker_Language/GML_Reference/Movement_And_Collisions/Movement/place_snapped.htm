place_snapped Avec cette fonction, vous pouvez vérifier et voir si l'origine d'une instance (ses positions x et y) est alignée sur une grille avec les valeurs hsnap et vsnap que vous avez spécifiées. Syntaxe:place_snapped(hsnap, vsnap); Argument Description hsnap L'accrochage horizontal à vérifier. vsnap L'accrochage vertical à vérifier. Retour:Boolean Exemple:with (obj_Pieces)     {     if !place_snapped(32, 32)         {         move_snap(32, 32);         }     } Le code ci-dessus vérifie toutes les instances de "obj_Pieces" pour voir si elles sont accrochées à une grille de 32x32 pixels, et si elles ne le sont pas, il les accroche à la position la plus proche dans cette grille.
