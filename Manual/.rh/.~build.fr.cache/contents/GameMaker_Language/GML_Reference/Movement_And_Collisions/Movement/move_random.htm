déplacer_aléatoire Avec cette fonction, vous pouvez placer l'instance à une position quelconque dans la pièce, mais alignée sur une grille "invisible". Ainsi, une valeur de 32 pour le hsnap et le vsnap placera l'instance à une position aléatoire qui est alignée sur une grille de 32x32 carrés. Vous pouvez définir ces valeurs à 1 pour obtenir une position n'importe où dans la pièce. REMARQUE: cette fonction se positionnera aux mêmes endroits à chaque nouvelle exécution du jeu, car GameMaker Studio 2 génère à chaque fois la même graine aléatoire initiale pour faciliter le débogage du code. Pour éviter ce comportement utilisez randomise() au début de votre jeu. Ceci n'est vrai que pour tester et déboguer le jeu, car l'exécutable final ne présentera pas ce comportement. ne présentera pas ce comportement et sera aléatoire à chaque partie. Syntaxe : move_random(hsnap, vsnap) ; Arguments Description hsnap L'accrochage horizontal (la taille en pixels entre les "cellules"). vsnap L'accrochage vertical (la taille en pixels entre les "cellules"). Les retours : N/A Exemple : if keyboard_check(vk_space) { move_random(1,1) ; } Ceci déplacera l'instance à une position aléatoire n'importe où dans la pièce lorsque la touche espace est pressée.
