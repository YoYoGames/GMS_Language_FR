move_bounce_solid Avec cette fonction, vous pouvez commander à une instance de ne rebondir que sur les instances marquées comme solides dans la pièce. Vous pouvez également lui dire d'utiliser la vérification de collision précise lorsqu'elle est activée, mais sachez que cela nécessite que toutes les instances aient des masques précis activés et ralentira considérablement votre jeu lorsque de nombreuses instances sont impliquées en raison de la quantité de traitement à effectuer. Cela devrait normalement aller dans l'événement d'étape d'une instance, mais peut également être utilisé de manière sélective dans un événement de collision. Syntaxe:move_bounce_solid(adv); Argument Description adv Activer le rebond avancé (vrai) ou non (faux). Retour:N/A Exemple:move_bounce_solid(false); Cela permettra le rebond non précis des instances marquées comme "solides".
