move_bounce_solid Avec cette fonction, vous pouvez commander à une instance de ne rebondir que sur les instances marquées comme solides dans la room. Vous pouvez également lui demander d'utiliser un contrôle de collision précis lorsqu'il est activé, mais sachez que cela nécessite que toutes les instances aient des masques précis activés. et ralentira considérablement votre jeu lorsque de nombreuses instances sont impliquées, en raison de la quantité de traitement à effectuer. Ceci devrait normalement aller dans l'événement step d'une instance, mais peut aussi être utilisé sélectivement dans un événement de collision. Syntaxe : move_bounce_solid(adv) ; Arguments Description adv Activation (true) ou non (false) du rebondissement avancé. Les retours : N/A Exemple : move_bounce_solid(false) ; Cela permettra un rebondissement non précis sur les instances marquées comme "solides".
