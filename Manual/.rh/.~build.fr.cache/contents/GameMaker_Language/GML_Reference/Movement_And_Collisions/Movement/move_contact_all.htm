déplacer_contact_tout Cette fonction déplacera l'instance exécutant le code d'un nombre défini de pixels dans la direction spécifiée jusqu'à ce qu'elle rencontre une autre instance avec un masque valide. Vous pouvez utiliser -1 ou 0 pour la distance maximale, qui est par défaut de 1000 pixels, c'est-à-dire que GameMaker Studio 2 déplacera l'instance continuellement de 1000 pixels jusqu'à ce qu'elle ne soit plus en collision. Syntaxe : move_contact_all(dir, maxdist) ; Arguments Description dir La direction à prendre. maxdist La distance maximale que l'objet peut parcourir (-1 ou 0 une valeur par défaut de 1000 pixels). Les retours : N/A Exemple : if !place_meeting(x, y + 1, all) { move_contact_all(270, -1) ; } Le code ci-dessus vérifiera sous l'instance s'il y a une collision, et s'il n'y en a pas, il la déplacera vers le bas jusqu'à ce qu'il y en ait une ou jusqu'à ce que 1000 pixels aient été couverts.
