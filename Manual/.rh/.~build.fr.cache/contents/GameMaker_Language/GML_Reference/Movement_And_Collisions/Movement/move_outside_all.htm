move_outside_all Avec cette fonction, vous pouvez dire à une instance de sortir d'une collision dans n'importe quelle direction et n'importe quel nombre de pixels à chaque étape, avec une valeur de -1 ou 0 pour maxdist étant une valeur par défaut de 1000px, c'est-à-dire: GameMaker Studio 2 déplacera l'instance jusqu'à 1000 pixels jusqu'à ce qu'il soit hors de collision. Syntaxe:move_outside_all(dir, maxdist); Argument Description dir La direction à prendre. maxdist La distance maximale que l'objet peut parcourir (-1 ou 0 une valeur par défaut de 1000 pixels). Retour:N/A Exemple:if place_meeting(x, y, all)    {    move_outside_all(90, 1);    } Le code ci-dessus déplacera l'instance d'un pixel vers le haut à chaque étape où elle est en collision avec toute autre instance.
