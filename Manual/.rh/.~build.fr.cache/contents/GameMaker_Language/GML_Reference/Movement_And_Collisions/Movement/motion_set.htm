motion_set Cette fonction définit une nouvelle direction de mouvement et une nouvelle vitesse pour l'instance exécutant le code. Notez que cela n'ajoute pas à la vitesse et à la direction actuelles des instances (pour cela, vous utiliseriez motion_add() ) mais le force plutôt aux nouveaux paramètres. Syntaxe:motion_set(dir, speed); Argument Description dir La nouvelle direction. speed La nouvelle vitesse. » Retour:N/A Exemple:if irandom(9) = 1     {     motion_set(random(360), 1 + random(3));     } Ce code ci-dessus fera changer la vitesse et la direction de l'instance à des intervalles aléatoires.
