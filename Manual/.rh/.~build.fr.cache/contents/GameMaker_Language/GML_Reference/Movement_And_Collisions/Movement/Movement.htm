Mouvement Dans n'importe quel jeu, le mouvement et la position sont d'une importance primordiale et GameMaker Studio 2 dispose donc d'une sélection complète de fonctions pour faire face à chaque situation. Les deux principales façons de déplacer une instance sont soit de définir la position réelle, soit de définir un vecteur vitesse / direction, et cela peut être fait soit en utilisant les variables d'instance de variables intégrées, soit en utilisant des fonctions de mouvement spécifiques. Ces deux options sont expliquées dans les sections ci-dessous: Les variables suivantes sont toutes incluses "intégrées" à chaque instance et peuvent être lues et modifiées pour changer différents comportements lorsque l'instance se déplace (elles affecteront les vecteurs de vitesse et de direction): directionfrictiongravitygravity_directionhspeedvspeedspeed Outre les variables mentionnées ci-dessus, vous pouvez également définir la position de l'instance directement à l'aide de ce qui suit: xyxpreviousypreviousxstartystart Les fonctions suivantes peuvent toutes être utilisées pour déplacer une instance d'une certaine manière, certaines affectant les vecteurs vitesse / direction et d'autres affectant directement la position x / y réelle dans la pièce: motion_addmotion_setmove_towards_pointmove_bounce_allmove_bounce_solidmove_contact_allmove_contact_solidmove_outside_allmove_outside_solidmove_randommove_snapmove_wrapplace_snapped Retour: Mouvement et collisions Suivant: Planification de mouvement © Copyright YoYo Games Ltd. 2021 Tous droits réservés
