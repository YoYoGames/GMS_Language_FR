déplacer_extérieur_solide Avec cette fonction, vous pouvez demander à une instance de sortir d'une collision avec n'importe quelle instance marquée comme étant solide dans n'importe quelle direction et n'importe quel nombre de pixels à chaque pas, avec une valeur de -1 ou 0 pour le maxdist. de -1 ou 0 pour la distance maximale, soit 1000 pixels par défaut, c'est-à-dire que GameMaker Studio 2 déplacera l'instance continuellement de 1000 pixels jusqu'à ce qu'elle soit hors collision. Syntaxe : move_outside_solid(dir, maxdist) ; Arguments Description dir La direction à prendre. maxdist La distance maximale que l'objet peut parcourir (-1 ou 0 une valeur par défaut de 1000 pixels). Les retours : N/A Exemple : if other.solid { var pdir ; pdir = point_direction(other.x, other.y, x, y) ; move_outside_solid(pdir, -1) ; } Le code ci-dessus se place dans un événement de collision et vérifie si l'"autre" instance de la collision est marquée comme "solide". Si c'est le cas, il déplace l'instance hors de la collision. Retour : Mouvement Suivant : move_random Copyright YoYo Games Ltd. 2021 Tous droits réservés
