déplacer_contact_solide Cette fonction déplacera l'instance qui exécute le code d'un nombre déterminé de pixels dans la direction spécifiée jusqu'à ce qu'elle rencontre une instance dont l'indicateur est solide marqué comme vrai. Vous pouvez Vous pouvez utiliser -1 ou 0 pour la distance maximale, qui est de 1000 pixels par défaut, c'est-à-dire que GameMaker Studio 2 déplacera l'instance continuellement de 1000 pixels jusqu'à ce qu'elle ne soit plus en collision. Syntaxe : move_contact_solid( dir, maxdist ) ; Arguments Description dir La direction à prendre. maxdist La distance maximale que l'objet peut parcourir (-1 ou 0 une valeur par défaut de 1000 pixels). Les retours : N/A Exemple : if !place_meeting(x, y + 1, obj_Floor) { move_contact_solid(270, -1) ; } Le code ci-dessus vérifiera sous l'instance s'il y a une collision avec "obj_Floor", et s'il n'y en a pas, il la déplacera vers le bas jusqu'à ce qu'il y ait une collision, ou que 1000 pixels aient été couverts.
