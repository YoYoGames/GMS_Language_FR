mp_linear_step_object Avec cette fonction, vous demandez à une instance de faire un "pas" vers un point spécifique, spécifié par les valeurs xgoal et ygoal. La taille du pas (c'est-à-dire le nombre de pixels que l'instance doit déplacer à chaque pas) est indiquée par stepsize, et si l'instance est déjà à la position voulue, elle s'arrêtera et ne se déplacera pas plus loin, contrairement à d'autres fonctions plus simples telles que move_towards_point(). Le stepsize est aussi la distance "en avant" que l'objet vérifiera à chaque pas pour s'assurer qu'il n'y a pas d'erreur. que l'objet vérifiera à chaque pas pour une collision, et vous pouvez choisir si l'instance s'arrête à une collision avec un objet ou une instance de votre choix. Syntaxe : mp_linear_step_object(xgoal, ygoal, stepsize, obj) ; Arguments Description xgoal La position x de la cible. ygoal La position y de la cible. taille d'étape La vitesse de déplacement de l'instance en pixels par pas. obj. L'objet qui doit bloquer le chemin. Il peut s'agir d'un index d'objet, d'un id d'instance ou du mot-clé spécial, tous Les retours : Booléen Exemple : if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall) { instance_create_layer(x, y, "Effects", obj_Explosion) ; instance_destroy() ; } Le code ci-dessus va faire en sorte que l'objet se déplace vers la souris à une vitesse de 5 pixels par pas, en vérifiant uniquement les collisions avec l'objet "obj_Wall". Une fois qu'il aura atteint la position de la souris, il créera un objet "obj_Explosion" et se détruira. se détruire.
