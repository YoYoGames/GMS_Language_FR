mp_linear_step Avec cette fonction, vous demandez à une instance de faire un "pas" vers un point spécifique, spécifié par les valeurs xgoal et ygoal. La taille du pas (c'est-à-dire le nombre de pixels que l'instance doit déplacer à chaque pas) est indiquée par la valeur stepsize, et si l'instance est déjà à la position voulue, elle s'arrêtera et ne se déplacera pas plus loin, contrairement à d'autres fonctions plus simples telles que move_towards_point(). Le stepsize est aussi la distance "en avant". que l'objet vérifiera à chaque étape pour une collision, et vous pouvez choisir si l'instance s'arrête à une collision avec n'importe quelle instance ou seulement celles qui sont marquées comme solides. La fonction retournera si elle a atteint la position cible (true) ou si elle a échoué (false). REMARQUE: cette fonction n'essaie pas de faire des détours si elle rencontre un obstacle, elle échoue simplement et arrête de se déplacer. Syntaxe : mp_linear_step(xgoal, ygoal, stepsize, checkall) ; Arguments Description xgoal La position x de la cible. ygoal La position y de la cible. taille d'étape La vitesse de déplacement de l'instance en pixels par pas. vérifier tout Si l'on veut vérifier toutes les instances (true) ou seulement celles qui sont solides (false). Les retours : Booléen Exemple : if mp_linear_step(mouse_x, mouse_y, 5, 0) { instance_create_layer(x, y, "Effects", obj_Explosion) ; instance_destroy() ; } Le code ci-dessus fera en sorte que l'objet se déplace vers la souris à une vitesse de 5 pixels par pas. Une fois qu'il aura atteint la position de la souris, il créera un objet "obj_Explosion" et se détruira.
