mp_grid_destroy Vous pouvez utiliser cette fonction pour détruire la mp_grid indiquée (telle que définie par la fonction mp_grid_create()) et libérer la mémoire qui lui est allouée. Il est essentiel d'appeler cette fonction chaque fois que la grille MP est terminée, sinon vous risquez d'obtenir une fuite de mémoire, ce qui signifie que votre jeu ralentira au fil du temps et finira par planter. REMARQUE: L'utilisation des fonctions mp_grid_* sur une grille après qu'elle ait été détruite entraînera une erreur. Syntaxe : mp_grid_destroy(id) ; Arguments Description id Index de la mp_grid qui doit être détruite Les retours : N/A Exemple : si timer = 0 { mp_grid_destroy(grid) ; room_goto(rm_Menu) ; } Le code ci-dessus va détruire la mp_grid indexée dans la variable "grid" si la variable "timer" est égale à 0, puis passer à une autre pièce.
