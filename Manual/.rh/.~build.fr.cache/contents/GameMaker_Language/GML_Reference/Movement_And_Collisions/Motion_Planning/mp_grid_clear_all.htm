mp_grid_clear_all Avec cette fonction, vous pouvez effacer une grille MP de toutes les cellules "interdites". Syntaxe:mp_grid_clear_all(id); Argument Description id Index du mp_grid à utiliser Retour:Boolean Exemple:if !instance_exists(obj_Player)     {     mp_grid_clear_all(grid);     } Le code ci-dessus effacera le mp_grid indexé dans la variable "grid", marquant toutes les cellules comme libres, si une instance de l'objet "obj_Player" n'existe plus dans la salle.
