mp_grid_clear_all Cette fonction permet de vider une grille MP de toutes les cellules "interdites". Syntaxe : mp_grid_clear_all(id) ; Arguments Description id Index de la grille mp_ qui doit être utilisée Les retours : Booléen Exemple : if !instance_exists(obj_Player) { mp_grid_clear_all(grid) ; } Le code ci-dessus effacera la grille mp_grid indexée dans la variable "grid", en marquant toutes les cellules comme libres, si une instance de l'objet "obj_Player" n'existe plus dans la pièce.
