mp_grid_get_cell Avec cette fonction, vous pouvez vérifier n'importe quelle cellule donnée de mp_grid pour voir si elle a été marquée comme occupée ou non, en donnant l'index du mp_grid (précédemment créé) et les coordonnées x et y de la cellule à vérifier. Si elle a été occupée ou si la position vérifiée est en dehors des limites de la grille, la fonction renverra -1 sinon elle renverra 0. Syntaxe:mp_grid_get_cell(id, x , y); Argument Description id Index du mp_grid à utiliser x1 La coordonnée x de la grille à vérifier. y1 La coordonnée y de la grille à vérifier. Retour:Real Exemple:if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1     {     image_blend = c_red;     } else     {     image_blend = c_lime;     } Le code ci-dessus vérifiera la cellule mp_grid qui correspond à la position de la souris et si elle est occupée, il définit le image_blend variable en rouge, et si elle n'est pas occupée, elle la met au vert.
