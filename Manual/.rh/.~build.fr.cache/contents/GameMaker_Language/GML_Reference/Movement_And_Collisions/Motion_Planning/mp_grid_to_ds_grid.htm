mp_grid_to_ds_grid Avec cette fonction, vous pouvez copier la grille MP donnée dans une grille DS. La grille DS doit être de la même taille que la grille MP, bien qu'elle ne doive pas nécessairement l'être (des données seront perdues si elle est plus petite, et si elle est plus grande, toutes les cellules supplémentaires de la grille seront à 0). Les cellules de la grille DS contiendront la valeur -1 si la cellule de la grille MP a été marquée comme étant occupée, ou 0 dans le cas contraire. Syntaxe : mp_grid_to_ds_grid(source, destination) ; Arguments Description source Index de la grille mp_ qui doit être utilisée destination Index de la grille ds_grid qui doit être utilisée pour copier les données de la grille. Les retours : Booléen Exemple : motion_grid = ds_grid_create(room_width / 32, room_height / 32) ; mp_grid_to_ds_grid(mp_grid, motion_grid) ; Le code ci-dessus va créer une nouvelle grille DS et ensuite copier les données de la grille MP contenues dans la variable "mp_grid" dans la nouvelle grille DS.
