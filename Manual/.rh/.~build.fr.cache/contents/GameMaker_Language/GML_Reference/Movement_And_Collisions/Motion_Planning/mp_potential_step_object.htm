mp_potential_step_object Cette fonction permet à l'instance de faire un pas vers une position particulière définie par xgoal/ygoal, tout en essayant d'éviter les obstacles. Lorsque l'instance rencontrera une instance de l'objet spécifié par l'argument "obj", elle changera la direction du mouvement pour essayer d'éviter cette instance et de la contourner. la direction du mouvement pour essayer d'éviter cette instance et de la contourner. Cette approche n'est pas garantie, mais dans la plupart des cas, elle permettra à l'instance de se déplacer vers le but. La fonction retourne si le but a été atteint ou non. Syntaxe : mp_potential_step_object(xgoal, ygoal, stepsize, obj) Arguments Description xgoal La position x de la cible. ygoal La position y de la cible. taille d'étape La vitesse de déplacement de l'instance en pixels par pas. obj. L'objet qui doit bloquer le chemin de l'instance qui exécute la fonction. Peut être un index d'objet, un identifiant d'instance ou le mot-clé spécial,all. Les retours : Booléen Exemple : if instance_exists(obj_Enemy) { var inst ; inst = instance_nearest(x, y, obj_Enemy) ; mp_potential_step_object(inst.x, inst.y, 5, obj_Wall) ; } Le code ci-dessus vérifie d'abord s'il y a des instances de "obj_Enemy" dans la pièce. S'il y en a, il trouve l'instance la plus proche et stocke son identifiant dans une variable. Cette variable est ensuite utilisée pour indiquer à mp_potential_step_object de déplacer l'instance avec le code vers la position x/y de l'objet qui a été trouvé à une vitesse de 5 pixels par pas tout en évitant seulement les instances de l'objet "obj_Wall".
