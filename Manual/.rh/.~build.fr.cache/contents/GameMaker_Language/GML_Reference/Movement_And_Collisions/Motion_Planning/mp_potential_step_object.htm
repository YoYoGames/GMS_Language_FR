mp_potential_step_object Cette fonction permet à l'instance de faire un pas vers une position particulière définie par xgoal / ygoal, tout en essayant d'éviter les obstacles. Lorsque l'instance se heurte à une instance de l'objet spécifié par l'argument "obj", elle change la direction du mouvement pour essayer d'éviter cette instance et se déplacer autour d'elle. Il n'est pas garanti que cette approche fonctionne, mais dans la plupart des cas simples, elle déplacera efficacement l'instance vers l'objectif. La fonction renvoie si l'objectif a été atteint ou non. Syntaxe:mp_potential_step_object(xgoal, ygoal, stepsize, obj) Argument Description xgoal La position x cible. ygoal La position y cible. stepsize Vitesse de déplacement de l'instance en pixels par étape. obj L'objet qui doit bloquer le chemin de l'instance exécutant la fonction. Peut être un index d'objet, un identifiant d'instance ou le mot-clé spécial, tout. Retour:Boolean Exemple:if instance_exists(obj_Enemy)     {     var inst;     inst = instance_nearest(x, y, obj_Enemy);     mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);     } Le code ci-dessus vérifie d'abord s'il y a des instances de "obj_Enemy" dans la salle. S'il y en a, il trouve le plus proche et stocke son identifiant dans une variable. Cette variable est ensuite utilisée pour dire mp_potential_step_object pour déplacer l'instance avec le code vers la position x / y de l'objet trouvé à une vitesse de 5 pixels par pas en évitant uniquement les instances de l'objet "obj_Wall".
