mp_linear_path Cette fonction calcule un chemin en ligne droite de la position actuelle de l'instance au point spécifié par les valeurs "xgoal" et "ygoal". Elle utilise la taille de pas indiquée, la même que dans la fonction mp_linear_step(). Le chemin indiqué doit déjà exister et sera écrasé par le nouveau chemin. La fonction retournera si un chemin complet a été trouvé (true) ou non (false). Si false est retourné, un chemin est tout de même généré, mais il n'ira que jusqu'à la position où le chemin a été bloqué. où le chemin a été bloqué. NOTE: Cette fonction ne déplace pas l'instance. Elle définit uniquement un chemin, et vous devez utiliser les fonctions Path pour le déplacement. Syntaxe : mp_linear_path(path, xgoal, ygoal, stepsize, checkall) ; Arguments Description chemin L'index du chemin à utiliser. xgoal La position x de la cible. ygoal La position y de la cible. taille d'étape La vitesse de déplacement de l'instance en pixels par pas. vérifier tout Si vous voulez vérifier les collisions avec toutes les instances (true) ou seulement les solides (false). Les retours : Booléen Exemple : if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0) { path_start(path, 2, 0, 0) ; } Cela permet à l'objet de vérifier s'il existe un chemin linéaire entre sa position actuelle et la position de "obj_Player".
