mp_grid_clear_rectangle Avec cette fonction, vous pouvez définir une zone en coordonnées de pièce qui effacera ensuite les cellules correspondantes dans la grille MP spécifiée. Même si une cellule tombe partiellement dans la zone rectangulaire définie, elle sera effacée. Syntaxe:mp_grid_clear_rectangle(id, x1, y1, x2, y2); Argument Description id Index du mp_grid à utiliser x1 La coordonnée x du côté gauche du rectangle à vérifier. y1 Coordonnée y du côté supérieur du rectangle à vérifier. x2 La coordonnée x du côté droit du rectangle à vérifier. y2 La coordonnée x du côté inférieur du rectangle à vérifier. Retour:Boolean Exemple:mp_grid_clear_rectangle(grid, 0, 0, 100, 200); Le code ci-dessus marquera comme libre toutes les cellules de mp_grid indexées dans la variable "grid" qui se situent dans la zone 0,0 à 100,200.
