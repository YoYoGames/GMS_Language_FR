gif_save_buffer Cette fonction permet d'enregistrer une animation GIF. Vous fournissez l'index GIF (tel que renvoyé par la fonction gif_open()) et la fonction renvoie un tampon de croissance aligné sur 1 octet contenant les données GIF. Notez que les données GIF finales auront été palettisées à l'aide de la palette Universal 884 (voir ici pour plus d'informations). Syntaxe : gif_save_buffer(gif_index) ; Arguments Description gif_index L'ID du gif à sauvegarder Les retours : Entier (ID de la mémoire tampon) Exemple : si save_gif == true { if count == 0 { gif_image = gif_open(room_width, room_height) ; } else if count < 30 { gif_add_surface(gif_image, application_surface, 6/100) ; } else { global.capture_buffer = gif_save_buffer(gif_image) ; count = 0 ; save_gif = false ; } count++ ; } Le code ci-dessus va créer un fichier image GIF avec 30 images prises sur la surface de l'application, puis l'enregistrer dans un tampon.
