afficher_get_gui_width Cette fonction permet d'obtenir la hauteur (en pixels) de l'interface graphique utilisée dans l'événement Draw GUI. Syntaxe : display_get_gui_height() ; Les retours : Real Exemple : ads_move(0, display_get_gui_height() - ads_get_display_height(0), 0) ; Le code ci-dessus permet d'afficher une publicité dans le coin inférieur gauche de l'écran.
