view_get_yport Cette fonction peut être utilisée pour récupérer la position y du port de vue donné. Syntaxe:view_get_yport(view_port) Argument Description view_port Le port de vue à cibler (0 - 7) Retour:Real Exemple:if view_get_yport(0) != (display_get_height() / 2) - (view_hport[0] / 2)     {     view_set_yport(0, (display_get_height() / 2) - (view_hport[0] / 2));     } Le code ci-dessus vérifiera la position y du port de vue [0] et s'il n'est pas là où il est requis d'être, il est réglé sur cette position.
