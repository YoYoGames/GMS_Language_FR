view_set_camera Cette fonction permet de définir une caméra sur un port de vue spécifique. Vous indiquez le port de vue à définir (de 0 à 7) et fournissez la valeur d'identification unique de la caméra (telle que renvoyée par les fonctions camera_create() ou lorsque vous utilisez la fonction view_get_camera()). Si vous donnez une valeur de -1 comme ID de caméra, vous supprimez une caméra du port de vue et notez que si ce port de vue est activé et visible, vous pouvez obtenir des résultats imprévisibles. Syntaxe : view_set_camera(view_port, camera_id) Arguments Description port_vue Le port de vue à cibler (0 - 7) camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. Les retours : N/A Exemple : var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1) ; view_set_camera(0, cam) ; Le code ci-dessus va créer une nouvelle caméra et lui attribuer le port de vue [0].
