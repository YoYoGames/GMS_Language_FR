view_enabled Cette variable contrôle si les ports de vue visibles dans la salle sont activés ou non. Si vous avez défini les ports de vue sur visible, puis désactivez cette option, toute la pièce sera dessinée sur l'écran à l'échelle de la taille de la fenêtre au lieu des différentes caméras dessinées à travers les ports de vue. Syntaxe:view_enabled; Retour:Boolean (true: enabled, false: disabled) Exemple:if !view_enabled     {     view_visible[0] = true;     view_enabled = true;     } Le code ci-dessus vérifie si les vues sont activées pour la salle, et si elles ne le sont pas, il s'assure que le port de vue [0] est visible, puis active les vues pour la salle.
