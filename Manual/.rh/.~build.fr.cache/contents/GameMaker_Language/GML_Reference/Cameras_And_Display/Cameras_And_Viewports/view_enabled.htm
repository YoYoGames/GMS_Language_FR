vue_activée Cette variable contrôle l'activation ou non des ports de vue visibles dans la pièce. Si les ports de vue sont définis comme visibles et que vous désactivez cette option, la salle entière sera dessinée à l'écran à l'échelle de la fenêtre au lieu que les différentes caméras soient dessinées à travers les ports de vue. au lieu de dessiner les différentes caméras à travers les ports de vue. Syntaxe : view_enabled ; Les retours : Booléen(vrai: activé, faux: désactivé) Exemple : if !view_enabled { view_visible[0] = true ; view_enabled = true ; } Le code ci-dessus vérifie si les vues sont activées pour la pièce, et si elles ne le sont pas, il s'assure que le port de vue [0] est visible, puis active les vues pour la pièce.
