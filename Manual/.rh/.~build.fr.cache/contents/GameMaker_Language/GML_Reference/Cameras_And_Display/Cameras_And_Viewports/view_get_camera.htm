vue_get_caméra Cette fonction peut être utilisée pour récupérer la valeur d'identification unique de la caméra affectée au port de vue donné (de 0 à 7). Si aucune caméra n'est affectée, la fonction renvoie -1. Syntaxe : view_get_camera(view_port) Arguments Description port_vue Le port de vue à cibler (0 - 7) Les retours : Indice de la caméra (réel) Exemple : var cam = view_get_camera(0) ; var cw = camera_get_view_width(cam) ; var ch = camera_get_view_height(cam) ; camera_set_view_pos(cam, mouse_x - (cw / 2), mouse_y - (ch / 2)) ; Le code ci-dessus récupère l'ID de la caméra affectée au port de vue [0], puis modifie sa position.
