view_set_visible Cette fonction peut être utilisée pour définir la visibilité d'un port de vue donné. La fonction prend l'index du port de vue (de 0 à 7) et un booléen true si le port est visible et false s'il ne l'est pas. Syntaxe : view_set_visible(view_port, visible) Arguments Description port_vue Le port de vue à cibler (0 - 7) visible Commutateur de visibilité(true est visible et false est invisible) Les retours : N/A Exemple : if !view_get_visible(0) { view_set_visible(0, true) ; } Le code ci-dessus vérifie si le port de vue [0] est visible et s'il ne l'est pas, il est défini comme visible.
