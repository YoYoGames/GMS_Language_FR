view_get_visible Cette fonction peut être utilisée pour vérifier la visibilité du port de vue donné. La fonction retournera true s'il est visible et false si ce n'est pas le cas. Syntaxe:view_get_visible(view_port) Argument Description view_port Le port de vue à cibler (0 - 7) Retour:Boolean Exemple:if !view_get_visible(0)     {     view_set_visible(0, true);     } Le code ci-dessus vérifiera si le port de vue [0] est visible et s'il ne l'est pas, il est défini sur visible.
