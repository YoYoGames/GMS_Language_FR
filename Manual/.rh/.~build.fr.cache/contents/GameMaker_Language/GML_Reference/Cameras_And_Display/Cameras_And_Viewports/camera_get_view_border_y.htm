camera_get_view_border_y Cette fonction peut être utilisée pour récupérer la valeur de la bordure pour l'objet/instance suivant de la caméra donnée le long de l'axe y (bordure verticale). La valeur de retour sera en pixels. Syntaxe : camera_get_view_border_y(camera_id) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. Les retours : Real Exemple : var xb = camera_get_view_border_x(view_camera[0]) ; var yb = camera_get_view_border_y(view_camera[0]) ; if xb != 200 || yb != 200 { camera_set_view_border(view_camera[0], 200, 200) ; } Le code ci-dessus récupère les valeurs xborder et yborder de la caméra affectée au port de vue [0], puis vérifie si elles correspondent à la valeur donnée. Si ce n'est pas le cas, la caméra de vue est définie sur la valeur donnée. Retour : Caméras et ports d'observation Suivant : camera_get_view_angle Copyright YoYo Games Ltd. 2021 Tous droits réservés
