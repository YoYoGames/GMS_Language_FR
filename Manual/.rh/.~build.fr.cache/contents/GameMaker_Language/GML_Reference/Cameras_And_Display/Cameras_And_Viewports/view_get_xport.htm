vue_get_xport Cette fonction peut être utilisée pour récupérer la position x du port de vue donné. Syntaxe : view_get_xport(view_port) Arguments Description port_vue Le port de vue à cibler (0 - 7) Les retours : Real Exemple : if view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2) { view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2)) ; } Le code ci-dessus vérifie la position x du port de vue [0] et s'il n'est pas là où il doit être, il est réglé sur cette position.
