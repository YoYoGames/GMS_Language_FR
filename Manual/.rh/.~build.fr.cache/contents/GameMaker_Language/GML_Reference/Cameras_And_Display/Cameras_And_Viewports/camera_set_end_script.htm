camera_set_end_script Cette fonction peut être utilisée pour définir une fonction de script qui sera appelée à la fin de chaque cadre de jeu auquel la caméra est affectée à un port de vue visible et actif, une fois que tout pour cette caméra de vue a été rendu. Vous donnez la valeur d'ID de caméra unique (telle qu'elle est renvoyée par les différents camera_create() fonctions) et l'ID de la fonction à appeler. L'ordre dans lequel les fonctions associées aux caméras et le rendu réel de la vue de la caméra sont les suivants: Les caméras de tous les ports de vue visibles et actifs ont leur fonction de mise à jour appelée Puis pour chaque vue: La fonction de démarrage de la caméra pour cette vue est appelée Les événements de dessin sont exécutés pour cette vue La fonction de fin de la caméra est appelée Passer à la vue visible active suivante et répéter... Syntaxe:camera_set_end_script(camera_id, script) Argument Description camera_id La valeur d'ID de caméra unique renvoyée lors de la création de la caméra. script Le script pour exécuter chaque cadre de jeu Retour:N/A Exemple:camera_set_end_script(view_camera[0], endCamera); Le code ci-dessus définit le script de fin de la caméra affectée au port d'affichage [0].
