camera_set_end_script Cette fonction peut être utilisée pour définir une fonction de script qui sera appelée à la fin de chaque image de jeu où la caméra est assignée à un port de vue visible et actif, après que tout ce qui concerne cette vue caméra a été rendu. Vous donnez la valeur unique de l'ID de la caméra (telle que renvoyée par les différentes fonctions camera_create() fonctions) et l'ID de la fonction à appeler. L'ordre dans lequel les fonctions attachées aux caméras et le rendu effectif de la vue de la caméra est le suivant : Les caméras de tous les ports de vue visibles et actifs ont leur fonction de mise à jour appelée Puis pour chaque vue : La fonction de démarrage de la caméra pour cette vue est appelée. Les événements de dessin sont exécutés pour cette vue La fonction de fin de la caméra est appelée Passez à la vue active visible suivante et répétez... Syntaxe : camera_set_end_script(camera_id, script) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. script Le script pour exécuter chaque cadre de jeu Les retours : N/A Exemple : camera_set_end_script(view_camera[0], endCamera) ; Le code ci-dessus définit le script de fin pour la caméra affectée au port de vue [0].
