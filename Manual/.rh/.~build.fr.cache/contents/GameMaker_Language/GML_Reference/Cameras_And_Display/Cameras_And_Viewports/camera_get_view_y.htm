camera_get_view_y Cette fonction peut être utilisée pour récupérer la position y de la vue pour une caméra donnée. Notez que cette fonction est uniquement valable pour les caméras créées à l' aide camera_create_view() ou pour ceux ajoutés dans l'éditeur de salle. Syntaxe:camera_get_view_y(camera_id) Argument Description camera_id La valeur d'ID de caméra unique renvoyée lors de la création de la caméra Retour:Real Exemple:var vx = camera_get_view_x(view_camera[0]); var vy = camera_get_view_y(view_camera[0]); draw_text(vx + 5, vy + 5, "SCORE" + string(score)); Le code ci-dessus récupère la position de la caméra affectée au port d'affichage [0], puis dessine le texte par rapport à cette position.
