view_set_surface_id Cette variable vous permet de définir le contenu d'un port de vue à dessiner sur une surface. Lorsque vous travaillez avec des surfaces, il est souvent nécessaire de capturer l'ensemble de la région visible de l'écran sur la surface, et donc de l'affecter à un port de vue en utilisant cette fonction. Cela signifie que tout ce qui est affiché dans la vue du port choisi sera désormais dessiné sur la surface assignée et que le contenu de ce port de vue ne sera plus affiché : Activer une nouvelle vue et dessiner la surface uniquement dans cette vue (en utilisant view_current pour vérifier quelle vue est dessinée) Dessinez la surface dans l'événement Draw GUI d'une instance, puisque la couche GUI est indépendante des vues. Lorsque vous utilisez cette fonction, vous donnez l'indice du port de vue (de 0 à 7) et un indice de surface (soit l'indice surface_de_l'application ou la valeur d'index unique retournée par la fonction surface_create()) ou, si une surface a déjà été attribuée et que vous voulez la supprimer, une valeur de -1. Pour plus d'informations sur le réglage du port de vue sur une surface, voir également la variable view_surface_id. Syntaxe : view_set_surface_id(view_port, surf) Arguments Description port_vue Le port de vue à cibler (0 - 7) Les retours : Real Exemple : si view_get_surface_id(0) == -1 { view_set_surface_id(0, global.vSurf) ; } Le code ci-dessus vérifiera si une surface a été assignée au port de vue [0] et si ce n'est pas le cas, une surface sera assignée.
