view_set_surface_id Avec cette variable, vous pouvez définir le contenu d'un port de vue pour dessiner sur une surface. Lorsque vous travaillez avec des surfaces, il est souvent nécessaire de capturer toute la zone visible de l'écran sur la surface, et vous devez donc l'attribuer à un port de vue à l'aide de cette fonction. Cela signifie que tout ce qui est affiché dans la vue de port choisie sera maintenant dessiné sur la surface affectée et le contenu de ce port de vue ne sera plus affiché, ce qui signifie que vous devrez soit: Activez une nouvelle vue et dessinez la surface uniquement dans cette vue (en utilisant view_current pour vérifier quelle vue est dessinée) Dessinez la surface dans l'événement Draw GUI d'une instance, car la couche GUI est indépendante des vues. Lorsque vous utilisez cette fonction, vous donnez l'index du port de vue (de 0 à 7) et un index de surface (soit le application_surface ou la valeur d'index unique renvoyée par la fonction surface_create() ) ou, si une surface a déjà été affectée et que vous souhaitez la supprimer, une valeur de -1. Pour plus d'informations sur la définition du port de vue sur une surface, voir également la variable view_surface_id. Syntaxe:view_set_surface_id(view_port, surf) Argument Description view_port Le port de vue à cibler (0 - 7) Retour:Real Exemple:if view_get_surface_id(0) == -1     {     view_set_surface_id(0, global.vSurf);     } Le code ci-dessus vérifiera si une surface a été assignée au port de vue [0] et si ce n'est pas le cas, une est assignée.
