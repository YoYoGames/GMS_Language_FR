view_set_wport Cette fonction peut être utilisée pour définir la largeur du port de vue donné. Vous donnez l'index du port de vue (de 0 à 7) et la nouvelle largeur à utiliser. Syntaxe:view_set_wport(view_port, w) Argument Description view_port Le port de vue à cibler (0 - 7) w La nouvelle largeur (en pixels) Retour:Real Exemple:if view_get_wport(0) != (display_get_width () / 2)     {     view_set_wport(0, display_get_width() / 2);     } Le code ci-dessus vérifiera la largeur du port de vue [0] et si ce n'est pas la moitié de la largeur d'affichage, il est défini sur cette valeur.
