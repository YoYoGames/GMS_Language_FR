camera_get_begin_script Cette fonction peut être utilisée pour récupérer l'ID de la fonction de script assignée comme script de début pour la caméra donnée. Si aucune fonction de script n'est attribuée, la fonction renvoie -1. Syntaxe : camera_get_begin_script(camera_id) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. Les retours : ID de la fonction du script ou -1 si aucun script n'est assigné Exemple : var scr = camera_get_begin_script(camera_view[0]) ; if scr != cutScene { camera_set_update_script(camera_view[0], cutScene) ; } Le code ci-dessus vérifie la fonction de script assignée en tant que script de début pour la caméra assignée au port de vue [0] et si ce n'est pas "cutScene", il est défini sur cette fonction.
