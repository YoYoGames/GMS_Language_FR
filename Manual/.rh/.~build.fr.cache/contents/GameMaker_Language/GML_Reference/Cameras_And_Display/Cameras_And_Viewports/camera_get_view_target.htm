camera_get_view_target Cette fonction peut être utilisée pour récupérer la cible de suivi de la caméra donnée. La valeur de retour peut être un index d'objet, un ID d'instance ou -1 si aucune cible de suivi n'a été définie. Syntaxe : camera_get_view_target(camera_id) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. Les retours : Index de l'objet, ID de l'instance, ou -1 si aucune cible n'est définie. Exemple : var target = camera_get_view_target(view_camera[0]) ; if target != obj_Player { camera_set_view_target(view_camera[0], obj_Player) ; } Le code ci-dessus récupère la cible de suivi de la caméra affectée au port de vue [0], puis vérifie si elle correspond à l'index d'objet "obj_Player". Si ce n'est pas le cas, la caméra de la vue est configurée pour suivre une instance de cet objet.
