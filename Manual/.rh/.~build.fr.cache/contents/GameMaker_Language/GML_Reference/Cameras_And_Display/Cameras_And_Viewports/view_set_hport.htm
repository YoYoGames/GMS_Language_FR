vue_set_hport Cette fonction peut être utilisée pour définir la hauteur du port de vue donné. Vous donnez l'index du port de vue (de 0 à 7) et la nouvelle hauteur à utiliser. Syntaxe : view_set_hport(view_port, h) Arguments Description port_vue Le port de vue à cibler (0 - 7) h La nouvelle hauteur (en pixels) Les retours : Real Exemple : if view_get_hport(0) != (display_get_height () / 2) { view_set_hport(0, display_get_height() / 2) ; } Le code ci-dessus vérifie la hauteur du port de vue [0] et si elle n'est pas égale à la moitié de la hauteur de l'écran, elle est fixée à cette valeur.
