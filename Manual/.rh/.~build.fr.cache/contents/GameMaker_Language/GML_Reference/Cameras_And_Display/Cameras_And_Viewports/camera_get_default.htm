camera_get_default Cette fonction peut être utilisée pour récupérer la valeur d'ID de caméra unique de la caméra par défaut (la caméra GameMaker Studio 2 lorsqu'aucune vue de caméra ou aucun port n'est actif dans une salle de jeux). Syntaxe:camera_get_default() Retour:Camera index (real). Exemple:var def = camera_get_default(); view_camera[0] = def; Le code ci-dessus obtient l'identifiant de la caméra par défaut et définit la caméra de vue pour le port [0] pour l'utiliser.
