camera_get_default Cette fonction peut être utilisée pour récupérer la valeur d'identification unique de la caméra par défaut (la caméra que GameMaker Studio 2 utilise lorsqu'aucune vue ou aucun port de caméra n'est actif dans une salle de jeu). Syntaxe : camera_get_default() Les retours : Indice de la caméra (réel). Exemple : var def = camera_get_default() ; view_camera[0] = def ; Le code ci-dessus obtient l'ID de la caméra par défaut et définit la caméra d'affichage pour le port [0] afin de l'utiliser.
