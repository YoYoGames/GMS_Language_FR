camera_set_view_speed Vous pouvez utiliser cette fonction pour mettre à jour la vitesse de la caméra de vision dans la pièce. Vous donnez la valeur d'ID de caméra unique (telle qu'elle est renvoyée par les différents camera_create() fonctions), puis donnez la vitesse x et y (horizontale et verticale) à laquelle il doit se déplacer lorsqu'il est réglé pour suivre une instance donnée. La vitesse est calculée en pixels par pas et peut être réglée sur "-1" pour faire bouger la caméra instantanément, mais si la caméra n'est pas réglée pour suivre une instance, les valeurs définies ici n'auront aucun effet visible. Syntaxe:camera_set_view_speed(camera_id, xspeed, yspeed) Argument Description camera_id La valeur d'ID de caméra unique renvoyée lors de la création de la caméra. xspeed La vitesse (nombre de pixels par image de jeu) à laquelle la vue doit se déplacer sur l'axe horizontal (x) yspeed La vitesse (nombre de pixels par image de jeu) à laquelle la vue doit se déplacer sur l'axe vertical (y) Retour:N/A Exemple:camera_set_view_speed(view_camera[0], 5, 5); Le code ci-dessus définira la vitesse de la caméra de visualisation pour la caméra affectée au port de visualisation [0].
