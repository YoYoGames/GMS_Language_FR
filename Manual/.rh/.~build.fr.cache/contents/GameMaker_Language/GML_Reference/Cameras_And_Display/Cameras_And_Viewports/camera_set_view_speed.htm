camera_set_view_speed Vous pouvez utiliser cette fonction pour mettre à jour la vitesse de la caméra de visualisation dans la pièce. Vous donnez la valeur unique de l'ID de la caméra (telle que retournée par les différentes fonctions de la fonction camera_create() fonctions), puis vous indiquez la vitesse x et y (horizontale et verticale) à laquelle la caméra doit se déplacer lorsqu'elle est réglée pour suivre une trajectoire donnée. et verticale) à laquelle elle doit se déplacer lorsqu'elle est configurée pour suivre une instance donnée. La vitesse est calculée en pixels par pas et peut être définie à "-1" pour que la caméra se déplace instantanément, mais si la caméra n'est pas définie pour suivre une instance, les valeurs définies ici n'auront aucun effet visible. valeurs définies ici n'auront aucun effet visible. Syntaxe : camera_set_view_speed(camera_id, xspeed, yspeed) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. xspeed La vitesse (nombre de pixels par image de jeu) à laquelle la vue doit se déplacer sur l'axe horizontal (x). yspeed La vitesse (nombre de pixels par image de jeu) à laquelle la vue doit se déplacer sur l'axe vertical (y). Les retours : N/A Exemple : camera_set_view_speed(view_camera[0], 5, 5) ; Le code ci-dessus définira la vitesse de la caméra de vue pour la caméra affectée au port de vue [0].
