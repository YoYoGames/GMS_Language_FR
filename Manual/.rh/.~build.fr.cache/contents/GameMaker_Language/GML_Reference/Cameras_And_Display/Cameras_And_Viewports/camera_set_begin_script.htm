camera_set_begin_script Cette fonction peut être utilisée pour définir une fonction de script qui sera appelée au début de chaque image de jeu où la caméra est assignée à un port de vue visible et actif. Vous donnez à la caméra une valeur unique unique de la caméra (telle que renvoyée par les différentes camera_create() fonctions) et l'ID de la fonction à appeler. L'ordre dans lequel les scripts attachés aux caméras et le rendu réel de la vue de la caméra sont les suivants : Les caméras de tous les ports de vue visibles et actifs ont leur fonction de mise à jour appelée Puis pour chaque vue : La fonction de démarrage de la caméra pour cette vue est appelée. Les événements de dessin sont exécutés pour cette vue La fonction de fin de la caméra est appelée Passez à la vue active visible suivante et répétez... Syntaxe : camera_set_begin_script(camera_id, script) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. script La fonction script pour exécuter chaque image du jeu Les retours : N/A Exemple : camera_set_begin_script(view_camera[0], beginCamera) ; Le code ci-dessus définit la fonction de script de début pour la caméra affectée au port de vue [0].
