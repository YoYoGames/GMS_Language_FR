camera_destroy Avec cette fonction, vous pouvez détruire n'importe quelle caméra. Lorsque vous appelez la fonction, vous fournissez la valeur unique de l'ID de la caméra, que vous obtenez à partir des fonctions camera_create_* ou du tableau view_camera si vous utilisez le Room Editor pour configurer le port d'affichage et la vue. Vous ne devez jamais détruire une caméra actuellement affectée à une vue visible, sauf si vous affectez une nouvelle caméra à cette vue au cours de la même étape . que vous avez créées par le biais du code lorsqu'elles ne sont plus nécessaires à votre jeu, afin d'éviter les fuites de mémoire. Vous pouvez également détruire les caméras par défaut si vous en avez assignées dans l'éditeur de salle, mais vous devez assigner une nouvelle caméra au port de vue (ou la désactiver), sinon vous obtiendrez des résultats étranges. ), sinon vous obtiendrez des résultats étranges. Syntaxe : camera_destroy(camera_id) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. Les retours : Real (valeur d'identification unique de la caméra) Exemple : camera_destroy(view_camera[0]) ; view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1) ; Le code ci-dessus détruit la caméra actuellement affectée au port de vue [0], puis crée une nouvelle caméra et affecte son ID à ce port.
