camera_destroy Avec cette fonction, vous pouvez détruire n'importe quel appareil photo. Lorsque vous appelez la fonction, vous indiquez la valeur d'ID de caméra unique, que vous obtenez camera_create_* fonctions ou depuis le view_camera tableau si vous utilisez l'éditeur de pièce pour configurer le port de vue et la vue. Vous ne devez jamais détruire une caméra actuellement affectée à une vue visible, à moins que vous n'assigniez une nouvelle caméra à cette vue au cours de la même étape, et vous devez toujours détruire toutes les caméras que vous avez créées via le code lorsqu'elles ne sont plus requises par votre jeu. pour éviter les fuites de mémoire, et vous pouvez également détruire les caméras par défaut si vous en avez assignées dans l'éditeur de pièce, mais vous devez attribuer une nouvelle caméra au port de vue (ou la désactiver) sinon vous obtiendrez des résultats étranges. Syntaxe:camera_destroy(camera_id) Argument Description camera_id La valeur d'ID de caméra unique renvoyée lors de la création de la caméra. Retour:Real (unique camera ID value) Exemple:camera_destroy(view_camera[0]); view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1); Le code ci-dessus détruit la caméra actuellement affectée au port d'affichage [0], puis crée une nouvelle caméra et attribue son ID à ce port.
