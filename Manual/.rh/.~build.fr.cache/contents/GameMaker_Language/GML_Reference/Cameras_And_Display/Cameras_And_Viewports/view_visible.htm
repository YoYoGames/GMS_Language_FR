view_visible Cette variable peut être utilisée pour savoir si un port de vue particulier est actuellement visible ou non. Vous pouvez également définir cette variable pour activer ou désactiver efficacement une vue en définissant la valeur sur true (visible) ou false (invisible). Notez que même si vous avez un port de vue défini sur visible, si les ports de vue ne sont pas activés (à l'aide de la variable intégrée view_enabled ou en les activant dans l'éditeur de salle), ils ne seront pas dessinés à l'écran. Syntaxe:view_visible[0 ... 7]; Retour:Boolean (true: enabled, false: disabled) Exemple:if !view_visible[0]     {     view_visible[0] = true;     } Le code ci-dessus vérifie si la vue [0] est visible ou non et si ce n'est pas le cas, il la définit alors comme visible.
