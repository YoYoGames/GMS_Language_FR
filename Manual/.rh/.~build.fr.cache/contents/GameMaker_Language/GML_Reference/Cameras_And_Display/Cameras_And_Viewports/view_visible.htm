vue_visible Cette variable peut être utilisée pour savoir si un port de vue particulier est actuellement visible ou non. Vous pouvez également définir cette variable pour activer ou désactiver efficacement une vue en définissant la valeur à true (visible) ou false (invisible) . Notez que même si un port de vue est défini comme visible, si les ports de vue ne sont pas activés (en utilisant la variable intégrée view_enabled ou en les activant dans l'éditeur de pièce), ils ne seront pas dessinés à l'écran. l'écran. Syntaxe : vue_visible [0 ... 7] ; Les retours : Booléen(vrai: activé, faux: désactivé) Exemple : if !view_visible[0] { view_visible[0] = true ; } Le code ci-dessus vérifie si view[0] est visible ou non et s'il ne l'est pas, il le rend visible.
