camera_set_view_target Vous pouvez utiliser cette fonction pour définir la cible de suivi de la caméra de vue dans la pièce. Vous donnez la valeur d'ID de caméra unique (telle qu'elle est renvoyée par les différents camera_create() fonctions), puis indiquez l'instance cible ou l'ID d'objet sur lequel vous souhaitez définir la vue de la caméra. Notez que si vous choisissez un ID d'objet et qu'il y a plus d'une instance de cet objet dans la pièce, il n'y a aucun moyen pour GameMaker Studio 2 de savoir quelle instance vous souhaitez suivre et donc il peut s'agir de l'une d'entre elles. Syntaxe:camera_set_view_target(camera_id, instance_id/object_id) Argument Description camera_id La valeur d'ID de caméra unique renvoyée lors de la création de la caméra instance_id/object_id Instance ou objet pour avoir la cible de la caméra à suivre Retour:N/A Exemple:camera_set_view_target(view_camera[0], id); Le code ci-dessus définira la cible de la caméra de vue pour la caméra affectée au port de visualisation [0] à celle de l'instance exécutant le code.
