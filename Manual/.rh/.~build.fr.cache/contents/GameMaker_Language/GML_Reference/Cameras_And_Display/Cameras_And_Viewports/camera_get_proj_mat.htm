camera_get_proj_mat Cette fonction peut être utilisée pour récupérer la matrice de projection. La fonction renvoie la valeur de l'ID de la matrice qui peut ensuite être utilisée dans d'autres fonctions matricielles ou pour définir la matrice de projection d'une autre caméra (à l'aide de la fonction d'identification). d'une autre caméra (en utilisant camera_set_proj_mat()). Syntaxe : camera_get_proj_mat(camera_id) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. Les retours : ID de la matrice (réel) Exemple : mat = camera_get_proj_mat(camera_view[0]) ; Le code ci-dessus obtient la matrice de projection pour la caméra affectée au port de vue [0].
