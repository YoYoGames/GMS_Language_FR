camera_get_proj_mat Cette fonction peut être utilisée pour récupérer la matrice de projection. La fonction renvoie la valeur de l'ID de matrice qui peut ensuite être utilisée dans d'autres fonctions de matrice ou pour définir la matrice de projection d'une autre caméra (en utilisant camera_set_proj_mat() ). Syntaxe: camera_get_proj_mat (camera_id) Argument Description camera_id La valeur d'ID de caméra unique renvoyée lors de la création de la caméra Retour:Matrix ID (real) Exemple:mat = camera_get_proj_mat(camera_view[0]); Le code ci-dessus obtient la matrice de projection de la caméra affectée au port de visualisation [0].
