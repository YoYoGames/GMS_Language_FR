camera_set_view_pos Vous pouvez utiliser cette fonction pour mettre à jour la position de la vue de la caméra dans la pièce. Vous donnez la valeur unique de l'ID de la caméra (telle que renvoyée par les différentes fonctions camera_create() fonctions), puis vous indiquez les positions x et y sur lesquelles sur lesquelles la caméra doit se positionner. Syntaxe : camera_set_view_pos(camera_id, x, y) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. x La position x à laquelle placer la vue (dans la pièce) y La position y à laquelle placer la vue (dans la pièce) Les retours : N/A Exemple : camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2)) ; Le code ci-dessus définira la position de la caméra de vue pour la caméra affectée au port de vue [0].
