camera_set_view_pos Vous pouvez utiliser cette fonction pour mettre à jour la position de la vue de la caméra dans la pièce. Vous donnez la valeur d'ID de caméra unique (telle qu'elle est renvoyée par les différents camera_create() fonctions), puis indiquez les positions x et y sur lesquelles régler la caméra. Syntaxe:camera_set_view_pos(camera_id, x, y) Argument Description camera_id La valeur d'ID de caméra unique renvoyée lors de la création de la caméra. x La position x sur laquelle placer la vue (dans la pièce) y La position y à laquelle placer la vue (dans la pièce) Retour:N/A Exemple:camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2)); Le code ci-dessus définira la position de la caméra de visualisation pour la caméra affectée au port de visualisation [0].
