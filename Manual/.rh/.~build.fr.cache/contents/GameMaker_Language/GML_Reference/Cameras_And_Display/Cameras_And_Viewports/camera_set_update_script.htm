camera_set_update_script Cette fonction peut être utilisée pour définir une fonction de script qui sera appelée à chaque image de jeu où la caméra est assignée à un port de vue visible et actif. Vous donnez la valeur unique de l'ID de la caméra (telle qu'elle est retournée par les différentes fonctions camera_create ) et l'ID de la fonction à appeler. L'ordre dans lequel les fonctions attachées aux caméras et le rendu réel de la vue de la caméra sont les suivants : Les caméras de tous les ports de vue visibles et actifs ont leur fonction de mise à jour appelée Ensuite, pour chaque vue : La fonction de démarrage de la caméra pour cette vue est appelée. Les événements de dessin sont exécutés pour cette vue La fonction de fin de la caméra est appelée Passez à la vue active visible suivante et répétez... Syntaxe : camera_set_update_script(camera_id, script) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. script La fonction script pour exécuter chaque image du jeu Les retours : N/A Exemple : camera_set_update_script(view_camera[0], updateCamera) ; Le code ci-dessus définit la fonction de mise à jour pour la caméra affectée au port de vue [0].
