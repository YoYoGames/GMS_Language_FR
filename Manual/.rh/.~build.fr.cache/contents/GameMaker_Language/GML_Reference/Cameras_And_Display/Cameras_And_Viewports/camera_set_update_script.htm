camera_set_update_script Cette fonction peut être utilisée pour définir une fonction de script qui sera appelée chaque cadre de jeu auquel la caméra est affectée à un port de vue visible et actif. Vous donnez la valeur d'ID de caméra unique (telle qu'elle est renvoyée par les différents camera_create fonctions) et l'ID de la fonction à appeler. L'ordre dans lequel les fonctions associées aux caméras et le rendu réel de la vue de la caméra sont les suivants: Les caméras de tous les ports de vue visibles et actifs ont leur fonction de mise à jour appelée Puis pour chaque vue: La fonction de démarrage de la caméra pour cette vue est appelée Les événements de dessin sont exécutés pour cette vue La fonction de fin de la caméra est appelée Passer à la vue visible active suivante et répéter... Syntaxe:camera_set_update_script(camera_id, script) Argument Description camera_id La valeur d'ID de caméra unique renvoyée lors de la création de la caméra. script La fonction de script pour exécuter chaque cadre de jeu Retour:N/A Exemple:camera_set_update_script(view_camera[0], updateCamera); Le code ci-dessus définit la fonction de mise à jour de la caméra affectée au port de visualisation [0].
