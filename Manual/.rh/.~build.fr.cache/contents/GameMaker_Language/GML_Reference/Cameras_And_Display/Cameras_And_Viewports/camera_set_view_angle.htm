camera_set_view_angle Vous pouvez utiliser cette fonction pour mettre à jour l'angle de la caméra de vue dans la pièce. Vous donnez la valeur unique de l'ID de la caméra (telle que renvoyée par les différentes fonctions de la fonction camera_create() fonctions), puis l'angle que vous souhaitez l'angle que vous souhaitez définir pour la caméra. La valeur par défaut est de 0°, les valeurs positives faisant tourner la caméra dans le sens inverse des aiguilles d'une montre. Par exemple, si la valeur est de 90, la caméra tournera de 90° vers la gauche. Syntaxe : camera_set_view_angle(camera_id, angle) Arguments Description camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra. angle Le nouvel angle à définir pour la vue de la caméra Les retours : N/A Exemple : camera_set_view_angle(view_camera[0], point_direction(x, y, mouse_x, mouse_y)) ; Le code ci-dessus définira l'angle de la caméra de vue pour la caméra affectée au port de vue [0].
