view_get_wport Cette fonction peut être utilisée pour récupérer la largeur du port de vue donné. Syntaxe : view_get_wport(view_port) Arguments Description port_vue Le port de vue à cibler (0 - 7) Les retours : Real Exemple : if view_get_wport(0) != (display_get_width () / 2) { view_set_wport(0, display_get_width() / 2) ; } Le code ci-dessus vérifie la largeur du port d'affichage [0] et si elle n'est pas égale à la moitié de la largeur de l'écran, elle est fixée à cette valeur.
