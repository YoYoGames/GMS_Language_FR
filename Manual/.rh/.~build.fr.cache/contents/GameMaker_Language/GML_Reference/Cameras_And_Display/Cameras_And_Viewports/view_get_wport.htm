view_get_wport Cette fonction peut être utilisée pour récupérer la largeur du port de vue donné. Syntaxe:view_get_wport(view_port) Argument Description view_port Le port de vue à cibler (0 - 7) Retour:Real Exemple:if view_get_wport(0) != (display_get_width () / 2)     {     view_set_wport(0, display_get_width() / 2);     } Le code ci-dessus vérifiera la largeur du port de vue [0] et si ce n'est pas la moitié de la largeur d'affichage, il est défini sur cette valeur.
