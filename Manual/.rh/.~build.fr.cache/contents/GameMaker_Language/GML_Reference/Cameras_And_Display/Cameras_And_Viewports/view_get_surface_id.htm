view_get_surface_id Cette fonction peut être utilisée pour récupérer la valeur d'ID unique pour la surface affectée au port de vue donné (retournera -1 si aucune surface n'a été affectée). Syntaxe:view_get_surface_id(view_port) Argument Description view_port Le port de vue à cibler (0 - 7) Retour:Real Exemple:if view_get_surface_id(0) == -1     {     view_set_surface_id(0, global.vSurf);     } Le code ci-dessus vérifiera si une surface a été assignée au port de vue [0] et si ce n'est pas le cas, une est assignée.
