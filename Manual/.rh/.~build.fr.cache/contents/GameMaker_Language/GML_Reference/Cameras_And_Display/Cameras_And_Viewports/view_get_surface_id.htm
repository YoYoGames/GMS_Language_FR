view_get_surface_id Cette fonction peut être utilisée pour récupérer la valeur d'identification unique de la surface attribuée au port de vue donné (elle renvoie -1 si aucune surface n'a été attribuée). Syntaxe : view_get_surface_id(view_port) Arguments Description port_vue Le port de vue à cibler (0 - 7) Les retours : Real Exemple : si view_get_surface_id(0) == -1 { view_set_surface_id(0, global.vSurf) ; } Le code ci-dessus vérifiera si une surface a été assignée au port de vue [0] et si ce n'est pas le cas, une surface sera assignée.
