vue_set_export Cette fonction peut être utilisée pour définir la position x du port de vue donné. Vous donnez l'index du port de vue (de 0 à 7) et la nouvelle position à laquelle le placer. Syntaxe : view_set_xport(view_port, x) Arguments Description port_vue Le port de vue à cibler (0 - 7) x La nouvelle position x Les retours : N/A Exemple : if view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2) { view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2)) ; } Le code ci-dessus vérifie la position x du port de vue [0] et s'il n'est pas là où il doit être, il est réglé sur cette position.
