caméra_get_active Cette fonction peut être utilisée pour récupérer la valeur d'identification unique de la caméra actuellement active. Syntaxe : camera_get_active() Les retours : Indice de la caméra (réel). Exemple : var active = camera_get_active() ; if active != view_camera[0] { view_camera[0] = active ; } Le code ci-dessus récupère l'ID de la caméra active et définit la caméra de vue pour le port [0] afin de l'utiliser.
