gif_save Avec cette fonction, vous pouvez enregistrer une animation GIF. Vous fournissez l'index GIF (tel que renvoyé par la fonction gif_open() ) ainsi qu'un nom de fichier pour l'enregistrer. Notez que GameMaker Studio 2 n'ajoute pas automatiquement le .gif l'extension de fichier, vous devez donc l'inclure dans la chaîne de nom de fichier si vous souhaitez que le fichier enregistré soit identifié en tant que GIF. Le GIF créé sera palettisé à l'aide de la palette Universal 884 (voir ici pour plus d'informations). Notez que si la fonction réussit et que le GIF est enregistré correctement, il renverra 0, sinon il renverra -1. Syntaxe:gif_save(gif_index, fname); Argument Description gif_index L'ID du gif à enregistrer fname Le nom de fichier à utiliser pour le gif Retour:Integer Exemple:if save_gif == true { if count == 0     {     gif_image = gif_open(room_width, room_height);     } else if count < 30     {     gif_add_surface(gif_image, application_surface, 6/100);     } else     {     gif_save(gif_image, "GameCapture.gif");     count = 0;     save_gif = false;     } count++; } Le code ci-dessus créera un fichier image GIF avec 30 images prises à partir de la surface de l'application, puis l'enregistrera.
