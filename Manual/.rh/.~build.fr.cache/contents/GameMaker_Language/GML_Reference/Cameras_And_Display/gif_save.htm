gif_save Cette fonction vous permet d'enregistrer une animation GIF. Vous fournissez l'index GIF (tel que renvoyé par la fonction gif_open()) ainsi qu'un nom de fichier pour l'enregistrer. Notez que GameMaker Studio 2 n'ajoute pas automatiquement automatiquement l'extension de fichier .gif, vous devez donc l'inclure dans la chaîne du nom de fichier si vous souhaitez que le fichier enregistré soit identifié comme un GIF. Le GIF créé sera palettisé à l'aide de la palette Universal 884 (voir ici pour plus d'informations). Notez que si la fonction réussit et que le GIF est enregistré correctement, elle renvoie 0, sinon elle renvoie -1. Syntaxe : gif_save(gif_index, fname) ; Arguments Description gif_index L'ID du gif à sauvegarder fname Le nom de fichier à utiliser pour le gif. Les retours : Entier Exemple : si save_gif == true { if count == 0 { gif_image = gif_open(room_width, room_height) ; } else if count < 30 { gif_add_surface(gif_image, application_surface, 6/100) ; } else { gif_save(gif_image, "GameCapture.gif") ; count = 0 ; save_gif = false ; } count++ ; } Le code ci-dessus va créer un fichier image GIF avec 30 images prises sur la surface de l'application, puis l'enregistrer.
