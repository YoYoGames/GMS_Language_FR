screen_save Avec cette fonction, vous pouvez enregistrer une capture d'écran du jeu tel qu'il apparaît actuellement dans la fenêtre du jeu. Le fichier sera enregistré dans le répertoire de travail du jeu, qui dans Windows se trouve à :/Users/YOURUSERNAME/AppData/Local/[Game Name]/ et sur Mac, ce serait ~/Library/Application Support/[Game Name]/. L'image capturée sera le rendu final de la surface d'application du jeu, ou (si la surface d'application a été désactivée) ce sera une image de l'ensemble de l'affichage ou de la fenêtre. Il est important de noter que la fonction est conçue pour être appelée dans l' événement de fin de l'interface graphique Draw, ce qui devrait donner des résultats cohérents sur toutes les plates-formes. Son utilisation dans tout autre événement peut ne pas fonctionner comme prévu et donner des résultats différents sur différentes cibles et même sur des appareils. Si vous avez besoin de plus de contrôle sur ce qui est enregistré, vous devez utiliser la fonction screen_save_part(). REMARQUE: cette fonction ne fonctionne pas sur la cible HTML5. Syntaxe:screen_save(fname); Argument Description fname Le nom de fichier de la capture d'écran à enregistrer Retour:N/A Exemple:if keyboard_check_pressed(ord("S"))     {     screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")     } Le code ci-dessus prend une capture d'écran du jeu lorsque l'utilisateur appuie sur la touche "S". Cette capture d'écran est enregistrée dans un répertoire spécifique et numérotée à l'aide d'une variable qui est ensuite incrémentée.
