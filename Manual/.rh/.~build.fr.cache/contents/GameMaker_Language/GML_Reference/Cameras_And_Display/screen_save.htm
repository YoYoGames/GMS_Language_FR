écran_sauvegarde Cette fonction vous permet d'enregistrer une capture d'écran du jeu telle qu'elle apparaît actuellement dans la fenêtre du jeu. Le fichier sera enregistré dans le répertoire de travail du jeu, qui, sous Windows, se trouve à :/Users/YOURUSERNAME/AppData/Local/[Nom du jeu]/ et sous Mac, à ~/Bibliothèque/Application Support/[Nom du jeu]/. L'image capturée sera le rendu final de la surface d'application du jeu, ou (si la surface d'application a été désactivée) ce sera une image de l'affichage ou de la fenêtre entière. Il est important de noter que la fonction est conçue pour être appelée dans l'événement Draw GUI End Event, ce qui devrait donner des résultats cohérents sur toutes les plateformes. Si vous l'utilisez dans un autre événement, vous risquez de ne pas obtenir les résultats escomptés et de ne pas obtenir les mêmes résultats sur différentes cibles et même sur différents périphériques. différentes cibles et même différents périphériques. Si vous avez besoin de plus de contrôle sur ce qui est sauvegardé, vous devez utiliser la fonction screen_save_part(). NOTE : Cette fonction ne fonctionne pas sur la cible HTML5. Syntaxe : screen_save(fname) ; Arguments Description fname Le nom de fichier de la capture d'écran à enregistrer. Les retours : N/A Exemple : if keyboard_check_pressed(ord("S")) { screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png") } Le code ci-dessus prend une capture d'écran du jeu lorsque l'utilisateur appuie sur la touche "S". Cette capture d'écran est enregistrée dans un répertoire spécifique et numérotée à l'aide d'une variable qui est ensuite incrémentée.
