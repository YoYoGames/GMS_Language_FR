display_mouse_set Cette fonction vous permet de modifier ou de définir la position de la souris dans l'affichage du jeu, ce qui peut être utile pour les jeux FPS, par exemple. La fonction ne fonctionne que lorsque le jeu est en cours et l'utilisation de ALT + Tab déverrouille la souris. Syntaxe : afficher_souris_set(x, y) ; Arguments Description x La coordonnée x dans l'affichage. y La coordonnée y dans l'affichage. Les retours : N/A Exemple : display_mouse_set(display_get_width() / 2, display_get_height() / 2) ; Le code ci-dessus permet de centrer la souris dans l'affichage du jeu.
