display_reset Cette fonction réinitialise les paramètres d'affichage à ceux qui ont été définis au démarrage du jeu, mais vous permet également de modifier le niveau actuel d'anti-crénelage plein écran utilisé et d'utiliser ou non la synchronisation verticale. Les niveaux d'anti-aliasing disponibles sont 0,2,4 et 8, la valeur de démarrage par défaut étant définie sur 0 et le paramètre v-sync par défaut est false (désactivé). Switch activation de la v-sync peut donner une expérience de jeu plus fluide, mais elle aura également besoin de plus de puissance de traitement et son impact doit donc être considéré avec prudence avant utilisation, et il en va de même pour l'anti-crénelage où plus le nombre est élevé, plus le traitement est obligatoire. Comme tous les appareils cibles ne sont pas identiques, certains peuvent ne pas prendre en charge l'anti-aliasing 8x ou 4x par exemple, et il existe donc une variable en lecture seule disponible pour obtenir les différents niveaux d'AA que l'appareil exécutant le jeu peut afficher:display_aa Cette variable renverra une valeur entière basée sur le paramétrage des bits pour les différents niveaux. Donc, pour seulement 2xAA, cela indiquera 2, pour la disponibilité 2x et 4x, il indiquera 6. Pour 8 et 4, il indiquera 12. Pour les 3 niveaux (2,4 et 8), il indiquera 14. Syntaxe:display_reset(aa, vsync); Argument Description aa Le niveau de filtrage anti-crénelage (0, 2, 4 ou 8). vsync Activez (vrai) ou désactivez la synchronisation verticale (faux). Retour:Real Exemple:if (display_aa > 12)     {     display_reset(8, true);     } Le code ci-dessus définira le niveau d'anti-aliasing sur 8 s'il est pris en charge et activera v-sync.
