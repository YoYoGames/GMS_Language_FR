afficher_get_largeur Cette fonction renvoie la largeur de l'affichage (en pixels). Notez que sur la cible HTML5, cette valeur est la largeur de la fenêtre du navigateur, plutôt que la taille physique de l'affichage. Syntaxe : display_get_width() ; Les retours : Real Exemple : mywidth = display_get_width() ; La variable "mywidth" serait ainsi définie sur la largeur de l'écran.
