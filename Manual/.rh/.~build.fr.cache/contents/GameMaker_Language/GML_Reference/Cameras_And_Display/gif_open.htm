gif_open Cette fonction permet de créer une image vide au format GIF, prête à recevoir des données. Lorsque vous appelez la fonction, vous devez spécifier la largeur et la hauteur du GIF (en pixels, et il n'y a pas de limite supérieure à la taille, sauf pour la mémoire disponible), et la fonction renverra la valeur d'identification unique utilisée pour identifier le cadeau dans les fonctions suivantes, ou elle renverra -1 si le gif n'a pas pu être initialisé (par exemple, si la largeur/hauteur est trop grande pour la mémoire disponible). Vous pouvez également spécifier un argument facultatif pour définir la "couleur claire" du GIF. Il s'agit d'une valeur RVB (sans composante alpha), et le GIF sera nettoyé à cette couleur avant que quoi que ce soit ne lui soit ajouté. Si vous ne fournissez pas d'argument pour la couleur d'effacement, la couleur par défaut, le noir, sera utilisée. de noir sera utilisée. Notez que lorsque vous utilisez cette fonction, vous devez appeler gif_save() pour terminer la création du GIF avant d'ouvrir un autre fichier pour l'enregistrement (ainsi, pour chaque gif_open(), il doit y avoir un gif_save() correspondant). Syntaxe : gif_open(width, height, [clear_colour]) ; Arguments Description largeur La largeur du gif à créer hauteur La hauteur du gif à créer [clear_colour] Facultatif] La couleur (RVB) pour effacer le gif. Les retours : Entier Exemple : si save_gif == true { if count == 0 { gif_image = gif_open(room_width, room_height) ; } else if count < 30 { gif_add_surface(gif_image, application_surface, 6/100) ; } else { gif_save(gif_image, "GameCapture.gif") ; count = 0 ; save_gif = false ; } count++ ; } Le code ci-dessus va créer un fichier image GIF avec 30 images prises sur la surface de l'application, puis l'enregistrer.
