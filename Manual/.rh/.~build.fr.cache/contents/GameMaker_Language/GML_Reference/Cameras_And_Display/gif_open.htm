gif_open Avec cette fonction, vous pouvez créer une image au format GIF vide, prête à recevoir des données. Lorsque vous appelez la fonction, vous devez spécifier la largeur et la hauteur du GIF (en pixels, et il n'y a pas de limite supérieure de taille à l'exception de la mémoire disponible), et la fonction renverra la valeur d'identifiant unique utilisée pour identifier le cadeau dans fonctions suivantes, ou il retournera -1 si le gif n'a pas pu être initialisé (par exemple, si la largeur / hauteur est trop grande pour la mémoire disponible). Vous pouvez également spécifier un argument facultatif pour définir la "couleur claire" du GIF. Il s'agit d'une valeur RVB (pas de composante alpha) et effacera le gif de cette couleur avant que quoi que ce soit ne soit ajouté. Si vous ne fournissez pas d'argument de couleur claire, la couleur par défaut du noir sera utilisée. Notez que lorsque vous utilisez cette fonction, vous devez appeler gif_save() pour terminer la création du GIF avant d'ouvrir un autre fichier pour l'enregistrement (donc pour chaque gif_open() il doit y avoir un accompagnement gif_save() ). Syntaxe:gif_open(width, height, [clear_colour]); Argument Description width La largeur du gif à créer height La hauteur du gif à créer [clear_colour] [ Facultatif ] La couleur (RVB) pour effacer le gif Retour:Integer Exemple:if save_gif == true { if count == 0     {     gif_image = gif_open(room_width, room_height);     } else if count < 30     {     gif_add_surface(gif_image, application_surface, 6/100);     } else     {     gif_save(gif_image, "GameCapture.gif");     count = 0;     save_gif = false;     } count++; } Le code ci-dessus créera un fichier image GIF avec 30 images prises à partir de la surface de l'application, puis l'enregistrera.
