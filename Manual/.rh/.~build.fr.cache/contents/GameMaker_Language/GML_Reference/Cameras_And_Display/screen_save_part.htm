screen_save_part Avec cette fonction, vous pouvez enregistrer une capture d'écran d'une partie du jeu telle qu'elle apparaît actuellement dans l'affichage du jeu. Les coordonnées de sélection sont basées sur la taille absolue de l'affichage (ou de la fenêtre si elle n'est pas en plein écran), donc (0,0) est toujours le coin supérieur gauche de l'écran (ou de la fenêtre). Voici une image qui montre comment cette fonction fonctionne: Le fichier sera enregistré dans le répertoire de travail du jeu. Il est important de noter que la fonction est conçue pour être appelée dans l' événement de fin de l'interface graphique Draw, ce qui devrait donner des résultats cohérents sur toutes les plates-formes. Son utilisation dans tout autre événement peut ne pas fonctionner comme prévu et donner des résultats différents sur différentes cibles et même sur des appareils. REMARQUE: cette fonction ne fonctionne pas sur la cible HTML5. Syntaxe:screen_save_part(fname,x,y,w,h); Argument Description fname Le nom de fichier de la capture d'écran à enregistrer x La coordonnée x du coin supérieur gauche de la zone de l'écran à capturer y La coordonnée y du coin supérieur gauche de la zone de l'écran à capturer w La largeur de la zone de l'écran à capturer h La hauteur de la zone de l'écran à capturer Retour:N/A Exemple:if keyboard_check_pressed(ord("S"))     {     screen_save_part("Screens\Screen_" + string(num++) + ".png", 0, 0, 100, 100)     } Le code ci-dessus prend une capture d'écran d'une partie de l'écran de jeu qui est de 100 pixels carrés lorsque l'utilisateur appuie sur la touche «S». Cette capture d'écran est enregistrée dans un dossier spécifique et numérotée à l'aide d'une variable qui est ensuite incrémentée.
