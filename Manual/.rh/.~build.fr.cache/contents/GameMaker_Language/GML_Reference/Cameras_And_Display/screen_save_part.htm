écran_save_part Cette fonction vous permet d'enregistrer une capture d'écran d'une partie du jeu telle qu'elle apparaît actuellement dans l'affichage du jeu. Les coordonnées de sélection sont basées sur la taille absolue de l'écran (ou de la fenêtre s'il ne s'agit pas d'un écran complet), donc (0,0) est toujours le coin supérieur gauche de l'écran (ou de la fenêtre). gauche de l'écran (ou de la fenêtre). L'image ci-dessous illustre le fonctionnement de cette fonction : Le fichier sera sauvegardé dans le répertoire de travail du le jeu. Il est important de noter que la fonction est conçue pour être appelée dans l'événement Draw GUI End Event, ce qui devrait donner des résultats cohérents sur toutes les plateformes. L'utiliser dans un autre événement peut ne pas fonctionner comme prévu et donner des résultats différents selon les cibles et même les appareils. résultats différents selon les cibles et même les appareils. NOTE : Cette fonction ne fonctionne pas sur la cible HTML5. Syntaxe : screen_save_part(fname,x,y,w,h) ; Arguments Description fname Le nom de fichier de la capture d'écran à enregistrer. x La coordonnée x du coin supérieur gauche de la zone de l'écran à capturer. y La coordonnée y du coin supérieur gauche de la zone de l'écran à capturer. w La largeur de la zone de l'écran à capturer h La hauteur de la zone de l'écran à capturer Les retours : N/A Exemple : if keyboard_check_pressed(ord("S")) { screen_save_part("Screens\Screen_" + string(num++) + ".png", 0, 0, 100, 100) } Le code ci-dessus prend une capture d'écran d'une partie de l'écran du jeu qui fait 100 pixels carrés lorsque l'utilisateur appuie sur la touche "S". Cette capture d'écran est enregistrée dans un dossier spécifique et numérotée à l'aide d'une variable qui est ensuite incrémentée.
