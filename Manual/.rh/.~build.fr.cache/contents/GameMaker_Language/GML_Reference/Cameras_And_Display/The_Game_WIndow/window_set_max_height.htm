fenêtre_set_max_height Cette fonction peut être utilisée pour définir une hauteur de fenêtre maximale pour votre jeu. Si vous activez l'option de redimensionnement de la fenêtre dans les options de jeu de la plate-forme cible, le joueur peut redimensionner la fenêtre du jeu à la taille qu'il souhaite. vous pouvez limiter la hauteur maximale à la taille que vous spécifiez. Si vous souhaitez revenir au comportement par défaut (c'est-à-dire aucun minimum), utilisez une valeur de -1. NOTE: Cette fonction n'est disponible que sur le Windows, macOS et Linux plateformes. Syntaxe : window_set_max_height(height) ; Arguments Description hauteur La hauteur maximale en pixels de la fenêtre de jeu. Les retours : N/A Exemple : window_set_min_width(640) ; window_set_min_height(480) ; window_set_max_width(1280) ; window_set_max_height(960) ; Le code ci-dessus permet de définir la largeur et la hauteur minimales et maximales de la fenêtre de jeu.
