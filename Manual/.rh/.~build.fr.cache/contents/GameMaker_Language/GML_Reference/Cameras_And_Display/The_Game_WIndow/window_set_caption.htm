window_set_caption Avec cette fonction, vous pouvez modifier ou définir la légende de la fenêtre de la pièce dans laquelle vous vous trouvez actuellement. Cette légende apparaît en haut de la fenêtre, à côté de l'icône du jeu, lorsque le jeu n'est pas en mode plein écran. Syntaxe:window_set_caption(caption); Argument Description caption La nouvelle légende. Retour:N/A Exemple:if window_get_caption() != ""     {     window_set_caption("");     } Le code ci-dessus vérifiera la légende de Windows pour voir si elle contient du texte ou non, et si elle le fait, elle la définit sur une chaîne vide afin qu'aucune légende ne soit affichée.
