window_views_mouse_get_x Cette fonction renvoie la coordonnée x de la souris par rapport à toutes les vues actives et renvoie la même valeur mouse_x. REMARQUE: pour les fonctions normales de la souris, reportez-vous à la section sur la saisie de la souris. Syntaxe:window_views_mouse_get_x(); Retour:Real Exemple:myx = window_views_mouse_get_x(); Cela définirait myx sur la coordonnée x de la souris dans la fenêtre par rapport à toutes les vues actives.
