fenêtre_get_couleur Cette fonction renvoie la couleur d'arrière-plan de la fenêtre de jeu. Cette couleur représente celle qui sera utilisée pour les zones de la fenêtre de jeu qui ne sont pas occupées par des vues. L'image suivante illustre cela : L'image ci-dessus comporte deux vues avec deux ports de vue, chacun dessiné à des positions différentes. La fenêtre de jeu est étirée pour accueillir les deux ports et la couleur de la fenêtre est utilisée pour colorer l'arrière-plan lorsqu'aucune vue n'est affichée. Syntaxe : window_get_colour() ; Les retours : N/A Exemple : if window_get_colour() != c_black { window_set_colour(c_black) ; } Le code ci-dessus vérifie la couleur de la fenêtre pour voir si elle est noire ou non, et si elle ne l'est pas, il la met en noir.
