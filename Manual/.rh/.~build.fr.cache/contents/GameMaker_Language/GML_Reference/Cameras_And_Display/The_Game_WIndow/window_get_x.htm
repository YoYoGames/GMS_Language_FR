fenêtre_get_x Avec cette fonction, vous pouvez obtenir la position x (en pixels) dans le navigateur s'il s'agit d'un jeu HTML5 ou dans l'affichage s'il s'agit d'un jeu Windows, Ubuntu(Linux) ou macOS. Syntaxe : window_get_x() ; Les retours : Real Exemple : wx = window_get_x() ; Le code ci-dessus stocke la position actuelle de la fenêtre sur l'axe x dans la variable "wx".
