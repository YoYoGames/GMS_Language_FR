fenêtre_vue_souris_get_y Cette fonction renvoie la coordonnée y de la souris par rapport à toutes les vues actives et renvoie la même valeur mouse_y. REMARQUE: Pour les fonctions normales de la souris, voir la section sur la saisie de la souris. Syntaxe : window_views_mouse_get_y() ; Les retours : Real Exemple : myy = window_views_mouse_get_y() ; Ceci définit myy comme la coordonnée y de la souris dans la fenêtre par rapport à toutes les vues actives.
