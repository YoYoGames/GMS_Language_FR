window_get_height Avec cette fonction, vous pouvez obtenir la hauteur actuelle (en pixels) de la fenêtre de jeu. Syntaxe:window_get_height(); Retour:Real Exemple:hh = window_get_height(); Le code ci-dessus stockera la hauteur actuelle de la fenêtre dans la variable "hh".
