fenêtre_a_focus Avec cette fonction, vous pouvez interroger l'état de la fenêtre (ou de l'onglet) et si elle perd le focus, la fonction retournera false, sinon elle retournera true. Dans la plupart des cas, vous pouvez simplement utiliser la os_is_paused() fonction mais dans certains cas très spécifiques (par exemple les jeux sur Chrome Apps), cette fonction ne se déclenchera pas, auquel cas vous devez utiliser cette fonction à la place. NOTE: Cette fonction n'est valable que pour le HTML5, Windows, et MacOS. Syntaxe : window_has_focus() ; Les retours : Booléen Exemple : if !window_has_focus() { scr_PauseGame() ; } Le code ci-dessus vérifiera si la fenêtre de jeu est en focus ou non, et si la fonction renvoie false, un script sera appelé.
