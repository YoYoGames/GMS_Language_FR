window_set_fullscreen Avec cette fonction, vous pouvez définir la fenêtre de jeu en plein écran (true ) ou non (false ). Veuillez noter que pour la macOS, vous devez avoir décoché l'option "Démarrer en plein écran" et coché l'option "Autoriser le joueur à redimensionner la fenêtre de jeu" dans les Options de jeu, sinon cette fonction échouera. Notez également que cette fonction ne fonctionnera pas sur HTML5 à moins d'être ajoutée en tant que rappel "cliquable" (voir ici pour plus de détails). Syntaxe:window_set_fullscreen(full); Argument Description full Indique s'il faut définir l'écran en plein écran (vrai) ou non (faux). Retour:N/A Exemple:if mouse_check_button_pressed(mb_left)     {     if window_get_fullscreen()         {         window_set_fullscreen(false);         }     else         {         window_set_fullscreen(true);         }     } Le code ci-dessus vérifie que vous appuyez sur le bouton de la souris, puis définit la fenêtre en plein écran si ce n'est pas déjà fait, ou la définit sur fenêtré si elle l'est déjà.
