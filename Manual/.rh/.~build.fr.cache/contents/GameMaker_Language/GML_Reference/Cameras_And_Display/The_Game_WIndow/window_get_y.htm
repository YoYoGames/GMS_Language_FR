window_get_y Avec cette fonction, vous pouvez obtenir la position y (en pixels) dans le navigateur s'il s'agit d'un jeu HTML5 ou dans l'affichage s'il s'agit d'un jeu Windows, Ubuntu ( Linux ) ou macOS. Syntaxe:window_get_y(); Retour:Real Exemple:wy = window_get_y(); Le code ci-dessus stocke la position actuelle de la fenêtre de l'axe y dans la variable "wy".
