window_device Cette fonction renverra le pointeur de périphérique d3d actuel, que vous pourrez ensuite (par exemple) transmettre à une DLL ou Dylib sous Windows et macOS. REMARQUE: cette fonction est obsolète dans GameMaker Studio 2 au profit de os_get_info (). Syntaxe:window_device(); Retour:Pointer Exemple:gfx_pointer = window_device() Le code ci-dessus stockera le pointeur de périphérique d3d dans une variable.
