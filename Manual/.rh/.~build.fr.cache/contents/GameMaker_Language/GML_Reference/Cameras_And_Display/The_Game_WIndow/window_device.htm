périphérique de fenêtre Cette fonction renvoie le pointeur de périphérique d3d actuel, que vous pouvez ensuite (par exemple) transmettre à une DLL ou à une Dylib sous Windows et macOS. REMARQUE: Cette fonction a été abandonnée dans GameMaker Studio 2 au profit de la fonction os_get_info(). Syntaxe : window_device() ; Les retours : Pointeur Exemple : gfx_pointer = window_device() Le code ci-dessus stockera le pointeur du périphérique d3d dans une variable.
