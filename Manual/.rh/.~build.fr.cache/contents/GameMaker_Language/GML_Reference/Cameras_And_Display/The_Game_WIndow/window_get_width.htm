window_get_width Avec cette fonction, vous pouvez obtenir la largeur actuelle (en pixels) de la fenêtre de jeu. Syntaxe:window_get_width(); Retour:Real Exemple:ww = window_get_width(); Le code ci-dessus stockera la largeur actuelle de la fenêtre dans la variable "ww".
