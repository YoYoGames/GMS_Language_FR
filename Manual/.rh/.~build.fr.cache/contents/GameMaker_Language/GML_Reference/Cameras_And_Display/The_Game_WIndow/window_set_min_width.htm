window_set_min_width Cette fonction peut être utilisée pour définir une largeur de fenêtre minimale pour votre jeu. Si vous activez l'option de redimensionnement de la fenêtre dans les Options de jeu pour la plate-forme cible, le joueur peut redimensionner la fenêtre de jeu à la taille de son choix, mais en utilisant cette fonction, vous pouvez limiter la largeur minimale à la taille que vous spécifiez. Si vous souhaitez revenir au comportement par défaut (c'est-à-dire: pas de minimum), utilisez une valeur de -1. REMARQUE: cette fonction n'est disponible que sur les Windows -formes macOS et Linux Syntaxe:window_set_min_width(width); Argument Description width La largeur minimale en pixels de la fenêtre de jeu. Retour:N/A Exemple:window_set_min_width(640); window_set_min_height(480); window_set_max_width(1280); window_set_max_height(960); Le code ci-dessus définira la largeur et la hauteur minimale et maximale de la fenêtre de jeu.
