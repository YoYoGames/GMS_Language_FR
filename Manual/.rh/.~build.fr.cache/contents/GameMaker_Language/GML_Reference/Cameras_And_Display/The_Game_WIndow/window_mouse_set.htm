fenêtre_souris_set Cette fonction vous permet de modifier ou de définir la position de la souris dans la fenêtre du jeu, ce qui peut être utile pour les jeux FPS, par exemple. La fonction ne fonctionne que lorsque le jeu est en cours et l'utilisation de alt + tab déverrouille la souris. NOTE: Pour les fonctions normales de la souris, voir la section sur la saisie de la souris. Syntaxe : window_mouse_set(x, y) ; Arguments Description x La coordonnée x dans la fenêtre. y La coordonnée y dans la fenêtre. Les retours : N/A Exemple : window_mouse_set(window_get_width() / 2, window_get_height() / 2) ; Le code ci-dessus permet de centrer la souris dans la fenêtre du jeu.
