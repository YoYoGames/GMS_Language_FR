window_center Avec cette fonction, vous pouvez centrer la fenêtre de jeu dans l'affichage lorsque le module cible est Windows, Ubuntu ( Linux ) ou macOS, ou vous pouvez le centrer dans le navigateur si le module cible est HTML5. Cette fonction n'a aucun effet sur aucun autre appareil. REMARQUE: Si vous avez redimensionné la fenêtre du jeu (par exemple, en passant du plein écran à la fenêtre), et que vous souhaitez centrer la nouvelle fenêtre, cette fonction doit être appelée dans une alarme [0] au moins un pas après la taille de la fenêtre changer, sinon cela ne fonctionnera pas correctement. Syntaxe:window_center(); Retour:N/A Exemple:if !window_get_fullscreen()     {     window_center();     } Le code ci-dessus centrera la fenêtre de jeu dans l'affichage si elle n'est pas en mode plein écran.
