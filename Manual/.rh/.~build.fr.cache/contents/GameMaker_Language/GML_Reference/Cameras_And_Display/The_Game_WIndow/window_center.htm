fenêtre_centre Avec cette fonction, vous pouvez centrer la fenêtre de jeu dans l'affichage lorsque le module cible est Windows, Ubuntu(Linux) ou macOS, ou vous pouvez la centrer dans le navigateur si le module cible est HTML5. Cette fonction n'a aucun effet sur les autres appareils. REMARQUE: si vous avez redimensionné la fenêtre du jeu (par exemple, en passant du plein écran au fenêtrage) et que vous souhaitez centrer la nouvelle fenêtre, cette fonction doit être appelée dans une alarme[0] au moins une étape après le changement de taille de la fenêtre, sinon sinon elle ne fonctionnera pas correctement. Syntaxe : window_center() ; Les retours : N/A Exemple : if !window_get_fullscreen() { window_center() ; } Le code ci-dessus permet de centrer la fenêtre du jeu dans l'écran si celui-ci n'est pas en mode plein écran.
