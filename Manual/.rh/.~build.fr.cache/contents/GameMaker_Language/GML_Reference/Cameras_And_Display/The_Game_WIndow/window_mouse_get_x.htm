fenêtre_souris_get_x Cette fonction permet d'obtenir la position x du curseur de la souris (en pixels) dans le navigateur s'il s'agit d'un jeu HTML5 ou dans l'affichage s'il s'agit d'un jeu Windows, Ubuntu(Linux) ou macOS. NOTE: Pour les fonctions normales de la souris, voir la section sur la saisie de la souris. Syntaxe : window_mouse_get_x() ; Les retours : Real Exemple : wx = window_mouse_get_x() ; Le code ci-dessus stocke la position actuelle de la souris sur l'axe x de la fenêtre dans la variable "wx".
