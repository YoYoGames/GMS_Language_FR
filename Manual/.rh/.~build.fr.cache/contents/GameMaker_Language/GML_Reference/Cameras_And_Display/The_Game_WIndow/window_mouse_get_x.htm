window_mouse_get_x Avec cette fonction, vous pouvez obtenir la position x du curseur de la souris (en pixels) dans le navigateur s'il s'agit d'un jeu HTML5 ou dans l'affichage s'il s'agit d'un jeu Windows, Ubuntu ( Linux ) ou macOS. REMARQUE: pour les fonctions normales de la souris, voir la section sur l' entrée de la souris Syntaxe:window_mouse_get_x(); Retour:Real Exemple:wx = window_mouse_get_x(); Le code ci-dessus stocke la position actuelle de la fenêtre sur l'axe x de la souris dans la variable "wx".
