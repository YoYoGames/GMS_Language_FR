fenêtre_handle Cette fonction permet d'obtenir la valeur interne de l'identifiant Windows (le HWND, un pointeur). Cette fonction n'est vraiment utile que pour les auteurs d'extensions qui ont besoin du handle de la fenêtre pour appeler les API Windows dans le code DLL (le pointeur renvoyé doit être converti en une chaîne de caractères, puis dans le C++, il suffit de le convertir en HWND). retourné doit être converti en une chaîne de caractères, puis dans le C++, il suffit de le convertir en un HWND). Le tableau ci-dessous montre les plates-formes supportées ainsi que ce qu'elles retournent : Plate-forme Renvoie à Windows Fenêtre HWND macOS Classe NSWindow Ubuntu(Linux) Manche de XWindow HTML5 ID du canevas Syntaxe : window_handle() ; Les retours : Pointeur Exemple : win_id = windows_handle() ; Le code ci-dessus stockera l'identifiant de la fenêtre de jeu dans une variable.
