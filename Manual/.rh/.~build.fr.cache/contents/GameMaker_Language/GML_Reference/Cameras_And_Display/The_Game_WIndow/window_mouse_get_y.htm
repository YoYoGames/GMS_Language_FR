fenêtre_souris_get_y Avec cette fonction, vous pouvez obtenir la position y du curseur de la souris (en pixels) dans le navigateur s'il s'agit d'un jeu HTML5 ou dans l'affichage s'il s'agit d'un jeu Windows, Ubuntu(Linux) ou macOS. NOTE: Pour les fonctions normales de la souris, voir la section sur la saisie de la souris. Syntaxe : window_mouse_get_y() ; Les retours : Real Exemple : wy = window_mouse_get_y() ; Le code ci-dessus stocke la position actuelle de la fenêtre sur l'axe y de la souris dans la variable "wy".
