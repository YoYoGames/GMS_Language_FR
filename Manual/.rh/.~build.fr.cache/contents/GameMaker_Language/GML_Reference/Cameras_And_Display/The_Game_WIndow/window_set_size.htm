fenêtre_set_size Cette fonction vous permet de modifier la taille de la fenêtre de jeu. Si vous utilisez cette fonction pour modifier la taille de la fenêtre de jeu, le contenu de la fenêtre sera mis à l'échelle pour s'adapter avec un rapport de 1:1 à la toile de fond, ce qui permettra de conserver une image nette et claire (bien que mise à l'échelle si la fenêtre est différente de la taille de la pièce ou de la vue). et claire (bien qu'elle soit redimensionnée si la fenêtre n'a pas la taille de la pièce ou de la vue). Par exemple, si vous avez une fenêtre de 200 x 150 pixels, et que le joueur étire la fenêtre en faisant glisser les bords, la taille finale de la fenêtre peut être de 400 x 300 pixels, mais le canevas de dessin est TOUJOURS de 200 x 150. est TOUJOURS de 200 x 150, ce qui donne une image floue car le rapport des pixels est maintenant de 1:2. Toutefois, si vous utilisez cette fonction pour définir la taille de la fenêtre, le rapport 1:1 est maintenu, même si l'image est mise à l'échelle pour s'adapter à la nouvelle taille de la fenêtre. Syntaxe : window_set_size( w, h ) ; Arguments Description w La nouvelle largeur de la fenêtre. h La nouvelle hauteur de la fenêtre. Les retours : N/A Exemple : if mouse_check_button_pressed(mb_left) { global.scale += 1 ; window_set_size(200 * global.scale, 150 * global.scale) ; } Le code ci-dessus modifie la taille de la fenêtre en fonction de la valeur de la variable "global.scale" lorsque le bouton de la souris est enfoncé.
