window_get_caption Cette fonction renvoie la légende de la fenêtre (c'est le texte qui apparaît en haut de la fenêtre, à côté de son icône) et par défaut cela montre la légende de la pièce dans laquelle vous vous trouvez actuellement. Syntaxe:window_get_caption(); Retour:String Exemple:if window_get_caption() != ""     {     window_set_caption("");     } Le code ci-dessus vérifiera la légende de Windows pour voir si elle contient du texte ou non, et si elle le fait, elle la définit sur une chaîne vide afin qu'aucune légende ne soit affichée.
