fenêtre_set_couleur Cette fonction permet de définir la couleur d'arrière-plan de la fenêtre de jeu. Cette couleur représente celle qui sera utilisée pour les zones de la fenêtre de jeu qui ne sont pas occupées par des vues. L'image suivante illustre cela : L'image ci-dessus comporte deux vues avec deux ports de vue, chacun dessiné à des positions différentes. Le site étire la fenêtre du jeu pour qu'elle puisse accueillir les deux ports et utilise la couleur de la fenêtre pour colorer l'arrière-plan lorsqu'aucune vue n'est affichée. Syntaxe : window_set_colour(colour) ; Arguments Description couleur La couleur pour définir la région. Les retours : N/A Exemple : if window_get_colour() != c_black { window_set_colour(c_black) ; } Le code ci-dessus vérifie la couleur de la fenêtre pour voir si elle est noire ou non, et si elle ne l'est pas, il la met en noir.
