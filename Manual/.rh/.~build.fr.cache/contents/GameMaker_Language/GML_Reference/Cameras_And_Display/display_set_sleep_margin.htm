display_set_sleep_margin Cette fonction peut être utilisée pour définir la valeur de la marge de sommeil utilisée pour la synchronisation du rendu de votre jeu et nécessite une valeur en millisecondes. Les valeurs par défaut sont les suivantes: Plate-forme Millisecondes Windows dix macOS dix Ubuntu dix HTML5 dix Android 4 iOS 4 Windows UWP dix Xbox dix PS4 dix Switch Nintendo dix Notez que même si la méthode de synchronisation est définie pour utiliser la synchronisation vsync, la marge de sommeil aura un effet sur le rendu du jeu, et vous devez donc faire attention lorsque vous définissez cette valeur. Pour plus d'informations sur la durée d'affichage, veuillez consulter display_set_timing_method(). REMARQUE: en plus de la marge de veille, vous pouvez contrôler davantage vos performances et votre consommation d'énergie sous Windows en ajustant la résolution du planificateur de threads runtime de l'exécution. Syntaxe:display_set_sleep_margin(milliseconds); Argument Description milliseconds Le nombre de millisecondes pour la marge de sommeil Retour:N/A Exemple:if display_get_timing_method() != tm_sleep { display_set_timing_method(tm_sleep); if display_get_sleep_margin() != 20     {     display_set_sleep_margin(20);     } } Le code ci-dessus vérifiera la méthode de chronométrage, puis s'il n'est pas défini sur tm_sleep il sera réglé sur cela et la marge de sommeil réglée sur 20.
