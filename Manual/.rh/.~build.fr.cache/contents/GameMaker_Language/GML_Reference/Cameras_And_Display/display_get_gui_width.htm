display_get_gui_width Avec cette fonction, vous pouvez obtenir la largeur (en pixels) de l'interface graphique telle qu'elle est utilisée dans l' événement Draw GUI. Syntaxe:display_get_gui_width(); Retour:Real Exemple:ads_move(display_get_gui_width() - ads_get_display_width(0), 0, 0); Le code ci-dessus définira une publicité à afficher en haut à droite de l'écran.
