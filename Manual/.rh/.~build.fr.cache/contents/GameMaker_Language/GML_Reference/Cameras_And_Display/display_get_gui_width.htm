afficher_get_gui_width Cette fonction permet d'obtenir la largeur (en pixels) de l'interface graphique telle qu'elle est utilisée dans l'événement Draw GUI. Syntaxe : display_get_gui_width() ; Les retours : Real Exemple : ads_move(display_get_gui_width() - ads_get_display_width(0), 0, 0) ; Le code ci-dessus permet d'afficher une publicité en haut à droite de l'écran.
