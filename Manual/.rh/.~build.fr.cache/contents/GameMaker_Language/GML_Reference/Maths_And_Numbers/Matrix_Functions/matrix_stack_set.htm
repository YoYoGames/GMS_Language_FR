matrix_stack_set Cette fonction écrase le sommet actuel de la pile de matrices par la matrice spécifiée. Syntaxe:matrix_stack_set(matrix); Argument Description matrix L'index de matrice à utiliser. Retour:N/A Exemple:var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1); matrix_stack_set(m); Le code ci-dessus construira une nouvelle matrice et stockera l'index de matrice résultant dans la variable "m" avant de remplacer le haut de la pile de matrices par elle.
