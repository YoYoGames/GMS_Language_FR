matrix_build Cette fonction peut être utilisée pour créer votre propre matrice personnalisée et renverra une valeur d'index pour cette nouvelle matrice qui devrait être stockée dans une variable pour référence et utilisation futures. La matrice elle-même est créée en tant que tableau de 16 valeurs où les 4 premiers éléments sont la ligne 1, les 4 seconds éléments sont la ligne 2 etc... d'une matrice 4x4. REMARQUE: lorsque vous créez une nouvelle matrice de cette manière, l'ordre de fonctionnement est YXZ. Syntaxe:matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale); Argument Description x La composante x du vecteur de traduction. y La composante y du vecteur de traduction. z La composante z du vecteur de traduction. xrotation L'angle de rotation autour de l'axe x (en degrés °). yrotation L'angle de rotation autour de l'axe y (en degrés °). xzotation L'angle de rotation autour de l'axe z (en degrés °). xscale Le montant de l'échelle x. yscale Le montant de l'échelle y. zscale Le montant de l'échelle z. Retour:Matrix index (4x4 Matrix) Exemple:t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1); Le code ci-dessus construira une nouvelle transformation matricielle et stockera l'index matriciel résultant dans la variable "t_matrix".
