matrix_get Cette fonction renvoie un tableau de 16 valeurs, correspondant au type de matrice 4x4 donné, où la ligne 1 correspond aux éléments [0 - 3], la ligne 2 aux éléments [4-7], etc... (voir l'image sur la page principale ). Les matrices disponibles sont la vue, la projection et le monde, pour lesquelles vous utiliseriez l'une des constantes suivantes: Constant Description matrix_view La matrice de vue actuelle matrix_projection La matrice de projection actuelle matrix_world La matrice mondiale actuelle Syntaxe:matrix_get(type); Argument Description type Le type de matrice pour obtenir les valeurs de (voir les constantes listées ci-dessus) Retour:Array (16 elements) Exemple:v_array = matrix_get(matrix_view); Le code ci-dessus obtiendra les valeurs de la matrice de vue courante et remplira le tableau "v_array" avec elles.
