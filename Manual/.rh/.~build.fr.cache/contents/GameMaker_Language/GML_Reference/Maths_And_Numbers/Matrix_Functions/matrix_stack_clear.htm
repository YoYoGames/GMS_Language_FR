matrix_stack_clear Cette fonction efface la pile de la matrice. Syntaxe : matrix_stack_clear() ; Les retours : N/A Exemple : if !matrix_stack_is_empty() { matrix_stack_clear() ; } Le code ci-dessus vérifie la pile de la matrice pour voir si elle est vide et si elle ne l'est pas, il l'efface.
