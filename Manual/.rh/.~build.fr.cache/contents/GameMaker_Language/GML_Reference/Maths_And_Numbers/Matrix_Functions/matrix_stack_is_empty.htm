matrix_stack_is_empty Cette fonction peut être utilisée pour vérifier si la pile de matrices est vide (renvoie true) ou non (renvoie false). Syntaxe : matrix_stack_is_empty() ; Les retours : Booléen Exemple : if !matrix_stack_is_empty() { matrix_stack_clear() ; } Le code ci-dessus vérifie la pile de la matrice pour voir si elle est vide et si elle ne l'est pas, il l'efface.
