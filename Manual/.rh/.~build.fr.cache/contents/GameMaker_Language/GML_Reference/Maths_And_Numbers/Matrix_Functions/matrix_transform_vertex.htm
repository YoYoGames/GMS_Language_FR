matrice_transformation_vertex Cette fonction transforme un vecteur par une matrice. Vous fournissez l'ID de la matrice de transformation (tel que retourné par la fonction matrice build()), ainsi que les valeurs x, y et z du vecteur à transformer. La fonction renvoie un tableau de 3 éléments où : tableau[0] = x tableau[1] = y tableau[2] = z. Syntaxe : matrix_transform_vertex(matrice, x, y, z) ; Arguments Description matrice La matrice à utiliser x La composante x du vecteur de transformation y La composante y du vecteur de transformation z La composante z du vecteur de transformation Les retours : Tableau (3 éléments) Exemple : t_matrix = matrix_build(0, 0, 0, 0, 90, 0, 1, 2, 1) ; verts = matrix_transform_vertex(t_matrix, x, y, z) ; Le code ci-dessus transforme les valeurs données en utilisant la matrice stockée dans la variable "t_matrix" et les stocke dans le tableau "verts".
