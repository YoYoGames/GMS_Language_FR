delta_time Le but du delta timing est d'éliminer les effets du décalage ou de la lenteur des ordinateurs qui essaient de gérer des graphiques complexes ou beaucoup de code. C'est une valeur qui peut être ajoutée à la vitesse des objets pour qu'ils finissent par se déplacer à la même vitesse, quel que soit le décalage. Ceci est réalisé dans GameMaker Studio 2 en utilisant la variable en lecture seule delta_time chaque étape car il mesure le temps qui s'est écoulé entre une étape et la suivante en microsecondes (1 microseconde équivaut à 1 000 000e de seconde). Par conséquent, la variable delta_time peut être utilisé pour calculer la vitesse à laquelle (par exemple) un personnage du jeu doit se déplacer pour compenser un pic de décalage dans le jeu. Syntaxe:delta_time Retour:Real (integer) Exemple:var _dt = delta_time / 1000000; speed = spd * dt; Le code ci-dessus définira la vitesse de l'instance en utilisant le delta-time pour corriger le décalage. La variable "_dt" serait utilisée pour contenir la valeur de temps delta précédemment calculée afin que le rapport puisse être utilisé pour multiplier la valeur de vitesse de base (contenue dans la variable "spd") et ainsi obtenir une vitesse cohérente pour l'instance.
