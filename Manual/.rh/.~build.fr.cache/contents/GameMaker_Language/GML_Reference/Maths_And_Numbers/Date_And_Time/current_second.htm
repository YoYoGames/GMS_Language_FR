current_second Cette variable en lecture seule renverra les secondes qui correspondent au moment actuel. Syntaxe:current_second; Retour:Real Exemple:draw_text(32, 32, "The time is " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second)); Le code ci-dessus dessinerait l'heure internationale actuelle sur l'écran.
