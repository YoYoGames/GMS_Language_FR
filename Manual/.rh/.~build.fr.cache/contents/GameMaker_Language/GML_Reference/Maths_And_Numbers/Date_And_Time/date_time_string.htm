date_heure_chaîne Avec cette fonction, vous pouvez créer une chaîne contenant l'heure donnée, formatée pour le système ou le périphérique qui exécute le jeu au moment où la fonction est appelée. Syntaxe : date_time_string( date ) ; Arguments Description date La date à utiliser. Les retours : Chaîne de caractères Exemple : str = date_time_string(date_current_datetime()) ; Cette opération donne à la variable donnée la valeur "11:36.00", en fonction des paramètres du système pour l'affichage de la date et de l'heure, ainsi que de la date et de l'heure actuelles.
