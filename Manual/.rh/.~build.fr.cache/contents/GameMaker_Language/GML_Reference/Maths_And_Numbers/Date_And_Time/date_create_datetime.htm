date_création_date Cette fonction va créer une valeur de date à partir du composant donné en argument. Syntaxe : date_create_datetime(année, mois, jour, heure, minute, seconde) ; Arguments Description année L'année à fixer. mois Le mois à définir. jour Le jour à régler. heure L'heure à régler. minute La minute à régler. deuxième Le deuxième à mettre en place. Les retours : Real Exemple : mydatetime = date_create_datetime(2011, 9, 15, 9, 43, 30) ; Cette opération définit "mydatetime" à la date et à l'heure données et stocke la valeur renvoyée dans la variable "mydatetime".
