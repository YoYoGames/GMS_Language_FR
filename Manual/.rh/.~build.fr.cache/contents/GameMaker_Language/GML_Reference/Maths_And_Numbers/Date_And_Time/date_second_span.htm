date_second_span Cette fonction permet d'obtenir le nombre de secondes entre deux dates. La valeur de retour est toujours positive et sera un nombre entier. Syntaxe : date_second_span(date1, date2) ; Arguments Description date1 La première date. date2 La date à laquelle il faut la comparer. Les retours : Real Exemple : diff = date_second_span(date_create_datetime(2011, 9, 15, 11, 4, 0 ), date_current_datetime()) ; Ainsi, "diff" correspondrait au nombre de secondes entre le 15 septembre 2011, 11:04 et la date et l'heure actuelles.
