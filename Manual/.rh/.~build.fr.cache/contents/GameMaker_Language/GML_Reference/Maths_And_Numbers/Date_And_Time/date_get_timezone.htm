date_get_timezone Obtient le fuseau horaire de base utilisé pour toutes les autres fonctions de date et d'heure. Ce fuseau horaire peut être soit local (tel que défini par le système) soit UTC, et la fonction renverra l'une des constantes suivantes : Constant Description fuseau horaire_local utiliser le fuseau horaire local tel que défini par le système fuseau horaire_utc utiliser le temps universel coordonné Syntaxe : date_get_timezone() ; Les retours : Constant Exemple : if (date_get_timezone() != timezone_utc) { date_set_timezone(timezone_utc) ; } Ce code vérifie le paramètre de base du fuseau horaire pour le jeu et s'il n'est pas UTC, il le change.
