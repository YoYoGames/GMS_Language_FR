current_year Cette variable en lecture seule renverra l'année en cours. Syntaxe:current_year Retour:Real Exemple:draw_text((32, 32, "Today is " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +"."); Le code ci-dessus dessinera le jour, le mois et l'année.
