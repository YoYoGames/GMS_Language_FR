heure actuelle Cette variable en lecture seule renvoie le nombre de millisecondes qui se sont écoulées depuis le début du jeu. Syntaxe : current_time ; Les retours : Real Exemple : if (current_time > 600000) { msg = show_question_async("Would you like to rate ?") ; } Le code ci-dessus vérifie si plus de 10 minutes se sont écoulées avant de poser une question à l'utilisateur.
