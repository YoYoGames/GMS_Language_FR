date_jours_dans_l'année Avec cette fonction, vous pouvez obtenir le nombre de jours que compte l'année donnée, en retournant 365 pour une année normale, et 366 pour une année bissextile. Syntaxe : date_days_in_year(date) ; Arguments Description date La date à utiliser. Les retours : Real Exemple : days = date_days_in_year(date_current_datetime()) ; Ainsi, "days" correspondrait au nombre de jours de l'année en cours.
