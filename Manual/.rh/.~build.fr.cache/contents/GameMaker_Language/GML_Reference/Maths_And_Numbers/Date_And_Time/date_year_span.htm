date_année_span Cette fonction permet d'obtenir le nombre d'années entre deux dates. Cette valeur est toujours positive, et les années incomplètes seront retournées sous forme de fraction. Syntaxe : date_année_span(date1, date2) ; Arguments Description date1 La première date. date2 La date à laquelle il faut la comparer. Les retours : Real Exemple : diff = date_year_span(date_create_datetime(2011, 9, 15, 11, 4, 0), date_current_datetime()) ; Cela correspondrait au nombre d'années entre le 15 septembre 2011, 11:04.0 et la date et l'heure actuelles.
