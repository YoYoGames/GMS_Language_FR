date_week_span Cette fonction permet d'obtenir le nombre de semaines entre deux dates. Cette valeur est toujours positive, et les semaines incomplètes seront retournées sous forme de fraction. Syntaxe : date_week_span(date1, date2) ; Arguments Description date1 La première date. date2 La date à laquelle il faut la comparer. Les retours : Real Exemple : diff = date_week_span( date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime() ) ; Ainsi, "diff" correspondrait au nombre de semaines entre le 15 septembre 2011, 11:04.0 et la date et l'heure actuelles.
