date_week_span Avec cette fonction, vous pouvez obtenir le nombre de semaines entre deux dates. Cette valeur est toujours positive et les semaines incomplètes seront renvoyées sous forme de fraction. Syntaxe:date_week_span(date1, date2); Argument Description date1 La première date / heure. date2 La date-heure à laquelle la comparer. Retour:Real Exemple:diff = date_week_span( date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime() ); Cela définirait "diff" sur le nombre de semaines entre le 15 septembre 2011, 11: 04.0 et la date et l'heure actuelles.
