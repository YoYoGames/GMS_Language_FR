date_get_jour_de_l'année Cette fonction renvoie le jour (de 1 à 366) dans l'année de la date donnée. Syntaxe : date_get_day_of_year( date ) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : mydayyear = date_get_day_of_year(date_current_datetime()) ; Ceci définit mydayyear comme le jour actuel de l'année.
