date_leap_year Cette fonction retournera true si la composante année de la valeur datetime donnée est une année bissextile ou false autrement. Cela peut être une fonction pratique pour des choses comme les œufs de Pâques dans vos jeux ou pour déverrouiller un contenu spécial. Syntaxe:date_leap_year(date); Argument Description date La date / heure à utiliser. Retour:Boolean Exemple:if date_leap_year(date_current_datetime())     {     if !global.ExtraContent         {         global.ExtraContent = true;         }     } Le code ci-dessus vérifiera la date / heure actuelle pour voir si l'année est une année bissextile ou non.
