date_année bissextile Cette fonction renvoie true (vrai) si la composante année de la valeur datetime donnée est une année bissextile, false (faux) sinon. Cette fonction peut être pratique pour des choses comme les œufs de Pâques dans vos jeux, ou pour débloquer du contenu spécial. Syntaxe : date_leap_year(date) ; Arguments Description date La date à utiliser. Les retours : Booléen Exemple : if date_leap_year(date_current_datetime()) { if !global.ExtraContent { global.ExtraContent = true ; } } Le code ci-dessus va vérifier la date actuelle pour voir si l'année est bissextile ou non.
