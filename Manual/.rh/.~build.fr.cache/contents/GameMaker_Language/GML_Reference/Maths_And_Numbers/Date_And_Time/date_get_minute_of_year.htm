date_get_minute_de_l'année Cette fonction renvoie la minute de la valeur de date donnée dans l'année (à partir du nombre total de minutes pour l'année, en tenant compte des années bissextiles). Syntaxe : date_get_minute_of_year(date) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : myminuteyear = date_get_minute_of_year(date_current_datetime()) ; Ceci définirait myminuteyear à la minute courante de l'année.
