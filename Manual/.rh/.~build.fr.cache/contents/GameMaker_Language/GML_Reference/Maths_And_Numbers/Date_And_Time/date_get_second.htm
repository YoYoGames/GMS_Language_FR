date_get_second Cette fonction renvoie la seconde de la valeur de la date donnée. Syntaxe : date_get_second(date) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : mysecond = date_get_second(date_current_datetime()) ; Cela mettrait ma seconde à la seconde actuelle.
