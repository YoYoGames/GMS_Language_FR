date_get_second Cette fonction renvoie la seconde de la valeur datetime donnée. Syntaxe:date_get_second(date); Argument Description date La date / heure à vérifier. Retour:Real Exemple:mysecond = date_get_second(date_current_datetime()); Cela définirait mysecond sur la seconde actuelle.
