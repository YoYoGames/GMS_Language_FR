date_get_day Cette fonction renvoie le jour (de 1 à 31) de la date donnée. Syntaxe : date_get_day(date) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : myday = date_get_day( date_current_datetime() ) ; Cela mettrait "myday" à la date du jour.
