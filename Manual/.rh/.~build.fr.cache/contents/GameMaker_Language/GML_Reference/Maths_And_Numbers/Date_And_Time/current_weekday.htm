jour de la semaine en cours Cette variable en lecture seule renvoie le jour de la semaine sous forme de valeur, où le dimanche est égal à 0 et le samedi à 6. Syntaxe : jour de la semaine en cours ; Les retours : Real Exemple : var day ; switch(current_weekday) { cas 0 : jour = "dimanche" ; pause ; cas 1 : jour = "lundi" ; pause ; cas 2 : jour = "mardi" ; pause ; cas 3 : jour = "mercredi" ; pause ; cas 4 : jour = "jeudi" ; pause ; cas 5 : jour = "vendredi" ; pause ; cas 6 : jour = "samedi" ; pause ; } draw_text(32, 32, "Today is " + day +".") ; Le code ci-dessus utilise la valeur current_weekday pour définir une variable avec le jour correct en texte, puis l'affiche pour que l'utilisateur puisse le voir.
