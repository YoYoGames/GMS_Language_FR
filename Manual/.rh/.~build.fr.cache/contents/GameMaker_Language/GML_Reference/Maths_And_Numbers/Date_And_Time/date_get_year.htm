date_get_year Cette fonction renvoie l'année de la date donnée. Syntaxe : date_get_year(date) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : monannée = date_get_year(date_current_datetime()) ; Ceci mettrait "myyear" à l'année en cours.
