date_inc_second Avec cette fonction, vous pouvez incrémenter une valeur de date / heure donnée d'un nombre spécifique de secondes, et elle renverra la nouvelle valeur de date / heure. Syntaxe:date_inc_second(date, amount); Argument Description date Date / heure à ajouter. amount Le nombre de secondes (doit être un entier) à ajouter. Retour:Real Exemple:mynewdatetime = date_inc_second(date_current_datetime(), 60); Cela définirait "mynewdatetime" à la date actuelle, mais avec 60 secondes supplémentaires.
