current_month Cette variable en lecture seule renvoie le mois en cours sous forme de valeur numérique, 1 étant janvier et 12 décembre. Syntaxe:current_month; Retour:Real Exemple:draw_text((32, 32, "Today is " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +"."); Le code ci-dessus dessinera le jour, le mois et l'année.
