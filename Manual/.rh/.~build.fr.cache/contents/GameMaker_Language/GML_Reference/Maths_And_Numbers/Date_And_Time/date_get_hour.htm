date_get_hour Cette fonction renvoie l'heure de la valeur datetime donnée. Syntaxe:date_get_hour(date); Argument Description date La date / heure à vérifier. Retour:Real Exemple:myhour = date_get_hour(date_current_datetime()); Cela définirait "myhour" à l'heure actuelle.
