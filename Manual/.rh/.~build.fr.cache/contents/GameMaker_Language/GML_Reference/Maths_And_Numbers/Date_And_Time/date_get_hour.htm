date_get_hour Cette fonction renvoie l'heure de la valeur de la date donnée. Syntaxe : date_get_hour(date) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : monheure = date_get_hour(date_current_datetime()) ; Ainsi, "myhour" sera réglé sur l'heure actuelle.
