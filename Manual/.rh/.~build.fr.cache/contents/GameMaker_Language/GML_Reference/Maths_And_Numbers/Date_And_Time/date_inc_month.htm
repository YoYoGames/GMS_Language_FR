date_inc_month Avec cette fonction, vous pouvez incrémenter une valeur de date / heure donnée d'un nombre spécifique de mois, et elle renverra la nouvelle valeur de date / heure. Syntaxe:date_inc_month( date, amount ); Argument Description date Date / heure à ajouter. amount Le nombre de mois (doit être un entier) à ajouter. Retour:Real Exemple:mynewdatetime = date_inc_month(date_current_datetime(), 12); Cela définirait "mynewdatetime" à la date actuelle, mais avec 12 mois ajoutés.
