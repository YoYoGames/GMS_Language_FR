date_inc_mois Avec cette fonction, vous pouvez incrémenter une valeur de date donnée d'un nombre spécifique de mois, et elle renverra la nouvelle valeur de date. Syntaxe : date_inc_month( date, montant ) ; Arguments Description date La date à laquelle il faut ajouter. montant Le nombre de mois (doit être un nombre entier) à ajouter. Les retours : Real Exemple : mynewdatetime = date_inc_month(date_current_datetime(), 12) ; Ceci définit "mynewdatetime" à la date actuelle mais avec 12 mois ajoutés.
