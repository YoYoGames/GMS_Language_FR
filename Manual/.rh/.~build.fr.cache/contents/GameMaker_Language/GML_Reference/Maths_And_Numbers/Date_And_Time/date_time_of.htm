date_heure_de Renvoie la valeur temporelle de l'heure donnée. L'heure renvoyée ignore l'heure d'été (et donc le temps universel) et doit normalement être utilisée en conjonction avec une autre fonction de gestion de la date et de l'heure. Syntaxe : date_heure_de(date) ; Arguments Description date La date à partir de laquelle l'heure doit être extraite. Les retours : Real Exemple : time = date_time_of(date_current_datetime()) ; Cela renvoie uniquement l'heure actuelle et stocke la valeur dans la variable "time".
