date_inc_heure Avec cette fonction, vous pouvez incrémenter une valeur de date donnée d'un nombre d'heures spécifique, et elle renverra la nouvelle valeur de date. Syntaxe : date_inc_hour(date, montant) ; Arguments Description date La date à laquelle il faut ajouter. montant Le nombre d'heures (doit être un nombre entier) à ajouter. Les retours : Real Exemple : mynewdatetime = date_inc_hour(date_current_datetime(), 24) ; Ceci définit mynewdatetime comme la date actuelle, mais avec 24 heures en plus.
