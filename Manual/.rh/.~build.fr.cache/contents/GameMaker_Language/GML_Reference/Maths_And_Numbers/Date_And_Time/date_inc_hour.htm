date_inc_hour Avec cette fonction, vous pouvez incrémenter une valeur de date / heure donnée d'un nombre spécifique d'heures, et elle renverra la nouvelle valeur de date / heure. Syntaxe:date_inc_hour(date, amount); Argument Description date Date / heure à ajouter. amount Le nombre d'heures (doit être un entier) à ajouter. Retour:Real Exemple:mynewdatetime = date_inc_hour(date_current_datetime(), 24); Cela définirait mynewdatetime à la date actuelle, mais avec 24 heures supplémentaires.
