date_compare_datetime Avec cette fonction, vous pouvez vérifier deux dates et heures pour voir laquelle est antérieure ou postérieure à l'autre. La fonction renvoie -1 si la date1 est antérieure, 1 si la date1 est postérieure et 0 s'il s'agit de la même chose. un 0 pour la correspondance est très rare (il faut littéralement que ce soit la même seconde du même jour du même mois de la même année pour retourner un 0). Syntaxe : date_compare_datetime(date1, date2) ; Arguments Description date1 La première date. date2 La date à laquelle il faut la comparer. Les retours : Real Exemple : d = date_compare_datetime(date_create_datetime(2011, 9, 15, 11, 4, 0), date_current_datetime()) ; Cela donnerait à d la valeur correspondante en fonction de la date la plus ancienne, probablement -1 puisque la date actuelle est plus avancée que le 15 septembre 2011.
