date_date_of Renvoie la valeur de date de la date / heure donnée. Syntaxe:date_date_of(date); Argument Description date Date / heure à partir de laquelle extraire la date. Retour:Real Exemple:today = date_date_of(date_current_datetime()); Cela renverrait la date actuelle uniquement et stockerait la valeur dans la variable «aujourd'hui».
