date_date_de Renvoie la valeur de la date de l'heure donnée. Syntaxe : date_date_de(date) ; Arguments Description date L'heure à partir de laquelle la date doit être extraite. Les retours : Real Exemple : today = date_date_of(date_current_datetime()) ; Cela renvoie uniquement la date du jour et stocke la valeur dans la variable "today".
