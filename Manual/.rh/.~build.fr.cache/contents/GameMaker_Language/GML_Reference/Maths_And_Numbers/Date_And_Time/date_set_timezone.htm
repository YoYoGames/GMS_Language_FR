date_set_timezone Définit le fuseau horaire de base à utiliser pour toutes les autres fonctions de date et d'heure. Ce fuseau horaire peut être soit local (tel que défini par le système) soit UTC, et vous utiliserez l'une des constantes suivantes pour définir lequel est utilisé (par défaut il s'agit de l'heure locale) : Constant Description fuseau horaire_local utiliser le fuseau horaire local tel que défini par le système fuseau horaire_utc utiliser le temps universel coordonné Syntaxe : date_set_timezone(timezone) ; Arguments Description fuseau horaire Le fuseau horaire à utiliser pour l'heure de base. Les retours : N/A Exemple : if (date_get_timezone() != timezone_utc) { date_set_timezone(timezone_utc) ; } Ce code vérifie le paramètre de base du fuseau horaire pour le jeu et s'il n'est pas UTC, il le change.
