date_get_minute Cette fonction renvoie la minute de la valeur de date donnée. Syntaxe : date_get_minute(date) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : myminute = date_get_minute(date_current_datetime()) ; Ceci définit "myminute" comme la minute actuelle.
