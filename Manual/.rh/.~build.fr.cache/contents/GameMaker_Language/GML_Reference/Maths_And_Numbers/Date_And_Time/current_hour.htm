heure_courante Cette variable en lecture seule renvoie l'heure qui correspond au moment présent en fonction du fuseau horaire par défaut du système (c'est-à-dire l'heure locale). Vous pouvez modifier le fuseau horaire de base à utiliser avec la fonction date_set_timezone(). Syntaxe : heure_actuelle ; Les retours : Real Exemple : draw_text(32, 32, "L'heure est " + string(current_hour) + " :" + string(current_minute) + "." + string(current_second))) ; Le code ci-dessus affiche l'heure internationale actuelle à l'écran.
