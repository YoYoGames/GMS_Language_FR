current_hour Cette variable en lecture seule renverra l'heure qui correspond au moment actuel en fonction du fuseau horaire par défaut du système (c'est-à-dire: l'heure locale). Vous pouvez modifier le fuseau horaire de base à utiliser avec la fonction date_set_timezone(). Syntaxe:current_hour; Retour:Real Exemple:draw_text(32, 32, "The time is " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second)); Le code ci-dessus dessinerait l'heure internationale actuelle sur l'écran.
