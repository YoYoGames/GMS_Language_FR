date_get_weekday Cette fonction renvoie la valeur du jour de la semaine de la date donnée. Il s'agira d'une valeur comprise entre 0 et 6. Syntaxe : date_get_weekday(date) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : monweekday = date_get_weekday(date_current_datetime()) ; Ainsi, "myweekday" sera défini comme le jour de la semaine en cours.
