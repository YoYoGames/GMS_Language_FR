minute en cours Cette variable en lecture seule renverra les minutes qui correspondent au moment actuel. Syntaxe : minute actuelle ; Les retours : Real Exemple : draw_text(32, 32, "L'heure est " + string(current_hour) + " :" + string(current_minute) + "." + string(current_second))) ; Le code ci-dessus affiche l'heure internationale actuelle à l'écran.
