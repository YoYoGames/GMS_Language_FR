date_days_in_month Avec cette fonction, vous pouvez obtenir le nombre de jours que compte le mois donné, soit 28, 29, 30 ou 31. Syntaxe:date_days_in_month(date); Argument Description date La date à utiliser. Retour:Real Exemple:days = date_days_in_month(date_current_datetime()); Cela définirait "jours" sur le nombre de jours du mois en cours.
