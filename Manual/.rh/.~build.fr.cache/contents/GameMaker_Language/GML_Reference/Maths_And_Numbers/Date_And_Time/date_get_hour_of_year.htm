date_get_hour_of_year Cette fonction renvoie l'heure de la valeur de date donnée dans l'année (à partir du nombre total d'heures pour l'année, en tenant compte des années bissextiles). Syntaxe : date_get_hour_of_year(date) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : myhouryear = date_get_hour_of_year(date_current_datetime()) ; Cela définirait "myhouryear" à l'heure actuelle de l'année.
