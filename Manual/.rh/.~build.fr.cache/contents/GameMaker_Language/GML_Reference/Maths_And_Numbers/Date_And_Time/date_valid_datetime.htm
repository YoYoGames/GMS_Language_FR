date_valide_datetime Avec cette fonction, vous pouvez vérifier une valeur de date-heure pour voir si elle est valide (renvoie true) ou non (renvoie false). Notez que cette fonction considère qu'une date est valide uniquement si elle est postérieure au 1/1/1970. retournera false, donc la date la plus proche que vous pouvez vérifier serait : date_valid_datetime(1970, 01, 01, 0, 0, 0) ; Syntaxe : date_valid_datetime(année, mois, jour, heure, minute, seconde) ; Arguments Description année L'année à vérifier. mois Le mois à vérifier. jour Le jour où il faut vérifier. heure L'heure à vérifier. minute La minute à vérifier. deuxième Le deuxième à vérifier. Les retours : Booléen Exemple : if date_valid_datetime(2011, 9, 15, 10, 3, 30) { mydatetime = date_create_datetime(2011, 9, 15, 10, 3, 30) ; } Ceci fixera "mydatetime" au 15 septembre 2011, 10:03.30, si c'est une valeur valide (ce qui est le cas).
