date_valid_datetime Avec cette fonction, vous pouvez vérifier une valeur datetime pour voir si elle est valide (renvoie true ) ou non (retourne false ). Notez que cette fonction ne considérera qu'une date / heure valide comme étant postérieure au 1/1/1970 et tout ce qui précède retournera false, donc le plus tôt que vous pouvez vérifier serait:date_valid_datetime(1970, 01, 01, 0, 0, 0); Syntaxe:date_valid_datetime(year, month, day, hour, minute, second); Argument Description year L'année à vérifier. month Le mois à vérifier. day Le jour pour vérifier. hour L'heure à vérifier. minute La minute pour vérifier. second Le second à vérifier. Retour:Boolean Exemple:if date_valid_datetime(2011, 9, 15, 10, 3, 30)     {     mydatetime = date_create_datetime(2011, 9, 15, 10, 3, 30);     } Cela définira "mydatetime" au 15 septembre 2011, 10: 03.30, s'il s'agit d'une valeur valide (ce qu'elle est).
