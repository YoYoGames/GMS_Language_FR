date_inc_minute Avec cette fonction, vous pouvez incrémenter une valeur de date donnée d'un nombre spécifique de minutes, et elle renverra la nouvelle valeur de date. Syntaxe : date_inc_minute( date, montant ) ; Arguments Description date La date à laquelle il faut ajouter. montant Le nombre de minutes (doit être un nombre entier) à ajouter. Les retours : Real Exemple : mynewdatetime = date_inc_minute(date_current_datetime(), 60) ; Cela définirait "mynewdatetime" à la date actuelle mais avec 60 minutes ajoutées.
