date_inc_day Avec cette fonction, vous pouvez incrémenter une valeur de date / heure donnée d'un nombre spécifique de jours, et elle renverra la nouvelle valeur de date / heure. Syntaxe:date_inc_day(date, amount); Argument Description date Date / heure à ajouter. amount Le nombre de jours (doit être un entier) à ajouter. Retour:Real Exemple:mynewdatetime = date_inc_day(date_current_datetime(), 365); Cela définirait "mynewdatetime" à la date actuelle, mais avec 365 jours ajoutés.
