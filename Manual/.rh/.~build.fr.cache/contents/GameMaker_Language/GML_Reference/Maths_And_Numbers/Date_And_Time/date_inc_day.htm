date_inc_jour Avec cette fonction, vous pouvez incrémenter une valeur de date donnée d'un nombre spécifique de jours, et elle renverra la nouvelle valeur de date. Syntaxe : date_inc_day(date, montant) ; Arguments Description date La date à laquelle il faut ajouter. montant Le nombre de jours (doit être un nombre entier) à ajouter. Les retours : Real Exemple : mynewdatetime = date_inc_day(date_current_datetime(), 365) ; Ainsi, "mynewdatetime" correspondrait à la date actuelle, mais avec 365 jours de plus.
