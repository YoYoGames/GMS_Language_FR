date_get_week Cette fonction renvoie la semaine de la valeur datetime donnée dans l'année. Syntaxe:date_get_week(date); Argument Description date La date / heure à vérifier. Retour:Real Exemple:myweek = date_get_week(date_current_datetime()); Cela définirait ma semaine sur la semaine en cours.
