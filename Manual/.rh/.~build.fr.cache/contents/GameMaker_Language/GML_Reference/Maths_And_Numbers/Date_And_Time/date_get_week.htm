date_get_week Cette fonction renvoie la semaine de la valeur de date donnée dans l'année. Syntaxe : date_get_week(date) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : ma semaine = date_get_week(date_current_datetime()) ; Ceci définit ma semaine comme la semaine en cours.
