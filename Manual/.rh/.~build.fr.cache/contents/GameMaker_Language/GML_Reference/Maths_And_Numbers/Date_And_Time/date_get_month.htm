date_get_month Cette fonction renvoie le mois de la valeur de la date donnée. Syntaxe : date_get_month(date) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : mymonth = date_get_month(date_current_datetime()) ; Ceci définit mymonth comme le mois en cours.
