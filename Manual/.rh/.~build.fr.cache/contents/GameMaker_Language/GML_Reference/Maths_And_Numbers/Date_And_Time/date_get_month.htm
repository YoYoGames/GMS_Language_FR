date_get_month Cette fonction renvoie le mois de la valeur datetime donnée. Syntaxe:date_get_month(date); Argument Description date La date / heure à vérifier. Retour:Real Exemple:mymonth = date_get_month(date_current_datetime()); Cela définirait mymonth sur le mois en cours.
