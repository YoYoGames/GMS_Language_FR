date_inc_année Avec cette fonction, vous pouvez incrémenter une valeur de date donnée d'un nombre spécifique d'années, et elle renverra la nouvelle valeur de date. Syntaxe : date_inc_year(date, montant) ; Arguments Description date La date à ajouter à montant Le nombre d'années (doit être un nombre entier) à ajouter. Les retours : Real Exemple : mynewdatetime = date_inc_year(date_current_datetime(), 1000) ; Ainsi, mynewdatetime correspondrait à la date actuelle, mais avec 1000 ans de plus.
