date_inc_semaine Avec cette fonction, vous pouvez incrémenter une valeur de date donnée d'un nombre spécifique de semaines, et elle renverra la nouvelle valeur de date. Syntaxe : date_inc_week(date, montant) ; Arguments Description date La date à laquelle il faut ajouter. montant Le nombre de semaines (doit être un nombre entier) à ajouter. Les retours : Real Exemple : mynewdatetime = date_inc_week(date_current_datetime(), 52) ; Ainsi, mynewdatetime correspondrait à la date actuelle, mais avec 52 semaines ajoutées.
