date_date_string Cette fonction permet de créer une chaîne contenant l'heure donnée, formatée pour le système ou le périphérique qui exécute le jeu au moment où la fonction est appelée. Syntaxe : date_datetime_string(date) ; Arguments Description date La date à utiliser. Les retours : Chaîne de caractères Exemple : str = date_datetime_string(date_current_datetime()) ; Cela donnerait à "str" la valeur "15 septembre 2011, 11:33.00", selon les paramètres du système pour l'affichage de la date et de l'heure et la date et l'heure actuelles.
