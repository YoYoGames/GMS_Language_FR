jour_courant Cette variable en lecture seule renvoie le jour sous la forme d'une valeur comprise entre 1 et 31, en fonction du mois. Syntaxe : jour_courant ; Les retours : Real Exemple : draw_text(32, 32, "Today is " + string(current_day) + "/" + string(current_month) + "/" + string(current_year) + ".") ; Le code ci-dessus affichera le jour, le mois et l'année.
