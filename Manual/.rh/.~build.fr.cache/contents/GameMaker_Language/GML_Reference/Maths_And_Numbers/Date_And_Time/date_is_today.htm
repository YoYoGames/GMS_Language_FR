date_est_aujourd'hui Cette fonction renvoie true si la valeur de date donnée correspond au jour où elle est vérifiée (c'est-à-dire aujourd'hui), ou false sinon. Cette fonction peut être pratique pour des choses comme les œufs de Pâques dans vos jeux, ou pour débloquer du contenu saisonnier. Notez que cette fonction sera affectée par le fuseau horaire défini (par défaut, l'heure locale), que vous pouvez modifier à l'aide de la fonction date_set_timezone() fonction. Syntaxe : date_is_today(date) ; Arguments Description date La date à utiliser. Les retours : Booléen Exemple : if date_is_today(global.Halloween) { global.Max_Levels = 200 ; } Le code ci-dessus vérifie la date stockée dans la variable globale "Halloween" pour voir si c'est aujourd'hui, et si c'est le cas, il attribue une nouvelle valeur à une autre variable globale.
