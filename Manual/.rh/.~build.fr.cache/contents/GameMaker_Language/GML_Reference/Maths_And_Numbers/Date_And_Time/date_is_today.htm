date_is_today Cette fonction retournera true si la valeur datetime donnée est le jour où elle est vérifiée (c'est-à-dire: aujourd'hui), ou false autrement. Cela peut être une fonction pratique pour des choses comme les œufs de Pâques dans vos jeux ou pour débloquer du contenu saisonnier. Notez que cette fonction sera affectée par le fuseau horaire défini (la valeur par défaut est l'heure locale) que vous pouvez modifier à l'aide du date_set_timezone() fonction. Syntaxe:date_is_today(date); Argument Description date La date / heure à utiliser. Retour:Boolean Exemple:if date_is_today(global.Halloween)     {     global.Max_Levels = 200;     } Le code ci-dessus vérifiera la date / heure stockée dans la variable globale "Halloween" pour voir si elle est aujourd'hui, et si c'est le cas, elle définit une autre variable globale sur une nouvelle valeur.
