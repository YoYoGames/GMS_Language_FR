date_get_second_of_year Cette fonction renvoie la seconde de la valeur de date donnée dans l'année (à partir du nombre total de secondes pour l'année, en tenant compte des années bissextiles). Syntaxe : date_get_second_of_year( date ) ; Arguments Description date La date à vérifier. Les retours : Real Exemple : mysecondyear = date_get_second_of_year(date_current_datetime()) ; Ainsi, "mysecondyear" correspondrait à la seconde de l'année en cours.
