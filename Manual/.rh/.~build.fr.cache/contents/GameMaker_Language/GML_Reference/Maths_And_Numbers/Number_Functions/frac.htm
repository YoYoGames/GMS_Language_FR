frac Cette fonction renvoie la partie fractionnaire de n, c'est-à-dire la partie derrière le point décimal. Il ne renverra que les décimales derrière le point d'une valeur, donc frac(3.125) renverra 0,125, frac(6.921) renverra 0.921, etc... Syntaxe:frac(n); Argument Description n Le nombre à changer. Retour:Real Exemple:val = frac(3.4); Cela définira val à 0,4. Retour: Fonctions numériques Suivant:abs © Copyright YoYo Games Ltd. 2021 Tous droits réservés
