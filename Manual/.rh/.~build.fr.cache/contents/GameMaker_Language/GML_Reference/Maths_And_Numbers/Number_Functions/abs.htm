abs Cette fonction renvoie la valeur absolue de l'argument d'entrée, donc si c'est une valeur positive, elle restera la même, mais si elle est négative, elle sera multipliée par -1 pour la rendre positive. Syntaxe : abs(val) Arguments Description val Le numéro à tourner en absolu. Les retours : Real Exemple : x += abs (x - mouse_x) Cela ajoutera une quantité égale à la valeur absolue de la position x actuelle de l'instance et de la position x de la souris.
