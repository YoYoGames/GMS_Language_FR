sol Renvoie le plancher de n, c'est-à-dire n arrondi à un nombre entier inférieur. Cette fonction est similaire à la fonction round() mais elle n'arrondit qu'à l'unité inférieure, quelle que soit la valeur décimale. Ainsi, floor(5.99999) renverra 5, de même que floor(5.2), floor(5.6457) etc... Syntaxe : plancher (n) ; Arguments Description n Le numéro de l'étage. Les retours : Real Exemple : val = floor( 3.9 ) ; La valeur de Val sera ainsi fixée à 3. Retour : Fonctions des chiffres Suivant : frac Copyright YoYo Games Ltd. 2021 Tous droits réservés
