moyenne Cette fonction fonctionne en additionnant toutes les valeurs d'entrée, puis en les divisant par leur propre nombre. Vous pouvez avoir autant d'arguments que vous le souhaitez (notez que plus d'arguments signifie que la fonction sera plus lente à analyser). Ainsi, mean(2, 6, 9, 32) renvoie 12,25 car 2+6+9+32=49 et 49/4=12,25. Syntaxe : mean(val1, val2, ... max_val) ; Arguments Description val0 ... max_val Les valeurs à comparer. Les retours : Real Exemple : xmiddle = mean(obj_player1.x, obj_player2.x, obj_player3.x) ; ymiddle = mean(obj_player1.y, obj_player2.y, obj_player3.y) ; Ceci définira xmiddle et ymiddle aux coordonnées x et y de la moyenne des coordonnées des trois objets joueurs, obj_player1, obj_player2 et obj_player3. Vous pourriez, par exemple, utiliser cette fonction pour que la caméra du jeu se concentre sur les trois joueurs au lieu d'un seul.
