random_get_seed Pour générer un nombre aléatoire, GameMaker Studio 2 démarre avec un nombre de départ aléatoire. Cette fonction vous permet de récupérer cette graine, qui peut ensuite être stockée pour reproduire une série spécifique d'événements aléatoires. Par exemple, supposons que vous ayez un jeu généré de façon mais que vous voulez que le joueur puisse retourner dans des zones précédemment découvertes. Au début de chaque zone, vous pouvez stocker la graine aléatoire avec cette fonction et ainsi, chaque fois que vous devez y retourner, vous pouvez utiliser la fonction random_set_seed() pour remettre la graine à ce qu'elle était auparavant et ainsi forcer GameMaker Studio 2 à reproduire le même ensemble de fonctions "aléatoires" qu'il utilisait précédemment. Syntaxe : random_get_seed() ; Les retours : Real Exemple : ini_open("Save.ini") ini_write_real("Levels", string(level), random_get_seed()) ; ini_close() ; Le code ci-dessus va obtenir la graine aléatoire et la stocker dans un fichier ini.
