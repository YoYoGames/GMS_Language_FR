min Cette fonction renvoie le minimum des valeurs d'entrée, qui peuvent être aussi nombreuses que vous le souhaitez (notez que plus d'arguments signifie que la fonction sera plus lente à analyser). Par exemple, min(12, 96, 32, 75) renverra 12, car c'est la valeur la plus basse de toutes les valeurs d'entrée. de toutes les valeurs d'entrée. Syntaxe : min(val1, val2, ... max_val) ; Arguments Description val0 ... max_val Les valeurs à comparer. Les retours : Real Exemple : x = min(x, room_width) ; Cela empêchera le joueur de sortir par la droite de la pièce. Cela fonctionne en fixant constamment son x à lui-même ou à la variable room_width, la plus petite des deux. Si le joueur sort par la droite, son x sera plus grand que la variable room_width, donc il sera remis en place.
