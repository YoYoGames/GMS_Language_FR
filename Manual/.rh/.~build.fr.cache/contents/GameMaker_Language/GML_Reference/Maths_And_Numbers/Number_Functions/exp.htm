exp Il s'agit de la fonction power(e,n), où e est approximativement 2,718281828 (également connu sous le nom de nombre d'Euler), et n est le nombre de fois qu'il doit être multiplié par lui-même. En gros, cela veut dire "combien de croissance j'obtiens après n unités de temps (et une croissance continue de 100%) ? (et une croissance continue de 100%)". Ainsi, exp(3) donnerait 20,08, ce qui signifie qu'après 3 unités de temps, nous avons 20,08 fois la quantité avec laquelle nous avons commencé. Syntaxe : exp(n) ; Arguments Description n Le pouvoir de l'e. Les retours : Real Exemple : val = exp(2) ; Cela fixera la valeur à e^2, soit environ 7,4.
