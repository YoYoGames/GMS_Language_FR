exp Il s'agit de la puissance de la fonction (e, n), où e est environ 2,718281828 (également connu sous le nom de nombre d'Euler ), et n est le nombre de fois où il doit être multiplié par lui-même. Cela dit essentiellement "combien de croissance est-ce que j'obtiens après n unités de temps (et 100% de croissance continue)?". Alors exp(3) retournerait 20,08, ce qui signifie qu'après 3 unités de temps, nous avons 20,08 fois le montant avec lequel nous avons commencé. Syntaxe:exp(n); Argument Description n La puissance de e. Retour:Real Exemple:val = exp(2); Cela mettra val à e ^ 2, autour de 7,4.
