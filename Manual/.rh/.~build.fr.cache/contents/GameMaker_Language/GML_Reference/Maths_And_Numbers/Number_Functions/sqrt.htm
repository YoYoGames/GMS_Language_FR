sqrt Si vous multipliez un nombre avec lui-même, vous obtenez le carré de ce nombre, mais parfois vous voulez faire l'inverse et obtenir la racine carrée d'un nombre. Donc, pour trouver quel nombre a été multiplié avec lui-même pour obtenir une valeur positive donnée, nous utilisons cette fonction. Par exemple:sqrt(9) renverra 3 puisque 3 * 3 = 9. Syntaxe:sqrt(val); Argument Description val Le nombre dont il faut obtenir la racine carrée. Retour:Real Exemple:num = sqrt(exp); Le code ci-dessus définira la variable "num" pour contenir la racine carrée de la valeur contenue dans "exp".
