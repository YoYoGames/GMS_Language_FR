choose Parfois, vous souhaitez spécifier autre chose que des nombres pour une sélection aléatoire, ou les nombres que vous voulez ne sont pas dans un ordre réel ou dans une plage définie. Dans ces cas, vous utiliseriez choose() pour générer un résultat aléatoire. Par exemple, disons que vous voulez créer un objet avec un sprite aléatoire au début, vous pouvez alors utiliser cette fonction pour définir l'index du sprite sur l'un d'un ensemble de sprites donnés. Notez que vous pouvez en avoir autant que vous le souhaitez (notez que plus d'arguments signifiera que la fonction sera plus lente à analyser). REMARQUE: Cette fonction renverra la même valeur à chaque fois que le jeu est réexécuté car GameMaker Studio 2 génère la même graine aléatoire initiale à chaque fois pour rendre le code de débogage beaucoup plus facile. Pour éviter ce comportement, utilisez randomise () au début de votre partie. Syntaxe:choose(val0, val1, val2... max_val); Argument Description val0... max_val Une valeur d'entrée qui peut être une chaîne, un entier, une variable ou une constante. Retour:One of the given arguments Exemple:sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub); hp = choose(5, 8, 15, 32, 40); name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen"); Le code ci-dessus utilise choisir de définir un certain nombre de propriétés pour l'instance.
