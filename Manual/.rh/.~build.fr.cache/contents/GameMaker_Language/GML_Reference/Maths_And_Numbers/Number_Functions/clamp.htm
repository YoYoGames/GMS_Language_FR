pince Cette fonction vous permet de maintenir une valeur d'entrée entre une plage spécifiée. Syntaxe : clamp(val, min, max) Arguments Description val La valeur à fixer. min La valeur minimale à fixer entre les deux. max La valeur maximale à fixer entre les deux. Les retours : Real Exemple : vitesse = clamp(vitesse, 1, 10) ; Le code ci-dessus bloquera la vitesse de sorte qu'elle ne descende jamais en dessous de 1 ou ne dépasse pas 10.
