log10 Cette fonction est similaire à la log2(n) fonction, seulement ça log10(n) obtient le nombre de 10 dont nous avons besoin pour multiplier ensemble pour obtenir n. Alors log10(100) renvoie 2, car multiplier 10 par lui-même donne 100, log10(1000) renvoie 3 comme 10x10x10 = 1000 etc... Le nombre ne doit pas nécessairement être divisible par dix, mais cela retournera toujours une valeur appropriée, par exemple si nous avons log10(101) nous obtenons 2.00432137... qui est le nombre de fois que vous pouvez diviser 101 par 10. Syntaxe:log10(n); Argument Description n La valeur d'entrée. Retour:Real Exemple:logval = log10(num); Le code ci-dessus obtient le log10 de la valeur stockée dans "num".
