sqr Multiplie un nombre par lui-même et renvoie ainsi le carré de ce nombre. PAR EXEMPLE:sqr(5) retournerait 25 puisque 5 * 5 = 25. Syntaxe:sqr(val); Argument Description val Le nombre au carré. Retour:Real Exemple:score += sqr(dmg); Le code ci-dessus ajoutera le carré de la variable "dmg" au score.
