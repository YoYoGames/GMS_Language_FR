sqr Multiplie un nombre par lui-même et renvoie donc le carré de ce nombre. Par exemple, sqr(5) renvoie 25 puisque 5*5=25. Syntaxe : sqr(val) ; Arguments Description val Le nombre à mettre au carré. Les retours : Real Exemple : score += sqr(dmg) ; Le code ci-dessus ajoutera le carré de la variable "dmg" au score.
