irandom_range Cette fonction est similaire à random_range() uniquement avec des valeurs entières comme entrée. Vous fournissez la valeur basse pour la plage ainsi que la valeur haute, et la fonction renverra une valeur entière aléatoire dans (et y compris) la plage donnée. Par exemple, irandom_range(10, 35) renverra un entier compris entre 10 et 35 inclus. Comme avec le irandom() fonction, des nombres réels peuvent être utilisés, auquel cas ils seront arrondis à l'entier inférieur le plus proche EG: irandom_range (6.2,9.9) donnera une valeur entre 6 et 9. REMARQUE: Cette fonction renverra la même valeur à chaque fois que le jeu est réexécuté car GameMaker: Studio génère la même graine aléatoire initiale à chaque fois pour rendre le code de débogage beaucoup plus facile. Pour éviter ce comportement, utilisez randomise () au début de votre partie. Syntaxe:irandom_range(n1, n2); Argument Description n1 L'extrémité inférieure de la plage à partir de laquelle le nombre aléatoire sera sélectionné. n2 L'extrémité supérieure de la plage à partir de laquelle le nombre aléatoire sera sélectionné. Retour:Integer Exemple:score += irandom_range(500, 600); Cela ajoutera une valeur entière comprise entre 500 et 600 (inclus) au score total.
