randomiser Cette fonction fixe la graine à une valeur aléatoire. Si vous avez besoin de conserver une valeur constante sur un certain nombre d'exécutions d'un jeu, vous devriez utiliser la fonction random_set_seed(). Veuillez noter que lorsque vous utilisez les fonctions de nombre aléatoire dans GameMaker Studio 2, la graine initiale est toujours la même, ce qui facilite grandement le dépistage des erreurs et le débogage. Si vous souhaitez tester avec un vrai nombre aléatoire, vous devez appeler cette fonction au début de votre jeu. La fonction renvoie la nouvelle valeur de la graine aléatoire (un nombre entier 32 bits non signé). Syntaxe : randomise() ; // ou randomize() ; Les retours : Entier (valeur non signée de 32 bits) Exemple : randomiser() ; Le code ci-dessus va randomiser la graine.
