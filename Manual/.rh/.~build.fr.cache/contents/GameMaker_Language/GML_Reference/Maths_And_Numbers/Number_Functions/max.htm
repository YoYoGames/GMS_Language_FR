max Cette fonction renvoie le maximum des valeurs d'entrée, dont elle peut avoir jusqu'à 16. Par exemple max(12, 96, 32, 75) renverra 96 car c'est la plus élevée de toutes les valeurs d'entrée. Syntaxe:max(val1, val2, ... max_val); Argument Description val0... max_val Les valeurs à comparer. Retour:Real Exemple:x = max(x, 0); Cela empêchera le joueur de quitter la gauche de la pièce. Cela fonctionne en définissant constamment son x sur lui-même ou sur 0, selon la valeur la plus élevée. Si le joueur sort par la gauche, son x sera plus petit que 0 (c'est-à-dire négatif), donc il sera remis en place.
