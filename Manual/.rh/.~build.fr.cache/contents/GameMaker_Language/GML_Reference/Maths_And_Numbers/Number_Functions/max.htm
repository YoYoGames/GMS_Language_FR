max Cette fonction renvoie le maximum des valeurs d'entrée, qui peuvent être au nombre de 16. Par exemple, max(12, 96, 32, 75) renverra 96, car c'est la valeur la plus élevée de toutes les valeurs d'entrée. Syntaxe : max(val1, val2, ... max_val) ; Arguments Description val0 ... max_val Les valeurs à comparer. Les retours : Real Exemple : x = max(x, 0) ; Cela empêchera le joueur de sortir par la gauche de la pièce. Cela fonctionne en réglant constamment son x sur lui-même ou sur 0, selon ce qui est le plus grand. Si le joueur sort par la gauche, son x sera plus petit que 0 (c'est-à-dire négatif), donc il sera remis à zéro. retour.
