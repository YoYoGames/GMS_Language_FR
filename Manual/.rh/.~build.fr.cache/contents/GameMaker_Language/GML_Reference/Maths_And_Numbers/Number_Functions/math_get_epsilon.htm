math_get_epsilon Cette fonction renvoie la valeur actuelle de l'epsilon pour la plate-forme cible. Pour plus d'informations sur l'epsilon, veuillez consulter la fonction math_set_epsilon(). Syntaxe : math_get_epsilon() ; Les retours : Real Exemple : var e = math_get_epsilon() ; if e !.= 0.000001 { math_set_epsilon(0.000001) ; } Ceci récupère la valeur actuelle de l'epsilon et la stocke dans une variable locale, qui est ensuite vérifiée et un nouvel epsilon est défini si nécessaire.
