math_get_epsilon Cette fonction renverra la valeur epsilon actuelle pour la plate-forme cible. Pour plus d'informations sur epsilon, veuillez consulter la fonction math_set_epsilon(). Syntaxe:math_get_epsilon(); Retour:Real Exemple:var e = math_get_epsilon(); if e != 0.000001     {     math_set_epsilon(0.000001);     } Cela récupérera la valeur epsilon actuelle et la stockera dans une variable locale, qui est ensuite vérifiée et un nouvel ensemble epsilon si nécessaire.
