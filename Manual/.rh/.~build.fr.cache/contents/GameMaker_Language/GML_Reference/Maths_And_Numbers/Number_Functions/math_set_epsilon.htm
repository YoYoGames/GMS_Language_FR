math_set_epsilon Epsilon est une valeur utilisée pour déterminer si deux nombres sujets à des erreurs d'arrondi sont suffisamment proches pour être considérés comme "égaux". Il est utile dans le cadre des mathématiques à virgule flottante, car il permet de réduire les "erreurs d'arrondi" qui font que certaines opérations renvoient des valeurs qui semblent incorrectes ou contraires à ce que l'on attend. opérations renvoient des valeurs qui semblent incorrectes ou contraires à ce que l'on attend. Par exemple, nous pouvons avoir ajouté une valeur à l'indice d'image d'un sprite et nous attendre à ce que le résultat soit un entier unique, mais en raison de la nature des mathématiques à virgule flottante, la valeur finale réelle est quelque chose comme 5. valeur finale réelle finit par être quelque chose comme 5.0000002, donc lorsque nous avons la vérification suivante : si image_index == 5 { //faire quelque chose } Le code ne se comporte pas comme prévu et renvoie false. Toutefois, si nous définissons la valeur d'epsilon à 0,000001, la valeur image_index sera arrondie au nombre réel le plus proche qui se situe à +/- 0,000001 de la valeur d'origine, ce qui fait que la comparaison ci-dessus renvoie vrai. La valeur epsilon sera utilisée pour tous les opérateurs entiers suivants : < : Moins de > : plus grand que == : égal à <= : Inférieur ou égal à >= : Supérieur ou égal à != : Non égal à Notez que la définition d'une valeur d'epsilon de 0 désactivera tous les arrondis, et l'utilisation d'une valeur de 1 donnera une erreur. Syntaxe : math_set_epsilon(epsilon) ; Arguments Description epsilon La nouvelle valeur d'epsilon (de 0 à 0,999999999). Les retours : Real Exemple : math_set_epsilon(0.0001) ; Ceci définira la valeur d'epsilon pour toutes les autres opérations en virgule flottante.
