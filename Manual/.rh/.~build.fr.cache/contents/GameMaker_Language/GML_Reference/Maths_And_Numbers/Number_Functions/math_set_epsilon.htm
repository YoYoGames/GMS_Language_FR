math_set_epsilon Epsilon est une valeur utilisée pour déterminer si deux nombres sujets à une erreur d'arrondi sont suffisamment proches pour être considérés comme "égaux". Il est utile lorsqu'il s'agit de calculs en virgule flottante car il peut réduire les "erreurs d'arrondi" qui font que certaines opérations renvoient des valeurs qui semblent incorrectes ou contraires à ce que nous attendons. Par exemple, nous pouvons avoir ajouté une valeur à l'index d'image d'un sprite et nous attendre à ce que le résultat soit un seul entier, mais en raison de la nature des mathématiques en virgule flottante, la valeur finale réelle finit par être quelque chose comme 5.0000002, donc quand nous faire le contrôle suivant:if image_index == 5     {     //do something     } Le code ne se comporte pas comme prévu et retourne false. Cependant, si nous définissons la valeur epsilon sur 0,000001, le image_index La valeur sera arrondie au nombre réel le plus proche qui est +/- 0,000001 de la valeur d'origine, ce qui rendra la comparaison ci-dessus vraie. La valeur epsilon sera utilisée pour tous les opérateurs entiers suivants:< : Moins que> : Plus grand que== : Égal à<= : Inférieur ou égal à>= : Plus grand ou égal à!= : Pas égal à Notez que la définition d'une valeur epsilon de 0 désactivera tous les arrondis et l'utilisation d'une valeur de 1 donnera une erreur. Syntaxe:math_set_epsilon(epsilon); Argument Description epsilon La nouvelle valeur epsilon (de 0 à 0,999999999). Retour:Real Exemple:math_set_epsilon(0.0001); Cela définira la valeur epsilon pour toutes les autres opérations en virgule flottante.
