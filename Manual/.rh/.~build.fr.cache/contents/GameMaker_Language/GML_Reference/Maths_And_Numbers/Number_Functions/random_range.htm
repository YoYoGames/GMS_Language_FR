random_range Cette fonction renvoie un nombre aléatoire entre la plage spécifiée et cette valeur de retour n'a pas besoin d'être un entier. Par exemple, random_range(20,50) renverra un nombre aléatoire de 20 à 50, mais la valeur peut être un nombre réel comme 38,65265. Les nombres réels peuvent également être utilisés comme arguments d'entrée. REMARQUE: Cette fonction renverra la même valeur à chaque fois que le jeu est réexécuté car GameMaker Studio 2 génère la même graine aléatoire initiale à chaque fois pour rendre le code de débogage beaucoup plus facile. Pour éviter ce comportement, utilisez randomise () au début de votre partie. Syntaxe:random_range(n1, n2); Argument Description n1 L'extrémité inférieure de la plage à partir de laquelle le nombre aléatoire sera sélectionné. n2 L'extrémité supérieure de la plage à partir de laquelle le nombre aléatoire sera sélectionné. Retour:Real Exemple:score += random_range(500, 600); Cela ajoutera entre 500 et 600, au score total. Retour: Fonctions numériques Suivant:irandom © Copyright YoYo Games Ltd. 2021 Tous droits réservés
