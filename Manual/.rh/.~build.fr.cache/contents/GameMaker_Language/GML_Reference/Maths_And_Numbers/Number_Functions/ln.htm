ln Le logarithme naturel ln(n) représente la quantité de temps nécessaire pour atteindre un certain niveau de croissance continue, où n est le niveau atteint. Ainsi, si nous voulons savoir combien d'unités de temps il nous faut pour obtenir une croissance de 20, nous utiliserons ln(20), qui donne qui donne 2,99 unités de temps pour obtenir cette quantité de croissance. Syntaxe : ln(n) ; Arguments Description n La valeur d'entrée. Les retours : Real Exemple : alarme[0] = ln(age) * room_speed ; Le code ci-dessus utilise le logarithme naturel de la valeur stockée dans la variable "age" pour déclencher une alarme.
