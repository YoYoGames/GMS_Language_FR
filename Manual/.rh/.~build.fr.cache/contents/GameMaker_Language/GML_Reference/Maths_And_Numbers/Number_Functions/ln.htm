ln Le logarithme naturel ln(n) est le temps nécessaire pour atteindre un certain niveau de croissance continue, où n est le niveau atteint. Donc, si nous voulons savoir combien d'unités de temps nous avons besoin pour obtenir 20 de croissance, nous utiliserions ln(20) qui renvoie 2,99 unités de temps pour obtenir cette quantité de croissance. Syntaxe:ln(n); Argument Description n La valeur d'entrée. Retour:Real Exemple:alarm[0] = ln(age) * room_speed; Le code ci-dessus utilise le logarithme naturel de la valeur stockée dans la variable «age» pour définir une alarme.
