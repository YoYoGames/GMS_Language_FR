au hasard Cette fonction est utile pour les probabilités où il n'est pas nécessaire de renvoyer un entier (nombre entier). Par exemple, random(100) renverra une valeur de 0 à 99, mais cette valeur peut être 22.56473 ! Vous pouvez également utiliser des nombres réels et non des entiers dans cette fonction comme ceci - random(0.5), qui renverra une valeur comprise entre 0 et 0.4999999. REMARQUE: cette fonction renverra la même valeur à chaque nouvelle exécution du jeu, car GameMaker Studio 2 génère à chaque fois la même graine aléatoire initiale afin de faciliter le débogage du code. Pour éviter ce comportement, utilisez randomise() au début de votre jeu. Syntaxe : aléatoire(n) ; Arguments Description n La fourchette supérieure dans laquelle le numéro aléatoire sera sélectionné. Les retours : Real Exemple : si (random(10) >= 9) { score += 100 ; } Cela donnera environ une chance sur dix d'ajouter 100 au score.
