signe Cette fonction indique si un nombre est positif, négatif ou ni l'un ni l'autre et renvoie respectivement 1, -1, 0. Par exemple, sign(458) renvoie 1, sign(-5) renvoie -1 et sign(0) renvoie 0. Syntaxe : signe(n) ; Arguments Description n Le nombre pour obtenir le signe de. Les retours : Entier Exemple : y += sign(y - mouse_y) ; Le code ci-dessus ajoutera 1, -1 ou 0 à y en fonction du résultat de y - mouse_y.
