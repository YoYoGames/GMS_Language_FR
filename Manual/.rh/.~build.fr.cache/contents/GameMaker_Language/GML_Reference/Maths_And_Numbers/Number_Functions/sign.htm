sign Cette fonction renvoie si un nombre est positif, négatif ou ni l'un ni l'autre et renvoie 1, -1, 0 respectivement. Par exemple -sign(458) renverra 1, sign(-5) retournera -1 et sign(0) renverra 0. Syntaxe:sign(n); Argument Description n Le nombre dont le signe doit être. Retour:Integer Exemple:y += sign(y - mouse_y); Le code ci-dessus ajoutera 1, -1 ou 0 sur y en fonction du résultat de y - mouse_y.
