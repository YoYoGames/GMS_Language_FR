log2 Cette fonction demande essentiellement "combien de 2 devez-vous multiplier pour obtenir n ?". Un exemple courant d'utilisation en programmation serait de calculer le nombre de bits nécessaires pour représenter un nombre. Pour ce faire, on calcule le log2(x) du nombre, on l'arrondit à l'inférieur et on ajoute 1. Par exemple, log2(100) renvoie 6,643856, ce qui, arrondi à l'inférieur, donne 6, auquel on ajoute 1 et on obtient 7. Nous avons donc besoin de 7 bits pour représenter 100. Il peut également être utilisé pour décrire une croissance ou une décroissance exponentielle. exponentielle, et est souvent appelé logarithme binaire. Syntaxe : log2(n) ; Arguments Description n La valeur d'entrée. Les retours : Real Exemple : colorbits = floor(log2(colour)) - 1 ; Le code ci-dessus stocke le nombre de bits nécessaires pour représenter la valeur stockée dans la variable "couleur".
