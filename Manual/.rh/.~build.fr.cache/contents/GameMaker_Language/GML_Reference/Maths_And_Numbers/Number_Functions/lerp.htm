lerp Avec cette fonction, vous pouvez trouver la valeur qui équivaut à la position entre deux autres valeurs pour un pourcentage donné. Ainsi, si vous faites, par exemple lerp(0, 10, 0.5) vous obtiendrez la valeur de retour 5, qui correspond à 50 % des valeurs d'entrée. Vous pouvez également extrapoler à l'aide de cette fonction, en fournissant une valeur positive ou négative pour le montant de l'interpolation, de sorte que vous puissiez faire quelque chose comme : lerp(0, 10, 2) retournera une valeur de 20. Syntaxe : lerp(a, b, amt) Arguments Description a La première valeur. b La deuxième valeur. amt Le montant à interpoler. Les retours : Real Exemple : xx = lerp(x, x + hspeed, room_speed) ; yy = lerp(y, y + vspeed, room_speed) ; Le code ci-dessus utilise la fonction d'interpolation linéaire pour prédire où une instance se serait déplacée après une seconde de jeu.
