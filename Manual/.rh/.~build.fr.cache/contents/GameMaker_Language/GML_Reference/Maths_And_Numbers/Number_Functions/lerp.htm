lerp Avec cette fonction, vous pouvez trouver la valeur qui correspond à la position entre deux autres valeurs pour un pourcentage donné. Donc si vous le faites, par exemple:lerp(0, 10, 0.5) vous obtiendrez la valeur de retour de 5, soit 50% des valeurs d'entrée. Vous pouvez également extrapoler à l'aide de cette fonction, en fournissant une valeur positive ou négative pour la quantité d'interpolation afin de faire quelque chose comme:lerp(0, 10, 2) renverra une valeur de 20. Syntaxe:lerp(a, b, amt) Argument Description a La première valeur. b La deuxième valeur. amt Le montant à interpoler. Retour:Real Exemple:xx = lerp(x, x + hspeed, room_speed); yy = lerp(y, y + vspeed, room_speed); Le code ci-dessus utilise la fonction d'interpolation linéaire pour prédire où une instance se serait déplacée après une seconde de temps de jeu.
