logn Cette fonction est similaire à la log2(n) et log10(n) fonctions, vous seul spécifiez la valeur de base du journal. Par exemple, logn(5,25) retournera combien de 5 nous devons multiplier pour obtenir 25 (ce qui fait 2). Syntaxe:logn(n, val); Argument Description n La base du journal. val La valeur d'entrée. Retour:Real Exemple:logval = logn(5, num); Le code ci-dessus obtient le log5 de la valeur stockée dans "num".
