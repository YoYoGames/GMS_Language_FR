logn Cette fonction est similaire aux fonctions log2(n) et log10(n) sauf que vous stipulez la valeur de base du logarithme. Par exemple, logn(5,25) renvoie le nombre de 5 qu'il faut multiplier pour obtenir 25 (soit 2). pour obtenir 25 (soit 2). Syntaxe : logn(n, val) ; Arguments Description n La base de bois. val La valeur d'entrée.
