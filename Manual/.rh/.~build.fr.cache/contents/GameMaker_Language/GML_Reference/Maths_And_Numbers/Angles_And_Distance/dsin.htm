dsin Dans un triangle rectangle dsin (val) = Opposé / Hypoténuse où "val" est l'un des trois angles. REMARQUE: cette fonction prend une valeur en degrés et non en radians Syntaxe:dsin(val); Argument Description val L'angle (en degrés) pour renvoyer le sinus. Retour:Real Exemple:val = dsin(90); Cela mettra "val" à 1. Retour: Angles et distance Suivant:dtan © Copyright YoYo Games Ltd. 2021 Tous droits réservés
