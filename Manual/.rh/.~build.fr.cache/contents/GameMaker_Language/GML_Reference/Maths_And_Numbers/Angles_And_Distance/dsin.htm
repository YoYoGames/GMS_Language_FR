dsin Dans un triangle rectangle, dsin(val) = Opposé / Hypoténuse où "val" est l'un des trois angles. NOTE: Cette fonction prend une valeur en degrés et non en radians. Syntaxe : dsin(val) ; Arguments Description val L'angle (en degrés) dont il faut retourner le sinus. Les retours : Real Exemple : val = dsin(90) ; Ceci mettra "val" à 1.
