degtorad Dans GM, toutes les fonctions trigonométriques fonctionnent en radians, mais la plupart des gens travaillent en degrés et cela signifie que pour convertir vos degrés en radians, vous devez utiliser cette fonction. Par exemple, degtorad(180)  renvoie 3,14159265 radians. Cette fonction convertit les degrés en radians en utilisant la formule:angle_radians = angle_degrees * pi / 180 Syntaxe:degtorad(deg); Argument Description deg Les degrés à convertir. Retour:Real Exemple:val = degtorad(90); Cela définira val à pi / 2. Retour: Angles et distance Suivant:radtodeg © Copyright YoYo Games Ltd. 2021 Tous droits réservés
