point_distance Cette fonction renvoie la longueur d'un vecteur formé par les composantes spécifiées [x1,y1] et [x2,y2]. Par exemple, dans l'image ci-dessous, si nous voulons obtenir la distance entre la position du vaisseau du joueur et la position de l'ennemi afin de pouvoir calculer si l'ennemi est suffisamment proche pour qu'on lui tire dessus, nous utiliserons cette fonction (le code exact est dans l'exemple). ennemi est assez proche pour qu'on lui tire dessus, nous utiliserons cette fonction (le code exact est dans l'exemple sous l'image) : Syntaxe : point_distance(x1, y1, x2, y2) ; Arguments Description x1 La coordonnée x de la première composante du vecteur y1 La coordonnée y de la première composante du vecteur x2 La coordonnée x de la deuxième composante du vecteur y2 La coordonnée y de la deuxième composante du vecteur Les retours : Real Exemple : var ex, ey ; ex = instance_nearest(x, y, enemy).x ; ey = instance_nearest(x, y, enemy).y ; if (point_distance(x, y, ex, ey) < 200) { instance_create_layer(x, y, "Bullets", obj_Missile) } Le code ci-dessus obtiendra les coordonnées x et y de l'ennemi le plus proche, puis les utilisera pour vérifier la distance (longueur) du vecteur formé par ces coordonnées et celles du joueur. Si la valeur est inférieure à 200, l'objet joueur créera une instance de "obj_Missile". Retour : Angles et distances Suivant : point_distance_3d Copyright YoYo Games Ltd. 2021 Tous droits réservés
