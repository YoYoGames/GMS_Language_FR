cos Dans un triangle à angle droit, cos(val) = Adjacent / Hypoténuse où val est l'un des trois angles. Cette fonction renvoie toujours un nombre compris entre 1 et -1. NOTE: Cette fonction prend une valeur en radians et non en degrés. Syntaxe : cos(val) ; Arguments Description val L'angle (en radians) dont il faut retourner le cosinus. Les retours : Real Exemple : val = cos( 0 ) ; Ceci mettra val à 1. Retour : Angles et distances Suivant : péché Copyright YoYo Games Ltd. 2021 Tous droits réservés
