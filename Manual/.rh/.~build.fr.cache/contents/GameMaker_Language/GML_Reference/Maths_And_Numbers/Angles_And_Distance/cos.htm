cos Dans un triangle rectangle cos (val) = Adjacent / Hypoténuse où val est l'un des trois angles. Cette fonction renverra toujours un nombre entre 1 et -1. REMARQUE: Cette fonction prend une valeur en radians et non en degrés. Syntaxe:cos(val); Argument Description val L'angle (en radians) de retour du cosinus. Retour:Real Exemple:val = cos( 0 ); Cela mettra val à 1. Retour: Angles et distance Suivant:sin © Copyright YoYo Games Ltd. 2021 Tous droits réservés
