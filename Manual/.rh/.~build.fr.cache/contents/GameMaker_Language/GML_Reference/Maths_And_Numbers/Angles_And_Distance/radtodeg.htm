radtodeg Une fois que vous avez effectué vos calculs en utilisant sin, ou cos etc... le résultat est en radians. Ce n'est pas toujours ce que vous voulez et donc pour transformer les radians en degrés, nous utilisons cette fonction. Par exemple, radtodeg(sin(180)) renverra -45 degrés. Cette fonction convertit les radians en degrés à l'aide de la formule : angle_degrés = angle_radians * 180 / pi ; Syntaxe : radtodeg(rad) ; Arguments Description rad Les radians à convertir. Les retours : Real Exemple : val = radtodeg( pi ) ; La valeur de Val sera ainsi fixée à 180. Retour : Angles et distances Suivant : point_direction Copyright YoYo Games Ltd. 2021 Tous droits réservés
