différence d'angle Cette fonction renvoie la plus petite différence d'angle entre deux angles sous la forme d'une valeur comprise entre -180 et 180 degrés (où un angle positif correspond au sens inverse des aiguilles d'une montre et un angle négatif au sens des aiguilles d'une montre). Syntaxe : différence_angle(ang1, ang2) Arguments Description ang1 Le premier angle à utiliser. ang2 Le deuxième angle à utiliser. Les retours : Real Exemple : var pd = point_direction(x, y, mouse_x, mouse_y) ; var dd = angle_difference(image_angle, pd) ; image_angle -= min(abs(dd), 10) * signe(dd) ; Le code ci-dessus obtiendra l'angle de direction entre l'instance et le curseur de la souris, puis la différence entre cet angle et l'angle actuel. image_angle, en utilisant cette valeur pour effectuer une rotation lente vers la souris.
