dcos Dans un triangle à angle droit, cos(val) = Adjacent / Hypoténuse où val est l'un des trois angles. Cette fonction renvoie toujours un nombre compris entre 1 et -1. NOTE: Cette fonction prend une valeur en degrés, pas en radians. Syntaxe : dcos(val) ; Arguments Description val L'angle (en degrés) dont il faut retourner le cosinus. Les retours : Real Exemple : val = dcos(45) ; La valeur de "val" sera ainsi fixée à 0,71. Retour : Angles et distances Suivant : dsin Copyright YoYo Games Ltd. 2021 Tous droits réservés
