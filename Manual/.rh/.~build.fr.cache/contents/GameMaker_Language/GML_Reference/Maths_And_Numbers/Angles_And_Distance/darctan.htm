darctan Renvoie la tangente inverse de x. Cette fonction acceptera n'importe quel nombre car, contrairement à dtan() darctan(), les asymptotes sont sur l'axe des y, ce qui signifie simplement que vous ne recevrez jamais un nombre supérieur à 90 ou inférieur à -90. NOTE: Cette fonction renvoie une valeur en degrés, et non en radians. Syntaxe : darctan(x) ; Arguments Description val La valeur dont on veut retourner la tangente inverse. Les retours : Real Exemple : val = darctan(1) ; La valeur de "val" sera ainsi fixée à -45. Retour : Angles et distances Suivant : darctan2 Copyright YoYo Games Ltd. 2021 Tous droits réservés
