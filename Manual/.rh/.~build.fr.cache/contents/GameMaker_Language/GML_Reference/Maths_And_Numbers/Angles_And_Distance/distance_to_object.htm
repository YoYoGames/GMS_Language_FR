distance_to_object Cette fonction calcule la distance entre le bord de la boîte englobante de l'instance appelante et le bord le plus proche de l'instance la plus proche de l'objet spécifié. L'objet peut être un index d'objet ou un ID d'instance spécifique ainsi que le mot -clé other, et la distance est renvoyée en pixels. Notez que si l'un des objets n'a aucun sprite ou aucun masque défini, les résultats seront incorrects. Syntaxe:distance_to_object(obj); Argument Description obj L'objet à rechercher. Retour:Real. Exemple:if (distance_to_object(obj_Player) < range)     {     canshoot = true;     } Le code ci-dessus vérifiera la distance par rapport à l'objet joueur et si elle est inférieure à la valeur stockée dans la variable «range», la variable «canshoot» est définie sur true.
