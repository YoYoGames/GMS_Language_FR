arcsin Renvoie le sinus inverse de x, c'est-à-dire que si sin(x)=n, arcsin(n)=x, le nombre résultant sera compris entre -pi/2 et pi/2. REMARQUE: Cette option n'accepte qu'une valeur comprise entre -1 et 1 (toute autre valeur entraînera une erreur). NOTE: Cette fonction renvoie une valeur en radians, et non en degrés. Syntaxe : arcsin(x) ; Arguments Description x La valeur dont on veut retourner le sinus inverse. Les retours : Real Exemple : val = arcsin(0) ; Cela mettra val à 0.
