arctan Renvoie la tangente inverse de x. Cette fonction accepte n'importe quel nombre car, contrairement à tan(), les asymptotes de arctan() sont sur l'axe des y, ce qui signifie simplement que vous ne recevrez jamais un nombre supérieur à pi/2 ou inférieur à -pi/2. NOTE: Cette fonction prend une valeur en radians, et non en degrés. Syntaxe : arctan(x) ; Arguments Description x L'angle (en radians) dont il faut retourner la tangente inverse. Les retours : Real Exemple : val = arctan( 0 ) ; Cela mettra val à 0. Retour : Angles et distances Suivant : arctan2 Copyright YoYo Games Ltd. 2021 Tous droits réservés
