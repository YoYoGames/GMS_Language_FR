sin Dans un triangle rectangle sin (val) = Opposé / Hypoténuse où val est l'un des trois angles. REMARQUE: Cette fonction prend une valeur en radians et non en degrés. Syntaxe:sin(val); Argument Description val L'angle (en radians) pour renvoyer le sinus. Retour:Real Exemple:val = sin( pi/2 ); Cela mettra val à 1.
