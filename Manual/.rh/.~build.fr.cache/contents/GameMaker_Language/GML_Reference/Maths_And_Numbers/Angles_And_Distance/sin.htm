sin Dans un triangle rectangle, sin(val) = Opposé / Hypoténuse où val est l'un des trois angles. NOTE: Cette fonction prend une valeur en radians et non en degrés. Syntaxe : sin(val) ; Arguments Description val L'angle (en radians) dont il faut retourner le sinus. Les retours : Real Exemple : val = sin( pi/2 ) ; Ceci mettra val à 1.
