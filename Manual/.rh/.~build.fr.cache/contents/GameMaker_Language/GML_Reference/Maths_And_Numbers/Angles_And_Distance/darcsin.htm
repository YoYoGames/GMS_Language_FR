darcsin Renvoie le sinus inverse de x, c'est-à-dire que si dsin(x)=n, darcsin(n)=x, et le nombre résultant sera compris entre -90 et 90. REMARQUE: Cette fonction n'accepte qu'un nombre compris entre -1 et 1 (toute autre valeur entraînera une erreur). NOTE: Cette fonction renvoie une valeur en degrés, et non en radians. Syntaxe : darcsin(x) ; Arguments Description val La valeur dont on veut retourner le sinus inverse. Les retours : Real Exemple : val = darcsin(-1) ; La valeur de "val" sera ainsi fixée à -90.
