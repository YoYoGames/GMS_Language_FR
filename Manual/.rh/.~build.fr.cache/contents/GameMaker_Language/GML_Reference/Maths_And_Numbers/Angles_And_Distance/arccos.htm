arccos Renvoie le cosinus inverse de x, c'est-à-dire que si cos(val)=n, arccos(n)=val, et le nombre résultant sera compris entre pi et 0. REMARQUE: Cette fonction n'accepte qu'un nombre compris entre -1 et 1 (toute autre valeur entraînera une erreur). NOTE: Cette fonction prend une valeur en radians, et non en degrés. Syntaxe : arccos(x) Arguments Description x L'angle (en radians) dont il faut retourner l'inverse du cosinus. Les retours : Real Exemple : val = arccos(0) ; Cela donnera à val la valeur pi/2.
