xboxlive_get_savedata_user Cette fonction renvoie le pointeur d'identification de l'utilisateur actuellement associé à l'enregistrement du fichier (ou la constante pointer_null si aucun ID utilisateur n'est actuellement utilisé). Voir xboxlive_set_savedata_user() pour plus de détails. Syntaxe : xboxlive_get_savedata_user() ; Les retours : Pointeur ou pointeur_null Exemple : if (xboxlive_get_savedata_user() != user_id[0]) { xboxlive_set_savedata_user(user_id[0]) ; } Le code ci-dessus vérifie si un utilisateur est actuellement affecté comme cible de sauvegarde, et s'il ne l'est pas, il est affecté.
