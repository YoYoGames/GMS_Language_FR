xboxlive_pad_for_user Les utilisateurs de la Xbox peuvent avoir plusieurs pads assignés à leur ID utilisateur, chacun remplissant un "slot". Ces emplacements sont numérotés à partir de 0 et peuvent contenir n'importe quel indice de pad. Ainsi, l'emplacement 0 peut être attribué au pad 3 et l'emplacement 1 au pad 2. Cette fonction peut être utilisée pour obtenir le pad qui est assigné à un slot donné, et est particulièrement utile si un utilisateur laisse une manette de jeu sans entrée pendant un certain temps, car elle s'éteindra d'elle-même, vous pouvez donc vérifier n'importe quel ID utilisateur donné en utilisant cette fonction pour vous assurer qu'il dispose d'une manette de jeu active et, si ce n'est pas le cas, réagir en conséquence (par exemple, en mettant votre jeu en pause). Si le pad est actif, la valeur de son index sera renvoyée, mais si ce n'est pas le cas, la fonction renverra -1. Notez que si un pad dans un slot devient inactif, les slots suivants vont "glisser" dans la position vacante, donc si vous avez trois pads actifs assignés aux slots 0, 1 et 2, et que le pad dans le slot 0 est inactif, les slots 1 et 2 deviendront maintenant les slots 0 et 1. 1, donc l'emplacement 0 devrait toujours contenir un pad actif, sauf si tous les pads sont désactivés (auquel cas il renvoie -1). Syntaxe : xboxlive_pad_for_user(user_id, slot) ; Arguments Description nom d'utilisateur L'ID utilisateur (un pointeur) à vérifier nom d'utilisateur L'index (un nombre entier) de l'emplacement de la manette de jeu à vérifier. Les retours : Pointeur Exemple : if (xboxlive_pad_for_user(user_id, 0) == -1) { for(var i = 0 ; i < gamepad_get_device_count() ; i++ ;) { var u_id = xboxlive_user_for_pad(i) ; if (u_id == -1) { async_ID = xboxlive_pad_for_user(i, u_id) ; } } } Le code ci-dessus vérifie l'emplacement 0 pour le pointeur d'ID utilisateur donné pour voir si des manettes de jeu ont été attribuées, et si ce n'est pas le cas, une manette est attribuée.
