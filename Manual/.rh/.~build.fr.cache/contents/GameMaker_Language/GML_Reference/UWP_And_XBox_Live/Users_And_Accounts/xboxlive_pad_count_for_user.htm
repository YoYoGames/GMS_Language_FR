xboxlive_pad_count_for_user Avec cette fonction, vous pouvez savoir combien de "slots" de pad sont actifs pour l'utilisateur actuel (voir xboxlive_pad_for_user() pour plus de détails). Syntaxe:xboxlive_pad_count_for_user(user_id, slot); Argument Description user_id L'ID utilisateur (un pointeur) à vérifier Retour:Integer Exemple:global.slots[0] = xboxlive_pad_count_for_user(user_id[0]) Le code ci-dessus stocke le nombre de plots associés au pointeur d'ID utilisateur donné.
