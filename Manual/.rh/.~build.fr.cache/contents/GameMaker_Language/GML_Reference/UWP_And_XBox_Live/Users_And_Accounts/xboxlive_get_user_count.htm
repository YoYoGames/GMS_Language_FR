xboxlive_get_user_count Avec cette fonction, vous pouvez trouver le nombre total d'utilisateurs actuellement connectés au système XBox. La valeur de retour sera une valeur entière, de 0 à N, où N est le nombre d'utilisateurs connectés -1. Syntaxe:xboxlive_get_user_count(); Retour:Real Exemple:for(var i = 0; i < xboxlive_get_user_count(); ++i;)     {     user_id[i] = xboxlive_get_user(i);     } Ce qui précède parcourt tous les utilisateurs connectés et stocke leur pointeur d'ID unique dans un tableau.
