xboxlive_generate_player_session_id Cette fonction générera une chaîne unique (contenant un GUID) pour la session actuelle du lecteur. Cela peut ensuite être utilisé avec la fonction xboxlive_fire_event(). Syntaxe:xboxlive_generate_player_session_id(); Retour:String Exemple:if !xboxlive_user_is_signed_in()     {     if !xboxlive_user_is_signing_in()         {         xboxlive_show_account_picker();         }     } else     {     global.GamerTag = xboxlive_gamertag_for_user();     } Le code ci-dessus vérifie si un utilisateur est connecté à XBox Live et s'il est connecté, le code obtiendra le gamertag de l'utilisateur et le stockera dans une variable globale pour une utilisation future. S'ils ne sont pas connectés, le code vérifie s'ils sont en train de se connecter, auquel cas rien ne se passera, et s'ils ne le sont pas, il ouvrira le sélecteur de compte, les invitant à signer -dans.
