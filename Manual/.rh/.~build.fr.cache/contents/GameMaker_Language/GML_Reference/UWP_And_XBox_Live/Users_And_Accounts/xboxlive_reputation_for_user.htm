xboxlive_reputation_pour_l'utilisateur Avec cette fonction, vous pouvez récupérer le score de réputation XBox Live pour le pointeur d'identification de l'utilisateur donné. Syntaxe : xboxlive_reputation_for_user(requesting_user_id) ; Arguments Description demandeur_identifiant L'ID utilisateur (un pointeur) de l'utilisateur à vérifier Les retours : Real Exemple : var _a = 0 ; var _num = xboxlive_get_user_count() ; for (var i = 0 ; i < _num ; ++i ;) { var _uid = xboxlive_get_user(i) ; if _uid != pointer_null { global.UserName[_a] = xboxlive_gamedisplayname_for_user(_uid) ; global.UserScore[_a] = xboxlive_gamerscore_for_user(_uid) ; global.UserRep[_a] = xboxlive_reputation_for_user(_uid) ; global.UserAvatar[_a] = xboxlive_sprite_add_from_gamerpicture(_uid, 256, 0, 0) ; ++a ; } } Le code ci-dessus passe en boucle les utilisateurs connectés et stocke leurs données de jeu dans divers tableaux globaux.
