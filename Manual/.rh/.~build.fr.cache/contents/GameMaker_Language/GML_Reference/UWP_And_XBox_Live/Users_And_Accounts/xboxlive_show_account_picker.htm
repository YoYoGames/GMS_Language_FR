xboxlive_show_account_picker La fonction ouvrira l'écran XBox Account Picker pour que l'utilisateur puisse se connecter s'il n'est pas déjà connecté (s'il l'est, la fonction ne fera rien). Notez que cette fonction n'est applicable que si vous utilisez la version UWP sur PC et non sur Xbox, car pour lancer un jeu sur Xbox, l'utilisateur doit être déjà connecté. Syntaxe : xboxlive_show_account_picker() ; Les retours : N/A Exemple : if !xboxlive_user_is_signed_in() { if !xboxlive_user_is_signing_in() { xboxlive_show_account_picker() ; } } else { global.GamerTag = xboxlive_gamertag_for_user() ; } Le code ci-dessus vérifie si un utilisateur est connecté à XBox Live et s'il l'est, le code récupère le gamertag de l'utilisateur et le stocke dans une variable globale pour une utilisation ultérieure. Si l'utilisateur n'est pas connecté, le code vérifie s'il est en train de se connecter. Si l'utilisateur n'est pas connecté, le code vérifie s'il est en train de se connecter, auquel cas rien ne se passe, et s'il ne l'est pas, il ouvre le sélecteur de compte, l'invitant à se connecter.
