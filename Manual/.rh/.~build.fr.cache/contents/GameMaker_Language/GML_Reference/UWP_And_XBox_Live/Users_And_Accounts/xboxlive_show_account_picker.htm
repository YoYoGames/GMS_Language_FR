xboxlive_show_account_picker La fonction ouvrira l'écran XBox Account Picker afin que l'utilisateur puisse se connecter s'il n'est pas déjà connecté (si tel est le cas, la fonction ne fera rien). Notez que cette fonction n'est applicable que lors de l'utilisation de la version UWP sur PC et non sur Xbox, car pour lancer un jeu sur Xbox, l'utilisateur doit déjà être connecté. Syntaxe:xboxlive_show_account_picker(); Retour:N/A Exemple:if !xboxlive_user_is_signed_in()     {     if !xboxlive_user_is_signing_in()         {         xboxlive_show_account_picker();         }     } else     {     global.GamerTag = xboxlive_gamertag_for_user();     } Le code ci-dessus vérifie si un utilisateur est connecté à XBox Live et s'il est connecté, le code obtiendra le gamertag de l'utilisateur et le stockera dans une variable globale pour une utilisation future. S'ils ne sont pas connectés, le code vérifie s'ils sont en train de se connecter, auquel cas rien ne se passera, et s'ils ne le sont pas, il ouvrira le sélecteur de compte, les invitant à signer -dans.
