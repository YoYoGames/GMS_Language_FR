xboxlive_set_rich_presence Cette fonction définira la chaîne de présence riche pour l'utilisateur donné. Une chaîne Présence riche montre l'activité d'un utilisateur dans le jeu après le nom du jeu auquel l'utilisateur joue, séparé par un trait d'union. Cette chaîne est affichée sous le gamertag d'un joueur dans la liste "Amis et clubs" ainsi que dans le profil utilisateur Xbox Live du joueur. Lorsque vous utilisez cette fonction, vous devez fournir le pointeur d'ID utilisateur pour l'utilisateur, puis vous pouvez marquer l'utilisateur comme étant actuellement actif dans le jeu ou non (en utilisant true /false ). Si défini sur false, alors la chaîne de présence riche sera ajoutée avec "/ afk" ou quelque chose d'approprié. L'argument suivant est l'ID de chaîne de présence riche à afficher, puis enfin vous pouvez (éventuellement) fournir un service_config_id chaîne de caractères. Notez qu'il s'agit d'un argument optionnel car si vous avez appelé xboxlive_stats_setup() vous n'avez pas besoin de passer le service_config_id ici. Pour plus d'informations sur la présence enrichie et comment configurer les chaînes à utiliser dans le centre des partenaires, consultez la documentation Microsoft suivante: Présentation des chaînes de présence Syntaxe:xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]) Argument Description user_id Le pointeur d'ID de l'utilisateur à vérifier. is_user_active Marquer l'utilisateur comme actif ou non. rich_presence_string ID de chaîne de présence riche à utiliser (tel que défini dans le centre des partenaires - 50 caractères maximum) service_config_id [FACULTATIF] Ceci est la chaîne config_id pour le jeu. Retour:N/A Exemple:var pad_uid = xboxlive_user_for_pad(0); xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge"); Le code ci-dessus obtient le pointeur d'ID utilisateur pour l'utilisateur affecté à l'emplacement de la manette de jeu [0], puis définit la chaîne de présence riche pour cet utilisateur.
