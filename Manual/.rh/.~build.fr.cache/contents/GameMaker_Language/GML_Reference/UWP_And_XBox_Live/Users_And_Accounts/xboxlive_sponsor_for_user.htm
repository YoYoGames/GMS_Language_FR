xboxlive_sponsor_for_user Cette fonction récupère l'ID du sponsor utilisateur donné. Syntaxe:xboxlive_sponsor_for_user(user_id); Argument Description user_id Le pointeur d'ID de l'utilisateur à vérifier. Retour:Real Exemple:var _uid = xboxlive_user_for_pad(0); sponsor = xboxlive_sponsor_for_user(_uid); Le code ci-dessus obtient le pointeur d'ID utilisateur pour l'utilisateur affecté à l'emplacement de la manette de jeu [0], puis utilise cet ID pour récupérer l'ID de sponsor de l'utilisateur.
