xboxlive_user_is_guest Cette fonction retournera true si le pointeur d'ID utilisateur donné est un utilisateur invité et false s'ils ne le sont pas. Vous pouvez obtenir un pointeur d'ID utilisateur avec la fonction xboxlive_get_user(). Syntaxe:xboxlive_user_is_guest(user_id); Argument Description user_id Le pointeur d'ID de l'utilisateur à vérifier. Retour:Boolean Exemple:if xboxlive_user_is_guest(user_id[0])     {     global.user_name[0] = "GUEST";     } else     {     global.user_name[0] = xboxlive_gamedisplayname_for_user(user_id[0]);     } Ce qui précède stocke le pointeur d'ID utilisateur d'activation dans une variable globale.
