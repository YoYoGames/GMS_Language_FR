xboxlive_user_is_guest Cette fonction retournera vrai si le pointeur d'ID d'utilisateur donné est un utilisateur invité et faux s'il ne l'est pas. Vous pouvez obtenir un pointeur d'ID utilisateur avec la fonction xboxlive_get_user(). Syntaxe : xboxlive_user_is_guest(user_id) ; Arguments Description nom d'utilisateur Le pointeur d'identification de l'utilisateur à vérifier. Les retours : Booléen Exemple : if xboxlive_user_is_guest(user_id[0]) { global.user_name[0] = "GUEST" ; } else { global.user_name[0] = xboxlive_gamedisplayname_for_user(user_id[0]) ; } L'exemple ci-dessus stocke le pointeur d'identification de l'utilisateur en cours d'activation dans une variable globale.
