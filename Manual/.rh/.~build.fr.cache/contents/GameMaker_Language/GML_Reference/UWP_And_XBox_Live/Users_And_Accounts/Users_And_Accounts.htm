Utilisateurs et comptes Cette section est destinée aux utilisateurs ciblant les plates-formes UWP ou Xbox One et décrit les fonctions XBox Live disponibles pour traiter les utilisateurs et les comptes d'utilisateurs. La Xbox One permet à plusieurs utilisateurs de se connecter simultanément à une console. Chaque utilisateur est associé à un contrôleur spécifique et toutes les données de sauvegarde de jeu sont également spécifiques à l'utilisateur. Pour tirer le meilleur parti de ce système, GameMaker Studio 2 introduit un certain nombre de fonctions XBox One uniquement liées à l'association entre les contrôleurs et les utilisateurs, vous permettant d'interroger des informations sur l'utilisateur et de contrôler également la manière dont les données de sauvegarde sont gérées par utilisateur.. Vous pouvez trouver plus d'informations sur ce système pour les développeurs dans le livre blanc officiel sur le site des développeurs Microsoft. REMARQUE: Notez que l'ID utilisateur est unique pour chaque individu et est perpétué sur différentes consoles, donc si le même utilisateur se connecte sur deux appareils différents, cet ID utilisateur sera le même.
