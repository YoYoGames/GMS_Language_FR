xboxlive_user_is_signed_in Cette fonction vérifie si un utilisateur est connecté et renvoie true si c'est le cas et false sinon. Si aucun utilisateur n'est connecté, vous pouvez appeler la fonction xboxlive_show_account_picker() pour ouvrir le sélecteur de compte et l'inviter à se connecter. Syntaxe : xboxlive_user_is_signed_in() ; Les retours : Booléen Exemple : if !xboxlive_user_is_signed_in() { if !xboxlive_user_is_signing_in() { xboxlive_show_account_picker() ; } } else { global.GamerTag = xboxlive_gamertag_for_user() ; } Le code ci-dessus vérifie si un utilisateur est connecté à XBox Live et s'il l'est, le code récupère le gamertag de l'utilisateur et le stocke dans une variable globale pour une utilisation ultérieure. Si l'utilisateur n'est pas connecté, le code vérifie s'il est en train de se connecter. Si l'utilisateur n'est pas connecté, le code vérifie s'il est en train de se connecter, auquel cas rien ne se passe, et s'il ne l'est pas, il ouvre le sélecteur de compte, l'invitant à se connecter.
