uwp_was_closed_by_user Avec cette fonction, vous pouvez savoir si le jeu a été précédemment arrêté par l'utilisateur pendant qu'il jouait (c'est-à-dire qu'il a été terminé correctement et non par le système). Si c'est le cas, la fonction renvoie true, sinon elle renvoie false. Syntaxe : uwp_was_closed_by_user() ; Les retours : Booléen Exemple : if uwp_was_closed_by_user() { global.GameLoaded = false ; } Le code ci-dessus vérifie si l'application a déjà été interrompue par l'utilisateur et, si c'est le cas, il définit une variable globale.
