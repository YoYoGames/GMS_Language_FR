uwp_was_closed_by_user Avec cette fonction, vous pouvez savoir si le jeu a été précédemment arrêté par l'utilisateur pendant le jeu (c'est-à-dire qu'il a été arrêté correctement et non par le système). Si tel est le cas, la fonction retournera true, sinon il reviendra false. Syntaxe:uwp_was_closed_by_user(); Retour:Boolean Exemple:if uwp_was_closed_by_user()     {     global.GameLoaded = false;     } Le code ci-dessus vérifie si l'application a déjà été arrêtée par l'utilisateur, et si tel est le cas, il définit une variable globale.
