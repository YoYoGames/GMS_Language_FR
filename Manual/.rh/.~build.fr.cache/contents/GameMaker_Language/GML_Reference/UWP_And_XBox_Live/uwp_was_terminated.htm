uwp_was_terminated Cette fonction permet de savoir si le jeu a été précédemment arrêté par le système (c'est-à-dire qu'il ne s'agit pas de la première exécution du jeu). Si c'est le cas, la fonction renvoie true, sinon elle renvoie false. Un exemple d'utilisation serait de vérifier si le jeu a été précédemment arrêté par le système, puis (si elle renvoie vrai) de charger l'étape précédente à laquelle le joueur se trouvait dans le jeu. Si l'on cible la XBox One, le comportement requis des jeux si cela est détecté sera différent entre les différentes versions des documents de certification, il est donc utile de vérifier la dernière version pour voir ce que le jeu doit faire. Syntaxe : uwp_was_terminated() ; Les retours : Booléen Exemple : if xboxlivewas_terminated() { buffer_load("Player_Save.sav") ; global.GameLoaded = true ; } Le code ci-dessus vérifie si l'application a déjà été arrêtée par le système, et si c'est le cas, il charge un tampon et définit une variable globale.
