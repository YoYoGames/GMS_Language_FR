xboxlive_matchmaking_set_joinable_session Cette fonction permet à une session (précédemment créée) d'être accessible aux autres utilisateurs via l'interface utilisateur du système. Un utilisateur n'a qu'une seule session joignable à la fois, et lorsqu'il quitte la session (ou que la session se termine), celle-ci est effacée. Cependant, vous pouvez l'effacer manuellement en passant -1 dans l'argument de la session. Syntaxe : xboxlive_matchmaking_set_joinable_session(local_user, session_that_is_joinable) ; Arguments Description Utilisateur local Le pointeur d'identification de l'utilisateur local. session_qui_est_joinable L'ID de session à rendre joignable, ou -1. Les retours : N/A Exemple : if (global.session_ID != -1) { xboxlive_matchmaking_set_joinable_session(xboxlive_user_for_pad(0), global.session_ID) ; } Le code ci-dessus vérifie l'existence d'une valeur d'ID de session valide, et si une telle valeur est détectée, il définit la session comme étant joignable.
