xboxlive_matchmaking_create Cette fonction créera une session multi-joueurs à l'aide du modèle de session, de la trémie de matchmaking et du modèle Secure Device Association, que vous auriez dû créer au préalable dans le tableau de bord XDP. L'ID utilisateur que vous pouvez récupérer pour une manette de jeu particulière en appelant le xboxlive_user_for_pad() la fonction et la visibilité seront l'une des constantes suivantes: Constant Description xboxlive_match_visibility_open précise que la session peut être rejointe par les autres. xboxlive_match_visibility_private spécifie que la session n'est pas visible pour les utilisateurs qui ne sont pas membres de la session. xboxlive_match_visibility_usetemplate spécifie que la valeur utilisée dans le modèle de session doit être utilisée (cela doit normalement être utilisé car vous ne pouvez pas remplacer la valeur d'un modèle si celui-ci a été défini dans XDP). La fonction renvoie une valeur d'ID de demande unique, qui peut ensuite être utilisée pour identifier l'événement social asynchrone correct pour cette fonction. Cet événement sera déclenché lorsque la session aura été créée et contiendra une DS Map dans la variable async_load avec les paires clé / valeur suivantes: "requestid" - contient l'ID de demande tel que renvoyé par la fonction appelante "status" - contient la chaîne "session_created" pour vous informer du type d'événement qui a été déclenché. "sessionid" - contient l'ID de session, ou -1 en cas d'échec "erreur" - contient 0 en cas de succès, ou -1 en cas d'échec "correlationid" - le descripteur de session pour la corrélation avec la fonction xboxlive_matchmaking_join_session() Syntaxe:xboxlive_matchmaking_create(user_id, visibility, template, hopper, sdatemplate, [matchattributes]); Argument Description user_id Le pointeur d'ID utilisateur à utiliser visibility L'une des constantes répertoriées ci-dessus template Le nom du modèle de session hopper Le nom de la trémie de matchmaking sdatemplate Le nom du modèle d'association d'appareils sécurisés matchattributes [OPTIONNEL] Retour:Request ID (integer) Exemple:var userID = xboxlive_user_for_pad(global.PadIndex); result = xboxlive_matchmaking_create(userID, xboxlive_match_visibility_usetemplate, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic"); Ce qui précède récupérera l'ID utilisateur de l'utilisateur sur la manette de jeu donnée, puis initialisera une session de matchmaking pour lui.
