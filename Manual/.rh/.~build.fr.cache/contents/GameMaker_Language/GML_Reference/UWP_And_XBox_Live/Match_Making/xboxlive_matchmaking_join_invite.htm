xboxlive_matchmaking_join_invite Cette fonction peut être appelée pour traiter une invitation de participation une fois que vous avez appelé avec succès la fonction xboxlive_matchmaking_start(). Vous fournissez l'ID utilisateur local et les valeurs de l'invitation (ID et ID hôte) qui ont été renvoyées lors de la réception de l'invitation, ainsi que le nom du modèle de session que vous avez créé (une chaîne). L'utilisateur sera alors joint à la session à laquelle il a été invité et vous recevrez des événements asynchrones sociaux comme si vous aviez appelé . si vous aviez appelé xboxlive_matchmaking_find(). Syntaxe : xboxlive_matchmaking_join_invite(user_who_received_invite, invitation_id, invitation_host, sda_template_name) ; Arguments Description utilisateur_qui_a_reçu_une_invitation Le pointeur d'identification de l'utilisateur local. invitation_id L'ID de l'invitation. invitation_host L'ID de l'hôte de l'invitation. sda_template_name Le nom du modèle de session. Les retours : N/A Exemple : var userID = xboxlive_user_for_pad(global.PadIndex) ; xboxlive_matchmaking_join_invite(userID, global.InviteID, global.InviteHost, global.SessionTemplate) ; L'exemple ci-dessus récupère l'ID de l'utilisateur pour le gamepad donné et tente ensuite de rejoindre la session à laquelle il a été invité.
