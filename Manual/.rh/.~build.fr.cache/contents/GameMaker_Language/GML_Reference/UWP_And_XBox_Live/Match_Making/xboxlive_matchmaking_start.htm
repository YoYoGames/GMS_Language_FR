xboxlive_matchmaking_start Cette fonction doit être appelée avant de pouvoir lancer une session multi-joueurs pour l'utilisateur. Elle prend le pointeur de l'ID utilisateur, que vous pouvez récupérer pour une manette de jeu particulière en appelant la fonction xboxlive_user_for_pad() et initialisera l'API multi-joueurs. Syntaxe : xboxlive_matchmaking_start(user_id) ; Arguments Description nom d'utilisateur Le pointeur d'identification de l'utilisateur à utiliser Les retours : N/A Exemple : global.UserID = xboxlive_user_for_pad(global.PadIndex) ; xboxlive_matchmaking_start(global.UserID) ; Le code ci-dessus récupère l'ID de l'utilisateur pour l'index du pad donné et initialise ensuite les capacités multijoueurs.
