xboxlive_matchmaking_stop Cette fonction peut être utilisée pour terminer une session de matchmaking pour l'utilisateur donné. Syntaxe:xboxlive_matchmaking_stop(user_id); Argument Description user_id Le pointeur d'ID utilisateur à utiliser Retour:N/A Exemple:if !xboxlive_user_is_signed_in(user_id[0])     {     xboxlive_matchmaking_stop(user_id[0]);     } Le code ci-dessus mettra fin à la session de matchmaking pour l'utilisateur donné.
