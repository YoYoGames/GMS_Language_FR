xboxlive_matchmaking_send_invites Cette fonction ouvre la boîte de dialogue du système Xbox Live pour permettre à un utilisateur d'envoyer des invitations à son jeu. Vous spécifiez l'utilisateur local qui doit contrôler la boîte de dialogue (et dont la liste d'amis sera utilisée par la boîte de dialogue), la valeur d'ID de session unique pour la session à laquelle il souhaite inviter des personnes et du texte par défaut à afficher dans la boîte de dialogue. Vous obtenez l'ID de session du async_load Carte DS générée par la fonction xboxlive_matchmaking_create(). La fonction renvoie -1 si les arguments ne sont pas valides ou si l'utilisateur n'est pas connecté à Xbox Live, ou une valeur supérieure à 0 dans le cas contraire. Syntaxe:xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite); Argument Description from_user L'ID utilisateur local pour contrôler le dialogue. session_to_invite_into ID de session unique pour la session à rejoindre. text_to_add_to_invite Un court texte à ajouter à l'invitation. Retour:Integer Exemple:if gamepad_button_check_pressed(0, gp_face1)     {     var _uid = xboxlive_user_for_pad(0);     xboxlive_matchmaking_send_invites(_uid, global.session_ID, "Come and join my game!");     } Le code ci-dessus vérifie si un bouton de la manette de jeu est pressé et si un bouton est détecté, la boîte de dialogue Xbox Live pour les invitations à une session sera ouverte.
