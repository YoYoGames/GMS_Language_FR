xboxlive_matchmaking_session_leave Cette fonction fait en sorte que l'utilisateur qui a créé ou trouvé la session spécifiée la quitte. Vous pouvez obtenir le pointeur d'ID de session à partir de la carte async_load carte DS qui est renvoyée dans l'événement social asynchrone lorsque vous avez créé ou trouvé une session (cf. xboxlive_matchmaking_create() pour plus de détails). Syntaxe : xboxlive_matchmaking_session_leave(session_id) ; Arguments Description identifiant de la session Le pointeur d'ID de session de la session à quitter. Les retours : N/A Exemple : if gamepad_button_check_pressed(0, gp_start) { xboxlive_matchmaking_session_leave(global.SessionID) ; } Le code ci-dessus vérifie s'il y a eu une pression sur un bouton de la manette de jeu et si c'est le cas, l'utilisateur sera déconnecté de la session de matchmaking donnée.
