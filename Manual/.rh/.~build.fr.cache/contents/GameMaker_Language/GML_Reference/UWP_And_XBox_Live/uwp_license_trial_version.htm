uwp_license_trial_version Cette fonction peut être utilisée pour vérifier si le jeu est sous licence d'essai ou non. Si c'est le cas, la fonction renvoie true, sinon false. Syntaxe : uwp_license_trial_version() ; Les retours : Booléen Exemple : if uwp_license_trial_version() { global.LevelCap = 10 ; } Le code ci-dessus vérifie si l'application est sous licence d'essai et, si c'est le cas, il définit une variable globale.
