xboxlive_stats_remove_user Cette fonction peut être utilisée pour supprimer (désenregistrer) un utilisateur donné du gestionnaire de statistiques, en effectuant un vidage des données statistiques sur le serveur en direct. Selon la documentation XBox, le jeu n'a pas à supprimer l'utilisateur du gestionnaire de statistiques, car le système d'exploitation XBox videra périodiquement les statistiques de toute façon. Vous fournissez l'ID utilisateur tel que renvoyé par (par exemple) la fonction xboxlive_get_user(), et la fonction renverra -1 s'il y a eu une erreur ou si l'ID utilisateur n'est pas valide, ou toute autre valeur si la fonction a été appelée avec succès. REMARQUE: la suppression de l'utilisateur peut renvoyer une erreur si les statistiques qui ont été définies sur l'utilisateur ne sont pas valides (telles que les noms de statistiques contenant des caractères non alphanumériques). La fonction générera un rappel qui déclenchera un événement asynchrone du système. Cet événement aura la carte spéciale DS async_load qui doit ensuite être analysé pour les clés suivantes: " id " - Tiendra la constante achievement_stat_event " eventname " - Contiendra la chaîne " LocalUserRemoved " " userid " - L'ID utilisateur associé à la demande " error " - 0 en cas de succès, une autre valeur en cas d'erreur " message d'erreur " - Une chaîne avec un message d'erreur, le cas échéant Notez que si vous souhaitez vider les données de statistiques sur le serveur en direct à tout moment sans supprimer l'utilisateur, vous pouvez utiliser la fonction xboxlive_stats_flush_user(). Syntaxe:xboxlive_stats_remove_user(user_id); Argument Description user_id L'ID utilisateur (un pointeur) à supprimer Retour:Real Exemple:for(var i = 0; i < xboxlive_get_user_count(); ++i;)     {     user_id[i] = xboxlive_get_user(i);     xboxlive_stats_remove_user(user_id[i]);     } Le code ci-dessus parcourt les ID de compte utilisateur stockés et supprime (désinscription) chaque utilisateur du gestionnaire de statistiques.
