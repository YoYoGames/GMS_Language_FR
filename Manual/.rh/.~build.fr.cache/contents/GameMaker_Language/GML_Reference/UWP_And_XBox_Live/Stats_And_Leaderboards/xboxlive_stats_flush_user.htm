xboxlive_stats_flush_user Cette fonction peut être utilisée pour vider les données de statistiques pour un utilisateur donné du gestionnaire de statistiques vers le serveur en direct, en s'assurant que le serveur est à jour avec les valeurs actuelles. Selon la documentation de XBox, les développeurs doivent faire attention à ne pas appeler cela trop souvent car la XBox limitera le taux des demandes, et le système d'exploitation XBox videra également automatiquement les statistiques environ toutes les 5 minutes automatiquement de toute façon. REMARQUE: la suppression de l'utilisateur peut renvoyer une erreur si les statistiques qui ont été définies sur l'utilisateur ne sont pas valides (telles que les noms de statistiques contenant des caractères non alphanumériques). Pour utiliser la fonction, vous fournissez l'ID utilisateur tel que renvoyé par (par exemple) la fonction xboxlive_get_user(), et une valeur de propriété (qui sera 1 pour une priorité élevée et 0 pour une priorité faible). La fonction renverra -1 s'il y a eu une erreur ou si l'ID utilisateur n'est pas valide, ou toute autre valeur si la fonction a été appelée avec succès. La fonction générera également un rappel qui déclenchera un événement asynchrone du système. Cet événement aura la carte spéciale DS async_load qui doit ensuite être analysé pour les clés suivantes: " id " - Tiendra la constante achievement_stat_event " eventname " - Contiendra la chaîne " StatisticUpdateComplete " " userid " - L'ID utilisateur associé à la demande " error " - 0 en cas de succès, une autre valeur en cas d'erreur " message d'erreur " - Une chaîne avec un message d'erreur, le cas échéant Notez que les statistiques seront automatiquement vidées pour vous lorsque vous supprimez un utilisateur du gestionnaire de statistiques. Voir xboxlive_stats_remove_user() pour plus d'informations. Syntaxe:xboxlive_stats_flush_user(user_id, priority); Argument Description user_id L'ID utilisateur (un pointeur) à utiliser priority La valeur de priorité (0 ou 1) Retour:Real Exemple:for(var i = 0; i < array_length(user_id); ++i;)     {     xboxlive_stats_flush_user(user_id[i], 0);     } Le code ci-dessus parcourt les ID de compte utilisateur stockés, puis vide les données du gestionnaire de statistiques pour chaque utilisateur.
