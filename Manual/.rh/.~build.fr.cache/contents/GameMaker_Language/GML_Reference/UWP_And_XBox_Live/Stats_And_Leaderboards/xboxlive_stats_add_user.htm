xboxlive_stats_add_user Cette fonction peut être utilisée pour ajouter un pointeur d'identification d'utilisateur donné au gestionnaire de statistiques. Cela doit être fait avant d'utiliser l'une des autres fonctions de statistiques pour synchroniser automatiquement le jeu avec le serveur XBox Live et récupérer les dernières valeurs. Vous fournissez le tel que renvoyé par (par exemple) la fonction xboxlive_get_user()et la fonction renverra -1 s'il y a eu une erreur ou si l'ID utilisateur n'est pas valide, ou toute autre valeur si la fonction a été appelée avec succès. La fonction va générer une callback qui va déclencher un événement asynchrone du système. Cet événement aura la carte spéciale DS async_load qui devra ensuite être analysée pour les clés suivantes : " id" - Contiendra la constante achievement_stat_event "Eventname" - Contient la chaîne "LocalUserAdded". " userid" - L'ID utilisateur associé à la demande "error" - 0 en cas de succès, une autre valeur en cas d'erreur. "errormessage" - Une chaîne contenant un message d'erreur, s'il y en a un. Syntaxe : xboxlive_stats_add_user(user_id) ; Arguments Description nom d'utilisateur L'identifiant de l'utilisateur (un pointeur) à ajouter Les retours : Real Exemple : for(var i = 0 ; i < xboxlive_get_user_count() ; ++i ;) { user_id[i] = xboxlive_get_user(i) ; xboxlive_stats_add_user(user_id[i]) ; } Le code ci-dessus obtiendra le nombre de comptes d'utilisateurs disponibles, puis les parcourra en boucle et assignera l'ID du compte à un tableau, ainsi que l'enregistrement de l'utilisateur avec le gestionnaire de statistiques.
