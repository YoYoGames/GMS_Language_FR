xboxlive_stats_delete_stat Cette fonction peut être utilisée pour supprimer une stat du gestionnaire de stat pour l'ID utilisateur donné. Vous fournissez l'ID utilisateur tel que retourné par (par exemple) la fonction xboxlive_get_user()puis la chaîne de caractères à supprimer. Cela efface la valeur de la stat et la supprime du gestionnaire de stat, ce qui signifie qu'elle ne sera plus renvoyée par les fonctions xboxlive_stats_get_stat_names() ou xboxlive_stats_get_stat(). La fonction retournera -1 s'il y a eu une erreur ou si l'ID utilisateur est invalide, ou toute autre valeur si la fonction a été appelée avec succès Syntaxe : xboxlive_stats_delete_stat(user_id, stat) ; Arguments Description nom d'utilisateur Le pointeur de l'ID de l'utilisateur pour supprimer le stat de stat La statistique à supprimer (une chaîne de caractères) Les retours : Real Exemple : for(var i = 0 ; i < xboxlive_get_user_count() ; i++ ;) { user_id[i] = xboxlive_get_user(i) ; xboxlive_stats_delete_stat(user_id[i], "HighScore") ; } Le code ci-dessus boucle à travers les utilisateurs connectés et supprime la stat spécifiée du gestionnaire de stat pour chacun d'entre eux.
