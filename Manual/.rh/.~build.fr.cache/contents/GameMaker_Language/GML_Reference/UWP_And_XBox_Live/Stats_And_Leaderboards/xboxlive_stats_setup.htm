xboxlive_stats_setup Cette fonction doit être appelée avant que vous puissiez utiliser l'une des autres fonctions de statistiques Xbox Live, et initialise simplement les bibliothèques requises sur le système. L'argument "user_id" est l'ID utilisateur brut tel que renvoyé par la fonction xboxlive_get_user(), tandis que les arguments "service_config" et "title_id" correspondent à l'ID unique de votre jeu sur le Dev Center de Xbox Live. Syntaxe : xboxlive_stats_setup(user_id, service_config_id, title_id) ; Arguments Description nom d'utilisateur Le pointeur d'identification de l'utilisateur à vérifier. service_config_id Il s'agit de la chaîne config_id du jeu. Titre_id L'identifiant unique de votre jeu sur le Xbox Dev Center. Les retours : N/A Exemple : var user = xboxlive_get_user(0) ; xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6) ; Le code ci-dessus initialise le système de statistiques pour l'ID utilisateur donné.
