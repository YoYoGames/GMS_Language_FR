xboxlive_read_player_leaderboard Cette fonction peut être utilisée pour déclencher un événement de statistiques. L'argument "event_name" est le nom de l'événement à déclencher tel que défini dans la console XDP de votre jeu, et les paramètres supplémentaires suivants dépendront également de ce que vous avez mis en place pour l'événement. La fonction retournera 0 si l'événement a été envoyé (et devrait être reçu/traité par le serveur) ou -1 s'il y a eu une erreur (ie : votre événement n'a pas été configuré comme le fichier manifeste de l'événement inclus dans le projet dit un autre nombre). Syntaxe : xboxlive_fire_event(nom_de_l'événement, [paramètres_de_stat]) ; Arguments Description nom de l'événement Le nom de l'événement à déclencher. stat_paramètres OPTIONNEL] Paramètres supplémentaires à ajouter à l'événement. Les retours : Chaîne de caractères Exemple : var _user = xboxlive_get_user(0) ; var _uid = xboxlive_user_id_for_user(_user) ; result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42) ; Le code ci-dessus obtient certaines informations sur l'utilisateur, puis déclenche un événement d'état avec ces données (et d'autres).
