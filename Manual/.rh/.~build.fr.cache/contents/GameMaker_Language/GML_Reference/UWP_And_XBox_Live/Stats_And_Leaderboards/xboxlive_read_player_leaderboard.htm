xboxlive_read_player_leaderboard Cette fonction renvoie les informations du classement pour le nom d'utilisateur donné. Le classement doit avoir été créé précédemment sur le tableau de bord XDP de votre jeu, et l'appel de la fonction déclenchera un callback Social Asynchronous Event qui contient la carte async_load map remplie avec les paires clé/valeur pertinentes. La clé "id" de cette carte DS est utilisée pour identifier le callback correct. (il peut y avoir plus d'une fonction de déclenchement pour un événement asynchrone donné), et IT sera associée à la constante achievement_leaderboard_info ainsi qu'à un certain nombre d'autres paires clé/valeur. Le contenu exact de la carte est présenté ci-dessous : "id" - Pour cette fonction, il s'agit de achievement_leaderboard_info. "leaderboardid" - L'ID unique du leaderboard tel que défini dans le tableau de bord du fournisseur. "numentries" - Le nombre d'entrées dans le classement que vous avez reçues. "PlayerN" - Le nom du joueur, où "N" est une valeur entière correspondant à sa position dans la liste des entrées du classement. "PlayeridN" - L'identifiant unique du joueur, "N". "RankN" - Le rang du joueur "N" dans le classement. "ScoreN" - Le score du joueur "N". La fonction vous demande d'indiquer l'une des constantes suivantes pour définir les propriétés du filtre : Constant Description filtre_réussite_tous_les_joueurs Obtenez tous les scores. filtre_réussite_amis_seulement Obtenez uniquement les scores des amis, dans l'ordre croissant. achievement_filter_favorites_only Obtenez uniquement les scores des favoris, dans l'ordre croissant. filtre_réussite_amis_alt Obtenez uniquement les scores des amis par ordre décroissant. achievement_filter_favorites_alt Obtenez uniquement les scores des favoris par ordre décroissant. Syntaxe : xboxlive_read_player_leaderboard(leaderboard_name, user_name, num_items, friend_filter) ; Arguments Description nom_du_classement Le nom du classement à lire, tel que configuré sur XDP. nom d'utilisateur Le nom de l'utilisateur à lire. nombre d'articles Le nombre d'éléments à récupérer. ami_filtre Une des constantes du filtre (voir la description, ci-dessus). Les retours : Real Exemple : var _user = xboxlive_user_for_pad(0) ; var _uid = xboxlive_user_id_for_user(_user) ; xboxlive_read_player_leaderboard("MyLeaderboard", _uid, 10, achievement_filter_all_players) ; Le code ci-dessus obtient l'ID de l'utilisateur et l'utilise pour demander toutes les données des joueurs pour les 10 premières places du classement donné.
