résolution de réseau Cette fonction renvoie l'adresse IP de l'URL donnée. Syntaxe : réseau_résolu(url) ; Arguments Description url L'URL pour obtenir l'IP de (une chaîne). Les retours : Chaîne de caractères Exemple : game_ip = network_resolve("www.macsweeneygames.com") ; Le code ci-dessus renvoie l'adresse IP de l'URL donnée et la stocke dans la variable "game_ip".
