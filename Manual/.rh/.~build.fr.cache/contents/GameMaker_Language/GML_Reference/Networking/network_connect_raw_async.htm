réseau_connexion_raw_async Avec cette fonction, vous pouvez envoyer une requête pour vous connecter à un serveur. La fonction prend l'identifiant du socket à utiliser pour la connexion (voir network_create_socket()) et vous demande de fournir l'adresse IP à laquelle se connecter (une chaîne) ainsi que le port à utiliser. Si la connexion échoue, une valeur inférieure à 0 sera renvoyée. La différence entre cette fonction et network_connect_async()est que cette fonction peut se connecter à n'importe quel serveur et ne fait rien aux données brutes, ce qui signifie que vous devez implémenter les protocoles vous-même du côté du serveur. Notez que cette fonction est asynchrone et génère un événement de mise en réseau asynchrone de type network_type_non_blocking_connect. Syntaxe : network_connect_raw_async(socket, url, port) ; Arguments Description prise de courant L'identifiant du socket à utiliser. url L'URL ou l'IP à laquelle se connecter (une chaîne). port Le port auquel se connecter. Les retours : Real Exemple : client = network_create_socket(network_socket_tcp) ; network_connect_raw_async(client, "www.macsweeneygames.com", 6510) ; Le code ci-dessus va créer un nouveau socket TCP puis tenter de créer une connexion asynchrone "brute" à travers ce socket vers l'URL donnée sur le port 6510.
