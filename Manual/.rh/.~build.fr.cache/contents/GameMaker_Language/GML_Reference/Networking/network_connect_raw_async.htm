network_connect_raw_async Avec cette fonction, vous pouvez envoyer une demande de connexion à un serveur. La fonction prend l' identifiant de socket pour se connecter (voir network_create_socket() ) et vous oblige à donner l'adresse IP à laquelle vous connecter (une chaîne) ainsi que le port de connexion, et si la connexion échoue, une valeur inférieure à 0 sera renvoyée. La différence entre cette fonction et network_connect_async() est que cette fonction peut se connecter à n'importe quel serveur et ne fait rien sur les données brutes, ce qui signifie que vous devez implémenter les protocoles vous-même du côté serveur. Notez que cette fonction est asynchrone, générant un événement de mise en réseau asynchrone du typenetwork_type_non_blocking_connect. Syntaxe:network_connect_raw_async(socket, url, port); Argument Description socket L'identifiant du socket à utiliser. url L'URL ou l'adresse IP à laquelle se connecter (une chaîne). port Le port auquel se connecter. Retour:Real Exemple:client = network_create_socket(network_socket_tcp); network_connect_raw_async(client, "www.macsweeneygames.com", 6510); Le code ci-dessus créera un nouveau socket TCP puis tentera de créer une connexion asynchrone "brute" via celle-ci vers l'URL donnée sur le port 6510.
