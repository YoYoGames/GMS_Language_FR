réseau_connexion_raw Avec cette fonction, vous pouvez envoyer une requête pour vous connecter à un serveur. La fonction prend l'identifiant de la socket à utiliser pour se connecter (voir réseau_créer_socket()) et vous demande d'indiquer l'adresse IP à laquelle se connecter (une chaîne de caractères) ainsi que le nom du serveur. Si la connexion échoue, une valeur inférieure à 0 sera renvoyée. La différence entre cette fonction et network_connect() est que cette fonction peut se connecter à n'importe quel serveur et ne fait rien aux données brutes, ce qui signifie que vous devez implémenter les protocoles vous-même du côté du serveur. Notez que, par défaut, la fonction est synchrone, ce qui signifie que votre jeu peut sembler se "bloquer" pendant que la connexion est établie. Vous pouvez définir une valeur de timeout pour la connexion, ou alternativement rendre la création asynchrone, en utilisant la fonction réseau_set_config()ou encore utiliser la fonction network_connect_async_raw() à la place. Syntaxe : network_connect_raw(socket, url, port) ; Arguments Description prise de courant L'identifiant du socket à utiliser. url L'URL ou l'IP à laquelle se connecter (une chaîne). port Le port auquel se connecter. Les retours : Real Exemple : client = network_create_socket(network_socket_tcp) ; network_connect_raw(client, "www.macsweeneygames.com", 6510) ; Le code ci-dessus créera un nouveau socket TCP puis tentera de créer une connexion "brute" à travers celui-ci vers l'URL donnée sur le port 6510.
