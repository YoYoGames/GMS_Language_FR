network_connect_raw Avec cette fonction, vous pouvez envoyer une demande de connexion à un serveur. La fonction prend l' identifiant de socket pour se connecter (voir network_create_socket() ) et vous oblige à donner l'adresse IP à laquelle vous connecter (une chaîne) ainsi que le port de connexion, et si la connexion échoue, une valeur inférieure à 0 sera renvoyée. La différence entre cette fonction et network_connect() est que cette fonction peut se connecter à n'importe quel serveur et ne fait rien sur les données brutes, ce qui signifie que vous devez implémenter les protocoles vous-même du côté serveur. Notez que par défaut, la fonction est synchrone, ce qui signifie que votre jeu peut sembler "se bloquer" lorsque la connexion est établie. Vous pouvez définir une valeur de délai d'expiration pour la connexion ou rendre la création asynchrone à l'aide de la fonction network_set_config(), ou utilisez la fonctionnetwork_connect_async_raw() au lieu. Syntaxe:network_connect_raw(socket, url, port); Argument Description socket L'identifiant du socket à utiliser. url L'URL ou l'adresse IP à laquelle se connecter (une chaîne). port Le port auquel se connecter. Retour:Real Exemple:client = network_create_socket(network_socket_tcp); network_connect_raw(client, "www.macsweeneygames.com", 6510); Le code ci-dessus créera un nouveau socket TCP puis tentera de créer une connexion "brute" via celle-ci vers l'URL donnée sur le port 6510.
