network_connect Avec cette fonction, vous pouvez envoyer une demande de connexion à un serveur. La fonction prend l' identifiant de socket pour se connecter (voir network_create_socket() ) et vous oblige à donner l'adresse IP à laquelle vous connecter (une chaîne) ainsi que le port de connexion, et si la connexion échoue, une valeur inférieure à 0 sera renvoyée. La connexion utilise un protocole spécial qui garantit que GameMaker Studio 2 connectent les uns aux autres, mais si vous devez vous connecter à un serveur qui n'est pas un jeu GameMaker Studio 2, vous pouvez utiliser network_connect_raw(). Notez que par défaut, la fonction est synchrone, ce qui signifie que votre jeu peut sembler "se bloquer" lorsque la connexion est établie. Vous pouvez définir une valeur de délai d'expiration pour la connexion ou rendre la création asynchrone à l'aide de la fonction network_set_config(), ou utilisez plutôt la fonction network_connect_async (). Syntaxe:network_connect(socket, url, port); Argument Description socket L'identifiant du socket à utiliser. url L'URL ou l'adresse IP à laquelle se connecter (une chaîne). port Le port auquel se connecter. Retour:Real Exemple:client = network_create_socket(network_socket_tcp); network_connect(client, "192.134.0.1", 6510); Le code ci-dessus créera un nouveau socket TCP, puis tentera de se connecter à l'adresse IP donnée sur le port 6510.
