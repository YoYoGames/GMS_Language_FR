réseau_connect Avec cette fonction, vous pouvez envoyer une requête pour vous connecter à un serveur. La fonction prend l'identifiant de la socket à utiliser pour se connecter (voir réseau_créer_socket()) et vous demande d'indiquer l'adresse IP à laquelle se connecter (une chaîne de caractères) ainsi que le nom du serveur. Si la connexion échoue, une valeur inférieure à 0 sera renvoyée. La connexion utilise un protocole spécial qui garantit que seuls les jeux GameMaker Studio 2 se connectent les uns aux autres. de vous connecter à un serveur qui n'est pas un jeu GameMaker Studio 2, vous pouvez utiliser la commande réseau_connect_raw(). Notez que, par défaut, la fonction est synchrone, ce qui signifie que votre jeu peut sembler " se bloquer " pendant que la connexion est établie. pendant que la connexion est établie. Vous pouvez définir une valeur de timeout pour la connexion, ou alternativement rendre la création asynchrone, en utilisant la fonction réseau_set_config()ou encore utiliser la fonction network_connect_async() à la place. Syntaxe : network_connect(socket, url, port) ; Arguments Description prise de courant L'identifiant du socket à utiliser. url L'URL ou l'IP à laquelle se connecter (une chaîne). port Le port auquel se connecter. Les retours : Real Exemple : client = network_create_socket(network_socket_tcp) ; network_connect(client, "192.134.0.1", 6510) ; Le code ci-dessus va créer un nouveau socket TCP et tenter de se connecter à travers celui-ci à l'adresse IP donnée sur le port 6510.
