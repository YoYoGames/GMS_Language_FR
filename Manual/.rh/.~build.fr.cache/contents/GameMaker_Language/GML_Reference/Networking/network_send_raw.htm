network_send_raw Avec cette fonction, vous pouvez envoyer un paquet de données "brutes" via le réseau. La fonction prend l' identifiant du socket pour se connecter, puis vous devez fournir l' identifiant du tampon qui contient les données à envoyer (pour plus d'informations sur les tampons, voir Référence - Tampons ) et enfin la taille (en octets) du paquet de données. Les données envoyées ne sont en aucun cas formatées par GameMaker Studio 2 et les appareils récepteurs les recevront sous forme de flux, ce qui signifie que vous devrez les gérer vous-même. La fonction renverra le nombre d'octets de données envoyés, ou un nombre inférieur à 0 si l'envoi a échoué. Syntaxe:network_send_raw(socket, buffer, size); Argument Description socket L'identifiant du socket à utiliser. buffer L'identifiant du tampon à partir duquel obtenir les données. size La taille (en octets) des données. Retour:Real Exemple:buff = buffer_load("player_save.dat"); network_send_raw(sock, buff, buffer_get_size(buff)); Les informations ci-dessus chargent des données de tampon précédemment enregistrées dans la mémoire et renvoient l'ID de tampon à stocker dans la variable "buff". Ce tampon complet est ensuite envoyé sous forme de paquet de données brutes sur le réseau en utilisant le socket identifié par la variable «sock».
