network_send_packet Avec cette fonction, vous pouvez envoyer un "paquet" de données via le réseau. La fonction prend l' identifiant du socket pour se connecter, puis vous devez fournir l' identifiant du tampon qui contient les données à envoyer (pour plus d'informations sur les tampons, voir Référence - Tampons ) et enfin la taille (en octets) du paquet de données. Les paquets envoyés avec cette fonction sont formatés de telle sorte que le jeu GameMaker Studio 2 recevant les données puisse «diviser» correctement les paquets, et la fonction renverra le nombre d'octets de données envoyés, ou un nombre inférieur à 0 si l'envoi a échoué. Il est à noter que la taille finale des données envoyées qui sont renvoyées par cette fonction comprendra également les GameMaker, qui représentent 12 octets supplémentaires. Syntaxe:network_send_packet(socket, buffer, size); Argument Description socket L'identifiant du socket à utiliser. buffer L'identifiant du tampon à partir duquel obtenir les données. size La taille (en octets) des données. Retour:Real Exemple:buff = buffer_load("player_save.dat"); network_send_packet(sock, buff, buffer_get_size(buff)); Les informations ci-dessus chargent des données de tampon précédemment enregistrées dans la mémoire et renvoient l'ID de tampon à stocker dans la variable "buff". Ce tampon complet est ensuite envoyé sous forme de paquet sur le réseau en utilisant le socket identifié par la variable "sock".
