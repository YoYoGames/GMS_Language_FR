network_connect_async Avec cette fonction, vous pouvez envoyer une demande de connexion à un serveur. La fonction prend l' identifiant de socket pour se connecter (voir network_create_socket() ) et vous oblige à donner l'adresse IP à laquelle vous connecter (une chaîne) ainsi que le port de connexion, et si la connexion échoue, une valeur inférieure à 0 sera renvoyée. La connexion utilise un protocole spécial qui garantit que GameMaker Studio 2 connectent les uns aux autres, mais si vous devez vous connecter à un serveur qui n'est pas un jeu GameMaker Studio 2, vous pouvez utiliser network_connect_async_raw(). Notez que cette fonction est asynchrone, générant un événement de mise en réseau asynchrone du typenetwork_type_non_blocking_connect. Syntaxe:network_connect_async(socket, url, port); Argument Description socket L'identifiant du socket à utiliser. url L'URL ou l'adresse IP à laquelle se connecter (une chaîne). port Le port auquel se connecter. Retour:Real Exemple:client = network_create_socket(network_socket_tcp); network_connect_async(client, "192.134.0.1", 6510); Le code ci-dessus créera un nouveau socket TCP, puis tentera d'établir une connexion asynchrone via celui-ci à l'adresse IP donnée sur le port 6510.
