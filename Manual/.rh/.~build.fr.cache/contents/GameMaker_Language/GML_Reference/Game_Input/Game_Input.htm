Entrée du jeu L'élément fondamental qui fait d'un jeu un jeu est l'interaction avec le joueur. Celle-ci s'effectue par divers moyens, avec le clavier et la souris sur les ordinateurs, l'écran tactile et l'inclinaison de l'appareil sur les mobiles, voire le gamepad... sur toutes les plateformes ! Pour commencer, dans GameMaker Studio 2, vous utilisiez les événements dédiés à la souris ou au clavier sur les PC ou les événements gestuels sur les mobiles, mais parfois vous avez besoin de plus de contrôle que ce que ces événements fournissent, ou vous voulez inclure des commandes de manette de jeu ou plusieurs options de contrôle différentes. Pour cela, vous pouvez utiliser les différentes fonctions d'entrée du jeu pour obtenir les boutons ou les touches qui ont été enfoncés ou relâchés, ainsi que pour récupérer les informations sur le jeu. appuyés ou relâchés, ainsi que pour récupérer les touches d'un écran ou le déplacement d'une manette de jeu. Il existe un grand nombre de fonctions de saisie et de contrôle dans vos jeux. Elles ont donc été divisées en sections liées à des méthodes de saisie spécifiques afin de faciliter la recherche de ce dont vous avez besoin.
