état_virtuel_du_clavier Cette fonction peut être utilisée pour obtenir le statut du clavier virtuel sur le périphérique exécutant le jeu. La fonction renverra vrai si le clavier virtuel du système d'exploitation est visible/affiché ou faux s'il est caché/effacé. Syntaxe keyboard_virtual_status() ; Les retours : Booléen Exemple : var _status = keyboard_virtual_status() ; if _status == false { keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false) ; } Le code ci-dessus affichera le clavier virtuel de l'OS si l'état actuel est faux.
