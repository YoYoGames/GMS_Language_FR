clavier_virtuel_cacher Cette fonction peut être utilisée pour masquer le clavier virtuel sur le périphérique exécutant le jeu. L'appel de cette fonction génère un événement asynchrone système, dans lequel la carte DS async_load DS map sera remplie avec les paires clé/valeur suivantes : "event_type" - ce sera "l'état du clavier virtuel" lorsqu'il sera déclenché par les fonctions du clavier virtuel. "screen_height " - la hauteur du clavier virtuel (en pixels). Elle sera de 0 si le clavier est invisible. "keyboard_status " - l'état actuel du clavier, renvoyé comme l'une des chaînes suivantes : "hiding" (caché) "caché" "visible" "visible" Syntaxe keyboard_virtual_hide() ; Les retours : N/A Exemple : if input == true { input = false ; keyboard_virtual_hide() ; } Le code ci-dessus masquera le clavier virtuel du système d'exploitation si la variable donnée n'est pas définie comme fausse.
