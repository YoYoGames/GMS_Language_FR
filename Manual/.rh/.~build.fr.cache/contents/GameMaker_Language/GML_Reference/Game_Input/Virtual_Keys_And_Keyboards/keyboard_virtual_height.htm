hauteur_virtuelle_du_clavier Cette fonction renvoie la hauteur actuelle en pixels du clavier virtuel, en fonction de la taille de l'écran. Si le clavier n'est pas visible, 0 sera renvoyé. Syntaxe keyboard_virtual_height() ; Les retours : Int Exemple : si keyboard_virtual_status() == true { key_h = keyboard_virtual_height() ; } Le code ci-dessus vérifiera l'état du clavier virtuel du système d'exploitation, et s'il est visible, il définira une variable à la hauteur du clavier.
