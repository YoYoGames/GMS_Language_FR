keyboard_virtual_height Cette fonction renverra la hauteur actuelle en pixels du clavier virtuel, en fonction de la taille de l' affichage. Si le clavier n'est pas visible, 0 sera renvoyé. Syntaxekeyboard_virtual_height(); Retour:Int Exemple:if keyboard_virtual_status() == true     {     key_h = keyboard_virtual_height();     } Le code ci-dessus vérifiera l'état du clavier virtuel du système d'exploitation et, s'il est visible, définissez une variable à la hauteur du clavier.
