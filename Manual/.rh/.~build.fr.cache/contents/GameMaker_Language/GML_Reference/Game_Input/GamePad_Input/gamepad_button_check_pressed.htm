gamepad_button_check_pressed Cette fonction renvoie vrai ou faux selon que le bouton du gamepad est détecté comme ayant été pressé ou non. Notez que cette fonction ne se déclenchera qu'une seule fois pour le bouton, la première fois qu'il sera pressé. Pour que Pour qu'elle se déclenche à nouveau, le bouton doit d'abord être relâché, puis réappuyé. Si vous avez besoin de vérifier une pression continue du bouton, vous devez utiliser la fonction gamepad_button_check(). Si vous vérifiez un bouton analogique, la vérification tiendra compte du réglage du seuil et ne renverra true que si la valeur brute de l'entrée dépasse le seuil donné. (vous pouvez obtenir cette valeur brute à l'aide de la fonction gamepad_button_value()). Syntaxe : gamepad_button_check_pressed(device, button) ; Arguments Description dispositif Quel "slot" de périphérique gamepad vérifier. bouton Quelle constante de bouton de manette à vérifier. Les retours : Booléen Exemple : if gamepad_button_check_pressed(0, gp_start) { audio_play_sound(snd_Start, 0, false) ; room_goto(rm_Level_1) ; } Le code ci-dessus détectera si le bouton "start" de la manette de jeu connectée au dispositif "slot" 0 a été pressé ou non et changera de pièce si c'est le cas.
