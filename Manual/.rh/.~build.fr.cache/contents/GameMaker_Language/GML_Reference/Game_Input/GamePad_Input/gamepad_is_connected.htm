gamepad_is_connected Cette fonction retournera si une manette de jeu est connectée au "slot" donné (retourne true ) ou non (retourne false ). Vous utiliseriez normalement cette fonction en conjonction avec le gamepad_get_device_count() fonction pour obtenir le nombre correct de manettes de jeu et / ou de "slots" de manettes de jeu disponibles. Notez qu'il peut y avoir un léger délai entre l'utilisateur connectant la manette de jeu et GameMaker Studio 2 détectant comme étant connectée (c'est particulièrement le cas lorsqu'il s'agit de manettes connectées Bluetooth). REMARQUE: sur certaines plates-formes, en particulier les consoles, cette fonction peut retourner false immédiatement après la connexion / la sélection d'une manette de jeu et peut nécessiter une vérification dans une alarme quelques images (étapes) plus tard pour qu'elle détecte correctement la manette de jeu. Syntaxe:gamepad_is_connected(numb); Argument Description device Quelle «fente» de manette de jeu vérifier. Retour:Boolean Exemple:var gp_num = gamepad_get_device_count(); for (var i = 0; i < gp_num; i++;)    {    if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;    } Le code ci-dessus parcourt les manettes de jeu disponibles (ou les emplacements pour manettes de jeu), puis vérifie chacune d'elles pour une manette de jeu connectée. la valeur retournée est ensuite utilisée pour définir un tableau global sur true ou false pour une utilisation lors de contrôles ultérieurs.
