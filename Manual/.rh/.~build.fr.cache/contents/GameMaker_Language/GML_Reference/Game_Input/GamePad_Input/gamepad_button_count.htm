gamepad_button_count Cette fonction renverra le nombre total de boutons disponibles pour la manette de jeu connectée au "slot" de l'appareil donné. Syntaxe:gamepad_button_count(device); Argument Description device Quel «slot» de périphérique de manette de jeu vérifier. Retour:Real Exemple:b_num = gamepad_button_count(0); Le code ci-dessus renverra le nombre de boutons disponibles sur la manette de jeu branchée sur le périphérique "slot" 0 et stockera la valeur dans la variable "b_num".
