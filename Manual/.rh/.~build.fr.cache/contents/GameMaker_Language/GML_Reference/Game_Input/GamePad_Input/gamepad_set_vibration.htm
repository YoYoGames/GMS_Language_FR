gamepad_set_vibration Avec cette fonction, vous pouvez définir la vibration des moteurs du gamepad, avec chaque moteur utilisant une valeur de 0 (aucune vibration) à 1 (vibration complète). Notez qu'il n'y a pas de limite de temps pour cette fonction, vous devrez donc utiliser une variable ou une alarme pour désactiver la vibration (mettre les moteurs à 0) après un certain temps, sinon vous ne pourrez pas utiliser cette fonction. Vous devrez donc utiliser une variable ou une alarme pour arrêter la vibration (mettre les moteurs à 0) après un certain temps, sinon le gamepad continuera à vibrer indéfiniment. REMARQUE: Cette fonction n'est actuellement disponible que pour les modules cibles standard Windows PS4 et Xbox One. Syntaxe : gamepad_set_vibration(device, left_motor, right_motor) ; Arguments Description dispositif Quel "slot" de périphérique gamepad vérifier. moteur_gauche La quantité de vibration du moteur gauche de 0 à 1. moteur_droit La quantité de vibration du moteur droit de 0 à 1. Les retours : N/A Exemple : si vies = 0 { gamepad_set_vibration(0, 1, 1) ; alarm[0] = room_speed / 2 ; } Le code ci-dessus serait utilisé (par exemple) dans un événement de collision pour faire vibrer la manette de jeu branchée sur le "slot" 0 pendant une demi-seconde, l'alarme définie étant utilisée pour la désactiver une fois ce temps écoulé.
