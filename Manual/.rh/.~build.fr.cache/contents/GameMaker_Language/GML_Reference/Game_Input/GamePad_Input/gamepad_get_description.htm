gamepad_get_description Cette fonction renvoie une chaîne de caractères avec la description de la manette de jeu donnée. Par exemple, une manette PS3 peut renvoyer une chaîne similaire à "PLAYSTATION(R)3 Controller", ou une manette XBox360 peut renvoyer "Xbox 360 Controller (XInput GAMEPAD STANDARD)". Cette chaîne dépend du matériel et la valeur renvoyée dépendra de la manette de jeu branchée dans le "slot" du périphérique qui est vérifié. Syntaxe : gamepad_get_description(device) ; Arguments Description dispositif Quel "slot" de gamepad pour obtenir le nom de. Les retours : Chaîne de caractères Exemple : var gp_num = gamepad_get_device_count() ; for (var i = 0 ; i < gp_num ; i++ ;) { if gamepad_is_connected(i) { draw_text(32, 32 + (i * 32), gamepad_get_description(i)) ; } else { draw_text(32, 32 + (i * 32), "No Gamepad Connected") ; } } Le code ci-dessus parcourt en boucle tous les emplacements de manettes de jeu pour vérifier si des périphériques sont connectés, puis affiche du texte à l'écran selon qu'une manette de jeu est connectée ou non à l'emplacement.
