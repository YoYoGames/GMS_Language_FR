gamepad_remove_mapping Cette fonction peut être utilisée pour supprimer le mappage de périphérique actuel de l'index de slot de gamepad donné. Une fois appelé, l'index du slot devra être remappé en utilisant la fonction gamepad_test_mapping() si vous voulez être en mesure d'utiliser les constantes du gamepad pour détecter correctement les entrées (les entrées directes peuvent toujours être récupérées à l'aide des fonctions gamepad_axis/button/hat_count() et gamepad_axis/button/hat_value() ensemble). Syntaxe : gamepad_remove_mapping(index) ; Arguments Description indice De quel "slot" de l'index du gamepad faut-il retirer le mapping ? Les retours : N/A Exemple : if remap == true { gamepad_remove_mapping(global.PadIndex) ; } Le code ci-dessus supprime le mappage de l'emplacement d'indexation du gamepad donné en fonction de la valeur d'une variable.
