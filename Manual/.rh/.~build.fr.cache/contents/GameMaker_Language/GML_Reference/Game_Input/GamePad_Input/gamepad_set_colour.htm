gamepad_set_colour Cette fonction peut être utilisée pour définir la couleur des voyants dans un contrôleur PlayStation. Vous spécifiez l'emplacement de périphérique à définir, puis donnez une couleur, qui peut être l'une des constantes de couleur ou une valeur de couleur créée à l'aide des fonctions de couleur spécifiques ou d'une valeur HEX (comme $FFFFFFF ). Syntaxe:gamepad_set_colour(device, colour); Argument Description device Quel «slot» de périphérique de manette de jeu définir. colour La couleur à utiliser. Retour:N/A Exemple:if health < 10    {    gamepad_set_colour(0, c_red);    } Le code ci-dessus définira les voyants du contrôleur PlayStation sur rouge si le health variable tombe en dessous de 10.
