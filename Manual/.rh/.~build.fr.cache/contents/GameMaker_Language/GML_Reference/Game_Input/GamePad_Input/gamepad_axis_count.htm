gamepad_axis_count Cette fonction renverra le nombre de commandes «axes» sur l'appareil en cours de vérification. Ces commandes sont les «manettes» de direction analogique sur la plupart des contrôleurs. Syntaxe:gamepad_axis_count(device); Argument Description device Quel «slot» de périphérique de manette de jeu vérifier. Retour:Real Exemple:axis = gamepad_axis_count(0)) Le code ci-dessus stocke le nombre d'axes disponibles pour la manette de jeu connectée à l'appareil "slot" 0 dans la variable "axis".
