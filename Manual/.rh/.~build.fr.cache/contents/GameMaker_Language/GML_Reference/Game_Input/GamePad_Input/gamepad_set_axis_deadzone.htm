gamepad_set_axis_deadzone Cette fonction peut être utilisée pour régler la "zone morte" de l'axe du joystick. Vous spécifiez l'emplacement de périphérique à définir, puis définissez une valeur de 0 à 1 et si la quantité d'entrée est inférieure à la valeur donnée, l'axe du joystick est considéré comme à 0. Notez qu'il s'agit d'un paramètre global qui affectera tous axe de tous les joysticks connectés à l'emplacement de périphérique spécifié. L'utilisation de cette fonction ne signifie pas que la valeur de l'axe commencera à partir de la valeur de la zone morte, car la fonctiongamepad_axis_value() renverra toujours une valeur normalisée entre -1 et 0 ou 0 et 1. Par exemple, régler la zone morte à 0,2 signifie que pousser le manche vers la droite ne commencera à renvoyer une valeur de 0 à 1 que lorsque la valeur brute de l' axe est supérieure à 0,2 - Ainsi, lorsque la valeur brute est 0,2, la valeur de retour sera 0, lorsque la valeur brute est à 0,5, la valeur de retour sera 0,375, ou si la valeur brute est 0,9, alors la valeur de retour sera 0,875. Syntaxe:gamepad_set_axis_deadzone(device, deadzone); Argument Description device Quel «slot» de périphérique de manette de jeu vérifier. deadzone La valeur de la zone morte de 0 à 1. Retour:N/A Exemple:if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05); if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05); if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05); if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05); Le code ci-dessus définira la zone morte de l'axe du joystick sur l'un des 4 appareils connectés à 0,05.
