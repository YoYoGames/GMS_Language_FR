gamepad_set_axis_deadzone Cette fonction peut être utilisée pour définir la "zone morte" de l'axe du joystick. Vous spécifiez le slot de périphérique à définir, puis vous définissez une valeur de 0 à 1. Si la quantité d'entrée est inférieure à la valeur donnée, l'axe du joystick est considéré comme étant à 0. Notez qu'il s'agit d'un réglage global qui affectera tous les axes de tous les joysticks connectés au slot de périphérique spécifié. L'utilisation de cette fonction ne signifie pas que la valeur de l'axe partira de la valeur de la zone morte, car la fonction gamepad_axis_value() renvoie toujours une valeur normalisée comprise entre -1 et 0 ou 0 et 1. Par exemple, si vous définissez la zone morte à 0,2, le fait de pousser le manche vers la droite ne commencera à renvoyer une valeur comprise entre 0 et 1 que lorsque la valeur brute de l'axe sera supérieure à 0,2 - ainsi, lorsque la valeur brute sera de 0,2, la valeur renvoyée sera 0, lorsque la valeur brute est à 0,5, la valeur de retour sera 0,375, ou si la valeur brute est 0,9, alors la valeur de retour sera 0,875. Syntaxe : gamepad_set_axis_deadzone(device, deadzone) ; Arguments Description dispositif Quel "slot" de périphérique gamepad vérifier. zone morte La valeur de la zone morte de 0 à 1. Les retours : N/A Exemple : if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05) ; if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05) ; if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05) ; if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05) ; Le code ci-dessus réglera la zone morte de l'axe du joystick sur l'un des 4 appareils connectés à 0,05.
