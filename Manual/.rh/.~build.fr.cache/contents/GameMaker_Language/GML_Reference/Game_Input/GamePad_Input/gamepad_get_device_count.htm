gamepad_get_device_count Cette fonction vous indiquera l'une des deux choses suivantes. Soit le nombre de manettes de jeu connectées, soit le nombre de "slots" disponibles pour les manettes de jeu à connecter. La valeur de retour réelle dépendra de la plate-forme et de la configuration interne de celle-ci. de cette plateforme et, en tant que telle, cette fonction doit être utilisée en conjonction avec la fonction gamepad_is_connected() pour s'assurer du nombre exact de pads connectés à tout moment. Syntaxe : gamepad_get_device_count() ; Les retours : Real Exemple : var gp_num = gamepad_get_device_count() ; for (var i = 0 ; i < gp_num ; i++ ;) { if gamepad_is_connected(i) { global.gp[i] = true ; } else { global.gp[i] = false ; } } Le code ci-dessus passe en boucle les game pads disponibles (ou les emplacements de gamepad) et vérifie pour chacun d'entre eux s'il existe un gamepad connecté. La valeur renvoyée est ensuite utilisée pour définir un tableau global sur true ou false à utiliser dans les vérifications futures.
