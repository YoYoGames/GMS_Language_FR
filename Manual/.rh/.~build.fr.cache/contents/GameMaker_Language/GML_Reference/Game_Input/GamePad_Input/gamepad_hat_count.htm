gamepad_hat_count Cette fonction renvoie le nombre total de chapeaux disponibles pour la manette de jeu connectée au "slot" de périphérique donné. Les chapeaux font généralement référence aux boutons haut/bas/gauche/droite, et notez que sur la cible Windows, les chapeaux ne sont disponibles que sur les contrôleurs DirectInput (donc, à partir du slot 4). Syntaxe : gamepad_hat_count(device) ; Arguments Description dispositif Quel "slot" de périphérique gamepad vérifier. Les retours : Real Exemple : h_num = gamepad_hat_count(4) ; Le code ci-dessus renvoie le nombre de chapeaux disponibles sur la manette de jeu branchée sur le périphérique "slot" 4 et stocke la valeur dans la variable "h_num".
