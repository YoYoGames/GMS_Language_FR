gamepad_hat_count Cette fonction renverra le nombre total de chapeaux disponibles pour la manette de jeu connectée au "slot" de l'appareil donné. Les chapeaux font généralement référence aux boutons haut / bas / gauche / droite, et notez que sur la Windows, les chapeaux ne sont disponibles que sur les contrôleurs DirectInput (donc, à partir du slot 4). Syntaxe:gamepad_hat_count(device); Argument Description device Quel «slot» de périphérique de manette de jeu vérifier. Retour:Real Exemple:h_num = gamepad_hat_count(4); Le code ci-dessus renverra le nombre de chapeaux disponibles sur la manette de jeu branchée sur le périphérique "slot" 4 et stockera la valeur dans la variable "h_num".
