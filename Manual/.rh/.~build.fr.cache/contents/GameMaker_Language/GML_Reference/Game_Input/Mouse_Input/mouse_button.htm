mouse_button Cette variable en lecture seule renvoie le bouton de la souris qui est actuellement enfoncé (actuellement, comme dans cette étape) et peut renvoyer n'importe laquelle des constantes spéciales de la souris sauf mb_any. Syntaxe:mouse_button; Retour:Constant (mb_left, mb_middle, mb_right, my_none) Exemple:if mouse_button = mb_left    {    x -= 1;    } Cela déplace l'instance actuelle vers la gauche si le bouton gauche de la souris est enfoncé.
