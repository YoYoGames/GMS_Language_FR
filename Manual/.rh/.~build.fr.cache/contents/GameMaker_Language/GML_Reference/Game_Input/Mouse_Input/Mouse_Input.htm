Entrée souris La saisie de la souris est acceptée sur toutes les plates-formes (sur les appareils mobiles, elle est acceptée comme un simple écran tactile. Si vous devez utiliser le multi-touch, vous devez utiliser les fonctions spécifiques à l'appareil) et comporte quelques constantes utilisées pour spécifier les boutons étant pressé. Ces constantes sont présentées dans le tableau suivant: Constant Description mb_left Le bouton gauche de la souris mb_middle Le bouton central de la souris (cela peut ne pas être valable pour toutes les plates-formes cibles) mb_right Le bouton droit de la souris mb_none Pas de bouton de la souris mb_any N'importe lequel des boutons de la souris Les fonctions suivantes existent pour les commandes de bouton de souris standard: mouse_buttonmouse_check_buttonmouse_check_button_pressedmouse_check_button_releasedmouse_clearmouse_lastbuttonmouse_wheel_upmouse_wheel_downmouse_xmouse_y Notez qu'il existe également un ensemble de fonctions de fenêtre liées à l'utilisation de la souris sur les cibles du bureau: window_mouse_get_xwindow_mouse_get_ywindow_mouse_setwindow_view_mouse_get_xwindow_view_mouse_get_ywindow_views_mouse_get_xwindow_views_mouse_get_y Retour: Entrée de jeu Suivant: Entrée de périphérique © Copyright YoYo Games Ltd. 2021 Tous droits réservés
