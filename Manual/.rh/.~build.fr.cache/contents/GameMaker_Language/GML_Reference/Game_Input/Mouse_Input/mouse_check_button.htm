Bouton de contrôle de la souris Cette fonction renvoie vrai si le bouton de la souris à vérifier est maintenu enfoncé ou faux s'il ne l'est pas. Vous fournissez le bouton de la souris à vérifier à partir de l'une des constantes suivantes : Constant Description mb_gauche Le bouton gauche de la souris mb_middle Le bouton central de la souris (ceci peut ne pas être valable pour toutes les plateformes cibles) mb_right Le bouton droit de la souris mb_side1* Bouton latéral de la souris 1 mb_side2* Bouton latéral de la souris 2 mb_any L'un des boutons de la souris mb_none Pas de bouton de souris * NOTE: Les boutons mb_side1 et mb_side2 ne sont disponibles que sous Windows, macOS, Ubuntu et HTML5. Syntaxe : mouse_check_button(numb) ; Arguments Description engourdi La constante du bouton de la souris à vérifier. Les retours : Booléen Exemple : if mouse_check_button(mb_left) { instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star) ; } Le code ci-dessus vérifiera la présence du bouton gauche de la souris et chaque fois qu'il sera maintenu enfoncé, il créera une instance de l'objet indexé dans "obj_Star".
