mouse_check_button Cette fonction retournera true si le bouton de la souris en cours de vérification est maintenu enfoncé ou false si ce n'est pas le cas. Vous fournissez le bouton de la souris pour vérifier à partir de l'une des constantes suivantes: Constant Description mb_left Le bouton gauche de la souris mb_middle Le bouton central de la souris (cela peut ne pas être valable pour toutes les plates-formes cibles) mb_right Le bouton droit de la souris mb_none Pas de bouton de la souris mb_any N'importe lequel des boutons de la souris Syntaxe:mouse_check_button(numb); Argument Description numb Quelle constante de bouton de la souris vérifier. Retour:Boolean Exemple:if mouse_check_button(mb_left)    {    instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);    } Le code ci-dessus vérifiera le bouton gauche de la souris et chaque étape où il est maintenu enfoncé créera une instance de l'objet indexé dans "obj_Star".
