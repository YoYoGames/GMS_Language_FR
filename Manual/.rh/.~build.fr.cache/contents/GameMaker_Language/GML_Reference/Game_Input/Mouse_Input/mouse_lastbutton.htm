mouse_lastbutton Cette variable renvoie le dernier bouton de la souris qui a été enfoncé et peut renvoyer n'importe laquelle des constantes spéciales de la souris (vous pouvez également définir cette variable sur l'une des constantes). La constante renvoyée (ou en cours de définition) sera l'une des suivantes: Constant Description mb_left Le bouton gauche de la souris mb_middle Le bouton central de la souris (cela peut ne pas être valable pour toutes les plates-formes cibles) mb_right Le bouton droit de la souris mb_none Pas de bouton de la souris Syntaxe:mouse_lastbutton; Retour:Constant Exemple:if mouse_lastbutton = mb_left    {    x -= 1;    } Cela déplace l'instance actuelle vers la gauche si le dernier bouton enfoncé était le bouton gauche de la souris.
