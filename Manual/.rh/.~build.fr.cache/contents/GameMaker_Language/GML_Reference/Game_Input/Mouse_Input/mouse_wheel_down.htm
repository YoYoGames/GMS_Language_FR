mouse_wheel_down Cette fonction renvoie true si la molette de la souris est tournée vers le bas et false autrement. Syntaxe:mouse_wheel_down(); Retour:Boolean Exemple:if mouse_wheel_down()    {    y += 10;    } Cela déplace l'instance actuelle vers le bas de l'écran si la molette de la souris est tournée vers le haut.
