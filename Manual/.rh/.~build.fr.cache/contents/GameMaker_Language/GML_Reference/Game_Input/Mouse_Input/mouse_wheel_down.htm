mouse_wheel_down Cette fonction renvoie vrai si la molette de la souris est tournée vers le bas et faux sinon. Syntaxe : mouse_wheel_down() ; Les retours : Booléen Exemple : if mouse_wheel_down() { y += 10 ; } Cela déplace l'instance actuelle vers le bas de l'écran si la molette de la souris est tournée vers le haut.
