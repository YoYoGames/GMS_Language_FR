mouse_check_button_pressed Cette fonction renvoie un message vrai si le bouton de la souris vérifié a été pressé ou faux s'il ne l'a pas été. Cette fonction ne sera déclenchée qu'une seule fois pour chaque bouton de la souris lorsqu'il est pressé pour la première fois. devra avoir été relâché et pressé à nouveau. Notez qu'elle sera considérée comme déclenchée pendant la durée de l'étape, et pour toutes les instances qui ont des événements de souris ou qui utilisent cette même fonction. Vous fournissez le bouton de la souris à vérifier à partir de l'une des constantes suivantes : Constant Description mb_gauche Le bouton gauche de la souris mb_middle Le bouton central de la souris (ceci peut ne pas être valable pour toutes les plateformes cibles) mb_right Le bouton droit de la souris mb_side1* Bouton latéral de la souris 1 mb_side2* Bouton latéral de la souris 2 mb_any L'un des boutons de la souris mb_none Pas de bouton de souris * NOTE: Les boutons mb_side1 et mb_side2 ne sont disponibles que sur Windows, macOS, Ubuntu et HTML5. Syntaxe : mouse_check_button_pressed(numb) ; Arguments Description engourdi La constante du bouton de la souris à vérifier. Les retours : Booléen Exemple : if mouse_check_button_pressed(mb_left) { score += 50 ; } Le code ci-dessus vérifie si le bouton gauche de la souris a été enfoncé et si c'est le cas, il ajoute 50 au score.
