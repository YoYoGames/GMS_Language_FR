mouse_clear Cette fonction efface l'état actuel du bouton de la souris. Cela signifie que la vérification du maintien du bouton enfoncé ne retournera pas vrai jusqu'à ce que le joueur relâche le bouton et appuie à nouveau dessus (mais l'état de relâchement sera toujours détecté si l'effacement est effectué alors que le bouton de la souris est maintenu enfoncé). Vous pouvez fournir une des constantes de bouton de souris listées sur cette page. Syntaxe : mouse_clear(bouton) ; Arguments Description bouton La constante du bouton de la souris à vérifier. Les retours : Booléen Exemple : mouse_clear(mb_any) ; Le code ci-dessus effacera l'état d'activation de tous les boutons de la souris.
