mouse_clear Cette fonction effacera l'état actuel de tous les boutons de la souris utilisés. Cela signifie que les chèques pour leur maintien ne reviendront pas true jusqu'à ce que le joueur relâche les boutons et les réprime (mais l'état de libération sera toujours détecté si l'effacement est effectué alors que le bouton de la souris est maintenu enfoncé). Vous fournissez le bouton de la souris pour vérifier à partir de l'une des constantes suivantes: Constant Description mb_left Le bouton gauche de la souris mb_middle Le bouton central de la souris (cela peut ne pas être valable pour toutes les plates-formes cibles) mb_right Le bouton droit de la souris mb_none Pas de bouton de la souris mb_any N'importe lequel des boutons de la souris Syntaxe:mouse_clear(button); Argument Description button Quelle constante de bouton de la souris vérifier. Retour:Boolean Exemple:mouse_clear(mb_any); Le code ci-dessus effacera l'état bas de tous les boutons de la souris.
