vérification du clavier Avec cette fonction, vous pouvez vérifier si une touche est maintenue enfoncée ou non. Contrairement à la fonction keyboard_check_pressed() ou keyboard_check_released() qui ne sont déclenchées qu'une seule fois lorsque la touche est enfoncée ou relâchée, cette fonction est déclenchée à chaque étape pendant laquelle la touche est maintenue enfoncée. La fonction prend une valeur de code de touche telle que retournée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'une des constantes vk_* listées sur la page principale de saisie du clavier. Syntaxe : keyboard_check(key) ; Arguments Description clé La clé pour vérifier l'état d'abattement de. Les retours : Booléen Exemple : if keyboard_check(vk_left) { x -= 5 ; } Le code ci-dessus vérifiera si la touche flèche est pressée et déplacera l'instance de 5 pixels vers la gauche à chaque fois qu'il retournera vrai.
