keyboard_check Avec cette fonction, vous pouvez vérifier si une touche est maintenue enfoncée ou non. Contrairement au keyboard_check_pressed() ou keyboard_check_released() fonctions qui ne sont déclenchées qu'une seule fois lorsque la touche est enfoncée ou relâchée, cette fonction est déclenchée à chaque étape pendant laquelle la touche est maintenue enfoncée. La fonction prendra une valeur de code clé telle que renvoyée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'un des vk_* constantes répertoriées sur la page principale de saisie au clavier. Syntaxe:keyboard_check(key); Argument Description key La clé pour vérifier l'état de. Retour:Boolean Exemple:if keyboard_check(vk_left)    {    x -= 5;    } Le code ci-dessus vérifiera si la touche fléchée est enfoncée et déplace l'instance de 5 pixels vers la gauche à chaque étape qu'elle renvoie true.
