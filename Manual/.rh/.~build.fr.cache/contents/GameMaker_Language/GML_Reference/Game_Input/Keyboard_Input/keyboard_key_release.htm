keyboard_key_release Avec cette fonction, vous pouvez simuler le relâchement de n'importe quelle touche du clavier. La fonction prendra une valeur de code clé telle que renvoyée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'un des vk_* constantes répertoriées sur la page principale de saisie au clavier. Syntaxe:keyboard_key_release(key); Argument Description key La clé pour simuler une version de. Retour:N/A Exemple:keyboard_key_release(vk_space); Cela simulera une version de la barre d'espace.
