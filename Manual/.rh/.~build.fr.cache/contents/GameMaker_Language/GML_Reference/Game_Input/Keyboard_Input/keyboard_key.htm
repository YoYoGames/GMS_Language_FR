keyboard_key Avec cette variable, vous pouvez obtenir le code de la touche qui est actuellement enfoncée et il renverra 0 si aucune touche n'est enfoncée lorsque la vérification est terminée. Syntaxe:keyboard_key; Retour:Real Exemple:switch (keyboard_key)    {    case vk_numpad1: gun = weapon[0][0]; break;    case vk_numpad2: gun = weapon[1][0]; break;    case vk_numpad3: gun = weapon[2][0]; break;    } Le code ci-dessus utilise la valeur de keyboard_key variable pour définir une variable sur la même valeur qu'un tableau.
