keyboard_unset_map Avec cette fonction, vous pouvez effacer toutes les clés remappées afin qu'elles reviennent à leur état par défaut, c'est-à-dire: toutes les clés à mapper à elles-mêmes. Syntaxe:keyboard_get_map(key); Retour:N/A Exemple:if keyboard_check_pressed(vk_escape)     {     keyboard_unset_map();     } L'exemple de code ci-dessus réinitialisera toutes les touches mappées à leurs paramètres par défaut si l'utilisateur appuie sur la touche «Échap».
