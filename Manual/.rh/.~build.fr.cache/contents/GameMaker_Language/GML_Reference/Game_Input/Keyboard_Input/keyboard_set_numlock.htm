keyboard_set_numlock Vous pouvez utiliser cette fonction pour activer ou désactiver le verrouillage numérique du clavier (défini sur vrai pour activé et faux pour désactivé). REMARQUE: cette fonctionnalité n'est disponible que dans les Windows cibles de Windows et ne fonctionnera sur aucun autre appareil. Syntaxe:keyboard_set_numlock(value); Argument Description value Définissez ceci sur true pour "on" et false pour "off" Retour:N/A Exemple:if keyboard_get_numlock()    {    keyboard_set_numlock(false);    } else    {    keyboard_set_numlock(true);    } L'exemple de code ci-dessus obtiendra l'état de la touche de verrouillage numérique et si elle est activée (true), elle la désactivera (false) et vice-versa.
