keyboard_get_numlock Vous pouvez utiliser cette fonction pour trouver le statut du verrouillage numérique du clavier avec true renvoyé pour activé et false renvoyé pour désactivé. REMARQUE: cette fonctionnalité est uniquement disponible dans les Windows exe et ne fonctionnera sur aucun autre appareil. Syntaxe:keyboard_get_numlock(); Retour:Boolean Exemple:if keyboard_get_numlock()    {    keyboard_set_numlock(false);    } else    {    keyboard_set_numlock(true);    } L'exemple de code ci-dessus obtiendra l'état de la touche de verrouillage numérique et si elle est activée (true), elle la désactivera (false) et vice-versa.
