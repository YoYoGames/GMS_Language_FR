keyboard_lastchar Cette variable stocke une chaîne de la dernière touche enfoncée. Cette variable n'est pas en lecture seule et vous pouvez la modifier, par exemple pour la définir sur "" (une chaîne vide) si vous l'avez déjà gérée. Syntaxe:keyboard_lastchar; Retour:String Exemple:if keyboard_lastkey != -1    {    str += keyboard_lastchar;    keyboard_lastkey = -1;    } Le code ci-dessus vérifie si la variable lastkey est supérieure à -1, et si c'est le cas, il ajoute ce que la dernière clé était sous forme de chaîne à la variable "str".
