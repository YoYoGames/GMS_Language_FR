clavier_dernière touche Cette variable fait référence à la valeur qui clé_clavier était dans l'image précédente, retournant le code de cette touche (toutes les constantes standard de code de touche sont retournées). Cette variable n' est pas en lecture seule et vous pouvez la modifier, par exemple pour la mettre à -1 si vous l'avez déjà manipulée. Syntaxe : Clavier_dernière touche ; Les retours : Real Exemple : if (keyboard_lastkey != -1) { str += keyboard_lastchar ; keyboard_lastkey = -1 ; } Le code ci-dessus vérifie si la variable lastkey n'est pas égale à -1, et si c'est le cas, il ajoute la dernière touche sous forme de chaîne à la variable "str", puis il réinitialise la variable keyboard_lastkey pour accepter d'autres entrées.
