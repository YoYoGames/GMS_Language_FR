gesture_drag_distance Cette fonction est utilisée pour définir la distance nécessaire pour qu'un événement de mouvement de glissement soit déclenché par le mouvement d'un toucher ou d'un clic. La distance est mesurée en pouces et a une valeur par défaut de 0,1. Syntaxe:gesture_drag_distance(distance); Argument Description distance Distance (en pouces) à définir pour la détection d'événement de mouvement de glissement. Retour:N/A Exemple:if gesture_get_drag_distance() != 0.1     {     gesture_drag_distance(0.1);     } Le code ci-dessus vérifie si la distance de glissement pour les gestes est définie sur 0,1 pouce et si ce n'est pas le cas, il la définit sur cette valeur.
