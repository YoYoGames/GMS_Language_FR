distance_dégagement Cette fonction permet de définir la distance nécessaire pour qu'un événement Geste de glissement soit déclenché par le mouvement d'une touche ou d'un clic. La distance est mesurée en pouces et a une valeur par défaut de 0,1. Syntaxe : gesture_drag_distance(distance) ; Arguments Description distance La distance (en pouces) à définir pour la détection des événements liés aux gestes de déplacement. Les retours : N/A Exemple : if gesture_get_drag_distance() !.= 0.1 { gesture_drag_distance(0.1) ; } Le code ci-dessus vérifie si la distance de déplacement pour les gestes est définie sur 0,1 pouce et si ce n'est pas le cas, il la définit sur cette valeur.
