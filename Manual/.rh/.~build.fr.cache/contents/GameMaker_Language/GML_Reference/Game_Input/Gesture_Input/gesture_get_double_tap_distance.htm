gesture_get_double_tap_distance Cette fonction est utilisée pour obtenir la distance à l'intérieur de laquelle vous devez toucher / cliquer à nouveau sur l'écran après un simple tapotement afin de déclencher un geste Double Tap. La distance est mesurée en pouces et a une valeur par défaut de 0,1. Syntaxe:gesture_get_double_tap_distance(); Retour:Real (inches) Exemple:if gesture_get_double_tap_distance() != 0.1     {     gesture_double_tap_distance(0.1);     } Le code ci-dessus vérifie si la distance de double pression pour les gestes est définie sur 0,1 pouce et si ce n'est pas le cas, il la définit sur cette valeur.
