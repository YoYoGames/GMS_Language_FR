gesture_flick_speed Cette fonction est utilisée pour définir la vitesse requise pour qu'un événement Flick Gesture soit déclenché lorsqu'un toucher ou un clic est relâché. La vitesse est mesurée en pouces par seconde et a une valeur par défaut de 2,0. Syntaxe:gesture_flick_speed(speed); Argument Description speed La vitesse (en pouces par seconde) à définir pour la détection d'événement de geste de mouvement. Retour:N/A Exemple:if gesture_get_flick_speed() != 2     {     gesture_flick_speed(2);     } Le code ci-dessus vérifie si la vitesse de défilement pour les gestes est définie sur 2 pouces par seconde et si ce n'est pas le cas, il la définit sur cette valeur.
