geste_double_tap_time Cette fonction est utilisée pour définir le temps qu'il faut entre deux touchers/clics pour déclencher un événement Geste de Double Tap. Le temps est mesuré en secondes et sa valeur par défaut est de 0,16. par défaut est de 0,16. Syntaxe : geste_double_tap_time(time) ; Arguments Description temps Le temps (en secondes) à définir pour la détection des événements liés au geste du double appui. Les retours : N/A Exemple : if gesture_get_double_tap_time() !.= 0.16 { gesture_get_double_tap_time(0.16) ; } Le code ci-dessus vérifie si la durée de la double pression pour les gestes est fixée à 0,16 seconde et, si ce n'est pas le cas, il la fixe à cette valeur.
