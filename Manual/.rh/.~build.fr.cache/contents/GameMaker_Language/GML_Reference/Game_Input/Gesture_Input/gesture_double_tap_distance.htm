distance_double_tap_giratoire Cette fonction permet de définir la distance à laquelle vous devez toucher/cliquer à nouveau sur l'écran après un simple tapotement afin de déclencher un geste de double tapotement. La distance est mesurée en pouces et sa valeur par défaut est de 0,1. Syntaxe : gesture_double_tap_distance(distance) ; Arguments Description distance La distance (en pouces) à définir pour la détection des événements liés au geste du double tapotement. Les retours : N/A Exemple : if gesture_get_double_tap_distance() !.= 0.1 { gesture_get_double_tap_distance(0.1) ; } Le code ci-dessus vérifie si la distance de double tapotement pour les gestes est définie sur 0,1 pouce et si ce n'est pas le cas, il la définit sur cette valeur.
