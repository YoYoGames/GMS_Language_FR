temps de rotation du geste Cette fonction est utilisée pour définir le temps pendant lequel une paire de touches doit tourner dans une direction cohérente pour qu'un geste de démarrage de rotation soit déclenché. Le temps est Le temps est mesuré en secondes et sa valeur par défaut est de 0,16s. IMPORTANT ! Actuellement, pour un jeu à 60 images par seconde, vous ne pouvez régler ce paramètre que sur un maximum d'une seconde, sinon aucune rotation ne sera détectée. Cette valeur augmentera pour un framrate inférieur, par exemple un jeu à 30fps peut avoir un temps maximum de 2 secondes. Syntaxe : gesture_rotate_time() ; Les retours : Réel (pouces) Exemple : if gesture_get_rotate_time() !.= 0.1 { gesture_rotate_time(0.1) ; } Le code ci-dessus vérifie si le temps de rotation des gestes est réglé sur 0,1 seconde et, si ce n'est pas le cas, il le règle sur cette valeur.
