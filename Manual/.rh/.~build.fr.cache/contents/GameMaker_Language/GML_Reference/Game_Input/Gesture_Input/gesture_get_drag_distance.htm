distance_get_drag Cette fonction est utilisée pour obtenir la distance nécessaire au déclenchement d'un événement de geste de traînage par le mouvement d'une touche ou d'un clic. La distance est mesurée en pouces et a une valeur par défaut de 0,1. Syntaxe : gesture_get_drag_distance() ; Les retours : Réel (pouces) Exemple : if gesture_get_drag_distance() !.= 0.1 { gesture_drag_distance(0.1) ; } Le code ci-dessus vérifie si la distance de déplacement pour les gestes est définie sur 0,1 pouce et si ce n'est pas le cas, il la définit sur cette valeur.
