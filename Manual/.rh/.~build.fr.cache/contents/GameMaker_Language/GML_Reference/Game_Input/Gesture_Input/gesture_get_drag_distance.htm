gesture_get_drag_distance Cette fonction est utilisée pour obtenir la distance nécessaire pour qu'un événement de geste de glissement soit déclenché par le mouvement d'un toucher ou d'un clic. La distance est mesurée en pouces et a une valeur par défaut de 0,1. Syntaxe:gesture_get_drag_distance(); Retour:Real (inches) Exemple:if gesture_get_drag_distance() != 0.1     {     gesture_drag_distance(0.1);     } Le code ci-dessus vérifie si la distance de glissement pour les gestes est définie sur 0,1 pouce et si ce n'est pas le cas, il la définit sur cette valeur.
