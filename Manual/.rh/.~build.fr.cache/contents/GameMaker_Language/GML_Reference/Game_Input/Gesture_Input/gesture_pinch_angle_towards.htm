gesture_pinch_angle_towards Cette fonction est utilisée pour définir l'angle dans lequel une touche doit se déplacer vers une autre touche avant qu'un geste de pincement puisse être déclenché. L'angle est mesuré en degrés et a une valeur par défaut de 45 °. Syntaxe:gesture_pinch_angle_towards(); Retour:Real (inches) Exemple:if gesture_get_pinch_angle_towards() != 45     {     gesture_pinch_angle_towards(45);     } Le code ci-dessus vérifie si l'angle de pincement pour les gestes est défini sur 45 ° et si ce n'est pas le cas, il le définit sur cette valeur.
