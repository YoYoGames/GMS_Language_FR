gesture_get_double_tap_time Cette fonction est utilisée pour obtenir le temps qu'il faut entre deux touches / clics pour déclencher un événement Double Tap Gesture. Le temps est mesuré en secondes et a une valeur par défaut de 0,16. Syntaxe:gesture_get_double_tap_time(); Retour:Real (seconds) Exemple:if gesture_get_double_tap_time() != 0.16     {     gesture_get_double_tap_time(0.16);     } Le code ci-dessus vérifie si le temps de double tap pour les gestes est défini sur 0,16 seconde et s'il ne l'est pas, il le définit sur cette valeur.
