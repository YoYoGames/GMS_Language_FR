temps_de_glissement Cette fonction est utilisée pour définir le temps nécessaire pour qu'un événement de geste de glissement soit déclenché par un toucher ou un clic. Ce délai affecte également le déclenchement de l'événement Tap. car un toucher/clic et un relâchement avant la fin de ce délai seront considérés comme une tape. Le temps est mesuré en secondes et sa valeur par défaut est de 0,16. Syntaxe : gesture_drag_time(time) ; Arguments Description temps Le temps (en secondes) à définir pour la détection des événements liés au geste de glisser. Les retours : N/A Exemple : if gesture_get_drag_time() !.= 0.16 { gesture_drag_time(0.16) ; } Le code ci-dessus vérifie si le temps de déplacement pour les gestes est fixé à 0,16 seconde et, si ce n'est pas le cas, il le fixe à cette valeur.
