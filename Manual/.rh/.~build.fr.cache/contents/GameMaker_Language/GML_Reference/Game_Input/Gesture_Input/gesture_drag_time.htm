gesture_drag_time Cette fonction est utilisée pour définir le temps nécessaire pour qu'un événement de mouvement de glissement soit déclenché par une pression ou un clic. Ce temps affectera également la façon dont l' événement Tap est déclenché par un toucher / clic et le relâchement avant la fin de ce temps sera considéré comme un Tap. Le temps est mesuré en secondes et a une valeur par défaut de 0,16. Syntaxe:gesture_drag_time(time); Argument Description time La durée (en secondes) à définir pour la détection d'événement de mouvement de glissement. Retour:N/A Exemple:if gesture_get_drag_time() != 0.16     {     gesture_drag_time(0.16);     } Le code ci-dessus vérifie si le temps de glissement pour les gestes est défini sur 0,16 seconde et si ce n'est pas le cas, il le définit sur cette valeur.
