geste_get_rotate_time Cette fonction est utilisée pour obtenir l'angle qu'une paire de touchers doit dépasser afin de déclencher un geste de démarrage de rotation. L'angle est mesuré en degrés et a une valeur par défaut de 5°. valeur par défaut de 5°. Syntaxe : gesture_get_rotate_angle() ; Les retours : Réel (pouces) Exemple : if gesture_get_rotate_angle() != 5 { gesture_rotate_angle(5) ; } Le code ci-dessus vérifie si l'angle de rotation pour les gestes est fixé à 5° et si ce n'est pas le cas, il le fixe à cette valeur.
