gesture_get_drag_time Cette fonction est utilisée pour obtenir le temps nécessaire pour qu'un événement de mouvement de démarrage par glisser soit déclenché par une pression ou un clic. Ce temps affectera également la façon dont l' événement Tap est déclenché par un toucher / clic et le relâchement avant la fin de ce temps sera considéré comme un Tap. Le temps est mesuré en secondes et a une valeur par défaut de 0,16. Syntaxe:gesture_get_drag_time(); Retour:Real (time in seconds) Exemple:if gesture_get_drag_time() != 0.16     {     gesture_drag_time(0.16);     } Le code ci-dessus vérifie si le temps de glissement pour les gestes est défini sur 0,16 seconde et s'il ne l'est pas, il le définit sur cette valeur.
