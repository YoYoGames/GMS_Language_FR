Heure du geste_get_drag Cette fonction est utilisée pour obtenir le temps qu'il faut pour qu'un événement Geste de démarrage par glissement soit déclenché par un toucher ou un clic. Ce temps affecte également le déclenchement de l'événement Tap. l'événement Tap est déclenché, car un toucher/clic et un relâchement avant la fin de ce temps seront considérés comme un Tap. Le temps est mesuré en secondes et sa valeur par défaut est de 0,16. Syntaxe : gesture_get_drag_time() ; Les retours : Real (temps en secondes) Exemple : if gesture_get_drag_time() !.= 0.16 { gesture_drag_time(0.16) ; } Le code ci-dessus vérifie si le temps de déplacement pour les gestes est fixé à 0,16 seconde et, si ce n'est pas le cas, il le fixe à cette valeur.
