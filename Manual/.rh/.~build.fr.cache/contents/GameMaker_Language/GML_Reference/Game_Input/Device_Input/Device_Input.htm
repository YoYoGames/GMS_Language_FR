Entrée de périphérique Lorsque votre jeu cible les plates-formes qui permettent le multi-touch, vous devriez utiliser ces fonctions. Ils fonctionnent de la même manière qu'un PC fonctionnerait s'il avait plusieurs souris attachées et les constantes qui sont vérifiées sont les mêmes que celles utilisées pour les fonctions de souris standard: Constant Description mb_left Le bouton gauche de la souris mb_middle Le bouton central de la souris (cela peut ne pas être valable pour toutes les plates-formes cibles) mb_right Le bouton droit de la souris mb_none Pas de bouton de la souris mb_any N'importe lequel des boutons de la souris Les fonctions peuvent toutes prendre une valeur «tactile» qui correspond à une pression sur l'écran de l'appareil, cependant le nombre maximum de touches pouvant être détectées dépendra beaucoup de l'appareil et du système d'exploitation en cours d'exécution. Par exemple, certains Android bas de gamme peuvent ne détecter que 2 ou 3 touches en même temps, tandis qu'un appareil UWP peut en détecter jusqu'à 10. Notez également que sur les appareils à écran tactile, un double tap est renvoyé comme l'équivalent d'un événement du bouton droit de la souris. Ainsi, si vous devez détecter plusieurs tapotements en succession rapide, par exemple, vous devrez vérifier les deux mb_left et mb_right. Cette fonctionnalité peut être activée ou désactivée à l'aide de device_mouse_dbclick_enable(). Vous trouverez ci-dessous une liste de toutes les fonctions de contrôle spécifiques à l'appareil: device_mouse_check_buttondevice_mouse_check_button_presseddevice_mouse_check_button_releaseddevice_mouse_dbclick_enabledevice_mouse_xdevice_mouse_ydevice_mouse_raw_xdevice_mouse_raw_ydevice_mouse_x_to_guidevice_mouse_y_to_guidevice_is_keypad_opendevice_get_tilt_xdevice_get_tilt_ydevice_get_tilt_z Veuillez noter que sur les appareils mobiles ( Android, iOS et Windows Phone ), vous pouvez utiliser le bouton Retour à tout moment, car il est mappé sur le retour arrière du clavier qui est la constante du clavier GameMaker Studio 2 vk_backspace, vous pouvez donc vérifier cette clé et effectuer des actions sur ces appareils comme si le bouton de retour avait été utilisé. Notez également que la cible HTML5 n'acceptera correctement l'entrée que pour "périphérique 0" lors de l'utilisation de ces fonctions.
