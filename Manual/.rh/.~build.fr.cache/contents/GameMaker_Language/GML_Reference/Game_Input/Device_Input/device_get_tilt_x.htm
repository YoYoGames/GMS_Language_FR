device_get_tilt_x Cette fonction renvoie une valeur comprise entre -1 et 1 en fonction de l'angle "d'inclinaison" de l'appareil. La corrélation réelle entre les degrés d'inclinaison et la valeur renvoyée dépend de l'appareil et du système d'exploitation qu'il utilise, mais généralement une valeur de 1 ou -1 équivaut à +/- 90 °. L'image ci-dessous montre comment chacune des fonctions disponibles se rapporte à l'appareil: Syntaxe:device_get_tilt_x() Retour:Real Exemple:if display_get_orientation() = display_landscape    {    x += sign(device_get_tilt_y());    } else    {    x += sign(device_get_tilt_x());    } Le code ci-dessus vérifie l'orientation de l'affichage, puis utilise la valeur d'inclinaison correspondante pour déplacer le lecteur le long de l'axe x.
