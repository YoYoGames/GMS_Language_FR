périphérique_souris_x Cette fonction renvoie la position x d'une touche sur le périphérique. Si vous exécutez cette fonction sur un module HTML5 ou PC et Mac, cette valeur est mise à jour en permanence, tant que le périphérique (généralement une souris) est branché, mais pour les périphériques mobiles, cette valeur ne sera mise à jour que lorsque l'écran sera touché. Cependant, pour les appareils mobiles, cette valeur ne sera mise à jour que lorsque l'écran est touché. Notez que le nombre maximum de touchers pouvant être détectés dépend beaucoup de l'appareil utilisé et du système d'exploitation qu'il exécute. Notez également que sur HTML5, seul "device 0" fonctionnera correctement. correctement et que les autres appareils peuvent donner des valeurs erronées. Syntaxe : device_mouse_x(device) ; Arguments Description dispositif Le dispositif (de 0 à n) qui est contrôlé. Les retours : Real Exemple : if device_mouse_check_button(0, mb_left) { if device_mouse_x(0) > x-32 && device_mouse_x(0) < x+32 { press = true ; } else { pressed = false ; } } Le code ci-dessus vérifie si le dispositif est pressé et si c'est le cas, il interroge la position x du dispositif pour voir si elle est dans les paramètres. Si c'est le cas, la variable "pressed" prend la valeur true, sinon, elle prend la valeur false.
