dispositif_est_clavier_ouvert Cette fonction vérifie la présence d'un clavier sur l'appareil et, s'il y en a un de disponible, elle renvoie true (vrai), sinon elle renvoie false (faux). Veuillez noter que cette fonction est principalement destinée aux appareils Android. Pour les utilisateurs du Sony Xperia Play, il existe une configuration de la carte des boutons et des touches dans GameMaker Studio 2. Vous pouvez donc utiliser les constantes clavier vk_up, vk_down, vk_left, vk_right pour les touches du joypad et Triangle est ord("T"), Carré est ord("S "), Cercle est vk_alt + vk_backspace tandis que le bouton arrière est simplement vk_backspace, Cross est vk_space, Select est vk_return et Start est vk_rshift, le Trigger L est ord("L") et le Trigger R est ord("R "). Syntaxe : dispositif_est_clavier_ouvert Les retours : Booléen Exemple : if device_is_keypad_open() { global.Setting = 2 ; } else { global.Setting = 1 ; } Le code ci-dessus vérifie la présence d'un clavier et modifie une variable globale en fonction de la valeur renvoyée.
