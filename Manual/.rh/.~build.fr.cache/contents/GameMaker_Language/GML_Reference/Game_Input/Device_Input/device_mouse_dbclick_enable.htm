dispositif_mouse_dbclick_enable Cette fonction peut être utilisée pour configurer le dispositif afin qu'il détecte un double tapotement du mb_left (bouton gauche de la souris) comme un tapotement du mb_right (bouton droit de la souris) ou non. Par défaut, cette fonction est réglée sur true (vrai), ce qui signifie que chaque fois que l'utilisateur touche l'écran du périphérique deux fois rapidement et consécutivement, la valeur de retour est la même que si le bouton droit de la souris avait été cliqué. Lorsque cette option est activée, le premier tapotement sera détecté comme mb_left, et le second comme mb_right. que le code que vous utilisez en tienne compte. Syntaxe : device_mouse_dbclick_enable(bool) ; Arguments Description bool Active(true) ou désactive(false) la détection du double-clic. Les retours : Booléen Exemple : if os_type == os_windows || os_type == os_mac { device_mouse_dbclick_enable(false) ; } Le code ci-dessus vérifie si l'appareil qui exécute le jeu est un PC Windows ou un Mac et s'il s'agit de l'un ou l'autre, il désactive la fonction de double-clic.
