dispositif_mouse_raw_x Cette fonction renvoie la position x brute d'un toucher sur le périphérique. Cela signifie qu'elle renvoie la définition réelle de l'appareil pour la position x qui est touchée, et non celle de GameMaker Studio 2. comme la position de la vue et la mise à l'échelle. Notez que le nombre maximum de touchers pouvant être détectés dépendra beaucoup de l'appareil utilisé et du système d'exploitation qu'il exécute. REMARQUE: Cette fonction est très dépendante du périphérique et vous devez d'abord expérimenter avec le module cible et le périphérique souhaités pour voir ce qui est retourné exactement. Syntaxe : device_mouse_raw_x(device) ; Arguments Description dispositif Le dispositif (de 0 à n) qui est contrôlé. Les retours : Real Exemple : if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left) { x_av = mean(device_mouse_raw_x(0), device_mouse_raw_x(1)) ; y_av = mean(device_mouse_raw_y(0), device_mouse_raw_y(1)) ; } Le code ci-dessus vérifie si les dispositifs 1 et 2 sont pressés, et si c'est le cas, il calcule la position moyenne des coordonnées x/y entre chaque point de pression.
