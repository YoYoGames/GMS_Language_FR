highscore_value Avec cette fonction, vous pouvez récupérer la valeur de score qui a été stockée dans la liste des meilleurs scores à la position donnée. Si aucun score n'a été entré, la fonction renverra 0. Syntaxe:highscore_value(place); Argument Description place La place sur la table (1-10). Retour:N/A Exemple:var i = 9; repeat(10)     {     scr[i] = highscore_value(i + 1);     i -= 1;     } Le code ci-dessus parcourra la liste des meilleurs scores et stockera tous les scores dans un tableau.
