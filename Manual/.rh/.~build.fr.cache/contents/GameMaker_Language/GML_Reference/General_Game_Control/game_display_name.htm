nom_affichage_du_jeu Cette variable en lecture seule renvoie le nom d'affichage de votre jeu pour la plate-forme cible, tel que défini dans les options du jeu. Syntaxe : nom_affichage_du_jeu Les retours : Chaîne de caractères Exemple : var name = game_display_name ; var = string(GM_version) ; draw_text(32, 32, name + " :" + ver) ; Le code ci-dessus récupère le nom d'affichage et le numéro de version du jeu et les affiche.
