game_display_name Cette variable en lecture seule renvoie le nom d'affichage de votre jeu pour la plate-forme cible, tel que défini dans les options de jeu. Syntaxe:game_display_name Retour:String Exemple:var name = game_display_name; var ver = string(GM_version); draw_text(32, 32, name + ":" + ver); Le code ci-dessus obtient le nom d'affichage et le numéro de version du jeu et les dessine.
