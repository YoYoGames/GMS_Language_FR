game_save_id Cette variable en lecture seule renvoie l'ID du chemin complet du répertoire utilisé par votre jeu pour sauvegarder les fichiers. Ce répertoire peut ou non être visible par d'autres applications, selon la plate-forme, et sur la cible HTML5, il retournera une chaîne vide. NOTE: Cette variable est purement informative et ne doit pas être utilisée pour obtenir un chemin d'accès à un fichier ou quoi que ce soit d'autre de similaire, car cela ne fonctionnera pas correctement sur toutes les plateformes. Syntaxe : game_save_id Les retours : Chaîne de caractères Exemple : save_dir = game_save_id ; Cela permettra de stocker le répertoire de sauvegarde des fichiers dans une variable pour une utilisation ultérieure.
