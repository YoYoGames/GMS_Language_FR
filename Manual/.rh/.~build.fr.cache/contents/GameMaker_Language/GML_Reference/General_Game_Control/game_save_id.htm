game_save_id Cette variable en lecture seule renverra l'ID de chemin complet du répertoire utilisé par votre jeu pour enregistrer les fichiers. Ce répertoire peut ou non être visible par d'autres applications, selon la plate-forme, et sur la cible HTML5, il renverra une chaîne vide. REMARQUE: Cette variable est purement informative et ne doit pas être utilisée pour obtenir un chemin de fichier ou autre chose similaire, car elle ne fonctionnera pas correctement sur toutes les plates-formes. Syntaxe:game_save_id Retour:String Exemple:save_dir = game_save_id; Cela stockera le répertoire pour enregistrer les fichiers dans une variable pour une utilisation future.
