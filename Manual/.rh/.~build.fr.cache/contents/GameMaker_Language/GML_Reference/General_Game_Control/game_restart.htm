game_restart Avec cette fonction, vous pouvez redémarrer le jeu. C'est essentiellement la même chose que d'exécuter le jeu pour la première fois et donc l' événement de début de jeu sera déclenché, ainsi que l' événement de fin de jeu. Cependant, il convient de noter que certaines choses ne seront pas réinitialisées lorsque cette fonction sera appelée: Les variables globales ne seront pas réinitialisées à moins qu'elles ne soient explicitement codées comme telles - par exemple, la variable globale intégrée score ne démarrera pas à zéro après un redémarrage du jeu s'il a déjà été modifié dans le jeu. L'état du GPU ne sera pas modifié (donc si vous avez défini la couleur de dessin ou l'alpha, par exemple, il restera à la valeur modifiée). La vitesse de jeu restera celle que vous avez définie dans votre code de jeu (si vous l'avez modifiée, ce changement sera perpétué). Tout élément du navigateur d'actifs qui a été modifié au moment de l'exécution dans le jeu - par exemple si vous modifiez l'origine d'une ressource de sprite ou changez la position d'une ressource de chemin - ne sera pas réinitialisé. Les ressources dynamiques telles que les tampons, les surfaces, les structures de données ou les sprites importés ne seront pas non plus nettoyées ou supprimées (bien que vous puissiez perdre des références à celles-ci, faites donc attention lorsque vous utilisez cette fonction pour utiliser des références globales pour la ressource dynamique, ou pour nettoyer avant que la fonction ne soit appelée). Syntaxe:game_restart(); Retour:N/A Exemple:if keyboard_check_pressed(ord("R")) game_restart(); Cela redémarrerait le jeu lorsque le joueur appuie sur la touche «R».
