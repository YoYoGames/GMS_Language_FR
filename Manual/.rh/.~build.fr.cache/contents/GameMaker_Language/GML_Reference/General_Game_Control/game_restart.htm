redémarrage du jeu Cette fonction vous permet de redémarrer le jeu. C'est essentiellement la même chose que de lancer le jeu pour la première fois et donc l'événement de début de jeu sera déclenché, ainsi que l'événement de fin de jeu. Cependant, il convient de noter que certaines choses ne seront pas réinitialisées lorsque cette fonction est appelée : Les variables globales ne seront pas réinitialisées sauf si elles sont explicitement codées comme telles - par exemple, la variable globale intégrée score ne repartira pas à zéro après un redémarrage du jeu si elle a déjà été modifiée dans le jeu. L'état du GPU ne sera pas modifié (ainsi, si vous avez défini la couleur de dessin ou l'alpha, par exemple, il restera à la valeur modifiée). La vitesse du jeu restera celle que vous avez définie dans votre code de jeu (si vous l'avez modifiée, ce changement sera perpétué). Toute ressource du navigateur d'actifs qui a été modifiée au moment de l'exécution dans le jeu - par exemple si vous changez l'origine d'une ressource sprite ou si vous déplacez la position d'une ressource chemin - ne sera pas réinitialisée. Les ressources dynamiques telles que les tampons, les surfaces, les structures de données ou les sprites importés ne seront pas non plus nettoyées ou supprimées (bien que vous puissiez perdre les références à celles-ci, donc faites attention lorsque vous utilisez cette fonction à utiliser des références globales pour la ressource dynamique, ou de les nettoyer avant l'appel de la fonction). Syntaxe : game_restart() ; Les retours : N/A Exemple : si keyboard_check_pressed(ord("R")) game_restart() ; Cela permettrait de redémarrer le jeu lorsque le joueur appuie sur la touche "R".
