nom_highscore Cette fonction permet de récupérer la chaîne de noms qui a été enregistrée dans la liste des meilleurs scores à la position donnée. Si aucun nom n'a été saisi, la chaîne "Inconnu" sera renvoyée. Syntaxe : highscore_name(place) ; Arguments Description place La place sur la table (1-10). Les retours : Chaîne de caractères Exemple : var i = 9 ; repeat(10) { name[i] = highscore_name(i + 1) ; i -= 1 ; } Le code ci-dessus va boucler la liste des meilleurs scores et stocker tous les noms dans un tableau.
