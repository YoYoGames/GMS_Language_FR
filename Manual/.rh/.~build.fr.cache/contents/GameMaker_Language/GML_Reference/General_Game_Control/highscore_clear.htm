valeur_highscore Cette fonction vous permet d'effacer la liste des meilleurs scores globaux internes, en réinitialisant tous les noms à la valeur par défaut "Inconnu" et tous les scores à la valeur par défaut 0. Syntaxe : highscore_clear() ; Les retours : N/A Exemple : highscore_clear() ; Cela effacerait la liste des meilleurs scores.
