gc_enable Cette fonction permet d'activer ou de désactiver le ramasse-miettes. L'appel de la fonction avec true comme argument l'active et l'utilisation de false le désactive (non recommandé). Il est activé par défaut. Syntaxe : gc_enable(enable) ; Arguments Description activer activer(true) ou désactiver(false) le garbage collector. Les retours : N/A Exemple : if (global.debug == true) { gc_enable(false) ; } Le code ci-dessus désactive la collecte des déchets si la variable globale donnée est vraie.
