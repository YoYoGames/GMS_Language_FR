gc_is_enabled Avec cette fonction, vous pouvez vérifier si le garbage collector est activé ou non. La fonction retournera true s'il est activé ou false sinon. Syntaxe : gc_is_enabled() ; Les retours : Bool Exemple : if (!gc_is_enabled()) { gc_enable(true) ; } Le code ci-dessus vérifie si le garbage collector est activé et s'il ne l'est pas, il l'active.
