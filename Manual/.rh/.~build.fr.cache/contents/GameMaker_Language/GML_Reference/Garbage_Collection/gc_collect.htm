gc_collect Avec cette fonction, vous pouvez déclencher le garbage collector, le forçant à s'exécuter à la fin de la trame (step) en cours. Il convient de noter que le garbage collector n'a pas besoin d'être actif pour que cela fonctionne. L'appel de cette fonction après la désactivation du le ramasseur d'ordures (à l'aide de la fonction gc_enable()) activera le ramasseur d'ordures pour une trame dans laquelle tous les objets qui ont été marqués pour la collecte seront retirés de la mémoire avant que le ramasseur d'ordures ne soit à nouveau désactivé. avant que le ramasseur d'ordures ne soit à nouveau désactivé. Syntaxe : gc_collect() ; Les retours : N/A Exemple : if (global.debug == true && keyboard_check_pressed(vk_f1)) { gc_collect() ; } Le code ci-dessus vérifie une variable globale et la pression d'une touche et si ces éléments sont vrais, le ramassage des déchets est déclenché pour la fin de la trame (étape).
