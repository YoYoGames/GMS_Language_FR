weak_ref_create Avec cette fonction, vous pouvez vérifier la référence faible de diverses structures pour voir si elles sont toujours "vivantes" ou non. Vous fournissez un tableau de références faibles à vérifier (chaque référence faible doit avoir été créée en utilisant la fonction). doit avoir été créée à l'aide de la fonction weak_ref_create()), et la fonction retournera true si n'importe quel des structs sont encore encore référencées quelque part ou false si elles ne le sont pas et ont été collectées. Notez que si vous fournissez un tableau dont l'une des valeurs n'est pas une référence faible, la fonction retournera undefined. Notez que la fonction prend également deux arguments facultatifs, le premier vous permettant de fournir un index initial dans le tableau à vérifier, ainsi qu'un argument pour définir le nombre de positions (longueur) à vérifier à partir de cet index. Seuls les indices du tableau compris dans ces paramètres seront vérifiés. Syntaxe : weak_ref_any_alive(weak_ref, [index], [length]) ; Arguments Description faible_réf La référence faible de la structure que vous voulez vérifier. [index] FACULTATIF ! L'index dans le tableau pour commencer la vérification [longueur] FACULTATIF ! Le nombre de positions à vérifier à partir de l'index donné. Les retours : Booléen (ou indéfini) Exemple : if weak_ref_any_alive(inventory_ref_array) { instance_destroy(obj_Inventory_Control) ; } Le code ci-dessus vérifie un tableau de références faibles et si l'une d'entre elles est encore vivante, une instance est détruite.
