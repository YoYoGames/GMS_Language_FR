Répertoires de fichiers Lorsque vous travaillez avec des fichiers, il arrive que vous deviez faire référence au répertoire dans lequel les fichiers sont stockés. GameMaker Studio 2 dispose de fonctions et de variables spéciales relatives aux différents répertoires possibles dont vous pouvez avoir besoin : répertoire_existe répertoire_créer répertoire_destruction répertoire temporaire répertoire de travail répertoire du programme REMARQUE: Vous ne pouvez pas créer ou détruire des répertoires sur HTML5.
