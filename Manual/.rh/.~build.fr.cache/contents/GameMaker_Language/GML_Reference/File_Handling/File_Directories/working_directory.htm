working_directory Working_directory peut en fait renvoyer deux valeurs différentes en fonction de l'utilisation que vous en faites. Si vous écrivez un fichier sur le disque, working_directory pointe vers la zone de stockage local réservée à votre jeu sur l'appareil cible (dans Windows, il s'agit de "%LOCALAPPDATA%/gamedir/ "où le" gamedir "est le répertoire avec le nom de votre jeu). Cependant, lors de la lecture à partir du disque, working_directory peut être le stockage local ou la zone où les fichiers inclus sont stockés. REMARQUE: le working_directory variable retournera le chemin, y compris la barre oblique inverse finale. Par exemple, si vous avez une valeur par défaut *.txt fichier inclus avec votre jeu et lisez-le en mémoire en utilisant working_directory, alors ce fichier sera extrait de la zone où les fichiers inclus sont stockés. Si vous choisissez ensuite d'écrire ces informations dans un fichier texte en utilisant working_directory, cela écrira le fichier sur le stockage local de la plate-forme cible, où il sera lu la prochaine fois, sauf si vous GameMaker Studio 2 de supprimer le fichier, auquel cas working_directory pointera une fois de plus vers la zone où les fichiers inclus sont stockés. Il est à noter qu'en général, vous n'avez pas besoin de l'utiliser et qu'il suffit d'appeler le nom du fichier (plus les informations de chemin supplémentaires), par exemple ceci:file_text_open_read(working_directory + "my_file.txt"); serait mieux fait comme ceci:file_text_open_read("my_file.txt"); ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur le système de fichiers pour plus d'informations. Syntaxe:working_directory Retour:String Exemple:ini_open(working_directory + "temp_ini.ini"); Cela ouvrira un fichier ini à partir du répertoire de travail du jeu (en le créant s'il n'existe pas déjà). Il peut s'agir du stockage local ou de la zone dans laquelle se trouvent les fichiers inclus, selon que le fichier recherché existe ou non à l'un ou l'autre endroit.
