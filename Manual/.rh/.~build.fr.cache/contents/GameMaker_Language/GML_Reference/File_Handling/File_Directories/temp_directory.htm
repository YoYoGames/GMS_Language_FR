répertoire temporaire Cette option peut être utilisée pour renvoyer le répertoire temporaire créé pour votre jeu à chaque fois qu'il est exécuté (la racine ne contient pas le "\" final). Ce répertoire contiendra des fichiers et pourra être consulté pendant l'exécution du jeu, mais il sera supprimé (avec tous les fichiers qu'il contient) lorsque le jeu sera fermé. avec tous les fichiers qu'il contient) lorsque le jeu est fermé. AVERTISSEMENT ! Il se peut que cette fonction ne fonctionne pas comme vous l'attendez, car GameMaker Studio 2 est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations. Syntaxe : répertoire temporaire Les retours : Chaîne de caractères Exemple : ini_open(temp_directory + "\temp_ini.ini") ; Cela ouvrira un fichier ini dans le répertoire temporaire du jeu (en le créant s'il n'existe pas déjà).
