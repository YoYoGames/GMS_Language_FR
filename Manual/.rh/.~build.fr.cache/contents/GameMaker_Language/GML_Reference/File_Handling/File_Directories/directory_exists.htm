répertoire_existe Cette fonction renvoie true si le répertoire indiqué existe ou false s'il n'existe pas. Le nom spécifié doit inclure le chemin d'accès complet, et non un chemin relatif. Par défaut, vous ne pouvez accéder à aucun répertoire en dehors du paquet de jeu car tous les jeux sont protégés par une sandbox (voir la section sur le système de fichiers pour plus d'informations). Syntaxe : directory_exists(dname) Arguments Description dname Le nom du répertoire à rechercher. Les retours : Booléen Exemple : if directory_exists(working_directory + "Saves/") { file = file_find_first(working_directory + "Saves/*.doc", fa_readonly) ; } Elle vérifiera si le répertoire spécifié existe et, si c'est le cas, elle s'y rendra et renverra le premier fichier doc "en lecture seule" trouvé.
