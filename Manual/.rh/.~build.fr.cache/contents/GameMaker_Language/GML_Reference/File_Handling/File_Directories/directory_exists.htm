directory_exists Cette fonction retournera true si le répertoire indiqué existe ou false si ça ne fait pas. Le nom spécifié doit inclure le chemin complet, pas un chemin relatif et par défaut, vous ne pouvez accéder à aucun répertoire à partir du bundle de jeu car tous les jeux sont en bac à sable (voir la section sur le système de fichiers pour plus d'informations). Syntaxe:directory_exists(dname) Argument Description dname Le nom du répertoire à rechercher. Retour:Boolean Exemple:if directory_exists(working_directory + "Saves/")    {    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);    } Cela vérifiera si le répertoire spécifié existe, puis, si c'est le cas, allez-y et retournez le premier "lecture seule "doc fichier trouvé.
