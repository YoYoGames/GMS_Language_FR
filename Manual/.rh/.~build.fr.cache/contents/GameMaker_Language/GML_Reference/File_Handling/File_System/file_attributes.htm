Attributs du fichier Vous pouvez utiliser cette fonction pour vérifier les attributs d'un fichier donné. Vous pouvez additionner les constantes suivantes pour voir le type de fichiers que vous souhaitez : Constant Description fa_readonly Fichiers en lecture seule fa_hidden Fichiers cachés fa_sysfile Fichiers système fa_volumeid Fichiers d'identification des volumes fa_directory Annuaires fa_archive Dossiers archivés REMARQUE: Cette fonction n'est disponible que sous Windows. Syntaxe : file_attributes(fname, attr) ; Arguments Description nom de famille Le nom du fichier à vérifier. attr Les attributs à vérifier. Les retours : Booléen Exemple : if !file_attributes(file, fa_hidden) { file_delete(file) ; } Cela permet de vérifier si un fichier est caché ou non, et si ce n'est pas le cas, il est supprimé.
