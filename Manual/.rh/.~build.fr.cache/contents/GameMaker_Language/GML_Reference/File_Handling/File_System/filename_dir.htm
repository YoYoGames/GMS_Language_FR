nom de fichier_répertoire Cette fonction renvoie la partie répertoire du nom de fichier indiqué, qui est normalement la même que le chemin d'accès sauf pour la barre oblique inverse finale. Syntaxe : filename_dir(fname) ; Arguments Description fname Le fichier à utiliser. Les retours : Chaîne de caractères Exemple : dir = filename_dir("Test.ini") ; Le code ci-dessus obtient le répertoire (sous forme de chaîne) du fichier spécifié.
