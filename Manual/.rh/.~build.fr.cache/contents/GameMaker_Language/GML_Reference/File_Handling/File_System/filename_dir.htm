filename_dir Cette fonction renvoie la partie répertoire du nom de fichier indiqué, qui est normalement le même que le chemin, à l'exception de la barre oblique inverse finale. Syntaxe:filename_dir(fname); Argument Description fname Le fichier à utiliser. Retour:String Exemple:dir = filename_dir("Test.ini"); Le code ci-dessus obtient le répertoire (sous forme de chaîne) du fichier spécifié.
