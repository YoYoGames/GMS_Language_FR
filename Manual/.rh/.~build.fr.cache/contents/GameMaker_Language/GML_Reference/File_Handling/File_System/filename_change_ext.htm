changement de nom de fichier_ext Cette fonction renvoie le nom de fichier indiqué avec l'extension (y compris le point) modifiée par la nouvelle extension. En utilisant une chaîne vide comme nouvelle extension, vous pouvez supprimer complètement la partie extension. Syntaxe : filename_change_ext(fname, newext) ; Arguments Description nom de famille Le fichier à utiliser. nouveau texte La nouvelle extension à utiliser. Les retours : Chaîne de caractères Exemple : ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "") ; Le code ci-dessus récupère le nom du fichier (sous forme de chaîne) sans l'extension.
