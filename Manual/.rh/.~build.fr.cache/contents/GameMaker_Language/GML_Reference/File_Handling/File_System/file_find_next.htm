fichier_trouver_suivant Cette fonction renvoie le nom du prochain fichier qui satisfait au masque donné précédemment et aux attributs (définis par file_find_first()). Si un tel fichier n'existe pas, la chaîne vide est retournée. AVERTISSEMENT ! Il se peut que cette fonction ne fonctionne pas comme vous l'attendez, car GameMaker Studio 2 est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations. REMARQUE: Cette fonction fonctionne sur toutes les plateformes cibles C++(Windows, Mac, iOS, Android et WinPhone), MAIS les drapeaux de filtrage ne fonctionnent que sur Windows. Syntaxe : file_find_next() ; Les retours : Chaîne de caractères Exemple : file = file_find_next() ; Cela renverra le prochain fichier qui correspond aux attributs donnés précédemment.
