filename_name L'utilisation de cette fonction renvoie la partie nom du fichier indiqué, avec l'extension mais sans le chemin Syntaxe:filename_name(fname); Argument Description fname Le fichier à utiliser. Retour:String Exemple:name = filename_name(file_find_first("C:/Games/*.doc", 0)); Le code ci-dessus obtient le nom (sous forme de chaîne) du premier fichier de type "doc" trouvé dans le répertoire spécifié.
