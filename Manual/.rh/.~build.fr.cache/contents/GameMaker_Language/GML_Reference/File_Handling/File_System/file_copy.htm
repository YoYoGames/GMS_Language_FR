fichier_copie Cette fonction permet de copier le fichier spécifié en lui donnant le nouveau nom que vous avez choisi. Si vous copiez un fichier sur Android depuis votre zone de fichiers inclus (qui se trouve dans votre APK), il sera d'abord copié dans la mémoire. que la copie de très gros fichiers à partir de l'APK peut entraîner un manque de mémoire sur les appareils. Sur la cible HTML5, nous utilisons l'API de stockage local HTML5 pour la sauvegarde côté client, et les fonctions ne prennent pas en charge les opérations sur les fichiers binaires. suppose que vous copiez un fichier texte. AVERTISSEMENT ! Il se peut que cette fonction ne fonctionne pas comme vous l'attendez, car GameMaker Studio 2 est en sandbox ! Veuillez consulter la section sur les limites du système de fichiers pour plus d'informations. Syntaxe : file_copy(fname, newname) ; Arguments Description fname Le nom du fichier à copier. nouveau nom Le nom du nouveau fichier, copié. Les retours : N/A Exemple : if file_exists("level1.txt") { file_copy("level1.txt", "level2.txt") ; } Cela permet de rechercher un fichier et s'il existe, il est copié.
