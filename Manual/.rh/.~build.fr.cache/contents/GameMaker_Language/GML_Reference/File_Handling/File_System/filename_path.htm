filename_path L'utilisation de cette fonction renvoie la partie du chemin du chemin de fichier indiqué, y compris la barre oblique inverse finale. Syntaxe:filename_path(fname); Argument Description fname Le nom du fichier et le chemin à utiliser. Retour:String Exemple:path = filename_path(working_directory + "Test.ini"); Le code ci-dessus obtient le chemin (sous forme de chaîne) du fichier spécifié.
