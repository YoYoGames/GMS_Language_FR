file_rename Cette fonction renomme le fichier spécifié avec le nom spécifié. La fonction retournera true si le fichier a été renommé avec succès, ou false dans toutes autres circonstances. ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur le système de fichiers pour plus d'informations. Syntaxe:file_rename(oldname, newname); Argument Description oldname Le nom du fichier à modifier. newname Le nouveau nom à donner au fichier. Retour:Boolean Exemple:if file_exists("level1.txt")     {     file_rename("level1.txt", "level.txt");     } Cela vérifierait un fichier et s'il existe, il est renommé.
