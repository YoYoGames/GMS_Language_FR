Système de fichiers Il existe dans GameMaker Studio 2 un certain nombre de fonctions qui vous permettent de manipuler des fichiers de tous types de différentes manières : fichier_existant fichier_supprimé renommer un fichier fichier_copie fichier_trouvé_premier fichier_trouver_suivant fichier_trouvé_fermé Attributs du fichier Les fonctions suivantes peuvent être utilisées pour modifier les noms de fichiers : NOTE: Ces fonctions ne fonctionnent pas sur les fichiers réels ! Elles ne traitent que les chaînes liées aux différents éléments du chemin d'accès et doivent être utilisées conjointement avec les autres fonctions de fichier. nom_de_fichier chemin d'accès au nom de fichier nom de fichier_répertoire nom de fichier_drive nom de fichier_ext changement de nom de fichier_ext Enfin, il existe quelques fonctions spéciales pour obtenir des noms de fichiers qui peuvent être utilisés pour charger et sauvegarder des fichiers avec des filtres de type fichier. Ces fonctions ouvrent une fenêtre de l'explorateur, ce qui fait que le jeu perd le focus. Cela signifie que toute entrée dont le Cela signifie que toute entrée dont dépend le jeu peut ne pas être correctement enregistrée, et nous recommandons donc (si vous utilisez ces fonctions dans un événement clavier) d'utiliser l'événement de la touche vers le haut et non les événements enfoncé ou bas. NOTE: Ces fonctions sont Windows et macOS uniquement. AVERTISSEMENT ! Il se peut que ces fonctions ne fonctionnent pas comme vous le souhaitez, car GameMaker Studio 2 est en sandbox !
