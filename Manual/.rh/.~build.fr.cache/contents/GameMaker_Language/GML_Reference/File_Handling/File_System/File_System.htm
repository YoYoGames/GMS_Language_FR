Système de fichiers Il existe dans GameMaker Studio 2 un certain nombre de fonctions qui vous permettent de manipuler des fichiers de tous types de différentes manières: file_existsfile_deletefile_renamefile_copyfile_find_firstfile_find_nextfile_find_closefile_attributes Les fonctions suivantes peuvent être utilisées pour modifier les noms de fichiers: REMARQUE: ces fonctions ne fonctionnent pas sur les fichiers réels! Ils ne traitent que les chaînes liées aux différents éléments de chemin et doivent être utilisés en conjonction avec les autres fonctions de fichier. filename_namefilename_pathfilename_dirfilename_drivefilename_extfilename_change_ext Enfin, il existe des fonctions spéciales pour obtenir des noms de fichiers qui peuvent être utilisées pour charger et enregistrer des fichiers avec des filtres de type de fichier. Ces fonctions ouvriront une fenêtre d'explorateur et entraîneront une perte de concentration du jeu. Cela signifie que toute entrée dont le jeu dépend peut ne pas être correctement enregistrée, et en tant que tel, nous vous recommandons (si vous utilisez ces fonctions dans un événement de clavier) d'utiliser l' événement touche haut et non les événements enfoncés ou descendus. REMARQUE: ces fonctions sont uniquement disponibles pour Windows et macOS ATTENTION! Ces fonctions peuvent ne pas fonctionner comme prévu en raison du GameMaker Studio 2!
