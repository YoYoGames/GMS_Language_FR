file_exists Cette fonction retournera true si le fichier spécifié existe et false si ça ne fait pas. Notez que la fonction ne peut être utilisée que pour vérifier les fichiers locaux, mais pas les fichiers stockés sur un serveur distant. Syntaxe:file_exists(fname); Argument Description fname Le nom du fichier à rechercher. Retour:Boolean Exemple:if file_exists("level.txt")     {     file = file_text_open_read("level.txt");     } Cela vérifierait un fichier et s'il existe, il est ouvert pour la lecture.
