filename_ext Cette fonction renvoie la partie d'extension du nom de fichier indiqué, y compris le point de début. Syntaxe:filename_ext(fname); Argument Description fname Le fichier à utiliser. Retour:String Exemple:ext = filename_ext(file_find_first("*.*", 0)); Le code ci-dessus obtient l'extension (sous forme de chaîne) du fichier spécifié.
