nom de fichier_ext Cette fonction renvoie la partie extension du nom de fichier indiqué, y compris le point de tête. Syntaxe : filename_ext(fname) ; Arguments Description nom de famille Le fichier à utiliser. Les retours : Chaîne de caractères Exemple : ext = filename_ext(file_find_first("*.*", 0)) ; Le code ci-dessus obtient l'extension (sous forme de chaîne) du fichier spécifié.
