ini_read_real Vous pouvez utiliser cette fonction pour lire un nombre à partir d'un fichier de données ini. Les fichiers ini sont divisés en sections, puis chaque section est ensuite divisée en paires clé-valeur. Ainsi, un fichier ini typique ressemblerait à quelque chose comme ceci : Syntaxe : ini_read_real(section, clé, défaut) ; Arguments Description section La section du .ini à lire. clé La clé dans la section pertinente du .ini à lire. par défaut La valeur à retourner si une valeur n'est pas trouvée à l'endroit défini (ou si le fichier .ini n'existe pas). Doit être un nombre réel. Les retours : Real Exemple : ini_open("savedata.ini") ; score = ini_read_real("save1", "score", 0 ) ; ini_close() ; Cela ouvrira le fichier "savedata.ini" et fixera le score à la valeur indiquée dans "save1" > "score", puis refermera le fichier .ini. S'il n'y a pas de valeur sous "save1" > "score", ou s'il n'y a pas de fichier "savedata.ini", le score sera mis à 0. n'est présent, le score sera fixé à 0.
