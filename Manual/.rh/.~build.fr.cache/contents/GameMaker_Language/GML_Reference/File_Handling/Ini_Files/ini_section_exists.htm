ini_section_existe Cette fonction vérifie si une section existe dans l'ini actuellement ouvert et renvoie true si c'est le cas ou false sinon. Ce n'est pas une vérification nécessaire pour éviter les erreurs car, lorsqu'une section n'existe pas, la lecture d'une section inexistante renverra simplement une valeur par défaut. inexistante renverra simplement une valeur par défaut, mais cela peut être utile pour voir si un fichier ini a enregistré des données spécifiques. Syntaxe : ini_section_exists(section) ; Arguments Description section La section à vérifier dans le fichier .ini ouvert. Les retours : Booléen Exemple : ini_open("savedata.ini") ; if !ini_section_exists("save1") { global.savegame = "1" ; } ini_close() ; Cela donnera à la variable global.savegame la valeur "1" si "save1" n'est PAS trouvé. Avec une boucle for, cela pourrait être utilisé comme un moyen de base pour déterminer combien de sauvegardes il y a pour un jeu (par exemple, créer un système pour plusieurs sauvegardes simultanées pour un jeu). simultanées pour un jeu).
