ini_section_delete Avec cette fonction, vous pouvez supprimer une section entière d'un fichier ini, ce qui supprimera également toutes les paires clé-valeur qui lui sont associées. Syntaxe:ini_section_delete(section); Argument Description section La section à supprimer. Retour:N/A Exemple:ini_open("savedata.ini"); ini_write_real("save1", "Score", score ); ini_section_delete("save1"); ini_close(); Cet exemple ouvrira "savedata.ini" et écrira une valeur dans "save1"> "Score".
