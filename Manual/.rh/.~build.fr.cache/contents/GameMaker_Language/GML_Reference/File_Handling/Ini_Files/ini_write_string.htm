ini_write_string Vous pouvez utiliser cette fonction pour écrire une chaîne (texte) dans un fichier de données ini. Les fichiers ini sont divisés en sections, puis chaque section est ensuite divisée en paires clé-valeur. Ainsi, un fichier ini typique ressemblerait à quelque chose comme ceci : Il convient de noter que l'utilisation de guillemets et de caractères d'échappement lors de l'écriture de chaînes ini peut poser des problèmes lors de la relecture ultérieure des données, celles-ci pouvant être tronquées prématurément. problèmes lors de la relecture des données, car elles peuvent être prématurément tronquées. Par exemple, écrire ceci posera un problème : ini_write_string("test2", "section", "hello \"Fritz\"") ; Syntaxe : ini_write_string(section, clé, valeur) ; Arguments Description section La section du fichier .ini dans laquelle il faut écrire. clé La clé dans la section pertinente du .ini pour écrire. valeur La chaîne à écrire à la destination pertinente. Les retours : N/A Exemple : ini_open("savedata.ini") ; ini_write_string("Save", "Player", global.Name) ; ini_close() ; Le code ci-dessus ouvre un fichier ini pour la lecture et l'écriture, puis écrit la chaîne stockée dans la variable globale "Name" dans la section "Save" avec la clé "Player" avant de refermer le fichier.
