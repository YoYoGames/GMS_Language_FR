ini_read_string Vous pouvez utiliser cette fonction pour lire une chaîne (texte) à partir d'un fichier de données ini. Les fichiers ini sont divisés en sections, puis chaque section est ensuite divisée en paires clé-valeur. Ainsi, un fichier ini typique ressemblerait à quelque chose comme ceci : Syntaxe : ini_read_string(section, clé, défaut) ; Arguments Description section La section du .ini à lire. clé La clé dans la section pertinente du .ini à lire. par défaut La chaîne à retourner si une chaîne n'est pas trouvée à l'endroit défini (ou si le fichier .ini n'existe pas). Il doit s'agir d'une chaîne de caractères. Les retours : Chaîne de caractères Exemple : ini_open("savedata.ini") ; global.name = ini_read_string("player", 'name', "Player1") ; ini_close() ; Cela ouvrira le fichier "savedata.ini" et définira global.name comme la chaîne de caractères sous "save1" > "Name", puis refermera le fichier .ini. S'il n'y a pas de chaîne sous "save1" > "Name", ou s'il n'y a pas de fichier "savedata.ini", global.name prendra la valeur "Player". n'est présent, global.name sera défini comme "Player1".
