Fichiers binaires Dans de rares cas, vous devrez peut-être lire des données à partir de fichiers binaires. Ces fonctions peuvent être utilisées à cette fin, bien qu'en général, nous vous recommandons d'utiliser les fonctions de tampon dédiées pour créer, enregistrer et charger des données binaires plutôt que ces fonctions. REMARQUE: ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.
