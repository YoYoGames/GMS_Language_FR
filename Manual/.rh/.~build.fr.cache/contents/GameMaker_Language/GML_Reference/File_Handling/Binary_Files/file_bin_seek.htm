fichier_bin_seek Cette fonction déplace la position de lecture actuelle dans le fichier à la position indiquée. Vous fournissez la valeur de l'ID du fichier, telle que retournée par la fonction file_bin_open()et pour ajouter un fichier, déplacez la position à la taille du fichier avant d'écrire. taille du fichier avant de l'écrire. REMARQUE: Ces fonctions ne fonctionnent pas lorsque le module cible est HTML5. Syntaxe : file_bin_seek(binfile, pos) ; Arguments Description binfile L'ID du fichier à lire. pos La position à laquelle il faut se déplacer dans le fichier. Les retours : N/A Exemple : file = file_bin_open("myfile.bin", 2) ; size = file_bin_size(file) ; file_bin_seek(file, size) ; Cela ouvrirait un fichier dans le même répertoire que le jeu, et attribuerait son index à la variable "file", puis obtiendrait la taille du fichier et fixerait la prochaine position d'écriture à cette taille.
