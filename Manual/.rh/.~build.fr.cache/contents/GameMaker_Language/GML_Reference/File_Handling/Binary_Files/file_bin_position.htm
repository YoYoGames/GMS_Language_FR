file_bin_position Cette fonction retourne la position actuelle en octets, où 0 est la première position, du fichier avec l'ID de fichier donné. Vous fournissez la valeur de l'ID de fichier, telle que renvoyée par la fonction file_bin_open(). REMARQUE: ces fonctions ne fonctionnent pas lorsque le module cible est HTML5. Syntaxe:file_bin_position(binfile); Argument Description binfile L'ID du fichier dans lequel obtenir la position. Retour:Real Exemple:pos = file_bin_position(file); Cela stockerait la position actuelle dans la variable "pos".
