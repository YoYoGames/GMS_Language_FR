file_bin_write_byte Cette fonction écrira un octet de données dans le fichier identifié par l'ID de fichier à la position d'écriture actuelle. Vous fournissez la valeur de l'ID de fichier, telle que renvoyée par la fonction file_bin_open() et l'octet de données à écrire. REMARQUE: ces fonctions ne fonctionnent pas lorsque le module cible est HTML5. Syntaxe:file_bin_write_byte(binfile, byte); Argument Description binfile L'ID du fichier dans lequel écrire. byte Les données à écrire. Retour:real Exemple:file_bin_write_byte(file, data); Cela écrirait un octet dans le fichier sélectionné.
