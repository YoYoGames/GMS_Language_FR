file_bin_open Cette fonction ouvre le fichier binaire portant le nom indiqué. Le mode indique ce qui peut être fait avec le fichier : 0 = lecture 1 = écriture 2 = lire et écrire à la fois Si le fichier n'existe pas, il est créé, et la fonction renvoie l'id du fichier qui doit être utilisé dans les autres fonctions. Vous pouvez ouvrir plusieurs fichiers en même temps (32 maximum), mais n'oubliez pas de les fermer une fois que vous en avez fini avec eux. AVERTISSEMENT ! Cette fonction peut ne pas fonctionner comme vous l'attendez, car GameMaker Studio 2 est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations. REMARQUE: Ces fonctions ne fonctionnent pas lorsque le module cible est HTML5. Syntaxe : file_bin_open(fname, mode) ; Arguments Description nom de famille Le nom du fichier à lire. mode l'indicateur de ce qui peut être fait avec le fichier. Les retours : Real Exemple : file = file_bin_open("myfile.bin", 2) ; Cela ouvrirait un fichier dans le même répertoire que le jeu, et attribuerait son index à la variable "file".
