réécriture de fichier_bin Cette fonction prend le nom du fichier tel que retourné par la fonction file_bin_open() et réécrit le fichier, en le vidant de toutes les données précédentes pour commencer à écrire depuis le début du fichier. REMARQUE: Ces fonctions ne fonctionnent pas lorsque le module cible est HTML5. Syntaxe : file_bin_rewrite(binfile) ; Arguments Description binfile L'ID du fichier à réécrire. Les retours : N/A Exemple : file = file_bin_open("myfile.bin", 2) ; file_bin_rewrite(file) ; Cela ouvrirait un fichier dans le même répertoire que le jeu, et attribuerait son index à la variable "file". Elle réécrit ensuite le fichier (en le vidant de ses données). Retour : Fichiers binaires Suivant : file_bin_close Copyright YoYo Games Ltd. 2021 Tous droits réservés
