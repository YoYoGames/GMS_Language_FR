taille_de_fichier Cette fonction renvoie la taille (en octets) d'un fichier qui a été ouvert en lecture et/ou en écriture. L'ID du fichier est la valeur renvoyée par la fonction file_bin_open(). REMARQUE: Ces fonctions ne fonctionnent pas lorsque le module cible est HTML5. Syntaxe : file_bin_size(binfile) ; Arguments Description binfile L'ID du fichier dont on veut obtenir la taille. Les retours : Real Exemple : file = file_bin_open("myfile.bin", 2) ; size = file_bin_size(file) ; file_bin_close(file) ; Cela ouvrirait un fichier dans le répertoire local du jeu, et attribuerait son index à la variable "file". Elle récupère ensuite la taille de ce fichier et le referme. Retour : Fichiers binaires Suivant : file_bin_position Copyright YoYo Games Ltd. 2021 Tous droits réservés
