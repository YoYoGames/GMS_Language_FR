file_text_write_real Avec cette fonction, vous pouvez écrire un nombre dans le fichier texte précédemment ouvert. Notez que comme la valeur à écrire peut être un nombre réel, toutes les décimales seront écrites avec un "." point comme séparateur. Si le fichier contient déjà des informations, ces informations seront effacées et la chaîne sera écrite au début du fichier, sauf si vous avez ouvert le fichier avec le file_text_open_append(). Vous pouvez également éviter cela en utilisant le file_text_readln() fonction avec le file_text_eof() pour parcourir le contenu du fichier jusqu'à la fin, puis commencer à écrire. Il est important de noter que lors de l'écriture de très grands nombres dans un fichier texte à l'aide de cette fonction, il peut être traduit dans un format standard simplifié, comme "6.6624e + 003", qui ne peut pas être lu correctement GameMaker Studio 2 Pour éviter des problèmes comme celui-ci, vous devez plutôt convertir la valeur en chaîne et utiliser la fonction file_text_write_string() au lieu. Syntaxe:file_text_write_real(fileid, val); Argument Description fileid L'identifiant du fichier à modifier. val La valeur réelle à écrire dans le fichier. Retour:Real Exemple:var file = file_text_open_write(working_directory + "hiscore.txt"); for (var i = 0; i < 10; ++i;)     {     file_text_write_real(file, scr[i]);     file_text_writeln(file);     file_text_write_string(file, scr_name[i]);     file_text_writeln(file);     } file_text_close(file); Le code ci-dessus ouvre un fichier pour l'écriture, puis effectue une boucle sur deux tableaux, écrivant chaque valeur de tableau sur une nouvelle ligne du fichier.
