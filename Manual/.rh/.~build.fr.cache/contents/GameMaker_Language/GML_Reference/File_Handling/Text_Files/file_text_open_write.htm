file_text_open_write Cette fonction ouvre le fichier texte avec le nom de fichier indiqué pour l' écriture uniquement (si le fichier n'existe pas, il est créé), retournant l' identifiant unique du fichier qui doit être stocké dans une variable car il sera utilisé pour toutes les actions ultérieures à voir avec ce fichier. Notez que si le fichier ne peut pas être créé (à cause d'un nom de fichier illégal, par exemple), la fonction renverra -1. REMARQUE: vous ne pouvez ouvrir qu'un maximum de 32 fichiers à la fois. Vous devez également toujours fermer les fichiers lorsque vous avez terminé, car cela écrit les informations et libère la mémoire associée au fichier. ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur le système de fichiers pour plus d'informations. Syntaxe:file_text_open_write(fname); Argument Description fname Le nom du fichier dans lequel écrire. Retour:Real Exemple:var file; file = file_text_open_write(working_directory + "level.txt"); file_text_write_string(file, level_data); file_text_close(file); Le code ci-dessus ouvrira le fichier "level.txt" pour l'écriture, puis écrira la chaîne stockée dans la variable "level_data" avant de finalement refermer le fichier.
