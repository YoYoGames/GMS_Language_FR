Fichiers texte Les fichiers texte sont un moyen utile de stocker de grandes quantités de données à l'extérieur de votre jeu. Contrairement aux fichiers ini, vous pouvez ouvrir jusqu'à 32 fichiers texte à la fois, et vous pouvez également y lire et y écrire de gros morceaux de données. Cependant, c'est à vous de décider comment ces données doivent être structurées car il n'y a pas de structure "section" ou "clé" comme avec *.inifichiers. Veuillez noter que pour les jeux avec des localisations, il est très important que les lettres accentuées puissent être lues à partir de fichiers externes. Cela signifie que vous devez créer le fichier txt au format UTF8 d' abord, puis l' ajouter dans GameMaker Studio 2 en tant que fichier inclus de sorte qu'il est exporté sur l' exécution du jeu et utilisé à la place du fichier format par défaut ANSI txt qui est créé par le GameMaker Studio 2 fonctionne lorsqu'aucun fichier n'est précédemment fourni. De cette façon, vous pouvez lire et écrire correctement avec tous les accents et les lettres non romaines conservés.
