fichier_texte_eoln Avec cette fonction, vous pouvez demander à GameMaker Studio 2 de vérifier dans le fichier actuellement ouvert si la ligne en cours de lecture est terminée. La fonction renvoie true si la fin de la ligne a été atteinte et false sinon. Syntaxe : file_text_eoln(fileid) ; Arguments Description fileid L'identifiant du fichier à vérifier. Les retours : Booléen Exemple : var file = file_text_open_read(working_directory + "Game_Data.txt") ; var num = 0 ; while (!file_text_eoln(file)) { score_array[num] = file_text_read_real(file) ; num++ ; } file_text_close(file) ; Le code ci-dessus ouvre un fichier en lecture puis lit les valeurs d'une seule ligne dans un tableau.
