file_text_open_append Cette fonction ouvre le fichier texte avec le nom de fichier indiqué pour l' écriture (si le fichier n'existe pas, il est créé), renvoyant l' identifiant unique du fichier qui doit être stocké dans une variable car il sera utilisé pour toutes les actions ultérieures pour faire avec ce fichier. La position dans le fichier pour l'écriture est définie sur la dernière ligne de texte que le fichier contient. Notez que si le fichier ne peut pas être créé (à cause d'un nom de fichier illégal, par exemple), la fonction renverra -1. REMARQUE: vous ne pouvez ouvrir qu'un maximum de 32 fichiers à la fois. Vous devez également toujours fermer les fichiers lorsque vous avez terminé, car cela écrit les informations et libère la mémoire associée au fichier. ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur le système de fichiers pour plus d'informations. Syntaxe:file_text_open_append(fname); Argument Description fname Le nom du fichier à ajouter. Retour:Real Exemple:file = file_text_open_append(working_directory + "save.txt"); Cela ouvrirait "save.txt" à partir du même répertoire que le jeu et stockerait l'identifiant du fichier dans la variable "file".
