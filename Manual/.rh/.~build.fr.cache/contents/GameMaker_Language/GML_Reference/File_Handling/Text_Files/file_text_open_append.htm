fichier_texte_ouvert_append Cette fonction ouvre le fichier texte avec le nom de fichier indiqué pour l'écriture (si le fichier n'existe pas, il est créé), renvoyant l'identifiant unique du fichier qui doit être stocké dans une variable car il sera utilisé pour toutes les actions ultérieures à faire avec ce fichier. à faire avec ce fichier. La position dans le fichier pour l'écriture est fixée à la dernière ligne de texte que le fichier contient. Notez que si le fichier ne peut pas être créé (à cause d'un nom de fichier illégal, par exemple), la fonction renvoie -1. REMARQUE: Vous ne pouvez avoir qu'un maximum de 32 fichiers ouverts à la fois. Vous devez également toujours fermer les fichiers lorsque vous avez terminé, car cela permet d'écrire les informations et de libérer la mémoire associée au fichier. AVERTISSEMENT ! Il se peut que cette fonction ne fonctionne pas comme vous l'attendez, car GameMaker Studio 2 est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations. Syntaxe : file_text_open_append(fname) ; Arguments Description fname Le nom du fichier à ajouter. Les retours : Real Exemple : file = file_text_open_append(working_directory + "save.txt") ; Cela ouvrirait "save.txt" dans le même répertoire que le jeu et enregistrerait l'identifiant du fichier dans la variable "file".
