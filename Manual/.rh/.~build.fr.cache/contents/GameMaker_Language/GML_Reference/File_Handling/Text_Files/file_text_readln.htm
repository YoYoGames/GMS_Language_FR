fichier_texte_lecture Avec cette fonction, vous pouvez sauter le reste de la ligne courante d'un fichier texte ouvert donné et passer au début de la suivante. La fonction renvoie également la ligne complète sous forme de chaîne de caractères, ce qui en fait un moyen facile de lire des "morceaux" complets de données pour les analyser ultérieurement. de données pour les analyser plus tard. Syntaxe : file_text_readln(fileid) ; Arguments Description fileid L'identifiant du fichier à lire. Les retours : chaîne de caractères Exemple : var file = file_text_open_read(working_directory + "hiscore.txt") ; for (var i = 0 ; i < 10 ; ++i ;) { scr[i] = file_text_read_real(file) ; file_text_readln(file) ; scr_name[i] = file_text_read_string(file) ; file_text_readln(file) ; } file_text_close(file) ; Le code ci-dessus ouvre un fichier pour le lire, puis boucle sur les lignes du fichier en lisant alternativement une valeur réelle et une chaîne de caractères dans deux tableaux différents pour une utilisation ultérieure. Le fichier est ensuite fermé lorsque la boucle est terminée. Retour : Fichiers texte Suivant : file_text_write_real Copyright YoYo Games Ltd. 2021 Tous droits réservés
