texte_fichier_eof Cette fonction renvoie vrai lorsque la fin d'un fichier texte ouvert donné a été atteinte ou faux sinon. Syntaxe : file_text_eof(fileid) ; Arguments Description fileid L'identifiant du fichier à vérifier. Les retours : Booléen Exemple : var num = 0 ; var file = file_text_open_read(working_directory + "Game_Data.txt") ; while (!file_text_eof(file)) { str[num++] = file_text_readln(file) ; } file_text_close(file) ; Le code ci-dessus ouvre un fichier pour l'écrire, puis passe en boucle les lignes de texte déjà écrites dans le fichier jusqu'à la fin, en stockant chaque ligne dans le tableau "str".
