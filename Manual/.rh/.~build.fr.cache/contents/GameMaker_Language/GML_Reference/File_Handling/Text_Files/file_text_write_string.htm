texte_fichier_écrit_chaîne Cette fonction permet d'écrire une chaîne de caractères dans un fichier texte préalablement ouvert. Si le fichier contient déjà des informations, celles-ci seront effacées et la chaîne sera écrite au début du fichier, sauf si vous avez ouvert le fichier avec la fonction file_text_open_append(). Vous pouvez également éviter cela en utilisant la fonction file_text_readln() avec la fonction fichier_texte_eof() pour parcourir en boucle le contenu du fichier jusqu'à la fin, puis commencer à écrire. Syntaxe : file_text_write_string(fileid, str) ; Arguments Description fileid L'identifiant du fichier à modifier. str La chaîne de caractères à écrire dans le fichier. Les retours : Real Exemple : var file = file_text_open_write(working_directory + "hiscore.txt") ; for (var i = 0 ; i < 10 ; ++i ;) { file_text_write_real(file, scr[i]) ; file_text_writeln(file) ; file_text_write_string(file, scr_name[i]) ; file_text_writeln(file) ; } file_text_close(file) ; Le code ci-dessus ouvre un fichier pour l'écriture et boucle ensuite sur deux tableaux, en écrivant chaque valeur de tableau sur une nouvelle ligne du fichier. Le fichier est ensuite fermé lorsque la boucle est terminée. Retour : Fichiers texte Suivant : file_text_writeln Copyright YoYo Games Ltd. 2021 Tous droits réservés
