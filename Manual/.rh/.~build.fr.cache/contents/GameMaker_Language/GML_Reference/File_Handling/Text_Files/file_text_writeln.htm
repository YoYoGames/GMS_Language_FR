file_text_writeln Cette fonction vous permet d'écrire une nouvelle ligne dans un fichier texte ouvert. De cette façon, vous pouvez sauter des lignes ou écrire des informations ligne par ligne (voir l'exemple de code ci-dessous). Syntaxe : file_text_writeln(fileid) ; Arguments Description fileid L'identifiant du fichier à modifier. Les retours : Real Exemple : var file = file_text_open_write(working_directory + "hiscore.txt") ; for (var i = 0 ; i < 10 ; ++i ;) { file_text_write_real(file, scr[i]) ; file_text_writeln(file) ; file_text_write_string(file, scr_name[i]) ; file_text_writeln(file) ; } file_text_close(file) ; Le code ci-dessus ouvre un fichier pour l'écriture et boucle ensuite sur deux tableaux, en écrivant chaque valeur de tableau sur une nouvelle ligne du fichier. Le fichier est ensuite fermé lorsque la boucle est terminée. Retour : Fichiers texte Suivant : file_text_eoln Copyright YoYo Games Ltd. 2021 Tous droits réservés
