base64_encode Cette fonction convertit une chaîne en une chaîne encodée au format base64. Il s'agit d'un schéma de codage couramment utilisé qui est souvent utilisé pour tout média devant être stocké ou transféré sur Internet sous forme de texte, et rend la sortie illisible à l'œil humain. Syntaxe:base64_encode(string) Argument Description string La chaîne à encoder. Retour:String Exemple:var str, file; str = base64_encode(game_data); file = file_text_open_write("save.txt"); file_text_write_string(file, str); file_text_close(file); Le code ci-dessus convertira la chaîne stockée dans "game_data" en une chaîne encodée en base64 qui est ensuite stockée dans un fichier texte externe.
