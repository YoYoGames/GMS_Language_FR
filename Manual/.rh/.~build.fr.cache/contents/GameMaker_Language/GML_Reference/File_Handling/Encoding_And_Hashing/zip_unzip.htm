zip_unzip Cette fonction ouvrira un fichier zip stocké et extraira son contenu dans le répertoire donné. Notez que si vous ne fournissez pas de chemin complet vers le répertoire ZIP, la racine du lecteur actuel sera utilisée, et si vous souhaitez la placer dans un chemin relatif vers le répertoire de travail du bundle de jeu, vous devez utiliser le working_directory variable dans le cadre du chemin (les chemins relatifs utilisant "." ou ".." ne fonctionneront pas comme prévu et doivent donc être évités). Notez aussi que le zip doit faire partie du bundle du jeu (c'est-à-dire: un fichier inclus ) ou avoir été téléchargé dans la zone de stockage en utilisant http_get_file (). La fonction retournera une valeur indiquant le nombre de fichiers extraits, ou elle retournera 0 ou moins si l'extraction a échoué. Syntaxe:zip_unzip(zip_file, target_directory) Argument Description zip_file Le fichier zip à ouvrir target_directory Le répertoire cible dans lequel extraire les fichiers Retour:Real Exemple:var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/"); if num <= 0    {    show_debug_message("Extraction Failed!");    } Le code ci-dessus ouvrira le fichier zip stocké dans le répertoire "téléchargements" et extraira son contenu dans le répertoire "extrait" (en créant ce répertoire s'il n'existe pas déjà), puis vérifiera que l'extraction a été correcte, montrant un message de débogage en cas d'échec.
