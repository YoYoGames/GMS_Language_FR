sha1_file En cryptographie, SHA-1 est une fonction de hachage cryptographique conçue par l'Agence de sécurité nationale des États-Unis et est employée dans plusieurs applications largement utilisées : populaire Git où elle est utilisée pour vérifier les changements de fichiers, et les protocoles TLS et SSL, PGP, SSH, S/MIME et IPsec. Cette fonction prend un fichier d'entrée et renvoie un condensé de message de 160 bits au format ASCII, unique à ce fichier, qui peut être utilisé ultérieurement pour vérifier son intégrité. NOTE: SHA-1 n'est pas complètement sécurisé et peut être cassé. Voir cette page pour plus d'informations. Syntaxe : sha1_file(nom du fichier) Arguments Description nom de fichier Le fichier pour lequel il faut générer le hachage sha1. Les retours : Chaîne de caractères Exemple : hash = sha1_file(working_directory + "game_data.ini") Le code ci-dessus va générer un hachage sha1 pour le fichier spécifié et stocker la valeur retournée dans la variable "hash".
