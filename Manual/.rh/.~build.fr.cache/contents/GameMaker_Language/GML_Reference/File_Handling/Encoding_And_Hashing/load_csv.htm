load_csv Cette fonction chargera un fichier au format CSV et le convertira en une grille DS, en renvoyant la valeur ID unique de la grille créée. Syntaxe : load_csv(nom du fichier) Arguments Description nom de fichier Le nom du fichier à ouvrir (sous forme de chaîne) Les retours : Index de la grille DS (réel) Exemple : file_grid = load_csv("spreadsheet.csv") ; var ww = ds_grid_width(file_grid) ; var hh = ds_grid_height(file_grid) ; var xx = 32 ; var yy = 32 ; for (var i = 0 ; i < ww ; i++ ;) { for (var j = 0 ; j < hh ; j++ ;) { draw_text(xx, yy, string(file_grid[# i, j])) ; yy += 32 ; } yy = 32 ; xx += 32 ; } Le code ci-dessus ouvrira le fichier CSV donné et stockera la grille DS retournée dans la variable "file_grid". Cette grille est ensuite analysée dans un couple de for lops et le contenu est affiché à l'écran. Retour : Codage des fichiers Suivant : json_encode Copyright YoYo Games Ltd. 2021 Tous droits réservés
