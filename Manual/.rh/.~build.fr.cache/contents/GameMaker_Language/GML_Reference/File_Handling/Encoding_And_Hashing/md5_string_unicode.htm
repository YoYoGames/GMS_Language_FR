md5_string_unicode En cryptographie, MD5 (Message-Digest algorithme 5) est une fonction de hachage cryptographique largement utilisée avec une valeur de hachage de 128 bits et a été utilisée dans une grande variété d'applications de sécurité. Il est également couramment utilisé pour vérifier l'intégrité des fichiers et des chaînes. Cette fonction prend une chaîne Unicode d'entrée (qui est de 16 bits pour chaque caractère) et renvoie le hachage MD5 hexadécimal de 32 caractères qui est unique à cette chaîne. De cette manière, vous pouvez générer une clé sécurisée qui peut être stockée et utilisée pour vérifier l'intégrité des informations envoyées (ou reçues depuis) un serveur externe (par exemple). REMARQUE: il existe deux formats pour l'encodage MD5, UTF-8 et unicode. Les deux sont fournis pour faciliter la communication avec différentes configurations de serveur, mais le plus courant à utiliser est l'unicode. Syntaxe:md5_string_unicode(string) Argument Description string La chaîne à hacher. Retour:String Exemple:var hash, str; str = base64_encode(game_data); hash = md5_string_unicode(str); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str); Le code ci-dessus encodera une chaîne en base64, puis générera un hachage MD5.
