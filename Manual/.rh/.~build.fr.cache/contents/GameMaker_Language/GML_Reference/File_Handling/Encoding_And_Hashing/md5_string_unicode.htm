md5_string_unicode En cryptographie, MD5 (Message-Digest algorithm 5) est une fonction de hachage cryptographique largement utilisée avec une valeur de hachage de 128 bits et a été employée dans une grande variété d'applications de sécurité. Elle est aussi couramment utilisée pour vérifier l'intégrité des fichiers et des chaînes de caractères. Cette fonction prend une chaîne unicode en entrée (qui comporte 16 bits pour chaque caractère) et renvoie le hachage MD5 hexadécimal de 32 caractères qui est unique à cette chaîne. De cette façon, vous pouvez générer une clé sécurisée qui peut être stockée et utilisée pour vérifier l'intégrité de l'information envoyée à (ou à l'étranger). des informations envoyées à (ou reçues de) un serveur externe (par exemple). REMARQUE : il existe deux formats pour l'encodage MD5, UTF-8 et unicode. Les deux sont fournis pour faciliter la communication avec différentes configurations de serveur, mais le plus courant à utiliser est unicode. REMARQUE: MD5 n'est pas complètement sécurisé et peut être cassé. Voir cette page pour plus d'informations. Syntaxe : md5_string_unicode(string) Arguments Description chaîne de caractères La chaîne à hacher. Les retours : Chaîne de caractères Exemple : var hash, str ; str = base64_encode(game_data) ; hash = md5_string_unicode(str) ; http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash) ; http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str) ; Le code ci-dessus va coder en base64 une chaîne de caractères, puis générer un hachage MD5.
