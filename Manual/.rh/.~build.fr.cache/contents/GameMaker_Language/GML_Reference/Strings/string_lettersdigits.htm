lettres et chiffres Cette fonction renvoie une copie d'une chaîne de caractères donnée dont on a supprimé toutes les lettres et tous les chiffres, ce qui signifie qu'elle peut être utilisée pour supprimer les caractères indésirables (comme "#" ou " ?"), par exemple, d'un nom de connexion ou d'un mot de passe. REMARQUE : cette fonction ne renvoie que les 26 lettres de l'alphabet anglais de A à Z (majuscules ou minuscules) et les chiffres de 0 à 9. Syntaxe : string_lettersdigits(string) ; Arguments Description chaîne de caractères La chaîne à convertir en lettres et chiffres. Les retours : Chaîne de caractères Exemple : si (string_length(password) > string_length(string_lettersdigits(password))) { draw_text(32,32, "Mot de passe invalide ! Seulement des lettres et des chiffres svp !") ; } Le code ci-dessus vérifie la longueur d'une chaîne de caractères par rapport à la longueur de la même chaîne, mais sans les symboles.
