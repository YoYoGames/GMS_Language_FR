string_supper Cette fonction permet de forcer une chaîne de caractères à ne contenir que des majuscules. REMARQUE : cette fonction ne détecte que les 26 lettres de l'alphabet anglais de A à Z. Syntaxe : string_upper(string) ; Arguments Description chaîne de caractères La chaîne de caractères à convertir en majuscules. Les retours : Chaîne de caractères Exemple : str1 = "Hello World" ; str2 = string_upper(str1) ; Le code ci-dessus donnera à la variable str2 la valeur "HELLO WORLD".
