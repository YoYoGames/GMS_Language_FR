string_upper Avec cette fonction, vous pouvez forcer une chaîne à ne contenir que des caractères majuscules. REMARQUE: Cette fonction détecte uniquement l'alphabet anglais de 26 lettres de A à Z. Syntaxe:string_upper(string); Argument Description string La chaîne à convertir en majuscules. Retour:String Exemple:str1 = "Hello World"; str2 = string_upper(str1); Le code ci-dessus définira la variable str2 à "HELLO WORLD".
