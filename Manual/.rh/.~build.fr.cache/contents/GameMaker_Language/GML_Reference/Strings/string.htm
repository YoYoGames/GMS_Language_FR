string Avec cette fonction, vous pouvez transformer n'importe quel nombre réel en une chaîne. Si le nombre réel est un entier, il sera enregistré sans décimales, sinon, il sera enregistré avec deux décimales. Si vous avez besoin de plus de décimales, utilisez la fonctionstring_format(). Notez également que l'utilisation de cette fonction sur une variable stockant un tableau, une structure de données ou une structure convertira le contenu de ces variables en une chaîne qui peut ensuite être sortie sur la console ou enregistrée dans un fichier pour le débogage. Syntaxe:string(val); Argument Description string Le nombre réel à transformer en chaîne. Retour:String Exemple:draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health)); Le code ci-dessus utilise le string fonction pour dessiner les nombres réels et les chaînes ensemble (tirage au sort n'acceptera une chaîne ou un vrai, mais pas les deux).
