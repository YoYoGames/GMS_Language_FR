chaîne de caractères Avec cette fonction, vous pouvez transformer n'importe quel nombre réel en une chaîne de caractères. Si le nombre réel est un nombre entier, il sera enregistré sans décimales, sinon, il sera enregistré avec deux décimales. Si vous avez besoin de plus de décimales, utilisez la fonction string_format(). Notez également que l'utilisation de cette fonction sur une variable stockant un tableau, une structure de données, ou une structure convertira le contenu de ces variables en une chaîne de caractères qui peut ensuite être affichée sur la console ou enregistrée dans un fichier pour le débogage. Syntaxe : string(val) ; Arguments Description chaîne de caractères Le nombre réel à transformer en chaîne de caractères. Les retours : Chaîne de caractères Exemple : draw_text(100, 100, "Score : " + string(score) + " / Santé : " + string(health)) ; Le code ci-dessus utilise la fonction string pour dessiner à la fois des nombres réels et des chaînes de caractères (draw n'accepte qu'une chaîne de caractères ou un réel, mais pas les deux).
