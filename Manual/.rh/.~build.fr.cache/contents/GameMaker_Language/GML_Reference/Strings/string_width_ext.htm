string_width_ext Cette fonction renvoie la largeur maximale (en pixels) de la chaîne de caractères en entrée, en tenant compte de la séparation des lignes et de la largeur du saut de ligne (qui est définie comme le nombre de pixels que la chaîne de caractères peut occuper avant l'insertion d'un saut de ligne). Cette fonction est très pratique pour calculer les distances entre les éléments de texte sur la base de la largeur maximale d'une chaîne de caractères répartie sur plusieurs lignes, telle qu'elle serait dessinée à l'aide de la fonction draw_text_ext() en utilisant la police actuellement définie actuellement définie. La séparation et la largeur peuvent être définies à -1 pour obtenir l'espacement par défaut. Syntaxe : string_width_ext(string, sep, w) ; Arguments Description chaîne de caractères La chaîne de caractères dont il faut mesurer la largeur. sep La distance en pixels entre les lignes de texte, comme si la chaîne était dessinée. w La largeur maximale (en pixels) de la chaîne avant un saut de ligne, comme si la chaîne était dessinée. Les retours : Real Exemple : var ww = string_width_ext(str_Story_Text[1], -1, 100) ; draw_text_ext(32, 32, str_Story_Text[1], -1, 100) ; draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100) ; Le code ci-dessus obtient la largeur de la chaîne de caractères donnée, en tenant compte de la largeur de la séparation et du saut de ligne, puis dessine deux lignes de texte en utilisant la largeur totale de la chaîne de caractères comme séparateur.
