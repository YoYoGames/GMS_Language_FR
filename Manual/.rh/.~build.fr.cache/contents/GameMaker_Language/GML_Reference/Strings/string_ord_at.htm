string_ord_at Vous pouvez utiliser cette fonction pour retourner un code de caractère spécifique à une position spécifique dans une chaîne de caractères, l'index commençant à 1 pour le premier caractère. Si aucun caractère n'est trouvé ou si la chaîne est plus courte que la valeur donnée à l'index, -1 est renvoyé. Syntaxe : string_ord_at(str, index) ; Arguments Description str La chaîne à vérifier. indice La position à partir de laquelle obtenir le code de caractère. Les retours : Real Exemple : str = "Hello World" ; char_code = string_ord_at(str, 7) ; Ceci va obtenir le code du septième caractère (où "H" compte comme le premier) dans la chaîne "str" et le stocker dans la variable "char_code".
