string_lower Avec cette fonction, vous pouvez forcer une chaîne à ne contenir que des caractères minuscules. REMARQUE: Cette fonction détecte uniquement l'alphabet anglais de 26 lettres de A à Z. Syntaxe:string_lower(string); Argument Description string La chaîne à convertir en minuscules. Retour:String Exemple:str1 = "Hello World"; str2 = string_lower(str1); Le code ci-dessus définira str2 à "bonjour le monde".
