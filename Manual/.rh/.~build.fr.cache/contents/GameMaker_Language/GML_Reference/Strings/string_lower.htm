string_lower Cette fonction permet de forcer une chaîne de caractères à ne contenir que des caractères minuscules. REMARQUE : cette fonction ne détecte que les 26 lettres de l'alphabet anglais de A à Z. Syntaxe : string_lower(string) ; Arguments Description chaîne de caractères La chaîne de caractères à convertir en minuscules. Les retours : Chaîne de caractères Exemple : str1 = "Hello World" ; str2 = string_lower(str1) ; Le code ci-dessus donnera à str2 la valeur "hello world".
