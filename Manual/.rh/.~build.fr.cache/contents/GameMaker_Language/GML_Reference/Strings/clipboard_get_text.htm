clipboard_get_text Cette fonction renverra une chaîne du texte contenu dans le presse-papiers. si aucun texte n'est stocké, il renverra une chaîne vide "". REMARQUE: cette fonction n'est valable que pour la cible Windows Syntaxe:clipboard_get_text(); Retour:String Exemple:if clipboard_has_text()    {    str = clipboard_get_text();    clipboard_set_text("");    } Le code ci-dessus vérifie le texte dans le presse-papiers et s'il en contient, il est lu comme une chaîne dans la variable "str". Enfin, le presse-papiers est effacé en le définissant sur une chaîne vide. Retour: Cordes Suivant:clipboard_set_text © Copyright YoYo Games Ltd. 2021 Tous droits réservés
