clipboard_get_text Cette fonction renvoie une chaîne de caractères du texte contenu dans le presse-papiers. Si aucun texte n'est stocké, elle renvoie une chaîne vide "". REMARQUE: Cette fonction n'est valable que pour la Windows cible. Syntaxe : clipboard_get_text() ; Les retours : Chaîne de caractères Exemple : if clipboard_has_text() { str = clipboard_get_text() ; clipboard_set_text("") ; } Le code ci-dessus vérifie la présence de texte dans le presse-papiers et s'il en contient, il est lu sous forme de chaîne dans la variable "str". Enfin, le presse-papiers est effacé en lui attribuant la valeur d'une chaîne vide. Retour : Cordes Suivant : clipboard_set_text Copyright YoYo Games Ltd. 2021 Tous droits réservés
