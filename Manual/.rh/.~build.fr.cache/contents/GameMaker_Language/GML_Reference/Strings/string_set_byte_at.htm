string_set_byte_at Cette fonction définit un octet directement dans une chaîne (basée sur le format UTF8) et renvoie une copie de la chaîne avec les modifications. REMARQUE: Cette fonction est incroyablement lente, alors considérez attentivement si elle est nécessaire et où vous l'utilisez. Syntaxe:string_set_byte_at(str, pos, byte); Argument Description str La chaîne dont vous souhaitez modifier l'octet. pos La position dans la chaîne (à partir de 1) pour changer l'octet. byte La nouvelle valeur d'octet. Retour:String Exemple:str = string_set_byte_at("hello", 2, 97); Le code ci-dessus changerait la valeur d'octet de la deuxième lettre de la chaîne, et donc définirait la variable str tenir "bonjour".
