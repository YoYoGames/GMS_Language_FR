string_set_byte_at Cette fonction définit directement un octet dans une chaîne de caractères (sur la base du format UTF8) et renvoie une copie de la chaîne avec les modifications. REMARQUE : cette fonction est incroyablement lente, aussi réfléchissez bien à sa nécessité et à l'endroit où vous l'utilisez. Syntaxe : string_set_byte_at(str, pos, byte) ; Arguments Description str La chaîne de caractères dont on veut changer l'octet. pos La position dans la chaîne (en commençant par 1) pour changer l'octet de. octet La nouvelle valeur de l'octet. Les retours : Chaîne de caractères Exemple : str = string_set_byte_at("hello", 2, 97) ; Le code ci-dessus modifierait la valeur de l'octet de la deuxième lettre de la chaîne, ce qui ferait en sorte que la variable str contienne "hallo".
