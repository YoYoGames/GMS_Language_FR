string_width Cette fonction retournera la largeur (en pixels) de la chaîne d'entrée, en tenant compte des sauts de ligne que le texte peut avoir. Il est très pratique pour calculer les distances entre les éléments de texte en fonction de la largeur totale des lettres qui composent la chaîne comme elle serait dessinée avec draw_text() en utilisant la police actuellement définie. Syntaxe:string_width(string); Argument Description string La chaîne pour mesurer la largeur de. Retour:Real Exemple:var ww = string_width(str_Name + " "); draw_text(32, 32, str_Name)); draw_text(32 + ww, 32, "has won the game!"); Le code ci-dessus obtiendra la largeur de la chaîne donnée, puis dessinera deux lignes de texte, en utilisant la largeur de chaîne retournée comme séparateur.
