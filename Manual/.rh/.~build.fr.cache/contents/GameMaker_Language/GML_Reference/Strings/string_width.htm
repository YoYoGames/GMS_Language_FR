largeur de la chaîne Cette fonction renvoie la largeur (en pixels) de la chaîne de caractères saisie, en tenant compte des éventuels sauts de ligne du texte. Elle est très pratique pour calculer les distances entre les éléments de texte sur la base de la largeur totale des lettres qui composent la chaîne de caractères telle qu'elle serait dessinée avec draw_text() en utilisant la police actuellement définie. Syntaxe : string_width(string) ; Arguments Description chaîne de caractères La chaîne de caractères dont il faut mesurer la largeur. Les retours : Real Exemple : var ww = string_width(str_Name + " ") ; draw_text(32, 32, str_Name)) ; draw_text(32 + ww, 32, "a gagné la partie !") ; Le code ci-dessus récupère la largeur de la chaîne de caractères donnée et dessine ensuite deux lignes de texte, en utilisant la largeur de la chaîne de caractères retournée comme séparateur.
