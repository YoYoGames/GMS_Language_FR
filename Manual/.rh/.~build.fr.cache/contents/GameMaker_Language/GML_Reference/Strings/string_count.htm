nombre de chaînes Cette fonction renvoie le nombre de fois que la sous-chaîne donnée apparaît dans une chaîne spécifique. De cette façon, vous pouvez vérifier combien de fois une lettre unique ou une phrase est répétée dans une section de texte stocké. Syntaxe : string_count(substr, str) ; Arguments Description substrat La sous-chaîne à vérifier dans la chaîne. str La chaîne à vérifier. Les retours : Real Exemple : str1 = "Hello World" ; ocount = string_count( "o", str1 ) ; Cela va définir la variable ocount au nombre de "o" dans str1, dans ce cas 2.
