string_count Cette fonction retournera le nombre de fois où la sous-chaîne donnée apparaît dans une chaîne spécifique. De cette façon, vous pouvez vérifier combien de fois une seule lettre ou une phrase est répétée dans une section de texte stocké. Syntaxe:string_count(substr, str); Argument Description substr La sous-chaîne pour laquelle vérifier la chaîne. str La chaîne à vérifier. Retour:Real Exemple:str1 = "Hello World"; ocount = string_count( "o", str1 ); Cela définira la variable ocount au nombre de "o" dans str1, dans ce cas 2.
