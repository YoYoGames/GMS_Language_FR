string_height Cette fonction renverra la hauteur (en pixels) de la chaîne d'entrée, en tenant compte de la séparation de ligne et de tout saut de ligne que le texte peut avoir. Il est très pratique pour calculer les distances entre les éléments de texte en fonction de la plus haute des lettres qui composent la chaîne comme elle serait dessinée avec draw_text() en utilisant la police actuellement définie. Syntaxe:string_height(string); Argument Description string La chaîne dont la hauteur doit être mesurée. Retour:Real Exemple:var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ"); draw_text(32, 32, string(score)); draw_text(32, 32 + hh, string(lives); Le code ci-dessus obtiendra la hauteur de la chaîne donnée, puis dessinera deux lignes de texte, en utilisant la hauteur de la chaîne retournée comme séparateur.
