lettres_chiffres Cette fonction vous permet de supprimer tous les caractères qui ne sont pas considérés comme des lettres. C'est pratique pour empêcher les gens de saisir des caractères indésirables dans une entrée de texte (comme saisir "#" pour forcer une nouvelle ligne). REMARQUE : cette fonction ne détecte et ne renvoie que l'alphabet anglais de 26 lettres de A à Z (majuscules ou minuscules). Syntaxe : lettres_chaîne(chaîne) ; Arguments Description chaîne de caractères La chaîne de caractères à convertir en lettres. Les retours : Chaîne de caractères Exemple : nom d'utilisateur = string_letters(nom d'utilisateur) ; Le code ci-dessus définit la variable "username" pour qu'elle ne contienne que les lettres de la chaîne originale (si la chaîne originale était "MacSw33ny", la chaîne retournée serait "MacSwny").
