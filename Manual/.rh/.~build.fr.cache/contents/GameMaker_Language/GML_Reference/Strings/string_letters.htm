string_letters Avec cette fonction, vous pouvez supprimer tous les caractères qui ne sont pas classés comme des lettres. Ceci est pratique pour empêcher les gens d'entrer des caractères indésirables dans une entrée de texte (comme entrer "#" pour forcer une nouvelle ligne). REMARQUE: Cette fonction détecte et renvoie uniquement l'alphabet anglais de 26 lettres de A à Z (majuscules ou minuscules). Syntaxe:string_letters(string); Argument Description string La chaîne à convertir en lettres. Retour:String Exemple:username = string_letters(username); Le code ci-dessus définira la variable "username" pour ne contenir que les lettres de la chaîne d'origine (id la chaîne d'origine était "MacSw33ny", la chaîne retournée serait "MacSwny").
