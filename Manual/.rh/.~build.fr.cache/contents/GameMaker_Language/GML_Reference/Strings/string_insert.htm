string_insert Cette fonction permet de créer une nouvelle chaîne de caractères composée de deux chaînes de caractères, l'une ayant été insérée dans l'autre à une position donnée. Cela peut être utile, par exemple, pour ajouter un nom d'utilisateur dans un texte prédéfini et ainsi faire en sorte que le joueur de votre jeu se sente plus impliqué dans l'action. plus impliqué dans l'action. Gardez à l'esprit, lorsque vous calculez la position à insérer, que les chaînes de caractères sont indexées à partir de 1, de sorte que le premier caractère d'une chaîne de caractères est 1 et non 0 comme vous pouvez vous y attendre. Syntaxe : string_insert(substr, str, index) ; Arguments Description substrat La sous-chaîne à insérer. str La chaîne de caractères à copier. indice La position en caractères de la chaîne pour insérer la sous-chaîne. Les retours : Chaîne de caractères Exemple : str2 = string_insert(username, "Hello, , how are you ?", 8) ; Cela insérera la chaîne de la variable "username" dans la phrase donnée, la chaîne résultante ressemblant à ceci : "Hello, NAME, how are you ?".
