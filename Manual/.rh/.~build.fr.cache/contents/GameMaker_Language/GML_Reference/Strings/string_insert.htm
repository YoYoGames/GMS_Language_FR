string_insert Avec cette fonction, vous pouvez créer une nouvelle chaîne composée de deux chaînes, où l'une a été insérée dans l'autre à une position donnée. Il peut être utile, par exemple, d'ajouter un nom d'utilisateur dans un texte prédéfini et ainsi faire en sorte que le joueur de votre jeu se sente plus impliqué dans l'action. Gardez à l'esprit que lorsque vous calculez la position à insérer, les chaînes sont indexées à partir de 1, de sorte que le premier caractère d'une chaîne est 1 et non 0 comme vous pouvez vous y attendre. Syntaxe:string_insert(substr, str, index); Argument Description substr La sous-chaîne à insérer. str La chaîne à copier. index La position en caractères de la chaîne pour insérer la sous-chaîne. Retour:String Exemple:str2 = string_insert(username, "Hello, , how are you?", 8); Cela insérera la chaîne dans la variable "username" dans la phrase donnée avec la chaîne résultante ressemblant à ceci: "Hello, NAME, how are you?".
