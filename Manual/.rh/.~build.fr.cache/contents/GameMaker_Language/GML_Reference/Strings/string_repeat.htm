chaîne_répétition Cette fonction renvoie simplement la même chaîne répétée un nombre donné de fois sur elle-même. Syntaxe : string_repeat(str, count) ; Arguments Description str La chaîne à répéter. compter Le nombre de fois que la chaîne doit être répétée. Les retours : Chaîne de caractères Exemple : str1 = "Hello World" ; str2 = string_repeat(str1, 3) ; Le code ci-dessus donnera à str2 la valeur "Hello WorldHello WorldHello World".
