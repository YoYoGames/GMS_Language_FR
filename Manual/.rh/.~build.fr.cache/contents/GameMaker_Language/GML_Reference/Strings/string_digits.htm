string_digits Vous pouvez utiliser cette fonction pour analyser une chaîne donnée et en extraire des nombres. Par exemple, disons que vous avez ce texte - "J'ai 81 ans". Avec cette fonction, vous obtiendrez une chaîne de retour de "81". Syntaxe:string_digits(string); Argument Description string La chaîne à partir de laquelle obtenir les chiffres. Retour:String Exemple:var t_str = string_digits(input_str); age = real(t_str); Le code ci-dessus prendra la chaîne d'entrée, la dépouillera de tous les caractères autres que les nombres, puis définira la variable age pour contenir la valeur de nombre réel de la chaîne de retour (ainsi, si la chaîne d'entrée était - par exemple - "J'ai 18 ans", la fonction renverrait "18").
