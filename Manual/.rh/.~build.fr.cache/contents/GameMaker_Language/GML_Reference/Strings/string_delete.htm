string_delete Vous pouvez utiliser cette fonction pour supprimer une partie spécifique d'une chaîne. Ainsi, vous fournissez la chaîne d'entrée et la position de début et de fin dans cette chaîne pour supprimer des caractères (l'index commence à 1) et la fonction renverra une nouvelle chaîne sans cette section. Syntaxe:string_delete(str, index, count); Argument Description str La chaîne à copier et à supprimer. index La position du premier caractère à supprimer. count Le nombre de caractères à supprimer. Retour:String Exemple:str1 = "Helloo World"; str2 = string_delete(str1, 5, 1); Cela définira str2 à "Hello World", car il supprime le "o" supplémentaire de "Hello", c'est-à-dire: la chaîne compte à partir de la première lettre 5 places, puis supprime 1 caractère).
