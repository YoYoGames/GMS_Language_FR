string_delete Vous pouvez utiliser cette fonction pour supprimer une partie spécifique d'une chaîne de caractères. Ainsi, vous fournissez la chaîne de caractères d'entrée et les positions de début et de fin dans cette chaîne pour supprimer des caractères (l'index commence à 1) et la fonction renverra une nouvelle chaîne sans cette section dans celle-ci. Syntaxe : string_delete(str, index, count) ; Arguments Description str La chaîne de caractères à copier et à supprimer. indice La position du premier caractère à supprimer. compter Le nombre de caractères à supprimer. Les retours : Chaîne de caractères Exemple : str1 = "Helloo World" ; str2 = string_delete(str1, 5, 1) ; Cela donnera à str2 la valeur "Hello World", puisqu'elle supprime le "o" supplémentaire de "Hello", c'est-à-dire que la chaîne compte à partir de la première lettre sur 5 positions, puis supprime 1 caractère).
