réel Cette fonction peut être utilisée pour transformer une chaîne de caractères donnée en un nombre réel. Lors de l'utilisation de cette fonction, les chiffres, les signes moins, les points décimaux et les parties exponentielles de la chaîne sont pris en compte, tandis que les autres caractères (tels que les lettres) provoqueront une erreur sera déclenchée. Si vous savez, ou soupçonnez, qu'une chaîne peut contenir d'autres caractères, vous pouvez utiliser la fonction string_digits() pour supprimer tous les caractères non numériques, avant d'utiliser cette fonction pour transformer la chaîne résultante en un nombre réel. Syntaxe : réel(string) ; Arguments Description chaîne de caractères La chaîne de caractères à convertir en valeur réelle. Les retours : Real Exemple : var t_str = string_digits(input_str) ; age = real(t_str) ; Le code ci-dessus prend la chaîne d'entrée, la débarrasse de tous les caractères autres que les chiffres, puis définit la variable "age" pour qu'elle contienne la valeur numérique réelle de la chaîne de retour.
