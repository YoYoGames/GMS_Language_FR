string_replace Vous pouvez utiliser cette fonction pour analyser une chaîne à la recherche d'une partie spécifique, qui peut ensuite être remplacée par la nouvelle chaîne que vous avez spécifiée. Syntaxe:string_replace(str, substr, newstr); Argument Description str La chaîne à copier. substr La sous-chaîne de la chaîne à remplacer. newstr La nouvelle sous-chaîne pour remplacer la précédente. Retour:String Exemple:str1 = "Hello Earth"; str2 = string_replace(str1, "Earth", "World"); Cela définira str2 à str1, mais avec son instance de «Terre» remplacée par «Monde», ce qui entraîne str2 étant "Hello World".
