longueur_chaîne Cette fonction renvoie le nombre de caractères d'une chaîne de caractères donnée. Elle peut être utile pour déterminer quand limiter la longueur des caractères d'une entrée de texte personnalisée (par exemple, limiter le nom d'un joueur à 10 caractères). N'oubliez pas que cette fonction est est différent de chaîne_largeur() en ce sens qu'elle mesure le nombre de caractères de la chaîne, et non sa largeur en pixels telle qu'elle est affichée à l'écran. Syntaxe : string_length(string) ; Arguments Description chaîne de caractères La chaîne de caractères dont on veut mesurer le nombre de caractères. Les retours : Real Exemple : if (string_length(name) > 10) { name = string_copy(name, 1, 10) ; } Cette opération vérifiera si la chaîne de caractères du nom est supérieure à dix et si c'est le cas, elle copiera et utilisera uniquement les dix premiers caractères.
