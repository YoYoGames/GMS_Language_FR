string_hash_to_newline Cette fonction renvoie une chaîne dans laquelle le symbole "#" a été converti en une nouvelle ligne. Donc, une chaîne au format:"Hello#World" Serait imprimé à l'écran comme:Hello World IMPORTANT! Cette fonction est fournie pour la compatibilité d'importation entre GameMaker Studio 2 et les versions précédentes du logiciel et en tant que telle, cette fonction ne doit pas être utilisée sauf lorsque cela est nécessaire pour reproduire des fonctionnalités obsolètes. Au lieu de cela, vous devriez utiliser les caractères d'échappement pour marquer une nouvelle ligne. Pour plus d'informations sur la compatibilité des importations, cliquez ici. Syntaxe:string_hash_to_newline(string); Argument Description string La chaîne à convertir sur plusieurs lignes Retour:String Exemple:var str = string_hash_to_newline("Hello#World"); draw_text(32, 32, str); Le code ci-dessus convertit la chaîne avec le symbole de hachage en une chaîne divisée sur deux lignes, puis l'imprime à l'écran.
