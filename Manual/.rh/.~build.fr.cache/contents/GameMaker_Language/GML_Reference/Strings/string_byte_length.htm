string_byte_length Cette fonction renvoie le nombre d'octets dans une chaîne, mais vous devez noter qu'en raison de leur contenu en UTF8, ce ne sera pas égal à leur longueur de chaîne. Syntaxe:string_byte_length(string); Argument Description string Chaîne pour laquelle mesurer le nombre d'octets. Retour:Real Exemple:bytesize = string_byte_length("Hello World"); Cela définirait bytesize au nombre d'octets dans "Hello World".
