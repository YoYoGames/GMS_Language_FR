chaîne_remplacer_tout Vous pouvez utiliser cette fonction pour analyser une chaîne de caractères à la recherche de parties spécifiques, qui peuvent ensuite être remplacées par la nouvelle chaîne de caractères que vous avez spécifiée à tous les endroits où elles apparaissent. Syntaxe : string_replace_all(str, substr, newstr) ; Arguments Description str La chaîne de caractères à copier. substrat La sous-chaîne de la chaîne de caractères à remplacer dans toutes les instances. newstr La nouvelle sous-chaîne qui remplace la ou les sous-chaînes précédentes. Les retours : Chaîne de caractères Exemple : str1 = "Hexxo Worxd" ; str2 = string_replace_all(str1, "x", "l") ; Cela va définir str2 comme str1, mais avec ses instances de "x" remplacées par "l", ce qui fait que str2 sera "Hello World".
