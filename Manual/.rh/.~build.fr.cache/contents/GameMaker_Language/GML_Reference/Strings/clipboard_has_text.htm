presse-papiers_a_texte Cette fonction retournera true si le presse-papiers contient du texte ou false si ce n'est pas le cas. REMARQUE: Cette fonction n'est valable que pour la Windows cible. Syntaxe : clipboard_has_text() ; Les retours : Booléen Exemple : if clipboard_has_text() { str = clipboard_get_text() ; clipboard_set_text("") ; } Le code ci-dessus vérifie la présence de texte dans le presse-papiers et s'il en contient, il est lu sous forme de chaîne dans la variable "str". Enfin, le presse-papiers est effacé en lui attribuant la valeur d'une chaîne vide. Retour : Cordes Suivant : clipboard_get_text Copyright YoYo Games Ltd. 2021 Tous droits réservés
