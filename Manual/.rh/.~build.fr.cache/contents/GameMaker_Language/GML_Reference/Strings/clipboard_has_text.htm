clipboard_has_text Cette fonction retournera true si le presse-papiers contient du texte ou false si ça ne fait pas. REMARQUE: cette fonction n'est valable que pour la cible Windows Syntaxe:clipboard_has_text(); Retour:Boolean Exemple:if clipboard_has_text()    {    str = clipboard_get_text();    clipboard_set_text("");    } Le code ci-dessus vérifie le texte dans le presse-papiers et s'il en contient, il est lu comme une chaîne dans la variable "str". Enfin, le presse-papiers est effacé en le définissant sur une chaîne vide. Retour: Cordes Suivant:clipboard_get_text © Copyright YoYo Games Ltd. 2021 Tous droits réservés
