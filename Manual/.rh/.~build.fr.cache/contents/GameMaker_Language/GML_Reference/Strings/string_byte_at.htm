string_byte_at Renvoie la valeur d'octet brute sous forme de valeur réelle à une position donnée dans la chaîne donnée. Syntaxe:string_byte_at(str, index); Argument Description str La chaîne à vérifier. index La position à partir de laquelle obtenir l'octet. Retour:Real Exemple:newbyte = string_byte_at("Hello World", 5); Cela définira newbyte à la valeur d'octet brut de la sixième lettre de "Hello World".
