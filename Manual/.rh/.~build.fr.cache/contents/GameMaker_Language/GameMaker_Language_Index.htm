Index deslangues de GameMaker Le GameMaker Language (également appelé simplement GML) est le langage de script propriétaire de GameMaker Studio 2. Ce langage est structuré pour permettre aux utilisateurs de créer leurs jeux de manière intuitive et flexible tout en tout en offrant toute la puissance de tout autre langage de programmation majeur. Il constitue également la base du Drag and Drop™ et peut être utilisé conjointement avec celui-ci si nécessaire.
