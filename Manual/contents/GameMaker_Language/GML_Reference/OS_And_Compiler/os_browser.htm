os_browser

Cette variable en lecture seule contient l'une des diverses constantes (énumérées ci-dessous) dont dispose GameMaker Studio 2 pour vous indiquer dans quel navigateur vous exécutez actuellement le jeu (le cas échéant).

Syntaxe :

os_browser

Les retours :

Constant

Constant Description
navigateur_not_a_browser Le jeu ne se déroule pas dans un navigateur
navigateur_inconnu Navigateur inconnu
navigateur_ie Internet Explorer
navigateur_ie_mobile Internet Explorer sur un appareil mobile
navigateur_firefox Mozilla Firefox
navigateur_chrome Google Chrome
navigateur_safari Safari
navigateur_safari_mobile Safari sur un appareil mobile
navigateur_opéra Opéra
browser_tizen Navigateur pour appareils mobiles Tizen
magasin_fenêtres_navigateur ApplicationWindows

Exemple :

if (os_browser == browser_not_a_browser)
{
global.Config = 0 ;
}
else
{
global.Config = 1 ;
}

Le code ci-dessus vérifie si le jeu est exécuté dans un navigateur ou non et définit une variable globale à une valeur dépendant du résultat de la vérification.
