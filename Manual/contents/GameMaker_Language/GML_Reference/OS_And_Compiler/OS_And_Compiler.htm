OS et compilateur

Lors de la création de jeux multiplateformes, il est souvent d'une importance vitale que vous obteniez les détails du système d'exploitation utilisé par l'appareil exécutant votre jeu. Des éléments tels que la langue, la version ou l'état du réseau peuvent tous être utilisés pour adapter votre jeu au système qui l'exécute et rendre l'expérience de l'utilisateur final la meilleure possible, ainsi GameMaker Studio 2 dispose d'un certain nombre de fonctions spéciales qui peuvent être utilisées pour obtenir les informations nécessaires.

Vous pouvez utiliser les fonctions et variables suivantes pour obtenir certains détails sur le système d'exploitation ou le navigateur qui exécute votre jeu:

  • os_browser
  • os_device
  • os_type
  • os_version
  • os_is_paused
  • os_is_network_connected
  • os_get_config
  • os_get_language
  • os_get_region
  • os_get_info
  • os_powersave_enable
  • os_lock_orientation
  • os_check_permission
  • os_request_permission

Certaines fonctions sont pré-compilées lorsque vous exécutez votre jeu (c'est-à-dire: elles sont utilisées pour définir comment le jeu final sera compilé), et certaines sont également à utiliser lorsque le jeu a été compilé pour obtenir des détails spécifiques sur l'environnement d' runtime Vous trouverez ici la liste complète de ces fonctions:

  • GM_build_date
  • GM_version
  • _Version d' runtime GM_
  • gml_release_mode
  • gml_pragma
  • parameter_count
  • parameter_string
  • environment_get_variable

Sous Windows, la résolution du Windows est définie sur 1 ms par défaut lorsque votre jeu s'exécute. Les fonctions suivantes sont disponibles pour récupérer et modifier la résolution du planificateur de thread Windows runtime de l'exécution:

  • scheduler_resolution_get
  • scheduler_resolution_set

REMARQUE: il s'agit de fonctions héritées et vous devez définir toutes les fonctions d'extension dans l'éditeur d'extension lui-même.

REMARQUE: ces fonctions sont Windows -formes cibles macOS

  • external_define
  • external_call
  • external_free
