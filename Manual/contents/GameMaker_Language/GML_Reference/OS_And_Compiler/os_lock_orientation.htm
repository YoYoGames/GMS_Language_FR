os_lock_orientation

Avec cette fonction, vous pouvez "verrouiller" votre appareil sur l'orientation actuelle jusqu'à ce que vous le "libériez" pour autoriser à nouveau toutes les orientations activées par les Options de jeu pour cette plate-forme cible. Notez que vous souhaiterez probablement confirmer que l'orientation est comme vous le souhaitez avant de verrouiller.

Syntaxe:

os_lock_orientation(flag)

Argument Description
flag Mis à true ou false pour activer ou désactiver le verrouillage d'orientation.

Retour:

N/A

Exemple:

if (os_type == os_android) || (os_type == os_ios)
   {
   os_lock_orientation(true);
   }

Le code ci-dessus vérifie le type de système d'exploitation et s'il s'agit d' Android ou d' iOS le verrouillage d'orientation est marqué comme true.

Retour: OS et compilateur
