externe_libre

Cette fonction libère la mémoire associée à la dll ou à la dylib portant le nom donné. Cela doit être fait lorsque le fichier en question n'est plus nécessaire dans le jeu, normalement (par exemple) dans un événement de fin de partie.

Syntaxe :

external_free(id) ;

Arguments Description
id Le nom de la dll ou de la dylib que vous voulez libérer.

Les retours :

N/A

Exemple :

external_free("MyDLL.dll") ;

L'exemple de code ci-dessus libérera la mémoire associée à la dll donnée.
