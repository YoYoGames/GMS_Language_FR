os_device

Cette variable en lecture seule contient l'une des diverses valeurs constantes énumérées ci-dessous pour vous indiquer sur quel périphérique vous exécutez actuellement le jeu. Notez que cette variable est dépréciée au profit de la fonction os_get_info() qui renvoie des informations plus précises sur le périphérique sur lequel le jeu est exécuté.

Syntaxe :

os_device

Les retours :

Constant

Constant Description
dispositif_ios_ipad iPad
appareil_ios_ipad_retina iPad plus récent avec écran Retina de 2048 x 1536
dispositif_ios_iphone6 iPhone6 avec un écran de 1334 x 750
dispositif_ios_iphone6plus Un iPhone 6 plus grand avec un écran 1920 x 1080
dispositif_ios_iphone5 iPhone5 avec une taille d'écran de 640 x 1136)
dispositif_ios_iphone iPhone/iPod Touch plus ancien (écran 480 x 320) ou téléphone Android
dispositif_ios_iphone_retina iPhone/iPod Touch plus récent avec écran Retina de 960 x 640
émulateur de périphérique L'appareil est en fait un émulateur(Windows Phone ou Android)
tablette_dispositif TabletteAndroid
périphérique_ios_inconnu Inconnu ou non iOS

Exemple :

if (os_browser = browser_not_a_browser)
{
switch (os_device)
{
case device_ios_ipad : global.Config = 2 ; break ;
case device_ios_iphone : global.Config = 3 ; break ;
case device_ios_iphone_retina : global.Config = 4 ; break ;
case device_ios_unknown : global.Config = 5 ; pause ;
}
}
else
{
global.Config = 1 ;
}

Le code ci-dessus vérifie si le jeu est exécuté dans un navigateur ou non et définit une variable globale à une valeur dépendant du résultat de la vérification.
