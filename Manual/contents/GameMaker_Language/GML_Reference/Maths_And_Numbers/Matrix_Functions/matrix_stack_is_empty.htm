matrix_stack_is_empty

Cette fonction peut être utilisée pour vérifier si la pile de matrices est vide (retourne true ) ou non (retourne false ).

Syntaxe:

matrix_stack_is_empty();

Retour:

Boolean

Exemple:

if !matrix_stack_is_empty()
    {
    matrix_stack_clear();
    }

Le code ci-dessus vérifie la pile de matrices pour voir si elle est vide et si ce n'est pas le cas, il l'efface.
