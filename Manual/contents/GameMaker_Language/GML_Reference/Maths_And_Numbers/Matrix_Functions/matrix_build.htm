matrix_build

Cette fonction peut être utilisée pour créer votre propre matrice personnalisée et renvoie une valeur d'index pour cette nouvelle matrice qui doit être stockée dans une variable pour une référence et une utilisation futures. La matrice elle-même est créée comme un tableau de 16 valeurs où les 4 premiers éléments sont la ligne 1, les 4 seconds éléments sont la ligne 2 etc... d'une matrice 4x4.

REMARQUE: Lorsque vous construisez une nouvelle matrice de cette manière, l'ordre des opérations est YXZ.

Syntaxe :

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale) ;

Arguments Description
x La composante x du vecteur de translation.
y La composante y du vecteur de translation.
z La composante z du vecteur de translation.
xrotation L'angle de rotation autour de l'axe des x (en degrés °).
yrotation L'angle de rotation autour de l'axe des y (en degrés °).
xzotation L'angle de rotation autour de l'axe z (en degrés °).
xscale Le montant de l'échelle x.
yscale Le montant de l'échelle y.
zscale Le montant de l'échelle z.

Les retours :

Index de la matrice (Matrice 4x4)

Exemple :

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1) ;

Le code ci-dessus construira une nouvelle matrice transformée et stockera l'indice de la matrice résultante dans la variable "t_matrix".
