matrix_get

Cette fonction renvoie un tableau de 16 valeurs, correspondant au type de matrice 4x4 donné, où la ligne 1 correspond aux éléments [0 - 3], la ligne 2 aux éléments [4 -7], etc... (voir l'image sur la page principale). Les matrices disponibles sont view, projection et world, pour lesquelles vous utiliserez l'une des constantes suivantes :

Constant Description
Vue de la matrice La matrice de vue actuelle
matrice_projection La matrice de projection actuelle
matrice_monde La matrice mondiale actuelle

Syntaxe :

matrix_get(type) ;

Arguments Description
type Le type de matrice pour obtenir les valeurs de (voir les constantes listées ci-dessus)

Les retours :

Tableau (16 éléments)

Exemple :

v_array = matrix_get(matrix_view) ;

Le code ci-dessus récupère les valeurs de la matrice de vue actuelle et remplit le tableau "v_array" avec ces valeurs.
