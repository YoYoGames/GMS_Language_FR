longueurdir_y

Cette fonction est utilisée pour obtenir la composante y d'une position "len" pixels du point de départ et dans la direction "dir". Si vous imaginez un cercle autour de votre instance, et que vous imaginez ensuite un point n'importe où sur ce cercle, pour se déplacer vers ce point, nous devons déplacer l'objet de tant de pixels dans cette direction. jusqu'à ce point, nous devons déplacer l'objet de tant de pixels dans cette direction... donc cette fonction (lorsqu'elle est utilisée avec lengthdir_x()) obtient la position de ce point sur le cercle pour être utilisée dans le code par l'instance. Voir l'image ci-dessous pour plus de détails :

Syntaxe :

lengthdir_y(len, dir) ;

Arguments Description
len La longueur de l'éloignement du point de retour.
dir La direction du point à retourner.

Les retours :

Real

Exemple :

var _xx = x + lengthdir_x(64, image_angle) ;
var _yy = y + lengthdir_y(64, image_angle) ;
instance_create_layer(_xx, _yy, "Bullets", obj_bullet) ;

Cela créera une instance de balle à(_xx, _yy), qui sera à 64 pixels de l'instance parent dans la direction de l'angle de l'image.
