floor

Renvoie le plancher de n, c'est-à-dire n arrondi à un entier inférieur. Ceci est similaire au round() fonction, mais il arrondit seulement vers le bas, quelle que soit la valeur décimale, donc floor(5.99999) retournera 5, tout comme floor(5.2), floor(5.6457) etc...

Syntaxe:

floor(n);

Argument Description
n Le numéro au sol.

Retour:

Real

Exemple:

val = floor( 3.9 );

Cela mettra val à 3.
