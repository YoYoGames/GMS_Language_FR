choisissez

Parfois, vous voulez spécifier autre chose que des nombres pour une sélection aléatoire, ou les nombres que vous voulez ne sont pas dans un ordre réel ou dans une plage définie. Dans ces cas, vous utiliserez choose() pour générer un résultat aléatoire. Par exemple, disons que vous voulez créer un objet avec un sprite aléatoire au début, alors vous pouvez utiliser cette fonction pour définir l'index du sprite sur un ensemble de sprites donnés. Notez que vous pouvez avoir autant d'arguments que vous le souhaitez (notez que plus d'arguments signifiera que la fonction sera plus lente à analyser).

REMARQUE: cette fonction renverra la même valeur à chaque nouvelle exécution du jeu, car GameMaker Studio 2 génère à chaque fois la même graine aléatoire initiale afin de faciliter le débogage du code. Pour éviter ce comportement, utilisez randomise() au début de votre jeu.

Syntaxe :

choisir(val0, val1, val2... max_val) ;

Arguments Description
val0... max_val Une valeur d'entrée qui peut être une chaîne de caractères, un nombre entier, une variable ou une constante.

Les retours :

Un des arguments donnés

Exemple :

sprite_index = choisir(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub) ;
hp = choisir(5, 8, 15, 32, 40) ;
name = choisir("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen") ;

Le code ci-dessus utilise le choix pour définir un certain nombre de propriétés pour l'instance.
