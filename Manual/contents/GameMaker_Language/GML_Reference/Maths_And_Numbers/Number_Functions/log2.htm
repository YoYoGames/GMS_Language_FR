log2

Cette fonction demande essentiellement "combien de 2 faut-il multiplier pour obtenir n?". Un exemple courant d'utilisation en programmation serait de calculer le nombre de bits nécessaires pour représenter un nombre. Pour ce faire, nous calculons le log2(x) du nombre, arrondissez-le vers le bas et ajoutez 1 - par exemple log2(100) renvoie 6,643856, arrondi à 6, auquel nous ajoutons un et cela nous donne 7. Nous avons donc besoin de 7 bits pour représenter 100. Il peut également être utilisé pour décrire la croissance exponentielle ou la décroissance, et est souvent appelé un logarithme binaire.

Syntaxe:

log2(n);

Argument Description
n La valeur d'entrée.

Retour:

Real

Exemple:

colourbits = floor(log2(colour)) - 1;

Le code ci-dessus stocke le nombre de bits nécessaires pour représenter la valeur stockée dans la variable "couleur".
