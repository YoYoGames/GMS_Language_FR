clamp

Avec cette fonction, vous pouvez maintenir une valeur d'entrée entre une plage spécifiée.

Syntaxe:

clamp(val, min, max)

Argument Description
val La valeur à serrer.
min La valeur minimale à fixer entre.
max La valeur maximale entre laquelle serrer.

Retour:

Real

Exemple:

speed = clamp(speed, 1, 10);

Le code ci-dessus limitera la vitesse afin qu'elle ne tombe jamais en dessous de 1 ou dépasse 10.
