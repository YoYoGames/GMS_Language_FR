gamme_aléatoire

Cette fonction renvoie un nombre aléatoire compris dans la plage spécifiée, et cette valeur de retour ne doit pas nécessairement être un nombre entier. Par exemple, random_range(20,50) renverra un nombre aléatoire compris entre 20 et 50, mais la valeur peut être un nombre réel comme 38.65265. Les nombres réels peuvent également être utilisés comme arguments d'entrée.

REMARQUE: cette fonction renverra la même valeur à chaque nouvelle exécution du jeu, car GameMaker Studio 2 génère à chaque fois la même graine aléatoire initiale afin de faciliter le débogage du code. Pour éviter ce comportement, utilisez randomise() au début de votre jeu.

Syntaxe :

random_range(n1, n2) ;

Arguments Description
n1 L'extrémité inférieure de la fourchette dans laquelle le numéro aléatoire sera sélectionné.
n2 L'extrémité supérieure de la fourchette dans laquelle le numéro aléatoire sera sélectionné.

Les retours :

Real

Exemple :

score += random_range(500, 600) ;

Cela ajoutera entre 500 et 600 points au score total.
