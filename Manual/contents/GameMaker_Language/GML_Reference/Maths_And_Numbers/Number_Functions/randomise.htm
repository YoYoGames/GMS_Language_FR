randomise

Cette fonction définit la valeur de départ sur une valeur aléatoire. Si vous avez besoin de conserver une valeur cohérente sur un certain nombre d'exécutions d'un jeu, vous devriez utiliser random_set_seed(). Veuillez noter que lorsque vous utilisez les fonctions de nombres aléatoires dans GameMaker Studio 2 la valeur initiale est toujours la même, car cela facilite le traçage des erreurs et le débogage. Si vous souhaitez tester avec un vrai aléatoire, vous devez appeler cette fonction au début de votre partie. La fonction renverra la nouvelle valeur de départ aléatoire (un entier 32 bits non signé).

Syntaxe:

randomise();
// or
randomize();

Retour:

Integer (unsigned 32 bit value)

Exemple:

randomise();

Le code ci-dessus va randomiser la graine.
