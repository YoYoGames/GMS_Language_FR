random

Cette fonction convient aux probabilités où le renvoi d'un entier (nombre entier) n'est pas nécessaire. Par exemple, random(100) renverra une valeur de 0 à 99, mais cette valeur peut être 22,56473! Vous pouvez également utiliser des nombres réels et non des entiers dans cette fonction comme ceci -random(0.5), qui renverra une valeur comprise entre 0 et 0,4999999.

REMARQUE: Cette fonction renverra la même valeur à chaque fois que le jeu est réexécuté car GameMaker Studio 2 génère la même graine aléatoire initiale à chaque fois pour rendre le code de débogage beaucoup plus facile. Pour éviter ce comportement, utilisez randomise () au début de votre partie.

Syntaxe:

random(n);

Argument Description
n La plage supérieure à partir de laquelle le nombre aléatoire sera sélectionné.

Retour:

Real

Exemple:

if (random(10) >= 9)
   {
   score += 100;
   }

Cela produira environ une chance sur dix d'ajouter 100 au score.
