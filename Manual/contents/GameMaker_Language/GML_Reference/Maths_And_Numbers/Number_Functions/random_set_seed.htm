random_set_seed

Pour générer un nombre aléatoire, GameMaker Studio 2 commence par un numéro de départ aléatoire. Avec cette fonction, vous pouvez définir cette graine à une valeur connue et ainsi "forcer" le résultat de tous les événements aléatoires par la suite à être le même à chaque fois que le programme est exécuté. Par exemple, cette fonction peut être utilisée en conjonction avec random_get_seed() pour créer du contenu généré de manière procédurale et enregistrer les résultats sans avoir d'énormes sauvegardes (vous enregistrez uniquement la graine, pas besoin d'autre chose). Si vous avez besoin de résultats vraiment aléatoires pour tout, vous devriez utiliser le randomise() fonction.

REMARQUE: Bien que cette graine donne des résultats cohérents sur chaque plate-forme cible, les résultats peuvent varier d'une plate-forme à l'autre en raison de la manière différente de chaque cible.

Syntaxe:

random_set_seed(val);

Argument Description
val La graine à planter.

Retour:

N/A

Exemple:

if debug
    {
    random_set_seed(1);
    }

Le code ci-dessus définit la valeur de départ aléatoire sur 1 uniquement si la variable «debug» est vraie.
