irandom

Cette fonction très utile ne renvoie que des entiers (nombres entiers). Ainsi, par exemple, pour obtenir un nombre aléatoire de 0 à 9, vous pouvez utiliser irandom(9) et il renverra un nombre de 0 à 9 inclus. Notez que les flottants peuvent également être utilisés mais la valeur supérieure après le point sera exclue, ainsi irandom(9.7) renverra une valeur comprise entre 0 et 9 uniquement. La fonction a une limite supérieure de $7fffffffffffffffLL, il faut donc faire attention si vous utilisez de très grands nombres.

REMARQUE : cette fonction renverra la même valeur à chaque nouvelle exécution du jeu, car GameMaker Studio 2 génère à chaque fois la même graine aléatoire initiale afin de faciliter le débogage du code. Pour éviter ce comportement, utilisez randomise() au début de votre jeu.

Syntaxe :

irandom(n) ;

Arguments Description
n La fourchette supérieure dans laquelle le numéro aléatoire sera sélectionné.

Les retours :

Entier

Exemple :

si (irandom(9) == 1)
{
score += 100 ;
}

Cela donnera une chance sur dix (puisque le 0 est inclus) d'ajouter 100 au score.
