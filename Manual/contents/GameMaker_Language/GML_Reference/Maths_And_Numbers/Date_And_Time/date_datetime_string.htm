date_datetime_string

Avec cette fonction, vous pouvez créer une chaîne contenant la date / heure donnée, formatée pour le système ou le périphérique qui exécute le jeu lorsque la fonction est appelée.

Syntaxe:

date_datetime_string(date);

Argument Description
date La date à utiliser.

Retour:

String

Exemple:

str = date_datetime_string(date_current_datetime());

Cela définirait "str" sur quelque chose comme "15 septembre 2011, 11: 33.00" en fonction des paramètres système pour l'affichage de la date / heure et de la date et de l'heure actuelles.
