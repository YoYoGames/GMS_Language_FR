date_jours_dans_mois

Avec cette fonction, vous pouvez obtenir le nombre de jours que compte le mois donné, soit 28, 29, 30 ou 31.

Syntaxe :

date_days_in_month(date) ;

Arguments Description
date La date à utiliser.

Les retours :

Real

Exemple :

days = date_days_in_month(date_current_datetime()) ;

Ainsi, "days" correspondrait au nombre de jours du mois en cours.
