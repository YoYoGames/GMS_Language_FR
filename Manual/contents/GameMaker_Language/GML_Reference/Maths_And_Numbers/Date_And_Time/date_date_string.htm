date_date_string

Avec cette fonction, vous pouvez créer une chaîne contenant la date donnée, au format jour / mois / année.

Syntaxe:

date_date_string(date);

Argument Description
date La date à utiliser.

Retour:

String

Exemple:

str = date_date_string(date_current_datetime());
draw_text(32, 32, str);

Cela définirait "str" pour contenir une chaîne formatée de la date et de l'heure actuelles, comme indiqué par le système.
