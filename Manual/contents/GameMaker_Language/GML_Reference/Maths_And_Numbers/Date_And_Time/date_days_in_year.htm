date_days_in_year

Avec cette fonction, vous pouvez obtenir le nombre de jours de l'année donnée, en renvoyant 365 pour une année normale et 366 pour une année bissextile.

Syntaxe:

date_days_in_year(date);

Argument Description
date La date à utiliser.

Retour:

Real

Exemple:

days = date_days_in_year(date_current_datetime());

Cela définirait "jours" au nombre de jours de l'année en cours.
