current_day

Cette variable en lecture seule renverra le jour sous la forme d'une valeur comprise entre 1 et 31, selon le mois.

Syntaxe:

current_day;

Retour:

Real

Exemple:

draw_text(32, 32, "Today is " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +".");

Le code ci-dessus dessinera le jour, le mois et l'année.

Retour: Date et heure
