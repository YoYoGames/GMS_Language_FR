current_time

Cette variable en lecture seule renverra le nombre de millisecondes écoulées depuis le démarrage du jeu.

Syntaxe:

current_time;

Retour:

Real

Exemple:

if (current_time > 600000)
    {
    msg = show_question_async("Would you like to rate?");
    }

Le code ci-dessus vérifie si plus de 10 minutes se sont écoulées avant de poser une question à l'utilisateur.

Retour: Date et heure
