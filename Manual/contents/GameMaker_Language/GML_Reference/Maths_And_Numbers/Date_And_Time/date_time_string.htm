date_time_string

Avec cette fonction, vous pouvez créer une chaîne contenant l'heure donnée, formatée pour le système ou l'appareil qui exécute le jeu lorsque la fonction est appelée.

Syntaxe:

date_time_string( date );

Argument Description
date La date / heure à utiliser.

Retour:

String

Exemple:

str = date_time_string(date_current_datetime());

Cela définirait la variable donnée à quelque chose comme "11: 36.00" en fonction des paramètres système pour l'affichage de la date / heure et de la date et de l'heure actuelles.
