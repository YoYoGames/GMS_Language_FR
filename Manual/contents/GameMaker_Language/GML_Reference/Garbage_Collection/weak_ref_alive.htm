weak_ref_create

Avec cette fonction, vous pouvez vérifier la référence faible d'une structure pour voir si elle est toujours "vivante" ou non. Vous fournissez la référence faible à vérifier (telle que retournée par la fonction weak_ref_create()), et la fonction renvoie true si la structure est toujours référencée quelque part ou false si elle ne l'est pas et a été collectée. Notez que si vous fournissez une valeur qui n'est pas une référence faible, la fonction retournera undefined.

Syntaxe :

weak_ref_alive(weak_ref) ;

Arguments Description
faible_réf La référence faible de la structure que vous voulez vérifier.

Les retours :

Booléen (ou indéfini)

Exemple :

if weak_ref_alive(inventory_ref)
{
inventory = -1 ;
}

Le code ci-dessus vérifie une référence faible à une structure et si elle est toujours vivante, la variable qui la référence est mise à -1, déréférençant la structure et lui permettant d'être collectée par les ordures.
