weak_ref_create

Avec cette fonction, vous pouvez vérifier la référence faible à diverses structures pour voir si elles sont toujours "vivantes" ou non. Vous fournissez un tableau de références faibles à vérifier (chaque référence faible doit avoir été créée à l'aide de la fonctionweak_ref_create() ), et la fonction retournera true si l' une des structures est toujours référencée quelque part ou false s'ils ne le sont pas et ont été ramassés. Notez que si vous fournissez un tableau où l'une des valeurs n'est pas une référence faible, la fonction retournera undefined.

Notez que la fonction prend également deux arguments facultatifs, où le premier vous permet de fournir un index initial dans le tableau à partir duquel vérifier, ainsi qu'un argument pour définir le nombre de positions (longueur) de cet index à vérifier. Seuls les indices de tableau dans ces paramètres seront vérifiés.

Syntaxe:

weak_ref_any_alive(weak_ref, [index], [length]);

Argument Description
weak_ref La référence faible à la structure que vous souhaitez vérifier.
[index] OPTIONNEL! L'index dans le tableau à partir duquel commencer la vérification
[length] OPTIONNEL! Le nombre de positions de l'index donné à vérifier

Retour:

Boolean (or undefined)

Exemple:

if weak_ref_any_alive(inventory_ref_array)
    {
    instance_destroy(obj_Inventory_Control);
    }

Le code ci-dessus vérifie un tableau de références faibles et si certaines sont toujours actives, une instance est détruite.
