weak_ref_create

Avec cette fonction, vous pouvez créer une référence faible à une structure qui peut ensuite être utilisée pour vérifier si la structure est toujours "vivante" (référencée) ou non dans le jeu. Vous fournissez la référence à la structure que vous souhaitez suivre, et la fonction renverra une autre structure qui est une référence faible à cette structure. Notez que vous pouvez vérifier si une référence est "forte" ou "faible" en utilisant la fonction instanceof(), car une référence forte renverra non plus "struct "ou le nom de la fonction constructeur qui a créé la structure, ou"weakref "si c'est une référence faible. Notez également que la structure de référence faible aura une variable"ref "auquel on peut accéder pour obtenir la référence de chaîne à la structure en question, sauf si elle a été collectée par garabage, auquel cas elle retournera undefined.

Syntaxe:

weak_ref_create(struct_to_track);

Argument Description
struct_to_track La structure pour laquelle vous souhaitez créer une référence faible

Retour:

Struct

Exemple:

inventory_ref = weak_ref_create(inventory);

Le code ci-dessus crée une référence faible à une structure et la stocke dans une variable d'instance pour une utilisation ultérieure.
