weak_ref_create

Avec cette fonction, vous pouvez créer une référence faible à une structure qui peut ensuite être utilisée pour vérifier si la structure est toujours "vivante" (référencée) ou non dans le jeu. Vous fournissez la à la structure que vous voulez suivre, et la fonction retournera une autre structure qui est une référence faible à cette structure. Notez que vous pouvez vérifier si une référence est "forte" ou "faible" en utilisant la fonction instanceof(), car une référence forte renverra soit "struct" ou le nom de la fonction de construction qui a créé la structure, soit "weakref" s'il s'agit d'une référence faible. Notez également que la structure de référence faible aura une variable "ref" à laquelle on peut accéder pour obtenir la référence de la chaîne de caractères de la structure en question, à moins qu'elle n'ait été collectée par garabage, auquel cas elle retournera undefined.

Syntaxe :

weak_ref_create(struct_to_track) ;

Arguments Description
struct_to_track La structure pour laquelle vous voulez créer une référence faible.

Les retours :

Structure

Exemple :

inventory_ref = weak_ref_create(inventory) ;

Le code ci-dessus crée une référence faible à une structure et la stocke dans une variable d'instance pour une utilisation ultérieure.
