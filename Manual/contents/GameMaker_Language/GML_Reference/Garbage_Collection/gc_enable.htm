gc_enable

Avec cette fonction, vous pouvez activer ou désactiver le ramasse-miettes. Appel de la fonction avec true comme l'argument le permet et en utilisant false le désactive (non recommandé). Il est activé par défaut.

Syntaxe:

gc_enable(enable);

Argument Description
enable permettre (true ) ou désactiver (false ) le ramasse-miettes.

Retour:

N/A

Exemple:

if (global.debug == true)
    {
    gc_enable(false);
    }

Le code ci-dessus désactive le garbage collection si la variable globale donnée est true.

Retour: Ramassage des ordures
