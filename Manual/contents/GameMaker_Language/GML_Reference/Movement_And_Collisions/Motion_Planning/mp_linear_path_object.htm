mp_linear_path_object

Cette fonction calcule un chemin en ligne droite de la position actuelle de l'instance au point spécifié par les valeurs "xgoal" et "ygoal". Elle utilise la taille de pas indiquée, la même que dans la fonction mp_linear_step(). Le chemin indiqué doit déjà exister et sera écrasé par le nouveau chemin et la fonction retournera si un chemin complet a été trouvé (true) ou non (false). Un chemin complet n'est trouvé que s'il n'y a pas eu de collision avec l'objet ou l'instance indiqué et si false est retourné, un chemin est tout de même généré, mais il n'ira que jusqu'à la position où le chemin a été bloqué.

NOTE: Cette fonction ne déplace pas l'instance. Elle définit uniquement un chemin, et vous devez utiliser les fonctions Path pour le déplacement.

Syntaxe :

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj) ;

Arguments Description
chemin L'index du chemin à utiliser
xgoal La position x de la cible.
ygoal La position y de la cible.
taille d'étape La vitesse de déplacement de l'instance en pixels par pas.
obj. L'objet qui doit bloquer le chemin. Peut être un index d'objet, un id d'instance ou le mot-clé spécial,all

Les retours :

Booléen

Exemple :

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
path_start(path, 4, 0, 0) ;
}

Le code ci-dessus vérifie s'il y a une collision avec "obj_Wall" le long du chemin entre l'objet qui exécute le code et la position x/y de la souris. S'il n'y a pas de collision et que le chemin complet est généré, l'objet démarre le long du chemin généré. chemin généré.
