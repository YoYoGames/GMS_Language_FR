mp_grid_add_instances

Cette fonction utilise le masque de collision de l'instance (déterminé par les propriétés du sprite ou l'index du masque de l'instance appelante) pour marquer les cellules comme interdites ou non dans une grille MP, où les cellules interdites ne peuvent être traversées par aucune des fonctions de recherche de chemin. Vous pouvez spécifier dans la fonction s'il faut considérer les collisions précises ou non et les cellules interdites marquées changeront en fonction de ce paramètre. Cette image illustre ce comportement :

Les deux instances ci-dessus ont été ajoutées à la grille du MP à l'aide de la fonction mp_grid_add_instances(), precise étant réglé sur true. Comme vous pouvez le constater, l'instance verte n'a marqué que les carrés de la grille qu'elle "touche" comme étant interdits, du fait que son masque de sprite est également défini comme précis. Cependant, la seconde instance a marqué d'autres cellules cellules qui ne semblent pas se toucher. Cela est dû au fait que le masque de sprite n' est pas défini comme précis, ce qui signifie que même si vous utilisez le paramètre précis dans la fonction, seule la boîte de délimitation du sprite sera prise en compte. Ceci est également ce qui se produirait si vous donniez la valeur false à l'argument precise de la fonction - même les instances avec un masque de sprite précis seront ajoutées dans la grille sur la base de leur boîte de délimitation.

Syntaxe :

mp_grid_add_instances(id, obj, prec) ;

Arguments Description
id Index de la grille mp_ qui doit être utilisée
obj. Index d'objet, ou identifiant d'instance, des instances à ajouter dans la grille mp_grid.
préc Si le contrôle est basé sur des collisions parfaites au niveau du pixel (true = slow) ou sur sa boîte de délimitation en général (false = fast).

Les retours :

N/A

Exemple :

mp_grid_add_instances(grid, obj_Wall, true) ;

Le code ci-dessus ajoutera toutes les instances de "obj_Wall" dans la mp_grid indexée dans la variable "grid" en utilisant le masque de collision précis plutôt que la boîte englobante.
