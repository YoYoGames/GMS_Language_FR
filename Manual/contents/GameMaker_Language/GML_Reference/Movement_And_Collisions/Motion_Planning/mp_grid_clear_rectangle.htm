mp_grid_clear_rectangle

Cette fonction vous permet de définir une zone en coordonnées de pièce qui effacera ensuite les cellules correspondantes dans la grille MP spécifiée. Même si une cellule se trouve partiellement dans la région rectangulaire définie, elle sera effacée.

Syntaxe :

mp_grid_clear_rectangle(id, x1, y1, x2, y2) ;

Arguments Description
id Index de la grille mp_ qui doit être utilisée
x1 La coordonnée x du côté gauche du rectangle à vérifier.
y1 La coordonnée y du côté supérieur du rectangle à vérifier.
x2 La coordonnée x du côté droit du rectangle à vérifier.
y2 La coordonnée x du côté inférieur du rectangle à vérifier.

Les retours :

Booléen

Exemple :

mp_grid_clear_rectangle(grid, 0, 0, 100, 200) ;

Le code ci-dessus marquera comme libres toutes les cellules de la grille mp_grid indexées dans la variable "grid" qui se trouvent dans la zone 0,0 à 100,200.
