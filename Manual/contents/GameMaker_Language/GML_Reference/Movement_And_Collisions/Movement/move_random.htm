move_random

Avec cette fonction, vous pouvez définir l'instance sur une position n'importe où dans la pièce, mais alignée sur une grille "invisible". Ainsi, une valeur de 32 pour hsnap et vsnap définira l'instance sur une position aléatoire alignée sur une grille de 32x32 carrés. vous pouvez définir ces valeurs sur 1 pour obtenir une position n'importe où dans la pièce.

REMARQUE: Cette fonction s'alignera sur les mêmes positions à chaque fois que le jeu est réexécuté car GameMaker Studio 2 génère la même graine aléatoire initiale à chaque fois pour rendre le code de débogage beaucoup plus facile. Pour éviter ce comportement, utilisez randomise () au début de votre partie. Cela n'est vrai que lors du test et du débogage du jeu, car le package exécutable final ne montrera pas ce comportement et sera aléatoire à chaque lecture.

Syntaxe:

move_random(hsnap, vsnap);

Argument Description
hsnap L'accrochage horizontal (la taille en pixels entre les «cellules»).
vsnap L'accrochage vertical (la taille en pixels entre les «cellules»).

Retour:

N/A

Exemple:

if keyboard_check(vk_space)
    {
    move_random(1,1);
    }

Cela déplacera l'instance vers une position aléatoire n'importe où dans la pièce lorsque la touche espace est enfoncée.
