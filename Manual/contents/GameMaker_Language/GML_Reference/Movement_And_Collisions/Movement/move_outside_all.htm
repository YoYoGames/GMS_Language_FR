move_outside_all

Avec cette fonction, vous pouvez indiquer à une instance de se déplacer hors d'une collision dans n'importe quelle direction et de n'importe quel nombre de pixels à chaque pas, avec une valeur de -1 ou 0 pour le maxdist qui est par défaut de 1000px, c'est-à-dire que GameMaker Studio 2 déplacera l'instance continuellement de 1000 pixels vers le haut jusqu'à ce qu'elle soit hors collision. vers le haut de 1000 pixels jusqu'à ce qu'elle soit sortie de la collision.

Syntaxe :

move_outside_all(dir, maxdist) ;

Arguments Description
dir La direction à prendre.
maxdist La distance maximale que l'objet peut parcourir (-1 ou 0 une valeur par défaut de 1000 pixels).

Les retours :

N/A

Exemple :

if place_meeting(x, y, all)
{
move_outside_all(90, 1) ;
}

Le code ci-dessus déplacera l'instance d'un pixel vers le haut à chaque pas de collision avec une autre instance.
