move_bounce_all

Avec cette fonction, vous pouvez commander à une instance de rebondir sur toutes les instances de la pièce, à la seule exception de celles qui n'ont pas d'index de sprite ou de masque assigné. Vous pouvez également lui demander d'utiliser le contrôle de collision précis lorsqu'il est activé, mais sachez que cela nécessite que toutes les instances aient des masques précis activés et que cela ralentira considérablement votre jeu lorsque de nombreuses instances sont impliquées, en raison de la quantité de traitement à effectuer. Ceci devrait normalement aller dans l'événement step d'une mais peut également être utilisé de manière sélective dans un événement de collision, comme l'illustre l'exemple de code ci-dessous.

Syntaxe :

move_bounce_all( adv ) ;

Arguments Description
adv Activation (true) ou non (false) du rebondissement avancé.

Les retours :

N/A

Exemple :

if other.visible
{
move_bounce_all(false) ;
}

Le code ci-dessus serait placé dans un événement de collision avec un autre objet. Il vérifie d'abord si l'objet est visible et s'il l'est, il exécute l'action move_bounce_all(). Notez que dans ce cas, le rebond est sélectif et ne sera calculé que pour cette collision. sera calculé uniquement pour cette collision, plutôt que pour chaque instance, à chaque étape.
