move_towards_point

Cette fonction indique simplement à l'instance exécutant le code de se déplacer vers un point de consigne à une vitesse définie. Gardez à l'esprit que c'est à vous de lui dire quoi faire une fois qu'il y sera, car il ne s'arrêtera pas tout seul. Cela signifie que vous pouvez obtenir une instance "vibrante" lorsqu'elle se déplace (par exemple) de 3 pixels vers l'avant, dépasse la position cible x / y de 2, puis recule de 3 pixels, dépassant le point cible de un et ainsi de suite.. Il existe un certain nombre de façons d'éviter cela, par exemple en utilisant distance_to_point() pour voir à quelle distance de l'emplacement spécifié se trouve l'instance avant de se déplacer.

REMARQUE: cette fonction définit la vitesse de l'instance, donc même si vous arrêtez d'utiliser ce code, l'instance continuera à se déplacer dans la direction précédente, il est donc nécessaire de définir lavitesse de l'instance sur 0 si vous souhaitez qu'elle s'arrête. Consultez l'exemple de code ci-dessous.

Syntaxe:

move_towards_point( x, y, sp );

Argument Description
x La position x du point vers lequel se déplacer.
y La position y du point vers lequel se déplacer.
sp Vitesse de déplacement en pixels par pas.

Retour:

N/A

Exemple:

if point_distance(x, y, target.x, target.y) > 5
    {
    move_towards_point(target.x, target.y, 5);
    }
else speed = 0;

Le code ci-dessus déplacera l'instance vers la position de l'instance indexée dans la variable "cible" à une vitesse de 5 pixels par étape. Une fois qu'il atteint cette position, la vitesse est réglée sur 0.

Retour: Mouvement
