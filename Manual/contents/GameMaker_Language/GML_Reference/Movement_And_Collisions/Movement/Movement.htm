Mouvement

Dans tout jeu, le mouvement et la position sont d'une importance capitale. GameMaker Studio 2 dispose donc d'une sélection complète de fonctions pour faire face à toutes les situations. Les deux principales manières de déplacer une instance sont soit de définir la Les deux principales façons de déplacer une instance sont soit de définir la position réelle, soit de définir un vecteur vitesse/direction, et cela peut être fait en utilisant les variables d'instance intégrées ou en utilisant des fonctions de mouvement spécifiques. Ces deux options sont expliquées dans les sections ci-dessous :

Les variables suivantes sont toutes incluses "intégrées" dans chaque instance et peuvent être lues et modifiées pour changer différents comportements lorsque l'instance se déplace (elles affecteront les vecteurs de vitesse et de direction) :

  • direction
  • friction
  • gravité
  • direction_gravité
  • hspeed
  • vspeed
  • vitesse

En plus des variables mentionnées ci-dessus, vous pouvez également définir la position de l'instance directement en utilisant ce qui suit :

  • x
  • y
  • xprécédent
  • yprécédent
  • xstart
  • ystart

Les fonctions suivantes peuvent toutes être utilisées pour déplacer une instance d'une manière ou d'une autre, certaines affectant les vecteurs de vitesse/direction et d'autres affectant directement la position x/y réelle dans la pièce :

  • motion_add
  • motion_set
  • déplacement vers un point
  • move_bounce_all
  • move_bounce_solid
  • déplacer_contact_tout
  • déplacer_contact_solide
  • move_outside_all
  • déplacer_extérieur_solide
  • déplacer_aléatoire
  • move_snap
  • move_wrap
  • place_snapped
