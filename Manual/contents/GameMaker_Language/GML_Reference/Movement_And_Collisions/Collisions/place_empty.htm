place_empty

Vous pouvez utiliser cette fonction pour vérifier et voir si l'instance appelante entrerait en collision avec une autre instance d'un objet ou toutes les instances de votre jeu. Pour que cette collision soit résolue correctement, l'instance exécutant le code doit avoir un masque de collision valide (soit pour le sprite lui-même, soit via le mask_index ) et il n'enregistrera que les collisions avec les instances qui ont également un masque valide.

La fonction teste s'il n'y a pas de collisions si l'instance appelante est placée à une position spécifique, et vous pouvez fournir un argument facultatif pour affiner la vérification afin de vérifier uniquement si une position est exempte de collisions avec des instances du type donné. Notez que si aucun ID d'objet facultatif n'est fourni, la vérification sera effectuée par rapport à toutes les instances de la salle. La vérification des collisions sera soit précise, soit basée sur la boîte englobante de l'instance, selon le type de masque de collision sélectionné, mais pour que les collisions précises fonctionnent correctement, toutes les instances de la vérification doivent avoir des masques de collision précis.

Notez que les coordonnées x / y données seront étalées à l'entier le plus proche avant que la vérification ne soit effectuée.

Syntaxe:

place_empty(x, y, [object_id]);

Argument Description
x La position x à vérifier.
y La position y à vérifier.
[object_id] Optionnel! L'objet à vérifier.

Retour:

Boolean

Exemple:

if place_empty(mouse_x, mouse_y, obj_Enemy)
   {
   x = mouse_x;
   y = mouse_y;
   }

Le code ci-dessus vérifiera une collision avec toute autre instance de l'objet "obj_Enemy", comme si l'instance appelante devait être placée à la même position que la souris. S'il n'y a pas de collision détectée, alors l'instance a ses coordonnées x / y définies sur celles de la souris.
