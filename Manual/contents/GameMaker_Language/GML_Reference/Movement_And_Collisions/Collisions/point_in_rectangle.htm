point_dans_rectangle

Lorsque vous utilisez cette fonction, vous définissez une zone rectangulaire et GameMaker Studio 2 déterminera si le point donné se trouve dans ses limites ou non. Si le point se trouve à l'intérieur du rectangle défini, la fonction renvoie true (vrai). la fonction renvoie false.

Syntaxe :

point_in_rectangle(px, py, x1, y1, x2, y2) ;

Arguments Description
px La coordonnée x du point à vérifier.
py La coordonnée y du point à vérifier.
x1 La coordonnée x du côté gauche du rectangle à vérifier.
y1 La coordonnée y du côté supérieur du rectangle à vérifier.
x2 La coordonnée x du côté droit du rectangle à vérifier.
y2 La coordonnée y du côté inférieur du rectangle à vérifier.

Les retours :

Booléen

Exemple :

if point_in_rectangle(mouse_x, mouse_y, x -10, y - 10, x + 10, y + 10)
{
audio_play_sound(snd_click, 0, false) ;
}

Ce court code vérifie la position de la souris par rapport à la zone rectangulaire définie et joue un son si elle se trouve dans les limites.
