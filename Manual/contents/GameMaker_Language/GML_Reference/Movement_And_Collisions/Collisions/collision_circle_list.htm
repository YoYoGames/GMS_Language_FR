collision_circle_list

Cette fonction est la même que la collision_circle(), mais au lieu de détecter une seule instance en collision à la fois, il détectera plusieurs instances. Vous fournissez la position x / y du centre de la zone circulaire à vérifier avec le rayon et l'objet à vérifier, et vous pouvez définir la vérification pour qu'elle soit précise (dans ce cas, toutes les instances vérifiées devront avoir une collision précise. masks) et si la vérification doit inclure ou non l'instance appelante. Vous fournissez également une liste DS qui sera remplie avec l'unique id valeurs de toutes les instances de l'objet considérées comme en collision avec l'instance appelante, et vous avez la possibilité de classer la liste en fonction de la distance entre l'origine de la zone circulaire et l'origine des instances trouvées en collision. La fonction renvoie le nombre d'instances trouvées, ou 0 si aucune n'est trouvée.

Notez qu'au lieu d'un index d'objet, vous pouvez fournir le mot-clé instanceall, pour rechercher toutes les instances de la salle actuelle, qui peuvent inclure l'instance exécutant le code (en fonction de la notme argument).

Syntaxe:

collision_circle_list(x1, y1, rad, obj, prec, notme, list, ordered);

Argument Description
x1 La coordonnée x du centre du cercle à vérifier.
y1 La coordonnée y du centre du cercle à vérifier.
rad Le rayon (distance en pixels de son centre à son bord).
obj L'objet à vérifier par exemple les collisions.
prec Si la vérification est basée sur des collisions précises (true, qui est plus lent) ou sa boîte englobante en général (false, plus rapide).
notme Si l'instance appelante, le cas échéant, doit être exclue (true ) ou non (false ).
list Liste DS à utiliser pour stocker les ID des instances en collision.
ordered Si la liste doit être triée par distance (true ) ou non (false ).

Retour:

Int (The number of instances found to be in collision)

Exemple:

var _list = ds_list_create();
var _num = collision_circle_list(x, y, 100, obj_Enemy, false, true, _list, false);
if (_num > 0)
    {
    for (var i = 0; i < _num; ++i;)
        {
        instance_destroy(_list[| i]);
        }
    }
ds_list_destroy(_list);

Le code ci-dessus vérifiera une zone circulaire avec un rayon de 100 pixels autour de la position de l'instance appelante pour les collisions avec les instances de "obj_Enemy". S'il y a des collisions, la liste pré-créée est bouclée et chaque instance qui était dans la collision est détruite.
