font_get_texture

Cette fonction renvoie un pointeur spécial pour la page de texture de la police. Cette valeur peut ensuite être utilisée dans d'autres fonctions de dessin, en particulier dans le dessin général lors de l'utilisation de primitives comme ainsi que les Ombrage ainsi que les fonctions Vous pouvez obtenir plus d'informations sur la page de texture renvoyée en utilisant les différentes fonctions texture_ que vous trouverez ici.

Syntaxe :

font_get_texture(font) ;

Arguments Description
police L'index de la police à utiliser.

Les retours :

Pointeur (ID de la page de texture)

Exemple :

tex = font_get_texture(fnt_Main) ;

Le code ci-dessus obtiendra le pointeur de texture pour la police indexée comme "fnt_Main".
