font_exists

Cette fonction renvoie si une police avec l'index spécifié existe ou non. Vous pouvez vérifier les index de polices tels que définis dans le navigateur de ressources ou les polices qui ont été ajoutées à l'aide de fonctions telles que font_add().

Syntaxe:

font_exists(ind);

Argument Description
ind Index de la police à vérifier.

Retour:

Boolean

Exemple:

if font_exists(fnt_Main)
   {
   draw_set_font(fnt_Main);
   }

Cela définira la police de dessin active sur fnt_Main s'il existe.

Retour: Polices
