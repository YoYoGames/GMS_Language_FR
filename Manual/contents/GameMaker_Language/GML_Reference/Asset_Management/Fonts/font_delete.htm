font_delete

Avec cette fonction, vous pouvez supprimer une ressource de police du jeu. Il s'agit d'une suppression permanente et les vestiaires ou le redémarrage du jeu ne ramèneront pas la police supprimée. Pour cela, le joueur devra quitter le jeu et redémarrer de cette façon, alors faites attention lorsque vous utilisez cette fonction. En général, il n'est nécessaire que pour libérer de la mémoire qui a été utilisée par une police ajoutée au jeu via les fonctions font_add() ou font_add_sprite().

Syntaxe:

font_delete(ind);

Argument Description
ind L'index de la police à supprimer.

Retour:

N/A

Exemple:

font_delete(global.Font);

Le code ci-dessus supprimera la police indexée dans la variable globale "Font".

Retour: Polices
