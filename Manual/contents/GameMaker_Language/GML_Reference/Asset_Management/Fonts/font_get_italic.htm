font_get_italic

Avec cette fonction, vous pouvez vérifier n'importe quelle ressource de police pour voir si elle a le drapeau italique ou non. Si c'est le cas, la fonction retournera true, sinon il reviendra false.

Syntaxe:

font_get_italic(ind);

Argument Description
ind Index de la police à vérifier.

Retour:

Boolean

Exemple:

if font_get_italic(fnt_Main)
   {
   draw_set_font(fnt_Main);
   }

Cela définira la police de dessin active sur fnt_Main s'il est mis en italique dans ses propriétés de police.
