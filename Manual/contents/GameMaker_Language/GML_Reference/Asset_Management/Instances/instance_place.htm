instance_place

Avec cette fonction, vous pouvez vérifier une position pour une collision avec une autre instance ou toutes les instances d'un objet en utilisant le masque de collision de l'instance qui exécute le code pour la vérification. Lorsque vous l'utilisez, vous demandez effectivement à GameMaker Studio 2 de déplacer l'instance vers la nouvelle position, de rechercher une collision, de revenir en arrière et de vous dire si une collision a été trouvée ou non. Cela fonctionnera pour des collisions précises, mais uniquement si l'instance et l'objet à vérifier ont tous deux des masques de collision précis sélectionnés, sinon seules les collisions de boîte englobante sont appliquées. cette fonction renverra l'instance unique id de l'objet en collision, mais si cela n'est pas nécessaire, il est légèrement plus rapide d'utiliser la fonction place_meeting(). Cette fonction accepte également les mots-clés spéciaux all et other et renverra le mot-clé noone si aucune collision ne se produit, ou la valeur d'ID d'instance unique de l'instance trouvée si une collision se produit.

Notez que les coordonnées x / y données seront arrondies à l'entier le plus proche avant que la vérification ne soit effectuée, donc si ce n'est pas ce dont vous avez besoin (ou si vous avez utilisé un ancien GameMaker ), vous devriez floor les coordonnées dans le chèque:

instance_place_list(floor(x), floor(y), obj_Enemy, list, true).

Syntaxe:

instance_place(x, y, obj);

Argument Description
x La position x pour rechercher des instances.
y La position y pour rechercher des instances.
obj Objet pour lequel rechercher des instances.

Retour:

Real (either an instance ID value or the keyword noone)

Exemple:

var _inst = instance_place(x, y, obj_Enemy);
if _inst != noone
    {
    hp -= _inst.dmg;
    instance_destroy(_inst);
    }

Le code ci-dessus vérifiera une collision avec les instances de "obj_Enemy" et s'il y en a une, il réduira la variable "hp" de la quantité stockée dans la variable "dmg" de l'instance en collision, puis détruira l'instance en collision.

Retour: Instances
