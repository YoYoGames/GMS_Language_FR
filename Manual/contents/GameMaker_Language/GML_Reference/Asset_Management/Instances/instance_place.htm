instance_place

Cette fonction vous permet de vérifier la présence d'une collision entre une position et une autre instance ou toutes les instances d'un objet en utilisant le masque de collision de l'instance qui exécute le code de vérification. Lorsque vous utilisez cette fonction, vous demandez effectivement à GameMaker Studio 2 de déplacer l'instance vers la nouvelle position, de vérifier s'il y a une collision, de revenir en arrière et de vous dire si une collision a été trouvée ou non. Cela fonctionnera pour les collisions précises, mais seulement si l'instance et l'objet à vérifier ont tous deux des masques de collisions précises de collision précis, sinon seules les collisions de la boîte englobante sont appliquées. id de l'objet en collision, mais si cela n'est pas nécessaire, il est légèrement plus rapide d'utiliser la fonction fonction place_meeting(). Cette fonction accepte également les mots-clés spéciaux all et other et renvoie le mot-clé retournera le mot-clé noone si aucune collision ne se produit, ou la valeur d'ID d'instance unique de l'instance trouvée si une collision se produit.

Notez que les coordonnées x/y données seront arrondies à l'entier le plus proche avant que la vérification ne soit effectuée, donc si ce n'est pas ce dont vous avez besoin (ou si vous avez utilisé un ancien produit GameMaker ), vous devez plancher les coordonnées dans l'option vérification :

instance_place_list(floor(x), floor(y), obj_Enemy, list, true).

Syntaxe :

instance_place(x, y, obj) ;

Arguments Description
x La position x à vérifier pour les instances.
y La position y pour vérifier les instances.
obj. L'objet dont il faut rechercher les instances.

Les retours :

Real (soit une valeur d'ID d'instance, soit le mot clé noone)

Exemple :

var _inst = instance_place(x, y, obj_Enemy) ;
if _inst != noone
{
hp -= _inst.dmg ;
instance_destroy(_inst) ;
}

Le code ci-dessus vérifiera s'il y a une collision avec des instances de "obj_Enemy" et s'il y en a une, il réduira la variable "hp" de la quantité stockée dans la variable "dmg" de l'instance en collision, puis détruira l'instance en collision. collision.
