instance_create_layer

Avec cette fonction, vous pouvez créer une nouvelle instance de l'objet spécifié à n'importe quel point donné dans la pièce et sur le calque spécifié. La couche peut être identifiée à l'aide de la valeur de l'ID de couche (telle que renvoyée par la fonction layer_create() ) ou par le nom du calque (sous forme de chaîne, par exemple "instance_layer") tel que défini dans l' éditeur de salle.

Cette fonction renvoie le id de la nouvelle instance qui peut ensuite être stockée dans une variable ou utilisée pour accéder à cette instance. Notez que cette fonction appellera également l' événement Create de l'instance en cours de création avant de continuer avec le code ou les actions pour l'événement qui a appelé la fonction.

IMPORTANT! Il y a une profondeur de couche minimale et maximale de -16 000 à 16 000. Tout ce qui est placé sur un calque en dehors de cette plage ne sera pas dessiné bien que tous les événements continuent de fonctionner normalement.

Syntaxe:

instance_create_layer(x, y, layer_id, obj);

Argument Description
x La position x à laquelle l'objet sera créé
y La position y à laquelle l'objet sera créé
layer_id L'ID de la couche (ou le nom) auquel attribuer l'instance créée
obj L'index d'objet de l'objet pour créer une instance de

Retour:

Real (instance ID value)

Exemple:

var inst = instance_create_layer(x, y, "instance_layer", obj_Bullet);
with (inst)
    {
    speed = other.shoot_speed;
    direction = other.image_angle;
    }

Le code ci-dessus crée une nouvelle instance de l'objet "obj_Bullet" et stocke l'ID d'instance dans une variable. Cette variable est ensuite utilisée pour attribuer speed et direction à la nouvelle instance.
