instance_créer_couche

Cette fonction permet de créer une nouvelle instance de l'objet spécifié en tout point de la pièce et sur la couche spécifiée. La couche peut être identifiée en utilisant la valeur d'identification de la couche (telle que retournée par la fonction layer_create()) ou par le nom de la couche (sous forme de chaîne, par exemple "instance_layer") tel que défini dans l'éditeur de la pièce .

Cette fonction renvoie l'identifiant id de la nouvelle instance qui peut alors être stockée dans une variable ou utilisée pour accéder à cette instance. Notez que cette fonction appellera également l'événement Create de l'instance en cours de création avant de poursuivre le code ou les actions de l'événement qui a appelé la fonction.

IMPORTANT ! Il existe une profondeur de couche minimale et maximale de -16000 à 16000. Tout ce qui est placé sur une couche en dehors de cette plage ne sera pas dessiné, mais tous les événements continueront à se dérouler normalement.

Syntaxe :

instance_create_layer(x, y, layer_id, obj) ;

Arguments Description
x La position x à laquelle l'objet sera créé
y La position y à laquelle l'objet sera créé
identifiant de la couche L'ID (ou le nom) de la couche à laquelle l'instance créée doit être affectée.
obj. L'index de l'objet dont il faut créer une instance.

Les retours :

Real (valeur de l'ID de l'instance)

Exemple :

var inst = instance_create_layer(x, y, "instance_layer", obj_Bullet) ;
with (inst)
{
speed = other.shoot_speed ;
direction = other.image_angle ;
}

Le code ci-dessus crée une nouvelle instance de l'objet "obj_Bullet" et stocke l'identifiant de l'instance dans une variable. Cette variable est ensuite utilisée pour attribuer la vitesse et la direction à la nouvelle instance.
