instance_find

Toutes les instances ont un identifiant unique (id) qui peut être utilisé pour les modifier et les manipuler pendant le déroulement d'un jeu, mais vous ne savez pas toujours quel est l'identifiant d'une instance spécifique et cette fonction peut vous aider car vous pouvez l'utiliser pour parcourir toutes les instances afin de trouver ce dont vous avez besoin. Vous spécifiez l'objet dont vous voulez trouver l'instance et un nombre, et s'il y a une instance à cette position dans la liste d'instances, la fonction retourne l'id de cette instance, sinon elle renvoie le mot-clé spécial noone. Vous pouvez également utiliser le mot-clé all pour itérer à travers toutes les instances d'un room, ainsi qu'un objet parent pour itérer parmi toutes les instances qui font partie de cette hiérarchie parent/enfant, et vous pouvez même spécifier une instance (si vous avez son id) pour vérifier si elle existe réellement dans le room actuel. elle existe réellement dans la pièce actuelle. Veuillez noter qu'étant donné que les instances sont triées de manière arbitraire, il n'y a pas d'ordre spécifique à la manière dont les instances sont vérifiées par cette fonction, et toute instance peut se trouver dans n'importe quelle position.

La valeur maximale de "n" dans cette fonction serait de

  • Pour le mot clé all: nombre d'instances - 1
  • Pour un indice d'objet : numéro_instance(OBJ) - 1

Syntaxe :

instance_find(obj, n) ;

Arguments Description
obj. L'objet pour trouver la nième instance de
n Le numéro de l'instance à trouver.

Les retours :

Real (soit une valeur d'ID d'instance, soit le mot clé noone)

Exemple :

for (var i = 0 ; i < instance_number(obj_Enemy) ; ++i ;)
{
enemy[i] = instance_find(obj_Enemy,i) ;
}

Le code ci-dessus utilisera une boucle for pour itérer à travers toutes les instances de "obj_Enemy" et stockera leur id dans le tableau "enemy[]".
