instance_position

Avec cette fonction, vous pouvez vérifier une position pour une collision avec une autre instance ou toutes les instances d'un objet. Lorsque vous utilisez cette fonction, vous vérifiez un point unique dans la room pour une instance ou un objet. Cette vérification sera effectuée par rapport au bounding de l'instance ou du masque de l'instance si cette instance a des collisions précises vérifiées et retournera l'instance unique. id. Si vous n'avez pas besoin de l'identifiant de l'instance en collision, vous devrait envisager d'utiliser position_réunion() à la place. Cette fonction accepte également les mots-clésspéciaux tous et other et renverra le mot-clé noone si aucune collision ne se produit ou la valeur d'identification unique de l'instance trouvée si une collision se produit.

Syntaxe :

instance_position( x, y, obj ) ;

Arguments Description
x La position x à vérifier pour les instances.
y La position y pour vérifier les instances.
obj. L'objet dont il faut rechercher les instances.

Les retours :

Real (soit une valeur d'ID d'instance, soit le mot clé noone)

Exemple :

var inst ;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button) ;
if (inst != noone)
{
with (inst) image_index=1 ;
instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu) ;
}

Le code ci-dessus vérifiera s'il y a une collision avec une instance de "obj_Pause_Button" à la position de la souris, et s'il y en a une, il utilisera l'id retourné pour définir son image_index à une nouvelle valeur avant de créer une nouvelle instance de l'objet "obj_Menu".
