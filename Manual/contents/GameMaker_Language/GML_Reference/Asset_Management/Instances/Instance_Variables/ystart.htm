ystart

Cette variable stocke la position y initiale de l'instance lorsqu'elle est créée pour la première fois dans la pièce. Ce n'est pas une variable en lecture seule et peut être définie aussi bien que lue.

Syntaxe:

ystart;

Retour:

Real (single precision floating point value)

Exemple:

if place_meeting(x, y, obj_spike)
    {
    score -= 100;
    x = xstart;
    y = ystart;
    }

Le code ci-dessus vérifiera une collision avec une instance de "obj_spike" et s'il y en a une, il en déduit 100 du score et ramène l'instance à sa position de départ.
