gravity

gravity est l'une des variables "intégrées" de toutes les instances et, une fois définie, appliquera une force constante dans le gravity_direction de l'instance, influençant à la fois l'instance speed  et direction. Notez que la gravité est une force cumulative et accélérera l'objet si vous choisissez de ne pas plafonner la vitesse finale, et il est habituel de définir cette variable sur de petites valeurs décimales telles que 0,01. Si vous définissez le gravity à 0, aucune gravité ne sera appliquée à l'instance (c'est la valeur par défaut).

Syntaxe:

gravity;

Retour:

Real (single precision floating point value)

Exemple:

if !place_meeting(x, y + 1, obj_Ground)
    {
    gravity = 0.01;
    }
else
    {
    gravity = 0;
    }

Le code ci-dessus n'appliquera la gravité que si l'instance ne trouve aucune instance de "obj_Ground" en dessous.
