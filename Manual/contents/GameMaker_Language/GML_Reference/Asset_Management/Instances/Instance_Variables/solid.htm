solid

Une instance peut être marquée comme solide via les propriétés de l' objet dans l'éditeur d'objets ou en modifiant la valeur de cette variable intégrée. Si solid est réglé sur true puis, lorsqu'une collision est détectée, l'instance en collision est automatiquement renvoyée à la position à laquelle elle se trouvait dans l'étape précédente à la collision (puis tout code ou action est exécuté dans l'événement de collision). S'il est réglé sur false, tout positionnement doit être traité lors de l'événement de collision.

Syntaxe:

solid;

Retour:

Boolean

Exemple:

if other.solid = true
    {
    x = xprevious;
    y = yprevious;
    }

Le code ci-dessus vérifiera le drapeau plein de l'instance "autre" en cas de collision et s'il est défini sur true ramener cette instance à sa position précédente.
