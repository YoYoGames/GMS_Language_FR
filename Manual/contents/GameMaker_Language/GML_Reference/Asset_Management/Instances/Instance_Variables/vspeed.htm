vspeed

vspeed est l'une des propriétés "intégrées" de toutes les instances et définit la vitesse de déplacement vertical (le long de l'axe y) de l'instance en pixels par pas. Donc, un vspeed de 3 signifie 3 pixels de mouvement vers le bas (+ y) à chaque pas, et un vspeed de -3 signifierait 3 pixels de mouvement vers le haut (-y) à chaque pas.

Notez que si vous définissez le speed et / ou direction variables puis les vspeed La valeur sera mise à jour automatiquement pour refléter ces changements, et de même, en changeant le vspeed la valeur affectera également le speed et direction valeurs. Notez également quegravity,gravity_direction etfriction peuvent tous modifier la valeur de cette variable lorsqu'ils sont utilisés dans vos jeux.

Syntaxe:

vspeed;

Retour:

Real (single precision floating point value)

Exemple:

if keyboard_check(vk_up) vspeed = -5;
if keyboard_check(vk_down) vspeed = 5;

Le code ci-dessus changera la vitesse verticale en fonction des touches enfoncées.
