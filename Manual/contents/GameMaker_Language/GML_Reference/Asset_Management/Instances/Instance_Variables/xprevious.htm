xprevious

Cette variable intégrée renvoie la position x précédente de l'instance. Cette variable sera définie juste avant le début de l' événement begin step, mais elle peut également être définie via le code à tout moment, ce qui signifie que vous pouvez lui donner votre propre valeur personnalisée si cela est nécessaire.

Syntaxe:

xprevious;

Retour:

Real (single precision floating point value)

Exemple:

if x != xprevious || y != yprevious
    {
    moved = true;
    }

Le code ci-dessus vérifie s'il y a une différence entre le xprevious et yprevious valeurs et les valeurs x et y actuelles. S'il y en a, il définit une variable sur true.
