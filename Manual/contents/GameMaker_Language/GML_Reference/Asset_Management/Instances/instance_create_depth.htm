instance_création_profondeur

Avec cette fonction, vous pouvez créer une nouvelle instance de l'objet spécifié en tout point de la pièce et à la profondeur spécifiée. La profondeur peut être n'importe quelle valeur, où plus la profondeur est faible, plus les objets seront dessinés "près" de la caméra et plus la profondeur est élevée, plus ils seront éloignés. plus la profondeur est élevée, plus l'objet est éloigné de la caméra. Ainsi, une instance située à une profondeur de -200 sera dessinée par rapport à une instance située à une profondeur de +300 (par exemple).

Notez que cette fonction créera en fait un calque de room pour l'instance, puisque toutes les instances doivent se trouver sur un calque de la room, mais comme il s'agit d'un calque géré (c'est-à-dire un calque que vous n'avez pas créé par le biais du code ou dans la room, mais un calque créé automatiquement par GameMaker Studio 2), vous ne pourrez pas y accéder. GameMaker Studio 2 l'a créé automatiquement), vous ne pouvez pas y accéder et la variable calque renverra la variable d'instance -1.

IMPORTANT ! Il existe une profondeur de couche minimale et maximale de -16000 à 16000. Tout ce qui est placé sur une profondeur en dehors de cette plage ne sera pas dessiné, mais tous les événements se dérouleront normalement.

Cette fonction renvoie l'identifiant id de la nouvelle instance qui peut alors être stockée dans une variable ou utilisée pour accéder à cette instance. Notez que cette fonction appellera également l'événement Create de l'instance en cours de création avant de poursuivre le code ou les actions de l'événement qui a appelé la fonction.

Syntaxe :

instance_create_depth(x, y, depth, obj) ;

Arguments Description
x La position x à laquelle l'objet sera créé
y La position y à laquelle l'objet sera créé
profondeur La profondeur à laquelle affecter l'instance créée
obj. L'index de l'objet dont il faut créer une instance.

Les retours :

Real (valeur de l'ID de l'instance)

Exemple :

var inst = instance_create_depth(x, y, -10000, obj_Bullet) ;
with (inst)
{
speed = other.shoot_speed ;
direction = other.image_angle ;
}

Le code ci-dessus crée une nouvelle instance de l'objet "obj_Bullet" et stocke l'identifiant de l'instance dans une variable. Cette variable est ensuite utilisée pour attribuer la vitesse et la direction à la nouvelle instance.
