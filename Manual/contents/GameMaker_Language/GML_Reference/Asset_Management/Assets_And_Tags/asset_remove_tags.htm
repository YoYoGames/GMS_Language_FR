supprimer_balises_de_l'actif

Avec cette fonction, vous pouvez supprimer une ou plusieurs chaînes de balises de n'importe quel poste du navigateur de postes. Vous fournissez soit le nom du bien (sous forme de chaîne), soit son index, ainsi qu'une seule chaîne de balises ou un tableau où chaque élément est une seule chaîne de balises. Si vous fournissez une valeur d'index d'asset, vous devrez fournir l'argument facultatif de type d'asset (une constante), car des assets de types différents peuvent avoir le même index, même s'ils ne peuvent pas avoir le même nom. Les types d'actifs disponibles sont répertoriés dans le tableau ci-dessous :

Constant Description
objet_actif Le nom donné fait référence à un objet.
actif_sprite Le nom donné fait référence à un sprite.
actif_son Le prénom fait référence à un son.
salle des actifs Le prénom fait référence à une pièce.
piliers_de_l'actif Le nom donné fait référence à un ensemble de carreaux.
chemin_de_l'actif Le nom donné fait référence à un chemin.
script_de_l'actif Le nom donné fait référence à un script.
asset_font Le nom donné fait référence à une police de caractères.
chronologie des actifs Le prénom fait référence à une ligne de temps.
asset_shader Le nom donné fait référence à un shader.
courbe d'animation de l'actif Le nom donné fait référence à une courbe d'animation.
séquence_d'actifs Le nom donné fait référence à une Séquence.

Si la fonction réussit à supprimer le(s) tag(s), elle renvoie true, sinon elle renvoie false.

Syntaxe :

asset_remove_tags(nom_ou_index, tags, [asset_type]) ;

Arguments Description
nom_ou_index Le nom du poste (une chaîne de caractères) ou sa valeur d'index (un nombre entier).
tags Une seule chaîne de tags d'actifs ou un tableau avec plusieurs tags d'actifs.
[type d'actif] FACULTATIF ! Le type de bien dont il faut supprimer les étiquettes, utilisé uniquement lorsqu'un index est fourni pour le premier argument.

Les retours :

Booléen

Exemple :

var _a = array_create(3) ;
_a[0] = "enemy" ;
_a[1] = "all_levels" ;
_a[2] = "boss" ;
asset_remove_tags(obj_Enemy_Boss_Parent, _a, asset_object) ;

Le code ci-dessus va créer un tableau de balises et les supprimer de l'objet donné.
