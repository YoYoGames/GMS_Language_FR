timeline_position

Cette variable contient la position actuelle (moment) à laquelle se trouve actuellement une ligne de temps. Vous pouvez modifier cette valeur pour sauter des parties de la ligne de temps, pour répéter des parties ou pour recommencer la ligne de temps depuis le début.

Syntaxe:

timeline_position;

Retour:

Real (single precision floating point value)

Exemple:

if !timeline_running
    {
    timeline_position = 0;
    timeline_running = true;
    }

Le code ci-dessus vérifiera si l'instance exécute une ligne de temps, et si ce n'est pas le cas, il réinitialise la ligne de temps assignée pour démarrer au premier moment, puis la démarre.
