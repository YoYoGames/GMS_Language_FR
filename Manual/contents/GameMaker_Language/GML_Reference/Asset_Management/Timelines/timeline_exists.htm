chronologie_existe

Avec cette fonction, vous pouvez vérifier et voir si une ligne de temps existe (renvoie true) ou non (renvoie false). Ceci est particulièrement utile lors de la création dynamique de lignes de temps à l'aide de la fonction timeline_add() fonction, mais vous devez noter que si vous recherchez une variable non initialisée (qui serait autrement assignée à l'index d'une ligne de temps), une erreur sera lancée.

Syntaxe :

timeline_exists(ind) ;

Arguments Description
ind L'indice de la ligne de temps à vérifier.

Les retours :

Booléen

Exemple :

if timeline_exists(global.tl)
{
timeline_delete(global.tl) ;
}

Le code ci-dessus vérifie si une ligne de temps existe et est indexée dans la variable globale "tl" et si c'est le cas, il supprime cette ligne de temps.
