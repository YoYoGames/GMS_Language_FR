timeline_moment_clear

Cette fonction permet d'effacer de tous les codes et actions un moment précis de toute ligne de temps préalablement définie.

Syntaxe :

timeline_moment_clear(ind, step) ;

Arguments Description
ind L'indice de la ligne de temps à effacer.
étape Le moment de dégager.

Les retours :

N/A

Exemple :

timeline_moment_clear(global.tl, room_speed * 30) ;

Le code ci-dessus effacera le moment spécifié de la ligne de temps indexée par la variable "global.tl".
