timeline_loop

Cette variable retournera si la ligne de temps est en boucle (vrai) ou non (faux). Vous pouvez changer cette variable pour activer ou désactiver la boucle et cela fonctionne avec une vitesse de ligne de temps négative (si la position de la ligne de temps passe en dessous de 0, elle recommencera au dernier moment défini).

Syntaxe:

timeline_loop;

Retour:

Boolean

Exemple:

if !timeline_loop
    {
    timeline_loop = true;
    }

Le code ci-dessus vérifie que la ligne de temps actuellement affectée à une instance va boucler ou non et si ce n'est pas le cas, il la définit sur true.
