chronologie_fonctionnement

Cette variable contient l'état actuel de la ligne de temps assignée et renvoie true si elle est en cours d'exécution et false si elle ne l'est pas. Vous pouvez également définir cette variable sur true ou false pour Il convient de noter qu'une ligne de temps arrêtée n' est pas réinitialisée, et donc la relancer ultérieurement la fera repartir à partir du moment exact où elle a été arrêtée.

Syntaxe :

la chronologie des opérations ;

Les retours :

Booléen

Exemple :

if !timeline_running
{
timeline_position = 0 ;
timeline_running = true ;
}

Le code ci-dessus vérifiera si l'instance exécute une ligne de temps, et si ce n'est pas le cas, il réinitialisera la ligne de temps assignée pour qu'elle démarre au premier moment, puis la lancera.
