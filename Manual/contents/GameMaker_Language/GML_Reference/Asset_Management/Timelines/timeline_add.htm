timeline_add

Avec cette fonction, vous pouvez ajouter une nouvelle ligne de temps (vide) dans votre jeu. la fonction renvoie l'index de cette nouvelle ligne de temps qui doit être stockée dans une variable pour être utilisée dans tous les autres appels de fonction qui impliquent cette nouvelle ligne de temps. Vous devez également vous assurer d'utiliser la fonction timeline_delete() chaque fois que vous ne souhaitez plus utiliser la chronologie (comme à la fin de la salle) afin d'éviter d'éventuelles fuites de mémoire qui pourraient ralentir ou planter votre jeu. Pour ajouter des moments à une chronologie créée de cette manière, consultez la fonctiontimeline_moment_add_script().

Syntaxe:

timeline_add();

Retour:

Real

Exemple:

global.tl = timeline_add();

Le code ci-dessus crée une nouvelle ligne de temps et stocke son index dans la variable "global.tl".
