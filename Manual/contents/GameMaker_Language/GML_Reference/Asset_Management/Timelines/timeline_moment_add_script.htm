timeline_moment_add_script

Avec cette fonction, vous pouvez ajouter dynamiquement une fonction de script aux Timelines à n'importe quel «moment» donné dans cette ligne de temps, où un «moment» est l'équivalent d'un tick (ou pas) de jeu. De cette façon, vous pouvez créer une nouvelle chronologie en utilisant le timeline_add() fonction et ajoutez différents comportements à tout moment, ou modifiez simplement une ressource chronologique créée précédemment avec de nouveaux comportements. Notez que la fonction ne peut pas nécessiter d'arguments supplémentaires lors de l'utilisation de cette fonction, et si vous l'utilisez pour modifier un élément de la chronologie présent dans le navigateur de ressources, toutes les instances qui utilisent cette chronologie seront affectées et la modification durera jusqu'à la fin de le jeu (appeler game_restart() ne réinitialisera pas non plus la modification).

Syntaxe:

timeline_moment_add_script(ind, step, script);

Argument Description
ind L'index de la chronologie à laquelle ajouter un moment.
step Le moment (étape) auquel ajouter.
script L'index de la fonction de script à ajouter dans l'instant.

Retour:

N/A

Exemple:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
    {
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
    }

Le code ci-dessus créera une nouvelle ligne de temps et stockera son index dans la variable "global.tl". Il ajoutera ensuite trois scripts à la chronologie, choisis au hasard, à une minute d'intervalle.
