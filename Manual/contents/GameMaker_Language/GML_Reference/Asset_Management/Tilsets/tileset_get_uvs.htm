tileset_get_uvs

Cette fonction renvoie un tableau avec les coordonnées UV et d'autres données pour la texture du jeu de tuiles donné sur la page de texture. La fonction renvoie un tableau avec les 8 éléments suivants:

  • [0] = gauche
  • [1] = haut
  • [2] = droite
  • [3] = bas
  • [4] = quantité de pixels que le compilateur d'éléments a supprimés du côté gauche de l'ensemble de tuiles
  • [5] = quantité de pixels que le compilateur d'éléments a supprimés du côté supérieur du jeu de tuiles
  • [6] = pourcentage normalisé de données de pixels par rapport à la largeur de l'ensemble de tuiles d'origine qui a été enregistrée sur la page de texture
  • [7] = pourcentage normalisé de données de pixels par rapport à la hauteur du jeu de mosaïques d'origine qui a été enregistrée sur la page de texture

Ce tableau peut ensuite être utilisé dans d'autres fonctions de dessin, en particulier lorsque vous travaillez en 3D ou que vous utilisez les fonctions primitives 2D, ainsi que lorsque vous travaillez avec les fonctions Shader

Syntaxe:

tileset_get_uvs(tileset);

Argument Description
tileset L'index du jeu de tuiles à utiliser.

Retour

Array (1D, 8 elements)

Exemple:

var tex = tileset_get_uvs(tl_Grass);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

Le code ci-dessus stockera les coordonnées UV pour le jeu de tuiles donné dans un tableau local, puis affectera les valeurs aux variables d'instance.

Retour: Ensembles de carreaux
