audio_emitter_position

Cette fonction vous permet de modifier la position d'un émetteur audio dans l'espace audio 3D. La position affectera le son de différentes manières en fonction de la position de l'auditeur dans l'espace audio. aussi (la position par défaut est (0, 0, 0). Voir audio_listener_position() pour plus d'informations). Ainsi, par exemple, si la position de l'émetteur est définie sur (100, 0, 0) et que l'auditeur actuel se trouve à (200, 0, 0), la position de l'émetteur sera différente. l'auditeur actuel se trouve à (200, 0, 0), l'audio diffusé par l'émetteur apparaîtra à gauche du champ audio. L'image ci-dessous montre une représentation visuelle des émetteurs et de leur position relative par rapport à l'auditeur :

Syntaxe :

audio_emitter_position(emitter, x, y, z) ;

Arguments Description
émetteur L'indice de l'émetteur à modifier.
x La position x.
y La position y.
z La position z.

Les retours :

N/A

Exemple :

si vitesse > 0
{
audio_emitter_position(s_emit, x, y, 0) ;
}

Le code ci-dessus vérifie si la vitesse de l'instance est supérieure à 0 et si c'est le cas, il met à jour l'émetteur audio indexé dans la variable "s_emit" à la position x/y actuelle.
