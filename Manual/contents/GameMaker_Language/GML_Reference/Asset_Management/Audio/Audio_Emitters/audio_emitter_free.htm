audio_emitter_free

Avec cette fonction, vous pouvez supprimer l'émetteur donné de la mémoire. Cela devrait toujours être fait chaque fois que l'émetteur ne sera plus utilisé dans la salle ou le jeu, c'est-à-dire: dans l' événement Destroy de l'instance ou dans l' événement de fin de salle, sinon vous risquez de vous retrouver avec une fuite de mémoire qui ralentira. vers le bas et finalement planter votre jeu.

Syntaxe:

audio_emitter_free(emitter);

Argument Description
emitter L'indice de l'émetteur à libérer.

Retour:

N/A

Exemple:

if lives = 0
   {
   audio_emitter_free(s_emit);
   room_goto(rm_Menu);
   }

Le code ci-dessus vérifie la valeur de la variable globale "lives" et s'il retourne 0, il détruira l'émetteur indexé dans la variable "s_emit" puis se rendra dans la pièce indexée dans la variable "rm_Menu".

Retour: Émetteurs audio
