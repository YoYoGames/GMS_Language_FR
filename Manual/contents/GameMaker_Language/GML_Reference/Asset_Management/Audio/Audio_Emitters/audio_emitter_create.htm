audio_emitter_create

Cette fonction crée un nouvel émetteur audio et renvoie son index. Cet index doit être stocké dans une variable pour toutes les autres fonctions liées à cet émetteur, puis lorsqu'il n'est plus nécessaire, il doit être supprimé de la mémoire à l'aide de la fonction audio_emitter_free() pour éviter les fuites de mémoire qui pourraient éventuellement planter votre jeu.

Syntaxe:

audio_emitter_create();

Retour:

Integer (emitter ID) 

Exemple:

s_emit = audio_emitter_create();

Le code ci-dessus crée un nouvel émetteur audio et assigne son index à la variable "s_emit".
