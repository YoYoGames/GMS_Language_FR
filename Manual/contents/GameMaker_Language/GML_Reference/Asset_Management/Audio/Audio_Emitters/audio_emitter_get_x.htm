audio_emitter_get_x

Cette fonction renvoie la position x actuelle de l'émetteur audio donné.

Syntaxe :

audio_emitter_get_x(emitter) ;

Arguments Description
émetteur L'indice de l'émetteur à utiliser.

Les retours :

Real

Exemple :

if audio_emitter_get_x(emitter_player) != x
{
audio_emitter_position(emitter_player, x, y, 0) ;
}

Le code ci-dessus vérifie la position x actuelle d'un émetteur donné et si elle n'est pas égale à la position x de l'instance, elle est fixée à la position de l'instance.
