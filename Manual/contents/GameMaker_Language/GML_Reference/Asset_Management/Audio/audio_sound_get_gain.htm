audio_sound_get_gain

Cette fonction renvoie la valeur de gain actuelle pour le son donné. Le son peut être soit un son référencé à partir d'un index pour un son individuel en cours de lecture qui a été stocké dans une variable lors de l'utilisation des fonctions audio_play_sound() ou audio_play_sound_at() ou un son réel provenant de l'Asset Browser. Le gain est généralement calculé comme une valeur comprise entre 0 et 1, mais sur certaines plates-formes, il est possible d'avoir un gain supérieur à 1. 1 est considérée comme le "plein volume" et toute valeur supérieure peut introduire un écrêtage audio.

Syntaxe :

audio_sound_get_gain(index) ;

Arguments Description
indice L'indice du son dont on veut obtenir le gain.

Les retours :

Real

Exemple :

if audio_sound_get_gain(snd_Music) != 1
{
audio_sound_gain(snd_Music, 1, 0) ;
}

Le code ci-dessus changera le gain de l'audio joué à partir du son indexé comme "snd_Music" si son gain n'est pas égal à 1.
