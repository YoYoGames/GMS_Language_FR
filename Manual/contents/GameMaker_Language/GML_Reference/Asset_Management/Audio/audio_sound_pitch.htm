audio_sound_pitch

Cette fonction peut être utilisée pour modifier la hauteur d'un son donné. Le son peut être soit un son référencé à partir d'un index pour un son individuel en cours de lecture qui a été stocké dans une variable lors de l'utilisation de la fonction audio_play_sound() ou audio_play_sound_at() ou un son réel provenant de l'explorateur d'objets. S'il s'agit de l'index d'un son en cours de lecture, seule cette instance sera modifiée, mais lorsqu'on utilise un actif sonore du navigateur d'actifs, toutes les autres instances de ce son seront modifiées. toutes les autres instances de ce son qui sont jouées seront modifiées.

L'argument pitch est un multiplicateur de hauteur, en ce sens que la valeur d'entrée multiplie la hauteur actuelle par cette quantité, de sorte que la valeur par défaut de 1 ne modifie pas la hauteur, tandis qu'une valeur inférieure à 1 abaisse la hauteur et supérieure à 1 l'élève. augmentera la hauteur de son. Il est préférable d'utiliser de petits incréments pour cette fonction car toute valeur inférieure à 0 ou supérieure à 5 peut ne pas être audible de toute façon. Il convient de noter que le changement de hauteur total autorisé est limité à (1/256) - 256 octaves, donc toute valeur supérieure ou inférieure à cette valeur ne sera pas enregistrée. n'est pas enregistrée.

REMARQUE : La valeur fixée ci-dessus est celle que GameMaker Studio 2 tente de fixer pour la gamme, mais cette valeur n'est pas garantie sur toutes les plates-formes cibles. iOS, par exemple, est fixé à (1/256) - 8, il se peut donc que vous deviez expérimenter sur chaque plate-forme cible et disposer de différentes versions d'une ressource sonore, chacune étant pré-décalée, si vous avez besoin de valeurs d'octave supérieures ou inférieures.

Syntaxe :

audio_sound_pitch(index, pitch) ;

Arguments Description
indice L'indice du son à modifier.
terrain Le multiplicateur de pas (par défaut 1).

Les retours :

N/A

Exemple :

var s_engine = audio_play_sound(snd_CarEngine, 10, false) ;
switch (gear)
{
case 1 : audio_sound_pitch(s_engine, 0.8) ; break ;
case 2 : audio_sound_pitch(s_engine, 0.9) ; break ;
case 3 : audio_sound_pitch(s_engine, 0.95) ; break ;
case 4 : audio_sound_pitch(s_engine, 1) ; break ;
case 5 : audio_sound_pitch(s_engine, 1.2) ; break ;
}

Le code ci-dessus changera la hauteur de l'audio joué à partir du son indexé dans la variable "s_engine" en fonction de la valeur de la variable "gear".
