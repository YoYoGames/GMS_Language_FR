audio_listener_set_position

Cette fonction vous permet de modifier la position d'un auditeur donné dans l'espace audio 3D. L'index par défaut de l'écouteur est 0, mais vous pouvez utiliser la fonction audio_get_listener_info() pour obtenir les différents indices disponibles pour la plate-forme cible. L'image d'exemple ci-dessous montre la position par défaut de l'écouteur dans l'espace audio :

Comme vous pouvez le constater, la position par défaut est (0, 0, 0) mais vous utiliserez normalement cette fonction pour déplacer l'écouteur avec l'objet joueur dans votre jeu et ainsi changer la façon dont l'audio créé par les émetteurs est entendu par le joueur, par exemple, dans l'image ci-dessous d'un jeu de haut en bas, l'instance du joueur définit l'écouteur qui fera en sorte que l'audio des différents émetteurs "change" à mesure que le joueur se déplace dans le niveau: :. l'audio des différents émetteurs à "changer" au fur et à mesure que le joueur se déplace dans le niveau :

Syntaxe :

audio_listener_set_position(index, x, y, z) ;

Arguments Description
indice L'écouteur pour lequel on veut obtenir les données (par défaut 0).
x La position x de l'écouteur (par défaut 0).
y La position y de l'écouteur (par défaut 0).
z La position z de l'écouteur (par défaut 0).

Les retours :

N/A

Exemple :

var _vmat = camera_get_view_mat(view_camera[0]) ;
audio_listener_set_position(global.Player_Listener, _vmat[0], _vmat[1], _vmat[2]) ;
audio_listener_set_orientation(info[ ? "index"], _vmat[3], _vmat[4], _vmat[5], _vmat[6], _vmat[7], _vmat[8]) ;

Le code ci-dessus récupère la matrice de vue pour la vue de la caméra [0] et l'utilise ensuite pour définir la position et l'orientation de l'écouteur audio pour l'écouteur dont l'ID est stocké dans la variable globale "Player_Listener".
