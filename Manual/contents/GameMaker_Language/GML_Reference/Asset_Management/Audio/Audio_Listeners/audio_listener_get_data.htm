audio_listener_get_data

Cette fonction va créer une carte DS et la remplir avec les valeurs de position, de vélocité et d'orientation pour l'écouteur donné. L'index par défaut de l'écouteur est 0, mais vous pouvez utiliser la fonction audio_get_listener_info() pour obtenir les différents indices disponibles pour la plateforme cible. Si vous fournissez un index d'écouteur incorrect, la fonction renverra -1.

REMARQUE : Vous êtes responsable de la destruction de la carte DS retournée en utilisant la fonction appropriée.

La carte DS contiendra les clés suivantes :

  • " x" - La position x de l'auditeur
  • "y" - La position y de l'auditeur.
  • "z" - La position z de l'auditeur.
  • "vx" - La vitesse de l'auditeur sur l'axe x.
  • "vy" - La vitesse sur l'axe y de l'auditeur.
  • "vz" - La vitesse de l'axe z de l'auditeur.
  • "Lookat_x" - La composante x du vecteur de visée de l'écouteur.
  • "Lookat_y" - La composante y du vecteur de visée de l'écouteur.
  • "Lookat_z" - La composante z du vecteur de visée de l'écouteur.
  • "up_x"- La composante x du vecteur haut de l'écouteur.
  • "up_y"- La composante y du vecteur haut de l'écouteur.
  • "up_z"- La composante z du vecteur haut de l'auditeur.

Syntaxe :

audio_listener_get_data(index) ;

Arguments Description
indice L'écouteur pour lequel on veut obtenir les données (par défaut 0).

Les retours :

ds_map

Exemple :

var num = audio_get_listener_count() ;
for(var i = 0 ; i < num ; ++i ;)
{
var info = audio_get_listener_info(i) ;
var data = audio_listener_get_data(info[ ? "index"]) ;
if data[ ? "x"] != 0
{
audio_listener_set_position(info[ ? "index"], 0, 0, 0) ;
}
ds_map_destroy(info) ;
ds_map_destroy(data) ;
}

Le code ci-dessus vérifie le nombre d'écouteurs disponibles, puis passe en boucle à travers eux et si leur position x est différente de 0, il définit leur position à 0, 0, 0.
