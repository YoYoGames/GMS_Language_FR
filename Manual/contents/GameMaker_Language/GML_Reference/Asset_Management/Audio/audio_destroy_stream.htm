flux audio_destroy_stream

Si vous avez précédemment créé un flux audio à partir d'un fichier en utilisant la fonction audio_create_stream() et que vous n'avez plus besoin de ce son, vous pouvez appeler cette fonction pour le supprimer de la mémoire. Tout autre appel à l'index du son après sa destruction donnera lieu à une erreur. Il convient de noter que cette fonction libère le flux, mais sur la plate-forme cible, cela peut ne pas apparaître dans un gestionnaire de mémoire. Cela est dû au fait que GameMaker Studio 2 met en commun afin d'éviter les surcharges d'allocation de mémoire, et la mémoire restera donc allouée jusqu'à ce qu'elle soit requise pour autre chose ou réutilisée pour un nouveau flux.

REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5.

Syntaxe :

audio_destroy_stream(nom du fichier) ;

Arguments Description
nom de fichier Le fichier à partir duquel le flux audio sera diffusé.

Les retours :

NA

Exemple :

audio_destroy_stream(snd) ;

Le code ci-dessus supprime de la mémoire le son indexé dans la variable "snd".
