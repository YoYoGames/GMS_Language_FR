audio_pause_all

Avec cette fonction, vous pouvez mettre en pause tous les sons en cours de lecture.

Syntaxe:

audio_pause_all();

Retour:

N/A

Exemple:

if keyboard_check_pressed(ord("P"))
   {
   global.Pause = !global.Pause;
   if global.Pause
      {
      audio_pause_all();
      }
   else
      {
      audio_resume_all();
      }
   }

Le code ci-dessus vérifie une pression sur la touche du clavier "P" et s'il en détecte une, il définit la variable globale "Pause" sur true ou false, puis suspend tous les sons ou redémarre tous les sons précédemment mis en pause.
