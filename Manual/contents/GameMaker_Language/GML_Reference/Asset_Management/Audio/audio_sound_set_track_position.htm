audio_sound_set_track_position

Cette fonction définit la position (en secondes) dans le fichier audio à partir de laquelle le son doit être lu. Le son peut être soit une seule instance d'un son (l'index des sons individuels en cours de lecture peut être stocké dans une variable lors de l'utilisation du audio_play_sound() ou audio_play_sound_at() fonctions) ou un atout solide. S'il s'agit d'un identifiant de son unique que vous utilisez, la position passera immédiatement à la nouvelle heure donnée, et s'il s'agit d'un actif du navigateur d'actifs, toutes les autres lectures du son donné commenceront à la nouvelle heure. La fonction ignorera toute valeur donnée qui est plus longue que la longueur du son donnée pour la valeur «index».

Syntaxe:

audio_sound_set_track_position(index, time);

Argument Description
index L'indice du son à changer.
time Le temps (en secondes) pour définir le point de départ.

Retour:

N/A

Exemple:

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

Le code ci-dessus lit un son, puis utilise la valeur ID renvoyée pour définir la position de départ du son sur 32 secondes.
