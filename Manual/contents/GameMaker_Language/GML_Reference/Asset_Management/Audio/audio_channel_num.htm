numéro_canal_audio

Cette fonction vous permet de définir le nombre de canaux audio disponibles pour la lecture de sons dans GameMaker Studio 2. Cela signifie essentiellement que vous indiquez le nombre de sons simultanés qui peuvent être joués à un moment donné. Si le nombre de sons dépasse cette valeur, les sons de moindre priorité sont arrêtés afin de libérer un canal pour les sons de priorité supérieure. Vous pouvez utiliser cette fonction pour optimiser votre jeu pour les périphériques car plus le nombre de canaux audio est faible, meilleures sont les performances, mais gardez à l'esprit que cela peut également entraîner une coupure notable de certains sons si beaucoup sont joués en même temps.

Syntaxe :

audio_channel_num(num) ;

Arguments Description
num Nombre de canaux audio disponibles (128 par défaut).

Les retours :

N/A

Exemple :

switch (os_browser)
{
cas browser_not_a_browser :
switch (os_type)
{
cas os_windows :
cas os_macos :
audio_channel_num(200) ;
break ;
défaut :
audio_channel_num(64) ;
break ;
}
break ;
par défaut :
audio_channel_num(16) ;
break ;
}

Le code ci-dessus vérifie la plate-forme sur laquelle le jeu est exécuté et modifie le nombre de canaux sonores disponibles afin d'augmenter les performances.
