audio_sound_get_listener_mask

Cette fonction renvoie les données de masque de bits qui définissent les écouteurs audio à partir desquels un son doit être joué. Consultez la section sur les écouteurs audio pour plus d'informations.

Syntaxe :

audio_sound_get_listener_mask(soundID) ;

Arguments Description
soundID L'ID unique du son dont on veut obtenir le masque.

Les retours :

Real

Exemple :

var snd = audio_play_sound(snd_PlayerDead, 10, false) ;
if audio_sound_get_listener_mask(snd) != global.PlayerMask
{
audio_sound_set_listener_mask(snd, global.PlayerMask) ;
}

Le code ci-dessus joue un son, puis vérifie les données du masque de l'auditeur pour le son, et si elles ne sont pas les mêmes que celles stockées dans une variable globale, il définit le ou les auditeurs à partir desquels jouer en utilisant les données du masque stockées dans la variable globale.
