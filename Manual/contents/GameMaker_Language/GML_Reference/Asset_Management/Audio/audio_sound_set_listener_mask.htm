audio_sound_set_listener_mask

Cette fonction peut être utilisée pour définir le masque binaire d'un son afin qu'il ne soit lu qu'à partir des auditeurs spécifiés. Vous pouvez créer un masque de bits en utilisant le audio_get_listener_info() puis en utilisant le bitwise or ("|") pour créer un masque personnalisé avec les écouteurs dont vous avez besoin pour la lecture du son, puis appliquez ce masque personnalisé au son après l'avoir lu. Ce masque remplacera le masque par défaut ou celui que vous avez peut-être défini à l'aide de la fonction audio_set_listener_mask().

REMARQUE: Cette fonction ne peut être utilisée sur un son qu'après avoir été configuré pour jouer et nécessite le soundID renvoyé par la fonction utilisée pour lire le son.

Syntaxe:

audio_sound_set_listener_mask(soundID, mask);

Argument Description
soundID L'identifiant unique du son pour définir le masque de
mask Les données de masque de bits à définir pour le son

Retour:

N/A

Exemple:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

Le code ci-dessus lit un son puis définit le ou les auditeurs à jouer à partir des données de masque stockées dans une variable globale.
