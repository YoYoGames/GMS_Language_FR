audio_group_unload

Cette fonction va décharger en mémoire tous les sons qui sont marqués comme appartenant au groupe audio donné. La fonction renvoie true (vrai) si le déchargement est initié et false (faux) si l'ID du groupe est invalide ou s'il a déjà été marqué pour le déchargement. groupe est invalide, ou s'il a déjà été marqué pour le déchargement. Notez que tout son en cours de lecture lorsque cette fonction est appelée sera arrêté.

Syntaxe :

audio_group_unload(groupID) ;

Arguments Description
groupID L'index du groupe audio à décharger (tel que défini dans la fenêtre Groupes audio).

Les retours :

Booléen

Exemple :

if audio_group_is_loaded(audiogroup_level1)
{
audio_group_unloaded(audiogroup_level1) ;
}

Le code ci-dessus vérifie si un groupe audio a été chargé et si oui, il le décharge.
