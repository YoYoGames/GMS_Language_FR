audio_group_load_progress

Cette fonction vérifie la progression du chargement d'un groupe audio et renvoie une valeur (approximative) entre 0 et 100.

Syntaxe:

audio_group_load_progress(groupID);

Argument Description
groupID L'index du groupe audio à vérifier (tel que défini dans la fenêtre Groupes audio )

Retour:

Real

Exemple:

if load
   {
   var pc = audio_group_load_progress(audiogroup_level1);
   draw_text(32, 32, "Loading: " + string(pc));
   }

Le code ci-dessus vérifie une variable et si elle retourne true puis un texte sera dessiné montrant la progression de l'audio en cours de chargement.

Retour: Groupes audio
