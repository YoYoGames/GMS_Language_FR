audio_sound_is_playable

Cette fonction peut être utilisée pour vérifier si l'indice sonore donné peut être lu actuellement. Cette fonction est nécessaire en raison des différentes façons dont la lecture du son en flux continu ou non continu est gérée sur la plate-forme cible HTML5. retournera true si le son peut être joué et false si ce n'est pas le cas. Notez que sur toutes les autres plateformes autres que HTML5, la fonction retournera toujours true...

Syntaxe :

audio_sound_is_playable(index) ;

Arguments Description
indice L'index du son à vérifier.

Les retours :

Real

Exemple :

if audio_sound_get_pitch(snd_Explode) != 1
{
audio_sound_pitch(snd_Explode, 1) ;
}

Le code ci-dessus changera la hauteur de l'audio joué à partir du son indexé comme "snd_Explode" si sa hauteur n'est pas égale à 1.
