audio_create_sync_group

Crée un groupe de synchronisation et renvoie une valeur d'identification unique pour celui-ci qui doit ensuite être utilisée dans tous les autres appels de fonction audio pour ce groupe. Si vous voulez que le groupe fasse une boucle, passez true, sinon passer false, mais notez que si vous voulez qu'ils soient mis en boucle, toutes les pistes ajoutées plus tard doivent avoir la même longueur. Notez que lorsque vous créez un groupe de synchronisation, vous devrez libérer la mémoire et les sons qui lui sont associés lorsqu'il n'est pas utilisé en utilisant le audio_destroy_sync_group() fonction - par exemple, dans les événements Room End ou Destroy.

REMARQUE: cette fonctionnalité n'est pas disponible pour la plate-forme cible HTML5.

Syntaxe:

audio_create_sync_group(loop);

Argument Description
loop Si les pistes du groupe doivent boucler (true ) ou non (false ).

Retour:

index

Exemple:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

Ce qui précède crée un nouveau groupe de synchronisation et affecte l'index du groupe à la variable "sg". Quatre sons sont ensuite ajoutés au groupe, le gain de trois d'entre eux étant réglé sur 0. Enfin, le groupe de synchronisation est joué.
