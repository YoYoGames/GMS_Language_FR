audio_sync_group_get_track_pos

Cette fonction renvoie la position de lecture actuelle du groupe de synchronisation donné. L'index du groupe est la valeur renvoyée lorsque vous avez créé le groupe à l'aide de la fonction audio_create_sync_group()et la valeur de retour est le temps en secondes pendant lequel les pistes ont été lues.

REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5.

Syntaxe :

audio_sync_group_get_track_pos(group_index) ;

Arguments Description
groupe_index L'indice du groupe pour obtenir la position de.

Les retours :

Real

Exemple :

var real_secs = audio_sync_group_get_track_pos(sg) ;
var secs = real_secs mod 60 ;
var mins = string(real_secs div 60) ;
if (secs > 9)
{
secs = string(secs) ;
}
else
{
secs = "0" + string(secs) ;
}
draw_text(32, 32, "Time = " + mins + " :" + secs) ;

Le code ci-dessus récupère la position temporelle pour le groupe de synchronisation indexé dans la variable "sg", puis utilise cette valeur pour dessiner le temps joué sur l'écran.
