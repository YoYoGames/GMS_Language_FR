audio_resume_sync_group

Cette fonction reprend le groupe de synchronisation donné s'il est en cours de lecture et a été précédemment mis en pause (à l'aide de la fonction audio_pause_sync_group). L'index du groupe est la valeur retournée lorsque vous avez créé le groupe groupe à l'aide de la fonction audio_create_sync_group().

REMARQUE: cette fonctionnalité n'est pas disponible pour la plateforme cible HTML5.

Syntaxe :

audio_resume_sync_group(group_index) ;

Arguments Description
groupe_index Le groupe index à reprendre.

Les retours :

N/A

Exemple :

if (keyboard_check_pressed((ord) "P"))
{
global.Pause = !global.Pause
if global.Pause
{
audio_pause_sync_group(sg) ;
}
else
{
audio_resume_sync_group(sg) ;
}
}

Le code ci-dessus vérifie si la touche "P" est pressée, et si elle est détectée, il bascule la variable "global.Pause" puis la vérifie pour mettre en pause ou reprendre le groupe de synchronisation indexé dans la variable "sg".
