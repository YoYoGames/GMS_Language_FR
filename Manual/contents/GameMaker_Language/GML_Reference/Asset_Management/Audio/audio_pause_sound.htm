audio_pause_sound

Avec cette fonction, vous pouvez mettre en pause tout son en cours de lecture. Le son peut être soit une seule instance d'un son (l'index des sons individuels en cours de lecture peut être stocké dans une variable lors de l'utilisation du audio_play_sound() ou audio_play_sound_at() fonctions) ou un élément audio, auquel cas toutes les instances du son donné seront mises en pause.

Syntaxe:

audio_pause_sound(index);

Argument Description
index L'index du son à mettre en pause.

Retour:

N/A

Exemple:

if keyboard_check_pressed(ord("P"))
   {
   global.Pause = !global.Pause;
   if global.Pause
      {
      audio_pause_sound(snd_Waterfall);
      }
   else
      {
      audio_resume_sound(snd_Waterfall);
      }
   }

Le code ci-dessus vérifie une pression sur la touche du clavier "P" et s'il en détecte une, il définit la variable globale "Pause" sur true ou false et puis soit met en pause le son indexé dans la variable "snd_Waterfall", soit il reprend le son à partir de son état de pause.

Retour: Audio
