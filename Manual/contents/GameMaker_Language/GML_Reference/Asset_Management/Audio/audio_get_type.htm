audio_get_type

Lorsque vous ajoutez du son à GameMaker Studio 2, il peut être diffusé en continu ou en mémoire. Si vous avez besoin de savoir si un index de son donné correspond à de l'audio en streaming ou non, vous pouvez utiliser cette fonction qui renverra 1 pour un son diffusé en continu, 0 pour un son en mémoire, et -1 s'il y a une erreur ou si l'index ne pointe pas vers une ressource sonore valide.

Syntaxe :

audio_get_type(index) ;

Arguments Description
indice L'index du son à vérifier.

Les retours :

Real

Exemple :

type = audio_get_type(snd_Music_1) ;

Le code ci-dessus vérifie le type d'audio indexé dans la variable "snd_Music_1" et stocke la valeur retournée dans la variable "type".
