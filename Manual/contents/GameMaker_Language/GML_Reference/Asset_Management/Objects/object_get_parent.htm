objet_get_parent

Cette fonction vous donnera l'object_index de tout parent qui a été assigné à l'objet spécifié, ou retournera -100 pour montrer que l'objet n'a pas de parent assigné, ou -1 si l'objet vérifié n'existe pas. Pour plus d'informations sur les parents, consultez la section sur l'éditeur d'objets.

Syntaxe :

objet_get_parent(obj) ;

Arguments Description
obj. L'index de l'objet à vérifier

Les retours :

Real

Exemple :

par = object_get_parent(object_index) ;

Le code ci-dessus récupère le parent de l'index de l'objet pour l'instance qui exécute le code et stocke la valeur de retour dans la variable "par".
