event_type

Cette variable en lecture seule renvoie le type d'événement en cours d'exécution, qui peut être l'une des constantes suivantes:

  • ev_create
  • ev_destroy
  • ev_step
  • ev_alarm
  • ev_keyboard
  • ev_keypress
  • ev_keyrelease
  • ev_mouse
  • ev_collision
  • ev_other
  • ev_gesture
  • ev_draw


Pour une liste complète des constantes liées aux différents événements, voir event_perform(), et si vous avez besoin du numéro d'événement (c'est-à-dire le sous-événement), vous devriez vérifier le event_number.

Syntaxe:

event_type;

Retour:

Real

Exemple:

show_debug_message("Current Event = " + string(event_type));

Le code ci-dessus affichera un message de débogage avec le type d'événement en cours d'exécution.
