event_object

Cette variable en lecture seule renvoie l'index d'objet de l'instance qui exécute l'événement en cours de vérification.

Syntaxe:

event_object;

Retour:

Real

Exemple:

global.obj = event_object;

Le code ci-dessus stocke l'index d'objet de l'instance exécutant l'événement dans une variable globale.
