event_action (obsolète)

Cette variable en lecture seule renvoyait l'index de l'action en cours d'exécution, commençant à 0 sur les versions précédentes de GameMaker. Cependant, il s'agit désormais d'une variable obsolète dans GameMaker Studio 2. Il a été laissé pour le support hérité uniquement et retournera toujours 0 car toutes les actions sont concaténées ensemble pour améliorer la vitesse d'exécution.

Syntaxe:

event_action;

Retour:

Real

Exemple:

num = event_action;

Le code ci-dessus stocke le numéro de l'action en cours dans la variable "num".
