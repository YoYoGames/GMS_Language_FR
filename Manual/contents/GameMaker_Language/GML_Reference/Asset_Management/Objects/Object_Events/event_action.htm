event_action (obsolète)

Cette variable en lecture seule renvoyait l'indice de l'action en cours d'exécution, en commençant par 0 sur les versions précédentes de GameMaker. Toutefois, cette variable est désormais obsolète dans GameMaker Studio 2. Elle n'a été conservée qu'à des fins de support et renvoie toujours 0, car toutes les actions sont concaténées pour améliorer la vitesse d'exécution.

Syntaxe :

événement_action ;

Les retours :

Real

Exemple :

num = action_événement ;

Le code ci-dessus stocke le numéro de l'action en cours dans la variable "num".
