objet_get_mask

Cette fonction vous dira si l'objet que vous vérifiez a un index de masque ou non, et si c'est le cas, elle retournera l'index de ce masque, ou -1 si ce n'est pas le cas. Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Vous pouvez avoir un objet sans sans masque alors qu'une instance de ce même objet peut en avoir un et vice-versa, ou ils peuvent même avoir des masques différents. Vous pouvez définir l'indice de masque d'une instance individuelle en utilisant la variable mask_index instance.

Syntaxe :

objet_get_mask(obj) ;

Arguments Description
obj. L'index de l'objet à vérifier

Les retours :

réel

Exemple :

if (mask_index != object_get_mask(object_index))
{
mask_index = object_get_mask(object_index) ;
}

L'exemple ci-dessus vérifiera l'index de masque de l'instance par rapport au masque de l'object_index de l'instance. S'ils ne sont pas identiques, il attribuera à l'instance le même masque que celui de l'index objet.
