objet_existant

Cette fonction renvoie si un objet avec l'indice spécifié existe ou non. Notez que cette fonction vérifie si un objet est présent dans le navigateur d'actifs, et non dans la salle de jeu. Pour cela, vous devez utiliser la fonction instance_exists().

Syntaxe :

objet_existant(obj) ;

Arguments Description
obj. L'index de l'objet à vérifier.

Les retours :

Booléen

Exemple :

if object_exists(obj_temp)
{
with (obj_Temp) scr_add_event() ;
}

Le code ci-dessus vérifie la variable "obj_temp" pour s'assurer qu'elle stocke l'index d'un objet et ensuite, si c'est le cas, il fait exécuter un script à l'objet.
