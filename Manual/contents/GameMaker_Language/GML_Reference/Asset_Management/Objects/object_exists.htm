object_exists

Cette fonction renvoie si un objet avec l'index spécifié existe ou non. Notez que cela vérifie si un objet est présent dans le navigateur d'actifs, et pas réellement dans la salle de jeux. Pour cela, vous devez utiliser la fonction instance_exists().

Syntaxe:

object_exists(obj);

Argument Description
obj L'index de l'objet à vérifier.

Retour:

Boolean

Exemple:

if object_exists(obj_temp)
    {
    with (obj_Temp) scr_add_event();
    }

Le code ci-dessus vérifie la variable "obj_temp" pour s'assurer qu'elle stocke l'index d'un objet et ensuite, si c'est le cas, il fait exécuter un script à l'objet.
