objet_set_visible

Cette fonction permet de marquer un objet spécifique comme étant visible ou non (pour plus d'informations sur le drapeau "visible", voir la section sur l'éditeur d'objets ). Cela signifie que toutes les instances de cet objet qui sont créées après que "visible" a été modifié seront créées avec ce nouvel état, tandis que les instances qui sont déjà dans la pièce peuvent ne pas être affectées.

Veuillez noter qu'il ne s'agit pas d'une fonction d'instance ! Vous pouvez définir l'indicateur de visibilité des instances individuelles à l'aide de la variable visible et ainsi avoir dix instances toutes marquées comme visibles même si même si l'objet en question a le drapeau à false dans l'IDE, et même si vous changez le drapeau visible de l'objet à true en utilisant cette fonction, toutes les instances qui sont actuellement dans la pièce resteront comme elles étaient, et seulement les instances créées après l'appel de la fonction commenceront avec le drapeau visible à true. l'appel de la fonction commenceront avec l'indicateur visible à true.

Syntaxe :

objet_set_visible( index, vis ) ;

Arguments Description
indice L'indice de l'objet à modifier.
vis La nouvelle visibilité de l'objet (true=visible, false=invisible).

Les retours :

N/A

Exemple :

if !object_get_visible(obj_Block)
{
object_set_visible(obj_Block, true) ;
}

Le code ci-dessus vérifie le drapeau visible de l'objet "obj_Block" et s'il est faux, il est changé en drapeau vrai.
