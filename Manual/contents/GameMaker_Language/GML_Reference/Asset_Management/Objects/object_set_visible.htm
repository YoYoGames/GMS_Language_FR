object_set_visible

Avec cette fonction, vous pouvez marquer un objet spécifique comme étant visible ou non (pour plus d'informations sur l'indicateur "visible", voir la section sur l' éditeur d'objets ). Cela signifie que toutes les instances de cet objet créées après la modification de visible seront créées avec ce nouvel état, tandis que les instances qui se trouvent déjà dans la salle peuvent ne pas être affectées.

Veuillez noter qu'il ne s'agit pas d'une fonction d'instance! Vous pouvez définir l'indicateur visible d'instances individuelles à l'aide de la visible variable et ainsi avoir dix instances toutes marquées comme visibles même si l'objet en question a l'indicateur défini sur false dans l'EDI, et même si vous changez l'indicateur visible de l'objet en true à l'aide de cette fonction, toutes les instances qui se trouvent actuellement dans la salle restera tel quel, et seules les instances créées après l'appel de la fonction commenceront par visible marqué comme vrai.

Syntaxe:

object_set_visible( index, vis );

Argument Description
index L'index de l'objet à modifier.
vis La nouvelle visibilité de l'objet (vrai = visible, faux = invisible).

Retour:

N/A

Exemple:

if !object_get_visible(obj_Block)
    {
    object_set_visible(obj_Block, true);
    }

Le code ci-dessus vérifie l'indicateur visible de l'objet "obj_Block" et s'il est faux, il est modifié pour être marqué comme vrai.
