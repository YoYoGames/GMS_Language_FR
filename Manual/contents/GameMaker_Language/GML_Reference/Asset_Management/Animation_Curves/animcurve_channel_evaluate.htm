animcurve_channel_evaluate

Cette fonction peut être utilisée pour obtenir la valeur à un moment précis dans le temps d'une structure de canal. Vous fournissez le pointeur de structure pour le canal (tel que retourné par la fonction animcurve_get_channel(), ou tel que renvoyé dans la structure de courbe d'animation par la fonction animcurve_get()) et la position "x" (temps) le long du canal de la courbe à évaluer. Cette position doit être comprise entre 0 et 1, et la fonction retournera la valeur de la courbe à cette position, ou elle retournera 0 si la structure de canal fournie n'est pas valide.

Syntaxe :

animcurve_channel_evaluate(channel_struct, posx) ;

Arguments Description
structure_canal Le pointeur de structure pour le canal à évaluer.
posx La position dans le temps à contrôler (de 0 à 1).

Les retours :

Real

Exemple :

var _channel = animcurve_get_channel(ac_AlphaCurve, 0) ;
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000)) ;
image_alpha = _val ;

Le code ci-dessus récupère la structure de canal pour l'actif de courbe d'animation "ac_AlphaCurve". Il utilise ensuite la valeur d'évaluation renvoyée pour définir l'alpha de l'image pour l'instance.

Retour : Courbes d'animation
