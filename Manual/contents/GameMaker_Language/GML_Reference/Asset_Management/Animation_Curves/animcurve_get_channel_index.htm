animcurve_get_channel_index

Cette fonction renvoie la valeur de l'index pour un canal de courbe d'animation donné. Vous devez fournir l'ID de la courbe d'animation ou le struct, où l'ID de la courbe est le nom de la courbe d'animation tel qu'il a été défini dans l'Asset Browser, ou le pointeur du struct tel qu'il est renvoyé par la fonction par la fonction animcurve_create(). Vous fournissez ensuite le nom du canal, sous forme de chaîne, et la fonction renvoie la valeur d'index associée à ce canal. Notez que si la courbe ou le canal n'existe pas n'existe pas, vous obtiendrez une erreur.

Syntaxe :

animcurve_get_channel_index(curve_struct_or_id, channel_name) ;

Arguments Description
curve_struct_or_id L'ID ou le pointeur de structure de la courbe d'animation à cibler.
nom_du_canal Le nom du canal (une chaîne).

Les retours :

Structure

Exemple :

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex) ;
var _points = _channeldata.points ;
for (var i = 0 ; i < array_length(_points) ; ++i ;)
{
_points[i].value += 1 ;
}

Le code ci-dessus récupère la structure de canal pour le canal nommé "x_pos" dans l'actif courbe "ac_ButtonTween", puis boucle sur les points de la courbe du canal et ajoute un à leur valeur.
