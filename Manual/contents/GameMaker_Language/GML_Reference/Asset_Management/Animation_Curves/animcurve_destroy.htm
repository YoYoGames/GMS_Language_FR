animcurve_destroy

Cette fonction peut être utilisée pour détruire toutes les courbes d'animation qui ont été précédemment créées avec la fonction animcurve_create(). L'appel de cette fonction supprimera la courbe d'animation de la mémoire et nettoiera également tous les canaux ou points qu'elle contient (ceux-ci n'ont jamais besoin d'être nettoyés manuellement). Notez que vous ne pouvez pas détruire les courbes d'animation créées dans le navigateur de ressources, uniquement celles créées lors de l' runtime.

Syntaxe:

animcurve_destroy(curve_struct);

Argument Description
curve_struct Le pointeur vers la structure de la courbe à détruire.

Retour:

N/A

Exemple:

animcurve_destroy(my_curve);

Le code ci-dessus détruira la structure de la courbe d'animation (créée précédemment) indexée dans la variable "my_curve ".

Retour: Courbes d'animation
