chemin_get_y

Avec cette fonction, vous pouvez obtenir la coordonnée y d'une position sur un chemin donné. 0 est le début du chemin, 1 est la fin du chemin, et tout ce qui se trouve entre les deux équivaut à la distance normalisée sur la longueur du chemin. Cette valeur n'a pas besoin d'être un point de définition de la trajectoire, elle peut être n'importe où sur celle-ci.

Syntaxe :

path_get_y(ind, pos) ;

Arguments Description
indice L'index du chemin à vérifier.
pos Jusqu'où dans le chemin à vérifier. Entre 0 (début) et 1 (fin).

Les retours :

Real

Exemple :

y = path_get_y(pth_WallWalk, 0.5) ;

Ceci définira le y de l'instance appelante à la coordonnée y du point situé exactement à mi-chemin du chemin donné.
