path_speed

Vous pouvez utiliser cette fonction pour obtenir ou définir la vitesse d'un chemin après son démarrage à l'aide de la fonction path_start(). Vous pouvez utiliser des valeurs négatives pour indiquer que l'instance doit suivre le chemin en sens inverse.

Syntaxe:

path_speed;

Retour:

Real

Exemple:

path_speed = -1 + random(2);

Le code ci-dessus définira l'instance pour parcourir le chemin à une vitesse aléatoire comprise entre -1 et 1.

Retour: Variables de chemin
