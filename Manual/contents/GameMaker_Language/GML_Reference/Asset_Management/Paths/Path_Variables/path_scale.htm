path_scale

Cette valeur peut être utilisée pour obtenir ou pour définir l'échelle du chemin actuellement affecté à l'instance (tel que défini par la fonction path_start() ) avec une valeur par défaut de 1. Il s'agit d'une valeur scalaire, donc 1 est une échelle de 1: 1, tandis que la définir à 2, par exemple, sera le double de l'échelle et la régler à 0,5 équivaudrait à diviser par deux l'échelle.

Syntaxe:

path_scale;

Retour:

Real

Exemple:

path_scale = 1 + random(2);

Le code ci-dessus définira l'échelle du chemin actuel sur une taille aléatoire comprise entre 1 (sa taille d'origine) et 3 (3 fois sa taille d'origine).
