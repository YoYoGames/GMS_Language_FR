path_positionprevious

Cette variable peut être utilisée pour obtenir ou définir la position d'une instance le long de son chemin actuel dans l'étape précédente. C'est une valeur normalisée entre 0 et 1, c'est-à-dire que 0 est la position de début du chemin et 1 la position de fin. Elle est similaire à la fonction xprécédent et yprevious dans son fonctionnement, mais elle est spécifique aux chemins. Elle peut être utile pour des choses comme l'arrêt temporaire d'un suiveur de chemin si quelque chose se trouve sur son chemin (voir l'exemple de code ci-dessous).

Syntaxe :

path_positionprevious ;

Les retours :

Real

Exemple :

var _x = x + lengthdir_x(16, direction) ;
var _y = y + lengthdir_y(16, direction) ;
if collision_circle(xx, yy, 16, obj_Player, false,true)
{
path_position = path_positionprevious ;
}

Le code ci-dessus vérifie qu'il n'y a pas de collision avec l'objet "obj_Player" dans une zone située devant l'instance et, s'il y en a une, il replace l'instance à la position précédente qu'elle occupait sur le chemin actuel, comme assigné par la fonction path_start().
