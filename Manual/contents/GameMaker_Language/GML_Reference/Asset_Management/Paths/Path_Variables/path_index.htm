path_index

Toutes les ressources de GameMaker Studio 2 ont un numéro d'identification unique et le path_index est une variable en lecture seule qui contient la valeur d'index pour une ressource de chemin donnée qui a été affectée à une instance à l'aide de path_start() fonction. Si l'instance n'a pas de chemin assigné, la variable sera définie sur -1.

Syntaxe:

path_index;

Retour:

integer (Path ID)

Exemple:

if (path_index == -1)
    {
    path_start(pth_enemy3, 4, path_action_reverse, 0);
    }

Le code ci-dessus vérifie si un chemin a été attribué à l'instance, et sinon il démarre un nouveau chemin (en l'attribuant au path_index à la fois).
