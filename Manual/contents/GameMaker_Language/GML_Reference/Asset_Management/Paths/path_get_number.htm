path_get_number

Cette fonction peut être utilisée pour renvoyer le nombre de points sur un chemin.

Syntaxe:

path_get_number(index);

Argument Description
index L'index du chemin à vérifier.

Retour:

Real

Exemple:

if (path_get_number(pth_AI) > 1)
     {
    path_start( pth_AI, 4, 3, 0);
    }

Le code ci-dessus vérifie si un chemin contient plus d'un point et si c'est le cas, il démarre l'instance le long de ce chemin.
