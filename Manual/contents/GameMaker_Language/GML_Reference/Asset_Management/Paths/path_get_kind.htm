path_get_kind

Les trajectoires peuvent être lisses ou droites (une trajectoire lisse calcule une trajectoire courbe autour des points de définition, tandis qu'une trajectoire droite va simplement d'un point à un autre). Cette fonction peut être utilisée pour savoir si le chemin donné est lisse (true ) ou non (false ).

Syntaxe:

path_get_kind(index);

Argument Description
index L'index du chemin à vérifier.

Retour:

Boolean

Exemple:

path_kind = path_get_kind(pth_Patrol);

Cela définira la variable "path_kind" sur true ou false selon le type de chemin, l'index de chemin donné est.
