path_get_kind

Les chemins peuvent être lisses ou droits (un chemin lisse calcule une trajectoire courbe autour des points de définition, tandis qu'un chemin droit va simplement d'un point à un autre). Cette fonction peut être utilisée pour déterminer si un chemin donné est lisse(vrai) ou non(faux).

Syntaxe :

path_get_kind(index) ;

Arguments Description
indice L'index du chemin à vérifier.

Les retours :

Booléen

Exemple :

path_kind = path_get_kind(pth_Patrol) ;

Ceci définira la variable "path_kind" à true ou false selon le type de chemin que l'index de chemin donné est.
