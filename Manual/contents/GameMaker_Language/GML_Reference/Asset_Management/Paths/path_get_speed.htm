chemin_get_speed

Cette fonction renvoie le facteur de vitesse d'une position donnée sur un chemin. La position doit être une valeur entre 0 et 1 (vous pouvez utiliser path_positionpar exemple, si vous avez besoin de la position actuelle de l'instance), 0 étant la position de départ et 1 la position d'arrivée. La valeur renvoyée sera le facteur par lequel la valeur actuelle de path_speed est défini. Par exemple, sur un chemin avec seulement deux points, 0 et 1, où le facteur de vitesse du point 0 est 100 et le facteur de vitesse du point 1 est 0, la fonction path_get_speed(path, 0.5) retournera 50.

Syntaxe :

path_get_speed(ind, pos) ;

Arguments Description
indice L'index du chemin à vérifier.
pos Jusqu'à quel point du chemin il faut vérifier. Entre 0 (début) et 1 (fin), peut être décimal.

Les retours :

Real

Exemple :

vitesse = path_get_speed(pth_Patrol, 0.5) / 20 ;

Ceci fixera la vitesse de l'instance appelante au facteur de vitesse du point situé exactement à mi-chemin du chemin donné, divisé par 20. Ainsi, si le facteur de vitesse trouvé est le facteur par défaut 100, la vitesse de l'instance appelante sera fixée à 5.
