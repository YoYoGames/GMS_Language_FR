path_get_point_speed

Avec cette fonction, vous pouvez obtenir la vitesse du point (tel que défini dans l'éditeur de trajectoire ou lorsque vous ajoutez dynamiquement un point de trajectoire à l'aide de la fonction path_add_point()) exprimée en pourcentage. Ainsi, si vous avez un point de chemin défini à 50 dans l'éditeur de chemin, cette fonction renverra 50 lorsqu'elle sera utilisée.

Syntaxe :

path_get_point_speed(index, n) ;

Arguments Description
indice L'index du chemin à vérifier.
n L'identité de point à vérifier.

Les retours :

Real

Exemple :

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
var _px = path_get_point_x(pth_Patrol, 0) ;
var _py = path_get_point_x(pth_Patrol, 0) ;
path_change_point(pth_Patrol, 0, _px, _py, 100) ;
}

Le code ci-dessus vérifie le point "0" du chemin indexé par "pth_Patrol" pour voir si la vitesse n'est pas égale à 100. Si ce n'est pas le cas, il fixe la vitesse de ce point à 100.
