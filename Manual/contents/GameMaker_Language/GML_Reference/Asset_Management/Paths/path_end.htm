path_end

L'appel de cette fonction mettra fin au chemin actuel que suit l'instance, tel que défini lorsque la fonctionpath_start() a été appelé..

Syntaxe:

path_end();

Retour:

N/A

Exemple:

if place_meeting(x, y, obj_Blocker)
    {
    path_end();
    }

Le code ci-dessus terminera le chemin actuel si l'instance détecte une collision avec une instance de l'objet donné.
