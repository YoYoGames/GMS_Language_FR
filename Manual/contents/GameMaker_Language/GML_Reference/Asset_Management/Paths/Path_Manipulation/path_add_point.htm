path_add_point

Avec cette fonction, vous pouvez ajouter un point au chemin spécifié et définir son facteur de vitesse. Ce point est ajouté à la fin du chemin, quelle que soit la position du point, et le facteur est égal au pourcentage de path_speed à laquelle l'instance suivante se rend réellement lorsqu'elle atteint ce point du chemin. Si vous souhaitez placer un point de chemin à une autre position qui n'est pas la fin, vous devez utiliser path_insert_point().

Si elle est utilisée sur un chemin du navigateur d'actifs, notez que la fonction changera l'actif réel, et affectera donc de manière permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, vous devez utiliser une fonction telle que path_duplicate() pour créer d'abord une copie du chemin, puis appelez cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

Syntaxe:

path_add_point(index, x, y, speed);

Argument Description
index L'index du chemin auquel ajouter le point.
x La coordonnée x du nouveau point.
y La coordonnée y du nouveau point.
speed Le facteur de vitesse du point (la valeur par défaut est 100).

Retour:

N/A

Exemple:

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
    {
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
    }

Le code ci-dessus créera un chemin circulaire autour de la position x / y de l'instance exécutant le code.
