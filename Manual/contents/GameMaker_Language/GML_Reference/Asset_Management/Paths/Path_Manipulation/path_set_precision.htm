path_set_precision

Avec cette fonction, vous pouvez modifier la "fluidité" d'un tracé. Cette valeur doit être comprise entre 1 et 8, avec une valeur faible créant des arêtes plus droites avec des courbes plus nettes entre les points, tandis qu'une valeur plus élevée arrondira les points et rendra le tracé beaucoup plus «sinueux». Notez que cette fonction n'aura aucun effet visible si le chemin n'a pas été défini pour lisser dans l'éditeur de chemin ou en utilisant la fonction path_set_kind(). Cette fonction modifie l'actif du chemin réel, et affectera donc de manière permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, vous devez utiliser une fonction telle que path_duplicate() pour créer d'abord une copie du chemin, puis appelez cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

Syntaxe:

path_set_precision(index, prec);

Argument Description
index L'index du chemin à changer.
prec La précision du chemin. Doit être un entier compris entre 1 et 8.

Retour:

N/A

Exemple:

path_set_precision(pth_Patrol, 2);

Cela définira la précision du chemin donné à 2.

Retour: Manipulation de chemin
