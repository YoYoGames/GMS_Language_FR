path_set_precision

Avec cette fonction, vous pouvez changer la "douceur" d'un chemin. Cette valeur doit être comprise entre 1 et 8, une valeur faible créant des bords plus droits avec des courbes plus nettes entre les points, tandis qu'une valeur plus élevée arrondit les points et rend le chemin beaucoup plus "sinueux". beaucoup plus "sinueux". Notez que cette fonction n'aura aucun effet visible si le chemin n'a pas été réglé sur lisse dans l'éditeur de chemin ou en utilisant la fonction path_set_kind(). Cette fonction modifie l'actif réel du chemin, et affectera donc de façon permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, utilisez une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appelez cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

Syntaxe :

path_set_precision(index, prec) ;

Arguments Description
indice L'index du chemin à modifier.
préc La précision du chemin. Doit être un nombre entier entre 1 et 8.

Les retours :

N/A

Exemple :

path_set_precision(pth_Patrol, 2) ;

Ceci fixera la précision du chemin donné à 2.
