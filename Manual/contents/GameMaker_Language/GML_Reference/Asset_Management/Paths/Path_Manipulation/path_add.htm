path_add

Avec cette fonction, vous pouvez créer un chemin dans GameMaker Studio 2 sans utiliser l'éditeur de chemin. cette fonction retournera l'index du chemin qui doit être stocké dans une variable et utilisé comme référence pour ce chemin à partir de là. Veuillez noter que le chemin créé est vide c'est-à-dire qu'il n'a pas de points définis, vous devrez donc utiliser les autres fonctions disponibles pour ajouter des points au chemin ou utiliser des grilles MP pour générer le chemin. Une fois que vous avez fini d'utiliser le chemin, ou que vous souhaitez en créer un nouveau et stocker son index dans la même variable, vous devez d'abord supprimer l'ancien chemin avec path_delete pour éviter les fuites de mémoire qui peuvent éventuellement planter votre jeu.

Syntaxe:

path_add();

Retour:

Real

Exemple:

global.newpath = path_add();

Cela créera un nouveau chemin et attribuera son index à global.newpath.

Retour: Manipulation de chemin
