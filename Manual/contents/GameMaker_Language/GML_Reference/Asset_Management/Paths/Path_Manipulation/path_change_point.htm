point_de_changement_de_chemin

Avec cette fonction, vous pouvez modifier la position et/ou le facteur de vitesse de n'importe quel point précédemment défini pour un chemin (le chemin peut avoir été créé dans l'éditeur de chemin ou par le code en utilisant la fonction path_add()). Si elle est utilisée sur un dans le navigateur d'actifs, notez que la fonction modifiera l'actif réel et affectera de façon permanente la façon dont le chemin est utilisé par toutes les instances du jeu, depuis le moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, alors vous devriez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appeler cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler d'appeler path_delete() sur le poste lorsqu'il n'est plus nécessaire).

Syntaxe :

path_change_point(index, n, x, y, vitesse) ;

Arguments Description
indice L'index de la trajectoire pour changer un point.
n Le point de définition pour changer les attributs de.
x La nouvelle coordonnée x (par rapport à la trajectoire) du point.
y La nouvelle coordonnée y (par rapport à la trajectoire) du point.
vitesse Le nouveau facteur de vitesse du point.

Les retours :

N/A

Exemple :

for (var i = 0 ; i < path_get_number(path0) ; ++i ;)
{
var px = path_get_point_x(pth_Patrol, i) + 64 - random(128) ;
var py = path_get_point_y(pth_Patrol, i) + 64 - random(128) ;
path_change_point(pth_Patrol, i, px, py, 100) ;
}

Le code ci-dessus boucle sur tous les points du chemin indexé comme "path0" et repositionne tous les points à une position aléatoire dans une zone de 128x128 pixels.
