path_change_point

Avec cette fonction, vous pouvez modifier la position et / ou le facteur de vitesse de tout point précédemment défini pour un chemin (le chemin peut avoir été créé dans l'éditeur de chemin ou via le code en utilisant path_add() ). Si elle est utilisée sur un chemin du navigateur d'actifs, notez que la fonction changera l'actif réel, et affectera donc de manière permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, vous devez utiliser une fonction telle que path_duplicate() pour créer d'abord une copie du chemin, puis appelez cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

Syntaxe:

path_change_point(index, n, x, y, speed);

Argument Description
index L'index du chemin pour changer un point de.
n Le point déterminant pour modifier les attributs de.
x La nouvelle coordonnée x (relative au chemin) du point.
y La nouvelle coordonnée y (par rapport au chemin) du point.
speed Le nouveau facteur de vitesse du point.

Retour:

N/A

Exemple:

for (var i = 0; i < path_get_number(path0); ++i;)
    {
    var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
    var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
    path_change_point(pth_Patrol, i, px, py, 100);
    }

Le code ci-dessus parcourt tous les points du chemin indexés comme "path0" et repositionne tous les points à une position aléatoire dans une zone de 128x128 pixels.
