Manipulation des chemins

Dans GameMaker Studio 2, vous n'êtes pas seulement limité aux chemins que vous créez à l'aide de l'éditeur de chemins. Il existe un certain nombre de fonctions qui peuvent être utilisées pour manipuler et modifier ces chemins lors de l'exécution, ou même pour en créer de nouveaux, puis les modifier et les manipuler.

Les fonctions suivantes sont utilisées pour changer de chemin :

  • chemin_existant
  • chemin_add
  • chemin_add_point
  • point_de_changement_de_chemin
  • path_insert_point
  • path_delete_point
  • chemin_points_clairs
  • chemin_append
  • path_assign
  • chemin_suppression
  • chemin_duplicate
  • path_flip
  • chemin_miroir
  • chemin_inverse
  • chemin_rotation
  • path_rescale
  • path_set_closed
  • path_set_kind
  • path_set_precision
  • chemin_décalage
