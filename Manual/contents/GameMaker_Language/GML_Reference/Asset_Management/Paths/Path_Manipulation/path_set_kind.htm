path_set_kind

Cette fonction peut être utilisée pour définir le type de chemin que vous souhaitez que la ressource spécifiée soit. Cela peut être un chemin en ligne droite (défini sur 0) ou un chemin lissé (défini sur 1), auquel cas la précision du chemin doit également être prise en compte (la précision peut également être définie en utilisant path_set_precision() ). Cette fonction modifie la ressource de chemin réelle, et affectera donc de manière permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu.Si ce n'est pas ce dont vous avez besoin, vous devez utiliser une fonction comme path_duplicate() pour créer d'abord une copie du chemin, puis appelez cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

Syntaxe:

path_set_kind(index, val);

Argument Description
index L'index du chemin à changer.
val Le type de chemin, 0 pour droit ou 1 pour lisse.

Retour:

N/A

Exemple:

path_set_kind(pth_Patrol, true);

Cela définira le chemin donné comme un chemin "lisse".
