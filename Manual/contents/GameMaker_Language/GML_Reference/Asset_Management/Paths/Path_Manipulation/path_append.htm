chemin_append

Avec ce code, vous pouvez ajouter un chemin à un autre, les joignant effectivement ensemble, et pour que cela fonctionne, les deux chemins doivent exister auparavant. En général, il est préférable d'utiliser cette fonction sur les chemins créés à l'aide de path_add(), car si vous l'utilisez sur une ressource de chemin, elle affectera de façon permanente le chemin pour les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu.

NOTE: Ceci ne supprimera pas le chemin ajouté du jeu. Il est toujours là, mais il n'a plus de points et si vous ne souhaitez plus l'utiliser, vous devez le supprimer avec la fonction path_delete().

Syntaxe :

path_append(index, path) ;

Arguments Description
indice Le nouvel index du chemin.
chemin L'index du chemin qui sera ajouté à 'index'.

Les retours :

N/A

Exemple :

path_append(mypath, mppath) ;
path_delete(mppath) ;

Le code ci-dessus ajoute les données de chemin indexées dans la variable "mppath" au chemin indexé dans la variable "mypath", puis supprime "mppath" de la mémoire.
