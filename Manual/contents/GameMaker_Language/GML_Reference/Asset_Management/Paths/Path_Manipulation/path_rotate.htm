path_rotate

Vous pouvez utiliser cette fonction pour faire pivoter un chemin donné autour de son centre. N'oubliez pas que dans GameMaker Studio 2 (sauf si vous utilisez la physique), les angles sont calculés dans le sens inverse des aiguilles d'une montre, donc une rotation du chemin de 90 degrés ferait pivoter le chemin vers la gauche. Cette fonction modifie la ressource de chemin réelle et affectera ainsi de manière permanente la façon dont le chemin est utilisé par toutes les instances du jeu à partir du moment où la fonction est utilisée jusqu'à la fin du jeu. Si ce n'est pas ce dont vous avez besoin, vous devez utiliser une fonction telle que path_duplicate() pour créer d'abord une copie du chemin, puis appelez cette fonction sur l'actif dupliqué (n'oubliez pas d'appeler path_delete() sur l'actif lorsqu'il n'est plus nécessaire).

REMARQUE: une fois qu'un chemin a été attribué à une instance à l'aide dupath_start() fonction, vous pouvez utiliser la path_orientation variable pour modifier également la rotation de la trajectoire.

Syntaxe:

path_rotate(index, angle);

Argument Description
index L'index du chemin à retourner.
angle L'angle des degrés de rotation.

Retour:

N/A

Exemple:

path_rotate(pth_Patrol, 90);

Cela ferait pivoter les 90 degrés dans le sens inverse des aiguilles d'une montre.

Retour: Manipulation de chemin
