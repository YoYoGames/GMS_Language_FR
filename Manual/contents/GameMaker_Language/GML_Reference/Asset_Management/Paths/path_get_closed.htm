path_get_closed

Cette fonction peut être utilisée pour retourner si le chemin est marqué comme fermé (vrai) ou ouvert (faux), c'est-à-dire si le chemin est en boucle ou s'il a un début et une fin définitifs.

Syntaxe:

path_get_closed(index);

Argument Description
index L'index du chemin à vérifier.

Retour:

Boolean

Exemple:

state = path_get_closed(pth_Patrol);

Cela définira "état" sur soit true ou false en fonction de l'état fermé du chemin indexé dans "pth_Patrol".
