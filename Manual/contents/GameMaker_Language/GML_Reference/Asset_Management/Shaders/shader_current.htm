shader _current

Cette fonction retournera la valeur de l'ID d'index du shader actuellement utilisé pour le rendu, ou elle retournera -1 si aucun shader n'est utilisé.

Syntaxe:

shader_current();

Retour:

Real (ID value of the shader, or -1 if no shader)

Exemple:

if (shader_current() == -1)
    {
    shader_set(sh_warp)
    }

Le code ci-dessus vérifiera quel est le shader actuel et s'il renvoie -1 (aucun shader n'est utilisé), un shader est défini.
