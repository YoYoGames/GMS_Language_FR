shader_set_uniform_matrix_array

Cette fonction réinitialise la cible de dessin et doit être appelée lorsque vous ne souhaitez plus utiliser le shader actuel (défini à l'aide de la commande shader_set()).

Syntaxe :

shader_reset() ;

Les retours :

N/A

Exemple :

shader_set(shader_Glass) ;
draw_self() ;
shader_reset() ;

Le code ci-dessus va définir un shader top à utiliser pour le dessin, puis dessiner le sprite courant utilisé pour l'instance qui l'utilise.
