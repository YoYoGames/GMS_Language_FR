shader_enable_corner_id

Avec cette fonction, vous pouvez définir un état global pour tous les shaders utilisés où, lorsqu'il est activé, le shader "vole" 2 bits des valeurs de couleur d'entrée. Le premier est le bit inférieur de la valeur de couleur rouge, et le second est le bit inférieur de la valeur de couleur bleue. bit inférieur de la valeur de la couleur bleue. Ces valeurs peuvent ensuite être récupérées dans le shader pour déterminer de quel vertex il s'agit (c'est-à-dire de quel coin).

Syntaxe :

shader_enable_corner_id(enable) ;

Arguments Description
activer Activez(true) ou désactivez(false) cette fonction.

Les retours :

N/A

Exemple :

shader_enable_corner_id(true) ;

Le code ci-dessus permettra l'utilisation de bits de couleur pour l'identifiant de coin pour tous les shaders.
