shader_set_uniform_matrix

Avec cette fonction, vous pouvez définir la valeur (ou les valeurs) d'une constante de shader à la matrice de transformation actuelle (telle que définie en utilisant les fonctions de matrice). Vous devez auparavant avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform().

NOTE: Tous les uniformes doivent être définis après avoir appelé la fonction shader_set()et avant d'appeler shader_reset().

Syntaxe :

shader_set_uniform_matrix(handle) ;

Arguments Description
poignée Le handle de la constante du shader à définir.

Les retours :

N/A

Exemple :

shader_set(shader_Glass) ;
shader_matrix= shader_get_uniform(shader_glass, "u_vMatrix") ;
shader_set_uniform_matrix(shader_matrix) ;
draw_self() ;
shader_reset() ;

Le code ci-dessus obtiendra le gestionnaire de la constante du shader "u_vMatrix" puis définira cette constante à la matrice de transformation actuelle.
