shader _set_uniform_matrix_array

Avec cette fonction, vous pouvez définir une shader pour contenir un tableau de valeurs de matrice. Vous devez au préalable avoir obtenu le "handle" de la constante en utilisant la fonction shader_get_uniform(), et vous devrez avoir préalablement initialisé le tableau comme un tableau de valeurs à virgule flottante, où chaque groupe successif de 16 flottants est une matrice 4x4.

REMARQUE: tous les uniformes doivent être définis après avoir appelé la fonction shader _set () et avant d'appeler shader _reset ().

Syntaxe:

shader_set_uniform_matrix_array(handle, array);

Argument Description
handle La poignée de la shader à définir.
array Un tableau précédemment initialisé de valeurs à virgule flottante.

Retour:

N/A

Exemple:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_tint, "cMatrixArray");
shader_set_uniform_matrix_array(shader_params, matrix_array);
draw_self();
shader_reset();

Le code ci-dessus obtiendra le handle de la shader "cMatrixArray" puis définira cette constante sur le tableau donné.
