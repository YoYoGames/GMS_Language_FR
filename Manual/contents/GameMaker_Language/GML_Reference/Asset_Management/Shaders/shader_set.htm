shader _set

Avec cette fonction, vous pouvez définir la cible de dessin sur le shader donné et tout autre dessin sera fait à l'aide de celui-ci. Vous pouvez mettre fin à l' shader avec la fonction shader_reset().

REMARQUE: vous ne pouvez définir un shader dans l' événement Draw d'une instance.

Syntaxe:

shader_set(shader);

Argument Description
shader L'index du shader à utiliser.

Retour:

N/A

Exemple:

shader_set(shader_Glass);
draw_self();
shader_reset();

Le code ci-dessus définira un shader top à utiliser pour le dessin, puis dessinera le sprite actuel utilisé pour l'instance qui l'utilise.
