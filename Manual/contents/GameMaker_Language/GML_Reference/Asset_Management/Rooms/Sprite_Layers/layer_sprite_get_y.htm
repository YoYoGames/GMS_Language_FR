couche_sprite_get_y

Cette fonction peut être utilisée pour obtenir la position y de l'élément sprite dans la pièce. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), et la fonction renverra la valeur de la position y.

Syntaxe :

layer_sprite_get_y(sprite_element_id) ;

Arguments Description
sprite_element_id La valeur d'identification unique de l'élément de sprite à partir duquel les informations doivent être obtenues.

Les retours :

Real

Exemple :

var lay_id = layer_get_id("sprite_sky") ;
var spr_id = layer_sprite_get_id(lay_id, "Clouds") ;
if layer_sprite_get_y(spr_id) < 0
{
layer_sprite_y(spr_id, room_height) ;
}

Le code ci-dessus obtiendra l'ID du calque nommé "sprite_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément sprite sur ce calque. Cet ID est ensuite utilisé pour vérifier la position y de l'élément et s'il est inférieur à 0, l'élément du calque est déplacé vers une position y différente. est déplacé vers une position y différente.
