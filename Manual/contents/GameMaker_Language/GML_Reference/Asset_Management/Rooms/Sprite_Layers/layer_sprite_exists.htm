layer_sprite_exists

Vous pouvez utiliser cette fonction pour vérifier et voir si un élément de sprite existe sur un calque donné. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ) ou le nom du calque (sous forme de chaîne - cela aura un impact sur les performances) et l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ) et la fonction retournera soit true si l'élément existe, ou false si ça ne fait pas.

NOTE: Cette fonction fonctionne dans le cadre de la salle cible actuelle - par défaut la salle dans laquelle la fonction est appelée - qui peut être définie à l'aide de la fonction layer_set_target_room ().

Syntaxe:

layer_sprite_exists(layer_id, sprite_element_id)

Argument Description
layer_id La valeur d'ID unique du calque à cibler (ou le nom du calque sous forme de chaîne)
sprite_element_id La valeur d'ID unique de l'élément de sprite à vérifier

Retour:

Boolean

Exemple:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
    {
    layer_sprite_destroy(lay_id, global.TreesSprites);
    }

Le code ci-dessus vérifie la couche "Asset_trees" pour voir si l'élément de sprite donné existe et si c'est le cas, alors il est détruit (mais pas la couche).
