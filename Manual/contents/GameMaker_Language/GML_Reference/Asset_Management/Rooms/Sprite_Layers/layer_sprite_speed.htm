vitesse_sprite_couche

Cette fonction peut être utilisée pour définir la vitesse d'une image animée qui a été assignée à un élément sprite. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis vous définissez la vitesse de l'animation. Si l'IDE a défini la vitesse d'animation du sprite utilisé pour qu'elle soit calculée en tant que Frames Per Game Frame, la valeur de la vitesse donnée est un multiplicateur, avec une valeur de 1 à 2. est un multiplicateur, 1 étant la valeur par défaut, et si vous la fixez à 0,5, la vitesse d'animation sera réduite de moitié, tandis que si vous la fixez à 1,5, elle sera doublée. Toutefois, si l'IDE a défini la vitesse d'animation du sprite utilisé pour qu'elle soit calculée en tant que Frames Per Second (images par seconde), alors la valeur de la vitesse donnée est une valeur littérale d'images par seconde, donc en la réglant sur 5, par exemple, l'animation tournera en boucle de 5 images par seconde pendant le jeu. Si le sprite utilisé ne comporte pas de sous-images, cette fonction n'aura aucun effet. n'aura aucun effet.

Syntaxe :

layer_sprite_speed(sprite_element_id, speed) ;

Arguments Description
sprite_element_id La valeur d'identification unique de l'élément du sprite à modifier.
vitesse Le multiplicateur/valeur de vitesse à utiliser pour l'élément.

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Assets_water") ;
var spr_id = layer_sprite_get_id(lay_id, "gfc_water") ;
layer_sprite_speed(spr_id, 0.5) ;

Le code ci-dessus obtiendra l'ID du calque nommé "Assets_water" et l'utilisera ensuite pour obtenir l'ID de l'élément sprite donné sur ce calque. Cet ID est ensuite utilisé pour modifier la vitesse de l'image de l'élément.
