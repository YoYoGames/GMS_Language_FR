layer_sprite_index

Cette fonction peut être utilisée pour définir l'index de l'image d'un actif sprite qui possède plusieurs sous-images sur une couche. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez l'indice d'image à utiliser. Si vous définissez une valeur en dehors de la plage des sous-images, l'indice d'image tournera en boucle.

Syntaxe :

layer_sprite_index(sprite_element_id, image_index) ;

Arguments Description
sprite_element_id La valeur d'identification unique de l'élément de sprite à définir.
indice L'index de l'image à utiliser pour le sprite

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("sprite_trees") ;
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees") ;
layer_sprite_index(spr_id, 1) ;

Le code ci-dessus obtiendra l'ID du calque nommé "sprite_trees", puis l'utilisera pour obtenir l'ID de l'élément sprite donné sur ce calque. Cet ID est ensuite utilisé pour modifier l'index de l'image de l'élément.
