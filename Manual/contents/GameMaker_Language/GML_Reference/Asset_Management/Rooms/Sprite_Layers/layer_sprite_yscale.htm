couche_sprite_échelle

En utilisant cette fonction, vous pouvez changer si l'élément sprite donné sur un calque doit être mis à l'échelle le long de l'axe y ou non. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur de l'échelle. Une échelle de 1 indique qu'il n'y a pas de mise à l'échelle (1:1), les valeurs plus petites seront réduites (0,5, par exemple, correspondra à la moitié de la hauteur du sprite utilisé), les valeurs plus grandes seront mises à l'échelle, et les valeurs négatives reflèteront le sprite et le mettront à l'échelle, sauf si la valeur utilisée est exactement -1 (dans ce cas, le sprite utilisé est simplement reflété de haut en bas autour de sa position (0, 0) sans mise à l'échelle).

Syntaxe :

layer_sprite_yscale(sprite_element_id, yscale) ;

Arguments Description
sprite_element_id La valeur d'identification unique de l'élément du sprite à modifier.
yscale La valeur de l'échelle des y (1 par défaut)

Les retours :

N/A

Exemple :

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees") ;
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
layer_sprite_xscale(asset_sprite, 1) ;
layer_sprite_yscale(asset_sprite, 1) ;
}

Le code ci-dessus vérifiera l'élément sprite attribué au calque sur lequel se trouve l'instance qui exécute le code et s'il est mis à l'échelle dans l'une ou l'autre direction, il définira l'échelle de l'axe des x et l'échelle de l'axe des y à 1.
