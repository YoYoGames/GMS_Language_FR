changement de couche_sprite

En utilisant cette fonction, vous pouvez changer la ressource sprite assignée à un élément sprite donné sur une couche. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite en utilisant la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis fournissez une ressource d'élément sprite indexée qui sera modifiée. Notez que si le sprite possède des sous-images, il s'animera également (ceci peut être contrôlé à l'aide des fonctions layer_sprite_index() et layer_sprite_speed() ). Notez que vous pouvez attribuer une valeur de -1 comme nouvel indice de sprite et aucun sprite ne sera affiché, bien que l'élément existe toujours et puisse être modifié ultérieurement.

Syntaxe :

layer_sprite_change(sprite_element_id, sprite_index)

Arguments Description
sprite_element_id La valeur d'identification unique de l'élément du sprite à modifier.
index_spirit Le nouvel index de sprite à utiliser

Les retours :

N/A

Exemple :

var s = layer_sprite_get_id(layer, global.Asset_sprite) ;
if layer_sprite_get_sprite(s) != spr_Nighttime
{
layer_sprite_change(s, spr_nighttime) ;
}

Le code ci-dessus vérifie l'index du sprite de l'élément avec l'ID stocké dans la variable globale global .Asset_sprite et si ce n'est pas "spr_Nighttime", alors ce sprite est attribué à l'élément.
