layer_sprite_get_yscale

Cette fonction peut être utilisée pour obtenir le multiplicateur d'échelle actuel le long de l'axe y pour l'élément sprite. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), et la fonction renverra la valeur réelle pour l'échelle y, où 1 n'est pas de mise à l'échelle.

Syntaxe:

layer_sprite_get_yscale(sprite_element_id);

Argument Description
sprite_element_id La valeur d'ID unique de l'élément de sprite à partir duquel obtenir les informations

Retour:

Real

Exemple:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_yscale(spr_id) != 1
    {
    layer_sprite_yscale(spr_id, 1);
    }

Le code ci-dessus obtiendra l'ID de la couche nommée "sprite_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément sprite sur cette couche. Cet ID est ensuite utilisé pour vérifier la valeur yscale et s'il est autre chose que 1, il est mis à 1.
