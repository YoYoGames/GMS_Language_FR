angle_sprite_couche

En utilisant cette fonction, vous pouvez changer l'angle de l'élément sprite donné sur une couche. Vous donnez l'ID de l'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), puis définissez la valeur de l'angle, de 0 à 359 dans le sens inverse des aiguilles d'une montre, où 0 est à droite, 90 est en haut, 180 est à gauche et 270 est en bas. Si vous définissez une valeur supérieure à 360, celle-ci sera mise en boucle pour la ramener dans la plage 0 - 359.

Syntaxe :

layer_sprite_angle(sprite_element_id, angle) ;

Arguments Description
Identifiant de l'élément de sprite La valeur d'identification unique de l'élément du sprite à modifier.
angle L'angle du sprite (0 par défaut)

Les retours :

N/A

Exemple :

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees") ;
if layer_sprite_get_angle(asset_sprite) != 0
{
layer_sprite_angle(asset_sprite, 0) ;
}

Le code ci-dessus vérifiera l'élément sprite attribué au calque sur lequel se trouve l'instance qui exécute le code et si son angle est différent de 0, il le mettra à 0.
