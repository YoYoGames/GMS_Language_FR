layer_sprite_get_sprite

Cette fonction peut être utilisée pour obtenir l'index actuel de l'élément sprite. Vous donnez l'ID de l'élément sprite (que vous obtenez lorsque vous créez un élément sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id()), et la fonction renverra une valeur réelle qui représente l'indice du sprite affiché. Si l'élément n'a pas de sprite assigné, la fonction renverra -1.

Syntaxe :

layer_sprite_get_sprite(sprite_element_id) ;

Arguments Description
sprite_element_id La valeur d'identification unique de l'élément de sprite à partir duquel les informations doivent être obtenues.

Les retours :

Index du sprite (réel) ou -1

Exemple :

var lay_id = layer_get_id("sprite_sky") ;
var spr_id = layer_sprite_get_id(lay_id, "Clouds") ;
if layer_sprite_get_sprite(spr_id) != spr_Clouds
{
layer_sprite_change(spr_id, spr_Clouds) ;
}

Le code ci-dessus récupère l'ID du calque nommé "sprite_sky" et l'utilise pour obtenir l'ID de l'élément sprite sur ce calque. Cet ID est ensuite utilisé pour vérifier le sprite assigné à l'élément, en le réglant sur le sprite "spr_Clouds", s'il ne l'est pas déjà. s'il ne l'est pas déjà.
