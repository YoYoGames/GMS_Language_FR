layer_sprite_get_sprite

Cette fonction peut être utilisée pour obtenir l'index de sprite actuel de l'élément sprite. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), et la fonction renverra une valeur réelle qui représente l'index du sprite affiché. Si l'élément n'a pas de sprite assigné, la fonction retournera -1.

Syntaxe:

layer_sprite_get_sprite(sprite_element_id);

Argument Description
sprite_element_id La valeur d'ID unique de l'élément de sprite à partir duquel obtenir les informations

Retour:

Sprite Index (Real) or -1

Exemple:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_sprite(spr_id) != spr_Clouds
    {
    layer_sprite_change(spr_id, spr_Clouds);
    }

Le code ci-dessus obtiendra l'ID de la couche nommée "sprite_sky" et l'utilisera ensuite pour obtenir l'ID de l'élément sprite sur cette couche. Cet identifiant est ensuite utilisé pour vérifier le sprite affecté à l'élément, en le définissant sur le sprite "spr_Clouds" si ce n'est déjà fait.

Retour: Couches de sprite
