layer_sprite_destroy

Cette fonction détruira l'élément de sprite donné. Vous fournissez l'ID de sprite (que vous obtenez lorsque vous créez le sprite en utilisant layer_sprite_create() ou lorsque vous utilisez l'ID de couche avec layer_get_sprite_id() ) et cela le supprimera. Notez que cela ne supprime pas le calque, mais uniquement le sprite, et si le sprite est celui qui a été ajouté dans l'éditeur de salle, la prochaine fois que vous quittez la pièce, puis que vous revenez, le sprite sera recréé à nouveau. Cependant, si la salle est persistante, le sprite sera supprimé à moins que la persistance de la pièce ne soit à nouveau désactivée.

Syntaxe:

layer_sprite_destroy(sprite_element_id)

Argument Description
sprite_element_id La valeur d'identifiant unique du sprite à détruire

Retour:

N/A

Exemple:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
    {
    layer_sprite_destroy(lay_id);
    }

Le code ci-dessus vérifie la couche "Asset_Trees" pour voir si l'élément de sprite donné existe et si c'est le cas, alors il est détruit (mais pas la couche).
