couche_sprite_destruction

Cette fonction détruira l'élément de sprite donné. Vous fournissez l'ID du sprite (que vous obtenez lorsque vous créez le sprite à l'aide de la fonction layer_sprite_create() ou lorsque vous utilisez l'ID du calque avec layer_get_sprite_id()) et cette fonction le supprimera. Notez que cela ne supprime pas le calque, mais seulement le sprite, et si le sprite a été ajouté dans l'éditeur de la pièce, la prochaine fois que vous quitterez la pièce et y reviendrez, le sprite sera recréé à nouveau. Cependant, si la pièce est persistante, le sprite sera supprimé à moins que la persistance de la pièce ne soit à nouveau désactivée.

Syntaxe :

layer_sprite_destroy(sprite_element_id)

Arguments Description
Identifiant de l'élément de sprite La valeur d'identification unique du sprite à détruire.

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Asset_Trees") ;
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
{
layer_sprite_destroy(lay_id) ;
}

Le code ci-dessus vérifie la couche "Asset_Trees" pour voir si l'élément sprite donné existe et si c'est le cas, il est détruit (mais pas la couche).
