layer_sprite_x

Cette fonction contrôle la position le long de l'axe des x de la pièce de l'élément de sprite d'actif sur le calque. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), puis définissez la valeur x à utiliser (en fonction des coordonnées de la pièce).

Syntaxe:

layer_sprite_x(sprite_element_id, x);

Argument Description
sprite_element_id La valeur d'ID unique de l'élément de sprite à modifier
x La position x du sprite

Retour:

N/A

Exemple:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_x(spr_id, irandom(room_width));

Le code ci-dessus obtient la valeur d'ID de l'élément de sprite "Clouds" attribué à la couche "Asset_sky", puis définit sa position x sur une valeur aléatoire comprise entre 0 et la largeur de la pièce.
