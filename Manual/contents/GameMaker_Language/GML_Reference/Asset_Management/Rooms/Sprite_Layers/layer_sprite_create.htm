couche_sprite_créer

Cette fonction vous permet d'affecter une ressource sprite à une couche à utiliser dans votre projet. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ou lorsque vous utilisez le nom de la couche avec layer_get_id()), une position dans la pièce et un index de sprite (qui serait le nom du sprite tel qu'il apparaît dans le navigateur d'actifs), et il sera ajouté à la couche. sera ajouté à la couche. La fonction renvoie la valeur d'identification unique de l'élément, qui peut ensuite être utilisée dans d'autres fonctions de calque pour les sprites.

Syntaxe :

layer_sprite_create(layer_id, x, y, sprite)

Arguments Description
identifiant de la couche La valeur d'identification unique de la couche à cibler
x La position x à utiliser
y La position y à utiliser
sprite L'index du sprite à utiliser

Les retours :

ID de l'élément Sprite (réel)

Exemple :

global.asset_layer = layer_create(10000) ;
for (var i = 0 ; i< 10 ; i++ ;)
{
var _x = random(room_width) ;
var _y = room_height - 100 ;
global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees) ;
}

Le code ci-dessus crée une nouvelle couche et y ajoute 10 nouveaux éléments de sprite, en stockant l'ID de chaque élément dans un tableau.
