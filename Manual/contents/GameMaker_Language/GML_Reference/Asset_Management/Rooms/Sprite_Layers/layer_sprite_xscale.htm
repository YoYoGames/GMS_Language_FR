layer_sprite_xscale

En utilisant cette fonction, vous pouvez changer si l'élément de sprite donné sur un calque doit être mis à l'échelle le long de l'axe x ou non. Vous donnez l'ID d'élément de sprite (que vous obtenez lorsque vous créez un élément de sprite en utilisant layer_sprite_create() ou lorsque vous utilisez la fonction layer_sprite_get_id() ), puis définissez la valeur d'échelle. Une échelle de 1 indique aucune mise à l'échelle (1: 1), des valeurs plus petites seront réduites (0,5, par exemple, la moitié de la largeur de l'image-objet utilisée), des valeurs plus grandes seront mises à l'échelle et des valeurs négatives inverseront l'image-objet et la mettront à l'échelle sauf si la valeur utilisée est exactement -1 (auquel cas le sprite utilisé est simplement retourné de droite à gauche autour de sa position d'origine sans mise à l'échelle).

Syntaxe:

layer_sprite_xscale(sprite_element_id, xscale);

Argument Description
sprite_element_id La valeur d'ID unique de l'élément de sprite à modifier
xscale La valeur xscale (la valeur par défaut est 1)

Retour:

N/A

Exemple:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
    {
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
    }

Le code ci-dessus vérifiera l'élément de sprite affecté à la couche sur laquelle se trouve l'instance exécutant le code et s'il est mis à l'échelle dans les deux sens, il définira à la fois l'échelle de l'axe x et l'échelle de l'axe y sur 1.
