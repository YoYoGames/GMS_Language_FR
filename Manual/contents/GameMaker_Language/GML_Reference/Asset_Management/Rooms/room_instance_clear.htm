room_instance_clear

Cette fonction permet de vider une room de toutes ses instances (aucun événement de destruction ne sera appelé pour les instances supprimées). Notez que l'appel de cette fonction sur un asset de room créé dans le navigateur d'assets videra définitivement la room de ses instances, et même l'appel à game_restart() ne ramènera pas la room à son état d'origine (seule la fin du jeu et sa réouverture permettront à la room de retrouver son état d'origine).

Syntaxe :

room_instance_clear(index) ;

Arguments Description
indice L'index de la pièce dans laquelle il faut effacer toutes les instances.

Les retours :

N/A

Exemple :

global.rm = room_add() ;
room_assign(rm_Base, global.rm) ;
room_instance_clear(global.rm) ;

Le code ci-dessus ajoutera une nouvelle room au jeu, puis y copiera le contenu de la room indexée comme "rm_Base". Il effacera ensuite toutes les instances de cette nouvelle room.
