largeur de la pièce

Cette variable contient la largeur de la pièce actuelle en pixels. Vous pouvez modifier cette variable pour changer la largeur de la pièce à tout moment.

Syntaxe :

largeur de la pièce ;

Les retours :

Real

Exemple :

si bbox_right > room_width
{
x += room_width - bbox_right ;
}

Le code ci-dessus vérifie si la zone de délimitation du sprite de l'instance actuelle est plus grande que la largeur de la pièce, et si c'est le cas, il déplace l'instance vers le haut.
