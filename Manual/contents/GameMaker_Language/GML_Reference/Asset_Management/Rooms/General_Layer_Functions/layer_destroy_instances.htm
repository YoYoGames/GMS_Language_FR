couche_destroy_instances

Cette fonction peut être utilisée pour détruire toutes les instances affectées à la couche donnée. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create()) ou le nom de la couche (sous la forme d'une chaîne de caractères - cela aura un impact sur les performances). aura un impact sur les performances), et toutes les instances qui sont sur la couche seront supprimées du jeu, en déclenchant leurs événements Destroy et Clean Up.

Syntaxe :

layer_destroy_instances(layer_id)

Arguments Description
identifiant de la couche La valeur d'identification unique de la couche dont il faut définir la profondeur (ou le nom de la couche sous forme de chaîne).

Les retours :

Real (valeur d'identification unique pour une instance)

Exemple :

if global.game_over
{
layer_destroy_instances(layer) ;
}

Le code ci-dessus vérifiera une variable globale et si elle est vraie, alors toutes les instances qui sont sur la couche de l'instance appelante seront détruites (y compris l'instance appelante).
