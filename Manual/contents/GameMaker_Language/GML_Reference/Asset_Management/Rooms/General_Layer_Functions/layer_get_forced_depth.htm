layer_is_draw_depth_forced

Cette fonction peut être utilisée pour récupérer la profondeur Z définie pour le rendu des couches dans la pièce. Voir layer_force_draw_depth() pour plus d'informations.

Syntaxe:

layer_get_forced_depth()

Retour:

Real

Exemple:

if layer_get_forced_depth() != 0
   {
   layer_force_draw_depth(true, 0);
   }

Le code ci-dessus vérifie si la profondeur de la couche Z est forcée à une valeur de 0 ou non et si ce n'est pas le cas, il définit la profondeur de la couche Z sur 0.
