layer_get_element_layer

Vous pouvez utiliser cette fonction pour obtenir l' ID de couche sur lequel se trouve l'élément donné. Vous fournissez la valeur d'ID d'élément unique (par exemple, telle que renvoyée par la fonction qui a créé l'élément ou à partir de l'éditeur de pièce) et la fonction renverra l'ID unique de la couche sur laquelle l'élément se trouve. Si l'ID d'élément donné n'est pas valide, la fonction renverra -1.

Syntaxe:

layer_get_element_layer(element_id)

Argument Description
element_id La valeur d'identifiant unique de l'élément pour obtenir la couche de

Retour:

Layer ID or -1 if the element is invalid (ie: does not exist)

Exemple:

element = layer_get_element_layer(asset_1);

Le code ci-dessus obtient l'ID de couche pour l'élément avec l'ID stocké dans la variable "asset_1" et le stocke dans une variable.
