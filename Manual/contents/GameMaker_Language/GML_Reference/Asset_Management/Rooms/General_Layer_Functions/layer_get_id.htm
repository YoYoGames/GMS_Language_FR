layer_get_id

Cette fonction peut être utilisée pour obtenir la valeur d'identifiant unique pour une couche donnée. Dans l'EDI, toutes les couches ont un nom et un type, et pour pouvoir les éditer ou les changer via le code, vous devez donner la valeur de l' ID de couche. Cette fonction est utilisée pour récupérer cet ID en utilisant le nom (une chaîne) de la couche (tel qu'écrit dans l'EDI). Si vous créez une nouvelle couche via du code à l'aide de la fonction layer_create() alors cette fonction renverra la valeur d'ID unique à la place (les couches créées dynamiquement n'obtiennent pas de noms). Notez que si vous donnez le nom d'un calque qui n'existe pas dans la pièce actuelle, vous obtiendrez une erreur et le projet plantera.

Syntaxe:

layer_get_id(layer_name)

Argument Description
layer_name Le nom du calque (une chaîne)

Retour:

Layer ID Value (Real), or -1 if the layer specified doesn't exist

Exemple:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Le code ci-dessus obtiendra d'abord l'index de l'instance la plus proche de la position x / y donnée et le stockera dans une variable locale. Il obtient ensuite l'ID de couche d'instance unique pour la couche nommée "Instances Front" et déplace l'instance trouvée sur cette couche.
