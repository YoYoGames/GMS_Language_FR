couche_add_instance

Cette fonction peut être utilisée pour déplacer une instance donnée de la couche sur laquelle elle se trouve actuellement vers une autre couche. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create()) ou le nom de la couche (sous forme de une chaîne - cela aura un impact sur les performances) et l'ID de l'instance à déplacer entre les couches. Par exemple, disons que votre lecteur se trouve sur un calque qui est à une profondeur inférieure à celle d'un autre calque et que vous voulez qu'il apparaisse derrière les calques entre les deux. les deux. Vous pouvez utiliser cette fonction pour faire passer le lecteur du calque de premier plan au calque d'arrière-plan et le faire apparaître derrière les autres calques dessinés.

NOTE : Cette fonction ne peut pas être utilisée pour ajouter une nouvelle instance à une couche. Vous devez avoir créé l'instance auparavant et enregistré son ID dans une variable.

REMARQUE: Cette fonction n'est valable que dans le cadre de la pièce actuelle et ne peut être utilisée lorsque le cadre a été modifié à l'aide de la fonction layer_set_target_room().

Syntaxe :

layer_add_instance(layer_id, instance_id)

Arguments Description
identifiant de la couche La valeur d'identification unique de la couche d'instance à cibler (ou le nom de la couche sous forme de chaîne).
identifiant de l'instance La valeur unique de l'ID de l'instance à déplacer vers la couche cible.

Les retours :

N/A

Exemple :

var near = instance_nearest(x, y, obj_Tree) ;
var layer_id = layer_get_id("Instances Front") ;
layer_add_instance(layer_id, near) ;

Le code ci-dessus va d'abord obtenir l'index de l'instance la plus proche de la position x/y donnée et le stocker dans une variable locale. Il obtient ensuite l'ID unique de la couche d'instance pour la couche nommée "Instances Front", et déplace l'instance trouvée sur cette couche. cette couche.
