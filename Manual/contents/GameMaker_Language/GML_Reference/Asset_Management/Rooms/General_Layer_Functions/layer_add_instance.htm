layer_add_instance

Cette fonction peut être utilisée pour déplacer une instance donnée du calque sur lequel elle se trouve actuellement vers un autre calque. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et l'ID d'instance de l'instance à déplacer entre les couches. Par exemple, supposons que votre lecteur se trouve sur un calque qui est à une profondeur inférieure à un autre calque et que vous voulez qu'il apparaisse derrière les calques entre les deux. Vous pouvez utiliser cette fonction pour faire passer le lecteur du calque de premier plan au calque d'arrière-plan et le faire apparaître derrière les autres calques en cours de dessin.

REMARQUE: cette fonction ne peut pas être utilisée pour ajouter une nouvelle instance à un calque. Vous devez avoir créé l'instance précédemment et stocké son ID dans une variable.

REMARQUE: Cette fonction n'est valable que dans le périmètre de la pièce actuelle et ne peut pas être utilisée lorsque le périmètre a été modifié à l'aide de la fonctionlayer_set_target_room().

Syntaxe:

layer_add_instance(layer_id, instance_id)

Argument Description
layer_id La valeur d'ID unique de la couche d'instance à cibler (ou le nom de la couche sous forme de chaîne)
instance_id La valeur d'ID d'instance unique de l'instance à déplacer vers la couche cible

Retour:

N/A

Exemple:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Le code ci-dessus obtiendra d'abord l'index de l'instance la plus proche de la position x / y donnée et le stockera dans une variable locale. Il obtient ensuite l'ID de couche d'instance unique pour la couche nommée "Instances Front" et déplace l'instance trouvée sur cette couche.
