couche_get_id_at_depth

Vous pouvez utiliser cette fonction pour obtenir les ID de toutes les couches affectées à une profondeur spécifique. Vous donnez la profondeur à vérifier et la fonction retournera un tableau avec 1 ou plusieurs entrées selon qu'il y a ou non des couches à la profondeur donnée. des couches à la profondeur donnée ou non. S'il n'y a pas de couches à la profondeur donnée, le tableau aura une seule entrée à la position [0] avec une valeur de -1, mais, s'il y a des couches à la profondeur, alors une entrée sera faite dans le tableau pour chaque couche trouvée. couche trouvée - la valeur de l'entrée sera la valeur d'identification unique pour une couche.

Syntaxe :

layer_get_id_at_depth(depth)

Arguments Description
profondeur La profondeur pour vérifier et récupérer les ID des couches.

Les retours :

Array

Exemple :

var a = layer_get_id_at_depth(0) ;
if a[0] != -1
{
for (var i = 0 ; i < array_length(a) ; i++ ;)
{
layer_destroy(a[i]) ;
}
}

Le code ci-dessus récupère les données sur les couches avec une profondeur de 0. Une vérification est effectuée pour voir si des couches existent à cette profondeur, et si c'est le cas, le tableau retourné est analysé et chacune des couches trouvées est détruite.
