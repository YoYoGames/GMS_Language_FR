layer_get_id_at_depth

Vous pouvez utiliser cette fonction pour obtenir les ID de toutes les couches auxquelles une profondeur spécifique est attribuée. Vous donnez la profondeur à vérifier et la fonction renverra un tableau avec 1 ou plusieurs entrées selon qu'il y a des couches à la profondeur donnée ou non. S'il n'y a pas de couches à la profondeur donnée, alors le tableau aura une seule entrée à la position [0] avec une valeur de -1, mais, s'il y a des couches à la profondeur, alors une entrée sera faite dans le tableau pour chaque couche trouvée - la valeur d'entrée sera la valeur d'ID unique pour une couche.

Syntaxe:

layer_get_id_at_depth(depth)

Argument Description
depth La profondeur à partir de laquelle vérifier et récupérer les ID de couche.

Retour:

Array

Exemple:

var a = layer_get_id_at_depth(0);
if a[0] != -1
    {
    for (var i = 0; i < array_length(a); i++;)
        {
        layer_destroy(a[i]);
        }
    }

Le code ci-dessus récupère les données sur les couches avec une profondeur de 0. Une vérification est effectuée pour voir si des couches existent à cette profondeur, et s'il y en a, le tableau retourné est analysé et chacune des couches trouvées est détruite.
