couche_get_élément_type

Vous pouvez utiliser cette fonction pour obtenir le type d'élément pour l'élément donné. Vous fournissez la valeur d'identification unique de l'élément (par exemple, telle que renvoyée par la fonction qui a créé l'élément ou par l'éditeur de pièce) et la fonction renverra l'une des valeurs suivantes (ou -1 si l'élément n'existe pas ou si la valeur de l'ID est erronée) :

Constant Description
layerelementtype_background L'élément est un fond.
layerelementtype_instance L'élément est une instance.
layerelementtype_sprite L'élément est un actif de type sprite.
layerelementtype_tilemap L'élément est une mosaïque.
layerelementtype_particlesystem L'élément est un système de particules.
type d'élément_carreau L'élément est une tuile d'arrière-plan héritée (ceci n'est valable que pour les projets qui ont été importés de versions précédentes de GameMaker).
séquence_type_élémentaire L'élément est un actif de séquence.

Notez que cette fonction est plus utile lorsque vous avez plusieurs types d'éléments différents assignés à une couche, et que vous pouvez obtenir une liste de tous les éléments sur une couche donnée en utilisant la fonction layer_get_all_elements().

Syntaxe :

layer_get_element_type(element_id)

Arguments Description
couche La valeur d'identification unique de l'élément dont on veut obtenir le type.

Les retours :

Constante (voir tableau ci-dessus) ou -1 si l'élément n'existe pas ou est invalide.

Exemple :

var a = layer_get_all_elements(layer) ;
for (var i = 0 ; i < array_length(a) ; i++ ;)
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_sprite_destroy(a[i])
}
}

Le code ci-dessus récupère les ID de tous les éléments d'instance affectés au layer de l'instance qui exécute le code. Le code vérifie ensuite si l'un des éléments retournés est un sprite et s'il l'est, il est détruit.
