couche_vitesse

Vous pouvez utiliser cette fonction pour définir la vitesse verticale (en pixels par image de jeu) de la couche dans la pièce actuellement délimitée. Vous fournissez l'ID du calque (que vous obtenez lorsque vous créez le calque en utilisant la fonction layer_create()) ou le nom du calque (sous forme de chaîne - cela aura un impact sur les performances) et la valeur de la vitesse à définir, où une valeur positive correspond à un mouvement vers le bas et une valeur négative vers le haut.

Syntaxe :

layer_vspeed(layer_id, vspd)

Arguments Description
identifiant de la couche La valeur d'identification unique de la couche pour définir la vitesse verticale de la couche.
vspd La vitesse verticale (en pixels par image de jeu) à définir.

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Sprites") ;
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
layer_hspeed(lay_id, 0) ;
layer_vspeed(lay_id, 0) ;
}

Le code ci-dessus vérifie les vitesses horizontale et verticale de la couche donnée et si elles ne sont pas toutes deux à 0, il les met à 0.
