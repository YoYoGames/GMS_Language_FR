layer_get_visible

Avec cette fonction, vous pouvez vérifier si un calque est visible ou non. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et la fonction retournera true s'il est visible, et false autrement.

Syntaxe:

layer_get_visible(layer_id)

Argument Description
layer_id La valeur d'ID unique du calque à cibler (ou le nom du calque sous forme de chaîne)

Retour:

Boolean

Exemple:

var lay_id = layer_get_id("Instances");
if layer_get_visible(lay_id)
   {
   layer_set_visible(lay_id, false);
   }
else
   {
   layer_set_visible(lay_id, true);
   }

Le code ci-dessus obtient la valeur d'ID pour la couche nommée «Instances» dans l'éditeur de salle, puis utilise l'ID pour vérifier si la couche est visible ou non, en basculant la visibilité de la couche en fonction de la valeur renvoyée.
