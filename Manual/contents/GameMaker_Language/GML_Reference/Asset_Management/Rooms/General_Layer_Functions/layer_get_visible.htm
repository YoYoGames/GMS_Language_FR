couche_get_visible

Avec cette fonction, vous pouvez vérifier si une couche est visible ou non. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create()) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances). performances) et la fonction retournera true si la couche est visible, false sinon.

Syntaxe :

layer_get_visible(layer_id)

Arguments Description
identifiant de la couche La valeur d'identification unique de la couche à cibler (ou le nom de la couche sous forme de chaîne).

Les retours :

Booléen

Exemple :

var lay_id = layer_get_id("Instances") ;
if layer_get_visible(lay_id)
{
layer_set_visible(lay_id, false) ;
}
else
{
layer_set_visible(lay_id, true) ;
}

Le code ci-dessus récupère la valeur de l'ID de la couche nommée "Instances" dans l'éditeur de pièce, puis utilise l'ID pour vérifier si la couche est visible ou non, en basculant la visibilité de la couche en fonction de la valeur retournée.
