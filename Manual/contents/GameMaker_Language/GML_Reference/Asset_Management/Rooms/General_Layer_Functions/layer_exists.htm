couche_existe

Cette fonction peut être utilisée pour vérifier si la couche donnée existe. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de la fonction layer_create()) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances). performance) et la fonction retournera une valeur booléenne de true si elle existe ou false si elle n'existe pas.

REMARQUE: Cette fonction fonctionne dans le cadre de la pièce cible actuelle - par défaut, la pièce dans laquelle la fonction est appelée - qui peut être définie à l'aide de la fonction layer_set_target_room().

Syntaxe :

layer_exists(nom_de_couche)

Arguments Description
nom_de_couche Le nom de la couche (une chaîne ou une valeur ID)

Les retours :

Booléen

Exemple :

if !layer_exists(global.tileLayer)
{
global.tileLayer = layer_create(1000) ;
}

Le code ci-dessus va vérifier si la couche stockée dans la variable globale existe réellement, et si ce n'est pas le cas, elle est créée.
