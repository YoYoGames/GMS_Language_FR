couche_get_all

Cette fonction renvoie un tableau contenant les valeurs d'identification uniques de chaque couche de la pièce.

Syntaxe :

layer_get_all()

Les retours :

Tableau (1D, rempli d'ID de couches)

Exemple :

var a = layer_get_all() ;
for (var i = 0 ; i < array_length(a) ; i++ ;)
{
layer_destroy(a[i]) ;
}

Le code ci-dessus récupère toutes les couches d'une pièce et ajoute leurs valeurs d'identification à un tableau. Ce tableau est ensuite analysé pour détruire les couches de la pièce.
