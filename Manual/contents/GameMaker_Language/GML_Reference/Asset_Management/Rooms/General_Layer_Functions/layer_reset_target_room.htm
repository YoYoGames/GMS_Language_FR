layer_get_target_room

Cette fonction est utilisée pour réinitialiser la cible du calque dans la pièce actuelle. Voir la fonction layer_set_target_room() pour plus d'informations.

Syntaxe:

layer_reset_target_room()

Retour:

N/A

Exemple:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
   {
   layer_sprite_create(l, random(room_get_width(rm_Game)), random(room_get_height(rm_Game)), spr_Trees);
   }
layer_reset_target_room();

Le code ci-dessus définit la salle cible sur la salle "rm_Game", puis obtient l'ID de couche pour la couche appelée "SpriteAssets" dans cette salle. Cet ID de couche est ensuite utilisé pour ajouter 50 ressources de sprite aléatoires à la couche, avant que la cible de la couche ne soit réinitialisée dans la pièce actuelle.
