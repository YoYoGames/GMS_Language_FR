couche_get_script_begin

Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create()) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et cette fonction renverra l'index de la fonction de script de la fonction assignée pour être exécutée au début du rendu pour cette couche, ou elle renverra -1 si aucune fonction n'est assignée. Vous pouvez attribuer des fonctions de script à une couche avec layer_script_begin() et layer_script_end().

Syntaxe :

layer_get_script_begin(layer_id) ;

Arguments Description
identifiant de la couche La valeur d'identification unique de la couche à cibler (ou le nom de la couche sous forme de chaîne).

Les retours :

Index de la fonction script (réel) ou -1 si aucun script n'est assigné.

Exemple :

si layer_get_script_begin(layer) == -1
{
layer_script_begin(layer, scr_SetShaderValues) ;
}

Le code ci-dessus vérifie si le calque de l'instance qui exécute le code a une fonction de script assignée et si ce n'est pas le cas, une fonction est assignée.
