couche_get_shader

Cette fonction peut être utilisée pour vérifier si un shader est attribué à un calque donné. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create()) ou le nom du calque (sous forme de chaîne de caractères - cela aura un impact sur les performances). cela aura un impact sur les performances), et la fonction retournera soit l'index du shader assigné, soit -1 si aucun shader n'est assigné.

Syntaxe :

layer_get_shader(layer_id)

Arguments Description
identifiant de la couche La valeur d'identification unique de la couche à cibler (ou le nom de la couche sous forme de chaîne).

Les retours :

Index dushader (ou -1 si aucun shader n'est attribué)

Exemple :

si layer_get_shader(layer) == -1
{
layer_shader(layer, shd_Sepia) ;
}

Le code ci-dessus vérifie si le calque de l'instance qui exécute le code a un shader assigné et si ce n'est pas le cas, un shader est assigné.
