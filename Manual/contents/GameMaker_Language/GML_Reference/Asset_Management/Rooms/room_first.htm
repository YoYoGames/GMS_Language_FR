room_first

Cette variable en lecture seule renvoie l'index de la toute première salle du jeu (celle-ci est définie par l'ordre dans lequel les salles apparaissent dans le Room Manager et non par l'ordre dans lequel elles ont été créées).

Syntaxe:

room_first;

Retour:

Index

Exemple:

if lives < 1
    {
    room_goto(room_first);
    }

Le code ci-dessus vérifiera la variable "lives" et si elle est inférieure à 1, allez dans la première salle du jeu.
