layer_sequence_get_angle

Avec cette fonction, vous fournissez l'ID d'élément de séquence - tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et il renverra l'angle actuel de l'élément de séquence dans la salle de jeux. Notez que les angles sont renvoyés en degrés, et 0º vers la droite, 90º vers le haut, 180º vers la gauche et 270º vers le bas.

Syntaxe:

layer_sequence_get_angle(sequence_element_id)

Argument Description
sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler

Retour:

Real

Exemple:

var _ang = layer_sequence_get_angle(title_sequence)
if _ang > 0
    {
    _ang -= 1;
    layer_sequence_angle(title_sequence, _ang);
    }

Le code ci-dessus récupère l'angle actuel de l'élément de séquence avec l'ID stocké dans la variable "title_sequence", et s'il n'est pas 0, alors 1 est soustrait de l'angle actuel.
