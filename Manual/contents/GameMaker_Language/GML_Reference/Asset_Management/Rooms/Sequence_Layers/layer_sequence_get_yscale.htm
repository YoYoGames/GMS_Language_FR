séquence_couche_get_échelle

Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle renverra l'échelle actuelle le long de l'axe Y de l'élément de séquence dans la salle de jeu.

Syntaxe :

layer_sequence_get_yscale(sequence_element_id)

Arguments Description
id_élément_de_séquence La valeur d'identification unique de l'élément de séquence à cibler.

Les retours :

Real

Exemple :

var _ys = layer_sequence_get_yscale(title_sequence)
if _ys < 1
{
_ys += 0,01 ;
layer_sequence_yscale(séquence_titre, _ys) ;
}

Le code ci-dessus récupère l'échelle actuelle le long de l'axe Y de l'élément de la séquence avec l'ID stocké dans la variable "title_sequence", et si elle est inférieure à 1, alors 0,01 est ajouté à l'échelle Y actuelle.
