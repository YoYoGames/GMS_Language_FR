pause_séquence_couche

Cette fonction permet de mettre en pause la lecture de la séquence donnée. Vous fournissez l'ID de l'élément de la séquence tel que renvoyé par couche_séquence_créer() ou par l'une des fonctions de l'élément de couche et la fonction mettra la séquence en pause jusqu'à ce que vous recommenciez la lecture à l'aide de la fonction layer_sequence_play().

Syntaxe :

layer_sequence_pause(sequence_element_id)

Arguments Description
id_élément_de_séquence La valeur d'identification unique de l'élément de séquence à cibler.

Les retours :

N/A

Exemple :

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground) ;
if global.Pause
{
layer_sequence_pause(_seq) ;
}

Le code ci-dessus crée une nouvelle séquence et stocke son ID dans une variable locale. Il vérifie ensuite si le jeu est en pause, et si c'est le cas, il met en pause la lecture de la séquence.
