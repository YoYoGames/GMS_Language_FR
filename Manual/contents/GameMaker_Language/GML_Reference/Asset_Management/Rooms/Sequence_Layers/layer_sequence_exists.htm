layer_sequence_exists

Avec cette fonction, vous pouvez vérifier si un élément de séquence existe sur le calque donné. Vous fournissez l'ID de couche qui peut être une chaîne du nom de couche - tel que défini dans l'éditeur de pièce - ou l'ID de couche unique - tel que renvoyé par la fonction layer_get_id(), ainsi que l'ID d'élément de séquence - tel que renvoyé par layer_sequence_create() ou par l'une des fonctions d'élément de couche - et il retournera true si l'élément donné existe ou false autrement.

Syntaxe:

layer_sequence_exists(layer_id, sequence_element_id)

Argument Description
layer_ID L'ID unique ou le nom de la couche à vérifier
sequence_element_id La valeur d'ID unique de l'élément de séquence à cibler

Retour:

Boolean

Exemple:

if !layer_sequence_exists("Background", my_seq)
    {
    my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
    layer_sequence_pause(my_seq);
    }

Le code ci-dessus vérifie si l'élément de séquence donné existe, et si ce n'est pas le cas, il crée une nouvelle séquence sur la couche donnée, puis la met en pause.
