couche_séquence_get_speedscale

Avec cette fonction, vous fournissez l'ID de l'élément de séquence - tel que retourné par layer_sequence_create() ou par l'une des fonctions de l'élément de couche - et elle renvoie l'échelle de vitesse de lecture actuelle. Il s'agit de la valeur du multiplicateur utilisé pour ralentir ou accélérer la vitesse de lecture. La valeur 1 est la valeur par défaut, et les valeurs inférieures à 1 signifient que la lecture est ralentie et les valeurs supérieures à 1 signifient que la lecture est accélérée. valeurs supérieures à 1 signifient que la lecture est accélérée.

Syntaxe :

layer_sequence_get_speedscale(sequence_element_id)

Arguments Description
id_élément_de_séquence La valeur d'identification unique de l'élément de séquence à cibler.

Les retours :

Real

Exemple :

if layer_sequence_get_speedscale(séquence_titre) != 1
{
layer_sequence_speedscale(séquence_titre, 1) ;
}

Le code ci-dessus vérifie l'échelle de vitesse de lecture actuelle de l'ID de l'élément de séquence stocké dans la variable "title_sequence", et si elle n'est pas égale à 1, elle est fixée à cette valeur.
