layer_sequence_create

Avec cette fonction, vous pouvez créer une instance d'un élément de séquence sur le calque donné. Vous fournissez l'ID de couche qui peut être une chaîne du nom de couche - tel que défini dans l'éditeur de pièce - ou l'ID de couche unique - tel que renvoyé par la fonction layer_get_id(), ainsi que la position X et Y dans la pièce où créer la séquence, et enfin l'ID de la séquence à créer. L'ID de séquence est la constante de nom que vous avez définie dans le navigateur de ressources pour la séquence. La fonction renverra l'ID unique de l'élément de séquence, qui peut ensuite être utilisé dans toutes les autres fonctions de couche pour les séquences, ou elle peut être utilisée pour récupérer la structure d'instance de séquence à l'aide de la fonction layer_sequence_get_instance().

Syntaxe:

layer_sequence_create(layer_id, x, y, sequence_id)

Argument Description
layer_id La valeur d'ID unique de la couche d'instance à cibler
x La position x dans la pièce pour créer la séquence à
y La position y dans la pièce pour créer la séquence à
sequence_id L'ID de séquence à créer, tel que défini dans le navigateur d'actifs

Retour:

Integer (element ID)

Exemple:

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

Le code ci-dessus crée une nouvelle séquence sur le calque "Arrière-plan" puis la met en pause.

Retour: Couches de séquence
