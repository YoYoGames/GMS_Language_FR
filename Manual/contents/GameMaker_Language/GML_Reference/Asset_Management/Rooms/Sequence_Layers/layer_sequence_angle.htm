angle_séquence_couche

Avec cette fonction, vous pouvez définir l'angle de rotation pour l'élément de séquence donné. Vous fournissez l'ID de l'élément de séquence tel que retourné par layer_sequence_create() ou par l'une des fonctions d'élément de calque avec le nouvel angle à définir et la séquence sera tournée autour de son origine vers la nouvelle position. Dans GameMaker, les angles sont calculés en partant de 0º à droite et en allant dans le sens inverse des aiguilles d'une montre (90º pour le haut, 180º pour la gauche et 270º pour le bas). pour une séquence serait de 0º.

Notez que vous ne devez pas modifier les variables image_xscale \ image_yscale \ image_angle \ x \ y pour toute instance dans une séquence qui utilise cette fonction, car elles seront écrasées dès que la séquence sera mise à jour après l'appel de la fonction.

Syntaxe :

layer_sequence_angle(sequence_element_id, angle)

Arguments Description
id_élément_de_séquence La valeur d'identification unique de l'élément de séquence à cibler.
angle Le nouvel angle de rotation de l'élément de la séquence.

Les retours :

N/A

Exemple :

if current_angle < 90
{
current_angle += 1 ;
layer_sequence_angle(my_seq, current_angle) ;
}

Le code ci-dessus vérifie la valeur contenue dans la variable current_angle, et si elle est inférieure à 90, il l'ajoute puis utilise la valeur pour définir l'angle de l'élément de séquence référencé dans la variable my_seq.
