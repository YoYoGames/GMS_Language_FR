redémarrage de la chambre

Cette fonction va redémarrer la pièce actuelle, comme si elle venait d'être entrée. Notez que la salle ne redémarrera pas avant la fin de l'événement où la fonction a été appelée, donc tout code après que celle-ci ait été appelée sera toujours exécuté s'il se trouve dans le même événement. L'avertissement de est que tout code qui crée des instances et qui est appelé après cette fonction, ne créera pas l'instance et doit donc être appelé avant de changer de room. Cette fonction déclenchera également l'événement Room End.

Syntaxe :

room_restart() ;

Les retours :

N/A

Exemple :

si vies < 1 room_restart() ;

Le code ci-dessus vérifie si la variable "lives" est inférieure à 1 et si c'est le cas, il redémarre la pièce.
