layer_background_create

Avec cette fonction, vous pouvez affecter une ressource de sprite à un calque à utiliser comme arrière-plan dans votre projet. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et un index de sprite (qui serait le nom de l'image-objet comme indiqué dans le navigateur d'actifs), et il sera ajouté à la couche. La fonction renvoie la valeur d'ID unique de l'élément, qui peut ensuite être utilisée dans d'autres fonctions de calque pour les arrière-plans.

Syntaxe:

layer_background_create(layer_id, sprite)

Argument Description
layer_id La valeur d'ID unique du calque à cibler (ou le nom du calque sous forme de chaîne)
sprite L'index de sprite à utiliser

Retour:

Background element ID (Real)

Exemple:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

Le code ci-dessus crée un nouveau calque, puis y ajoute un nouvel élément d'arrière-plan, définissant un sprite comme l'image d'arrière-plan utilisée.

Retour: Calques d'arrière-plan
