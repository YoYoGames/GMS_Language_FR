couche_arrière-plan_get_stretch

Cette fonction peut être utilisée pour obtenir l'état étiré du sprite de l'élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), et la fonction renverra soit true si le sprite de l'élément est actuellement étiré pour s'adapter à la pièce, soit false s'il ne l'est pas. pas.

Syntaxe :

layer_background_get_stretch(background_element_id) ;

Arguments Description
id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à partir duquel les informations doivent être obtenues.

Les retours :

Booléen

Exemple :

var lay_id = layer_get_id("Background_sky") ;
var back_id = layer_background_get_id(lay_id) ;
if layer_background_get_stretch(back_id)
{
layer_background_stretch(back_id, false) ;
}

Le code ci-dessus récupère l'ID du calque nommé "Background_sky" et l'utilise ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier si le sprite de l'élément sera étiré pour s'adapter à la pièce. et s'il est étiré, cette propriété est mise à false.
