layer_background_alpha

Cette fonction contrôle l'alpha (transparence) du sprite d'arrière-plan. Vous donnez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), puis définissez la valeur alpha à utiliser. Alpha peut être compris entre 0 (entièrement transparent) et 1 (entièrement opaque), la valeur alpha par défaut de l'élément d'arrière-plan étant 1. Notez que si le calque auquel l'élément d'arrière-plan a été affecté n'est pas visible - ou l'élément lui-même a été créé invisible - vous ne verrez aucune différence avec cette fonction tant que le calque ou l'élément n'a pas été rendu visible à nouveau.

Syntaxe:

layer_background_alpha(background_element_id, alpha);

Argument Description
background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à modifier
alpha L'alpha du sprite d'arrière-plan, de 0 à 1 (la valeur par défaut est 1)

Retour:

N/A

Exemple:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

Le code ci-dessus obtient la valeur d'ID de l'arrière-plan affectée au calque "Background_sky", puis définit son alpha sur une valeur aléatoire entre 0 et 1.

Retour: Calques d'arrière-plan
