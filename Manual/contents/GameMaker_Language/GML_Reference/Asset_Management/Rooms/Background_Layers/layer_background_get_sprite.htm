couche_arrière-plan_get_sprite

Cette fonction peut être utilisée pour obtenir la valeur actuelle de l'index du sprite de l'élément d'arrière-plan. Vous donnez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), et la fonction renverra une valeur réelle qui représente l'index du sprite affiché. Si l'élément n'a pas de sprite assigné, la fonction renverra -1.

Syntaxe :

layer_background_get_sprite(background_element_id) ;

Arguments Description
id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à partir duquel les informations doivent être obtenues.

Les retours :

Index du sprite (réel) ou -1

Exemple :

var lay_id = layer_get_id("Background_sky") ;
var back_id = layer_background_get_id(lay_id) ;
if layer_background_get_sprite(back_id) != spr_Clouds
{
layer_background_sprite(back_id, spr_Clouds) ;
}

Le code ci-dessus récupère l'ID du calque nommé "Background_sky" et l'utilise ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour vérifier le sprite assigné à l'élément, en le réglant sur le sprite "spr_Clouds", s'il ne l'est pas déjà. s'il ne l'est pas déjà.
