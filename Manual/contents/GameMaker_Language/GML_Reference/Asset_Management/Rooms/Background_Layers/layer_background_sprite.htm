layer_background_sprite

En utilisant cette fonction, vous pouvez définir l'index du sprite de l'élément d'arrière-plan. Vous fournissez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), puis donnez un index de sprite à utiliser. L'image de l'élément d'arrière-plan sera remplacée par le nouveau sprite. Si vous donnez une valeur de -1, l'élément n'aura pas de sprite assigné (mais existera toujours et pourra avoir un sprite assigné à nouveau plus tard).

Syntaxe:

layer_background_sprite(background_element_id, sprite_index)

Argument Description
background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à modifier
sprite_index L'index de sprite du sprite à utiliser pour l'élément d'arrière-plan

Retour:

N/A

Exemple:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
    {
    layer_background_sprite(back_id, spr_Clouds);
    }

Le code ci-dessus obtiendra l'ID de la couche nommée "Background_sky", puis l'utilisera pour obtenir l'ID de l'élément d'arrière-plan sur cette couche. Cet identifiant est ensuite utilisé pour vérifier le sprite affecté à l'élément, en le définissant sur le sprite "spr_Clouds" si ce n'est déjà fait.
