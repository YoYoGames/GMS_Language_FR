layer_background_index

Cette fonction peut être utilisée pour définir l'index d'image du sprite d'arrière-plan qui a plusieurs sous-images. Vous donnez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), puis définissez l'index d'image à utiliser. Si vous définissez une valeur en dehors de la plage des sous-images, l'index de l'image sera en boucle.

Syntaxe:

layer_background_index(background_element_id, image_index);

Argument Description
background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à modifier
index L'index d'image à utiliser pour l'arrière-plan

Retour:

N/A

Exemple:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

Le code ci-dessus obtiendra l'ID de la couche nommée "Background_trees", puis l'utilisera pour obtenir l'ID de l'élément d'arrière-plan sur cette couche. Cet identifiant est ensuite utilisé pour modifier l'index d'image de l'élément.

Retour: Calques d'arrière-plan
