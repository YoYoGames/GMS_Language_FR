layer_background_visible

En utilisant cette fonction, vous pouvez basculer la visibilité des éléments d'arrière-plan. Vous fournissez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), puis définissez l'argument visible sur true ou false. Lorsqu'il est réglé sur true l'élément sera affiché, et lorsqu'il est réglé sur false, l'élément ne sera pas affiché. Notez que cela dépend de la visibilité du calque, et même si l'élément d'arrière-plan est marqué comme visible, il ne sera pas dessiné si le calque sur lequel il se trouve est marqué comme non visible.

Syntaxe:

layer_background_visible(background_element_id, visible)

Argument Description
background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à modifier
visible La bascule de visibilité, qui peut être true ou false

Retour:

N/A

Exemple:

if mouse_check_button_pressed(mb_left)
    {
    var back = layer_background_get_id(layer);
    if layer_background_get_visible(back)
        {
        layer_background_visible(back, false);
        }
    else
        {
        layer_background_visible(back, true);
        }
    }

Le code ci-dessus vérifie si un bouton de la souris est enfoncé et si un bouton est détecté, il basculera la visibilité d'arrière-plan de l'élément d'arrière-plan affecté à la couche sur laquelle se trouve l'instance exécutant le code.
