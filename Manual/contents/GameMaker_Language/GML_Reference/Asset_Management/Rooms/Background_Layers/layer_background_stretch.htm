layer_background_stretch

Cette fonction vous permet de faire en sorte qu'un sprite d'élément d'arrière-plan s'étire pour s'adapter à la pièce ou reste à l'échelle 1:1 de la résolution. Vous fournissez l'ID de l'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de la fonction layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id()), puis définissez l'argument d'étirement à true ou false. Lorsqu'il est défini sur true, le sprite de l'élément sera étiré pour s'adapter à la pièce (soit à l'échelle, soit à l'échelle inférieure). la pièce (à l'échelle supérieure ou inférieure selon la taille du sprite et de la pièce), et à l'échelle inférieure, le sprite de l'élément sera affiché à sa résolution native.

Syntaxe :

layer_background_stretch(background_element_id, stretch)

Arguments Description
id_élément_fond La valeur d'identification unique de l'élément d'arrière-plan à modifier.
extensible Le bouton d'étirement, qui peut être vrai ou faux.

Les retours :

N/A

Exemple :

if mouse_check_button_pressed(mb_left)
{
var back = layer_background_get_id(layer) ;
if layer_background_get_stretch(back)
{
layer_background_stretch(back, false) ;
}
else
{
layer_background_stretch(back, true) ;
}
}

Le code ci-dessus vérifie si un bouton de la souris a été enfoncé et, s'il est détecté, il fait basculer l'étirement du sprite de l'élément d'arrière-plan attribué au calque sur lequel se trouve l'instance qui exécute le code.
