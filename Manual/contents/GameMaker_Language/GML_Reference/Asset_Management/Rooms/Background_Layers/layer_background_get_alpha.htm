layer_background_get_alpha

Cette fonction peut être utilisée pour obtenir la valeur alpha de l'élément d'arrière-plan. Vous donnez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), et la fonction renverra une valeur comprise entre 0 (totalement transparent) et 1 (totalement opaque).

Syntaxe:

layer_background_get_alpha(background_element_id);

Argument Description
background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à partir duquel obtenir les informations

Retour:

Real (from 0 to 1)

Exemple:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_alpha(back_id) < 0.1
    {
    layer_background_destroy(back_id);
    }

Le code ci-dessus obtiendra l'ID de la couche nommée "Background_sky", puis l'utilisera pour obtenir l'ID de l'élément d'arrière-plan sur cette couche. Cet ID est ensuite utilisé pour vérifier l'élément alpha et s'il est inférieur à 0,1, alors l'élément de couche est détruit.
