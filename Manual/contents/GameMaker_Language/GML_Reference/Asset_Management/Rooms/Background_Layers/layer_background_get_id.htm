layer_background_get_id

Cette fonction peut être utilisée pour récupérer la valeur d'identification unique de l'élément d'arrière-plan sur une couche. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous utilisez le nom de la couche avec la fonction layer_get_id()) et la fonction renvoie la valeur d'identification associée à l'élément d'arrière-plan sur la couche. Notez que cette fonction est spécifiquement conçue pour être utilisée avec des arrière-plans qui ont été ajoutés dans l'IDE, car si vous ajoutez un arrière-plan à un calque par le biais de la fonction code en utilisant la fonction layer_background_create()la fonction renverra l'ID unique de l'élément d'arrière-plan ajouté.

Syntaxe :

layer_background_get_id(layer_id)

Arguments Description
identifiant de la couche La valeur d'identification unique de la couche à cibler

Les retours :

Valeur d'identification de l'élément de fond (réel)

Exemple :

var lay_id = layer_get_id("Background_trees") ;
var back_id = layer_background_get_id(lay_id) ;
layer_background_sprite(back_id, bck_Trees_Winter) ;

Le code ci-dessus récupère l'ID du calque nommé "Background_trees" et l'utilise ensuite pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour changer le sprite de l'élément.
