tilemap_get_at_pixel

Cette fonction permet de récupérer les données des tuiles à partir d'une position (dans la pièce) de l'élément de carte des tuiles. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()) ainsi que la position x et y dans la pièce pour obtenir les données des tuiles et la fonction renverra le "blob" de données des tuiles. Ces données sont essentiellement un masque binaire contenant l'index de la tuile, les booléens d'inversion, de rotation et de miroir, ainsi que les données de masque qui ont été appliquées (cf. tilemap_set_mask() pour plus de détails), et la valeur des données résultantes peut alors être utilisée dans les fonctions tile_* pour modifier les propriétés d'une tuile. Si vous avez besoin d'obtenir les données d'une tuile à partir d'une cellule de tuile spécifique, vous devriez utiliser la fonction tilemap_get() à la place.

IMPORTANT ! Si les tuiles de la carte des tuiles sont restées inchangées (c'est-à-dire qu'elles n'ont pas été tournées ou retournées, etc.), la valeur de retour des données du jeu de tuiles sera exactement égale à l'index de la tuile dans le jeu de tuiles. Vous pouvez donc créer des "cartes de collision en utilisant une tuile à l'index 1 dans le jeu de tuiles - par exemple - puis utiliser cette fonction pour vérifier 1 ou 0 (une tuile vide) pour calculer les collisions.

Syntaxe :

tilemap_get_at_pixel(tilemap_element_id, x, y) ;

Arguments Description
tilemap_element_id La valeur d'identification unique de l'élément de la carte des tuiles pour obtenir les données des tuiles.
x La position le long de l'axe des x pour obtenir les données de la tuile (en pixels de la pièce).
y La position le long de l'axe des y pour obtenir les données de la tuile (en pixels de la pièce)

Les retours :

Réel (-1 s'il y a une erreur)

Exemple :

var lay_id = layer_get_id("Tiles_sky") ;
var map_id = layer_tilemap_get_id(lay_id) ;
var data = tilemap_get_at_pixel(map_id, 64, 128) ;
data = tile_set_flip(data, true) ;
tilemap_set_at_pixel(map_id, data, 64, 128) ;

Le code ci-dessus obtient l'ID de la carte de tuiles "Nuages" sur la couche "Tuiles_Sky" et l'utilise ensuite pour obtenir les données de la tuile à la position (64, 128). Ces données de tuile sont ensuite retournées avant d'être utilisées pour définir la tuile sur la carte de tuiles à nouveau.
