tilemap_get_at_pixel

En utilisant cette fonction, vous pouvez récupérer les données de mosaïque à partir d'une position (dans la pièce) de l'élément de carte de mosaïque. Vous donnez l'ID d'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id() ) ainsi que la position x et y dans la pièce pour obtenir les données de tuile et la fonction renverra les données de tuile "blob". Ces données sont essentiellement un masque de bits qui contient l'index de la tuile, les booléens flip / rotate / mirror et toutes les données de masque qui ont été appliquées (voir tilemap_set_mask() pour plus de détails), et la valeur de données résultante peut ensuite être utilisée dans le tile_* fonctions pour modifier les propriétés d'une tuile. Si vous avez besoin d'obtenir les données de mosaïque à partir d'une cellule de mosaïque spécifique, vous devez utiliser la fonction tilemap_get() au lieu.

IMPORTANT! Si les tuiles de la carte de tuiles n'ont pas été modifiées (c'est-à-dire qu'elles ne sont pas tournées ou retournées, etc.), alors la valeur de retour des données du jeu de tuiles sera exactement égale à l'indice de la tuile sur le jeu de tuiles. Ainsi, vous pouvez créer des "cartes de collision" de tuiles en utilisant une tuile à l'index 1 dans l'ensemble de tuiles - par exemple - puis utiliser cette fonction pour vérifier 1 ou 0 (une tuile vide) pour calculer les collisions.

Syntaxe:

tilemap_get_at_pixel(tilemap_element_id, x, y);

Argument Description
tilemap_element_id La valeur d'ID unique de l'élément de carte de tuile pour obtenir les données de tuile de
x La position le long de l'axe des x à partir de laquelle obtenir les données de mosaïque (en pixels de la pièce)
y La position le long de l'axe y à partir de laquelle obtenir les données de mosaïque (en pixels de la pièce)

Retour:

Real (-1 if there is an error)

Exemple:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

Le code ci-dessus obtient l'ID de la carte de tuiles "Nuages" sur la couche "Tiles_Sky" et l'utilise ensuite pour obtenir les données de la tuile à la position (64, 128). Ces données de tuile sont ensuite retournées avant d'être utilisées pour redéfinir la tuile sur la carte de tuiles.
