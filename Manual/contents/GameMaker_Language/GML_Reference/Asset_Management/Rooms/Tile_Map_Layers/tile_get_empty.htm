tile_get_empty

Cette fonction peut être utilisée pour vérifier si un ensemble donné de données de tuile est pour une tuile vide ou non. Vous donnez les données de tuile, qui peuvent être récupérées à l'aide de la fonction tilemap_get(), et la fonction retournera true si la tuile est vide, ou false s'il existe un index de tuiles.

Syntaxe:

tile_get_empty(tiledata)

Argument Description
tiledata les données de tuile à vérifier

Retour:

Boolean

Exemple:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
for (var i = 0; i < tilemap_get_width(map_id); i++;)
    {
    for (var j = 0; j < tilemap_get_height(map_id); j++;)
        {
        var data = tilemap_get(map_id, i, j);
        if !tile_get_empty(data)
            {
            data = tile_set_empty(data)
            tilemap_set(map_id, data, i, j);
            }
        }
    }

Le code ci-dessus obtient l'ID de carte de tuile de la couche donnée, puis vérifie chaque cellule de tuile sur la carte pour voir si elle contient des données ou non. Si tel est le cas, la vignette est définie comme vide.
