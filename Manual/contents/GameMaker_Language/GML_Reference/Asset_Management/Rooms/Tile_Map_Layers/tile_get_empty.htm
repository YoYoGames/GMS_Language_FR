tuile_get_empty

Cette fonction peut être utilisée pour vérifier si un ensemble donné de données de tuiles correspond à une tuile vide ou non. Vous donnez les données des tuiles, qui peuvent être récupérées en utilisant la fonction tilemap_get()et la fonction retournera vrai si la tuile est vide, ou faux s'il y a un index de tuile.

Syntaxe :

tile_get_empty(tiledata)

Arguments Description
données en mosaïque les données de la tuile à vérifier

Les retours :

Booléen

Exemple :

var lay_id = layer_get_id("Tiles_sky") ;
var map_id = layer_tilemap_get_id(lay_id) ;
for (var i = 0 ; i < tilemap_get_width(map_id) ; i++ ;)
{
for (var j = 0 ; j < tilemap_get_height(map_id) ; j++ ;)
{
var data = tilemap_get(map_id, i, j) ;
if !tile_get_empty(data)
{
data = tile_set_empty(data)
tilemap_set(map_id, data, i, j) ;
}
}
}

Le code ci-dessus récupère l'ID de la carte des tuiles à partir de la couche donnée, puis vérifie chaque cellule de la carte pour voir si elle contient des données ou non. Si c'est le cas, la tuile est définie comme vide.
