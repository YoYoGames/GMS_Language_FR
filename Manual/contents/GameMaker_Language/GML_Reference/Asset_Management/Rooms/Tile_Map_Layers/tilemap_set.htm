tilemap_set

Cette fonction peut être utilisée pour définir n'importe quelle cellule (carré de la grille) dans l'élément de carte de tuiles sur la couche sur une nouvelle tuile. Vous donnez l'ID d'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id() ), puis les données de mosaïque à définir ainsi que la position dans la carte de mosaïques. Vous pouvez récupérer des données de mosaïque à l'aide de la fonction tilemap_get() puis utilisez le tile_ fonctions pour le changer avant de régler la cellule à l'aide de cette fonction. La fonction retournera true si la tuile a été correctement installée et false s'il y avait un problème et qu'il n'a pas été réglé.

Syntaxe:

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)

Argument Description
tilemap_element_id La valeur d'identifiant unique de l'élément de mosaïque à modifier
tiledata Les données de tuile à définir
xcell La position de la cellule (grille) à définir le long de l'axe des x
ycell La position de la cellule (grille) à définir le long de l'axe y

Retour:

Boolean

Exemple:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Le code ci-dessus obtient l'ID de la carte de tuiles sur la couche "Tiles_Sky" et l'utilise ensuite pour obtenir les données de la tuile à la cellule (0, 0). Ces données de tuile sont ensuite retournées avant d'être utilisées pour redéfinir la tuile sur la carte de tuiles.
