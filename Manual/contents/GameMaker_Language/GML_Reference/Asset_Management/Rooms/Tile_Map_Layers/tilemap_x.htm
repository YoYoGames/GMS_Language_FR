tilemap_x

Cette fonction contrôle la position sur l'axe des x de la pièce de l'élément de carte à carreaux de l'actif sur la couche. Vous donnez l'ID de l'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), puis définissez la valeur x à utiliser (en fonction des coordonnées de la pièce).

Syntaxe :

tilemap_x(tilemap_element_id, x) ;

Arguments Description
tilemap_element_id La valeur d'identification unique de l'élément de la carte des tuiles à modifier.
x La position x de la carte des tuiles

Les retours :

N/A

Exemple :

var lay_id = layer_get_id("Asset_sky") ;
var map_id = layer_tilemap_get_id(lay_id) ;
tilemap_x(map_id, irandom(room_width)) ;

Le code ci-dessus récupère la valeur d'identification de l'actif de la carte de carreaux attribué au calque "Asset_sky", puis définit sa position x à une valeur aléatoire comprise entre 0 et la largeur de la pièce.
