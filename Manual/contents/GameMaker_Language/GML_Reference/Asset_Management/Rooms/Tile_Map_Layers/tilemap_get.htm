tilemap_get

À l'aide de cette fonction, vous pouvez récupérer les données de mosaïque à partir d'une cellule de l'élément de carte de mosaïque. Vous donnez l'ID d'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id() ), ainsi que la position x et y de la cellule et la fonction renverra les données de tuile "blob". Ces données sont essentiellement un masque de bits qui contient l'index de la tuile, les booléens flip / rotate / mirror et toutes les données de masque qui ont été appliquées (voir tilemap_set_mask() pour plus de détails), et la valeur de données résultante peut ensuite être utilisée dans le tile_* fonctions pour modifier les propriétés d'une tuile. Notez que nous utilisons des positions de cellule x / y basées sur la taille de la grille de la carte de tuiles et non comme une position dans la pièce, donc "cell_x" est une valeur comprise entre 0 et la largeur de la carte de tuiles, et "cell_y" est une valeur de 0 à la hauteur de la carte de tuiles. Si vous avez besoin d'obtenir les données pour une tuile à une position de pièce spécifique, vous devez utiliser la fonction tilemap_get_at_pixel().

IMPORTANT! Si les tuiles de la carte de tuiles n'ont pas été modifiées (c'est-à-dire qu'elles ne sont pas tournées ou retournées, etc.), alors la valeur de retour des données du jeu de tuiles sera exactement égale à l'indice de la tuile sur le jeu de tuiles. Ainsi, vous pouvez créer des "cartes de collision" de tuiles en utilisant une tuile à l'index 1 dans l'ensemble de tuiles - par exemple - puis utiliser cette fonction pour vérifier 1 ou 0 (une tuile vide) pour calculer les collisions.

Syntaxe:

tilemap_get(tilemap_element_id, cell_x, cell_y);

Argument Description
tilemap_element_id La valeur d'ID unique de l'élément de carte de tuile pour obtenir les données de tuile de
x_cell La cellule le long de l'axe des x pour obtenir les données de la vignette
y_cell La cellule le long de l'axe des y pour obtenir les données de la vignette

Retour:

Real (-1 if there is an error)

Exemple:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Le code ci-dessus obtient l'ID de la carte de tuiles sur la couche "Tiles_Sky" et l'utilise ensuite pour obtenir les données de la tuile à la cellule (0, 0). Ces données de tuile sont ensuite retournées avant d'être utilisées pour redéfinir la tuile sur la carte de tuiles.
