tilemap_get_y

Cette fonction vous permet de récupérer la position y (dans la pièce) de l'élément de carte de carrelage. Vous donnez l'ID de l'élément de carte de carrelage (que vous obtenez lorsque vous créez un élément de carte de carrelage à l'aide de la fonction layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id()), et la fonction renvoie la position sur l'axe des y.

Syntaxe :

tilemap_get_y(tilemap_element_id) ;

Arguments Description
tilemap_element_id La valeur d'identification unique de l'élément de la carte de la tuile pour obtenir la position y de

Les retours :

Real

Exemple :

var lay_id = layer_get_id("Tiles_Walls") ;
var map_id = layer_tilemap_get_id(lay_id) ;
var _x = tilemap_get_x(map_id) ;
var _y = tilemap_get_y(map_id) ;
tilemap_x(map_id, _x + 10) ; tilemap_y(map_id, _y + 10) ;

Le code ci-dessus utilise l'ID de la carte des tuiles récupérée pour obtenir la position x et y de la carte des tuiles, puis utilise ces valeurs pour la déplacer.

Retour : Carrelage des couches de la carte
