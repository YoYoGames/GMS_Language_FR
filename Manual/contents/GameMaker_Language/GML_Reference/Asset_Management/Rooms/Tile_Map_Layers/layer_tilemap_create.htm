layer_tilemap_create

Avec cette fonction, vous pouvez affecter une ressource de jeu de tuiles à une couche à utiliser comme carte de tuiles dans votre projet. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances), puis une position initiale (x, y) pour ajouter la carte de tuiles à la pièce, la ressource de l'ensemble de tuiles à utiliser, puis la largeur et la hauteur de la carte de tuiles en cellules (c'est-à-dire: une largeur de 20 et une hauteur de 10 créera une carte de tuiles avec 200 cellules de 20 cellules de largeur et 10 cellules de hauteur), la taille des cellules elles-mêmes étant définie par le jeu de tuiles choisi. Il est à noter que vous ne pouvez pas placer de tuiles à des positions négatives dans la carte de tuiles, donc toutes les tuiles doivent être placées dans les espaces de cellule 0 à largeur - 1, 0 à hauteur - 1.

Syntaxe:

layer_tilemap_create(layer_id, x, y, tile set, width, height)

Argument Description
layer_id La valeur d'ID unique du calque à cibler (ou le nom du calque sous forme de chaîne)
x La position x de la carte de tuiles dans la pièce
y La position y de la carte de tuiles dans la pièce
tile set L'index du jeu de tuiles à utiliser
width La carte de tuiles de largeur (dans les cellules)
height La hauteur de la carte de tuiles (en cellules)

Retour:

tile map element ID (Real)

Exemple:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

Le code ci-dessus crée une nouvelle couche, puis y ajoute un nouvel élément de carte de tuiles, en définissant la position de la carte de tuiles sur (0,0) ainsi que le jeu de tuiles à utiliser et la largeur et la hauteur de la carte de tuiles.
