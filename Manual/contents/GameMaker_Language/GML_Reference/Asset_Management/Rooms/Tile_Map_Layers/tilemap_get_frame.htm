tilemap_get_frame

Étant donné que les tuiles peuvent être animées, il peut parfois être utile de savoir quelle image est actuellement dessinée et de réagir en conséquence, donc avec cette fonction, vous pouvez récupérer l'index de l'image actuelle pour une carte de tuiles donnée. Vous donnez l'ID d'élément de carte de tuiles (que vous obtenez lorsque vous créez un élément de carte de tuiles à l'aide de layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id() ), et la fonction renverra l'index de l'image.

Syntaxe:

tilemap_get_frame(tilemap_element_id)

Argument Description
tilemap_element_id La valeur d'ID unique de l'élément de mappage de tuiles pour obtenir l'index d'image de

Retour:

Real (between 0 (inclusive) and the maximum number of frames of animation (exclusive))

Exemple:

var lay_id = layer_get_id("Tiles_Traps");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_frame(map_id) >= 2 && tilemap_get_frame(map_id) < 4
    {
    global.activate = true;
    }
else
    {
    global.activate = false;
    }

Le code ci-dessus vérifie l'image d'animation actuelle pour la carte de tuiles donnée sur la couche donnée et définit une variable globale basée sur la valeur de retour.
