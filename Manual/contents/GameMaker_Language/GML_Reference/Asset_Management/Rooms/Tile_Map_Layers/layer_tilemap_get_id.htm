layer_tilemap_get_id

Cette fonction peut être utilisée pour récupérer la valeur d'ID unique de l'élément de carte de tuiles sur une couche. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ) ou le nom de la couche (sous forme de chaîne - cela aura un impact sur les performances) et la fonction renverra la valeur d'ID associée à l'élément de carte de tuiles sur la couche. Notez que cette fonction est spécialement conçue pour être utilisée avec les cartes de tuiles qui ont été ajoutées dans l'EDI, comme si vous ajoutiez une carte de tuiles à une couche via du code à l'aide de la fonction layer_tilemap_create(), puis il renverra l'ID unique de l'élément de mosaïque ajouté. Si l'ID de tilemap donné est incorrect ou si le tilemap n'existe pas, la fonction renverra -1.

Syntaxe:

layer_tilemap_get_id(layer_id)

Argument Description
layer_id La valeur d'ID unique du calque à cibler (ou le nom du calque sous forme de chaîne)

Retour:

tile map element ID value (Real) or -1

Exemple:

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

Le code ci-dessus obtiendra l'ID de couche pour la couche nommée "tilemap_trees" et l'utilisera ensuite pour obtenir l'ID de l'élément de carte de tuiles sur cette couche. Cet ID est ensuite utilisé pour supprimer la carte de tuiles de la couche.

Retour: Couches de carte de mosaïque
