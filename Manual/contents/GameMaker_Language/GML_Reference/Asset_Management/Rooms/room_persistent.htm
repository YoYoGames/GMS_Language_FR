chambre_persistante

Cette variable peut être utilisée pour obtenir et définir l'indicateur de persistance pour la room actuelle. Si elle est définie à true, la room est considérée comme persistante, dans ce cas, chaque fois que vous quittez la room et que vous revenez, l'état des instances de cette room aura été maintenu. En revanche, s'il est défini sur false, chaque fois que vous revenez dans la room, celle-ci sera réinitialisée à son état initial. Vous devez noter qu'une salle persistante utilise beaucoup plus de mémoire qu'une salle normale et il n'est pas recommandé d'en avoir trop dans votre système. d'en avoir trop dans votre jeu.

Syntaxe :

chambre_persistante ;

Les retours :

Booléen

Exemple :

si vies < 1
{
room_persistent = false ;
room_goto(rm_start) ;
}

Le code ci-dessus vérifie la variable "lives" et si elle est inférieure à 1, il met la persistance de la chambre à false et change ensuite de chambre.
