room_set_viewport

Avec cette fonction, vous pouvez définir les propriétés du port d'affichage pour n'importe quelle pièce de votre jeu, à l'exception de celle actuelle. Vous fournissez l'index de la salle pour définir le port de vue dans, l'index du port de vue (de 0 à 7) et ensuite si le port de vue doit être visible au début de la salle ou non (défini sur true pour rendre le port visible, et false autrement). Après cela, vous définissez les coordonnées x et y (correspondant à la position supérieure gauche du port de vue), puis la largeur et la hauteur en pixels.

Syntaxe:

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);

Argument Description
rm L'index de la pièce à définir
vind L'index du port de vue à définir
vis La visibilité du port de vue (true est visible, false is invisible )
xport La position x du port de vue dans la pièce
yport La position y du port de vue dans la pièce
wport La largeur (en pixels) du port de vue
hport La hauteur (en pixels) du port de vue

Retour:

N/A

Exemple:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

Cela définira les propriétés du port de vue [0] dans la salle indexée dans la variable "global.myroom".
