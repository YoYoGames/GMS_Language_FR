Sprites

Les sprites sont généralement les représentations visuelles des objets dans le jeu que vous créez, bien qu'ils puissent être utilisés à de nombreuses autres fins. Un sprite est soit une seule image, soit un ensemble d'images qui, lorsqu'elles sont lues les unes après les autres, ressemblent à une animation en boucle. Les sprites peuvent être créés dans de nombreux formats, le sprite "traditionnel" étant composé de cadres d'images pixel, importés dans GameMaker Studio 2 comme *.png des dossiers. Cependant, vous pouvez également importer des sprites vectoriels qui ont été enregistrés dans le *.swf format ou des sprites d'animation squelettiques qui ont été enregistrés sous *.json fichiers avec accompagnement *.atlas et *.png fichiers (pour plus d'informations sur ces types de fichiers image et comment les importer dans GameMaker Studio 2 veuillez consulter la section Importer des sprites non bitmap ).

Étant donné que les sprites font partie intégrante de n'importe quel jeu, GameMaker Studio 2 dispose d'un grand nombre de fonctions et de variables qui peuvent être utilisées pour obtenir des informations sur des sprites spécifiques, ou pour modifier ceux qui sont dans votre jeu, ou même pour charger des sprites personnalisés lors de l' runtime à partir d'un fichier image source.
