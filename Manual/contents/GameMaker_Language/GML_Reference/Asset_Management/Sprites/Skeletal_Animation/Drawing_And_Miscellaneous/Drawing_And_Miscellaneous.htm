Dessin et divers

Les fonctions suivantes sont des outils supplémentaires pour obtenir différents ensembles de données sur un sprite d'animation de squelette :

  • squelette_get_minmax
  • squelette_get_num_bounds
  • squelette_get_bounds

Vous pouvez également utiliser diverses fonctions de dessin conçues spécifiquement pour ces types de sprites (en dehors des fonctions ordinaires draw_sprite() ordinaires). Ces fonctions vous donnent plus de contrôle sur ce qui est dessiné et vous permettent de modifier des caractéristiques spécifiques à l'animation, telles que le timing et les skins :

  • squelette_collision_draw_set
  • dessiner_squelette
  • draw_skeleton_instance
  • dessiner_squelette_collision
  • draw_skeleton_time
