draw_skeleton_instance

Cette fonction est uniquement destinée à être utilisée avec des instances auxquelles des sprites ont été affectés créés à l'aide du programme d'animation squelettique Spine. Bien que vous puissiez dessiner ces sprites en utilisant les fonctions de dessin normales (comme draw_self() ), il peut arriver que vous souhaitiez dessiner une seule image ou afficher un changement de texture de la peau sans modifier réellement les propriétés de sprite d'une instance. Dans ces cas, vous pouvez dessiner le sprite qui a été affecté à l'instance avec cette fonction, où vous donnez l'ID d'instance unique, puis le nom de l'animation à partir de laquelle obtenir l'image (une chaîne, telle que définie dans le programme utilisé pour créer le sprite). Ensuite, vous donnez le nom du skin à utiliser (encore une fois, tel que défini lors de la création du sprite), et une valeur de cadre. La valeur d'image sert à indiquer à GameMaker Studio 2 partie de l'animation à afficher et est une valeur comprise entre 0 et le image_number. Les autres arguments sont les mêmes que ceux utilisés pour le dessin de sprite normal et changeront l'échelle, l'angle, la couleur de fusion et l'alpha de l'image d'animation finale dessinée.

ATTENTION! En raison de la façon dont les animations squelettiques sont interpolées entre diverses images «clés», il y a une surcharge supplémentaire associée à cet appel et il est donc recommandé d'éviter de dessiner des sprites basés sur des squelettes en utilisant cette méthode, sauf si c'est absolument nécessaire.

Syntaxe:

draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

Argument Description
instance L'index de l'instance à partir de laquelle dessiner le sprite.
animname Le nom de l'animation à partir de laquelle obtenir l'image (une chaîne).
skinname Le nom du skin à utiliser (une chaîne).
frame L'image d'animation à dessiner (de 0 à image_number - 1).
x La coordonnée x de l'endroit où dessiner le sprite.
y La coordonnée y de l'endroit où dessiner le sprite.
xscale La mise à l'échelle horizontale du sprite, sous forme de multiplicateur: 1 = mise à l'échelle normale, 0,5 est la moitié etc...
yscale La mise à l'échelle verticale du sprite, en tant que multiplicateur: 1 = mise à l'échelle normale, 0,5 est la moitié etc...
rot La rotation du sprite. 0 = normal, 90 = tourné de 90 degrés dans le sens inverse des aiguilles d'une montre, etc.
colour La couleur avec laquelle mélanger le sprite.
alpha L'alpha du sprite (de 0 à 1 où 0 est transparent et 1 opaque).

Retour:

N/A

Exemple:

draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1);

Le code ci-dessus dessinera la première image du sprite d'animation squelettique assigné à l'instance donnée, en utilisant le jeu d'animation "jump" et le skin défini comme "skin1".
