squelette_collision_draw_set

Avec cette fonction, vous pouvez activer(true) ou désactiver(false) l'affichage des données de collision pour le sprite d'animation squelettique actuel utilisé par l'instance. Si cette fonction est activée, la boîte de délimitation et le masque de collision précis seront dessinés comme des contours autour du sprite. seront dessinés comme des contours autour du sprite.

Syntaxe :

skeleton_collision_draw_set(flag) ;

Arguments Description
drapeau Défini à true pour activer le dessin, et false pour le désactiver.

Les retours :

N/A

Exemple :

if debug_mode == true
{
skeleton_collision_draw_set(true) ;
}

Le code ci-dessus vérifie si le jeu est exécuté en mode débogage et si c'est le cas, il bascule les données de collision du squelette pour l'instance à afficher.
