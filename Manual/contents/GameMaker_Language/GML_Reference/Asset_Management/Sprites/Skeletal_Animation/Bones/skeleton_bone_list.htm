liste de squelettes

Avec cette fonction, vous pouvez remplir une liste DS (pré-créée) avec tous les noms des os utilisés dans le sprite de l'animation squelettique. Les noms seront des chaînes de caractères et pourront ensuite être utilisés dans les autres fonctions d'os d'animation squelettique pour ces types de sprites.

Syntaxe :

skeleton_bone_list(sprite, list) ;

Arguments Description
sprite L'index du sprite de l'animation squelettique Spine pour obtenir la liste.
liste L'ID de la liste DS à remplir avec les noms des os.

Les retours :

N/A

Exemple :

bone_list = ds_list_create() ;
skeleton_bone_list(sprite_index, bone_list) ;

Le code ci-dessus crée une liste DS puis la remplit avec les noms des os pour une utilisation ultérieure.
