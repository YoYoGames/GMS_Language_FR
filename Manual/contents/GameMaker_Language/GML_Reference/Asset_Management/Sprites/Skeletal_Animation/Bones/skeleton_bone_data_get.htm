skeleton_bone_data_get

Votre animation squelettique est composée d'un certain nombre de "bones", que vous aurez définis et nommés dans votre programme d'animation. Cette fonction peut être utilisée pour obtenir des informations sur l'os nommé à tout moment et stocker les données dans une carte DS vide, que vous devez avoir créée auparavant. Notez que ces données font référence à la pose par défaut du squelette, et non à la pose actuelle en cours de dessin. Si vous avez besoin des données de la pose actuelle, utilisez skeleton_bone_state_get().

Lorsque vous appelez cette fonction, la carte sera remplie avec les touches suivantes:

  • "x": la position x locale de l'os par rapport à l'os parent.
  • "y": la position y locale de l'os par rapport à l'os parent.
  • "angle": La rotation locale de l'os par rapport à l'os parent.
  • "xscale": l'échelle horizontale locale de l'os, en référence à l'os parent.
  • «yscale»: l'échelle verticale locale de l'os, en référence à l'os parent.
  • "parent": le nom (une chaîne) de l'os parent.

Syntaxe:

skeleton_bone_data_get(bone, map);

Argument Description
bone Le nom (sous forme de chaîne) de l'os.
map La carte DS (créée précédemment) qui stocke les données d'os.

Retour:

N/A

Exemple:

bone_map = ds_map_create();
skeleton_bone_data_get("leftarm", bone_map);

Le code ci-dessus crée une carte DS, puis la remplit avec les données de l'os nommé "bras gauche".

Retour: Os
