liste d'animation du squelette

Avec cette fonction, vous pouvez remplir une liste DS (pré-créée) avec tous les noms des animations incluses dans le sprite d'animation squelettique. Les noms seront des chaînes de caractères et pourront ensuite être utilisés dans les autres fonctions d'animation pour ces types de sprites.

Syntaxe :

skeleton_animation_list(sprite, liste) ;

Arguments Description
sprite L'index du sprite de l'animation squelettique Spine pour obtenir la liste.
liste L'ID de la liste DS à remplir avec les noms des animations.

Les retours :

N/A

Exemple :

var list = ds_list_create() ;
skeleton_animation_list(sprite_index, list) ;
for (var i = 1 ; i < ds_list_size(list) ; i++ ;)
{
skeleton_animation_mix(list[| 0], list[| i], 0.5) ;
}
ds_list_destroy(list) ;

Le code ci-dessus crée une liste DS de tous les noms d'animation pour le sprite utilisé par l'instance. Il les parcourt ensuite en boucle et fixe la valeur de mixage de toutes les animations avec la première à 0.5.
