skeleton_animation_get

Avec cette fonction, vous pouvez obtenir le jeu d'animation actuel utilisé par votre sprite d'animation squelettique. La valeur de retour est une chaîne, qui sera le nom de l'ensemble tel que vous l'avez défini dans votre programme d'animation squelettique.

Syntaxe:

skeleton_animation_get();

Retour:

String

Exemple:

if keyboard_check_pressed(vk_space)
    {
    if skeleton_animation_get() != "jump"
        {
        skeleton_animation_set("jump");
        }
    }

Le code ci-dessus changera le jeu d'animation utilisé en jeu de "saut" lorsque la touche d'espace est pressée, mais seulement si le jeu actuel utilisé n'est pas déjà "saut".
