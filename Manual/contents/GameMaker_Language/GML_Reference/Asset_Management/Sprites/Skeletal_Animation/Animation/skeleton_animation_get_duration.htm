animation_squelette_get_duration

Cette fonction renvoie le temps nécessaire à l'exécution de l'animation donnée avant de revenir au début. La valeur de retour est exprimée en secondes.

Syntaxe :

skeleton_animation_get_duration(animname) ;

Arguments Description
nom de l'animateur Le nom (une chaîne) du jeu d'animation à utiliser.

Les retours :

Real

Exemple1 :

time += delta_time / 1000000 ;
var duration = skeleton_animation_get_duration(skeleton_animation_get()) ;
var frame = floor((image_number * (mTime / duration))) + 0,5) % nombre_d'images ;
indice_d'images = cadre ;
draw_self() ;

Le code ci-dessus définira l'image_index à la valeur correcte pour le sprite d'animation squelettique actuellement attribué.

Exemple2 :

time += delta_time / 1000000 ;
var d = skeleton_animation_get_duration("walk") ;
if time > d time -= d ;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_blanc) ;

Le code ci-dessus dessine le sprite d'animation squelettique donné en utilisant le delta-time pour définir l'image à dessiner.
