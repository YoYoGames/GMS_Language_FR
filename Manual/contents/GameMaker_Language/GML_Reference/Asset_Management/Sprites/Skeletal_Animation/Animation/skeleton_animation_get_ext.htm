animation_squelette_get_ext

Un seul sprite d'animation squelettique peut avoir plusieurs jeux d'animation, et ces jeux peuvent être affectés à différentes pistes afin que vous puissiez "mélanger et assortir" les jeux d'animation. Cette fonction renvoie le nom du jeu d'animation actuellement utilisé par le numéro de piste numéro de piste donné (tel que défini par la fonction skeleton_animation_set_ext).

Syntaxe :

skeleton_animation_get_ext(piste) ;

Arguments Description
piste Le numéro de la piste pour obtenir le nom du jeu d'animation.

Les retours :

Chaîne de caractères

Exemple :

if skeleton_animation_get_ext(1) != "bodyfight"
{
skeleton_animation_set_ext("bodyfight", 1) ;
}

Le code ci-dessus changera le jeu d'animation utilisé par la piste 1 en "bodyfight" s'il ne l'est pas déjà.
