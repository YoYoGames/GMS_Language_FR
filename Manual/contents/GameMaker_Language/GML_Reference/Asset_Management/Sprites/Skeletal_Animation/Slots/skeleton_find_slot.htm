skeleton_find_slot

Avec cette fonction, vous pouvez trouver les slots qui se trouvent à une position spécifiée de l'espace de la pièce dans le sprite Spine associé à l'instance actuelle. Vous créez une liste de DS et fournissez son ID ainsi qu'une position x/y à vérifier. avec une position x/y à vérifier et la liste sera remplie avec une chaîne de noms pour chacun des slots d'attachement disponibles pour le sprite (y compris toute modification d'attachement). Notez que la liste est toujours triée dans l'ordre décroissant en partant du l'emplacement le plus élevé.

Syntaxe :

squelette_find_slot(x, y, liste) ;

Arguments Description
x La position x dans la pièce à vérifier.
y La position y dans la pièce à vérifier.
liste L'ID de la liste DS à remplir avec les cartes DS.

Les retours :

N/A

Exemple :

slot_list = ds_list_create() ;
skeleton_find_slot(mouse_x, mouse_y, slot_list) ;

Le code ci-dessus crée une liste de DS et la remplit ensuite avec les noms des emplacements trouvés à la position de la souris dans la pièce.
