La structure des neuf tranches

Une structure Nine Slice peut être créée en utilisant sprite_nineslice_create(), ou récupéré à partir d'un sprite à l'aide de sprite_get_nineslice(). Cette structure contiendra des propriétés qui peuvent être modifiées et appliquées à un sprite en utilisant sprite_set_nineslice().

Si extrait d'une image - objet, l' évolution des valeurs dans cette struct affectera l'image - objet d' origine.

Les propriétés suivantes seront disponibles dans cette structure:

  • enabled: ce booléen sera vrai si Nine Slicing est activé pour le sprite, et faux s'il est désactivé. Vous pouvez le modifier pour activer ou désactiver Nine Slicing pour l'image-objet.
  • left: C'est le décalage du guide gauche sur l'axe x, par rapport au bord gauche de l'image-objet.
  • right: C'est le décalage du guide droit sur l'axe x, par rapport au bord droit de l'image-objet.
  • top: C'est le décalage du guide supérieur sur l'axe y, par rapport au bord supérieur de l'image-objet.
  • bottom: C'est le décalage du guide inférieur sur l'axe y, par rapport au bord inférieur de l'image-objet.
  • tilemode: Ceci renvoie un tableau contenant les modes de mosaïque pour les tranches de bord et de centre.

Le mode Tile de chaque tranche peut être lu ou modifié en utilisant ces constantes comme index du tableau:

Constant Description
nineslice_left La tranche du bord gauche
nineslice_top La tranche du bord supérieur
nineslice_right La tranche du bord droit
nineslice_bottom La tranche de bord inférieur
nineslice_centre La tranche centrale

Le mode mosaïque d'une tranche sera (ou peut être défini sur) l'une des constantes suivantes:

Constant Description
nineslice_stretch La tranche sera étirée
nineslice_repeat La tranche sera répétée
nineslice_mirror La tranche sera répétée par mise en miroir
nineslice_blank La tranche ne sera pas étirée ou répétée, ce qui entraînera une zone vide après elle
nineslice_hide La tranche n'apparaîtra pas du tout

Puisque le tilemode est un tableau, vous devez utiliser la syntaxe du tableau pour changer le mode de mosaïque d'une tranche. L'exemple de code ci-dessous définit le mode Mosaïque de la tranche supérieure sur Répéter:

Exemple:

var _nineslice = sprite_get_nineslice(spr_box);

_nineslice.tilemode[nineslice_top] = nineslice_repeat;
