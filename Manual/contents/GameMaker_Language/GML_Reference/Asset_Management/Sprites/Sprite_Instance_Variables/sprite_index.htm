sprite_index

Cette variable renvoie l'index du sprite actuel pour l'instance, ou -1 si l'instance n'a aucun sprite associé. Vous pouvez le modifier pour donner à l'instance un sprite différent en lui donnant le nom d'un sprite de l'arborescence des ressources ou en utilisant une variable dans laquelle un sprite chargé en externe est indexé. Changer le sprite ne change pas l'index du cadre actuellement visible, donc si vous changez le sprite sur l'image numéro 3, le nouveau sprite sera dessiné avec ce cadre visible (en supposant qu'il a le même nombre d'images).

Syntaxe:

sprite_index;

Retour:

Real (integer - resource index value)

Exemple:

with (obj_Check)
    {
    if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
        {
        sprite_index = spr_spotted;
        }
    else
        {
        sprite_index = spr_clear;
        }
    }

Le code ci-dessus boucle à travers toutes les instances de "obj_Check" vérifiant une ligne de collision entre elles et l'instance exécutant le code. Le sprite de ces instances sera modifié en fonction de la valeur de retour (vrai ou faux) pour la ligne de collision.

Retour: Variables d'instance de sprite
