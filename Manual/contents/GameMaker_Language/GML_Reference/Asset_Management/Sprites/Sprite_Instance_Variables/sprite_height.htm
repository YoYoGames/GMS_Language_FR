sprite_height

Cette variable en lecture seule renvoie la hauteur de l'image-objet qui a été affectée à l'instance. Cette hauteur est renvoyée en pixels et dépendra de la image_yscale. Si vous avez besoin de la hauteur non mise à l'échelle, vous devez utiliser sprite_get_height().

Syntaxe:

sprite_height;

Retour:

Real (single precision floating point value)

Exemple:

if sprite_height != sprite_get_height(sprite_index)
    {
    image_yscale = 1;
    }

Le code ci-dessus vérifie la hauteur du sprite affecté à l'instance exécutant le code par rapport à la hauteur de la ressource sprite et si ce n'est pas le même, il réinitialise le image_yscale à 1,
