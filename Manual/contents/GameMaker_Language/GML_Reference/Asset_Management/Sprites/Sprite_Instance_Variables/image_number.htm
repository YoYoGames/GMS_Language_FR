numéro de l'image

Cette variable en lecture seule peut être utilisée pour obtenir le nombre de sous-images dans un sprite qui a été attribué à une instance (si vous avez besoin du nombre de sous-images pour un sprite autre que celui attribué à l'instance, vous devez utiliser la variable devez utiliser sprite_get_number()). Veuillez noter que lorsqu'il y a (par exemple) 1 sous-image, cette variable retournera "1" mais l'image_index de cette sous-image est égal à 0.

Syntaxe :

numéro d'image ;

Les retours :

Réel (valeur à virgule flottante de simple précision)

Exemple :

si image_speed > 0
{
if image_index > image_number - 1 instance_destroy() ;
}

Le code ci-dessus vérifie si le sprite est en cours d'animation, et si c'est le cas, il vérifie ensuite si l'image_index actuelle est supérieure au nombre de sous-images et si c'est le cas, il détruit l'instance.
