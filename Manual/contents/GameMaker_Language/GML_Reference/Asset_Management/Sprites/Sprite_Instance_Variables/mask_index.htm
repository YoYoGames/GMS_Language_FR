mask_index

Lorsque vous définissez un objet dans GameMaker Studio 2 vous pouvez lui attribuer un masque à utiliser pour les collisions plutôt que celui qui correspond à l'image-objet définie. Cette variable peut être utilisée pour trouver lesprite_index de ce masque (ou il retournera -1 si aucun sprite n'a été attribué) ou pour définir le masque d'une instance sur le sprite choisi. La définition de l'index de masque signifie que vous pouvez avoir, par exemple, un sprite pour l'instance avec une forme irrégulière, tout en lui donnant un masque de collision circulaire qui est obtenu à partir d'un sprite différent.

Syntaxe:

mask_index;

Retour:

Real (integer - resource index value)

Exemple:

mask_index = spr_Round;

Le code ci-dessus définit le masque de l'instance sur celui du sprite "spr_Round".
