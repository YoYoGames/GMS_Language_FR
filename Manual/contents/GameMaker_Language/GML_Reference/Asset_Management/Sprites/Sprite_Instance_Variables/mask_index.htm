masque_index

Lorsque vous définissez un objet dans GameMaker Studio 2, vous pouvez lui attribuer un masque à utiliser pour les collisions plutôt que celui qui correspond au sprite défini. Cette variable peut être utilisée pour trouver le sprite_index de ce masque (ou elle retournera -1 si aucun sprite n'a été assigné) ou pour définir le masque d'une instance au sprite choisi. Définir l'index du masque signifie que vous pouvez avoir, par exemple, un sprite pour l'instance avec une forme irrégulière, et pourtant lui donner un masque de collision circulaire obtenu à partir d'un sprite différent.

Syntaxe :

mask_index ;

Les retours :

Real (nombre entier - valeur de l'index de la ressource)

Exemple :

mask_index = spr_Round ;

Le code ci-dessus définit le masque de l'instance à celui du sprite "spr_Round".
