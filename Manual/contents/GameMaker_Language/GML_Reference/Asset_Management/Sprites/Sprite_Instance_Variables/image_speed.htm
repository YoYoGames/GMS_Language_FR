vitesse de l'image

Cette variable détermine la vitesse à laquelle GameMaker Studio 2 fera défiler les sous-images du sprite d'instance actuel. La valeur de la vitesse est un multiplicateur, 1 étant la valeur par défaut. Si vous la réglez sur 0,5, la vitesse de l'animation sera réduite de moitié (comme définie dans l'éditeur de sprites ou l'éditeur d'images). La valeur de vitesse indiquée est un multiplicateur, 1 étant la valeur par défaut. Si le sprite utilisé n'a n'a pas de sous-images, cette variable n'aura aucun effet.

Syntaxe :

image_speed ;

Les retours :

Réel (valeur à virgule flottante de précision simple)

Exemple :

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
image_speed = 0.5 ;
}

Le code ci-dessus crée une instance de l'objet "obj_Explosion" et définit son image_speed à 0,5 (ce qui réduit de moitié la vitesse de déplacement).
