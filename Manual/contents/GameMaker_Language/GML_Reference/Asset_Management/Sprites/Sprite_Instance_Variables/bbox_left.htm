bbox_left

Cette variable en lecture seule renvoie la position (le long de l'axe des x) dans la pièce du cadre de sélection de gauche pour l'instance, où le cadre de sélection est défini par la largeur et la hauteur maximales du masque pour l'instance (telles que définies par lesprite_index ou par le mask_index ). Même lorsqu'un sprite a un masque de collision précis, le cadre de sélection existe et est utilisé pour certaines choses, et vous pouvez donc utiliser cette variable pour le trouver. Veuillez noter que lorsque l'instance n'a pas de sprite affecté, la valeur renvoyée sera la même que la position X de l'instance.

Syntaxe:

bbox_left;

Retour:

Real (integer)

Exemple:

if bbox_left < 0
    {
    x = sprite_xoffset;
    }

Le code ci-dessus vérifie si la boîte englobante de l'instance se trouve à l'extérieur de la pièce et si c'est le cas, il définit l'instance sur une nouvelle position.
