Variables d'instance de sprite

Ces variables sont incluses par défaut avec GameMaker Studio 2 et sont tous liés à l'image-objet assignée à l'instance dans la salle. Ces variables sont accessibles à tout moment dans le code ou les actions d'une instance et utilisées pour modifier ce qui est dessiné et comment.

  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • image_alpha
  • image_angle
  • image_blend
  • image_index
  • image_number
  • image_speed
  • image_xscale
  • image_yscale
  • mask_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top
