sprite_yoffset

Cette variable en lecture seule renvoie le décalage y local (la composante y de l'origine telle que définie dans l'éditeur de sprite) du sprite qui a été attribué à l'instance. Ce décalage est renvoyé en pixels et dépendra de la valeur de la variable image_yscale, de sorte que (par exemple) si un sprite carré de 32x32 est défini comme ayant une origine y à 16, lorsque ce sprite est mis à l'échelle par 2, cette variable renverra 32. Si vous avez besoin de l'origine y non mise à l'échelle, vous devez utiliser sprite_get_yoffset().

Syntaxe :

sprite_yoffset ;

Les retours :

Réel (valeur à virgule flottante de précision simple)

Exemple :

si y - sprite_yoffset < 0
{
y = sprite_yoffset ;
}

Le code ci-dessus vérifie si le bord supérieur du sprite de l'instance est à l'extérieur de la pièce et si c'est le cas, il déplace l'instance afin que le sprite entier soit visible.
