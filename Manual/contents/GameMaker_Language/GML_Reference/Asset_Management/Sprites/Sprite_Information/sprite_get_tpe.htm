sprite_get_tpe

Cette fonction renverra une valeur spéciale qui se rapporte à la position du sprite et de la sous-image donnés sur la page de texture utilisée par votre jeu. De cette façon, vous pouvez transmettre la valeur à des extensions à utiliser en dehors du jeu compilé, en contournant efficacement les GameMaker Studio 2 et en permettant au sprite d'être utilisé dans le contenu DOM qui peut ensuite être dessiné n'importe où dans la fenêtre contenant le canevas du jeu.. Cette fonction est particulièrement intéressante pour ceux qui souhaitent créer des boutons et autres médias interactifs en dehors de l' GameMaker Studio 2 sur leur page hôte à l'aide de la fonction clickable_add().

REMARQUE: cette fonction est pour HTML5 uniquement.

Syntaxe:

sprite_get_tpe(sprite, index);

Argument Description
sprite L'index du sprite pour trouver l'entrée de la page de texture.
index L'index d'image du sprite.

Retour

Real

Exemple:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Le code ci-dessus crée un bouton DOM à la position (32, 32) de la page sur laquelle le canevas de jeu s'exécute. Le bouton utilise le sprite référencé à partir de la page de texture comme "spr_MS_Home" et lorsque vous cliquez dessus, le bouton ouvrira une nouvelle fenêtre pour l'URL spécifiée et avec les propriétés définies pour cette fenêtre.
