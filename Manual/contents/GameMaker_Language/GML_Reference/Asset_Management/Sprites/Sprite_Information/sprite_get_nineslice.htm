sprite_get_nineslice

Cette fonction renvoie la structure Nine Slice d'un sprite, contenant toutes ses propriétés Nine Slice telles que définies dans l'éditeur de sprites ou à l'aide de sprite_set_nineslice(). Le contenu de cette structure est détaillé sur cette page.

Si une structure Nine Slice n'est pas affectée au sprite fourni, une nouvelle structure avec des propriétés Nine Slice par défaut sera créée pour le sprite et renvoyée.

La modification de toute valeur dans cette structure modifiera les propriétés Nine Slice du sprite d'origine, ce qui affectera tous les futurs appels de dessin effectués avec ce sprite.

Syntaxe :

sprite_get_nineslice(ind) ;

Arguments Description
ind L'index du sprite à partir duquel la structure Nine Slice sera récupérée.

Les retours :

Structure (ou -1 si le sprite n'existe pas)

Exemple :

var _box_nineslice = sprite_get_nineslice(spr_box_0) ;

_box_nineslice.enabled = true ;
_box_nineslice.left = 10 ;
_box_nineslice.right = 10 ;
_box_nineslice.top = 10 ;
_box_nineslice.bottom = 10 ;

Le code ci-dessus récupère la structure Nine Slice d'un sprite, active Nine Slicing pour celui-ci et définit les décalages du guide.
