sprite_get_name

Cette fonction renverra le nom sous forme de chaîne du sprite spécifié. Ce nom est celui qui a été spécifié pour le sprite dans l'arborescence des ressources de la GameMaker Studio 2. Veuillez noter qu'il ne s'agit que d' une chaîne et ne peut pas être utilisée pour référencer directement le sprite - pour cela, vous auriez besoin de l' index du sprite. Vous pouvez cependant utiliser cette chaîne pour obtenir l' index du sprite en utilisant la chaîne retournée avec la fonction asset_get_index().

Syntaxe:

sprite_get_name(index);

Argument Description
index L'index du sprite pour lequel obtenir le nom.

Retour

String

Exemple:

str = sprite_get_name(sprite_index);

Le code ci-dessus obtiendra le nom de l'index du sprite pour l'instance exécutant le code et stockera la valeur de retour dans la variable "str".
