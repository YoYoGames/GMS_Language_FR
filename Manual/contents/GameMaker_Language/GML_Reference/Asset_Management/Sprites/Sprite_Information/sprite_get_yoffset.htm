sprite_get_yoffset

Lorsque vous définissez un sprite dans l'éditeur de sprites, vous devez définir l'origine de ce sprite. Il s'agit du point auquel le sprite sera "attaché" ou "dessiné" lorsqu'il sera utilisé avec une instance. Cette fonction renvoie le décalage relatif pour l'axe des y de l'origine, basé sur le coin supérieur gauche étant la position 0,0, avec un +y étant vers le bas et un -y étant vers le haut. L'image suivante illustre cela :

Syntaxe :

sprite_get_yoffset(index) ;

Arguments Description
indice L'index du sprite pour trouver le décalage en y.

Renvoie à

Real

Exemple :

si y - sprite_get_yoffset(sprite_index) < 0
{
y = sprite_get_yoffset(sprite_index) ;
}

Le code ci-dessus garantit qu'une instance est maintenue dans la pièce en fonction du sprite (de sorte que le sprite soit toujours visible).
