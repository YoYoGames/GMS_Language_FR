sprite_get_uvs

Cette fonction renvoie un tableau contenant les coordonnées UV et d'autres données pour la texture de la sous-image du sprite sur la page de texture. La fonction renvoie un tableau avec les 8 éléments suivants :

  • [0] = gauche
  • [1] = haut
  • [2] = droite
  • [3] = fond
  • 4] = nombre de pixels que le compilateur d'actifs a enlevé du côté gauche du sprite.
  • 5] = nombre de pixels que le compilateur d'actifs a enlevé de la partie supérieure du sprite.
  • 6] = pourcentage normalisé des données de pixel de la largeur originale du sprite qui ont été sauvegardées dans la page de texture.
  • 7] = pourcentage normalisé des données de pixels de la hauteur du sprite original qui ont été sauvegardées dans la page de texture.

Ce tableau peut ensuite être utilisé dans d'autres fonctions de dessin, en particulier lorsque vous travaillez en 3D ou que vous utilisez les fonctions primitives 2D, ainsi que lorsque vous travaillez avec les fonctions Shader fonctions.

REMARQUE: cette fonction ne fonctionne pas avec les sprites vectoriels ou les sprites d'animation de squelette.

Syntaxe :

sprite_get_uvs(sprite, subimage) ;

Arguments Description
sprite L'index du sprite à utiliser.
sous-image La sous-image du sprite à utiliser.

Renvoie à

Tableau (1D, 8 éléments)

Exemple :

var tex = sprite_get_uvs(sprite, 0) ;
tex_left = tex[0] ;
tex_top = tex[1] ;
tex_right = tex[2] ;
tex_bottom = tex[3] ;

Le code ci-dessus stocke les coordonnées UV du sprite donné dans un tableau local, puis affecte les valeurs aux variables d'instance.
