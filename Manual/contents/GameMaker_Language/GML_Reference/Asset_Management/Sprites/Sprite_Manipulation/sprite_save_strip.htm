sprite_save_strip

Cette fonction créera une bande d'image à partir de toutes les sous-images d'un sprite, en la sauvegardant sur le disque avec le nom de fichier spécifié. Le sprite doit avoir été ajouté au moment de l'exécution (vous ne pouvez pas sauvegarder les sprites ajoutés via l'IDE) et le fichier doit être enregistré avec une extension *.png. doit être enregistré avec une extension *.png.

REMARQUE: selon la plate-forme cible choisie, vous êtes limité quant à l'endroit où vous pouvez enregistrer et charger des fichiers. Voir Manipulation des fichiers pour plus d'informations.

Syntaxe :

sprite_save_strip(ind, filename) ;

Arguments Description
ind L'index du sprite dont il faut modifier la carte de transparence.
nom de fichier Le nom de fichier de la bande de sprites enregistrée.

Renvoie à

N/A

Exemple :

var spr, i ;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16) ;
for (i = 1 ; i < 8 ; i +=1)
{
sprite_add_from_surface(spr, i, 0, 32, 32, vrai, vrai, 16, 16) ;
}
sprite_save_strip(spr, "Custom_Player_Sprite.png") ;
sprite_delete(spr) ;

Le code ci-dessus crée un sprite à partir de la surface indexée dans la variable "surf", en affectant son index à la variable locale "spr_Custom", puis utilise une boucle for pour se déplacer sur la surface et capturer diverses sections qui sont ajoutées au sprite en tant que sous-images. Ce nouveau sprite est ensuite enregistré sous forme de bande PNG avant d'être supprimé de la mémoire.
