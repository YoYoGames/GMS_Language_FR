sprite_set_speed

Cette fonction peut être utilisée pour définir la vitesse et le type d'animation de base pour la ressource sprite. Ces paramètres sont normalement définis dans l'éditeur de sprites, mais il peut arriver que vous deviez les définir manuellement pour une ressource sprite donnée. de les définir manuellement pour une ressource sprite donnée. Vous devez fournir l'index du sprite à définir, la vitesse d'animation et le type de vitesse d'animation à appliquer. Le type de vitesse d'animation peut être l'une des constantes suivantes :

Constant Description
spritespeed_framespersecond Le sprite a été défini avec une animation en images par seconde.
spritespeed_framespergameframe Le sprite a été défini avec une animation en frames par frame de jeu.


La sélection du type de vitesse à utiliser pour calculer les animations est très importante car il existe une énorme différence entre les deux options. Les deux exemples suivants illustrent cette différence :

  • Si vous avez un sprite qui dessine 1 image par seconde et que vous réglez la vitesse de l'image sur 0,5, il sera dessiné à 0,5 image par seconde. Si la fréquence d'images de votre jeu est de 60 images par seconde, le sprite dessinera une image toutes les 120 images du jeu. images du jeu.
  • Si vous avez un sprite qui dessine 1 image par image du jeu et que vous réglez la vitesse de l'image sur 0,5, il dessinera 0,5 image par image du jeu. Si la fréquence d'images de votre jeu est de 60 images par seconde, le sprite affichera 30 images pour 60 images de jeu. toutes les 60 images du jeu.

Vous pouvez trouver la valeur de la vitesse utilisée pour l'animation du sprite en utilisant la fonction sprite_get_speed()et vous pouvez obtenir le type de vitesse d'animation à l'aide de la fonction sprite_get_speed_type().

IMPORTANT ! Cette fonction affectera la ressource sprite, donc tout changement effectué avec cette fonction affectera toutes les instances qui utilisent le sprite et toutes les instances futures également. instances futures.

Syntaxe :

sprite_set_speed(index, vitesse, type)

Arguments Description
indice L'index du sprite dont on veut obtenir le type d'animation

Les retours :

Constant (voir ci-dessus)

Exemple :

sprite_set_speed(sprite_index, 10, spritespeed_framesperseconde) ;

Le code ci-dessus définit la vitesse d'animation de base du sprite affecté à l'instance à 10 images par seconde.
