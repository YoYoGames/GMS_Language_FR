sprite_set_speed

Cette fonction peut être utilisée pour définir la vitesse et le type d'animation de base comme pour la ressource de sprite. Ceci est normalement défini dans l' éditeur de sprite, mais il peut y avoir des moments où vous devez le définir manuellement pour une ressource de sprite donnée. Vous devez fournir l'index du sprite à définir, la vitesse d'animation et le type de vitesse d'animation à appliquer. Le type de vitesse d'animation peut être l'une des constantes suivantes:

Constant Description
spritespeed_framespersecond Le sprite a été défini avec une animation en images par seconde.
spritespeed_framespergameframe Le sprite a été défini avec une animation en images par image de jeu.


La sélection du type de vitesse à utiliser pour calculer les animations est très importante car il existe une énorme différence entre les deux options. Les deux exemples suivants illustrent cette différence:

  • Si vous avez un sprite qui dessine 1 image par seconde et définissez la vitesse de l'image à 0,5, il dessinera à 0,5 image par seconde. Si la fréquence d'images de votre jeu est de 60 images par seconde, le sprite dessinera 1 image pour 120 images de jeu.
  • Si vous avez un sprite qui dessine 1 image par image de jeu et définissez la vitesse d'image sur 0,5, il dessinera 0,5 image par image de jeu. Si la fréquence d'images de votre jeu est de 60 images par seconde, alors le sprite dessinera 30 images pour 60 images de jeu.

Vous pouvez savoir quelle était la valeur de vitesse utilisée pour l'animation de sprite en utilisant la fonction sprite_get_speed(), et vous pouvez obtenir le type de vitesse d'animation en utilisant sprite_get_speed_type().

IMPORTANT! Cette fonction affectera la ressource de sprite donc toutes les modifications que vous apportez avec cette fonction affecteront toutes les instances qui utilisent le sprite et toutes les instances futures aussi.

Syntaxe:

sprite_set_speed(index, speed, type)

Argument Description
index L'index du sprite pour obtenir le type d'animation de

Retour:

Constant (see above)

Exemple:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

Le code ci-dessus définit la vitesse d'animation de base pour le sprite affecté à l'instance à 10 images par seconde.
