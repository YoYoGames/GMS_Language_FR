sprite_flush_multi

Avec cette fonction, vous pouvez supprimer de la mémoire de texture un nombre quelconque de pages de texture pour les sprites donnés. Vous fournissez les indices des sprites sous la forme d'un tableau et les pages de texture qui leur sont attribuées seront supprimées de la mémoire de texture. Notez que si l'une des pages de texture est utilisée ailleurs dans la room (par un autre sprite d'instance ou un arrière-plan, etc...), vous pouvez avoir un léger impact sur les performances, car la page est immédiatement rechargée en mémoire, il faut donc faire attention en utilisant cette fonction. Notez que la fonction retournera -1 si le flush n'est pas supporté pour les ressources choisies, ou elle retournera 0 si tout a fonctionné correctement.

Syntaxe :

sprite_flush_multi(array) ;

Arguments Description
tableau Tableau avec les indices des sprites à purger

Les retours :

Int (-1 ou 0)

Exemple :

spr_a[0] = spr_Player_Aura1 ;
spr_a[1] = spr_Player_Aura2 ;
spr_a[2] = spr_Player_Aura3 ;
spr_a[3] = spr_Player_Aura4 ;
sprite_flush_multi(spr_a) ;

Le code ci-dessus crée un tableau dont chaque élément contient un index de sprite. Ce tableau est ensuite utilisé pour effacer ces textures de sprites de la mémoire.
