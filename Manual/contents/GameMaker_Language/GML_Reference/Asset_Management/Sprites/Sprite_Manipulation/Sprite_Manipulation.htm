Manipulation des sprites

Les fonctions ci-dessous sont toutes liées à la modification de l'actif de sprite (ressource) lui-même. Cela signifie que la majorité de ces fonctions affectera toutes les instances qui référencent le sprite en cours de modification, cela signifie également que vous devez prendre grand soin de vous assurer qu'un sprite existe avant de le référencer, ou de vous assurer qu'il n'est pas référencé avant de le modifier.
