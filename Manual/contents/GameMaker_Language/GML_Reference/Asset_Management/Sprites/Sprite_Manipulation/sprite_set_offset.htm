sprite_set_offset

Cette fonction peut être utilisée pour définir l'origine x et y d'un sprite, et prend des valeurs relatives basées sur la position (0,0) qui est le coin supérieur gauche du sprite. L'image suivante illustre cela :

REMARQUE : Cette fonction affecte l'actif du sprite de sorte que toutes les autres instances avec ce sprite auront le même décalage.

Syntaxe :

sprite_set_offset(ind, xoff, yoff) ;

Arguments Description
ind L'index du sprite dont il faut modifier le décalage.
xoff La position x de l'origine.
yoff La position y de l'origine.

Renvoie à

N/A

Exemple :

sprite_assign(spr_Custom, spr_Base) ;
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base)) ;

Le code ci-dessus affecte le sprite indexé dans "spr_Base" au sprite indexé dans "spr_Custom", puis utilise les valeurs de décalage x et y de "spr_Base" pour définir l'origine du nouveau sprite.
