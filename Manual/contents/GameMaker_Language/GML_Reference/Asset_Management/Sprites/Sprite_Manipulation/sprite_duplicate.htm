sprite_duplicate

Cette fonction renverra l'index d'un sprite nouvellement créé qui est une copie (copie) de la première entrée en tant qu'argument "index". Si la fonction échoue, -1 est renvoyé. Cette fonction doit être utilisée pour copier les sprites des ressources d'origine avant que des transformations puissent être effectuées sur eux. Par exemple, si vous souhaitez modifier le cadre de sélection d'un sprite ou définir son alpha à partir d'un autre sprite, vous devez d'abord le dupliquer, puis effectuer l'opération sur le sprite dupliqué et l'utiliser. Un sprite dupliqué sera placé sur sa propre page de texture unique lors de sa création, ce qui signifie que la duplication de plusieurs sprites créera plusieurs pages de texture et aura un impact sur les performances, utilisez donc cette fonction uniquement lorsque cela est nécessaire.

REMARQUE: lorsque vous dupliquez un sprite dans GameMaker Studio 2 vous devez vous rappeler de le supprimer à nouveau (avec sprite_delete () ) lorsqu'il n'est plus nécessaire, sinon il y a un risque de fuite de mémoire qui ralentira et finira par planter votre jeu.

Syntaxe:

sprite_duplicate(index);

Argument Description
index L'index du sprite à dupliquer.

Retour

Real (unique sprite ID value)

Exemple:

new_spr = sprite_duplicate(sprite_index)

Le code ci-dessus duplique le sprite actuellement utilisé comme index de sprite de l'instance et stocke l'index de ce nouveau sprite dans la variable "new_spr".
