sprite_duplicate

Cette fonction renverra l'index d'un sprite nouvellement créé qui est un duplicata (copie) de celui qui est entré comme argument "index". Si la fonction échoue, -1 est renvoyé. Cette fonction doit être utilisée pour copier tout sprites des actifs d'origine avant que toute transformation puisse être effectuée sur eux. Par exemple, si vous souhaitez modifier le rectangle de délimitation d'un sprite, ou définir son alpha à partir d'un autre sprite, vous devez d'abord le dupliquer, puis effectuer l'opération sur le sprite dupliqué et l'utiliser. dupliqué et l'utiliser. Un sprite dupliqué sera placé sur sa propre page de texture unique lors de sa création, ce qui signifie que la duplication de plusieurs sprites créera de multiples pages de texture et aura un impact sur les performances, donc utilisez cette fonction uniquement que lorsque cela est nécessaire.

REMARQUE: Lorsque vous dupliquez un sprite dans GameMaker Studio 2, vous devez vous souvenir de le supprimer à nouveau (avec la commande sprite_delete()) lorsqu'il n'est plus nécessaire, sinon il y a un risque de fuite de mémoire qui ralentira et finira par faire planter votre jeu.

Syntaxe :

sprite_duplicate(index) ;

Arguments Description
indice L'index du sprite à dupliquer.

Renvoie à

Réel (valeur d'identification unique du sprite)

Exemple :

nouveau_spr = sprite_duplicate(sprite_index)

Le code ci-dessus duplique le sprite actuellement utilisé comme index de sprite de l'instance et stocke l'index de ce nouveau sprite dans la variable "new_spr".
