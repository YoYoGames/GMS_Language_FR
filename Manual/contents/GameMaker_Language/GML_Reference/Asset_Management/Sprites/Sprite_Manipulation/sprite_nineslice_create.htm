sprite_nineslice_create

Ledécoupage en neuf est une technique utilisée pour mettre à l'échelle des images rectangulaires afin de préserver les détails, ce qui leur permet de conserver leur forme originale après la mise à l'échelle. Voir cette page pour des informations sur le découpage en neuf.

Cette fonction est utilisée pour créer une structure Nine Slice qui peut être modifiée et ensuite appliquée à un sprite. Le contenu de cette structure est détaillé sur cette page.

Vous pouvez stocker cette structure dans une variable, modifier ses propriétés et l'appliquer à n'importe quel sprite en utilisant sprite_set_nineslice().

Syntaxe :

sprite_nineslice_create() ;

Les retours :

Structure

Exemple :

var _nineslice = sprite_nineslice_create() ;

_nineslice.enabled = true ;
_nineslice.left = 10 ;
_nineslice.right = 10 ;
_nineslice.top = 10 ;
_nineslice.bottom = 10 ;

sprite_set_nineslice(spr_box_0, _nineslice) ;

Le code ci-dessus crée une nouvelle structure Nine Slice, l'active et définit les décalages pour les guides. La structure est ensuite appliquée à un sprite, modifiant ses propriétés Nine Slice.
