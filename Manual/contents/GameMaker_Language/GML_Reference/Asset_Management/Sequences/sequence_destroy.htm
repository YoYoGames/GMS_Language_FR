séquence_destruction

Cette fonction permet de détruire un objet séquence qui a été créé dynamiquement. Vous fournissez soit la structure de l'objet séquence (telle que retournée par la fonction séquence_créer()) ou l'ID de la séquence (tel que retourné par la fonction layer_sequence_get_sequence() ou à partir de la propriété sequence de la structure d'instance de la séquence). Cette fonction doit être utilisée lorsqu'une séquence créée dynamiquement n'est plus nécessaire afin de libérer de l'espace. n'est plus nécessaire afin de libérer la mémoire qui lui est associée.

Syntaxe :

sequence_destroy(sequence_struct_or_id) ;

Arguments Description
séquence_structure_ou_id La structure ou l'ID de l'objet de la séquence à détruire

Les retours :

N/A

Exemple :

if sequence_exists(my_seq)
{
sequence_destroy(my_seq) ;
}

Le code ci-dessus vérifie si l'objet séquence donné existe et si c'est le cas, il est détruit.
