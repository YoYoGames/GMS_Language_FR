sequence_instance_override_object

Avec cette fonction, vous pouvez remplacer (remplacer) toutes les instances d'un objet utilisé dans une séquence par une autre. Vous fournissez l'ID de structure d'instance de séquence (tel que renvoyé lorsque l'instance de séquence a été créée dans la pièce ou en utilisant l'une des fonctions de couche de pièce - voir ici ), ainsi que l'index d'objet (tel que défini dans le navigateur d'actifs) pour l'objet que vous souhaitez remplacer. Enfin, vous donnez un index d'objet ou un ID d'instance à utiliser comme objet qui va remplacer la séquence (la fourniture d'un ID d'instance utilisera simplement l'objet à partir duquel l'instance a été créée comme remplacement). Notez que cela ne peut être fait que sur des instances de séquence (pas sur des objets de séquence) et doit être fait avant que la séquence ne commence à jouer, sinon cela ne fonctionnera pas.

Syntaxe:

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

Argument Description
sequence_instance_struct Structure d'instance de séquence à modifier.
object_id Index d'objet de l'objet dans la séquence à remplacer.
instance_or_object_id L'index d'objet ou l'ID d'instance à utiliser pour remplacer les objets de séquence.

Retour:

N/A

Exemple:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq); sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

Le code ci-dessus crée une nouvelle instance de séquence sur le calque donné, puis la modifie afin que toutes les instances de l'objet "obj_Trees_Winter" soient remplacées par des instances de l'objet "obj_Trees_Summer".
