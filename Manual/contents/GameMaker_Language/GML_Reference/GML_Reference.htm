Référence du GameMaker Language

Cette section du manuel est un guide de référence pour le GameMaker Studio 2 ( GML ). Vous pouvez trouver toutes les fonctions disponibles documentées ici, ainsi que les arguments requis et des exemples de code pour montrer comment elles peuvent être utilisées. Pour plus d'informations sur l'utilisation de GML veuillez consulter la section Présentation de GML du manuel.

REMARQUE: Partout où le manuel a "N / A" comme valeur de retour pour une fonction, cela signifie que la fonction n'est pas censée renvoyer une valeur dans le cadre de son opération. Si vous essayez de vérifier une valeur de retour à partir de ces fonctions, vous obtiendrez la valeur undefined.
