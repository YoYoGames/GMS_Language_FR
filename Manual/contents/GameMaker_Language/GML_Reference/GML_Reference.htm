Référence dulangage GameMaker

Cette section du manuel est un guide de référence pour le logiciel GameMaker Studio 2(GML). Toutes les fonctions disponibles sont documentées ici, avec les arguments requis et des exemples de code pour montrer comment les utiliser. Pour plus d'informations sur l'utilisation du GML, consultez la section Présentation deGML du manuel.

REMARQUE: si le manuel indique "N/A" comme valeur de retour d'une fonction, cela signifie que la fonction n'est pas censée renvoyer de valeur dans le cadre de son fonctionnement. Si vous essayez de vérifier la valeur de retour de ces fonctions, vous obtenez la valeur suivante vous obtiendrez la valeur undefined.
