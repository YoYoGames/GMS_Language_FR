réseau_connexion_asynchrone

Avec cette fonction, vous pouvez envoyer une requête pour vous connecter à un serveur. La fonction prend l'identifiant de la socket à utiliser pour se connecter (voir réseau_créer_socket()) et vous demande d'indiquer l'adresse IP à laquelle se connecter (une chaîne de caractères) ainsi que le nom du serveur. Si la connexion échoue, une valeur inférieure à 0 sera renvoyée. La connexion utilise un protocole spécial qui garantit que seuls les jeux GameMaker Studio 2 se connectent les uns aux autres. de vous connecter à un serveur qui n'est pas un jeu GameMaker Studio 2, vous pouvez utiliser la commande network_connect_async_raw(). Notez que cette fonction est asynchrone et génère un événement de mise en réseau asynchrone de type network_type_non_blocking_connect.

Syntaxe :

network_connect_async(socket, url, port) ;

Arguments Description
prise de courant L'identifiant du socket à utiliser.
url L'URL ou l'IP à laquelle se connecter (une chaîne).
port Le port auquel se connecter.

Les retours :

Real

Exemple :

client = network_create_socket(network_socket_tcp) ;
network_connect_async(client, "192.134.0.1", 6510) ;

Le code ci-dessus créera un nouveau socket TCP puis tentera d'établir une connexion asynchrone à travers celui-ci vers l'adresse IP donnée sur le port 6510.
