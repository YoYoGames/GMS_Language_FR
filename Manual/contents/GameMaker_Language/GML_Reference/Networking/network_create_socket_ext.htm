réseau_créer_socket_ext

Cette fonction est utilisée pour créer un nouveau socket client pour que votre jeu puisse communiquer sur le réseau. Vous devez définir le protocole de la socket (voir la liste des constantes ci-dessous) et donner un port à utiliser, et la fonction retournera un identifiant unique qui devra être utilisé dans tous les appels de fonction ultérieurs pour cette socket ou une valeur inférieure à 0 si la connexion échoue. être utilisé dans tous les autres appels de fonction pour ce socket, ou une valeur inférieure à 0 si la connexion échoue.

Constant Description
réseau_socket_tcp Créer un socket en utilisant TCP.
réseau_socket_udp Créer un socket en utilisant UDP.
réseau_socket_ws Créez un socket web (uniquement pour les projets fonctionnant sur HTML5), en utilisant TCP.

Syntaxe :

réseau_créer_socket_ext(protocole, port) ;

Arguments Description
protocole Le protocole réseau à utiliser
port Le port à utiliser

Les retours :

Real

Exemple :

client = network_create_socket_ext(network_socket_udp, 6510) ;

Le code ci-dessus va créer un nouveau socket UDP sur le port 6510.
