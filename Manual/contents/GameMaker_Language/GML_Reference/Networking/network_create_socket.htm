réseau_créer_socket

Cette fonction est utilisée pour créer un nouveau socket client pour que votre jeu puisse communiquer sur le réseau. Vous devez définir le type de socket (voir la liste des constantes ci-dessous) et la fonction retournera un identifiant unique pour ce socket, qui devra être utilisé dans tous les autres appels de fonction pour ce socket, ou une valeur inférieure à 0 si la connexion échoue.

REMARQUE: Vous pouvez utiliser l'adresse IP "127.0.0.1" pour vous reconnecter au même appareil qui exécute le jeu.

Constant Description
réseau_socket_tcp Créer un socket en utilisant TCP.
réseau_socket_udp Créer un socket en utilisant UDP.
réseau_socket_ws Créez un socket web (uniquement pour les projets fonctionnant sur HTML5), en utilisant TCP.
réseau_socket_bluetooth Créer un socket Bluetooth(actuellement non disponible !).

Syntaxe :

réseau_créer_socket(type) ;

Arguments Description
type Le type de connexion de socket à créer (voir les constantes listées ci-dessous).

Les retours :

Real

Exemple :

if os_browser == browser_not_a_browser
{
client = network_create_socket(network_socket_tcp) ;
network_connect( client, "192.134.0.1", 6510 ) ;
}
else
{
client = network_create_socket(network_socket_ws) ;
network_connect( client, "192.134.0.1", 6520 ) ;
}

Le code ci-dessus vérifiera si le jeu est exécuté dans un navigateur ou non et créera un nouveau socket TCP ou Web avant de tenter de se connecter à travers celui-ci à l'adresse IP donnée sur le port donné.
