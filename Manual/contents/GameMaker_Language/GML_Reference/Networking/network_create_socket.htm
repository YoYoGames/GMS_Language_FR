network_create_socket

Cette fonction est utilisée pour créer une nouvelle socket client pour que votre jeu communique sur le réseau. Vous devez définir le type de socket (voir la liste des constantes ci-dessous) et la fonction renverra un identifiant unique pour cette socket, qui devrait être utilisé dans tous les autres appels de fonction pour cette socket, ou une valeur inférieure à 0 si la connexion échoue.

REMARQUE: vous pouvez utiliser l'IP «127.0.0.1» pour vous reconnecter au même appareil qui exécute le jeu.

Constant Description
network_socket_tcp Créez un socket en utilisant TCP.
network_socket_udp Créez un socket en utilisant UDP.
network_socket_ws Créez un socket Web ( uniquement pour les projets exécutés sur HTML5), à l'aide de TCP.
network_socket_bluetooth Créez une prise Bluetooth ( actuellement indisponible! ).

Syntaxe:

network_create_socket(type);

Argument Description
type Le type de connexion socket à créer (voir les constantes ci-dessous).

Retour:

Real

Exemple:

if os_browser == browser_not_a_browser
    {
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
    }
else
    {
    client = network_create_socket(network_socket_ws);
    network_connect( client, "192.134.0.1", 6520 );
    }

Le code ci-dessus vérifiera si le jeu fonctionne dans un navigateur ou non et créera un nouveau socket TCP ou Web avant d'essayer de se connecter via celui-ci à l'adresse IP donnée sur le port donné.
