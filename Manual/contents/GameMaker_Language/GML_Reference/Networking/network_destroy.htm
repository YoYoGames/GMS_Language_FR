network_destroy

Avec cette fonction, vous pouvez supprimer une connexion de socket réseau de votre jeu.

REMARQUE: cette fonction ne fonctionnera pas lorsqu'elle est utilisée dans un projet exécuté sur la cible HTML5.

Syntaxe:

network_destroy(socket);

Argument Description
socket L'identifiant du socket à supprimer.

Retour:

N/A

Exemple:

if !os_is_network_connected()
   {
   network_destroy(sock);
   }

Le code ci-dessus vérifiera s'il y a une connexion de données et si aucune n'est trouvée, détruisez la socket avec l'id "chaussette".
