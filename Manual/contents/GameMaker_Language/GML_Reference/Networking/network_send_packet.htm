réseau_envoi_paquet

Cette fonction permet d'envoyer un "paquet" de données sur le réseau. La fonction prend l'identifiant de la socket à laquelle se connecter, puis vous devez fournir l'identifiant du tampon qui contient les données à envoyer (pour plus d'informations sur les tampons voir Référence - Tampons) et enfin la taille (en octets) du paquet de données. Les paquets envoyés avec cette fonction sont formatés de manière à ce que le jeu GameMaker Studio 2 qui reçoit les données puisse "découper" les paquets correctement. paquets correctement, et la fonction renvoie le nombre d'octets de données envoyés, ou un nombre inférieur à 0 si l'envoi a échoué. Il convient de noter que la taille finale des données envoyées, renvoyée par cette fonction, comprendra également les informations d'en-tête de GameMaker. informations d'en-tête GameMaker, qui représentent 12 octets supplémentaires.

Syntaxe :

réseau_send_packet(socket, buffer, size) ;

Arguments Description
prise de courant L'identifiant du socket à utiliser.
tampon L'identifiant du tampon à partir duquel les données sont récupérées.
taille La taille (en octets) des données.

Les retours :

Real

Exemple :

buff = buffer_load("player_save.dat") ;
network_send_packet(sock, buff, buffer_get_size(buff)) ;

Les informations ci-dessus chargent en mémoire un tampon de données précédemment enregistré et renvoient l'identifiant du tampon à stocker dans la variable "buff". Ce tampon complet est ensuite envoyé sous forme de paquet sur le réseau en utilisant le socket identifié par la variable "sock".
