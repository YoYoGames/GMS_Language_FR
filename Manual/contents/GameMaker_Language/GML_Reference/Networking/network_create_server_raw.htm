réseau_créer_serveur_raw

Cette fonction est utilisée pour créer un nouveau serveur réseau pour votre jeu, en utilisant l'un des protocoles de connexion autorisés (voir les constantes listées ci-dessous). Vous fournissez le type de serveur, puis vous lui donnez un port à utiliser, et enfin le nombre maximum de connexions autorisées à tout moment sur le serveur (notez que ce nombre dépend de vous). qui devraient être autorisées à tout moment sur le serveur (notez que ce nombre est libre, mais un trop grand nombre de clients connectés saturera le réseau ou le processeur du périphérique ne sera pas en mesure de gérer le traitement de ce nombre de joueurs, donc à utiliser avec prudence). La fonction renvoie un identifiant unique qui doit être stocké dans une variable et utilisé pour identifier le serveur dans toutes les autres fonctions réseau, ou une valeur inférieure à 0 si la connexion échoue.

Constant Description
réseau_socket_tcp Créer un socket en utilisant TCP.
réseau_socket_udp Créer un socket en utilisant UDP.
réseau_socket_ws Créer un socket web(uniquement pour se connecter aux projets HTML5), en utilisant TCP.



Comme cela crée un serveur "brut", il n'acceptera ni n'utilisera aucun des en-têtes de données intégrés à GameMaker pour la communication, et vous devrez donc utiliser les fonctions réseau_send_raw() et réseau_send_udp_raw() pour envoyer des données non formatées au serveur créé.

Syntaxe :

network_create_server_raw(type, port, max_client) ;

Arguments Description
type Le type de serveur à créer (voir les constantes listées ci-dessous).
port Le port que le serveur utilisera.
max_client Le nombre maximum de clients qui peuvent se connecter en même temps.

Les retours :

Real

Exemple :

var port = 6510 ;
server = network_create_server_raw(network_socket_tcp, port, 32) ;
while (server < 0 && port < 65535)
{
port++
server = network_create_server(network_socket_tcp, port, 32) ;
}

Le code ci-dessus va essayer de créer un serveur en utilisant le port TCP 6510. Si ce port n'est pas disponible, il parcourra les ports en boucle pour en trouver un qui le soit.
