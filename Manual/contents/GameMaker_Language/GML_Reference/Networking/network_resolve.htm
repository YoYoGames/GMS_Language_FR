network_resolve

Cette fonction renverra l'adresse IP de l'URL donnée.

Syntaxe:

network_resolve(url);

Argument Description
url L'URL pour obtenir l'IP de (une chaîne).

Retour:

String

Exemple:

game_ip = network_resolve("www.macsweeneygames.com");

Le code ci-dessus renverra l'adresse IP de l'URL donnée et la stockera dans la variable "game_ip".
