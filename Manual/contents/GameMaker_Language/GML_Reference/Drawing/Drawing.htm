Dessin

Cette section contient toutes les fonctions liées au dessin dans la salle de jeu, ainsi que pour contrôler la façon dont les choses seront dessinées (mélange, alpha, abattage, etc...). Il existe un grand nombre de fonctions différentes pour le dessin, et elles sont réparties dans les catégories suivantes pour faciliter la recherche de ce dont vous avez besoin:

  • Couleur et alpha
  • Contrôle GPU
  • Mipmapping
  • Formulaires de base
  • Sprites et tuiles
  • Texte
  • Primitives et formats de sommets
  • Surfaces
  • Éclairage
  • Particules
  • Textures
  • Shaders

Deux fonctions spéciales sont également disponibles, l'une utilisée pour activer ou désactiver l' événement de dessin pour votre projet, et l'autre pour vider tout le pipeline de dessin:

  • draw_enable_drawevent
  • draw_flush
