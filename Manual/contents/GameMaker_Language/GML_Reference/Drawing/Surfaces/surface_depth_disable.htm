surface_depth_disable

Cette fonction désactive la génération automatique du tampon de profondeur pour toutes les surfaces créées après sa désactivation. Normalement, toutes les surfaces ont des tampons de profondeur, donc si vous leur dessinez des objets 3D, il les triera correctement par profondeur afin de maintenir la parité fonctionnelle entre le rendu sur les surfaces et le rendu vers le tampon arrière, cependant l'allocation de tampons de profondeur double essentiellement la quantité de mémoire. requis pour les surfaces, ce qui pourrait être une surcharge excessive et inutile, surtout si votre jeu est très gourmand en mémoire, et pour les jeux 2D, ils ne sont généralement pas nécessaires.

Pour utiliser cette fonction, vous l'appelez avant de créer une surface, puis vous pouvez la réactiver après (si nécessaire). Toutes les surfaces créées lorsque le tampon de profondeur est désactivé n'utiliseront plus de tampon de profondeur lorsqu'elles sont dessinées, ce qui fait que les objets dessinés sur la surface ont la parité de profondeur. Les images ci-dessous illustrent cela dans la pratique:

Dans l'image ci-dessus, nous dessinons un rectangle et un triangle à l'aide d'un tampon de sommets en 3D. Les coordonnées du triangle rouge le placent derrière le rectangle vert, mais le code pour dessiner le triangle est appelé après le code du rectangle, et lorsque le tampon de profondeur est activé, le triangle est toujours dessiné derrière le rectangle car la profondeur 3D est être respecté. Cependant, lorsque nous désactivons le tampon de profondeur pour la surface, seul l'ordre du code est respecté et ainsi le triangle est dessiné sur le rectangle.

REMARQUE: pour conserver correctement la profondeur en 3D lors de l'utilisation de surfaces, vous devez non seulement activer le tampon de profondeur, mais également activer l'écriture z et le test z. Voir les fonctions gpu_set_ztestenable() et gpu_set_zwriteenable() pour plus d'informations.

REMARQUE: Si votre jeu est 100% 2D ou que vous savez que vous n'avez pas besoin d'un tampon de profondeur de surface, alors il devrait être bon d'appeler simplement cette fonction une fois au début du jeu pour désactiver le tampon.

Syntaxe:

surface_depth_disable(disable);

Argument Description
disable Si défini sur true le tampon de profondeur sera désactivé pour les surfaces créées.

Retour:

N/A

Exemple:

var _depth_setting = surface_get_depth_disable();
surface_depth_disable(false);
surface_with_depth = surface_create(room_width, room_height);
surface_depth_disable(true);
surface_without_depth = surface_create(room_width, room_height);
surface_depth_disable(_depth_setting);

Le code ci-dessus stocke le paramètre de mémoire tampon de profondeur actuel dans une variable, puis crée deux surfaces, une avec la mémoire tampon de profondeur activée et une sans, avant de restaurer à nouveau le paramètre d'origine.
