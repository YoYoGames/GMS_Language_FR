surface_get_texture

Cette fonction renvoie un pointeur spécial pour la texture de la surface. Cette valeur peut ensuite être utilisée dans d'autres fonctions de dessin, en particulier en 3D générale et certaines des fonctions primitives 2D.

REMARQUE: lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment car elles sont stockées dans la mémoire de texture. Vous devez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists () avant de les référencer directement.

Syntaxe:

surface_get_texture(surface_id);

Argument Description
surface_id ID de la surface dont vous souhaitez obtenir la largeur.

Retour:

Pointer

Exemple:

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

Le code ci-dessus dessinera une bande de triangle de 4 sommets texturée avec la texture contenue dans la variable "tex", qui est elle-même prise à partir d'une surface créée précédemment.
