application_surface_enable

Avec cette fonction, vous pouvez activer ou désactiver la surface d'application. Par défaut, la surface de l'application est activée et tous les dessins pour les événements de dessin seront faits à cela et la surface dessinée à l'écran. Cependant, sur certains appareils plus anciens et certains chipsets, cela peut entraîner de moins bonnes performances. Dans ces cas, vous pouvez désactiver le dessin sur la surface de l'application et tout le dessin d'événement Draw sera alors effectué directement dans la mémoire tampon de l'écran.

Vous pouvez utiliser cette fonction pour activer ou désactiver la surface d'application à tout moment dans votre jeu, mais vous devez noter qu'elle nécessite au moins une image (étape) pour que les modifications prennent effet. Switch désactiver désactivera toutes les corrections de rapport hauteur / largeur, comme défini dans les options de jeu de la plate-forme cible. Cela signifie que vous devez mettre en place une méthode permettant d'adapter le jeu aux dimensions requises de l'appareil, de la fenêtre ou de l'écran. Si vous le réactivez, les options de jeu reprendront effet à l'étape suivante.

Il est recommandé d'utiliser cette fonction avec précaution car elle aura un impact sur la façon dont tout est dessiné à l'écran. En général, vous voudriez l'avoir comme option dans un menu de votre jeu, ou l'appeler une seule fois au début de votre jeu et écrire votre code pour savoir si vous l'avez activé ou désactivé.

Syntaxe:

application_surface_enable(enable);

Argument Description
enable Définir pour activer (true ) ou désactiver (false ) la surface d'application.

Retour:

N/A

Exemple:

if keyboard_check_pressed(vk_space)
    {
    if application_surface_is_enabled()
        {
        application_surface_enable(false);
        }
    else
        {
        application_surface_enable(true);
        }
    }

Le code ci-dessus vérifie la pression d'une touche et active ou désactive la surface de l'application en fonction de son état (comme dans un menu d'options).
