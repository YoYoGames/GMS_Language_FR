surface_get_depth_disable

Cette fonction vérifie si la génération automatique de tampon de profondeur pour les surfaces est activée. Normalement, toutes les surfaces ont des tampons de profondeur, donc si vous leur dessinez des objets 3D, cela les triera correctement par profondeur, mais l'allocation de tampons de profondeur double essentiellement la taille des surfaces, ce qui pourrait être une surcharge excessive et inutile, surtout si votre jeu est très mémoire. intensif ou principalement 2D. Dans ces cas, vous pouvez vérifier cela à l'aide de cette fonction et désactiver le tampon de profondeur pour les surfaces si nécessaire à l'aide de la fonction surface_depth_disable().

Syntaxe:

surface_get_depth_disable();

Retour:

Boolean

Exemple:

if (!surface_exists(surf))
    {
    if surface_get_depth_disable() == false
        {
        surface_depth_disable(true);
        }
    surf = surface_create(room_width, room_height);
    }

Le code ci-dessus vérifiera si la surface donnée existe, et si ce n'est pas le cas, il vérifie l'état actuel du tampon de profondeur de surface et s'il est activé, il le désactivera à la place, avant de finalement créer la surface.
