surface_get_depth_disable

Cette fonction vérifie si la génération automatique de tampons de profondeur pour les surfaces est activée. Normalement, toutes les surfaces ont des tampons de profondeur, donc si vous dessinez des objets 3D sur celles-ci, ils seront triés correctement par profondeur. Cependant, l'allocation de tampons de profondeur double essentiellement la taille des surfaces, ce qui peut constituer une surcharge excessive et inutile, en particulier si votre jeu est très gourmand en mémoire ou principalement en 2D. Dans ces cas, vous pouvez vérifier cela en utilisant cette fonction et désactiver le tampon de profondeur pour les surfaces. si nécessaire, en utilisant la fonction surface_depth_disable().

Syntaxe :

surface_get_depth_disable() ;

Les retours :

Booléen

Exemple :

if (!surface_exists(surf))
{
if surface_get_depth_disable() == false
{
surface_depth_disable(true) ;
}
surf = surface_create(room_width, room_height) ;
}

Le code ci-dessus vérifie si la surface donnée existe, et si ce n'est pas le cas, il vérifie l'état actuel du tampon de profondeur de la surface et s'il est activé, il le désactive à la place, avant de créer finalement la surface.
