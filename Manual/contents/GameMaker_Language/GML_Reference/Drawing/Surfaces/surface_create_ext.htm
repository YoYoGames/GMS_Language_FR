surface_create_ext

Cette fonction vous permet d'attacher une surface à un élément de canevas qui existe déjà dans votre page Web, ce qui signifie que vous pouvez effectivement diviser des parties de votre jeu pour les dessiner à différents endroits de la page. À cette fin, vous devez avoir défini correctement l'élément canvas dans la page *.html de votre jeu en utilisant les tailles et les noms corrects qui correspondent aux surfaces que vous souhaitez créer. Ainsi, vous disposez d'un canevas "principal" et de surfaces secondaires. qui seront affectés aux surfaces correctes à l'aide de cette fonction. La mise en page peut être délicate, surtout si vous souhaitez que tous les éléments s'alignent correctement. Mais une fois la tâche difficile de la mise en page terminée, vous pouvez ajouter ce fichier html comme fichier de page par défaut pour le jeu en utilisant l'onglet HTML5 des options du jeu. La prochaine chose à faire est de configurer votre salle et vos vues, car chaque surface devra être associée à une vue spécifique pour "capturer" les images du jeu (voir la variable de vue view_surface_id[0...7]). Enfin, vous utiliserez cette fonction pour créer les surfaces, le nom étant le même que celui utilisé pour les éléments du canevas et la taille correspondant à la taille de ce même canevas. utilisé pour les éléments du canevas et la taille correspondant à la taille de ce même canevas. La fonction renvoie l'indice de la surface, qui doit être stocké dans une variable pour les futurs appels de fonction. Lorsque la surface est créée pour la première fois, elle peut contenir du "bruit", car il s'agit en fait d'une zone de mémoire réservée à cet effet (et cette mémoire peut encore contenir des informations). Vous pouvez donc effacer la surface avant de l'utiliser avec une fonction telle que draw_clear_alpha().

REMARQUE: Cette fonction n'est disponible qu'avec le module HTML5.

Syntaxe :

surface_create_ext(name, w, h) ;

Arguments Description
nom Le nom de l'élément du canevas auquel la surface doit être liée.
w La largeur de la surface à créer.
h La hauteur de la surface à créer.

Les retours :

Real

Exemple :

s1 = surface_create_ext("surface1", 192, 550) ;
s2 = surface_create_ext("surface2", 608, 186) ;
view_surface_id[1] = s1 ;
view_surface_id[2] = s2 ;

Le code ci-dessus crée deux surfaces de tailles différentes, en les affectant chacune à un élément de canevas différent, puis ces surfaces sont affectées à deux vues afin de capturer la bonne partie de la pièce.
