surface_save_part

Cette fonction permet de sauvegarder une partie d'une surface sur le disque en utilisant le nom de fichier donné. La surface doit être sauvegardée comme un fichier au format *.png, et la position (x,y) doit être donnée comme des coordonnées locales de la surface, en gardant à l'esprit que le coin supérieur gauche de la surface est toujours (0,0). de la surface est toujours (0,0).

Syntaxe :

surface_save_part(surface_id, fname, x, y, width, height) ;

Arguments Description
surface_id L'ID de la surface à définir comme cible du dessin.
nom de famille Le nom du fichier image enregistré.
x La position x de départ dans la surface.
y La position y de départ dans la surface.
largeur La largeur de la pièce à sauvegarder.
hauteur La hauteur de la pièce à sauvegarder.

Les retours :

N/A

Exemple :

if keyboard_check_pressed(ord("S")
{
surface_save_part(surf, "test.png", 0, 0, 100, 100) ;
}

Le code ci-dessus vérifie si l'utilisateur appuie sur la touche "S" du clavier et si c'est le cas, il enregistre une partie de la surface indexée dans la variable "surf" sur le disque.
