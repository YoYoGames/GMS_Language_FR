application_surface_draw_enable

Vous pouvez utiliser cette fonction pour activer ou désactiver le dessin automatique de la surface d'application. Par défaut, cela est activé, mais dans de nombreux cas, vous voudrez prendre en charge quand et comment la surface est dessinée (lors de l'utilisation de shaders par exemple), auquel cas vous utiliseriez cette fonction pour la définir sur false afin de pouvoir la dessiner vous-même quand et comment vous en avez besoin. Il est important de noter que lorsque vous désactivez le dessin automatique et dessinez vous-même la surface d'application, vous pouvez voir certains problèmes avec le composant alpha des sprites et la surface elle-même. En effet, GameMaker Studio 2 dessinera la surface de l'application sans fusion alpha lorsque le dessin automatique est activé. Si vous désactivez le dessin automatique, vous devez le gérer vous-même, en utilisant le code suivant (par exemple):

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

Syntaxe:

application_surface_draw_enable(flag);

Argument Description
flag Le drapeau sera soit true (activé, la valeur par défaut) ou false (désactivée).

Retour:

N/A

Exemple:

application_surface_draw_enable(false);

Le code ci-dessus désactivera le dessin automatique de la surface d'application.
