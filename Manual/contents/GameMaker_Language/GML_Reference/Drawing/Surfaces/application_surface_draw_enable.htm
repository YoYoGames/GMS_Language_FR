application_surface_draw_enable

Vous pouvez utiliser cette fonction pour activer ou désactiver le dessin automatique de la surface de l'application. Par défaut, cette fonction est activée, mais dans de nombreux cas, vous souhaiterez prendre en charge le moment et la manière dont la surface est dessinée (lorsque vous utilisez des shaders, par exemple). vous utiliserez cette fonction pour la définir sur false afin de pouvoir la dessiner vous-même quand et comment vous le souhaitez. Il est important de noter que lorsque vous désactivez le dessin automatique et dessinez vous-même la surface de l'application, vous pouvez rencontrer certains problèmes avec la composante alpha des sprites et de la surface elle-même. En effet, GameMaker Studio 2 dessine la surface de l'application sans mélange alpha lorsque le dessin automatique est activé. Si vous désactivez le dessin automatique, vous devez gérer cela vous-même, en utilisant le code suivant (par exemple) :

gpu_set_blendenable(false) ;
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1) ;
gpu_set_blendenable(true) ;

Syntaxe :

application_surface_draw_enable(flag) ;

Arguments Description
drapeau L'indicateur sera soit vrai (activé, la valeur par défaut) soit faux (désactivé).

Les retours :

N/A

Exemple :

application_surface_draw_enable(false) ;

Le code ci-dessus désactivera le dessin automatique de la surface de l'application.
