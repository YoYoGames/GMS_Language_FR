surface_get_width

Cette fonction renvoie simplement la largeur, en pixels, de la surface indexée. Il convient de noter que si vous appelez ceci pour vérifier le application_surface immédiatement après avoir changé sa taille en utilisant surface_resize() il ne renverra pas la nouvelle valeur car la modification nécessite une ou deux étapes pour être entièrement traitée. Après avoir attendu une étape, il devrait renvoyer correctement la nouvelle taille.

REMARQUE: lorsque vous travaillez avec des surfaces, il est possible qu'elles cessent d'exister à tout moment car elles sont stockées dans la mémoire de texture. Vous devez TOUJOURS vérifier qu'une surface existe en utilisant surface_exists () avant de les référencer directement.

Syntaxe:

surface_get_width(surface_id);

Argument Description
surface_id ID de la surface dont vous souhaitez obtenir la largeur.

Retour:

Real

Exemple:

sw = surface_get_width(surf);

Le code ci-dessus stockera la largeur de la surface indexée dans la variable "surf" dans la variable "sw".
