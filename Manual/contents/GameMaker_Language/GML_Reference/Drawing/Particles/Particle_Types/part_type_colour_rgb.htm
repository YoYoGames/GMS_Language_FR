partie_type_couleur_rgb

Avec cette fonction, vous pouvez définir le mélange de couleurs rouge, vert et bleu pour toutes les particules créées du type donné. Vous fournissez une valeur minimale et une valeur maximale pour chacun des trois composants et les particules créées auront une couleur aléatoire basée sur la gamme de paramètres donnée. sur la base de la plage de paramètres donnée. Toutes les valeurs doivent être comprises entre 0 et 255.

Syntaxe :

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Arguments Description
ind L'indice du type de particule à modifier.
rmin Le minimum que peut atteindre la composante rouge de la couleur finale.
rmax Le maximum que peut atteindre la composante rouge de la couleur finale.
gmin Le minimum que peut atteindre la composante verte de la couleur finale.
gmax Le maximum que peut atteindre la composante verte de la couleur finale.
bmin Le minimum que peut atteindre la composante bleue de la couleur finale.
bmax Le maximum que peut atteindre la composante bleue de la couleur finale.

Les retours :

N/A

Exemple :

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0, 0) ;

Le code ci-dessus définit chaque particule émise du type de particule indexé dans la variable globale "Blood_Part" pour qu'elle soit uniquement de différentes nuances de rouge.
