part_type_colour1

Cette fonction est utilisée pour définir un type de particule comme une seule couleur pour la durée totale de la vie de chaque particule individuelle.

Syntaxe:

part_type_colour1(ind, colour1);

Argument Description
ind L'indice du type de particule à modifier.
colour1 La couleur unique pour créer le type de particule.

Retour:

N/A

Exemple:

part_type_colour1(global.Snow_Part, c_white);

Le code ci-dessus définira toutes les particules créées du type de particule indexé dans la variable globale "Snow_Part" pour être blanches uniquement.
