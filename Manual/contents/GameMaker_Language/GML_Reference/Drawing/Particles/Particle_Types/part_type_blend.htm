part_type_blend

Avec cette fonction, vous pouvez dessiner vos particules avec un mode de fusion additif (true ) ou non (false ). Le mélange additif est un mode de fusion spécial qui ajoute les valeurs de luminosité de chaque particule lorsqu'elles se chevauchent, de sorte que les couleurs claires deviennent progressivement plus lumineuses (jusqu'à ce qu'elles apparaissent blanches) lorsqu'elles se chevauchent, et les couleurs sombres deviennent de plus en plus transparentes, le noir étant presque invisible.. Cette fonction peut donner de beaux effets de particules mais n'est pas toujours nécessaire. Par exemple, une traînée de fumée aurait un mélange additif pour garder l'effet gris, mais un effet de flamme l'aurait probablement activé pour rendre les particules plus translucides et plus lumineuses.

Syntaxe:

part_type_blend(ind, additive);

Argument Description
ind L'indice du type de particule à modifier.
additive Si les particules doivent être mélangées de manière additive (vrai) ou normalement (faux).

Retour:

N/A

Exemple:

part_type_blend(part_Fire, true);

Le code ci-dessus définira toutes les particules créées du type de particule indexé dans la variable "part_Fire" pour avoir un mode de fusion additif.
