type_partie_mélange

Avec cette fonction, vous pouvez faire en sorte que vos particules soient dessinées avec un mode de fusion additif(true) ou non(false). Le mélange additif est un mode de mélange spécial qui ajoute les valeurs de luminosité de chaque particule au fur et à mesure qu'elles se chevauchent. deviennent progressivement plus claires (jusqu'à paraître blanches) au fur et à mesure qu'elles se chevauchent, et les couleurs sombres deviennent de plus en plus transparentes, le noir étant presque invisible. Cette fonction peut donner de magnifiques effets de particules, mais n'est pas toujours nécessaire. Par exemple, Par exemple, pour une traînée de fumée, le mélange additif sera désactivé pour que l'effet reste gris, mais pour un effet de flamme, il sera probablement activé pour rendre les particules plus translucides et plus brillantes.

Syntaxe :

partie_type_mélange(ind, additif) ;

Arguments Description
ind L'indice du type de particule à modifier.
additif Si les particules doivent être mélangées de manière additive (vrai) ou normale (faux).

Les retours :

N/A

Exemple :

part_type_blend(part_Fire, true) ;

Le code ci-dessus définira toutes les particules créées du type de particule indexé dans la variable "part_Fire" pour avoir un mode de fusion additif.
