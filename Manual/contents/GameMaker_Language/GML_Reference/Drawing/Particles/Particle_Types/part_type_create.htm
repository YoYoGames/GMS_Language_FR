part_type_create

Avec cette fonction, vous pouvez créer un nouveau type de particule et la valeur de retour doit être stockée dans une variable pour être utilisée dans toutes les fonctions de particules suivantes.

Syntaxe:

part_type_create();

Retour:

Integer (particle ID)

Exemple:

mypart = part_type_create();

Cela créera un nouveau type de particule, en stockant son index dans la variable "mypart".

Retour: Types de particules
