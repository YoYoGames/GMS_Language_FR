part_type_alpha2

Cette fonction peut être utilisée pour définir un gradient de valeur alpha (transparence) pour chaque particule créée du type donné. Le premier alpha est celui avec lequel toutes les particules commenceront, et le second alpha est celui avec lequel la particule se terminera, et un changement de gradient progressif se produira vers l'alpha au cours de la durée de vie des particules de l'un à l'autre. Cela peut être compris entre 0 (transparent) et 1 (opaque).

Syntaxe:

part_type_alpha2(ind, alpha1, alpha2);

Argument Description
ind L'indice du type de particule à modifier.
alpha1 Le début alpha de la particule.
alpha2 La fin alpha de la particule.

Retour:

N/A

Exemple:

part_type_alpha2(global.Snow_Part, 0, 1);

Le code ci-dessus définira toutes les particules créées du type de particule indexé dans la variable globale "Snow_Part" pour avoir une valeur alpha de 0 (transparent) et ensuite se fondre pour avoir un alpha de 1 (opaque) à la fin de leur durée de vie.
