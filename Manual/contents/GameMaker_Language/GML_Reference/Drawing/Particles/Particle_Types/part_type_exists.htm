partie_type_existe

Avec cette fonction, vous pouvez vérifier si le type de particule indexé existe ou non dans le jeu. Notez que si la variable vérifiée est une variable non initialisée (à laquelle un type de particule aurait normalement été assigné), une erreur sera générée. une erreur.

Syntaxe :

partie_type_existe(ind) ;

Arguments Description
ind L'indice du type de particule à vérifier.

Les retours :

Booléen

Exemple :

if !part_type_exists(global.p1)
{
global.p1 = part_type_create() ;
}

Le code ci-dessus vérifie si la variable globale "p1" stocke l'index d'un type de particule et si ce n'est pas le cas, il en crée un.
