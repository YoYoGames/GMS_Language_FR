partie_system_position

Avec cette fonction, vous pouvez définir la position de base du système de particules par rapport à la position (0,0) de la pièce, ce qui signifie que toutes les autres fonctions de particules relatives à ce système seront désormais dessinées par rapport à la nouvelle position. Par défaut, cette Par défaut, cette position est toujours (0,0), mais dans certaines circonstances très particulières, vous pouvez souhaiter la modifier.

REMARQUE: Cette fonction modifie tout dans le système de particules, donc si vous avez un emitter à la position (100,100) et que vous réglez la position du système de particules à (0,100), l'emitter sera maintenant dessiné à (100,200). Il en va de même si vous décalez le système, puis créez l'émetteur, car même si vous le créez à (100,100), il sera dessiné à (100,200).

Syntaxe :

part_system_position(ind, x, y) ;

Arguments Description
ind L'indice du système de particules à modifier.
x La nouvelle coordonnée x du système de particules.
y La nouvelle coordonnée y du système de particules.

Les retours :

N/A

Exemple :

if mouse_check_button_pressed(mb_left)
{
part_system_position(global.Sname, mouse_x, mouse_y) ;
}

Le code ci-dessus vérifie si le bouton de la souris est pressé et s'il le détecte, le système de particules indexé dans la variable globale "Sname" est déplacé à la position x/y de la souris.
