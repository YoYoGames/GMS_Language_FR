couche_système_partiel

Cette fonction peut être utilisée pour faire passer un système de particules de sa couche actuelle à une nouvelle couche. Vous fournissez la valeur de l'ID du système de particules (telle que retournée par la fonction partie_system_create_layer()) et ensuite l'ID unique de la couche (tel que retourné par la fonction couche_créer() ou le nom de la couche à utiliser sous forme de chaîne - par exemple "instance_layer" -. tel que défini dans l'éditeur d'instance), et le système sera déplacé vers la nouvelle couche.

Syntaxe :

part_system_layer(ps, layer) ;

Arguments Description
ps L'ID du système de particules
couche L'ID de la nouvelle couche

Les retours :

N/A

Exemple :

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
part_system_layer(global.p_sys, "effects_layer" ;
}

Le code ci-dessus vérifiera un système de particules pour voir sur quelle couche il se trouve et s'il n'est pas sur la couche donnée, il sera changé.
