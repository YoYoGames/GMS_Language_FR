part_system_create_layer

Cette fonction créera un nouveau système de particules sur une couche donnée. Vous donnez l'ID de couche unique renvoyé par la fonction layer_create() ou le nom de la couche à utiliser comme chaîne - par exemple "instance_layer" - puis marquez le système comme persistant ou non. Si le système n'est pas marqué comme persistant, il sera automatiquement détruit à la fin de la pièce dans laquelle il a été créé (c'est la même chose que si vous aviez appelé la fonction part_system_destroy() et détruira également tous les émetteurs associés au système). Cependant, lorsqu'il est marqué comme persistant, le système sera transporté dans la pièce suivante lorsque la pièce est changée, et si la pièce suivante n'a pas de couche avec le même nom ou la même profondeur que celle assignée, alors une nouvelle couche sera créée. pour le système qui persiste dans les pièces, et il portera le même nom que le calque d'origine. Lorsque vous changez de pièce, s'il existe un autre calque dans les pièces suivantes portant le même nom, l'instance persistante sera affectée au calque portant ce nom, quelle que soit la profondeur du calque. Enfin, si un système persistant se déplace vers une pièce avec un calque à la même profondeur que le calque sur lequel le système a été créé, il ne sera pas affecté à ce calque, mais un nouveau calque sera créé à la même profondeur (avec le même nom que le calque d'origine).

La fonction renverra une valeur d'ID unique pour le système de particules qui doit être utilisée dans tous les autres appels de fonction où vous devez donner un ID système.

IMPORTANT! Si vous marquez le système de particules comme persistant, il (et tous les émetteurs qui lui sont assignés) devront être nettoyés manuellement à l'aide des fonctions de destruction appropriées lorsqu'il n'est pas utilisé, sinon vous risquez une fuite de mémoire qui aura un impact négatif sur votre jeu final.

Syntaxe:

part_system_create_layer(layer_id, persistent);

Argument Description
layer La valeur ou la chaîne de l'ID de couche à laquelle attribuer le système de particules (peut être n'importe quel type de couche)
persistent Marquer le système de particules comme persistant (défini sur true ) ou non (défini sur false )

Retour:

Real (unique particle system ID value)

Exemple:

global.p_sys = part_system_create_layer("effects_layer", true);

Le code ci-dessus créera un nouveau système de particules sur le calque donné et le marquera comme persistant sur les pièces suivantes. L'ID du système de particules est stocké dans une variable de portée globale pour référence future.
