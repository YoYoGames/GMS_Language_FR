ordre de tirage des pièces

Avec cette fonction, vous pouvez définir la façon dont les particules sont dessinées lorsqu'elles sont créées à l'écran. Le système par défaut utilise un look ancien > nouveau (la fonction est définie à true), où les anciennes particules sont dessinées à une profondeur plus élevée que les nouvelles et apparaissent donc "sous" les nouvelles particules. Les anciennes particules sont dessinées à une profondeur plus élevée que les nouvelles et apparaissent donc "sous" les nouvelles particules, mais en définissant cette fonction à false vous pouvez inverser cet ordre et avoir les nouvelles particules dessinées plus haut et donc apparaître "sous" les anciennes. Les images ci-dessous illustrent cela, avec l'image de gauche étant la valeur par défaut de true et l'image de droite étant false:

NOTE: Lorsque les particules sont dessinées avec un mode de fusion additif, l'effet de cette fonction n'est pas toujours évident.

Syntaxe :

part_system_draw_order(ind, oldtonew) ;

Arguments Description
ind L'indice du système de particules à modifier.
oldtonew Si les anciennes particules doivent être dessinées derrière les plus récentes (true) ou vice versa (false).

Les retours :

N/A

Exemple :

mysystem = part_system_create() ;
part_system_draw_order(mysystem, true) ;

Cela créera un nouveau système de particules avec l'index " mysystem", et ensuite, il définira les particules pour dessiner les nouvelles particules au dessus des anciennes.
