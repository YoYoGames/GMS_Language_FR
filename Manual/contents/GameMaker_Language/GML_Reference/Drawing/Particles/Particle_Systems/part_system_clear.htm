part_system_clear

Avec cette fonction, vous pouvez remettre le système indexé à son état par défaut, en supprimant tous les émetteurs et en réinitialisant la profondeur et la position (si elles ont été modifiées). Soyez prudent en utilisant cette fonction car si vous avez un réglage d'instance ou un éclatement ou toute autre action utilisant un emitter associé à ce système, vous obtiendrez une erreur à moins que vous n'utilisiez la fonction partie_émetteur_existe() pour vérifier. Il n'est pas non plus nécessaire d'appeler la partie_émetteur_destroy() fonction car cela est également pris en charge automatiquement.

NOTE: cette fonction efface les particules visibles dans la pièce, mais elle n' efface pas les propriétés des particules, ni ne les supprime de la mémoire. Pour cela, vous devez utiliser les fonctions part_type_clear() et part_type_destroy().

Syntaxe :

part_system_clear(ind) ;

Arguments Description
ind L'index du système de particules à effacer.

Les retours :

N/A

Exemple :

part_system_clear(global.Sname) ;

Le code ci-dessus remet le système de particules indexé dans la variable globale "Sname" à son état par défaut.
