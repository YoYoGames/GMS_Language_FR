part_system_clear

Avec cette fonction, vous pouvez réinitialiser le système indexé à son état par défaut, en supprimant tous les émetteurs et en réinitialisant la profondeur et la position (si elles avaient été modifiées). Soyez prudent en utilisant cette fonction, car si vous avez un paramètre d'instance ou une rafale ou toute autre action utilisant un émetteur associé à ce système, vous obtiendrez une erreur sauf si vous utilisez le part_emitter_exists() fonction à vérifier. Il n'est pas non plus nécessaire d'appeler le part_emitter_destroy() fonction car cela est également pris en charge automatiquement.

REMARQUE: cette fonction efface les particules visibles dans la pièce, mais elle n'efface pas les propriétés des particules et ne les supprime pas de la mémoire. Pour cela, vous devez utiliser les fonctions part_type_clear () et part_type_destroy ().

Syntaxe:

part_system_clear(ind);

Argument Description
ind L'indice du système de particules à effacer.

Retour:

N/A

Exemple:

part_system_clear(global.Sname);

Le code ci-dessus effacera le système de particules indexé dans la variable globale "Sname" à son état par défaut.

Retour: Systèmes de particules
