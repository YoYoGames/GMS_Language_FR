part_particles_create

Cette fonction est idéale pour les effets qui ne nécessitent aucune des fonctionnalités offertes par les émetteurs de particules (par exemple, pour créer de la fumée à partir d'un missile, ou un simple effet d'explosion) car elle vous permet de créer rapidement et facilement des particules à n'importe quelle position dans la salle de jeux. Vous pouvez également colorer les particules "à la volée" au fur et à mesure qu'elles sont créées avec cette fonction, et cette couleur remplacera la couleur prédéfinie de la particule, mais elle ne mélange pas cette couleur pendant la durée de vie des particules. Notez que vous devez avoir créé le système de particules et le type de particule au préalable pour que cette fonction soit utilisée.

Syntaxe:

part_particles_create_colour(ind, x, y, parttype, colour, number);

Argument Description
ind L'indice du système de particules.
x Coordonnée x de l'endroit où créer les particules.
y Coordonnée y de l'endroit où créer les particules.
parttype L'index (type) des particules à créer.
colour Le mélange de couleurs pour les particules.
number Le nombre de particules à créer.

Retour:

N/A

Exemple:

if (speed > 1)
    {
    var _c = choose(c_aqua, c_lime, c_fuschia, c_orange);
    part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
    }

Le code ci-dessus générera une seule particule à chaque étape où l'instance avec le code a une vitesse supérieure à un. Ces particules seront d'une couleur aléatoire.
