part_system_exists

Avec cette fonction, vous pouvez vérifier si le système de particules indiqué existe dans le jeu ou non. Notez que si la variable vérifiée est une variable non initialisée (à laquelle un système de particules aurait autrement son index assigné), cela générera une erreur.

Syntaxe:

part_system_exists(ind);

Argument Description
ind L'indice du système de particules à vérifier.

Retour:

Boolean

Exemple:

if !part_system_exists(global.Sname)
    {
    global.Sname = part_system_create();
    }

Le code ci-dessus vérifie si le système de particules référencé dans la variable globale existe et s'il ne l'est pas, il est créé.
