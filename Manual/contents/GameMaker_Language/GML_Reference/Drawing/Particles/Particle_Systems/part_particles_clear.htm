part_particles_clear

Avec cette fonction, vous pouvez effacer toutes les particules actuellement créées par le système dans la pièce. Elle ne réinitialise pas ou ne supprime pas les types de particules eux-mêmes, juste leur représentation visuelle, et si vous avez un objet qui diffuse des particules à partir d'un émetteur, ces particules disparaissent mais recommenceront à apparaître à l'étape suivante après l'appel de ce code. un émetteur, ces particules disparaissent mais recommenceront à apparaître à l'étape suivante après avoir appelé ce code.

Syntaxe :

part_particles_clear(ind) ;

Arguments Description
ind L'indice du système de particules.

Les retours :

N/A

Exemple :

if (vies <= 0)
{
part_particles_clear(global.Sname) ;
room_goto(rm_intro) ;
}

Le code ci-dessus vérifie la valeur de la variable "vies" et si elle est égale à 0, il efface toutes les particules du système et change ensuite de pièce.
