partie_système_destruction

Avec cette fonction, vous pouvez détruire un système de particules donné et le retirer de la mémoire. Cette fonction doit toujours être appelée lorsque le système n'est plus nécessaire, comme à la fin d'une room ou dans l'événement de destruction d'une instance. des particules apparaissant dans des pièces ultérieures sans pouvoir les supprimer, ainsi qu'une fuite de mémoire qui finira par faire planter votre jeu.

REMARQUE: Cette fonction détruit et supprime également tous les émetteurs qui ont pu être créés et associés au système en cours de destruction.

Syntaxe :

part_system_destroy(ind) ;

Arguments Description
ind L'indice du système de particules à détruire.

Les retours :

N/A

Exemple :

if (global.player_hp <= 0)
{
part_system_destroy(p_sys) ;
room_goto_next() ;
}

Le code ci-dessus vérifie si la valeur d'une variable globale est inférieure ou égale à zéro, et si c'est le cas, il détruit le système de particules référencé dans la variable donnée, puis passe à la pièce suivante.
