partie_émetteur_destruction

Cette fonction supprimera l'émetteur spécifié du système donné et l'effacera de la mémoire (cela empêchera également toute particule d'être produite par l'émetteur donné, mais ne les supprimera PAS de la pièce). Cette fonction devrait toujours être appelée lorsque l'émetteur donné n'est plus nécessaire pour le système afin d'éviter les fuites de mémoire et les erreurs.

Syntaxe :

part_emitter_destroy( ps, ind ) ;

Arguments Description
ps Le système de particules dont il faut détruire l'émetteur.
ind L'index de l'émetteur à détruire.

Les retours :

N/A

Exemple :

if part_emitter_exists(global.Sname, p_emit)
{
part_emitter_destroy(global.Sname, p_emit1) ;
}

Le code ci-dessus va vérifier si l'émetteur de particules indexé dans la variable "p_emit" existe dans le système de particules donné et si c'est le cas, il est détruit.
