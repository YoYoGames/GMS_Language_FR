effect_clear

Cela effacera toutes les particules actuellement visibles dans la pièce qui ont été fabriquées en utilisant soit effect_create_above() ou effect_create_below().

Syntaxe:

effect_clear();

Retour:

N/A

Exemple:

effect_clear();

L'appel de ceci effacera la pièce de toutes les particules faites avec le effect_create_* les fonctions.
