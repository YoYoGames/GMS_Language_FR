draw_get_swf_aa_level

Cette fonction peut être utilisée pour obtenir le niveau d'anti-aliasing (AA) pour les sprites vectoriels au format SWF. La valeur de retour sera comprise entre 0 et 1 et montre comment "lisser" les bords de ces sprites seront dessinés. Vous pouvez régler le niveau AA à l'aide de la fonction draw_set_swf_aa_level().

Syntaxe:

draw_get_swf_aa_level();

Retour:

Real

Exemple:

if draw_get_swf_aa_level() == 0
    {
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
    }

Le code ci-dessus vérifiera la valeur AA pour les sprites au format SWF, et si elle est égale à 0, il active AA et définit la valeur sur 0,5.
