draw_sprite_pos

Cette fonction permet de dessiner un sprite déformé sur la zone définie par les quatre coordonnées d'angle. Les deux premiers arguments sont le sprite à dessiner et la sous-image du sprite (la même que dans la fonction draw_sprite()) mais les arguments suivants sont ceux qui définissent la position de chacun des quatre coins de la boîte englobante du sprite donné. Elles doivent être données dans le sens des aiguilles d'une montre, donc la première coordonnée est le coin supérieur gauche, puis le coin supérieur droit, puis le coin inférieur droit et enfin le coin inférieur gauche. droite et enfin le bas gauche. Vous pouvez également définir une valeur pour l'alpha du sprite afin de le dessiner avec transparence. L'image ci-dessous illustre le fonctionnement de cette fonction :

AVERTISSEMENT ! L'image ci-dessus est uniquement destinée à des fins d'illustration. Si vous utilisez cette fonction sur un sprite, vous obtiendrez des résultats différents et vous risquez de subir un "cisaillement" de la texture en raison de la manière dont un sprite est construit à partir d'un quadrant de primitives.

REMARQUE: Cette fonction n'est utile que pour les sprites bitmap et ne fonctionnera pas avec les sprites SWF ou JSON (Spine).

REMARQUE: lorsque vous utilisez cette fonction, le paramètre Recadrage automatique doit être désactivé pour les pages de texture.

Syntaxe :

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha) ;

Arguments Description
sprite L'index du sprite à dessiner.
sous-image La sous-image (frame) du sprite à dessiner (image_index ou -1 correspondent à la frame d'animation actuelle de l'objet).
x1 La première coordonnée x.
y1 La première coordonnée y.
x2 La deuxième coordonnée x.
y2 La deuxième coordonnée y.
x3 La troisième coordonnée x.
y3 La troisième coordonnée y.
x4 La quatrième coordonnée x.
y4 La quatrième coordonnée y.
alpha L'alpha du sprite (de 0 à 1 où 0 est transparent et 1 opaque).

Les retours :

N/A

Exemple :

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1) ;

Le code ci-dessus dessinera le sprite associé à l'instance exécutant le code déformé autour de la position x/y de l'instance et avec un alpha totalement opaque.
