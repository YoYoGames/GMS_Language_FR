draw_tilemap

Cette fonction peut être utilisée pour dessiner un tilemap donné n'importe où dans la pièce. Vous donnez l'ID d'élément tilemap (que vous obtenez lorsque vous créez un élément tilemap à l'aide de layer_tilemap_create() ou lorsque vous utilisez la fonction layer_tilemap_get_id() ), puis donnez la position x / y pour le dessin (dans la pièce). Notez que cela dessinera simplement le tilemap au point spécifié, en utilisant la profondeur de couche de l'instance qui appelle la fonction. Cela ne déplace pas le tilemap - ni ne le modifie en aucune façon - et peu importe que le tilemap soit marqué comme visible ou non.

Syntaxe:

draw_tilemap(tilemap_element_id, x, y);

Argument Description
tilemap_element_id La valeur d'ID unique de l'élément tilemap pour obtenir la position y de la cellule
x La position x dans la pièce à laquelle dessiner
y La position y dans la pièce à laquelle dessiner

Retour:

N/A

Exemple:

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

Le code ci-dessus obtient l'ID de couche puis l'utilise pour obtenir un ID de tilemap spécifique qui à son tour est utilisé pour dessiner le tilemap à la position de la souris.
