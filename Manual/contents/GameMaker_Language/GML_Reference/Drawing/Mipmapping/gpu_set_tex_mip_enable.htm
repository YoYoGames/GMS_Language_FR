gpu_set_tex_mip_enable

Avec cette fonction, vous pouvez changer si le mipmapping est désactivé, activé pour tout, ou activé uniquement pour les groupes de textures sélectionnés dans le gestionnaire de groupes de textures. Cette fonction requiert l'une des constantes énumérées ci-dessous.

Constant Description
mip_off Le Mipmapping est désactivé.
mip_on Le Mipmapping pour toutes les textures est activé.
mip_markedonly Le Mipmapping est activé pour les textures qui l'ont activé dans les options du groupe de textures (par défaut).

Syntaxe :

gpu_set_tex_mip_enable(paramètre) ;

Arguments Description
paramètre Le paramètre mipmap (une constante, par défaut : mip_markedonly)

Les retours :

N/A

Exemple :

if gpu_get_tex_mip_enable != mip_on
{
gpu_set_tex_mip_enable(mip_on) ;
}

Le code ci-dessus vérifiera si le mipmapping est activé et s'il ne l'est pas, il l'activera.
