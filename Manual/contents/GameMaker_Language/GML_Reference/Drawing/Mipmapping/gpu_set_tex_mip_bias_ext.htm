gpu_set_tex_mip_bias_ext

Avec cette fonction, vous pouvez définir la valeur du biais de mipmap pour un échantillonneur de shader donné. Vous fournissez la valeur de l'index de l'échantillonneur d'ombrage (tel que retourné par la fonction shader_get_sampler_index(), et ensuite la valeur du biais, où 0 est pour aucun biais, 1 équivaut à la première mipmap, 2 équivaut à la seconde mipmap etc... Ceci contrôle la vitesse à laquelle la mipmap est échangée et rendra généralement les textures du shader plus floues plus la valeur est élevée et plus la "distance" vue est grande. plus la valeur est élevée et plus la "distance" vue est grande. Notez que cette fonction peut aussi prendre des valeurs négatives, dans ce cas les textures du shader seront plus nettes sur une plus grande distance plus la valeur est faible.

Syntaxe :

gpu_set_tex_mip_bias_ext(sampler_index, bias) ;

Arguments Description
sampler_index L'index de l'échantillonneur de shader à obtenir
biais La valeur du biais de mipmap à utiliser (par défaut : 0)

Les retours :

N/A

Exemple :

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background") ;
var _spriteTex = sprite_get_texture(sprite_index, 0) ;
shader_set (shd_Glass) ;
if gpu_get_tex_mip_bias_ext(_sampleIndex) != 0
{
gpu_set_tex_mip_bias_ext(_sampleIndex, 0) ;
}
texture_set_stage(_sampleIndex , _spriteTex) ;
shader_reset() ;

Le code ci-dessus définit le biais du filtre mip à 0 pour l'échantillonneur de texture du shader donné s'il n'a pas déjà été défini à 0.
