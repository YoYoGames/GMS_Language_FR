gpu_set_tex_max_mip

Avec cette fonction, vous pouvez définir le niveau de mipmap maximum actuellement défini qui doit être utilisé, où une valeur de 0 est la résolution la plus élevée, 1 est d'utiliser le premier mipmap, 2 est le deuxième etc... Notez que cela peut être assez utile pour éviter les artefacts de saignement lors du rendu des textures, par exemple, définir la bordure de la page de texture sur 8px, puis définir le niveau maximal de mipmap sur 3 vous assurera de ne pas avoir de problèmes de saignement à des distances de rendu plus grandes.

Syntaxe:

gpu_set_tex_max_mip(maxmip);

Argument Description
maxmip Le niveau maximum de mipmap (par défaut: 16)

Retour:

N/A

Exemple:

if gpu_get_tex_max_mip() != 4
    {
    gpu_set_tex_max_mip(4);
    }

Le code ci-dessus vérifiera le niveau maximum actuel de mipmap et s'il n'est pas 4, il est mis à 4.
