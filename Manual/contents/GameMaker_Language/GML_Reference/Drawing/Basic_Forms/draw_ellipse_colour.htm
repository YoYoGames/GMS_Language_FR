draw_ellipse_colour

Avec cette fonction, vous pouvez dessiner le contour d'une ellipse ou une ellipse pleine en définissant une zone rectangulaire qui aura ensuite l'ellipse créée pour s'adapter. S'il est rempli, vous pouvez définir les couleurs de remplissage intérieur et extérieur. Si ces couleurs ne sont pas les mêmes, vous obtiendrez un effet de dégradé de l'une à l'autre (les paramètres de couleur remplaceront la couleur de base définie avec la fonction draw_set_colour() ). Vous pouvez définir la précision du dessin avec la fonction draw_set_circle_precision().

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises, et le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous devez dessiner des formes de cette manière, le shader doit être personnalisé dans cet esprit.

Syntaxe:

draw_ellipse_colour(x1, y1, x2, y2, col1, col2, outline);

Argument Description
x1 La coordonnée x de la gauche de l'ellipse.
y1 La coordonnée y du haut de l'ellipse.
x2 La coordonnée x de la droite de l'ellipse.
y2 La coordonnée y du bas de l'ellipse.
col1 La couleur du centre.
col2 La couleur du bord.
outline Indique si l'ellipse est un contour (vrai) ou remplie (faux).

Retour:

N/A

Exemple:

draw_ellipse_colour(50, 50, 200, 100, c_black, c_white, false);

Cela dessinerait une ellipse remplie et étirée horizontalement avec un centre noir et un bord blanc.

Retour: Formulaires de base
