dessiner_ellipse

Avec cette fonction, vous pouvez dessiner le contour d'une ellipse ou une ellipse remplie en définissant une zone rectangulaire dans laquelle l'ellipse sera ensuite créée. Vous pouvez définir la précision du dessin à l'aide de la fonction draw_set_circle_precision().

NOTE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.

Syntaxe :

draw_ellipse(x1, y1, x2, y2, outline) ;

Arguments Description
x1 La coordonnée x de la gauche de l'ellipse.
y1 La coordonnée y du sommet de l'ellipse.
x2 La coordonnée x de la droite de l'ellipse.
y2 La coordonnée y du bas de l'ellipse.
aperçu Indique si l'ellipse est dessinée remplie (false) ou comme un contour d'un pixel de large (true).

Les retours :

N/A

Exemple :

draw_set_colour(c_white) ;
draw_ellipse(100, 100, 300, 200, false) ;

Ceci dessinera une ellipse remplie à l'intérieur de la zone rectangulaire définie.
