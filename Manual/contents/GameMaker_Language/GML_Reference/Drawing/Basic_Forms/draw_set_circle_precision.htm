draw_set_circle_precision

Pour optimiser les performances lors du dessin de cercles, GameMaker Studio 2 dessine une forme polygonale avec suffisamment de côtés pour qu'elle paraisse circulaire. Toutefois, en fonction de la taille des cercles que vous souhaitez créer, il se peut que la modification de cette valeur vous permette d'améliorer les performances de votre jeu ou d'améliorer l'apparence des cercles. cette valeur peut contribuer à améliorer les performances de votre jeu ou à donner un meilleur aspect aux cercles. La valeur de précision que vous pouvez entrer dans cette fonction est le nombre de côtés du polygone circulaire, et ce nombre peut être compris entre 4 et 64, mais doit être un nombre divisible par 4, avec une valeur par défaut de 24.

NOTE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.

Syntaxe :

draw_set_circle_precision(precision) ;

Arguments Description
précision La précision de tous les cercles dessinés après l'appel de cette fonction.

Les retours :

N/A

Exemple :

draw_set_circle_precision(12) ;

Le code ci-dessus dessine tous les cercles en utilisant un polygone à 12 côtés.
