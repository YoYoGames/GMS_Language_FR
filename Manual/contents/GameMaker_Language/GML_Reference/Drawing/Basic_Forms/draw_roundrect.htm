draw_roundrect

Avec cette fonction, vous pouvez dessiner le contour d'un rectangle arrondi ou un rectangle arrondi rempli où la position (x1, y1) est le coin supérieur gauche et la position (x2, y2) est le coin inférieur droit. Vous pouvez définir la précision du dessin des coins avec la fonction draw_set_circle_precision(), mais cette fonction utilise un rayon fixe pour eux (si vous devez changer le rayon du coin, utilisez la fonction draw_roundrect_ext() ). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/- 1 sur le x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les différentes plates-formes prises en charge.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises, et le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous devez dessiner des formes de cette manière, le shader doit être personnalisé dans cet esprit.

Syntaxe:

draw_roundrect(x1, y1, x2, y2, outline);

Argument Description
x1 La coordonnée x du coin supérieur gauche du rectangle arrondi (comme s'il s'agissait d'un rectangle normal).
y1 La coordonnée y du coin supérieur gauche du rectangle arrondi (comme s'il s'agissait d'un rectangle normal).
x2 La coordonnée x du coin inférieur droit du rectangle arrondi (comme s'il s'agissait d'un rectangle normal).
y2 La coordonnée y du coin inférieur droit du rectangle arrondi (comme s'il s'agissait d'un rectangle normal).
outline Indique si le rectangle arrondi est dessiné rempli (faux) ou sous la forme d'un contour large d'un pixel (vrai).

Retour:

N/A

Exemple:

draw_set_colour(c_lime);
draw_roundrect(50, 100, 250, 200, 0);

Cela dessinera un rectangle arrondi rempli, avec son coin supérieur gauche théorique (avant d'être courbé) à (50,100) et son coin inférieur droit théorique à (250,200).
