dessiner_rectangle

Avec cette fonction, vous pouvez dessiner soit le contour d'un rectangle, soit un rectangle rempli dont la position (x1,y1) est le coin supérieur gauche et la position (x2,y2) le coin inférieur droit. Veuillez noter que le rectangle dessiné peut avoir besoin de différentes (+/-1 sur les x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.

Syntaxe :

draw_rectangle(x1, y1, x2, y2, outline) ;

Arguments Description
x1 La coordonnée x du coin supérieur gauche du rectangle.
y1 La coordonnée y du coin supérieur gauche du rectangle.
x2 La coordonnée x du coin inférieur droit du rectangle.
y2 La coordonnée y du coin inférieur droit du rectangle.
aperçu Si le rectangle est dessiné rempli (false) ou comme un contour d'un pixel de large (true).

Les retours :

N/A

Exemple :

draw_set_colour(c_yellow) ;
draw_rectangle(100, 100, 300, 200, true) ;

Cela va dessiner le contour d'un rectangle, avec son coin supérieur gauche à (100,100) et son coin inférieur droit à (300,200).
