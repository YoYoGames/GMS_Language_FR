draw_roundrect_ext

Avec cette fonction, vous pouvez dessiner soit le contour d'un rectangle arrondi, soit un rectangle arrondi rempli dont la position (x1,y1) est le coin supérieur gauche et la position (x2,y2) le coin inférieur droit. Vous devez également fournir des valeurs de rayon pour les axes x et y (en pixels) et les coins seront arrondis de ces valeurs. Vous pouvez définir la précision du dessin des coins à l'aide de la fonction draw_set_circle_precision(). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/-1 sur les axes x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.

Syntaxe :

draw_roundrect_ext(x1, y1, x2, y2, xrad, yrad, outline) ;

Arguments Description
x1 La coordonnée x du coin supérieur gauche du rectangle arrondi (comme s'il s'agissait d'un rectangle normal).
y1 La coordonnée y du coin supérieur gauche du rectangle arrondi (comme s'il s'agissait d'un rectangle normal).
x2 La coordonnée x du coin inférieur droit du rectangle arrondi (comme s'il s'agissait d'un rectangle normal).
y2 La coordonnée y du coin inférieur droit du rectangle arrondi (comme s'il s'agissait d'un rectangle normal).
xrad Le rayon de la courbe le long de l'axe x à partir des coins du rectangle.
yrad Le rayon de la courbe le long de l'axe des y à partir des coins du rectangle.
aperçu Indique si le rectangle arrondi est dessiné rempli (false) ou comme un contour d'un pixel de large (true).

Les retours :

N/A

Exemple :

var dist = point_distance(x, y, mouse_x, mouse_y) / 10 ;
draw_set_colour(c_lime) ;
draw_roundrect_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, 0) ;

Ceci dessinera un rectangle arrondi rempli dont les coins seront arrondis d'une quantité basée sur la position de la souris par rapport à l'instance effectuant le dessin.
