dessiner_rectangle_couleur

Avec cette fonction, vous pouvez dessiner le contour d'un rectangle ou un rectangle rempli en définissant une zone dont la position (x1,y1) est le coin supérieur gauche et la position (x2,y2) le coin inférieur droit. S'il est rempli, vous pouvez définir les pour chaque point d'angle et si ces couleurs ne sont pas les mêmes, vous obtiendrez un effet de gradient de l'un à l'autre (les paramètres de couleur remplaceront la couleur de base définie avec la fonction draw_set_colour()). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/-1 sur les x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge.

NOTE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.

Syntaxe :

draw_rectangle_colour(x1, y1, x2, y2, col1, col2, col3, col4, outline) ;

Arguments Description
x1 La coordonnée x de la partie gauche du rectangle.
y1 La coordonnée y du haut du rectangle.
x2 La coordonnée x de la droite du rectangle.
y2 La coordonnée y du bas du rectangle.
col1 La couleur du coin supérieur gauche.
col2 La couleur du coin supérieur droit.
col3 La couleur du coin inférieur droit.
col4 La couleur du coin inférieur gauche.
aperçu Indique si le rectangle est un contour (true) ou rempli (false).

Les retours :

N/A

Exemple :

draw_rectangle_colour(50, 50, 200, 200, c_black, c_white, c_black, c_white, false) ;

Cela dessine un carré rempli avec un dégradé noir/blanc entre les coins.
