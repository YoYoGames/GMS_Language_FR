draw_roundrect_colour

Avec cette fonction, vous pouvez dessiner le contour d'un rectangle arrondi ou un rectangle arrondi rempli où la position (x1, y1) est le coin supérieur gauche et la position (x2, y2) est le coin inférieur droit. S'il est rempli, vous pouvez définir les couleurs individuelles pour le centre et les bords, et si ces couleurs ne sont pas les mêmes, vous obtiendrez un effet de dégradé de l'une à l'autre (les paramètres de couleur remplaceront la couleur de base définie avec la fonction draw_set_colour() Vous pouvez définir la précision du dessin des coins avec la fonction draw_set_circle_precision(), mais les coins sont toujours dessinés avec un rayon fixe. Si vous avez besoin de changer le rayon du coin, vous devez utiliser la fonction draw_roundrect_colour_ext(). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/- 1 sur le x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les différentes plates-formes prises en charge.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises, et le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous devez dessiner des formes de cette manière, le shader doit être personnalisé dans cet esprit.

Syntaxe:

draw_roundrect_colour(x1, y1, x2, y2, col1, col2, outline);

Argument Description
x1 Coordonnée x de la gauche du rectangle arrondi.
y1 Coordonnée y du haut du rectangle arrondi.
x2 Coordonnée x de la droite du rectangle arrondi.
y2 Coordonnée y du bas du rectangle arrondi.
col1 La couleur centrale.
col2 La couleur du bord extérieur.
outline Si le rectangle est un contour (vrai) ou rempli (faux).

Retour:

N/A

Exemple:

draw_roundrect_colour(50, 50, 200, 200, c_black, c_white, false);

Cela dessinerait un carré rempli avec des coins arrondis et avec un dégradé noir / blanc lisse du centre vers les bords.

Retour: Formulaires de base
