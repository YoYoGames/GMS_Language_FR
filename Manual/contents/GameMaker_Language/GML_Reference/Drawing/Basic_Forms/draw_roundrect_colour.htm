dessiner_arrondi_couleur

Avec cette fonction, vous pouvez dessiner soit le contour d'un rectangle arrondi, soit un rectangle arrondi rempli dont la position (x1,y1) est le coin supérieur gauche et la position (x2,y2) le coin inférieur droit. S'il est rempli, vous pouvez définir les couleurs individuelles pour le centre et les bords, et si ces couleurs ne sont pas les mêmes, vous obtiendrez un effet de gradient de l'une à l'autre (les paramètres de couleur remplacent la couleur de base définie avec la fonction draw_set_colour()).vous Vous pouvez définir le degré de précision du dessin des angles avec la fonction draw_set_circle_precision()Vous pouvez définir le degré de précision du dessin des coins avec la fonction "-", mais les coins sont toujours dessinés avec un rayon fixe. Si vous souhaitez modifier le rayon des coins vous devez utiliser la fonction draw_roundrect_colour_ext(). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/-1 sur les x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge. dimensions souhaitées, en raison des différences entre les diverses plates-formes prises en charge.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.

Syntaxe :

draw_roundrect_colour(x1, y1, x2, y2, col1, col2, outline) ;

Arguments Description
x1 La coordonnée x de la partie gauche du rectangle arrondi.
y1 La coordonnée y du haut du rectangle arrondi.
x2 La coordonnée x de la droite du rectangle arrondi.
y2 La coordonnée y du bas du rectangle arrondi.
col1 La couleur centrale.
col2 La couleur du bord extérieur.
aperçu Indique si le rectangle est un contour (true) ou rempli (false).

Les retours :

N/A

Exemple :

draw_roundrect_colour(50, 50, 200, 200, c_black, c_white, false) ;

Cela dessine un carré rempli avec des coins arrondis et un gradient lisse noir/blanc du centre vers les bords.
