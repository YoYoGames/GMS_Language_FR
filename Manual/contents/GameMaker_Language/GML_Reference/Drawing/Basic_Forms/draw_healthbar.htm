dessiner_barre de santé

Cette fonction vous permet de dessiner une barre de couleur pour afficher une valeur constante. Bien que la fonction utilise le mot "healthbar", vous pouvez l'utiliser pour afficher ce que vous voulez, à condition que la quantité à afficher soit un pourcentage compris entre 0 et 100. 100 (un pourcentage de n'importe quelle valeur peut être calculé avec la formule (CURRENT_Value / MAXIMUM_value) * 100), donc, par exemple, vous pouvez l'utiliser pour afficher la puissance, la santé, le mana, le temps ou tout autre élément qui bénéficierait d'un affichage en barre.

Vous pouvez définir différentes choses qui modifient l'aspect visuel de la barre et celles-ci sont illustrées dans l'image ci-dessous (notez que les couleurs utilisées ici sont c_green pour la valeur maximale et c_red pour la valeur minimale, mais vous pouvez utiliser les couleurs que vous souhaitez): : c_green pour la valeur maximale et c_red pour la valeur minimale mais vous pouvez utiliser les couleurs que vous souhaitez) :

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.

Syntaxe :

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder) ;

Arguments Description
x1 La coordonnée x de la gauche de la barre de santé
y1 La coordonnée y du haut de la barre de santé
x2 La coordonnée x de la droite de la barre de santé
y2 La coordonnée y du bas de la barre de santé
montant La variable qui définit la santé totale (entre 0 et 100)
backcol La couleur de l'arrière-plan de la barre
mincol La couleur de la barre lorsqu'elle n'a pas de santé (0)
maxcol La couleur de la barre lorsqu'elle est à pleine santé (100)
direction Où la barre est "ancrée" (0 = gauche, 1 = droite, 2 = haut, 3 = bas)
showback Si l'arrière-plan de la barre est dessiné (true) ou non (false). Si false, l'argument backcol n'est pas pris en compte.
montrer la frontière Indique si les éléments de la barre ont une bordure noire de 1px de large (true) ou non (false).

Les retours :

N/A

Exemple :

var pc ;
pc = (hp / max_hp) * 100 ;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

Le code ci-dessus utilise la valeur en pourcentage que vous obtenez des variables "hp" et "hp_max" pour dessiner une barre de santé rouge/verte standard.
