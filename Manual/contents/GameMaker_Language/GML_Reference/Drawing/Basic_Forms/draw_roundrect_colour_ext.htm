draw_roundrect_colour_ext

Avec cette fonction, vous pouvez dessiner le contour d'un rectangle arrondi ou un rectangle arrondi rempli où la position (x1, y1) est le coin supérieur gauche et la position (x2, y2) est le coin inférieur droit. Si le rectangle est rempli, les arguments de couleur seront utilisés pour générer un dégradé de couleur du centre vers les bords, où la couleur 1 est la couleur centrale et la couleur 2 la couleur des bords. Vous devez également fournir des valeurs de rayon pour les axes x et y (en pixels) et les coins seront arrondis de ces montants. Vous pouvez définir la précision du dessin des coins avec la fonction draw_set_circle_precision(). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/- 1 sur le x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les différentes plates-formes prises en charge.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises, et le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous devez dessiner des formes de cette manière, le shader doit être personnalisé dans cet esprit.

Syntaxe:

draw_roundrect_colour_ext(x1, y1, x2, y2, xrad, yrad, col1, col2, outline);

Argument Description
x1 Coordonnée x de la gauche du rectangle arrondi.
y1 Coordonnée y du haut du rectangle arrondi.
x2 Coordonnée x de la droite du rectangle arrondi.
y2 Coordonnée y du bas du rectangle arrondi.
xrad Le rayon de la courbe le long de l'axe x à partir des coins du rectangle.
yrad Le rayon de la courbe le long de l'axe y à partir des coins du rectangle.
col1 La couleur centrale.
col2 La couleur du bord extérieur.
outline Si le rectangle est un contour (vrai) ou rempli (faux).

Retour:

N/A

Exemple:

var dist = point_distance(x, y, mouse_x, mouse_y) / 10;
var col = make_colour_hsv(clamp(dist, 0, 255), 255, 255);
draw_roundrect_colour_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, col, c_white, 0);

Cela dessinerait un carré rempli avec des coins arrondis, dont le rayon est défini par la distance entre l'occurrence faisant le dessin et la souris. Cette valeur est également utilisée pour calculer la couleur de fusion centrale.
