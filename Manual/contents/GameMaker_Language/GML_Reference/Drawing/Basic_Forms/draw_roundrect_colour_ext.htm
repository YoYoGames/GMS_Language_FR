draw_roundrect_colour_ext

Cette fonction permet de dessiner soit le contour d'un rectangle arrondi, soit un rectangle arrondi rempli dont la position (x1,y1) est le coin supérieur gauche et la position (x2,y2) le coin inférieur droit. Si le rectangle est rempli, alors les arguments de couleur seront utilisés pour générer un gradient de couleur du centre vers les bords, où la couleur 1 est la couleur du centre et la couleur 2 la couleur des bords. Vous devez également fournir des valeurs de rayon pour les axes x et y (en pixels) et les coins seront arrondis de ces valeurs. de ces valeurs. Vous pouvez définir la précision du dessin des coins à l'aide de la fonction draw_set_circle_precision(). Veuillez noter que le rectangle dessiné peut nécessiter des valeurs différentes (+/-1 sur les x, y, ou la largeur ou la hauteur) pour être dessiné avec les dimensions souhaitées en raison des différences entre les diverses plateformes prises en charge.

NOTE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes : vertex, texture, couleur. Cependant, en utilisant cette fonction, seules les données de vertex et de couleur sont transmises, et donc le shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). shader peut ne rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.

Syntaxe :

draw_roundrect_colour_ext(x1, y1, x2, y2, xrad, yrad, col1, col2, outline) ;

Arguments Description
x1 La coordonnée x de la partie gauche du rectangle arrondi.
y1 La coordonnée y du haut du rectangle arrondi.
x2 La coordonnée x de la droite du rectangle arrondi.
y2 La coordonnée y du bas du rectangle arrondi.
xrad Le rayon de la courbe le long de l'axe x à partir des coins du rectangle.
yrad Le rayon de la courbe le long de l'axe des y à partir des coins du rectangle.
col1 La couleur centrale.
col2 La couleur du bord extérieur.
aperçu Indique si le rectangle est un contour (true) ou rempli (false).

Les retours :

N/A

Exemple :

var dist = point_distance(x, y, mouse_x, mouse_y) / 10 ;
var col = make_colour_hsv(clamp(dist, 0, 255), 255, 255) ;
draw_roundrect_colour_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, col, c_white, 0) ;

Cela dessine un carré rempli avec des coins arrondis, dont le rayon est défini par la distance entre l'instance qui effectue le dessin et la souris. Cette valeur est également utilisée pour calculer la couleur de fusion centrale.
