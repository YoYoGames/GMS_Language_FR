draw_vertex_texture

Cette fonction définit la position d'un sommet texturé pour une primitive. L'aspect final de la primitive dépendra du type primitif choisi pour dessiner (voir draw_primitive_begin() pour plus d'informations), l'ordre dans lequel vous ajoutez les sommets et la position du point de départ que vous avez défini pour la texture. Pour terminer et dessiner la primitive, vous devez appeler draw_primitive_end().

Vous pouvez également choisir de ne dessiner qu'une partie de la texture sur la surface de la primitive en définissant la position xtex / ytex appropriée à échantillonner (ces coordonnées sont également appelées points UV ). Ce point est défini par une valeur comprise entre 0 et 1, le coin supérieur gauche de l'image à utiliser étant le point (0,0) et le coin inférieur droit étant le point (1,1). De cette manière, vous pouvez texturer une primitive avec seulement une partie de l'image source.

Lorsque vous dessinez une primitive texturée, la texture sera mélangée avec la couleur définie par draw_set_colour. Cela signifie que pour dessiner la texture sans mélange, cela doit être réglé sur c_white, comme illustré dans l'exemple ci-dessous.

NOTE: Pour qu'une texture se répète, elle doit avoir une puissance de deux, c'est-à-dire: 32x32, 128x128, etc...

Syntaxe:

draw_vertex_texture(x, y, xtex, ytex)

Argument Description
X La coordonnée x du sommet.
y La coordonnée y du sommet.
xtex La largeur de l'échantillon de texture à utiliser.
ytex La hauteur de l'échantillon de texture à utiliser.

Retour:

N/A

Exemple:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

Le code ci-dessus dessinera une bande de triangle de 4 sommets (faisant un rectangle) texturée avec la texture contenue dans la variable "tex", et la texture entière sera utilisée pour couvrir la primitive terminée.
