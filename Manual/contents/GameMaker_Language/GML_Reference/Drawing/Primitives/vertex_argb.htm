vertex_argb

Cette fonction définira les valeurs ARGB pour le sommet actuellement défini pour la primitive personnalisée. Vous fournissez le tampon dans lequel écrire les données ainsi que les valeurs rouge, verte, bleue et alpha que vous souhaitez utiliser sous la forme d'un entier non signé de 32 bits. que vous souhaitez utiliser sous la forme d'un seul entier non signé de 32 bits - échantillon alpha dans les 8 bits les plus élevés, suivi de l'échantillon rouge, de l'échantillon vert et enfin de l'échantillon bleu dans les 8 bits les plus bas. Vous pouvez utiliser la notation hexadécimale ($AARRGGBB), un nombre réel ou utiliser l'une des fonctions make_colour_*. l'une des fonctions make_colour_*() pour définir la valeur de la couleur.

Syntaxe :

vertex_argb(buffer, argb) ;

Arguments Description
tampon Le tampon dans lequel écrire l'information.
a La valeur alpha de la couleur (0 - 255).
r La partie de la composante rouge d'une couleur (0 - 255).
g La partie de la composante verte d'une couleur (0 - 255).
b La partie de la composante bleue d'une couleur (0 - 255).

Les retours :

N/A

Exemple :

vertex_argb(buff, $FFFFFFFF) ;

Le code ci-dessus définira les valeurs ARGB du sommet défini comme étant blanc.
