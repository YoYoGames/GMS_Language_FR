tampon de création de vertex

Cette fonction vous permet de créer un nouveau tampon de sommets. Il s'agit d'un tampon de croissance spécial créé par GameMaker Studio 2 qui est préformaté pour être utilisé lors de la construction de primitives (pour être utilisé avec les shaders, par exemple). La fonction renvoie un "Pointeur" (index) pour le tampon qui devra ensuite être utilisé dans tous les appels ultérieurs à cette fonction.

Lorsque vous utilisez un tampon de sommets créé avec cette fonction, il vous suffit d'appeler vertex_begin() pour commencer à lui assigner des données de vertex afin de commencer à définir votre primitive personnalisée, qui sera ensuite maintenue dans le tampon, prête à être soumise au shader. prêt à être soumis au shader. Le tampon peut être réutilisé si nécessaire (sauf si vous avez utilisé la fonction vertex_freeze() ), chaque appel de vertex_begin() efface les données précédentes du tampon données du tampon, prêt à accepter les nouvelles données.

Syntaxe :

vertex_create_buffer() ;

Les retours :

Pointeur

Exemple :

v_buff = vertex_create_buffer() ;

Le code ci-dessus va créer un nouveau tampon de sommets et stocker son identifiant dans la variable "v_buff".
