vertex_colour

Si votre format de sommet défini prend une valeur de couleur, vous pouvez utiliser cette fonction pour ajouter ces données au sommet défini pour la primitive actuelle. La fonction a besoin d'un tampon pour stocker les données et prendra soit une constante de couleur, soit une valeur hexadécimale (en utilisant le GameMaker Studio 2 de BGR, par exemple: $FF0000 pour le bleu) ainsi qu'une valeur alpha de 0 (transparent ) à 1 (complètement opaque).

Syntaxe:

vertex_colour(buffer, colour, alpha);

Argument Description
buffer Le tampon dans lequel écrire les informations.
colour La couleur de ce sommet (peut être une constante ou une valeur hexadécimale).
alpha La valeur alpha du sommet (de 0 à 1).

Retour:

N/A

Exemple:

vertex_colour(b, c_white, 1);

Le code ci-dessus définira le sommet défini comme étant blanc avec une valeur alpha de 1.
