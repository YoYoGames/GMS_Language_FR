vertex_format_delete

Cette fonction doit être appelée chaque fois que vous avez fini d'utiliser les formats vectoriels créés. Vous fournissez la valeur de l'ID de format (telle que renvoyée par la fonction vector_format_end() ), et cette fonction libérera la mémoire qui lui est associée. Notez que si vous essayez à nouveau d'utiliser ce format après avoir appelé cette fonction, vous obtiendrez une erreur.

Syntaxe:

vertex_format_delete(formatID);

Retour:

N/A

Exemple:

vertex_format_delete(myFormat);

Le code ci-dessus supprimera le format de sommet créé dans la variable "myFormat" de la mémoire.

Retour: Primitives et formats de sommets
