vertex_position_3d

Cette fonction ajoutera des données de position 3D au sommet en cours de définition pour la primitive personnalisée. Vous fournissez le tampon dans lequel écrire les données ainsi que les positions x, y et z pour le dessin.

Syntaxe:

vertex_position_3d(buffer, x, y, z);

Argument Description
buffer Le tampon dans lequel écrire les informations.
x La position x.
y La position y.
z La position z.

Retour:

N/A

Exemple:

vertex_position_3d(buff, x - 100, y - 125, 0);

Le code ci-dessus définira la position du sommet en cours de définition.

Retour: Primitives et formats de sommets
