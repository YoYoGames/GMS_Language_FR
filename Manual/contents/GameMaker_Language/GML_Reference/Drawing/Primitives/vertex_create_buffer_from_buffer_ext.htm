Création de tampon de sommet à partir d'un tampon externe

Comme avec la fonction vertex_create_buffer()cette fonction va créer un nouveau tampon de sommets, mais les données de sommets qu'il stocke sont copiées à partir du tampon normal spécifié comme source. Le tampon est préformaté avec le format de sommet pour construire des primitives à utiliser avec (par exemple) des shaders, et vous pouvez également fournir un décalage dans le tampon source à copier et le nombre de sommets que le tampon final doit avoir. Notez que si le nombre de sommets ne correspond pas à ceux qui sont copiés, vous pouvez obtenir des données de sommets corrompues.

Syntaxe :

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num) ;

Arguments Description
tampon Le tampon à partir duquel créer le tampon des sommets.
format Le format de sommet primitif à utiliser.
src_offset Le décalage dans le tampon source à copier.
vert_num Le nombre de sommets que le tampon doit avoir.

Les retours :

Pointeur

Exemple :

vertex_format_begin() ;
vertex_format_add_position_3d() ;
vertex_format_add_colour() ;
vertex_format_add_textcoord() ;
var my_format = vertex_format_end() ;
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512) ;

Le code ci-dessus crée un nouveau format de vertex puis crée un nouveau tampon de vertex à partir d'un tampon régulier précédemment créé, en lui appliquant le format de vertex personnalisé avec un décalage de 0. La fonction indique au nouveau tampon de sommets qu'il doit s'attendre à 512 sommets.
