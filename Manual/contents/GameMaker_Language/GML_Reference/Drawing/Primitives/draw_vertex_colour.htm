dessiner_vertex_couleur

Cette fonction définit la position d'un sommet pour une primitive, avec sa propre couleur et son paramètre alpha. L'aspect final de la primitive dépendra du type de primitive choisi pour la dessiner et de l'ordre dans lequel vous lui ajoutez les sommets (voir draw_primitive_begin() pour plus d'informations). Les sommets ayant des couleurs et des alphas différents se fondront en douceur les uns dans les autres. Pour terminer et dessiner la primitive, vous devez appeler draw_primitive_end().

Syntaxe :

draw_vertex_colour(x, y, col, alpha)

Arguments Description
x La coordonnée x du sommet.
y La coordonnée y du sommet.
col La couleur avec laquelle ce sommet doit être dessiné.
alpha L'alpha avec lequel ce sommet doit être dessiné (0-1).

Les retours :

N/A

Exemple :

draw_primitive_begin(pr_trianglelist) ;
draw_vertex_colour(100, 100, c_blue, 0.1) ;
draw_vertex_colour(100, 200, c_rouge, 0.1) ;
draw_vertex_colour(150, 150, c_vert, 1) ;
draw_primitive_end() ;

Le code ci-dessus dessine un triangle semi-transparent dont chaque sommet est coloré d'une couleur différente.
