vertex_texcoord

Cette fonction va définir les coordonnées de la texture à utiliser pour le sommet actuellement défini pour la primitive personnalisée. Vous fournissez le tampon dans lequel écrire les données ainsi que la position UV dans la texture à utiliser.

Syntaxe :

vertex_texcoord(buffer, u, v) ;

Arguments Description
tampon Le tampon dans lequel écrire l'information.
u La première coordonnée de texture à utiliser (0 - 1).
v La deuxième coordonnée de texture à utiliser (0 - 1).

Les retours :

N/A

Exemple :

vertex_texcoord(buff, 0, 0) ;

Le code ci-dessus définira les valeurs UV du sommet en cours de définition.
