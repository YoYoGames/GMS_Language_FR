vertex_end

Avec cette fonction, vous terminez la construction de la primitive personnalisée. Une fois que vous avez appelé cette commande, la primitive peut être utilisée dans la fonction vertex_submit() fonction pour être utilisée dans un shader ou vous pouvez geler le buffer (rendant le tampon de vertex utilisé en lecture seule et beaucoup plus rapide) en utilisant vertex_freeze().

Syntaxe :

vertex_end(buffer) ;

Arguments Description
tampon Le tampon dans lequel il faut terminer l'écriture.

Les retours :

N/A

Exemple :

vertex_end(buff) ;

Le code ci-dessus met fin à la définition d'une primitive personnalisée.
