make_colour_rgb

GameMaker Studio 2 propose cette fonction (ainsi que d'autres) pour permettre à l'utilisateur de créer ses propres couleurs. Cette fonction particulière prend trois composants, le rouge, le vert et le bleu de la couleur que vous souhaitez créer. que vous souhaitez créer. Ces valeurs sont comprises entre 0 et 255, ce qui vous permet de créer 16 777 216 (256*256*256) couleurs ! Syntaxe :

make_colour_rgb(rouge, vert, bleu) ;

Arguments Description
rouge La composante rouge de la couleur
vert La composante verte de la couleur
bleu La composante bleue de la couleur

Les retours :

Int

Exemple :

col = make_colour_rgb(100, 145, 255) ;

Le code ci-dessus utilise la fonction pour créer une couleur et stocker sa valeur dans la variable "col" pour une utilisation ultérieure.
