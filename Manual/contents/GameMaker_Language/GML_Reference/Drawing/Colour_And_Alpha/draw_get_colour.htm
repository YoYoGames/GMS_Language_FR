draw_get_colour

Cette fonction renvoie la couleur de dessin actuelle qui est utilisée pour dessiner des formes, du texte, des primitives et des modèles 3D non texturés. Cela peut être défini avec le draw_set_colour() fonction.

Syntaxe:

draw_get_colour()

Retour:

Real

Exemple:

if draw_get_colour() != c_black
   {
   draw_set_colour(c_black);
   }

Le code ci-dessus vérifie d'abord la couleur de dessin actuelle, puis s'il n'est pas défini sur c_black il le met à c_black.
