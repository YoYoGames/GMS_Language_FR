draw_getpixel

Avec cette fonction, vous pouvez obtenir la valeur de couleur de tout pixel dessiné vers la cible de rendu actuelle. Cela signifie que les résultats dépendront de l'événement dans lequel la fonction est appelée, ainsi que de la surface cible utilisée. Notez que cela ne renverra aucune valeur alpha, pour cela vous devez utiliser draw_getpixel_ext().

REMARQUE: Cette fonction entraînera un énorme impact sur les performances et ne doit donc être utilisée que lorsque cela est absolument nécessaire.

Syntaxe:

draw_getpixel(x, y);

Argument Description
x La coordonnée x du pixel à vérifier
y La coordonnée y du pixel à vérifier

Retour:

Real

Exemple:

draw_set_colour(draw_getpixel(500,200));

Le code ci-dessus définira la couleur du dessin pour tous les textes, formulaires et primitives sur la couleur du pixel à (500 200) sur l'écran de jeu.
