Couleur et alpha

GameMaker Studio 2 dispose d'une sélection complète de fonctions permettant de manipuler la couleur et l'alpha de ce qui est dessiné à l'écran (pour le mélange d'images à l'aide de modes de fusion spéciaux et d'autres fonctions spécifiques à l'unité de traitement graphique, consultez la section sur le contrôle GPU). voir la section sur le contrôle GPU. Ces fonctions vous permettent de séparer les couleurs en leurs composants, de créer de nouvelles couleurs et de manipuler l'alpha du dessin pour modifier la façon dont les différents éléments de votre jeu sont dessinés ensemble. pour modifier la façon dont les différents éléments de votre jeu sont dessinés ensemble. Notez que vous pouvez également créer des couleurs à partir de leur valeur hexadécimale en utilisant au préalable le symbole "$", et ces valeurs peuvent également être utilisées dans les fonctions de couleur pour obtenir les teintes des composants ou pour les mélanges, etc... La valeur hexadécimale de ces couleurs est divisée en trois parties avec le format Bleu / Vert / Rouge. Ainsi, par exemple, pour définir une couleur violette, vous pouvez utiliser :

col = 983c95

Où 98 est la composante bleue, 3c est la composante verte et 95 est la composante rouge.

Les fonctions suivantes peuvent être utilisées pour obtenir les teintes composantes, la teinte composée, la saturation et la luminosité d'une couleur sélectionnée ainsi que l'alpha et d'autres propriétés :

  • couleur_get_bleu
  • couleur_get_green
  • couleur_get_rouge
  • couleur_get_hue
  • couleur_get_saturation
  • couleur_get_valeur
  • draw_getpixel
  • draw_getpixel_ext
  • draw_get_colour
  • draw_get_alpha

Ces fonctions peuvent être utilisées pour créer des couleurs à partir de valeurs d'entrée brutes pour les composantes de couleur :

  • make_colour_hsv
  • make_colour_rgb
  • fusionner_couleur

Les fonctions suivantes peuvent être utilisées pour définir les différentes options de dessin à l'écran, notamment l'alpha, la couleur et le mélange :

  • dessiner_clair
  • draw_clear_alpha
  • draw_set_alpha
  • draw_set_colour
