colour_get_hue

Cette fonction renverra la teinte de la couleur donnée. Il s'agit de la tonalité de couleur "pure" qui fait partie de la méthode de teinte, de saturation et de valeur (luminosité) pour définir une couleur. L'image suivante montre comment cette valeur correspond à l'échelle de couleur HSV:

Syntaxe:

colour_get_hue(col);

Argument Description
col La couleur à vérifier

Retour:

Int

Exemple:

col = merge_colour(colour_get_hue(c_teal), c_white, 0.5);

Le code ci-dessus récupère la teinte utilisée pour rendre la couleur constante "c_teal", puis la fusionne avec du blanc à un rapport 50:50, en stockant la couleur résultante dans la variable "col".
