draw_clear_alpha

Cette fonction peut être utilisée pour effacer tout l'écran avec une couleur donnée et la composante alpha de la destination est définie sur la valeur que vous avez défini - cette fonction ne fait aucun mélange que cela fonctionne mais les opérations de mélange suivantes peut être mis en place pour utilisez l'alpha de destination que vous avez défini. Ceci n'est utilisé que dans l'événement de dessin d'une instance (il ne s'affichera pas s'il est utilisé dans un autre événement), et cela peut également être très utile pour effacer les surfaces lorsqu'elles sont nouvellement créées.

Il est à noter que lorsque vous utilisez le module HTML5, vous pouvez utiliser cette fonction pour rendre l'arrière-plan du canevas transparent, ce qui permettra ensuite à toutes les images utilisées dans le HTML hôte d'être vues avec le jeu en cours d'exécution. Pour obtenir cet effet, vous devez d'abord accéder à l'éditeur de salle et supprimer les calques d'arrière-plan de l' éditeur de pièce, puis décocher l'option Effacer l'arrière-plan de la vue dans les propriétés de la vue. Enfin, vous devrez créer un nouvel objet, le placer dans la première salle du jeu et lui attribuer une profondeur très élevée (par exemple, 10000). Ensuite, dans l' événement Pre Draw, utilisez cette fonction pour définir l'alpha du canevas sur la couleur et la transparence que vous souhaitez qu'il ait.

Syntaxe:

draw_clear_alpha(col, alpha);

Argument Description
col La couleur avec laquelle l'écran sera effacé
alpha La transparence de la couleur avec laquelle l'écran sera effacé

Retour:

N/A

Exemple:

draw_clear_alpha(c_red, 0.5);

Cela effacera l'écran avec une transparence de 50% (0,5) avec la couleur rouge.
