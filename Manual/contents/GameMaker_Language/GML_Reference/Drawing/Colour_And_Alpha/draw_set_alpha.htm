draw_set_alpha

Avec cette fonction, vous pouvez définir l'alpha de base du dessin pour le jeu. Cette valeur peut être définie de 0 à 1, 0 étant totalement transparent et 1 étant totalement opaque (la valeur par défaut), et affectera tous les autres dessins, y compris les arrière-plans, les sprites, les polices, les primitives et la 3D.

Notez que si vous dessinez l'un des actifs et que vous fournissez un argument alpha dans le cadre de la fonction (par exemple, lorsque vous utilisez la commande draw_sprite_ext()), alors la valeur alpha définie ici sera Ainsi, dessiner un sprite à l'aide de draw_sprite_ext() avec un argument alpha de 1 rendra le sprite complètement opaque, quel que soit l'alpha défini avec draw_set_alpha() (cela fonctionne de la même manière lorsque vous utilisez draw_self() et en définissant le paramètre image_alpha de l'instance). Notez également que lorsqu'une instance est dessinée par défaut (c'est-à-dire qu'elle possède un sprite mais pas d'événement de dessin), la variable image_alpha sera fixée à l'alpha défini avec draw_set_alpha(), de sorte qu'une image_alpha de 1 et un draw alpha de 0,5 dessineront le sprite à l'alpha 0,5, tandis qu'une image_alpha de 0,25 et un draw alpha de 0,5 dessineront le sprite à l'alpha 0,5. de 0,5 dessineront le sprite avec un alpha de 0,25.

Syntaxe :

draw_set_alpha(alpha) ;

Arguments Description
alpha L'alpha à définir (entre 0 et 1)

Les retours :

N/A

Exemple :

draw_set_alpha(0.5) ;
draw_set_colour(c_black) ;
draw_text(x+5, y+5, "LEVEL 1") ;
draw_set_alpha(1) ;
draw_set_colour(c_white) ;
draw_text(x, y, "LEVEL 1") ;

Le code ci-dessus dessine un texte à la position spécifiée avec un effet d'ombre créé en modifiant l'alpha et la couleur du texte.
