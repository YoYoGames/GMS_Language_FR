colour_get_green

Cette fonction renvoie la quantité de vert utilisée pour créer la couleur donnée, avec une valeur comprise entre 0 et 255, où 0 signifie pas de vert et 255 est tout vert. L'image suivante illustre cela:

Syntaxe:

colour_get_green(col);

Argument Description
col La couleur à vérifier

Retour:

Int

Exemple:

g_comp = colour_get_green(c_teal);

Le code ci-dessus obtiendra la composante verte de la constante de couleur "c_teal" et stockera la valeur dans la variable "g_comp".

Dos: couleur et alpha
