colour_get_blue

Cette fonction renvoie la quantité de bleu utilisée pour créer la couleur donnée, avec une valeur comprise entre 0 et 255, où 0 signifie pas de bleu et 255 est tout bleu. L'image suivante illustre cela:

Syntaxe:

colour_get_blue(col);

Argument Description
col La couleur à vérifier

Retour:

Int

Exemple:

b_comp = colour_get_blue(c_teal);

Le code ci-dessus obtiendra la composante bleue de la constante de couleur "c_teal" et stockera la valeur dans la variable "b_comp".

Dos: couleur et alpha
