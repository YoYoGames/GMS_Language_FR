couleur_get_bleu

Cette fonction renvoie la quantité de bleu utilisée pour obtenir la couleur donnée, la valeur étant comprise entre 0 et 255, où 0 correspond à aucun bleu et 255 à tout le bleu. L'image suivante illustre cela :

Syntaxe :

couleur_get_bleu(col) ;

Arguments Description
col La couleur à vérifier

Les retours :

Int

Exemple :

b_comp = couleur_get_bleu(c_teal) ;

Le code ci-dessus récupère la composante bleue de la constante de couleur "c_teal" et enregistre la valeur dans la variable "b_comp".
