texture_is_ready

Cette fonction peut être utilisée pour vérifier si une page de texture spécifique a été décompressée et est prête à être utilisée, ou si un groupe de pages de texture a été décompressé et est prêt à être utilisé. Vous fournissez l' ID de page de texture unique (tel que renvoyé par le texturegroup_* fonctions) ou la chaîne d'ID du groupe de textures (tel que défini dans l'éditeur de groupe de textures), et la fonction retournera true s'ils ont été déballés, ou false autrement.

Syntaxe:

texture_is_ready(tex_id);

Argument Description
tex_id L'ID de page de texture (un entier) ou un ID de groupe de texture (une chaîne)

Retour:

Boolean

Exemple:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
    {
    if !texture_is_ready(_tex_array[i])
        {
        texture_prefetch(_tex_array[i]);
        }
    }

Le code ci-dessus récupérera les identifiants de page de texture pour le groupe de texture "MainMenu", puis vérifiera s'ils sont décompressés, et s'ils ne le sont pas, ils sont pré-lus dans la VRAM.
