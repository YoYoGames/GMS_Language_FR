texture_get_uvs

Cette fonction renvoie un tableau 1D avec 4 éléments représentant les coordonnées UV de l'image sur la page de texture, en remplissant le tableau avec les valeurs suivantes:

  • [0] = gauche
  • [1] = haut
  • [2] = droite
  • [3] = bas
  • [4] = quantité de pixels que le compilateur d'éléments a supprimés du côté gauche des images-objets (éléments d'image-objet uniquement)
  • [5] = quantité de pixels que le compilateur d'éléments a supprimés du côté supérieur des sprites (éléments de sprite uniquement)
  • [6] = pourcentage normalisé de données de pixels par rapport à la largeur des sprites d'origine qui a été enregistrée sur la page de texture (éléments de sprite uniquement)
  • [7] = pourcentage normalisé de données de pixels par rapport à la hauteur des sprites d'origine qui a été enregistrée sur la page de texture (éléments de sprite uniquement)

Cette valeur peut ensuite être utilisée dans d'autres fonctions de dessin, en particulier en 3D générale et certaines des fonctions primitives 2D, ainsi que les fonctions Shader Si vous avez besoin de l'UVS pour un sprite, vous pouvez utiliser le sprite_get_uvs() et pour une police, font_get_uvs().

Syntaxe:

texture_get_uvs(texid)

Argument Description
texid L'élément d'image à partir duquel obtenir la texture

Retour:

1D Array (4 - 8 elements)

Exemple:

var tex = texture_get_uvs(surf_back);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

Le code ci-dessus stockera les coordonnées UV pour la texture de surface donnée dans un tableau local, puis affectera les valeurs aux variables d'instance.
