texturegroup_get_fonts

Avec cette fonction, vous pouvez récupérer l'index des polices de chacune des polices affectées aux pages de texture dans le groupe de textures donné. Vous fournissez la chaîne d'ID de groupe de texture (telle que définie dans l'éditeur de groupe de texture) et la fonction renverra un tableau 1D où chaque entrée contient l'index de police pour une ressource de police. Si la fonction échoue - c'est-à-dire: un groupe invalide est donné, ou le groupe n'a pas de texture qui lui est assignée - alors le tableau sera vide (longueur 0).

Syntaxe:

texturegroup_get_fonts(tex_id);

Argument Description
tex Le nom du groupe de textures à vérifier (une chaîne)

Retour:

1D array

Exemple:

var _tex_array = texturegroup_get_fonts( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
    {
    show_debug_message("Font " + string(i) + " Index:" + string(tex_array[i]));
    }

Le code ci-dessus récupérera les index de polices pour le groupe de textures "MainMenu", puis affichera ces ID dans la fenêtre de sortie de la console.
