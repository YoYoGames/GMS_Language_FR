texturegroup_get_fonts

Cette fonction permet de récupérer l'index de chacune des polices affectées aux pages de texture dans un groupe de texture donné. Vous fournissez la chaîne d'identification du groupe de texture (telle que définie dans l'éditeur de groupe de texture) et la fonction renvoie un tableau 1D où chaque entrée contient l'index de police pour une ressource de police. Si la fonction échoue - c'est-à-dire si un groupe invalide est fourni, ou si le groupe n'a pas de texture assignée - alors le tableau sera vide (longueur 0).

Syntaxe :

texturegroup_get_fonts(tex_id) ;

Arguments Description
tex Le nom du groupe de texture à vérifier (une chaîne)

Les retours :

Tableau 1D

Exemple :

var _tex_array = texturegroup_get_fonts( "MainMenu") ;
for (var i = 0 ; i < array_length(_tex_array) ; ++i ;)
{
show_debug_message("Font " + string(i) + " Index :" + string(tex_array[i])) ;
}

Le code ci-dessus récupère les index des polices pour le groupe de textures "MainMenu", puis affiche ces ID dans la fenêtre de sortie de la console.
