largeur_de_la_texture

Renvoie la largeur de la texture avec l'identifiant donné, qui est toujours une valeur comprise entre 0 et 1. Cette valeur peut ensuite être utilisée lors du mappage des textures aux modèles ou aux primitives.

Syntaxe :

texture_get_width(tex) ;

Arguments Description
tex Le pointeur de ressource de la page de texture à utiliser

Les retours :

Real

Exemple :

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0)) ;

Le code ci-dessus obtiendra la largeur de la texture prise à partir d'un actif de sprite.
