texture_get_height

Renvoie la hauteur de la texture avec l'identifiant donné, qui est toujours une valeur comprise entre 0 et 1. Cette valeur peut ensuite être utilisée lors du mappage des textures aux modèles ou aux primitives.

Syntaxe :

texture_get_height(tex) ;

Arguments Description
tex Le pointeur de ressource de la page de texture à utiliser

Les retours :

Real

Exemple :

tex_h = texture_get_height(surface_get_texture(global.Surf)) ;

Le code ci-dessus permet d'obtenir la hauteur de la texture prise dans une surface précédemment créée.
