texture_get_height

Renvoie la hauteur de la texture avec l'identifiant donné, qui est toujours une valeur comprise entre 0 et 1. Cela peut ensuite être utilisé lors du mappage de textures sur des modèles ou des primitives.

Syntaxe:

texture_get_height(tex);

Argument Description
tex Le pointeur de ressource de page de texture à utiliser

Retour:

Real

Exemple:

tex_h = texture_get_height(surface_get_texture(global.Surf));

Le code ci-dessus obtiendra la hauteur de la texture prise à partir d'une surface créée précédemment.
