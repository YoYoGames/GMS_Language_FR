texture_get_texel_width

Un texel, ou élément de texture est l'unité fondamentale de l'espace de texture utilisée dans l'infographie. Les textures sont représentées par des tableaux de texels, tout comme les images sont représentées par des tableaux de pixels, et cette fonction renvoie la largeur d'un seul texel à partir de la page de texture de l'élément d'image utilisé.

Syntaxe:

texture_get_texel_width(tex);

Argument Description
tex Le pointeur de ressource de page de texture à utiliser

Retour:

Real

Exemple:

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

Le code ci-dessus obtiendra la largeur et la hauteur du texel de la texture provenant d'un élément de sprite.
