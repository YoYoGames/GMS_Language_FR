texture_prefetch

Cette fonction peut être utilisée pour "prélever" une page de texture ou un groupe de pages de texture, c'est-à-dire les charger dans la VRAM quand cela est nécessaire. Vous fournissez l'ID unique de la page de texture (tel que renvoyé par les fonctions texturegroup_* ) pour prélever une seule page ou vous pouvez fournir le nom d' un groupe de textures (tel que défini dans l'éditeur de groupe de textures) pour précharger toutes les pages de textures du groupe.

Syntaxe :

texture_prefetch(tex_id) ;

Arguments Description
tex_id L'ID de la page de texture (un nombre entier) ou l'ID d'un groupe de texture (une chaîne de caractères).

Les retours :

N/A

Exemple :

var _tex_array = texturegroup_get_textures( "MainMenu") ;
for (var i = 0 ; i < array_length(_tex_array) ; ++i ;)
{
texture_prefetch(_tex_array[i]) ;
}

Le code ci-dessus va prélever toutes les pages de texture sous le groupe de texture "MainMenu".
