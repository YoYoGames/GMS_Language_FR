draw_set_lighting

Cette fonction est utilisée pour activer tous les effets d'éclairage. La valeur par défaut est désactivée (false ).

Syntaxe:

draw_set_lighting(enable);

Argument Description
enable activer ou désactiver tous les éclairages (true ou fausse)

Retour:

N/A

Exemple:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Le code ci-dessus activera l'éclairage pour toute la scène, définira ensuite une lumière directionnelle blanche dans l'espace de la pièce, puis allumera finalement cette lumière.

Retour: Éclairage
