draw_light_define_direction

Cette fonction permet de définir une lumière directionnelle, où la direction et l'intensité sont définies par les valeurs entrées pour le vecteur unitaire x, y et z (ce qui signifie qu'elles doivent être comprises entre -1 et 1, où 1 ou -1 est la pleine intensité, et 0 correspond à aucune intensité). Les lumières peuvent également recevoir une couleur, ce qui affectera également l'intensité perçue de la lumière car certaines couleurs semblent plus «foncées» que d'autres. Vous devez également donner à la lumière un numéro d'index qui sera utilisé dans d'autres fonctions pour la référencer.

REMARQUE: Il n'y a que 8 voyants matériels disponibles, donc seuls 8 voyants définis peuvent être activés à la fois (bien que d'autres puissent être définis).

Syntaxe:

draw_light_define_direction(ind, x, y, z, col);

Argument Description
ind Le numéro d'index de la lumière (arbitraire)
x La composante x du vecteur lumière
y La composante y du vecteur lumière
z La composante z du vecteur lumière
col La couleur à utiliser pour la lumière (une valeur constante, réelle ou hexadécimale).

Retour:

N/A

Exemple:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Le code ci-dessus activera l'éclairage pour toute la scène, définira ensuite une lumière directionnelle blanche dans l'espace de la pièce, puis allumera finalement cette lumière.
