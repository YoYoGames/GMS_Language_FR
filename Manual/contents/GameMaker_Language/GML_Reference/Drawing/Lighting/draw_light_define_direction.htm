draw_light_define_direction

Cette fonction permet de définir une lumière directionnelle, dont la direction et l'intensité sont définies par les valeurs saisies pour le vecteur unitaire x, y et z (c'est-à-dire qu'elles doivent être comprises entre -1 et 1, où 1 ou -1 correspond à une intensité maximale et 0 à une intensité nulle). Les lumières de peuvent également être dotées d'une couleur, qui affectera également l'intensité perçue de la lumière, certaines couleurs apparaissant plus "sombres" que d'autres. Vous devez également donner à la lumière un numéro d'index qui sera utilisé dans d'autres fonctions pour la référencer. l'utiliser.

REMARQUE: il n'y a que 8 lumières matérielles disponibles, donc seulement 8 lumières définies peuvent être activées à un moment donné (bien que d'autres puissent être définies).

Syntaxe :

draw_light_define_direction(ind, x, y, z, col) ;

Arguments Description
ind Le numéro d'index du feu (arbitraire)
x La composante x du vecteur lumière
y La composante y du vecteur lumière
z La composante z du vecteur lumière
col La couleur à utiliser pour la lumière (soit une constante, un réel ou une valeur hexadécimale).

Les retours :

N/A

Exemple :

draw_set_lighting(true) ;
draw_light_define_direction(1, 0, 1, 0, c_white) ;
draw_light_enable(1, true) ;

Le code ci-dessus va activer l'éclairage de l'ensemble de la scène, puis définir une lumière blanche directionnelle dans l'espace de la pièce, et enfin allumer cette lumière.
