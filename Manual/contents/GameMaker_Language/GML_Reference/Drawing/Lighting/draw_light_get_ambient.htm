draw_light_get_ambient

Cette fonction retournera la couleur actuelle utilisée pour l'éclairage ambiant.

Syntaxe:

draw_light_get_ambient()

Retour:

Real

Exemple:

light_col = draw_light_get_ambient();

Le code ci-dessus obtiendra la couleur actuelle de la lumière ambiante et la stockera dans une variable.

Retour: Éclairage
