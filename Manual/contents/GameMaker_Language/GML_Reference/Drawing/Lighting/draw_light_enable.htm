draw_light_enable

Cette fonction est utilisée pour activer un feu défini. Lorsque vous définissez un feu positionnel ou directionnel, vous devez lui attribuer un numéro d'index qui est ensuite utilisé par cette fonction pour allumer ou éteindre le feu. La valeur par défaut est désactivée(false).

REMARQUE: il n'y a que 8 lumières matérielles disponibles, donc seulement 8 lumières définies peuvent être activées à un moment donné (bien que d'autres puissent être définies).

Syntaxe :

draw_light_enable(ind, enable) ;

Arguments Description
ind Le numéro d'index du feu (de 0 à 7)
activer Activer ou désactiver tous les éclairages(vrai ou faux)

Les retours :

N/A

Exemple :

draw_set_lighting(true) ;
draw_light_define_direction(1, 0, 1, 0, c_white) ;
draw_light_enable(1, true) ;

Le code ci-dessus va activer l'éclairage de l'ensemble de la scène, puis définir une lumière blanche directionnelle dans l'espace de la pièce, et enfin allumer cette lumière.
