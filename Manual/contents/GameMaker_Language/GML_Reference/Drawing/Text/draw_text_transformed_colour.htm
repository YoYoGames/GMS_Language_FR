draw_text_transformed_colour

Cette fonction est une combinaison de la base draw_text() fonction avec le draw_text_transformed() et draw_text_colour() fonctions, vous permettant de mettre à l'échelle et de faire pivoter le texte ainsi que de le colorer avec un remplissage dégradé et de modifier sa valeur alpha, en ignorant les paramètres alpha et de couleur de base pour le dessin.

REMARQUE: la fusion de dégradé n'est pas disponible pour la cible HTML5 sauf si WebGL est activé, bien que vous puissiez toujours définir les couleurs de fusion et il fusionnera la police avec la première couleur donnée. Cependant, tout mélange de cette manière crée une police en double qui est ensuite stockée dans le cache et utilisée si nécessaire, ce qui est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de vos jeux. Vous pouvez définir la taille du cache de polices pour essayer de limiter cela si nécessaire en utilisant la fonction font_set_cache_size ().

Syntaxe:

draw_text_transformed_colour(x, y, string, xscale, yscale, angle, c1, c2, c3, c4, alpha);

Argument Description
x La coordonnée x de la chaîne dessinée.
y La coordonnée y de la chaîne dessinée.
string La chaîne à dessiner.
xscale L'échelle horizontale.
yscale L'échelle verticale.
angle L'angle du texte.
c1 La couleur du coin supérieur gauche du texte dessiné.
c2 La couleur du coin supérieur droit du texte dessiné.
c3 La couleur du coin inférieur droit du texte dessiné.
c4 La couleur du coin inférieur gauche du texte dessiné.
alpha L'alpha du texte.

Retour:

N/A

Exemple:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed_colour(room_width / 2, room_height / 2, keyboard_string, 2, 2, image_angle, c_red, c_red, c_yellow, c_yellow, 0.5);

Le code ci-dessus dessinera le texte donné au milieu de la pièce, en tournant et en le redimensionnant à deux fois sa taille d'origine, avec un dégradé de couleur allant du jaune au rouge ainsi qu'un alpha de 0,5.
