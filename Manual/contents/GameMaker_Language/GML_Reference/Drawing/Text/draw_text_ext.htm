draw_text_ext

Cette fonction dessine le texte de la même manière que draw_text () ce n'est que maintenant que vous pouvez définir l'espace entre chaque ligne de texte - si le texte occupe plus d'une ligne - et limiter la largeur (en pixels) de la chaîne par ligne afin que si une ligne dépasse cette valeur, GameMaker Studio 2 se divise automatiquement le texte à la ligne suivante à l'espace blanc disponible le plus proche (si le texte n'a pas d'espaces blancs, il dépassera cette valeur de largeur maximale). Notez que tout espace blanc placé au début de la chaîne sera supprimé avant d'être analysé pour le dessin à cause de cela. Notez également qu'une valeur de -1 pour l'argument de séparation de ligne sera par défaut une séparation basée sur la hauteur du caractère "M" dans la police choisie.

Syntaxe:

draw_text_ext(x, y, string, sep, w);

Argument Description
x La coordonnée x de la chaîne dessinée.
y La coordonnée y de la chaîne dessinée.
string La chaîne à dessiner.
sep La distance en pixels entre les lignes de texte.
w La largeur maximale en pixels de la chaîne avant un saut de ligne.

Retour:

N/A

Exemple:

draw_text_ext(100, 50, keyboard_string, 3, 300);

Le code ci-dessus dessinera le texte que l'utilisateur tape sur le clavier, le divisant en nouvelles lignes chaque fois que la longueur de la chaîne pour cette ligne dépasse 300 pixels. le code maintiendra également une séparation de 3 pixels entre les lignes si cela se produit.
