draw_get_halign

Cette fonction est utilisée pour obtenir le paramètre d'alignement du texte le long de l'axe horizontal et renvoie l'une des constantes répertoriées ci-dessous.

Syntaxe:

draw_get_halign();

Retour:

Constant

Constant Alignement
fa_left
fa_center
fa_right

Exemple:

if draw_get_halign() != fa_left
    {
    draw_set_halign(fa_left);
    }
draw_text(100, 32, "Score: " + string(score));

Le code ci-dessus vérifiera si l'alignement actuel du texte est aligné à gauche, et si ce n'est pas le cas, il sera défini et du texte sera dessiné.
