dessiner_texte_couleur

Cette fonction dessine du texte de manière similaire à la fonction dessiner_texte() sauf que maintenant vous pouvez choisir les couleurs à utiliser pour colorer le texte ainsi que la valeur alpha, et ces nouvelles valeurs seront utilisées à la place de la couleur et de l'alpha du dessin de base. Ces nouvelles valeurs seront utilisées à la place de la couleur et de l'alpha du dessin de base.

REMARQUE: le mélange par gradient n'est pas disponible pour la cible HTML5, sauf si WebGL est activé, mais vous pouvez toujours définir les couleurs de mélange et la police sera mélangée avec la première couleur donnée. Cependant, tous les mélanges de cette manière créent une police dupliquée qui est ensuite stockée dans le cache. qui est ensuite stockée dans le cache et utilisée au besoin, ce qui est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de votre jeu. Vous pouvez définir la taille du cache des polices pour essayer de limiter ce phénomène si nécessaire. en utilisant la fonction font_set_cache_size().

Syntaxe :

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha) ;

Arguments Description
x La coordonnée x de la chaîne dessinée.
y La coordonnée y de la chaîne dessinée.
chaîne de caractères La ficelle à tirer.
c1 La couleur de la partie supérieure gauche du texte dessiné.
c2 La couleur de la partie supérieure droite du texte dessiné.
c3 La couleur de la partie inférieure droite du texte dessiné.
c4 La couleur de la partie inférieure gauche du texte dessiné.
alpha L'alpha du texte.

Les retours :

N/A

Exemple :

draw_set_colour(c_white) ;
draw_text(100, 100, "Health") ;
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1) ;

Le code ci-dessus permet de dessiner deux sections de texte sur la même ligne, le premier texte étant dessiné en blanc (puisqu'il s'agit de la couleur de base du dessin) et le second texte étant dessiné avec un dégradé de vert citron à vert normal.
