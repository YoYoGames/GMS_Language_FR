draw_text_colour

Cette fonction dessine le texte de la même manière que draw_text () Seulement maintenant, vous pouvez choisir les couleurs à utiliser pour colorer le texte ainsi que la valeur alpha, et ces nouvelles valeurs seront utilisées à la place de la couleur de dessin de base et de l'alpha.

REMARQUE: la fusion de dégradé n'est pas disponible pour la cible HTML5 sauf si WebGL est activé, bien que vous puissiez toujours définir les couleurs de fusion et il fusionnera la police avec la première couleur donnée. Cependant, tout mélange de cette manière crée une police en double qui est ensuite stockée dans le cache et utilisée si nécessaire, ce qui est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de vos jeux. Vous pouvez définir la taille du cache de polices pour essayer de limiter cela si nécessaire en utilisant la fonction font_set_cache_size ().

Syntaxe:

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha);

Argument Description
x La coordonnée x de la chaîne dessinée.
y La coordonnée y de la chaîne dessinée.
string La chaîne à dessiner.
c1 La couleur du coin supérieur gauche du texte dessiné.
c2 La couleur du coin supérieur droit du texte dessiné.
c3 La couleur du coin inférieur droit du texte dessiné.
c4 La couleur du coin inférieur gauche du texte dessiné.
alpha L'alpha du texte.

Retour:

N/A

Exemple:

draw_set_colour(c_white);
draw_text(100, 100, "Health");
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1);

Le code ci-dessus dessinera deux sections de texte sur la même ligne, le premier texte étant dessiné en blanc (car c'est la couleur de dessin de base) et le second texte dessiné avec un dégradé vert lime à vert normal.
