gpu_get_blendenable

Cette fonction peut être utilisée pour récupérer l'état de l'alpha-blending. Si elle renvoie true, alors l'alpha-blending est activé, et si elle renvoie false, il est désactivé. Par défaut, il est activé et la fonction renvoie donc true.

Syntaxe :

gpu_get_blendenable() ;

Les retours :

Booléen

Exemple :

if gpu_get_blendenable() == false
{
gpu_set_blendenable(true) ;
}

Le code ci-dessus vérifie l'état du mélange alpha et s'il est désactivé, il est réactivé.
