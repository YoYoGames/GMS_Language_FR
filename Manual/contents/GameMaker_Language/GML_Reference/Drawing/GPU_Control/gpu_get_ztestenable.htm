gpu_get_ztestenable

Cette fonction peut être utilisée pour récupérer si le z-testing est activé (la fonction renvoie true) ou non (la fonction renvoie false). La valeur par défaut est que le z-testing est désactivé, donc la fonction renvoie false.

Syntaxe :

gpu_get_ztestenable() ;

Les retours :

Booléen

Exemple :

if gpu_get_ztestenable() == false
{
gpu_set_ztestenable(true) ;
}

Le code ci-dessus vérifie si le test Z est activé ou non et s'il est désactivé, il est alors réactivé.
