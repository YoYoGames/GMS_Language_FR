Contrôle GPU

Lorsque vous dessinez des objets dans une pièce, vous aurez souvent besoin d'un contrôle plus granulaire sur la manière exacte dont le dessin sera fait. Vous souhaiterez peut-être qu'il n'y ait pas de fusion alpha, ou vous pourriez avoir besoin d'une élimination, ou vous devrez peut-être activer ou désactiver l'interpolation de texture. Tout cela est possible - et bien plus encore! - en utilisant les fonctions du contrôleur GPU répertoriées ci-dessous. Notez que la plupart de ces fonctions ont une portée globale et qu'une fois définies ou modifiées, elles affecteront la façon dont tout est dessiné à partir de là.

Les fonctions suivantes peuvent être utilisées pour obtenir les valeurs de différentes propriétés de GPU:

  • gpu_get_blendenable
  • gpu_get_ztestenable
  • gpu_get_zfunc
  • gpu_get_zwriteenable
  • gpu_get_fog
  • gpu_get_cullmode
  • gpu_get_blendmode
  • gpu_get_blendmode_ext
  • gpu_get_blendmode_ext_sepalpha
  • gpu_get_blendmode_src
  • gpu_get_blendmode_dest
  • gpu_get_blendmode_srcalpha
  • gpu_get_blendmode_destalpha
  • gpu_get_colourwriteenable
  • gpu_get_alphatestenable
  • gpu_get_alphatestref
  • gpu_get_texfilter
  • gpu_get_texfilter_ext
  • gpu_get_texrepeat
  • gpu_get_texrepeat_ext

Les fonctions suivantes peuvent être utilisées pour définir les valeurs de différentes propriétés GPU:

  • gpu_set_blendenable
  • gpu_set_ztestenable
  • gpu_set_zfunc
  • gpu_set_zwriteenable
  • gpu_set_fog
  • gpu_set_cullmode
  • gpu_set_blendmode
  • gpu_set_blendmode_ext
  • gpu_set_blendmode_ext_sepalpha
  • gpu_set_colourwriteenable
  • gpu_set_alphatestenable
  • gpu_set_alphatestref
  • gpu_set_texfilter
  • gpu_set_texfilter_ext
  • gpu_set_texrepeat
  • gpu_set_texrepeat_ext

Veuillez noter qu'avant d'utiliser les fonctions du mode de fusion, il est important de comprendre leur fonctionnement afin d'obtenir les bons résultats lors de leur application. Pour cela, nous avons créé le guide suivant:

  • Guide d'utilisation des modes de fusion

Enfin, il existe les fonctions suivantes qui peuvent être utilisées pour manipuler la pile GPU:

  • gpu_push_state
  • gpu_pop_state
  • gpu_get_state
  • gpu_set_state
