gpu_set_texrepeat

Cette fonction permet d'indiquer à GameMaker Studio 2 si les textures doivent se répéter(true) ou non(false) lorsqu'elles sont utilisées pour des modèles et des primitives. Normalement, la largeur et la hauteur des textures se situent dans la plage 0 - 1, mais si vous spécifiez une valeur supérieure à 1, par défaut, le reste n'est pas dessiné (la texture entière sera dessinée avec une section "vide" correspondant au pourcentage supérieur à 1 qui est utilisé. Ainsi, une valeur de 2 pour la largeur ou la hauteur d'une texture dessinerait la texture dans la moitié de l'espace et laisserait l'autre moitié vide. la moitié de l'espace et laisserait l'autre moitié vide). Cependant, en utilisant cette fonction pour définir repeat sur true, la texture est répétée automatiquement sur tous les modèles et primitives. Le cas le plus probable d'utilisation de ces fonctions est la répétition d'une texture en 3D mais pour que cela fonctionne et ne tire pas les images du reste de la page de texture, le sprite utilisé devra être marqué comme étant sur une "page de texture séparée" dans l'éditeur de sprites.

NOTE: Les sprites et les arrière-plans sont toujours dessinés sans répétition, donc une fois que vous dessinez un sprite ou un arrière-plan, cette valeur est réinitialisée à false. Ce comportement par défaut n'affectera pas les paramètres de la scène de texture (c'est-à-dire que l'appel de la fonction mais pas le réglage interne à false).

REMARQUE: Ce paramètre remplacera les répétitions d'étage de texture définies pour les shaders lors de l'utilisation de la fonction gpu_set_texrepeat_ext().

Syntaxe :

gpu_set_texrepeat(enable) ;

Arguments Description
activer Activer ou désactiver le filtrage des textures(true / false)

Les retours :

N/A

Exemple :

if gpu_get_texrepeat()
{
gpu_set_texrepeat(false) ;
}
else
{
gpu_set_texrepeat(true) ;
}

Le code ci-dessus vérifie si la répétition de la texture est activée ou non et l'active en conséquence.
