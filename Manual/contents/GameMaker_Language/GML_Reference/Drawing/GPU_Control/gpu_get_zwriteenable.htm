gpu_get_zwriteenable

Cette fonction peut être utilisée pour savoir si l'écriture z est activée (la fonction renvoie true ) ou non (la fonction renvoie false ). La valeur par défaut est que l'écriture z est activée, donc la fonction retournera true.

Syntaxe:

gpu_get_zwriteenable();

Retour:

Boolean

Exemple:

if gpu_get_zwriteenable() == false
    {
    gpu_set_zwriteenable(true);
    }

Le code ci-dessus vérifie si l'écriture z est activée ou non et si elle est désactivée, elle est ensuite à nouveau activée.

Retour: Contrôle GPU
