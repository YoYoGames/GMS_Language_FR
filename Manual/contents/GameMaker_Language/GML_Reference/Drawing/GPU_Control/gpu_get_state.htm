gpu_get_state

Cette fonction obtiendra l'état actuel du GPU, le retournant sous forme de carte DS. Cela peut ensuite être manipulé ou même enregistré, et vous pouvez renvoyer cette carte au GPU en utilisant la fonction gpu_set_state().

Syntaxe:

gpu_get_state();

Retour:

Integer (DS map ID)

Exemple:

gpu_map = gpu_get_state();

Le code ci-dessus stocke l'état actuel du GPU dans une variable.
