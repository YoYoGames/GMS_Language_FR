gpu_set_blendmode

Lorsque GameMaker Studio 2 dessine un pixel, il y a une couleur source (la couleur du pixel que nous allons dessiner) et une couleur de destination (la couleur qui se trouve déjà dans le pixel que nous dessinons). le pixel, les couleurs source et destination sont calculées en fonction du mode de fusion choisi. Chaque composante des couleurs est stockée sous la forme d'une valeur à virgule flottante comprise entre 0 et 1, et la nouvelle couleur est calculée en multipliant chaque composante de la couleur source par un facteur et en multipliant chaque composante de la couleur de destination par un autre facteur. source par un certain facteur et en multipliant chaque composante de la couleur de destination par un autre facteur, puis en additionnant les résultats composante par composante.

Cette fonction vous permet de définir le mode de fusion de quatre façons de base en utilisant les constantes suivantes :

Constant Description Mode de mélange étendu
bm_normal Mélange normal (le mode de mélange par défaut). (bm_src_alpha, bm_inv_src_alpha)
bm_add Mélange additif. Les valeurs de luminosité des zones claires sont ajoutées. (bm_src_alpha, bm_one)
bm_subtract Mélange soustractif. Les valeurs de luminosité des zones claires sont soustraites. (bm_zero, bm_inv_src_colour)
bmax Mélange maximal. Similaire au mélange additif. (bm_src_alpha, bm_inv_src_colour)

Comme vous pouvez le voir dans le tableau ci-dessus, ces quatre modes de fusion sont en fait des composites de modes de fusion étendus que l'on peut trouver sur la page décrivant gpu_set_blendmode_ext().

Pour vous aider à tirer le meilleur parti des modes de fusion et à comprendre comment ils fonctionnent et comment ils affectent l'image finale dessinée à l'écran, nous vous recommandons de lire le guide suivant :

  • Guide d'utilisation des Blendmodes

Syntaxe :

gpu_set_blendmode(mode) ;

Arguments Description
mode Le mode de fusion à utiliser (voir le tableau ci-dessus)

Les retours :

N/A

Exemple :

gpu_set_blendmode(bm_add) ;
draw_circle_colour(100, 100, 50, c_white, c_black, 0) ;
gpu_set_blendmode(bm_normal) ;

Cela transformera le noir en transparence, créant un effet de "lueur" dû au fait que le blanc est fort à l'extérieur et s'affaiblit progressivement à partir du centre du cercle. Le mode de fusion est réinitialisé après le tracé du cercle afin de s'assurer que la fusion additive n'est pas appliquée à tout ce qui suit. ne soit pas également appliqué à tout ce qui suit.
