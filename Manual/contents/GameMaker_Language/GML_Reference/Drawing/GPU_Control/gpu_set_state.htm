gpu_set_state

Cette fonction définira l'état actuel du GPU à l'aide de la DS Map transmise. La carte fournie peut être créée à l'aide de la fonction gpu_get_state().

Syntaxe:

gpu_set_state(ds_map);

Argument Description
ds_map L'état du GPU à définir comme DS Map.

Retour:

N/A

Exemple:

gpu_set_state(gpu_map);

Le code ci-dessus définit l'état du GPU en utilisant la carte fournie dans la variable "gpu_map".

Retour: Contrôle GPU
