uwp_license_trial_user

Cette fonction renverra le pointeur de l'ID utilisateur dont la licence d'essai est actuellement utilisée, ou elle retournera pointer_null si l'utilisateur n'est pas actuellement actif ou si le jeu n'est pas actuellement en mode d'essai.

Syntaxe:

uwp_license_trial_user();

Retour:

Pointer

Exemple:

if gamepad_button_check_pressed(0, gp_start)
    {
    if uwp_license_trial_version()
        {
        var _uid = uwp_license_trial_user();
        uwp_show_help(_uid);
        }
    }

Le code ci-dessus vérifie si un bouton est enfoncé, puis vérifie l'application pour voir si elle fonctionne avec une licence d'essai. Si tel est le cas, il obtient l'ID utilisateur de la licence et ouvre le fichier d'aide de cet utilisateur.
