xboxlive_matchmaking_join_invite

Cette fonction peut être appelée pour traiter une invitation à rejoindre une fois que vous avez appelé avec succès xboxlive_matchmaking_start(). Vous fournissez l'ID utilisateur local et les valeurs d'invitation (ID et ID d'hôte) qui ont été renvoyées lors de la réception de l'invitation, ainsi que le nom du modèle de session que vous avez créé (une chaîne). Il rejoindra ensuite l'utilisateur à la session à laquelle il a été invité et vous recevrez des événements sociaux asynchrones comme si vous aviez appelé xboxlive_matchmaking_find().

Syntaxe:

xboxlive_matchmaking_join_invite(user_who_received_invite, invitation_id, invitation_host, sda_template_name);

Argument Description
user_who_received_invite Le pointeur d'ID utilisateur local.
invitation_id L'ID d'invitation.
invitation_host L'ID d'hôte de l'invitation.
sda_template_name Le nom du modèle de session.

Retour:

N/A

Exemple:

var userID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_join_invite(userID, global.InviteID, global.InviteHost, global.SessionTemplate);

Ce qui précède obtient l'ID utilisateur pour la manette de jeu donnée, puis tente de rejoindre la session à laquelle ils ont été invités.
