xboxlive_matchmaking_stop

Cette fonction peut être utilisée pour mettre fin à une session de jumelage pour l'utilisateur donné.

Syntaxe :

xboxlive_matchmaking_stop(user_id) ;

Arguments Description
nom d'utilisateur Le pointeur d'identification de l'utilisateur à utiliser

Les retours :

N/A

Exemple :

if !xboxlive_user_is_signed_in(user_id[0])
{
xboxlive_matchmaking_stop(user_id[0]) ;
}

Le code ci-dessus mettra fin à la session de jumelage pour l'utilisateur donné.
