xboxlive_stats_set_stat_string

Cette fonction peut être utilisée pour définir une statistique sur une chaîne pour l'ID utilisateur donné. Vous fournissez l'ID utilisateur tel que renvoyé par (par exemple) la fonction xboxlive_get_user(), puis la chaîne de statistiques à définir ( si la chaîne de statistiques n'existe pas déjà, une nouvelle statistique sera créée et définie sur la valeur donnée ), puis la chaîne sur laquelle définir la statistique. Notez que le nom de la statistique doit être alphanumérique uniquement, sans symboles ni espaces, mais la valeur de chaîne que vous définissez peut contenir des espaces et des caractères spéciaux.

Lors de la définition de la chaîne de statistiques, toute chaîne précédente sera remplacée, il vous appartient donc de déterminer si la chaîne de statistiques doit être mise à jour ou non en la comparant à la chaîne de statistiques actuelle avec la nouvelle avant de la définir.

La fonction renverra -1 s'il y a eu une erreur ou si l'ID utilisateur n'est pas valide, ou toute autre valeur si la fonction a été appelée avec succès.

Syntaxe:

xboxlive_stats_set_stat_string(user_id, stat, value);

Argument Description
user_id Le pointeur d'ID utilisateur pour définir la statistique pour
stat La statistique à définir (une chaîne)
value La valeur à laquelle définir la statistique (une chaîne)

Retour:

Real

Exemple:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

Le code ci-dessus définit la statistique "LevelName" sur la chaîne stockée dans la variable globale donnée.
