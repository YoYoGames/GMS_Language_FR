xboxlive_stats_set_stat_real

Cette fonction peut être utilisée pour définir la valeur d'une statistique pour l'ID utilisateur donné. Vous fournissez l'ID utilisateur tel que retourné par (par exemple) la fonction xboxlive_get_user()puis la chaîne de statuts à définir(si la chaîne de statuts n'existe pas encore, un nouveau statut sera créé et défini à la valeur donnée) et une valeur (un réel) à définir pour le statut. Notez que le nom de la statue doit être uniquement alphanumérique, sans symboles ni espaces.

Lors de la définition de la valeur de la stat, toute valeur précédente sera remplacée, c'est donc à vous de déterminer si la valeur de la stat doit être mise à jour ou non (par exemple, vérifier que le meilleur score est bien le plus élevé) en comparant la valeur de la stat actuelle avec la nouvelle valeur avant de la définir. la nouvelle valeur avant de la définir.

La fonction renverra -1 s'il y a eu une erreur ou si l'ID utilisateur est invalide, ou toute autre valeur si la fonction a été appelée avec succès.

Syntaxe :

xboxlive_stats_set_stat_real(user_id, stat, value) ;

Arguments Description
nom d'utilisateur Le pointeur d'identification de l'utilisateur pour définir le statut de l'utilisateur.
stat La statistique à définir (une chaîne de caractères)
valeur La valeur à donner à la stat (un réel)

Les retours :

Real

Exemple :

if (game_over == true)
{
if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
{
var _val = (global.LevelsFinished / global.LevelsTotal) * 100 ;
xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val) ;
}
}

Le code ci-dessus vérifie une variable et si elle est vraie , il vérifie la valeur de la statistique "PercentDone". Si cette valeur est inférieure à 100, une valeur est alors générée et la stat est fixée à cette valeur.
