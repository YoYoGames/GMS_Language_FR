xboxlive_achievements_set_progress

Cette fonction peut être utilisée pour mettre à jour la progression d'une réalisation. Vous fournissez l'ID utilisateur tel qu'il est retourné, par exemple, par la fonction xboxlive_get_user()et ensuite la chaîne de réalisation (il s'agit de l'identifiant du succès tel qu'il a été attribué dans le centre de développement XDP/Windows lors de sa création). Enfin, vous définissez la valeur de progression sur une valeur comprise entre 0 et 100. Notez que le système de réalisation refusera les mises à jour qui sont inférieures à la valeur de progression actuelle. actuelle.

Syntaxe :

xboxlive_achievements_set_progress(user_id, achievement, progress) ;

Arguments Description
nom d'utilisateur L'ID de l'utilisateur pour lequel il faut définir la réalisation.
réalisation La réalisation à définir (une chaîne de caractères)
progrès La valeur de progression à définir (de 0 à 100)

Les retours :

N/A

Exemple étendu :

Voici un exemple détaillé de la façon dont cette fonction peut être utilisée. Pour commencer, vous l'appelez dans un événement comme Room Start ou Create :

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false) ;

Le code ci-dessus sera appelé pour obtenir une liste de toutes les positions du classement mondial pour le jeu, et générera un appel à l'événement social asynchrone que nous traiterons de la manière suivante :

var _progress = (global.Level / global.LevelMax) * 100 ;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress) ;

Le code ci-dessus crée une valeur en pourcentage basée sur certaines variables globales et l'utilise ensuite pour définir la progression de l'accomplissement "Game_Completed".
