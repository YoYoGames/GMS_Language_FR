xboxlive_achievements_set_progress

Cette fonction peut être utilisée pour mettre à jour la progression d'une réalisation. Vous fournissez l'ID utilisateur tel que renvoyé, par exemple, par la fonction xboxlive_get_user(), puis la chaîne de réussite (il s'agit de l'ID de réussite tel qu'attribué dans le Windows lors de sa création). Enfin, vous définissez la valeur de progression sur une valeur comprise entre 0 et 100. Notez que le système de réussite refusera les mises à jour inférieures à la valeur de progression actuelle.

Syntaxe:

xboxlive_achievements_set_progress(user_id, achievement, progress);

Argument Description
user_id L'ID utilisateur de l'utilisateur pour lequel définir la réussite
achievement La réalisation à définir (une chaîne)
progress La valeur de progression à définir (de 0 à 100)

Retour:

N/A

Exemple étendu:

Voici un exemple détaillé de la façon dont cette fonction peut être utilisée. Pour commencer, vous l'appelez dans un événement comme Room Start ou Create:

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

Le code ci-dessus serait appelé pour obtenir une liste de toutes les positions de classement mondial pour le jeu, et générera un rappel d'événement social asynchrone que nous traiterions de la manière suivante:

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

Le code ci-dessus crée une valeur de pourcentage basée sur certaines variables globales, puis l'utilise pour définir la progression de la réussite pour la réalisation "Game_Completed".
