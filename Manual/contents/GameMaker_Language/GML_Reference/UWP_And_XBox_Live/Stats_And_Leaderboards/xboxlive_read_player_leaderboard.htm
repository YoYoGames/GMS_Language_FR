xboxlive_read_player_leaderboard

La fonction renverra des informations de classement pour le nom d'utilisateur donné. Le classement doit avoir été créé au préalable sur le tableau de bord XDP de votre jeu, et l'appel de la fonction déclenchera un événement social asynchrone de rappel qui contient le async_load carte remplie avec les paires clé / valeur pertinentes. La touche "id" de cette carte DS est utilisée pour identifier le rappel correct (il peut y avoir plus d'une fonction de déclenchement pour un événement asynchrone donné), et le service informatique sera associé à la constante achievement_leaderboard_info ainsi qu'un certain nombre d'autres paires clé / valeur. Le contenu exact de la carte est indiqué ci-dessous:

  • " id " - Pour cette fonction, il devrait être achievement_leaderboard_info
  • " leaderboardid " - ID unique du classement tel que défini sur le tableau de bord du fournisseur.
  • " numentries " - Le nombre d'entrées dans le classement que vous avez reçues.
  • " PlayerN " - Le nom du joueur, où "N" est une valeur entière correspondant à sa position dans la liste des entrées du classement.
  • " PlayeridN " - L'identifiant unique du joueur, "N".
  • " RankN " - Le rang du joueur "N" dans le classement.
  • " ScoreN " - Le score du joueur "N".


La fonction nécessite que vous fournissiez l'une des constantes suivantes pour définir les propriétés du filtre:

Constant Description
achievement_filter_all_players Obtenez tous les scores.
achievement_filter_friends_only Obtenez uniquement les scores de vos amis, par ordre croissant.
achievement_filter_favorites_only Obtenez uniquement les scores favoris, par ordre croissant
achievement_filter_friends_alt Obtenez uniquement les scores de vos amis dans l'ordre décroissant.
achievement_filter_favorites_alt Obtenez uniquement les scores des favoris par ordre décroissant.

Syntaxe:

xboxlive_read_player_leaderboard(leaderboard_name, user_name, num_items, friend_filter);

Argument Description
leaderboard_name Le nom du classement à lire, tel que configuré sur XDP.
user_name Le nom de l'utilisateur à partir duquel lire.
num_items Le nombre d'éléments à récupérer.
friend_filter L'une des constantes de filtre (voir la description ci-dessus).

Retour:

Real

Exemple:

var _user = xboxlive_user_for_pad(0);
var _uid = xboxlive_user_id_for_user(_user);
xboxlive_read_player_leaderboard("MyLeaderboard", _uid, 10, achievement_filter_all_players);

Le code ci-dessus obtient l'ID utilisateur, puis l'utilise pour demander toutes les données du joueur pour les 10 premières places du classement donné.
