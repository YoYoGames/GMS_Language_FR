xboxlive_stats_get_stat

Cette fonction peut être utilisée pour récupérer une seule valeur de stat dans le gestionnaire de stat pour un utilisateur donné. Vous fournissez l'ID de l'utilisateur tel que renvoyé par (par exemple) la fonction xboxlive_get_user(), puis la chaîne de caractères de la statistique telle qu'elle a été définie lorsque vous l'avez créée à l'aide de l'une des fonctions xboxlive_stats_set_stat_*. La valeur de retour peut être une chaîne de caractères ou un réel (en fonction de la statistique vérifiée) ou la constante GML undefined si la statistique n'existe pas ou s'il y a eu une erreur.

Syntaxe :

xboxlive_stats_get_stat(user_id, stat) ;

Arguments Description
nom d'utilisateur Le pointeur d'identification de l'utilisateur pour obtenir les noms de stat de
stat La statistique à définir (une chaîne de caractères)

Les retours :

Réel, chaîne de caractères ou indéfini

Exemple :

if (game_over == true)
{
if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
{
var _val = (global.LevelsFinished / global.LevelsTotal) * 100 ;
xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val) ;
}
}

Le code ci-dessus vérifie une variable et si elle est vraie , il vérifie la valeur de la statistique "PercentDone". Si cette valeur est inférieure à 100, une valeur est alors générée et la stat est fixée à cette valeur.
