xboxlive_stats_setup

Cette fonction doit être appelée avant de pouvoir utiliser l'une des autres fonctions statistiques Xbox Live et initialise simplement les bibliothèques requises sur le système. L'argument "user_id" est l'ID utilisateur brut renvoyé par la fonction xboxlive_get_user(), tandis que "service_config" et "title_id" sont l'identifiant unique de votre jeu sur le Xbox Live Dev Center.

Syntaxe:

xboxlive_stats_setup(user_id, service_config_id, title_id);

Argument Description
user_id Le pointeur d'ID de l'utilisateur à vérifier.
service_config_id Il s'agit de la chaîne config_id du jeu.
title_id L'ID unique de votre jeu sur le Xbox Dev Center.

Retour:

N/A

Exemple:

var user = xboxlive_get_user(0); xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

Le code ci-dessus initialise le système de statistiques pour l'ID utilisateur donné.
