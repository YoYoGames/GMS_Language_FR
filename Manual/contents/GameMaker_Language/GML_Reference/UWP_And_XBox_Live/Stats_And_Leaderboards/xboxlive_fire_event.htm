xboxlive_read_player_leaderboard

Cette fonction peut être utilisée pour déclencher un événement statistique. L'argument "event_name" est le nom de l'événement à déclencher tel que défini dans la console XDP de votre jeu, et les paramètres supplémentaires suivants dépendront également de ce que vous avez configuré pour la statistique. La fonction renverra 0 si l'événement a été envoyé (et devrait être reçu / traité par le serveur) ou -1 s'il y a eu une erreur (c'est-à-dire: votre événement n'a pas été configuré car le fichier manifeste d'événement inclus dans le projet dit un autre numéro).

Syntaxe:

xboxlive_fire_event(event_name, [stat_parameters]);

Argument Description
event_name Le nom de l'événement à déclencher.
stat_parameters [FACULTATIF] Paramètres supplémentaires à ajouter à l'événement.

Retour:

String

Exemple:

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42);

Le code ci-dessus obtient des informations sur l'utilisateur, puis déclenche un événement statistique avec ces données (et d'autres).
