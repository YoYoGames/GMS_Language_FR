uwp_is_constrained

Cette fonction renvoie true (vrai) si le jeu fonctionne dans un mode restreint en ressources ou false (faux) s'il fonctionne normalement. Le mode restreint en ressources peut être activé lorsque le jeu n'a pas le focus, et dans ce mode, le jeu a des ressources réduites pour le GPU. ressources du GPU. La réponse à cette situation devrait normalement être de mettre le jeu en pause (puisque l'utilisateur n'a plus le contrôle). De même, la désactivation de tout effet particulièrement éprouvant peut être une bonne idée si un ralentissement est observé dans ce mode. Notez que contrairement à la fonction uwp_is_suspending() cette fonction retournera vrai continuellement tant que le jeu est en mode contraint (et pas seulement une fois quand cela se produit pour la première fois).

Syntaxe :

uwp_is_constrained() ;

Les retours :

Booléen

Exemple :

if xboxliveis_constrained()
{
if !instance_exists(obj_PauseMenu)
{
instance_create(0,0,obj_PauseMenu) ;
}
}

Le code ci-dessus vérifie si l'application est contrainte et si elle l'est, il crée une instance d'un objet (utilisé pour mettre le jeu en pause).
