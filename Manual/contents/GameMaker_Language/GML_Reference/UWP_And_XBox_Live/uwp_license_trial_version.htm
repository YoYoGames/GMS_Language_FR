uwp_license_trial_version

Cette fonction peut être utilisée pour vérifier si le jeu est sous licence d'essai ou non. Si c'est le cas, la fonction retournera true, ou false autrement.

Syntaxe:

uwp_license_trial_version();

Retour:

Boolean

Exemple:

if uwp_license_trial_version()
    {
    global.LevelCap = 10;
    }

Le code ci-dessus vérifie si l'application est sous une licence d'essai et, si tel est le cas, il définit une variable globale.

Retour: UWP et XBox Live
