xboxlive_get_savedata_user

Cette fonction renvoie le pointeur d'ID utilisateur actuellement associé à l'enregistrement du fichier (ou la constante pointer_null si aucun ID utilisateur n'est actuellement utilisé). Voir xboxlive_set_savedata_user() pour plus de détails.

Syntaxe:

xboxlive_get_savedata_user();

Retour:

Pointer or pointer_null

Exemple:

if (xboxlive_get_savedata_user() != user_id[0])
    {
    xboxlive_set_savedata_user(user_id[0]);
    }

Le code ci-dessus vérifie si un utilisateur est actuellement affecté en tant que cible de sauvegarde, et si ce n'est pas le cas, il est affecté.
