xboxlive_sponsor_for_user

Cette fonction récupère l'ID du sponsor de l'utilisateur donné.

Syntaxe :

xboxlive_sponsor_for_user(user_id) ;

Arguments Description
nom d'utilisateur Le pointeur d'identification de l'utilisateur à vérifier.

Les retours :

Real

Exemple :

var _uid = xboxlive_user_for_pad(0) ;
sponsor = xboxlive_sponsor_for_user(_uid) ;

Le code ci-dessus récupère le pointeur de l'ID de l'utilisateur affecté à l'emplacement [0] de la manette de jeu, puis utilise cet ID pour récupérer l'ID du sponsor de l'utilisateur.
