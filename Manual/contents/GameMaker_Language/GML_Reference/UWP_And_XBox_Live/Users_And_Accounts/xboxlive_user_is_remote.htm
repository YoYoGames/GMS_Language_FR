xboxlive_user_is_remote

Cette fonction vérifiera l'ID utilisateur donné et retournera true si le lecteur est un lecteur distant, ou false autrement.

Syntaxe:

xboxlive_user_is_remote(user_id);

Argument Description
user_id Le pointeur d'ID de l'utilisateur à vérifier.

Retour:

Boolean

Exemple:

global.PlayerNum = 0;
global.Player_ID = array_create();
for(var i = 0; i < xboxlive_get_user_count(); ++i;)
    {
    var _uid = xboxlive_get_user(i);
    if xboxlive_user_is_active(_uid)
        {
        if !xboxlive_user_is_remote(_uid)
            {
            global.Remote_Player_ID[global.PlayerNum++] = _uid;
            }
        }
    }

Le code ci-dessus parcourt les comptes d'utilisateurs, puis vérifie si l'un d'entre eux est actif. S'ils le sont, leur ID utilisateur est ajouté dans un tableau global, uniquement s'ils ne sont pas des utilisateurs distants.

Retour: Utilisateurs et comptes
