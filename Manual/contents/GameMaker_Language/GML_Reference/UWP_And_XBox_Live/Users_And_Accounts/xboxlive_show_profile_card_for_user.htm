xboxlive_show_profile_card_for_user

Avec cette fonction, vous pouvez demander à la XBox d'afficher la carte de profil pour le pointeur de l'ID utilisateur cible. La fonction nécessite à la fois le pointeur d'ID utilisateur pour l'utilisateur qui demande les informations ainsi que le pointeur d'ID utilisateur de l'utilisateur à cibler et à obtenir la carte de profil.

Syntaxe:

xboxlive_show_profile_card_for_user(requesting_user_id, target_user_id);

Argument Description
requesting_user_id L'ID utilisateur (un pointeur) de l'utilisateur demandeur
target_user_id L'ID utilisateur (un pointeur) de l'utilisateur pour obtenir la carte de profil de

Retour:

N/A

Exemple:

if (gamepad(0, gp_start))
    {
    xboxlive_show_profile_card_for_user(user[0], user[1]);
    }

Le code ci-dessus vérifie si un bouton de la manette de jeu est pressé et s'il en détecte un, il affiche la carte de profil de l'utilisateur donné.
