xboxlive_user_id_for_user

Cette fonction convertit le pointeur de l'ID utilisateur en une chaîne.

Syntaxe:

xboxlive_user_id_for_user(user_id);

Argument Description
user_id Le pointeur d'ID de l'utilisateur à vérifier.

Retour:

String

Exemple:

var _uid = xboxlive_user_for_pad(0);
uid_string = xboxlive_user_id_for_user(_uid);

Le code ci-dessus obtient le pointeur d'ID utilisateur pour l'utilisateur affecté à l'emplacement de la manette de jeu [0], puis enregistre cet ID utilisateur sous forme de chaîne dans une variable d'instance.
