xboxlive_sprite_add_from_gamerpicture

Avec cette fonction, vous pouvez obtenir l'image du joueur pour un pointeur d'identification d'utilisateur donné. La fonction fonctionne de manière asynchrone et déclenchera un événement Image Loaded asynchrone pour vous informer de la fin de la fonction (comme avec les appels HTML5/URL). pour vous informer que la fonction est terminée (comme avec les appels HTML5/URL à sprite_add()). L'événement Asynchronous Image Loaded aura une carte DS stockée dans la variable intégrée async_load. Cette carte contient les informations suivantes :

  • "Nom du fichier": Le chemin d'accès complet au fichier que vous avez demandé.
  • " id " : L'ID de la ressource que vous avez chargée. Ce sera le même que celui de la variable à laquelle vous avez attribué la ressource.
  • "status " : Renvoie une valeur inférieure à 0 en cas d'erreur.

REMARQUE: Lorsque vous chargez dynamiquement un sprite dans votre jeu au moment de l'exécution, vous devez vous souvenir de le supprimer à nouveau à l'aide de la méthode sprite_delete() lorsqu'il n'est plus nécessaire, sinon il y a un risque de fuite de mémoire qui ralentira et finira par faire planter votre jeu.

Syntaxe :

xboxlive_sprite_add_from_gamerpicture(user_id, imagesize, xorig, yorig) ;

Arguments Description
nom d'utilisateur L'identifiant de l'utilisateur (un pointeur) pour obtenir l'image du joueur.
imagesize Taille en pixels du sprite à renvoyer
xorig Indique la position x de l'origine dans le sprite
yorig Indique la position y de l'origine dans le sprite

Les retours :

Real

Exemple :

var _a = 0 ;
var _num = xboxlive_get_user_count() ;
for (var i = 0 ; i < _num ; ++i ;)
{
var _uid = xboxlive_get_user(i) ;
if _uid != pointer_null
{
global.UserName[_a] = xboxlive_gamedisplayname_for_user(_uid) ;
global.UserScore[_a] = xboxlive_gamerscore_for_user(_uid) ;
global.UserRep[_a] = xboxlive_reputation_for_user(_uid) ;
global.UserAvatar[_a] = xboxlive_sprite_add_from_gamerpicture(_uid, 256, 0, 0) ;
++a ;
}
}

Le code ci-dessus passe en boucle les utilisateurs connectés et stocke leurs données de jeu dans divers tableaux globaux.
