xboxlive_get_activating_user

Avec cette fonction, vous pouvez récupérer le pointeur de l'ID utilisateur de l'utilisateur qui a lancé le jeu depuis la console.

REMARQUE: la façon dont vous utilisez ce pointeur dépendra de ce que vous voulez faire dans le jeu, mais vous ne devez pas vous fier à ce qu'il s'agisse de l'ID utilisateur en cours de lecture, car l'utilisateur activant peut se déconnecter pendant que le jeu est en cours d'exécution.

Syntaxe:

xboxlive_get_activating_user();

Retour:

Pointer

Exemple:

global.activUser = xboxlive_get_activating_user();

Ce qui précède stocke le pointeur d'ID utilisateur d'activation dans une variable globale
