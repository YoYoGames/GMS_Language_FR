xboxlive_gamedisplayname_for_user

Cette fonction permet de récupérer le nom d'affichage - sous forme de chaîne - à partir du pointeur d'identification de l'utilisateur. Notez que si l'utilisateur est local, la valeur retournée est simplement une chaîne de son nom d'affichage, mais s'il s'agit d'un utilisateur distant, la chaîne sera son gamertag. sera son gamertag, qui peut alors être utilisé, par exemple, pour activer/désactiver le mode silencieux dans le chat vocal.

Syntaxe :

xboxlive_gamedisplayname_for_user(user_id) ;

Arguments Description
nom d'utilisateur Le pointeur d'identification de l'utilisateur à vérifier.

Les retours :

Chaîne de caractères

Exemple :

for(var i = 0 ; i < xboxlive_get_user_count() ; ++i ;)
{
user_id[i] = xboxlive_get_user(i) ;
user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]) ;
}

Le code ci-dessus récupère le pointeur d'identification de l'utilisateur pour chaque compte d'utilisateur disponible, puis stocke le pointeur ainsi que le nom d'affichage dans deux tableaux.
