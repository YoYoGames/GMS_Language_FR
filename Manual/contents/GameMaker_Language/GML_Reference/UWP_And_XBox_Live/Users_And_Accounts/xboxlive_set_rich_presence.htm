xboxlive_set_rich_presence

Cette fonction définit la chaîne de présence riche pour l'utilisateur donné. Une chaîne de présence riche affiche l'activité d'un utilisateur dans le jeu après le nom du jeu auquel l'utilisateur joue, séparé par un trait d'union. Cette chaîne est affichée sous le gamertag du joueur Gamertag du joueur dans la liste "Amis et clubs" ainsi que dans le profil utilisateur Xbox Live du joueur.

Lorsque vous utilisez cette fonction, vous devez fournir le pointeur d'identification de l'utilisateur, puis vous pouvez indiquer si l'utilisateur est actuellement actif dans le jeu ou non (en utilisant true/false). Si cette fonction est définie sur false, la chaîne de présence riche sera sera complétée par "/afk" ou quelque chose d'approprié. L'argument suivant est l'ID de la chaîne de présence riche à afficher, et enfin vous pouvez (facultativement) fournir une chaîne service_config_id. Notez qu'il s'agit d'un argument optionnel car si vous avez appelé xboxlive_stats_setup() vous n'avez pas besoin de passer le service_config_id ici.

Pour plus d'informations sur la présence riche et sur la manière de configurer les chaînes à utiliser dans le centre de partenaires, veuillez consulter la documentation Microsoft suivante :

  • Aperçu des cordes de présence

Syntaxe :

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id])

Arguments Description
nom d'utilisateur Le pointeur d'identification de l'utilisateur à vérifier.
est_utilisateur_actif Marquer l'utilisateur comme actif ou non.
chaîne_de_présence_riche L'ID de la chaîne de présence riche à utiliser (tel que défini dans le Centre des partenaires - 50 caractères maximum).
service_config_id OPTIONNEL] Il s'agit de la chaîne config_id du jeu.

Les retours :

N/A

Exemple :

var pad_uid = xboxlive_user_for_pad(0) ;
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge") ;

Le code ci-dessus récupère le pointeur d'identification de l'utilisateur affecté à l'emplacement [0] du gamepad, puis définit la chaîne de présence riche pour cet utilisateur.
