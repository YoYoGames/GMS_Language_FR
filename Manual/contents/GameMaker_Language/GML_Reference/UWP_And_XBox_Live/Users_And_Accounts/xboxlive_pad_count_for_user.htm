xboxlive_pad_count_for_user

Cette fonction vous permet de savoir combien de "slots" de pad sont actifs pour l'utilisateur actuel (cf. xboxlive_pad_for_user() pour plus de détails).

Syntaxe :

xboxlive_pad_count_for_user(user_id, slot) ;

Arguments Description
nom d'utilisateur L'ID utilisateur (un pointeur) à vérifier

Les retours :

Entier

Exemple :

global.slots[0] = xboxlive_pad_count_for_user(user_id[0])

Le code ci-dessus stocke le nombre de tampons associés au pointeur d'identification de l'utilisateur donné.
