xboxlive_user_for_pad

Cette fonction renverra le pointeur de l'ID utilisateur associé à la valeur d'index de la manette de jeu donnée, ou pointer_null si aucun utilisateur n'y est associé. Notez que cette fonction ne doit être utilisée qu'avec les index de gamepad, et non avec xboxlive_get_user_count().

Syntaxe:

xboxlive_user_for_pad(pad_index);

Argument Description
pad_index L'index (un entier) de l'emplacement de la manette de jeu à vérifier

Retour:

Pointer

Exemple:

for(var i = 0; i < 11; ++i;)
    {
    var u_id = xboxlive_user_for_pad(i);
    if u_id != pointer_null
        {
        pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);
        }
    else
        {
        pad_name[i] = "Press Play";
        }
    }

Le code ci-dessus parcourt tous les emplacements de pad disponibles et vérifie si un pointeur d'ID utilisateur lui est associé. Si tel est le cas, un tableau est défini avec le nom d'affichage de l'utilisateur, sinon le tableau est défini sur un texte par défaut.
