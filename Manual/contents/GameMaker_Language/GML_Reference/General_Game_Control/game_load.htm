game_load

Avec cette fonction, vous pouvez charger un jeu qui a été sauvegardé précédemment (en utilisant game_save() ).

Syntaxe:

game_load(filename);

Argument Description
filename Le nom du fichier à charger.

Retour:

N/A

Exemple:

if keyboard_check_pressed(ord("L"))
    {
    if global.Save game_load("Save.dat");
    }

Le code ci-dessus chargera une partie précédemment sauvegardée si la variable globale est true lorsque le joueur appuie sur la touche "L".

Retour: Contrôle général du jeu
