identifiant du jeu

Cette variable en lecture seule renvoie l'identifiant unique du jeu que vous avez créé. Vous pouvez l'utiliser si vous avez besoin d'un nom de fichier unique ou de tout autre élément permettant d'identifier uniquement votre jeu. Cette variable peut être définie dans les options du jeu.

Syntaxe :

identifiant du jeu

Les retours :

Real

Exemple :

ini_open("Score.ini") ;
ini_write_real("Scores", "0",score + game_id) ;
ini_close() ;

Le code ci-dessus effectue un cryptage très basique sur le score en y ajoutant le game_id avant de le sauvegarder. Lors de la relecture dans le jeu, il faut déduire l'identifiant de jeu pour obtenir à nouveau la valeur correcte.
