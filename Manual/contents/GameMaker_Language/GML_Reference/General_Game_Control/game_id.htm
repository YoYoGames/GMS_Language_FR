game_id

Cette variable en lecture seule renvoie l'identifiant unique du jeu que vous avez créé. Vous pouvez l'utiliser si vous avez besoin d'un nom de fichier unique ou de tout autre élément nécessitant quelque chose pour identifier uniquement votre jeu. Cela peut être défini dans les options de jeu.

Syntaxe:

game_id

Retour:

Real

Exemple:

ini_open("Score.ini");
ini_write_real("Scores","0",score + game_id);
ini_close();

Le code ci-dessus effectue un cryptage très basique sur le score en y ajoutant le game_id avant de l'enregistrer. En le relisant dans le jeu, vous déduiriez le game_id pour obtenir à nouveau la valeur correcte.
