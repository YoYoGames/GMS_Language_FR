game_project_name

Cette variable en lecture seule renvoie le nom d'affichage de votre jeu pour la plate-forme cible dans un format "sauvegarde facile" pour la plate-forme cible. Si le nom d'affichage contient des caractères non autorisés pour un nom de fichier, ils seront automatiquement remplacés par "_". Le nom d'affichage peut être défini dans les options de jeu. Notez que puisqu'il n'y a pas de restrictions sur les noms de fichiers pour HTML5, cette chaîne sera probablement la même que celle renvoyée par game_display_name.

Syntaxe:

game_project_name

Retour:

String

Exemple:

var file = game_project_name + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();

Le code ci-dessus obtient le nom du projet et l'utilise pour ouvrir (ou créer) un fichier ini qui est ensuite écrit.

Retour: Contrôle général du jeu
