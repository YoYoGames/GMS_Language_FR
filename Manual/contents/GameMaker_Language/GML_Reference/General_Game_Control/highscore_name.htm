highscore_name

Avec cette fonction, vous pouvez récupérer la chaîne de nom qui a été stockée dans la liste des meilleurs scores à la position donnée. Si aucun nom n'a été saisi, la chaîne "Inconnu" sera renvoyée.

Syntaxe:

highscore_name(place);

Argument Description
place La place sur la table (1-10).

Retour:

String

Exemple:

var i = 9;
repeat(10)
    {
    name[i] = highscore_name(i + 1);
    i -= 1;
    }

Le code ci-dessus parcourra la liste des meilleurs scores et stockera tous les noms dans un tableau.

Retour: Contrôle général du jeu
