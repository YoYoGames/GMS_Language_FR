valeur_highscore

Cette fonction vous permet de récupérer la valeur du score qui a été enregistrée dans la liste des meilleurs scores à la position donnée. Si aucun score n'a été saisi, la fonction renvoie 0.

Syntaxe :

highscore_value(place) ;

Arguments Description
place La place sur la table (1-10).

Les retours :

N/A

Exemple :

var i = 9 ;
repeat(10)
{
scr[i] = highscore_value(i + 1) ;
i -= 1 ;
}

Le code ci-dessus va boucler la liste des meilleurs scores et stocker tous les scores dans un tableau.
