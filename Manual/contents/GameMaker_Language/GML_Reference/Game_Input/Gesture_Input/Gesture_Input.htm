Les fonctions énumérées ici sont toutes conçues pour vous permettre de mieux contrôler comment et quand ces événements seront déclenchés.

IMPORTANT ! Ces fonctions ne fonctionneront pas sur la plate-forme cible HTML5. Elles ne donneront pas d'erreur si elles sont utilisées, mais seront simplement ignorées.
