Entrée gestuelle

Les fonctions d'entrée de périphérique sont toutes parfaites pour gérer de simples touches à l'écran ou pour obtenir l'orientation de l'appareil, mais parfois vous voudrez que vos projets détectent des choses plus complexes, comme pouvoir faire glisser ou «faire glisser» une instance dans une pièce.. Ces choses peuvent être faites avec les fonctions de l'appareil, mais pour rendre les choses plus faciles, GameMaker Studio 2 dispose d'une gamme complète d' événements de geste pour détecter les doubles tapotements, les glissements, les flicks, les pincements et les rotations, à la fois au niveau d'une instance et au niveau global, ainsi retourner des informations sur l'événement. Les fonctions répertoriées ici sont toutes conçues pour vous permettre de mieux contrôler comment et quand ces événements seront déclenchés.

IMPORTANT! Ces fonctions ne fonctionneront pas sur la plate-forme cible HTML5. Ils ne donneront pas d'erreur s'ils sont utilisés, mais seront simplement ignorés.

Les différentes fonctions gestuelles suivantes sont disponibles:

  • gesture_get_drag_time
  • gesture_get_drag_distance
  • gesture_get_flick_speed
  • gesture_get_double_tap_time
  • gesture_get_double_tap_distance
  • gesture_get_tap_count
  • gesture_get_pinch_distance
  • gesture_get_pinch_angle_towards
  • gesture_get_pinch_angle_away
  • gesture_get_rotate_time
  • gesture_get_rotate_angle
  • gesture_drag_time
  • gesture_drag_distance
  • gesture_flick_speed
  • gesture_double_tap_time
  • gesture_double_tap_distance
  • gesture_tap_count
  • gesture_pinch_distance
  • gesture_pinch_angle_towards
  • gesture_pinch_angle_away
  • gesture_rotate_time
  • gesture_rotate_angle
