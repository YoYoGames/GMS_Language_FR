gesture_pinch_angle_away

Cette fonction est utilisée pour définir l'angle dans lequel un toucher doit s'éloigner d'un autre toucher avant qu'un geste de pincement puisse être déclenché. L'angle est mesuré en degrés et a une valeur par défaut de 45 °.

Syntaxe:

gesture_pinch_angle_away();

Retour:

Real (inches)

Exemple:

if gesture_get_pinch_angle_away() != 45
    {
    gesture_pinch_angle_away(45);
    }

Le code ci-dessus vérifie si l'angle de pincement pour les gestes est défini sur 45 ° et s'il ne l'est pas, il le définit sur cette valeur
