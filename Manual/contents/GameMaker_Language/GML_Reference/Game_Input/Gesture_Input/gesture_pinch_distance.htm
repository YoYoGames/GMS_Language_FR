gesture_pinch_distance

Cette fonction est utilisée pour définir la distance à l'intérieur de laquelle vous devez toucher / cliquer sur l'écran et vous déplacer avec deux doigts avant de déclencher un geste de pincement. La distance est mesurée en pouces et a une valeur par défaut de 0,1.

Syntaxe:

gesture_pinch_distance();

Retour:

Real (inches)

Exemple:

if gesture_get_pinch_distance() != 0.1
    {
    gesture_pinch_distance(0.1);
    }

Le code ci-dessus vérifie si la distance de pincement pour les gestes est définie sur 0,1 pouce et si ce n'est pas le cas, il la définit sur cette valeur.
