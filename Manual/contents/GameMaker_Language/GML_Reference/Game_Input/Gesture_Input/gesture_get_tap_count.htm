gesture_get_tap_count

Cette fonction permet de vérifier si le comptage des prises est activé ou désactivé. La fonction retournera true s'il est activé, et false sinon et vous pouvez activer ou désactiver le comptage des prises à l'aide de la fonction gesture_tap_count(). Lorsqu'il est activé, chaque événement de tap aura une entrée supplémentaire de carte DS "tapcount" qui aura enregistré le nombre de taps, et lorsqu'il sera activé, cela signifie que tous les événements taps seront déclenchés, c'est-à-dire: deux taps déclencheront à la fois l'événement tap unique et l'événement de double tap, avec le compte de tap d'événement de tap unique étant 1 et le nombre de tap d'événement de double tap étant 2. La valeur de comptage de tap sera remise à 0 après le temps défini pour une détection de double tap (voir la fonction gesture_double_tap_time() ). Si le comptage des taps est désactivé, alors le tap initial ne sera pas enregistré tant que le temps du double tap n'est pas écoulé et qu'aucun deuxième tap n'a été détecté. Notez que cela est activé par défaut.

Syntaxe:

gesture_get_tap_count();

Retour:

Bool

Exemple:

if !gesture_get_tap_count()
    {
    getsure_tap_count(true);
    }

Le code ci-dessus vérifie si le comptage des prises est activé et si ce n'est pas le cas, il est activé.
