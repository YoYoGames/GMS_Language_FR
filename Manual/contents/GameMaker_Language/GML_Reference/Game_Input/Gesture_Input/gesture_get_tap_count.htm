geste_get_tap_count

Cette fonction est utilisée pour vérifier si le comptage des appuis est activé ou désactivé. La fonction retournera true si elle est activée, et false sinon. Vous pouvez activer ou désactiver le comptage des tapotements en utilisant la fonction gesture_tap_count(). Lorsqu'il est activé, chaque événement de tapotement aura une entrée supplémentaire "tapcount" dans la carte DS qui aura enregistré le nombre de tapotements, et lorsqu'il est activé, cela signifie que tous les événements de tapotement seront déclenchés, c'est-à-dire que deux tapotements déclencheront à la fois l'événement de tapotement simple et l'événement de tapotement double. Deux tapotements déclenchent à la fois l'événement de tapotement simple et l'événement de tapotement double, le nombre de tapotements de l'événement de tapotement simple étant de 1 et celui de l'événement de tapotement double de 2. La valeur du compteur de coups de robinet sera remise à 0 après que le temps défini pour la détection d'un double coup de robinet se soit écoulé (voir la fonction gesture_double_tap_time()). Si le comptage des tapotements est désactivé, le premier tapotement ne sera pas enregistré tant que le temps de double tapotement ne sera pas écoulé et qu'aucun deuxième tapotement n'aura été détecté. Notez que cette fonction est activée par défaut.

Syntaxe :

gesture_get_tap_count() ;

Les retours :

Bool

Exemple :

if !gesture_get_tap_count()
{
getsure_tap_count(true) ;
}

Le code ci-dessus vérifie si le comptage par tapotement est activé et s'il ne l'est pas, il est activé.
