gesture_get_rotate_time

Cette fonction est utilisée pour obtenir l'angle qu'une paire de touches doit dépasser pour déclencher un geste de rotation de départ. L'angle est mesuré en degrés et a une valeur par défaut de 5 °.

Syntaxe:

gesture_get_rotate_angle();

Retour:

Real (inches)

Exemple:

if gesture_get_rotate_angle() != 5
    {
    gesture_rotate_angle(5);
    }

Le code ci-dessus vérifie si l'angle de rotation des gestes est défini sur 5 ° et s'il ne l'est pas, il le définit sur cette valeur.
