données de l'événement

Cette variable a une portée globale et est utilisée pour contenir une carte DS lorsqu'elle est utilisée dans les événements gestuels, et -1 à tous les autres moments. tous les autres moments. Le contenu réel de la carte DS dépendra du type d'événement gestuel qui l'a déclenché, alors reportez-vous aux sections individuelles pour ces événements.

Syntaxe :

données de l'événement ;

Les retours :

Réel (valeur entière)

Exemple :

isFlick = event_data[ ? "isflick"] ;
if (isFlick)
{
flickVelX = event_data[ ? "diffX"] ;
flickVelY = event_data[ ? "diffY"] ;
}
else
{
flickVelX = 0 ;
flickVelY = 0 ;
}

Le code ci-dessus est extrait de l'événement Drag End gesture et vérifie si l'événement est un événement "flick". Si c'est le cas, il extrait les données requises de la carte DS event_data et les utilise pour définir certaines variables. Si un événement " flick n'est pas détecté, les mêmes variables sont définies sur 0.
