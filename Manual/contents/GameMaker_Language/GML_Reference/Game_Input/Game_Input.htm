Entrée de jeu

La chose fondamentale qui fait d'un jeu un jeu est l'interaction avec le joueur. Ceci est réalisé par une variété de moyens, avec le clavier et la souris sur les ordinateurs et grâce à l'inclinaison de l'écran tactile et de l'appareil sur mobile, voire via une manette de jeu sur toutes les plates-formes!

Pour commencer dans GameMaker Studio 2 vous utiliseriez les événements dédiés de la souris ou du clavier sur les PC ou les événements de gestes sur mobile, mais parfois vous avez besoin de plus de contrôle que ces événements ne fournissent, ou vous souhaitez inclure des commandes de manette de jeu ou plusieurs options de contrôle différentes. Pour cela, vous pouvez utiliser les différentes fonctions d'entrée du jeu pour obtenir quels boutons ou touches ont été enfoncés ou relâchés ainsi que pour récupérer les touches sur un écran ou combien un joystick de manette de jeu a été déplacé.

Il existe un grand nombre de fonctions pour la saisie et le contrôle du jeu dans vos jeux, et elles ont donc été divisées en sections liées à des méthodes de saisie spécifiques pour vous permettre de trouver plus facilement ce dont vous avez besoin. Les sections disponibles sont:

  • La saisie au clavier
  • Entrée souris
  • Entrée de périphérique
  • Entrée gestuelle
  • Entrée GamePad
  • Clés virtuelles et claviers virtuels
