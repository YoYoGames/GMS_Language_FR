device_get_tilt_z

Cette fonction renvoie une valeur comprise entre -1 et 1 en fonction de l'angle "d'inclinaison" de l'appareil. La corrélation réelle entre les degrés d'inclinaison et la valeur renvoyée dépend de l'appareil et du système d'exploitation qu'il utilise, mais généralement une valeur de 1 ou -1 équivaut à +/- 90 °. L'image ci-dessous montre comment chacune des fonctions disponibles se rapporte à l'appareil:

Syntaxe:

device_get_tilt_z()

Retour:

Real

Exemple:

y += sign(device_get_tilt_z());

Le code ci-dessus utilise la valeur d'inclinaison pour déplacer le lecteur le long de l'axe y.
