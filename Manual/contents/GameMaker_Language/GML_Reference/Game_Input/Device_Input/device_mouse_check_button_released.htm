dispositif_mouse_check_button_released

Cette fonction renvoie vrai ou faux selon que la touche de périphérique que vous spécifiez a été libérée ou non. L'argument du périphérique fait référence au numéro de touche du périphérique, qui peut être compris entre 0 et n. Notez que le nombre maximum de touches pouvant être détectées dépendra beaucoup du périphérique utilisé et du système d'exploitation qu'il exécute. touches qui peuvent être détectées dépend beaucoup du dispositif utilisé et du système d'exploitation qu'il exécute (la plupart des dispositifs supportent au moins 4 touches simultanées). Cette fonction n'est déclenchée qu'une seule fois par l'action de relâchement. sur cette page peuvent être utilisées pour vérifier les boutons de la souris. Notez que mb_droite n'aura été détecté que si une double touche est détectée et maintenue sur la deuxième touche avant le relâchement (ce comportement peut être désactivé à l'aide de la fonction device_mouse_dbclick_enable()).

Syntaxe :

device_mouse_check_button_released(device, button) ;

Arguments Description
dispositif Le dispositif (de 0 à n) qui est contrôlé.
bouton Le bouton du dispositif en cours de vérification.

Les retours :

Booléen

Exemple :

if device_mouse_check_button_released(0, mb_left)
{
press=false ;
}

Le code ci-dessus vérifie si le dispositif a été pressé et définit une variable à false si c'est le cas.
