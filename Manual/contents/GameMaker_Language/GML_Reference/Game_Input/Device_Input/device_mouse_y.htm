périphérique_souris_y

Cette fonction renvoie la position y d'une touche sur le périphérique. Si vous exécutez cette fonction sur un module HTML5 ou PC et Mac, cette valeur est mise à jour en permanence, tant que le périphérique (généralement une souris) est branché, mais pour les périphériques mobiles, cette valeur ne sera mise à jour que lorsque l'écran sera touché. Cependant, pour les appareils mobiles, cette valeur ne sera mise à jour que lorsque l'écran est touché. Notez que le nombre maximum de touchers pouvant être détectés dépend beaucoup de l'appareil utilisé et du système d'exploitation qu'il exécute. Notez également que sur HTML5, seul "device 0" fonctionnera correctement. correctement et que les autres appareils peuvent donner des valeurs erronées.

Syntaxe :

device_mouse_y(device) ;

Arguments Description
dispositif Le dispositif (de 0 à n) qui est contrôlé.

Les retours :

Real

Exemple :

if device_mouse_check_button(0, mb_left)
{
if device_mouse_y(0) > y-32 && device_mouse_y(0) < y+32
{
press = true ;
}
else
{
pressed = false ;
}
}

Le code ci-dessus vérifie si le dispositif est pressé et si c'est le cas, il interroge la position y du dispositif pour voir si elle est dans les paramètres. Si c'est le cas, il définit la variable "pressed" comme vraie, sinon il la définit comme fausse.
