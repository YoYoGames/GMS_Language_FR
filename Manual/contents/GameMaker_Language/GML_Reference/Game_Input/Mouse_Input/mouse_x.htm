mouse_x

Cette variable en lecture seule renvoie la position actuelle de l'axe x de la souris dans la pièce.

Syntaxe:

mouse_x;

Retour:

Real

Exemple:

x = median(64, mouse_x, room_width - 64);

Le code ci-dessus maintiendra l'instance à la position x de la souris tant qu'elle se trouve dans les limites de 64 pixels de chaque côté de la pièce.

Retour: Entrée souris
