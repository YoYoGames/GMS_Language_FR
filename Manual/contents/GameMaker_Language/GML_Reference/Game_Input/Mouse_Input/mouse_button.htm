bouton de souris

Cette variable en lecture seule renvoie le bouton de la souris qui est actuellement pressé (actuellement, comme dans cette étape) et peut renvoyer toutes les constantes spéciales de la souris, sauf mb_any.

Syntaxe :

bouton de la souris ;

Les retours :

Constant

Exemple :

if mouse_button = mb_left
{
x -= 1 ;
}

Ceci déplace l'instance actuelle vers la gauche si le bouton gauche de la souris est enfoncé.
