gamepad_set_option

Cette fonction peut être utilisée pour définir l'une des options de manette de jeu disponibles. Vous fournissez le "slot" de la manette de jeu pour activer l'option, ainsi que la chaîne d'options à définir et la valeur à utiliser. La chaîne d'option disponible dépendra de la plate-forme sur laquelle le projet est exécuté, comme indiqué ci-dessous:

Option Plate-forme Description Évaluer
"allow_rotation" tvOS Définissez si la télécommande Siri sera verrouillée en rotation (c'est-à-dire qu'elle ne peut être utilisée que verticalement) ou non. Une fois déverrouillé (une valeur de true ) la rotation de la commande changera l'entrée, de telle sorte qu'une rotation de 90 ° vers la gauche fera revenir le côté "droit" du pavé tactile "vers le haut", etc. de sorte que toutes les entrées soient relatives à l'orientation du dispositif. Lorsqu'il est verrouillé (une valeur de false ), alors l'entrée ne sera pas modifiée par rapport à la position du contrôleur et sera toujours traitée comme si le contrôleur était en position verticale. Booléen
"dpad_absolute" tvOS Définissez si le pavé tactile Siri Remote accepte une entrée absolue ou une entrée relative. En mode absolu (une valeur de true ) le centre du pavé tactile est considéré comme la position (0,0) - c'est-à-dire: là où le stick analogique est au repos ou où se trouverait le centre du pavé tactile - et le mouvement autour de cette position générera une entrée. Lorsqu'il est défini sur relatif (une valeur de false alors la position (0,0) sera considérée comme étant l'endroit où le contact initial sur le pad est effectué et toutes les entrées seront générées à partir de cette position, pas du centre du pad. Booléen

Syntaxe:

gamepad_set_option(device, option_name, value);

Argument Description
device Quel «slot» de périphérique de manette de jeu définir.
option_name Le nom de l'option à définir (une chaîne, voir le tableau ci-dessus)
value La valeur à laquelle définir l'option (peut être booléen, réel, entier ou chaîne - voir le tableau ci-dessus)

Retour:

N/A

Exemple:

for(var i = 0; i < 12; ++i;)
    {
    if gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote"
        {
        if gamepad_get_option(i, "allow_rotation") == false
            {
            gamepad_set_option(i, "allow_rotation", true);
            }
        }
    }

Le code ci-dessus parcourt tous les emplacements de manette de jeu et vérifie la présence de «Siri Remote» sur la plate-forme tvOS Si un est détecté, il règle alors la télécommande pour permettre la rotation d'entrée.

Retour: Entrée de la manette de jeu
