gamepad_get_description

Cette fonction renverra une chaîne avec la description de la manette de jeu donnée, par exemple, un contrôleur PS3 peut renvoyer une chaîne similaire à "PLAYSTATION (R) 3 Controller", ou un contrôleur XBox360 peut renvoyer "Xbox 360 Controller (XInput STANDARD GAMEPAD) ". Cette chaîne dépend du matériel et la valeur renvoyée dépendra de la manette de jeu branchée dans le "slot" du périphérique qui est en cours de vérification.

Syntaxe:

gamepad_get_description(device);

Argument Description
device De quel "slot" de manette de jeu obtenir le nom.

Retour:

String

Exemple:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
   {
   if gamepad_is_connected(i)
      {
      draw_text(32, 32 + (i * 32), gamepad_get_description(i));
      }
   else
      {
      draw_text(32, 32 + (i * 32), "No Gamepad Connected");
      }
   }

Le code ci-dessus passera en boucle dans tous les emplacements de la manette de jeu pour vérifier les appareils connectés, puis dessinera du texte à l'écran en fonction du fait qu'une manette de jeu est connectée à la fente ou non.

Retour: Entrée de la manette de jeu
