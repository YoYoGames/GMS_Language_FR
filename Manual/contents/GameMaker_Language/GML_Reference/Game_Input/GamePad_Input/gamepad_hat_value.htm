gamepad_hat_value

Avec cette fonction, vous pouvez obtenir la valeur actuelle d'un "chapeau" de gamepad. Chaque valeur de chapeau est un masque de bit pour les différentes directions où :

  • Haut = 1
  • Droit = 2
  • Bas = 4
  • Gauche = 8

Notez que ces éléments peuvent être combinés (par exemple, fournir un indice de chapeau de 3 reviendrait à vérifier vers le haut et vers la droite) mais seuls 2 bits au maximum peuvent être pressés à la fois. La fonction renverra une valeur réelle entre 0 et 1, où 0 signifie que le bouton n'a pas été enfoncé et 1 qu'il l'a été complètement. (et il peut y avoir des valeurs intermédiaires selon que la manette de jeu supporte ou non l'entrée analogique pour les chapeaux).

Syntaxe :

gamepad_hat_value(device, hatindex) ;

Arguments Description
dispositif Quel "slot" de périphérique gamepad vérifier.
hatindex Quel(s) chapeau(x) de manette à vérifier.

Les retours :

Real

Exemple :

var _dir = gamepad_hat_value(global.PadIndex, 0) ;
switch (_dir)
{
cas 1 : y -= 3 ; break ;
cas 2 : x += 3 ; break ;
cas 3 : y -= 3 ; x += 3 ; break ;
cas 4 : x -= 3 ; break ;
cas 6 : y += 3 ; x += 3 ; break ;
cas 8 : y += 3 ; break ;
cas 9 : y -= 3 ; x -= 3 ; break ;
cas 12 : y += 3 ; x -= 3 ; break ;
}

Le code ci-dessus stocke l'état du chapeau "0" dans une variable locale, puis vérifie quelle est la valeur de retour et agit en conséquence.
