gamepad_is_supported

Avec cette fonction, vous pouvez savoir si la plate-forme cible prend en charge les manettes de jeu (renvoie true ) ou non (retourne false ).

Syntaxe:

gamepad_is_supported();

Retour:

Boolean

Exemple:

global.GP = gamepad_is_supported();

Le code ci-dessus vérifie si une manette de jeu est prise en charge et stocke la valeur de retour dans une variable globale pour de futures vérifications.

Retour: Entrée de la manette de jeu
