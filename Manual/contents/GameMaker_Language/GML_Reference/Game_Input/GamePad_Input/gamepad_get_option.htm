gamepad_get_option

Cette fonction peut être utilisée pour récupérer la valeur d'option donnée. Vous fournissez le "slot" de la manette de jeu à vérifier, ainsi que la chaîne d'options et la fonction renverra la valeur associée. La chaîne d'option disponible dépendra de la plate-forme sur laquelle le projet est exécuté, comme indiqué ci-dessous:

Option Plate-forme Description Retour
"allow_rotation" tvOS Renvoie si la télécommande Siri sera verrouillée en rotation (c'est-à-dire qu'elle ne peut être utilisée que verticalement) ou non. Lorsqu'elle est déverrouillée, la rotation de la commande changera l'entrée, de sorte qu'une rotation de 90 ° vers la gauche fera revenir le côté "droit" du pavé tactile "vers le haut", etc. de sorte que toutes les entrées soient relatives à l'orientation de l'appareil Booléen
"dpad_absolute" tvOS Renvoie si le pavé tactile Siri Remote accepte une entrée absolue ou une entrée relative. En mode absolu, le centre du pavé tactile est considéré comme la position (0,0) - c'est-à-dire: là où le stick analogique est au repos ou où se trouverait le centre du pavé tactile - et le mouvement autour de cette position générera une entrée. Lorsqu'elle est réglée sur relative, alors la position (0,0) sera considérée comme étant là où le contact initial sur le pad est effectué et toutes les entrées seront générées à partir de cette position, pas du centre du pad. Booléen

Syntaxe:

gamepad_get_option(device, option_name);

Argument Description
device Quel «slot» de périphérique de manette de jeu vérifier.
option_name Le nom de l'option à partir de laquelle obtenir la valeur (une chaîne, voir le tableau ci-dessus)

Retour:

Int, Bool, Real, String, N/A (depends on platform)

Exemple:

for(var i = 0; i < 12; ++i;)
    {
    if gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote"
        {
        if gamepad_get_option(i, "allow_rotation") == false
            {
            gamepad_set_option(i, "allow_rotation", true);
            }
        }
    }

Le code ci-dessus parcourt tous les emplacements de manette de jeu et vérifie la présence de «Siri Remote» sur la plate-forme tvOS Si un est détecté, il règle alors la télécommande pour permettre la rotation d'entrée.

Retour: Entrée de la manette de jeu
