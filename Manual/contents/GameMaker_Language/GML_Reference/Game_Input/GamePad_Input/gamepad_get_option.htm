gamepad_get_option

Cette fonction peut être utilisée pour récupérer la valeur d'une option donnée. Vous fournissez le "slot" du gamepad à vérifier, ainsi que la chaîne d'option et la fonction renverra la valeur associée. La chaîne d'options disponible dépendra de la plateforme sur laquelle sur laquelle le projet est exécuté, comme indiqué ci-dessous :

Option Plate-forme Description Renvoie à
"allow_rotation" tvOS Indique si la télécommande Siri est verrouillée en rotation (c'est-à-dire qu'elle ne peut être utilisée que verticalement) ou non. Lorsqu'elle est déverrouillée, la rotation de la commande modifiera l'entrée, de sorte qu'une rotation de 90º vers la gauche fera revenir le côté "droit" du pavé tactile vers le haut, etc. de sorte que toutes les entrées seront relatives à l'orientation de l'appareil. Booléen
"dpad_absolu" tvOS Renvoie si le pavé tactile de la télécommande Siri accepte une entrée absolue, ou une entrée relative. En mode absolu, le centre du pavé tactile est considéré comme la position (0,0) - c'est-à-dire où le stick analogique est au repos ou où le centre du dpad serait - et tout mouvement autour de cette position génère une entrée. Lorsqu'il est réglé sur relatif, la position (0,0) est considérée comme étant l'endroit où le toucher initial du pad est effectué et toutes les entrées seront générées à partir de cette position, et non du centre du pad. du pavé. Booléen

Syntaxe :

gamepad_get_option(device, option_name) ;

Arguments Description
dispositif Quel "slot" de périphérique gamepad vérifier.
nom_de_l'option Le nom de l'option pour obtenir la valeur de (une chaîne, voir le tableau ci-dessus)

Les retours :

Int, Bool, Real, String, N/A (dépend de la plateforme)

Exemple :

for(var i = 0 ; i < 12 ; ++i ;)
{
if gamepad_is_connected(i) && gamepad_get_description(i) == " tvOS SiriRemote "
{
if gamepad_get_option(i, "allow_rotation") == false
{
gamepad_set_option(i, "allow_rotation", true) ;
}
}
}

Le code ci-dessus passe en revue tous les emplacements de manettes de jeu et vérifie la présence de la "télécommande Siri" sur la plate-forme tvOS . S'il en détecte une, il configure alors la télécommande pour permettre la rotation des entrées.
