gamepad_axis_count

Cette fonction renvoie le nombre de commandes d'"axes" sur le dispositif vérifié. Ces commandes sont les "pouces" de direction analogiques de la plupart des contrôleurs.

Syntaxe :

gamepad_axis_count(device) ;

Arguments Description
dispositif Quel "slot" de périphérique gamepad vérifier.

Les retours :

Real

Exemple :

axis = gamepad_axis_count(0))

Le code ci-dessus stocke le nombre d'axes disponibles pour la manette de jeu connectée au périphérique "slot" 0 dans la variable "axis".
