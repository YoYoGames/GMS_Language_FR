gamepad_axis_value

Vous pouvez utiliser cette fonction pour obtenir la valeur des différents axes d'un emplacement de manette de jeu donné. Vous fournissez l'index de l'emplacement de la manette de jeu à vérifier, avec soit une constante d'index d'axe (comme indiqué ici ), soit une valeur entière comprise entre 0 et gamepad_axis_count() -1. La valeur de retour sera comprise entre -1 et 1 pour chacun des axes horizontaux et verticaux disponibles.

Notez que la valeur renvoyée est normalisée pour prendre en compte la valeur définie pour la zone morte de l'axe (en utilisant la fonction gamepad_set_axis_deadzone() ), et sera donc toujours entre -1 et 0, et 0 et 1. Par exemple, régler la zone morte sur 0,2 signifiera que pousser la manette vers la droite ne commencera à renvoyer une valeur de 0 à 1 que lorsque la valeur brute de l' axe est terminée 0,2 - donc, lorsque la valeur brute est 0,2, la valeur de retour sera 0, lorsque la valeur brute est à 0,5, la valeur de retour sera de 0,375, ou si la valeur brute est de 0,9, la valeur de retour sera de 0,875.

Syntaxe:

gamepad_axis_value(device, axisIndex);

Argument Description
device Quel «slot» de périphérique de manette de jeu vérifier.
axisIndex L'index de l'axe à vérifier (voir la liste des constantes ).

Retour:

real

Exemple:

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

Le code ci-dessus définit la vitesse et la direction de l'instance en fonction des valeurs renvoyées du contrôleur analogique gauche de la manette de jeu branchée sur le «slot» 0 du périphérique.

Retour: Entrée de la manette de jeu
