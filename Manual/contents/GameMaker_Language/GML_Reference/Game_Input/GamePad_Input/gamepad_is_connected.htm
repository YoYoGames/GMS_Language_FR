gamepad_is_connected

Cette fonction indique si une manette de jeu est connectée au "slot" donné (renvoie true) ou non (renvoie false). Vous utiliserez normalement cette fonction en conjonction avec la fonction gamepad_get_device_count() pour obtenir le nombre correct de manettes de jeu et/ou de "slots" disponibles. Notez qu'il peut y avoir un léger délai entre le moment où l'utilisateur connecte la manette de jeu et celui où GameMaker Studio 2 la détecte comme étant connectée (c'est notamment le cas lorsqu'il s'agit de manettes connectées par Bluetooth). cas lorsqu'il s'agit de manettes connectées par Bluetooth).

REMARQUE: Sur certaines plates-formes - en particulier les consoles - cette fonction peut renvoyer un faux immédiatement après qu'une manette de jeu ait été connectée/sélectionnée et peut devoir être vérifiée dans une alarme quelques images (étapes) plus tard pour qu'elle détecte correctement la manette de jeu. gamepad.

Syntaxe :

gamepad_is_connected(numb) ;

Arguments Description
dispositif Quel est le "slot" du gamepad à vérifier.

Les retours :

Booléen

Exemple :

var gp_num = gamepad_get_device_count() ;
for (var i = 0 ; i < gp_num ; i++ ;)
{
if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false ;
}

Le code ci-dessus passe en boucle les game pads disponibles (ou les emplacements de gamepad) et vérifie pour chacun d'entre eux s'il existe un gamepad connecté. La valeur renvoyée est ensuite utilisée pour définir un tableau global sur true ou false à utiliser dans les vérifications futures.
