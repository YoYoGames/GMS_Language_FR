gamepad_set_button_threshold

Cette fonction peut être utilisée pour définir le paramètre de seuil actuel des touches analogiques pour un appareil donné. Le seuil par défaut pour tous les boutons analogiques est de 0,5, la plage allant de 0 à 1. Le seuil définit à quel point le bouton est considéré comme étant "enfoncé" pour les jeux qui nécessitent qu'ils agissent comme un bouton numérique. Notez que cette fonction affectera les états de vérification, enfoncé et relâché pour les boutons analogiques, mais n'affectera pas la valeur renvoyée par la fonctiongamepad_button_check(), qui renverra toujours la valeur brute du bouton.

Syntaxe:

gamepad_set_button_threshold(device, threshold);

Argument Description
device Quel «slot» de périphérique de manette de jeu vérifier.
threshold La nouvelle valeur de seuil (de 0 à 1, 0,5 par défaut).

Retour:

N/A

Exemple:

if gamepad_get_button_threshold(0) != 0.5
   {
   gamepad_set_button_threshold(0, 0.5);
   }

Le code ci-dessus vérifie le seuil du bouton analogique de la manette de jeu connectée à l'appareil "slot" 0 et si ce n'est pas la valeur par défaut de 0,5, il est réglé sur cette valeur.

Retour: Entrée de la manette de jeu
