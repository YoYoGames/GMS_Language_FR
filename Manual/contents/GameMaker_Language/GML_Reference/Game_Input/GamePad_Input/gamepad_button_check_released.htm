gamepad_button_check_released

Cette fonction renvoie vrai ou faux selon que le bouton de la manette de jeu est détecté comme ayant été relâché ou non. Notez que cette fonction ne se déclenchera qu'une seule fois pour le bouton au moment où il a été relâché. Pour qu'elle se déclenche à nouveau, le bouton doit d'abord être enfoncé, puis relâché une nouvelle fois. Si vous vérifiez un bouton analogique, la vérification prendra en compte le réglage du seuil et ne renverra que ne renvoie true que lorsque la valeur d'entrée brute passe sous le seuil donné (vous pouvez obtenir cette valeur brute à l'aide de la fonction gamepad_button_value()).

Syntaxe :

gamepad_button_check_released(device, button) ;

Arguments Description
dispositif Quel "slot" de périphérique gamepad vérifier.
bouton Quelle constante de bouton de manette à vérifier.

Les retours :

Booléen

Exemple :

if gamepad_button_check_released(0, gp_select)
{
audio_play_sound(snd_Button, 0, false) ;
global.Pause = !global.Pause ;
}

Le code ci-dessus détectera si le bouton "select" de la manette de jeu connectée au périphérique "slot" 0 a été pressé ou non et basculera la variable globale "Pause".
