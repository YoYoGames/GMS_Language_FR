gamepad_get_guid

Avec cette fonction, vous pouvez récupérer le GUID de la manette de jeu connectée à l'index de slot donné. Vous fournissez l'emplacement de la manette de jeu à vérifier (de 0 à 11), et la fonction renverra l'une des chaînes suivantes:

  • " aucun " si aucun GUID n'est disponible ou qu'aucune manette de jeu n'est connectée
  • « index de l'appareil hors de portée » si l'index de l'emplacement de la manette de jeu est hors de portée
  • La chaîne GUID, quelque chose comme "050000005e040000fd020000ffff3f00", si l'emplacement a une manette de jeu attribuée et peut obtenir le GUID correspondant

Cette fonction est généralement utilisée en conjonction avec gamepad_get_description() pour générer une partie de la chaîne SDL requise pour le remappage des contrôleurs sous Windows, Android et macOS.

Syntaxe:

gamepad_get_guid(index);

Argument Description
index Quel index de «slot» de manette de jeu vérifier

Retour:

String

Exemple:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

Le code ci-dessus obtient le GUID et les chaînes de description de nom, puis les concatène et stocke la chaîne finale dans une variable globale pour référence future.

Retour: Entrée de la manette de jeu
