clavier_check_direct

Cette fonction retournera vrai si la touche avec le code particulier est pressée, ou faux si elle ne l'est pas, en vérifiant le matériel directement. Elle permet quelques vérifications supplémentaires, en particulier vous pouvez utiliser les keycodes vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol et vk_ralt pour vérifier si la touche shift, control ou alt gauche ou droite est enfoncée. La fonction prendra une valeur de code de touche telle que retournée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'une des constantes vk_* listées sur la page principale de saisie du clavier.

REMARQUE : Cette fonction n'est disponible que pour la cible Windows standard et le résultat est indépendant de l'application qui a le focus.

Syntaxe :

keyboard_check_direct(key) ;

Arguments Description
clé La clé pour vérifier l'état d'abattement de.

Les retours :

Booléen

Exemple :

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
{
crouch = true ;
}

Le code ci-dessus vérifie si les touches Alt gauche ou droite ont été pressées, et si c'est le cas, il met la variable "crouch" à true.
