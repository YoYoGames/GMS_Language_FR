Touche clavier

Avec cette variable, vous pouvez obtenir le code de la touche qui est actuellement pressée et elle retournera 0 si aucune touche n'est pressée lorsque la vérification est effectuée.

Syntaxe :

touche_clavier ;

Les retours :

Real

Exemple :

switch (keyboard_key)
{
case vk_numpad1 : gun = weapon[0][0] ; break ;
case vk_numpad2 : gun = weapon[1][0] ; break ;
case vk_numpad3 : gun = weapon[2][0] ; break ;
}

Le code ci-dessus utilise la valeur de la variable keyboard_key pour définir une variable à la même valeur qu'un tableau.
