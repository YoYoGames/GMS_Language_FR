keyboard_check_pressed

Avec cette fonction, vous pouvez vérifier si une touche a été pressée ou non. Contrairement à la fonction vérification du clavier() cette fonction ne s'exécutera qu'une fois pour chaque fois que la touche est enfoncée, donc pour qu'elle se déclenche à nouveau, la touche doit d'abord être relâchée, puis enfoncée à nouveau. La fonction prendra une valeur de code de touche telle que retournée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'une des constantes vk_* constantes listées sur la page principale de saisie du clavier.

Syntaxe :

keyboard_check_pressed(key) ;

Arguments Description
clé La touche pour vérifier l'état enfoncé de.

Les retours :

Booléen

Exemple :

if keyboard_check_pressed(vk_anykey)
{
room_goto_next() ;
}

Le code ci-dessus permet de passer à la pièce suivante si le joueur appuie sur l'une des touches du clavier (ce qui fonctionne comme une invite "Press Any Key to Continue").
