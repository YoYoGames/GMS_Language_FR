keyboard_clear

Avec cette fonction, vous pouvez effacer l'état actuel du clavier, ce qui signifie essentiellement que si la touche est maintenue enfoncée, elle ne sera plus reconnue jusqu'à ce qu'elle soit relâchée à nouveau (ce qui ne déclenchera pas non plus l'événement Keyboard Key Released à cette occasion) et pressé à nouveau. La fonction prendra une valeur de code clé telle que renvoyée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'un des vk_* constantes répertoriées sur la page principale de saisie au clavier.

Syntaxe:

keyboard_clear(key);

Argument Description
key La clé pour effacer.

Retour:

N/A

Exemple:

keyboard_clear(vk_space);

Le code ci-dessus efface l'état de la barre d'espace.

Retour: Entrée clavier
