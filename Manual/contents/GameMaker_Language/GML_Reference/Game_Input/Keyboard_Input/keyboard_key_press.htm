keyboard_key_press

Avec cette fonction, vous pouvez simuler la pression de n'importe quelle touche du clavier. Lorsqu'elle est utilisée, la touche sera marquée comme étant enfoncée jusqu'à ce que la fonction de libération correspondante soit appelée (voir keyboard_key_release() ). La fonction prendra une valeur de code clé telle que renvoyée par la fonction ord() (uniquement les lettres majuscules de A à Z ou les chiffres de 0 à 9), ou l'un des vk_* constantes répertoriées sur la page principale de saisie au clavier.

Notez qu'après l'appel de cette fonction, si la touche est physiquement enfoncée sur le clavier, cet événement de pression ne sera pas enregistré à nouveau tant que la touche n'aura pas été physiquement relâchée (déclenchant l'événement de libération et arrêtant cette fonction), ou la fonction de touche de libération correspondante été appelé, et la touche a été à nouveau enfoncée.

Syntaxe:

keyboard_key_press(key);

Argument Description
key La touche pour simuler une pression sur.

Retour:

N/A

Exemple:

keyboard_key_press(vk_space);

Cela simulera une pression sur la barre d'espace.

Retour: Entrée clavier
