touche_clavier_release

Avec cette fonction, vous pouvez simuler le relâchement de n'importe quelle touche du clavier. La fonction prendra une valeur de code de touche telle que retournée par la fonction ord() (uniquement les lettres majuscules de A-Z ou les chiffres de 0-9), ou l'une des constantes vk_* listées sur la page principale de saisie du clavier.

Syntaxe :

keyboard_key_release(key) ;

Arguments Description
clé La clé pour simuler une libération de.

Les retours :

N/A

Exemple :

touche_clavier_release(vk_space) ;

Cela simulera le relâchement de la barre d'espacement.

Retour : Entrée du clavier
