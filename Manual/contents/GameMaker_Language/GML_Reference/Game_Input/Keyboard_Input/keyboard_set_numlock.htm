clavier_set_numlock

Cette fonction permet d'activer ou de désactiver le verrouillage numérique du clavier ( true pour on, false pour off).

REMARQUE : cette fonctionnalité n'est disponible que dans les versions cibles de Windows et ne fonctionnera sur aucun autre appareil.

Syntaxe :

keyboard_set_numlock(value) ;

Arguments Description
valeur Définissez ce paramètre sur true pour "on" et false pour "off".

Les retours :

N/A

Exemple :

if keyboard_get_numlock()
{
keyboard_set_numlock(false) ;
}
else
{
keyboard_set_numlock(true) ;
}

L'exemple de code ci-dessus récupère l'état de la touche de verrouillage numérique et si elle est activée (vrai), elle est désactivée (faux) et vice-versa.
