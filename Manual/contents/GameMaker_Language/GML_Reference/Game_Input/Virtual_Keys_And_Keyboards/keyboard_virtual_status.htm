keyboard_virtual_status

Cette fonction peut être utilisée pour obtenir l'état du clavier virtuel sur l'appareil exécutant le jeu. La fonction retournera true si le clavier virtuel du système d'exploitation est visible / affiché ou false s'il est caché / caché.

Syntaxe

keyboard_virtual_status();

Retour:

Boolean

Exemple:

var _status = keyboard_virtual_status();
if _status == false
    {
    keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false);
    }

Le code ci-dessus affichera le clavier virtuel du système d'exploitation si l'état actuel est false.
