virtual_key_delete

Si votre jeu a différentes salles ou instances pour contrôler les menus, le jeu et d'autres choses, il est probable que vous deviez changer la position et les cartes des touches de vos touches virtuelles à un moment donné. Pour cela, vous pouvez utiliser cette fonction pour supprimer les anciens avant d'en créer les nouveaux (si nécessaire). La fonction nécessite que vous fournissiez l'ID de la clé virtuelle à supprimer - tel que renvoyé parvirtual_key_add().

Notez que toutes les clés virtuelles seront automatiquement supprimées de la pièce lorsque la pièce sera changée, cette fonction n'est donc nécessaire que lorsque vous souhaitez retirer manuellement les clés avant que la pièce elle-même ne soit terminée.

Syntaxe

virtual_key_delete(index);

Argument Description
index L'index de la clé virtuelle à supprimer.

Retour:

N/A

Exemple:

virtual_key_delete(global.Left);

Le code ci-dessus supprime la clé virtuelle indexée dans la variable globale "global.Left".
