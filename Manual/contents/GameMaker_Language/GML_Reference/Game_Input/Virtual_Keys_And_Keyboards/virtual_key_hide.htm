virtual_key_hide

Une fois que vous avez créé votre clé virtuelle pour vos appareils, vous devrez peut-être déboguer et tester qu'elle est correctement positionnée et qu'elle fonctionne correctement et pour cela, vous appelleriez la fonction virtual_key_show(). Cependant, vous devrez peut-être activer et désactiver cette vue, de sorte que vous pouvez également masquer la touche virtuelle avec cette fonction, en utilisant l'index de la clé virtuelle que vous souhaitez masquer (précédemment créée et stockée à l'aide de virtual_key_add() ). Une fois que vous êtes satisfait des choses, il est recommandé de créer et de placer votre propre graphique dans sa zone.

Syntaxe

virtual_key_hide(index);

Argument Description
index L'index de la clé virtuelle à masquer.

Retour:

N/A

Exemple:

if global.Debug
   {
   virtual_key_show(global.Left);
   }
else
   {
   virtual_key_hide(global.Left);
   }

Le code ci-dessus vérifie la variable globale "global.Debug" et si elle teste true, la clé virtuelle indexée dans la variable "global.Left" sera dessinée à l'écran, et si elle est fausse, la clé sera masquée.
