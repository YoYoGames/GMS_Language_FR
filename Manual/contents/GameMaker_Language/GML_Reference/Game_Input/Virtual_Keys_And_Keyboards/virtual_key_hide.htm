clé_virtuelle_cacher

Une fois que vous avez créé votre clé virtuelle pour vos périphériques, vous pouvez avoir besoin de déboguer et de tester qu'elle est correctement positionnée et qu'elle fonctionne correctement et pour cela, vous devez appeler la fonction virtual_key_show(). Cependant, il se peut que vous ayez besoin d'activer et de désactiver cette vue, vous pouvez donc également masquer la clé virtuelle avec cette fonction, en utilisant l'index de la clé virtuelle que vous souhaitez masquer (précédemment créée et stockée à l'aide de la fonction virtual_key_add()). Une fois que vous êtes satisfait des choses, il est recommandé de créer et de placer votre propre graphique dans sa zone.

Syntaxe

virtual_key_hide(index) ;

Arguments Description
indice L'index de la clé virtuelle à cacher.

Les retours :

N/A

Exemple :

if global.Debug
{
virtual_key_show(global.Left) ;
}
else
{
virtual_key_hide(global.Left) ;
}

Le code ci-dessus vérifie la variable globale "global.Debug" et si elle est vraie, la touche virtuelle indexée dans la variable "global.Left" sera dessinée à l'écran, et si elle est fausse, la touche sera cachée.
