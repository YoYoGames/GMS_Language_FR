chaîne_copie

Avec cette fonction, vous pouvez facilement sélectionner un certain nombre de caractères dans une chaîne de caractères pour les copier dans une autre. Le premier caractère d'une chaîne de caractères est toujours indexé à 1 et non à 0 comme on pourrait s'y attendre, donc pour copier (par exemple) les cinq premiers caractères d'une chaîne de caractères, il faudrait écrire string_copy(str 1, 5). de la chaîne de caractères, il faudrait écrire string_copy(str, 1, 5). Un autre exemple est fourni ci-dessous.

Syntaxe :

string_copy(str, index, count) ;

Arguments Description
str La chaîne de caractères à copier.
indice La position du premier caractère de la chaîne à copier (numéroté à partir de 1)
compter Le nombre de caractères à copier, en partant de la position du premier.

Les retours :

Chaîne de caractères

Exemple :

name = keyboard_string ;
if (string_length(name) > 15)
{
keyboard_string = string_copy(name, 1, 15) ;
}

Le code ci-dessus permet au joueur d'entrer une chaîne de caractères au moyen du clavier, qui est ensuite stockée dans la variable nom. Cette variable est ensuite vérifiée pour voir si elle dépasse la longueur maximale de 15 caractères et si c'est le cas, la chaîne keyboard_string est remplacée par une copie de 15 caractères de la chaîne du nom. Cela limite effectivement l'entrée du joueur à 15 caractères.
