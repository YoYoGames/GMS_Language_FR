string_copy

Avec cette fonction, vous pouvez facilement sélectionner un certain nombre de caractères dans une chaîne à copier dans une autre. Le premier caractère d'une chaîne est toujours indexé sur 1 et non sur 0 comme vous pouvez vous y attendre, donc pour copier (par exemple) les cinq premiers caractères de la chaîne que vous auriez string_copy(str, 1, 5). Un autre exemple est fourni ci-dessous.

Syntaxe:

string_copy(str, index, count);

Argument Description
str La chaîne à partir de laquelle copier.
index La position du premier caractère de la chaîne à copier (numéroté à partir de 1)
count Le nombre de caractères, à partir de la position du premier, à copier

Retour:

String

Exemple:

name = keyboard_string;
if (string_length(name) > 15)
      {
      keyboard_string = string_copy(name, 1, 15);
      }

Le code ci-dessus permettra au joueur de saisir une chaîne via le clavier qui est ensuite stockée dans la variable name. Cette variable est ensuite vérifiée pour voir si elle dépasse la longueur maximale de 15 caractères et si c'est le cas, la chaîne keyboard_string est remplacée par une copie de 15 caractères du name chaîne de caractères. Cela limite effectivement l'entrée du joueur à 15 caractères.
