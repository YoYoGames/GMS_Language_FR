clipboard_set_text

Cette fonction définira le presse-papiers pour contenir la chaîne définie. Vous pouvez le définir sur une chaîne vide "" pour effacer efficacement le presse-papiers du texte.

REMARQUE: cette fonction n'est valable que pour la cible Windows

Syntaxe:

clipboard_set_text(string);

Argument Description
string Le texte (sous forme de chaîne) que le presse-papiers doit contenir.

Retour:

N/A

Exemple:

if clipboard_has_text()
    {
    str = clipboard_get_text();
    clipboard_set_text("");
    }

Le code ci-dessus vérifie le texte dans le presse-papiers et s'il en contient, il est lu comme une chaîne dans la variable "str". Enfin, le presse-papiers est effacé en le définissant sur une chaîne vide.
