clipboard_set_text

Cette fonction va définir le presse-papiers pour contenir la chaîne de caractères définie. Vous pouvez la définir comme une chaîne vide "" pour vider le presse-papiers du texte.

NOTE: Cette fonction n'est valable que pour la Windows cible.

Syntaxe :

clipboard_set_text(string) ;

Arguments Description
chaîne de caractères Le texte (sous forme de chaîne de caractères) que le presse-papiers doit contenir.

Les retours :

N/A

Exemple :

if clipboard_has_text()
{
str = clipboard_get_text() ;
clipboard_set_text("") ;
}

Le code ci-dessus vérifie la présence de texte dans le presse-papiers et s'il en contient, il est lu sous forme de chaîne dans la variable "str". Enfin, le presse-papiers est effacé en lui attribuant la valeur d'une chaîne vide.
