remplacer la chaîne

Vous pouvez utiliser cette fonction pour analyser une chaîne de caractères à la recherche d'une partie spécifique, qui peut ensuite être remplacée par la nouvelle chaîne de caractères que vous avez spécifiée.

Syntaxe :

string_replace(str, substr, newstr) ;

Arguments Description
str La chaîne de caractères à copier.
substrat La sous-chaîne de la chaîne à remplacer.
newstr La nouvelle sous-chaîne qui remplace la précédente.

Les retours :

Chaîne de caractères

Exemple :

str1 = "Hello Earth" ;
str2 = string_replace(str1, "Earth", "World") ;

Cela va définir str2 comme str1, mais avec son instance de "Earth" remplacée par "World", ce qui fait que str2 sera "Hello World".
