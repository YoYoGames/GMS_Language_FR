string_replace_all

Vous pouvez utiliser cette fonction pour analyser une chaîne à la recherche de parties spécifiques, qui peuvent ensuite être remplacées par la nouvelle chaîne que vous avez spécifiée à tous les endroits où elles apparaissent.

Syntaxe:

string_replace_all(str, substr, newstr);

Argument Description
str La chaîne à copier.
substr La sous-chaîne de la chaîne à remplacer dans toutes les instances.
newstr La nouvelle sous-chaîne pour remplacer la ou les précédentes.

Retour:

String

Exemple:

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l");

Cela définira str2 à str1, mais avec ses instances de "x" remplacées par "l", ce qui entraîne str2 étant "Hello World".
