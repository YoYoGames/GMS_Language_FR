ord

Cette fonction prend une chaîne d'entrée de caractère unique et renvoie la valeur Unicode (UTF8) de ce caractère. Notez que lorsqu'il est utilisé avec le keyboard_check* fonctions, la chaîne d'entrée ne peut comporter qu'un seul caractère et ne peut être qu'un nombre compris entre 0 et 9 ou un caractère romain majuscule de A à Z.

Syntaxe:

ord(string);

Argument Description
string La chaîne avec laquelle trouver le code Unicode.

Retour:

Real

Exemple:

if keyboard_check(ord("W"))
    {
    y -= 4;
    }

Cela déplacera l'instance appelante de quatre pixels vers le haut si la touche W est maintenue enfoncée.
