chr

Cette fonction renvoie une chaîne contenant le caractère qui se rapporte au code Unicode d'entrée à afficher. Ce caractère dépend de la page de codes du jeu de caractères des polices de dessin actuelles et si aucune police n'est définie, il utilisera la page de codes par défaut de la machine.

Syntaxe:

chr(val);

Argument Description
val La valeur du code Unicode à partir de laquelle obtenir la chaîne

Retour:

String

Exemple:

mystring = chr(53) + chr(48);

Cela définirait mystring à "50" (une chaîne, pas un entier) car chr (53) est "5" et chr (48) est "0".
