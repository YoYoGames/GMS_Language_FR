string_repeat

Cette fonction renvoie simplement la même chaîne répétée un certain nombre de fois sur elle-même.

Syntaxe:

string_repeat(str, count);

Argument Description
str La chaîne à répéter.
count Le nombre de répétitions de la chaîne.

Retour:

String

Exemple:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

Le code ci-dessus définira str2 à "Hello WorldHello WorldHello World".
