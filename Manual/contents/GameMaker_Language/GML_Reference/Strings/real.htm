real

Cette fonction peut être utilisée pour transformer une chaîne donnée en un nombre réel. Lors de l'utilisation de cette fonction, les nombres, les signes moins, les points décimaux et les parties exponentielles de la chaîne sont pris en compte, tandis que d'autres caractères (tels que des lettres) provoqueront une erreur. Si vous savez ou soupçonnez qu'une chaîne peut avoir d'autres caractères, vous pouvez utiliser string_digits() pour supprimer tous les caractères non numériques, avant d'utiliser cette fonction pour transformer la chaîne résultante en un nombre réel.

Syntaxe:

real(string);

Argument Description
string La chaîne à convertir en valeur réelle.

Retour:

Real

Exemple:

var t_str = string_digits(input_str);
age = real(t_str);

Le code ci-dessus prendra la chaîne d'entrée, la dépouillera de tous les caractères autres que les nombres, puis définira la variable "age" pour qu'elle contienne la valeur réelle de la chaîne de retour.
