longueur_octet_chaîne

Cette fonction renvoie le nombre d'octets dans une chaîne de caractères, mais vous devez noter qu'en raison de leur maintien en UTF8, ce nombre ne sera pas égal à leur longueur.

Syntaxe :

string_byte_length(string) ;

Arguments Description
chaîne de caractères La chaîne de caractères dont on veut mesurer le nombre d'octets.

Les retours :

Real

Exemple :

bytesize = string_byte_length("Hello World") ;

Ceci définit bytesize comme le nombre d'octets dans "Hello World".

Retour : Cordes
