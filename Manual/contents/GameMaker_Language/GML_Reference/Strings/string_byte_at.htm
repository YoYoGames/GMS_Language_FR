string_byte_at

Renvoie la valeur brute de l'octet en tant que valeur réelle à une position donnée dans la chaîne donnée.

Syntaxe :

string_byte_at(str, index) ;

Arguments Description
str La chaîne à vérifier.
indice La position à partir de laquelle l'octet doit être récupéré.

Les retours :

Real

Exemple :

newbyte = string_byte_at("Hello World", 5) ;

Cela donnera à newbyte la valeur brute de l'octet de la sixième lettre de "Hello World".
