ansi_char

Cette fonction renvoie une chaîne contenant le caractère avec la valeur BYTE brute définie. Cela ne sera pas, et ne devrait pas, être affiché, mais il sera correctement enregistré sur le disque pour une utilisation dans le codage.

Syntaxe:

ansi_char(val);

Argument Description
val La valeur d'octet brut.

Retour:

String (Single character)

Exemple:

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

Le code ci-dessus crée une chaîne à partir de données d'octets brutes et l'écrit dans un fichier (précédemment ouvert).
