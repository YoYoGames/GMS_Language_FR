string_length

Cette fonction renvoie le nombre de caractères composant une chaîne donnée. Cela peut être utile pour des choses comme déterminer quand limiter la longueur des caractères d'une entrée de texte personnalisée (par exemple: limiter le nom d'un joueur à 10 caractères). N'oubliez pas que c'est différent de string_width() en ce qu 'il mesure le nombre de caractères de la chaîne, et non sa largeur telle que dessinée à l'écran en pixels.

Syntaxe:

string_length(string);

Argument Description
string Chaîne pour laquelle mesurer le nombre de caractères.

Retour:

Real

Exemple:

if (string_length(name) > 10)
   {
   name = string_copy(name, 1, 10);
   }

Cela vérifiera si la chaîne du nom est supérieure à dix caractères et si c'est le cas, il copiera et utilisera uniquement les dix premiers caractères.
