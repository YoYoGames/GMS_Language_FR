string_char_at

Vous pouvez utiliser cette fonction pour renvoyer un caractère spécifique à une position spécifique dans une chaîne, avec l'index commençant à 1 pour le premier caractère. Si aucun caractère n'est trouvé ou si la chaîne est plus courte que la valeur donnée à index, une chaîne vide "" est renvoyée.

Syntaxe:

string_char_at(str, index);

Argument Description
str La chaîne à vérifier.
index La position à partir de laquelle obtenir le personnage.

Retour:

String

Exemple:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

Cela définira str2 au septième caractère ("H" comptant comme le premier) dans la chaîne str1, dans ce cas "W".
