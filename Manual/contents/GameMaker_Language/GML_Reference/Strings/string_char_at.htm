chaîne_caractère_at

Vous pouvez utiliser cette fonction pour retourner un caractère spécifique à une position spécifique dans une chaîne de caractères, l'index commençant à 1 pour le premier caractère. Si aucun caractère n'est trouvé ou si la chaîne de caractères est plus courte que la valeur donnée à l'index, une chaîne vide """ est renvoyée. est renvoyée.

Syntaxe :

string_char_at(str, index) ;

Arguments Description
str La chaîne à vérifier.
indice La position pour obtenir le personnage à partir de.

Les retours :

Chaîne de caractères

Exemple :

str1 = "Hello World" ;
str2 = string_char_at(str1, 7) ;

Cela va définir str2 sur le septième caractère ("H" comptant comme le 1er) de la chaîne str1, dans ce cas "W".
