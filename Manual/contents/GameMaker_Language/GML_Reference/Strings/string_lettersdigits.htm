string_lettersdigits

Cette fonction renverra une copie d'une chaîne donnée avec tout sauf ses lettres et chiffres supprimés, ce qui signifie qu'elle peut être utilisée pour supprimer tous les caractères indésirables (comme "#" ou "?"), Par exemple, un nom de connexion ou un le mot de passe.

REMARQUE: Cette fonction renvoie uniquement l'alphabet anglais de 26 lettres de A à Z (majuscules ou minuscules) et les chiffres de 0 à 9.

Syntaxe:

string_lettersdigits(string);

Argument Description
string La chaîne à convertir en lettres et chiffres.

Retour:

String

Exemple:

if (string_length(password) > string_length(string_lettersdigits(password)))
   {
   draw_text(32,32,"Invalid Password! Only Letters and numbers please!");
   }

Le code ci-dessus vérifiera la longueur d'une chaîne par rapport à la longueur de la même chaîne, mais avec tous les symboles supprimés. S'ils ne sont pas les mêmes, un message est dessiné à l'écran.
