string_ord_at

Vous pouvez utiliser cette fonction pour renvoyer un code de caractère spécifique à une position spécifique dans une chaîne, avec l'index commençant à 1 pour le premier caractère. Si aucun caractère n'est trouvé ou si la chaîne est plus courte que la valeur donnée à index, -1 est renvoyé.

Syntaxe:

string_ord_at(str, index);

Argument Description
str La chaîne à vérifier.
index La position à partir de laquelle obtenir le code de caractère.

Retour:

Real

Exemple:

str = "Hello World";
char_code = string_ord_at(str, 7);

Cela obtiendra le code de caractère pour le septième caractère (où "H" compte comme le premier) dans la chaîne "str" et le stockera dans la variable "char_code".
