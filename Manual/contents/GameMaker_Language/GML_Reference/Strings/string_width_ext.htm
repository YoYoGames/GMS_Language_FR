string_width_ext

Cette fonction retournera la largeur maximale (en pixels) de la chaîne d'entrée, en tenant compte de la séparation de ligne et de la largeur de saut de ligne (qui est définie comme le nombre de pixels que la chaîne peut occuper avant qu'un saut de ligne ne soit inséré). Il est très pratique pour calculer les distances entre les éléments de texte en fonction de la largeur maximale d'une chaîne qui est divisée sur plusieurs lignes comme elle serait dessinée avec draw_text_ext() en utilisant la police actuellement définie. La séparation et la largeur peuvent être définies sur -1 pour obtenir l'espacement par défaut.

Syntaxe:

string_width_ext(string, sep, w);

Argument Description
string La chaîne pour mesurer la largeur de.
sep La distance en pixels entre les lignes de texte comme si la chaîne était dessinée.
w La largeur maximale (en pixels) de la chaîne avant un saut de ligne comme si la chaîne était dessinée.

Retour:

Real

Exemple:

var ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100);

Le code ci-dessus obtiendra la largeur de la chaîne donnée, en tenant compte de la séparation de ligne et de la largeur du saut de ligne, puis dessinera deux lignes de texte, en utilisant la largeur de chaîne totale renvoyée comme séparateur.
