camera_set_proj_mat

Cette fonction définira la matrice de projection pour une caméra donnée. Vous donnez la valeur d'ID de caméra unique (telle qu'elle est renvoyée par les différents camera_create() fonctions) et une matrice de vue à appliquer. Vous pouvez en savoir plus sur la création de matrices de projection dans la section Fonctions de matrice, en particulier matrix_build_projection_perspective() et matrix_build_projection_ortho(). Notez que si votre caméra effectue un suivi automatique des objets, c'est-à-dire qu'elle a été créée en utilisant camera_create_view() avec un index d'objet / ID d'instance qui n'est pas -1, ou vous définissez une caméra définie dans l'éditeur de salle, ou vous définissez la caméra par défaut - alors cette matrice sera écrasée la prochaine image de jeu.

Syntaxe:

camera_set_proj_mat(camera_id, matrix)

Argument Description
camera_id La valeur d'ID de caméra unique renvoyée lors de la création de la caméra.
matrix ID de matrice unique renvoyé lors de la création de la matrice.

Retour:

N/A

Exemple:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Le code ci-dessus crée une nouvelle caméra et l'affecte au port d'affichage [0]. Cette caméra a alors ses matrices de vue et de projection définies.
