camera_get_view_mat

Cette fonction peut être utilisée pour récupérer la matrice de vue. Elle renvoie la valeur de l'ID de la matrice, qui peut ensuite être utilisée dans d'autres fonctions matricielles ou pour définir la matrice de vue d'une autre caméra (à l'aide de la fonction une autre caméra (en utilisant camera_set_view_mat()). Notez que cette fonction est généralement utilisée avec les caméras créées à l'aide de la fonction caméra_create()mais elle peut également être utilisée sur celles créées dans l'éditeur de salle (ou avec camera_create_view()) - dans ce cas, la matrice renvoyée ne sera valide que lorsque la caméra donnée sera utilisée pour la première fois dans un port de vue.

Syntaxe :

camera_get_view_mat(camera_id)

Arguments Description
camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra.

Les retours :

ID de la matrice (réel)

Exemple :

mat = camera_get_view_mat(camera_view[0]) ;

Le code ci-dessus obtient la matrice de vue pour la caméra assignée au port de vue [0].
