vue_get_visible

Cette fonction peut être utilisée pour vérifier la visibilité du port de vue donné. La fonction retournera true si le port est visible et false s'il ne l'est pas.

Syntaxe :

view_get_visible(view_port)

Arguments Description
port_vue Le port de vue à cibler (0 - 7)

Les retours :

Booléen

Exemple :

if !view_get_visible(0)
{
view_set_visible(0, true) ;
}

Le code ci-dessus vérifie si le port de vue [0] est visible et s'il ne l'est pas, il est défini comme visible.
