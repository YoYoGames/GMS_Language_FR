vue_caméra

Ce tableau contient l'ID unique de la caméra attribuée au port de vue donné, et peut être défini pour une nouvelle caméra ou lu pour obtenir la caméra actuelle, renvoyant -1 si aucune caméra n'est attribuée. Vous pouvez avoir jusqu'à 8 ports de vue actifs dans une pièce (valeurs de tableau de 0 à 7). 7), et vous pouvez affecter une caméra à l'un d'entre eux en définissant simplement cette variable sur la valeur de l'ID de la caméra (tel que renvoyé par les fonctions camera_create() ou camera_create_view()). Si vous avez défini la caméra sur un port de vue via l'éditeur de salle, cette variable contient l'ID de la caméra pour ce port.

Syntaxe :

view_camera [0 ... 7] ;

Les retours :

Index de la caméra ou -1 si aucune caméra n'est assignée

Exemple :

si view_camera[0] == -1
{
view_camera[0] = camera_create_view(0, 0, room_width, room_height) ;
}

Le code ci-dessus vérifie si une caméra est attachée au port de vue [0] et si ce n'est pas le cas, une caméra est créée et fixée à ce port.
