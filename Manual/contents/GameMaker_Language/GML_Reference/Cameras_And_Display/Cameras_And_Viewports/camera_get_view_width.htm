camera_get_view_width

Cette fonction peut être utilisée pour récupérer la largeur (en pixels) de la caméra donnée. Notez que cette fonction est uniquement valable pour les caméras créées à l' aide camera_create_view() ou pour ceux ajoutés dans l'éditeur de salle.

Syntaxe:

camera_get_view_width(camera_id)

Argument Description
camera_id La valeur d'ID de caméra unique renvoyée lors de la création de la caméra

Retour:

Real

Exemple:

var vw = camera_get_view_width(view_camera[0]) / 2;
var vh = camera_get_view_height(view_camera[0]) / 2;
camera_set_view_pos(view_camera[0], x - vw, y - vh);

Le code ci-dessus récupère la largeur et la hauteur de la caméra affectée au port de vue [0], puis définit sa position par rapport au centre.
