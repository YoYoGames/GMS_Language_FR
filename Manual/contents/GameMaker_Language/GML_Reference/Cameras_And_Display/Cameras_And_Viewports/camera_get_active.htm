camera_get_active

Cette fonction peut être utilisée pour récupérer la valeur d'ID de caméra unique de la caméra actuellement active.

Syntaxe:

camera_get_active()

Retour:

Camera index (real).

Exemple:

var active = camera_get_active();
if active != view_camera[0]
    {
    view_camera[0] = active;
    }

Le code ci-dessus obtient l'ID de caméra pour la caméra active et définit la caméra de vue pour le port [0] pour l'utiliser.
