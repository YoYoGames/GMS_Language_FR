camera_set_view_target

Vous pouvez utiliser cette fonction pour définir la cible de suivi de la caméra de vue dans la pièce. Vous donnez la valeur unique de l'ID de la caméra (telle que renvoyée par les différentes fonctions de la fonction camera_create() fonctions), puis l'instance cible ou l'ID de l'objet ou l'ID de l'objet cible sur lequel vous souhaitez définir la vue de la caméra. Notez que si vous choisissez un ID d'objet et qu'il y a plus d'une instance de cet objet dans la pièce, GameMaker Studio 2 n'a aucun moyen de savoir quelle instance vous souhaitez suivre. Il peut donc s'agir de n'importe laquelle d'entre elles.

Syntaxe :

camera_set_view_target(camera_id, instance_id/object_id)

Arguments Description
camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra.
instance_id/object_id Instance ou objet à cibler par la caméra pour le suivi.

Les retours :

N/A

Exemple :

camera_set_view_target(view_camera[0], id) ;

Le code ci-dessus définira la cible de la caméra de vue pour la caméra affectée au port de vue [0] à celle de l'instance qui exécute le code.
