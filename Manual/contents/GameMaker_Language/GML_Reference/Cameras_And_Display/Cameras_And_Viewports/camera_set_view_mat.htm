camera_set_view_mat

Cette fonction va définir la matrice de vue pour une caméra donnée. Vous donnez la valeur unique de l'ID de la caméra (telle que retournée par les différentes fonctions) et la matrice de vue à appliquer. camera_create() fonctions) et une matrice de vue à appliquer. Vous pouvez en savoir plus d'informations sur la création de matrices de vue dans la section Fonctions de matrice, en particulier matrix_build_lookat(). Notez que si votre caméra effectue un suivi automatique des objets, c'est-à-dire qu'elle a été créée à l'aide de la fonction camera_create_view() avec un indice d'objet / ID d'instance qui n'est pas -1, ou si vous configurez une caméra définie dans l'éditeur de salle, ou encore si vous configurez une matrice de vue. l'éditeur de salle, ou vous définissez la caméra par défaut - alors cette matrice sera écrasée à la prochaine image du jeu.

Syntaxe :

camera_set_view_mat(camera_id, matrix)

Arguments Description
camera_id La valeur d'identification unique de la caméra renvoyée lorsque vous avez créé la caméra.
matrice L'identifiant unique de la matrice renvoyé lorsque vous avez créé la matrice.

Les retours :

N/A

Exemple :

view_camera[0] = camera_create() ;
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0) ;
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0) ;
camera_set_view_mat(view_camera[0], viewmat) ;
camera_set_proj_mat(view_camera[0], projmat) ;

Le code ci-dessus crée une nouvelle caméra et l'affecte au port de vue [0]. Cette caméra a ensuite ses matrices de vue et de projection définies.
