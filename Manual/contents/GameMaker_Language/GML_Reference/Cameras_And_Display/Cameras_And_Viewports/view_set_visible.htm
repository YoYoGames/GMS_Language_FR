view_set_visible

Cette fonction peut être utilisée pour définir la visibilité du port de vue donné. La fonction prend l'index du port de vue (de 0 à 7) et un booléen true s'il est visible et false si ce n'est pas le cas.

Syntaxe:

view_set_visible(view_port, visible)

Argument Description
view_port Le port de vue à cibler (0 - 7)
visible Bascule de visibilité (true est visible et false est invisible)

Retour:

N/A

Exemple:

if !view_get_visible(0)
    {
    view_set_visible(0, true);
    }

Le code ci-dessus vérifiera si le port de vue [0] est visible et s'il ne l'est pas, il est défini sur visible.
