afficher_get_sleep_margin

Cette fonction peut être utilisée pour obtenir la valeur actuelle de la marge de sommeil utilisée pour le timing de rendu de votre jeu, et renverra une valeur en millisecondes. Pour plus d'informations sur la synchronisation de l'affichage, veuillez consulter display_set_timing_method().

Syntaxe :

afficher_get_sleep_margin() ;

Les retours :

Real

Exemple :

if display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep) ;
if display_get_sleep_margin() != 20
{
display_set_sleep_margin(20) ;
}
}

Le code ci-dessus vérifiera la méthode de synchronisation et si elle n'est pas définie sur tm_sleep , elle sera définie sur cette méthode et la marge de sommeil sera définie sur 20.
