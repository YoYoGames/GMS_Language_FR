display_get_gui_width

Avec cette fonction, vous pouvez obtenir la hauteur (en pixels) de l'interface graphique telle qu'elle est utilisée dans l' événement Draw GUI.

Syntaxe:

display_get_gui_height();

Retour:

Real

Exemple:

ads_move(0, display_get_gui_height() - ads_get_display_height(0), 0);

Le code ci-dessus définira une publicité à afficher dans le coin inférieur gauche de l'écran.
