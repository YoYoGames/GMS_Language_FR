display_set_ui_visibility

Cette fonction peut être utilisée pour afficher ou masquer l'interface utilisateur du système sur Android seulement. La fonction vous demande de fournir un ou plusieurs drapeaux système sous forme de valeur entière. Lorsque vous utilisez plus d'un drapeau, ceux-ci doivent être fusionnés à l'aide de la méthode bitwise "ou", comme le montre l'exemple ci-dessous. Vous pouvez trouver une liste des indicateurs système Android ici.

Syntaxe :

display_set_ui_visibility(flags) ;

Arguments drapeaux
drapeaux Les drapeaux du SYSTÈME Android à utiliser (sous forme de valeur entière).

Les retours :

N/A

Exemple :

var flags = 1024|4096 ;
display_set_ui_visibility(flags) ;

Le code ci-dessus utilise les drapeaux du système Android "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" et "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" pour définir la visibilité de l'affichage.
