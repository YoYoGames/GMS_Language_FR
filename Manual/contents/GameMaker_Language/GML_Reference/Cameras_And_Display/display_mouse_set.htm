display_mouse_set

Avec cette fonction, vous pouvez modifier ou définir la position de la souris dans l'affichage du jeu, ce qui peut être utile pour les jeux FPS, par exemple. La fonction ne fonctionnera que lorsque le jeu est mis au point et utilise ALT + Tab déverrouillera la souris.

Syntaxe:

display_mouse_set(x, y);

Argument Description
x La coordonnée x dans l'affichage.
y La coordonnée y dans l'affichage.

Retour:

N/A

Exemple:

display_mouse_set(display_get_width() / 2, display_get_height() / 2);

Le code ci-dessus centrerait la souris dans l'affichage du jeu.
