window_mouse_set

Avec cette fonction, vous pouvez modifier ou définir la position de la souris dans la fenêtre de jeu, ce qui peut être utile pour les jeux FPS, par exemple. La fonction ne fonctionnera que pendant la mise au point du jeu et l'utilisation de alt + tab déverrouillera la souris.

REMARQUE: pour les fonctions normales de la souris, voir la section sur l' entrée de la souris

Syntaxe:

window_mouse_set(x, y);

Argument Description
x La coordonnée x dans la fenêtre.
y La coordonnée y dans la fenêtre.

Retour:

N/A

Exemple:

window_mouse_set(window_get_width() / 2, window_get_height() / 2);

Le code ci-dessus centrerait la souris dans la fenêtre de jeu.

Retour: La fenêtre de jeu
