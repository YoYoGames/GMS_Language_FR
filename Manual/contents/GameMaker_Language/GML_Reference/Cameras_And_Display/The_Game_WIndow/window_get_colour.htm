window_get_colour

Cette fonction renvoie la couleur d'arrière-plan de la fenêtre de jeu. Cette couleur représente celle qui sera utilisée pour les zones de la fenêtre de jeu qui ne sont occupées par aucune vue. L'image suivante illustre cela:

L'image ci-dessus a deux vues avec deux ports de vue, chacun étant dessiné à des positions différentes. Cela étend la fenêtre de jeu pour accueillir les deux ports et utilise la couleur de la fenêtre pour colorer l'arrière-plan là où aucune vue n'est affichée.

Syntaxe:

window_get_colour();

Retour:

N/A

Exemple:

if window_get_colour() != c_black
    {
    window_set_colour(c_black);
    }

Le code ci-dessus vérifiera la couleur de la fenêtre pour voir si elle est définie comme noire ou non, et si ce n'est pas le cas, elle la définit sur noir.
