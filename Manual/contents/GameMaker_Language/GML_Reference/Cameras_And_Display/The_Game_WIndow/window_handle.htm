window_handle

Avec cette fonction, vous pouvez obtenir la Windows interne (le HWND, un pointeur ). Cette fonction n'est vraiment utile que pour les auteurs d'extensions qui ont besoin du window handle pour appeler les Windows dans le code DLL (le pointeur renvoyé doit être converti en chaîne, puis dans le C ++, il suffit de le convertir en HWND). Le tableau ci-dessous montre les plates-formes prises en charge avec ce qu'elles renvoient:

Plate-forme Retour
Windows Fenêtre HWND
macOS Classe NSWindow
Ubuntu ( Linux ) Poignée XWindow
HTML5 Identifiant du canevas

Syntaxe:

window_handle();

Retour:

Pointer

Exemple:

win_id = windows_handle();

Le code ci-dessus stockera l'identifiant de la fenêtre de jeu dans une variable.

Retour: La fenêtre de jeu
