window_set_position

Avec cette fonction, vous pouvez définir la fenêtre du jeu sur une position spécifique dans l'affichage (sous macOS, Linux ( Ubuntu ) et Windows ) ou dans le navigateur (HTML5).

REMARQUE: si votre jeu HTML5 utilise un index.html et qui place le canevas à une position fixe, cette fonction n'aura aucun effet sur la position de la fenêtre.

Syntaxe:

window_set_position(x, y);

Argument Description
x Coordonnée x de l'emplacement de la fenêtre.
y La coordonnée y de l'emplacement de la fenêtre.

Retour:

N/A

Exemple:

window_set_position(0, 0);

Le code ci-dessus positionnera la fenêtre du jeu dans le coin supérieur gauche du navigateur ou de l'affichage (selon le module cible utilisé).

Retour: La fenêtre de jeu
