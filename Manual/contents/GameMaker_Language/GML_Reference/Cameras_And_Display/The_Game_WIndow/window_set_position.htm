fenêtre_set_position

Cette fonction permet de placer la fenêtre de jeu à une position spécifique dans l'affichage (sur macOS, Linux(Ubuntu) et Windows) ou dans le navigateur (HTML5).

REMARQUE: si votre jeu HTML5 utilise un index.html personnalisé et que celui-ci définit le canevas à une position fixe, cette fonction n'aura aucun effet sur la position de la fenêtre.

Syntaxe :

window_set_position(x, y) ;

Arguments Description
x La coordonnée x de l'endroit où positionner la fenêtre.
y La coordonnée y de l'endroit où positionner la fenêtre.

Les retours :

N/A

Exemple :

window_set_position(0, 0) ;

Le code ci-dessus positionnera la fenêtre de jeu dans le coin supérieur gauche du navigateur ou de l'écran (selon le module cible utilisé).
