window_set_fullscreen

Cette fonction vous permet de définir la fenêtre de jeu en plein écran(true) ou non(false). Veuillez noter que pour la cible macOS cible, vous devez avoir décoché l'option "Démarrer en plein écran" et coché l'option "Autoriser le joueur à redimensionner la fenêtre de jeu" dans les Options du jeu. le joueur à redimensionner la fenêtre de jeu" dans les options du jeu , sinon cette fonction échouera. Notez également que cette fonction ne fonctionnera pas en HTML5 si elle n'est pas ajoutée en tant que "clickable". (voir ici pour plus de détails).

Syntaxe :

window_set_fullscreen(full) ;

Arguments Description
complet Si l'écran doit passer en mode plein écran (true) ou non (false).

Les retours :

N/A

Exemple :

if mouse_check_button_pressed(mb_left)
{
if window_get_fullscreen()
{
window_set_fullscreen(false) ;
}
else
{
window_set_fullscreen(true) ;
}
}

Le code ci-dessus vérifie si le bouton de la souris a été enfoncé et fait passer la fenêtre en plein écran si ce n'est pas déjà le cas, ou en mode fenêtré si c'est déjà le cas.
