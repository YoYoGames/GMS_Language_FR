window_get_fullscreen

Cette fonction renvoie si la fenêtre de jeu est en mode plein écran (true ) ou non (false ).

Syntaxe:

window_get_fullscreen();

Retour:

Boolean

Exemple:

if window_get_fullscreen()
    {
    draw_text(32, 32, "Fullscreen is ON");
    }
else     {
    draw_text(32, 32, "Fullscreen is OFF");
    }

Le code ci-dessus vérifie si la fenêtre est en mode plein écran ou non et dessine un message texte en fonction de la valeur renvoyée.

Retour: La fenêtre de jeu
