La fenêtre de jeu

Le jeu que vous créez se déroule dans une fenêtre (même en plein écran), et cette fenêtre a un certain nombre de propriétés, comme la position, la taille, si elle est en plein écran, etc... Ces détails sont normalement définis automatiquement pour vous en fonction de la taille de la pièce et des ports de vue activés en combinaison avec les paramètres des Options de jeu pour la plate-forme cible, mais vous pouvez les modifier pendant le jeu en utilisant les fonctions listées sur cette page.

NOTE: Ces fonctions sont pour Windows, Ubuntu(Linux), macOS et les modules cibles HTML5 uniquement et peuvent ne pas fonctionner sur tout autre appareil.
