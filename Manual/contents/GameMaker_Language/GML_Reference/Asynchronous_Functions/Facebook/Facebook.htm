Facebook

Facebook est l'un des sites de médias sociaux les plus importants sur Internet et de nombreux jeux et développeurs ont actuellement leurs propres pages pour publier des jeux et des médias. GameMaker Studio 2 a une extension qui vous permet de vous connecter à Facebook et d'effectuer certaines actions et tâches que leur API permet. Il est recommandé de visiter la page Facebook Developers avant de commencer à utiliser l'extension, car il est essentiel de bien comprendre le fonctionnement de Facebook Graph et les appels et autorisations disponibles pour que vous puissiez tirer le meilleur parti de ces fonctions.

Vous pouvez découvrir comment configurer et utiliser l'extension Facebook pour toutes les plates-formes disponibles à partir du lien suivant:

  • Android, iOS et HTML5: intégrer Facebook

Les fonctions suivantes sont obsolètes qui n'étaient valables que pour l'ancienne API Facebook. En tant que tel, vous ne pouvez plus utiliser ces fonctions dans vos projets et elles sont répertoriées ici au cas où vous auriez importé un ancien projet avec elles. Pour la fonctionnalité Facebook actuelle, vous devez avoir activé l' API Facebook dans les options de jeu appropriées pour la plate-forme cible, ce qui vous demandera de télécharger et d'installer une extension. L'extension est livrée avec un *.pdf format "mini-manuel" expliquant comment l'utiliser ainsi que toutes les définitions de fonctions et exemples. Toutes les références à l'ancien facebook_*() les fonctions devront être remplacées (et mises à jour) pour utiliser l'extension fb_*() équivalents.

Les fonctions répertoriées ci-dessous concernaient l'intégration de votre jeu avec Facebook:

  • facebook_init
  • facebook_login
  • facebook_request_publish_permissions
  • facebook_request_read_permissions
  • facebook_check_permission
  • facebook_status
  • facebook_accesstoken
  • facebook_user_id
  • facebook_graph_request
  • facebook_dialog
  • facebook_send_invite
  • facebook_post_message
  • facebook_logout
  • facebook_launch_offerwall
