achievement_login_status

Cette fonction n'est actuellement utile que pour Google Play sur la Android forme Android. Il vérifiera si l'utilisateur est déjà connecté aux services Google, auquel cas il reviendra true, ou pas, retour false. S'il revient true il n'est pas nécessaire d'appeler le achievement_login() fonction.

Syntaxe:

achievement_login_status();

Retour:

Boolean

Exemple:

if !achievement_login_status()
    {
    achievement_login();
    }

Le code ci-dessus vous connectera au classement et au service de réussite choisis.
