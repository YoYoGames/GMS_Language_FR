achievement_show_challenge_notifications

Avec cette fonction, vous pouvez afficher ou supprimer les différentes notifications contextuelles "toast" relatives aux défis. Lorsque les arguments sont définis sur true ces messages apparaîtront, informant le joueur de tous les défis locaux ou distants reçus ainsi que des défis qui ont été complétés, tout en les définissant sur false supprimera ces notifications.

Syntaxe:

achievement_show_challenge_notifications(receive, local, remote)

Argument Description
receive Afficher les défis reçus pop-up (true ) ou non (false ).
local Afficher la fenêtre contextuelle des défis locaux terminés (true ) ou non (false ).
remote Afficher la fenêtre contextuelle des défis à distance terminés (true ) ou non (false ).

Retour:

N/A

Exemple:

achievement_show_challenge_notifications(true, false, false);

Le code ci-dessus définira les notifications toast de défi à afficher uniquement pour les défis reçus.
