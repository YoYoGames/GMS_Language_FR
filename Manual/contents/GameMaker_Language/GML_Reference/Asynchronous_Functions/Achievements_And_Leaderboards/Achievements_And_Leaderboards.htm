Réalisations et classements

Les jeux conçus pour cibler des appareils exécutant iOS ou Android peuvent également se connecter à l'une des deux API de classement et de réussite. Pour iOS vous avez iTunes Connect, et pour Android vous pouvez utiliser Google Play, les deux utilisant les fonctions de cette page. Ces fonctions permettent à votre jeu d'envoyer des informations sur les scores et les réalisations aux classements que vous avez configurés dans le compte Développeurs pour le jeu que vous créez et constituent un excellent moyen d'impliquer les joueurs dans le côté social du jeu comme ils peuvent alors le voir. ce que les autres personnes qui ont joué à votre jeu ont réalisé et essaient de les améliorer.

Lorsque vous utilisez ces fonctions, vous devez noter qu'elles ne suivent pas automatiquement les réalisations et les scores pour vous. Ils affichent uniquement les données que vous leur envoyez à partir de votre jeu, vous devez donc coder vos propres réalisations et votre système de hi-score pour suivre le statut des individus dans le jeu, puis utiliser ces fonctions pour ajouter les données au réseau choisi (quelle que soit la façon dont vous peut demander des informations sur la progression des réalisations en utilisant le achievement_load_progress() fonction).

Certaines de ces fonctions déclencheront un événement asynchrone social de rappel, qui contiendra un DS Map de données relatives à la fonction qui l'a déclenchée. Les fonctions qui déclenchent l'événement détaillent également les paires clé / valeur de la carte renvoyées par le rappel, mais un événement social de rappel peut également être déclenché par votre jeu recevant des notifications externes sur les défis, etc.

REMARQUE: pour que ces fonctions fonctionnent avec iOS vous devez disposer d'un compte Game Center valide et d'un identifiant d'application iOS pour votre jeu.

REMARQUE: pour que ces fonctions fonctionnent avec Android vous devez soumettre votre jeu à Google Play, avoir configuré les sections nécessaires et activer l'API dans les options de jeu Android pour votre jeu.

Les fonctions répertoriées ci-dessous sont celles de base utilisées pour intégrer votre jeu à la plate-forme de jeu social cible choisie:

  • achievement_login_status
  • achievement_login
  • achievement_logout
  • achievement_available
  • achievement_post
  • achievement_post_score
  • achievement_increment
  • achievement_show_achievements
  • achievement_show_leaderboards
  • achievement_show_challenge_notifications
  • achievement_get_challenges
  • achievement_get_pic
  • achievement_load_friends
  • achievement_load_leaderboard
  • achievement_load_progress
  • achievement_send_challenge
  • achievement_reset
