Réalisations et classements

Les jeux destinés aux appareils fonctionnant sous iOS ou Android peuvent également se connecter à l'une des deux API de classement et de réussite. Pour iOS, vous disposez d'iTunes Connect, et pour Android, vous pouvez utiliser Google Play. les deux utilisent les fonctions trouvées sur cette page. Ces fonctions permettent à votre jeu d'envoyer des informations sur les scores et les réalisations aux classements que vous avez configurés dans le compte de développeur pour le jeu que vous créez. Il s'agit d'un excellent moyen d'impliquer les joueurs dans l'aspect social du jeu, car ils peuvent alors voir les résultats obtenus par d'autres personnes ayant joué à votre jeu et essayer de les améliorer.

Lorsque vous utilisez ces fonctions, vous devez noter qu'elles ne suivent pas les réalisations et les scores automatiquement pour vous. Elles n'affichent que les données que vous leur envoyez à partir de votre jeu, vous devez donc coder vos propres réalisations et votre propre système de score pour suivre le statut des individus dans le jeu, puis utiliser ces fonctions pour ajouter les données. Vous devez donc coder votre propre système de réussite et de score pour suivre l'état des individus dans le jeu, puis utiliser ces fonctions pour ajouter les données au réseau choisi (vous pouvez toutefois demander des informations sur la progression des réalisations à l'aide de la fonction achievement_load_progress() ).

Certaines de ces fonctions déclencheront un événement social asynchrone(callback), qui contiendra une carte DS Map de données relatives à la fonction qui l'a déclenchée. à la fonction qui l'a déclenché. Ces fonctions qui déclenchent l'événement détaillent également les paires clé/valeur de la carte qui sont renvoyées par le callback, mais un événement social de callback peut également être déclenché par votre jeu en recevant notifications externes concernant des défis, etc...

REMARQUE: pour que ces fonctions fonctionnent avec iOS, vous devez avoir un compte Game Center valide et un identifiant d'application iOS pour votre jeu.

REMARQUE: pour que ces fonctions fonctionnent avec Android, vous devez soumettre votre jeu à Google Play, configurer les sections nécessaires et activer l'API dans les options de jeu d'Android pour votre jeu. votre jeu.
