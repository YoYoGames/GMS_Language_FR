Sauvegarde des nuages

Lors de la création de jeux, il est souvent nécessaire de stocker des informations sur l'état du jeu dans un fichier quelconque, mais stocker ces informations sur l'appareil n'est pas toujours la meilleure option car, si le joueur supprime le jeu et le réinstalle ensuite, ces informations peuvent être perdues. peuvent être perdues. Pour éviter cela, vous pouvez utiliser différents services en nuage, qui permettent de stocker des données sur Internet afin de les récupérer et de les modifier à tout moment. Vous devez noter que cette fonction est limitée à un seul "blob" de données par jeu. Vous devez noter que cette fonction est limitée à un seul "blob" de données par jeu, de sorte que chaque fois que vous envoyez une nouvelle sauvegarde au service en nuage, qu'il s'agisse d'une chaîne ou d'un fichier, elle écrasera toutes les données précédemment stockées.

REMARQUE : Actuellement, seule la cible Android cible utilise ces fonctions, et vous devez cocher la case Activer la sauvegarde dans le nuage de Google dans la section Sociale des options du jeu Android. Cela vous invitera à télécharger l'extension Google Services, qui est nécessaire au fonctionnement de la sauvegarde dans le nuage sur Android. Pour plus d'informations sur la sauvegarde dans le nuage, veuillez consulter l'article suivant du service d'assistance : AndroidCloud Saving
