shop_leave_rating

Cette fonction ouvre une boîte de dialogue dépendant du système d'exploitation dans laquelle vous pouvez demander à l'utilisateur de publier une évaluation ou un commentaire sur une page particulière. Vous pouvez définir le texte qui doit apparaître dans la boîte de dialogue, ainsi que le texte que vous souhaitez voir apparaître sur les deux boutons et l'URL où le commentaire doit être posté.

REMARQUE: Cette fonction est uniquement pour les cibles Android et iOS.

IMPORTANT ! Sur iOSlorsque vous utilisez TestFlight, l'invite s'affiche pendant le développement mais vous ne pouvez rien faire. Une fois que votre application est dans l'App Store, ce bouton devient actif et l'utilisateur a la possibilité de d'écrire une critique ou de laisser une note. Notez que si vous distribuez le jeu par le biais de TestFlight, cette fenêtre peut même ne pas s'afficher.

Syntaxe :

shop_leave_rating(texte, yes_string, no_string, url)

Arguments Description
texte Texte qui apparaît dans la boîte de dialogue.
oui_chaîne Texte qui apparaît sur le bouton "oui".
sans_chaîne Texte qui apparaît sur le bouton "non".
url L'URL complète où le commentaire doit être envoyé.

Les retours :

N/A

Exemple :

if timer <= 0
{
shop_leave_rating("Merci d'avoir joué le jeu ! Pourquoi ne pas laisser un commentaire ?", "Ok !", "Pas aujourd'hui !", "http://MacSweeney/comments") ;
timer = 10000000 ;
}
else
{
timer -= 1 ;
}

Le code ci-dessus demandera à l'utilisateur de laisser un commentaire si la variable "timer" a décompté jusqu'à 0.
