ds_map_read

Cette fonction va prendre une chaîne de caractères qui a été précédemment créée par la fonction ds_map_write() et la lira dans une carte DS précédemment créée. Si la carte dans laquelle la chaîne contient des paires clé/valeur, celles-ci seront d'abord effacées avant que la carte sauvegardée ne soit reconstruite. Notez que si la structure de données a été créée avec des versions antérieures de GameMaker Studio 2, vous devez ajouter l'argument facultatif "legacy", en lui attribuant la valeur true car le format de la chaîne a changé avec cette version.

Syntaxe :

ds_map_read(id, str [, legacy]) ;

Arguments Description
id L'identifiant de la structure de données dans laquelle lire la chaîne de caractères.
str La chaîne à lire
héritage(facultatif) Peut être soit vrai, soit faux, soit complètement omis.

Les retours :

N/A

Exemple :

inventaire = ds_map_create() ;
ini_open("map.ini") ;
var t_string = ini_read_string("Saved", "0", "") ;
if (t_string != "")
{
ds_map_read(inventory, t_string) ;
}
ini_close() ;

Le code ci-dessus crée une nouvelle carte DS et stocke son index id dans la variable "inventory". Il ouvre ensuite un fichier ini et lit une chaîne de ce fichier dans la variable temporaire "t_string". Enfin, il vérifie que la est valide (pas la valeur par défaut de "") et si c'est le cas, elle lit la chaîne dans la carte DS nouvellement créée avant de refermer le fichier ini.
