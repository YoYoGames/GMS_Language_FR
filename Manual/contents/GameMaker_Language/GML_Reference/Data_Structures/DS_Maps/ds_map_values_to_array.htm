ds_map_values_to_array

Cette fonction permet de récupérer toutes les valeurs contenues dans une carte DS pour chaque clé de la carte. Vous fournissez l'ID de la carte DS pour récupérer les clés (comme retourné par ds_map_create()) et la fonction renvoie un tableau dont chaque entrée correspond à une valeur de la carte DS (par exemple, si la carte comporte 10 clés, un tableau de 10 entrées sera renvoyé, chaque entrée correspondant à une valeur de clé). à une valeur pour une clé). La fonction dispose d'un deuxième argument facultatif où vous pouvez fournir un tableau que vous avez créé, auquel cas les données de la valeur de la clé de la carte seront ajoutées à toute donnée existante dans le tableau. Notez que la fonction modifiera le tableau fourni directement, mais renvoie également une référence à celui-ci (ou une référence à un nouveau tableau si aucun n'est fourni).

Syntaxe :

ds_map_values_to_array(id, [array])

Arguments Description
id L'identifiant de la carte à utiliser.
[tableau] FACULTATIF ! Le tableau auquel ajouter les valeurs de la carte DS.

Les retours :

Array
