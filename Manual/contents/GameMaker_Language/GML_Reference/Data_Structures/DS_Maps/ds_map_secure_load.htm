ds_map_secure_load

Cette fonction chargera une carte DS précédemment enregistrée à partir du fichier donné. Ce fichier doit avoir été créé au préalable à l'aide du ds_map_secure_save() et, une fois chargée, la fonction renverra l' index de la carte DS qui a été créée à partir des données chargées. Cet index de carte DS doit être stocké dans une variable et utilisé pour tous les autres appels de fonction à cette carte. Notez que si la carte DS en cours de chargement a été enregistrée avec un tableau comme l'une des valeurs clés, ce tableau aura été converti en liste DS lors du chargement.

IMPORTANT! L'une des caractéristiques d'un fichier enregistré sécurisé est qu'il est verrouillé sur l'appareil sur lequel il a été créé, de sorte que vous ne pouvez pas charger un fichier enregistré sur un appareil dans un projet exécuté sur un autre appareil.

Syntaxe:

ds_map_secure_load(filename);

Argument Description
filename Le nom du fichier à partir duquel charger les données cartographiques.

Retour:

Integer (DS map index)

Exemple:

p_map = ds_map_secure_load("p_data.dat");

Le code ci-dessus chargera une carte DS enregistrée en toute sécurité et stockera sa valeur d'index dans une variable pour une utilisation future.
