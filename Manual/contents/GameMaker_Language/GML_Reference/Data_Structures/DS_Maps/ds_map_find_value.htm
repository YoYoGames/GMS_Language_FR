ds_map_find_value

Avec cette fonction, vous pouvez obtenir la valeur d'une clé spécifiée. Les valeurs d'entrée de la fonction sont la carte DS (créée précédemment) à utiliser et la clé à vérifier.

REMARQUE: si aucune clé n'existe, la fonction retournera <undefined>. Vous devriez toujours vérifier cela en utilisant la fonction is_undefined ()

Syntaxe:

ds_map_find_value(id, key)

Argument Description
id L'identifiant de la carte à utiliser.
key La clé à trouver.

Retour:

Real, string or undefined

Exemple:

amount = ds_map_find_value(inventory, "food");

Ou, en utilisant l' accesseur de carte "?":

amount = inventory[? "food"];

Le code ci-dessus obtiendra la valeur de la clé «nourriture» et la stockera dans la variable «montant».
