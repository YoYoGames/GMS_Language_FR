ds_map_find_value

Avec cette fonction, vous pouvez obtenir la valeur d'une clé spécifiée. Les valeurs d'entrée de la fonction sont la carte DS (précédemment créée) à utiliser et la clé à vérifier.

REMARQUE: Si une telle clé n'existe pas, la fonction renverra un message non défini. Vous devez toujours vérifier cela en utilisant la is_undefined() fonction

Syntaxe :

ds_map_find_value(id, key)

Arguments Description
id L'identifiant de la carte à utiliser.
clé La clé pour trouver.

Les retours :

Réel, chaîne de caractères ou indéfini

Exemple :

amount = ds_map_find_value(inventory, "food") ;

Ou, en utilisant l'accesseur de carte " ?":

montant = inventaire[ ? "nourriture"] ;

Le code ci-dessus récupère la valeur de la clé "food" et la stocke dans la variable "amount".
