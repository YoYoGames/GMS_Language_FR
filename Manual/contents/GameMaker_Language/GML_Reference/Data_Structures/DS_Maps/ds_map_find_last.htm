ds_map_find_last

Cette fonction renvoie la dernière clé stockée dans la carte DS. Ce n'est pas la dernière clé dans l'ordre dans lequel vous les avez ajoutées! Les cartes DS ne sont pas stockées sous une forme linéaire, pour cela utilisent des listes DS, il suffit donc de trouver la dernière clé stockée par l'ordinateur. Cela peut être utile si vous devez parcourir la carte DS à la recherche de quelque chose, mais cela doit être évité si possible car cela peut être lent.

Syntaxe:

ds_map_find_last(id);

Argument Description
id L'identifiant de la carte à utiliser.

Retour:

Real or String

Exemple:

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
    {
    if (key != "gold")
        {
        key = ds_map_find_previous(inventory, key);
        }
    else break;
    }

Le code ci-dessus crée des variables temporaires, puis obtient la taille de la carte DS et trouve la dernière clé stockée par l'ordinateur dans la carte. Il utilise ensuite un for boucle pour parcourir la carte DS à la recherche de la valeur clé "or". S'il le trouve, il brise la boucle.
