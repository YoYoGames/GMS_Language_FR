ds_map_delete

Avec cette fonction, vous pouvez supprimer n'importe quelle clé donnée (et sa valeur correspondante) de la carte DS donnée précédemment créée.

Syntaxe:

ds_map_delete(id, key);

Argument Description
id L'identifiant de la carte à modifier.
key La clé (avec sa valeur associée) à supprimer.

Retour:

N/A

Exemple:

ds_map_delete(inventory, "shield");

Le code ci-dessus supprimera la clé "bouclier" (et la valeur avec laquelle elle est associée) de la carte DS (inventaire).
