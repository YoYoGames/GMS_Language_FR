ds_map_replace

Cette fonction permet de modifier la valeur de la clé donnée dans une carte DS. Vous fournissez l'index de la carte (tel que retourné par la fonction ds_map_create()) puis la clé à remplacer - soit une chaîne de caractères, soit un nombre entier - et la valeur par laquelle remplacer la valeur de la clé. Si la clé donnée n'existe pas , elle sera créée pour vous, et si elle existe, la valeur actuelle sera remplacée par la nouvelle valeur. valeur. La fonction renvoie true si la clé existe et que la valeur est remplacée, et false si la clé n'existe pas et qu'une nouvelle clé a été créée avec la valeur.

Syntaxe :

ds_map_replace( id, key, val ) ;

Arguments Description
id L'identifiant de la carte à modifier.
clé La clé avec la valeur qui doit être remplacée par la nouvelle.
val La nouvelle valeur pour remplacer la valeur donnée par

Les retours :

Booléen

Exemple :

ds_map_replace(inventory, "torso", 55) ;

Le code ci-dessus recherche la clé "torso" dans la carte DS et lorsqu'il la trouve (ou qu'elle est créée si elle n'existe pas), la valeur actuelle est remplacée par celle spécifiée.
