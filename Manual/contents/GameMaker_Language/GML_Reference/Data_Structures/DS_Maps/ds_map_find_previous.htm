ds_map_find_previous

Cette fonction renvoie la clé précédente stockée dans la carte DS avant celle spécifiée dans la fonction. Cela peut être utile si vous devez parcourir la carte DS à la recherche de quelque chose, mais cela doit être évité si possible car cela peut être lent. Si aucune clé n'existe, la fonction retournera <undefined>. Vous devriez toujours vérifier cela en utilisant le is_undefined() fonction.

Syntaxe:

ds_map_find_previous(id, key);

Argument Description
id L'identifiant de la carte à utiliser.
key La clé pour trouver le précédent.

Retour:

Real, string or undefined

Exemple:

var size = ds_map_size(inventory) - 1;
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
    {
    if (key != "gold")
        {
        key = ds_map_find_previous(inventory, key);
        }
    else break;
    }

Le code ci-dessus crée des variables temporaires, puis obtient la taille ds_map et trouve la dernière clé stockée par l'ordinateur dans la carte. Il utilise ensuite un for boucle pour parcourir le ds_map à la recherche de la valeur de clé "or". S'il le trouve, il brise la boucle.
