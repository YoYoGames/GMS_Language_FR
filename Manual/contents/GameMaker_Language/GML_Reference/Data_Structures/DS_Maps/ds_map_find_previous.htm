ds_map_find_previous

Cette fonction renvoie la clé précédente stockée dans la carte DS avant celle spécifiée dans la fonction. Cela peut être utile si vous devez itérer dans la carte DS à la recherche de quelque chose, mais devrait être évité si possible car cela peut être lent. Si aucune clé de ce type n'existe, la fonction renvoie undefined. Vous devriez toujours vérifier cela en utilisant la fonction is_undefined() fonction.

Syntaxe :

ds_map_find_previous(id, key) ;

Arguments Description
id L'identifiant de la carte à utiliser.
clé La clé pour trouver le précédent.

Les retours :

Réel, chaîne de caractères ou indéfini

Exemple :

var size = ds_map_size(inventory) - 1 ;
var key = ds_map_find_last(inventory) ;
for (var i = size ; i > 0 ; i-- ;)
{
if (key != "gold")
{
key = ds_map_find_previous(inventory, key) ;
}
else break ;
}

Le code ci-dessus crée quelques variables temporaires, puis récupère la taille de ds_map et trouve la dernière clé stockée par l'ordinateur dans la carte. Il utilise ensuite une boucle for pour parcourir la ds_map à la recherche de la valeur "gold". Si elle la trouve, elle interrompt la boucle.
