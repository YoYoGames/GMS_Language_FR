ds_map_replace_map

Avec cette fonction, vous pouvez remplacer un DS Map qui a été stocké dans la "clé" donnée avec une autre carte qui a été créée précédemment. Cette fonction est conçue pour créer des cartes compatibles JSON que vous encoderiez ensuite en utilisant json_encode() et ne doit être utilisé qu'en conjonction avec cette fonctionnalité.

Syntaxe:

ds_map_replace_map(id, key, value)

Argument Description
id L'identifiant du ds_map à utiliser.
key La clé à remplacer.
value L'id du ds_map à utiliser pour remplacer celui précédemment stocké dans la clé donnée.

Retour:

N/A

Exemple:

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

Le code ci-dessus créera un DS Map et remplissez-le avec un tableau et un DS List avant de remplacer une carte précédemment stockée dans DS Map "j_map".
