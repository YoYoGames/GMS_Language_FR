ds_map_empty

Cette fonction renvoie simplement false si la carte DS spécifiée (précédemment créée) contient des paires clé/valeur, ou true si elle n'en contient pas.

Syntaxe :

ds_map_empty(id) ;

Arguments Description
id L'identifiant de la structure de données à vérifier.

Les retours :

Booléen

Exemple :

if ds_map_empty(inventaire)
{
poids = 0 ;
}

Le code ci-dessus vérifie si la carte DS indexée dans la variable "inventory" contient des paires clé/valeur et si ce n'est pas le cas, il définit la variable "weight" à 0.
