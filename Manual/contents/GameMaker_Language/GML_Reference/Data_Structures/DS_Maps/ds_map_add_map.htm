ds_map_add_map

Cette fonction permet d'affecter une carte DS (précédemment créée) à une clé de la carte DS donnée. Cette fonction est conçue pour créer des cartes compatibles JSON que vous encoderez ensuite à l'aide de la fonction json_encode() et ne doit être utilisée qu'en conjonction avec cette fonctionnalité. Si une carte DS comporte une autre carte ajoutée de cette manière, la destruction de la carte parent détruira également les cartes contenues et libérera leur mémoire, et l'appel de la fonction ds_map_clear() sur détruira également les cartes contenues et libérera leur mémoire, et l'appel à la carte parent détruira et nettoiera toutes les cartes signalées.

Syntaxe :

ds_map_add_map(id, key, value)

Arguments Description
id L'identifiant de la carte à utiliser.
clé La clé pour la carte ajoutée.
valeur L'identifiant de la carte à ajouter.

Les retours :

N/A

Exemple :

var j_map = ds_map_create() ;
var j_list = ds_list_create() ;
var sub_map = ds_map_create() ;
ds_map_add_list(sub_map, "list", j_list) ;
ds_map_add(sub_map, "array", j_array) ;
ds_map_add_map(j_map, "map", sub_map) ;
var j = json_encode(j_map) ;
ds_map_destroy(j_map) ;

Le code ci-dessus va créer deux maps DS, puis remplir l'une d'elles avec une liste et un tableau avant de l'ajouter à la seconde, qui est ensuite encodée en une chaîne JSON. La carte est ensuite détruite pour la supprimer, ainsi que toutes les autres cartes ou listes qu'elle contient, de la mémoire.
