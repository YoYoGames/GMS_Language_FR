ds_priority_delete_value

Cette fonction supprime simplement la valeur donnée, ainsi que sa priorité, de la file d'attente indexée des priorités.

Syntaxe :

ds_priority_delete_value(id,val) ;

Arguments Description
id L'identifiant de la file d'attente prioritaire à utiliser.
val La valeur à supprimer de la file d'attente prioritaire.

Les retours :

N/A

Exemple :

if (ai_move == false)
{
ds_priority_delete_value(ai_priority, AI_Move) ;
}

Le code ci-dessus vérifie une variable d'instance et s'il renvoie false, il retire la fonction de script indexée de la file d'attente prioritaire.
