ds_priority_copy

Cette fonction peut être utilisée pour copier le contenu d'une file prioritaire dans une autre. Notez que cela ne supprime PAS le contenu de la file prioritaire d'origine, et ne détruit pas non plus la file prioritaire d'origine. Lorsque vous utilisez cette fonction, la file prioritaire doit avoir été créée au préalable et si elle contenait des éléments avant la copie, ceux-ci seront effacés en premier (ce qui signifie que ces informations seront perdues).

Syntaxe :

ds_priority_copy(id, source) ;

Arguments Description
id L'identifiant de la file d'attente prioritaire à copier.
source L'identifiant de la file d'attente prioritaire à copier.

Les retours :

N/A

Exemple :

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
p_queue = ds_priority_create() ;
ds_priority_copy(p_queue, other.p_queue) ;
}

La fonction ci-dessus crée une nouvelle instance, puis dans cette instance, elle crée une nouvelle file d'attente prioritaire DS et copie le contenu de la file d'attente prioritaire de l'instance qui exécute le bloc de code, dans la file d'attente prioritaire de l'instance nouvellement créée.
