ds_priority_find_min

Avec cette fonction, vous pouvez trouver la valeur stockée dans la file d'attente des priorités avec la priorité la plus basse, et si plus d'une valeur a la même priorité, alors n'importe laquelle d'entre elles peut être retournée dans n'importe quel ordre. Cependant, contrairement à ds_priority_delete_min(), cette fonction ne supprimera pas la valeur de la file d'attente.

Syntaxe :

ds_priority_find_min(id) ;

Arguments Description
id L'identifiant de la file d'attente prioritaire à utiliser.

Les retours :

Réel ou Chaîne

Exemple :

if ai_move
{
script_execute(ds_priority_find_min(ai_priority)) ;
}

Le code ci-dessus vérifie une variable d'instance et s'il retourne vrai, il exécutera une fonction de script indexée dans la file d'attente de priorité avec la valeur de priorité la plus basse.
