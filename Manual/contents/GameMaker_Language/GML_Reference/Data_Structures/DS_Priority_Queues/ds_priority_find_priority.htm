ds_priority_find_priority

Avec cette fonction, vous pouvez récupérer la priorité d'une valeur donnée. Si la valeur n'existe pas dans la file d'attente prioritaire, alors undefined sera retourné.

Syntaxe:

ds_priority_find_priority(id, val);

Argument Description
id L'ID de la file d'attente prioritaire à utiliser.
val La valeur pour laquelle rechercher la priorité.

Retour:

Real or undefined

Exemple:

p = ds_priority_find_priority(ai_priority, "intelligence");

Le code ci-dessus stockera la priorité renvoyée pour la valeur donnée dans la variable d'instance "p".
