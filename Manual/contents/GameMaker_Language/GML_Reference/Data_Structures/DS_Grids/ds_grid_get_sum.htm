ds_grid_get_sum

Cette fonction peut être utilisée pour ajouter toutes les valeurs de toutes les cellules trouvées dans la région définie d'une grille ensemble, comme indiqué dans l'image ci-dessous:

Syntaxe:

ds_grid_get_sum(index, x1, y1, x2, y2);

Argument Description
index L'index de la grille.
x1 La colonne de cellule gauche de la région.
y1 La rangée de cellules supérieure de la région.
x2 La colonne de cellule de droite de la région.
y2 La rangée de cellules inférieure de la région.

Retour:

Real

Exemple:

val = ds_grid_get_sum(grid, 0, 0, 5, 5)

Le code ci-dessus définira la variable «val» à la somme de toutes les valeurs contenues dans la région donnée de la grille DS indexée dans la variable «grille».
