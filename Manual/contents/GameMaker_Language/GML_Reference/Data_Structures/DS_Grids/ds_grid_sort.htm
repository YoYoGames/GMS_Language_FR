ds_grid_sort

Cette fonction peut être utilisée pour trier une grille DS en fonction des valeurs d'une colonne donnée (de la même manière que vous trieriez les fichiers par date, taille, etc. dans l'explorateur de fichiers du système d'exploitation). L'image suivante montre un exemple:

Syntaxe:

ds_grid_sort(index, column, ascending);

Argument Description
index L'index de la grille à trier.
column La colonne à utiliser pour trier les lignes
ascending S'il faut trier du plus bas au plus élevé (true ), ou du plus haut au plus bas (false ).

Retour:

N/A

Exemple:

ds_grid_sort(grid, 3, false)

Cela prendrait toutes les valeurs de la grille DS indexées dans la variable «grille» et les trierait en fonction des valeurs trouvées dans la 3ème colonne de la grille (comme indiqué dans l'image ci-dessus).
