ds_grid_width

Cette fonction retournera la largeur de la grille donnée. Cette valeur est le nombre de cellules de la grille le long de l'axe des x et est toujours un entier, comme indiqué dans l'image ci-dessous:

Syntaxe:

ds_grid_width(index);

Argument Description
index Cet index de la grille pour trouver la largeur de.

Retour:

Real

Exemple:

for (var i = 0; i < ds_grid_width(grid); ++i;)
   {
   for (var j = 0; j < ds_grid_height(grid); ++j;)
      {
      if (ds_grid_get(grid, i, j) == 1)
         {
         instance_create_layer(i * 32, j * 32, "Walls", obj_Wall);
         }
      }
   }

Le code ci-dessus va parcourir la grille DS indexée dans la variable "grid" et si la valeur trouvée dans une cellule spécifique est égale à 1, il créera alors une instance de "obj_Wall" à la position appropriée dans la pièce.
