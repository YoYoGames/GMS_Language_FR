ds_grid_set_disk

Avec cette fonction, vous pouvez définir une région circulaire d'une grille sur une certaine valeur. Vous devez fournir une cellule de grille de départ (sous forme de coordonnées des axes x et y) ainsi que le rayon du disque à définir et la valeur que vous souhaitez également définir pour le disque, comme le montre l'illustration ci-dessous:

Syntaxe:

ds_grid_set_disk(index, xm, ym, r, val);

Argument Description
index L'index de la grille.
xm La position x du disque sur la grille.
ym La position y du disque sur la grille.
r Le rayon du disque sur la grille.
val La valeur à définir avec les cellules du disque.

Retour:

N/A

Exemple:

ds_grid_set_disk(grid, ds_grid_width(grid) div 2, ds_grid_height(grid) div 2, 5, -4)

Le code ci-dessus définira une région circulaire avec un rayon de 5 cellules dans la grille DS indexée dans la variable «grille» à une valeur de -4.
