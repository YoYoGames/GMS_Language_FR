ds_grid_set_disk

Cette fonction vous permet de définir une région circulaire d'une grille à une certaine valeur. Vous devez fournir une cellule de grille de départ (en tant que coordonnée d'axe x et y) ainsi que le rayon du disque à définir et la valeur que vous souhaitez donner au disque, comme le montre l'illustration suivante comme le montre l'illustration ci-dessous :

Syntaxe :

ds_grid_set_disk(index, xm, ym, r, val) ;

Arguments Description
indice L'index de la grille.
xm La position x du disque sur la grille.
ym La position y du disque sur la grille.
r Le rayon du disque sur la grille.
val La valeur à définir avec les cellules du disque.

Les retours :

N/A

Exemple :

ds_grid_set_disk(grid, ds_grid_width(grid) div 2, ds_grid_height(grid) div 2, 5, -4)

Le code ci-dessus va définir une région circulaire avec un rayon de 5 cellules dans la grille DS indexée dans la variable "grid" à une valeur de -4.
