ds_grid_create

Avec cette fonction, vous pouvez créer une nouvelle structure de données de grille DS de la largeur et de la hauteur de cellule spécifiées. Cette fonction renvoie un identifiant qui doit être utilisé dans toutes les autres fonctions qui traitent de cette grille DS.

IMPORTANT! Lorsque vous créez une structure de données, la valeur d'index pour l'identifier est une valeur entière commençant à 0. Cela signifie que les structures de données de différents types peuvent avoir la même valeur d'index, donc en cas de doute, vous devriez utiliser la fonction ds_exists () avant y accéder. Notez également que les index sont réutilisés, de sorte qu'une valeur d'index de structure de données détruite peut être utilisée par une valeur nouvellement créée par la suite.

Syntaxe:

ds_grid_create(w, h);

Argument Description
w La largeur de la grille à créer.
h La hauteur de la grille à créer.

Retour:

Real

Exemple:

mygrid = ds_grid_create(10, 10)

Cela crée une grille de 10 cellules de haut et 10 cellules de large.
