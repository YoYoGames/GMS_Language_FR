ds_grid_set_grid_region

Cette fonction peut être utilisée pour copier le contenu d'une zone rectangulaire de cellules de grille d'une grille DS (précédemment définie) vers une autre, ou elle peut être utilisée pour copier une région à partir de la même grille. Les images suivantes illustrent les deux manières d'utiliser cette fonction:

Syntaxe:

ds_grid_set_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);

Argument Description
index L'index de la grille de destination.
source L'index de la grille source.
x1 La position gauche de la région de cellules à copier à partir de la grille source.
y1 La position supérieure de la région de cellules à copier à partir de la grille source.
x2 La bonne position de la région de cellules à copier à partir de la grille source.
y2 La position inférieure de la région de cellules à copier à partir de la grille source.
xpos Position x sur la grille de destination vers laquelle copier la région source.
ypos La position y sur la grille de destination vers laquelle copier la région source.

Retour:

N/A

Exemple:

ds_grid_set_grid_region(grid, t_grid, 0, 0, 5, 5, 10, 10)

Cela copierait la région des cellules de (0,0) à (5,5) de la grille indexée dans la variable "t_grid" et les copierait à la position (10,10) dans la grille indexée dans la variable "grid".
