ds_grid_add

Cette fonction peut être utilisée pour ajouter une valeur donnée (réel ou chaîne) à la valeur de la cellule donnée dans la grille. La valeur à ajouter doit être du même type que celle contenue dans la cellule de la grille, c'est-à-dire que vous ne pouvez pas ajouter une chaîne de caractères à un réel ou vice-versa. chaînes de caractères, cela correspond à une concaténation.

Syntaxe :

ds_grid_add(index, x, y, val) ;

Arguments Description
indice L'index de la grille.
x La position x de la cellule dans la grille.
y La position y de la cellule dans la grille.
val La valeur à ajouter à la cellule.

Les retours :

N/A

Exemple :

ds_grid_add(grid, 5, 5, 6)

Cela ajouterait 6 à la cellule donnée dans la grille DS référencée par la variable "grid".
