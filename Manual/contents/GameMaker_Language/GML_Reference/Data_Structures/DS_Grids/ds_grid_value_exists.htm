ds_grid_value_exists

Avec cette fonction, vous pouvez vérifier si une valeur spécifique (réel ou chaîne de caractères) est présente dans une zone rectangulaire d'une grille DS donnée. Si la valeur est présente, la fonction renverra true, sinon elle renverra false.

Syntaxe :

ds_grid_value_exists(index, x1, y1, x2, y2, val) ;

Arguments Description
indice L'index de la grille.
x1 La position x de la partie gauche de la région dans la grille.
y1 La position y du haut de la région dans la grille.
x2 La position x de la droite de la région dans la grille.
y2 La position y du bas de la région dans la grille.
val La valeur à trouver.

Les retours :

Booléen

Exemple :

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val) ;
ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val) ;
}

Le code ci-dessus vérifie une grille DS pour une valeur spécifique dans une région rectangulaire. Si elle est trouvée, il stocke alors la position x et y de la valeur dans deux variables pour une utilisation ultérieure.
