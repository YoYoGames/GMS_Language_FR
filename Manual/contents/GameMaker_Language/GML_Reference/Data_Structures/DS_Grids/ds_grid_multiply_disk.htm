ds_grid_multiply_disk

Cette fonction prend toutes les valeurs dans une région donnée en forme de disque de la grille DS, et multiplie chacune d'entre elles par le montant donné.

REMARQUE : Cette fonction ne fonctionne qu'avec des nombres réels, pas avec des chaînes de caractères.

Syntaxe :

ds_grid_multiply_disk(index, xm, ym, r, val) ;

Arguments Description
indice L'index de la grille.
xm La position x du disque sur la grille.
ym La position y du disque sur la grille.
r Le rayon du disque sur la grille.
val La valeur avec laquelle il faut multiplier les cellules du disque.

Les retours :

N/A

Exemple :

ds_grid_multiply_disk(mygrid, 5, 5, 5, 2)

Le code ci-dessus prend toutes les valeurs trouvées dans la zone de la grille circulaire et les multiplie par 2.
