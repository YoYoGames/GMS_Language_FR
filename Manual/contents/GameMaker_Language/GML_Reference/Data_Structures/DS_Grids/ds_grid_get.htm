ds_grid_get

Cette fonction peut être utilisée pour obtenir la valeur (soit un nombre réel, soit une chaîne de caractères) de n'importe quelle cellule de la grille DS donnée. Si vous transmettez des coordonnées de grille non valides à la fonction, la valeur renvoyée sera indéfinie et une erreur sera affichée dans la fenêtre de sortie.

Syntaxe :

ds_grid_get(index, x, y) ;

Arguments Description
indice L'index de la grille.
x La position x de la cellule dont vous voulez trouver la valeur.
y La position y de la cellule dont vous voulez trouver la valeur.

Les retours :

Réel ou Chaîne

Exemple :

var xx = irandom(ds_grid_width(grid) - 1) ;
var yy = irandom(ds_grid_height(grid) - 1) ;
val = ds_grid_get(grid, xx, yy)

Le code ci-dessus sélectionne une cellule aléatoire dans la grille DS indexée dans la variable "grid" et stocke sa valeur dans la variable "val".
