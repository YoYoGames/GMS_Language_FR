ds_queue_copy

Cette fonction peut être utilisée pour copier le contenu d'une file d'attente dans une autre. Notez que cela ne supprime PAS le contenu de la file d'origine, et ne détruit pas non plus la file d'origine. Lors de l'utilisation de cette fonction, la file d'attente copiée doit avoir été créée précédemment. précédemment créée et si elle contenait des éléments avant la copie, ceux-ci seront effacés en premier (ce qui signifie que ces informations seront perdues).

Syntaxe :

ds_queue_copy(id, source) ;

Arguments Description
id L'identifiant de la nouvelle file d'attente.
source La file d'attente ORIGINALE à copier vers id.

Les retours :

N/A

Exemple :

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
queue = ds_queue_create() ;
ds_queue_copy(queue, other.queue) ;
}

La fonction ci-dessus crée une nouvelle instance, puis dans cette instance, elle crée une nouvelle file d'attente DS et copie le contenu de la file d'attente de l'instance qui exécute le bloc de code, dans la file d'attente d'instance nouvellement créée.
