ds_queue_tail

Cette fonction ne lira que la dernière valeur de la file d'attente (celle qui est "à la queue"). Il ne retirera pas la valeur de la file d'attente, ce qui signifie qu'elle pourra encore être lue à l'avenir par cette fonction ou le ds_queue_dequeue().

Syntaxe:

ds_queue_tail(id);

Argument Description
id L'identifiant de la structure de données à partir de laquelle lire.

Retour:

Data type value stored in the queue

Exemple:

num = ds_queue_tail(control_queue);

Le code ci-dessus lira la valeur de la file d'attente indexée dans la variable "control_queue" et stockera la valeur de retour dans la variable "num".
