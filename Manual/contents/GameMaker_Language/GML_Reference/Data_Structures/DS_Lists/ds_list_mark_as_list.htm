ds_list_mark_as_list

Cette fonction "marque" (ou "signale") une position donnée dans une liste DS précédemment créée comme contenant une autre liste DS. Cette fonctionnalité est nécessaire lors de l'encodage de chaînes JSON (que vous pouvez créer à l'aide de la commande json_encode()), mais elle peut également être utile lors de l'imbrication de structures de données, car les éléments marqués de cette manière seront automatiquement récupérés (détruits) lorsque la liste DS parente sera détruite. Cela signifie que vous n'avez pas à parcourir manuellement le contenu de la liste et à détruire individuellement les structures de données marquées avant de détruire la liste "parent". Toutefois, si vous supprimez la position de la liste individuellement, la liste DS qu'elle contenait ne sera pas récupérée et vous devez appeler la fonction de destruction de la liste DS appropriée avant de la supprimer. DS appropriée avant de supprimer l'élément de la position de la liste parent. Notez également que si vous appelez la fonction ds_list_clear() sur une liste, tous les éléments marqués comme étant des listes seront également détruits lorsque la liste est effacée.

Il convient également de noter que si la position qui a été marquée change au sein de la liste, la "marque" se déplacera avec elle. Ainsi, si vous avez marqué la position 3 dans la liste (par exemple) et que vous insérez ensuite 2 éléments supplémentaires sous cette position pour qu'elle passe à la position 5, elle restera marquée comme une liste. 5, la liste sera toujours marquée comme une liste.

Syntaxe :

ds_list_mark_as_list(id, pos) ;

Arguments Description
id L'identifiant de la liste à marquer.
pos La position à marquer dans la liste.

Les retours :

Real (entier - ID de la liste DS si elle est réussie ou -1 si elle échoue)

Exemple :

var j_list = ds_list_create() ;
var sub_list = ds_list_create() ;
ds_list_add(sub_list, health) ;
ds_list_add(sub_list, lives) ;
ds_list_add(sub_list, score) ;
ds_list_add(j_list, sub_list) ;
ds_list_mark_as_list(j_list, 0) ;

Le code ci-dessus crée deux listes DS, puis en remplit une avec diverses valeurs provenant de variables globales. Cette liste est ensuite ajoutée à la deuxième liste, et la position est "marquée" comme telle afin qu'elle puisse être correctement encodée par la suite.
