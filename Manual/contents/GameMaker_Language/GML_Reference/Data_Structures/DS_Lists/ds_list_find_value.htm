ds_list_find_value

Avec cette fonction, vous pouvez vérifier la position de la liste donnée et la valeur contenue dans la liste pour cette position sera retournée. Notez que si vous donnez une position qui est en dehors de la taille de la liste donnée (par exemple, la position 11 dans une liste de 10 valeurs), la fonction peut retourner indéfini ou 0. peut retourner undefined ou 0. Ceci est dû au fait que lorsque vous créez la liste, les premières entrées de la liste sont mises à 0 pour minimiser les problèmes de performance lors de l'ajout initial d'éléments à la liste (bien que la fonction ds_list_size() retournera toujours 0 sur une liste nouvellement créée). Si vous souhaitez vous assurer que la liste est "réellement" vide lors de sa création, vous devez appeler la fonction ds_list_clear() après avoir créé la liste, ce qui signifie que signifie que toutes les valeurs renvoyées pour les emplacements de liste non remplis seront indéfinies, ce que vous pouvez ensuite vérifier de manière cohérente en utilisant la fonction is_undefined() fonction.

Syntaxe :

ds_list_find_value(id, pos) ;

Arguments Description
id L'identifiant de la liste à utiliser.
pos La position à regarder, où 0 correspond au tout début de la liste et la position finale est ds_list_size(id)-1.

Les retours :

Réel, chaîne ou indéfini

Exemple :

val = ds_list_find_value(list, ds_list_size(list) - 1) ;

Le code ci-dessus vérifie la liste indexée dans la variable "list" à la dernière position dans la liste et stocke la valeur retournée dans la variable "val".
