ds_list_find_index

Avec cette fonction, vous pouvez vérifier la présence d'une valeur dans la liste donnée et la position de cette valeur dans la liste sera renvoyée. Notez que s'il y a plus d'une entrée dans la liste avec la même valeur, la position de n'importe laquelle d'entre elles peut être retournée, et que si la valeur n'existe pas, alors -1 sera renvoyé. Notez que la valeur peut être un tableau, que vous pouvez vérifier avec la fonction is_array().

Syntaxe :

ds_list_find_index(id, val) ;

Arguments Description
id L'identifiant de la liste à utiliser.
val La valeur à trouver.

Les retours :

Real

Exemple :

pos = ds_list_find_index(list, "Player1") ;

Le code ci-dessus vérifie la liste indexée dans la variable "list" pour la valeur "Player1" et stocke la position retournée dans la variable "pos".
