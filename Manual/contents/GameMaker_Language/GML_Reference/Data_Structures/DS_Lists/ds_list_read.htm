ds_list_read

Avec cette fonction, vous pouvez recréer une liste de DS sauvegardée (une liste qui a été précédemment écrite sous forme de chaîne de caractères à l'aide de la fonction ds_list_write()). Vous devez d'abord créer une nouvelle liste de DS pour y lire la chaîne de caractères. existe déjà et contient des informations, celles-ci seront effacées avant la lecture. Cette fonction est d'une importance vitale lors de la création de mécanismes de sauvegarde/chargement pour votre jeu. GameMaker vous devez ajouter l'argument optionnel "legacy", en le mettant à true car le format de la chaîne a changé avec cette version.

Syntaxe :

ds_list_read(id, str [, legacy]) ;

Arguments Description
id L'identifiant de la structure de données à lire.
str La chaîne de caractères à lire.
héritage(facultatif) Peut être soit vrai, soit faux, soit complètement omis.

Les retours :

N/A

Exemple :

list = ds_list_create() ;
ini_open("save.ini") ;
var str = ini_read_string("Lists", "0", "") ;
if str != ""
{
ds_list_read(list, str) ;
}
ini_close() ;

Le code ci-dessus crée une liste et stocke l'index dans la variable "list". Il ouvre ensuite un fichier ini et lit une chaîne de caractères à partir de celui-ci, en vérifiant d'abord que la chaîne n'est pas renvoyée comme vide. Cette chaîne est ensuite lue dans la liste ds_list nouvellement créée.
