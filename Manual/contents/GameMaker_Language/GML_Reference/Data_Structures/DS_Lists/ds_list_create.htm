ds_list_create

Cette fonction créera une nouvelle structure de données de liste et retournera la valeur d'index. Cette valeur doit être stockée dans une variable et utilisée dans tous les autres appels de fonction relatifs à la liste. Notez que si vous essayez d'accéder à une valeur dans la liste avant qu'elle n'ait été remplie - c'est-à-dire que la liste est "vide" - alors la valeur de retour peut être 0 car en interne les premières entrées de la liste sont définies lors de leur création pour minimiser les problèmes de performances lors de l'ajout initial d'éléments à la liste (bien que le ds_list_size() la fonction retournera toujours 0 sur une liste nouvellement créée). Si vous souhaitez vous assurer que la liste est "vraiment" vide lors de la création, vous devez appeler ds_list_clear() après avoir créé la liste, ce qui signifiera alors que toutes les valeurs renvoyées pour les emplacements de liste non remplis seront undefined.

IMPORTANT! Lorsque vous créez une structure de données, la valeur d'index pour l'identifier est une valeur entière commençant à 0. Cela signifie que les structures de données de différents types peuvent avoir la même valeur d'index, donc en cas de doute, vous devriez utiliser la fonction ds_exists () avant y accéder. Notez également que les index sont réutilisés, de sorte qu'une valeur d'index de structure de données détruite peut être utilisée par une valeur nouvellement créée par la suite.

Syntaxe:

ds_list_create();

Retour:

Real

Exemple:

list = ds_list_create();

Cela créera une nouvelle liste et attribuera son identifiant d'index à la variable d'instance "list".
