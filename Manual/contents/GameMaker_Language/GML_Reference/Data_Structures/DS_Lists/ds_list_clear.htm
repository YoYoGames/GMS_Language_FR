ds_list_clear

Avec cette fonction, vous pouvez effacer toutes les données de la structure de données de la liste donnée. Cela ne détruit PAS la structure de données (pour cela vous devriez utiliser ds_list_destroy()), elle efface seulement toutes les données de celle-ci et rend la liste vide (de taille nulle). Notez que l'effacement d'une liste déréférencera toutes les structures de données qui y sont stockées, ce qui entraînera une fuite de mémoire. pour éviter cela. Le seul cas où cela n'est pas nécessaire est celui où vous avez marqué des éléments de la liste comme étant une autre liste DS ou une carte DS. et leur mémoire sera nettoyée automatiquement lorsque le parent sera effacé.

Syntaxe :

ds_list_clear(id) ;

Arguments Description
id L'identifiant de la structure de données à effacer.

Les retours :

N/A

Exemple :

if (count == 15) && (!ds_list_empty(command_list))
{
ds_list_clear(command_list) ;
alarm[0] = room_speed ;
ai_count = 0 ;
}

Le code ci-dessus vérifie une variable pour voir si elle a atteint une valeur spécifique et si c'est le cas, il efface la liste DS indexée dans la variable "command_list", déclenche une alarme et remet la variable à 0.
