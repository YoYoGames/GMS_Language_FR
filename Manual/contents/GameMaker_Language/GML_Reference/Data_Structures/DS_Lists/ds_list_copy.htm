ds_list_copy

Cette fonction permet de copier le contenu d'une liste dans une autre. Les deux listes doivent avoir été créées précédemment et si la liste à copier contient déjà des informations, elle sera effacée en premier. Le résultat final est deux listes indépendantes indépendantes qui contiennent les mêmes informations.

Syntaxe :

ds_list_copy( id, source ) ;

Arguments Description
id L'identifiant de la liste à copier.
source L'identifiant de la liste à copier.

Les retours :

N/A

Exemple :

if !ds_list_empty(main_list)
{
old_list = ds_list_create() ;
ds_list_copy(old_list, main_list) ;
ds_list_clear(main_list) ;
}

Le code ci-dessus vérifie une liste DS pour voir si elle est vide. Si elle n'est pas vide, elle est copiée dans une autre liste DS (qui a été créée précédemment), puis la liste originale est effacée.
