ds_list_sort

Avec cette fonction, vous pouvez trier toutes les valeurs d'une liste, par ordre croissant ou décroissant. Si la liste contient des chaînes, celles-ci seront triées par ordre alphabétique, en fonction de l'alphabet anglais de 26 lettres.

Syntaxe:

ds_list_sort(id, ascend);

Argument Description
id L'identifiant de la liste à trier.
ascend Si les valeurs doivent être ascendantes (true ) ou décroissant (false ) ordre.

Retour:

N/A

Exemple:

if newgame
    {
    ds_list_sort(name_list, true);
    }

Le code ci-dessus triera la liste indexée dans la variable "name_list" si la variable "newgame" est marquée comme true.
