ds_stack_empty

Avec cette fonction, vous pouvez vérifier la pile DS donnée pour voir si elle est vide (renvoie true) ou non (renvoie false).

Syntaxe :

ds_stack_empty(id) ;

Arguments Description
id L'identifiant de la structure de données à vérifier.

Les retours :

Booléen

Exemple :

if (ai_count == 15) && (!ds_stack_empty(AI_stack))
{
ds_stack_clear(AI_stack) ;
alarm[0] = room_speed ;
ai_count = 0 ;
}

Le code ci-dessus vérifie une variable pour voir si elle a atteint une valeur spécifique et si c'est le cas, il efface la pile DS indexée dans la variable "AI_stack", déclenche une alarme et remet la variable à 0.
