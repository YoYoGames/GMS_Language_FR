ds_stack_top

Cette fonction ne lira que la première valeur de la pile (celle qui est "en haut"). La valeur n'apparaîtra pas, ce qui signifie qu'elle pourra encore être lue à l'avenir par cette fonction ou le ds_stack_pop(). Si la pile est vide, la fonction retournera la constante undefined, sinon il renverra la valeur réelle ou chaîne contenue dans la pile.

Syntaxe:

ds_stack_top(id);

Argument Description
id L'identifiant de la structure de données à partir de laquelle lire.

Retour:

Data type value stored in the stack or undefined

Exemple:

num = ds_stack_top(control_stack);

Le code ci-dessus lira la valeur de la pile indexée dans la variable "control_stack" et stockera la valeur de retour dans la variable "num".
