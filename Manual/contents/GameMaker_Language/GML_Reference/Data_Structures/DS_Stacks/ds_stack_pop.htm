ds_stack_pop

Cette fonction extrait la valeur supérieure de la pile DS, la supprime de la pile et renvoie la valeur à stocker dans une variable. Si la pile est vide, la fonction renvoie la constante indéfinie, sinon elle renvoie la valeur réelle ou la chaîne de caractères contenue dans la pile. la valeur réelle ou la chaîne de caractères contenue dans la pile.

Syntaxe :

ds_stack_pop(id) ;

Arguments Description
id L'identifiant de la structure de données à extraire.

Les retours :

Valeur du type de données qui est stockée dans la pile ou indéfinie

Exemple :

if !ds_stack_empty(move_stack)
{
var xx = ds_stack_pop(move_stack) ;
var yy = ds_stack_pop(move_stack) ;
move_towards_point(xx, yy, 4) ;
}

Le code ci-dessus vérifie la pile DS indexée dans la variable "move_stack" pour voir si elle est vide, et si elle ne l'est pas, il extrait les deux valeurs supérieures de la pile et les utilise pour définir une direction pour le mouvement.
