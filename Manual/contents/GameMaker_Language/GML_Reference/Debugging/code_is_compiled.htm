code_is_compiled

Cette fonction retournera true si la plate-forme compile en dehors de la machine virtuelle, comme pour les plates-formes YYC et JS.

Syntaxe:

code_is_compiled();

Retour:

Boolean

Exemple:

if code_is_compiled()
   {
   show_debug_message("Compiler okay!");
   }

Le code ci-dessus vérifie si le jeu a été compilé à l'aide du YoYo et affiche un message de débogage s'il l'était.

Retour: Débogage
