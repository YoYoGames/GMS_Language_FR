code_est_compilé

Cette fonction retournera vrai si la plateforme compile en dehors de la machine virtuelle, comme pour les plateformes YYC et JS.

Syntaxe :

code_is_compiled() ;

Les retours :

Booléen

Exemple :

if code_is_compiled()
{
show_debug_message("Compilateur ok !") ;
}

Le code ci-dessus vérifie si le jeu a été compilé avec le compilateur YoYo et affiche un message de débogage si c'est le cas.

Retour : Débogage
