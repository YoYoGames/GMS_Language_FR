clickable_exists

Cette fonction retourne si une icône DOM cliquable a été créée avec l'indice spécifié existe ou non. Veuillez noter que la valeur utilisée pour la vérification doit avoir été initialisée précédemment, sinon vous obtiendrez une erreur entraînant la fermeture de GameMaker Studio 2.

Syntaxe :

clickable_exists(index) ;

Arguments Description
indice L'index du cliquable à vérifier.

Les retours :

Booléen

Exemple :

if !clickable_exists(home_but)
{
home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0") ;
}

Le code ci-dessus vérifie la variable "home_but" pour voir si elle existe déjà, et si ce n'est pas le cas, il crée une icône DOM cliquable à la position (32, 32) de la page sur laquelle le canevas de jeu est exécuté. L'icône utilise le sprite référencé dans la page la page de texture en tant que "spr_MS_Home" et, lorsqu'elle est cliquée, l'icône ouvre une nouvelle fenêtre pour l'URL spécifiée et avec les propriétés définies pour cette fenêtre.
