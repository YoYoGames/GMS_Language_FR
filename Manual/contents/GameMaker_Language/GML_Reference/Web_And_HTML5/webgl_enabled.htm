webgl_enabled

Cette variable en lecture seule indique si WebGL est activé(true) ou non(false) pour votre jeu. Elle ne fonctionnera que pour les jeux fonctionnant via un navigateur (c'est-à-dire HTML5), et pour toutes les autres plateformes, elle renverra true.

Syntaxe :

webgl_enabled

Les retours :

Booléen

Exemple :

if webgl_enabled
{
global.quality = 1 ;
}
else global.quality = 0 ;

Le code ci-dessus vérifie l'indicateur WebGL et définit ensuite la variable globale "qualité" en conséquence.
