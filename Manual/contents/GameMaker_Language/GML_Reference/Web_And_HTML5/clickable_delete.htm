clickable_delete

Cette fonction doit être utilisée pour supprimer une icône cliquable précédemment créée avec clickable_add() de la fenêtre de jeu.

Syntaxe :

clickable_delete(index) ;

Arguments Description
indice Index de l'icône cliquable à supprimer.

Les retours :

N/A

Exemple :

if clickable_exists(global.Help_Icon)
{
clickable_delete(global.Help_Icon) ;
}

Le code ci-dessus supprime l'icône cliquable indexée dans la variable globale "Help_Icon" de la fenêtre du jeu, si elle existe.
