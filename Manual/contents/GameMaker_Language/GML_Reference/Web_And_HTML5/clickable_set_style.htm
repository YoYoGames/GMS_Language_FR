clickable_set_style

Cette fonction vous permet de définir les propriétés de style CSS pour le bouton donné via les paires clé / valeur dans le DS Map. Vous devez avoir créé au préalable l'élément bouton (en utilisant clickable_add() ) et le DS Map et fournissez les indices stockés à chacun comme arguments.

Syntaxe:

clickable_set_style(index, map)

Argument Description
index L'index de l'icône cliquable à styliser.
map L'index du DS Map pour définir le style à partir de.

Retour:

N/A

Exemple:

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

Le code ci-dessus changera le style du bouton cliquable avec l'index stocké dans la variable "bouton".
