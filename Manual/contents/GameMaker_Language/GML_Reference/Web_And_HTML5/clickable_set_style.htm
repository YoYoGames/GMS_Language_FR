Style de jeu cliquable

Cette fonction vous permet de définir les propriétés de style CSS pour le bouton donné via les paires clé/valeur dans le fichier Carte DS. Vous devez avoir préalablement créé l'élément bouton (à l'aide de la fonction clickable_add()) et la carte DS et fournir les indices stockés pour chacun d'eux comme arguments.

Syntaxe :

clickable_set_style(index, map)

Arguments Description
indice L'index de l'icône cliquable à styliser.
carte L'index de la carte DS pour définir le style.

Les retours :

N/A

Exemple :

var map = ds_map_create() ;
ds_map_add(map, "opacity", "0.5") ;
clickable_set_style(button, map) ;
ds_map_destroy() ;

Le code ci-dessus changera le style du bouton cliquable avec l'indice stocké dans la variable "bouton".
