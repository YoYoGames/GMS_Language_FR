url_open

Cela ouvrira l'URL spécifiée sur le navigateur de l'appareil cible choisi ou, si vous utilisez le module HTML5, dans le navigateur actuellement ouvert.

Syntaxe :

url_open(url) ;

Arguments Description
url L'URL (adresse du site web) vers lequel le lien doit être établi.

Les retours :

N/A

Exemple :

url_open("http://yoyogames.com") ;

Cela ouvrirait la page d'accueil de YoYo Games dans la fenêtre actuelle.
