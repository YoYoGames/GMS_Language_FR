clickable_add

Avec cette fonction, vous pouvez créer une icône personnalisée, cliquable, qui agit comme un "bouton" en dehors du canevas du jeu, mais dans la fenêtre du navigateur elle-même. La fonction renverra une valeur d'identifiant qui doit être stockée dans une variable pour être utilisée dans toutes les autres fonctions de bouton faisant référence à la nouvelle icône. Cette fonction est très utile pour empêcher le navigateur de créer une fenêtre contextuelle en cliquant sur le bouton car tout est fait via GameMaker Studio 2 lui-même.

Vous devez d'abord définir la position, qui est relative au (0,0) de la fenêtre du navigateur (coin supérieur gauche), puis donner l'entrée de page de texture pour le sprite que vous souhaitez que l'icône du bouton ait (voir:sprite_get_tpe() ). Vous pouvez ensuite spécifier une URL à ouvrir ou un script (à partir du navigateur de GameMaker Studio 2 ) à ouvrir ou à exécuter. Le paramètre "cible" que vous spécifiez est le même que la valeur "nom" JavaScript standard lorsque vous utilisez le open() (sachez que tout sauf '_self' peut entraîner le blocage du navigateur, ou demander à l'utilisateur s'il souhaite l'autoriser) et l'argument "params" est le même que le paramètre JavaScript "specs" pour contrôler les propriétés du nouveau la fenêtre / l'onglet doit s'afficher (tous les navigateurs ne prennent pas en charge toutes les fonctionnalités) Si vous utilisez l'icône pour exécuter un script interne, ces deux derniers arguments peuvent être laissés sous forme de chaînes vides "".

Les cibles valides sont:

Cibler Description
_Vide Ouvre le document lié dans une nouvelle fenêtre ou un nouvel onglet.
_soi Ouvre le document lié dans le même cadre dans lequel il a été cliqué (il s'agit de la valeur par défaut).
_parent Ouvre le document lié dans le cadre parent.
_Haut Ouvre le document lié dans le corps entier de la fenêtre.

Les paramètres valides sont:

Paramètres Description
'height = [ px ]' La hauteur de la fenêtre, la valeur minimale étant 100.
'width = [ px ]' La largeur de la fenêtre, la valeur minimale étant 100.
"gauche = [ px ]" La position gauche de la fenêtre.
"top = [ px ]" La position supérieure de la fenêtre (IE uniquement).
'location = [ booléen ]' Afficher ou non le champ d'adresse (la valeur par défaut est 1).
'menubar = [ booléen ]' Afficher ou non la barre de menus (la valeur par défaut est 1).
'redimensionnable = [ booléen ]' Si la fenêtre est redimensionnable ou non (la valeur par défaut est 1).
'scrollbars = [ booléen ]' Afficher ou non les barres de défilement (la valeur par défaut est 1).
'status = [ booléen ]' Ajout ou non d'une barre d'état (la valeur par défaut est 1).
'titlebar = [ booléen ]' Afficher ou non la barre de titre. Ceci est ignoré sauf si l'application appelante est une application HTML ou une boîte de dialogue approuvée (la valeur par défaut est 1);
'toolbar = [ booléen ]' Afficher ou non la barre d'outils du navigateur (par défaut, oui).


Il est également intéressant de noter que si l'argument URL que vous passez est le nom d'une fonction de script (c'est-à-dire qu'il ne commence pas par "http: //") ET que le script est appelé gmcallback_* alors vous pouvez obtenir des flux directs dans un événement html5. Les noms des fonctions de script GameMaker Studio 2 ne sont pas masqués s'ils commencent par gmcallback_.

Syntaxe:

clickable_add(x, y, tpe, url, target, params);

Argument Description
x La position x dans la fenêtre.
y La position y dans la fenêtre.
tpe L'entrée de la page de texture pour le sprite à utiliser.
url L'URL (adresse du site Web) vers laquelle créer un lien ou le script à exécuter.
target Il s'agit de la zone cible dans laquelle ouvrir l'URL (voir description).
params Les différents paramètres de l'icône (voir description).

Retour:

Real

Exemple:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Le code ci-dessus crée une icône DOM cliquable à la position (32, 32) de la page sur laquelle le canevas de jeu s'exécute. L'icône utilise le sprite référencé à partir de la page de texture comme "spr_MS_Home" et lorsque vous cliquez dessus, l'icône ouvrira une nouvelle fenêtre pour l'URL spécifiée et avec les propriétés définies pour cette fenêtre.

Retour: Web et HTML5
