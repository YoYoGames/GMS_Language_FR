événement_analytique

Cette fonction enverra le texte spécifié au fournisseur de données analytiques que vous avez configuré dans les options de jeu HTML5. Cette fonction peut être utilisée pour créer une analyse personnalisée afin de suivre quelque chose en dehors de la portée du fournisseur utilisé. en dehors de la portée du fournisseur utilisé.

Syntaxe :

analytics_event(string) ;

Arguments Description
chaîne de caractères Une chaîne à envoyer au fournisseur

Les retours :

N/A

Exemple :

if hs_new
{
analytics_event("Nouveau hiscore de " + string (score)) ;
}

Le code ci-dessus vérifiera une variable pour voir si elle est vraie et si elle l'est, un événement analytique spécial sera déclenché avec la chaîne spécifiée.
