url_open_full

Cela ouvrira l'URL spécifiée sur le navigateur de l'appareil cible choisi ou, si vous utilisez le module HTML5, dans le navigateur actuellement ouvert. Le paramètre "cible" que vous spécifiez est le même que la valeur "nom" JavaScript standard lorsque vous utilisez le open() (sachez que tout sauf " _self " peut entraîner le blocage du navigateur, ou demander à l'utilisateur s'il souhaite l'autoriser) et le paramètre "options" est le même que le paramètre JavaScript "specs" pour contrôler les propriétés de la nouvelle fenêtre / tab doit s'afficher (tous les navigateurs ne prennent pas en charge toutes les fonctionnalités).

Les cibles valides sont:

Cibler Description
_Vide Ouvre le document lié dans une nouvelle fenêtre ou un nouvel onglet (cela ne fonctionnera pas si les fenêtres contextuelles sont bloquées par l'utilisateur, auquel cas vous pouvez utiliser le clickable_* fonctions à la place).
_soi Ouvre le document lié dans le même cadre dans lequel il a été cliqué (il s'agit de la valeur par défaut).
_parent Ouvre le document lié dans le cadre parent.
_Haut Ouvre le document lié dans le corps entier de la fenêtre.

Les options valides sont:

Paramètres Description
'height = [ px ]' La hauteur de la fenêtre, la valeur minimale étant 100.
'width = [ px ]' La largeur de la fenêtre, la valeur minimale étant 100.
"gauche = [ px ]" La position gauche de la fenêtre.
"top = [ px ]" La position supérieure de la fenêtre (IE uniquement).
'location = [ booléen ]' Afficher ou non le champ d'adresse (la valeur par défaut est 1).
'menubar = [ booléen ]' Afficher ou non la barre de menus (la valeur par défaut est 1).
'redimensionnable = [ booléen ]' Si la fenêtre est redimensionnable ou non (la valeur par défaut est 1).
'scrollbars = [ booléen ]' Afficher ou non les barres de défilement (la valeur par défaut est 1).
'status = [ booléen ]' Ajout ou non d'une barre d'état (la valeur par défaut est 1).
'titlebar = [ booléen ]' Afficher ou non la barre de titre. Ceci est ignoré sauf si l'application appelante est une application HTML ou une boîte de dialogue approuvée (la valeur par défaut est 1);
'toolbar = [ booléen ]' Afficher ou non la barre d'outils du navigateur (par défaut, oui).

Syntaxe:

url_open_full(url, target, options);

Argument Description
url L'URL (adresse du site Web) vers laquelle créer un lien.
target Il s'agit de la zone cible dans laquelle ouvrir l'URL (voir description).
options Options de navigateur standard (voir description).

Retour:

N/A

Exemple:

url_open_full("http://yoyogames.com", "_blank", "resizable=0, height=200, scrollbars=0");

Cela ouvrirait la YoYo Games dans une nouvelle fenêtre qui ne peut pas être redimensionnée, a une hauteur de 200 pixels et aucune barre de défilement.
