url_open_ext

Cela ouvrira l'URL spécifiée sur le navigateur de l'appareil cible choisi ou, si vous utilisez le module HTML5, dans le navigateur actuellement ouvert. Le paramètre "target" que vous spécifiez est le même que la valeur standard JavaScript "name" lorsque vous utilisez la méthode open(). standard de JavaScript lorsque vous utilisez la méthode open(), et vous devez savoir que tous les paramètres, à l'exception de " _self", peuvent entraîner le blocage du navigateur ou demander à l'utilisateur s'il souhaite l'autoriser.

Les cibles valides sont :

Cible Description
_blank Ouvre le document lié dans une nouvelle fenêtre ou un nouvel onglet (cela ne fonctionnera pas si les fenêtres pop-up sont bloquées par l'utilisateur, auquel cas vous pouvez utiliser les fonctions cliquable_* à la place).
_self Ouvre le document lié dans le même cadre que celui dans lequel il a été cliqué (c'est la valeur par défaut).
_parent Ouvre le document lié dans le cadre parent.
_haut Ouvre le document lié dans le corps complet de la fenêtre.

Syntaxe :

url_open_ext(url, cible) ;

Arguments Description
url L'URL (adresse du site web) vers lequel le lien doit être établi.
cible Il s'agit de la zone cible dans laquelle il faut ouvrir l'URL.

Les retours :

N/A

Exemple :

url_open_ext("http://yoyogames.com", "_blank") ;

Cela ouvrira la page d'accueil de YoYo Games dans une nouvelle fenêtre.
