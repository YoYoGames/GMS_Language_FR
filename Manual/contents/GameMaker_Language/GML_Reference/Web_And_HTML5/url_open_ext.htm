url_open_ext

Cela ouvrira l'URL spécifiée sur le navigateur de l'appareil cible choisi ou, si vous utilisez le module HTML5, dans le navigateur actuellement ouvert. Le paramètre "cible" que vous spécifiez est le même que la valeur "nom" JavaScript standard lorsque vous utilisez le open() et vous devez être conscient que tout sauf " _self " peut entraîner le blocage du navigateur, ou demander à l'utilisateur s'il souhaite l'autoriser.

Les cibles valides sont:

Cibler Description
_Vide Ouvre le document lié dans une nouvelle fenêtre ou un nouvel onglet (cela ne fonctionnera pas si les fenêtres contextuelles sont bloquées par l'utilisateur, auquel cas vous pouvez utiliser le clickable_* fonctions à la place).
_soi Ouvre le document lié dans le même cadre dans lequel il a été cliqué (il s'agit de la valeur par défaut).
_parent Ouvre le document lié dans le cadre parent.
_Haut Ouvre le document lié dans le corps entier de la fenêtre.

Syntaxe:

url_open_ext(url, target);

Argument Description
url L'URL (adresse du site Web) vers laquelle créer un lien.
target Il s'agit de la zone cible dans laquelle ouvrir l'URL.

Retour:

N/A

Exemple:

url_open_ext("http://yoyogames.com", "_blank");

Cela ouvrirait la YoYo Games dans une nouvelle fenêtre.
