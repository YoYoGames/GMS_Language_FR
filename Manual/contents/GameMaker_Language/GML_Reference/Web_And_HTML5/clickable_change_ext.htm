Texte de changement cliquable

Avec cette fonction, vous pouvez changer le sprite et la position d'une icône cliquable précédemment créée avec clickable_add(). Gardez à l'esprit que la position est basée sur la position (0,0) de la fenêtre et non du canevas. et que le sprite doit être référencé directement à partir de la page de texture (voir : sprite_get_tpe()). Cette fonction vous permet également de modifier l'échelle du sprite utilisée (sous forme de multiplicateur, de sorte que 1 est la valeur par défaut, 0,5 la moitié et 2 le double) et la valeur alpha de 0 (totalement transparent) à 1 (totalement opaque) pour l'icône finale à l'écran.

Syntaxe :

clickable_change_ext(index, tpe, x, y, alpha, scale)

Arguments Description
indice L'indice de l'icône cliquable à modifier.
tpe L'entrée de la page de texture pour le sprite à utiliser.
x La nouvelle position x dans la fenêtre.
y La nouvelle position y dans la fenêtre.
échelle L'échelle de l'icône (par défaut 1).
alpha L'alpha de l'image de l'icône (par défaut 1).

Les retours :

N/A

Exemple :

switch (room)
{
case rm_Menu : clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1) ; break ;
case rm_Game : clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5) ; break ;
}

Le code ci-dessus changera l'index de l'image et la position de l'icône cliquable indexée dans la variable "global.Help" en fonction de la pièce actuelle, en modifiant également l'échelle et l'alpha de l'icône.
