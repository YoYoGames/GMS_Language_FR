largeur du navigateur

Cette variable contient la largeur (en pixels) du navigateur dans lequel le jeu est exécuté. Si aucun navigateur n'est présent, la taille de la fenêtre est renvoyée. L'image suivante illustre la relation entre cette variable et le reste des fonctions de positionnement disponibles par le biais des fonctions de la fenêtre.

Syntaxe :

largeur du navigateur

Les retours :

Real

Exemple :

si (browser_height > window_get_height()) || (browser_width > window_get_width())
{
var xx, yy ;
if (browser_width > window_get_width())
{
xx = (largeur du navigateur - largeur de la fenêtre_get_width()) / 2 ;
}
else
{
xx = 0 ;
}
if (browser_height > window_get_height())
{
yy = (browser_height - window_get_height()) / 2 ;
}
else
{
yy = 0 ;
}
window_set_position(xx, yy) ;
}

Le code ci-dessus vérifie la largeur et la hauteur du navigateur pour voir si elles sont supérieures à la largeur et à la hauteur de la fenêtre actuelle. Si c'est le cas, il calcule alors une nouvelle position pour la fenêtre de jeu dans le navigateur.
