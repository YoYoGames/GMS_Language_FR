test_physique_chevauchement

Cette fonction peut être utilisée pour vérifier si un corps physique (c'est-à-dire l'objet fixe d'une instance) se chevauche, ou se chevauchera, lorsqu'il est tourné et placé à une position donnée dans la pièce. l'argument "angle" est l'angle de rotation que l'instance appelante a (ou aura) à la position à vérifier. angle de rotation que l'instance appelante a (ou aura) à la position à vérifier, et l'argument "obj" peut être soit un seul identifiant d'instance, et un index d'objet, soit l'un ou l'autre. mots-clés tous ou autre.

Syntaxe :

physics_test_overlap(xpos, ypos, angle, obj) ;

Arguments Description
xpos La position x dans la pièce à vérifier
ypos La position y dans la pièce à vérifier
angle L'angle à vérifier (de l'instance appelante)
obj. L'objet à vérifier

Les retours :

Booléen

Exemple :

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
alarme[0] = room_speed ;
ignited = true ;
}

Le code ci-dessus vérifie qu'il n'y a pas de chevauchement entre les appareils de physique dans une position et, s'il y en a un, il définit une variable et une alarme.
