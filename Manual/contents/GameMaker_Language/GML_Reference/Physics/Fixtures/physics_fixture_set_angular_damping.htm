physics_fixture_set_angular_damping

Si vous pensez à un objet en rotation dans le «monde réel», à moins qu'il n'ait un moteur ou qu'il ne soit dans l'espace, il ralentit avec le temps en raison de l'influence de forces externes (comme le frottement avec l'air qui l'entoure). Nous pouvons utiliser la fonction physics_fixture_set_angular_damping() dans GameMaker Studio 2 pour simuler cet effet et réduire la vitesse de rotation des instances dans le monde de la physique, car sans elle, toute instance en rotation continuerait à tourner à l'infini. Les paramètres d'amortissement doivent être compris entre 0 et l'infini, 0 signifiant aucun amortissement et l'infini signifiant un amortissement complet. Normalement, vous utiliserez une valeur d'amortissement comprise entre 0 et 1, mais vous pouvez utiliser n'importe quelle valeur non négative si nécessaire.

Syntaxe:

physics_fixture_set_angular_damping(fixture, damping)

Argument Description
fixture l'index du luminaire
damping l'amortissement angulaire du luminaire, généralement compris entre 0 et 1

Retour:

N/A

Exemple:

physics_fixture_set_angular_damping(fix_Ball, 0.1);

Le code ci-dessus définira l'amortissement angulaire de l'appareil indexé dans "fix_ball" à 0,1.
