physique_get_restitution

Lorsque vous liez un dispositif à une instance à l'aide de la méthode physics_fixture_bind() cela renvoie un "id" pour le fixture lié. Vous pouvez utiliser cet identifiant pour obtenir la restitution (la propriété "bounciness") du dispositif lié, et non du dispositif "de base", à tout moment en utilisant cette fonction.

Syntaxe :

physics_get_restitution(fixture)

Arguments Description
L'appareil l'identifiant du dispositif lié

Les retours :

Real

Exemple :

var rest = physics_get_restitution(fix_id) ;
physics_set_restitution(fix_id, rest * 2) ;

Le code ci-dessus obtient la valeur de restitution actuelle pour les propriétés physiques liées de l'instance, puis les définit à une valeur différente.
