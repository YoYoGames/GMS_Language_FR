physics_fixture_set_polygon_shape

Cette fonction définit la forme d'un polygone pour votre appareil, mais vous devrez utiliser la fonction physics_fixture_add_point() pour définir la forme de ce polygone par rapport à l'origine de l'appareil. Le polygone est fermé lorsque le dispositif est lié à une instance. Vous devez également noter que cette fonction doit être appelée avant de définir des points, et que vous devez avoir défini au moins trois points pour votre polygone avant de le lier à une instance, sinon vous obtiendrez une erreur.

Syntaxe :

physics_fixture_set_polygon_shape(fixture)

Arguments Description
L'appareil l'indice du dispositif

Les retours :

N/A

Exemple :

physics_fixture_set_polygon_shape(fix_Ship) ;
physics_fixture_add_point(fix_Ship, 0,0) ;
physics_fixture_add_point(fix_Ship, -40, 100) ;
physics_fixture_add_point(fix_Ship, 40, 100) ;

Le code ci-dessus applique une forme polygonale au dispositif indexé dans la variable "fix_Ship", puis définit trois points pour créer une forme triangulaire.
