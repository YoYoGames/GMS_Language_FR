physics_fixture_set_awake

Lorsque vous liez un dispositif à une instance à l'aide de la méthode physics_fixture_bind() cela renvoie un "id" pour le dispositif lié. Vous pouvez utiliser cet identifiant pour obtenir la valeur de friction du dispositif lié(et non du dispositif "de base") à tout moment en utilisant cette fonction.

Syntaxe :

physics_get_friction(fixture)

Arguments Description
L'appareil l'identifiant du dispositif lié

Les retours :

Real

Exemple :

var fric = physics_get_friction(fix_id) ;
physics_set_friction(fix_id, fric + 0.1) ;

Le code ci-dessus obtient la valeur de friction actuelle pour les propriétés physiques liées de l'instance, puis les définit à une valeur différente.
