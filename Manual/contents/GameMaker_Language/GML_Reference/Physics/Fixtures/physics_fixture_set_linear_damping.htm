physics_fixture_set_linear_damping

L'amortissement est utilisé pour réduire la vitesse du monde physique des instances, et diffère du frottement en ce que le frottement ne se produit que lorsque deux instances avec des fixations sont en contact. L'amortissement est également beaucoup moins cher à simuler que le frottement, mais veuillez noter que l'amortissement ne remplace pas le frottement; les deux effets peuvent et doivent être utilisés ensemble. Voici une image pour illustrer l'amortissement:

Les paramètres d'amortissement doivent être compris entre 0 et l'infini, 0 signifiant aucun amortissement et l'infini signifiant un amortissement complet. Normalement, vous utiliserez une valeur d'amortissement comprise entre 0 et 1, mais vous pouvez utiliser n'importe quelle valeur non négative si nécessaire.

Syntaxe:

physics_fixture_set_linear_damping(fixture, damping)

Argument Description
fixture l'index du luminaire
damping l'amortissement du luminaire, généralement entre 0 et 1

Retour:

N/A

Exemple:

physics_fixture_set_linear_damping(fix_Ball, 0.1);

Le code ci-dessus définira l'amortissement linéaire de l'appareil indexé dans "fix_ball" à 0,1.
