physics_fixture_set_chain_shape

Cette fonction définit une forme de fixation "en chaîne". Un dispositif de fixation à chaîne est composé d'un certain nombre de points, qui sont ensuite reliés entre eux en utilisant des formes de bord pour les joindre. La fonction elle-même prend l'index (ID) de l'appareil et peut dire à l'appareil de boucler ou non. Définir ceci sur true connectera le dernier point au premier point avec une arête, tout en le définissant sur false Ne fera pas. Notez que cette fonction sur prépare l'appareil à accepter les points nécessaires pour faire la chaîne, et ceux-ci doivent être ajoutés après avoir appelé cette fonction en utilisant physics_fixture_add_point(), comme vous le feriez lors de la construction d'un appareil polygonal.

Lors de la création d'un montage en chaîne, vous devez lui donner au moins deux points mais vous n'êtes pas limité dans le nombre de points suivants que vous pouvez ajouter pour constituer le luminaire fini.

Syntaxe:

physics_fixture_set_chain_shape(fixture, loop)

Argument Description
fixture L'index du luminaire
loop S'il faut fermer la chaîne (true ) ou non (false ).

Retour:

N/A

Exemple:

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
    {
    physics_fixture_add_point(fix, xx, yy);
    xx += 50 + random(150);
    yy = (room_height / 2) - 250 + irandom(500);
    }
physics_fixture_bind(fix, id);

Le code ci-dessus créera une ligne de luminaires «de bord» avec une variété de hauteurs sur toute la longueur de la pièce.
