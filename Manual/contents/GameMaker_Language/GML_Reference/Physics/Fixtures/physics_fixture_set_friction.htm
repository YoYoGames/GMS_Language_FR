physics_fixture_set_friction

Le frottement est la force qui résiste au mouvement relatif des éléments matériels glissant les uns contre les autres, ce qui, dans le GameMaker Studio 2, se traduit par la perte d'élan causée par la collision de deux instances avec des appareils qui leur sont liés. Ainsi, lorsque deux instances entrent en collision, leur mouvement est affecté par cette valeur, un frottement élevé provoquant une perte d'élan plus importante qu'une valeur inférieure. Notez que le frottement est généralement défini sur une valeur comprise entre 0 et 1, mais vous pouvez utiliser n'importe quelle valeur non négative si nécessaire.

Syntaxe:

physics_fixture_set_friction(fixture, friction)

Argument Description
fixture l'index du luminaire
friction le frottement de la fixation

Retour:

N/A

Exemple:

physics_fixture_set_friction(fix_Brick, 0.1);

Le code ci-dessus définira le frottement de l'appareil indexé dans "fix_brick" à 0,1.
