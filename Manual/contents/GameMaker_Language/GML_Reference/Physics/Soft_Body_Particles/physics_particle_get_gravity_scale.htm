physics_particle_get_gravity_scale

Avec cette fonction, vous pouvez savoir quel est le facteur d'échelle de gravité actuel pour les particules dans la simulation physique (vous pouvez définir cette valeur en utilisant physics_particle_set_gravity_scale() ).

Syntaxe:

physics_particle_get_gravity_scale()

Retour:

Real

Exemple:

physics_particle_set_gravity_scale(physics_particle_get_gravity_scale() + 0.1);

Le code ci-dessus définira l'échelle de gravité.

Retour: Particules corporelles molles
