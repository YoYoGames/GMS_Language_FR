physics_particle_get_density

Cette fonction permet de connaître la densité de courant des particules dans la simulation physique (vous pouvez définir cette valeur à l'aide de la fonction physics_particle_set_density()).

Syntaxe :

physics_particle_get_damping()

Les retours :

Real

Exemple :

if (physics_particle_get_density() < 1)
{
physics_particle_set_density(physics_particle_get_density() + 0.01) ;
}

Le code ci-dessus vérifiera la valeur de la densité actuelle pour toutes les particules du système et si elle est inférieure à 1, il y ajoutera 0,01.
