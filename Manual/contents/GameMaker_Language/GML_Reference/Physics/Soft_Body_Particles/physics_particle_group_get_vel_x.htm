physics_particle_group_get_vel_x

Avec cette fonction, vous pouvez récupérer la vitesse horizontale d'un groupe de particules. Le group valeur est celle qui a été renvoyée lors de la création du groupe de particules à l'aide de la fonction physics_particle_group_end(), et la fonction renverra une valeur qui est la valeur combinée des indicateurs actuellement définis.

Syntaxe:

physics_particle_group_get_vel_x(group)

Argument Description
group Le groupe de particules à obtenir.

Retour:

Real

Exemple:

vx = physics_particle_group_get_vel_x(group1);
vy = physics_particle_group_get_vel_y(group1);

Le code ci-dessus obtiendra la vitesse horizontale et verticale du groupe de particules indexées dans la variable "group1" et les stockera dans des variables.

Retour: Particules corporelles molles
