physics_particle_group_delete

Cette fonction permet de supprimer (retirer) un groupe de particules de la simulation physique dans la pièce actuelle. La fonction prend l'ID unique du groupe à supprimer, tel que renvoyé par la fonction physics_particle_group_end().

Syntaxe :

physics_particle_group_delete(ind)

Arguments Description
ind L'index (ID) du groupe de particules à supprimer.

Les retours :

N/A

Exemple :

physics_particle_group_delete(gp1) ;

Le code ci-dessus supprimera de la simulation toutes les particules qui composent le groupe dont l'ID est stocké dans la variable "gp1".
