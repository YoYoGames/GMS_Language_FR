physics_joint_get_value

En utilisant une série de constantes prédéfinies, vous pouvez demander à GameMaker Studio 2 de vous dire un certain nombre de choses sur l'état d'une articulation donnée. Cette fonction est très utile car elle vous permet de supprimer des articulations ou de modifier le comportement d'une instance en fonction de vos besoins du moment. en fonction de vos besoins du moment. Il y a un certain nombre de constantes qui peuvent être utilisées dans cette fonction et elles peuvent être trouvées ici : Constantes des articulations physiques, mais sachez que des calculs complexes sont effectués lorsque vous les appelez. lorsque vous les appelez, elles doivent donc être utilisées avec précaution et uniquement lorsque cela est nécessaire. Notez que beaucoup d'entre elles sont uniques à un type de joint spécifique.

Si la propriété n'existe pas (si, par exemple, vous vérifiez la translation prismatique d'une articulation de poulie), la valeur de retour sera 0.

Syntaxe :

physics_joint_get_value(joint, valeur)

Arguments Description
joint L'indice de l'articulation que vous souhaitez tester.
valeur La constante du bien commun que vous souhaitez tester.

Les retours :

Real

Exemple :

if shipJoint
{
var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x) ;
var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y) ;
var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY) ;
if reactionForce > 2
{
physics_joint_delete(shipJoint) ;
shipJoint = -1 ;
}
}

Le code ci-dessus vérifie si la variable "shipJoint" contient un indice d'articulation et si c'est le cas, il calcule la force appliquée à cette articulation à l'aide des deux constantes. Enfin, si la force totale est supérieure à 2, l'articulation est supprimée.
