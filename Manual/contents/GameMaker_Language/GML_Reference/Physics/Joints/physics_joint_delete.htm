physics_joint_delete

Une fois que deux instances avec des représentations physiques ont été liées par une liaison, celle-ci peut être supprimée à nouveau à tout moment. Normalement, cela se produit automatiquement lorsque l'une des deux instances est détruite, ou lorsque la salle se termine, mais il y a des moments où vous voudrez peut-être le faire manuellement. Dans ces cas, vous utiliseriez cette fonction.

Il convient également de noter que chaque fois qu'une instance qui fait partie d'une articulation d'engrenage est détruite, l'articulation d'engrenage doit être supprimée à l'aide de cette fonction avant l' une des instances impliquées dans la formation de l'engrenage (mais les articulations restantes seront supprimées automatiquement), pour exemple dans l'événement destroy de l'instance.

Syntaxe:

physics_joint_delete(joint)

Argument Description
joint L'index de l'articulation que vous souhaitez supprimer

Retour:

N/A

Exemple:

var reactionForceX, reactionForceY, reactionForce;
if shipJoint
    {
    reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    reactionForce = sqrt((reactionForceX + reactionForceX) + (reactionForceY + reactionForceY));
    if reactionForce > 2
        {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
        }
    }

Le code ci-dessus vérifie si la variable "shipJoint" contient un index d'articulation et si c'est le cas, il calcule alors la force appliquée à cette articulation en utilisant les deux constantes. Enfin, si la force totale est supérieure à 2, l'articulation est supprimée.
