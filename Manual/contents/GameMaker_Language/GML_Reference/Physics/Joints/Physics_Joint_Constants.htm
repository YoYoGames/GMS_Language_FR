Constantes articulaires de physique

Il existe un grand nombre de constantes incluses dans la GML GameMaker Studio 2 qui sont spécifiques aux articulations entre les appareils dans le monde de la physique. Celles-ci peuvent être utilisées en conjonction avec de nombreuses fonctions communes différentes pour définir ou obtenir des informations à partir d'elles en temps réel pendant que la simulation physique est en cours. Vous devez être conscient, cependant, que des calculs complexes sont effectués lorsque vous les appelez, ils doivent donc être utilisés avec précaution et uniquement lorsque cela est nécessaire et notez que beaucoup sont uniques à un type d'articulation spécifique. Notez également que même si vous pouvez obtenir toutes ces valeurs avec la fonction appropriée, vous ne pouvez définir que celles qui sont marquées comme n'étant pas en lecture seule.

En général, ces constantes seraient utilisées en conjonction avec les fonctions suivantes:

  • physics_joint_get_value
  • physics_joint_set_value

Les constantes suivantes peuvent être appliquées à tous les types de jonction disponibles:

Constant Description Lecture seulement
phy_joint_anchor_1_x La coordonnée x du premier point d'ancrage du joint dans la pièce Yes
phy_joint_anchor_1_y La coordonnée y du premier point d'ancrage du joint dans la pièce Yes
phy_joint_anchor_2_x La coordonnée x du deuxième point d'ancrage du joint dans la pièce Yes
phy_joint_anchor_2_y La coordonnée y du deuxième point d'ancrage du joint dans la pièce Yes
phy_joint_reaction_force_x Il s'agit de la force de réaction appliquée à la seconde instance dans une articulation à la position d'ancrage x Yes
phy_joint_reaction_force_y Il s'agit de la force de réaction appliquée à la seconde instance dans une articulation à la position d'ancrage y Yes
phy_joint_reaction_torque Il s'agit du couple appliqué à la seconde instance dans une articulation à la position d'ancrage Yes


Ces constantes concernent les articulations auxquelles est attaché un moteur (révolutionnaire, prismatique, roue):

Constant Description Lecture seulement
phy_joint_max_motor_force La valeur spécifiée lors de la création de l'articulation pour la force maximale du moteur No
phy_joint_max_motor_torque La valeur spécifiée lors de la création de l'articulation pour le couple moteur maximal No
phy_joint_motor_force La force actuelle du moteur Yes
phy_joint_motor_speed La vitesse actuelle du moteur No
phy_joint_motor_torque Le couple moteur actuel Yes


Pour une articulation révolutionnaire, vous pouvez utiliser la constante suivante (ainsi que les constantes du moteur s'il y en a une):

Constant Description Lecture seulement
phy_joint_angle L'angle qu'une ligne entre les deux points d'ancrage du joint fait. Ceci est calculé en utilisant les coordonnées du monde physique
( pas les coordonnées de la pièce GameMaker Studio 2		 Yes
phy_joint_angle_limits Activez ou désactivez la limitation d'angle pour l'articulation. Définissez la valeur sur true pour activer ou false pour désactiver. No
phy_joint_upper_angle_limit La limite d'angle supérieure de l'articulation en degrés. No
phy_joint_lower_angle_limit La limite inférieure de l'angle de l'articulation en degrés. No


Pour un joint prismatique, vous pouvez utiliser la constante suivante:

Constant Description Lecture seulement
phy_joint_translation Obtient la distance entre les coordonnées x / y de l'ancre et les coordonnées x / y locales. Yes
phy_joint_speed La vitesse de mouvement actuelle de l'articulation. Yes


Pour une distance, une soudure et des joints de roue, vous pouvez utiliser les constantes suivantes (ainsi que celles des joints de poulie):

Constant Description Lecture seulement
phy_joint_damping_ratio Le rapport d'amortissement est non dimensionnel et définit la "souplesse" du joint. La valeur de cette constante est généralement comprise entre 0 et 1, mais peut être plus grande, et à 1, l'amortissement est critique, ce qui signifie que toutes les oscillations doivent disparaître. No
phy_joint_frequency Cela retournera (ou définira) la fréquence d'oscillation pour l'articulation, en hertz, et généralement la fréquence doit être inférieure à la moitié de la fréquence du pas de temps, comme défini par la fonction physics_world_update_speed(). No
phy_joint_length_1 Cela renverra la longueur de l'articulation des premières coordonnées x / y locales aux premières coordonnées x / y d'ancrage (liaisons à distance uniquement, ne peut être lue qu'à partir de) Yes
phy_joint_length_2 Cela renverra la longueur de l'articulation à partir des secondes coordonnées x / y locales vers les secondes coordonnées x / y d'ancrage (liaisons à distance uniquement, ne peut être écrite que dans) No


Pour un joint de friction, vous pouvez utiliser les constantes suivantes:

Constant Description Lecture seulement
phy_joint_max_torque La valeur de couple maximum pour l'articulation. No
phy_joint_max_force La valeur de force maximale pour l'articulation. No


Pour un joint de corde, vous pouvez utiliser la constante suivante:

Constant Description Lecture seulement
phy_joint_max_length L'extension maximale de la connexion entre les deux points d'ancrage. No
