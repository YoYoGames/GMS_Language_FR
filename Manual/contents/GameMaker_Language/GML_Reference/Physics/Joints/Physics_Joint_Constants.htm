Constantes communes de la physique

Le GML de GameMaker Studio 2 contient un grand nombre de constantes spécifiques aux articulations entre les appareils dans le monde physique. Elles peuvent être utilisées en conjonction avec de nombreuses fonctions de jointure pour les définir ou obtenir des informations en temps réel pendant l'exécution de la simulation physique. Sachez toutefois que des calculs complexes sont effectués lorsque vous appelez ces fonctions. et seulement lorsque cela est nécessaire, et notez que beaucoup d'entre eux sont uniques à un type spécifique de joint. Notez également que si vous pouvez obtenir toutes ces valeurs avec la fonction appropriée, vous ne pouvez définir que celles qui sont marquées comme n'étant pas en lecture seule.

En général, ces constantes sont utilisées en conjonction avec les fonctions suivantes :

  • physics_joint_get_value
  • physics_joint_set_value

Les constantes suivantes peuvent être appliquées à tous les types de joints disponibles :

Constant Description Lecture seulement
phy_joint_anchor_1_x La coordonnée x du premier point d'ancrage du joint dans la pièce. Oui
phy_joint_anchor_1_y La coordonnée y du premier point d'ancrage du joint dans la pièce Oui
phy_joint_anchor_2_x La coordonnée x du deuxième point d'ancrage du joint dans la pièce Oui
phy_joint_anchor_2_y Coordonnée y du deuxième point d'ancrage du joint dans la pièce Oui
phy_joint_réaction_force_x C'est la force de réaction appliquée à la deuxième instance dans une articulation à la position d'ancrage x. Oui
phy_joint_reaction_force_y Il s'agit de la force de réaction appliquée à la deuxième instance dans une articulation à la position d'ancrage y. Oui
phy_joint_reaction_torque Il s'agit du couple appliqué à la deuxième instance dans une articulation à la position d'ancrage. Oui


Ces constantes sont destinées aux articulations auxquelles un moteur est attaché (rotative, prismatique, roue) :

Constant Description Lecture seulement
phy_joint_max_motor_force La valeur spécifiée lors de la création de l'articulation pour la force maximale du moteur. Non
phy_joint_max_motor_torque La valeur spécifiée lors de la création du joint pour le couple moteur maximum Non
phy_joint_motor_force La force actuelle du moteur Oui
phy_joint_motor_speed La vitesse actuelle du moteur Non
phy_joint_motor_torque Le couple actuel du moteur Oui


Pour une articulation rotative, vous pouvez utiliser la constante suivante (ainsi que les constantes du moteur si un moteur a été ajouté) :

Constant Description Lecture seulement
phy_joint_angle L'angle que fait une ligne entre les deux points d'ancrage de l'articulation. Il est calculé à l'aide des coordonnées du monde physique
( et non des coordonnées de la pièce de GameMaker Studio 2 ) en radians. ) en radians.		 Oui
Limites de l'angle de jointure des phares Active ou désactive la limitation de l'angle pour l'articulation. Définissez la valeur sur true pour activer ou false pour désactiver. Non
phy_joint_angle_supérieur_limite La limite supérieure de l'angle de l'articulation en degrés. Non
phy_joint_angle_inférieur_limite La limite inférieure de l'angle de l'articulation en degrés. Non


Pour un joint prismatique, vous pouvez utiliser la constante suivante :

Constant Description Lecture seulement
phy_joint_translation Obtient la distance entre les coordonnées x/y de l'ancre et les coordonnées x/y locales. Oui
phy_joint_speed La vitesse actuelle du mouvement de l'articulation. Oui


Pour une distance, une soudure et des joints de roue, vous pouvez utiliser les constantes suivantes (ainsi que celles pour les joints de poulie) :

Constant Description Lecture seulement
phy_joint_damping_ratio Le rapport d'amortissement est non dimensionnel et définit le "ressort" de l'articulation. La valeur de cette constante est généralement comprise entre 0 et 1, mais peut être plus grande. À 1, l'amortissement est critique, ce qui signifie que toutes les oscillations doivent disparaître. Non
phy_joint_fréquence Ceci renverra (ou définira) la fréquence d'oscillation de l'articulation, en hertz, et typiquement la fréquence devrait être inférieure à la moitié de la fréquence du pas de temps, comme défini par la fonction physics_world_update_speed(). Non
phy_joint_longueur_1 Cette fonction renvoie la longueur de l'articulation entre les premières coordonnées locales x/y et les premières coordonnées x/y de l'ancre (uniquement pour les articulations à distance, qui peuvent être lues). Oui
phy_joint_longueur_2 Ceci renverra la longueur du joint entre les deuxièmes coordonnées locales x/y et les deuxièmes coordonnées x/y de l'ancre (joints de distance uniquement, ne peuvent être écrits). Non


Pour un joint de friction, vous pouvez utiliser les constantes suivantes :

Constant Description Lecture seulement
phy_joint_max_torque La valeur maximale du couple pour l'articulation. Non
phy_joint_max_force La valeur de la force maximale pour l'articulation. Non


Pour un joint de corde, vous pouvez utiliser la constante suivante :

Constant Description Lecture seulement
phy_joint_max_length L'extension maximale de la connexion entre les deux points d'ancrage. Non
