physics_pause_enable

Normalement, pour qu'une simulation physique fonctionne, elle doit être continue et ne peut pas être arrêtée et démarrée, ou avoir des instances déplacées soudainement d'un endroit à l'autre de la pièce. Cependant, il y a des moments où vous devez mettre en pause la simulation comme (par par exemple, le dispositif os s'est arrêté, et vous utiliserez alors cette fonction. Elle met en pause la simulation si le drapeau est à true et aucun autre calcul physique ne sera effectué jusqu'à ce que le drapeau soit à nouveau à false.

REMARQUE: cette fonction est particulièrement utile si vous souhaitez désactiver toutes les instances d'une pièce car, même désactivé, un corps physique continuera à être calculé et simulé dans le monde physique.

Syntaxe :

physics_pause_enable(flag)

Arguments Description
drapeau Cette valeur peut être définie sur true pour mettre la simulation en pause, ou false pour la relancer.

Les retours :

N/A

Exemple :

if os_is_paused()
{
physics_pause_enable(true) ;
instance_deactivate_all(true) ;
instance_create_layer(x, y, "Controllers", obj_PauseMenu) ;
}

Le code ci-dessus vérifie si l'OS a été mis en pause et si c'est le cas, il met en pause le monde physique, désactive tout sauf lui-même, puis crée une instance de menu de pause.
