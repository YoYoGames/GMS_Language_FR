Le monde de la physique

Avant que toute réaction physique ne puisse se produire dans GameMaker Studio 2, vous devez d'abord définir le monde physique dans lequel elle va se produire. Si vous ne le faites pas, GameMaker Studio 2 utilisera par défaut les fonctions de collision de base. GameMaker Studio 2 utilisera par défaut les fonctions de collision de base, et tout autre code ou paramètre lié à la physique sera ignoré. Vous pouvez configurer le monde physique à partir de l'éditeur de pièce, mais vous pouvez également définir les mêmes propriétés en utilisant du code avec les fonctions suivantes :

  • physique_monde_créer
  • physics_world_gravity
  • physics_world_update_iterations
  • physics_world_update_speed
  • physics_world_draw_debug
  • physics_draw_debug
  • physics_pause_enable
