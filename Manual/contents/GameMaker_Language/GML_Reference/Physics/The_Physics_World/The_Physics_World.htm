Le monde de la physique

Avant que des réactions physiques ne puissent se produire dans GameMaker Studio 2 vous devez d'abord définir le monde physique dans lequel elles vont se produire. Si vous ne le faites pas, GameMaker Studio 2 utilisera par défaut les fonctions de collision de base et tous les autres codes ou paramètres liés à la physique seront ignorés. Vous pouvez configurer le monde physique à partir de l' éditeur de salle, mais vous pouvez également définir les mêmes propriétés à l'aide de code avec les fonctions suivantes:

  • physics_world_create
  • physics_world_gravity
  • physics_world_update_iterations
  • physics_world_update_speed
  • physics_world_draw_debug
  • physics_draw_debug
  • physics_pause_enable
