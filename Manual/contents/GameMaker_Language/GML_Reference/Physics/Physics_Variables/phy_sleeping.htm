phy_sleeping

Cette variable en lecture seule indique si l'instance est actuellement "endormie" (true) ou non(false). Une instance"endormie" est une instance qui n'est pas activement engagée dans une simulation physique. GameMaker Studio 2 met les objets en sommeil pour économiser des cycles de simulation lorsqu'une instance est au repos et n'entre pas en collision avec une autre instance.

Syntaxe :

phy_sleeping ;

Les retours :

Booléen (ou indéfini si l'instance n'est pas activée pour la physique)

Exemple :

if phy_sleeping
{
instance_destroy() ;
}

Le code ci-dessus vérifie si l'objet est activement simulé ou non et s'il ne l'est pas, il est détruit.
