phy_position_yprevious

Cette variable peut être utilisée pour obtenir (ou pour définir) la position y précédente de l'instance dans le monde physique de la salle de jeux. Il s'agit de la position de l'instance dans le monde de la physique de l'étape précédente à l'étape actuelle

Syntaxe:

phy_position_yprevious;

Retour:

Real (single precision floating point value, or undefined if the instance is not physics enabled)

Exemple:

xx = phy_position_xprevious;
yy = phy_position_yprevious;

Le code ci-dessus stocke la position x et y précédente pour l'instance activée pour la physique dans deux variables.
