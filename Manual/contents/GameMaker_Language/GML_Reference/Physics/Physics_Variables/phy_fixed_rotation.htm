phy_fixed_rotation

Cette variable peut être utilisée pour définir si l'instance peut ou non être affectée par les forces de rotation (la valeur par défaut est false ). Si ce paramètre est défini sur true, aucune force externe (que ce soit des impulsions ou des forces codées, ou des collisions) n'affectera la valeur de rotation de l'instance et cela devrait être réglé manuellement à l'aide du phy_rotation variable

Syntaxe:

phy_fixed_rotation;

Retour:

Boolean (or undefined if the instance is not physics enabled)

Exemple:

phy_rotation = 0;
phy_fixed_rotation = true;

Le code ci-dessus changera l'instance pour avoir une rotation fixe, puis définira l'angle de rotation.
