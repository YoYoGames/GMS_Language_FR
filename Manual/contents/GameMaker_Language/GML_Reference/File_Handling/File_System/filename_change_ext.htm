filename_change_ext

Cette fonction renvoie le nom de fichier indiqué avec l'extension (y compris le point) remplacée par la nouvelle extension. En utilisant une chaîne vide comme nouvelle extension, vous pouvez supprimer la partie d'extension tous ensemble.

Syntaxe:

filename_change_ext(fname, newext);

Argument Description
fname Le fichier à utiliser.
newext La nouvelle extension à utiliser.

Retour:

String

Exemple:

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

Le code ci-dessus obtient le nom du fichier (sous forme de chaîne) avec la partie d'extension supprimée.
