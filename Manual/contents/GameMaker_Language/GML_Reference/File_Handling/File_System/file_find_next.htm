file_find_next

Cette fonction renvoie le nom du fichier suivant qui satisfait le masque précédemment donné et les attributs (définis par file_find_first() ). Si aucun fichier de ce type n'existe, la chaîne vide est renvoyée.

ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

REMARQUE: Cette fonction fonctionne sur toutes les plates-formes cibles C ++ ( Windows, Mac, iOS, Android et WinPhone), MAIS les indicateurs de filtre ne fonctionnent que sous Windows.

Syntaxe:

file_find_next();

Retour:

String

Exemple:

file = file_find_next();

Cela renverra le fichier suivant qui correspond aux attributs précédemment donnés.
