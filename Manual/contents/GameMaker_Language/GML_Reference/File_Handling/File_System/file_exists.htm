fichier_existant

Cette fonction renvoie vrai si le fichier spécifié existe et faux s'il n'existe pas. Notez que cette fonction ne peut être utilisée que pour vérifier les fichiers locaux, mais pas les fichiers stockés sur un serveur distant.

Syntaxe :

file_exists(fname) ;

Arguments Description
nom de famille Le nom du fichier à vérifier.

Les retours :

Booléen

Exemple :

if file_exists("level.txt")
{
file = file_text_open_read("level.txt") ;
}

Ceci vérifie l'existence d'un fichier et s'il existe, il est ouvert à la lecture.
