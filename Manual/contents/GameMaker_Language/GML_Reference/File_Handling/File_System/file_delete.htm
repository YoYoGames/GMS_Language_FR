file_delete

Cette fonction supprimera le fichier spécifié du système. Il convient de noter que cette fonction supprimera uniquement les fichiers que GameMaker Studio 2 est capable de créer et d'analyser: les fichiers ini, les fichiers texte et les fichiers binaires, ou les fichiers créés pour stocker des ressources créées par le jeu telles que des sprites ou des surfaces. Cependant, il ne supprimera aucun autre fichier. La fonction retournera également true si le fichier a été supprimé avec succès, ou false dans toutes autres circonstances.

ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur les limites du système de fichiers pour plus d'informations.

Syntaxe:

file_delete(fname);

Argument Description
fname Le nom du fichier à supprimer.

Retour:

Boolean

Exemple:

if file_exists("level.txt")
    {
    file_delete("level.txt");
    }

Cela vérifierait un fichier et s'il existe, il est supprimé.
