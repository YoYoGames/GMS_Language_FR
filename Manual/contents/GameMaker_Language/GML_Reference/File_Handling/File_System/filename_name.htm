nom_de_fichier

L'utilisation de cette fonction renvoie la partie nom du fichier indiqué, avec l'extension mais sans le chemin.

Syntaxe :

nom_de_fichier(fname) ;

Arguments Description
nom de famille Le fichier à utiliser.

Les retours :

Chaîne de caractères

Exemple :

name = nom de fichier(file_find_first("C:/Games/*.doc", 0)) ;

Le code ci-dessus obtient le nom (sous forme de chaîne) du premier fichier de type "doc" trouvé dans le répertoire spécifié.
