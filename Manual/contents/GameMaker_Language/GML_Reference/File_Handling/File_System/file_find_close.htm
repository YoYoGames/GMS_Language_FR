fichier_trouvé_fermé

Doit être appelé après avoir traité les fichiers ouverts avec file_find_first() et file_find_next() pour libérer la mémoire. Les fonctions de recherche de fichiers ouvrent des handles dans le répertoire des fichiers et ceux-ci occupent une quantité minimale de mémoire, qui s'accumule (avec le temps). Par conséquent, vous devez toujours appeler cette fonction après avoir trouvé les fichiers dont vous avez besoin pour "fermer" ces handles.

Syntaxe :

file_find_close() ;

Renvoie à

N/A

Exemple :

file_find_close() ;

Ceci fermera et libérera la mémoire associée aux poignées de la fonction de recherche de fichiers.
