file_find_close

Doit être appelé après la gestion des fichiers ouverts à l'aide de file_find_first() et file_find_next() fonctions pour libérer de la mémoire. Les fonctions de recherche de fichiers ouvrent des poignées dans le répertoire de fichiers et celles-ci occupent une quantité minimale de mémoire, qui (au fil du temps) monte. Par conséquent, vous devez toujours appeler cette fonction après avoir trouvé les fichiers dont vous avez besoin pour «fermer» ces poignées.

Syntaxe:

file_find_close();

Retour

N/A

Exemple:

file_find_close();

Cela fermera et libérera la mémoire associée aux poignées de la fonction de recherche de fichiers.
