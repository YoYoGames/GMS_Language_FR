filename_drive

Cette fonction renvoie les informations de lecteur du nom de fichier.

Syntaxe:

filename_drive(fname);

Argument Description
fname Le fichier à utiliser.

Retour:

String

Exemple:

drive = filename_drive(file_find_first(working_directory + "*.doc", 0));

Le code ci-dessus obtient les informations de lecteur (sous forme de chaîne) du fichier spécifié.

Retour: Système de fichiers
