get_save_filename_ext

Cette fonction ouvre une boîte de dialogue et demande au joueur un nom de fichier à enregistrer avec le filtre donné. Le filtre a la forme " nom1 | masque1 | nom2 | masque2 |... ", où le masque peut contenir les différentes options avec un point-virgule entre elles et vous pouvez également utiliser un "*" pour désigner n'importe quelle chaîne. Par exemple: " bitmaps | *.bmp; *. Wmf ", demanderait à l'utilisateur de sélectionner le nom d'un fichier bitmap dans le système, mais uniquement ceux qui ont les deux extensions spécifiées (bmp et png ). cette fonction n'enregistre pas le fichier lui-même! Il renvoie uniquement une chaîne avec le nom complet et le chemin du fichier. Si l'utilisateur appuie sur " Annuler ", une chaîne vide "" est renvoyée. Vous pouvez également ajouter un répertoire à partir duquel démarrer la recherche et une légende à afficher en haut de la fenêtre au lieu de celle par défaut.

Une chose importante à noter lors de l'utilisation de cette fonction est qu'elle vous accorde certaines autorisations pour ce fichier, pour la durée de votre partie. Ainsi, une fois que vous avez obtenu le chemin du fichier de l'utilisateur, vous pouvez y accéder encore et encore sans avoir à demander, et l'autorisation s'applique également à la lecture et à l' écriture.

REMARQUE: cette fonction n'est valide que sur les cibles Windows et macOS

Syntaxe:

get_save_filename_ext(filter, fname, directory, caption);

Argument Description
filter Le filtre et / ou le nom du type de fichier.
fname Le nom de fichier suggéré à utiliser.
directory Le répertoire à partir duquel démarrer la recherche.
caption La légende de la fenêtre.

Retour:

String

Exemple:

var file;
file = get_save_filename_ext("screenshot|*.png", "", working_directory, "Save a level here");
if file != ""
    {
    screen_save(file);
    }

Le code ci-dessus demandera à l'utilisateur de sélectionner un fichier de nom de fichier image à enregistrer dans le répertoire de travail, puis de vérifier le nom et le chemin du fichier renvoyé.Si ce n'est pas une chaîne vide, il prend une capture d'écran et l'enregistre en utilisant le nom donné.
