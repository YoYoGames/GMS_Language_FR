file_bin_size

Cette fonction retournera la taille (en octets) d'un fichier qui a été ouvert en lecture et / ou en écriture. L'ID de fichier est la valeur renvoyée par la fonction file_bin_open().

REMARQUE: ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.

Syntaxe:

file_bin_size(binfile);

Argument Description
binfile L'ID du fichier dont la taille doit être obtenue.

Retour:

Real

Exemple:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_close(file);

Cela ouvrirait un fichier à partir du répertoire local du jeu et attribuerait son index à la variable "fichier". Il obtiendrait alors la taille de ce fichier et le fermerait à nouveau.

Retour: Fichiers binaires
