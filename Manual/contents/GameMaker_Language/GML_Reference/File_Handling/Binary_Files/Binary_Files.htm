Fichiers binaires

Dans de rares cas, vous devrez peut-être lire des données à partir de fichiers binaires. Ces fonctions peuvent être utilisées à cette fin, bien qu'en général, nous vous recommandons d'utiliser les fonctions de tampon dédiées pour créer, enregistrer et charger des données binaires plutôt que ces fonctions.

REMARQUE: ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.

Les routines de bas niveau suivantes existent pour cela:

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
