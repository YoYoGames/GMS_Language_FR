Fichiers binaires

Dans de rares situations, vous pouvez avoir besoin de lire des données à partir de fichiers binaires. Ces fonctions peuvent être utilisées à cette fin, bien qu'en général, nous vous recommandons d'utiliser les fonctions de tampon dédiées pour créer, enregistrer et charger des données binaires plutôt que ces fonctions. et charger des données binaires plutôt que ces fonctions.

REMARQUE: Ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.

Les routines de bas niveau suivantes existent pour cela :

  • file_bin_open
  • réécriture de fichier_bin
  • file_bin_close
  • taille_de_fichier
  • file_bin_position
  • fichier_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
