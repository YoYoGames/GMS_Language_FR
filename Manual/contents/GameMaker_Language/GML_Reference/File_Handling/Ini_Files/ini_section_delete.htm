ini_section_delete

Avec cette fonction, vous pouvez supprimer une section entière d'un fichier ini, ce qui supprimera également toutes les paires clé-valeur qui lui sont associées.

Syntaxe :

ini_section_delete(section) ;

Arguments Description
section La section à supprimer.

Les retours :

N/A

Exemple :

ini_open("savedata.ini") ;
ini_write_real("save1", "Score", score ) ;
ini_section_delete("save1") ;
ini_close() ;

Cet exemple va ouvrir "savedata.ini" et écrire une valeur dans "save1" > "Score". Il supprimera ensuite la section "save1" et fermera le fichier .ini.
