ini_key_delete

Cette fonction vous permet de supprimer la clé sélectionnée (et sa valeur correspondante) d'un fichier ini.

Syntaxe :

ini_key_delete(section, clé) ;

Arguments Description
section La section dont il faut supprimer une clé.
clé La clé pour supprimer.

Les retours :

N/A

Exemple :

ini_open("savedata.ini") ;
ini_write_real("save1", "Score",score) ;
ini_key_delete("save1", "Score") ;
ini_close() ;

Cet exemple va ouvrir "savedata.ini" et écrire une valeur dans "save1" > "Score". Il supprimera ensuite la clé "Score" et fermera le fichier .ini.
