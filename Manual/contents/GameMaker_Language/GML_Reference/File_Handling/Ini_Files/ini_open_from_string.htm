ini_open_from_string

Cette fonction va créer un fichier ini à partir d'une chaîne de caractères et l'ouvrir en lecture/écriture. La chaîne de caractères doit être correctement formatée comme un fichier ini (c'est-à-dire avec des sections, des clés de retour à la ligne et des valeurs), sinon le fichier ini ne sera pas créé correctement. Notez que ce fichier ini est temporaire et qu'il sera supprimé de la mémoire au moment où il sera fermé, perdant ainsi toute information qui y était stockée. ini_close() renvoie une chaîne de caractères du fichier ini complet qui peut être qui peut ensuite être sauvegardé sur un serveur ou sur le disque.

Syntaxe :

ini_open_from_string(string) ;

Arguments Description
chaîne de caractères La chaîne de caractères contenant toutes les informations ini.

Les retours :

N/A

Exemple :

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true) ;
ini_close() ;

Le code ci-dessus définit un ini temporaire pour contenir les informations de la chaîne "str", puis lit l'ini avant de le refermer.
