file_text_eof

Cette fonction renvoie true lorsque la fin d'un fichier texte ouvert donné est atteinte ou false sinon.

Syntaxe:

file_text_eof(fileid);

Argument Description
fileid L'identifiant du fichier à vérifier.

Retour:

Boolean

Exemple:

var num = 0;
var file = file_text_open_read(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
    {
    str[num++] = file_text_readln(file);
    }
file_text_close(file);

Le code ci-dessus ouvre un fichier pour l'écriture puis boucle à travers les lignes de texte déjà écrites dans le fichier jusqu'à ce qu'il atteigne la fin, en stockant chaque ligne dans le tableau "str".
