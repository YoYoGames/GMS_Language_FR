file_text_open_from_string

Cette fonction créera un fichier texte à partir d'une chaîne et l'ouvrira pour la lecture, renvoyant le fichier "handle" qui devrait être utilisé dans tous les autres appels de fonction de fichier à lire à partir de ce fichier. Notez que ce fichier est temporaire et en lecture seule, et en tant que tel, il sera supprimé de la mémoire au moment de sa fermeture.

REMARQUE: vous ne pouvez ouvrir qu'un maximum de 32 fichiers à la fois. Vous devez également toujours fermer les fichiers lorsque vous avez terminé, car cela écrit les informations et libère la mémoire associée au fichier.

ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

Syntaxe:

file_text_open_from_string(string);

Argument Description
string Chaîne à partir de laquelle créer le fichier.

Retour:

Real

Exemple:

file = file_text_open_from_string(reset_str);

Le code ci-dessus prend la chaîne stockée dans la variable "reset_str" et l'utilise pour créer un fichier texte en lecture seule. Le «descripteur» de ce fichier est ensuite stocké dans la variable «fichier» pour toutes les autres fonctions de fichier à utiliser.

Retour: Fichiers texte
