Fichiers texte

Les fichiers texte sont un moyen utile de stocker de grandes quantités de données en dehors de votre jeu. Contrairement aux fichiers ini, vous pouvez avoir jusqu'à 32 fichiers texte ouverts en même temps, et vous pouvez également y lire et y écrire de grandes quantités de données. Veuillez noter que pour les jeux avec localisations, il est très important que les lettres accentuées puissent être lues à partir de fichiers externes. De cette façon, vous pouvez le lire et l'écrire correctement en conservant tous les accents et les lettres non romaines.
