fichier_texte_ouvert_écrit

Cette fonction ouvre le fichier texte avec le nom de fichier indiqué pour l'écriture seulement (si le fichier n'existe pas, il est créé), en retournant l'identifiant unique du fichier qui doit être stocké dans une variable car il sera utilisé pour toutes les autres actions à faire avec ce fichier. actions à faire avec ce fichier. Notez que si le fichier ne peut pas être créé (à cause d'un nom de fichier illégal, par exemple), la fonction retournera -1.

REMARQUE: Vous ne pouvez avoir qu'un maximum de 32 fichiers ouverts à la fois. Vous devez également toujours fermer les fichiers lorsque vous avez terminé, car cela permet d'écrire les informations et de libérer la mémoire associée au fichier.

AVERTISSEMENT ! Il se peut que cette fonction ne fonctionne pas comme vous l'attendez, car GameMaker Studio 2 est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

Syntaxe :

file_text_open_write(fname) ;

Arguments Description
nom de famille Le nom du fichier dans lequel il faut écrire.

Les retours :

Real

Exemple :

var file ;
file = file_text_open_write(working_directory + "level.txt") ;
file_text_write_string(file, level_data) ;
file_text_close(file) ;

Le code ci-dessus ouvrira le fichier "level.txt" pour l'écrire, puis écrira la chaîne stockée dans la variable "level_data" avant de refermer le fichier.
