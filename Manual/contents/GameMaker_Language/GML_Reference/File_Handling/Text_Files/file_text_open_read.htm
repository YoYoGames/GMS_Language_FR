file_text_open_read

Cette fonction ouvre le fichier texte avec le nom de fichier indiqué pour la lecture uniquement, retournant l'identifiant unique du fichier qui doit être stocké dans une variable car il sera utilisé pour toutes les autres actions à faire avec ce fichier. Si le fichier n'existe pas, la fonction renverra la valeur -1.

REMARQUE: vous ne pouvez ouvrir qu'un maximum de 32 fichiers à la fois. Vous devez également toujours fermer les fichiers lorsque vous avez terminé, car cela écrit les informations et libère la mémoire associée au fichier.

ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

Syntaxe:

file_text_open_read(fname);

Argument Description
fname Le nom du fichier à lire.

Retour:

Real

Exemple:

file = file_text_open_read(working_directory + "level.txt");

Cela ouvrirait "level.txt" à partir du même répertoire que le jeu et stockerait l'identifiant du fichier dans la variable "file".
