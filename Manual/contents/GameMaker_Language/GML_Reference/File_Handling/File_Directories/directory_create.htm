directory_create

Cette fonction crée un répertoire avec le nom donné dans la zone de sauvegarde.

REMARQUE: cette fonction ne fonctionnera pas sur HTML5 car vous ne pouvez pas créer ou supprimer de répertoires dans le stockage local du navigateur.

ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

Syntaxe:

directory_create(dname)

Argument Description
dname Le nom du répertoire à créer.

Retour:

N/A

Exemple:

if !directory_exists("Games")
   {
   directory_create("Games");
   }

Cela vérifiera si le répertoire spécifié existe dans le dossier de données local et, si ce n'est pas le cas, il le crée pour vous.
