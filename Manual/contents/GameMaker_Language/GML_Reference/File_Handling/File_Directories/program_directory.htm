program_directory

Cela renverra le répertoire où l'exécutable du jeu est stocké. Cependant, cela peut ne pas toujours être utile, d'autant plus que certains périphériques exécutent l'exe à partir d'un fichier *.zip, donc cela retournerait le même, peu importe où le jeu est réellement exécuté.

ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

REMARQUE: Si vous désactivez le bac à sable, vous ne pourrez toujours pas écrire dans ce répertoire pour éviter tout problème éventuel de suppression des fichiers de jeu requis.

Syntaxe:

program_directory

Retour:

String

Exemple:

dir = program_directory;

Cela stockera le répertoire où l'exécutable est stocké dans une variable.
