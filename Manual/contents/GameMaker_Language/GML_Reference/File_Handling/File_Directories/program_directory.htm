répertoire du programme

Cela renverra le répertoire où l'exécutable du jeu est stocké. Cependant, cela n'est pas toujours utile, notamment parce que certains périphériques exécutent l'exe à partir d'un fichier *.zip. Cette option renvoie donc le même résultat, quel que soit l'endroit où le jeu est exécuté.

AVERTISSEMENT ! Il se peut que cette fonction ne fonctionne pas comme vous l'attendez, car GameMaker Studio 2 est en sandbox ! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

REMARQUE: Si vous désactivez le bac à sable, vous ne pourrez toujours pas écrire dans ce répertoire afin d'éviter tout problème de suppression des fichiers de jeu nécessaires.

Syntaxe :

répertoire du programme

Les retours :

Chaîne de caractères

Exemple :

dir = répertoire du programme ;

Le répertoire où est stocké l'exécutable sera ainsi stocké dans une variable.
