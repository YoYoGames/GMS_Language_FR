Répertoires de fichiers

Lorsque vous travaillez avec des fichiers, il y aura des moments où vous aurez besoin de référencer le répertoire dans lequel les fichiers sont stockés et GameMaker Studio 2 a des fonctions spéciales et des variables relatives aux différents répertoires possibles dont vous pourriez avoir besoin:

  • directory_exists
  • directory_create
  • directory_destroy
  • temp_directory
  • working_directory
  • program_directory

REMARQUE: vous ne pouvez pas créer ou détruire des répertoires sur HTML5.
