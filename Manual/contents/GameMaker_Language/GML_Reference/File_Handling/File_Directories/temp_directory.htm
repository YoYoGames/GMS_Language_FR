temp_directory

Cela peut être utilisé pour renvoyer le répertoire temporaire créé pour votre jeu à chaque exécution (la racine ne contient pas le "\" final). Ce répertoire contiendra des fichiers et sera accessible pendant que le jeu est en cours d'exécution, mais il sera supprimé (avec tous les fichiers qu'il contient) à la fermeture du jeu.

ATTENTION! Cette fonction peut ne pas fonctionner comme prévu en raison du GameMaker Studio 2! Veuillez consulter la section sur le système de fichiers pour plus d'informations.

Syntaxe:

temp_directory

Retour:

String

Exemple:

ini_open(temp_directory + "\temp_ini.ini");

Cela ouvrira un fichier ini dans le répertoire temporaire du jeu (en le créant s'il n'existe pas déjà).
