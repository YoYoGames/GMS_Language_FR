md5_file

En cryptographie, MD5 (Message-Digest algorithme 5) est une fonction de hachage cryptographique largement utilisée avec une valeur de hachage de 128 bits et a été utilisée dans une grande variété d'applications de sécurité. Il est également couramment utilisé pour vérifier l'intégrité des fichiers et des chaînes. Cette fonction prendra le fichier donné et générera un MD5 unique pour ce fichier qui pourra ensuite être stocké pour une utilisation ultérieure.

Syntaxe:

md5_file(filename)

Argument Description
filename Le fichier pour lequel générer le hachage MD5.

Retour:

String

Exemple:

hash = md5_file(working_directory + "game_data.ini")

Le code ci-dessus générera un hachage MD5 pour le fichier spécifié et stockera la valeur renvoyée dans la variable «hachage».
