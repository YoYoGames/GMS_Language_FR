sha1_file

En cryptographie, SHA-1 est une fonction de hachage cryptographique conçue par l' Agence nationale de sécurité des États-Unis et est utilisée dans plusieurs applications largement utilisées, Git où il est utilisé pour vérifier les modifications de fichiers, ainsi que les protocoles TLS et SSL, PGP, SSH, S / MIME et IPsec. Cette fonction prend un fichier d'entrée et renvoie un résumé de message de 160 bits au format ASCII unique à ce fichier pour être utilisé pour la vérification de l'intégrité à une date ultérieure.

Syntaxe:

sha1_file(filename)

Argument Description
filename Le fichier pour lequel générer le hachage sha1.

Retour:

String

Exemple:

hash = sha1_file(working_directory + "game_data.ini")

Le code ci-dessus générera un hachage sha1 pour le fichier spécifié et stockera la valeur renvoyée dans la variable «hachage».
