steam _get_achievement

Avec cette fonction, vous pouvez vérifier l' Steam pour voir si un exploit spécifique a été attribué. Le succès doit avoir été défini au préalable sur la Steam Works pour votre jeu et la chaîne transmise à la fonction doit correspondre à celle utilisée comme nom d'API sur le panneau de contrôle.

Syntaxe:

steam_get_achievement(ach_name);

Argument Description
ach_name Le nom de la réalisation à obtenir (chaîne).

Retour:

Boolean

Exemple:

if hp <= 0
    {
    global.Deaths += 1;
    if global.Deaths == 10
        {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
        }
    }

Le code ci-dessus récompensera le joueur un exploit si la variable globale "Décès" est égal à 10 et si le succès n'a pas déjà été décerné.
