Steam

Cette section est destinée aux utilisateurs qui ont eu accès à l' Steam pour publier votre jeu sur cette plate-forme. Pour pouvoir utiliser ces fonctions, vous devez avoir été préalablement accepté sur Steam, soit via un éditeur ou via le système d'auto-édition, et avoir configuré correctement les préférences Steam et les options de jeu Steam

Il existe un grand nombre de fonctions différentes liées à l' Steam, nous les avons donc divisées dans les sections suivantes pour faciliter la navigation:

  • API Steam générale
  • La superposition de Steam
  • Classements
  • Réalisations et statistiques
  • Nuage de Steam
  • DLC (contenu téléchargeable)
  • UGC (contenu généré par l'utilisateur)
