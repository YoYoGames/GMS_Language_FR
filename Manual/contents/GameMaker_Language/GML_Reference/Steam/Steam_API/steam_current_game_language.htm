steam _current_game_language

Cette fonction permet de récupérer la langue courante utilisée par Steam (sous forme de chaîne), par exemple "anglais".

Syntaxe:

steam_current_game_language();

Retour:

String

Exemple:

language = steam_current_game_language();

Le code ci-dessus obtient la langue utilisée pour le jeu en cours.

Retour: API Steam générale
