API général de la vapeur

L'ensemble des fonctions suivantes servent à vérifier la disponibilité de certains aspects de l'API du client ou du serveur Steam, et renvoient soit vrai, soit faux, en fonction de facteurs comme l'occupation du serveur, la connexion de l'utilisateur, etc. comme le serveur étant occupé ou l'utilisateur étant connecté etc... Cela signifie que ces fonctions doivent être utilisées avant tout autre appel de fonction de l'API Steam pour s'assurer que la configuration client/serveur est correcte et qu'elle communique avec votre jeu :

  • vapeur_initialisé
  • steam_stats_ready
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • steam_get_persona_name
  • steam_get_user_persona_name
  • steam_is_user_logged_on
  • steam_langue_du_jeu_actuel
  • steam_available_languages
