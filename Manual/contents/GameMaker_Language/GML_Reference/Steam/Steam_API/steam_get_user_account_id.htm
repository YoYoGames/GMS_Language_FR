steam_get_user_account_id

Cette fonction est utilisée pour récupérer l'identifiant unique que Steam attribue à chaque utilisateur, qui est nécessaire pour utiliser certaines des fonctions de contenu généré par l'utilisateur.

Syntaxe :

steam_get_user_account_id() ;

Les retours :

Real

Exemple :

global.user_id = steam_get_user_account_id() ;

Le code ci-dessus récupère l'ID utilisateur unique de la personne qui possède le jeu et le stocke dans une variable globale.
