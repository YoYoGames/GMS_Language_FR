steam _initialisé

Lors de l'utilisation de l' Steam, cette fonction peut être appelée pour vérifier que l' Steam a été correctement initialisée avant de faire tout autre appel aux Steam dans votre jeu.

Syntaxe:

steam_initialised();

Retour:

Boolean

Exemple:

global.steam_api = false;
if steam_initialised()
    {
    if steam_stats_ready() && steam_is_overlay_enabled()
        {
        global.steam_api = true;
        }
    }

Le code ci-dessus définira une variable globale sur true si l' Steam est correctement initialisée, avec les Steam et la fonctionnalité de superposition, ou elle définira la variable sur false autrement.

Retour: API Steam générale
