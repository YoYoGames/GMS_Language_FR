vapeur_initialisé

Lorsque vous utilisez l'API Steam, cette fonction peut être appelée pour vérifier que l'API client Steam a été initialisée correctement avant d'effectuer tout autre appel à des fonctions spécifiques à Steam dans votre jeu.

Syntaxe :

steam_initialised() ;

Les retours :

Booléen

Exemple :

global.steam_api= false ;
if steam_initialised()
{
if steam_stats_ready() && steam_is_overlay_enabled()
{
global.steam_api= true ;
}
}

Le code ci-dessus définira une variable globale à true si l'API client Steam est correctement initialisée, ainsi que les statistiques Steam et la fonctionnalité de superposition, ou il définira la variable à false sinon.
