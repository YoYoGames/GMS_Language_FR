steam _get_user_ steam _id

Vous pouvez utiliser cette fonction pour renvoyer l' identifiant Steam de l'utilisateur actuellement connecté au client Steam Il s'agit d'une valeur unique de 64 bits qui peut être utilisée pour identifier des utilisateurs individuels et non leur nom d'utilisateur à l'écran (cela peut être trouvé à l'aide de la fonction steam_get_persona_name() ). Comme il s'agit d'un int64, vous devrez peut-être le convertir en tant que tel lors de son utilisation dans des fonctions ultérieures en utilisant le int64() fonction.

Syntaxe:

steam_get_user_steam_id();

Retour:

Integer (int64)

Exemple:

if steam_initialised()
    {
    global.u_id = steam_get_user_steam_id();
    }

Le code ci-dessus définira une variable globale sur l' Steam unique de l'utilisateur actuel si l' Steam est correctement initialisée.
