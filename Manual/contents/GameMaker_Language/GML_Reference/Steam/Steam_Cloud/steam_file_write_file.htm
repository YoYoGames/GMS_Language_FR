steam_file_write_file

Avec cette fonction, vous pouvez copier le contenu d'un fichier enregistré localement vers un fichier synchronisé avec le Steam Cloud. Le fichier local doit exister avant d'utiliser cette fonction, qui renvoie une valeur de 0 si elle échoue pour une raison quelconque et une valeur supérieure à 0 si elle réussit. une valeur supérieure à 0 si elle réussit.

Syntaxe :

steam_file_write_file(steam_filename, local_filename) ;

Arguments Description
nom du fichier vapeur Le fichier Steam Cloud à copier.
nom_de_film_local Le fichier local à utiliser pour la copie.

Les retours :

réel

Exemple :

steam_file_write_file("rm_koala.png", "Koala2.png") ;

Le code ci-dessus copiera le contenu du fichier "Koala2.png" dans le fichier "rm_koala.png" du Steam Cloud.
