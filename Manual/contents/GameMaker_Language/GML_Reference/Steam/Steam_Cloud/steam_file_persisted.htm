steam _file_persisted

Avec cette fonction, vous pouvez vérifier le fichier donné pour voir s'il a été synchronisé avec le Steam Cloud. Une valeur de retour de true signifie que c'est le cas, tandis que false signifie que ce n'est pas le cas.

Syntaxe:

steam_file_persisted(filename);

Argument Description
filename Le nom du fichier à vérifier.

Retour:

Boolean

Exemple:

if !steam_file_persisted("Save.txt")
    {
    steam_file_share("Save.txt");
    }

Le code ci-dessus vérifiera si un fichier a été stocké sur le Steam Cloud, et si ce n'est pas le cas, il le synchronisera.

Retour: Steam Cloud
