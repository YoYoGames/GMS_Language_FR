steam_file_persisted

Avec cette fonction, vous pouvez vérifier le fichier donné pour voir s'il a été synchronisé avec le Steam Cloud. Une valeur de retour de true signifie que c'est le cas, tandis que false signifie que ça ne l'est pas.

Syntaxe :

steam_file_persisted(nom du fichier) ;

Arguments Description
nom de fichier Le nom du fichier à vérifier.

Les retours :

Booléen

Exemple :

if !steam_file_persisted("Save.txt")
{
steam_file_share("Save.txt") ;
}

Le code ci-dessus vérifiera si un fichier a été stocké dans le Steam Cloud, et s'il ne l'a pas été, il le synchronisera.
