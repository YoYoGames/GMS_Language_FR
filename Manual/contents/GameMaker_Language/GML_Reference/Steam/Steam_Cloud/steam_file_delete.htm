steam _file_delete

Cette fonction supprimera le fichier donné du Steam Cloud. La fonction renverra une valeur de 0 si elle échoue pour une raison quelconque et une valeur supérieure à 0 si elle réussit.

Syntaxe:

steam_file_delete(filename);

Argument Description
filename Le nom du fichier à supprimer.

Retour:

Real

Exemple:

if steam_file_exists("Save.txt")
    {
    steam_file_delete("Save.txt");
    }

Le code ci-dessus vérifiera si un fichier existe, et si c'est le cas, il supprime le fichier du Steam Cloud.
