steam_file_exists

Avec cette fonction, vous pouvez vérifier si un fichier du Steam Cloud existe ou non, avec une valeur de retour de true s'il existe, ou false sinon.

Syntaxe :

steam_file_exists(nom du fichier) ;

Arguments Description
nom de fichier Le nom du fichier à vérifier.

Les retours :

Booléen

Exemple :

if steam_file_exists("Save.txt")
{
save_str = steam_file_read("Save.txt") ;
}

Le code ci-dessus vérifie si un fichier existe sur le Steam Cloud et si c'est le cas, il l'ouvre et lit son contenu dans la variable "save_str".
