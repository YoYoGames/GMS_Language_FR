steam _file_exists

Avec cette fonction, vous pouvez vérifier si un fichier du Steam Cloud existe ou non, avec une valeur de retour de true s'il existe, ou false autrement.

Syntaxe:

steam_file_exists(filename);

Argument Description
filename Le nom du fichier à rechercher.

Retour:

Boolean

Exemple:

if steam_file_exists("Save.txt")
    {
    save_str = steam_file_read("Save.txt");
    }

Le code ci-dessus vérifie si un fichier existe sur le Steam Cloud et si c'est le cas, il l'ouvre et lit son contenu dans la variable "save_str".

Retour: Steam Cloud
