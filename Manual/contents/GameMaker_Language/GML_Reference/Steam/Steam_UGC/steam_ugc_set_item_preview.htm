steam_ugc_set_item_preview

Cette fonction va définir l'image de prévisualisation à utiliser pour l'élément donné. L'image doit être fournie sous la forme d'un fichier au format PNG, JPG ou GIF d'une taille maximale de 1 Mo. Le chemin d'accès à l'image doit être un chemin relatif dans le dossier de sauvegarde ou le paquet de jeu (c'est-à-dire un chemin d'accès relatif). (c'est-à-dire un fichier inclus). Le gestionnaire de mise à jour est la valeur renvoyée lorsque vous avez appelé la fonction steam_ugc_start_item_update(). La fonction renvoie un message vrai si l'API a été accédée avec succès et un message faux en cas de problème. et faux s'il y a eu un problème.

Syntaxe :

steam_ugc_set_item_preview(ugc_update_handle, preview) ;

Arguments Description
ugc_update_handle L'identifiant unique de l'UGC à mettre à jour.
aperçu L'image d'aperçu (JPG, GIF ou PNG - taille maximale de 1 Mo) à utiliser pour l'article.

Les retours :

Booléen

Exemple :

var app_id = steam_get_app_id() ;
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID) ;
steam_ugc_set_item_title(updateHandle, "Mon article d'atelier(3) !") ;
steam_ugc_set_item_description( updateHandle, "testant atelier...") ;
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public) ;
var tagArray ;
tagArray[0] = "Test" ;
tagArray[1] = "New" ;
steam_ugc_set_item_tags( updateHandle, tagArray) ;
steam_ugc_set_item_preview (updateHandle, "promo.jpg") ;
steam_ugc_set_item_content(updateHandle, "WorkshopContent1") ;
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2") ;

Le code ci-dessus récupère l'ID du jeu, puis l'utilise avec l'ID du fichier publié précédemment stocké pour générer un identifiant de mise à jour pour l'élément. Ce gestionnaire est ensuite utilisé pour mettre à jour divers éléments d'information avant que la mise à jour ne soit envoyée aux serveurs de l'atelier. des serveurs.
