steam _ugc_set_item_visibility

Cette fonction définira la visibilité de l'élément donné, définie à l'aide de l'une des constantes suivantes:

Constant Description
ugc_visibility_public Définir l'élément pour qu'il soit visible publiquement
ugc_visibility_friends_only Définir l'élément pour qu'il ne soit visible que par les personnes figurant dans la liste d'amis des utilisateurs
ugc_visibility_private Définir l'élément comme privé

Le handle de mise à jour est la valeur renvoyée lorsque vous avez appelé la fonction steam_ugc_start_item_update(), et la fonction retournera true si l'accès à l'API a réussi et false s'il y avait un problème.

Syntaxe:

steam_ugc_set_item_visibility(ugc_update_handle, visibility);

Argument Description
ugc_update_handle Le handle unique de l'UGC à mettre à jour.
visibility La visibilité (une constante, répertoriée ci-dessous) à utiliser pour l'élément.

Retour:

Boolean

Exemple:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "My workshop item(3)!");
steam_ugc_set_item_description( updateHandle, "testing workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "New";
steam_ugc_set_item_tags(updateHandle, tagArray);
steam_ugc_set_item_preview(updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

Le code ci-dessus obtient l'ID du jeu, puis l'utilise avec un ID de fichier publié précédemment stocké pour générer une poignée de mise à jour pour l'élément. Cette poignée est ensuite utilisée pour mettre à jour diverses informations avant que la mise à jour ne soit transmise aux serveurs de l'Atelier.

Retour: UGC
