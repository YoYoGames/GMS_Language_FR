steam_ugc_download

Cette fonction vous permet de télécharger une image de prévisualisation pour tout élément UGC donné. L'ugc_handle est la valeur d'identification unique de l'image (que vous pouvez obtenir à l'aide de la fonction steam_ugc_send_query()), et le nom du fichier de destination est le nom (et le chemin d'accès local dans la sandbox Steam ) que vous souhaitez donner au fichier image une fois le téléchargement terminé.

Lorsque cette fonction est utilisée, elle déclenche un événement asynchroneSteam pour rapporter les détails du fichier image demandé contenant les paires clé/valeur suivantes dans le champ async_load ds_map :

  • "id" - L'identifiant asynchrone renvoyé par la fonction appelante.
  • "resultat" - Le résultat de l'opération (une valeur réelle). Ce sera soit la constante GML ugc_result_success ou un autre nombre réel. Vous devez donc vérifier cette constante pour vous assurer que l'appel a réussi, et si sinon, quelque chose n'a pas fonctionné correctement. Le reste des valeurs possibles retournées sont affichées comme le résultat de la valeur "EResult" de Steam et vous devriez voir steamclientpublic .h dans les en-têtes du SDK pour les 89 valeurs possibles.
  • "event_type" - Cette clé contient la valeur "ugc_download".
  • "original_filename" - Cette clé contient le nom original du fichier image sur le serveur (une chaîne de caractères ).
  • "dest_filename" - Cette clé contient le nom du fichier image que vous avez transmis (une chaîne).
  • "ugc_handle" - Cette clé contient la valeur ugc_handle que vous avez passée à la fonction appelante.

Syntaxe :

steam_ugc_download(ugc_handle, dest_filename) ;

Arguments Description
ugc_handle L'identifiant unique de l'aperçu à télécharger.
nom du fichier de destination Le nom du fichier avec lequel enregistrer l'aperçu.

Les retours :

ID asynchrone

Exemple étendu :

Dans cet exemple, nous appelons d'abord la fonction et stockons la valeur de l'ID asynchrone dans une variable :

steam_get= steam_ugc_download(steam_handle, "\UGC\Preview_file.png") ;

Cela enverra ensuite une demande de fichier à l'API Steam, générant un événement asynchrone que nous traiterons comme suit :

var event_id = async_load[ ? "id"] ;
if event_id == steam_get
{
var type = async_load[ ? "event_type"] ;
if type == "ugc_download"
{
sprite_delete(preview_sprite) ; preview_sprite = sprite_add(async_load[ ? "dest_filename"], 0, false, false, 0, 0) ;
}
}

Le code ci-dessus vérifie le type d'événement et crée ensuite un sprite à partir de l'image téléchargée.
