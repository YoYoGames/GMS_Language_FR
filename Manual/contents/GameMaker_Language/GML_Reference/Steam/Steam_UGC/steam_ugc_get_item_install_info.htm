steam_ugc_get_item_install_info

Cette fonction peut être utilisée pour récupérer des informations sur un élément de fichier publié donné qui a été souscrit et téléchargé dans la zone de stockage locale de Steam pour votre jeu. Vous donnez l'ID de l'élément et fournissez l'index d'une carte DS vide qui sera ensuite remplie avec les paires clé/valeur suivantes si l'élément existe :

  • "size_on_disk" - La taille du fichier sur le disque (en octets).
  • " legacy_item" - Sera vrai ou faux selon qu'il s'agit d'un fichier ancien ou non.
  • "folder" - Il s'agit du chemin complet vers le contenu installé (veuillez vous référer à "Item Installation" dans la documentation du SDK Steam, car les éléments "anciens " téléchargés avec l'ancienne méthode sont traités différemment).

Si l'élément existe, la fonction renvoie true et remplit la carte, sinon elle renvoie false et la carte reste vide.

Syntaxe :

steam_ugc_get_item_install_info(published_file_id, info_map) ;

Arguments Description
fichier_publié_id L'identifiant unique de l'UGC à mettre à jour.
info_map Un index de carte DS (précédemment créé).

Les retours :

Booléen

Exemple :

var item_map = ds_map_create() ;
steam_ugc_get_item_install_info(global.fileID, item_map) ;

Le code ci-dessus interroge l'état d'installation de l'élément indexé dans la variable globale "fileID", en utilisant un DS Map pour stocker l'information.

Retour : UGC
