steam_download_scores_autour_de_l'utilisateur

Cette fonction est utilisée pour récupérer les entrées du classement relatives à l'entrée de l'utilisateur actuel. Le paramètre range_start est le nombre d'entrées à récupérer avant l' entrée de l'utilisateur actuel, et le paramètre range_end est le nombre d'entrées après l'entrée de l'utilisateur actuel. d'entrées après l'entrée de l'utilisateur actuel, et l'entrée de l'utilisateur actuel est toujours incluse dans les résultats. Par exemple, si l'utilisateur actuel est numéro 5 dans un classement donné, le fait de définir la plage de début à -2 et la plage de fin à 2, vous obtiendrez 5 entrées : 3 à 7. S'il n'y a pas assez d'entrées dans le classement avant ou après l'entrée de l'utilisateur, Steam ajustera la plage pour essayer de retourner le nombre d'entrées demandé. Par exemple, si l'utilisateur est numéro 1 au classement, le début est défini à -2 et la fin à 2, Steam renverra les 5 premières entrées du classement.

Syntaxe :

steam_download_scores_around_user(lb_name, range_start, range_end) ;

Arguments Description
lb_name Le nom du classement à partir duquel vous téléchargez les scores (une chaîne).
Début de plage La position de départ dans le classement.
fin de plage La position finale dans le classement.

Les retours :

Real

Exemple :

score_get = steam_download_scores_around_user("Game Scores", -4, 5) ;

Cette méthode enverra une requête au serveur Steam pour une série de 10 scores du classement "Game Scores", centrée sur le joueur et stockera l'identifiant asynchrone de la requête dans la variable "score_get". Celle-ci sera ensuite traitée dans l'application Événement asynchroneSteam comme indiqué dans l'exemple étendu pour steam_download_scores().
