buffer_save

Avec cette fonction, vous pouvez enregistrer le contenu d'un tampon dans un fichier, prêt à être relu en mémoire à l'aide du buffer_load() fonction.

REMARQUE: sur HTML5, le contenu du tampon sera enregistré sous forme de chaînes codées en base64 lors de l'utilisation de cette fonction.

Syntaxe:

buffer_save(buffer, filename);

Argument Description
buffer L'index du tampon à enregistrer.
filename Le nom du fichier sous lequel enregistrer.

Retour:

N/A

Exemple:

buffer_save(buff, "Player_Save.sav");

Enregistre le contenu actuel du tampon avec l'ID stocké dans la variable "buff" dans un fichier.

Retour: Tampons
