tampon_async_groupe_début

Cette fonction est appelée lorsque vous voulez commencer la sauvegarde de plusieurs tampons vers plusieurs fichiers. Le "groupname" est une chaîne de caractères et sera utilisé comme nom de répertoire pour l'endroit où les fichiers seront sauvegardés, et devra être utilisé comme partie du chemin d'accès au fichier lors du chargement des fichiers dans le système. lorsque vous rechargerez les fichiers dans l'IDE par la suite (en utilisant l'une des fonctions buffer_load() ). Cette fonction est uniquement à utiliser avec la buffer_save_async() fonction et vous devez également terminer la définition de la sauvegarde en appelant la fonction buffer_async_group_end() sinon les fichiers ne seront pas sauvegardés.

Notez que pour les plateformes de console (comme la PS4 par exemple), le "nom du groupe" sera utilisé comme description de l'emplacement de sauvegarde, et l'utilisation de cette fonction peut vous aider à éviter d'avoir l'interface utilisateur affichée pour chaque fichier qui est sauvegardé. Notez également que lorsque vous l'utilisation de l'UWP, vous pouvez faire circuler vos sauvegardes entre les fenêtres natives de la Xbox et de la boutique Microsoft en créant un groupe identique à un groupe ERA Xbox, par exemple :

if os_type == os_uwp
{
buffer_async_group_begin("root/yourgroupname") ;
}
else
{
buffer_async_group_begin("votre nom de groupe") ;
}

Syntaxe :

buffer_async_group_begin(nom du groupe) ;

Arguments Description
nom de groupe Le nom du groupe (sous forme de chaîne).

Les retours :

N/A

Exemple :

buffer_async_group_begin("SaveGame") ;
save1 = buffer_save_async(buff1, "Player_Save1.sav", 0, 16384) ;
save2 = buffer_save_async(buff2, "Player_Save2.sav", 0, 16384) ;
save3 = buffer_save_async(buff3, "Player_Save3.sav", 0, 16384) ;
save4 = buffer_save_async(buff4, "Player_Save4.sav", 0, 16384) ;
buffer_async_group_end() ;

Le code ci-dessus démarre un groupe de tampons puis le définit pour sauvegarder 4 fichiers de manière asynchrone. La définition du groupe est ensuite terminée (à ce moment-là, la sauvegarde commencera).
