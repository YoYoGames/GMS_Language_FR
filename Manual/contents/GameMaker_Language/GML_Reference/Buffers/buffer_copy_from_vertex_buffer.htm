buffer_copy_from_vertex_buffer

Cette fonction peut être utilisée pour copier une partie (ou la totalité) des données de sommets stockées dans un tampon de sommets dans un tampon régulier précédemment créé. Lors de la copie d'un tampon de sommets dans un tampon normal avec cette fonction, les deux tampons doivent avoir été créés au préalable (en utilisant le vertex_create_buffer() et buffer_create() fonctions, par exemple). Vous pouvez spécifier la plage de données de sommets que vous souhaitez copier dans le tampon, où le sommet de départ peut être n'importe où entre 0 et le nombre de sommets -1, et vous pouvez donner le nombre de sommets à copier à partir de ce point. Vous pouvez utiliser la fonction vertex_get_number() sur le tampon de sommets pour obtenir le nombre total de sommets stockés. Enfin, vous donnez l'index de tampon dans lequel copier les données de sommet, ainsi qu'un décalage de données pour définir la position vers laquelle copier les données de sommet dans le tampon de destination.

Syntaxe:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

Argument Description
vertex_buffer Index du tampon de sommets à partir duquel copier.
start_vertex Le sommet de départ.
num_vertices Le nombre total de sommets à utiliser.
dest_buffer L'index de la mémoire tampon pour copier à.
dest_offset Position de décalage vers laquelle copier les données (en octets).

Retour:

N/A

Exemple:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

Le code ci-dessus copiera les données de vertex stockées dans le tampon de vertex indexé dans la variable "model_buffer", puis les collera dans le tampon indexé dans la variable "player_buffer".
