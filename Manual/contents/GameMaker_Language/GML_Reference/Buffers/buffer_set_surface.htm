buffer_set_surface

Avec cette fonction, vous pouvez écrire les informations d'un tampon sur une surface donnée. Le tampon en cours de lecture et la surface en cours d'écriture doivent avoir été créés précédemment, et vous pouvez fournir un décalage dans le tampon pour commencer la lecture. Notez que la lecture commencera toujours au début du tampon plus la valeur de décalage et non à la position de recherche actuelle plus la valeur de décalage.

REMARQUE: cette fonction a changé depuis la mise à jour 2.3.1 de GameMaker Studio Les versions précédentes utilisaient la constante buffer_surface_copy, qui est désormais obsolète.

Syntaxe:

buffer_set_surface(buffer, surface, offset);

Argument Description
buffer L'index du tampon à utiliser.
surface L'indice de la surface à utiliser.
offset La valeur de décalage des données.

Retour:

N/A

Exemple:

buffer_set_surface(buff, application_surface, 0);

Le code ci-dessus copiera toutes les données stockées dans le tampon indexé dans la variable "buff" vers la surface d'application sans décalage.
