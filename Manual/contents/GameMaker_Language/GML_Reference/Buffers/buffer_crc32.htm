buffer_crc32

Cette fonction prend les données d'entrée d'un tampon et renvoie un hachage de somme de contrôle crc32. Vous spécifiez l'ID du tampon à utiliser, puis une valeur de décalage (en octets) pour savoir où commencer, et enfin une taille (toujours en octets) pour la région à hacher. la fonction renvoie une valeur entière de 32 bits pour cette région.

Syntaxe :

buffer_crc32(buffer, offset, size) ;

Arguments Description
tampon L'index du tampon à utiliser.
décalage La valeur du décalage des données.
taille La taille du tampon.

Les retours :

Int 32

Exemple :

check_val = buffer_crc32(buff, 0, buffer_get_size(buff)) ;

Le code ci-dessus créera une valeur de hachage de checksum crc32 pour les données complètes stockées dans le tampon indexé par la variable "buff", et stockera la valeur de hachage entière retournée dans la variable "check_val".
