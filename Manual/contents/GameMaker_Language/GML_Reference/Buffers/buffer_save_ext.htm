tampon_save_ext

Avec cette fonction, vous pouvez sauvegarder une partie du contenu d'un tampon dans un fichier, prêt à être relu en mémoire à l'aide de la fonction buffer_load() en mémoire. L'"offset" définit la position de départ dans la mémoire tampon pour la sauvegarde (en octets), et la "taille" est la taille de la zone du tampon à sauvegarder à partir de ce décalage (également en octets).

Syntaxe :

buffer_save_ext(buffer, filename, offset, size) ;

Arguments Description
tampon L'index du tampon à sauvegarder.
nom de fichier Le nom du fichier à enregistrer sous.
décalage Le décalage dans la mémoire tampon à sauvegarder (en octets).
taille La taille de la zone tampon à sauvegarder (en octets).

Les retours :

N/A

Exemple :

buffer_save_ext(buff, "Player_Save.sav", 0, 16384) ;

Sauvegarde une partie du contenu actuel du tampon avec l'id stocké dans la variable "buff" dans un fichier.
