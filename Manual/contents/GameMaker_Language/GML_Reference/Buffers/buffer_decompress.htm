tampon_décompression

Avec cette fonction, vous pouvez décompresser un tampon précédemment compressé en utilisant la compression zlib. Vous fournissez l'ID du tampon à décompresser, et la fonction renverra un nouvel ID de tampon contenant les données non compressées. données non compressées. Si la décompression a échoué (par exemple, vous fournissez un tampon qui n'a pas été compressé), la fonction renverra une valeur inférieure à 0.

Syntaxe :

buffer_decompress(buffer) ;

Arguments Description
tampon L'index du tampon à décompresser.

Les retours :

ID de la mémoire tampon

Exemple :

var cmpBuff = buffer_load("Player_Save.sav") ;
var srcBuff = buffer_decompress(cmpBuff) ;
global.DataString = buffer_read(srcBuff, buffer_string) ;

Le code ci-dessus va d'abord charger un tampon enregistré, puis le décompresser et enfin lire les données de la chaîne du tampon décompressé dans une variable globale.
