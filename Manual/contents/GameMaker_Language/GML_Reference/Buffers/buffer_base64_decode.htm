buffer_base64_decode

Avec cette fonction, vous pouvez décoder une chaîne de caractères encodée en base64 (créée à l'aide de la fonction buffer_base64_encode() ) dans un tampon. Cette fonction va créer le tampon (comme un tampon "grow" aligné sur 1 octet) et renvoie l'index unique pour le tampon qui doit être utilisé dans tous les autres appels de fonction.

REMARQUE: il est important de supprimer de la mémoire toute ressource créée dynamiquement comme celle-ci lorsque vous n'en avez plus besoin afin d'éviter les fuites de mémoire. Ainsi, lorsque vous avez fini d'utiliser le tampon que vous avez créé, vous devez le libérer à nouveau. en utilisant buffer_delete().

Syntaxe :

buffer_base64_decode(string) ;

Arguments Description
chaîne de caractères La chaîne encodée en base64 à décoder

Les retours :

Real

Exemple :

ini_open("Save.ini") ;
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", "")) ;
ini_close() ;

Le code ci-dessus ouvrira un fichier ini, puis lira une chaîne de caractères dans la fonction de décodage. Cette fonction renverra un index de tampon, qui est stocké dans la variable "buff", contenant les données précédemment encodées et sauvegardées. Le fichier ini est ensuite fermé.
