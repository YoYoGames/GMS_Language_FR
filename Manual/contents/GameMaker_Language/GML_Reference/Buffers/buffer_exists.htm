tampon_existant

Cette fonction peut être utilisée pour vérifier une variable pour voir si elle contient une valeur d'ID de tampon valide ou non. Si c'est le cas, la fonction renvoie true, sinon elle renvoie false.

Syntaxe :

buffer_exists(buffer)

Arguments Description
tampon L'index du tampon à vérifier.

Les retours :

Booléen

Exemple :

if buffer_exists(buff)
{
buffer_delete(buff) ;
}

Le code ci-dessus vérifie si la variable " buff " contient un ID de tampon et si c'est le cas, le tampon est supprimé.
