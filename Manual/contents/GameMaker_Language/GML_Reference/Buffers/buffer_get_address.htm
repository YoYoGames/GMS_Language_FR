buffer_get_address

Avec cette fonction, vous pouvez obtenir le pointeur a vers l'adresse de tampon brute et alignée. Ceci est principalement destiné à être utilisé avec les extensions car vous pouvez leur transmettre cette valeur, ce qui leur permet d'accéder aux données de la mémoire tampon. Notez qu'il y a certaines choses que vous ne pouvez pas faire avec le tampon lors de l'accès à partir d'une extension:

  • Vous ne pouvez pas libérer la mémoire.
  • Vous ne pouvez pas redimensionner la mémoire.
  • Vous ne pouvez pas ne pas écrire au-delà de l'adresse du tampon plus la taille du tampon (vous pouvez utiliser le buffer_get_size() fonction pour cela) ou vous obtiendrez une erreur hors limites.

Syntaxe:

buffer_get_address(buffer);

Argument Description
buffer L'index du tampon à utiliser.

Retour:

pointer

Exemple:

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

Le code ci-dessus obtiendra l'adresse mémoire du tampon indexé dans la variable "buff_model" puis obtiendra l'adresse mémoire pour la fin du tampon en utilisant le buffer_get_size() fonction, en stockant les deux valeurs dans des variables locales pour une utilisation ultérieure.
