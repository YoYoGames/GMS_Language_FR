buffer_resize

Avec cette fonction, vous pouvez redimensionner un tampon donné à la taille (en octets) que vous spécifiez.

Syntaxe:

buffer_resize(buffer, newsize);

Argument Description
buffer Index de la mémoire tampon dont la taille doit être modifiée.
newsize La nouvelle taille du tampon (en octets).

Retour:

N/A

Exemple:

if (buffer_get_size(buff) < 16384)
    {
    buffer_resize(buff, 16384);
    }

Le code ci-dessus vérifiera la taille du tampon indexé dans la variable "buff" et si elle est inférieure à la valeur donnée, le tampon est redimensionné.
