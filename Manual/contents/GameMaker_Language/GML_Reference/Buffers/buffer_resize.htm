redimensionnement du tampon

Avec cette fonction, vous pouvez redimensionner un tampon donné pour qu'il ait la taille (en octets) que vous spécifiez.

Syntaxe :

buffer_resize(buffer, newsize) ;

Arguments Description
tampon L'index du tampon dont on veut changer la taille.
nouvelle taille La nouvelle taille du tampon (en octets).

Les retours :

N/A

Exemple :

si (buffer_get_size(buff) < 16384)
{
buffer_resize(buff, 16384) ;
}

Le code ci-dessus va vérifier la taille du tampon indexé dans la variable "buff" et si elle est inférieure à la valeur donnée, le tampon est redimensionné.
