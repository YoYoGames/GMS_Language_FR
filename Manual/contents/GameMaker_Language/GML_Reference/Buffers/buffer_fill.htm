buffer_fill

Cette fonction peut être utilisée pour remplir un tampon créé précédemment avec un type de données et une valeur donnés. Les données avec lesquelles vous remplissez le tampon doivent être en accord avec l'argument "type" de cette fonction, ce qui signifie que vous ne pouvez pas essayer de remplir avec une chaîne et d'utiliser le type entier 16 bits non signé, par exemple. Les constantes de type sont les mêmes que celles utilisées par le buffer_read() et buffer_write() les fonctions. La "taille" est la taille du tampon (en octets) que vous souhaitez remplir, tandis que le décalage est la valeur de décalage (également en octets) à partir du début du tampon à partir duquel commencer le remplissage.

Syntaxe:

buffer_fill(buffer, offset, type, value, size);

Argument Description
buffer L'index du tampon à remplir.
offset La valeur de décalage des données (en octets).
type Le type de données à écrire dans le tampon (voir la liste des constantes ici ).
value Les données à écrire.
size La taille du tampon (en octets) que vous souhaitez remplir.

Retour:

N/A

Exemple:

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

Le code ci-dessus trouve le début du tampon avec l'ID stocké dans la variable "buff" et y écrit une série de valeurs entières 16 bits signées.
