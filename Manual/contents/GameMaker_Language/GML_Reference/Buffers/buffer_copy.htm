buffer_copy

Cette fonction peut être utilisée pour copier un segment (ou la totalité) des données stockées dans un tampon vers un autre. Lors de l'utilisation de deux tampons et de la copie de l'un à l'autre, les deux tampons doivent avoir été créés au préalable à l'aide du buffer_create() (par exemple), et vous pouvez spécifier un décalage de données (en octets) pour le point de départ des données à copier à partir du tampon source par rapport au début du tampon, ainsi qu'un autre décalage de données pour définir la position à copiez les données dans la mémoire tampon de destination.

REMARQUE: vous ne pouvez pas copier dans le même tampon.

Syntaxe:

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

Argument Description
src_buffer L'index du tampon à partir duquel copier.
src_offset Décalage des données à partir duquel commencer la copie (en octets).
size La taille des données à copier (en octets).
dest_buffer L'index de la mémoire tampon pour copier à.
dest_offset Position de décalage vers laquelle copier les données (en octets).

Retour:

N/A

Exemple:

buffer_copy(buff1, 0, 2048, buff2, 2048);

Le code ci-dessus copiera les données stockées dans le tampon indexé dans la variable "buff1", puis les collera dans le tampon indexé dans la variable "buff2", mais décalé de 2048 octets.

Retour: Tampons
