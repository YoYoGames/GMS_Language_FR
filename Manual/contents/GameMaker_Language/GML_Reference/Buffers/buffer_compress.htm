tampon_compressé

Avec cette fonction, vous pouvez compresser une partie (ou la totalité) d'un tampon en utilisant la compression zlib. Vous fournissez l'ID du tampon à compresser (tel que renvoyé par la fonction buffer_create()), le décalage dans le tampon à utiliser en octets, et la taille des données du tampon à compresser (également en octets). La fonction renverra une nouvelle valeur d'ID de tampon pour le tampon compressé, ou une valeur inférieure à 0 si elle a échoué pour une raison quelconque. Cette fonction ne modifiera pas le tampon d'origine.

Syntaxe :

buffer_compress(buffer, offset, size) ;

Arguments Description
tampon L'index du tampon à compresser.
décalage Le décalage dans le tampon à compresser (en octets).
taille La taille de la zone tampon à compresser (en octets).

Les retours :

Int (valeur de l'ID du tampon)

Exemple :

var srcBuff = buffer_create(1024, buffer_grow, 1) ;
buffer_write(srcBuff, global.DataString) ;
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff)) ;
buffer_save(cmpBuff, "Player_Save.sav") ;
buffer_delete(srcBuff) ;
buffer_delete(cmpBuff) ;

Le code ci-dessus va créer un tampon puis le remplir avec les données d'une chaîne. Ce tampon est ensuite compressé et sauvegardé, puis les tampons source et compressé sont supprimés.
