ensemble_de_structures_variables

Avec cette fonction, vous pouvez définir la valeur d'une variable donnée dans une structure. Vous fournissez la référence de la structure ainsi que le nom de la variable dont vous voulez définir la valeur sous la forme d'une chaîne de caractères (voir l'exemple de code ci-dessous), et enfin la valeur à définir (peut être n'importe quel type de données valide). tout type de données valide). Si la variable n'existe pas déjà dans la structure, elle sera créée et la valeur lui sera attribuée.

Syntaxe :

variable_struct_set(struct, name, val) ;

Arguments Description
struct La référence de la structure à définir
nom Le nom de la variable à définir (sous forme de chaîne)
val La valeur à donner à la variable

Les retours :

N/A

Exemple :

if !variable_struct_exists(mystruct, "shields")
{
variable_struct_set(mystruct, "shields", 0) ;
}

Le code ci-dessus va vérifier si la variable donnée existe dans la structure donnée et si ce n'est pas le cas, elle est créée et mise à 0.
