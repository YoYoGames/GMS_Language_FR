bool

Cette fonction tente de convertir une valeur donnée en un type de données booléen, où la valeur sera renvoyée comme vraie si elle est supérieure à 0,5 et fausse sinon.

Syntaxe :

bool(n) ;

Arguments Description
n La valeur à convertir.

Les retours :

Booléen

Exemple :

if !is_bool(val)
{
val = bool(val) ;
}

Le code ci-dessus vérifie la variable "val" pour voir si c'est un booléen et si ce n'est pas le cas, elle est convertie en un booléen.
