tableau_get

Avec cette fonction, vous pouvez récupérer la valeur d'un index dans un tableau. La fonction nécessite que vous fournissiez une variable qui contient le tableau et l'index pour récupérer la valeur dans ce tableau. Notez que si l'indice du tableau donné est hors limites alors le jeu se plantera avec une erreur. Cette fonction peut également être utilisée pour des tableaux multidimensionnels, à condition de spécifier quelle dimension vous souhaitez obtenir lorsque vous fournissez l'indice du tableau, en suivant ce modèle :

// Tableau 1D
array_get(my_array, 0) ;
// Tableau 2D
array_get(my_array[0], 0) ;
// Tableau 3D
array_get(my_array[0][0], 0) ;
// etc...

Syntaxe :

array_get(variable, index) ;

Arguments Description
variable La variable qui contient le tableau.
indice L'index du tableau pour obtenir la valeur.

Les retours :

Tout type de données valide qu'un tableau peut contenir.

Exemple :

for (var i = 0 ; i < 10 ; ++i ;)
{
show_debug_message(array_get(my_array, i)) ;
}

Le code ci-dessus affichera les 10 premiers éléments de la liste donnée dans la console.
