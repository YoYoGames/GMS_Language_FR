array_copy

Avec cette fonction, vous pouvez copier tout ou partie d'un tableau dans un autre tableau à n'importe quelle position. Vous devez fournir à la fois les tableaux source et de destination (les deux doivent avoir été créés précédemment), ainsi qu'une position dans le tableau source à partir de laquelle copier et une position dans le tableau de destination vers lequel copier. Enfin, vous devez spécifier la longueur du tableau (ou la longueur de la partie que vous souhaitez) à copier. Si les données copiées dépassent la longueur du tableau de destination, le tableau sera étendu pour accepter les données.

Cette fonction peut également être utilisée pour les tableaux multidimensionnels, à condition de spécifier la dimension à copier lorsque vous fournissez l'index du tableau, en suivant ce modèle:

// Copy from the first dimension of an array
// to the second dimension of an array
array_copy(item_array, 0, inventory_array[0], 0, len);

// Copy from the third dimension of an array
// to the first dimension of an array
array_copy(item_array[0][0], 0, inventory_array, 0, len);

// etc...

Syntaxe:

array_copy(dest, dest_index, src, src_index, length);

Argument Description
dest L'ID de la baie vers laquelle effectuer la copie.
dest_index L'index dans le tableau vers lequel copier.
src L'ID de la baie à partir de laquelle effectuer la copie.
src_index L'index avec le tableau à partir duquel commencer la copie.
length La longueur (nombre d'indices de tableau) à copier.

Retour:

N/A

Exemple:

if !array_equals(inventory_array, item_array)
   {
   var len = array_length(inventory_array);
   array_copy(item_array, 0, inventory_array, 0, len);
   }

Le code ci-dessus vérifiera deux tableaux pour voir s'ils contiennent des valeurs équivalentes, et si ce n'est pas le cas, le code copiera tout le contenu du tableau "inventaire_array" dans le tableau "item_array".
