array_insert

Avec cette fonction, vous pouvez insérer une valeur (ou des valeurs) dans un tableau à n'importe quelle position donnée. La fonction vous oblige à fournir une variable contenant le tableau, l'index (position) dans le tableau à insérer, ainsi qu'au moins une valeur à insérer, bien que vous puissiez éventuellement fournir d'autres arguments et ils seront tous insérés dans le tableau dans un ordre consécutif à partir de l'index donné. Notez que si l'index donné est en dehors de la longueur du tableau, les valeurs seront simplement ajoutées à l'index donné, et tous les "espaces" dans le tableau entre la dernière valeur et les valeurs insérées seront définis sur une valeur par défaut de 0.

Syntaxe:

array_insert(variable, index, value, [value], [value], [etc...]);

Argument Description
variable La variable qui contient le tableau.
index L'index (position) dans le tableau pour insérer la ou les valeurs
value La valeur à insérer
[value], [value], [etc...] OPTIONNEL! Autres valeurs à pousser sur le tableau

Retour:

N/A

Exemple:

array_insert(score_array, score_rank, current_core);

Le code ci-dessus insérera une valeur dans le tableau donné à l'index contenu dans la variable "score_rank".
