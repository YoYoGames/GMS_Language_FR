typeof

Cette fonction renvoie le type de données de toute variable donnée sous forme de chaîne. Les valeurs de retour possibles sont répertoriées dans le tableau ci-dessous:

Chaîne de caractères Description
number La variable contient un nombre réel (à virgule flottante) - cela peut inclure NaN et infinity
string La variable contient une chaîne
array La variable fait référence à un tableau
bool La variable contient un booléen (true /false )
int32 La variable contient un entier 32 bits
int64 La variable contient un entier de 64 bits
ptr La variable contient un pointeur
undefined La variable n'est pas définie
null La variable contient une valeur nulle (cela ne devrait pas être vu normalement)
vec3 La variable contient un vecteur de 3 valeurs
vec4 La variable contient un vecteur de 4 valeurs
method La variable contient une référence de fonction
struct La variable contient une référence struct
unknown La valeur est inconnue. Cela ne devrait jamais être vu et signifie que quelque chose s'est mal passé au niveau le plus élémentaire, comme un écrasement de mémoire

Veuillez noter qu'il existe des cas où cette fonction peut ne pas renvoyer la valeur correcte pour une méthode. Considérez les deux définitions de fonction suivantes:

a = function()
    {
    // something
    }

function b()
    {
    // Something
    }

Techniquement, ces deux méthodes sont considérées comme des méthodes car elles lient une fonction à une variable de portée d'instance, mais en appelant typeof() en fonction b renverra "number" et non "method", en l'appelant a retournera "méthode". Cela est dû au fait que les méthodes créées comme celle pour b se voient attribuer des indices de script (qui sont des valeurs entières), car c'est ainsi que le compilateur reconnaît les fonctions de script, même si la fonction n'a pas été définie dans un script.

Syntaxe:

typeof(variable);

Argument Description
variable La variable pour obtenir le type de données.

Retour:

String (see table, above)

Exemple:

var _str = typeof(global.ExtensionInput);
show_debug_message(" global.ExtensionInput is a " + _str);

Le code ci-dessus récupère le type de données détenu par la variable globale donnée et renvoie la chaîne à une variable locale qui est ensuite utilisée pour envoyer un message à la console.
