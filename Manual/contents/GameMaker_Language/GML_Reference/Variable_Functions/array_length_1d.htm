array_length_1d (obsolète)

ATTENTION! Cette fonction est obsolète car les tableaux ne sont plus limités à seulement 1 ou 2 dimensions, et en tant que telle, cette fonction n'est fournie que pour la prise en charge des projets hérités. Tous les nouveaux projets doivent utiliser la méthode actuelle de création et d'accès aux tableaux multidimensionnels (voir ici pour plus d'informations).

Avec cette fonction, vous pouvez obtenir la longueur (nombre d'entrées) d'un tableau 1D. Pour les tableaux 2D, vous devez utiliser le array_height_2d et array_length_2d les fonctions.

IMPORTANT!: Si le tableau a plus de 32 000 entrées, cette fonction renverra une valeur erronée et ne doit pas être utilisée.

Syntaxe:

array_length_1d(array_index);

Argument Description
array_index L'index du tableau à vérifier.

Retour:

Real

Exemple:

for (var i = array_length_1d(a) - 1; i > -1; i--;)
   {
   a[i] = -1;
   }

Le code ci-dessus va parcourir un tableau et définir chaque entrée sur -1.
