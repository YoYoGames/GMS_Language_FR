variable_instance_get

Cette fonction vous permet d'obtenir la valeur d'une variable nommée donnée. Vous fournissez la valeur unique de l'ID de l'instance (qui peut être trouvée dans les propriétés de l'instance dans l'éditeur room ou est retournée lorsque vous appelez la fonction instance_create_layer()) ainsi que le nom de la variable dont vous souhaitez obtenir la valeur sous forme de chaîne de caractères (voir l'exemple de code ci-dessous). La fonction renverra la valeur détenue par la variable, ou undefined si la variable n'existe pas.

IMPORTANT ! Si la variable que vous obtenez n'existe pas, la fonction renverra le mot clé undefined et vous risquez d'obtenir des erreurs qui empêcheront le jeu de fonctionner, donc en cas de doute, utilisez d'abord la fonction variable_instance_exists premièrement.

Syntaxe :

variable_instance_get(instance_id, name) ;

Arguments Description
identifiant de l'instance La valeur d'identification unique de l'instance à utiliser
nom Le nom de la variable à récupérer (sous forme de chaîne)

Les retours :

Valeur (tout type de données) ou indéfinie (si la variable nommée n'existe pas)

Exemple :

if variable_instance_exists(id, "shields")
{
var ss = variable_instance_get(id, "shields") ;
}
else
}
var ss = -1 ;
}

Le code ci-dessus vérifie si une variable existe et si c'est le cas, la valeur qu'elle contient est récupérée et stockée dans une variable locale. Si elle n'existe pas, la variable locale prend la valeur -1.
