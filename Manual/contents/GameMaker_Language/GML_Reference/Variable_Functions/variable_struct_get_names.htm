variable_struct_get_names

Avec cette fonction, vous pouvez récupérer un tableau rempli des noms de variables d'une structure. Vous passez la référence de la structure à vérifier, et chaque entrée du tableau sera une chaîne de noms de variables que la structure contient. contient.

Syntaxe :

variable_struct_get_names(struct) ;

Arguments Description
struct La référence de la structure à vérifier.

Les retours :

Tableau (chaque entrée est une chaîne de caractères)

Exemple :

var str = "" ;
var array = variable_struct_get_names(mystruct) ;
show_debug_message("Variables for struct : " + string(array)) ;
for (var i = 0 ; i < array_length(array) ; i++ ;)
{
str = array[i] + " :" + string(variable_struct_get(mystruct, array[i])) ;
show_debug_message(str) ;
}

Le code ci-dessus récupère un tableau des noms de variables pour la structure donnée et les affiche avec leurs valeurs dans la sortie de débogage.
