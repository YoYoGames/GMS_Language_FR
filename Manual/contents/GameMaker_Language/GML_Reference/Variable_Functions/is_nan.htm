is_nan

Cette fonction renvoie si une variable donnée est NaN (pas un nombre) ou pas, retour true si c'est le cas, et false si ce n'est pas le cas.

Syntaxe:

is_nan(n);

Argument Description
n L'argument à vérifier.

Retour:

Boolean

Exemple:

if is_nan(global.value)
    {
    show_debug_message("Value is not a number");
    }

Le code ci-dessus vérifie la variable globale "valeur" pour voir s'il s'agit d'un nombre ou non et affiche un message de débogage si ce n'est pas le cas.
