is_nan

Cette fonction indique si une variable donnée est NaN (pas un nombre) ou non, en retournant true si c'est le cas, et false si ce n'est pas le cas.

Syntaxe :

is_nan(n) ;

Arguments Description
n L'argument à vérifier.

Les retours :

Booléen

Exemple :

if is_nan(global.value)
{
show_debug_message("Value is not a number") ;
}

Le code ci-dessus vérifie la variable globale "value" pour voir si c'est un nombre ou non et affiche un message de débogage si ce n'est pas le cas.
