is_struct

Cette fonction vérifie si la valeur fournie est une structure. Elle retourne true si c'est le cas, sinon elle retourne false.

Notez que les variables de méthodes renvoient également true, et les instances d'objets false.

Syntaxe :

is_struct(val) ;

Arguments Description
val La valeur à vérifier.

Les retours :

Booléen

Exemple :

if is_struct(a)
{
delete(a) ;
}

Le code ci-dessus vérifie une variable pour voir si c'est un struct, et si la fonction renvoie vrai, le struct est supprimé.
