is_numeric

Cette fonction renvoie si une variable donnée est une valeur numérique (réelle, int32, int64 ou booléenne) ou non. Dans certains cas, vous souhaitez vérifier et voir si une variable dans GameMaker Studio 2 contient une valeur numérique, et c'est à ce moment que vous utiliserez cette fonction. La fonction retournera true si l'entrée donnée est numérique, et false autrement.

Syntaxe:

is_numeric(n);

Argument Description
n L'entrée à vérifier.

Retour:

Boolean

Exemple:

if is_numeric(val)
    {
    current_val += val;
    }

Le code ci-dessus vérifie la variable "val" pour voir si c'est une valeur numérique et si c'est le cas, elle l'ajoute à la variable donnée.
