variable_struct_exists

Avec cette fonction, vous pouvez vérifier si une variable existe ou non dans une structure donnée. Vous fournissez la référence de la structure à utiliser ainsi que le nom de la variable à vérifier sous forme de chaîne de caractères (voir l'exemple de code ci-dessous). La fonction retournera vrai si une variable avec le nom donné existe pour la structure et faux sinon.

Syntaxe :

variable_struct_exists(struct, nom) ;

Arguments Description
struct La référence de la structure à vérifier
nom Le nom de la variable de structure à vérifier (sous forme de chaîne).

Les retours :

Booléen

Exemple :

if !variable_struct_exists(mystruct, "shields"))
{
mystruct.shields = 0 ;
}

Le code ci-dessus va vérifier si la variable appelée "shields" existe dans la structure donnée et si ce n'est pas le cas, elle est créée et initialisée à 0.
