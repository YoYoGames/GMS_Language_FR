variable_struct_exists

Avec cette fonction, vous pouvez vérifier si une variable existe dans la structure donnée ou non. Vous fournissez la référence de structure à utiliser ainsi que le nom de la variable à vérifier sous forme de chaîne (voir l'exemple de code ci-dessous). La fonction retournera true si une variable avec le nom donné existe pour la structure et false autrement.

Syntaxe:

variable_struct_exists(struct, name);

Argument Description
struct La référence struct à vérifier
name Le nom de la variable struct à vérifier (sous forme de chaîne)

Retour:

Boolean

Exemple:

if !variable_struct_exists(mystruct, "shields"))
{
mystruct.shields = 0;
}

Le code ci-dessus vérifiera si la variable appelée "shields" existe dans la structure donnée et si ce n'est pas le cas, elle est créée et initialisée à 0.
