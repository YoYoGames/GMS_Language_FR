variable_global_exists

Avec cette fonction, vous pouvez vérifier si une variable de portée globale existe ou non. Vous fournissez le nom de la variable globale à vérifier sous forme de chaîne (voir l'exemple de code ci-dessous) et la fonction retournera true si une variable globale avec le nom donné existe ou false autrement.

Syntaxe:

variable_global_exists(name);

Argument Description
name Le nom de la variable globale à vérifier (sous forme de chaîne)

Retour:

Boolean

Exemple:

if !variable_global_exists("enemy_num"))
{
global.enemy_num = instance_number(obj_Enemey_Parent);
}

Le code ci-dessus vérifiera si la variable globale appelée "ennemi_num" existe et si ce n'est pas le cas, elle est créée et définit une valeur.
