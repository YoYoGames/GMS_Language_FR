array_sort

Avec cette fonction, vous pouvez trier un tableau par ordre croissant ou décroissant ou en utilisant une fonction personnalisée pour définir l'ordre de tri. La fonction vous oblige à fournir une variable contenant le tableau, puis soit:

  • Les constantes true ou false pour indiquer ascendant (true ) ou décroissant (false ) ordre de tri
  • Une fonction pour définir l'ordre de tri des tableaux

Si vous utilisez une fonction pour créer l'ordre de tri, elle doit prendre 2 arguments (ceux-ci correspondront à deux éléments du tableau), et la fonction doit renvoyer 0 pour l'égalité, inférieure à 0 ou supérieure à 0. La valeur de retour sera puis être utilisé pour déterminer l'ordre de tri (voir l'exemple ci-dessous).

Notez que si le tableau contient un ensemble de chaînes, les chaînes seront triées par ordre alphabétique en fonction de l'alphabet anglais lors de l'utilisation du type de tri croissant / décroissant par défaut. Tous les autres types de données seront triés en fonction de la valeur numérique, dont les valeurs exactes dépendront du type de données lui-même (par exemple, un tableau de tampons serait trié en fonction de la valeur numérique convertie de leur adresse mémoire).

Syntaxe:

array_sort(variable, sorttype_or_function);

Argument Description
variable La variable qui contient le tableau.
sorttype_or_function Le type de tri ( truepour monter ou false pour décroissant) ou une référence de fonction à utiliser pour le tri.

Retour:

N/A

Exemple:

var _xx, _yy, _a;
for (var i = 0; i < 10; ++i;)
    {
    _xx = irandom(room_width);
    _yy = irandom(room_height)
    _a[i] = instance_create_layer(_xx, _yy, layer, obj_Bullet);
    }

show_debug_message(_a);

var _f = function(inst1, inst2)
    {
    return inst1.x - inst2.x;
    }

show_debug_message(_a);

Le code ci-dessus créera un tableau de 10 instances placées à des positions aléatoires dans la salle. Le message de débogage affichera quelque chose comme ce qui suit:

[ 100003,100004,100005,100006,100007,100008,100009,100010,100011,100012 ]

Le code crée ensuite une méthode à utiliser dans le array_sort() qui renvoie la différence entre chacune des positions X des instances dans la pièce et la array_sort() La fonction est exécutée à l'aide de la méthode, en triant les instances par X du plus bas au plus élevé. La sortie du tableau ressemblerait alors à ceci:

[ 100011,100004,100009,100006,100012,100008,100010,100005,100007,100003 ]
