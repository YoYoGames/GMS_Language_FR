is_method

Cette fonction peut être utilisée pour vérifier et voir si une variable est une variable de méthode (elle retournera true ) ou non (auquel cas il retournera false ).

Syntaxe:

is_method(n);

Argument Description
n La variable à vérifier.

Retour:

Boolean

Exemple:

if is_method(get_vec)
    {
    show_debug_message("Method variable!");
    }

Le code ci-dessus vérifie une variable pour voir s'il s'agit d'une méthode et si la fonction retourne true, puis un message de débogage est envoyé à la console.
