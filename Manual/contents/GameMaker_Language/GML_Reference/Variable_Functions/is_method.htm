est_méthode

Cette fonction peut être utilisée pour vérifier et voir si une variable est une variable de méthode (elle renverra true) ou non (dans ce cas, elle renverra false).

Syntaxe :

est_méthode(n) ;

Arguments Description
n La variable à vérifier.

Les retours :

Booléen

Exemple :

if is_method(get_vec)
{
show_debug_message("Méthode variable !") ;
}

Le code ci-dessus vérifie une variable pour voir s'il s'agit d'une méthode, et si la fonction renvoie vrai, alors un message de débogage est affiché dans la console.
