variable_instance_set

Cette fonction vous permet de définir la valeur d'une variable donnée dans une instance. Vous fournissez la valeur unique de l'ID de l'instance (qui peut être trouvée à partir des propriétés de l'instance dans l ou est retournée lorsque vous appelez la fonction instance_create_layer()) ainsi que le nom de la variable dont vous souhaitez définir la valeur sous forme de chaîne de caractères (voir l'exemple de code ci-dessous), et enfin la valeur à définir (peut être n'importe quel type de données valide). Si la variable n'existe pas déjà dans l'instance, elle sera créée et la valeur lui sera attribuée.

Syntaxe :

variable_instance_set(instance_id, name, val) ;

Arguments Description
identifiant de l'instance La valeur d'identification unique de l'instance à utiliser
nom Le nom de la variable à définir (sous forme de chaîne)
val La valeur à donner à la variable

Les retours :

N/A

Exemple :

if !variable_instance_exists(id, "shields")
{
variable_instance_set(id, "shields", 0) ;
}

Le code ci-dessus va vérifier si une variable d'instance existe dans l'instance appelante et si ce n'est pas le cas, elle est créée et mise à 0.
