int64

Cette fonction tentera de convertir une valeur donnée en un entier de 64 bits, où la valeur doit être un réel, une chaîne, un int64, une int32, ou un ptr. Toute autre chose provoquera le crash du jeu avec un message d'erreur. Vous pouvez vérifier si une variable contient un int64 en utilisant la fonction is_int64().

Syntaxe:

int64(val);

Argument Description
val La valeur à convertir.

Retour:

64bit integer

Exemple:

steam_handle = int64(global.fileReadString);

Le code ci-dessus convertit la valeur contenue dans la variable globale en un entier 64 bits.
