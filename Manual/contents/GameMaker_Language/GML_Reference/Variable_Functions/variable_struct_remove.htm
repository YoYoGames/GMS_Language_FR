suppression_de_la_structure_variable

Cette fonction permet de supprimer une variable d'une structure. Vous fournissez l'ID de la structure dont vous voulez supprimer la variable et le nom de la variable à supprimer.

Syntaxe :

variable_struct_set(struct, nom) ;

Arguments Description
struct La référence de la structure dont il faut retirer la variable
nom Le nom de la variable à supprimer (sous forme de chaîne)

Les retours :

N/A

Exemple :

if variable_struct_exists(mystruct, "shields")
{
variable_struct_remove(mystruct, "shields") ;
}

Le code ci-dessus va vérifier si la variable donnée existe dans la structure donnée et si c'est le cas, elle est supprimée.
