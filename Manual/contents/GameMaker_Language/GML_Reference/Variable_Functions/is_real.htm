is_real

Cette fonction renvoie si une variable donnée est un nombre réel (simple, double ou entier) ou non. Dans certains cas, vous souhaitez vérifier et voir si une variable dans GameMaker Studio 2 contient un nombre réel, et c'est à ce moment que vous utiliserez cette fonction. Il ne renvoie pas le nombre réel mais plutôt true ou false, donc une valeur de, par exemple, "poisson" pour n reviendra false, mais une valeur de 200 pour n reviendra true.

Syntaxe:

is_real(n);

Argument Description
n L'argument à vérifier.

Retour:

Boolean

Exemple:

if is_real(val)
    {
    score += val;
    }

Le code ci-dessus vérifie la variable "val" pour voir si c'est un nombre réel et si c'est le cas, il l'ajoute au score.
