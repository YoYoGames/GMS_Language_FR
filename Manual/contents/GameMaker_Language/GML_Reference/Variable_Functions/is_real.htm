est_réel

Cette fonction renvoie si une variable donnée est un nombre réel (simple, double ou entier) ou non. Dans certains cas, vous voulez vérifier si une variable contient un nombre réel, et c'est là que vous utiliserez cette fonction. Elle ne renvoie pas le nombre réel mais plutôt vrai ou faux, donc une valeur de, par exemple, "poisson" renverra faux, mais une valeur de 200 renverra vrai.

REMARQUE: Cette fonction renverra false pour toutes les valeurs d'enum car elles sont stockées en tant que int64. La fonction correcte pour vérifier ce type de données est la suivante is_int64().

Syntaxe :

est_réel(n) ;

Arguments Description
n L'argument à vérifier.

Les retours :

Booléen

Exemple :

if is_real(val)
{
score += val ;
}

Le code ci-dessus vérifie la variable "val" pour voir si c'est un nombre réel et si c'est le cas, il l'ajoute au score.
