array_delete

Avec cette fonction, vous pouvez supprimer une valeur (ou des valeurs) d'un tableau à n'importe quelle position donnée. La fonction nécessite que vous fournissiez une variable contenant le tableau, l'index (position) du tableau à supprimer, ainsi que le nombre de valeurs à supprimer.

Syntaxe:

array_delete(variable, index, number);

Argument Description
variable La variable qui contient le tableau.
index L'index (position) dans le tableau à partir duquel supprimer la ou les valeurs.
number Le nombre de valeurs à supprimer.

Retour:

N/A

Exemple:

array_delete(score_array, 0, 10);

Le code ci-dessus supprimera toutes les valeurs du tableau donné de l'index 0 à l'index 9.
