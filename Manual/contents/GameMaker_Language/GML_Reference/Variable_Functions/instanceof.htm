instanceof

Cette fonction peut être utilisée pour obtenir le nom de la fonction qui a été utilisée pour créer une structure lorsque celle-ci a été créée à l'aide d'une fonction de construction et que l'attribut nouveau et l'opérateur Vous fournissez la variable avec la référence de la structure à vérifier et la fonction retournera soit une chaîne avec le nom de la fonction, soit undefined. Notez que si vous passez à la fonction un littéral de structure (c'est-à-dire une structure qui a été créée sans utiliser de fonction de construction), elle retournera simplement la chaîne "struct". Cette fonction peut également être utilisée pour vérifier si une référence de structure est un objet référence faible ou Dans ce cas, la fonction renvoie la chaîne "weakref" au lieu du nom de la fonction qui a créé la structure (pour plus d'informations, voir la fonction weak_ref_create()).

Syntaxe :

instanceof(struct_id) ;

Arguments Description
struct La référence de la structure à utiliser.

Les retours :

Chaîne ou indéfini

Exemple étendu :

Dans cet exemple, nous devons d'abord définir la fonction qui sera utilisée comme constructeur de la structure :

function init_struct(_a, _b, _c) constructor
{
a = _a ;
b = _b ;
c = _c ;
}

Cette fonction peut ensuite être utilisée avec l'opérateur new pour créer une structure et la remplir avec les variables définies par les valeurs des arguments utilisés dans la fonction :

mystruct = new init_struct(10, 100, "Hello World") ;

Nous pouvons alors obtenir le nom de la fonction qui a été utilisée comme ceci :

var _name = instanceof(mystruct) ;
if is_string(_name)
{
show_debug_message(_name) ;
}
