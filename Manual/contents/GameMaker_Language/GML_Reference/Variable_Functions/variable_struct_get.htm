variable_struct_get

Avec cette fonction, vous pouvez obtenir la valeur d'une variable nommée donnée dans une structure. Vous fournissez la référence de la structure ainsi que le nom de la variable dont vous voulez obtenir la valeur sous forme de chaîne de caractères (voir l'exemple de code ci-dessous). La fonction retournera la valeur détenue par la variable ou indéfinie si la variable nommée n'existe pas.

IMPORTANT ! Si la variable que vous obtenez n'existe pas, la fonction renverra le mot-clé undefined et vous pourrez obtenir des erreurs qui empêcheront le jeu de fonctionner, donc en cas de doute, utilisez d'abord la fonction variable_struct_exists() d'abord.

Syntaxe :

variable_struct_get(struct, nom) ;

Arguments Description
struct La référence de la structure à utiliser
nom Le nom de la variable à récupérer (sous forme de chaîne)

Les retours :

Valeur (tout type de données) ou indéfinie

Exemple :

if variable_struct_exists(mystruct, "shields")
{
var ss = variable_struct_get(mystruct, "shields") ;
}
else
}
var ss = -1 ;
}

Le code ci-dessus vérifie si une variable existe dans la structure donnée et si c'est le cas, la valeur qu'elle contient est récupérée et stockée dans une variable locale. Si elle n'existe pas, la variable locale est définie à -1.
