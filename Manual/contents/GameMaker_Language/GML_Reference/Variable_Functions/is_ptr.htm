est_ptr

Cette fonction renvoie si une variable donnée est un pointeur ou non. Dans certains cas, vous souhaitez vérifier le type de données d'une variable dans GameMaker Studio 2 et c'est dans ce cas que vous utiliserez cette fonction. Elle renvoie true ou false selon selon que la valeur est un pointeur ou non.

Syntaxe :

est_ptr(n) ;

Arguments Description
n L'argument à vérifier.

Les retours :

Booléen

Exemple :

if !is_ptr(val)
{
show_debug_message("Not a valid texture !") ;
}

Le code ci-dessus vérifie la variable "val" pour voir si elle contient un pointeur et si ce n'est pas le cas, il affiche un message dans la console de débogage.
