nombre_de_noms_d_instances_variables

Avec cette fonction, vous pouvez trouver le nombre total de variables définies pour une instance. Vous fournissez l'ID de l'instance à vérifier, et la fonction retournera une valeur entière pour le nombre de variables rencontrées, ou (si aucune instance de l'ID donné n'existe) -1. ID donnée) -1.

Syntaxe :

variable_instance_names_count(instance_id) ;

Arguments Description
identifiant de l'instance La valeur d'identification unique de l'instance à vérifier.

Les retours :

Entier

Exemple :

var _num = variable_instance_names_count(myinst) ;
show_debug_message("Instance Variables = " + string(_num)) ;

Le code ci-dessus récupère le nombre de variables dans l'instance donnée et affiche un message de débogage dans la sortie console avec cette valeur.
