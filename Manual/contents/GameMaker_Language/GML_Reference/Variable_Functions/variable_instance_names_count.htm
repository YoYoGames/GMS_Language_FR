variable_instance_names_count

Avec cette fonction, vous pouvez trouver le nombre total de variables définies pour une instance. Vous fournissez l'ID d'instance à vérifier et la fonction renverra une valeur entière pour le nombre de variables rencontrées, ou (si aucune instance de l'ID donné n'existe) -1.

Syntaxe:

variable_instance_names_count(struct_id);

Argument Description
instance_id La valeur d'ID unique de l'instance à vérifier.

Retour:

Integer

Exemple:

var _num = variable_instance_names_count(mystruct);
show_debug_message("Instance Variables = " + string(_num));

Le code ci-dessus récupérera le nombre de variables dans l'instance donnée et affichera un message de débogage dans la sortie de la console avec cette valeur.
