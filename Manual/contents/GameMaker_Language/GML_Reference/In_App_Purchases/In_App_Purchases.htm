Achats dans les applications

Vous pouvez ajouter des achats intégrés à vos jeux en utilisant les extensions disponibles sur la place de Marketplace et en suivant les guides de configuration répertoriés ci-dessous (chaque extension est livrée avec son propre «mini-manuel» qui explique comment l'utiliser et répertorie toutes les fonctions qu'elle contient):

  • Android: configuration des IAP
  • iOS: configuration des IAP
  • macOS: configuration des IAP

Les versions précédentes de GameMaker STudio avaient les fonctions d'achat intégrées dans le runner, mais cela était impossible à maintenir et ces fonctions sont désormais obsolètes et ne fonctionneront plus:

  • iap_data
  • iap_activate
  • iap_status
  • iap_enumerate_products
  • iap_restore_all
  • iap_acquire
  • iap_consume
  • iap_product_details
  • iap_purchase_details
