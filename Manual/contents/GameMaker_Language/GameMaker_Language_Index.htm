Index des GameMaker Language

Le GameMaker Language (également appelé simplement GML ) est le propriétaire Langage de script GameMaker Studio 2. Ce langage est structuré pour permettre aux utilisateurs de créer leurs jeux de manière intuitive et flexible tout en offrant toute la puissance de tout autre langage de programmation majeur. C'est également la base du Drag and Drop™ et peut être utilisé en conjonction avec cela si nécessaire.
