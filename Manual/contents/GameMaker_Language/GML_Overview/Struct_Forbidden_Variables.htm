Variables interdites de structure

Lors de la création d'une structure, il est possible d'utiliser certaines variables intégrées comme noms de variables membres, par exemple:

mystruct = {
object_index : obj_Player,
speed : 3,
image_blend : c_red
}

Cependant, seules les variables intégrées de portée locale et d'instance peuvent être utilisées de cette manière et attribuer des valeurs comme s'il s'agissait de variables membres de structure régulières. L'utilisation de variables intégrées à portée globale est interdite et causera des problèmes avec votre jeu. Vous trouverez ci-dessous une liste complète de ces variables afin de savoir lesquelles éviter.
