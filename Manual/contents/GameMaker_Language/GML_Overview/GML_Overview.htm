GameMaker Language

Cette section du manuel contient toutes les informations nécessaires pour comprendre et utiliser le langage de script GameMaker Studio 2 GML . Les runtime de ce langage peuvent être utilisées pour créer vos jeux et sont ajoutées aux objets à partir de l' éditeur d'objets , normalement lorsque vous avez choisi de créer un nouveau GameMaker Language , bien qu'il puisse également être utilisé avec DnD™ ( Drag and Drop ) . Ci-dessous, vous pouvez voir une image typique d'un objet avec l'éditeur de code ouvert sur un événement:

Chaque événement a son propre onglet dans l'éditeur et vous pouvez y ajouter, modifier ou supprimer du code à tout moment (pour plus d'informations sur les événements, voir Evénements d'objets ). Le code lui-même doit avoir une structure de base et peut contenir des indices de ressources, des variables, des fonctions, des expressions, des mots-clés... etc... qui sont tous expliqués dans les sections ci-dessous. Si vous êtes novice dans la programmation ou si vous passez de DnD™, il est recommandé de commencer par la page sur la structure de code de base , puis de lire toutes les autres pages de cette section, en testant le code de chacune dans GameMaker Studio 2 lui-même..
