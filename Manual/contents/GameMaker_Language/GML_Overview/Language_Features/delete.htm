delete

Le delete L'opérateur est utilisé pour dé-référencer une structure et a la syntaxe suivante:

delete <variable>;

Cela signifie que vous appelez l'opérateur de suppression avec la variable qui contient une structure et cela supprimera la référence spécifique à la structure stockée dans la variable donnée (une référence est simplement une valeur qui pointe vers la zone de mémoire qui stocke le struct et son contenu). Si cette référence était la dernière référence à la structure dans votre jeu, le garbage collector supprimera la structure de la mémoire dans une itération suivante, généralement au tout début de l'étape suivante.

REMARQUE:delete ne peut être utilisé qu'avec des variables et non des expressions

Par défaut, les structures sans autres références dans le code seront automatiquement récupérées dans les étapes suivant leur utilisation, mais il est recommandé d'utiliser cet opérateur pour les marquer explicitement pour que le ramasse-miettes les supprime de la mémoire au tout début de ce qui suit étape, libérant la mémoire rapidement et plus efficacement.

L'exemple suivant montre une structure en cours de création, par exemple dans l'événement Create d'une instance:

mystruct = {
    a : 0,
    b : 100,
    c : "Hello World"
    }

Cette structure sera ensuite utilisée dans les événements d'instance comme d'habitude, avant d'être finalement marquée pour le garbage collection dans l' événement Clean Up, comme ceci:

delete mystruct;
