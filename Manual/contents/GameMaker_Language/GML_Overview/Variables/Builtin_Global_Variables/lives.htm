lives

Cette variable a une portée globale et est utilisée pour contenir une valeur numérique qui est généralement utilisée pour la vie du joueur. Cette variable est uniquement conçue pour prendre en charge les projets hérités des versions précédentes de GameMaker et ne doit pas être utilisée dans de nouveaux projets car elle pourrait être obsolète à l'avenir.

Syntaxe:

lives;

Retour:

Real (single precision floating point value)

Exemple

if (lives < 5)
    {
    lives += 1;
    }

Le code ci-dessus vérifie le lives variable et s'il est inférieur à 5, 1 lui est ajouté.
