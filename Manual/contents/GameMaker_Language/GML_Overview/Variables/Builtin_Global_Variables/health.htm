health

Cette variable a une portée globale et est utilisée pour contenir une valeur numérique qui est généralement utilisée pour la santé du joueur. Cette variable est uniquement conçue pour prendre en charge les projets hérités des versions précédentes de GameMaker et ne doit pas être utilisée dans de nouveaux projets car elle pourrait être obsolète à l'avenir.

Syntaxe:

health;

Retour:

Real (single precision floating point value)

Exemple

if (health <= 0)
    {
    global.state = "Game Over";
    instance_destroy();
    }

Le code ci-dessus vérifie le health variable et si elle est inférieure ou égale à 0, une variable globale est définie et l'instance est détruite.
