score

Cette variable a une portée globale et est utilisée pour contenir une valeur numérique qui est généralement utilisée pour le score du joueur. Cette variable est uniquement conçue pour prendre en charge les projets hérités des versions précédentes de GameMaker et ne doit pas être utilisée dans de nouveaux projets car elle pourrait être obsolète à l'avenir.

Syntaxe:

score;

Retour:

Real (single precision floating point value)

Exemple

switch (object_index)
    {
    case obj_Enemy_Fighter:
        score += 10;
        break;
    case obj_Enemy_Mage:
        score += 25;
        break;
    case obj_Enemy_Boss:
        score += 100;
        break;
    }
instance_destroy();

Le code ci-dessus vérifie l'index d'objet de l'instance exécutant le code à l'aide d'unswitch, puis ajoute des montants différents au score variable en fonction de quel objet il s'agit.
