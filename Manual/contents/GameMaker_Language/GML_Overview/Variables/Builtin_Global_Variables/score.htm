score

Cette variable a une portée globale et est utilisée pour contenir une valeur numérique qui est généralement utilisée pour le score du joueur. Cette variable est uniquement conçue pour prendre en charge les projets hérités des versions antérieures de GameMaker et devrait ne pas utiliser dans les nouveaux projets car elle pourrait être obsolète à l'avenir.

Syntaxe :

score ;

Les retours :

Réel (valeur à virgule flottante de précision simple)

Exemple

switch (object_index)
{
case obj_Enemy_Fighter :
score += 10 ;
break ;
case obj_Enemy_Mage :
score += 25 ;
break ;
case obj_Enemy_Boss :
score += 100 ;
break ;
}
instance_destroy() ;

Le code ci-dessus vérifie l'index de l'objet de l'instance qui exécute le code à l'aide d'un commutateur, puis ajoute différents montants à la variable score en fonction de l'objet en question.
