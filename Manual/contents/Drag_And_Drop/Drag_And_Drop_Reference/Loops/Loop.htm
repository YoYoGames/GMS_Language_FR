Boucle

L' action Boucle boucle indéfiniment jusqu'à ce qu'une action Break soit appelée. Des précautions doivent être prises avec cette action pour s'assurer que quelles que soient les actions placées dans la boucle, une pause sera toujours appelée, sinon le jeu provoquera une erreur "ne répond pas" sur le système d'exploitation cible et cessera de fonctionner. Vous ajoutez cette action dans l'espace de travail du code d'action, puis ajoutez les actions à effectuer dans la boucle, tout comme vous ajouteriez une action à un "si", c'est-à-dire en la déposant sur le côté de l'action plutôt qu'en dessous:
