Bibliothèque d'actions de boucle

La bibliothèque d'actions de boucle contient les actions requises pour créer des boucles dans le code d'action de votre jeu. Une boucle est simplement un moyen d'itérer sur le même ensemble d'actions une ou plusieurs fois en fonction de certaines conditions et est pratique dans toutes les situations où vous devez répéter les mêmes actions encore et encore.

La bibliothèque de boucles comporte les actions suivantes:

Boucle
Répéter
Tandis que
Pour
Casser
