Répéter

Cette action répétera toutes les actions qui lui sont ajoutées le nombre de fois spécifié. Vous indiquez le nombre de fois que les actions doivent être répétées, puis ajoutez l'action à la boucle de répétition, de la même manière que vous ajouteriez une action à un "si", c'est-à-dire: le déposer sur le côté de l'action plutôt que sous celui-ci:

Toutes les actions ajoutées sur le côté seront incluses dans la répétition et seront exécutées avant que l'événement ou le script ne se poursuive avec les actions ajoutées sous (comme indiqué dans l'exemple ci-dessous). Notez que vous pouvez mettre fin prématurément à n'importe quelle boucle si vous appelez l'action Break.

Syntaxe de l'action:

Arguments:

Argument Description
Times Le nombre de répétitions (une valeur entière entière)

Exemple:

Le code de bloc d'action ci-dessus obtient le nombre d'éléments dans une liste et stocke la valeur dans une variable locale temporaire. Cette variable est ensuite utilisée pour définir le nombre de fois qu'une boucle de répétition est exécutée, et dans la boucle de répétition, les entrées de la liste sont lues dans une autre variable temporaire qui sert ensuite à détruire les instances auxquelles elles font référence. Une fois la boucle terminée, la structure de données est libérée.
