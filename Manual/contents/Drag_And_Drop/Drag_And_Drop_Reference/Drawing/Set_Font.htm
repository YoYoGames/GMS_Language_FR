Définir la police

Avec cette action, vous pouvez définir la police à utiliser pour toutes les actions suivantes de dessin de texte. Cliquez sur la boîte de dialogue pour ouvrir la fenêtre suivante dans laquelle vous pouvez ensuite sélectionner la police que vous souhaitez utiliser:

Notez que cela définira la police à utiliser pour tous les appels de texte de dessin ultérieurs, et peut donc être appelée une fois au début du jeu - dans tous les cas, pas seulement l'événement de dessin - et toutes les actions de dessin de texte utiliseront la police définie. Cependant, si vous avez besoin de plusieurs polices dans votre jeu, vous devrez appeler cette action avant de dessiner un texte avec la police requise.
