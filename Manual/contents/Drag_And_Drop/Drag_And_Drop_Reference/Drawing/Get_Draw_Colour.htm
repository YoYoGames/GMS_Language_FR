Obtenir la couleur de dessin

Avec cette action, vous pouvez obtenir la couleur utilisée pour toutes les formes (rectangle / ellipse / ligne) ou pour tout texte dessiné. L'action renverra la couleur à la variable cible que vous spécifiez, qui peut avoir été créée précédemment ou peut être une nouvelle temporaire (si vous cochez la case "Temp").
