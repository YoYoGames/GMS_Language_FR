Dessiner un rectangle de dégradé

Cette action dessinera un rectangle à une position donnée dans la pièce, en utilisant un ensemble de couleurs de fusion pour créer un dégradé. Vous donnez la position en haut à gauche et en bas à droite et le rectangle sera dessiné entre elles, et la position peut être soit une position absolue dans la pièce, soit une position par rapport à l'instance appelant l'action. Vous définissez ensuite les couleurs à mélanger à partir de chacun des coins et pouvez également définir si le rectangle peut être rempli ou délimité en cochant la case de remplissage en bas.

REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de tirage au sort et ne tirera rien si elle est utilisée ailleurs.

Syntaxe de l'action:

Arguments:

Argument Description
Left La position de gauche à partir de laquelle commencer à dessiner
Top La position supérieure à partir de laquelle commencer à dessiner
Right La bonne position pour terminer le dessin
Bottom La position inférieure pour terminer le dessin à partir de
Top Left La couleur à mélanger à partir du coin supérieur gauche
Top Right La couleur à mélanger dans le coin supérieur droit
Bottom Left La couleur à mélanger à partir du coin inférieur gauche
Bottom Right La couleur à mélanger dans le coin inférieur droit
