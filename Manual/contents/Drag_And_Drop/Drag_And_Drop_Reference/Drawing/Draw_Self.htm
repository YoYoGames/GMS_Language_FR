Dessiner soi-même

Lorsque vous ajoutez un type d'action ou de code à l'événement Draw d'un objet, vous GameMaker Studio 2 que vous prenez le relais et que vous lui direz quoi dessiner à partir de maintenant (par défaut, une instance dessinera le sprite assigné dans l'éditeur d'objets lorsqu'il n'y a aucune action dans l'événement principal de dessin). Cela signifie que tout sprite assigné ne sera plus dessiné à moins que vous ne le disiez explicitement, c'est à ce moment que vous utiliseriez cette action. Cela dessine simplement le sprite attribué à l'objet avec toutes les transformations qui peuvent avoir été définies dans des événements précédents (comme la mise à l'échelle, la fusion ou les modifications alpha) à la sous-image actuelle.

REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de tirage au sort et ne tirera rien si elle est utilisée ailleurs.
