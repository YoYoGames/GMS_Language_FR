Obtenir l'alignement du texte

Cette action obtiendra l'alignement de la police pour toutes les actions de texte de dessin en cours. Vous pouvez choisir de récupérer l'alignement horizontal ou vertical à vérifier, et l'action renverra l'une des valeurs constantes suivantes, qui représentent un autre type d'alignement horizontal ou vertical:

Le texte est aligné horizontalement à gauche
Le texte est aligné horizontalement au centre
Le texte est aligné horizontalement à droite
Le texte est aligné verticalement vers le haut
Le texte est aligné verticalement au milieu
Le texte est aligné verticalement vers le bas

La constante de retour sera stockée dans la variable cible que vous spécifiez, qui peut avoir été créée précédemment ou peut être une nouvelle temporaire (si vous cochez la case "Temp").
