Dessiner un sprite

Cette action va dessiner un sprite quelque part dans la room. Vous spécifiez le sprite, la position (soit une position absolue dans la room, soit une position relative à l'instance effectuant le dessin) et le cadre d'animation du sprite, s'il a plusieurs cadres (notez que les cadres d'image sont numérotés de 0 à 5). cadres (notez que les cadres d'image sont numérotés à partir de 0). Si vous avez un sprite animé et que vous voulez qu'il dessine les frames comme s'il était assigné à l'objet, alors utilisez -1 pour l'index de frame ou la variable d'instance image_index, Mais notez que si l'instance n'a pas de sprite assigné, elle ne s'animera pas, et si le sprite assigné a un nombre de frames différent de celui qui est dessiné, le sprite sera dessiné avec des frames supplémentaires ou des frames manquées.

IMPORTANT : Cette action ne dessinera pas le sprite transformé, même si vous avez modifié le mélange ou l'échelle de l'image. Pour cela, voir l'action Draw Sprite Transformed.

NOTE: Cette action est uniquement destinée à être utilisée dans les différents événements de tirage au sort, et ne tirera rien si elle est utilisée ailleurs.

Arguments :
