Aller au début

Cette action permet de déplacer une instance de sa position actuelle à la position où elle a été créée à l'origine. La position d'origine où une instance est créée est stockée dans deux variables d'instance appelées xstart et ystart. Ces variables peuvent être utilisées dans toute autre action et peuvent également être définies pour modifier la position de départ enregistrée.
