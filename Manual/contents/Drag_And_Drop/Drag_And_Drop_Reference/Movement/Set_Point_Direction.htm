Direction du point de consigne

Cette action permet de définir une direction à partir d'un vecteur formé par la position actuelle de l'instance et la position donnée. Vous fournissez les positions x et y pour "pointer" sur - elles peuvent être marquées comme relatives à la position de l'instance - et la direction de l'instance vers ce point sera utilisée pour l'instance.

Syntaxe de l'action:

Arguments:

Argument Description
x La composante x du vecteur de direction.
y La composante y du vecteur de direction.

Exemple:

Le code du bloc d'action ci-dessus vérifie si une instance de l'objet "obj_Player "existe dans la pièce et si tel est le cas, définit la direction de l'instance pour qu'elle pointe vers la position x / y de"obj_Player ".
