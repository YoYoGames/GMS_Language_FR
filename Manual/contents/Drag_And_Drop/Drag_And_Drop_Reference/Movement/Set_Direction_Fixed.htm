Définir la direction fixe

Cette action est utilisée pour définir une direction de mouvement fixe. Les directions disponibles sont indiquées par les différents boutons dans le bloc d'action, et cliquer sur l' un d'entre eux définira cette direction. Notez que les directions dans GameMaker Studio 2 sont mesurées en degrés, la droite étant à 0 °, le haut étant à 90 °, la gauche à 180 ° et le bas à 270 °.
