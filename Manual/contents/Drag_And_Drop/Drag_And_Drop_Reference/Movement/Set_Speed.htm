Régler la vitesse

Cette action permet de définir la vitesse de déplacement de l'instance. La vitesse calculée comme le nombre de pixels qu'une instance doit se déplacer dans une seule image de jeu (étape). Par défaut, la valeur de vitesse sera ajoutée au vecteur de direction - donc si vous définissez la vitesse sur 2 et la direction sur 45 °, l'instance se déplacera vers le haut et la droite de 2 pixels à chaque image du jeu - mais vous pouvez également cliquer sur le bouton direction et sélectionnez la vitesse horizontale ou verticale.

Le choix de l'un ou l'autre de ces paramètres définira la vitesse uniquement pour les vecteurs horizontaux ou verticaux. Notez que vous pouvez marquer la valeur comme étant relative à la vitesse actuelle, ce qui ajoutera ou soustrayera la valeur de la vitesse réelle de l'instance, et vous pouvez également fournir des nombres négatifs, ce qui la fera se déplacer dans la direction opposée au courant. valeur de direction.

Syntaxe de l'action:

Arguments:

Argument Description
speed La valeur de la vitesse (peut être négative).

Le code du bloc d'action ci-dessus vérifie si un bouton gauche de la souris appuie sur l'instance et si un est détecté, il définit la direction vers le bas (270 °) et la vitesse sur 5.
