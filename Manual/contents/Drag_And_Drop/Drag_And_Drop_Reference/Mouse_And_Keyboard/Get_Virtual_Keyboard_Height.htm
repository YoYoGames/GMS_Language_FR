Obtenir la hauteur du clavier virtuel

Cette action renverra la hauteur actuelle en pixels du clavier virtuel, en fonction de la taille de l' affichage. Si le clavier n'est pas visible, 0 sera renvoyé.

Syntaxe de l'action:

Exemple:

Le code de bloc d'action ci-dessus vérifie si le clavier virtuel du système d'exploitation est visible, et s'il l'est, il obtient la hauteur du clavier (en pixels d'écran), puis dessine du texte au-dessus avec l'entrée du clavier.
