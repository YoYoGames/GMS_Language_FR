Si le clavier virtuel s'affiche

Cette action peut être utilisée pour vérifier si le clavier virtuel du système d'exploitation est affiché ou non. L'action évaluera à true lorsque le clavier est ouvert ou s'ouvre, ou false si le clavier virtuel est fermé ou se ferme. Si vous activez le not modificateur, cette action sera inversée et vous pourrez stocker le retour true /false valeur dans une variable cible, qui peut être marquée comme temp (local).

Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme indiqué dans l'image ci-dessous:

Ces actions seront désormais exécutées si le "si" est évalué à true, tandis que toutes les actions déposées ailleurs seront exécutées après le bloc "if".
