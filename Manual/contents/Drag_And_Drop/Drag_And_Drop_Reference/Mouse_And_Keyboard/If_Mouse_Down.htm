Si la souris est enfoncée

Avec cette action, vous pouvez interroger la souris pour voir si le bouton sélectionné est actuellement maintenu enfoncé. L'action reviendra true chaque étape pendant laquelle le bouton est maintenu enfoncé, ou false sinon, bien que si vous cliquez sur le not modificateur vous pouvez alors vérifier si le bouton n'est pas maintenu, c'est à dire: l'action reviendra true alors qu'aucun bouton n'est enfoncé et false s'il y a. Si vous n'avez besoin de vérifier qu'un seul bouton, utilisez l'action Si la souris est enfoncée.

Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme indiqué dans l'image ci-dessous:

Ces actions seront désormais exécutées si le "si" est évalué à true, tandis que toutes les actions déposées ailleurs seront exécutées après le bloc "if".

Syntaxe de l'action:

Arguments:

Argument Description
Mouse Button Le nom du bouton de la souris à vérifier (gauche, milieu, droite).
Not Annuler le chèque (vrai devient faux et vice versa)

Le code de bloc d'action ci-dessus interroge l'état du bouton de la souris à chaque étape et s'il est maintenu enfoncé, il vérifie la pression initiale de la souris. Si le chèque est true lors de la pression initiale vers le bas, la vitesse de l'instance est définie, puis, pendant que le bouton est enfoncé, le mélange de sprite passe au rouge. Si la souris n'est pas maintenue enfoncée, une vérification est effectuée au relâchement de la souris pour réinitialiser la couleur de fusion au blanc et régler la vitesse sur 0.
