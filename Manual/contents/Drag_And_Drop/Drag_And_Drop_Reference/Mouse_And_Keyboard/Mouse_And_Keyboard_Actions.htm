Bibliothèque des actions de la souris et du clavier

Le site Souris et clavier est l'endroit où vous pouvez trouver les actions requises pour interroger et traiter les entrées du clavier ou de la souris. Les actions disponibles sont toutes répertoriées ci-dessous :

Si la souris est pressée
Si la souris est en bas
Si la souris est libérée
Si la touche est pressée
Si la clé est enfoncée
Si la clé est libérée

Les trois autres actions font référence au clavier virtuel. Si vous avez besoin de saisir du texte ou des chiffres dans votre jeu sur un appareil à écran tactile, vous pouvez utiliser les actions du clavier virtuel fournies pour utiliser le clavier du système d'exploitation de l'appareil. Ces actions vont appeler Ces actions appelleront ou masqueront le clavier du système d'exploitation et accepteront les entrées de celui-ci de la même manière qu'un clavier physique. Ainsi, les actions clavier habituelles de GameMaker Studio 2 devraient toutes fonctionner avec les claviers virtuels (par exemple, l'action " Si la touche est relâchée "), et toute entrée provenant du clavier virtuel sera stockée dans la mémoire de GameMaker Studio 2. provenant du clavier virtuel sera stockée dans les différentes variables de clavier, telles que chaîne_clavier ou clavier_dernière_caractère, etc...).

L'ouverture ou la fermeture du clavier virtuel générera des événements asynchrones du système, dans lesquels la carte DS sera remplie avec les paires clé/valeur suivantes async_load DS map sera remplie avec les paires clé/valeur suivantes :

  • " event_type" - le type d'événement système reçu, qui sera " virtualkeyboard status " pour les claviersvirtuels.
  • "screen_height " - la hauteur du clavier virtuel (en pixels). Elle sera de 0 si le clavier est invisible.
  • "keyboard_status" - l'état actuel du clavier, renvoyé sous la forme d'une des chaînes suivantes :
    • "caché"
    • "caché"
    • "visible"
    • "visible"

Les actions du clavier virtuel disponibles sont les suivantes :

Afficher le clavier virtuel
Masquer le clavier virtuel
Obtenir la hauteur du clavier virtuel
Si le clavier virtuel est affiché
