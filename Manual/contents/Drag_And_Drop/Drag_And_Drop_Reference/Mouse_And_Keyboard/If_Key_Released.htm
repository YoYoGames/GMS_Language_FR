Si la clé est relâchée

Cette action vérifie si une touche du clavier a été relâchée. Elle retournera vrai sur le seul pas de jeu (frame) où la touche est relâchée et faux à tous les autres moments. Si vous activez le modificateur not, cette action sera inversée, et vérifiera si la touche n'a pas été relâchée, en retournant false sur le pas de jeu (frame) où elle a été relâchée et true à tous les autres moments.

Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous :

Ces actions seront maintenant exécutées si le "if" vaut true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if".

Syntaxe de l'action :

Arguments :

Arguments Description
Clé Le nom de la clé à vérifier.
Non Négation de la vérification (vrai devient faux et vice versa)

Le code du bloc d'action ci-dessus interroge l'état du clavier à chaque étape et si une touche est maintenue enfoncée, il vérifie l'appui initial sur la touche. Si la vérification est vraie lors de l'appui initial sur la touche, un son est joué, puis, tant que la touche est maintenue enfoncée, le mélange du sprite passe au rouge. Si la touche n'est pas maintenue la touche n'est pas maintenue enfoncée, une vérification est effectuée au moment du relâchement de la touche pour réinitialiser la couleur de mélange en blanc.
