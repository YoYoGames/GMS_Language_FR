Définir le sprite de particules

Cette action peut être utilisée pour définir le sprite qu'une particule peut utiliser lors du dessin. Vous fournissez la valeur d'ID unique du type pour définir l'image-objet (telle qu'elle est renvoyée lorsque vous avez créé le type avec l'action Créer un type de particule ) et la ressource d'image-objet à utiliser. Si l'image-objet contient des sous-images, la particule s'animera à travers les images à la vitesse définie dans l'éditeur d'images pour l'image-objet. Notez également que si vous définissez une valeur de couleur pour la particule (à l'aide de l' action Définir la couleur de la particule ), cette couleur sera fusionnée avec l'image-objet utilisée, tout comme si vous définissez le mélange d'image sur une instance, une couleur sera mélangée avec l'image-objet d'instance (utilisez blanc pour aucun mélange).

REMARQUE: si vous utilisez cette action, vous ne devez pas utiliser l'action Définir la forme de particule car vous ne pouvez avoir qu'un sprite ou une forme, mais pas les deux.

Le code de bloc d'action ci-dessus créera un nouveau type de particule et attribuera sa valeur d'identifiant unique à une variable globale. Il procède ensuite à la définition de toutes les propriétés du type de particule.
