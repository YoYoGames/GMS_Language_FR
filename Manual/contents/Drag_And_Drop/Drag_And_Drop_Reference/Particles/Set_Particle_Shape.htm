Définir la forme des particules

Cette action peut être utilisée pour définir la forme de sprite intégrée qu'une particule doit utiliser lors du dessin. Vous fournissez la valeur d'ID unique du type pour définir la forme (telle que renvoyée lorsque vous avez créé le type avec l'action Créer un type de particule ) et la forme à utiliser. REMARQUE: si vous utilisez cette action, vous ne devez pas utiliser l'action Définir le sprite de particules car vous ne pouvez avoir qu'un sprite ou une forme, mais pas les deux.
