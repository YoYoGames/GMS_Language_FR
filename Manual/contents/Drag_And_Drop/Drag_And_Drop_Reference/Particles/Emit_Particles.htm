Émettre des particules

Cette action est utilisée pour créer des particules à partir d'un émetteur. Vous fournissez l'ID du système de particules auquel l'émetteur appartient (tel que renvoyé par l'action Créer un système de particules ), puis l'ID réel de l'émetteur (tel que renvoyé par l'action Créer un émetteur de particules ) et enfin le type de particule à créer (tel que renvoyé par le action Créer un type de particule ). Vous pouvez alors sélectionner "Rafale" ou "Diffuser" les particules de l'émetteur, ainsi que donner le nombre d'entre elles que vous souhaitez créer.

Lorsque vous choisissez de diffuser des particules, vous n'avez pas besoin de placer cette action dans un événement d'image de jeu comme l'événement Step ou l'événement Draw, mais appelez-la simplement une fois dans l'événement Create (par exemple) et l'émetteur diffusera des particules tous les cadre de jeu jusqu'à ce que vous appeliez à nouveau cette action pour l'arrêter (en définissant la valeur de comptage sur 0). Si vous choisissez d'éclater les particules, vous pouvez l'appeler dans n'importe quel événement de votre choix et les particules seront créées une fois.

Syntaxe de l'action:

Arguments:

Argument Description
System L'ID du système dans lequel se trouve l'émetteur
Emitter L'ID de l'émetteur à utiliser
Type L'ID du type de particule à créer
Event Le type d'événement pour les particules, soit "Rafale" ou "Flux"
Count Le nombre de particules à éclater ou à ruisseler

Exemple:

Le code de bloc d'action ci-dessus créera un émetteur de particules, définira sa région, puis le définira pour émettre un flux de particules.
