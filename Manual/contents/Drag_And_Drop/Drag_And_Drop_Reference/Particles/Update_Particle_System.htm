Mettre à jour le système de particules

Cette action forcera le système de particules donné à mettre à jour la position, la couleur, l'alpha, etc... Vous fournissez la valeur d'ID unique pour le système à mettre à jour (tel que renvoyé lorsque vous avez créé le système avec l'action Créer un système de particules ), et chaque lorsque vous appelez l'action, le système donné mettra à jour une image de jeu, ce qui signifie que vous pouvez - par exemple - appeler cette action dans une boucle de répétition et faire mettre à jour le système de particules plusieurs fois dans une seule image de jeu (utile lorsque vous voulez un niveau pour avoir un effet qui nécessite la présence de particules au tout début, comme la neige ou la pluie). Vous pouvez également utiliser cette action lorsque le système de particules a été suspendu à l'aide du système de particules Pause pour mettre à jour manuellement les particules dans le système.

Syntaxe de l'action:

Arguments:

Argument Description
System La valeur d'identifiant unique du système à mettre à jour
