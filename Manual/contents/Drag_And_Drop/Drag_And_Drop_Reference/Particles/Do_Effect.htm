Faire effet

Cette action créera un effet de particule intégré, au-dessus ou en dessous des calques dessinés dans votre salle de jeux. Vous choisissez le type d'effet à créer dans la liste déroulante (affichée dans l'image ci-dessous), puis définissez la position (qui peut être relative à la position de l'instance) et sélectionnez une taille - petite, moyenne ou grande - pour l'effet. Vous pouvez également définir la couleur à utiliser - la valeur alpha sera ignorée - et décider si vous souhaitez que l'effet soit créé sous tous les calques de la pièce ou au-dessus de tous les calques de la pièce. Notez que si vous choisissez d'utiliser les effets de pluie ou de neige, les valeurs de position et de taille ne seront pas prises en compte car l'effet est conçu pour être visible sur tout l'écran.

Syntaxe de l'action:

Arguments:

Argument Description
Type Le type d'effet de particules à créer (voir image ci-dessus)
X La position x dans la pièce pour créer l'effet (en pixels)
Y La position y dans la pièce pour créer l'effet (en pixels)
Size La taille de l'effet (petit, moyen, grand)
Colour La couleur de l'effet (l'alpha sera ignoré)
Where Que ce soit pour créer l'effet au-dessus ou en dessous des couches de la pièce
