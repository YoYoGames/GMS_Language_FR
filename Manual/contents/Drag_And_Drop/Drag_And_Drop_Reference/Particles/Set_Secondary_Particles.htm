Définir les particules secondaires

Cette action vous permet d'utiliser un deuxième type de particule avec la particule principale. Vous donnez la valeur de l'ID du type de particule principale (qui doit être la même que la valeur utilisée lorsque vous avez créé la particule à l'aide de l' action Créer un type de particule ), puis vous pouvez choisir de lui faire émettre un type de particule secondaire à chaque image de jeu qu'elle existe et / ou à la fin de la durée de vie de chaque particule primaire. Vous devez donner une autre valeur d'ID de particule pour le "Secondaire", puis un nombre à créer dans le "Compte".

Si l'événement sélectionné est "Par étape", alors chaque image de jeu où une particule primaire existe à l'écran émettra le nombre donné de particules secondaires - par exemple, vous pourriez avoir une fusée avec des particules "flamme", et chaque particule de flamme émet un particule de "fumée" secondaire. Si l'événement est défini sur «À la mort», les particules secondaires ne seront désormais émises que lorsqu'une particule du type primaire atteint la fin de sa durée de vie - par exemple, vous pourriez avoir une particule «explosion» qui émet une «poussière "particule en fin de vie.

Il est important de noter que vous ne pouvez pas utiliser le même ID de type de particule que la particule principale que pour les particules secondaires, et que vous pouvez très rapidement rencontrer des problèmes de performances lors de l'utilisation de cette action car il est facile de créer des milliers de particules très rapidement de cette manière - donc veillez à limiter la durée de vie et le nombre de particules créées par les types de particules primaires et secondaires. Notez également que vous pouvez attribuer une particule secondaire à n'importe quel type de particule et ainsi créer des «chaînes de particules» où une particule en crée une autre à chaque étape, puis cette particule en crée une autre lorsqu'elle atteint la fin de sa durée de vie, etc., des précautions doivent être prises lors de l'enchaînement des particules de cette manière pour s'assurer que les performances ne souffrent pas. Vous pouvez également appeler cette action deux fois pour un seul type de particule et la diffuser à l'étape et éclater à la mort.

Syntaxe de l'action:

Arguments:

Argument Description
Type ID La valeur ID pour le type de particule à donner à la particule secondaire
Event Quand créer la particule secondaire, "Par étape" ou "À la mort"
Death Type La valeur d'ID du type de particule à émettre dans l'événement secondaire choisi
Count Le nombre de particules secondaires à créer dans l'événement secondaire choisi
