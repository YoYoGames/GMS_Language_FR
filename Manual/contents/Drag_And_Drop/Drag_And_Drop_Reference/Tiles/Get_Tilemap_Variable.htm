Obtenir la variable Tilemap

Avec cette action, vous pouvez récupérer l'une quelconque d'un nombre donné de variables pour l' élément de carte de tuiles sur une couche. Lorsque vous créez une couche de tuiles dans l'éditeur de pièces, cette couche contient un élément de carte de tuiles qui est ensuite rempli avec les tuiles d'un ensemble de tuiles. Cet élément de carte de tuiles a certaines valeurs qui lui sont associées, comme une position de décalage, le jeu de tuiles utilisé, la largeur, la hauteur, etc... La liste complète des valeurs que vous pouvez obtenir est:

  • Coordonnées X : la position le long de l'axe x sur le calque où l'élément de carte de tuiles a été placé.
  • Coordonnée Y : position le long de l'axe y sur le calque où l'élément de carte de tuiles a été placé.
  • Colonnes : La largeur (définie comme le nombre de colonnes de cellules de tuiles) de l'élément de carte de tuiles.
  • Lignes : La hauteur (définie comme le nombre de lignes de cellules de tuiles) de l'élément de carte de tuiles.
  • Largeur de tuile : La largeur d'une seule cellule de tuile.
  • Hauteur de la tuile : La hauteur d'une seule cellule de tuile.
  • Ressource de l'ensemble de tuiles : ressource de l'ensemble de tuiles qui a été affectée à l'élément de carte de tuiles pour le dessin.
  • Image actuelle : l'image en cours de dessin pour les tuiles animées.
  • Masque : les données de masque pour l'élément de mosaïque.
  • Masque global : les données de masque global pour tous les éléments de la mosaïque.

Lorsque vous utilisez cette action, vous indiquez le nom de la couche (une chaîne, telle que définie dans l'éditeur de pièce) à partir de laquelle obtenir les données d'élément de carte de tuile, puis le type de données que vous souhaitez récupérer (comme indiqué dans la liste ci-dessus). La valeur retournée sera alors stockée dans la variable cible qui peut avoir été créée précédemment ou peut être une nouvelle temporaire (si vous cochez la case "Temp"). Notez que vous pouvez récupérer des valeurs supplémentaires en cliquant sur l'icône plus à côté de l'action, et en sélectionnant une autre variable et en donnant une autre variable pour stocker la valeur renvoyée.
