Obtenir l'index des tuiles au pixel

Cette action est utilisée pour obtenir un index de tuile sur une couche donnée. Le calque est spécifié à partir du nom du calque donné (une chaîne, comme utilisé pour nommer le calque dans l'éditeur de pièce), puis vous donnez une position x et y pour trouver la tuile pour obtenir l'index. La tuile sera celle dans laquelle la position donnée se situe dans la pièce, et l'index sera renvoyé à la variable cible, où 0 est aucune tuile et un entier positif est l'index de la tuile. Les positions x et y peuvent être marquées comme étant relatives à la position de l'instance appelante, et la variable cible peut être marquée comme une variable temporaire (locale) qui peut être utilisée jusqu'à la fin de l'événement.

Syntaxe de l'action:

Arguments:

Argument Description
Layer Le calque sur lequel définir la vignette
x La position x à vérifier
y La position y à vérifier
Target La variable cible pour stocker l'index des tuiles

Exemple:
