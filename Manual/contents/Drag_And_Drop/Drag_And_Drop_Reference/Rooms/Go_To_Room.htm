Aller dans la chambre

Cette action mettra fin à la salle actuelle et ira à la salle spécifiée par l'index d'entrée. La salle doit exister ou bien une erreur sera donnée, et si vous utilisez la même salle que la salle actuelle, cela aura le même effet que l'utilisation de Redémarrer la salle. Notez que la salle ne changera pas avant la fin de l'événement ou du script où l'action a été appelée, donc toutes les actions après celle-ci seront toujours exécutées.

Syntaxe de l'action:

Arguments:

Argument Description
Room La pièce où aller

Exemple:

Le code de bloc d'action ci-dessus récupère une variable globale puis vérifie si la valeur qu'elle contient est inférieure ou égale à 0. Si c'est le cas, le jeu ira dans une autre salle.
