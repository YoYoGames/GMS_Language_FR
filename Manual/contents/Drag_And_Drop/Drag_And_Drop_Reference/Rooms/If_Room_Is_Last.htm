Si la pièce est la dernière

Cette action vérifiera si la salle actuelle est la dernière pièce répertoriée dans le Gestionnaire de salle. L'action reviendra true si la salle actuelle est la dernière du Room Manager et false si ce n'est pas le cas. Vous pouvez cocher la case non pour le transformer en "Si la pièce n'est PAS la dernière".

Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme indiqué dans l'image ci-dessous:

Ces actions seront désormais exécutées si le "si" est évalué à true, tandis que toutes les actions déposées ailleurs seront exécutées après le bloc "if".
