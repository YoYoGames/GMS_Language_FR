Si la chambre est la dernière

Cette action va vérifier si la pièce actuelle est la dernière pièce listée dans le Room Manager. L'action retournera vrai si la pièce actuelle est la dernière dans le Room Manager et faux si elle ne l'est pas. Vous pouvez cocher la case " non" pour transformer cette action en "Si la pièce n'est pas la dernière".

Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous :

Ces actions seront maintenant exécutées si le "if" vaut true, tandis que toute action déposée ailleurs sera exécutée après le bloc "if". déposée ailleurs sera exécutée après le bloc "if".
