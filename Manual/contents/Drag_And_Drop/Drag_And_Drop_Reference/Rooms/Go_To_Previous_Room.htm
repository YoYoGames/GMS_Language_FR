Aller à la pièce précédente

Cette action fera en sorte que votre jeu se rende dans la salle précédente telle qu'elle était listée dans le gestionnaire de salles au moment où le jeu a été compilé. Si cette salle n'existe pas, une erreur sera lancée et le jeu sera forcé de se fermer. fermer. Notez que la salle ne changera pas avant la fin de l'événement ou du script où l'action a été appelée, donc tout code après cette action sera toujours exécuté.
