Obtenir un nombre aléatoire

Cette action générera un nombre aléatoire dans les limites d'un ensemble de valeurs minimales et maximales. Vous pouvez choisir le type de nombre à générer (décimal ou entier), puis définir la plage du nombre à générer en fournissant une valeur minimale et maximale. La valeur aléatoire générée sera renvoyée à la variable cible (qui peut être marquée comme une variable locale temporaire).

REMARQUE: Lors de la génération d'un entier aléatoire, l'action renverra une valeur entière comprise entre la valeur minimale et la valeur maximale incluse, de sorte qu'une plage de 5 à 10 pourrait renvoyer 5 ou 10 ainsi que tout entier entre les deux. Cependant, la génération d'une décimale aléatoire ne retournera jamais la valeur maximale, de sorte que la plage 5 à 10 peut renvoyer 5 mais ne renverra pas 10 (bien qu'elle puisse renvoyer 9,999999999999).

Syntaxe de l'action:

Arguments:

Argument Description
Type Le type de nombres à utiliser (entier ou décimal)
Minimum Le nombre minimum pour la plage aléatoire
Maximum Le nombre maximum pour la plage aléatoire
Target La variable cible pour stocker la valeur renvoyée

Exemple:

Le code de bloc d'action ci-dessus génère une valeur aléatoire et l'affecte à une variable locale temporaire. Cette valeur est ensuite utilisée dans une boucle de répétition où d'autres appels aléatoires sont effectués pour générer des positions dans la pièce. Ces positions sont ensuite utilisées pour créer un certain nombre d'instances.
