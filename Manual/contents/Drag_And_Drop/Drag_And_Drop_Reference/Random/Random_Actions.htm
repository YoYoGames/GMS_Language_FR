Bibliothèque d'actions aléatoires

La bibliothèque Random propose les actions suivantes pour gérer la génération de nombres aléatoires et les opérations aléatoires:

Obtenir un nombre aléatoire
Randomiser
Choisir

Il est à noter que GameMaker Studio 2 utilisera la même graine lors de la génération de nombres aléatoires chaque fois que vous exécuterez le jeu à partir de l'EDI. Ceci est pour faciliter le débogage et signifie que vous obtiendrez les mêmes résultats initiaux lors de la génération de valeurs aléatoires, mais cela ne se produira pas lorsque vous aurez compilé le jeu dans un exécutable (une nouvelle graine sera générée chaque fois que le jeu est lancé). Si vous ne souhaitez pas ce comportement lors du test, vous devez appeler l' action Randomiser une fois au début du jeu, avant de générer des valeurs aléatoires.
