Si la séquence existe

Cette action peut être utilisée pour vérifier si l'élément de séquence donné existe dans la salle de jeu. Vous donnez l'ID de l'élément, tel que retourné par l'action Create Sequence, et si l'élément de séquence existe dans la salle, alors l'action retournera true sinon elle retournera false.

Si vous indiquez l'argument " Not ", l'action vérifiera si un tel élément de séquence n'existe pas et si aucun n'est trouvé, elle retournera true.
