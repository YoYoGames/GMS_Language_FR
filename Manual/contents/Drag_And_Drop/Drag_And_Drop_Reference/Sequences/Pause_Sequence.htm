Jouer la séquence

Avec cette action, vous pouvez mettre en pause une séquence de lecture. Vous fournissez l'ID d'élément de séquence (tel que renvoyé par l'action Créer une séquence ), et l'action mettra la lecture en pause.

Syntaxe de l'action:

Arguments:

Argument Description
Variable L'ID de l'élément de séquence à mettre en pause.

Exemple:

Le code de bloc d'action ci-dessus vérifie si une variable globale est true. S'il s'agit de l'élément de séquence référencé dans la variable "my_seq "est mis en pause, sinon il est configuré pour jouer.
