Obtenir la tête de séquence

Avec cette action, vous pouvez récupérer la position actuelle de la tête de lecture de l'élément de séquence donné dans la pièce. Vous fournissez l'ID d'élément de séquence (tel que renvoyé par l'action Créer une séquence ), et cette action renverra l'image actuelle sur laquelle se trouve la tête de lecture de la séquence, dans la variable cible. Pour plus d'informations sur la tête de lecture, voir Utilisation de la Dope Sheet.

Notez que la variable cible peut être marquée comme temporaire (locale), qui sera créée pour l'action et peut être utilisée dans toutes les actions suivantes pour l'événement dans lequel se trouve l'action.

Syntaxe de l'action:

Arguments:

Argument Description
Variable L'ID de l'élément de séquence pour obtenir la position de la tête.
Target La variable cible pour maintenir la position de la tête.

Exemple:

Le bloc d'action ci-dessus récupère la position actuelle de la tête de l'ID d'élément de séquence stocké dans la variable "seq_id ". Cette valeur est ensuite utilisée pour définir la position de la tête du même élément de séquence, en l'avançant de 4 images.
