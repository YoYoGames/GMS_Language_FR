Obtenir la longueur de la séquence

Avec cette action, vous pouvez récupérer la longueur (en images) de l'élément de séquence donné dans la pièce. Vous fournissez l'ID d'élément de séquence (tel que renvoyé par l'action Créer une séquence ), et cette action renverra sa longueur dans la variable cible. Ce sera le nombre d'images pendant lesquelles la séquence s'exécute.

Notez que la variable cible peut être marquée comme temporaire (locale), qui sera créée pour l'action et peut être utilisée dans toutes les actions suivantes pour l'événement dans lequel se trouve l'action.

Syntaxe de l'action:

Arguments:

Argument Description
Variable L'ID de l'élément de séquence dont la longueur doit être obtenue.
Target La variable cible pour contenir la longueur.

Exemple:

Le bloc d'action ci-dessus récupère la longueur de l'ID d'élément de séquence stocké dans la variable "seq_id ". Cette valeur est ensuite utilisée pour définir la position de la tête du même élément de séquence à 10 images avant sa fin.
