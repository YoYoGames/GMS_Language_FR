Détruire l'élément de séquence

Vous pouvez appeler cette action chaque fois que vous souhaitez "détruire" un élément de séquence créé précédemment. Vous donnez l'ID d'élément, tel que renvoyé par l'action Créer un élément de séquence, et cette action le détruira, le supprimant de la pièce.

Syntaxe de l'action:

Arguments:

Argument Description
Variable L'ID de l'élément de séquence à détruire.

Exemple:

Le code du bloc d'action ci-dessus vérifie si l'élément de séquence stocké dans la variable "my_seq "existe et s'il existe, il le détruit.
