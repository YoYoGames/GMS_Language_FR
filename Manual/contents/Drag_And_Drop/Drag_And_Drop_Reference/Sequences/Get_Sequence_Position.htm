Obtenir la position de la séquence

Avec cette action, vous pouvez récupérer la position actuelle de l'élément de séquence donné dans la pièce. Vous fournissez l'ID d'élément de séquence (tel que renvoyé par l'action Créer une séquence ), et cette action renverra sa position X et Y dans les variables cibles. Notez que les variables cibles peuvent être marquées comme des variables temporaires (locales), qui seront créées pour l'action et peuvent être utilisées dans toutes les actions suivantes pour l'événement dans lequel l'action se trouve.

Syntaxe de l'action:

Arguments:

Argument Description
Variable L'ID de l'élément de séquence dont il faut obtenir la position.
Target X La variable cible pour maintenir la position x.
Target Y La variable cible pour maintenir la position y.
