Définir la couleur de l'instance

Ce bloc d'action définit la couleur image_blend pour le "mélange" avec le sprite de l'instance. La valeur par défaut est le code hexadécimal $FFFFFFFF (qui représente la couleur blanche au format ARGB ) et ceci dessinera le sprite sans mélange et avec un alpha complet. Toute autre valeur (y compris les constantes de couleur internes comme c_red ou c_aqua) mélangera la couleur spécifiée avec le sprite original et définira la composante image_alpha pour l'instance, mais uniquement lorsque la case Utiliser l'alpha est cochée. Cela écrasera toute valeur alpha définie précédemment à l'aide de l'action Set Instance Alpha.

Veuillez noter que pour que les modifications apportées à cette action soient visibles, l'instance ne doit pas avoir d'événement de dessin (et GameMaker Studio 2 dessinera donc le sprite par défaut) ou doit être dessinée à l'aide de l'action Draw Self. Il est important de noter également que vous devez essayer de limiter les mélanges sur la plate-forme HTML5 (à moins d'utiliser WebGL), car chaque sprite mélangé doit être mis en cache séparément et le fait d'avoir de nombreux sprites mélangés peut nuire aux performances.
