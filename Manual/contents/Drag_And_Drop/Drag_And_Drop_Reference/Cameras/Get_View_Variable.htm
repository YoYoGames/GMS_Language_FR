Obtenir la variable de vue

Cette action vous permet d'obtenir la valeur de certaines variables intégrées liées à un port de vue donné. Vous sélectionnez la variable de port de vue à récupérer, puis donnez le port de vue à la cible (de 0 à 7) et enfin la variable cible pour contenir la valeur renvoyée (cela peut être marqué comme étant une variable locale temporaire). Le port de vue est la zone de l'écran qui sera utilisée pour dessiner une vue de caméra donnée (comme configuré dans l'éditeur de salle), et les différentes variables que vous pouvez obtenir de cette manière sont décrites dans le tableau ci-dessous:

Variable Description
Camera La valeur renvoyée sera la valeur d'ID unique de la caméra à assigner au port (voir ici pour plus d'informations).
Visibility Cela renverra une valeur de true ou false où true est que le port de vue est visible (c'est-à-dire qu'il est dessiné sur l'écran), et false ce qui signifie que rien n'est dessiné.
View Port X Coordinate Il s'agit de la position X du port de vue dans la fenêtre de jeu.
View Port Y Coordinate Il s'agit de la position Y du port de vue dans la fenêtre de jeu.
View Port Width Il s'agit de la largeur (en pixels) du port de vue.
View Port Height Il s'agit de la hauteur (en pixels) du port de vue.
View Port Surface ID Cela renverra la valeur d'ID unique pour la surface affectée au port de vue ou -1 si aucune surface n'a été affectée.

Syntaxe de l'action:

Arguments:

Argument Description
Variable La variable de vue intégrée pour obtenir
View Le port de vue à cibler (de 0 à 7)
Target La variable cible contenant la valeur renvoyée (peut être marquée comme une variable locale temporaire)
