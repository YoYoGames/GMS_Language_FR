Définir la variable de vue

Cette action vous permet de définir certaines variables intégrées liées à la façon dont le port de vue affichera les vues de la caméra de la pièce. Vous sélectionnez la variable de port de vue à éditer, puis donnez le port de vue à la cible (de 0 à 7) et enfin la nouvelle valeur de la variable. Le port de vue est la zone de l'écran qui sera utilisée pour dessiner une vue de caméra donnée (comme configuré dans l'éditeur de salle), et les différentes variables que vous pouvez définir de cette manière sont décrites dans le tableau ci-dessous:

Variable Description
Camera Ceci permet de configurer la fenêtre pour afficher une vue de caméra différente. Vous devez fournir la valeur d'identifiant unique de la caméra vers laquelle passer (voir ici pour plus d'informations).
Visibility Il s'agit de la bascule de visibilité pour le port de vue et attend une valeur de true ou false où le mettre à true rendra le port de vue visible (c'est-à-dire: il s'affichera à l'écran), et une valeur de false ne dessine rien à l'écran.
View Port X Coordinate Il s'agit de la position X du port de vue dans la fenêtre de jeu. Notez cependant que ce n'est vraiment nécessaire que lorsque plus d'un port d'affichage est actif - par exemple pour créer un effet d'écran partagé - et s'il est utilisé lorsqu'un seul port est actif, vous obtenez des résultats inattendus.
View Port Y Coordinate Il s'agit de la position Y du port de vue dans la fenêtre de jeu. Notez cependant que ce n'est vraiment nécessaire que lorsque plus d'un port d'affichage est actif - par exemple pour créer un effet d'écran partagé - et s'il est utilisé lorsqu'un seul port est actif, vous obtenez des résultats inattendus.
View Port Width Il s'agit de la largeur (en pixels) du port de vue. La largeur du port de vue (ou des ports de vue combinés si plusieurs sont actifs) définit la largeur de la fenêtre de jeu ou de la toile d'arrière-plan au début du jeu, donc changer cette valeur après le début du jeu n'aura aucun effet visible sur la taille de la fenêtre de jeu sauf si elle est appelée avec la fonction GML window_set_size(). Notez également que si vous avez une taille de port plus grande ou plus petite que celle attribuée à la caméra, la vue de la caméra sera réduite - ou augmentée - pour s'adapter.
View Port Height Il s'agit de la hauteur (en pixels) du port de vue. La hauteur du port de vue (ou des ports de vue combinés si plusieurs sont actifs) définit la hauteur de la fenêtre de jeu ou de la toile d'arrière-plan au début du jeu, donc changer cette valeur après le début du jeu n'aura aucun effet visible sur la taille de la fenêtre de jeu sauf si elle est appelée avec la fonction GML window_set_size(). Notez également que si vous avez une taille de port plus grande ou plus petite que celle attribuée à la vue de la caméra, la vue de la caméra sera réduite - ou augmentée - pour s'adapter.
View Port Surface ID Avec cette variable, vous pouvez définir le contenu d'un port de vue donné à dessiner sur une surface, la valeur que vous fournissez étant celle de la surface à utiliser pour dessiner le port de vue. Pour plus d'informations, cliquez ici.
