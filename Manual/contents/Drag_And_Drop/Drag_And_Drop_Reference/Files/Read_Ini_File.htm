Lire le fichier Ini

Avec cette action, vous pouvez lire une donnée à partir d'un fichier Ini. Vous devez au préalable avoir ouvert le fichier Ini en lecture / écriture à l'aide de l'action Ouvrir un fichier Ini, puis avec cette action, vous indiquez le type de données à lire (soit un nombre réel soit une chaîne entre ""), ainsi que le en-tête de section et la clé dans cette section pour obtenir les données. Vous pouvez également fournir une valeur par défaut qui peut être soit un nombre réel de chaîne, et le type doit coïncider avec le "Type" défini dans l'action. Cette action par défaut sera renvoyée par l'action si l'une des situations suivantes se produit:

  • Le fichier Ini que vous avez ouvert n'existe pas
  • L'en-tête de section auquel vous faites référence n'existe pas
  • La clé à partir de laquelle vous essayez d'obtenir des données n'existe pas

Dans l'une des circonstances ci-dessus, vous obtiendrez la valeur par défaut renvoyée plutôt qu'une valeur du fichier Ini. Quelle que soit la valeur renvoyée, elle sera stockée dans la variable cible que vous spécifiez, qui peut également être créée à ce moment si elle est marquée comme variable locale temporaire.

IMPORTANT! essayer de lire des données à partir d'un fichier Ini qui n'a pas été ouvert auparavant à l'aide de l' action Ouvrir un fichier Ini provoquera une erreur grave et plantera votre jeu.

Syntaxe de l'action:

Arguments:

Argument Description
Type Le type de données à lire, soit un string (entre guillemets ""), ou un real number
Section La section du fichier Ini à lire (sous forme de chaîne)
Key La clé dans la section à partir de laquelle lire (sous forme de chaîne)
Default La valeur par défaut à renvoyer si le fichier, la section ou la clé n'existent pas
Target La variable cible pour contenir la valeur de retour de l'action
