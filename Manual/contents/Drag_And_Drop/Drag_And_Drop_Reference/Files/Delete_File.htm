Supprimer le fichier

Avec cette action, vous pouvez supprimer un fichier qui a été créé précédemment. Vous fournissez le nom de fichier (sous forme de chaîne) du fichier à supprimer. Notez que vous ne pouvez supprimer aucun fichier inclus dans l'offre groupée du jeu, uniquement ceux qui ont été créés à l'aide des actions Close Ini ou Save Buffer (voir la section sur le système de fichiers pour plus d'informations).

Syntaxe de l'action:

Arguments:

Argument Description
Filename Le nom (sous forme de chaîne) du fichier à supprimer

Exemple:

Le code de bloc d'action ci-dessus vérifiera si le fichier "checkpoint.sav" existe, et s'il le fait alors vérifie si le fichier "checkpoint_OLD.sav" existe. Si ce fichier existe également, il est supprimé, puis le fichier d'origine "checkpoint.sav" est renommé "checkpoint_OLD.sav". Enfin, un tampon est enregistré sous "checkpoint.sav" (essentiellement, nous sauvegardons un fichier tampon enregistré à chaque fois que nous le sauvegardons, de sorte qu'il y ait toujours une sauvegarde "actuelle" et un "ancien" fichier de sauvegarde).
