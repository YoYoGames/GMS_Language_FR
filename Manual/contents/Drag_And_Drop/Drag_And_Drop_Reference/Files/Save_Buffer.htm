Enregistrer le tampon

Avec cette action, vous pouvez enregistrer un fichier contenant les données d'un tampon. Vous fournissez l'ID de tampon unique (tel que renvoyé lorsque vous avez créé le tampon avec Create Buffer ) et donnez un nom au fichier (sous forme de chaîne, avec l'extension de votre choix) et le tampon sera écrit dans ce fichier prêt à être chargé à nouveau en utilisant Load Buffer.

Syntaxe de l'action:

Arguments:

Argument Description
Buffer La valeur d'ID de tampon unique du tampon à enregistrer
Filename Le nom (sous forme de chaîne) du fichier à enregistrer

Exemple:

Le code de bloc d'action ci-dessus créera une variable pour contenir les données de tampon, puis vérifiera si un fichier de sauvegarde de tampon a été créé précédemment. Si le fichier existe, il est chargé dans la variable de tampon que nous avons créée, mais s'il n'existe pas, il est créé, écrit puis sauvegardé, prêt à être chargé la prochaine fois.
