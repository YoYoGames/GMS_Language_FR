Si le fichier existe

Avec cette action, vous pouvez vérifier si un fichier avec le nom donné existe. Vous sélectionnez le fichier à vérifier (comme une piqûre et avec l'extension) et la fonction reviendra true s'il existe, et false si ça ne fait pas. Notez que cela reviendra true si le fichier existe dans l'ensemble du jeu en tant que fichier inclus ou si le fichier a été enregistré pendant que le jeu était en cours d'exécution (voir la section sur le système de fichiers pour plus d'informations).
