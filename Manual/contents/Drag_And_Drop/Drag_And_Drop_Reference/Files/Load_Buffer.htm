Charger le tampon

Avec cette action, vous pouvez charger un tampon qui a été précédemment enregistré avec l'action Save Buffer. L'action créera un nouveau tampon, y chargera le fichier de données et renverra la valeur d'ID unique pour identifier le tampon dans la variable. Vous fournissez le nom du fichier à charger (sous forme de chaîne) et une variable dans laquelle stocker l'ID de tampon renvoyé. Notez que si vous marquez la variable comme une variable locale temporaire, vous devez vous assurer de supprimer le tampon créé par cette fonction. avant la fin du script d'action ou de l'événement, sinon vous perdrez la référence à celui-ci et obtiendrez une fuite de mémoire lorsque la variable est supprimée.

Syntaxe de l'action:

Arguments:

Argument Description
Filename Le nom (sous forme de chaîne) du fichier à ouvrir
Target La variable cible pour contenir la valeur d'ID unique pour le tampon créé lors du chargement

Exemple:

Le code de bloc d'action ci-dessus créera une variable pour contenir les données de tampon, puis vérifiera si un fichier de sauvegarde de tampon a été créé précédemment. Si le fichier existe, il est chargé dans la variable de tampon que nous avons créée, mais s'il n'existe pas, il est créé, écrit puis sauvegardé, prêt à être chargé la prochaine fois.
