Tampon de chargement

Avec cette action, vous pouvez charger un tampon qui a été précédemment sauvegardé avec l'action Save Buffer. L'action créera un nouveau tampon, chargera le fichier de données dans celui-ci, et retournera la valeur ID unique pour identifier le tampon à la variable variable. Vous fournissez le nom du fichier à charger (sous forme de chaîne de caractères) et une variable pour stocker l'ID du tampon retourné. Notez que si vous marquez la variable comme une variable locale temporaire, vous devez vous assurer de supprimer la mémoire tampon créée par cette fonction avant avant la fin du script d'action ou de l'événement, sinon vous perdrez la référence à celui-ci et obtiendrez une fuite de mémoire lorsque la variable sera supprimée.

Arguments :

Arguments Description
Nom de fichier Le nom (sous forme de chaîne) du fichier à ouvrir
Cible La variable cible qui contient la valeur d'identification unique pour le tampon créé lors du chargement.
