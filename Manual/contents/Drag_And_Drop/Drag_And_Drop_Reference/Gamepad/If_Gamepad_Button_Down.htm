Si le bouton du Gamepad est enfoncé

Avec cette action, vous pouvez interroger un gamepad pour savoir si un bouton donné est actuellement maintenu enfoncé. L'action retournera vrai à chaque étape où le bouton est maintenu enfoncé, ou faux sinon, bien que si vous cliquez sur le modificateur not vous pouvez alors vérifier si le bouton n'est pas maintenu enfoncé, c'est-à-dire que l'action renverra vrai si aucun bouton n'est enfoncé et faux s'il l'est. Si vous n'avez besoin de vérifier qu'un seul bouton est enfoncé, utilisez l'action If Gamepad Button Pressed.

Si le bouton vérifié est un bouton analogique, il ne sera pas considéré comme maintenu enfoncé tant que la valeur du bouton n'aura pas dépassé le seuil défini à l'aide de l'action Set Gamepad Button Threshold.

Arguments :

Arguments Description
Manette de jeu L'index du gamepad.
Bouton Le nom du bouton à vérifier.
Non Négation de la vérification (vrai devient faux et vice versa)
