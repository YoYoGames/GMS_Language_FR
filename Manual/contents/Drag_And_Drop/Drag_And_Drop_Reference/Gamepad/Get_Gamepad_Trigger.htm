Obtenez Gamepad Trigger

Vous pouvez utiliser cette action pour obtenir la valeur des différents déclencheurs d'une manette de jeu donnée. Vous fournissez l'index de la manette de jeu (c'est le nombre de "slots" qu'une manette de jeu occupe) et sélectionnez le déclencheur à vérifier (gauche ou droite). La valeur renvoyée sera comprise entre 0 et 1, où 0 correspond à aucune pression sur la gâchette et à 1 pleine pression. Cette valeur renvoyée sera stockée dans la variable cible que vous fournissez pour une utilisation future. Notez que cette fonction ne prend pas en compte le réglage du seuil.

Syntaxe de l'action:

Arguments:

Argument Description
Gamepad L'index de la manette de jeu.
Trigger Le déclencheur à vérifier (gauche / droite)
Target La variable cible dans laquelle stocker la valeur renvoyée.
