Quitter le jeu

Avec cette action, vous pouvez mettre fin au jeu, déclenchant l' événement Autre - Fin de partie. Notez que cela ne se produit pas au moment où l'action est appelée, mais plutôt à la fin de l'événement ou du script en cours, de sorte que toutes les actions que vous avez après l'appel de cette action seront toujours exécutées jusqu'à la fin de l'événement / du script..

Veuillez noter que cette action comporte les restrictions suivantes:

  • Il signalera une erreur sur la iOS car elle est contre les conditions de leur marché.
  • Il n'aura aucun effet lorsqu'il est appelé sur les cibles HTML5 et Windows UWP
  • Il fonctionnera sur Windows (y compris les jeux conçus pour Steam ), Android, Ubuntu ( Linux ) et Mac.
