Exécuter le script

Cette action peut être utilisée pour appeler une fonction de script ou une fonction définie par l'utilisateur. En général, cette action ne serait utilisée que pour le support hérité et ne devrait pas vraiment être utilisée dans les projets actuels ou nouveaux. Pour utiliser l'action, vous donnez l'actif de script à appeler - ou le nom de la variable de fonction définie par l'utilisateur à appeler - puis fournissez tous les arguments nécessaires, c'est-à-dire: toutes les valeurs (chaînes, entiers, indices de ressources, etc.) que vous souhaitez saisir dans le script ou la fonction afin qu'il puisse les utiliser. Vous pouvez ajouter des arguments supplémentaires si nécessaire en cliquant sur l'icône plus à côté de l'action.

Certains scripts et fonctions définies par l'utilisateur peuvent également renvoyer une valeur pour une utilisation ultérieure dans la liste des blocs d'actions, et vous pouvez donc spécifier la variable cible qui doit stocker cette valeur de retour. Cela peut avoir été créé précédemment ou peut être un nouveau local temporaire (si vous cochez la case "Temp "). Les champs cible et argument peuvent être laissés vides s'ils ne sont pas obligatoires.

Notez qu'il est généralement préférable d'utiliser l'action Appel de fonction pour appeler des fonctions définies par l'utilisateur et que vous ne devriez généralement pas avoir besoin d'utiliser cette action ou d'appeler un élément de script directement dans le code.

Syntaxe de l'action:

Arguments:

Argument Description
Script Le nom du script ou de la fonction définie par l'utilisateur à appeler.
Argument0... Argument3 Les différents arguments (valeurs) à transmettre au script ou à la fonction (les arguments inutilisés peuvent être laissés vides)
Target Le nom de la variable à cibler pour toutes les valeurs renvoyées (peut être laissé vide)
