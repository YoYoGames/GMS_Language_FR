Si variable

Cette action est utilisée pour vérifier l'état d'une variable donnée par rapport à une valeur donnée à l'aide d'une expression spécifique. Une instruction "if" retournera toujours soit true ou false et les expressions de vérification peuvent être l'une des suivantes:

  • Égale à - La variable et la valeur sont toutes deux exactement égales
  • Inférieur à - La variable est inférieure à la valeur
  • Supérieur à - La variable est supérieure à la valeur
  • Inférieur ou égal à - La variable est inférieure ou égale à la valeur
  • Supérieur ou égal à - La variable est supérieure ou égale à la valeur

Si vous marquez l'argument "Non", alors ce qui précède sera des expressions négatives, par exemple "égal à" devient " non égal à". Notez également que ni la variable ni les arguments de valeur ne sont limités aux valeurs numériques et peuvent être une chaîne, un pointeur, un ID de ressource ou tout autre élément qu'une fonction peut renvoyer ou utiliser (voir la section sur les types de données pour plus d'informations), mais vous ne pouvez pas comparer deux types de données différents sans obtenir une erreur - par exemple, si la variable contient une chaîne et que vous essayez de vérifier si elle est égale à un nombre, vous obtiendrez une erreur de syntaxe.

REMARQUE IMPORTANTE: en raison de problèmes de précision en virgule flottante, vérifier si deux valeurs sont exactement égales peut renvoyer false, puisque l'un peut être exactement 1, tandis que l'autre peut être 1.00000000000001. Cela peut être évité en utilisant l' action Decimal to Integer avant de vérifier ou d'utiliser les expressions «supérieur à» ou «inférieur à».
