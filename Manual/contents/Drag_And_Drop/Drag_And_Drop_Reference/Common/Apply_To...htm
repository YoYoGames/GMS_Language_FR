Postuler à...

Cette action peut être utilisée pour modifier la portée de l'action pour toutes les autres actions à enchaîner à celle-ci. Comme avec unif action, si vous déposez d'autres actions à droite de l' action Appliquer à..., elles seront exécutées à partir de l'objet ou de l'instance de portée. La portée est choisie dans le menu contextuel affiché lorsque vous cliquez sur la petite flèche en haut à droite de l'action, et de cette façon, vous pouvez indiquer à un autre ensemble d'objets - ou même à une seule instance si vous avez l'ID d'instance - à exécutez plusieurs actions avant de continuer avec le code de l'objet actuel.

Notez que pour ajouter des actions dans le bloc "S'applique à", elles doivent être déposées sur le côté de l'action, comme indiqué dans l'image ci-dessous:

Ces actions seront désormais exécutées à partir de l'objet ou de l'instance ciblée par le bloc «S'applique à», tandis que toutes les actions déposées ailleurs seront exécutées après le bloc «S'applique à».
