Expression If

Cette action est utilisée pour vérifier qu'une expression est vraie ou fausse, et vous pouvez vérifier n'importe quelle expression logique ici, tant qu'elle donne l'une ou l'autre valeur. Par exemple, vous pouvez utiliser l'une des expressions suivantes :

x < (room_width / 2)
a == b
++val >= 100

Si vous marquez l'argument " Not ", alors les expressions ci-dessus seront des expressions négatives, par exemple "==" devient "not=". Notez également que l'expression n'est pas limitée aux valeurs numériques et que vous pouvez vérifier les chaînes de caractères, les pointeurs, les ID de ressources ou tout autre élément qu'une fonction peut retourner ou utiliser (voir la section sur les types de données pour plus d'informations), mais vous ne pouvez pas comparer deux types de données différents sans obtenir une erreur - par exemple, si vous essayez d'utiliser une chaîne de caractères et un entier dans l'expression, vous obtiendrez une erreur. Pour plus d'informations sur les expressions, voir ici.

REMARQUE IMPORTANTE: En raison de problèmes de précision de la virgule flottante, vérifier si deux valeurs sont exactement égales peut renvoyer un résultat faux, car l'une peut être exactement 1, alors que l'autre peut être 1.00000000000001. Vous pouvez éviter ce problème en utilisant l'action Decimal to Integer avant de vérifier ou en utilisant les expressions "plus grand que (>)" ou "moins grand que (<)".

Cela permettra de vérifier si toutes les expressions données retournent vrai en une seule fois et d'exécuter le code d'action ajouté uniquement si c'est le cas (voir l'exemple ci-dessous).
