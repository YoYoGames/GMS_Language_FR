Appel de fonction

Cette action peut être utilisée pour appeler n'importe quelle fonction GML ou n'importe quelle fonction définie par l'utilisateur. Vous donnez la fonction à appeler puis fournissez tous les arguments supplémentaires dont elle a besoin, c'est-à-dire: toutes les valeurs (chaînes, entiers, indices de ressources, etc...) que vous souhaitez entrer dans la fonction afin qu'elle puisse les utiliser. Vous pouvez ajouter des entrées d'argument supplémentaires en cliquant sur l'icône plus à côté de l'action, et vous devez vous assurer que votre nombre d'arguments correspond à celui de la fonction utilisée (c'est-à-dire: si la fonction nécessite 3 arguments, vous devez fournir 3 arguments). Certaines fonctions peuvent également renvoyer une valeur pour une utilisation ultérieure dans la liste des blocs d'actions, et vous pouvez donc également spécifier la variable cible qui doit stocker cette valeur de retour, qui peut avoir été créée précédemment ou peut être une nouvelle variable locale temporaire (si vous cochez la case à cocher "Temp"). Les champs cible et argument peuvent être laissés vides s'ils ne sont pas obligatoires.

Syntaxe de l'action:

Arguments:

Argument Description
Function La GML à appeler.
Argument 0... Argument X Les différents arguments (valeurs) à transmettre à la fonction.
Target Le nom de la variable à cibler pour toutes les valeurs renvoyées (peut être laissé vide).
