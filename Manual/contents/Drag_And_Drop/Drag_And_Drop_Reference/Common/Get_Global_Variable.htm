Obtenir une variable globale

Cette action permet de récupérer la valeur d'une variable globale et de l'affecter à une autre instance ou à une variable temporaire (locale). La variable à laquelle vous attribuez la valeur globale peut être celle qui a été précédemment créée, ou vous pouvez cocher la case "Temp" et donner un nouveau nom de variable pour que l'action crée une nouvelle variable temporaire (locale) spécifiquement pour contenir le évaluer.

Notez que vous pouvez ajouter des variables supplémentaires dans la même action en cliquant sur l'icône plus à côté de l'action, et en donnant un autre nom de variable globale et un nom de variable locale temporaire. Pour des informations plus avancées sur les variables, veuillez consulter la section Variables et portée des variables.

Syntaxe de l'action:

Arguments:

Argument Description
Name Le nom de la variable globale pour laquelle obtenir la valeur.
Target La variable cible contenant la valeur renvoyée - elle aurait dû être créée précédemment si "Temp" n'est pas coché (voir ci-dessous).
Temp Créez une variable temporaire (locale) pour contenir la valeur renvoyée (désactivée par défaut)
