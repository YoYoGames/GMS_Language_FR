Revenir

Cette action est destinée à être utilisée lors de la création de fonctions de bloc d'action et est utilisée pour renvoyer une valeur à l'instance appelant le script. La valeur de retour peut être tout ce qu'une variable peut contenir - comme une chaîne, une valeur numérique, un pointeur ou un ID de ressource - ou elle peut être une variable elle-même. Notez que lorsque nous appelons l'action de retour dans une fonction, la fonction sera quittée - renvoyant la valeur donnée - et aucune autre action après le retour ne sera appelée, bien que si vous avez des actions dans l' événement après avoir appelé la fonction, celles-ci seront toujours courir.

Syntaxe de l'action:

Arguments:

Argument Description
Return La valeur de retour.

Exemple:

La fonction de bloc d'action ci-dessus définit une variable temporaire sur le mot-clé noone, puis vérifie une variable d'entrée pour voir quelle valeur elle a actuellement. Si la variable est supérieure à 100, une instance est créée et son ID d'instance unique est stocké dans la variable temporaire, qui est ensuite renvoyée par la fonction.
