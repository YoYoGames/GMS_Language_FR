Nouveau

Cette action est uniquement utilisé lors de la création de structures à l'aide d'une fonction qui a été marquée comme fonction de constructeur (voir Déclarer une nouvelle fonction pour plus d'informations). Cette action prend la méthode de fonction pour une fonction précédemment définie, ainsi qu'un certain nombre d'arguments, que vous pouvez développer pour en ajouter si nécessaire à l'aide du icône sur la gauche. Ces arguments doivent correspondre aux entrées requises par la fonction et seront utilisés pour remplir la structure en cours de création. La structure sera retournée à la variable cible, qui peut être marquée comme une variable locale temporaire ou non.

Syntaxe de l'action:

Arguments:

Argument Description
Script Le nom du script ou de la fonction définie par l'utilisateur à appeler.
Argument0... Argument3 Les différents arguments (valeurs) à transmettre au script ou à la fonction (les arguments inutilisés peuvent être laissés vides)
Target Le nom de la variable à cibler pour toutes les valeurs renvoyées (peut être laissé vide)
