L'action "else" est uniquement utilisée en conjonction avec If Variable (ou toute autre action "if"), et est un moyen de définir un ensemble d'actions qui doivent être effectuées si l'évaluation "if" retourne false. Par exemple, vous pouvez avoir une action "si" qui vérifie si la variable de santé des joueurs est inférieure ou égale à 0, et si cela donne la valeur true, vous détruisez l'instance. Cependant, si vous souhaitez effectuer un autre ensemble d'actions de code si la santé du joueur n'est pas inférieure à 0, vous pouvez ajouter un «autre» dans, puis spécifier les actions à effectuer tandis que le if s'évalue à false.
