Définir la variable globale

Cette action permet soit d'affecter une valeur à une variable globale (en la créant dans le processus), soit de modifier la valeur d'une variable globale existante. Les variables globales sont des variables qui "n'appartiennent" à aucune instance spécifique, mais plutôt à l'ensemble du projet, et en tant que telles peuvent être consultées et modifiées par n'importe quelle instance, script ou code. Pour plus d'informations sur les variables globales, cliquez ici.

Vous donnez le nom de la variable globale puis la valeur, qui peut être ajoutée par rapport à la valeur d'origine si nécessaire. Si vous donnez un nouveau nom de variable (c'est-à-dire: une variable qui n'a jamais été assignée auparavant dans tout le jeu), cela créera la nouvelle variable globale et lui assignera la valeur que vous donnez, tandis que si vous donnez le nom d'une variable globale existante, il changera simplement la valeur pour la nouvelle. Notez que la vérification de Relatif ajoutera la valeur à la valeur de la variable globale existante (utilisez un nombre négatif pour soustraire) et vous pouvez également utiliser l' expression s et les variables créées précédemment comme valeur.

Il est à noter que lorsque nous parlons de «valeurs», nous ne parlons pas uniquement de valeurs numériques, car une variable peut être une chaîne, un pointeur, un ID de ressource ou tout autre élément qu'une fonction peut renvoyer ou utiliser. Dans le cas d'une chaîne, la vérification de Relative concaténera la valeur avec la variable précédemment contenue, donc si votre variable était "Hello" et que vous lui attribuez "World" en utilisant le contrôle relatif, la valeur finale de la variable sera "Hello World ". Il faut cependant veiller à ce que les valeurs des variables soient toutes les deux du même type de données, car essayer d'ajouter, par exemple, un entier et une chaîne, entraînera une erreur du compilateur.

Vous pouvez également ajouter des variables supplémentaires dans la même action en cliquant sur l'icône plus à côté de l'action, et en donnant un autre nom et une autre valeur de variable globale. Pour des informations plus avancées sur les variables, veuillez consulter la section Variables et portée des variables.
