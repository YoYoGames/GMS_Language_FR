Définir des vies

Avec cette action, vous pouvez créer ou définir une variable d'instance pour des vies. Vous fournissez la valeur pour définir la variable lives sur ou pour l'ajouter à la variable lives (si vous cochez la case relative, la valeur sera ajoutée à la valeur "lives" actuelle). Notez que contrairement aux variables d'instance régulières, le nom de la variable pour "lives" est stocké en interne. Pour récupérer la valeur du score, vous devez utiliser l'action Get Lives.

Syntaxe de l'action:

Arguments:

Argument Description
Lives Le montant auquel définir les vies (ou à ajouter aux vies si la case relative est cochée)
