Définir la santé

Avec cette action, vous pouvez créer ou définir une variable d'instance pour la santé. Vous fournissez la valeur pour définir la variable de santé ou pour l'ajouter à la variable de santé (si vous cochez la case relative, la valeur sera ajoutée à la valeur actuelle de "santé"). Notez que si vous souhaitez utiliser l' action Draw Instance Healthbar, cette valeur doit être comprise entre 0 et 100, sinon il peut s'agir de tout montant dont vous avez besoin. Notez également que contrairement aux variables d'instance régulières, le nom de la variable "santé" est stocké en interne. Pour récupérer la valeur du score, vous devez utiliser l'action Get Health.

Syntaxe de l'action:

Arguments:

Argument Description
Health Le montant auquel définir la santé (ou à ajouter à la santé si le paramètre relatif est coché)
