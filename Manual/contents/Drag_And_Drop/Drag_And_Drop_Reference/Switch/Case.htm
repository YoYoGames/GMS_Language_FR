Case

Cette action créera un nouveau cas dans une Switch. Vous donnez la valeur du cas et elle sera comparée à la valeur d'entrée pour le commutateur, et si elles correspondent, les actions ajoutées sous le cas seront exécutées. Les actions qui sont ajoutées à une affaire pour être exécutées uniquement dans cette affaire doivent être placées à côté de l'action, comme vous ajouteriez une action à un "si" :

Notez qu'une valeur Case doit être une constante, qui peut être de n'importe quel type de données (réel, chaîne de caractères, etc...), tant qu'elle est évaluée à une seule valeur à la compilation - vous ne pouvez donc pas utiliser d'expressions qui dépendent de variables. Notez également que Case ne peut pas être utilisé en dehors d'un Switch et que si vous essayez de le faire, vous obtiendrez une erreur.
