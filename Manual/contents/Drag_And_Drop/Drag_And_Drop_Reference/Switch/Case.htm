Cas

Cette action créera un nouveau cas dans un Switch. Vous donnez la valeur pour le cas et elle sera comparée à la valeur d'entrée pour le Switch, et si elles correspondent, les actions ajoutées sous le cas seront exécutées. Les actions qui sont ajoutées à un cas à exécuter uniquement dans ce cas doivent être supprimées du côté de l'action, tout comme vous ajouteriez une action à un "si":

Notez qu'une valeur Case doit être une constante, qui peut être n'importe quel type de données (réel, chaîne, etc...), tant qu'elle s'évalue à une valeur unique lors de la compilation - vous ne pouvez donc pas utiliser d'expressions qui dépendent de variables. Notez également que Case ne peut pas être utilisé en dehors d'un Switch et essayer de le faire vous donnera une erreur.
