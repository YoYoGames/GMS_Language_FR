Switch

Cette action créera une nouvelle Switch à laquelle vous pourrez ensuite ajouter d'autres cas à évaluer. Vous donnez la valeur à utiliser pour comparer aux cas dans le Switch, puis ajoutez les différentes actions de cas, de la même manière que vous ajouteriez une action à un "si", c'est-à-dire: la déposer sur le côté de l'action plutôt que dessous:

Notez que vous ne pouvez ajouter aucun autre type d'action dans un commutateur sans avoir au préalable ajouté au moins un cas pour le contenir.

Syntaxe de l'action:

Arguments:

Argument Description
Value La valeur à utiliser par le commutateur pour l'évaluation

Exemple:

Le code de bloc d'action ci-dessus récupère la valeur stockée dans une variable, puis la compare aux différents cas possibles. Si l'un des cas est identique à la valeur, l'action dans ce cas sera exécutée, sinon le cas suivant sera évalué. Une fois l'interrupteur en marche, une alarme est réglée sur 30.
