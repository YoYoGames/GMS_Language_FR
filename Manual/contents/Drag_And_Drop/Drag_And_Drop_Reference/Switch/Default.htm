Défaut

Cette action créera un nouveau cas par défaut dans un Switch. Le cas par défaut est celui qui sera exécuté si la valeur d'entrée ne correspond à aucune autre constante de cas. Les actions ajoutées à un cas par défaut doivent être supprimées du côté de l'action, tout comme vous ajouteriez une action à un "si":

Le code de bloc d'action ci-dessus récupère la valeur stockée dans une variable globale, puis la compare aux différents cas possibles. Si l'un des cas est identique à la valeur, l'action dans ce cas sera exécutée, sinon le cas suivant sera évalué. Si aucun cas ne correspond à la valeur d'entrée, le cas par défaut sera exécuté à la place.
