Définir la vitesse de suivi du chemin

Cette action définira la vitesse, en pixels par image de jeu, à laquelle une instance doit suivre un chemin (notez que vous devez avoir appelé l' action Démarrer le chemin suivant pour attribuer un chemin à l'instance pour que cela ait un effet). Vous pouvez utiliser des valeurs négatives pour signifier que l'instance doit suivre le chemin en sens inverse et il convient de noter que la vitesse que vous définissez ici peut être modifiée par le modificateur de vitesse défini dans l'éditeur de chemin pour chaque point de chemin, donc si vous définissez une vitesse de chemin à 2 et sur le chemin, l'un des points a un modificateur de vitesse de 200%, la vitesse réelle des pixels par image du jeu à ce moment-là sera de 4.

Syntaxe de l'action:

Arguments:

Argument Description
Speed La vitesse (en pixels par image de jeu) à laquelle l'instance doit suivre le chemin

Exemple:

Le code de bloc d'action ci-dessus démarre une instance en suivant un chemin, puis génère une valeur aléatoire entre 2 et 4 avant d'utiliser cette valeur pour définir la vitesse à laquelle l'instance se déplace le long du chemin.
