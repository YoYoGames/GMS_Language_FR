Obtenir la vitesse de suivi du chemin

Cette action obtiendra la vitesse, en pixels par image de jeu, à laquelle une instance est définie pour suivre un chemin (notez que vous devez avoir appelé l' action Démarrer le chemin suivant pour attribuer un chemin à l'instance pour que cela renvoie une valeur autre que 0). La valeur renvoyée sera stockée dans la variable cible que vous donnez, ou vous pouvez marquer la variable comme une variable locale temporaire pour la créer et affecter la valeur.

Syntaxe de l'action:

Arguments:

Argument Description
Target La variable cible pour contenir la vitesse de suivi de chemin renvoyée
