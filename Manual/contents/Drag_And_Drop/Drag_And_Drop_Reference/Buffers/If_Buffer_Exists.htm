Si le tampon existe

Cette action peut être utilisée pour vérifier si un ID de tampon donné est valide et si le tampon existe réellement en mémoire. Si l'ID de tampon est valide et existe, l'action retournera true sinon il reviendra false. Vous pouvez marquer "non" pour que l'action vérifie si un tampon n'existe pas.
