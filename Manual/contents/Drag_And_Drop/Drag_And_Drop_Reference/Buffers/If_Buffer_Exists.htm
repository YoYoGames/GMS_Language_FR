Si le tampon existe

Cette action peut être utilisée pour vérifier si un ID de tampon donné est valide et si le tampon existe réellement en mémoire. Si l'ID du tampon est valide et existe, l'action renverra true, sinon elle renverra false. Vous pouvez marquer "not" (pas) pour que l'action vérifie si un tampon n' existe pas.
