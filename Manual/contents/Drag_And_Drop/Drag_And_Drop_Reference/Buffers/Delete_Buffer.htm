Supprimer le tampon

Cette action peut être utilisée pour supprimer un tampon de la mémoire, libérer les ressources utilisées pour le créer et supprimer toutes les données qu'il peut actuellement contenir. Vous fournissez la valeur d'ID de tampon unique renvoyée par l'action Créer un tampon. Notez que vous pouvez sélectionner plusieurs tampons à supprimer en cliquant sur l'icône plus à côté de l'action et en ajoutant un autre ID de tampon.

REMARQUE: il est important de toujours supprimer de la mémoire toutes les ressources créées dynamiquement lorsque vous n'en avez plus besoin pour éviter les fuites de mémoire.
