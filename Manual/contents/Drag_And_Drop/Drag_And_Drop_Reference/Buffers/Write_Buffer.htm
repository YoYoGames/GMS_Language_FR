Tampon d'écriture

Cette action peut être utilisée pour écrire des données dans un tampon. Vous fournissez l'ID de tampon unique dans lequel écrire (tel que renvoyé par l'action Créer un tampon ), puis la valeur à écrire, qui peut être un réel ou une chaîne, mais notez que toutes les valeurs sont stockées sous forme de chaînes, donc lorsque vous lisez à partir du buffer (à l'aide du tampon de lecture ), vous devrez peut-être utiliser l'une des actions de type de données pour le convertir d'une chaîne à la valeur d'origine. Gardez à l'esprit également que les tampons sont lus et écrits séquentiellement et que l'écriture dans un tampon déplacera la position «de recherche» le long d'un, ce qui signifie que pour écrire divers éléments de données, vous appelez simplement cette action le nombre de fois requis.

Arguments:

Argument Description
Buffer L'index du tampon (stocké dans une variable)
