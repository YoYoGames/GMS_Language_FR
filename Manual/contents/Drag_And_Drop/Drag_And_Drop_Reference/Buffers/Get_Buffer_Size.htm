Obtenir la taille de la mémoire tampon

Cette action peut être utilisée pour récupérer la taille d'un tampon, en octets. Vous fournissez la valeur unique de l'ID du tampon telle qu'elle est renvoyée par l'action Créer un tampon, puis vous donnez une variable cible à laquelle renvoyer la valeur de la taille. La variable cible peut être marquée comme étant une variable locale temporaire, auquel cas elle sera créée pour vous et supprimée de la mémoire à la fin du script ou de l'événement de l'action en cours.

Syntaxe de l'action :

Arguments :

Arguments Description
Tampon L'index du tampon (stocké dans une variable)
Cible La variable cible qui contient la valeur retournée
