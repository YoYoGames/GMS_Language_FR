Tampon de copie

Cette action peut être utilisée pour copier toutes les données stockées dans un tampon vers un autre. Lors de l'utilisation de deux tampons et de la copie de l'un à l'autre, les deux tampons doivent avoir été créés au préalable à l'aide de l' action Créer un tampon et vous fournissez l'ID de tampon unique renvoyé en tant qu'arguments from / to.

REMARQUE: vous ne pouvez pas copier dans le même tampon.

Syntaxe de l'action:

Arguments:

Argument Description
Buffer L'index du tampon (stocké dans une variable) à partir duquel copier
Copy To L'indice de tampon (stocké dans une variable) à copier

Exemple:

Le code de bloc d'action ci-dessus créera un nouveau tampon et stockera son ID dans une variable temporaire. Il vérifie ensuite si le tampon indexé dans la variable globale donnée existe, et si c'est le cas, il est copié dans le tampon temporaire et la position de lecture / écriture est réinitialisée au début, et si elle n'existe pas, elle est créé.
