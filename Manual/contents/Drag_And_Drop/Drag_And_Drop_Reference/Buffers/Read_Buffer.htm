Tampon de lecture

Cette action peut être utilisée pour lire des données à partir d'un tampon précédemment créé. Vous fournissez la valeur unique de l'ID du tampon (telle que renvoyée par l'action Créer un tampon) et vous donnez une variable pour stocker la valeur renvoyée. Si vous marquez la variable comme une variable locale temporaire, elle sera créée pour vous et supprimée à la fin de l'événement ou du script d'action. Toutes les données sont renvoyées sous forme de chaîne de caractères. Si vous avez stocké des nombres réels, vous devrez utiliser l'une des actions de type de données pour les convertir de la chaîne de caractères en valeur. Gardez également à l'esprit que les tampons sont lus et écrits de manière séquentielle et que la lecture d'un tampon déplacera la position de "recherche" vers l'entrée suivante, ce qui signifie que pour lire divers éléments de données il suffit d'appeler cette action le nombre de fois nécessaire et de stocker chaque élément de données renvoyé dans une variable.
