If End Of Buffer

Cette action peut être utilisée pour vérifier si la position de lecture/écriture a atteint la fin du tampon. Vous fournissez la valeur unique de l'ID du tampon telle qu'elle est retournée par l'action Create Buffer et l'action retournera vrai si la position de lecture/écriture a atteint la fin de toutes les données disponibles dans le tampon, ou faux si ce n'est pas le cas. Vous pouvez marquer "not" pour que l'action vérifie si la position de lecture/écriture n'est pas à la fin du tampon.
