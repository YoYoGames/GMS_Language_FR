Tampon de recherche

Cette action peut être utilisée pour se déplacer (rechercher) vers une position dans un tampon. Vous fournissez la valeur unique de l'ID du tampon telle qu'elle est renvoyée par l'action Créer un tampon, puis la valeur de décalage en octets vers laquelle vous souhaitez vous déplacer. La valeur de décalage sera sera appliquée à partir du début du tampon, donc l'appel de cette fonction déplacera la position de lecture/écriture au début du tampon, puis déplacera le nombre d'octets donné et positionnera la nouvelle position de lecture/écriture à cet endroit. Cependant, si vous marquez le décalage comme relatif, la position de lecture/écriture sera déplacée vers l'avant (ou vers l'arrière si le décalage est une valeur négative) à partir de la position actuelle.

Syntaxe de l'action :

Arguments :

Arguments Description
Tampon L'index du tampon (stocké dans une variable)
décalage Le décalage (en octets) par rapport à la position de recherche actuelle.
