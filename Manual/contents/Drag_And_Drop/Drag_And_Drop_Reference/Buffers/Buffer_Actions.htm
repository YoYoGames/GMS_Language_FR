Bibliothèque d'actions de tampons

Un tampon est une région de la mémoire physique utilisée pour contenir temporairement des données pendant qu'elles sont déplacées d'un endroit à un autre ou manipulées d'une manière ou d'une autre, par exemple, vous pouvez utiliser une mémoire tampon pour conserver les données d'état du jeu. GameMaker Studio 2 propose un certain nombre d'actions spécifiques liées à la création, à la modification et au chargement / enregistrement de tampons, mais avant de les utiliser, veuillez noter les points suivants:

  • Lorsqu'un tampon est créé, il est automatiquement effacé et rempli à 0.
  • L'argument "taille" des fonctions de tampon se réfère toujours à la taille en octets du tampon.
  • DnD tampons DnD™ sont toujours alignés sur 1 octet. L '«alignement» fait référence à la façon dont les données sont stockées dans un tampon et est utilisé lors de l'utilisation des fonctions de tampon GML, mais pour le Drag And Drop, ce n'est pas pertinent.
  • Les données dans les tampons sont lues et écrites de manière séquentielle (c'est-à-dire: vous appelez l' action Write Buffer trois fois, puis elle écrira les premières données, avancera, rectifiera les secondes données, avancera puis écrira les dernières données), avec quelque chose appelé la valeur "seek" pour vous dire où dans le tampon la prochaine lecture / écriture aura lieu.
  • Toutes les données sont écrites dans la mémoire tampon et renvoyées à partir de la mémoire tampon sous forme de chaîne.Par conséquent, si vous avez stocké des nombres réels, vous devrez utiliser l'une des actions de type de données pour les convertir de la chaîne en valeur.

La mémoire utilisée pour créer des tampons est la mémoire système, donc même lorsque le jeu n'a pas le focus, toutes les données stockées dans une mémoire tampon doivent être en sécurité, mais si l'application est fermée ou redémarrée, elle sera perdue. Il convient également de noter que les tampons peuvent également être enregistrés sous forme de fichiers, puis chargés à nouveau dans le jeu à l'aide des actions sur les fichiers.
