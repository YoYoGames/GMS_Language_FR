Chaîne en nombre

Cette action peut convertir une chaîne en un nombre. Vous fournissez la chaîne à modifier, puis donnez une variable cible à laquelle renvoyer le nombre réel (cette variable peut être marquée comme une variable locale temporaire). Notez que si la chaîne contient des caractères autres que des nombres et une virgule décimale, elle ne sera pas convertie, bien que si la chaîne commence par un nombre, ceux-ci seront convertis et le reste de la chaîne exclu. Il est important de noter que lors du passage de décimales dans cette action, la chaîne renvoyée sera arrondie à deux décimales uniquement. Vous trouverez ci-dessous un tableau des valeurs d'entrée et ce que vous attendez d'un retour

Chaîne de caractères Valeur retournée
"123" 123
"123 Hello World" 123
"Hello World" 0 (par défaut si la chaîne ne peut pas être convertie, 0 sera renvoyé)
"Hello World 123" 0
"123.45678" 123,46

Syntaxe de l'action:

Arguments:

Argument Description
Value La chaîne à convertir
Target La variable cible contenant la valeur renvoyée
