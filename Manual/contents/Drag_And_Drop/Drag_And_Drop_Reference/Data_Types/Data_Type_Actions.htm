Bibliothèque d'actions de types de données

GameMaker Studio 2 a un certain nombre de types de données différents, comme des entiers, des chaînes, des pointeurs, etc... et certains d'entre eux peuvent être convertis en d'autres. Cette section du manuel présente les différentes actions disponibles pour traiter et convertir différents types de données.

REMARQUE: vous pouvez en savoir plus sur les différents types de données ici.

La bibliothèque Types a les actions suivantes:

Décimal en entier
Chaîne en nombre
Nombre en chaîne
