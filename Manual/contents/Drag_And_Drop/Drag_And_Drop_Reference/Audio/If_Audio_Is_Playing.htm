Si l'audio est en cours de lecture

Cette action peut être utilisée pour vérifier si une instance d'une ressource sonore est en cours de lecture dans votre jeu. Vous donnez la ressource sonore à vérifier à partir de l'explorateur d'actifs et la fonction retournera true si elle joue et false si ce n'est pas le cas, bien que si vous vérifiez l'indicateur "not", cela sera inversé et la fonction renverra true si le son n'est pas joué et false sinon.

Notez que pour ajouter des actions dans le bloc "if", elles doivent être déposées sur le côté de l'action, comme le montre l'image ci-dessous :

Ces actions seront désormais exécutées si le "if" vaut true, tandis que toute actions déposées ailleurs seront exécutées après le bloc "if".
