Arrêter l'audio

Cette action peut être utilisée pour arrêter la lecture d'un son. Vous sélectionnez le son dans l'explorateur de ressources et toutes les instances de ce son s'arrêteront de jouer (ainsi, si vous avez défini le même son pour qu'il soit joué dans trois événements différents, tous s'arrêteront).
