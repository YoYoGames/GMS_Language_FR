Suspendre l'audio

Cette action peut être utilisée pour mettre en pause le son choisi s'il est en cours de lecture. Notez que si vous avez joué plusieurs instances du même son, tous est mise en pause, et cela ne pause pas de sons joués après l'action a été appelé. Vous pouvez relancer la lecture d'un son mis en pause à l'aide de l'action Reprendre l'audio.

Syntaxe de l'action:

Arguments:

Argument Description
Sound La ressource sonore pour faire une pause

Le code de bloc d'action ci-dessus vérifiera une variable globale et s'il est évalué à true puis le son donné est mis en pause, sinon il est repris.
