Obtenir la hauteur du son

Cette action récupère la hauteur de la ressource sonore que vous fournissez. La valeur de retour sera un multiplicateur pour la hauteur réelle enregistrée de l'échantillon, où une valeur de 1 ne représente aucun changement. Pour plus d'informations sur le pitch, voir l'action Set Audio Pitch.

Arguments :

Arguments Description
Son La ressource sonore pour obtenir la hauteur de
Cible La variable cible pour stocker le pas retourné
