Définir l'état de la fenêtre

Avec cette action, vous pouvez définir l'état de la fenêtre de jeu sur plein écran ou fenêtré. Il y a trois options ici:

  • Basculer - Lorsque cette option est sélectionnée, chaque fois que vous appelez l'action, la fenêtre bascule entre les états plein écran et fenêtré
  • Plein écran - Cela fera passer la fenêtre en plein écran à partir d'un état fenêtré
  • Fenêtre - Cela fera revenir la fenêtre à un état fenêtré si elle est en plein écran

Notez que l'apparence du jeu en plein écran dépendra des options de jeu - Paramètres graphiques pour vos plates-formes cibles, car si le jeu ne dispose pas d'un mécanisme pour s'adapter à différents rapports d'aspect et résolutions, vous pouvez vous étirer (si le jeu est définie sur Pleine échelle ), ou vous pouvez ajouter des barres noires (si l'option de jeu est définie sur Conserver le rapport hauteur / largeur ). En général, si vous n'avez rien en place pour gérer différentes résolutions, vous devriez utiliser Keep Aspect Ratio pour les plates-formes cibles sur lesquelles vous souhaitez compiler.
