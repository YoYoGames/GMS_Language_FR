Obtenir l'index de l'élément de liste

Avec cette action, vous pouvez récupérer la position d'index d'un élément dans la liste donnée. Vous fournissez la variable qui stocke l'index de la liste (tel que renvoyé par l'action Créer une liste ) et la valeur dont vous souhaitez trouver la position d'index dans la liste, ainsi qu'une variable cible pour stocker la position d'index renvoyée pour l'élément (qui peut être marqué comme une variable locale temporaire à utiliser jusqu'à la fin du script ou de l'événement). Notez que si la valeur que vous recherchez n'existe pas dans la liste, il n'y aura pas de position d'index de liste à renvoyer, et donc la valeur de retour sera -1.

Syntaxe de l'action:

Arguments:

Argument Description
List L'index (stocké dans une variable) de la liste à supprimer
Value La valeur pour laquelle vérifier la liste
Target La variable cible pour stocker l'index renvoyé

S'il existe, l'élément est supprimé de la liste et l'instance détruite.
