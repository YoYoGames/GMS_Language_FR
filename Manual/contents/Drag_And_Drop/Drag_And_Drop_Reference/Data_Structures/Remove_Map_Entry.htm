Supprimer une entrée de carte

Avec cette action, vous pouvez supprimer une clé (et sa valeur correspondante) d'une structure de données cartographiques. Vous fournissez la variable qui contient l'index de la carte (tel que renvoyé par l'action Créer une carte ), puis donnez une "clé" (qui est l'identificateur dans la carte pour une valeur), et l'action supprimera cette clé de la carte. Notez que cette action ne retourne rien, et si vous avez besoin de la valeur de la clé donnée, vous devez utiliser la Obtenir carte valeur l' action avant de le retirer.

Syntaxe de l'action:

Arguments:

Argument Description
Map L'index (stocké dans une variable) de la carte pour supprimer la valeur de
Key La clé à utiliser (réelle ou chaîne)
