Pile pop

Avec cette action, vous pouvez faire sauter une valeur du haut d'une structure de données de pile. Vous fournissez la variable qui contient l'index de la pile (tel que renvoyé par l'action Créer une pile ), puis donnez la variable cible qui doit contenir la variable retournée (qui peut être marquée comme une variable locale temporaire).

Syntaxe de l'action:

Arguments:

Argument Description
Stack L'index (stocké dans une variable) de la pile à utiliser
Target La variable cible contenant la valeur renvoyée

Exemple:

Le code de bloc d'action ci-dessus vérifie si la pile donnée est vide, et si ce n'est pas le cas, il supprime la valeur supérieure et la stocke dans une variable temporaire. Ceci est ensuite utilisé pour définir une alarme, tandis que si la pile est vide, 10 valeurs aléatoires sont poussées sur la pile.
