Si la structure de données existe

Avec cette action, vous pouvez vérifier une variable pour voir si elle indexe une structure de données ou non. Vous devez d'abord choisir le type de structure de données à vérifier (soit une Liste, une Carte, une Grille ou une Pile) et ensuite vous fournissez la variable qui contient l'index de la structure de données (tel que retourné par les actions Créer une liste, Créer une carte, Créer une grille et Créer une pile). L'action retournera true si la structure de données existe, et false si elle n'existe pas. Notez que vous pouvez marquer "not" dans l'action pour vérifier si une variable ne contient pas d'index de structure de données.
