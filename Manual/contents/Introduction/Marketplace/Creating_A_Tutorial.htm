Tutoriel sur la création d'une place de marché

La création d'un tutoriel est similaire à la création d'un package de ressources Marketplace ordinaire, sauf que vous devez maintenant ajouter des fichiers inclus spécifiques ainsi qu'une page initiale de tutoriel (et une page vidéo facultative). L'espace de travail Tutoriel ressemblera à ceci :Comme expliqué ci-dessus pour les ressources générales du Marché, vous devez sélectionner un éditeur, puis donner à votre didacticiel un nom et un ID unique (format URL inversé), et enfin, dans la liste déroulante "type", sélectionner Tutoriel. Vous disposerez alors d'options supplémentaires pour ajouter un fichier HTML de base à partir duquel le didacticiel pourra démarrer, ainsi qu'un fichier vidéo (facultatif). ainsi qu'un fichier vidéo (facultatif) et une URL. Le fichier vidéo doit utiliser le modèle fourni par le marché, tout comme le reste du tutoriel, que vous pouvez obtenir ici, et lorsque l'utilisateur charge le didacticiel, la vidéo sera chargée dans un cadre distinct de l'EDI. Notez que les pages que vous donnez pour la page de démarrage et la page de la vidéo sont des chemins relatifs dans les fichiers inclus, et vous pouvez ouvrir un explorateur pour choisir un explorateur pour choisir autre chose que les pages par défaut avec lesquelles GameMaker Studio 2 remplit les rubriques.

Pour créer votre tutoriel, vous devez ajouter des fichiers inclus dans votre projet de tutoriel, et ceux-ci doivent être ajoutés d'une manière très spécifique pour que l'IDE détecte correctement les fichiers à utiliser pour le tutoriel. utilisés pour le tutoriel. La structure de base des dossiers est présentée dans l'image ci-dessous :

Nous avons donc deux dossiers principaux :

  • TutorialContent: Tous les fichiers de votre tutoriel HTML, le site CSS pour les styliser et les images requises (pas les images requises pour les ressources, mais celles qui seront affichées dans le tutoriel), ainsi que le modèle de vidéo, le cas échéant. dans le tutoriel), ainsi que le modèle de vidéo si nécessaire, vont ici. Vous pouvez également avoir des sous-dossiers si vous souhaitez diviser le didacticiel en chapitres, par exemple, mais vous devez avoir au moins un fichier de base HTML qui sera utilisé comme page de démarrage du didacticiel.
  • TutorialResources: Ce dossier contient deux autres sous-dossiers pour les images et les ressources sonores . Si votre didacticiel exige que l'utilisateur ajoute un sprite (par exemple) et que vous souhaitez l'inclure dans le didacticiel alors vous placerez la ressource image dans le dossier Images. Ces dossiers sont importés dans le projet du didacticiel et lorsque l'utilisateur ouvre l'éditeur de sprites et va ajouter une image, le chemin par défaut de la fenêtre de l'explorateur sera cet emplacement, ce qui permet à l'utilisateur de trouver plus facilement les ressources nécessaires.

Pour que votre tutoriel fonctionne dans l'EDI, vous devez avoir cette structure de dossier ainsi qu'une page de démarrage de base HTML avec un fichier CSS pour le style. Il existe un certain nombre de liens spéciaux qui peuvent être utilisés dans vos projets de didacticiel et qui permettent de passer à certaines parties d'une vidéo ou d'ouvrir le manuel sur une section ou une fonction spécifique. Ces liens sont tous fournis dans les modèles de didacticiel officiels du marché.

IMPORTANT ! Nous vous conseillons vivement de télécharger les modèles de didacticiels sur le marché avant d'essayer de créer vos propres didacticiels. Ils contiennent des exemples de toutes les parties importantes du code HTML ainsi que des CSS et des images pour recréer des boutons et des menus dans le didacticiel, ce qui vous permet de vous concentrer sur le contenu plutôt que sur la façon de l'afficher.

Une fois que vous avez configuré les fichiers inclus et fourni les informations requises, vous pouvez cliquer sur Okay et le tutoriel sera téléchargé sur le Marketplace, ouvrant un espace de travail sur la liste des actifs une fois terminé afin que vous puissiez le modifier et ajouter des icônes, du texte, des captures d'écran, etc... Une fois le didacticiel publié, les utilisateurs pourront le trouver dans la section Didacticiels de la page de démarrage, où il pourra être téléchargé et installé.
