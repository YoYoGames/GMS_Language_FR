Votre bibliothèque Marketplace

La fenêtre Bibliothèque vous montre les packages que vous avez achetés et vous donne la possibilité de les télécharger et / ou de les ajouter à votre projet. Sur la gauche se trouvent les différents filtres de catégorie que vous pouvez utiliser pour voir ce que vous avez dans chacun d'eux, puis au milieu vous avez la liste des actifs de votre bibliothèque. Chacun peut être sélectionné afin que vous puissiez voir un aperçu de ce qu'il fait (à droite) et ils ont tous des icônes pour afficher, télécharger ou installer le package d'actifs dans un projet. Notez que vous avez deux filtres supplémentaires en haut de la fenêtre principale dans lesquels vous pouvez choisir de ne voir que les actifs qui ont été créés spécifiquement pour GameMaker Studio 2, ainsi qu'un filtre pour n'afficher que les actifs que vous avez déjà téléchargés sur votre machine. Vous pouvez également actualiser la liste des actifs affichés à tout moment en cliquant sur le bouton Actualiser.

Pour ajouter un package d'actifs à votre projet actuel, vous devez d'abord le télécharger en cliquant sur le bouton Télécharger et une fois le téléchargement terminé, vous pouvez cliquer sur le bouton Importer pour l'ajouter à votre projet. L'ajout d'un package de cette manière créera une extension dans votre projet actuel et vous demandera de sélectionner parmi les différentes ressources du package d'actifs pour choisir celles que vous souhaitez ajouter (ou vous pouvez les importer toutes). Vous pouvez également sélectionner l'option Importer toutes les ressources dans un nouveau projet, et au lieu d'ajouter les actifs à votre projet actuel, GameMaker Studio 2 vous invitera à donner un nouveau nom de projet et à enregistrer l'actif dans ce projet à la place. Notez que si vous accédez directement à la bibliothèque à partir de la page de démarrage, vous constaterez que la plupart des éléments ne peuvent pas être ajoutés individuellement, et à la place, vous serez obligé d'utiliser le bouton Ajouter tout. Cela est dû au fait que vous n'avez pas de projet GameMaker Studio 2 doit donc créer un nouveau projet auquel vous pourrez ajouter tous les actifs du package (lorsque vous cliquez sur le bouton Importer, vous serez invité à enregistrer le nouveau projet avec le nom de votre choix). Une fois les actifs ajoutés à votre navigateur d'actifs, vous pouvez désormais les utiliser dans votre jeu comme vous le feriez pour les actifs que vous avez créés vous-même. Veuillez noter que les actifs téléchargés à partir de Marketplace sont destinés à un usage individuel et ne peuvent pas être partagés avec des tiers - bien que vous puissiez les sauvegarder et les copier vers d'autres emplacements pour des raisons de sécurité - à moins que l'actif ne dispose d'un CLUF personnalisé indiquant le contraire.

REMARQUE: Les personnes qui créent des packages d'actifs pour la place de Marketplace travaillent très dur pour créer une excellente ressource à utiliser.Veuillez donc envisager d'évaluer les actifs que vous achetez lorsque vous les avez testés ou si vous les utilisez. Cela encourage le développeur et constitue également un guide utile pour les autres utilisateurs quant à la qualité des actifs disponibles. Vous pouvez facilement évaluer vos ressources téléchargées à partir de la fenêtre Bibliothèque.
