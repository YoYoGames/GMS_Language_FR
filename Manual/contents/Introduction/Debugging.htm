Mode débogage

Si vous avez besoin de vérifier les choses plus attentivement, vous pouvez exécuter le jeu en mode débogage en cliquant sur le bouton debug run . Le module de débogage s'ouvre alors en même temps que votre jeu. Ce module vous permet de contrôler le fonctionnement de votre jeu, depuis les performances globales jusqu'aux valeurs des variables individuelles dans les différentes instances. Vous trouverez de plus amples informations sur le module de débogage, ses possibilités et la façon de l'utiliser à la page suivante :

  • Le débogueur

Notez que lorsque vous utilisez le module de débogage, vous pouvez également ajouter des points d'arrêt à votre code ou à votre DnD™. Un point d'arrêt est un point de la boucle de jeu où vous voulez que le module de débogage fasse une pause et vous permette de parcourir le code de jeu réel, ligne par ligne. ligne à la fois. Vous pouvez ajouter des points d'arrêt en utilisant la touche "F9", et ils seront affichés dans la fenêtre de sortie où vous pouvez les activer/désactiver selon vos besoins.
