Achat d'application

Cet événement ne peut être déclenché que par les différentes extensions IAP (In App Purchase ) pour les différentes plates-formes cibles disponibles sur la Marketplace. Cet événement générera toujours une carte DS exclusive à cet événement et stockée dans la variable spéciale async_load. Cette carte DS aura différentes paires clé / valeur en fonction de la fonction d'extension qui l'a déclenchée, mais il y aura toujours un "id "qui peut ensuite être utilisée pour identifier le type d'événement d'achat intégré à l'application. Pour plus de détails sur les valeurs de retour possibles et les fonctions qui génèrent cet événement, veuillez consulter les manuels PDF individuels inclus avec chacune des extensions.
