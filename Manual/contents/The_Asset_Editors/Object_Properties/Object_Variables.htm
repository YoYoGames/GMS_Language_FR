Variables d'objet

Lorsque vous cliquez sur le bouton Variables il va ouvrir la fenêtre objet Variables, que vous pouvez ensuite utiliser pour générer un certain nombre de variables avant l'événement Créer de l'objet:

Dans cette fenêtre, vous pouvez cliquer sur le bouton Ajouter pour ajouter une nouvelle variable à la liste. Vous pouvez ensuite nommer la variable ainsi que définir son type et ajuster ses propriétés:

Le nom de la variable doit commencer par une lettre et ne peut contenir que des lettres, des chiffres et le symbole de soulignement «_» avec une longueur maximale de 64 symboles. Ainsi, les variables valides sont des choses comme fish, foo_bar, num1, et les variables non valides seraient 6fish, foo bar, ou *num. Une fois que vous avez donné le nom de la variable, vous devez définir son type, qui peut être l'un des suivants: Il est à noter que cette fonctionnalité est particulièrement utile lorsque vous travaillez avec des instances parent / enfant - puisque vous pouvez avoir un objet parent avec un ensemble de variables définies et ensuite simplement les modifier dans l'objet enfant (voir la section Variables d'objet et objets enfants ci-dessous pour plus de détails) - et lorsque vous ajoutez des instances dans l' éditeur de salle (voir la section Calques > Calque d'instance ), car vous pouvez également modifier ces variables pour des instances individuelles qui ont été placées dans la salle.

Les différents types de variables que vous pouvez créer sont répertoriés ci-dessous:

  • Réel: un nombre réel est un nombre qui n'est pas un entier entier et qui peut être positif ou négatif. Alors, 124.5, 45639.566546456, 0.9, -45.5 etc... sont tous des exemples de nombres réels. Tous les nombres réels sont stockés sous forme de valeurs à virgule flottante 64 bits (doubles). Le réel (et entier ) les types de données ont une option pour définir une plage de valeurs accessible en cliquant sur Options bouton: Lorsque vous cochez cela, vous pouvez alors entrer une valeur de début et une valeur de fin et au lieu d'avoir une valeur fixe affichée pour la variable dans la fenêtre Variables d'objet, vous aurez un curseur qui est fixé à ces valeurs:
  • Entier: un entier est un nombre entier et peut être positif ou négatif, par exemple 30004, 19, 0, -300. Notez que vous pouvez définir une plage de valeurs pour l'entier de sortie, comme expliqué ci-dessus pour les nombres réels.
  • Chaîne: une chaîne est tout ce qui a été placé entre guillemets doubles, par exemple "poisson", "Hello World" ou "12345".
  • Boolean: un booléen est une valeur qui est vraie ou fausse. Dans la fenêtre Variables de l'éditeur d'objets, cela s'affiche simplement sous la forme d'une case dans laquelle vous cochez la case true et décochez la case false.
  • Expression: une expression est une phrase mathématique qui peut contenir des nombres ordinaires, des variables, des chaînes ou des GML ainsi qu'un ou plusieurs opérateurs. Par exemple sqrt(85 * 6) + 5.5 est une expression.
  • Ressource: une ressource est simplement l'une des ressources que vous avez définies dans le navigateur d'actifs. Lorsque vous sélectionnez ce type de ressource, vous pouvez ensuite cliquer sur Open Asset Explorer pour ouvrir l'explorateur d'actifs et sélectionner la ressource dont vous avez besoin.
  • Liste: La sélection d'un type d'entrée de liste signifie que vous pouvez ensuite créer une sélection de valeurs (celles-ci peuvent être des chaînes, des réels, des expressions etc...) dont vous pouvez ensuite en choisir une ou plusieurs pour la variable à renvoyer. Pour définir les éléments qui vont dans la liste, vous devez d'abord les ajouter dans les Options (cliquez sur le bouton):
    Vous pourrez choisir l'élément comme valeur par défaut pour la variable dans la fenêtre principale à partir du menu déroulant:
    Notez que si vous avez coché la sélection multiple dans les options de liste, la variable deviendra un tableau de toutes les options sélectionnées:
  • Couleur: Le type de couleur vous permet de définir une valeur de couleur à stocker dans la variable. Vous pouvez entrer un nombre réel (de 0 à 16777216, cela aura une valeur alpha de 255), ou une valeur hexadécimale (au format $ RRGGBBAA), ou vous pouvez double-cliquer sur l'échantillon de couleur pour ouvrir le sélecteur de couleurs et y définir la couleur.

Variables d'objets et objets enfants

Une caractéristique importante des variables d'objet est qu'elles sont héritées par tous les objets enfants que vous avez dans le navigateur d'actifs (voir la section sur les objets parents pour plus d'informations), ce qui signifie que vous pouvez alors choisir de remplacer ou de modifier tout ou partie des eux si vous le souhaitez.

Lorsque vous créez un objet enfant d'une autre ressource objet pour laquelle des variables sont définies, celles-ci s'affichent dans la fenêtre Variables d'objet comme suit:

Vous pouvez voir dans l'image du haut que l'objet Parent a quatre variables d'objet et dans l'image du bas, celles-ci sont également affichées, uniquement "grisées" et ont le parent hérité icône pour indiquer qu'ils ont été hérités. Ces variables héritées peuvent ensuite être modifiées si vous cliquez sur la variable Remplacer bouton, de sorte que vous pouvez ensuite ajuster le curseur de plage ou les valeurs, ou sélectionner différents éléments dans les listes, etc... Notez que lorsque vous modifiez une variable parente, vous ne pouvez modifier que la valeur définie mais pas le nom, ni la variable options. Notez également que vous pouvez également ajouter de nouvelles variables d'objet aux objets enfants (et dans l'exemple d'image ci-dessus, vous pouvez voir que "Type de caractère" est une nouvelle variable d'objet uniquement pour l'objet enfant).

Si vous avez modifié une variable objet héritée, vous pouvez cliquer sur le bouton Supprimer pour supprimer la modification, mais pas la variable. La variable sera toujours héritée du parent, mais utilisera à nouveau la valeur par défaut du parent. Si vous devez supprimer complètement la variable, vous devez le faire à partir de l'objet parent.
