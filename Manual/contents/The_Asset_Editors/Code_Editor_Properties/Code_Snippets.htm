Extraits de code

L'utilisation d'extraits de code est un outil très pratique dont vous disposez lors de l'édition de votre code. Lorsque vous utilisez l'éditeur de script, vous pouvez appuyer sur F4 pour ouvrir la fenêtre contextuelle de l'extrait de code, qui vous permet de sélectionner l'une des méthodes de code couramment utilisées dans une liste:

Dans cette fenêtre, vous pouvez sélectionner avec la souris l'extrait de code à utiliser, ou vous pouvez appuyer sur le raccourci clavier associé répertorié à droite. Cela ajoutera l'extrait de code à votre code pour le modifier.

Vous pouvez également définir vos propres extraits de code si vous le souhaitez. Avant de faire cela, vous devrez créer un fichier appelé "snippets.txt "dans l'un des répertoires suivants:

  • Windows: %ProgramData%\GameMaker Studio2 \ Utilisateur \
  • macOS: /Users/Shared/GameMaker Studio2 / Utilisateur /

Ce dossier n'est pas modifié lors d'une mise à jour de GameMaker Studio 2, donc votre fichier modifié restera intact (mais toutes les modifications apportées au fichier d'extraits de code de base dans le répertoire d'installation seront annulées).

Une fois le fichier créé dans le répertoire User, vous pouvez le modifier avec n'importe quel éditeur de texte en suivant ces règles:

  • Chaque extrait de code doit être sur une ligne distincte (il ne doit pas y avoir de lignes vides)
  • Chaque extrait de code doit commencer par le raccourci clavier à utiliser suivi de "- "puis le nom de l'extrait (qui est ce qui est affiché dans le menu) suivi de deux points": ", par exemple:

I - Créer une instance:

Après les deux points, vous ajoutez l'extrait de code

I - Création d'instance: instance_create_layer (x, y, | layer |, object);

Le code que vous ajoutez doit également suivre un format spécifique où:

  • Tout le code est sur une seule ligne
  • Tu utilises "$ "pour une nouvelle ligne
  • Vous placez le texte qui doit être sélectionné en premier entre "| "symboles (le curseur sélectionnera ce texte pour le modifier lorsque l'extrait de code sera ajouté)

Consultez les exemples qui se trouvent déjà dans le fichier d'extraits de code de base pour voir comment il est configuré en suivant les règles ci-dessus. Vous pouvez trouver le fichier de base dans le répertoire d'installation de GameMaker Studio 2

  • \GameMaker Studio 2 \ TextEditor \ snippets.txt
