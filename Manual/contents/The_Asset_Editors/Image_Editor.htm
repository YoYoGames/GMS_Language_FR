L'éditeur d'images

L' éditeur I mage est l'endroit où vous pouvez créer vos propres sprites ou modifier ceux que vous avez importés ou créés précédemment. Si vous avez utilisé d'autres outils d'image, cela devrait vous être familier, mais il y a quelques points à noter lors de l'utilisation de l' GameMaker Studio 2 qui peuvent ne pas être immédiatement apparents:

  • Lorsque vous créez plusieurs cadres, vous pouvez dessiner dessus pendant qu'ils s'animent! Donc, si vous appuyez sur le bouton de lecture dans la barre de contrôle du cadre, puis sélectionnez un pinceau et commencez à dessiner dans la fenêtre de l'éditeur, vous verrez que les cadres s'animeront dans l'éditeur et ce que vous dessinez sera ajouté à l'image correspondante à ce moment-là. Vous pouvez expérimenter différentes fréquences d'images jusqu'à ce que vous obteniez quelque chose que vous aimez, mais une fois que vous aurez compris comment cela fonctionne, vous créerez des sprites et des effets dynamiques en un rien de temps.
  • Vous pouvez extraire l'onglet Editeur d'image de la fenêtre principale de l'espace de travail dans sa propre fenêtre. Cela peut être pratique lors de l'utilisation de plusieurs moniteurs, par exemple.
  • Lorsque vous utilisez les fonctions Copier pour, par exemple, copier une sélection d'une image, la sélection copiée est ajoutée à votre liste de pinceaux. Cela signifie que vous pouvez sélectionner une partie d'une image, puis l'utiliser dans le reste des outils de peinture pour dessiner des lignes, des polygones ou simplement peindre librement. Le pinceau peut également être mis à l'échelle et mélangé avec une couleur. Notez que la fonctionnalité Coller n'est pas destinée à être utilisée en interne dans l'éditeur car tout ce que vous copiez est ajouté à la liste des pinceaux et peut donc être sélectionné pour "coller" de cette façon. Au lieu de cela, coller est simplement pour vous de capturer quelque chose à partir d'un programme externe et de le coller dans l'éditeur d'images.
  • Vous pouvez avoir plusieurs couches pour chaque sprite grâce à l' éditeur de couches pratique. Ces couches peuvent être ordonnées et même placées dans des dossiers pour faciliter la création d'images plus complexes.
  • Vous pouvez passer rapidement d'une sous-image (cadres) à une autre dans l'éditeur en utilisant le "1 " et "2 "touches du clavier.

Ce ne sont là que quelques-unes des fonctionnalités les plus avancées offertes par l'éditeur d'images. Pour découvrir la gamme complète des fonctionnalités, veuillez lire les éléments ci-dessous détaillant les fonctions de chaque partie de l'éditeur:

Vue du cadre

Cette partie de la fenêtre montre les cadres dans l'ordre dans lequel ils seront animés. Vous pouvez cliquer sur le bouton vers la gauche pour ajouter d'autres images - ou utilisez le menu Image - et vous pouvez également cliquer et faire glisser l'une des images pour changer leur position le long de la chronologie. En haut à gauche de la vue cadre, vous pouvez activer la pelure d'oignon, ce qui signifie que dans une seule image de n'importe quelle animation de sprite, vous pouvez choisir de voir l'une des images précédentes ou suivantes comme un calque semi-transparent sous l'image que vous êtes en train de modifier. La grande chose à ce sujet est que chaque image est dessinée légèrement plus claire que la précédente afin que vous puissiez voir comment l'animation a progressé d'une image à l'autre tout en dessinant la suivante.

L'activation de cette option affichera des contrôles supplémentaires sur les cadres sur lesquels vous pouvez cliquer et faites glisser pour définir le nombre d'images précédentes à afficher ainsi que le nombre d'images suivantes.

Vous pouvez définir certaines propriétés pour la pelure d'oignon à partir du menu Vue contextuelle, qui est expliqué plus loin dans cette page.

Sous le bouton Pelure d'oignon, vous avez le bouton Contrôle de boucle utilisé pour définir si le sprite boucle (retourne à l'image 1 lorsque la dernière image est atteinte) ou ping-pongs (retourne à travers les images lorsque la dernière image est atteinte). Le bouton Play en bas est pour démarrer / arrêter l'aperçu de l'animation.

Vous pouvez cliquer sur n'importe quel cadre pour le sélectionner pour l'édition, et un cadre sélectionné aura un bouton Supprimer dans le coin supérieur sur lequel vous pouvez cliquer pour supprimer le cadre du sprite. Vous pouvez également cliquer sur / + sur plusieurs cadres pour les sélectionner ensemble, auquel cas le fait de cliquer sur le bouton Supprimer supprimera tous les cadres sélectionnés. Enfin, vous pouvez utiliser les raccourcis clavier standard pour Couper, Copier et Coller pour modifier les images sélectionnées: / + "X ", / + "C ", et / + "V ".

Informations sur le cadre

Cette barre contient les informations sur le cadre. Ici, vous pouvez définir la vitesse à laquelle les images d'animation doivent être lues à l'aide de la zone de saisie sur la gauche. La vitesse d'image peut être basée sur « Images par seconde » ou « Images par image de jeu ». Le nombre total d'images disponibles est affiché à droite avec l'image actuelle qui a été sélectionnée, et vous pouvez réduire cette barre (la masquer) en cliquant sur le icône à gauche. Vous pouvez définir cette option pour qu'elle démarre toujours ouverte ou toujours fermée à partir des préférences de l'éditeur d'images.

Notez que la valeur utilisée ici pour la vitesse d'animation définira également la façon dont le sprite est animé lorsqu'il est utilisé dans un jeu, soit en tant qu'actif de sprite sur un calque ou en tant que sprite affecté à une instance, bien que vous puissiez le modifier au runtime utilisant les fonctions et variables appropriées.

Contrôles du canevas

Les commandes du canevas traitent de divers aspects de l'affichage du canevas principal et de son contenu. Les options sont:

  • Toggle Grille de canevas: Cela activera / désactivera la grille du canevas. Il s'agit d'une grille que GameMaker Studio 2 dessine sur le canevas principal pour la diviser en sections, et sa taille par défaut est de 1 px. Cependant, si vous cliquez sur l'icône du menu Grille vous ouvrirez les options de la grille:
    Ces options vous permettent de définir la couleur et l'alpha de la grille, ainsi que les valeurs de cellule de la grille le long des axes X et Y. Vous avez également une option pour activer ou désactiver l'accrochage à la grille ici (désactivé par défaut).
  • Commandes de zoom du canevas: Ces boutons contrôlent le niveau de zoom actuel du canevas. Vous pouvez effectuer un zoom avant ou arrière et en cliquant sur le Le bouton réinitialise le canevas à 1: 1 avec l'image en cours de modification. Vous pouvez également cliquer sur le bouton Ajuster la fenêtre pour que le canevas de l'image entière rentre dans l'espace de travail de l'éditeur actuel (cela effectuera un zoom avant / arrière selon les besoins pour l'adapter). Notez que vous pouvez également effectuer un zoom avant et arrière à l'aide de / et la molette de la souris, et en appuyant sur / + définit le canevas de la pièce sur 1: 1 avec l'image.
  • Toile divisée: En cliquant dessus, vous alternerez entre le fractionnement horizontal du canevas, le gardant verticalement comme un seul canevas. En vue de canevas fractionnée, chaque canevas aura son propre ensemble de commandes canevas afin que vous puissiez effectuer un zoom avant et arrière, par exemple, et vous pouvez modifier l'un ou l'autre canevas avec les changements reflétés dans l'autre canevas. Pour changer la bascule de la vue du canevas, vous pouvez cliquer sur les options de fractionnement et sélectionnez celui que vous souhaitez utiliser ou cliquez simplement sur le bouton Diviser le canevas jusqu'à ce qu'il passe à celui que vous souhaitez.

Dessiner une toile

Il s'agit de la fenêtre d'édition principale de vos images. Ici, vous pouvez appliquer l'un des outils de droite pour peindre sur le calque actuellement sélectionné. Vous pouvez peindre soit avec la gauche ou le droit bouton de la souris (et attribuez une couleur à chacun à partir du sélecteur de couleurs) et utilisez l'un des pinceaux définis en haut à droite de l'éditeur. Vous pouvez aussi utiliser / et la molette de la souris pour effectuer un zoom avant ou arrière, et appuyez sur / + définira le canevas sur 1: 1 avec l'image (ou vous pouvez utiliser les commandes du canevas comme expliqué ci-dessus). Notez également que vous pouvez vous déplacer dans le canevas à l'aide du bouton central de la souris ou +.

Valeurs du canevas

Cette petite barre en bas montre les différentes valeurs pour la position de la souris et la largeur et la hauteur des outils de sélection dans l'éditeur.

Brosses

Ici, vous pouvez sélectionner le pinceau avec lequel peindre. Par défaut, vous avez une sélection de pinceaux carrés et ronds de différentes tailles à utiliser, mais vous pouvez facilement créer votre propre pinceau en peignant sur le calque d'image, puis en sélectionnant la partie que vous souhaitez utiliser pour le nouveau pinceau et en utilisant / + "C "pour le copier. La section copiée sera ensuite ajoutée à vos pinceaux et vous pourrez peindre avec elle comme vous le feriez avec n'importe quel autre pinceau, et cela inclut l'utilisation des différents outils de peinture disponibles comme dessiner une ligne.

Notez que la couleur de fusion par défaut pour le nouveau pinceau sera le blanc, mais si vous changez la couleur avec le sélecteur de couleur et peignez à l'aide du pinceau, elle sera mélangée avec cette nouvelle couleur. Si vous souhaitez supprimer un pinceau de la liste, cliquez simplement dessus et appuyez sur le bouton clé et il sera supprimé.

REMARQUE: les pinceaux ne sont pas conservés entre les sessions, la fermeture de votre projet supprimera tous les pinceaux que vous avez créés.

Outils

Ci-dessous, nous listons les outils qui sont disponibles par défaut avec GameMaker Studio 2 - notez que tous les outils peuvent être utilisés avec la gauche et à droite bouton de la souris, et la plupart d'entre eux sont associés à des raccourcis clavier (les plus importants sont répertoriés ici avec l'outil, mais pour une liste complète, veuillez consulter la page sur les raccourcis ). La plupart des outils auront également des options supplémentaires qui seront visibles en haut de l'éditeur d'image, comme indiqué ci-dessous:

Crayon Ceci est l'outil crayon. Il utilise le pinceau sélectionné pour peindre dans la couleur sélectionnée pour le bouton de la souris, et vous pouvez définir la taille à partir de la barre supérieure de l'éditeur d'image ainsi qu'activer / désactiver le lissage (interpolation). Notez que vous pouvez également redimensionner et mettre à l'échelle n'importe quel pinceau en maintenant + "Z "puis en cliquant et en faisant glisser la souris pour modifier les proportions du pinceau, ou bien sélectionnez l'un des pinceaux prédéfinis en haut à droite.
La gomme Ceci est l'outil de gomme. Il utilise le pinceau sélectionné pour effacer une zone en cliquant sur l'un des boutons de la souris. La sélection de cette option ajoutera également des outils supplémentaires à la barre supérieure de l'éditeur pour vous permettre de mettre à l'échelle le pinceau ou d'activer / désactiver le lissage ( interpolation linéaire ).
Peinture / Remplissage Il s'agit de l'outil pot de peinture (ou remplissage). Cela remplira une zone de l'image en fonction du paramètre de tolérance alpha choisi, que vous pouvez modifier à l'aide du curseur affiché dans la barre supérieure de l'éditeur.
Supprimer la couleur Il s'agit de l'outil de suppression de couleur. Lorsque vous le sélectionnez, vous pouvez ensuite cliquer sur ou sur n'importe quel pixel de l'image et tous les pixels de la même couleur seront supprimés (ils sont techniquement remplacés par la couleur noire à un alpha de 0). La barre supérieure a un curseur de tolérance où vous pouvez définir la tolérance de couleur des pixels - plus cette valeur est basse, plus la couleur doit correspondre exactement à celle sur laquelle vous avez cliqué, tandis qu'une valeur plus élevée l'étendra pour inclure les couleurs similaires à la on a cliqué. Si vous avez précédemment effectué une sélection à l'aide de l'un des outils de sélection, seul un clic sur les pixels de la sélection aura un effet, et seuls les pixels de la sélection seront supprimés.
Remplacer la couleur Il s'agit de l'outil de remplacement de couleur. Lorsque vous le sélectionnez, vous pouvez ensuite cliquer sur ou sur tout pixel de l'image et tous les pixels de même couleur seront remplacés par la couleur attribuée au bouton de la souris utilisé. La barre supérieure a un curseur de tolérance où vous pouvez définir la tolérance de couleur des pixels - plus cette valeur est basse, plus la couleur doit correspondre exactement à celle sur laquelle vous avez cliqué, tandis qu'une valeur plus élevée l'étendra pour inclure les couleurs similaires à la on a cliqué. Si vous avez précédemment effectué une sélection à l'aide de l'un des outils de sélection, alors seul un clic sur les pixels de la sélection aura un effet, et seuls les pixels de la sélection seront affectés par le changement de couleur.
Ligne Cet outil trace une ligne entre deux points. Cliquez une fois avec l'un ou l'autre des boutons pour créer le premier point, puis cliquez à nouveau ailleurs pour créer un autre point et les joindre. La ligne sera dessinée à l'aide du pinceau sélectionné pour créer le contour, et vous pourrez éditer certaines des propriétés du pinceau à l'aide de la barre supérieure de l'éditeur (taille, interpolation linéaire, etc...).
Rectangle Cet outil dessinera un rectangle plein ou un rectangle avec contour. Si vous cliquez en haut à gauche de l'icône, elle sera mise en évidence, tout en cliquant en bas à droite sera remplie et si vous avez besoin des deux, vous pouvez sélectionner cette option dans la barre supérieure de l'éditeur d'image (ou utiliser + pour sélectionner l'un puis l'autre sur l'icône de la boîte à outils), ainsi que définir la taille et le lissage du pinceau utilisé. Le rectangle sera ensuite dessiné à l'aide du pinceau sélectionné. Notez que lorsque vous dessinez, vous pouvez maintenir et cliquez pour dessiner à partir du centre de la position de la souris.
Ellipse Cet outil dessinera une ellipse pleine ou délimitée. Si vous cliquez en haut à gauche de l'icône, elle sera mise en évidence, tout en cliquant en bas à droite sera remplie et si vous avez besoin des deux, vous pouvez sélectionner cette option dans la barre supérieure de l'éditeur d'image (ou utiliser + pour sélectionner l'un puis l'autre sur l'icône de la boîte à outils), ainsi que définir la taille et le lissage du pinceau utilisé. L'ellipse sera dessinée à l'aide du pinceau sélectionné pour créer le contour. Notez que lorsque vous dessinez, vous pouvez maintenir et cliquez pour dessiner à partir du centre de la position de la souris.
Polygone Cet outil dessinera un polygone plein ou un polygone avec contour. Si vous cliquez en haut à gauche de l'icône, elle sera mise en évidence, tout en cliquant en bas à droite sera remplie et si vous avez besoin des deux, vous pouvez sélectionner cette option dans la barre supérieure de l'éditeur d'image (ou utiliser + pour sélectionner l'un puis l'autre sur l'icône de la boîte à outils), ainsi que définir la taille et le lissage du pinceau utilisé. Vous cliquez ou pour ajouter des points au polygone à l'aide du pinceau sélectionné et lorsque vous êtes satisfait des résultats, vous devez sélectionner un outil différent pour valider les modifications et générer l'image. Le polygone sera dessiné à l'aide du pinceau sélectionné. Vous pouvez appuyer sur à tout moment pour annuler la définition du polygone.
Spline C'est l'outil spline. Il utilise le pinceau sélectionné pour créer une spline. Vous cliquez et maintenez l'un des boutons de la souris pour définir un point, puis faites glisser la souris pour générer la spline, relâchez la souris et répétez pour ajouter d'autres points et courbes. La spline sera dessinée à l'aide du pinceau sélectionné, qui peut être édité à l'aide de la barre supérieure de l'éditeur pour définir la taille et le lissage. La barre supérieure a également un paramètre pour les itérations qui définira la fluidité de la spline résultante (les valeurs inférieures seront plus grossières, plus lisses et la valeur par défaut est 10).
Texte
Ceci est l'outil de saisie de texte. Il crée un nouveau calque de texte dans lequel vous pouvez taper tout ce dont vous avez besoin. Vous pouvez définir la police à utiliser ainsi que la mise en forme à partir de la barre supérieure de l'éditeur d'image.
Compte-gouttes oculaire L'outil compte-gouttes vous permet d'échantillonner une couleur de l'image en cours de modification.
Sélection de rectangle Avec cet outil, vous pouvez générer une sélection rectangulaire. Vous pouvez annuler une sélection dans le menu Image ou en appuyant sur.
Sélection de peinture Avec cet outil, vous pouvez "peindre" une zone sélectionnée en utilisant l'un des pinceaux, y compris les pinceaux personnalisés. Les mêmes raccourcis clavier s'appliquent que pour l'outil Crayon. Vous pouvez annuler une sélection dans le menu Image ou en appuyant sur
Baguette magique L'outil de sélection de la baguette magique sélectionnera des zones de la même couleur en fonction de la valeur de tolérance définie dans la barre supérieure de l'éditeur. Si vous cochez "Couleur uniquement", la valeur alpha ne sera pas prise en compte. Vous pouvez annuler une sélection dans le menu Image ou en appuyant sur.
Tourner C'est l'outil de rotation. Si vous avez sélectionné une zone à l'aide de l'un des outils de sélection, cliquer dessus vous permettra de faire pivoter la zone sélectionnée d'une valeur arbitraire de 0 ° à 360 °. Si aucune zone de l'image n'est sélectionnée, cela vous permettra de faire pivoter le pinceau actuel. Pour faire pivoter, maintenez simplement le bouton gauche de la souris enfoncé et faites glisser vers la gauche ou la droite ou définissez la valeur directement dans la barre supérieure. Pour accepter les modifications, vous pouvez appuyer sur ou sélectionnez un autre outil. Notez que l'acceptation de la modification ne changera pas le matériau source, mais créera plutôt une nouvelle ressource Pinceau que vous utiliserez avec l'image sélectionnée ou un pinceau pivoté de la quantité que vous avez définie. Notez également que le raccourci clavier "Z "peut être utilisé pour faire pivoter n'importe quelle sélection ou pinceau de 90 ° et que cette option ne crée pas de nouveau pinceau et n'affectera que la façon dont le pinceau actuel est dessiné.
Miroir C'est l'outil miroir. La sélection de cette option reflétera le pinceau actuel le long de l'axe vertical. Cette option ne crée pas de nouveau pinceau et n'affectera que la façon dont le pinceau actuel est dessiné.
Retourner Ceci est l'outil de retournement. La sélection de cette option reflétera le pinceau actuel le long de l'axe horizontal. Cette option ne crée pas de nouveau pinceau et n'affectera que la façon dont le pinceau actuel est dessiné.
Poêle C'est l'outil de panoramique. Lorsque vous sélectionnez ceci, vous pouvez ensuite cliquer sur et faites glisser la souris pour déplacer le contenu du calque dans le calque. Notez que tous les pixels en dehors du canevas de dessin ne seront pas visibles et l'image sera rognée à la taille du canevas de dessin lorsque vous changerez d'outil à nouveau. Notez également que si vous avez effectué une sélection, les pixels sélectionnés seront déplacés plutôt que le contenu du calque.

Éditeur de calques

Vous pouvez créer des calques dans l'éditeur d'image pour différentes parties de votre image, et ces calques peuvent être manipulés et dessinés à l'aide des différents pinceaux et outils. Pour créer un nouveau calque, cliquez simplement sur le icône, et pour le supprimer à nouveau, icône. Les calques peuvent également être classés en cliquant sur sur un calque, puis en faisant glisser le calque vers le haut ou le bas jusqu'à la position souhaitée, et vous pouvez organiser davantage les calques en créant des dossiers à l'aide du puis en y faisant glisser des calques. Si vous double-cliquez sur un calque (ou utilisez et sélectionnez Modifier les propriétés de la couche ), vous aurez accès à la fenêtre Propriétés de la couche:

Ici, vous pouvez nommer le calque ainsi que définir le mode de fusion (la façon dont il doit être "mélangé" avec les calques en dessous), en choisissant parmi les mélanges normaux, additifs, soustractifs ou multipliés. Enfin, vous pouvez définir l'opacité (transparence) du calque avec un curseur, mais notez que cela est cumulatif avec la valeur alpha réelle de pixels individuels - par exemple, si vous avez des pixels opaques à 50% et définissez l'opacité du calque sur 50 %, les pixels finaux seront dessinés avec une opacité de 25%.

Image

Le menu Image a les options suivantes (dont certaines peuvent ou non être disponibles selon le focus de l'éditeur actuel):

Lorsque vous coupez ou copiez dans le presse-papiers à partir d'une image dans l'éditeur d'images, cela est ajouté aux pinceaux en haut à droite de l'éditeur, et vous devez sélectionner un pinceau à dessiner avec la sélection coupée / copiée plutôt que d'essayer de le coller dans l'image. Notez également qu'un certain nombre de programmes auront des problèmes pour coller les transparents du presse-papiers dans l' GameMaker Studio 2 raison du format utilisé par le presse-papiers.

  • Convertir en cadres - Cette option convertit une seule image en un certain nombre d'images séparées. Vous devez d'abord avoir créé une seule image avec toutes les parties de l'animation (ou importé une seule image de bande) comme celle illustrée ci-dessous:
    Comme il s'agit d'une seule image et que nous voulons la convertir en cadres individuels, nous pouvons utiliser cette option qui ouvrira alors la fenêtre suivante:
    Ici, vous pouvez définir comment l'image unique doit être divisée pour créer les cadres, en définissant le nombre de cadres, la largeur et la hauteur des cadres et tout y compris les valeurs de décalage.
  • Importer une image de bande - Cette option ouvrira un explorateur de fichiers et vous invitera à rechercher un fichier d'image à bande unique à utiliser pour créer des images à partir de. Une fois l'image sélectionnée, la fenêtre Importer la bande s'ouvre, exactement comme pour l' option Convertir en images ci-dessus.
  • Exporter au format PNG - La sélection de cette option exportera le sprite en un seul PNG image. Si le sprite a plus d'une seule image, alors un sprite "strip" sera créé avec toutes les images, l'une après l'autre, et le nom du fichier de sauvegarde sera automatiquement ajouté avec "_stripX ", où "X "est le nombre d'images exportées. Par exemple, une image-objet de cinq images sera exportée en tant que"sprite_name_strip5.png ".
  • Redimensionner toutes les images - Lorsque vous sélectionnez cette option, la fenêtre suivante s'ouvre dans laquelle vous pouvez définir la manière dont vous souhaitez redimensionner toutes les images d'animation qui composent l'image-objet:
    Vous pouvez choisir de mettre l'image à l' échelle ou de redimensionner le canevas. Si vous choisissez de mettre l'image à l'échelle, il vous suffit de sélectionner la nouvelle largeur et / ou la nouvelle hauteur pour chaque image, si vous souhaitez mettre à l'échelle en utilisant le même rapport hauteur / largeur, et également si vous souhaitez mettre à l'échelle en pixels ou en pourcentage. Notez que lors de la mise à l'échelle, vous pouvez activer ou désactiver l'interpolation linéaire, où son activation entraînera un flou ou un "maculage" des pixels lorsque l'image est mise à l'échelle (en général, pour l'art à haute résolution, vous voulez cette option, mais pour le pixel art, vous veux que cela soit désactivé ).

    L'option de redimensionnement du canevas ne modifiera pas les pixels de l'image mais augmentera ou diminuera la taille du canevas sur lequel les images sont dessinées. Vous pouvez définir l'emplacement de «l'ancre» pour le redimensionnement à l'aide des icônes fléchées, puis définir la nouvelle taille (en pixels ou en pourcentage) et définir s'il faut conserver le rapport hauteur / largeur des images d'origine. Notez que rendre le cadre plus petit que les images originales du sprite rognera les pixels.
  • Recadrer tous les cadres à la sélection - Cette option n'est disponible que si une zone de l'image est sélectionnée dans le canevas de dessin principal. Lorsque vous choisissez ensuite cette option, l'image en cours de modification (et toutes les autres images de l'animation) seront rognées jusqu'aux limites de la sélection.
  • Découper automatiquement toutes les images - Cette option vous permet de découper tous les pixels à zéro alpha autour des bords d'un sprite. Cela prendra en compte tous les cadres d'image dans le sprite, de sorte que chaque côté sera rogné sur le bord de l'image qui a le pixel alpha différent de zéro le plus éloigné.
  • Inverser les images - Sélectionnez cette option pour inverser simplement l'ordre des images pour l'image-objet.
  • Miroir - Cette option vous permet de refléter soit l'image actuelle, toutes les images qui composent l'image-objet le long de l'axe vertical.
  • Retourner - Cette option vous permet de retourner soit l'image actuelle, toutes les images qui composent l'image-objet le long de l'axe horizontal.
  • Faire pivoter toutes les images (90 ° dans le sens des aiguilles d'une montre) - Cette option fait pivoter toutes les images de l'animation de 90 ° dans le sens des aiguilles d'une montre.
  • Faire pivoter toutes les images (90 ° dans le sens inverse des aiguilles d'une montre) - Cette option fait pivoter toutes les images de l'animation de 90 ° dans le sens inverse des aiguilles d'une montre.

Vue

Le menu Affichage propose les options suivantes:

  • 1: 1 - Mettez l'image à l'échelle 1: 1 avec la résolution (c'est-à-dire: taille réelle) et centrez-la dans la fenêtre d'édition d'image.
  • Ajuster l'écran - Mettez l'image à l'échelle pour remplir la fenêtre d'édition d'image.
  • Afficher la grille - Bascule la grille de pixels. C'est le même que le bouton de grille trouvé dans les contrôles de canevas (expliqué plus loin dans cette page).
  • Paramètres de peau d'oignon - Modifiez les paramètres de peau d'oignon. Cela ouvrira la fenêtre suivante où vous pouvez définir les cadres affichés par le bouton de peau d'oignon, ainsi que l'opacité et la couleur des cadres avec peau.

Effets

Le menu Effets est l'endroit où vous pouvez ajouter vos propres plugins d'effets personnalisés, et a les effets suivants par défaut:

  • Flou - Appliquez un effet de flou à l'image. Vous pouvez définir l'intensité ainsi que le type de flou à utiliser (boîte ou gaussien) et choisir de ne flouter que le long de l'axe horizontal ou vertical, ou le long des deux. Vous pouvez ensuite choisir de l'appliquer uniquement au calque actuel, aux calques visibles ou à tous les calques.
  • Niveaux de gris - Désaturer l'image d'une quantité déterminée. Vous pouvez choisir de l'appliquer uniquement au calque actuel, aux calques visibles ou à tous les calques.
