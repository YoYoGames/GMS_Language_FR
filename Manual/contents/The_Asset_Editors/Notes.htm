L'éditeur de notes

L' éditeur de notes ouvrira une fenêtre de code / script dans laquelle vous pourrez conserver des notes et des extraits de code ou toute autre information que vous jugez pertinente pour le projet. Vous pouvez créer plusieurs notes et les nommer à partir du bouton droit de la souris dans le navigateur de ressources. Les notes peuvent également être regroupées et réorganisées comme n'importe quel autre élément.

Si vous avez importé un projet créé à l'aide d'une version précédente de GameMaker Studio, vous trouverez également un rapport de compatibilité inclus sous forme de note dans cette section.
