Signets

Une autre méthode de navigation dans l'espace de travail consiste à utiliser les signets. Vous pouvez ajouter jusqu'à 9 «signets» aux fenêtres IDE ou aux sections de code / DnD™ dans des scripts ou des événements d'objet, puis vous pouvez rapidement passer de l'un à l'autre en utilisant la combinaison de touches appropriée.

Pour ajouter un signet, sélectionnez simplement la fenêtre ou la ligne de code que vous souhaitez étiqueter puis + + NumberKey[1 - 9] sur Windows ou + +NumberKey[1 - 9] sur Mac et cela définira le signet, étiquetant l'élément dans l'EDI comme suit:

Dans l'image ci-dessus, nous avons attribué des signets numérotés 1, 2 et 4 à différentes parties de l'EDI, et nous pouvons maintenant quitter ces fenêtres ou même les fermer complètement, car le signet nous permettra d'y revenir directement. à tout moment plus tard. Cela se fait en appuyant sur / + NumberKey[1 - 9], et si l'élément marqué a été fermé, il sera ouvert, sinon l'espace de travail affichera le signet sélectionné.

IMPORTANT! Les signets ne sont pas enregistrés avec le projet ou lorsque l'EDI est fermé. Ils ne sont valides que pour la session dans laquelle ils ont été créés.

Si vous souhaitez supprimer un signet, il vous suffit de sélectionner la fenêtre ou la ligne de code du signet, puis de répéter le + / + NumberKey[1 - 9] que vous avez utilisé pour le créer.
