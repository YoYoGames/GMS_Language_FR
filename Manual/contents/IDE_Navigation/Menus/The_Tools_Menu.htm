Le menu Outils

Ce menu vous permet d'ouvrir les différents outils autonomes disponibles dans GameMaker Studio 2, avec les options suivantes:

  • Groupes de textures - Affichez la fenêtre Groupe de textures.
  • Groupes audio - Affichez la fenêtre Groupe audio.
  • Sound Mixer - Affichez la fenêtre Sound Mixer.
  • Générer des images de projet - Ouvre la fenêtre Générer des images de projet.
  • Créer un package local - Ouvre la fenêtre Asset Packager, afin que vous puissiez créer et enregistrer des packages d'actifs locaux pour une utilisation future.
  • Importer un package local - Ouvre une fenêtre d'explorateur de fichiers dans laquelle vous pouvez ensuite sélectionner un package d'actifs locaux et l'importer, soit en l'ajoutant au projet actuel, soit en créant un nouveau projet pour celui-ci.
