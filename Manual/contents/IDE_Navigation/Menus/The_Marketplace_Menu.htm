Le menu du Marketplace

Outre la création de vos propres ressources et code pour vos projets, vous avez également accès au Marketplace GameMaker Studio 2 où vous pouvez télécharger des ressources gratuites et payantes à ajouter à vos projets. Vous pouvez en savoir plus sur le Marketplace dans la section sur le Marketplace GameMaker, et les options de ce menu sont données ci-dessous:

  • Accédez à la place de Marketplace - Ouvrez la Marketplace laquelle vous pouvez rechercher des actifs.
  • Ma bibliothèque - Affichez votre bibliothèque de contenu Marketplace
  • Mon historique des achats - Cette option ouvrira un navigateur et vous amènera à la page du compte de l'utilisateur connecté afin que vous puissiez voir l'historique des achats pour ce compte.
  • Mettre à jour le package existant - Cette option ouvrira l' assistant de mise à jour de package afin que vous puissiez mettre à jour un Marketplace existant que vous avez précédemment créé.
  • Créer un package - Avec cette option, vous pouvez ouvrir l' assistant de création de package pour créer un nouveau package d'actifs ou un didacticiel pour la place de Marketplace.
