Le menu fichier

Le menu Fichier est l'endroit où vous pouvez choisir de modifier, enregistrer ou importer des projets ainsi que définir les préférences de l'EDI GameMaker Studio 2 Les options disponibles sont:

  • Nouveau projet - Ceci permet de créer un nouveau projet à partir de zéro dans GameMaker Studio 2. Il ouvrira la boîte de dialogue «Enregistrer» et vous invitera à choisir un emplacement pour enregistrer le nouveau projet. Une fois que vous avez donné un nom au projet et un emplacement pour l'enregistrer, vous serez redirigé vers un nouvel espace de travail vide pour commencer à travailler dessus. Les fichiers de projet sont enregistrés avec le *.yyp extension de fichier.
  • Ouvrir un projet - Cela ouvrira un explorateur de fichiers pour vous permettre de sélectionner un *.yyz fichier de projet pour le chargement. Veuillez noter qu'à l'importation, si le projet a été créé avec une version inférieure à 2.3.0, vous obtiendrez le message d'avertissement suivant: Si vous choisissez de continuer, un explorateur de fichiers sera ouvert et vous pourrez donner au projet un nom et un emplacement mis à jour pour l'enregistrer, afin que le projet d'origine ne soit pas écrasé. Une fois que le projet a été converti au nouveau format 2.3.0+, il s'ouvrira dans l'EDI et vous recevrez un message indiquant que la conversion a réussi. Vous devez procéder à la vérification du rapport de compatibilité qui a été généré par le processus de conversion et voir s'il y a quelque chose que vous devez corriger avant d'essayer d'exécuter ou de compiler le projet. Notez que si vous obtenez des erreurs répétées en essayant d'ouvrir un projet réalisé avec une version antérieure à 2.3.0, vous devez alors contacter YoYo Games (en utilisant l'option "Signaler un bug" du menu Aide ), en fournissant l'original *.yyp fichier que vous essayez de convertir.
  • Importer un projet - Ici, vous pouvez importer des projets précédemment exportés en tant que *.yyz (voir «Exporter le projet» ci-dessous), ou - si vous avez un projet réalisé avec le GameMaker: Studio 1.4 de YoYo Games - vous pouvez sélectionner cette option pour importer soit le *.gmx ou la *.gmz fichier de cette gamme de produits. Comme pour l'ouverture d'un projet, si le *.yyz a été créé avec une version de GameMaker Studio 2 antérieure à la mise à jour 2.3.0, il devra être converti car le format de fichier du projet a changé avec cette mise à jour.

REMARQUE: pour importer un projet GameMaker GameMaker Studio 2 il doit avoir été créé à l'aide de la version 1.4, et même dans ce cas, il peut ne pas fonctionner directement lors de l'importation. Beaucoup de choses sont différentes entre GameMaker Studio 2 et GameMaker: Studio 1.4, et donc vos projets plus anciens peuvent avoir besoin d'être peaufinés pour qu'ils fonctionnent. Veuillez consulter l'article Portage d'un jeu GM: S 1.4 vers GameMaker Studio 2 sur la YoYo Games et la section du manuel qui couvre les fonctions obsolètes et les scripts de compatibilité.

IMPORTANT! Importer un *.yyz projet qui contient des ressources de polices peut vous obliger à «régénérer» les polices en les supprimant / en les rajoutant, ou en modifiant la police dans l'éditeur de polices, puis en la redéfinissant. En effet, les informations de police existantes sont conservées par le *.yyz lorsqu'il est exporté et, lors d'une importation ultérieure ou sur une autre machine, le projet peut ne pas avoir les polices requises déjà installées (ou peut avoir une police différente du même nom). Cela ne devrait être fait que si vous rencontrez des problèmes avec le rendu de la police après l'importation.

  • Projets récents - Vous pouvez utiliser cette option pour afficher les projets récemment utilisés. En le sélectionnant, vous rouvrirez la page de démarrage (si vous l'avez fermée) et vous pourrez sélectionner un projet récemment utilisé dans la liste qui s'y trouve.
  • Enregistrer le projet - Cela enregistrera votre jeu au même emplacement que celui que vous avez défini lors de la création du projet (ou à partir de l'endroit où vous l'avez ouvert).
  • Enregistrer le projet sous - Comme ci-dessus, vous seul spécifiez l'emplacement et le nouveau nom du projet.
  • Exporter le projet - Cela exportera le projet en un seul *.yyz déposer. Très utile pour envoyer des copies à des personnes ou pour enregistrer des sauvegardes.
  • Préférences - En sélectionnant cette option, vous accédez à la fenêtre Préférences où vous pouvez configurer l'apparence et la convivialité de GameMaker Studio 2
  • Nouvel IDE - Lorsque vous sélectionnez cette option, une nouvelle instance de l'EDI sera ouverte pour que vous puissiez l'utiliser (elle s'ouvrira sur la page de démarrage).
  • Mon compte - Cette option ouvrira votre navigateur Web par défaut et vous YoYo page de votre compte YoYo.
  • Déconnexion - Sélectionnez cette GameMaker Studio 2. Vous devrez vous reconnecter avec la même licence (ou toute autre) pour pouvoir utiliser le produit plus tard.

IMPORTANT! Si vous avez utilisé une licence d'essai pour GameMaker Studio 2 et ensuite acheter une des autres licences (Desktop, Mobile, UWP, Web ou console), vous devrez peut - être vous déconnecter de l'IDE, puis redémarrez GameMaker Studio 2 et reconnectez- vous à nouveau pour vous assurer que la nouvelle licence est appliquée correctement (bien que cela devrait se produire automatiquement).

  • Quitter - Quittez GameMaker Studio 2 sur le bureau.
