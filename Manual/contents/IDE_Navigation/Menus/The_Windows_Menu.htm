Le menu Windows

Ce menu sert à ouvrir les fenêtres IDE ancrées générales. Étant donné que les quais peuvent être réorganisés et que le contenu peut être supprimé en tant que fenêtres autonomes ou même fermé entièrement, cette fenêtre vous permet de les rouvrir si vous trouvez que vous en avez besoin pour quelque chose. Les fenêtres peuvent être ouvertes d'ici sont le navigateur Asset fenêtre et toutes les différentes sorties fenêtres (sortie, recherche, contrôle de source, points d' arrêt et Compile / erreurs de syntaxe).

Ensuite, vous pouvez ouvrir la fenêtre Inspecteur. Cette fenêtre est utilisée pour inspecter les propriétés de l'élément IDE actuellement sélectionné, et ces propriétés peuvent ensuite être modifiées selon les besoins.

Après l'option Inspecteur, vous avez la Windows récentes, et vous avez également la possibilité d'ouvrir la fenêtre Variables d'environnement où vous pouvez voir les différents chemins vers lesquels pointent les différentes variables d'environnement utilisées dans les préférences. Ces variables ne peuvent pas être modifiées (ni leur nom ni leur chemin / paramètre) mais vous pouvez sélectionner et copier / coller l'une d'entre elles pour l'utiliser ailleurs.

Enfin, vous avez la possibilité d'ouvrir les Windows didacticiel. Cela rouvrira toutes les fenêtres liées à un projet de didacticiel que vous avez ouvert, mais uniquement si le projet est celui qui a été créé dans le cadre d'un didacticiel (sinon, cette option ne sera pas disponible). Pour plus d'informations sur la création de vos propres projets de didacticiel, veuillez consulter la section sur la place de Marketplace (car tous les didacticiels sont créés, hébergés et téléchargés à partir de là).
