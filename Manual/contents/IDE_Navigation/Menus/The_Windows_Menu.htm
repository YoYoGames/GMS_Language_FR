Le menu Windows

Ce menu permet d'ouvrir les fenêtres générales de l'IDE ancré. Puisque les docks peuvent être réorganisés et que leur contenu peut être supprimé en tant que fenêtres autonomes ou même fermé entièrement, cette fenêtre vous permet de les rouvrir si vous constatez que vous en avez besoin pour quoi que ce soit. Les fenêtres qui peuvent être ouvertes à partir d'ici sont la fenêtre Asset Browser et toutes les différentes fenêtres de sortie (Output, Search, Source Control, Points d'arrêt et Erreurs de compilation/syntaxe).

Ensuite, vous pouvez ouvrir la fenêtre Inspecteur. Cette fenêtre est utilisée pour inspecter les propriétés de l'élément de l'IDE actuellement sélectionné, et ces propriétés peuvent ensuite être modifiées si nécessaire.

Après l'option Inspecteur, vous avez la liste des fenêtres récentes, et il y a aussi l'option d'ouvrir la fenêtre Variables d'environnement où vous pouvez voir les différents chemins vers lesquels pointent les diverses variables d'environnement. variables d'environnement utilisées dans les Préférences. Ces variables ne peuvent pas être modifiées (ni leur nom, ni leur chemin/réglage) mais vous pouvez les sélectionner et les copier/coller pour les utiliser ailleurs. ailleurs.

Enfin, vous avez la possibilité d'ouvrir les fenêtres du didacticiel. Cette option permet de rouvrir toutes les fenêtres liées à un projet de didacticiel que vous avez ouvert, mais seulement si le projet a été créé dans le cadre d'un tutoriel (sinon, cette option ne sera pas disponible). Pour plus d'informations sur la création de vos propres projets de didacticiels, veuillez consulter la section sur le site Web de la Commission européenne. Place de marché (car tous les didacticiels sont créés, hébergés et téléchargés à partir de là).
