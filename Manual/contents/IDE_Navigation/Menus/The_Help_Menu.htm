Le menu d'aide

Ce menu permet d'obtenir de l'aide lors de l'utilisation de GameMaker Studio 2. Il contient les éléments de menu suivants :

  • Page d'accueil - Cela ouvrira un nouvel espace de travail Page d'accueil.
  • Page d'accueil - Ceci ouvrira le diaporama de la page d'accueil initiale dans un nouvel espace de travail.
  • Ouvrir le manuel - Ouvre le document d'aide. Notez qu'en cliquant sur le bouton d'aide dans la barre de menu, vous ouvrirez également le document, de même que si vous appuyez sur .
  • Base de connaissances - Vous accéderez au centre d'aide deGameMaker Studio 2, où vous trouverez des articles sur la configuration de la plate-forme, les problèmes connus, l'utilisation de GameMaker Studio 2 et bien d'autres choses encore. autres sujets.
  • Signaler un bug - Si vous trouvez des bugs ou rencontrez des problèmes lors de l'utilisation de GameMaker Studio 2, vous pouvez les signaler ici.
  • YoYoGames.com - Ouvre la page d'accueil principale deYoYo Games.
  • GameMaker Community - Cela ouvrira les forums deGameMaker Community dans votre navigateur. Veuillez noter qu'il s'agit d'un forum géré par la communauté . technique, vous devez d'abord vous rendre sur le site d'assistance officiel (voir ci-dessus).
  • Notes de publication - Cette option ouvre les notes de publication de GameMaker Studio 2 dans votre navigateur. Si vous devez annuler une mise à jour, vous pouvez obtenir des programmes d'installation de la version précédente en cliquant sur le lien du numéro de version sur cette page.
  • Notes de mise à jour du Runtime actuel - Comme le Runtime peut se mettre à jour indépendamment de l'IDE, il possède son propre ensemble de Runtime Release Notes. La sélection de cette option les ouvrira dans votre navigateur, mais notez que le runtime actuel peut ne pas être celui que vous utilisez. Pour savoir quel runtime vous utilisez, ou pour obtenir des informations sur la façon d'ajouter, de supprimer ou d'activer des runtimes, veuillez consulter la section PréférencesRuntime.
  • SDK requis - En raison de la nature multiplateforme de GameMaker Studio 2, un certain nombre de SDK tiers sont nécessaires. En cliquant sur ce lien, vous accéderez à un article du forum d'assistance qui détaille ce dont vous avez besoin pour chaque plateforme.
  • Ouvrir le projet dans l'explorateur - Ceci ouvrira l'emplacement de sauvegarde du projet actuellement chargé dans l'explorateur de fichiers du système d'exploitation.
  • À propos de - Ouvre la fenêtre "À propos de GameMaker Studio 2" dans laquelle vous pouvez obtenir des informations sur la version du produit (indispensable pour signaler tout bogue ou problème).
