Le menu de construction

Le menu de construction est l'endroit où vous pouvez choisir de créer votre projet pour le test, le débogage ou en tant qu'exécutable final. Les options disponibles sont:

  • Exécuter - Exécutez votre projet actuel sur la plate-forme cible choisie pour le test. Pour les différentes options de compilation disponibles lors de la compilation s'il vous plaît voir la section sur Compiling. Notez que cette commande équivaut à appuyer sur le bouton Exécuter dans l'IDE.
  • Déboguer - Exécutez votre projet actuel pour le tester avec le débogueur afin que vous puissiez voir des informations détaillées sur les performances et déboguer les erreurs ou problèmes que vous pourriez rencontrer. Pour plus d'informations sur le débogage, consultez la section Débogage. Notez que cette commande équivaut à appuyer sur le bouton Déboguer dans l'IDE.
  • Nettoyer - GameMaker Studio 2 conserve un cache de fichiers pour accélérer le temps de compilation lors des tests. Cependant, parfois, un cache "périmé" peut conduire à des erreurs étranges et inattendues dans votre jeu, auquel cas la première chose à faire est de nettoyer le cache d'ici et de recommencer le test. Il est également judicieux de nettoyer le cache avant de créer un exécutable final pour la plate-forme cible. Notez que cette commande équivaut à appuyer sur le bouton Nettoyer le cache dans l'IDE.
  • Créer un exécutable - Cela compilera le projet actuel et créera un package exécutable (ou des packages) pour la plate-forme cible. Il ouvrira d'abord l'explorateur de fichiers afin que vous puissiez sélectionner l'emplacement où vous souhaitez enregistrer le projet, puis compiler le jeu. Voir la section sur Compiling pour plus d' informations sur la mise en place des objectifs de compilation. Notez que cette commande équivaut à appuyer sur le bouton Créer un exécutable dans l'IDE.
  • Show Remote Worker Installer - Cela ouvrira une fenêtre d'explorateur de fichiers sur l'emplacement du programme d'installation de l' outil Remote Worker (voir ici pour plus d'informations sur les travailleurs à distance).
