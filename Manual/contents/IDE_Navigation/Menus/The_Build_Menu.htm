Le menu de construction

Le menu de construction est l'endroit où vous pouvez choisir de construire votre projet soit pour le tester, le déboguer ou comme exécutable final. Les options disponibles sont les suivantes :

  • Run - Exécutez votre projet actuel sur la plate-forme cible choisie pour le tester. Pour connaître les différentes options disponibles lors de la compilation, veuillez consulter la section sur la compilation. Notez que cette est la même chose que d'appuyer sur le bouton Run dans l'IDE.
  • Débogage - Exécutez votre projet actuel pour le tester avec le débogueur afin que vous puissiez voir des informations détaillées sur les performances et déboguer toute erreur ou tout problème que vous pourriez rencontrer. Pour plus d'informations sur le débogage, veuillez consulter la section Débogage. Notez que cette commande est la même que celle qui consiste à appuyer sur le bouton Debug dans l'IDE.
  • Clean - GameMaker Studio 2 conserve un cache de fichiers pour accélérer le temps de compilation lors des tests. Cependant, un cache "périmé" peut parfois entraîner des erreurs étranges et inattendues dans votre jeu. dans votre jeu. Dans ce cas, la première chose à faire est de nettoyer le cache à partir d'ici et de refaire le test. C'est aussi une bonne idée de nettoyer le cache avant de créer un exécutable final pour la plateforme cible. Notez que cette commande est la même que celle qui consiste à appuyer sur le bouton Clean Cache dans l'IDE.
  • Create Executable - Ceci compilera le projet actuel et créera un paquet exécutable (ou des paquets) pour la plateforme cible. Elle ouvrira d'abord l'explorateur de fichiers afin que vous puissiez sélectionner l'endroit où vous souhaitez enregistrer le projet, puis compiler le jeu. Consultez la section sur la compilation pour plus d'informations sur la configuration des cibles de compilation. Notez que cette commande est la la même chose que d'appuyer sur le bouton Create Executable dans l'IDE.
  • Afficher l'installateur de l'outil Remote Worker - Cela ouvrira une fenêtre d'exploration de fichiers à l'emplacement de l'installateur de l'outil Remote Worker (voir ici pour plus d'informations sur les travailleurs à distance). travailleurs à distance).
