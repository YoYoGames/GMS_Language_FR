Le menu des mises en page

Ce menu est lié à la sauvegarde et au chargement de différentes dispositions IDE. Vous pouvez configurer l'EDI comme vous le souhaitez, avec certains éléments (comme le navigateur d'actifs ou les Windows sortie) disposés différemment par défaut. Cela vous permet de configurer GameMaker Studio 2 en fonction de votre flux de travail et avec les options de ce menu, vous pouvez enregistrer les dispositions actuelles du projet en tant que *.xml fichier qui peut ensuite être chargé dans n'importe quel projet pour que les mises en page soient les mêmes. Les options de ce menu sont les suivantes:

  • Charger la mise en page - Cela ouvrira un explorateur de fichiers dans lequel vous pourrez sélectionner un fichier de mise en page *.xml précédemment enregistré pour le chargement.
  • Enregistrer la mise en page - Cela ouvrira un explorateur de fichiers dans lequel vous pourrez nommer et enregistrer la mise en page IDE actuelle sous forme de fichier *.xml.
  • Réinitialiser la mise en page - Cela réinitialisera la mise en page IDE actuelle aux paramètres par défaut.
