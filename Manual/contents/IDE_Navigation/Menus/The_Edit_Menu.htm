Le menu d'édition

Le menu d'édition est l'endroit où vous pouvez utiliser les outils d'édition générale de l'interface GameMaker Studio 2 Les options disponibles sont:

  • Annuler - Annule une action.
  • Refaire - Refait une action précédente.
  • Rechercher et remplacer - Cela ouvrira une fenêtre de recherche qui vous permettra de rechercher dans tout votre code (y compris le code de création de salle, les scripts et les événements) pour une entrée donnée - voir ci-dessous pour plus d'informations.

Lorsque vous sélectionnez Rechercher et remplacer dans le menu Edition, la fenêtre suivante s'ouvre:

Vous pouvez l'utiliser pour rechercher dans tous les éléments scriptés de votre jeu, y compris le code de salle, le code d'événement, les scripts et les shaders. Lors de la recherche, vous pouvez taper un nouveau terme de recherche dans la zone en haut, ou vous pouvez cliquer sur l'icône en forme de flèche à droite pour ouvrir une liste des termes recherchés précédemment. Vous avez également la possibilité de donner un terme de remplacement qui sera utilisé pour remplacer le terme de recherche initial, et cela a également une option pour utiliser une chaîne de remplacement précédemment utilisée à partir du bouton fléché sur la droite.

Les recherches peuvent être générales ou vous pouvez les affiner à l'aide des options ci-dessous les entrées des termes de recherche:

  • Vous pouvez choisir de rendre la recherche sensible à la casse, auquel cas elle n'inclura que les mots qui correspondent exactement à la casse de la chaîne de recherche, c'est-à-dire: rechercher "My_Var" sans cette option et vous obtiendrez des résultats avec "my_var", "mY_VAR", ou "Ma_Var", mais avec cette option cochée, seul "Ma_Var" sera inclus dans les résultats de la recherche.
  • Vous pouvez choisir de n'inclure que les résultats de recherche qui incluent uniquement le mot entier, auquel cas tous les résultats qui incluent le terme de recherche mais qui ne correspondent pas exactement seront ignorés, c'est-à-dire: une recherche sur "jeu" sans cette case cochée apparaîtra des choses comme la fonction "game_end ()" ou des variables comme "game_state", tout en vérifiant ceci et en effectuant la recherche ne montreront que les résultats qui sont "jeu" lui-même.
  • Vous pouvez choisir que la recherche ignore les commentaires. Lorsque cette option est cochée, tous les commentaires de votre code contenant le terme de recherche seront simplement ignorés et ne seront pas affichés dans les résultats de sortie.
  • Vous pouvez appliquer un filtre sur les résultats de la recherche en fonction des différents types d'actifs dans le navigateur d'actifs. Par défaut, ce paramètre est défini sur "Tout", mais en cliquant dessus, vous ouvrirez le menu suivant dans lequel vous pourrez cocher ou décocher les éléments à inclure dans la recherche:

Une fois que vous avez donné un terme de recherche (et éventuellement un terme de remplacement) et configuré les options de recherche, vous pouvez utiliser les différents boutons en bas pour choisir comment effectuer la recherche et les actions à entreprendre. Ces boutons sont:

  • Rechercher tout: recherchez tous les termes de recherche et affichez-les dans la fenêtre de sortie.
  • Rechercher précédent: recherchez l'instance précédente du terme de recherche. Cela ouvrira l'actif contenant la référence dans l'espace de travail (s'il n'est pas déjà ouvert) et le panoramique.
  • Rechercher suivant: recherchez l'instance suivante du terme de recherche. Cela ouvrira l'actif contenant la référence dans l'espace de travail (s'il n'est pas déjà ouvert) et le panoramique.
  • Remplacer tout: Cela effectuera une recherche pour le terme donné avec les options définies et le remplacera également dans les actifs par le nouveau terme de remplacement que vous avez spécifié.
  • Remplacer: Cela remplacera le terme actuellement sélectionné dans l'éditeur actuel où il est mis en surbrillance par le terme de remplacement sélectionné.
  • Arrêter la recherche: arrête la recherche en cours si elle est en cours.

Tous les résultats de l'utilisation de la fenêtre Rechercher / Remplacer seront affichés dans la fenêtre Sortie de recherche au bas de l'EDI. Chaque résultat peut être double-cliqué pour ouvrir la ressource avec le terme dans l'espace de travail actuel et faire un panoramique dessus, prêt pour l'édition.
