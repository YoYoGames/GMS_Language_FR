Notes de version

Chaque version de GameMaker Studio 2 est accompagnée de son propre ensemble de notes de mise à jour. Ceux-ci contiennent des informations détaillées sur les modifications apportées ainsi qu'une liste complète de tous les bogues corrigés. C'est un outil précieux pour décider de mettre à jour votre version actuelle ou lorsque vous trouvez quelque chose qui peut être un bogue après une publication.

Les notes de publication sont divisées en deux ensembles différents en raison du fait que l'EDI et les Runtimes réels peuvent être mis à jour indépendamment. Vous pouvez trouver les notes de version actuelles de l'EDI à partir du lien ci-dessous:

  • Notes de version de l'EDI

Vous pouvez trouver les notes de version actuelles des Runtimes partir du lien ci-dessous:

  • Notes de version d' Runtime
