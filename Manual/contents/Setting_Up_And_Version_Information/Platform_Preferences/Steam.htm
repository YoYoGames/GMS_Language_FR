Préférences Steam

Le Steam est utilisé pour les Windows, macOS et Ubuntu ( Linux ) qui doivent être distribuées via la Steam forme Steam. Cette section contient les options de configuration nécessaires pour le faire fonctionner dans vos projets:

  • Chemin d'accès au SDK Steam: le chemin du système vers lequel vous avez installé le SDK Steam Une fois que vous avez défini ce chemin vers le SDK, vous devrez ensuite activer Steam dans les options de jeu, puis pour chaque plate-forme, ajouter l' ID d'application.
