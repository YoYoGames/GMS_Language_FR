Préférences Windows

Les Windows ont les options suivantes:

  • Chemin Visual Studio: cette option permet de pointer GameMaker Studio 2 vers l' Visual Studio sur le PC de construction.
  • Type d'empaquetage par défaut: lors de la création d'un Windows final, vous pouvez choisir entre la création d'un programme d'installation (à l'aide de NSIS ) ou d'un fichier compressé. ZIP fichier qui contiendra le jeu et tous les fichiers supplémentaires qu'il contient. Par défaut, vous aurez le choix parmi ces deux options que vous souhaitez choisir lors de la compilation, mais vous pouvez définir cette préférence pour toujours utiliser l'une ou l'autre. Notez que lorsque cette option est définie sur Afficher le dialogue, dans la fenêtre de dialogue qui apparaît lors de la compilation, choisir "Mémoriser l’option d’emballage" définira cette préférence sur l’option sélectionnée.
