Préférences tvOS

Les tvOS ont les options suivantes:

  • Réponse automatique à l'avertissement de limitation de stockage persistant tvOS: les tvOS ont un stockage local limité pour les fichiers, et lorsque vous compilez pour cette cible, un avertissement s'affiche par défaut pour vous rappeler ce fait. Vous pouvez modifier le paramètre ici pour supprimer cet avertissement en sélectionnant "OK". Le paramètre par défaut est "Afficher le message".
  • Supprimer la génération et l'exécution: par défaut, lorsque vous créez un tvOS, puis que vous construisez un package exécutable final, GameMaker Studio 2 construira le package, puis tentera de l'exécuter sur tous les appareils connectés. Cocher cette option supprimera la création de l'application et son exécution sur l'appareil et créera simplement un projet XCode sur le Mac hôte.
  • Chemin d'installation macOS: chemin d'installation des fichiers de projet sur le Mac de construction.
  • Identifiant d'équipe par défaut: ici, vous pouvez ajouter votre identifiant d'équipe par défaut, tel qu'il vous a été attribué par Apple. Cet ID d'équipe sera utilisé lorsque vos fichiers de jeu sont envoyés à Xcode pour créer l'application et permettra à Xcode de générer les certificats de signature requis. Notez que ce paramètre sera appliqué par défaut à tous les jeux construits pour tvOS, mais il peut être écrasé par projet à partir du Général tvOS tvOS.htm "> Options de jeu.
