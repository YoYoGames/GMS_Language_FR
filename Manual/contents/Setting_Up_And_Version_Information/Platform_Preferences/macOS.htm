Préférences demacOS

Les préférences de macOS comportent les options suivantes :

  • Identifiant d'équipe par défaut: Vous pouvez ajouter ici votre identifiant d'équipe par défaut, tel qu'il vous a été attribué par Apple. Cet identifiant d'équipe sera utilisé lorsque vos fichiers de jeu seront envoyés à Xcode pour construire l'application, et permettra à Xcode de générer les certificats de signature requis. Notez que ce paramètre sera appliqué par défaut à tous les jeux construits pour macOS, mais qu'il peut être modifié pour chaque projet à partir des options générales de macOS. Options du jeu.
  • Choix de l'emballage automatique: vous pouvez définir ici une préférence par défaut pour la création d'un exécutable pour la cible macOS. L'option "Show Dialog" affichera toujours une boîte de dialogue avant de créer un exécutable, vous demandant de choisir entre DMG et Zip, mais vous pouvez définir cette option sur l'un ou l'autre dans les Préférences pour ignorer cette boîte de dialogue. Notez que l'activation de l'option "Remember Package Option" sur la boîte de dialogue définira également cette option sur le type de paquet sélectionné.
