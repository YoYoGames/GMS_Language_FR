Préférences Windows UWP

Les Windows UWP comportent les options suivantes:

  • Emplacement de Visual Studio: cette option permet de pointer GameMaker Studio 2 vers l' Visual Studio sur le PC de construction.
  • Emplacement des outils Win UWP: cette option permet de pointer GameMaker Studio 2 vers le chemin d'installation des outils UWP sur le PC de construction.
