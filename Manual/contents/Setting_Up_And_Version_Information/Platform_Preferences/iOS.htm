PréférencesiOS

Les préférences iOS comportent les options suivantes :

  • Supprimer la construction et l'exécution: Par défaut, lorsque vous créez un projet iOS, puis que vous construisez un package exécutable final, GameMaker Studio 2 construit le package, puis tente de l'exécuter sur tous les appareils connectés. En cochant cette option, vous supprimera la construction et l'exécution de l'application sur l'appareil et créera simplement un projet XCode sur le Mac hôte.
  • Chemin d'installation demacOS: Le chemin d'installation des fichiers du projet sur le Mac de construction.
  • Identifiant d'équipe par défaut: Vous pouvez ajouter ici votre identifiant d'équipe par défaut, tel qu'il vous a été attribué par Apple. Cet identifiant d'équipe sera utilisé lorsque les fichiers de votre jeu seront envoyés à Xcode pour construire l'application, et permettra à Xcode de générer les certificats de signature requis. Notez que ce paramètre sera appliqué par défaut à tous les jeux construits pour iOS, mais qu'il peut être modifié pour chaque projet à partir des Options générales des jeux iOS.
