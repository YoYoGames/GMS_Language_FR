Préférences générales

Les préférences générales sont utilisées pour définir le comportement général de l'IDE GameMaker Studio 2. Les plus générales sont listées sur la page principale, tandis que de nombreuses autres sont divisées en plusieurs sous-catégories, expliquées plus bas sur cette page.

  • Langue de l'IDE: Cette option vous permet de sélectionner la langue de l'IDE à partir de la liste d'options fournie. La langue par défaut est l'anglais.
  • Skin IDE: Vous pouvez choisir ici parmi les différents habillages que l'IDE peut utiliser. Par défaut, GameMaker Studio 2 est livré avec deux apparences, une claire et une foncée, et la configuration initiale utilise l'apparence foncée.
  • Fréquence d'images maximale de l'éditeur GameMaker Studio 2: Cette option vous permet de définir la fréquence d'images maximale à laquelle l'IDE de GameMaker Studio 2 se mettra à jour, ce qui vous permet de régler la fréquence d'images pour qu'elle corresponde à la vitesse de mise à jour de votre écran. La valeur par défaut de valeur par défaut est 60.
  • Périphérique audio par défaut: Vous pouvez choisir ici le périphérique audio par défaut à utiliser par l'IDE GameMaker Studio 2. Vous pouvez actualiser la liste en cliquant sur le bouton d'actualisation .
  • Réponse automatique à la confirmation de sortie: Cette option vous permet de définir si un message de confirmation doit s'afficher lorsque vous quittez GameMaker Studio 2. Par défaut, l'option est réglée sur " Afficher le message ", mais elle peut être réglée pour ignorer le message comme si vous aviez cliqué sur OK. message comme si vous aviez cliqué sur OK.
  • Automatic Response to The System Font Changed Dialog (Réponse automatique à la boîte de dialogue de modification de la police système) : Cette option vous permet de définir si un message de confirmation doit être affiché lorsqu'une police système a été ajoutée, supprimée ou modifiée. Par défaut, cette option est réglée sur "show message", mais elle peut être configurée de manière à ignorer le message comme si vous aviez cliqué sur OK.
  • Activer les notifications: En cochant cette case, l'IDE pourra afficher des notifications sur les nouveaux runtimes, etc... Cette option est activée par défaut.
  • Délai d'affichage de la notification (ms): Définit la durée (en millisecondes) d'affichage d'une notification. La valeur par défaut est 6000.
  • Déconnexion à la sortie: si cette option est cochée, l'IDE se déconnecte automatiquement de la session utilisateur lorsque GameMaker Studio 2 est fermé. Cette option est principalement destinée aux utilisateurs du secteur de l'éducation car, lorsqu'elle est cochée, cela signifie qu'à chaque fois que le programme se ferme, l'utilisateur est déconnecté. l'utilisateur sera déconnecté, ce qui libérera le siège de la session pour d'autres utilisateurs. Cette option est désactivée par défaut.
  • Activer le remplacement du DPI: Cette option vous permet de remplacer le paramètre DPI par défaut par le vôtre. Vous devez d'abord activer l'option et ensuite définir soit la valeur Raw DPi Override (la valeur par défaut est 96), soit le pourcentage du DPi natif (valeur par défaut 100%). Une fois la modification effectuée, vous serez invité à redémarrer l'IDE, ce qui peut vous obliger à sauvegarder tout travail effectué sur le projet actuellement ouvert. Notez que le DPi a un pourcentage minimum de 50% et maximum de 500%, et si vous vous passez la souris sur l'icône d'information, vous pouvez voir le paramètre maximum recommandé pour l'écran utilisé :

    . Notez que si vous dépassez le paramètre DPi recommandé pour le moniteur, vous verrez apparaître une notification qui vous permet de réinitialiser le DPi.
  • Afficher l'écran de bienvenue au démarrage: Cette option est activée par défaut et indique simplement à GameMaker Studio 2 d'afficher un bref écran de démarrage lorsque vous lancez le programme. Si vous décochez cette option, l'écran d'accueil ne s'affichera pas.
  • Autoriser la fermeture des onglets vides via un raccourci: vous pouvez fermer tous les onglets ouverts à l'aide du raccourci clavier / + W, mais si vous désactivez cette option, ce raccourci ne fonctionnera plus. Cette option est activée par défaut.
  • Ouvrir de nouvelles ressources à éditer en faisant glisser des fichiers dans l'IDE: Certaines ressources (comme les sprites ou les sons) peuvent être glissées sur l'espace de travail principal de l'IDE de GameMaker Studio 2 et seront détectées et ajoutées en tant que nouvelle ressource dans le navigateur Asset Navigateur. Par défaut, l'éditeur de ressources correspondant sera également ouvert pour modification lorsque cela se produit, mais si vous décochez cette option, les éditeurs ne seront pas ouverts et vous devrez les ouvrir manuellement.
  • Ouvrir les nouvelles ressources pour l'édition à la création: Si cette option est cochée, chaque fois que vous créez une nouvelle ressource dans le navigateur d'actifs, l'éditeur de ressources correspondant sera ouvert automatiquement. Cette option est cochée par défaut et si vous la décochez, vous devrez ouvrir manuellement toutes les ressources nouvellement créées. signifie que vous devez ouvrir manuellement toutes les ressources nouvellement créées.
  • Recharger automatiquement les fichiers modifiés: Si cette case est cochée, l'IDE analysera et vérifiera automatiquement les fichiers pour voir si l'un d'entre eux a été modifié en dehors de GameMaker Studio 2 et les rechargera si c'est le cas.
  • Désactiver les animations de transition de l'IDE: L'IDE de GameMaker Studio 2 comporte un certain nombre de transitions animées pour des choses comme l'ouverture de fenêtres, etc... Cependant, elles ne sont pas du goût de tout le monde et vous pouvez bénéficier d'une amélioration des performances si elles sont désactivées sur des machines plus anciennes ou moins puissantes. sont désactivées sur les machines anciennes ou sous-puissantes. Vous pouvez les désactiver ici, et elles sont activées par défaut.
  • Accepter de participer aux enquêtes de satisfaction: Lorsque cette option est cochée, vous indiquez à GameMaker Studio 2 que vous souhaitez participer à des enquêtes de satisfaction, ce qui signifie qu'à l'occasion, pendant que vous travaillez, un message s'affichera sur la zone sur laquelle vous travaillez et vous demandera une évaluation et des commentaires. sur lequel vous travaillez et vous demandera une évaluation et des commentaires. Cette option est activée par défaut.
