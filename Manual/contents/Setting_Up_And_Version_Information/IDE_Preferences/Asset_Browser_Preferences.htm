Préférences du navigateur d'actifs

Les préférences du navigateur d'actifs permettent de définir certaines propriétés du fonctionnement du navigateur d'actifs et de ses éléments. Les principales options sont:

  • Afficher l'accès rapide: lorsqu'elle est activée, cette option affichera la section «Accès rapide» en haut du navigateur de ressources. Cette option est activée par défaut.
  • Créer des dossiers par défaut dans un nouveau projet: lorsqu'elle est activée, cette option fera démarrer tous les nouveaux projets avec les dossiers d'actifs par défaut déjà créés, et lorsqu'elle est désactivée, seul un dossier de salle sera créé avec la salle par défaut requise par tous les projets. Cette option est activée par défaut.
  • Nombre maximum d'actifs à ouvrir simultanément sans avertissement: vous pouvez sélectionner plusieurs actifs à ouvrir à la fois en utilisant / + pour les sélectionner puis faire un clic droit et en choisissant ouvrir tout. Si vous avez sélectionné plus que le nombre spécifié dans cette option, un avertissement s'affichera, car avoir trop d'actifs ouverts en même temps peut entraîner des problèmes de performances sur les ordinateurs bas de gamme. La valeur par défaut est 10.
  • Nombre maximal d'éléments récemment modifiés: ce paramètre régit le nombre d'actifs répertoriés dans la section «Récents» du navigateur d'actifs. La valeur par défaut est 10.
  • Taille de la palette de couleurs: ici, vous pouvez définir le nombre de couleurs disponibles dans la palette du navigateur d'actifs pour les groupes d'actifs. La valeur par défaut est 16, avec une valeur minimale de 16 et une valeur maximale de 100.
  • Appliquer les couleurs d'actif dans l'accès rapide: lorsqu'elle est cochée, cette option utilisera les couleurs définies pour un type d'actif donné pour colorer l'actif dans la section Accès rapide du navigateur. Le fait de décocher cette option n'utilisera pas la couleur de l'élément. C'est activé par défault.
  • Les couleurs de groupe remplacent les couleurs enfants: lorsque cette option est cochée, cette option utilisera la couleur de groupe de niveau supérieur pour colorer tous les éléments du groupe, en ignorant toute couleur définie pour l'élément individuel. Ceci est désactivé par défaut.
  • Créer un style de dialogue d'actif: Lorsque vous cliquez sur le bouton "Ajouter un actif" dans le navigateur d'actifs, une fenêtre vous montre les différents actifs qui peuvent être ajoutés à votre projet. Cette fenêtre peut être présentée dans une vue de liste ou une vue de grille, et ce paramètre régit ce qui est affiché par défaut. La valeur par défaut est la grille.
  • Ordre de tri par défaut: Ceci peut être utilisé pour définir l'ordre de tri par défaut des actifs dans le navigateur, soit par ordre alphabétique de A à Z, soit de Z à A. La valeur par défaut est de A à Z.
  • Tri par défaut des dossiers en premier: lorsque cette option est activée, le navigateur d'actifs triera d'abord les dossiers, puis les actifs «libres» en second, tandis que si cette option est désactivée, tous les actifs et dossiers seront triés ensemble. C'est activé par défault.
  • Sprite double-clic ouvre l'éditeur d'image: Par défaut, double-clic sur n'importe quel actif ouvrira cet actif pour modification, mais souvent lorsqu'il s'agit de sprites, après la configuration initiale lorsque vous créez le nouvel actif, vous l'ouvrez pour modifier l'image plutôt que les propriétés du sprite. La sélection de cette option ouvrira l'éditeur d'image pour la ressource de sprite au lieu de la ressource de sprite elle-même. Vous pouvez toujours ouvrir l'éditeur de sprite en cliquant avec le bouton droit de la souris et en sélectionnant Modifier. Cette option est désactivée par défaut.
  • Un double-clic sur un objet ouvre tous les événements de l'objet: par défaut, un double-clic sur n'importe quel actif ouvrira cet actif pour modification, mais souvent lorsqu'il s'agit d'objets, après la configuration initiale lorsque vous créez le nouvel actif, vous l'ouvrez pour modifier les événements uniquement plutôt que les propriétés de l'objet. La sélection de cette option ouvrira les différents événements d'objet dans un nouvel espace de travail, chaque événement ayant son propre onglet dans le nouvel espace de travail. Vous pouvez toujours ouvrir l'éditeur d'objets en cliquant avec le bouton droit de la souris et en sélectionnant Modifier. Cette option est désactivée par défaut.
  • La dissociation de l'ordre des salles supprime le groupe vide: lorsque cette option est activée, la dissociation de toutes les salles appartenant à un dossier de groupe supprimera également le dossier de groupe. Lorsqu'elle est désactivée, la même action dissocie les salles mais laisse le dossier de groupe vide dans le navigateur de ressources. C'est activé par défault.
  • Zoom par défaut (%): Ceci définit le pourcentage de zoom par défaut pour le navigateur de ressources lorsque vous démarrez un nouveau projet. La valeur par défaut est 100%.
  • Zoom minimum (%): Ceci définit la valeur de zoom minimum autorisée pour le navigateur de ressources. La valeur par défaut est 50%.
  • Zoom maximum (%): Ceci définit la valeur de zoom maximum autorisée pour le navigateur de ressources. La valeur par défaut est 250%.
  • Afficher les lignes de l'arborescence: ici, vous pouvez activer / désactiver les lignes de l'arborescence qui sont affichées à côté des actifs dans le navigateur d'actifs. Ceci est coché par défaut pour afficher les lignes d'arbre.
  • Développer l'arborescence lors du changement de filtre: cette option forcera le navigateur d'actifs à développer tous les nœuds d'arborescence réduits lorsque vous modifiez les options de filtre. La désactivation de l'option empêchera cela. La valeur par défaut est activée.

Confirmer les boîtes de dialogue

Ici, vous pouvez sélectionner le comportement des boîtes de dialogue de confirmation lorsque vous travaillez avec un actif à partir du navigateur d'actifs. Les options disponibles sont:

  • Réponse automatique à la suppression d'actif: lorsque vous supprimez un actif du navigateur d'actifs, vous recevez normalement un avertissement vous demandant si vous êtes sûr de vouloir continuer. En définissant cette option sur "OK", le message sera supprimé comme si vous aviez cliqué sur le bouton "OK" pour continuer. La valeur par défaut est "Afficher le message".
  • Réponse automatique à la suppression du groupe Ordre des salles: Lorsque vous supprimez un dossier de groupe contenant des salles du navigateur de ressources, vous recevez normalement un avertissement vous demandant si vous êtes sûr et que vous voulez procéder (qui répertorie toutes les salles du groupe qui seront supprimées). En définissant cette option sur "OK", le message sera supprimé comme si vous aviez cliqué sur le bouton OK pour continuer et supprimer toutes les salles du groupe ainsi que le dossier du groupe lui-même. La valeur par défaut est "Afficher le message".
  • Réponse automatique à l'ouverture de plusieurs actifs: comme indiqué ci-dessus dans les préférences générales du navigateur d'actifs, lorsque vous ouvrez plusieurs actifs à la fois à partir du navigateur d'actifs, vous pouvez recevoir un avertissement si le nombre dépasse le nombre spécifié. En définissant cette option sur "OK", l'avertissement sera supprimé comme si vous aviez cliqué sur le bouton "OK" pour continuer. La valeur par défaut est "Afficher le message".
  • Réponse automatique à la DnD™ en GameMaker Language: Lorsque vous sélectionnez l'option de conversion d'un DnD™ en GML, vous serez invité à confirmer la modification car elle ne peut pas être annulée. En définissant cette option sur "OK", l'avertissement sera supprimé comme si vous aviez cliqué sur le bouton "OK" pour continuer. La valeur par défaut est "Afficher le message".
  • Réponse automatique à la définition d'un parent de pièce par glissement de nœud: Dans la section "Héritage" de la fenêtre du Gestionnaire de salle, vous pouvez faire glisser une pièce vers une autre pièce pour la définir comme enfant de la pièce vers laquelle elle est déplacée. En faisant cela, vous recevrez un avertissement vous demandant si vous êtes sûr et que vous voulez procéder. En définissant cette option sur "OK", le message sera supprimé comme si vous aviez cliqué sur le bouton OK pour continuer. La valeur par défaut est "Afficher le message".
  • Réponse automatique à l'erreur de nom de groupe en double: lorsque vous créez un dossier de groupe pour organiser vos actifs et que vous le nommez comme un autre dossier dans le navigateur, un message s'affiche vous demandant si vous souhaitez continuer et avoir deux dossiers avec le même nom. En définissant cette option sur "OK", l'avertissement sera supprimé comme si vous aviez cliqué sur le bouton "OK" pour continuer. La valeur par défaut est "Afficher le message".
  • Réponse automatique à la définition d'un parent de pièce via le glissement de nœud: lorsque vous faites glisser une pièce sur une autre pièce dans le Gestionnaire de salle, puis que vous la relâchez, un message s'affiche vous demandant si vous souhaitez faire de la pièce un «enfant» de la pièce en cours. tombé sur. En définissant cette option sur "OK", l'avertissement sera supprimé comme si vous aviez cliqué sur le bouton "OK" pour continuer. La valeur par défaut est "Afficher le message"
