Préférences de l'espace de travail

Les préférences ici affecteront la façon dont vous vous déplacez dans les différents espaces de travail GameMaker Studio 2. Les options sont:

  • Plage de degrés de navigation du clavier de l'espace de travail: dans un espace de travail, vous pouvez ouvrir plusieurs fenêtres pour différents objets et ressources.Pour vous déplacer rapidement entre eux, vous pouvez utiliser le raccourci clavier / + + <Arrow Keys> se déplacer entre eux. Étant donné que les fenêtres dans un espace de travail ne sont pas alignées sur la grille, l'EDI doit choisir la fenêtre que vous voulez dire lorsque vous appuyez sur l'une des 8 directions possibles, et cette option vous permet de définir "l'angle de vue" utilisé. La valeur par défaut est de 40 °.
  • Remplissage des colonnes de la chaîne de l'espace de travail (px): les éléments de l'espace de travail sont souvent «enchaînés» les uns aux autres (par exemple, l'éditeur de code est enchaîné à l'éditeur d'événements est enchaîné à l'éditeur d'objets), et cette option vous permet de définir la distance horizontale entre deux fenêtres chaînées quelconques. La valeur par défaut est 30px.
  • Rembourrage des lignes de la chaîne de l'espace de travail (px): Comme mentionné ci-dessus, les éléments de l'espace de travail sont souvent «enchaînés» les uns aux autres et cette option vous permet de définir la distance verticale entre deux fenêtres chaînées. La valeur par défaut est 20px.
  • Nombre de segments de rendu de la chaîne de l'espace de travail: vous pouvez définir ici le nombre maximum de segments pour les chaînes de rendu. La réduction de cette valeur peut améliorer les performances sur les machines bas de gamme. La valeur par défaut est 20.
  • Couleur des maillons de la chaîne de l'espace de travail: il s'agit de la couleur qui sera utilisée pour afficher les chaînes entre les éléments de l'espace de travail. La couleur par défaut est ARGB $99DD00.
  • La navigation au clavier dans l'espace de travail réinitialise le zoom: comme mentionné ci-dessus, vous pouvez passer d'une fenêtre à l'autre dans l'espace de travail à l'aide de raccourcis clavier. Vous pouvez également effectuer un zoom avant ou arrière sur l’espace de travail à l’aide de / + la molette de la souris. La définition de cette option signifie que si vous avez agrandi l'espace de travail et que vous utilisez les raccourcis pour passer à une autre fenêtre, l'espace de travail reviendra à une vue 1: 1 et sera centré sur la fenêtre sélectionnée. Cette option est désactivée par défaut.
  • Les chaînes d'espace de travail peuvent se chevaucher: lorsqu'elle est activée, cette option permet aux chaînes d'espace de travail de se croiser et de se chevaucher. Ceci est désactivé par défaut.
