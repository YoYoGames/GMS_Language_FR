Aller aux préférences

Les préférences ici concernent la fenêtre Aller à, que vous pouvez ouvrir à l'aide du raccourci clavier / + T.

  • Nombre maximal de résultats de recherche à afficher: cette option contrôle le nombre maximal de résultats renvoyés lorsque vous utilisez la fenêtre de recherche Atteindre.
  • Activer le style sur les résultats de recherche (désactiver pour de meilleures performances): cette option active ou désactive le style du texte pour les éléments trouvés dans la fenêtre Atteindre. La valeur par défaut est activée.
  • Rechercher dans les actifs: la désactivation de cette option signifie que la fenêtre Aller à omettra tous les éléments du navigateur d'actifs qui coïncident avec la phrase de recherche donnée. La valeur par défaut est que cela est activé.
  • Rechercher dans les préférences: la désactivation de cette option signifie que la fenêtre Aller à omettra tous les éléments des préférences qui coïncident avec la phrase de recherche donnée. La valeur par défaut est que cela est activé.
  • Rechercher dans les options de jeu: la désactivation de cette option signifie que la fenêtre Aller à omettra tous les éléments des options de jeu qui coïncident avec la phrase de recherche donnée. La valeur par défaut est que cela est activé.
