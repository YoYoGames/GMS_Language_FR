Préférences d'aide

Le fichier d'aide est conçu pour fonctionner comme une page Web et même si les fichiers sont stockés localement, il nécessite l'utilisation d'un navigateur externe pour les afficher. Les paramètres suivants liés au fichier d'aide peuvent être modifiés ici:

  • Langue du manuel: Ceci définit la langue à utiliser par l'EDI. La valeur par défaut est «anglais».
  • Port manuel d'aide: Ceci définit le numéro de port que le micro-serveur Web manuel doit utiliser. La valeur par défaut est 51291.
