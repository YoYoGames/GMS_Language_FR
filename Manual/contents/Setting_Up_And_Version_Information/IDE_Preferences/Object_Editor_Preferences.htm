Préférences de l'éditeur d'objets

Les préférences de l'éditeur d'objets sont utilisées pour définir certaines propriétés de la fenêtre de l' éditeur d'objets. La section principale contient l'option suivante:

  • Cliquez pour ouvrir Méthode d'événement: Cette option contrôle si vous devez effectuer un double clic ou un simple clic sur un événement d'objet pour l'ouvrir. Par défaut, cela est configuré pour double-cliquer.
  • Evénements hérités alpha: Cette option vous permet de modifier l'alpha avec lequel tous les événements hérités ¡seront dessinés dans la liste d'événements de l'objet. Par défaut, il est défini sur 0,5 (les valeurs doivent être comprises entre 0 et 1).
  • Format d'étiquette d'événement: Ici, vous pouvez activer ou désactiver les différents textes possibles qui peuvent être appliqués à l'étiquette d'événement dans la liste d'événements de l'éditeur d'objets. Par défaut, la description de l'événement est activée uniquement.
  • Ajouter un contenu d'événement par défaut: par défaut, cette option est activée et il suffit d'ajouter un simple commentaire dans l'éditeur de code pour tous les nouveaux événements pour vous rappeler que vous pouvez y ajouter votre code.
  • Contenu d'événement par défaut: Ici, vous pouvez modifier les commentaires qui sont ajoutés par défaut dans la fenêtre de code pour les nouveaux événements.

La sous-catégorie suivante existe également pour l'éditeur d'objets:

Confirmer les boîtes de dialogue

Cette section traite des boîtes de dialogue qui peuvent être affichées lors de l'utilisation de l'éditeur d'objets, avec l'option suivante disponible:

  • Réponse automatique à la DnD™ en Gamemaker Language: vous pouvez utiliser le bouton droit de la souris dans l' Drag and Drop™ pour sélectionner l'option permettant de transformer tous les Drag and Drop™ de cet événement en code GML L'action par défaut ici est d'afficher un message d'avertissement vous demandant si vous voulez continuer, mais vous pouvez le définir sur "OK" et la boîte de dialogue ne s'affichera pas avant la conversion.
  • Réponse automatique en cas de dépassement de la limite de vertex physique: cette option peut être utilisée pour contrôler le message contextuel qui s'affiche lorsque vous dépassez le nombre de vertex autorisé dans l'éditeur de forme physique. L'action par défaut ici est d'afficher un message d'avertissement, mais vous pouvez le définir sur "OK" et la boîte de dialogue ne s'affichera pas.
  • Choix automatique lors de la suppression d'événement: cette option peut être utilisée pour contrôler le message contextuel qui s'affiche lorsque vous supprimez un événement dans la liste des événements de l'éditeur d'objets. L'action par défaut ici est d'afficher un message d'avertissement, mais vous pouvez le définir sur "OK" et la boîte de dialogue ne s'affichera pas.
