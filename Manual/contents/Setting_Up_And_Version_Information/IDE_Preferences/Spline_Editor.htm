Préférences de l'éditeur de spline

Les préférences de l'éditeur de spline sont principalement utilisées pour définir l' apparence des chemins lisses dans l'éditeur de chemin, bien qu'elles affectent certains autres aspects de l' EDI (comme l'éditeur de forme physique). Les options suivantes existent:

  • Insérer la précision du point de chemin (px): Ceci définit la valeur de précision (en pixels) pour insérer des points sur un chemin, c'est-à-dire: la souris doit être dans cette plage pour insérer le point plutôt que d'en ajouter un nouveau. La valeur par défaut est 20 px.
  • Largeur du point d'ancrage: définissez la largeur des points d'ancrage, en pixels. La valeur par défaut est 8px;
  • Hauteur du point d'ancrage: définissez la hauteur des points d'ancrage, en pixels. La valeur par défaut est 8px;
  • Couleur du chemin: la couleur à utiliser pour afficher le chemin lorsque vous le modifiez. La valeur par défaut est ( ARVB ) #FFFFFFFF.
  • Couleur du segment de chemin en surbrillance: couleur à utiliser pour mettre en évidence les segments de chemin sélectionnés. La valeur par défaut est (ARVB) #FFFFFFFF.
  • Couleur de chemin non valide: couleur à utiliser pour mettre en évidence un chemin qui n'est pas valide. La valeur par défaut est (ARVB) # FF0000FF.
  • Couleur des points d'ancrage: couleur à utiliser pour dessiner les points d'ancrage du tracé. La valeur par défaut est (ARGB) # FFC0C0C0.
  • Couleur des points d'ancrage non valide: couleur à utiliser pour dessiner les points d'ancrage du tracé. La valeur par défaut est (ARVB) # FF800000.
  • Couleur du point d'ancrage sélectionné: couleur à utiliser pour dessiner les points d'ancrage du tracé qui ont été mis en surbrillance (sélectionnés). La valeur par défaut est (ARVB) # FF00A4FF.
  • Couleur de point d'ancrage non valide sélectionnée: couleur à utiliser pour dessiner les points d'ancrage qui sont mis en surbrillance mais non valides. La valeur par défaut est (ARVB) # FFA400FF.
  • Couleur du point d'ancrage au survol: couleur à utiliser pour dessiner les points d'ancrage actifs. La valeur par défaut est (ARVB) #FFFFFFFF.
  • Couleur du point d'ancrage non valide au survol: couleur à utiliser pour dessiner les points d'ancrage qui sont actifs mais non valides. La valeur par défaut est (ARVB) # FF0000FF.
  • Couleur de superposition de masque: définissez la couleur de superposition des masques s'ils sont activés (voir ci-dessus). La valeur par défaut est (ARVB) # FF404040.
