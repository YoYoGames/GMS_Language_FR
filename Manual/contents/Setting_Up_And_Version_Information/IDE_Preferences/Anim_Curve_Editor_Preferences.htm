Préférences de l'éditeur de courbes d'animation

Les préférences de courbes d'animation sont utilisées pour définir certaines propriétés lorsque vous travaillez avec l' éditeur de courbes d'animation. Les options disponibles sont:

  • Réponse automatique à la suppression de la courbe: cette option peut être utilisée pour contrôler le message contextuel qui s'affiche lorsque vous supprimez une courbe de l'éditeur de courbe d'animation. L'action par défaut ici est d'afficher un message d'avertissement, mais vous pouvez le définir sur "OK" et la boîte de dialogue ne s'affichera pas.
