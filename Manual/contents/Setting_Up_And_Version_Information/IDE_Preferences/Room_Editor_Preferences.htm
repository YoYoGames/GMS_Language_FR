Préférences de l'éditeur de salle

Les préférences de l'éditeur de pièce sont utilisées pour définir l'apparence et le comportement de l' éditeur de pièce. Les paramètres généraux sont les suivants:

  • Distance de départ du fondu de la chaîne (px): définit la distance à partir de la fenêtre parente à partir de laquelle le maillon de la chaîne doit disparaître. La valeur par défaut est 10 px.
  • Distance d'arrêt du fondu de la chaîne (px): définit la distance à partir de la fenêtre parente pour le maillon de la chaîne vers lequel se termine le fondu. La valeur par défaut est 100 px.
  • Chaîne alpha minimum: définit la valeur alpha finale après le point d'atténuation du fondu sortant pour l'alpha de la chaîne. La valeur par défaut est 0,3.
  • Espacement de profondeur de calque par défaut: lorsque vous créez plusieurs calques dans l'éditeur de pièce, une profondeur leur est automatiquement attribuée en fonction de leur position dans la liste des calques. Cette profondeur est incrémentée du montant indiqué ici. La valeur par défaut est 100, donc chaque calque sera le calque précédent + 100 en profondeur.
  • Largeur de pièce par défaut: définit la largeur initiale qui sera sélectionnée par défaut pour une nouvelle pièce. La valeur par défaut est 1024px.
  • Hauteur de pièce par défaut: définit la hauteur initiale qui sera sélectionnée par défaut pour une nouvelle pièce. La valeur par défaut est 768px.
  • Ajouter des couches de pièce par défaut: lorsque vous créez une nouvelle ressource de salle, GameMaker Studio 2 génère pour vous quelques couches «de base» - une couche d'arrière-plan et une couche d'instance. Vous pouvez désactiver ce comportement ici afin que les nouvelles pièces n'aient pas de couches prédéfinies. Ceci est activé par défaut.
  • Afficher les vues de la pièce: si vous avez des vues actives dans une pièce, cela est indiqué par un contour où les limites de la vue seraient dans la pièce, et la visibilité de cette zone peut être basculée à l'aide du bouton dédié dans l'éditeur lui-même. Cependant, vous pouvez choisir de rendre les limites de vue invisibles lorsque vous ouvrez une pièce en décochant cette option (le bouton peut toujours être utilisé pour les afficher à nouveau). La valeur par défaut est activée, ce qui rend les limites de la vue visibles lorsque vous ouvrez une pièce.
  • Options de couleur générales: elles sont liées aux différentes couleurs utilisées dans l'interface de l'éditeur de salle, et si vous double-cliquez sur l'un des échantillons, vous pouvez modifier la couleur de ce paramètre.
  • Réglage du volume principal: Le réglage du volume principal pour la lecture de l'aperçu dans l'éditeur de pièce, une valeur comprise entre 0 et 1. La valeur par défaut est 1.

Confirmer les boîtes de dialogue

Les préférences de confirmation des boîtes de dialogue sont utilisées pour modifier les messages qui s'affichent dans l'éditeur de salle lorsque vous effectuez certaines actions. Les messages que vous pouvez modifier sont:

  • Réponse automatique à la mise à jour des profondeurs de calque: lorsque vous modifiez les profondeurs de calque, un message s'affiche pour vous avertir que cela mettra également à jour automatiquement toutes les autres profondeurs de calque. Avec cette option, vous pouvez choisir d'afficher le message, de le mettre à jour comme si vous aviez cliqué sur "Ok" ou de ne rien mettre à jour comme si vous aviez cliqué sur "Non".
  • Réponse automatique à l'avertissement de calque verrouillé: Si vous essayez de supprimer un calque dans la pièce et qu'il est verrouillé, un message s'affichera par défaut. Cependant, la définition de cette option sur "Ok" n'affichera pas le message et ne vous permettra tout simplement pas de supprimer le calque.
  • Réponse automatique à l'avertissement d'élément verrouillé: Si vous essayez de supprimer un élément, il se trouve être verrouillé, un message s'affichera par défaut. Cependant, la définition de cette option sur "Ok" n'affichera pas le message et ne vous permettra tout simplement pas de supprimer l'élément.
  • Réponse automatique à l'avertissement du calque du propriétaire de l'élément verrouillé: Si vous essayez de supprimer un élément sur un calque et que le calque est verrouillé (même si l'élément ne l'est pas), un message s'affiche par défaut pour vous informer qu'il peut pas être fait. Cependant, la définition de cette option sur "Ok" n'affichera pas le message et ne vous permettra tout simplement pas de supprimer l'élément.
  • Créez un nouveau calque d'actifs lorsque vous faites glisser des séquences / sprites sur des calques inappropriés: Si vous essayez d'ajouter un sprite (ou une séquence) à un calque qui n'est pas un calque d'actifs, un message s'affiche par défaut pour vous informer qu'il peut ne soyez pas fait et demandez si vous souhaitez créer un nouveau calque de ressources pour l'élément en cours de glissement. Cependant, définir cette option sur "Ok" n'affichera pas le message et créera automatiquement un nouveau calque pour les sprites (et les séquences).
  • Réponse automatique à la suppression d'un élément de la chambre parent: Ici, vous pouvez choisir ce qui se passe lorsque vous supprimez un élément qui fait partie d'une chambre parent (et donc présent dans toutes les chambres enfants). La valeur par défaut est de vous montrer un message d'avertissement et de vous laisser décider quoi faire, mais vous pouvez le définir sur "Supprimer tout" pour supprimer le message et continuer avec la suppression de l'élément de la salle parent et de tous ses enfants, ou vous pouvez sélectionnez "Garder tout" pour supprimer le message et annuler la suppression.
  • Réponse automatique à la suppression du calque de la pièce parent: Ici, vous pouvez choisir ce qui se passe lorsque vous supprimez un calque qui fait partie d'une pièce parent (et donc présent dans toutes les pièces enfants). La valeur par défaut est de vous montrer un message d'avertissement et de vous laisser décider quoi faire, mais vous pouvez le définir sur "Supprimer tout" pour supprimer le message et continuer avec la suppression du calque de la salle parent et de tous ses enfants, ou vous pouvez sélectionnez "Garder tout" pour supprimer le message et annuler la suppression.
  • Réponse automatique à une erreur de glissement d'actif non valide: si vous essayez d'ajouter un actif à une couche qui n'est pas désignée comme étant pour ce type d'actif (comme l'ajout d'un chemin à une couche d'instance), un message s'affiche par défaut pour informer vous que cela ne peut pas être fait. Cependant, la définition de cette option sur "Ok" n'affichera pas le message et ne vous permettra tout simplement pas d'ajouter l'élément. Notez que cette option ne s'applique pas aux instances, sprites ou séquences.
  • Créez un nouveau calque d'instance lorsque vous faites glisser des objets sur des calques inappropriés: si vous essayez d'ajouter une occurrence d'objet à un calque qui n'est pas un calque d'instance, un message vous sera affiché par défaut pour vous informer que cela ne peut pas être fait et demandez si vous souhaitez créer un nouveau calque d'occurrence pour l'élément en cours de déplacement. Cependant, la définition de cette option sur "Ok" n'affichera pas le message et créera automatiquement un nouveau calque pour l'instance.
  • Réponse automatique à une erreur de glissement de sélection de calque non valide: ici, vous pouvez choisir ce qui se passe lorsque vous faites glisser une ressource vers l'éditeur de salle lorsque vous avez sélectionné plusieurs calques dans l'éditeur de calques (ou que vous n'avez sélectionné aucun calque du tout). La valeur par défaut est d'afficher un message d'avertissement, mais vous pouvez définir cette option sur "OK" pour supprimer le message et continuer, sans ajouter de ressources.
  • Réponse automatique à une erreur de glissement de calque verrouillé ou invisible: ici, vous pouvez choisir ce qui se passe lorsque vous faites glisser une ressource sur un calque qui a été verrouillé ou qui est marqué comme invisible. La valeur par défaut est d'afficher un message d'avertissement, mais vous pouvez définir cette option sur "OK" pour supprimer le message et continuer, sans ajouter de ressources.

Paramètres de la grille

Les préférences Paramètres de la grille sont utilisées pour modifier le fonctionnement de la superposition de la grille, avec les options suivantes disponibles:

  • Espacement horizontal de la grille (px): Cela définira l'espace horizontal des cellules ceinturées de la grille de la pièce en pixels. La valeur par défaut est 32px.
  • Espacement vertical de la grille (px): Cela définira l'espace vertical des cellules ceinturées de la grille de la pièce en pixels. La valeur par défaut est 32px.
  • Afficher la grille: cette option active ou désactive la grille de la pièce lorsque vous ouvrez une pièce pour la première fois. Il est activé par défaut, mais même s'il est désactivé, vous pouvez le réactiver à l'aide du bouton approprié dans l'éditeur de salle lui-même.
  • Accrocher à la grille: Avec cette option, vous pouvez définir si une pièce aura des actifs alignés sur la grille de la pièce par défaut ou non lors de l'ouverture initiale d'une pièce. Il est activé par défaut, mais même s'il est désactivé, vous pouvez le réactiver à l'aide du bouton approprié dans l'éditeur de salle lui-même.
  • Couleur de la ligne de la grille: cette option vous permet de définir la couleur (y compris la composante alpha) des lignes de la grille de la pièce. La valeur par défaut est ( ARVB ) # 30FFFFFF.



Guides intelligents

Cette section détaille les préférences des guides intelligents pour l'éditeur de pièce. Les options disponibles sont:

  • Activer les guides intelligents: Ceci est utilisé pour activer / désactiver l'utilisation des guides intelligents dans l'éditeur de pièce. La valeur par défaut est activée.

  • Activer les règles: Cette option activera / désactivera les règles latérales dans l'éditeur de salle. La valeur par défaut est activée.

  • Aligner les règles sur les bords: lorsqu'elle est activée, cette option aligne les guides glissés dans l'éditeur de pièce depuis les règles vers tous les éléments placés dans la pièce. La valeur par défaut est activée.

  • Distance d'accrochage: ce paramètre indique à l'éditeur de pièce à quelle distance un guide doit être d'un élément avant qu'il ne «s'aligne» sur le côté de l'élément. La valeur par défaut est 6px.
