Préférences de l'éditeur de sprites

Les préférences de l'éditeur de sprites sont utilisées pour définir l'apparence et les fonctions de l' éditeur de sprites. Les options suivantes existent:

  • Origine de sprite par défaut: cette option vous permet de définir la position d'origine par défaut pour toutes les nouvelles ressources de sprite que vous créez. La valeur par défaut est le coin supérieur gauche.
  • Type de masque de collision de sprite par défaut: Cette option vous permet de définir le type de masque de collision par défaut à utiliser pour les nouveaux sprites. Les choix sont: rectangle, rectangle pivoté, ellipse, losange ou précis / précis par image. La valeur par défaut est rectangle.
  • Mode d'animation de sprite par défaut: cette option vous permet de définir le mode d'animation par défaut pour les sprites dans l'éditeur de sprite. Vous pouvez choisir entre « Images par seconde » ou « Images par image de jeu », le paramètre par défaut étant « Images par seconde ».
  • Vitesse d'animation de sprite par défaut: cette option vous permet de définir la valeur de vitesse d'animation par défaut lorsque vous créez une nouvelle ressource de sprite. Notez que cette valeur est liée au mode animation et - par exemple - 3 images par seconde n'est pas la même chose que 30 images par image de jeu, assurez-vous donc de définir correctement cette valeur en référence au mode préféré. La valeur par défaut est 30.
  • Largeur par défaut du sprite: cette option vous permet de définir la largeur par défaut de tous les nouveaux sprites créés. La valeur par défaut est 64.
  • Hauteur par défaut des sprites: cette option vous permet de définir la hauteur par défaut de tous les nouveaux sprites. La valeur par défaut est 64.
  • Appliquer le redimensionnement et l'échelle à l'origine du sprite: lorsque cette option est cochée, cette option changera l'origine de l'image-objet lorsque l'image-objet est redimensionnée ou mise à l'échelle, la nouvelle origine étant déplacée / mise à l'échelle pour correspondre aux nouvelles valeurs de l'image-objet. Notez que cela ne s'applique qu'aux origines de sprite personnalisées qui ne sont pas verrouillées dans l'éditeur de sprite. Si l'origine est verrouillée ou si le sprite utilise l'une des positions par défaut - par exemple: "Centre" ou "En haut à gauche", etc... - alors cette option sera grisée et indisponible pour la modification. La valeur par défaut est activée.

La sous-catégorie suivante existe également pour l'éditeur de sprites:

Confirmer les boîtes de dialogue

Les options disponibles ici contrôlent les messages qui s'affichent lorsqu'il est nécessaire de confirmer une action. Vous pouvez modifier ces paramètres pour afficher toujours le message ou le rejeter avec une action spécifique (Oui ou Non):

  • Réponse automatique à l'importation de fichiers: lorsque vous importez une nouvelle image et qu'il en existe déjà une, par défaut, vous serez invité à poursuivre l'importation et à écraser l'image existante ou à annuler l'opération. Cette option vous permet de supprimer ce message et d'annuler toujours l'importation en sélectionnant "Non", ou d'écraser le sprite actuel en sélectionnant "Oui". La valeur par défaut est "Afficher le message".
  • Réponse automatique en cas d'échec de chargement d'image: lorsque vous importez une nouvelle image et que l'importation échoue, un message d'avertissement s'affiche par défaut pour vous informer qu'il y a eu un échec. Cette option vous permet de supprimer ce message en sélectionnant "OK". La valeur par défaut est "Afficher le message".
