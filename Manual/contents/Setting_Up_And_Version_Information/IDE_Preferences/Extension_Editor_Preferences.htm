Préférences de l'éditeur d'extensions

Les préférences de l'éditeur d'extensions couvrent quelques options liées aux messages qui vous sont affichés lors de son utilisation, avec les détails exacts décrits ci-dessous:

  • Réponse automatique à la suppression du fichier: le paramètre par défaut ici est d'afficher un message vous demandant si vous êtes sûr de vouloir supprimer, mais vous pouvez le définir sur "OK" afin que toutes les suppressions soient effectuées sans ce message comme si vous aviez cliqué sur OK déjà.
  • Réponse automatique à la suppression du fichier proxy: comme ci-dessus, cela contrôle le message qui s'affiche lors de la suppression de tout fichier proxy.
