Préférences IDE

La fenêtre Préférences est une partie essentielle de GameMaker Studio 2, car c'est là que vous pouvez configurer et personnaliser votre expérience à l'aide de l'EDI ainsi que configurer n'importe quel SDK de plate-forme spécifique, etc.Dans cette section, nous couvrons les domaines spécifiques de l'IDE de la fenêtre des préférences - pour les plates-formes cibles individuelles, veuillez consulter la section sur les préférences de la plate-forme cible. Notez qu'après avoir modifié les préférences de cette fenêtre, vous devez cliquer sur le bouton Appliquer pour qu'elles prennent effet.

Si, après avoir modifié les options ici pour personnaliser GameMaker Studio 2 selon vos propres goûts, vous souhaitez sauvegarder les préférences que vous avez définies pour accéder au dossier:

%AppData%/GameMakerStudio2/<your_account_name>/

sous Windows, ou le dossier:

~/.config/GameMakerStudio2

sur Mac, puis copiez les deux fichiers suivants dans un endroit sûr:

  • local_settings.json
  • cloud_settings.json

Cependant, si vous souhaitez réinitialiser les préférences que vous avez modifiées, vous devez cliquer sur le bouton Restaurer les paramètres par défaut en haut de la fenêtre. Notez que cela ne restaurera que les valeurs par défaut de la section en cours d'édition.

Vous pouvez ouvrir les Préférences de GameMaker Studio 2 partir du menu Fichier en haut à gauche de l'EDI:

Une fois la fenêtre Préférences ouverte, vous aurez les différentes options suivantes (cette liste omet toutes les préférences spécifiques à la plate-forme, pour celles-ci, veuillez voir ici ):

  • Général
  • Drag and Drop™
  • Éditeur d'images
  • Éditeur de sprite
  • Éditeur d'objets
  • Éditeur de polices
  • Éditeur de salle
  • Éditeur de spline
  • Éditeur de jeu de tuiles
  • Délais
  • Éditeur d'extensions
  • Éditeurs de texte
  • Langues
  • Marketplace
  • Plugins
  • Flux d' Runtime
  • Éditeur de courbes d'animation
  • Navigateur d'actifs
  • Éditeur de séquences
