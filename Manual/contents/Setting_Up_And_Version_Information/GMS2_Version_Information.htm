Configuration de GMS2 et informations sur la version

Il existe différentes versions des licences que vous pouvez obtenir pour GameMaker Studio 2, dont le choix dépendra largement du public que vous souhaitez cibler et du fait que vous souhaitiez simplement vous amuser en créant des jeux ou que vous souhaitiez les vendre et les distribuer. les vendre et les distribuer. Vous trouverez ci-dessous les détails de chaque type de licence :

Procès

L'enregistrement d'un compte YoYo vous permet d'obtenir une licence d'essai gratuite de 30 jours pour démarrer GameMaker Studio 2, puis d'acheter différentes mises à niveau en fonction de vos besoins. La licence d'essai vous offre une grande partie de l'expérience de GameMaker Studio 2 et peut être exécutée sur un ordinateur portable. macOS ou PCWindows, mais elle est limitée à 30 jours d' utilisation. Une fois les 30 jours écoulés, vous devrez passer à l'une des licences payantes pour continuer à utiliser le produit. Notez que la licence d'essai présente les limitations suivantes :

  • Il ne permet l'accès à GMS2 que pendant 30 jours à partir de la date à laquelle vous avez activé votre compte YoYo via votre e-mail de confirmation.
  • Vous ne pouvez pas compiler un exécutable final pour n'importe quelle plateforme cible.
  • Vous n'avez pas accès aux exportations cibles "YYC", plus performantes, pour les tester.
  • Vous ne pouvez pas désactiver les notifications de l'IDE concernant l'expiration prochaine de votre licence d'essai (mais les autres notifications seront désactivées si vous désactivez les notifications dans les Préférences).
  • Vous ne pouvez pas créer un compte d'éditeur sur la place de marché (mais votre compte YoYo peut accéder à la place de marché et acheter des ressources payantes ou gratuites).


Créateur

Avec la version Creator de GameMaker Studio 2, la limite de 30 jours de la licence d'essai (voir ci-dessus) est supprimée et vous pouvez exporter vos jeux pour la plate-forme choisie, qui peut être soit macOS ou Windows. La plate-forme cible disponible dépend de la version de l'IDE que vous avez achetée ( Creator pour macOS ou Creator pour Windows) et utilisera la VM (un runner + du code interprété) pour exécuter et construire vos jeux. Pour obtenir des informations sur l'installation, consultez la section du service d'assistance sur les SDK requis. SDK requis ou la section sur les Cibles de bureau. Notez qu'il y a deux limites mineures imposées à la licence Creator :

  • Vous ne pouvez pas modifier l'écran d'accueil par défaut pour le chargement de votre jeu.
  • Vous n'avez pas accès à la possibilité de construire vos projets en utilisant les exportations cibles "YYC" (YoYo Compiler) plus performantes.


Bureau

Avec la version version Desktop de GameMaker Studio 2, toutes les limitations de la licence d'essai (voir ci-dessus) sont supprimées et vous pouvez compiler des projets vers les cibles Desktop suivantes :

  • Windows
  • macOS
  • Ubuntu(Linux)

Toutes les cibles sont fournies avec deux options de compilation, l'une est la VM qui utilise un runner et du code interprété et l'autre qui est le YYC (YoYo Compiler) qui crée du code natif et un exécutable compilé. Notez que l'option YYC nécessite une configuration supplémentaire. Pour plus d'informations, consultez la section sur SDKs requis ou la section sur Cibles de bureau.


Web

Avec la version Web de GameMaker Studio 2, toutes les limitations de la licence d'essai (voir ci-dessus) sont supprimées et vous pouvez compiler des projets vers la cible cible HTML5. Les jeux sont compilés en utilisant JavaScript. Vous pouvez trouver plus d'informations auprès du service d'assistance de YoYo Games ici : Configuration pour le Web (HTML5).


Mobile

Avec la version mobile de GameMaker Studio 2, toutes les limitations de la licence d'essai (voir ci-dessus) sont supprimées et vous pouvez compiler des projets vers les cibles suivantes :

  • Android
  • iOS
  • tvOS

Toutes les cibles sont fournies avec deux options de compilation, l'une est la VM qui utilise un runner et du code interprété et l'autre qui est le YYC (YoYo Compiler) qui crée du code natif et un exécutable compilé. Notez que l'option YYC nécessite une configuration supplémentaire. Pour plus d'informations sur la configuration, voir la section sur les SDK requis ou la section du service d'assistance sur les Cibles mobiles.


UWP

Avec la version UWP de GameMaker Studio 2, toutes les limitations de la licence d'essai (voir ci-dessus) sont supprimées et vous pouvez compiler des projets sous forme d'exécutables compatibles UWP. Ces exécutables peuvent être configurés pour fonctionner sur l'une des deux plates-formes UWP :

  • Windows 10
  • XBox One

La cible UWP est fournie avec deux options de compilation, l'une est la VM qui utilise un runner et un code interprété et l'autre qui est le YYC (YoYo Compiler) qui crée du code natif et un exécutable compilé. Notez que l'option YYC nécessite une configuration supplémentaire. Pour plus d'informations sur la configuration, voir la section sur les SDKs requis ou la section du helpdesk sur les Cibles UWP.


Consoles

Avec la Consoles de GameMaker Studio 2, toutes les limitations de la licence d'essai (voir ci-dessus) sont supprimées et vous pouvez compiler des projets pour une ou plusieurs des plateformes cibles suivantes :

  • XBox One
  • PlayStation 4
  • Nintendo Switch

Chacune des cibles de la console est fournie avec deux options de compilation, l'une est la VM qui utilise un runner et du code interprété et l'autre qui est le YYC (YoYo Compiler) qui crée du code natif et un exécutable compilé. Notez que l'option YYC nécessite une configuration supplémentaire. Pour plus d'informations sur la configuration, consultez la section sur les SDKs requis.

IMPORTANT ! En raison des restrictions liées aux différents accords de confidentialité associés aux exportations de consoles, le manuel de GameMaker Studio 2 ne contient aucune information sur la configuration ou la programmation de ces cibles. Une fois que vous êtes un développeur pour la plateforme choisie, veuillez contacter le service d'assistance via votre compte compteYoYo et demandez l'accès à la documentation appropriée.


Éducation

GameMaker Studio 2 peut également être utilisé dans un environnement scolaire ou universitaire. Pour plus d'informations à ce sujet, veuillez contacter YoYo Games via le portail éducatif de YoYo Games.
