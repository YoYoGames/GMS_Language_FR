Objets et instances

Avec les ressources que vous avez vues jusqu'à présent, vous pouvez ajouter de belles images et de beaux sons dans le navigateur de ressources du jeu, mais elles ne font encore rien dans le jeu lui-même ! Pour cela, nous avons besoin de l'actif le plus important de tous dans GameMaker Studio 2, l'actif Objet . Les objets sont des ressources spéciales que nous utilisons pour contrôler certains aspects d'un jeu et pour faire des choses spécifiques. La plupart du temps, un sprite leur est associé afin que vous puissiez les voir dans les salles de jeu. Mais parfois, ils sont utilisés comme des objets de contrôle "en coulisses" pour faire des choses comme suivre les statistiques, ou définir les temps pour les cut-scenes, etc... On peut donner des comportements aux objets et ils peuvent réagir à certains événements ainsi qu'entre eux, et la plupart des choses que vous voyez dans un jeu sont basées sur les objets et leurs comportements. un jeu sont basées sur les objets et leurs interactions avec l'utilisateur et entre eux.

On ne peut pas parler d'objets sans parler d'instances . Dans un jeu réalisé avec GameMaker Studio 2, les personnages, les monstres, les balles, les murs, etc... que vous voyez dans le jeu sont en fait tous considérés comme des instances. Une instance est essentiellement une "copie" d'un objet que l'on trouve dans l'Asset Browser et que l'on place dans une pièce (pensez aux objets comme à des modèles d 'instances). Ainsi, lorsque nous parlons de quelque chose qui affecte ou modifie une instance, nous voulons dire qu'une copie particulière d'un objet dans une room est affectée alors que toutes les autres ne le sont pas, mais lorsque nous parlons d'affecter ou de modifier un objet, nous voulons dire que tout ce que nous faisons sur lui sera reflété dans toutes les instances créées à partir de ce point aussi. à partir de ce point. Ainsi, l'objet est le modèle de l'instance, et l'instance est ce que nous plaçons dans une salle pour créer notre jeu.

Les événements sont des moments discrets dans la boucle du jeu où les choses se produisent en fonction de ce que vous avez programmé pour elles.
